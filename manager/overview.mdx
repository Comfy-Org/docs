**ComfyUI-Manager** is an extension designed to enhance the usability of [ComfyUI](https://github.com/comfyanonymous/ComfyUI). It offers management functions to **install, remove, disable, and enable** various custom nodes of ComfyUI. Furthermore, this extension provides a hub feature and convenience functions to access a wide range of information within ComfyUI.

![ComfyUI-Manager dialog](https://raw.githubusercontent.com/ltdrdata/ComfyUI-extension-tutorials/refs/heads/Main/ComfyUI-Manager/images/dialog.jpg)

## Key features

- **Custom node management**: Install, update, remove, and disable custom nodes
- **Model management**: Download and manage models from various sources
- **Snapshot management**: Save and restore installation states
- **Workflow sharing**: Share workflows to platforms like ComfyWorkflows, OpenArt, and YouML
- **Missing node detection**: Automatically detect and install missing nodes from workflows
- **Component sharing**: Copy, paste, and drag-drop workflow components

## UI modes

ComfyUI-Manager has two UI modes:

- **New UI**: The default modern interface integrated with ComfyUI
- **Legacy UI**: The original manager interface for users who prefer the classic experience

You can switch between modes using the `--enable-manager-legacy-ui` flag when starting ComfyUI.

## Sharing workflows

Share workflows by clicking the **Share** button at the bottom of the main menu.

![Share dialog](https://raw.githubusercontent.com/ltdrdata/ComfyUI-extension-tutorials/Main/ComfyUI-Manager/images/share.jpg)

Supported platforms:
- [ComfyWorkflows](https://comfyworkflows.com/)
- [OpenArt](https://openart.ai/workflows/dev)
- [YouML](https://youml.com)
- Matrix channel

You can configure share behavior in the Manager menu under **Share settings**:
- `None`: Hide from main menu
- `All`: Show a dialog where you can select a title for sharing

## Component sharing

ComfyUI-Manager supports sharing workflow components through copy/paste and drag-drop.

**Copy & Paste**: Components can be pasted from the clipboard in JSON format. See the [demo page](https://ltdrdata.github.io/component-demo/) for examples.

**Drag & Drop**: Drag and drop `.pack` or `.json` files to add components.

## Double-click node title

You can configure double-click behavior in the Manager menu:

| Option | Description |
|--------|-------------|
| `Copy All Connections` | Copies all connections from the nearest node (within 1000 pixels) |
| `Copy Input Connections` | Copies only input connections from the nearest node |
| `Possible Input Connections` | Connects all outputs that match the closest type from nodes on the left |
| `Possible(left) + Copy(right)` | Left half of title uses "Possible Input", right half uses "Copy All" |
