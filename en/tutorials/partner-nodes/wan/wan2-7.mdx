Wan2.7 is Alibaba's latest video generation model, now available in ComfyUI through Partner Nodes. It represents a comprehensive upgrade from version 2.6, delivering significant improvements in quality, audio, motion dynamics, stylization, and consistency.

This release brings a fully upgraded multimodal video pipeline directly into your node graph, supporting text, image, audio, and video inputs across five task types.

## Key Features

- **Image-to-Video**: First-frame, first+last frame, and audio-driven generation
- **Text-to-Video**: Pure text prompt generation with optional audio input and multi-shot narration
- **Video Continuation**: Extend existing clips with new content guided by text prompts
- **Reference-to-Video**: Reference both visual appearance and voice characteristics of subjects; supports up to 5 real-person inputs and multi-character interaction
- **Video Editing**: Edit or clone videos via text prompts, reference images, or style transfer

## Highlights

- Supports up to **5 real-person image inputs** for multi-character scenes
- Voice reference for consistent audio-visual identity
- 3x3 grid-based image generation
- Significant improvements in motion dynamics, stylization, and consistency over Wan2.6
