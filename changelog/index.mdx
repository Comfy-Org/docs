# ComfyUI v0.3.40

New features and improvements for enhanced workflow capabilities

• Added new ImageStitch node for concatenating images and GetImageSize node for retrieving image dimensions
• Introduced regex replace operation node for advanced text processing workflows
• Enhanced frontend with Help Menu in Node Library and multiple stability fixes
• Improved ROCm support and custom node compatibility with pyproject.toml configuration
• Fixed security vulnerability (XSS) and deprecated Pillow API usage for better stability
</Update>
