<Update label="v0.3.41" description="December 19, 2024">
# ComfyUI v0.3.41

Major update featuring new AI model support, improved GPU performance, and enhanced training capabilities

• **New Model Support**: Added support for Cosmos predict2 text-to-image (2B and 14B) and image-to-video models with enhanced memory optimization
• **Enhanced GPU Performance**: Enabled FP8 operations by default on AMD gfx1201 GPUs and improved PyTorch attention for better performance
• **Advanced Training Features**: Introduced LoRA Trainer node for weight adapter scheme training with improved filename formatting
• **Improved Sampling**: Added new multistep DPM++ SDE samplers for Rectified Flow models and generalized SEEDS samplers
• **Bug Fixes**: Resolved black image issues with some Flux models in FP16 mode and fixed WebcamCapture signature problems
</Update>
