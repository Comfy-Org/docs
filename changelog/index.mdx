<Update label="v0.3.50" description="August 13, 2025">

**Advanced Model Integration & Performance Enhancements**

This release significantly expands ComfyUI's model ecosystem with enhanced Qwen support, async API capabilities, and critical stability improvements for complex workflows:

### Qwen Model Ecosystem
- **Enhanced Qwen Image Model Support**: Comprehensive integration improvements including proper LoRA loading and model merging capabilities, enabling more sophisticated vision workflows with Qwen-based models
- **Qwen Model Merging Node**: New dedicated node for merging Qwen image models, allowing creators to combine different model strengths for optimized generation results

- **Async API Nodes**: Revolutionary introduction of asynchronous API nodes, enabling non-blocking workflow execution for better performance with external service integrations
- **Improved Memory Handling**: Enhanced RepeatLatentBatch node now properly handles multi-dimensional latents, fixing critical workflow interruptions for batch processing operations
- **WAN 2.2 Fun Control Support**: Added comprehensive support for WAN 2.2 fun control features, expanding creative control options for video generation workflows

### Hardware Optimization & Compatibility
- **AMD GPU Improvements**: Enhanced AMD Radeon support with improved FP16 accuracy handling and better performance optimization for ROCM users
- **RDNA3 Architecture Fixes**: Resolved specific issues with gfx1201 GPUs when using Flux models with PyTorch attention, improving stability for latest AMD hardware
- **Updated PyTorch Support**: Bumped CUDA and ROCM PyTorch versions with comprehensive testing on Python 3.13 and CUDA 12.9

### Developer Experience Enhancements
- **Cleaner Logging**: Feature flags now only display in verbose mode, reducing console noise while maintaining debugging capabilities
- **Audio Processing Safety**: Enhanced torchaudio import safety checks prevent crashes when audio dependencies are unavailable
- **Kling API Improvements**: Fixed image type parameter handling in Kling Image API nodes for more reliable external service integration

### Workflow Benefits
- **Async Workflow Execution**: New async API capabilities enable more responsive workflows when integrating external services or APIs
- **Enhanced Model Flexibility**: Expanded Qwen support allows for more diverse vision-language workflows with improved LoRA compatibility
- **Better Hardware Utilization**: AMD GPU optimizations and updated PyTorch support improve performance across diverse hardware configurations
- **Improved Batch Processing**: Fixed RepeatLatentBatch ensures reliable operation with complex multi-dimensional data structures
- **Professional Video Control**: WAN 2.2 fun control features provide advanced creative control for professional video generation workflows

This release strengthens ComfyUI's position as a comprehensive AI workflow platform, particularly benefiting creators working with cutting-edge vision models, external APIs, and diverse hardware configurations while improving overall system stability and performance.
