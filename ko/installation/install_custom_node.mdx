## 커스텀 노드란 무엇인가요?

커스텀 노드는 ComfyUI의 확장 기능으로, 고급 이미지 처리, 머신러닝 미세조정, 색상 조정 등 새로운 기능을 추가합니다. 커뮤니티에서 개발한 이러한 노드들은 ComfyUI의 핵심 기능을 크게 확장할 수 있습니다.

<Warning>
커스텀 노드를 설치하기 전에 주의 깊게 검토하는 것이 중요합니다. ComfyUI는 오픈소스 프로젝트이므로 악성 플러그인이 커스텀 노드를 악용할 가능성이 있습니다:
1. 신뢰할 수 있는 저자와 커뮤니티에서 일반적으로 사용되는 커스텀 노드만 설치하세요.
2. 플러그인의 기능을 이해한 후 설치하고, 알 수 없는 출처는 피하여 시스템 보안을 유지하세요.
3. 낯선 또는 의심스러운 플러그인은 설치하지 마세요—검증되지 않은 플러그인은 시스템 침해로 이어질 수 있는 보안 위험을 초래할 수 있습니다.
</Warning>

모든 커스텀 노드 설치에는 다음 두 단계를 완료해야 합니다:

1. 노드 코드를 `ComfyUI/custom_nodes` 디렉터리에 복제하기
2. 필요한 파이썬 종속성을 설치하기

이 가이드에서는 세 가지 설치 방식을 비교합니다. **ComfyUI 매니저**는 대부분의 설정에서 현재 ComfyUI 릴리스에 내장되어 있습니다—무엇을 하는지와 활성화 방법은 [ComfyUI 매니저 개요](/ko/manager/overview)를 참조하세요. 노드가 레지스트리에 없거나 특정 버전이 필요할 때는 **Git 복제** 또는 **ZIP**를 사용하세요.

| 방법                        | 장점                                                    | 단점                                                |
| ---------------------------- | ------------------------------------------------------------- | ----------------------------------------------------------- |
| **ComfyUI 매니저**(권장) | 1. 자동 설치<br/>2. 종속성 관리<br/>3. GUI 인터페이스 | 레지스트리에 등록되지 않은 노드를 직접 검색할 수 없음 |
| **Git 복제**                | 레지스트리에 등록되지 않은 노드도 설치 가능               | 1. Git 지식 필요<br/>2. 수동 종속성 관리<br/>3. 설치 위험 |
| **레포지토리 ZIP 다운로드**   | 1. Git 필요 없음<br/>2. 수동 제어                      | 1. 수동 종속성 관리<br/>2. 버전 관리 없음<br/>3. 설치 위험 |

팁: 커스텀 노드를 설치하기 전에 플러그인의 README 파일을 확인하여 설치 방법, 사용법, 특정 모델이나 종속성 버전 요구사항, 일반적인 문제 해결 방법 등을 숙지하세요.

## 방법 1: ComfyUI 매니저(권장)

<Note>
이 방법을 사용하려면 환경에 ComfyUI 매니저가 있어야 합니다. 없거나 비활성화된 경우 [ComfyUI 매니저 개요](/ko/manager/overview)와 [설치 가이드](/ko/manager/install)를 참고하세요.
</Note>

ComfyUI 매니저는 커스텀 노드를 관리하는 두 가지 UI 버전을 제공합니다. 인터페이스에 맞는 가이드를 선택하세요:

<Columns cols={2}>
  <Card title="새 UI(데스크탑 기본)" icon="sparkles" href="/ko/manager/pack-management">
    데스크탑 사용자를 위해 기본적으로 활성화된 새 노드 관리 인터페이스입니다.
  </Card>
  <Card title="레거시 UI" icon="rectangle-history" href="/ko/manager/legacy-ui">
    커스텀 노드 설치를 위한 클래식 매니저 인터페이스입니다.
  </Card>
</Columns>

## 방법 2: Git를 이용한 수동 설치

매니저에 없는 새로운 노드나 특정 버전이 필요한 경우에 적합합니다. 시스템에 [Git](https://git-scm.com/)가 설치되어 있어야 합니다.

<Steps>
  <Step title="레포지토리 URL 얻기">
    GitHub에서 "코드" 버튼을 클릭하고 HTTPS 링크를 복사하세요
  </Step>
  <Step title="custom_nodes 디렉터리로 이동">
 ```bash
 cd /path/to/ComfyUI/custom_nodes
 ```
  </Step>
  <Step title="레포지토리 복제">
 ```bash
 git clone [레포지토리 URL]
 ```
  </Step>
  <Step title="종속성 설치">
  종속성은 ComfyUI 환경에 설치해야 합니다—시스템 환경과 혼동되지 않도록 주의하세요
   <Tabs>
    <Tab title="Windows 포터블">
    Windows 포터블 버전의 경우 내장된 Python 환경에 종속성을 설치하세요
      ```bash
      python_embeded\python.exe -m pip install -r ComfyUI\custom_nodes\[노드 디렉터리]\requirements.txt
      ```
    </Tab>
    <Tab title="수동 설치">
    ComfyUI 환경에 종속성을 설치하세요
      ```bash
      cd [노드 디렉터리]
      pip install -r requirements.txt
      ```
    </Tab>
  </Tabs>
  </Step>
  <Step title="ComfyUI 재시작 및 브라우저 새로고침">
    ComfyUI를 재시작하고 브라우저를 새로고침하세요. 시작 로그를 확인해 'import failed' 오류가 있는지 확인하세요
  </Step>
</Steps>

## 방법 3: ZIP 다운로드 설치
Git나 매니저를 사용할 수 없는 사용자에게 적합합니다

<Warning>
이 설치 방법은 버전 관리 기능을 잃게 되므로 권장하지 않습니다
</Warning>

<Steps>
  <Step title="GitHub에서 '코드' → 'ZIP 다운로드' 클릭">
    GitHub 페이지에서 '코드' → 'ZIP 다운로드'를 클릭하세요
    ![GitHub에서 코드 → ZIP 다운로드 클릭](/images/installation/custom_nodes/install-custom-nodes-by-zip.jpg)
  </Step>
  <Step title="ZIP 파일 압축 해제">
    다운로드한 ZIP 파일을 압축 해제하세요
  </Step>
  <Step title="압축 해제된 폴더를 'ComfyUI/custom_nodes/' 디렉터리로 복사">
    압축 해제된 폴더를 'ComfyUI/custom_nodes/' 디렉터리로 복사하세요
  </Step>
  <Step title="수동으로 종속성 설치(위의 Git 방법 4단계와 동일)">
    ComfyUI를 재시작하고 브라우저를 새로고침하세요
  </Step>
  <Step title="성공적인 설치 확인">
    재시작 후 ComfyUI 매니저를 확인해 플러그인이 성공적으로 설치되었으며 'import failed' 오류가 없는지 확인하세요
  </Step>
</Steps>

## 커스텀 노드 리소스

ComfyUI에서는 기본 노드 확장 기능 외에도 커스텀 노드는 다음과 같은 추가 리소스를 포함할 수 있습니다:
- [노드 문서](/ko/custom-nodes/help_page): 모든 커스텀 노드와 기본 노드를 지원합니다. 노드 문서를 보고 노드의 목적과 사용법을 이해하며, 작성자에게 PR을 통해 문서를 기여할 수 있습니다.
- [커스텀 노드 워크플로우 템플릿](/ko/custom-nodes/workflow_templates): 노드 작성자가 제공하는 예시 워크플로우 템플릿으로, ComfyUI 템플릿에서 탐색하고 불러올 수 있습니다.
- [다국어 지원](/ko/custom-nodes/i18n)

커스텀 노드 개발자라면 이러한 리소스를 추가해 사용자 친화성을 높일 수 있습니다.
