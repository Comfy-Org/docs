**ComfyUI 데스크톱**은 일반 소프트웨어처럼 설치할 수 있는 독립형 설치 버전입니다. **파이썬 환경과 의존성**의 빠른 설치와 자동 구성이 지원되며, 기존 ComfyUI 설정, 모델, 워크플로우 및 파일을 원클릭으로 가져올 수 있습니다. 기존 [ComfyUI 포터블 버전](/installation/comfyui_portable_windows)에서 데스크톱 버전으로 빠르게 이전할 수 있습니다.

ComfyUI 데스크톱은 오픈소스 프로젝트이며, 전체 코드는 [여기](https://github.com/Comfy-Org/desktop)를 방문해 주세요.

ComfyUI 데스크톱 하드웨어 요구사항:
- NVIDIA GPU (CUDA 지원)
- AMD GPU (ROCm 지원)

이 튜토리얼에서는 소프트웨어 설치 과정을 안내하고 관련 구성 세부사항을 설명합니다.

<Warning>**ComfyUI 데스크톱**은 아직 **베타** 상태이므로 실제 설치 과정은 변경될 수 있습니다</Warning>

## ComfyUI 데스크톱 제거 방법

**ComfyUI 데스크톱**은 Windows 설정의 시스템 제거 기능을 이용해 소프트웨어 제거를 완료할 수 있습니다.

![ComfyUI 데스크톱 제거](/images/desktop/win-uninstall-comfyui.jpg)

모든 **ComfyUI 데스크톱** 파일을 완전히 제거하려면 다음 폴더를 수동으로 삭제할 수 있습니다:
- C:\Users\<YOUR_USERNAME>\AppData\Local\@comfyorgcomfyui-electron-updater
- C:\Users\<YOUR_USERNAME>\AppData\Local\Programs\@comfyorgcomfyui-electron
- C:\Users\<YOUR_USERNAME>\AppData\Roaming\ComfyUI

위 작업은 다음 폴더를 삭제하지 않습니다. 만약 해당 파일을 삭제해야 한다면 수동으로 삭제하세요:
- 모델 파일
- 커스텀 노드
- 입력/출력 디렉토리

## 문제 해결

### 지원되지 않는 장치 표시

![ComfyUI 설치 단계 - 지원되지 않는 장치](/images/desktop/win-comfyui-desktop-0.jpg)

ComfyUI 데스크톱은 Windows에서 **NVIDIA GPU**와 **AMD GPU**를 지원합니다. 사용하시는 장치가 지원되는 GPU 유형이 아닐 경우 이 화면이 나타날 수 있습니다.
- 지원되는 장치로 교체해 주세요.
- 또는 [ComfyUI 포터블](/installation/comfyui_portable_windows)이나 [수동 설치](/installation/manual_install)를 통해 ComfyUI를 사용해 보세요.

### 백신이 `uv.exe`를 악성코드로 표시

ComfyUI 데스크톱은 파이썬 환경 관리를 위해 [`uv`](https://github.com/astral-sh/uv) (Astral에서 제공)를 번들로 포함하고 있습니다. 일부 백신 제품—예를 들어 **360 Total Security**, **Windows Defender**, **BitDefender**, **DeepInstinct**—는 간혹 `uv.exe`를 악성코드로 표시합니다. 이는 ComfyUI가 아닌 `uv` 자체에 영향을 미치는 **알려진 오진**입니다.

**증상:**
- 백신이 설치 또는 업데이트 중 `uv.exe`를 격리하거나 삭제함
- 파이썬 환경 설정 중간에 설치가 실패함
- 백신 경고 이후 "파이썬 환경 설치 실패" 또는 유사한 오류가 발생함

**해결책:** ComfyUI 설치 디렉토리를 백신의 신뢰 목록/화이트리스트에 추가한 후 설치를 다시 진행하거나 복구하세요. 화이트리스트에 추가할 디렉토리는 다음과 같습니다:

- 선택한 ComfyUI 설치 디렉토리 (설치 단계에서 선택한 대로)
- `C:\Users\<YOUR_USERNAME>\AppData\Local\Programs\@comfyorgcomfyui-electron`
- `C:\Users\<YOUR_USERNAME>\AppData\Roaming\ComfyUI`

**참고:** `uv` 프로젝트에서 상위 버전으로 추적됨—[astral-sh/uv#1587](https://github.com/astral-sh/uv/issues/1587), [#10079](https://github.com/astral-sh/uv/issues/10079), [#12621](https://github.com/astral-sh/uv/issues/12621), 그리고 [#15011](https://github.com/astral-sh/uv/issues/15011) 참조.
