**Comfy Desktop**은 하나의 런처에서 여러 ComfyUI 인스턴스를 실행할 수 있는 멀티 설치 관리자입니다. 이 가이드에서는 Windows에서 Comfy Desktop을 설치하고 제거하는 방법을 다룹니다.

## 시스템 요구 사항

- **OS:** Windows 10 이상
- **아키텍처:** x64 또는 ARM64
- **GPU:** 전용 GPU(NVIDIA / AMD)를 권장하지만, 성능이 충분하다면 없어도 사용 가능
- **디스크 공간:** 설치당 최소 4.85GB 이상 권장

## 설치

다운로드한 `.exe` 설치 파일(NSIS 인스톨러)을 실행하고 안내에 따라 진행하세요. 설치가 끝나면 시작 메뉴나 바탕화면 바로가기를 통해 Comfy Desktop을 실행합니다.

![Windows 설정](/images/desktop/installation/windows_1_install.png)
앱을 처음 열면 **시작** 화면이 나타나며, 여기서 첫 번째 설치를 생성할 수 있습니다.

## 제거

### Comfy Desktop 제거

1. **설정 → 앱 → 설치된 앱**으로 이동합니다.
2. **Comfy Desktop**을 찾아 **제거**를 클릭합니다.

![Windows에서 Comfy Desktop 제거](/images/desktop/installation/windows_2_uninstall.png)

### 앱 제거 시 삭제되지 않는 항목

Comfy Desktop을 제거하면 앱 자체만 삭제됩니다. 다음 데이터는 **제거되지 않으므로** 완전히 정리하려면 수동으로 삭제해야 합니다.

| 데이터 | 위치 | 설명 |
|------|----------|-------------|
| 설치 | `%USERPROFILE%\ComfyUI-Installs` | 생성한 각 ComfyUI 설치(앱 내에서도 삭제 가능) |
| 공유 모델 및 출력 | `%USERPROFILE%\ComfyUI-Shared` | 공유 모델 라이브러리, 입력/출력 디렉터리 |
| 앱 설정 및 레지스트리 | `%APPDATA%\Comfy Desktop` | 설정, 설치 기록, 로그 |

모든 항목을 완전히 제거하려면:

1. **앱 데이터 폴더 삭제** – `%APPDATA%\Comfy Desktop`
2. 앱 내에서 **개별 설치를 삭제**하거나 `%USERPROFILE%\ComfyUI-Installs`를 수동으로 삭제
3. `%USERPROFILE%\ComfyUI-Shared`에 있는 **공유 모델 라이브러리**를 삭제(해당 파일이 더 이상 필요하지 않은 경우)
