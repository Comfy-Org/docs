**ComfyUI 데스크톱**은 일반 소프트웨어처럼 설치할 수 있는 독립형 설치 버전입니다.  
빠른 설치와 **파이썬 환경 및 종속성의 자동 구성**을 지원하며, 기존 ComfyUI 설정, 모델, 워크플로우 및 파일을 원클릭으로 가져올 수 있습니다.

ComfyUI 데스크톱은 오픈소스 프로젝트로, 전체 코드는 [여기](https://github.com/Comfy-Org/desktop)에서 확인해 주세요.

<Note>ComfyUI 데스크톱(MacOS)은 Apple Silicon만 지원합니다</Note>

이 튜토리얼에서는 소프트웨어 설치 과정을 안내하고 관련 구성 세부사항을 설명합니다.

<Warning>ComfyUI 데스크톱은 아직 **베타** 상태이므로 실제 설치 과정이 변경될 수 있습니다</Warning>

## Homebrew를 통한 설치

ComfyUI 데스크톱은 [Homebrew](https://brew.sh/)를 통해서도 설치할 수 있습니다:

```
brew install comfyui
```

## ComfyUI 데스크톱 설치 단계

다운로드한 설치 패키지 파일을 더블클릭하세요. 이미지에 표시된 대로, 화살표에 따라 **ComfyUI** 애플리케이션을 **응용프로그램** 폴더로 드래그하세요.

![ComfyUI 설치 패키지](/images/desktop/mac-comfyui-desktop-0.png)

설치 패키지를 열었을 때 아이콘에 금지 표시가 나타나는 경우, 현재 시스템 버전이 **ComfyUI 데스크톱**과 호환되지 않는다는 의미입니다.

![ComfyUI 로고](/images/desktop/mac-comfyui-desktop-0-1.png)

그런 다음 **Launchpad**에서 **ComfyUI 아이콘**을 찾아 클릭하여 ComfyUI 초기화 설정으로 들어가세요.

![ComfyUI Launchpad](/images/desktop/mac-comfyui-desktop-1.jpg)

   세 가지 옵션은 다음과 같습니다:
   1. **MPS(권장):** Metal Performance Shaders(MPS)는 Apple의 프레임워크로, Apple 디바이스에서 GPU를 활용해 컴퓨팅 및 머신러닝 작업을 가속화하며 PyTorch와 같은 프레임워크를 지원합니다.
   2. **수동 구성:** 파이썬 런타임 환경을 수동으로 설치하고 구성해야 합니다. 구성 방법을 잘 알고 있지 않다면 이 옵션을 선택하지 마세요.
   3. **CPU 모드 활성화:** 개발자 및 특별한 경우에만 선택하세요. 꼭 필요하다는 확신이 없으면 선택하지 마세요.

    특별한 상황이 아니라면, 아래와 같이 **MPS**를 선택하고 **다음**을 클릭해 진행하세요.

  </Step>
  <Step title="설치 위치">
  ![ComfyUI 설치 단계 - 설치 위치](/images/desktop/mac-comfyui-desktop-4.png)

  이 단계에서는 ComfyUI의 다음 관련 내용에 대한 설치 위치를 선택하게 됩니다:
  - **파이썬 환경**
  - **모델 파일**
  - **커스텀 노드 커스텀 노드**

  권장 사항:
  - ComfyUI의 설치 디렉터리를 별도의 빈 폴더로 생성하세요.
  - 최소 **5G**의 디스크 공간이 있어야 **ComfyUI 데스크톱**의 정상적인 설치가 보장됩니다.

  <Note>모든 파일이 이 디렉터리에 설치되는 것은 아니며, 일부 파일은 MacOS 시스템 디렉터리에 저장됩니다. 이 가이드의 제거 섹션을 참고해 ComfyUI 데스크톱 버전의 제거를 완료할 수 있습니다</Note>

  </Step>
  <Step title="기존 설치에서 이전(선택사항)">
  ![ComfyUI 설치 단계 - 파일 이전](/images/desktop/mac-comfyui-desktop-5.png)

  이 단계에서는 기존 ComfyUI 설치 내용을 ComfyUI 데스크톱으로 이전할 수 있습니다. 기존 ComfyUI 설치 디렉터리를 선택하면 설치 프로그램이 자동으로 인식합니다:
  - **사용자 파일**
  - **모델:** 복사되지 않고 데스크톱 버전과 링크됩니다.
  - **커스텀 노드:** 노드가 다시 설치됩니다.

  걱정하지 마세요. 이 단계에서는 모델 파일이 복사되지 않습니다. 필요에 따라 옵션을 체크하거나 해제할 수 있습니다. **다음**을 클릭해 계속 진행하세요.

  </Step>
  <Step title="데스크톱 설정">
  ![ComfyUI 설치 단계 - 데스크톱 설정](/images/desktop/mac-comfyui-desktop-6.png)

   다음은 기본 설정입니다:
   1. **자동 업데이트:** ComfyUI 업데이트가 있을 때 자동 업데이트를 설정할지 여부
   2. **사용 통계:** 활성화되면 **익명의 사용 데이터**를 수집해 ComfyUI 개선에 도움을 줍니다.
   3. **미러 설정:** 프로그램이 인터넷에 접속해 파이썬을 다운로드하고 환경 설치를 완료해야 하므로, 설치 중 빨간색 ❌ 표시가 나타나 설치 실패를 유발할 수 있다면 아래 단계를 따르세요.

## ComfyUI 데스크톱 제거 방법

**ComfyUI 데스크톱**의 경우, **응용프로그램** 폴더에서 **ComfyUI**를 바로 삭제할 수 있습니다.

모든 **ComfyUI 데스크톱** 파일을 완전히 제거하려면 다음 폴더를 수동으로 삭제할 수 있습니다:
```
~/Library/Application Support/ComfyUI
```

위 작업은 다음 폴더를 삭제하지 않습니다. 해당 파일을 삭제하려면 수동으로 삭제하세요:
- 모델 파일
- 커스텀 노드
- 입력/출력 디렉터리
