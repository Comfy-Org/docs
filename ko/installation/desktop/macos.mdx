**Comfy Desktop**은 하나의 런처로 여러 ComfyUI 인스턴스를 실행할 수 있는 다중 설치 관리자입니다. 이 가이드는 macOS에서 Comfy Desktop을 설치하고 제거하는 방법을 다룹니다.

## 시스템 요구 사항

- **OS:** macOS 13 (Ventura) 이상
- **하드웨어:** Apple Silicon (M1 이상)
- **디스크 공간:** 설치당 최소 4.85GB 권장

<Card title="Comfy Desktop 다운로드" icon="download" href="https://comfy.org/download">
    macOS용 최신 설치 프로그램을 다운로드하세요. 페이지가 자동으로 사용 중인 플랫폼을 감지합니다.
</Card>

## 설치

1. 다운로드한 `.dmg` 파일을 엽니다.
2. **Comfy Desktop**을 **응용 프로그램** 폴더로 드래그합니다.
3. Launchpad 또는 Spotlight에서 실행합니다.

![Desktop MacOS](/images/desktop/installation/mac_1_install.png)
처음 실행 시 macOS에서 보안 경고가 표시될 수 있습니다. **시스템 설정 → 개인정보 보호 및 보안**으로 이동하여 **그냥 열기**를 클릭해 허용하세요.

## 제거

### Comfy Desktop 제거

1. Finder에서 **응용 프로그램** 폴더를 엽니다.
2. **Comfy Desktop**을 마우스 오른쪽 버튼으로 클릭하고 **휴지통으로 이동**을 선택합니다.

![macOS에서 Comfy Desktop 제거하기](/images/desktop/installation/mac_2_uninstall.png)

### 앱 제거가 제거하지 않는 항목

Comfy Desktop을 제거하면 런처 앱 자체만 제거됩니다. 다음 데이터는 **제거되지 않으며** 완전한 정리를 원할 경우 수동으로 삭제해야 합니다:

| 데이터 | 기본 위치 | 설명 |
|------|----------|-------------|
| 설치본 | `~/ComfyUI-Installs` | 생성한 각 ComfyUI 설치본 (앱 내에서도 삭제 가능). 사용자 지정 설치 경로도 제거되지 않습니다. |
| 공유 모델 및 출력 | `~/ComfyUI-Shared` | 공유 모델 라이브러리, 입력/출력 디렉토리 |
| 앱 설정 및 레지스트리 | `~/Library/Application Support/Comfy Desktop` | 설정, 설치 기록, 로그 |

<Warning>위에 나열된 디렉터리에는 **개인 ComfyUI 데이터** — 모델, 출력물, 설정 및 설치본이 포함되어 있습니다. 이들은 Comfy Desktop 애플리케이션 파일이 **아닙니다**. 삭제하면 데이터가 영구적으로 제거됩니다.</Warning>

모든 것을 완전히 제거하려면:

1. **앱 데이터 폴더 삭제** – `~/Library/Application Support/Comfy Desktop`
2. 앱 내에서 **개별 설치본을 삭제**하거나 `~/ComfyUI-Installs`를 수동으로 삭제합니다.
3. `~/ComfyUI-Shared`의 **공유 모델 라이브러리 삭제** (이 파일들이 더 이상 필요하지 않은 경우에만)
