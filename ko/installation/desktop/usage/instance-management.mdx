Comfy Desktop에서는 선택기 화면에서 여러 ComfyUI 인스턴스를 관리할 수 있습니다.

### 인스턴스 유형

Comfy Desktop은 네 가지 인스턴스 유형을 지원합니다. 각 유형은 카드와 제목 표시줄에 고유한 아이콘을 표시합니다.

| 유형 | 아이콘 | 설명 |
|------|------|-------------|
| **독립형** | 💻 | Comfy Desktop이 생성한 자체 완비형 ComfyUI 인스턴스입니다. 번들 Python, 자동 업데이트, 전체 스냅샷 지원이 포함됩니다. **새 설치**를 클릭했을 때의 기본값입니다. |
| **클라우드** | ☁️ | Comfy Cloud 인스턴스입니다. Comfy Cloud 계정이 필요합니다. 클라우드 인스턴스는 Comfy 서버에서 실행되며 별도로 청구됩니다. |
| **원격** | 🌐 | 네트워크의 다른 컴퓨터에서 실행되는 ComfyUI 인스턴스에 연결합니다. 설정 중에 원격 호스트와 포트를 구성합니다. |
| **추적형** | — | 기존 ComfyUI 설치를 추적합니다(Git 클론 또는 포터블). **독립형**과 동일한 아이콘을 사용하며, 추적형 인스턴스를 위한 별도의 아이콘은 없습니다. 업데이트는 수동으로 관리합니다. [기존 설치 추적하기](#tracking-an-existing-installation)를 참조하세요. |

아래 이미지는 각 인스턴스 카드에서 사용할 수 있는 주요 작업을 표시합니다.

![인스턴스 관리 작업](/images/desktop/interface/interface_instance_menu.png)

1. **새 설치**: 완전히 새로운 ComfyUI 인스턴스를 만듭니다. [새 인스턴스 생성하기](#creating-a-new-instance)를 참조하세요. [스냅샷 파일에서 인스턴스 생성하기](/ko/installation/desktop/usage/snapshots#create-an-instance-from-a-snapshot-file)도 가능합니다.
2. **⋮ 메뉴**: 세 점 아이콘을 클릭하여 인스턴스 작업 메뉴를 엽니다.
3. **관리**: 관리 패널을 열어 인스턴스를 구성합니다(경로, 실행 인자, 모델 디렉터리, 환경 변수). [관리](/ko/installation/desktop/usage/manage) 페이지를 참조하세요.
4. **스냅샷 복원**: 인스턴스를 이전 스냅샷으로 되돌립니다. [스냅샷](/ko/installation/desktop/usage/snapshots) 페이지를 참조하세요.
5. **공유**: 인스턴스의 스냅샷을 내보내 다른 사람과 공유합니다.
6. **설치 복사**: 기존 독립형 설치를 자체 Python 환경과 커스텀 노드를 포함하여 복제합니다. [설치 복사하기](#copying-an-installation)를 참조하세요.
7. **Forget**: 파일을 삭제하지 않고 선택기에서 인스턴스를 제거합니다(Untrack과 동일).
8. **Uninstall**: 인스턴스와 모든 파일을 디스크에서 영구히 삭제합니다.
   Forget과 Uninstall에 대한 자세한 내용은 [인스턴스 제거하기](#uninstalling-an-instance)를 참조하세요.

## 새 인스턴스 생성하기

선택기 화면에서 **+** 카드(왼쪽 상단)를 클릭하여 새 ComfyUI 인스턴스를 추가합니다.

<Steps>
  <Step title="Comfy Desktop 구성하기">

<Tabs>
  <Tab title="Windows">
    ![Windows에서 Comfy Desktop 구성](/images/desktop/usage/new_install_01_settings_windows.png)
  </Tab>
  <Tab title="macOS">
    ![macOS에서 Comfy Desktop 구성](/images/desktop/usage/new_install_01_settings_mac.png)
  </Tab>
</Tabs>

1. **설치 이름**: 인스턴스의 이름입니다. 기본값은 "ComfyUI"입니다.
2. **설치 위치**: ComfyUI를 설치할 위치를 선택합니다. Windows에서는 **SSD**를 사용하고 C: 드라이브의 루트를 피할 것을 강력히 권장합니다.
3. **고급**: 클릭하여 추가 옵션에 접근합니다.
4. **독립형**: 권장 옵션입니다. 번들 Python, 자동 업데이트, 스냅샷 지원이 포함된 자체 완비형 설치입니다.
5. **원격 연결**: 다른 컴퓨터에서 실행되는 ComfyUI 인스턴스에 연결합니다.

  </Step>
  <Step title="설치 중">

![ComfyUI 설치 중](/images/desktop/usage/new_install_02_installing.png)

Comfy Desktop이 나머지 작업을 처리합니다: Python 다운로드, 환경 설정, ComfyUI 설치. 진행 화면에 현재 단계가 표시됩니다.

  </Step>
  <Step title="템플릿 선택하기">
