왼쪽 상단의 ☰ 메뉴(1)를 클릭하고 **Desktop Settings**(2)를 선택하여 전역 설정 패널을 엽니다.

![Access Desktop Settings](/images/desktop/usage/comfy_desktop_settings_00_menu.png)

설정 패널은 네 개의 탭으로 구성되어 있습니다. 사이드바에서 탭을 클릭하여 뷰를 전환하세요.

## 환경설정

![Preferences tab](/images/desktop/usage/comfy_desktop_settings_01_preferences.png)

- **언어**: 표시 언어 선택: **English** 또는 **中文**.
- 
  **마지막 인스턴스 자동 재시작** — 마지막으로 사용한 인스턴스를 실행 시 자동으로 시작합니다.
- **마지막 창에서 Comfy Desktop 종료 (대시보드로 돌아가지 않음)**: 마지막 창을 닫을 때 대시보드로 돌아가지 않고 완전히 종료합니다.
- **인스턴스 선택기에서 클라우드 숨기기**: 선택기 보기에서 클라우드 인스턴스를 숨깁니다.
- **익명 사용 데이터 공유**: 익명 사용 데이터를 공유하여 Comfy Desktop 개선에 도움을 줍니다.
- **Comfy Desktop**: ComfyDesktop GitHub 저장소 링크.

## 업데이트

![Updates tab](/images/desktop/usage/comfy_desktop_settings_02_updates.png)

- **상태**: Comfy Desktop의 최신 여부, 설치된 버전, 마지막 확인 시간을 표시합니다.
- **업데이트 확인**: 새 버전을 수동으로 확인합니다. 업데이트가 있을 경우 버튼이 **다운로드** → **설치**로 바뀝니다.
- **자동으로 데스크톱 업데이트 설치**: 백그라운드에서 업데이트를 자동 다운로드 및 설치합니다.

## 저장소

저장소 탭에서는 Comfy Desktop이 파일을 찾고 저장하는 위치를 구성합니다. 핵심 개념은 **공유**입니다: 각 인스턴스가 자체적으로 격리된 모델, 입력, 출력을 가지는 대신 모든 인스턴스에서 접근 가능한 공유 디렉토리를 설정합니다.

![Storage tab](/images/desktop/usage/comfy_desktop_settings_03_storage.png)

- **공유 모델**: 여기에 나열된 각 디렉토리는 모든 인스턴스가 모델을 검색합니다. **주**로 표시된 디렉토리가 기본적으로 새 모델을 저장하는 위치입니다.

  ![Make directory primary](/images/desktop/usage/comfy_desktop_settings_03-2_storage.png)

  - 폴더 아이콘을 클릭하면 파일 관리자에서 디렉토리를 엽니다.
  - 디렉토리의 **...** 메뉴(1)를 클릭한 다음, **주로 설정**(2)을 선택하여 새 모델을 저장할 디렉토리를 변경합니다.
  - **+ 디렉토리 추가**를 클릭하여 모델 디렉토리를 더 추가합니다.
- **입력 디렉토리**: 모든 인스턴스가 입력 파일(이미지, 비디오 등)을 찾는 공유 경로입니다.
- **출력 디렉토리**: 모든 인스턴스가 생성된 결과물을 저장하는 공유 경로입니다. 이 경로를 사용자 지정하면 모든 결과를 한 곳에 보관할 수 있습니다.
- 폴더 아이콘을 클릭하여 새 경로를 찾아 선택합니다. **?** 도구 설명에 추가 정보가 표시됩니다.

## 고급

![Advanced tab](/images/desktop/usage/comfy_desktop_settings_04_advanced.png)

- **기본 설치 위치**: 새 독립형 인스턴스가 생성되는 위치입니다. 폴더 아이콘을 클릭하여 변경합니다.
- **PyPI 미러 URL**: 사용자 지정 pip 미러 (예: `https://mirrors.aliyun.com/pypi/simple/`).
- **중국 미러 사용 (Git & PyPI)**: Git 및 PyPI에 중국 미러 서버를 사용합니다. 중국 본토에서 유용합니다.
- **캐시 디렉토리**: Comfy Desktop이 임시 및 캐시 데이터를 저장하는 위치입니다. 폴더 아이콘을 클릭하여 변경합니다.

## 데이터 위치

Comfy Desktop은 플랫폼별 디렉토리에 데이터를 저장합니다.

<Tabs>
  <Tab title="Windows">
```
App data:  %APPDATA%\Comfy Desktop
Logs:      %APPDATA%\Comfy Desktop\logs
```
  </Tab>
  <Tab title="macOS">
```
App data:  ~/Library/Application Support/Comfy Desktop
Logs:      ~/Library/Application Support/Comfy Desktop/logs
```
  </Tab>
  <Tab title="Linux">
```
Config:  ~/.config/Comfy Desktop
Data:    ~/.local/share/Comfy Desktop
Cache:   ~/.cache/Comfy Desktop
State:   ~/.local/state/Comfy Desktop
```
  </Tab>
</Tabs>

## 키보드 단축키

| 단축키 | 동작 |
|----------|--------|
| `Ctrl+R` / `Cmd+R` | 현재 ComfyUI 인스턴스 새로고침 |
| `Ctrl+W` / `Cmd+W` | 현재 창 닫기 |
| `Escape` | 대화 상자나 팝업 닫기 |
| `Ctrl+,` / `Cmd+,` | Comfy Desktop 설정 열기 |
