<Warning>
  **실험적 API:** 이 API는 실험적인 것으로, 변경될 수 있습니다. 엔드포인트, 요청/응답 형식 및 동작은 사전 통지 없이 수정될 수 있습니다.
</Warning>

Comfy Cloud API는 Comfy Cloud 인프라에서 워크플로를 프로그래밍 방식으로 실행할 수 있는 접근 권한을 제공합니다.

<Note>
  **크리에이터 또는 프로 구독 필요:** API 접근 권한은 **크리에이터** 및 **프로** 요금제에서만 제공됩니다. 무료 및 스탠다드 요금제에서는 API 접근 권한이 포함되지 않습니다. 자세한 내용은 [가격 정책](https://www.comfy.org/cloud/pricing?utm_source=docs)을 참조하십시오.
</Note>

## 엔드포인트 카테고리

| 카테고리 | 설명 |
|----------|-------------|
| 워크플로 | 실행을 위해 워크플로 제출 |
| 작업 | 작업 상태 모니터링 및 대기열 관리 |
| 자산 | 파일 업로드 및 다운로드 |
| 모델 | 사용 가능한 AI 모델 탐색 |
| 노드 | 사용 가능한 노드 정보 가져오기 |
| 사용자 | 계정 정보 및 개인 데이터 |
| 시스템 | 서버 상태 및 건강 검사 |
