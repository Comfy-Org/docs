About Panel Badges API를 사용하면 확장 프로그램에서 ComfyUI의 정보 페이지에 맞춤형 배지를 추가할 수 있습니다. 이러한 배지는 확장 프로그램에 대한 정보를 표시하고, 문서, 소스 코드 또는 기타 리소스로 연결되는 링크를 포함할 수 있습니다.

## 기본 사용법

```javascript
app.registerExtension({
  name: "MyExtension",
  aboutPageBadges: [
    {
      label: "문서",
      url: "https://example.com/docs",
      icon: "pi pi-file"
    },
    {
      label: "GitHub",
      url: "https://github.com/username/repo",
      icon: "pi pi-github"
    }
  ]
});
```

## 배지 구성

각 배지에는 다음 속성이 모두 필요합니다:

```javascript
{
  label: string,           // 배지에 표시할 텍스트
  url: string,             // 배지를 클릭했을 때 열리는 URL
  icon: string             // 아이콘 클래스 (예: PrimeVue 아이콘)
}
```

## 아이콘 옵션

배지 아이콘은 PrimeVue의 아이콘 세트를 사용합니다. 다음은 자주 사용되는 아이콘들입니다:

- 문서: `pi pi-file` 또는 `pi pi-book`
- GitHub: `pi pi-github`
- 외부 링크: `pi pi-external-link`
- 정보: `pi pi-info-circle`
- 다운로드: `pi pi-download`
- 웹사이트: `pi pi-globe`
- 디스코드: `pi pi-discord`

사용 가능한 아이콘의 전체 목록은 [PrimeVue 아이콘 문서](https://primevue.org/icons/)를 참조하십시오.

## 예제

```javascript
app.registerExtension({
  name: "BadgeExample",
  aboutPageBadges: [
    {
      label: "웹사이트",
      url: "https://example.com",
      icon: "pi pi-home"
    },
    {
      label: "기부하기",
      url: "https://example.com/donate",
      icon: "pi pi-heart"
    },
    {
      label: "문서",
      url: "https://example.com/docs",
      icon: "pi pi-book"
    }
  ]
});
```

배지는 ComfyUI 인터페이스의 오른쪽 상단 모서리에 있는 톱니바퀴 아이콘을 통해 접근할 수 있는 설정 대화상자의 정보 패널에 나타납니다.
