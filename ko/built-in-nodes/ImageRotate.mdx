ImageRotate 노드는 입력 이미지를 지정된 각도만큼 회전시킵니다. 네 가지 회전 옵션을 지원하며, 회전 없음, 90도 시계방향, 180도, 270도 시계방향이 있습니다. 이 회전은 이미지 데이터의 무결성을 유지하는 효율적인 텐서 연산을 통해 수행됩니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `image` | IMAGE | 예 | - | 회전할 입력 이미지 |
| `rotation` | STRING | 예 | "없음"<br />"90도"<br />"180도"<br />"270도" | 이미지에 적용할 회전 각도 (기본값: "없음") |

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `image` | IMAGE | 회전된 출력 이미지 |
