Wan22ImageToVideoLatent 노드는 이미지로부터 비디오 잠재 표현을 생성합니다. 지정된 크기의 빈 비디오 잠재 공간을 생성하며, 선택적으로 시작 이미지 시퀀스를 초기 프레임에 인코딩할 수 있습니다. 시작 이미지를 제공하면 해당 이미지를 잠재 공간에 인코딩하고, 치환된 영역에 대응하는 노이즈 마스크를 생성합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `vae` | VAE | 예 | - | 이미지를 잠재 공간으로 인코딩하는 데 사용되는 VAE 모델 |
| `width` | INT | 예 | 32부터 MAX_RESOLUTION까지 | 출력 비디오의 너비(픽셀 단위) (기본값: 1280, 증분: 32) |
| `height` | INT | 예 | 32부터 MAX_RESOLUTION까지 | 출력 비디오의 높이(픽셀 단위) (기본값: 704, 증분: 32) |
| `length` | INT | 예 | 1부터 MAX_RESOLUTION까지 | 비디오 시퀀스의 프레임 수 (기본값: 49, 증분: 4) |
| `batch_size` | INT | 예 | 1부터 4096까지 | 생성할 배치 수 (기본값: 1) |
| `start_image` | IMAGE | 아니요 | - | 비디오 잠재 공간에 인코딩할 시작 이미지 시퀀스 (선택적) |

**참고:** `start_image`를 제공하면 노드가 이미지 시퀀스를 잠재 공간의 초기 프레임에 인코딩하고, 해당하는 노이즈 마스크를 생성합니다. 너비와 높이 매개변수는 적절한 잠재 공간 크기를 위해 16으로 나누어져야 합니다. `length` 매개변수는 비디오 잠재 공간의 프레임 수를 결정하며, 잠재 공간의 시간 차원은 `((length - 1) // 4) + 1`로 계산됩니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `samples` | LATENT | 생성된 비디오 잠재 표현 |
| `noise_mask` | LATENT | 생성 과정에서 어떤 영역을 노이즈 제거해야 하는지 나타내는 노이즈 마스크 |
