형태학 노드는 이미지에 다양한 형태학적 연산을 적용하며, 이는 이미지 속 도형을 처리하고 분석하는 데 사용되는 수학적 연산입니다. 침식, 팽창, 열림, 닫힘 등과 같은 연산을 사용자 정의 가능한 커널 크기를 통해 효과의 강도를 조절하면서 수행할 수 있습니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `image` | IMAGE | 예 | - | 처리할 입력 이미지 |
| `operation` | STRING | 예 | `"erode"`<br />`"dilate"`<br />`"open"`<br />`"close"`<br />`"gradient"`<br />`"bottom_hat"`<br />`"top_hat"` | 적용할 형태학적 연산 (기본값: "erode") |
| `kernel_size` | INT | 예 | 3-999 | 구조 요소 커널의 크기 (기본값: 3). 반드시 홀수여야 합니다. |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `image` | IMAGE | 형태학적 연산을 적용한 후 처리된 이미지 |
