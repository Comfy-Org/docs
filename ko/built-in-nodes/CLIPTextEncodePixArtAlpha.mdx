텍스트를 인코딩하고 PixArt Alpha용 해상도 조건부 데이터를 설정합니다. 이 노드는 텍스트 입력을 처리하고 너비와 높이 정보를 추가하여 PixArt Alpha 모델 전용 조건부 데이터를 생성합니다. PixArt Sigma 모델에는 적용되지 않습니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `width` | INT | 예 | 0부터 MAX_RESOLUTION까지 | 해상도 조건부를 위한 너비 치수 (기본값: 1024) |
| `height` | INT | 예 | 0부터 MAX_RESOLUTION까지 | 해상도 조건부를 위한 높이 치수 (기본값: 1024) |
| `text` | STRING | 예 | - | 인코딩할 텍스트 입력으로, 다중 줄 입력과 동적 프롬프트를 지원합니다 |
| `clip` | CLIP | 예 | - | 토큰화 및 인코딩에 사용되는 CLIP 모델 |

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `CONDITIONING` | CONDITIONING | 텍스트 토큰과 해상도 정보를 포함한 인코딩된 조건부 데이터 |
