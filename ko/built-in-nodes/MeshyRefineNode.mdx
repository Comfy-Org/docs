Meshy: 초안 모델 정밀화 노드는 이전에 생성된 3D 초안 모델을 받아 품질을 개선하고, 선택적으로 텍스처를 추가합니다. 이 노드는 Meshy API에 정밀화 작업을 제출하며, 처리가 완료되면 최종 3D 모델 파일을 반환합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `model` | COMBO | 예 | `"latest"` | 정밀화에 사용할 AI 모델을 지정합니다. 현재는 "latest" 모델만 사용 가능합니다. |
| `meshy_task_id` | MESHY_TASK_ID | 예 | - | 정밀화하려는 초안 모델의 고유한 작업 ID입니다. |
| `enable_pbr` | BOOLEAN | 아니요 | - | 기본 색상 외에도 PBR 맵(금속성, 거칠기, 법선)을 생성합니다. 참고: Sculpture 스타일을 사용할 경우 이 옵션은 false로 설정해야 합니다. Sculpture 스타일은 자체적인 PBR 맵 세트를 생성하기 때문입니다. (기본값: `False`) |
| `texture_prompt` | STRING | 아니요 | - | 텍스처링 과정을 안내할 텍스트 프롬프트를 제공하세요. 최대 600자입니다. `texture_image`와 동시에 사용할 수 없습니다. (기본값: 빈 문자열) |
| `texture_image` | IMAGE | 아니요 | - | `texture_image`와 `texture_prompt` 중 하나만 동시에 사용할 수 있습니다. |

**참고:** `texture_prompt`와 `texture_image` 입력은 서로 배타적입니다. 동일한 작업에서 텍스처링을 위해 텍스트 프롬프트와 이미지를 모두 제공할 수 없습니다.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `model_file` | STRING | 생성된 GLB 모델의 파일명입니다. (후방 호환성을 위해만 제공됩니다) |
| `meshy_task_id` | MESHY_TASK_ID | 제출된 정밀화 작업의 고유한 작업 ID입니다. |
| `GLB` | FILE3DGLB | GLB 형식의 최종 정밀화된 3D 모델입니다. |
| `FBX` | FILE3DFBX | FBX 형식의 최종 정밀화된 3D 모델입니다. |
