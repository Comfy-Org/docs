Gemini API와 함께 사용할 수 있도록 입력 파일을 로드하고 형식화합니다. 이 노드를 사용하면 사용자가 텍스트(.txt) 및 PDF(.pdf) 파일을 Gemini 모델의 입력 컨텍스트로 포함시킬 수 있습니다. 파일은 API에서 요구하는 적절한 형식으로 변환되며, 여러 파일을 하나의 요청에 포함시키기 위해 체인 방식으로 연결할 수 있습니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `file` | COMBO | 예 | 여러 옵션 제공 (20 MB 미만의 .txt 및 .pdf 파일만 가능) | 모델의 컨텍스트로 포함할 입력 파일입니다. 현재 텍스트(.txt) 및 PDF(.pdf) 파일만 지원됩니다. 파일 크기는 최대 입력 파일 크기 제한(20 MB)보다 작아야 합니다. |
| `GEMINI_INPUT_FILES` | GEMINI_INPUT_FILES | 선택 | N/A | 이 노드에서 로드된 파일과 함께 배치할 추가 파일(옵션). 단일 메시지에 여러 입력 파일을 포함하도록 입력 파일을 체인 방식으로 연결할 수 있습니다. |

**참고:** `file` 매개변수에는 20 MB 미만의 텍스트(.txt) 및 PDF(.pdf) 파일만 표시됩니다. 파일은 자동으로 필터링되고 이름순으로 정렬됩니다.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `GEMINI_INPUT_FILES` | GEMINI_INPUT_FILES | Gemini LLM 노드와 함께 사용할 준비가 된 형식화된 파일 데이터로, 적절한 API 형식으로 로드된 파일 내용을 포함합니다. |
