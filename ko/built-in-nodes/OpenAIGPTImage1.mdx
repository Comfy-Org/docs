> 이 문서는 AI로 생성되었습니다. 오류를 발견하거나 개선할 점이 있다면 언제든지 기여해 주세요! [GitHub에서 편집하기](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/OpenAIGPTImage1/en.md)

OpenAI의 GPT 이미지 엔드포인트를 통해 동기식으로 이미지를 생성합니다. 이 노드는 텍스트 프롬프트를 사용해 새로운 이미지를 만들거나, 입력 이미지와 선택적 마스크를 제공하면 기존 이미지를 편집할 수 있습니다. gpt-image-1, gpt-image-1.5, gpt-image-2 등 여러 GPT 이미지 모델을 지원합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `prompt` | 문자열 | 예 | - | GPT 이미지용 텍스트 프롬프트 (기본값: "") |
| `seed` | 정수 | 아니요 | 0부터 2147483647까지 | 생성을 위한 난수 시드 (기본값: 0) - 아직 백엔드에서는 구현되지 않았습니다 |
| `quality` | 콤보 | 아니요 | "낮음"<br />"중간"<br />"높음" | 이미지 품질, 비용과 생성 시간에 영향을 줍니다 (기본값: "낮음") |
| `background` | 콤보 | 아니요 | "자동"<br />"불투명"<br />"투명" | 배경 포함 또는 제외된 이미지 반환 (기본값: "자동") |
| `size` | 콤보 | 아니요 | "자동"<br />"1024x1024"<br />"1024x1536"<br />"1536x1024"<br />"2048x2048"<br />"2048x1152"<br />"1152x2048"<br />"3840x2160"<br />"2160x3840"<br />"사용자 지정" | 이미지 크기. "사용자 지정"을 선택하면 맞춤 너비와 높이를 사용할 수 있습니다 (GPT 이미지 2만 해당) (기본값: "자동") |
| `n` | 정수 | 아니요 | 1부터 8까지 | 몇 장의 이미지를 생성할지 (기본값: 1) |
| `image` | 이미지 | 아니요 | - | 이미지 편집을 위한 선택적 참조 이미지 |
| `mask` | 마스크 | 아니요 | - | 인페인팅을 위한 선택적 마스크 (흰색 영역은 교체됩니다) |
| `model` | 콤보 | 아니요 | "gpt-image-1"<br />"gpt-image-1.5"<br />"gpt-image-2" | 사용할 GPT 이미지 모델 (기본값: "gpt-image-2") |
| `custom_width` | 정수 | 아니요 | 1024부터 3840까지 | `size`가 "사용자 지정"일 때만 사용됩니다. 반드시 16의 배수여야 합니다 (GPT 이미지 2만 해당) (기본값: 1024) |
| `custom_height` | 정수 | 아니요 | 1024부터 3840까지 | `size`가 "사용자 지정"일 때만 사용됩니다. 반드시 16의 배수여야 합니다 (GPT 이미지 2만 해당) (기본값: 1024) |

**매개변수 제약사항:**

- `image`가 제공되면 노드는 이미지 편집 모드로 전환됩니다.
- `mask`는 `image`가 제공될 때만 사용 가능합니다.
- `mask`를 사용할 경우 단일 이미지만 지원됩니다 (배치 크기는 1이어야 함).
- `mask`와 `image`는 반드시 같은 크기여야 합니다.
- 사용자 지정 해상도(`size` = "사용자 지정")는 gpt-image-2 모델에서만 지원됩니다.
- 사용자 지정 너비와 높이는 반드시 16의 배수여야 합니다.
- 사용자 지정 해상도의 종횡비는 3:1을 초과해서는 안 됩니다.
- 사용자 지정 해상도의 총 픽셀 수는 655,360에서 8,294,400 사이여야 합니다.
- 투명 배경은 gpt-image-2 모델에서 지원되지 않습니다.
- 1536x1024보다 큰 크기(예: 2048x2048, 3840x2160)는 gpt-image-2 모델에서만 지원됩니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-----------|-----------|-------------|
| `IMAGE` | 이미지 | 생성되거나 편집된 이미지 |
