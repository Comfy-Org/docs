PikaScenes v2.2 노드는 여러 이미지를 결합하여 모든 입력 이미지의 객체를 포함한 동영상을 만듭니다. 최대 다섯 장의 서로 다른 이미지를 업로드하여 이를 매끄럽게 혼합한 고품질의 동영상을 생성할 수 있습니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `prompt_text` | 문자열 | 예 | - | 생성할 내용에 대한 텍스트 설명 |
| `negative_prompt` | 문자열 | 예 | - | 생성 시 피해야 할 내용에 대한 텍스트 설명 |
| `seed` | 정수 | 예 | - | 생성을 위한 난수 시드 값 |
| `resolution` | 문자열 | 예 | - | 출력 동영상의 해상도 |
| `duration` | 정수 | 예 | - | 생성된 동영상의 지속 시간 |
| `ingredients_mode` | 문자열 | 아니요 | "creative"<br />"precise" | 재료를 결합하는 모드 (기본값: "creative") |
| `aspect_ratio` | 부동소수점 | 아니요 | 0.4 - 2.5 | 종횡비 (가로 / 세로) (기본값: 1.778) |
| `image_ingredient_1` | 이미지 | 아니요 | - | 동영상 생성에 사용될 재료 이미지 |
| `image_ingredient_2` | 이미지 | 아니요 | - | 동영상 생성에 사용될 재료 이미지 |
| `image_ingredient_3` | 이미지 | 아니요 | - | 동영상 생성에 사용될 재료 이미지 |
| `image_ingredient_4` | 이미지 | 아니요 | - | 동영상 생성에 사용될 재료 이미지 |
| `image_ingredient_5` | 이미지 | 아니요 | - | 동영상 생성에 사용될 재료 이미지 |

**참고:** 최대 5개의 이미지 재료를 제공할 수 있지만, 동영상을 생성하려면 적어도 하나의 이미지는 반드시 필요합니다. 노드는 제공된 모든 이미지를 사용해 최종 동영상 구성물을 만들어냅니다.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `output` | 동영상 | 모든 입력 이미지를 결합한 생성된 동영상 |
