WanSoundImageToVideo 노드는 이미지로부터 비디오 콘텐츠를 생성하며, 선택적 오디오 조정 기능을 제공합니다. 이 노드는 긍정적 및 부정적 조정 프롬프트와 VAE 모델을 받아 비디오 랜트를 생성하고, 참조 이미지, 오디오 인코딩, 제어 비디오 및 모션 참조를 포함하여 비디오 생성 과정을 안내할 수 있습니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `positive` | CONDITIONING | 예 | - | 생성된 비디오에 어떤 콘텐츠가 나타나야 하는지를 안내하는 긍정적 조정 프롬프트 |
| `negative` | CONDITIONING | 예 | - | 생성된 비디오에서 피해야 할 콘텐츠를 지정하는 부정적 조정 프롬프트 |
| `vae` | VAE | 예 | - | 비디오 랜트 표현을 인코딩하고 디코딩하는 데 사용되는 VAE 모델 |
| `width` | INT | 예 | 16부터 MAX_RESOLUTION까지 | 출력 비디오의 너비(픽셀 단위) (기본값: 832, 반드시 16으로 나누어져야 함) |
| `height` | INT | 예 | 16부터 MAX_RESOLUTION까지 | 출력 비디오의 높이(픽셀 단위) (기본값: 480, 반드시 16으로 나누어져야 함) |
| `length` | INT | 예 | 1부터 MAX_RESOLUTION까지 | 생성된 비디오의 프레임 수 (기본값: 77, 반드시 4로 나누어져야 함) |
| `batch_size` | INT | 예 | 1부터 4096까지 | 동시에 생성할 비디오의 수 (기본값: 1) |
| `audio_encoder_output` | AUDIOENCODEROUTPUT | 아니요 | - | 사운드 특성에 따라 비디오 생성에 영향을 줄 수 있는 선택적 오디오 인코딩. 제공될 경우, 오디오 특성이 보간되어 비디오 생성에 조정으로 사용됩니다. |
| `ref_image` | IMAGE | 아니요 | - | 비디오 콘텐츠에 시각적 안내를 제공하는 선택적 참조 이미지. 이미지는 지정된 너비와 높이에 맞게 확대된 후 랜트 표현으로 인코딩됩니다. |
| `control_video` | IMAGE | 아니요 | - | 생성된 비디오의 움직임과 구조를 안내하는 선택적 제어 비디오. 비디오는 확대되고 인코딩된 후 출력 조정에 사용됩니다. 첫 번째 `length` 프레임만 사용됩니다. |
| `ref_motion` | IMAGE | 아니요 | - | 비디오의 움직임 패턴에 안내를 제공하는 선택적 모션 참조. 입력이 73프레임 이상일 경우, 마지막 73프레임만 사용됩니다. 73프레임 미만일 경우, 중립 프레임으로 시퀀스를 채웁니다. |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `positive` | CONDITIONING | 비디오 생성을 위해 수정된 긍정적 조정, 오디오 임베딩, 참조 랜트, 모션 참조 및 제어 비디오 조정 포함 |
| `negative` | CONDITIONING | 비디오 생성을 위해 수정된 부정적 조정, 오디오 임베딩(0으로 설정), 참조 랜트, 모션 참조 및 제어 비디오 조정 포함 |
| `latent` | LATENT | 최종 비디오 프레임으로 디코딩할 수 있는 랜트 공간의 생성된 비디오 표현. 랜트 텐서의 형태는 [batch_size, 16, latent_t, height/8, width/8]이며, latent_t는 length 매개변수에서 파생됩니다 |
