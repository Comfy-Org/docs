## 개요

이 노드는 추적된 객체의 비디오 미리보기를 생성하며, 각 추적된 객체에 고유한 색상 오버레이와 숫자 레이블을 표시합니다. 이 노드는 이미지나 비디오 텐서를 출력하지 않으며, 대신 결과 미리보기 비디오를 바로 임시 파일에 저장합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `track_data` | TRACK_DATA | 예 | - | SAM3 추적 노드에서 패킹된 마스크와 객체 정보를 포함한 추적 데이터입니다. |
| `images` | IMAGE | 아니요 | - | 미리보기의 배경으로 사용할 선택적 입력 이미지입니다. 제공되지 않을 경우 검은색 배경이 사용됩니다. |
| `opacity` | FLOAT | 아니요 | 0.0부터 1.0까지 (단계: 0.05) | 추적된 객체에 적용되는 색상 오버레이의 불투명도입니다 (기본값: 0.5). |
| `fps` | FLOAT | 아니요 | 1.0부터 120.0까지 (단계: 1.0) | 출력 비디오의 프레임 속도입니다 (기본값: 24.0). |

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `ui` | PREVIEW_VIDEO | 생성된 미리보기 비디오를 표시하는 UI 요소입니다. 텐서 데이터는 반환되지 않습니다. |
