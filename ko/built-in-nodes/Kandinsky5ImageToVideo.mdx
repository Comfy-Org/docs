Kandinsky5ImageToVideo 노드는 Kandinsky 모델을 사용해 비디오 생성을 위한 조건부 및 잠재 공간 데이터를 준비합니다. 이 노드는 빈 비디오 잠재 텐서를 생성하며, 선택적으로 시작 이미지를 인코딩하여 생성된 비디오의 초기 프레임을 안내하고, 이에 따라 긍정적·부정적 조건부를 수정할 수 있습니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `positive` | CONDITIONING | 예 | N/A | 비디오 생성을 안내하는 긍정적 조건부 프롬프트입니다. |
| `negative` | CONDITIONING | 예 | N/A | 특정 개념에서 벗어나도록 비디오 생성을 유도하는 부정적 조건부 프롬프트입니다. |
| `vae` | VAE | 예 | N/A | 선택적 시작 이미지를 잠재 공간으로 인코딩하는 데 사용되는 VAE 모델입니다. |
| `width` | INT | 아니요 | 16~8192 (단계 16) | 출력 비디오의 너비(픽셀), 기본값: 768. |
| `height` | INT | 아니요 | 16~8192 (단계 16) | 출력 비디오의 높이(픽셀), 기본값: 512. |
| `length` | INT | 아니요 | 1~8192 (단계 4) | 비디오의 프레임 수, 기본값: 121. |
| `batch_size` | INT | 아니요 | 1~4096 | 동시에 생성할 비디오 시퀀스의 수, 기본값: 1. |
| `start_image` | IMAGE | 아니요 | N/A | 선택적 시작 이미지입니다. 제공되면 인코딩되어 모델 출력 잠재값의 잡음 있는 시작 부분을 대체하는 데 사용됩니다. |

**참고:** `start_image`가 제공되면, 이 이미지는 자동으로 지정된 `width`와 `height`에 맞게 이중선형 보간법을 사용해 크기가 조정됩니다. 이미지 배치의 처음 `length` 프레임이 인코딩에 사용됩니다. 이후 인코딩된 잠재값은 `positive`와 `negative` 조건부 모두에 삽입되어 비디오의 초기 모습을 안내합니다.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `positive` | CONDITIONING | 수정된 긍정적 조건부, 선택적으로 인코딩된 시작 이미지 데이터로 업데이트될 수 있습니다. |
| `negative` | CONDITIONING | 수정된 부정적 조건부, 선택적으로 인코딩된 시작 이미지 데이터로 업데이트될 수 있습니다. |
| `latent` | LATENT | 지정된 차원에 맞춰 0으로 채워진 빈 비디오 잠재 텐서입니다. |
| `cond_latent` | LATENT | 제공된 시작 이미지의 깨끗하고 인코딩된 잠재 표현입니다. 이는 내부적으로 생성된 비디오 잠재값의 잡음 있는 시작 부분을 대체하는 데 사용됩니다. |
