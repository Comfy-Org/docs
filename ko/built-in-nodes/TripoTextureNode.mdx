TripoTextureNode는 Tripo API를 사용해 질감이 적용된 3D 모델을 생성합니다. 모델 작업 ID를 입력받아 PBR 재료, 질감 품질 설정, 정렬 방법 등 다양한 옵션을 통해 질감 생성을 적용합니다. 이 노드는 Tripo API와 통신하여 질감 생성 요청을 처리하고, 결과 모델 파일과 작업 ID를 반환합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `model_task_id` | MODEL_TASK_ID | 예 | - | 질감을 적용할 모델의 작업 ID |
| `texture` | BOOLEAN | 아니요 | - | 질감 생성 여부 (기본값: True) |
| `pbr` | BOOLEAN | 아니요 | - | PBR(물리 기반 렌더링) 재료 생성 여부 (기본값: True) |
| `texture_seed` | INT | 아니요 | - | 질감 생성을 위한 난수 시드 (기본값: 42) |
| `texture_quality` | COMBO | 아니요 | "standard"<br />"detailed" | 질감 생성의 품질 수준 (기본값: "standard"). "detailed" 옵션은 0.20 USD, "standard" 옵션은 0.10 USD입니다. |
| `texture_alignment` | COMBO | 아니요 | "original_image"<br />"geometry" | 질감 정렬 방법 (기본값: "original_image"). "original_image"는 원본 입력 이미지에 맞춰 질감을 정렬하며, "geometry"는 3D 지오메트리에 맞춰 정렬합니다. |

*참고: 이 노드는 인증 토큰과 API 키가 필요하며, 시스템에서 자동으로 처리됩니다.*

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `model_file` | STRING | 질감이 적용된 생성된 모델 파일 (후향적 호환성을 위해만 제공) |
| `model task_id` | MODEL_TASK_ID | 질감 생성 과정을 추적하는 작업 ID |
| `GLB` | FILE3DGLB | 질감이 적용된 GLB 형식의 생성된 3D 모델 |
