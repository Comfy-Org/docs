텍스트 교체 노드는 간단한 텍스트 치환 작업을 수행합니다. 입력된 텍스트 내에서 지정된 텍스트를 검색하고, 모든 일치하는 부분을 새 텍스트로 교체합니다. 이 작업은 노드에 제공된 모든 텍스트 입력에 적용됩니다.

**참고:** 이 노드는 더 이상 사용되지 않으며, 다른 Replace Text 노드로 대체되었습니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `text` | 문자열 | 예 | - | 처리할 텍스트입니다. |
| `find` | 문자열 | 예 | - | 찾을 텍스트 (기본값: 빈 문자열). |
| `replace` | 문자열 | 예 | - | 교체할 텍스트 (기본값: 빈 문자열). |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-----------|-----------|-------------|
| `text` | 문자열 | `find` 텍스트의 모든 일치 부분이 `replace` 텍스트로 교체된 처리된 텍스트입니다. |
