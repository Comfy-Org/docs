TextGenerate 노드는 CLIP 모델을 사용하여 사용자의 프롬프트를 기반으로 텍스트를 생성합니다. 선택적으로 이미지, 비디오 또는 오디오를 추가적인 맥락으로 활용해 텍스트 생성을 유도할 수 있습니다. 출력 길이를 조정하고, 지원되는 모델에서는 사고 모드를 활성화하며, 다양한 설정을 통해 랜덤 샘플링을 사용할지 또는 샘플링 없이 텍스트를 생성할지 선택할 수 있습니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `clip` | CLIP | 예 | N/A | 프롬프트를 토큰화하고 텍스트를 생성하는 데 사용되는 CLIP 모델입니다. |
| `prompt` | 문자열 | 예 | N/A | 생성을 유도하는 텍스트 프롬프트입니다. 이 필드는 여러 줄과 동적 프롬프트를 지원합니다. 기본값은 빈 문자열입니다. |
| `image` | 이미지 | 아니요 | N/A | 텍스트 프롬프트와 함께 사용해 생성된 텍스트에 영향을 줄 수 있는 선택적 이미지입니다. |
| `video` | 이미지 | 아니요 | N/A | 비디오 프레임을 이미지 배치로 제공합니다. 24 FPS로 가정되며, 내부적으로 1 FPS로 하향 샘플링됩니다. |
| `audio` | 오디오 | 아니요 | N/A | 텍스트 프롬프트와 함께 사용해 생성된 텍스트에 영향을 줄 수 있는 선택적 오디오 입력입니다. |
| `max_length` | 정수 | 예 | 1~2048 | 모델이 생성할 최대 토큰 수입니다. 기본값은 256입니다. |
| `sampling_mode` | 콤보 | 예 | `"on"`<br />`"off"` | 텍스트 생성 시 랜덤 샘플링 사용 여부를 제어합니다. "on"으로 설정하면 샘플링을 제어하는 추가 매개변수가 사용 가능해집니다. 기본값은 "on"입니다. |
| `thinking` | 부울 | 아니요 | True 또는 False | 모델이 지원하는 경우 사고 모드로 작동합니다. 기본값은 False입니다. |
| `use_default_template` | 부울 | 아니요 | True 또는 False | 모델에 내장된 시스템 프롬프트/템플릿을 사용할지 여부입니다. 기본값은 True입니다. 이는 고급 매개변수입니다. |
| `temperature` | 실수 | 아니요 | 0.01~2.0 | 출력의 무작위성을 제어합니다. 낮은 값은 출력을 더 예측 가능하게 만들고, 높은 값은 더 창의적으로 만듭니다. 이 매개변수는 `sampling_mode`가 "on"일 때만 사용 가능합니다. 기본값은 0.7입니다. |
| `top_k` | 정수 | 아니요 | 0~1000 | 가장 가능성이 높은 다음 토큰 상위 K개로 샘플링 풀을 제한합니다. 값이 0이면 이 필터가 비활성화됩니다. 이 매개변수는 `sampling_mode`가 "on"일 때만 사용 가능합니다. 기본값은 64입니다. |
| `top_p` | 실수 | 아니요 | 0.0~1.0 | 핵심 샘플링을 사용해 누적 확률이 이 값보다 작은 토큰으로 선택을 제한합니다. 이 매개변수는 `sampling_mode`가 "on"일 때만 사용 가능합니다. 기본값은 0.95입니다. |
| `min_p` | 실수 | 아니요 | 0.0~1.0 | 토큰이 고려되기 위한 최소 확률 임계값을 설정합니다. 이 매개변수는 `sampling_mode`가 "on"일 때만 사용 가능합니다. 기본값은 0.05입니다. |
| `repetition_penalty` | 실수 | 아니요 | 0.0~5.0 | 이미 생성된 토큰을 페널티해 반복을 줄입니다. 값이 1.0이면 페널티가 적용되지 않습니다. 이 매개변수는 `sampling_mode`가 "on"일 때만 사용 가능합니다. 기본값은 1.05입니다. |
| `presence_penalty` | 실수 | 아니요 | 0.0~5.0 | 새로 생성된 토큰이 현재까지 텍스트에 나타난 적이 있는지에 따라 페널티를 적용해 모델이 새로운 주제를 다루도록 유도합니다. 이 매개변수는 `sampling_mode`가 "on"일 때만 사용 가능합니다. 기본값은 0.0입니다. |
| `seed` | 정수 | 아니요 | 0~18446744073709551615 | 샘플링이 "on"일 때 재현 가능한 결과를 위해 난수 생성기를 초기화하는 데 사용되는 숫자입니다. 기본값은 0입니다. |

**참고:** `temperature`, `top_k`, `top_p`, `min_p`, `repetition_penalty`, `presence_penalty`, `seed` 매개변수는 `sampling_mode`가 "on"으로 설정된 경우에만 노드 인터페이스에서 활성화되고 표시됩니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-----------|-----------|-------------|
| `generated_text` | 문자열 | 입력 프롬프트와 선택적 이미지, 비디오 또는 오디오를 기반으로 모델이 생성한 텍스트입니다. |
