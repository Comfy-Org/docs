밝기 조정 노드는 입력 이미지의 밝기를 수정합니다. 각 픽셀 값을 지정된 요인으로 곱한 뒤, 결과값이 유효한 범위 내에 있도록 클램핑하여 작동합니다. 요인 값이 1.0이면 이미지가 변경되지 않고, 1.0 미만이면 어두워지고, 1.0 초과이면 밝아집니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `image` | IMAGE | 예 | - | 조정할 입력 이미지입니다. |
| `factor` | FLOAT | 아니요 | 0.0 - 2.0 | 밝기 요인. 1.0 = 변화 없음, &lt;1.0 = 어둡게, >1.0 = 밝게. (기본값: 1.0) |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `image` | IMAGE | 밝기가 조정된 출력 이미지입니다. |
