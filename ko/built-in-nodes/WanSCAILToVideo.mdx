WanSCAILToVideo 노드는 비디오 생성을 위해 조건부와 빈 잠재 공간을 준비합니다. 참조 이미지, 포즈 비디오, CLIP 비전 출력과 같은 선택적 입력을 처리하여 이를 비디오 모델의 양성 및 음성 조건부에 포함시킵니다. 이 노드는 수정된 조건부와 지정된 비디오 크기의 빈 잠재 텐서를 출력합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `positive` | CONDITIONING | 예 | - | 양성 조건부 입력입니다. |
| `negative` | CONDITIONING | 예 | - | 음성 조건부 입력입니다. |
| `vae` | VAE | 예 | - | 이미지와 비디오 프레임을 인코딩하는 데 사용되는 VAE 모델입니다. |
| `width` | INT | 예 | 32부터 MAX_RESOLUTION까지 | 출력 비디오의 너비(픽셀 단위, 기본값: 512). 8로 나누어져야 합니다. |
| `height` | INT | 예 | 32부터 MAX_RESOLUTION까지 | 출력 비디오의 높이(픽셀 단위, 기본값: 896). 8로 나누어져야 합니다. |
| `length` | INT | 예 | 1부터 MAX_RESOLUTION까지 | 비디오의 프레임 수(기본값: 81). 4로 나누어져야 합니다. |
| `batch_size` | INT | 예 | 1부터 4096까지 | 한 번에 생성할 비디오 수(기본값: 1). |
| `clip_vision_output` | CLIP_VISION_OUTPUT | 아니요 | - | 조건부를 위한 선택적 CLIP 비전 출력입니다. |
| `reference_image` | IMAGE | 아니요 | - | 조건부를 위한 선택적 참조 이미지입니다. |
| `pose_video` | IMAGE | 아니요 | - | 포즈 조건부에 사용되는 비디오입니다. 메인 비디오 해상도의 절반으로 축소됩니다. |
| `pose_strength` | FLOAT | 예 | 0.0부터 10.0까지 | 포즈 잠재 변수의 강도(기본값: 1.0). |
| `pose_start` | FLOAT | 예 | 0.0부터 1.0까지 | 포즈 조건부를 사용할 시작 단계(기본값: 0.0). |
| `pose_end` | FLOAT | 예 | 0.0부터 1.0까지 | 포즈 조건부를 사용할 끝 단계(기본값: 1.0). |

**참고:** `pose_video` 입력은 처음 `length` 프레임에만 처리됩니다. `reference_image`는 배치의 첫 번째 이미지에만 처리됩니다. `reference_image`가 제공되면 동일한 크기의 0으로 채워진 잠재 변수가 음성 조건부에 사용됩니다. `clip_vision_output`이 제공되면 양성 및 음성 조건부 모두에 적용됩니다. `pose_video`는 인코딩 전에 메인 비디오 해상도의 절반으로 축소됩니다.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `positive` | CONDITIONING | 수정된 양성 조건부로, 잠재적으로 참조 이미지 잠재 변수, CLIP 비전 출력 또는 포즈 비디오 잠재 변수가 포함될 수 있습니다. |
| `negative` | CONDITIONING | 수정된 음성 조건부로, 잠재적으로 참조 이미지 잠재 변수, CLIP 비전 출력 또는 포즈 비디오 잠재 변수가 포함될 수 있습니다. |
| `latent` | LATENT | `[batch_size, 16, ((length - 1) // 4) + 1, height // 8, width // 8]` 크기의 빈 잠재 텐서입니다. |
