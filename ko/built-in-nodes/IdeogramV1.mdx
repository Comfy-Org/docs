IdeogramV1 노드는 API를 통해 Ideogram V1 모델을 사용하여 이미지를 생성합니다. 텍스트 프롬프트와 다양한 생성 설정을 받아 사용자의 입력에 따라 하나 이상의 이미지를 생성합니다. 이 노드는 다양한 종횡비와 생성 모드를 지원하여 출력을 맞춤화할 수 있습니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `prompt` | 문자열 | 예 | - | 이미지 생성을 위한 프롬프트 (기본값: 빈 문자열) |
| `turbo` | 부울 | 예 | - | 터보 모드 사용 여부 (더 빠른 생성, 잠재적으로 낮은 품질) (기본값: False) |
| `aspect_ratio` | 콤보 | 아니요 | "1:1"<br />"4:3"<br />"3:4"<br />"16:9"<br />"9:16"<br />"2:1"<br />"1:2"<br />"3:2"<br />"2:3"<br />"4:5"<br />"5:4" | 이미지 생성 시 종횡비 (기본값: "1:1") |
| `magic_prompt_option` | 콤보 | 아니요 | "AUTO"<br />"ON"<br />"OFF" | 생성 시 MagicPrompt 사용 여부 결정 (기본값: "AUTO") |
| `seed` | 정수 | 아니요 | 0-2147483647 | 생성을 위한 임의의 시드 값 (기본값: 0) |
| `negative_prompt` | 문자열 | 아니요 | - | 이미지에서 제외할 내용 설명 (기본값: 빈 문자열) |
| `num_images` | 정수 | 아니요 | 1-8 | 생성할 이미지 수 (기본값: 1) |

**참고:** `num_images` 매개변수는 한 번의 생성 요청당 최대 8개의 이미지를 생성할 수 있습니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `output` | 이미지 | Ideogram V1 모델에서 생성된 이미지(들) |
