이 노드는 SDXL 아키텍처에 특별히 맞춤화된 CLIP 모델을 사용하여 텍스트 입력을 인코딩하도록 설계되었습니다. 이 노드는 듀얼 인코더 시스템(CLIP-L 및 CLIP-G)을 사용해 텍스트 설명을 처리하며, 이를 통해 보다 정확한 이미지 생성을 가능하게 합니다.

## 입력

| 매개변수 | 데이터 타입 | 설명 |
|-----------|-----------|-------------|
| `clip` | CLIP | 텍스트 인코딩에 사용되는 CLIP 모델 인스턴스입니다. |
| `width` | INT | 이미지 너비를 픽셀 단위로 지정합니다. 기본값은 1024입니다. |
| `height` | INT | 이미지 높이를 픽셀 단위로 지정합니다. 기본값은 1024입니다. |
| `crop_w` | INT | 자르기 영역의 너비를 픽셀 단위로 지정합니다. 기본값은 0입니다. |
| `crop_h` | INT | 자르기 영역의 높이를 픽셀 단위로 지정합니다. 기본값은 0입니다. |
| `target_width` | INT | 출력 이미지의 목표 너비를 지정합니다. 기본값은 1024입니다. |
| `target_height` | INT | 출력 이미지의 목표 높이를 지정합니다. 기본값은 1024입니다. |
| `text_g` | STRING | 전체 장면 설명을 위한 글로벌 텍스트 설명입니다. |
| `text_l` | STRING | 세부 설명을 위한 로컬 텍스트 설명입니다. |

## 출력

| 매개변수 | 데이터 타입 | 설명 |
|-----------|-----------|-------------|
| `CONDITIONING` | CONDITIONING | 이미지 생성에 필요한 인코딩된 텍스트와 조건 정보를 포함합니다. |
