폴리엑스포넨셜스케줄러 노드는 폴리엑스포넨셜 노이즈 스케줄을 기반으로 노이즈 레벨(시그마)의 시퀀스를 생성하도록 설계되었습니다. 이 스케줄은 시그마의 로그값에 대한 다항식 함수로, 확산 과정 전반에 걸쳐 노이즈 레벨이 유연하고 맞춤형으로 진행되도록 합니다.

## 입력

| 매개변수   | 데이터 타입 | 설명                                                                                                                                                                                                                                                                                                                                                      |
|-------------|-------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| `steps`     | INT         | 확산 과정의 단계 수를 지정하며, 생성되는 노이즈 레벨의 세밀도에 영향을 줍니다.                                                                                                                                                                                                                                                                        |
| `sigma_max` | FLOAT       | 최대 노이즈 레벨로, 노이즈 스케줄의 상한 값을 설정합니다.                                                                                                                                                                                                                                                                                                                                 |
| `sigma_min` | FLOAT       | 최소 노이즈 레벨로, 노이즈 스케줄의 하한 값을 설정합니다.                                                                                                                                                                                                                                                                                                                                 |
| `rho`       | FLOAT       | 폴리엑스포넨셜 노이즈 스케줄의 형태를 제어하는 매개변수로, 최소값과 최대값 사이에서 노이즈 레벨이 어떻게 진행되는지에 영향을 줍니다.                                                                                                                                                                                                               |

## 출력

| 매개변수 | 데이터 타입 | 설명                                                                 |
|-----------|-------------|-----------------------------------------------------------------------------|
| `sigmas`  | SIGMAS      | 출력은 지정된 폴리엑스포넨셜 노이즈 스케줄에 맞춘 노이즈 레벨(시그마)의 시퀀스입니다. |
