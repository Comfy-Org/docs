DualCFGGuider 노드는 이중 분류자 없는 가이던스 샘플링을 위한 가이던스 시스템을 생성합니다. 두 개의 양수 조건 입력과 하나의 음수 조건 입력을 결합하며, 각 조건 쌍에 서로 다른 가이던스 스케일을 적용하여 생성된 출력에 각 프롬프트가 미치는 영향을 제어합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `model` | MODEL | 예 | - | 가이던스에 사용할 모델 |
| `cond1` | CONDITIONING | 예 | - | 첫 번째 양수 조건 입력 |
| `cond2` | CONDITIONING | 예 | - | 두 번째 양수 조건 입력 |
| `negative` | CONDITIONING | 예 | - | 음수 조건 입력 |
| `cfg_conds` | FLOAT | 예 | 0.0 - 100.0 | 첫 번째 양수 조건에 대한 가이던스 스케일 (기본값: 8.0) |
| `cfg_cond2_negative` | FLOAT | 예 | 0.0 - 100.0 | 두 번째 양수 및 음수 조건에 대한 가이던스 스케일 (기본값: 8.0) |
| `style` | COMBO | 예 | "regular"<br />"nested" | 적용할 가이던스 스타일 (기본값: "regular"). "nested"로 설정하면 가이던스가 중첩된 방식으로 적용됩니다 |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `GUIDER` | GUIDER | 사용 가능한 구성된 가이던스 시스템 |
