이 노드는 수학적 정규화 과정을 통해 입력 이미지의 픽셀 값을 조정합니다. 각 픽셀에서 지정된 평균값을 뺀 후, 그 결과를 지정된 표준편차로 나눕니다. 이는 이미지 데이터를 다른 머신러닝 모델에 준비시키기 위한 일반적인 전처리 단계입니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `image` | IMAGE | 예 | - | 정규화할 입력 이미지입니다. |
| `mean` | FLOAT | 아니요 | 0.0 - 1.0 | 정규화용 평균값 (기본값: 0.5). |
| `std` | FLOAT | 아니요 | 0.001 - 1.0 | 정규화용 표준편차 (기본값: 0.5). |

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `image` | IMAGE | 정규화 과정이 적용된 결과 이미지입니다. |
