`FeatherMask` 노드는 주어진 마스크의 가장자리에 퍼짐 효과를 적용하여, 각 가장자리에서 지정된 거리에 따라 불투명도를 조정함으로써 마스크의 가장자리를 부드럽게 전환합니다. 이로 인해 더욱 부드럽고 자연스러운 경계 효과를 얻을 수 있습니다.

## 입력

| 매개변수 | 데이터 유형 | 설명 |
|-----------|--------------|-------------|
| `mask`    | MASK         | 퍼짐 효과가 적용될 마스크입니다. 이미지의 어느 부분이 퍼짐 효과의 영향을 받을지 결정합니다. |
| `left`    | INT          | 왼쪽 가장자리로부터 적용될 퍼짐 효과의 거리를 지정합니다. |
| `top`     | INT          | 위쪽 가장자리로부터 적용될 퍼짐 효과의 거리를 지정합니다. |
| `right`   | INT          | 오른쪽 가장자리로부터 적용될 퍼짐 효과의 거리를 지정합니다. |
| `bottom`  | INT          | 아래쪽 가장자리로부터 적용될 퍼짐 효과의 거리를 지정합니다. |

## 출력

| 매개변수 | 데이터 유형 | 설명 |
|-----------|--------------|-------------|
| `mask`    | MASK         | 출력은 입력 마스크를 수정한 버전으로, 가장자리에 퍼짐 효과가 적용되었습니다. |
