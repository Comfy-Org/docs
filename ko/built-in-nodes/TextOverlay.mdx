이 노드는 이미지 또는 이미지 배치 위에 텍스트를 그립니다. 사용자 정의 가능한 글꼴 크기, 색상, 위치, 정렬 및 선택적 윤곽선을 사용하여 텍스트 오버레이를 생성한 후, 텍스트를 원본 이미지 위에 합성합니다.

## 입력

| 매개변수 | 설명 | 데이터 타입 | 필수 | 범위 |
|-----------|-------------|-----------|----------|-------|
| `images` | 텍스트를 그릴 입력 이미지 또는 이미지 배치 | IMAGE | 예 | |
| `text` | 이미지 위에 오버레이할 텍스트 (기본값: "") | STRING | 예 | |
| `font_size` | 이미지 높이 대비 글꼴 크기 백분율 (기본값: 5.0) | FLOAT | 예 | 0.5 ~ 50.0 |
| `color` | 텍스트 색상 (기본값: "#ffffff") | STRING | 예 | |
| `position` | 이미지에서 텍스트의 세로 위치 (기본값: "top") | COMBO | 예 | `"top"`<br />`"bottom"` |
| `align` | 텍스트의 가로 정렬 (기본값: "left") | COMBO | 예 | `"left"`<br />`"center"`<br />`"right"` |
| `outline` | 텍스트 주위에 검은색 윤곽선 그리기 (기본값: True) | BOOLEAN | 예 | |

## 출력

| 출력 이름 | 설명 | 데이터 타입 |
|-------------|-------------|-----------|
| `images` | 텍스트 오버레이가 합성된 입력 이미지 | IMAGE |
