이 노드는 이미지 또는 이미지 배치 위에 텍스트를 그립니다. 사용자 지정 텍스트, 글꼴 크기, 색상, 위치, 정렬 및 선택적 윤곽선을 지원하며, 원본 이미지 위에 텍스트를 합성합니다.

## 입력

| 매개변수 | 설명 | 데이터 타입 | 필수 | 범위 |
|-----------|-------------|-----------|----------|-------|
| `images` | 텍스트를 그릴 입력 이미지 또는 이미지 배치입니다. | IMAGE | 예 | |
| `text` | 오버레이할 텍스트 문자열입니다. 줄바꿈(`\n`) 및 탭(`\t`) 문자를 지원합니다. 기본값: "" | STRING | 예 | |
| `font_size` | 이미지 높이 대비 백분율로 표시된 글꼴 크기입니다. 기본값: 5.0 | FLOAT | 예 | 0.5 ~ 50.0 |
| `color` | 텍스트의 색상입니다. 기본값: "#ffffff"(흰색) | STRING | 예 | |
| `position` | 이미지에서 텍스트의 세로 위치입니다. 기본값: "top" | COMBO | 예 | "top"<br />"bottom" |
| `align` | 텍스트의 가로 정렬입니다. 기본값: "left" | COMBO | 예 | "left"<br />"center"<br />"right" |
| `outline` | 텍스트 주위에 검은색 윤곽선을 그립니다. 기본값: True | BOOLEAN | 예 | |

## 출력

| 출력 이름 | 설명 | 데이터 타입 |
|-------------|-------------|-----------|
| `images` | 텍스트 오버레이가 합성된 이미지 또는 이미지 배치입니다. | IMAGE |
