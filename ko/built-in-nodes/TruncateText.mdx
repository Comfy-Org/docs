이 노드는 지정된 최대 길이로 텍스트를 잘라내어 짧게 만듭니다. 어떤 입력 텍스트든 받아들여 설정한 문자 수까지의 앞부분만 반환합니다. 특정 크기를 초과하지 않도록 텍스트를 간단히 조정하는 방법입니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `text` | STRING | 예 | N/A | 잘라낼 텍스트 문자열입니다. |
| `max_length` | INT | 예 | 1~10000 | 최대 텍스트 길이입니다. 이 문자 수 이후부터 텍스트가 잘려나갑니다(기본값: 77). |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `string` | STRING | 잘려진 텍스트로, 입력 텍스트에서 처음 `max_length`자만 포함됩니다. |
