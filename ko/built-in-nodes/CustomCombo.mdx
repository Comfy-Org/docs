Custom Combo 노드를 사용하면 자신만의 텍스트 옵션 목록으로 맞춤형 드롭다운 메뉴를 만들 수 있습니다. 이 노드는 프론트엔드 중심의 노드로, 워크플로우 내에서 호환성을 보장하기 위해 백엔드 표현을 제공합니다. 드롭다운에서 옵션을 선택하면 노드는 해당 텍스트를 문자열로 출력하고, 해당 인덱스 위치도 함께 출력합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `choice` | COMBO | 예 | 사용자 정의 | 맞춤형 드롭다운에서 선택한 텍스트 옵션입니다. 사용 가능한 옵션 목록은 노드의 프론트엔드 인터페이스에서 사용자가 정의합니다. |
| `index` | INT | 아니요 | 0 | 인덱스를 지정하는 데 사용할 수 있는 정수 값입니다. 기본값: 0. |

**참고:** 이 노드의 입력 검증은 의도적으로 비활성화되어 있습니다. 이를 통해 백엔드에서 미리 정의된 목록에 속한 선택인지 확인하지 않고도 프론트엔드에서 원하는 모든 맞춤형 텍스트 옵션을 정의할 수 있습니다.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `STRING` | STRING | 맞춤형 콤보 박스에서 선택한 옵션의 텍스트 문자열입니다. |
| `INDEX` | INT | 드롭다운 목록에서 선택한 옵션의 인덱스 위치입니다. |
