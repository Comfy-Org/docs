## 개요

이 노드는 Luma UNI-1 모델을 사용하여 텍스트 설명으로부터 이미지를 생성합니다. 텍스트 프롬프트와 가로세로 비율 및 스타일과 같은 선택적 설정을 받아 Luma API로 요청을 보내 이미지를 생성합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `prompt` | 문자열 | 예 | 1~6000자 | 원하는 이미지의 텍스트 설명. |
| `model` | 콤보 | 예 | `"uni-1"`<br />`"uni-1-max"` | 생성에 사용할 모델. 모델을 선택하면 해당 모델에 대한 추가 설정이 표시됩니다. |
| `seed` | 정수 | 예 | 0~2147483647 | 시드는 노드가 다시 실행될지 여부를 제어하며, 시드와 상관없이 결과는 비결정적입니다. (기본값: 0) |

### 모델별 입력

`model` 매개변수로 `"uni-1"` 또는 `"uni-1-max"`를 선택한 경우 다음 입력이 사용 가능해집니다:

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `aspect_ratio` | 콤보 | 예 | `"auto"`<br />`"3:1"`<br />`"2:1"`<br />`"16:9"`<br />`"3:2"`<br />`"1:1"`<br />`"2:3"`<br />`"9:16"`<br />`"1:2"`<br />`"1:3"` | 출력 이미지의 가로세로 비율. `"auto"`는 프롬프트에 따라 모델이 선택하도록 합니다. (기본값: `"auto"`) |
| `style` | 콤보 | 예 | `"auto"`<br />`"manga"` | 생성된 이미지의 시각적 스타일. (기본값: `"auto"`) |
| `web_search` | 부울 | 예 | true / false | 모델이 추가적인 문맥을 위해 웹 검색을 허용할지 여부. (기본값: false) |
| `image_ref` | 이미지 | 아니오 | 최대 9개의 이미지 | 생성을 안내할 참조 이미지. |

**`style` 및 `aspect_ratio` 제약 사항 참고:** `style`을 `"manga"`로 설정한 경우 `aspect_ratio`는 `"auto"`이거나 다음 세로 비율 중 하나여야 합니다: `"2:3"`, `"9:16"`, `"1:2"`, `"1:3"`. `"manga"` 스타일과 함께 가로 또는 정사각형 비율을 사용하면 오류가 발생합니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `image` | 이미지 | 생성된 이미지를 텐서로 반환합니다. |
