이 노드는 이미지 주변에 패딩을 추가하여 아웃페인팅 과정에 적합한 이미지를 준비하도록 설계되었습니다. 이미지 크기를 조정해 아웃페인팅 알고리즘과의 호환성을 높이고, 원래 이미지 경계를 넘어 확장된 이미지 영역을 생성하는 데 도움을 줍니다.

## 입력

| 매개변수 | 데이터 타입 | 설명 |
|-----------|-------------|-------------|
| `image`   | `IMAGE`     | 'image' 입력은 아웃페인팅을 위해 준비할 기본 이미지로, 패딩 작업의 기준이 됩니다. |
| `left`    | `INT`       | 이미지의 왼쪽에 추가할 패딩의 양을 지정하며, 아웃페인팅을 위한 확장 영역에 영향을 줍니다. |
| `top`     | `INT`       | 이미지의 상단에 추가할 패딩의 양을 결정하며, 아웃페인팅을 위한 수직 확장에 영향을 줍니다. |
| `right`   | `INT`       | 이미지의 오른쪽에 추가할 패딩의 양을 정의하며, 아웃페인팅을 위한 수평 확장에 영향을 줍니다. |
| `bottom`  | `INT`       | 이미지의 하단에 추가할 패딩의 양을 나타내며, 아웃페인팅을 위한 수직 확장에 기여합니다. |
| `feathering` | `INT` | 원본 이미지와 추가된 패딩 간의 전환 부드러움을 제어하며, 아웃페인팅 시 시각적 통합성을 높여줍니다. |

## 출력

| 매개변수 | 데이터 타입 | 설명 |
|-----------|-------------|-------------|
| `image`   | `IMAGE`     | 출력 'image'는 패딩이 추가된 이미지를 나타내며, 아웃페인팅 과정에 바로 사용할 수 있습니다. |
| `mask`    | `MASK`      | 출력 'mask'는 원본 이미지와 추가된 패딩 영역을 표시하며, 아웃페인팅 알고리즘의 안내에 유용합니다. |
