> 이 문서는 AI로 생성되었습니다. 오류를 발견하거나 개선할 점이 있으면 언제든지 기여해 주세요! [GitHub에서 편집하기](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/HitPawVideoEnhance/en.md)

HitPaw Video Enhance 노드는 외부 API를 사용하여 동영상의 품질을 향상시킵니다. 저해상도 동영상을 고해상도로 업스케일링하고, 시각적 아티팩트를 제거하며, 노이즈를 줄입니다. 처리 비용은 입력 동영상의 초당 길이에 따라 계산됩니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `model` | DYNAMIC COMBO | 예 | `"Portrait Restore Model (1x)"`<br />`"Portrait Restore Model (2x)"`<br />`"General Restore Model (1x)"`<br />`"General Restore Model (2x)"`<br />`"General Restore Model (4x)"`<br />`"Ultra HD Model (2x)"`<br />`"Generative Model (1x)"` | 동영상 향상에 사용할 AI 모델입니다. 모델을 선택하면 중첩된 `resolution` 매개변수가 나타납니다. 사용 가능한 모델과 지원 해상도는 다릅니다. |
| `model.resolution` | COMBO | 예 | `"Generative Model (1x)"`의 경우: `"original"`<br />`"720p"`<br />`"1080p"`<br />`"2K/QHD"`<br />`"4K/UHD"`<br /><br />다른 모든 모델의 경우: `"original"`<br />`"720p"`<br />`"1080p"`<br />`"2K/QHD"`<br />`"4K/UHD"`<br />`"8K"` | 향상된 동영상의 목표 해상도입니다. `"8K"` 옵션은 `"Generative Model (1x)"`가 아닌 모델에서만 사용 가능합니다. |
| `video` | VIDEO | 예 | N/A | 향상시킬 입력 동영상 파일입니다. |

**제약사항:**

* 입력 `video`의 길이는 0.5초에서 60분(3600초) 사이여야 합니다.
* 선택한 `resolution`은 입력 동영상의 크기보다 커야 합니다. 동영상이 정사각형인 경우, 선택한 해상도는 가로/세로 크기보다 커야 합니다. 정사각형이 아닌 동영상의 경우, 선택한 해상도는 동영상의 짧은 쪽 크기보다 커야 합니다. 목표 해상도가 더 작으면 오류가 발생합니다. 입력 동영상의 해상도를 유지하려면 `"original"`을 선택하세요.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `video` | VIDEO | 향상된 동영상 파일입니다. |
