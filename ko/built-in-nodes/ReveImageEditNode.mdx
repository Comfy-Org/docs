Reve Image Edit 노드를 사용하면 텍스트 설명을 기반으로 기존 이미지를 수정할 수 있습니다. Reve API를 이용해 사용자의 지시를 해석하고 제공한 이미지에 요청된 변경 사항을 적용합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `image` | IMAGE | 예 | - | 수정할 이미지입니다. |
| `edit_instruction` | STRING | 예 | - | 이미지를 어떻게 수정할지에 대한 텍스트 설명입니다. 최대 2560자입니다. |
| `model` | MODEL | 예 | `"reve-edit@20250915"`<br />`"reve-edit-fast@20251030"` | 수정에 사용할 모델 버전입니다. |
| `model.aspect_ratio` | COMBO | 아니요 | `"auto"`<br />`"16:9"`<br />`"9:16"`<br />`"3:2"`<br />`"2:3"`<br />`"4:3"`<br />`"3:4"`<br />`"1:1"` | 수정된 이미지의 종횡비입니다. "auto"로 설정하면 종횡비가 자동으로 결정됩니다. |
| `model.test_time_scaling` | FLOAT | 아니요 | - | 모델의 테스트 시간 스케일링 요인입니다. 값이 높을수록 품질은 향상되지만 처리 시간도 늘어납니다. |
| `upscale` | COMBO | 아니요 | `"disabled"`<br />`"enabled"` | 생성된 이미지를 확대할지 여부를 제어합니다. |
| `upscale.upscale_factor` | FLOAT | 아니요 | - | 확대가 활성화된 경우 이미지를 몇 배로 확대할지 결정하는 요인입니다. |
| `remove_background` | BOOLEAN | 아니요 | - | 생성된 이미지에서 배경을 제거할지 여부를 제어합니다. |
| `seed` | INT | 아니요 | 0부터 2147483647까지 | 시드는 노드가 다시 실행될지 여부를 제어하며, 시드와 관계없이 결과는 비결정적입니다. (기본값: 0) |

**참고:** `upscale.upscale_factor` 매개변수는 `upscale` 매개변수가 "enabled"로 설정된 경우에만 관련이 있습니다.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `image` | IMAGE | 지침에 따라 생성된 수정된 이미지입니다. |
