PerpNegGuider 노드는 수직적 부정 조건부를 사용해 이미지 생성을 제어하는 가이던스 시스템을 생성합니다. 이 노드는 긍정적, 부정적, 빈 조건부 입력을 받아 효율성을 위해 한 번의 배치에서 세 가지 노이즈 예측을 모두 계산하는 특수한 가이던스 알고리즘을 적용합니다. 이 노드는 실험적 테스트를 위해 설계되었으며, 가이던스 강도와 부정 스케일링을 세밀하게 제어할 수 있습니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `model` | MODEL | 예 | - | 가이던스 생성에 사용할 모델 |
| `positive` | CONDITIONING | 예 | - | 원하는 콘텐츠를 향해 생성을 유도하는 긍정적 조건부 |
| `negative` | CONDITIONING | 예 | - | 원치 않는 콘텐츠를 멀리하도록 유도하는 부정적 조건부 |
| `empty_conditioning` | CONDITIONING | 예 | - | 수직적 부정 가이던스의 기준점으로 사용되는 빈 또는 중립적 조건부 |
| `cfg` | FLOAT | 예 | 0.0 - 100.0 | 조건부가 생성에 얼마나 강하게 영향을 미치는지를 제어하는 분류기 없는 가이던스 스케일 (기본값: 8.0) |
| `neg_scale` | FLOAT | 예 | 0.0 - 100.0 | 수직적 부정 효과의 강도를 조정하는 부정 스케일링 요소 (기본값: 1.0) |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `guider` | GUIDER | 생성 파이프라인에서 바로 사용할 수 있도록 구성된 가이던스 시스템 |
