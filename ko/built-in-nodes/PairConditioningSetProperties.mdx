**PairConditioningSetProperties** 노드를 사용하면 양성 및 음성 조건화 쌍의 속성을 동시에 수정할 수 있습니다. 이 노드는 강도 조정, 조건화 영역 설정, 그리고 선택적 마스크 또는 타이밍 제어를 두 조건화 입력에 적용하며, 수정된 양성 및 음성 조건화 데이터를 반환합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `positive_NEW` | CONDITIONING | 예 | - | 수정할 양성 조건화 입력 |
| `negative_NEW` | CONDITIONING | 예 | - | 수정할 음성 조건화 입력 |
| `strength` | FLOAT | 예 | 0.0부터 10.0까지 | 조건화에 적용되는 강도 승수 (기본값: 1.0) |
| `set_cond_area` | STRING | 예 | "default"<br />"mask bounds" | 조건화 영역 계산 방식 결정 (기본값: "default") |
| `mask` | MASK | 아니요 | - | 조건화 영역을 제한하는 선택적 마스크 |
| `hooks` | HOOKS | 아니요 | - | 고급 조건화 수정을 위한 선택적 후크 그룹 |
| `timesteps` | TIMESTEPS_RANGE | 아니요 | - | 조건화 적용 시점을 제한하는 선택적 타임스텝 범위 |

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `positive` | CONDITIONING | 적용된 속성이 반영된 수정된 양성 조건화 |
| `negative` | CONDITIONING | 적용된 속성이 반영된 수정된 음성 조건화 |
