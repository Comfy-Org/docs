SamplerDPMPP_2M_SDE 노드는 확산 모델용 DPM++ 2M SDE 샘플러를 생성합니다. 이 샘플러는 2차 미분 방정식 해법과 확률적 미분 방정식을 사용하여 샘플을 생성합니다. 샘플링 과정을 제어하기 위해 다양한 해법 유형과 노이즈 처리 옵션을 제공합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `solver_type` | 문자열 | 예 | `"midpoint"`<br />`"heun"` | 샘플링 과정에 사용할 미분 방정식 해법의 유형 |
| `eta` | 부동소수점 | 예 | 0.0 - 100.0 | 샘플링 과정의 확률성을 제어합니다(기본값: 1.0) |
| `s_noise` | 부동소수점 | 예 | 0.0 - 100.0 | 샘플링 중 추가되는 노이즈의 양을 제어합니다(기본값: 1.0) |
| `noise_device` | 문자열 | 예 | `"gpu"`<br />`"cpu"` | 노이즈 계산이 수행되는 장치입니다. "cpu"로 설정하면 CPU 기반 노이즈 생성을 사용하고, "gpu"로 설정하면 GPU 기반 노이즈 생성을 사용해 더 빠른 성능을 얻을 수 있습니다(기본값: "gpu") |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `sampler` | 샘플러 | 샘플링 파이프라인에서 바로 사용할 수 있도록 구성된 샘플러 객체 |
