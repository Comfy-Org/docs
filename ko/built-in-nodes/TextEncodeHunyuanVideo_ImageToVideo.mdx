TextEncodeHunyuanVideo_ImageToVideo 노드는 텍스트 프롬프트와 이미지 임베딩을 결합하여 비디오 생성을 위한 조건 데이터를 생성합니다. CLIP 모델을 사용해 텍스트 입력과 CLIP 비전 출력의 시각적 정보를 처리한 후, 지정된 이미지 교차 설정에 따라 두 소스를 혼합한 토큰을 생성합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `clip` | CLIP | 예 | - | 토크나이징 및 인코딩에 사용되는 CLIP 모델 |
| `clip_vision_output` | CLIP_VISION_OUTPUT | 예 | - | CLIP 비전 모델에서 제공하는 시각적 임베딩으로, 이미지 컨텍스트를 제공합니다 |
| `prompt` | 문자열 | 예 | - | 비디오 생성을 안내하는 텍스트 설명입니다. 다중 줄 입력과 동적 프롬프트를 지원합니다. 프롬프트는 참조 이미지를 기반으로 비디오를 설명하도록 모델에 요청하는 템플릿 형식으로 작성되며, 주요 내용, 객체 세부사항, 행동, 배경, 카메라 앵글 등의 요소를 포함합니다. |
| `image_interleave` | 정수 | 예 | 1-512 | 이미지가 텍스트 프롬프트보다 얼마나 영향을 미치는지 설정합니다. 숫자가 높을수록 텍스트 프롬프트의 영향력이 더 커집니다. (기본값: 2) |

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `CONDITIONING` | CONDITIONING | 비디오 생성을 위해 텍스트와 이미지 정보를 결합한 조건 데이터 |
