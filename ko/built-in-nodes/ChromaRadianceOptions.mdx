ChromaRadianceOptions 노드를 사용하면 Chroma Radiance 모델의 고급 설정을 구성할 수 있습니다. 이 노드는 기존 모델을 래핑하고, 시그마 값에 따라 노이즈 제거 과정 중 특정 옵션을 적용하여 NeRF 타일 크기와 기타 복사광 관련 매개변수를 세밀하게 조정할 수 있게 해줍니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `model` | MODEL | 예 | - | Chroma Radiance 옵션을 적용할 모델 |
| `preserve_wrapper` | BOOLEAN | 아니요 | - | 활성화 시, 기존 모델 함수 래퍼가 존재하는 경우 이를 위임합니다. 일반적으로 활성화된 상태로 두는 것이 좋습니다. (기본값: True) |
| `start_sigma` | FLOAT | 아니요 | 0.0~1.0 | 해당 옵션이 적용되는 첫 번째 시그마 값입니다. (기본값: 1.0) |
| `end_sigma` | FLOAT | 아니요 | 0.0~1.0 | 해당 옵션이 적용되는 마지막 시그마 값입니다. (기본값: 0.0) |
| `nerf_tile_size` | INT | 아니요 | -1 이상 | 기본 NeRF 타일 크기를 재정의할 수 있습니다. -1은 기본값(32)을 사용함을 의미합니다. 0은 비타일링 모드를 사용함을 의미하며, 이 경우 많은 VRAM이 필요할 수 있습니다. (기본값: -1) |

**참고:** Chroma Radiance 옵션은 현재 시그마 값이 `end_sigma`와 `start_sigma` 사이에 포함될 때만 적용됩니다. `nerf_tile_size` 매개변수는 0 또는 그 이상의 값으로 설정된 경우에만 적용됩니다.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `model` | MODEL | Chroma Radiance 옵션이 적용된 수정된 모델 |
