이 노드는 ComfyUI의 입력 디렉터리 내 특정 하위 폴더에서 여러 이미지를 로드합니다. 선택한 폴더를 스캔하여 일반적인 이미지 파일 형식을 찾고 이를 리스트로 반환하므로, 일괄 처리나 데이터셋 준비에 유용합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `folder` | 문자열 | 예 | *다양한 옵션 제공* | 이미지를 로드할 폴더입니다. 옵션은 ComfyUI의 기본 입력 디렉터리에 존재하는 하위 폴더들입니다. |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `images` | 이미지 | 로드된 이미지 목록입니다. 노드는 선택한 폴더에서 발견된 유효한 이미지 파일(PNG, JPG, JPEG, WEBP)을 모두 로드합니다. |
