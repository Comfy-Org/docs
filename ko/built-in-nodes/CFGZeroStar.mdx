CFGZeroStar 노드는 확산 모델에 특화된 가이던스 스케일링 기법을 적용합니다. 조건부 예측과 무조건적 예측 간의 차이를 기반으로 최적화된 스케일 팩터를 계산하여 분류기 없는 가이던스 과정을 수정합니다. 이 방법은 최종 출력을 조정해 생성 과정에 대한 제어력을 강화하면서도 모델의 안정성을 유지합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `model` | MODEL | 예 | - | CFGZeroStar 가이던스 스케일링 기법으로 수정될 확산 모델 |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `patched_model` | MODEL | CFGZeroStar 가이던스 스케일링이 적용된 수정된 모델 |
