RenormCFG 노드는 조건부 스케일링과 정규화를 적용하여 확산 모델의 분류자 없는 안내(CFG) 과정을 수정합니다. 지정된 타임스텝 임계값과 재정규화 계수에 따라 노이즈 제거 과정을 조정하여 이미지 생성 중 조건부 예측과 무조건적 예측 간의 영향력을 제어합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `model` | MODEL | 예 | - | 재정규화된 CFG를 적용할 확산 모델 |
| `cfg_trunc` | FLOAT | 아니요 | 0.0 - 100.0 | CFG 스케일링을 적용할 타임스텝 임계값. 현재 타임스텝이 이 값보다 낮으면 CFG 스케일링이 적용되고, 그렇지 않으면 조건부 예측만 사용됩니다(기본값: 100.0) |
| `renorm_cfg` | FLOAT | 아니요 | 0.0 - 100.0 | CFG 스케일링된 예측의 최대 노름을 원래 조건부 예측과 비교해 제한하는 재정규화 계수. 값이 0.0이면 재정규화가 비활성화됩니다(기본값: 1.0) |

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `model` | MODEL | 재정규화된 CFG 기능이 적용된 수정된 모델 |
