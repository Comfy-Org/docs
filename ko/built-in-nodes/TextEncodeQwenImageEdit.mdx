TextEncodeQwenImageEdit 노드는 텍스트 프롬프트와 선택적 이미지를 처리하여 이미지 생성 또는 편집을 위한 컨디셔닝 데이터를 생성합니다. 이 노드는 CLIP 모델을 사용해 입력을 토큰화하며, 선택적으로 VAE를 사용해 참조 이미지를 인코딩하여 참조 랜트를 생성할 수 있습니다. 이미지를 제공하면 노드가 자동으로 이미지를 리사이즈하여 일정한 처리 크기를 유지합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `clip` | CLIP | 예 | - | 텍스트 및 이미지 토큰화에 사용되는 CLIP 모델 |
| `prompt` | 문자열 | 예 | - | 컨디셔닝 생성을 위한 텍스트 프롬프트, 다중 줄 입력 및 동적 프롬프트 지원 |
| `vae` | VAE | 아니요 | - | 참조 이미지를 랜트로 인코딩하는 데 사용되는 선택적 VAE 모델 |
| `image` | 이미지 | 아니요 | - | 참조 또는 편집 목적으로 제공되는 선택적 입력 이미지 |

**참고:** `image`와 `vae`가 모두 제공되면 노드는 이미지를 참조 랜트로 인코딩하고 이를 컨디셔닝 출력에 첨부합니다. 이미지는 자동으로 리사이즈되어 약 1024x1024 픽셀의 일정한 처리 스케일을 유지합니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-----------|-----------|-------------|
| `CONDITIONING` | CONDITIONING | 텍스트 토큰과 선택적 참조 랜트를 포함한 이미지 생성용 컨디셔닝 데이터 |
