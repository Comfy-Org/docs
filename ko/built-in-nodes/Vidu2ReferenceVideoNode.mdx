> 이 문서는 AI로 생성되었습니다. 오류를 발견하거나 개선할 점이 있다면 언제든지 기여해 주세요! [GitHub에서 편집하기](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/Vidu2ReferenceVideoNode/en.md)

Vidu2 참조 동영상 생성 노드는 텍스트 프롬프트와 여러 참조 이미지를 바탕으로 동영상을 생성합니다. 최대 7개의 주제를 정의할 수 있으며, 각 주제마다 고유한 참조 이미지 세트를 제공하고, 프롬프트에서 `@subject{subject_id}`를 사용해 참조할 수 있습니다. 이 노드는 지정된 지속 시간, 종횡비 및 움직임을 설정하여 동영상을 생성합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `model` | COMBO | 예 | `"viduq2"` | 동영상 생성에 사용할 AI 모델입니다. |
| `subjects` | AUTOGROW | 예 | N/A | 각 주제별로 최대 3개의 참조 이미지를 제공하세요(모든 주제 합쳐 총 7개 이미지). 프롬프트에서 `@subject{subject_id}`를 통해 참조하세요. |
| `prompt` | STRING | 예 | N/A | 동영상 생성을 안내하는 텍스트 설명입니다. `audio` 매개변수가 활성화되면, 이 프롬프트를 기반으로 생성된 음성과 배경음악이 포함된 동영상이 됩니다. |
| `audio` | BOOLEAN | 아니요 | N/A | 활성화하면, 프롬프트를 기반으로 생성된 음성과 배경음악이 포함된 동영상이 됩니다(기본값: `False`). |
| `duration` | INT | 아니요 | 1~10 | 생성된 동영상의 길이(초)(기본값: `5`). |
| `seed` | INT | 아니요 | 0~2147483647 | 재현 가능한 결과를 위해 생성의 무작위성을 제어하는 데 사용되는 숫자입니다(기본값: `1`). |
| `aspect_ratio` | COMBO | 아니요 | `"16:9"`<br />`"9:16"`<br />`"4:3"`<br />`"3:4"`<br />`"1:1"` | 동영상 프레임의 비율입니다. |
| `resolution` | COMBO | 아니요 | `"720p"`<br />`"1080p"` | 출력 동영상의 픽셀 해상도입니다(기본값: `"720p"`). |
| `movement_amplitude` | COMBO | 아니요 | `"auto"`<br />`"small"`<br />`"medium"`<br />`"large"` | 프레임 내 객체의 움직임 진폭을 제어합니다(기본값: `"auto"`). |

**제약사항:**

* `prompt`는 1~2000자 사이여야 합니다.
* 여러 주제를 정의할 수 있지만, 모든 주제를 합친 참조 이미지의 총 개수는 7개를 초과해서는 안 됩니다.
* 각 주제별로 참조 이미지는 최대 3개까지 가능합니다.
* 각 참조 이미지는 가로세로 비율이 1:4에서 4:1 사이여야 합니다.
* 각 참조 이미지는 가로와 세로가 각각 최소 128픽셀 이상이어야 합니다.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-----------|-----------|-------------|
| `output` | VIDEO | 생성된 동영상 파일입니다. |
