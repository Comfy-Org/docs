> 이 문서는 AI로 생성되었습니다. 오류를 발견하거나 개선할 점이 있다면 언제든지 기여해 주세요! [GitHub에서 편집하기](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/RecraftTextToImageNode/en.md)

프롬프트와 해상도를 기반으로 이미지를 동기식으로 생성합니다. 이 노드는 Recraft API와 연결되어 지정된 크기와 선택적 스타일 및 제어 매개변수를 사용해 텍스트 설명으로부터 이미지를 생성합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `prompt` | STRING | 예 | - | 이미지 생성을 위한 프롬프트입니다. (기본값: "") |
| `size` | COMBO | 예 | "1024x1024"<br />"1152x896"<br />"896x1152"<br />"1216x832"<br />"832x1216"<br />"1344x768"<br />"768x1344"<br />"1536x640"<br />"640x1536" | 생성될 이미지의 크기입니다. (기본값: "1024x1024") |
| `n` | INT | 예 | 1-6 | 생성할 이미지의 수입니다. (기본값: 1) |
| `seed` | INT | 예 | 0-18446744073709551615 | 노드가 다시 실행될지 여부를 결정하는 시드입니다. 실제 결과는 시드와 관계없이 비결정적입니다. (기본값: 0) |
| `recraft_style` | RECRAFT_STYLE | 아니요 | 여러 옵션 제공 | 이미지 생성을 위한 선택적 스타일입니다. 제공되지 않으면 기본적으로 "realistic_image" 스타일이 적용됩니다. |
| `negative_prompt` | STRING | 아니요 | - | 이미지에 포함되지 않기를 원하는 요소에 대한 선택적 텍스트 설명입니다. (기본값: "") |
| `recraft_controls` | RECRAFT_CONTROLS | 아니요 | 여러 옵션 제공 | Recraft Controls 노드를 통해 생성 과정을 추가로 제어할 수 있습니다. |

**참고:** `seed` 매개변수는 노드가 다시 실행될 때만 제어하며, 이미지 생성을 결정적으로 만드는 것은 아닙니다. 같은 시드 값이라도 실제 출력 이미지는 달라질 수 있습니다.

**참고:** `prompt` 매개변수는 길이가 1~1000자 사이여야 합니다.

**참고:** Infinite Style Library의 `style_id`를 사용할 경우, 벡터 아트 스타일이 아닌지 확인하세요. 그렇지 않으면 이미지 대신 SVG 데이터가 반환되어 오류가 발생할 수 있습니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `IMAGE` | IMAGE | 생성된 이미지들을 배치된 텐서 형태로 출력합니다. 이미지가 여러 개 생성될 경우(n > 1), 배치 차원을 따라 연결됩니다. |
