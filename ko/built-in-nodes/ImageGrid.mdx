Image Grid 노드는 여러 이미지를 하나의 정리된 격자 또는 콜라주로 결합합니다. 이미지 목록을 받아 지정된 열 수로 배열하고, 각 이미지를 정해진 셀 크기에 맞게 리사이즈하며, 선택적으로 이미지 간에 패딩을 추가합니다. 결과는 모든 입력 이미지를 격자 레이아웃으로 포함한 단일 새 이미지입니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `images` | IMAGE | 예 | - | 격자에 배열할 이미지 목록입니다. 노드가 작동하려면 최소 한 개의 이미지가 필요합니다. |
| `columns` | INT | 아니요 | 1 - 20 | 격자의 열 수입니다(기본값: 4). |
| `cell_width` | INT | 아니요 | 32 - 2048 | 격자의 각 셀의 너비(픽셀)입니다(기본값: 256). |
| `cell_height` | INT | 아니요 | 32 - 2048 | 격자의 각 셀의 높이(픽셀)입니다(기본값: 256). |
| `padding` | INT | 아니요 | 0 - 50 | 격자 내 이미지 간에 넣을 패딩의 양(픽셀)입니다(기본값: 4). |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `image` | IMAGE | 모든 입력 이미지를 격자로 배열한 단일 출력 이미지입니다. |
