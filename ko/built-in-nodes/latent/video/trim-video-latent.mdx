TrimVideoLatent 노드는 잠재 공간(LATENT)에서 비디오 프레임을 자르는 데 사용됩니다. 비디오 잠재 시퀀스를 처리할 때, 시작 부분의 원치 않는 프레임을 제거하여 비디오를 '앞쪽으로 자르는' 작업에 일반적으로 사용됩니다.

기본 사용법: 자를 비디오 잠재 데이터를 samples에 입력하고, trim_amount에 자를 프레임 수를 설정하세요. 노드는 비디오의 시작 부분에서 지정된 수만큼 프레임을 잘라내고, 남은 잠재 시퀀스를 출력합니다.

일반적인 시나리오: 비디오 생성, 비디오 편집 등에서 원치 않는 앞부분 프레임을 제거하거나, 다른 노드와 함께 사용해 비디오 세그먼트를 이어붙이고 처리하는 데 활용됩니다.

## 매개변수

### 입력 매개변수
| 매개변수   | 유형   | 필수 | 기본값 | 설명                           |
| ----------- | ------ | -------- | ------- | ------------------------------------- |
| samples     | LATENT | 예      | 없음    | 입력 잠재 비디오 데이터               |
| trim_amount | INT    | 예      | 0       | 자를 프레임 수 (시작부터) |

### 출력 매개변수
| 매개변수 | 유형   | 설명               |
| --------- | ------ | ------------------------- |
| samples   | LATENT | 자른 비디오 잠재 데이터 |

### 소스 코드

```python
class TrimVideoLatent:
    @classmethod
    def INPUT_TYPES(s):
        return {"required": { "samples": ("LATENT",),
                              "trim_amount": ("INT", {"default": 0, "min": 0, "max": 99999}),
                             }}

    RETURN_TYPES = ("LATENT",)
    FUNCTION = "op"

    CATEGORY = "latent/video"

    EXPERIMENTAL = True

    def op(self, samples, trim_amount):
        samples_out = samples.copy()

        s1 = samples["samples"]
        samples_out["samples"] = s1[:, :, trim_amount:]
        return (samples_out,)
```
