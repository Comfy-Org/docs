이 노드를 사용하면 두 이미지를 지정된 방향(위, 아래, 왼쪽, 오른쪽)으로 이어붙일 수 있으며, 크기 맞춤과 이미지 간의 간격 설정도 지원합니다.

## 입력

| 매개변수 이름 | 데이터 타입 | 입력 유형 | 기본값 | 범위 | 설명 |
|---------------|-----------|-------------|---------|--------|-------------|
| `image1` | IMAGE | 필수 | - | - | 이어붙일 첫 번째 이미지 |
| `image2` | IMAGE | 선택 | 없음 | - | 이어붙일 두 번째 이미지로, 제공되지 않으면 첫 번째 이미지만 반환됩니다 |
| `direction` | STRING | 필수 | right | right/down/left/up | 두 번째 이미지를 이어붙이는 방향: 오른쪽, 아래, 왼쪽 또는 위 |
| `match_image_size` | BOOLEAN | 필수 | True | True/False | 두 번째 이미지를 첫 번째 이미지의 크기에 맞춰 조정할지 여부 |
| `spacing_width` | INT | 필수 | 0 | 0-1024 | 이미지 간의 간격 너비로, 반드시 짝수여야 합니다 |
| `spacing_color` | STRING | 필수 | white | white/black/red/green/blue | 이어붙인 이미지 간의 간격 색상 |

> `spacing_color`에서 "white/black" 이외의 색상을 사용할 경우, `match_image_size`가 false로 설정되면 패딩 영역은 검은색으로 채워집니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `IMAGE` | IMAGE | 이어붙인 이미지 |

## 워크플로우 예시

아래 워크플로우에서는 서로 다른 크기의 3개의 입력 이미지를 예시로 사용합니다:

- image1: 500x300
- image2: 400x250
- image3: 300x300

![workflow](/images/built-in-nodes/ImageStitch/workflow.webp)

**첫 번째 이미지 이어붙이기 노드**

- `match_image_size`: false, 이미지는 원본 크기 그대로 이어붙입니다.
- `direction`: up, `image2`는 `image1` 위에 배치됩니다.
- `spacing_width`: 20
- `spacing_color`: 검은색

출력 이미지 1:

![output1](/images/built-in-nodes/ImageStitch/output-1.webp)

**두 번째 이미지 이어붙이기 노드**

- `match_image_size`: true, 두 번째 이미지는 첫 번째 이미지의 높이나 폭에 맞춰 스케일링됩니다.
- `direction`: right, `image3`는 오른쪽에 나타납니다.
- `spacing_width`: 20
- `spacing_color`: 흰색
