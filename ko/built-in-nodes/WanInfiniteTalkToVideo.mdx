WanInfiniteTalkToVideo 노드는 오디오 입력을 기반으로 비디오 시퀀스를 생성합니다. 하나 또는 두 명의 화자로부터 추출된 오디오 특성에 조건부로 설정된 비디오 확산 모델을 사용하여 말하는 얼굴 비디오의 잠재적 표현을 생성합니다. 이 노드는 새로운 시퀀스를 생성하거나 이전 프레임을 사용해 기존 시퀀스를 확장하여 모션 컨텍스트를 제공할 수 있습니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `mode` | COMBO | 예 | `"single_speaker"`<br />`"two_speakers"` | 오디오 입력 모드입니다. `"single_speaker"`는 하나의 오디오 입력을 사용합니다. `"two_speakers"`는 두 번째 화자와 해당 마스크에 대한 입력을 지원합니다. |
| `model` | MODEL | 예 | - | 기본 비디오 확산 모델입니다. |
| `model_patch` | MODELPATCH | 예 | - | 오디오 투영 레이어를 포함한 모델 패치입니다. |
| `positive` | CONDITIONING | 예 | - | 생성을 안내하는 긍정적 조건부입니다. |
| `negative` | CONDITIONING | 예 | - | 생성을 안내하는 부정적 조건부입니다. |
| `vae` | VAE | 예 | - | 이미지를 잠재 공간과 상호 변환하는 데 사용되는 VAE입니다. |
| `width` | INT | 아니요 | 16 - MAX_RESOLUTION | 출력 비디오의 너비(픽셀 단위). 16으로 나누어져야 합니다. (기본값: 832) |
| `height` | INT | 아니요 | 16 - MAX_RESOLUTION | 출력 비디오의 높이(픽셀 단위). 16으로 나누어져야 합니다. (기본값: 480) |
| `length` | INT | 아니요 | 1 - MAX_RESOLUTION | 생성할 프레임 수입니다. (기본값: 81) |
| `clip_vision_output` | CLIPVISIONOUTPUT | 아니요 | - | 추가적인 조건부를 위한 옵션 CLIP 비전 출력입니다. |
| `start_image` | IMAGE | 아니요 | - | 비디오 시퀀스를 초기화하기 위한 선택적 시작 이미지입니다. |
| `audio_encoder_output_1` | AUDIOENCODEROUTPUT | 예 | - | 첫 번째 화자에 대한 특성을 포함한 기본 오디오 인코더 출력입니다. |
| `motion_frame_count` | INT | 아니요 | 1 - 33 | 시퀀스를 확장할 때 모션 컨텍스트로 사용할 이전 프레임의 수입니다. (기본값: 9) |
| `audio_scale` | FLOAT | 아니요 | -10.0 - 10.0 | 오디오 조건부에 적용되는 스케일링 요소입니다. (기본값: 1.0) |
| `previous_frames` | IMAGE | 아니요 | - | 확장할 이전 비디오 프레임입니다. |
| `audio_encoder_output_2` | AUDIOENCODEROUTPUT | 아니요 | - | 두 번째 오디오 인코더 출력입니다. `mode`가 `"two_speakers"`로 설정된 경우 필요합니다. |
| `mask_1` | MASK | 아니요 | - | 첫 번째 화자용 마스크로, 두 개의 오디오 입력을 사용할 경우 필요합니다. |
| `mask_2` | MASK | 아니요 | - | 두 번째 화자용 마스크로, 두 개의 오디오 입력을 사용할 경우 필요합니다. |

**매개변수 제약사항:**

* `mode`가 `"two_speakers"`로 설정된 경우, `audio_encoder_output_2`, `mask_1`, `mask_2` 매개변수가 필수로 변경됩니다.
* `audio_encoder_output_2`가 제공되면, `mask_1`과 `mask_2`도 반드시 제공되어야 합니다.
* `mask_1`과 `mask_2`가 제공되면, `audio_encoder_output_2`도 반드시 제공되어야 합니다.
* `previous_frames`가 제공되면, 이전 프레임의 수가 `motion_frame_count`에 지정된 수 이상이어야 합니다.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `model` | MODEL | 오디오 조건부가 적용된 패치된 모델입니다. |
| `positive` | CONDITIONING | 긍정적 조건부로, 추가적인 컨텍스트(예: 시작 이미지, CLIP 비전)로 수정될 수 있습니다. |
| `negative` | CONDITIONING | 부정적 조건부로, 추가적인 컨텍스트로 수정될 수 있습니다. |
| `latent` | LATENT | 생성된 비디오 시퀀스의 잠재 공간 표현입니다. |
| `trim_image` | INT | 모션 컨텍스트의 시작부터 트리밍해야 할 프레임 수입니다. |
