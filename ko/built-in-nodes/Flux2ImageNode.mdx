## 개요

텍스트 프롬프트와 선택적 참조 이미지를 사용하여 Flux.2 [pro] 또는 Flux.2 [max] 모델을 이용해 이미지를 생성합니다. 이 노드는 요청을 BFL API로 보내고 결과를 폴링한 후 생성된 이미지를 텐서로 반환합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `prompt` | 문자열 | 예 | 없음 | 이미지 생성 또는 편집을 위한 프롬프트 (기본값: 빈 문자열). |
| `model` | 콤보 | 예 | `"Flux.2 [pro]"`<br />`"Flux.2 [max]"` | 사용할 Flux.2 모델 버전입니다. 모델을 선택하면 너비, 높이 및 선택적 참조 이미지에 대한 추가 매개변수가 활성화됩니다. |
| `seed` | 정수 | 예 | 0부터 18446744073709551615까지 | 노이즈 생성에 사용되는 난수 시드입니다. 각 생성 후 무작위로 설정할 수 있습니다 (기본값: 0). |

**추가 매개변수 (모델 선택에 따라 활성화):**

모델을 선택하면 다음 매개변수가 사용 가능해집니다:

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `model.width` | 정수 | 예 | 256부터 1440까지 | 생성된 이미지의 너비(픽셀). |
| `model.height` | 정수 | 예 | 256부터 1440까지 | 생성된 이미지의 높이(픽셀). |
| `model.images` | 이미지 | 아니요 | 0부터 8개까지 | 생성을 안내하는 선택적 참조 이미지입니다. 최대 8개의 이미지를 지원합니다. |

**제약사항:**
- 참조 이미지의 최대 개수는 8개입니다. 8개 이상의 이미지를 제공하면 오류가 발생합니다.
- `model.width`와 `model.height` 값은 생성 비용에 영향을 미칩니다 (소스 코드의 가격 배지 로직 참조).

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `image` | 이미지 | BFL API 결과로부터 다운로드된 생성된 이미지 텐서. |
