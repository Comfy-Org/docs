HunyuanRefinerLatent 노드는 정제 작업을 위해 컨디셔닝과 잠재적 입력을 처리합니다. 이 노드는 양성 및 음성 컨디셔닝 모두에 노이즈 증강을 적용하며, 잠재 이미지 데이터를 통합하여 특정 크기의 새로운 잠재적 출력을 생성하여 추가 처리를 위한 준비를 마칩니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `positive` | CONDITIONING | 예 | - | 처리할 양성 컨디셔닝 입력 |
| `negative` | CONDITIONING | 예 | - | 처리할 음성 컨디셔닝 입력 |
| `latent` | LATENT | 예 | - | 잠재적 표현 입력 |
| `noise_augmentation` | FLOAT | 예 | 0.0 - 1.0 | 적용할 노이즈 증강의 양 (기본값: 0.10) |

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `positive` | CONDITIONING | 노이즈 증강과 잠재 이미지 연결이 적용된 처리된 양성 컨디셔닝 |
| `negative` | CONDITIONING | 노이즈 증강과 잠재 이미지 연결이 적용된 처리된 음성 컨디셔닝 |
| `latent` | LATENT | [batch_size, 32, 높이, 너비, 채널] 크기를 가진 새로운 잠재적 출력 |
