![비디오 생성](/images/built-in-nodes/image/video/create-video.jpg)

CreateVideo 노드는 이미지 시퀀스를 선택적 오디오와 함께 비디오로 결합하는 데 사용됩니다. 프레임 속도(fps)를 설정하여 재생 속도를 조정할 수 있습니다. 이 노드는 일반적으로 생성된 이미지 시퀀스를 표준 비디오 파일로 내보내어 공유 및 후처리에 활용합니다.

## 매개변수 설명

### 입력 매개변수

| 매개변수 | 유형   | 필수 | 기본값 | 설명                                                                |
| --------- | ------ | -------- | ------- | -------------------------------------------------------------------------- |
| images    | IMAGE  | 예     | 없음    | 비디오 생성을 위한 이미지 시퀀스, 순서대로 배열됨                       |
| fps       | FLOAT  | 예     | 30.0    | 비디오 프레임 속도, 초당 표시되는 프레임을 제어하며 일반적인 범위는 1~120    |
| audio     | AUDIO  | 아니요   | 없음    | 선택적 매개변수, 비디오에 추가할 오디오 트랙                   |

images: 이미지 시퀀스를 반드시 입력해야 하며, 그 순서가 비디오 재생 순서를 결정합니다.
fps: 프레임 속도가 높을수록 비디오는 더 부드럽게 보이지만 파일 크기도 커집니다. 일반적인 값은 24, 30, 60 등입니다.
audio: 선택적 매개변수로, 배경 음악이나 나레이션을 추가할 수 있습니다. 오디오가 필요하지 않다면 비워둘 수 있습니다.

### 출력 매개변수
| 출력 | 유형  | 설명      |
| ------ | ----- | ---------------- |
| video  | VIDEO | 생성된 비디오  |

## 소스 코드

[2025년 5월 15일 업데이트]

```python
class CreateVideo(ComfyNodeABC):
    @classmethod
    def INPUT_TYPES(cls):
        return {
            "required": {
                "images": (IO.IMAGE, {"tooltip": "비디오를 생성할 이미지들."}),
                "fps": ("FLOAT", {"default": 30.0, "min": 1.0, "max": 120.0, "step": 1.0}),
            },
            "optional": {
                "audio": (IO.AUDIO, {"tooltip": "비디오에 추가할 오디오."}),
            }
        }

    RETURN_TYPES = (IO.VIDEO,)
    FUNCTION = "create_video"

    CATEGORY = "image/video"
    DESCRIPTION = "이미지로부터 비디오를 생성합니다."

    def create_video(self, images: ImageInput, fps: float, audio: Optional[AudioInput] = None):
        return (VideoFromComponents(
            VideoComponents(
            images=images,
            audio=audio,
            frame_rate=Fraction(fps),
            )
        ),)
```
