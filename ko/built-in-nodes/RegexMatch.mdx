RegexMatch 노드는 텍스트 문자열에 주어진 정규 표현식 패턴과 일치하는 부분이 있는지 확인합니다. 입력 문자열을 검색하여 텍스트 내 어디서든 해당 패턴이 발견되었는지 여부를 나타내는 간단한 예/아니요 결과를 반환합니다. 대소문자 구분 없이 일치하거나 다중 줄 모드와 같은 옵션을 활성화하여 검색 방식을 조정할 수 있습니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `string` | STRING | 예 | - | 일치 항목을 찾을 텍스트 문자열 |
| `regex_pattern` | STRING | 예 | - | 문자열과 일치시킬 정규 표현식 패턴 |
| `case_insensitive` | BOOLEAN | 아니요 | - | 일치 시 대소문자를 무시할지 여부 (기본값: True) |
| `multiline` | BOOLEAN | 아니요 | - | 정규 표현식 일치 시 다중 줄 모드를 활성화할지 여부 (기본값: False) |
| `dotall` | BOOLEAN | 아니요 | - | 정규 표현식 일치 시 dotall 모드를 활성화할지 여부 (기본값: False) |

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `matches` | BOOLEAN | 입력 문자열의 어느 부분과도 정규 표현식 패턴이 일치하면 True를 반환하고, 그렇지 않으면 False를 반환합니다 |
