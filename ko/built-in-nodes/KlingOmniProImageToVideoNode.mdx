> 이 문서는 AI로 생성되었습니다. 오류를 발견하거나 개선할 점이 있다면 언제든지 기여해 주세요! [GitHub에서 편집하기](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/KlingOmniProImageToVideoNode/en.md)

이 노드는 Kling AI 모델을 사용하여 텍스트 프롬프트와 최대 7개의 참조 이미지를 기반으로 비디오를 생성합니다. 비디오의 종횡비, 지속 시간, 해상도를 제어할 수 있으며, 선택적으로 스토리보드를 사용하거나 오디오를 생성할 수도 있습니다. 노드는 요청을 외부 API로 보내고 생성된 비디오를 반환합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `model_name` | COMBO | 예 | `"kling-v3-omni"`<br />`"kling-video-o1"` | 비디오 생성에 사용할 특정 Kling 모델 (기본값: "kling-v3-omni"). |
| `prompt` | STRING | 예 | - | 비디오 내용을 설명하는 텍스트 프롬프트입니다. 긍정적 및 부정적 설명을 모두 포함할 수 있습니다. 텍스트는 자동으로 정규화되며, 1~2500자 사이여야 합니다. 스토리보드가 활성화된 경우 무시됩니다. |
| `aspect_ratio` | COMBO | 예 | `"16:9"`<br />`"9:16"`<br />`"1:1"` | 생성할 비디오의 원하는 종횡비입니다. |
| `duration` | INT | 예 | 3~15 | 비디오의 길이(초)입니다. 슬라이더로 값을 조정할 수 있습니다 (기본값: 5). |
| `reference_images` | IMAGE | 예 | - | 최대 7개의 참조 이미지입니다. 각 이미지는 최소 300x300픽셀이어야 하며, 종횡비는 1:2.5에서 2.5:1 사이여야 합니다. |
| `resolution` | COMBO | 아니요 | `"4k"`<br />`"1080p"`<br />`"720p"` | 비디오의 출력 해상도입니다. 이 매개변수는 선택적입니다 (기본값: "1080p"). |
| `storyboards` | DYNAMIC_COMBO | 아니요 | `"disabled"`<br />`"1 스토리보드"`<br />`"2 스토리보드"`<br />`"3 스토리보드"`<br />`"4 스토리보드"`<br />`"5 스토리보드"`<br />`"6 스토리보드"` | 개별 프롬프트와 지속 시간을 가진 일련의 비디오 세그먼트를 생성합니다. `kling-v3-omni`에서만 지원됩니다. 활성화 시 전역 `prompt`는 무시되며, 모든 스토리보드 세그먼트의 총 지속 시간은 전역 `duration`과 같아야 합니다. |
| `generate_audio` | BOOLEAN | 아니요 | `true`<br />`false` | 비디오용 오디오를 생성합니다. `kling-v3-omni`에서만 지원됩니다 (기본값: false). |
| `seed` | INT | 아니요 | 0~2147483647 | 시드는 노드가 다시 실행될지 여부를 제어합니다. 시드와 상관없이 결과는 비결정적입니다 (기본값: 0). |

**참고:** `reference_images` 입력은 최대 7개의 이미지를 허용합니다. 더 많은 이미지를 제공하면 노드는 오류를 발생시킵니다. 각 이미지는 최소 크기와 종횡비를 검증합니다.

**모델별 제한사항:**
- `kling-video-o1`은 10초 이상의 지속 시간을 지원하지 않습니다.
- `kling-video-o1`은 오디오 생성을 지원하지 않습니다.
- `kling-video-o1`은 4k 해상도를 지원하지 않습니다.
- `kling-video-o1`은 스토리보드를 지원하지 않습니다.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `output` | VIDEO | 생성된 비디오 파일입니다. |
