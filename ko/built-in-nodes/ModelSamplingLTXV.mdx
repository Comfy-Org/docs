ModelSamplingLTXV 노드는 토큰 수를 기반으로 모델에 고급 샘플링 매개변수를 적용합니다. 입력 잠재 변수의 토큰 수에 따라 기본값과 최대 변위 값 간의 선형 보간을 통해 변위 값을 계산합니다. 그런 다음 이 노드는 특화된 모델 샘플링 구성 설정을 만들어 입력 모델에 적용합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `model` | MODEL | 예 | - | 샘플링 매개변수를 적용할 입력 모델 |
| `max_shift` | FLOAT | 예 | 0.0부터 100.0까지 | 선형 보간 계산에 사용되는 최대 변위 값 (기본값: 2.05) |
| `base_shift` | FLOAT | 예 | 0.0부터 100.0까지 | 선형 보간 계산에 사용되는 기본 변위 값 (기본값: 0.95) |
| `latent` | LATENT | 아니요 | - | 변위 계산을 위한 토큰 수를 결정하는 데 사용되는 선택적 잠재 입력입니다. 제공되지 않으면 기본 토큰 수인 4096이 사용됩니다 |

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `model` | MODEL | 샘플링 매개변수가 적용된 수정된 모델 |
