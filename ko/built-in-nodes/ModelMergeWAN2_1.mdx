ModelMergeWAN2_1 노드는 두 개의 WAN2.1 모델을 병합하며, 각 구성 요소를 가중 평균을 이용해 블렌딩합니다. 이 노드는 30블록을 가진 1.3B 모델과 40블록을 가진 14B 모델 등 다양한 모델 크기를 지원하며, 이미지에서 비디오로 변환하는 모델에는 추가적인 이미지 임베딩 구성 요소가 포함되어 특별히 처리됩니다. 각 모델의 구성 요소는 개별적으로 가중치를 설정하여 두 입력 모델 간의 블렌딩 비율을 조정할 수 있습니다.
