TextEncodeAceStepAudio1.5 노드는 AceStepAudio 1.5 모델과 함께 사용하기 위해 텍스트와 오디오 관련 메타데이터를 준비합니다. 설명 태그, 가사 및 음악적 파라미터를 입력받아 CLIP 모델을 사용해 오디오 생성에 적합한 조건화 형식으로 변환합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `clip` | CLIP | 예 | N/A | 입력 텍스트를 토큰화하고 인코딩하는 데 사용되는 CLIP 모델입니다. |
| `tags` | STRING | 예 | N/A | 장르, 분위기, 악기 등 오디오에 대한 설명 태그입니다. 다중 줄 입력 및 동적 프롬프트를 지원합니다. |
| `lyrics` | STRING | 예 | N/A | 오디오 트랙의 가사입니다. 다중 줄 입력 및 동적 프롬프트를 지원합니다. |
| `seed` | INT | 아니요 | 0부터 18446744073709551615까지 | 재현 가능한 생성을 위한 임의의 시드 값입니다. control_after_generate 위젯이 있습니다. 기본값: 0. |
| `bpm` | INT | 아니요 | 10부터 300까지 | 생성된 오디오의 분당 박자 수(BPM)입니다. 기본값: 120. |
| `duration` | FLOAT | 아니요 | 0.0부터 2000.0까지 | 오디오의 원하는 지속 시간(초)입니다. 기본값: 120.0. |
