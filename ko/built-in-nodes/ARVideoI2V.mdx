## 개요

이 노드는 AR(Auto-Regressive) 비디오 모델을 위한 이미지-비디오 생성 설정을 준비합니다. 시작 이미지를 받아 VAE를 사용해 잠재 공간으로 인코딩하고, 인코딩된 이미지를 모델의 구성에 저장합니다. 이를 통해 비디오 샘플링 과정에서 이미지를 첫 프레임으로 사용할 수 있어 별도의 이미지-비디오 모델 아키텍처가 필요하지 않습니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `model` | MODEL | 예 | - | 생성에 사용될 AR 비디오 모델입니다. |
| `vae` | VAE | 예 | - | 시작 이미지를 잠재 공간으로 인코딩하는 데 사용되는 VAE 모델입니다. |
| `start_image` | IMAGE | 예 | - | 생성될 비디오의 첫 프레임 역할을 할 초기 이미지입니다. |
| `width` | INT | 예 | 16~8192 (단계: 16) | 생성될 비디오 프레임의 너비입니다(기본값: 832). |
| `height` | INT | 예 | 16~8192 (단계: 16) | 생성될 비디오 프레임의 높이입니다(기본값: 480). |
| `length` | INT | 예 | 1~1024 (단계: 4) | 생성될 비디오의 총 프레임 수입니다(기본값: 81). |
| `batch_size` | INT | 예 | 1~64 | 한 번에 생성할 비디오 시퀀스의 수입니다(기본값: 1). |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `MODEL` | MODEL | 인코딩된 시작 이미지를 구성에 저장한 복제된 모델입니다. |
| `LATENT` | LATENT | 비디오 생성 과정에 맞는 올바른 크기의 빈 잠재 텐서입니다. |
