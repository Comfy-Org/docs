Ideogram V2 노드는 Ideogram V2 AI 모델을 사용해 이미지를 생성합니다. 텍스트 프롬프트와 다양한 생성 설정을 받아 API 서비스를 통해 이미지를 생성합니다. 이 노드는 다양한 종횡비, 해상도 및 스타일 옵션을 지원하여 출력 이미지를 맞춤화할 수 있습니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `prompt` | 문자열 | 예 | - | 이미지 생성을 위한 프롬프트 (기본값: 빈 문자열) |
| `turbo` | 부울 | 아니요 | - | 터보 모드 사용 여부 (더 빠른 생성, 잠재적으로 낮은 품질) (기본값: False) |
| `aspect_ratio` | 콤보 | 아니요 | "1:1"<br />"4:3"<br />"3:4"<br />"16:9"<br />"9:16"<br />"2:1"<br />"1:2"<br />"3:2"<br />"2:3"<br />"4:5"<br />"5:4" | 이미지 생성 시 종횡비. 해상도가 AUTO로 설정되지 않은 경우 무시됩니다. (기본값: "1:1") |
| `resolution` | 콤보 | 아니요 | "자동"<br />"512 x 1536"<br />"576 x 1408"<br />"576 x 1472"<br />"576 x 1536"<br />"640 x 1024"<br />"640 x 1344"<br />"640 x 1408"<br />"640 x 1472"<br />"640 x 1536"<br />"704 x 1152"<br />"704 x 1216"<br />"704 x 1280"<br />"704 x 1344"<br />"704 x 1408"<br />"704 x 1472"<br />"720 x 1280"<br />"736 x 1312"<br />"768 x 1024"<br />"768 x 1088"<br />"768 x 1152"<br />"768 x 1216"<br />"768 x 1232"<br />"768 x 1280"<br />"768 x 1344"<br />"832 x 960"<br />"832 x 1024"<br />"832 x 1088"<br />"832 x 1152"<br />"832 x 1216"<br />"832 x 1248"<br />"864 x 1152"<br />"896 x 960"<br />"896 x 1024"<br />"896 x 1088"<br />"896 x 1120"<br />"896 x 1152"<br />"960 x 832"<br />"960 x 896"<br />"960 x 1024"<br />"960 x 1088"<br />"1024 x 640"<br />"1024 x 768"<br />"1024 x 832"<br />"1024 x 896"<br />"1024 x 960"<br />"1024 x 1024"<br />"1088 x 768"<br />"1088 x 832"<br />"1088 x 896"<br />"1088 x 960"<br />"1120 x 896"<br />"1152 x 704"<br />"1152 x 768"<br />"1152 x 832"<br />"1152 x 864"<br />"1152 x 896"<br />"1216 x 704"<br />"1216 x 768"<br />"1216 x 832"<br />"1232 x 768"<br />"1248 x 832"<br />"1280 x 704"<br />"1280 x 720"<br />"1280 x 768"<br />"1280 x 800"<br />"1312 x 736"<br />"1344 x 640"<br />"1344 x 704"<br />"1344 x 768"<br />"1408 x 576"<br />"1408 x 640"<br />"1408 x 704"<br />"1472 x 576"<br />"1472 x 640"<br />"1472 x 704"<br />"1536 x 512"<br />"1536 x 576"<br />"1536 x 640" | 이미지 생성 시 해상도. AUTO로 설정되지 않은 경우, 이 설정이 aspect_ratio 설정을 덮어씁니다. (기본값: "자동") |
| `magic_prompt_option` | 콤보 | 아니요 | "AUTO"<br />"ON"<br />"OFF" | 생성 시 MagicPrompt 사용 여부 결정 (기본값: "AUTO") |
| `seed` | 정수 | 아니요 | 0-2147483647 | 생성을 위한 난수 시드 (기본값: 0) |
| `style_type` | 콤보 | 아니요 | "AUTO"<br />"GENERAL"<br />"REALISTIC"<br />"DESIGN"<br />"RENDER_3D"<br />"ANIME" | 생성 시 스타일 유형 (V2 전용) (기본값: "NONE") |
| `negative_prompt` | 문자열 | 아니요 | - | 이미지에서 제외할 내용 설명 (기본값: 빈 문자열) |
| `num_images` | 정수 | 아니요 | 1-8 | 생성할 이미지 수 (기본값: 1) |

**참고:** `resolution`이 "자동"으로 설정되지 않은 경우, `aspect_ratio` 설정을 덮어씁니다. `num_images` 매개변수는 한 번의 생성에 최대 8개의 이미지를 생성할 수 있습니다.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `output` | 이미지 | Ideogram V2 모델에서 생성된 이미지 |
