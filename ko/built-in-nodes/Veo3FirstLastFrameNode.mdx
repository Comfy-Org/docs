Veo3FirstLastFrameNode는 Google의 Veo 3 모델을 사용하여 텍스트 프롬프트를 기반으로 동영상을 생성하며, 제공된 첫 번째 및 마지막 프레임은 동영상 시퀀스의 시작과 끝을 정의합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `prompt` | 문자열 | 예 | N/A | 동영상의 텍스트 설명 (기본값: 빈 문자열). |
| `negative_prompt` | 문자열 | 아니요 | N/A | 동영상에서 피해야 할 내용을 안내하는 부정적 텍스트 프롬프트 (기본값: 빈 문자열). |
| `resolution` | 콤보 | 예 | `"720p"`<br />`"1080p"`<br />`"4k"` | 출력 동영상의 해상도. |
| `aspect_ratio` | 콤보 | 아니요 | `"16:9"`<br />`"9:16"` | 출력 동영상의 종횡비 (기본값: "16:9"). |
| `duration` | 정수 | 아니요 | 4~8 | 출력 동영상의 지속 시간(초) (기본값: 8, 단계: 2). |
| `seed` | 정수 | 아니요 | 0~4294967295 | 동영상 생성을 위한 시드 값 (기본값: 0). |
| `first_frame` | 이미지 | 예 | N/A | 동영상의 시작 프레임. |
| `last_frame` | 이미지 | 예 | N/A | 동영상의 끝 프레임. |
| `model` | 콤보 | 아니요 | `"veo-3.1-generate"`<br />`"veo-3.1-fast-generate"`<br />`"veo-3.1-lite"` | 생성에 사용할 특정 Veo 3 모델 (기본값: "veo-3.1-generate"). |
| `generate_audio` | 불리언 | 아니요 | N/A | 동영상에 오디오를 생성할지 여부 (기본값: True). |

**참고:** `veo-3.1-lite` 모델은 4K 해상도를 지원하지 않습니다. `veo-3.1-lite`와 4K 해상도를 선택하면 오류가 발생합니다.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `output` | 동영상 | 생성된 동영상 파일. |
