이 노드는 체크포인트 가중치를 로드하고 모델과 CLIP 구성 요소 모두에 강도 조정을 적용하여 LoRA(저랭크 적응) 방식으로 후크 모델을 생성합니다. 후크 기반 접근 방식을 통해 기존 모델에 LoRA 스타일의 수정을 적용할 수 있으며, 이를 통해 모델을 영구적으로 변경하지 않고도 세밀한 조정과 적응이 가능합니다. 이 노드는 이전 후크와 결합할 수 있으며, 효율성을 위해 로드된 가중치를 캐시합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `ckpt_name` | 문자열 | 예 | 여러 옵션 제공 | 가중치를 로드할 체크포인트 파일 (사용 가능한 체크포인트 중 선택) |
| `strength_model` | 부동소수점 | 예 | -20.0부터 20.0까지 | 모델 가중치에 적용되는 강도 배율 (기본값: 1.0) |
| `strength_clip` | 부동소수점 | 예 | -20.0부터 20.0까지 | CLIP 가중치에 적용되는 강도 배율 (기본값: 1.0) |
| `prev_hooks` | HOOKS | 아니요 | - | 새로 생성된 LoRA 후크와 결합할 수 있는 이전 후크 (선택사항) |

**매개변수 제약사항:**

- `ckpt_name` 매개변수는 사용 가능한 체크포인트 폴더에서 체크포인트를 로드합니다.
- 두 강도 매개변수 모두 -20.0부터 20.0까지 값을 받으며, 단계는 0.01씩 증가합니다.
- `prev_hooks`가 제공되지 않을 경우, 노드는 새로운 후크 그룹을 생성합니다.
- 노드는 같은 체크포인트를 여러 번 다시 로드하지 않도록 로드된 가중치를 캐시합니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `HOOKS` | HOOKS | 생성된 LoRA 후크들로, 이전 후크가 제공된 경우 이들과 결합됩니다 |
