> 이 문서는 AI로 생성되었습니다. 오류를 발견하거나 개선할 점이 있다면 언제든지 기여해 주세요! [GitHub에서 편집하기](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/Vidu2StartEndToVideoNode/en.md)

이 노드는 제공된 시작 프레임과 끝 프레임 사이를 보간하여 비디오를 생성하며, 텍스트 프롬프트에 따라 안내됩니다. 지정된 Vidu 모델을 사용해 두 이미지 간에 일정한 시간 동안 부드러운 전환을 만들어냅니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `model` | COMBO | 예 | `"viduq2-pro-fast"`<br />`"viduq2-pro"`<br />`"viduq2-turbo"` | 비디오 생성에 사용할 Vidu 모델입니다. |
| `first_frame` | IMAGE | 예 | - | 비디오 시퀀스의 시작 이미지입니다. 단일 이미지만 허용됩니다. |
| `end_frame` | IMAGE | 예 | - | 비디오 시퀀스의 끝 이미지입니다. 단일 이미지만 허용됩니다. |
| `prompt` | STRING | 예 | - | 비디오 생성을 안내하는 텍스트 설명(최대 2000자). |
| `duration` | INT | 아니요 | 2~8 | 생성된 비디오의 길이(초)(기본값: 5). |
| `seed` | INT | 아니요 | 0~2147483647 | 재현 가능한 결과를 위해 난수 생성을 초기화하는 데 사용되는 숫자(기본값: 1). |
| `resolution` | COMBO | 아니요 | `"720p"`<br />`"1080p"` | 생성된 비디오의 출력 해상도입니다. |
| `movement_amplitude` | COMBO | 아니요 | `"auto"`<br />`"small"`<br />`"medium"`<br />`"large"` | 프레임 내 객체의 움직임 진폭입니다. |

**참고:** `first_frame`과 `end_frame` 이미지는 비율이 비슷해야 합니다. 노드는 두 이미지의 가로세로 비율이 0.8에서 1.25의 상대적 범위 내에 있는지 검증합니다.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `output` | VIDEO | 생성된 비디오 파일입니다. |
