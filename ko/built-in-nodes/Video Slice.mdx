비디오 슬라이스 노드를 사용하면 비디오에서 특정 구간을 추출할 수 있습니다. 시작 시간과 지속 시간을 정해 비디오를 자를 수도 있고, 처음 프레임을 건너뛸 수도 있습니다. 요청한 지속 시간이 남은 비디오보다 길 경우, 노드는 가능한 부분을 반환하거나 오류를 발생시킬 수 있습니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `video` | VIDEO | 예 | - | 슬라이스할 입력 비디오입니다. |
| `start_time` | FLOAT | 아니요 | -1e5부터 1e5까지 | 초 단위의 시작 시간(기본값: 0.0). |
| `duration` | FLOAT | 아니요 | 0.0 이상 | 초 단위의 지속 시간, 또는 무한 지속 시간을 위해 0을 입력합니다(기본값: 0.0). |
| `strict_duration` | BOOLEAN | 아니요 | - | True로 설정하면, 지정된 지속 시간이 불가능할 경우 오류가 발생합니다(기본값: False). |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-----------|-----------|-------------|
| `video` | VIDEO | 잘라낸 비디오 구간입니다. |
