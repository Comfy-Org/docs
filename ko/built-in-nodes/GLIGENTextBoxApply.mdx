`GLIGENTextBoxApply` 노드는 텍스트 기반 컨디셔닝을 생성 모델의 입력에 통합하도록 설계되었으며, 특히 텍스트 박스 매개변수를 적용하고 이를 CLIP 모델을 사용해 인코딩합니다. 이 과정은 공간적 정보와 텍스트 정보를 통해 컨디셔닝을 풍부하게 만들어 보다 정밀하고 문맥을 고려한 생성을 가능하게 합니다.

## 입력

| 매개변수            | Comfy 데이터형        | 설명 |
|----------------------|--------------------|-------------|
| `conditioning_to`     | `CONDITIONING`     | 텍스트 박스 매개변수와 인코딩된 텍스트 정보가 추가될 초기 컨디셔닝 입력을 지정합니다. 새로운 컨디셔닝 데이터를 통합하여 최종 출력을 결정하는 데 핵심적인 역할을 합니다. |
| `clip`               | `CLIP`             | 제공된 텍스트를 생성 모델에서 사용 가능한 형식으로 인코딩하는 데 사용되는 CLIP 모델입니다. 텍스트 정보를 호환 가능한 컨디셔닝 형식으로 변환하는 데 필수적입니다. |
| `gligen_textbox_model` | `GLIGEN`         | 텍스트 박스를 생성하는 데 사용할 특정 GLIGEN 모델 구성입니다. 원하는 사양에 따라 텍스트 박스가 생성되도록 보장하는 데 중요합니다. |
| `text`               | `STRING`           | 인코딩되어 컨디셔닝에 통합될 텍스트 내용입니다. 생성 모델을 안내하는 의미적 정보를 제공합니다. |
| `width`              | `INT`              | 텍스트 박스의 가로 크기(픽셀 단위). 생성된 이미지 내에서 텍스트 박스의 공간적 크기를 정의합니다. |
| `height`             | `INT`              | 텍스트 박스의 세로 크기(픽셀 단위). 가로 크기와 마찬가지로 생성된 이미지 내에서 텍스트 박스의 공간적 크기를 정의합니다. |
| `x`                  | `INT`              | 생성된 이미지 내에서 텍스트 박스의 왼쪽 상단 모서리의 x좌표. 텍스트 박스의 수평 위치를 지정합니다. |
| `y`                  | `INT`              | 생성된 이미지 내에서 텍스트 박스의 왼쪽 상단 모서리의 y좌표. 텍스트 박스의 수직 위치를 지정합니다. |

## 출력

| 매개변수            | Comfy 데이터형        | 설명 |
|----------------------|--------------------|-------------|
| `conditioning`        | `CONDITIONING`     | 원래의 컨디셔닝 데이터와 새로 추가된 텍스트 박스 매개변수 및 인코딩된 텍스트 정보를 포함한 풍부해진 컨디셔닝 출력입니다. 생성 모델이 문맥을 고려한 출력을 생성하도록 안내하는 데 사용됩니다. |
