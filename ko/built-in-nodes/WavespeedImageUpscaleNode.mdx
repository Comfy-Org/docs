# WaveSpeed 이미지 업스케일 노드

WaveSpeed 이미지 업스케일 노드는 외부 AI 서비스를 사용하여 이미지의 해상도와 품질을 향상시킵니다. 단일 입력 사진을 받아 2K, 4K 또는 8K와 같은 더 높은 목표 해상도로 업스케일하여 더 선명하고 세부적인 결과물을 생성합니다.

## 입력

| 매개변수 | 설명 | 데이터 타입 | 필수 여부 | 범위 |
| --- | --- | --- | --- | --- |
| `model` | 업스케일에 사용할 AI 모델입니다. "SeedVR2"와 "Ultimate"는 서로 다른 품질과 가격 체계를 제공합니다. | STRING | 예 | `"SeedVR2"`<br />`"Ultimate"` |
| `image` | 업스케일할 입력 이미지입니다. | IMAGE | 예 |  |
| `목표 해상도` | 업스케일된 이미지의 원하는 출력 해상도입니다. | STRING | 예 | `"2K"`<br />`"4K"`<br />`"8K"` |

**참고:** 이 노드는 정확히 하나의 입력 이미지가 필요합니다. 이미지 배치를 제공하면 오류가 발생합니다.

## 출력

| 출력 이름 | 설명 | 데이터 타입 |
| --- | --- | --- |
| `image` | 업스케일된 고해상도 출력 이미지입니다. | IMAGE |
