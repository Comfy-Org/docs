TripoRetargetNode는 모션 데이터를 리타겟팅하여 사전 정의된 애니메이션을 3D 캐릭터 모델에 적용합니다. 이 노드는 이전에 리깅된 3D 모델을 받아 여러 사전 설정된 애니메이션 중 하나를 적용하고, 애니메이션 처리된 3D 모델 파일을 출력으로 생성합니다. 이 노드는 Tripo API와 통신하여 애니메이션 리타겟팅 작업을 처리합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `original_model_task_id` | RIG_TASK_ID | 예 | - | 애니메이션을 적용할 이전에 처리된 리깅된 3D 모델의 작업 ID |
| `animation` | STRING | 예 | "preset:idle"<br />"preset:walk"<br />"preset:run"<br />"preset:dive"<br />"preset:climb"<br />"preset:jump"<br />"preset:slash"<br />"preset:shoot"<br />"preset:hurt"<br />"preset:fall"<br />"preset:turn"<br />"preset:quadruped:walk"<br />"preset:hexapod:walk"<br />"preset:octopod:walk"<br />"preset:serpentine:march"<br />"preset:aquatic:march" | 3D 모델에 적용할 애니메이션 사전 설정입니다. 옵션에는 휴머노이드 애니메이션(idle, walk, run, dive, climb, jump, slash, shoot, hurt, fall, turn)과 크리처 애니메이션(사족 보행, 육각 보행, 팔발 보행, 뱀 모양 행진, 수중 행진)이 포함됩니다. |
| `auth_token_comfy_org` | AUTH_TOKEN_COMFY_ORG | 아니요 | - | Comfy.org API 접속을 위한 인증 토큰 (숨겨진 매개변수) |
| `api_key_comfy_org` | API_KEY_COMFY_ORG | 아니요 | - | Comfy.org 서비스 접속을 위한 API 키 (숨겨진 매개변수) |
| `unique_id` | UNIQUE_ID | 아니요 | - | 작업 추적을 위한 고유 식별자 (숨겨진 매개변수) |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `model_file` | STRING | 생성된 애니메이션 처리된 3D 모델 파일 (후방 호환성용만) |
| `retarget task_id` | RETARGET_TASK_ID | 리타겟팅 작업을 추적하는 작업 ID |
| `GLB` | FILE3DGLB | GLB 형식의 애니메이션 처리된 3D 모델 |
