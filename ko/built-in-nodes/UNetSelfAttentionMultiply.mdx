UNetSelfAttentionMultiply 노드는 UNet 모델의 자기 주의 메커니즘의 쿼리, 키, 값 및 출력 구성 요소에 곱셈 요인을 적용합니다. 이를 통해 주의 계산의 각 부분을 스케일링하여 주의 가중치가 모델의 동작에 어떻게 영향을 미치는지 실험할 수 있습니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `model` | MODEL | 예 | - | 주의 스케일링 요인으로 수정할 UNet 모델 |
| `q` | FLOAT | 아니요 | 0.0 - 10.0 | 쿼리 구성 요소에 적용되는 곱셈 요인 (기본값: 1.0) |
| `k` | FLOAT | 아니요 | 0.0 - 10.0 | 키 구성 요소에 적용되는 곱셈 요인 (기본값: 1.0) |
| `v` | FLOAT | 아니요 | 0.0 - 10.0 | 값 구성 요소에 적용되는 곱셈 요인 (기본값: 1.0) |
| `out` | FLOAT | 아니요 | 0.0 - 10.0 | 출력 구성 요소에 적용되는 곱셈 요인 (기본값: 1.0) |

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `MODEL` | MODEL | 주의 구성 요소가 스케일링된 수정된 UNet 모델 |
