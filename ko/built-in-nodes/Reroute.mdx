노드 이름: 리루트 노드
노드 용도: 주로 ComfyUI 워크플로우에서 지나치게 긴 연결선의 로직을 정리하는 데 사용됩니다.

## 리루트 노드 사용 방법

| 메뉴 옵션 | 설명 |
| --- | --- |
| 유형 표시 | 노드의 유형 속성을 표시합니다 |
| 기본적으로 유형 숨기기 | 기본적으로 노드의 유형 속성을 숨깁니다 |
| 세로 설정 | 노드의 배선 방향을 세로로 설정합니다 |
| 가로 설정 | 노드의 배선 방향을 가로로 설정합니다 |

배선 로직이 너무 길고 복잡해 인터페이스를 깔끔하게 정리하고 싶을 때, 두 연결 지점 사이에 ```리루트``` 노드를 삽입할 수 있습니다. 이 노드의 입력과 출력은 유형이 제한되지 않으며, 기본 스타일은 가로 방향입니다. 마우스 오른쪽 버튼 메뉴를 통해 배선 방향을 세로로 변경할 수 있습니다.
