이 노드는 Stable Diffusion 3 컨디셔닝에 ControlNet 가이드라인을 적용합니다. 긍정적이고 부정적인 컨디셔닝 입력과 ControlNet 모델 및 이미지를 받아들여 조정 가능한 강도와 타이밍 매개변수를 통해 제어 가이드라인을 적용하여 생성 과정에 영향을 줍니다.

**참고:** 이 노드는 더 이상 사용되지 않는 것으로 표시되었으며, 향후 버전에서는 제거될 수 있습니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `positive` | CONDITIONING | 예 | - | ControlNet 가이드라인을 적용할 긍정적 컨디셔닝 |
| `negative` | CONDITIONING | 예 | - | ControlNet 가이드라인을 적용할 부정적 컨디셔닝 |
| `control_net` | CONTROL_NET | 예 | - | 가이드라인에 사용할 ControlNet 모델 |
| `vae` | VAE | 예 | - | 처리 과정에 사용되는 VAE 모델 |
| `image` | IMAGE | 예 | - | ControlNet가 가이드라인으로 사용할 입력 이미지 |
| `strength` | FLOAT | 예 | 0.0 - 10.0 | ControlNet 효과의 강도 (기본값: 1.0) |
| `start_percent` | FLOAT | 예 | 0.0 - 1.0 | 생성 과정에서 ControlNet가 적용되기 시작하는 지점 (기본값: 0.0) |
| `end_percent` | FLOAT | 예 | 0.0 - 1.0 | 생성 과정에서 ControlNet가 적용을 멈추는 지점 (기본값: 1.0) |

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `positive` | CONDITIONING | ControlNet 가이드라인을 적용한 수정된 긍정적 컨디셔닝 |
| `negative` | CONDITIONING | ControlNet 가이드라인을 적용한 수정된 부정적 컨디셔닝 |
