> 이 문서는 AI로 생성되었습니다. 오류를 발견하거나 개선할 점이 있다면 언제든지 기여해 주세요! [GitHub에서 편집하기](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/StabilityStableImageUltraNode/en.md)

프롬프트와 해상도를 기반으로 이미지를 동기식으로 생성합니다. 이 노드는 Stability AI의 Stable Image Ultra 모델을 사용해 이미지를 생성하며, 텍스트 프롬프트를 처리하고 지정된 종횡비와 스타일로 해당하는 이미지를 생성합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `prompt` | 문자열 | 예 | - | 출력 이미지에서 보고 싶은 내용입니다. 요소, 색상, 주제를 명확히 정의하는 강력하고 구체적인 프롬프트를 사용하면 더 나은 결과를 얻을 수 있습니다. 특정 단어의 가중치를 조정하려면 `(단어:가중치)` 형식을 사용하세요. 여기서 `단어`는 가중치를 조정하고자 하는 단어이고, `가중치`는 0과 1 사이의 값입니다. 예를 들어, `The sky was a crisp (blue:0.3) and (green:0.8)`는 하늘이 파란색과 녹색이지만, 녹색이 파란색보다 더 강하게 나타나도록 표현합니다. |
| `aspect_ratio` | 콤보 | 예 | `"1:1"`<br />`"16:9"`<br />`"21:9"`<br />`"2:3"`<br />`"3:2"`<br />`"4:5"`<br />`"5:4"`<br />`"9:16"`<br />`"9:21"` | 생성된 이미지의 종횡비(기본값: "1:1"). |
| `style_preset` | 콤보 | 아니요 | `"3d-model"`<br />`"analog-film"`<br />`"anime"`<br />`"cinematic"`<br />`"comic-book"`<br />`"digital-art"`<br />`"enhance"`<br />`"fantasy-art"`<br />`"isometric"`<br />`"line-art"`<br />`"low-poly"`<br />`"modeling-compound"`<br />`"neon-punk"`<br />`"origami"`<br />`"photographic"`<br />`"pixel-art"`<br />`"tile-texture"` | 생성된 이미지의 원하는 스타일(선택사항). 아무 스타일도 적용하지 않으려면 "None"을 선택하세요. |
| `seed` | 정수 | 예 | 0 - 4294967294 | 노이즈 생성에 사용되는 난수 시드입니다. |
| `image` | 이미지 | 아니요 | - | 이미지 대 이미지 생성을 위한 선택적 입력 이미지입니다. |
| `negative_prompt` | 문자열 | 아니요 | - | 출력 이미지에서 보고 싶지 않은 내용을 설명하는 짧은 문구입니다. 이는 고급 기능입니다. |
| `image_denoise` | 실수 | 아니요 | 0.0 - 1.0 | 입력 이미지의 노이즈 제거 정도; 0.0은 입력 이미지와 동일한 이미지를 생성하고, 1.0은 이미지를 전혀 제공하지 않은 것과 같습니다(기본값: 0.5). |

**참고:** 입력 이미지를 제공하지 않을 경우, `image_denoise` 매개변수는 자동으로 비활성화되고 무시됩니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-----------|-----------|-------------|
| `output` | 이미지 | 입력 매개변수를 기반으로 생성된 이미지입니다. |
