텍스트 프롬프트와 원본 이미지의 종횡비에 따라 이미지를 동기적으로 수정합니다. 이 노드는 입력 이미지를 받아 제공된 프롬프트에 따라 변환하며, 구성 가능한 이미지 가중치를 사용해 원본 이미지가 얼마나 변경될지 조정할 수 있습니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `image` | IMAGE | 예 | - | 수정할 입력 이미지 |
| `prompt` | STRING | 예 | - | 이미지 생성을 위한 프롬프트 (기본값: "") |
| `image_weight` | FLOAT | 아니요 | 0.0-0.98 | 이미지의 가중치; 1.0에 가까울수록 이미지가 덜 수정됩니다 (기본값: 0.1). 내부적으로 이 값은 반전되어(1.0 - image_weight) 0.0과 0.98 사이로 제한됩니다. |
| `model` | STRING | 예 | `"photon-flash-1"`<br />`"photon-1"`<br />`"photon"` | 이미지 수정에 사용할 Luma 모델입니다. 모델마다 비용이 다릅니다. |
| `seed` | INT | 아니요 | 0-18446744073709551615 | 노드를 다시 실행할지 결정하는 시드; 실제 결과는 시드와 관계없이 비결정적입니다 (기본값: 0) |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `image` | IMAGE | Luma 모델이 생성한 수정된 이미지 |
