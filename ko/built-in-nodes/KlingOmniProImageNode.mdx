Kling Omni Image (Pro) 노드는 최신 Kling AI 모델을 사용해 이미지를 생성하거나 편집합니다. 이 노드는 텍스트 설명을 기반으로 이미지를 생성하며, 선택적으로 참조 이미지를 사용해 스타일이나 내용을 안내할 수 있습니다. 노드는 외부 API에 요청을 보내고, API가 작업을 처리한 후 최종 이미지를 반환합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
| :--- | :--- | :--- | :--- | :--- |
| `model_name` | COMBO | 예 | `"kling-v3-omni"`<br />`"kling-image-o1"` | 이미지 생성에 사용할 특정 Kling AI 모델입니다. |
| `prompt` | STRING | 예 | - | 이미지의 내용을 설명하는 텍스트 프롬프트입니다. 긍정적 및 부정적 설명을 모두 포함할 수 있습니다. 텍스트 길이는 1~2500자 사이여야 합니다. 업로드된 이미지나 동영상을 참조하려면 `@image`, `@image1`, `@image2` 등과 같은 형식 또는 `@video`, `@video1`, `@video2` 등과 같은 형식을 사용할 수 있습니다. |
| `resolution` | COMBO | 예 | `"1K"`<br />`"2K"`<br />`"4K"` | 생성할 이미지의 목표 해상도입니다. 참고: `kling-image-o1` 모델에서는 4K 해상도가 지원되지 않습니다. |
| `aspect_ratio` | COMBO | 예 | `"16:9"`<br />`"9:16"`<br />`"1:1"`<br />`"4:3"`<br />`"3:4"`<br />`"3:2"`<br />`"2:3"`<br />`"21:9"` | 생성할 이미지의 원하는 가로세로 비율(가로:세로)입니다. |
| `series_amount` | COMBO | 예 | `"disabled"`<br />`"2"`<br />`"3"`<br />`"4"`<br />`"5"`<br />`"6"`<br />`"7"`<br />`"8"`<br />`"9"` | 이미지 시리즈를 생성합니다. 이 기능은 `kling-image-o1` 모델에서는 지원되지 않습니다. (기본값: "disabled") |
| `reference_images` | IMAGE | 아니요 | - | 최대 10개의 추가 참조 이미지입니다. 각 이미지는 가로와 세로가 각각 300픽셀 이상이어야 하며, 가로세로 비율은 1:2.5에서 2.5:1 사이여야 합니다. |
| `seed` | INT | 아니요 | 0부터 2147483647까지 | 시드는 노드가 다시 실행될지 여부를 제어합니다. 시드에 관계없이 결과는 비결정적입니다. (기본값: 0) |

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
| :--- | :--- | :--- |
| `image` | IMAGE | Kling AI 모델이 생성하거나 편집한 최종 이미지입니다. 시리즈를 요청한 경우 여러 이미지가 일괄로 반환됩니다. |
