ResolutionSelector 노드는 선택한 종횡비와 목표 총 해상도(MP)를 기반으로 이미지의 픽셀 너비와 높이를 계산합니다. 이 노드는 Empty Latent Image 노드와 같은 다른 노드에서 일정한 크기를 생성하는 데 유용합니다. 출력 크기는 항상 8의 배수로 반올림됩니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `aspect_ratio` | COMBO | 예 | `"1:1 (정사각형)"`<br />`"3:2 (사진)"`<br />`"4:3 (표준)"`<br />`"16:9 (와이드스크린)"`<br />`"21:9 (울트라와이드)"`<br />`"2:3 (세로 사진)"`<br />`"3:4 (세로 표준)"`<br />`"9:16 (세로 와이드스크린)"` | 출력 크기의 종횡비입니다(기본값: `"1:1 (정사각형)"`). |
| `megapixels` | FLOAT | 예 | 0.1 - 16.0 | 목표 총 메가픽셀 수입니다. 1.0 MP ≈ 1024×1024(정사각형 종횡비)입니다(기본값: 1.0). |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `width` | INT | 계산된 너비(픽셀), 8의 배수입니다. |
| `height` | INT | 계산된 높이(픽셀), 8의 배수입니다. |
