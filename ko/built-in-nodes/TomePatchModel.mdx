TomePatchModel 노드는 Token Merging(ToMe)을 확산 모델에 적용하여 추론 시 계산 요구량을 줄입니다. 이는 어텐션 메커니즘에서 유사한 토큰들을 선택적으로 병합함으로써 모델이 더 적은 수의 토큰을 처리하도록 하면서도 이미지 품질을 유지하는 방식입니다. 이 기술은 큰 품질 손실 없이 생성 속도를 빠르게 하는 데 도움을 줍니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `model` | MODEL | 예 | - | 토큰 병합을 적용할 확산 모델 |
| `ratio` | FLOAT | 예 | 0.0 - 1.0 | 병합할 토큰의 비율 (기본값: 0.3). 값이 높을수록 더 많은 토큰이 병합되어 속도가 크게 향상되지만 품질은 다소 저하될 수 있습니다. |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `model` | MODEL | 토큰 병합이 적용된 수정된 모델 |
