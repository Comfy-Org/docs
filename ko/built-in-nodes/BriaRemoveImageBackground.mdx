이 노드는 Bria RMBG 2.0 서비스를 사용하여 이미지의 배경을 제거합니다. 이미지를 외부 API로 전송해 처리한 후, 배경이 제거된 결과를 반환합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `image` | IMAGE | 예 | - | 배경을 제거할 입력 이미지입니다. |
| `moderation` | COMBO | 아니요 | `"false"`<br />`"true"` | 모더레이션 설정입니다. `"true"`로 설정하면 추가적인 모더레이션 옵션이 사용 가능해집니다. |
| `visual_input_moderation` | BOOLEAN | 아니요 | - | 입력 이미지에 시각적 콘텐츠 모더레이션을 활성화합니다. 이 매개변수는 `moderation`이 `"true"`로 설정된 경우에만 사용 가능합니다. 기본값: `False`. |
| `visual_output_moderation` | BOOLEAN | 아니요 | - | 출력 이미지에 시각적 콘텐츠 모더레이션을 활성화합니다. 이 매개변수는 `moderation`이 `"true"`로 설정된 경우에만 사용 가능합니다. 기본값: `True`. |
| `seed` | INT | 아니요 | 0부터 2147483647까지 | 노드가 다시 실행될지 여부를 제어하는 시드 값입니다. 시드 값과 관계없이 결과는 비결정적입니다. 기본값: `0`. |

**참고:** `visual_input_moderation` 및 `visual_output_moderation` 매개변수는 `moderation` 매개변수에 의존합니다. `moderation`이 `"true"`로 설정된 경우에만 활성화되고 필요합니다.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `image` | IMAGE | 배경이 제거된 처리된 이미지입니다. |
