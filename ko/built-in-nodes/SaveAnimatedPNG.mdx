SaveAnimatedPNG 노드는 프레임 시퀀스에서 애니메이션 PNG 이미지를 생성하고 저장하는 데 사용됩니다. 이 노드는 개별 이미지 프레임을 하나의 일관된 애니메이션으로 조합하며, 프레임 지속 시간, 반복 여부 및 메타데이터 포함 여부를 맞춤 설정할 수 있습니다.

## 입력

| 필드             | 데이터 타입 | 설명                                                                         |
|-------------------|-------------|-------------------------------------------------------------------------------------|
| `images`          | `IMAGE`     | 처리하여 애니메이션 PNG로 저장할 이미지 목록입니다. 목록의 각 이미지는 애니메이션의 한 프레임을 나타냅니다. |
| `filename_prefix` | `STRING`    | 출력 파일의 기본 이름을 지정하며, 생성된 애니메이션 PNG 파일의 접두사로 사용됩니다. |
| `fps`             | `FLOAT`     | 애니메이션의 초당 프레임 수로, 프레임이 얼마나 빠르게 표시되는지를 제어합니다. |
| `compress_level`  | `INT`       | 애니메이션 PNG 파일에 적용되는 압축 수준으로, 파일 크기와 이미지 선명도에 영향을 줍니다. |

## 출력

| 필드 | 데이터 타입 | 설명                                                                       |
|-------|-------------|-----------------------------------------------------------------------------------|
| `ui`  | N/A         | 생성된 애니메이션 PNG 이미지를 표시하고, 애니메이션이 단일 프레임인지 다중 프레임인지 알려주는 UI 구성 요소를 제공합니다. |
