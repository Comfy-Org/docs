PrimitiveStringMultiline 노드는 워크플로우를 통해 문자열 값을 입력하고 전달하기 위한 다중 줄 텍스트 입력 필드를 제공합니다. 이 노드는 여러 줄로 구성된 텍스트 입력을 받아들여 동일한 문자열 값을 변경 없이 출력합니다. 긴 텍스트 콘텐츠나 여러 줄에 걸쳐 형식화된 텍스트를 입력해야 할 때 유용합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `value` | STRING | 예 | N/A | 여러 줄로 구성될 수 있는 텍스트 입력 값 |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `output` | STRING | 입력으로 제공된 동일한 문자열 값 |
