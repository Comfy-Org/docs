StableZero123_Conditioning 노드는 입력 이미지와 카메라 각도를 처리하여 3D 모델 생성을 위한 컨디셔닝 데이터와 잠재 표현을 생성합니다. 이 노드는 CLIP 비전 모델을 사용해 이미지 특징을 인코딩하고, 고도와 방위각에 기반한 카메라 임베딩 정보와 결합한 후, 긍정적·부정적 컨디셔닝과 하위 3D 생성 작업을 위한 잠재 표현을 생성합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `clip_vision` | CLIP_VISION | 예 | - | 이미지 특징을 인코딩하는 데 사용되는 CLIP 비전 모델 |
| `init_image` | IMAGE | 예 | - | 처리하고 인코딩할 입력 이미지 |
| `vae` | VAE | 예 | - | 픽셀을 잠재 공간으로 인코딩하는 데 사용되는 VAE 모델 |
| `width` | INT | 예 | 16부터 MAX_RESOLUTION까지 | 잠재 표현의 출력 너비 (기본값: 256, 8로 나누어져야 함) |
| `height` | INT | 예 | 16부터 MAX_RESOLUTION까지 | 잠재 표현의 출력 높이 (기본값: 256, 8로 나누어져야 함) |
| `batch_size` | INT | 예 | 1부터 4096까지 | 배치에서 생성할 샘플 수 (기본값: 1) |
| `elevation` | FLOAT | 예 | -180.0부터 180.0까지 | 카메라 고도각 (도 단위, 기본값: 0.0) |
| `azimuth` | FLOAT | 예 | -180.0부터 180.0까지 | 카메라 방위각 (도 단위, 기본값: 0.0) |

**참고:** `width`와 `height` 매개변수는 노드가 자동으로 8로 나누어 잠재 표현의 차원을 생성하므로 반드시 8로 나누어져야 합니다.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `positive` | CONDITIONING | 이미지 특징과 카메라 임베딩을 결합한 긍정적 컨디셔닝 데이터 |
| `negative` | CONDITIONING | 특징이 0으로 초기화된 부정적 컨디셔닝 데이터 |
| `latent` | LATENT | 차원이 [batch_size, 4, height//8, width//8]인 잠재 표현 |
