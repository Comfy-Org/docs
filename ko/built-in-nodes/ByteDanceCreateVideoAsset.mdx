> 이 문서는 AI로 생성되었습니다. 오류를 발견하거나 개선할 점이 있다면 언제든지 기여해 주세요! [GitHub에서 편집하기](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/ByteDanceCreateVideoAsset/en.md)

이 노드는 Seedance 2.0용 개인 동영상 자산을 생성합니다. 입력 동영상을 업로드하고 지정된 자산 그룹 내에 등록합니다. 그룹 ID를 제공하지 않으면 브라우저에서 실제 사람 인증 과정을 거쳐 먼저 새 그룹을 생성하도록 안내합니다.

## 입력값

| 매개변수 | 데이터 타입 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `video` | VIDEO | 예 | - | 개인 자산으로 등록할 동영상입니다. |
| `group_id` | STRING | 아니요 | - | 이미 존재하는 Seedance 자산 그룹 ID를 재사용하여 동일한 사람에 대한 반복적인 인간 인증을 생략할 수 있습니다. 비워두면 브라우저에서 실제 사람 인증을 수행하고 새 그룹을 생성합니다. (기본값: 빈 문자열) |

**동영상 제약 조건:**
*   **길이:** 2초에서 15초 사이여야 합니다.
*   **크기:** 가로와 세로 각각 300~6000픽셀 사이여야 합니다.
*   **종횡비:** 가로 대 세로 비율은 0.4에서 2.5 사이여야 합니다.
*   **총 픽셀 수:** 총 픽셀 수(가로 × 세로)는 409,600에서 927,408 사이여야 합니다.
*   **프레임 속도:** 초당 24~60프레임(FPS) 사이여야 합니다.

## 출력값

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `asset_id` | STRING | 새로 생성된 동영상 자산의 고유 식별자입니다. |
| `group_id` | STRING | 새 동영상을 포함하는 자산 그룹의 식별자입니다. 제공된 `group_id` 또는 새로 생성된 그룹이 됩니다. |
