> 이 문서는 AI로 생성되었습니다. 오류를 발견하거나 개선할 점이 있으면 언제든지 기여해 주세요! [GitHub에서 편집하기](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/TripoTextToModelNode/en.md)

Tripo의 API를 사용하여 텍스트 프롬프트를 기반으로 3D 모델을 동기적으로 생성합니다. 이 노드는 텍스트 설명을 입력받아 선택적 텍스처와 재료 속성을 포함한 3D 모델을 생성합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `prompt` | 문자열 | 예 | - | 3D 모델 생성을 위한 텍스트 설명 (멀티라인 입력) |
| `negative_prompt` | 문자열 | 아니요 | - | 생성된 모델에서 피하고자 하는 내용의 텍스트 설명 (멀티라인 입력) |
| `model_version` | 콤보 | 아니요 | `v1.4`<br />`v2.0`<br />`v2.5-20250123`<br />`v3.0` | 생성에 사용할 Tripo 모델의 버전 (기본값: v2.5-20250123) |
| `style` | 콤보 | 아니요 | `None`<br />`sculpture`<br />`low-poly`<br />`voxel`<br />`brick`<br />`clay`<br />`marble`<br />`origami`<br />`papercraft`<br />`pixel-art`<br />`tile`<br />`cartoon`<br />`fantasy`<br />`hand-drawn`<br />`illustration`<br />`isometric`<br />`pixel-art`<br />`sketch`<br />`stained-glass`<br />`steampunk`<br />`vector`<br />`watercolor` | 생성된 모델의 스타일 설정 (기본값: "None") |
| `texture` | 부울 | 아니요 | - | 모델에 텍스처를 생성할지 여부 (기본값: True) |
| `pbr` | 부울 | 아니요 | - | PBR(물리 기반 렌더링) 재료를 생성할지 여부 (기본값: True) |
| `image_seed` | 정수 | 아니요 | - | 이미지 생성용 난수 시드 (기본값: 42) |
| `model_seed` | 정수 | 아니요 | - | 모델 생성용 난수 시드 (기본값: 42) |
| `texture_seed` | 정수 | 아니요 | - | 텍스처 생성용 난수 시드 (기본값: 42) |
| `texture_quality` | 콤보 | 아니요 | `standard`<br />`detailed` | 텍스처 생성의 품질 수준 (기본값: "standard") |
| `face_limit` | 정수 | 아니요 | -1부터 2000000까지 | 생성된 모델의 최대 면 수, 한계 없음은 -1 (기본값: -1) |
| `quad` | 부울 | 아니요 | - | 삼각형 대신 사각형 기반 지오메트리를 생성할지 여부 (기본값: False) |
| `geometry_quality` | 콤보 | 아니요 | `standard`<br />`detailed` | 지오메트리 생성의 품질 수준 (기본값: "standard") |

**참고:** `prompt` 매개변수는 필수이며 비워둘 수 없습니다. 프롬프트가 제공되지 않으면 노드는 오류를 발생시킵니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `model_file` | 문자열 | 생성된 3D 모델 파일 (후향 호환성 전용) |
| `model task_id` | MODEL_TASK_ID | 모델 생성 과정의 고유한 작업 식별자 |
| `GLB` | FILE3DGLB | GLB 형식으로 생성된 3D 모델 |
