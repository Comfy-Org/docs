## 개요

Beeble SwitchX를 사용하여 단일 이미지를 편집하세요. 이 노드는 장면의 모든 요소(배경, 조명, 의상)를 교체하면서도 원래 피사체의 픽셀은 그대로 유지합니다. 참조 이미지 및/또는 텍스트 프롬프트를 제공하여 새로운 모습을 설명하세요. 최대 해상도는 약 277만 화소입니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `image` | IMAGE | 예 | - | 편집할 소스 이미지입니다. |
| `prompt` | STRING | 예 | - | 원하는 새로운 모습에 대한 텍스트 설명(예: "빛나는 갑옷을 입은 기사"). |
| `alpha_mode` | COMBO | 예 | `"select"`<br />`"fill"`<br />`"custom"` | 알파 마티를 처리하는 방법입니다. "select"는 키프레임을 사용해 피사체를 선택하고, "fill"은 별도의 마티 없이 이미지를 전부 교체하며, "custom"은 사용자가 제공한 마스크를 사용합니다. |
| `max_resolution` | COMBO | 예 | `"1080p"`<br />`"720p"` | 출력 이미지의 최대 해상도입니다. 해상도가 높을수록 크레딧 비용이 더 듭니다. |
| `seed` | INT | 예 | - | 재현성을 위한 시드 값입니다. |
| `reference_image` | IMAGE | 선택 | - | 새 장면 요소의 스타일이나 모양을 안내하기 위한 선택적 참조 이미지입니다. |

**`alpha_mode` 관련 참고사항:** `alpha_mode`를 "select"로 설정하면 `alpha_keyframe`(피사체를 선택하는 데 사용되는 키프레임 이미지)도 제공해야 합니다. "custom"으로 설정하면 `alpha_mask`(사용자가 만든 마스크)를 제공해야 합니다. "fill"로 설정하면 알파 입력이 필요하지 않습니다.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `image` | IMAGE | 장면 요소가 교체된 편집된 이미지입니다. |
| `alpha` | MASK | Beeble이 사용하는 알파 마티입니다. "fill" 모드에서는 별도의 마티가 없으므로 비어 있습니다. |
