TripoRigNode는 원본 모델 작업 ID로부터 리깅된 3D 모델을 생성합니다. Tripo API에 요청을 보내 Tripo 사양을 사용해 GLB 형식의 애니메이션 리그를 생성한 후, 리그 생성 작업이 완료될 때까지 API를 폴링합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `original_model_task_id` | MODEL_TASK_ID | 예 | - | 리깅할 원본 3D 모델의 작업 ID |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `model_file` | STRING | 생성된 리깅된 3D 모델 파일 (후방 호환성을 위해 유지) |
| `rig task_id` | RIG_TASK_ID | 리그 생성 과정을 추적하는 작업 ID |
| `GLB` | FILE3DGLB | 생성된 리깅된 3D 모델의 GLB 형식 |
