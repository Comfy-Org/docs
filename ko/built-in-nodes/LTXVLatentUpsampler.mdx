LTXVLatentUpsampler 노드는 비디오 잠재 표현의 공간 해상도를 두 배로 높입니다. 이 노드는 특수한 업스케일링 모델을 사용하여 잠재 데이터를 처리하며, 먼저 정규화 해제된 후 제공된 VAE의 채널 통계를 이용해 다시 정규화됩니다. 이 노드는 잠재 공간 내 비디오 워크플로우를 위해 설계되었습니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `samples` | LATENT | 예 | | 업스케일링할 비디오의 입력 잠재 표현입니다. |
| `upscale_model` | LATENT_UPSCALE_MODEL | 예 | | 잠재 데이터에 2배 업스케일링을 수행하는 데 사용되는 로드된 모델입니다. |
| `vae` | VAE | 예 | | 업스케일링 전에 입력 잠재를 정규화 해제하고, 이후에 출력 잠재를 다시 정규화하는 데 사용되는 VAE 모델입니다. |

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `LATENT` | LATENT | 업스케일링된 잠재 표현으로, 공간 차원이 입력보다 두 배 커졌습니다. 출력 잠재는 입력과 동일한 배치 크기, 채널 수, 시간 길이를 가집니다. 입력에 있던 `noise_mask`는 출력에서 제거됩니다. |
