> 이 문서는 AI로 생성되었습니다. 오류를 발견하거나 개선할 점이 있다면 언제든지 기여해 주세요! [GitHub에서 편집하기](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/LTXVAddGuide/en.md)

LTXVAddGuide 노드는 잠재적 시퀀스에 비디오 조건부 가이드라인을 추가하기 위해 입력 이미지 또는 비디오를 인코딩하고 이를 키프레임으로 조건부 데이터에 포함시킵니다. 이 노드는 VAE 인코더를 통해 입력을 처리하고, 결과로 얻은 잠재적 값을 지정된 프레임 위치에 전략적으로 배치하며, 긍정적 및 부정적 조건부 정보를 모두 키프레임 정보와 함께 업데이트합니다. 또한 이 노드는 프레임 정렬 제약을 처리하고 조건부 영향의 강도를 제어할 수 있도록 합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `positive` | CONDITIONING | 예 | - | 키프레임 가이드라인으로 수정될 긍정적 조건부 입력 |
| `negative` | CONDITIONING | 예 | - | 키프레임 가이드라인으로 수정될 부정적 조건부 입력 |
| `vae` | VAE | 예 | - | 입력 이미지/비디오 프레임을 인코딩하는 데 사용되는 VAE 모델 |
| `latent` | LATENT | 예 | - | 조건부 프레임을 받게 될 입력 잠재적 시퀀스 |
| `image` | IMAGE | 예 | - | 잠재적 비디오를 조건부로 만들 이미지 또는 비디오. 8*n + 1 프레임이어야 합니다. 비디오가 8*n + 1 프레임이 아닐 경우, 가장 가까운 8*n + 1 프레임으로 자릅니다. |
| `frame_idx` | INT | 아니요 | -9999부터 9999까지 | 조건부를 시작할 프레임 인덱스. 단일 프레임 이미지나 1~8 프레임의 비디오는 어떤 frame_idx 값이나 가능합니다. 9프레임 이상의 비디오에서는 frame_idx가 8로 나누어져야 하며, 그렇지 않으면 가장 가까운 8의 배수로 반올림됩니다. 음수 값은 비디오 끝에서부터 계산됩니다. (기본값: 0) |
| `strength` | FLOAT | 아니요 | 0.0부터 10.0까지 | 조건부 영향의 강도, 1.0은 완전한 조건부를 적용하고 0.0은 조건부를 전혀 적용하지 않습니다 (기본값: 1.0) |
| `attention_mask` | MASK | 아니요 | - | 선택적 픽셀 공간 공간 마스크. 자기 주의를 통해 각 지역별 조건부 영향을 제어하며, 강도와 곱해집니다. |
| `iclora_parameters` | IC_LORA_PARAMETERS | 아니요 | - | Get IC-LoRA Parameters 노드에서 가져온 선택적 IC-LoRA 매개변수. 특정 IC-LoRA(예: reference_downscale_factor > 1인 경우)에 따라 가이드 처리를 조정하는 데 사용됩니다. 체인으로 연결된 경우 각 LTXVAddGuide는 자신에게 연결된 매개변수만 사용합니다. |

**참고:** 입력 이미지/비디오는 프레임 수가 8*n + 1 패턴을 따라야 합니다(예: 1, 9, 17, 25 프레임). 입력이 이 패턴을 초과하면 자동으로 가장 가까운 유효한 프레임 수로 자릅니다.

**iclora_parameters 관련 참고사항:** reference_downscale_factor가 1보다 큰 IC-LoRA 매개변수를 사용할 때, 잠재적 공간 차원(너비와 높이)은 해당 요소로 나누어져야 합니다. 이 조건을 만족하지 않으면 노드는 오류를 발생시킵니다.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `positive` | CONDITIONING | 키프레임 가이드라인 정보로 업데이트된 긍정적 조건부 |
| `negative` | CONDITIONING | 키프레임 가이드라인 정보로 업데이트된 부정적 조건부 |
| `latent` | LATENT | 조건부 프레임이 포함되고 노이즈 마스크가 업데이트된 잠재적 시퀀스 |
