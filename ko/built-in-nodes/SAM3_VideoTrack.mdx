## 개요

SAM3의 메모리 기반 트래커를 사용하여 비디오 프레임 간에 객체를 추적합니다. 이 노드는 비디오 프레임 시퀀스를 처리하고, 초기 마스크 또는 텍스트 프롬프트를 사용해 각 프레임 간에 객체의 정체성을 유지하며 추적할 대상을 정의합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `images` | IMAGE | 예 | 배치된 비디오 프레임 | 배치된 이미지 형태의 비디오 프레임 |
| `model` | MODEL | 예 | SAM3 모델 | 추적에 사용할 SAM3 모델 |
| `initial_mask` | MASK | 아니요 | 객체당 하나의 마스크 | 추적할 첫 번째 프레임의 마스크(객체당 하나씩). `conditioning`이 제공되지 않을 경우 필수입니다. |
| `conditioning` | CONDITIONING | 아니요 | 텍스트 조건부 | 추적 중 새 객체를 탐지하는 데 사용할 텍스트 조건부. `initial_mask`가 제공되지 않을 경우 필수입니다. |
| `detection_threshold` | FLOAT | 아니요 | 0.0~1.0 (기본값: 0.5) | 텍스트 프롬프트 기반 탐지의 점수 임계값 |
| `max_objects` | INT | 아니요 | 0~64 (기본값: 0) | 최대 추적 객체 수. 초기 마스크도 이 한도에 포함됩니다. 0일 경우 내부 제한인 64개를 사용합니다. |
| `detect_interval` | INT | 아니요 | 1~무제한 (기본값: 1) | N번째 프레임마다 탐지를 실행합니다 (1=매 프레임). 값이 클수록 계산 자원을 절약할 수 있습니다. |

**참고:** `initial_mask`와 `conditioning` 중 하나는 반드시 제공해야 합니다. 둘 다 생략되면 노드는 오류를 발생시킵니다.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `track_data` | SAM3TrackData | 모든 비디오 프레임에 걸친 객체 마스크와 메타데이터를 포함한 추적 데이터 |
