Kling 이미지에서 비디오 카메라 제어 노드는 정지된 이미지를 전문적인 카메라 움직임으로 영화 같은 비디오로 변환합니다. 이 특수한 이미지-비디오 노드를 사용하면 줌, 회전, 패닝, 틸트 및 1인칭 시점과 같은 가상 카메라 동작을 제어하면서 원본 이미지에 초점을 유지할 수 있습니다. 현재 카메라 제어는 kling-v1-5 모델의 프로 모드에서만 5초 길이로 지원됩니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `start_frame` | IMAGE | 예 | - | 참조 이미지 - URL 또는 Base64 인코딩된 문자열, 크기는 10MB를 초과해서는 안 되며 해상도는 300x300px 이상, 종횡비는 1:2.5에서 2.5:1 사이여야 합니다. Base64에는 data:image 접두사가 포함되어서는 안 됩니다. |
| `prompt` | STRING | 예 | - | 원하는 비디오 내용을 설명하는 긍정적 텍스트 프롬프트입니다. 최대 길이는 500자입니다. |
| `negative_prompt` | STRING | 예 | - | 생성된 비디오에서 피해야 할 사항을 설명하는 부정적 텍스트 프롬프트입니다. 최대 길이는 500자입니다. |
| `cfg_scale` | FLOAT | 아니요 | 0.0부터 1.0까지 | 텍스트 지침의 강도를 조절합니다. 값이 높을수록 출력물이 프롬프트를 더 잘 따르게 됩니다(기본값: 0.75) |
| `aspect_ratio` | COMBO | 아니요 | `"16:9"`<br />`"9:16"`<br />`"1:1"` | 생성된 비디오의 종횡비(기본값: "16:9") |
| `camera_control` | CAMERA_CONTROL | 예 | - | Kling 카메라 제어 노드를 사용해 생성할 수 있습니다. 비디오 생성 중 카메라의 움직임과 동작을 제어합니다. |

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `output` | VIDEO | 생성된 비디오 출력 |
| `video_id` | STRING | 생성된 비디오의 고유 식별자 |
| `duration` | STRING | 생성된 비디오의 길이 |
