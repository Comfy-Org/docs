> 이 문서는 AI로 생성되었습니다. 오류를 발견하거나 개선할 점이 있다면 언제든지 기여해 주세요! [GitHub에서 편집하기](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/ByteDanceFirstLastFrameNode/en.md)

이 노드는 텍스트 프롬프트와 첫 번째 및 마지막 프레임 이미지를 사용해 동영상을 생성합니다. 사용자의 설명과 두 개의 핵심 프레임을 받아 그 사이를 부드럽게 전환하는 완전한 동영상 시퀀스를 만들어냅니다. 이 노드는 동영상의 해상도, 종횡비, 지속 시간 및 기타 생성 매개변수를 제어할 수 있는 다양한 옵션을 제공합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `model` | COMBO | 예 | `"seedance-1-5-pro-251215"`<br />`"seedance-1-0-pro-250528"`<br />`"seedance-1-0-lite-i2v-250428"` | 동영상 생성에 사용할 모델 (기본값: `"seedance-1-0-lite-i2v-250428"`). |
| `prompt` | STRING | 예 | - | 동영상 생성에 사용할 텍스트 프롬프트입니다. |
| `first_frame` | IMAGE | 예 | - | 동영상에 사용될 첫 번째 프레임입니다. 300x300~6000x6000 픽셀 사이이며, 종횡비는 0.4~2.5 사이여야 합니다. |
| `last_frame` | IMAGE | 예 | - | 동영상에 사용될 마지막 프레임입니다. 300x300~6000x6000 픽셀 사이이며, 종횡비는 0.4~2.5 사이여야 합니다. |
| `resolution` | COMBO | 예 | `"480p"`<br />`"720p"`<br />`"1080p"` | 출력 동영상의 해상도입니다. |
| `aspect_ratio` | COMBO | 예 | `"adaptive"`<br />`"16:9"`<br />`"4:3"`<br />`"1:1"`<br />`"3:4"`<br />`"9:16"`<br />`"21:9"` | 출력 동영상의 종횡비 (기본값: `"adaptive"`). |
| `duration` | INT | 예 | 3~12 | 출력 동영상의 지속 시간(초)입니다 (기본값: 5). 참고: `seedance-1-5-pro-251215` 모델의 경우 최소 지원 지속 시간은 4초입니다. |
| `seed` | INT | 아니요 | 0~2147483647 | 생성에 사용할 시드입니다 (기본값: 0). |
| `camera_fixed` | BOOLEAN | 아니요 | - | 카메라를 고정할지 여부를 지정합니다. 플랫폼은 카메라를 고정하라는 지시를 프롬프트에 추가하지만, 실제로 효과가 보장되지는 않습니다 (기본값: False). |
| `watermark` | BOOLEAN | 아니요 | - | 동영상에 "AI 생성" 워터마크를 추가할지 여부입니다 (기본값: False). |
| `generate_audio` | BOOLEAN | 아니요 | - | 이 매개변수는 `seedance-1-5-pro-251215` 모델을 제외한 모든 모델에서는 무시됩니다 (기본값: False). |
