SVD_img2vid_Conditioning 노드는 Stable Video Diffusion을 이용해 비디오 생성을 위한 조건 데이터를 준비합니다. 초기 이미지를 받아 CLIP 비전과 VAE 인코더를 통해 긍정적·부정적 조건 쌍을 생성하고, 비디오 생성을 위한 빈 잠재 공간도 함께 만들어냅니다. 이 노드는 생성된 비디오의 모션, 프레임 속도 및 증강 수준을 제어하는 데 필요한 매개변수를 설정합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `clip_vision` | CLIP_VISION | 예 | - | 입력 이미지를 인코딩하는 CLIP 비전 모델 |
| `init_image` | IMAGE | 예 | - | 비디오 생성의 시작점으로 사용할 초기 이미지 |
| `vae` | VAE | 예 | - | 이미지를 잠재 공간으로 인코딩하는 VAE 모델 |
| `width` | INT | 예 | 16부터 MAX_RESOLUTION까지 | 출력 비디오 너비 (기본값: 1024, 단계: 8) |
| `height` | INT | 예 | 16부터 MAX_RESOLUTION까지 | 출력 비디오 높이 (기본값: 576, 단계: 8) |
| `video_frames` | INT | 예 | 1부터 4096까지 | 비디오에 생성할 프레임 수 (기본값: 14) |
| `motion_bucket_id` | INT | 예 | 1부터 1023까지 | 생성된 비디오의 모션 양을 제어합니다 (기본값: 127) |
| `fps` | INT | 예 | 1부터 1024까지 | 생성된 비디오의 초당 프레임 수 (기본값: 6) |
| `augmentation_level` | FLOAT | 예 | 0.0부터 10.0까지 | 입력 이미지에 적용할 노이즈 증강 수준 (기본값: 0.0, 단계: 0.01) |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `positive` | CONDITIONING | 이미지 임베딩과 비디오 파라미터를 포함한 긍정적 조건 데이터 |
| `negative` | CONDITIONING | 임베딩이 0으로 설정되고 비디오 파라미터를 포함한 부정적 조건 데이터 |
| `latent` | LATENT | 비디오 생성을 위해 준비된 빈 잠재 공간 텐서 |
