텍스트를 소문자로 변환하는 노드는 텍스트 문자열을 입력받아 모든 문자를 소문자로 변환합니다. 이는 텍스트의 대소문자를 통일하는 간단한 유틸리티입니다.

> **참고:** 이 노드는 더 이상 사용되지 않으며, 대신 텍스트 대소문자 변환 노드가 사용됩니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `text` | STRING | 예 | 임의의 텍스트 문자열 | 소문자로 변환할 텍스트 문자열입니다. |

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `text` | STRING | 입력된 텍스트를 모든 문자를 소문자로 변환한 결과입니다. |
