Text to Lowercase 노드는 텍스트 문자열을 입력받아 모든 문자를 소문자로 변환합니다. 텍스트 대소문자를 표준화하는 간단한 유틸리티입니다.

## 입력

| 매개변수 | 설명 | 데이터 타입 | 필수 | 범위 |
| --- | --- | --- | --- | --- |
| `text` | 소문자로 변환할 텍스트 문자열입니다. | STRING | 예 | 모든 텍스트 문자열 |

## 출력

| 출력 이름 | 설명 | 데이터 타입 |
| --- | --- | --- |
| `texts` | 모든 문자가 소문자로 변환된 입력 텍스트입니다. | STRING |
