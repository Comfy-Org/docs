CheckpointLoader 노드는 사전 훈련된 모델 체크포인트와 그 구성 파일을 로드합니다. 구성 파일과 체크포인트 파일을 입력으로 받아, 워크플로우에서 사용할 수 있는 메인 모델, CLIP 모델, VAE 모델을 포함한 로드된 모델 구성 요소를 반환합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `config_name` | 문자열 | 예 | 사용 가능한 구성 파일 | 모델 아키텍처와 설정을 정의하는 구성 파일 |
| `ckpt_name` | 문자열 | 예 | 사용 가능한 체크포인트 파일 | 훈련된 모델 가중치와 파라미터를 포함한 체크포인트 파일 |

**참고:** 이 노드는 구성 파일과 체크포인트 파일을 모두 선택해야 합니다. 구성 파일은 로드되는 체크포인트 파일의 아키텍처와 일치해야 합니다.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `MODEL` | MODEL | 추론에 준비된 로드된 메인 모델 구성 요소 |
| `CLIP` | CLIP | 텍스트 인코딩을 위한 로드된 CLIP 모델 구성 요소 |
| `VAE` | VAE | 이미지 인코딩 및 디코딩을 위한 로드된 VAE 모델 구성 요소 |

**중요 참고사항:** 이 노드는 더 이상 사용되지 않으며, 향후 버전에서는 제거될 수 있습니다. 새로운 워크플로우에는 대체 로드 노드를 사용하는 것을 고려해 주세요.
