Moonvalley Marey 텍스트 to 비디오 노드는 Moonvalley API를 사용해 텍스트 설명을 기반으로 비디오 콘텐츠를 생성합니다. 이 노드는 텍스트 프롬프트를 받아 해상도, 품질, 스타일 등 맞춤 설정이 가능한 비디오로 변환합니다. 노드는 생성 요청을 보내는 것부터 최종 비디오 출력을 다운로드하는 전 과정을 처리합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `prompt` | 문자열 | 예 | - | 생성할 비디오 콘텐츠의 텍스트 설명 |
| `negative_prompt` | 문자열 | 아니요 | - | 부정적 프롬프트 텍스트 (기본값: 합성, 장면 컷, 아티팩트, 노이즈 등 제외 요소의 광범위한 목록) |
| `resolution` | 문자열 | 아니요 | "16:9 (1920 x 1080)"<br />"9:16 (1080 x 1920)"<br />"1:1 (1152 x 1152)"<br />"4:3 (1536 x 1152)"<br />"3:4 (1152 x 1536)"<br />"21:9 (2560 x 1080)" | 출력 비디오의 해상도 (기본값: "16:9 (1920 x 1080)") |
| `prompt_adherence` | 부동소수점 | 아니요 | 1.0-20.0 | 생성 제어를 위한 가이던스 스케일 (기본값: 4.0) |
| `seed` | 정수 | 아니요 | 0-4294967295 | 랜덤 시드 값 (기본값: 9) |
| `steps` | 정수 | 아니요 | 1-100 | 추론 단계 (기본값: 33) |

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `video` | 비디오 | 텍스트 프롬프트를 기반으로 생성된 비디오 출력 |
