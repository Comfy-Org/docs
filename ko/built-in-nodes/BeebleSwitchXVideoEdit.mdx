# Beeble SwitchX 비디오 편집

Beeble SwitchX로 비디오를 편집하세요. 이 노드는 장면의 모든 요소(배경, 조명, 의상)를 교체하면서도 원래 피사체의 픽셀과 움직임은 그대로 유지합니다. 새로운 모습을 설명하는 참조 이미지 및/또는 텍스트 프롬프트를 제공하세요.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `video` | VIDEO | 예 | N/A | 편집할 입력 비디오입니다. 최대 240프레임, 한 프레임당 최대 약 2.77메가픽셀입니다. |
| `prompt` | STRING | 예 | N/A | 장면에 원하는 새로운 모습을 설명하는 텍스트입니다. |
| `alpha_mode` | COMBO | 예 | `"fill"`<br />`"select"`<br />`"custom"` | 알파 마티 모드입니다. "fill" 모드는 별도의 마티 없이 전체 프레임을 채웁니다. "select" 모드는 단일 키프레임 이미지를 사용해 편집할 영역을 정의합니다. "custom" 모드는 전체 알파 비디오를 사용해 프레임별로 편집할 영역을 정의합니다. |
| `max_resolution` | COMBO | 예 | `"720p"`<br />`"1080p"` | 출력 비디오의 최대 해상도입니다(기본값: "1080p"). |
| `seed` | INT | 예 | 0부터 2147483647까지 | 재현성을 위한 시드 값입니다. 동일한 입력과 동일한 시드를 사용하면 동일한 결과가 나옵니다. |
| `reference_image` | IMAGE | 선택 | N/A | 장면에 원하는 새로운 모습을 설명하는 선택적 참조 이미지입니다. |

### 알파 모드 세부사항

`alpha_mode` 매개변수는 비디오의 어떤 부분을 편집할지 제어합니다:

- **fill**: 비디오 프레임 전체가 편집됩니다. 별도의 알파 마티는 생성되지 않습니다.
- **select**: 편집할 영역을 정의하는 단일 키프레임 이미지를 제공합니다. 노드는 이를 사용해 비디오의 어느 부분을 변경할지 결정합니다.
- **custom**: 프레임별로 편집할 영역을 정의하는 전체 알파 비디오를 제공합니다. 각 프레임의 어느 부분을 편집할지 정밀하게 제어할 수 있습니다.

`select` 모드를 사용할 때는 `alpha_keyframe` 이미지를 반드시 제공해야 합니다. `custom` 모드를 사용할 때는 `alpha_mask` 비디오를 반드시 제공해야 합니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `video` | VIDEO | 장면 변경이 적용된 편집된 비디오입니다. |
| `alpha` | VIDEO | Beeble에서 사용하는 알파 마티입니다. "fill" 모드에서는 별도의 마티가 없으므로 빈 값입니다. |
