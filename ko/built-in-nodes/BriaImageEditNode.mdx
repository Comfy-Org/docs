> 이 문서는 AI로 생성되었습니다. 오류를 발견하거나 개선할 점이 있다면 언제든지 기여해 주세요! [GitHub에서 편집하기](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/BriaImageEditNode/en.md)

Bria FIBO 이미지 편집 노드를 사용하면 텍스트 지침을 통해 기존 이미지를 수정할 수 있습니다. 이 노드는 이미지와 사용자의 프롬프트를 Bria API로 전송하며, API는 FIBO 모델을 사용해 요청에 따라 이미지의 새 버전을 생성합니다. 또한 마스크를 제공하여 특정 영역에만 편집을 제한할 수도 있습니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `model` | COMBO | 예 | `"FIBO"` | 이미지 편집에 사용할 모델 버전입니다. |
| `image` | IMAGE | 예 | - | 편집하려는 입력 이미지입니다. |
| `prompt` | STRING | 아니요 | - | 이미지를 어떻게 편집할지 설명하는 텍스트 지침입니다(기본값: 비어 있음). |
| `negative_prompt` | STRING | 아니요 | - | 편집된 이미지에 나타나지 않기를 원하는 내용을 설명하는 텍스트입니다(기본값: 비어 있음). |
| `structured_prompt` | STRING | 아니요 | - | JSON 형식으로 된 구조화된 편집 프롬프트를 포함한 문자열입니다. 일반적인 프롬프트 대신 이를 사용하면 정밀하고 프로그래밍 방식으로 제어할 수 있습니다(기본값: 비어 있음). |
| `seed` | INT | 예 | 1부터 2147483647까지 | 무작위 생성을 초기화하는 데 사용되는 숫자로, 재현 가능한 결과를 보장합니다(기본값: 1). |
| `guidance_scale` | FLOAT | 예 | 3.0부터 5.0까지 | 생성된 이미지가 프롬프트를 얼마나 충실히 따르는지 제어합니다. 값이 높을수록 더 강하게 준수됩니다(기본값: 3.0). |
| `steps` | INT | 예 | 20부터 50까지 | 모델이 수행할 노이즈 제거 단계의 수입니다(기본값: 50). |
| `moderation` | DYNAMICCOMBO | 예 | `"false"`<br />`"true"` | 콘텐츠 감시를 활성화하거나 비활성화합니다. `"true"`를 선택하면 프롬프트 콘텐츠, 시각적 입력 및 시각적 출력에 대한 추가 감시 옵션이 표시됩니다. |
| `mask` | MASK | 아니요 | - | 선택적 마스크 이미지입니다. 제공되면 이미지의 마스크된 영역에만 편집이 적용됩니다. |

**중요한 제약사항:**

* `prompt` 또는 `structured_prompt` 중 적어도 하나는 반드시 제공해야 합니다. 두 입력 모두 비어 있으면 안 됩니다.
* 정확히 하나의 `image` 입력이 필요합니다.
* `moderation` 매개변수가 `"true"`로 설정된 경우, 세 가지 추가적인 불리언 입력이 사용 가능해집니다: `prompt_content_moderation`(기본값: false), `visual_input_moderation`(기본값: false), 그리고 `visual_output_moderation`(기본값: true).

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `IMAGE` | IMAGE | Bria API에서 반환한 편집된 이미지입니다. |
| `structured_prompt` | STRING | 편집 과정에서 사용되었거나 생성된 구조화된 프롬프트입니다. |
