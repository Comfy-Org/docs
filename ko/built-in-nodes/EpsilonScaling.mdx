연구 논문 "확산 모델의 노출 편향 규명"에서 소개된 에피소드 스케일링 방법을 구현했습니다. 이 방법은 샘플링 과정 중 예측된 노이즈를 스케일링하여 샘플의 품질을 향상시킵니다. 확산 모델의 노출 편향을 완화하기 위해 균일한 스케줄을 사용합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `model` | MODEL | 예 | - | 에피소드 스케일링을 적용할 모델 |
| `scaling_factor` | FLOAT | 아니요 | 0.5 - 1.5 | 예측된 노이즈를 스케일링하는 데 사용되는 요인 (기본값: 1.005) |

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `model` | MODEL | 에피소드 스케일링이 적용된 모델 |
