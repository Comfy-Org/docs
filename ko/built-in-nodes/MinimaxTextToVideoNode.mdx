> 이 문서는 AI로 생성되었습니다. 오류를 발견하거나 개선할 점이 있다면 언제든지 기여해 주세요! [GitHub에서 편집하기](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/MinimaxTextToVideoNode/en.md)

MiniMax의 API를 사용하여 텍스트 프롬프트와 선택적 매개변수를 기반으로 동기식으로 비디오를 생성합니다. 이 노드는 MiniMax의 텍스트-to-비디오 서비스에 연결하여 텍스트 설명을 바탕으로 비디오 콘텐츠를 생성합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `prompt_text` | 문자열 | 예 | - | 비디오 생성을 안내하는 텍스트 프롬프트 |
| `model` | 콤보 | 아니요 | "T2V-01"<br />"T2V-01-Director" | 비디오 생성에 사용할 모델 (기본값: "T2V-01") |
| `seed` | 정수 | 아니요 | 0부터 18446744073709551615까지 | 노이즈 생성에 사용되는 난수 시드 (기본값: 0) |

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `output` | 비디오 | 입력 프롬프트를 기반으로 생성된 비디오 |
