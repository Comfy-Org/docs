WavespeedFlashVSRNode는 저해상도 또는 흐릿한 영상을 고해상도로 높이고 선명도를 복원하는 빠르고 고품질의 비디오 업스케일러입니다. 이 노드는 비디오 입력을 처리하여 사용자가 선택한 더 높은 해상도의 새 비디오를 출력합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `video` | VIDEO | 예 | N/A | 업스케일링할 입력 비디오 파일입니다. MP4 컨테이너 형식이어야 하며, 길이는 5초에서 10분 사이여야 합니다. |
| `target_resolution` | STRING | 예 | `"720p"`<br />`"1080p"`<br />`"2K"`<br />`"4K"` | 업스케일링된 출력 비디오의 원하는 해상도입니다. |

**입력 제약사항:**

* 입력 `video` 파일은 반드시 MP4 컨테이너 형식이어야 합니다.
* 입력 `video`의 길이는 5초에서 10분(600초) 사이여야 합니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `output` | VIDEO | 선택한 목표 해상도의 업스케일링된 비디오 파일입니다. |
