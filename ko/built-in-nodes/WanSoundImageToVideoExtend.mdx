WanSoundImageToVideoExtend 노드는 기존 비디오 잠재표상을 확장하여 추가 프레임을 생성하며, 선택적으로 오디오, 참조 이미지 및 제어 비디오에 의해 안내받을 수 있습니다. 시작 비디오 잠재표상을 받아 더 긴 비디오 시퀀스를 생성하며, 제공된 조건과 오디오 신호를 활용해 새로운 콘텐츠를 유도합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `positive` | CONDITIONING | 예 | - | 비디오에 포함되어야 할 내용을 안내하는 양성 조건 프롬프트 |
| `negative` | CONDITIONING | 예 | - | 비디오에 포함되지 말아야 할 내용을 지정하는 음성 조건 프롬프트 |
| `vae` | VAE | 예 | - | 비디오 프레임 인코딩 및 디코딩에 사용되는 변분 오토인코더 |
| `length` | INT | 예 | 1부터 MAX_RESOLUTION까지 | 비디오 시퀀스에 생성할 총 프레임 수 (기본값: 77, 단계: 4) |
| `video_latent` | LATENT | 예 | - | 확장을 위한 시작점 역할을 하는 초기 비디오 잠재표상. 너비, 높이, 배치 크기 및 프레임 오프셋은 이 잠재표상에서 파생됩니다. |
| `audio_encoder_output` | AUDIOENCODEROUTPUT | 아니요 | - | 선택적 오디오 임베딩으로, 사운드 특성에 따라 비디오 생성에 영향을 줄 수 있습니다. 제공될 경우 오디오는 보간 처리되어 조건에 추가되는 오디오 임베딩 버킷으로 사용됩니다. |
| `ref_image` | IMAGE | 아니요 | - | 선택적 참조 이미지로, 비디오 생성에 시각적 안내를 제공합니다. 이미지는 대상 크기에 맞게 상향 스케일링되고 잠재표상으로 인코딩된 후 양성 및 음성 조건에 추가됩니다. |
| `control_video` | IMAGE | 아니요 | - | 선택적 제어 비디오로, 생성된 비디오의 모션과 스타일을 안내할 수 있습니다. 비디오는 상향 스케일링되고 인코딩된 후 양성 및 음성 조건에 추가됩니다. 제어 비디오는 지정된 `length`로 잘립니다. |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `positive` | CONDITIONING | 비디오 컨텍스트가 적용된 처리된 양성 조건, 오디오 임베딩, 참조 잠재표상, 참조 모션 및 제공된 경우 제어 비디오 포함 |
| `negative` | CONDITIONING | 비디오 컨텍스트가 적용된 처리된 음성 조건, 오디오 임베딩(0으로 설정), 참조 잠재표상, 참조 모션 및 제공된 경우 제어 비디오 포함 |
| `latent` | LATENT | 확장된 비디오 시퀀스를 담고 있는 생성된 비디오 잠재표상으로, 입력 `video_latent`와 목표 `length`에서 파생된 크기를 가진 0으로 초기화됨 |
