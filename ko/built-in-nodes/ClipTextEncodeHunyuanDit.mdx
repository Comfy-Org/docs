`CLIPTextEncodeHunyuanDiT` 노드는 텍스트 설명을 HunyuanDiT 모델이 이해할 수 있는 형식으로 변환합니다. 이 노드는 HunyuanDiT의 이중 텍스트 인코더 아키텍처를 위해 설계된 고급 조건부 노드로, 서로 다른 토크나이저를 통해 두 개의 별도 텍스트 입력을 처리합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `clip` | CLIP | 예 | - | 텍스트 토크나이징과 인코딩에 사용되는 CLIP 모델 인스턴스로, 조건 생성의 핵심입니다. |
| `bert` | STRING | 예 | - | BERT 토크나이저를 통해 인코딩할 텍스트 입력입니다. 문구와 키워드를 선호하며, 다중행 및 동적 프롬프트를 지원합니다. |
| `mt5xl` | STRING | 예 | - | mT5-XL 토크나이저를 통해 인코딩할 텍스트 입력입니다. 다중행 및 동적 프롬프트(다국어)를 지원하며, 완전한 문장과 복잡한 설명도 사용 가능합니다. |

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `CONDITIONING` | CONDITIONING | BERT와 mT5-XL 토크나이징된 텍스트를 결합한 인코딩된 조건 출력으로, 생성 작업에서 추가 처리에 사용됩니다. |
