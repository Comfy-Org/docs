## 개요

이 노드는 HappyHorse 모델을 사용하여 참조 이미지를 기반으로 사람이나 객체가 등장하는 비디오를 생성합니다. 단일 캐릭터 또는 서로 상호작용하는 여러 캐릭터를 포함한 비디오 제작을 지원합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `model` | COMBO | 예 | `"happyhorse-1.0-r2v"` | 비디오 생성에 사용할 HappyHorse 모델입니다. |
| `prompt` | STRING | 예 | 없음 | 생성하고자 하는 비디오에 대한 텍스트 설명입니다. 'character1' 및 'character2'와 같은 식별자를 사용해 참조 캐릭터를 지정하세요. |
| `resolution` | COMBO | 예 | `"720P"`<br />`"1080P"` | 생성된 비디오의 해상도입니다. |
| `ratio` | COMBO | 예 | `"16:9"`<br />`"9:16"`<br />`"1:1"`<br />`"4:3"`<br />`"3:4"` | 생성된 비디오의 종횡비입니다. |
| `duration` | INT | 예 | 3~15 | 생성된 비디오의 길이(초)입니다(기본값: 5). |
| `reference_images` | IMAGE | 예 | 1~9 | 비디오에 등장할 사람이나 객체의 참조 이미지입니다. 최소 한 장의 이미지를 제공해야 합니다. |
| `seed` | INT | 아니요 | 0~2147483647 | 재현 가능한 생성을 위한 시드 값입니다(기본값: 0). 시드는 생성마다 자동으로 변경되도록 설정할 수 있습니다. |
| `watermark` | BOOLEAN | 아니요 | True 또는 False | 결과 비디오에 AI가 생성한 워터마크를 추가할지 여부입니다(기본값: False). |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `VIDEO` | VIDEO | 생성된 비디오 파일입니다. |
