> 이 문서는 AI로 생성되었습니다. 오류를 발견하거나 개선할 점이 있다면 언제든지 기여해 주세요! [GitHub에서 편집하기](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/TripoImageToModelNode/en.md)

Tripo의 API를 사용하여 단일 이미지를 기반으로 3D 모델을 동기식으로 생성합니다. 이 노드는 입력 이미지를 받아 텍스처, 품질 및 모델 속성에 대한 다양한 맞춤 설정 옵션을 통해 3D 모델로 변환합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `image` | IMAGE | 예 | - | 3D 모델 생성에 사용되는 입력 이미지 |
| `model_version` | COMBO | 아니요 | `"v1.4"`<br />`"v3.0"`<br />`"v3.5"`<br />`"v3.6"` | 생성에 사용할 모델 버전 |
| `style` | COMBO | 아니요 | `"None"`<br />`"realistic"`<br />`"cartoon"`<br />`"sculpture"`<br />`"low_poly"` | 생성된 모델의 스타일 설정 (기본값: "None") |
| `texture` | BOOLEAN | 아니요 | - | 모델에 텍스처를 생성할지 여부 (기본값: True) |
| `pbr` | BOOLEAN | 아니요 | - | 물리 기반 렌더링을 사용할지 여부 (기본값: True) |
| `model_seed` | INT | 아니요 | - | 모델 생성용 난수 시드 (기본값: 42) |
| `orientation` | COMBO | 아니요 | `"default"`<br />`"front"`<br />`"back"`<br />`"left"`<br />`"right"`<br />`"top"`<br />`"bottom"` | 생성된 모델의 방향 설정 (기본값: "default") |
| `texture_seed` | INT | 아니요 | - | 텍스처 생성용 난수 시드 (기본값: 42) |
| `texture_quality` | COMBO | 아니요 | `"standard"`<br />`"detailed"` | 텍스처 생성의 품질 수준 (기본값: "standard") |
| `texture_alignment` | COMBO | 아니요 | `"original_image"`<br />`"geometry"` | 텍스처 매핑의 정렬 방법 (기본값: "original_image") |
| `face_limit` | INT | 아니요 | -1부터 500000까지 | 생성된 모델의 최대 면 수, 한계 없음은 -1 (기본값: -1) |
| `quad` | BOOLEAN | 아니요 | - | 삼각형 대신 사각형 면을 사용할지 여부 (기본값: False) |
| `geometry_quality` | COMBO | 아니요 | `"standard"`<br />`"detailed"` | 지오메트리 생성의 품질 수준 (기본값: "standard") |

**참고:** `image` 매개변수는 필수이며, 노드가 작동하려면 반드시 제공해야 합니다. 이미지를 제공하지 않으면 노드는 RuntimeError를 발생시킵니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `model_file` | STRING | 생성된 3D 모델 파일 (후방 호환성을 위해만 제공) |
| `model task_id` | MODEL_TASK_ID | 모델 생성 과정을 추적하는 작업 ID |
| `GLB` | FILE3DGLB | GLB 형식으로 생성된 3D 모델 |
