WanFunInpaintToVideo 노드는 시작 이미지와 끝 이미지 사이를 인페인팅하여 비디오 시퀀스를 생성합니다. 이 노드는 양성 및 음성 조건과 선택적 프레임 이미지를 받아 비디오 랜트를 생성합니다. 노드는 구성 가능한 크기와 길이 매개변수를 통해 비디오 생성을 처리합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `positive` | CONDITIONING | 예 | - | 비디오 생성을 위한 양성 조건 프롬프트 |
| `negative` | CONDITIONING | 예 | - | 비디오 생성 시 피해야 할 음성 조건 프롬프트 |
| `vae` | VAE | 예 | - | 인코딩/디코딩 작업을 위한 VAE 모델 |
| `width` | INT | 예 | 16부터 MAX_RESOLUTION까지 | 출력 비디오 너비(픽셀) (기본값: 832, 단계: 16) |
| `height` | INT | 예 | 16부터 MAX_RESOLUTION까지 | 출력 비디오 높이(픽셀) (기본값: 480, 단계: 16) |
| `length` | INT | 예 | 1부터 MAX_RESOLUTION까지 | 비디오 시퀀스의 프레임 수 (기본값: 81, 단계: 4) |
| `batch_size` | INT | 예 | 1부터 4096까지 | 한 번에 생성할 비디오 수 (기본값: 1) |
| `clip_vision_output` | CLIP_VISION_OUTPUT | 선택 | - | 추가 조건을 위한 옵션 CLIP 비전 출력 |
| `start_image` | IMAGE | 선택 | - | 비디오 생성을 위한 선택적 시작 프레임 이미지 |
| `end_image` | IMAGE | 선택 | - | 비디오 생성을 위한 선택적 끝 프레임 이미지 |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `positive` | CONDITIONING | 처리된 양성 조건 출력 |
| `negative` | CONDITIONING | 처리된 음성 조건 출력 |
| `latent` | LATENT | 생성된 비디오 랜트 표현 |
