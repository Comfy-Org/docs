문자열 노드는 워크플로우에서 텍스트 데이터를 간단히 입력하고 전달하는 방법을 제공합니다. 이 노드는 텍스트 문자열을 입력으로 받아 동일한 문자열을 그대로 출력하므로, 문자열 매개변수가 필요한 다른 노드에 텍스트 입력을 제공하는 데 유용합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `value` | STRING | 예 | 임의의 텍스트 | 노드를 통과할 텍스트 문자열 |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `output` | STRING | 입력으로 제공된 동일한 텍스트 문자열 |
