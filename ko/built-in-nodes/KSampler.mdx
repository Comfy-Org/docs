KSampler는 다음과 같이 작동합니다: 제공된 원본 잠재 이미지 정보를 특정 모델과 긍정적 및 부정적 조건에 따라 수정합니다.
먼저, 설정된 **시드**와 **노이즈 강도**에 따라 원본 이미지 데이터에 노이즈를 추가한 다음, 미리 설정된 **모델**과 **긍정적** 및 **부정적** 가이드 조건을 결합해 이미지를 생성합니다.

## 입력

| 매개변수 이름         | 데이터 유형    | 필수 | 기본값 | 범위/옵션            | 설명                                                                        |
| ---------------------- | ------------ | -------- | ------- | ------------------------ | ---------------------------------------------------------------------------------- |
| 모델                  | checkpoint   | 예     | 없음    | -                        | 노이즈 제거 과정에 사용되는 입력 모델                                         |
| 시드                   | 정수          | 예     | 0       | 0 ~ 18446744073709551615 | 랜덤 노이즈 생성에 사용되며, 동일한 "시드"를 사용하면 동일한 이미지 생성 가능 |
| 단계                  | 정수          | 예     | 20      | 1 ~ 10000                | 노이즈 제거 과정에 사용할 단계 수, 단계가 많을수록 더 정확한 결과 얻음     |
| cfg                    | 실수        | 예     | 8.0     | 0.0 ~ 100.0              | 생성된 이미지가 입력 조건과 얼마나 일치하는지 제어하며, 6~8 권장           |
| 샘플러 이름           | UI 옵션    | 예     | 없음    | 여러 알고리즘      | 노이즈 제거용 샘플러 선택, 생성 속도와 스타일에 영향                          |
| 스케줄러              | UI 옵션    | 예     | 없음    | 여러 스케줄러      | 노이즈 제거 방식 제어, 생성 과정에 영향                           |
| 긍정                 | 컨디셔닝    | 예     | 없음    | -                        | 노이즈 제거를 안내하는 긍정적 조건, 이미지에 나타나기를 원하는 내용        |
| 부정                 | 컨디셔닝    | 예     | 없음    | -                        | 노이즈 제거를 안내하는 부정적 조건, 이미지에 나타나기를 원하지 않는 내용  |
| 잠재 이미지           | 잠재         | 예     | 없음    | -                        | 노이즈 제거에 사용되는 잠재 이미지                                                    |
| 노이즈 제거            | 실수        | 아니요   | 1.0     | 0.0 ~ 1.0                | 노이즈 제거 비율 결정, 값이 낮을수록 입력 이미지와의 연관성이 줄어듦     |
| 제어 후 생성           | UI 옵션    | 아니요   | 없음    | 랜덤/증가/감소/유지      | 각 프롬프트 이후 시드 변경 가능성을 제공                                  |

## 출력

| 매개변수 | 기능                                   |
| -------------- | ------------------------------------------ |
| 잠재         | 샘플러 노이즈 제거 후 잠재 출력 |

def common_ksampler(model, seed, steps, cfg, sampler_name, scheduler, positive, negative, latent, denoise=1.0, disable_noise=False, start_step=None, last_step=None, force_full_denoise=False):
    latent_image = latent["samples"]
    latent_image = comfy.sample.fix_empty_latent_channels(model, latent_image)

    if disable_noise:
        noise = torch.zeros(latent_image.size(), dtype=latent_image.dtype, layout=latent_image.layout, device="cpu")
    else:
        batch_inds = latent["batch_index"] if "batch_index" in latent else None
        noise = comfy.sample.prepare_noise(latent_image, seed, batch_inds)

    noise_mask = None
    if "noise_mask" in latent:
        noise_mask = latent["noise_mask"]

    callback = latent_preview.prepare_callback(model, steps)
    disable_pbar = not comfy.utils.PROGRESS_BAR_ENABLED
    samples = comfy.sample.sample(model, noise, steps, cfg, sampler_name, scheduler, positive, negative, latent_image,
                                  denoise=denoise, disable_noise=disable_noise, start_step=start_step, last_step=last_step,
                                  force_full_denoise=force_full_denoise, noise_mask=noise_mask, callback=callback, disable_pbar=disable_pbar, seed=seed)
    out = latent.copy()
    out["samples"] = samples
    return (out, )
