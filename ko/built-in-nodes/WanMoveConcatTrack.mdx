WanMoveConcatTrack 노드는 두 세트의 모션 트래킹 데이터를 하나의 더 긴 시퀀스로 결합합니다. 이 노드는 입력 트랙의 트랙 경로와 가시성 마스크를 각각의 차원에 따라 연결하여 작동합니다. 만약 입력 트랙이 하나만 제공된다면, 해당 데이터를 그대로 출력으로 전달합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `tracks_1` | TRACKS | 예 | | 결합할 첫 번째 모션 트래킹 데이터 세트입니다. |
| `tracks_2` | TRACKS | 아니요 | | 선택적 두 번째 모션 트래킹 데이터 세트입니다. 제공되지 않으면 `tracks_1`이 그대로 출력으로 전달됩니다. |

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `tracks` | TRACKS | 결합된 모션 트래킹 데이터로, 입력에서 합쳐진 `track_path`와 `track_visibility`를 포함합니다. |
