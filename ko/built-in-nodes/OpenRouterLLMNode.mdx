## 개요

OpenRouter LLM 노드는 OpenRouter 서비스를 통해 제공되는 인기 있는 언어 모델들의 선별된 세트에 텍스트 프롬프트를 전송하고, 생성된 텍스트 응답을 반환합니다. xAI, DeepSeek, Qwen, Mistral, Z.AI (GLM), Moonshot (Kimi), Perplexity Sonar와 같은 공급업체의 모델을 지원하며, 요청에 이미지나 동영상도 선택적으로 포함할 수 있습니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `prompt` | 문자열 | 예 | N/A | 모델에 입력하는 텍스트입니다. |
| `model` | 문자열 | 예 | 여러 옵션 가능 (아래 참고 참조) | 응답을 생성하는 데 사용되는 OpenRouter 모델입니다. |
| `seed` | 정수 | 예 | 0부터 2147483647까지 | 샘플링을 위한 시드입니다. 0으로 설정하면 생략됩니다. 대부분의 모델에서는 이 값을 단순히 참고용으로만 취급합니다. (기본값: 0) |
| `system_prompt` | 문자열 | 아니요 | N/A | 모델의 동작을 지시하는 기본적인 지침입니다. (기본값: "") |

**`model` 매개변수 관련 참고사항:** 사용 가능한 모델 옵션은 동적으로 구성되며, 서로 다른 기능을 가진 모델들이 포함될 수 있습니다. 일부 모델은 추론 노력, 웹 검색, 이미지·동영상 입력과 같은 추가 기능을 지원합니다. 노드는 제공된 이미지 또는 동영상의 수가 모델이 지원하는 최대 수를 초과하지 않도록 확인합니다.

**`seed` 매개변수 관련 참고사항:** `seed` 매개변수는 "control_after_generate" 동작을 가지며, 사용자의 위젯 설정에 따라 각 노드 실행 후 자동으로 변경(예: 무작위화, 증가, 고정 등)될 수 있습니다.

**`system_prompt` 관련 참고사항:** 이 매개변수는 선택적이며, 사용자 인터페이스에서는 고급 매개변수로 표시됩니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `output` | 문자열 | OpenRouter 모델에서 생성된 텍스트 응답입니다. |
