> 이 문서는 AI로 생성되었습니다. 오류를 발견하거나 개선할 점이 있다면 언제든지 기여해 주세요! [GitHub에서 편집하기](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/Wan2TextToVideoApi/en.md)

이 노드는 Wan 2.7 모델을 사용하여 텍스트 설명을 기반으로 비디오를 생성합니다. 요청을 외부 API로 전송하면, 해당 API가 프롬프트를 처리한 후 비디오 파일을 반환합니다. 선택적으로 오디오 클립을 제공하여 비디오의 움직임과 타이밍에 영향을 줄 수 있습니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `model` | COMBO | 예 | `"wan2.7-t2v"` | 비디오 생성에 사용할 특정 모델입니다. |
| `model.prompt` | STRING | 예 | - | 비디오에 포함하고자 하는 요소와 시각적 특성에 대한 설명입니다. 영어와 중국어를 지원합니다. |
| `model.negative_prompt` | STRING | 아니요 | - | 생성된 비디오에서 피하고자 하는 요소나 특성에 대한 설명입니다. |
| `model.resolution` | COMBO | 예 | `"720P"`<br />`"1080P"` | 출력 비디오의 해상도입니다. |
| `model.ratio` | COMBO | 예 | `"16:9"`<br />`"9:16"`<br />`"1:1"`<br />`"4:3"`<br />`"3:4"` | 출력 비디오의 종횡비입니다. |
| `model.duration` | INT | 예 | 2~15 | 비디오 길이(초 단위, 기본값: 5). |
| `audio` | AUDIO | 아니요 | - | 비디오 생성을 위한 오디오 파일로, 립싱크나 박자에 맞춘 움직임을 구현하는 데 사용됩니다. 제공되지 않으면 모델이 자동으로 적합한 배경음악이나 효과음을 생성합니다. 오디오 길이는 1.5초에서 60초 사이여야 합니다. |
| `seed` | INT | 아니요 | 0~2147483647 | 생성의 무작위성을 제어하는 데 사용되는 숫자로, 재현 가능한 결과를 보장합니다(기본값: 0). |
| `prompt_extend` | BOOLEAN | 아니요 | - | 활성화 시, AI의 도움을 받아 프롬프트가 강화됩니다(기본값: True). |
| `watermark` | BOOLEAN | 아니요 | - | 활성화 시, AI가 생성한 워터마크가 결과에 추가됩니다(기본값: False). |

**참고:** `audio` 매개변수는 선택사항입니다. 제공될 경우 길이는 1.5초에서 60초 사이여야 합니다. 생략하면 모델이 자동으로 오디오를 생성합니다.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `output` | VIDEO | 생성된 비디오 파일입니다. |
