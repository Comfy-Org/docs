개별 빨강, 초록, 파랑 값을 지정하여 Recraft 색상을 생성하세요. 이 노드는 RGB 정수 값(0-255)을 받아 다른 Recraft 작업에서 사용할 수 있는 Recraft 색상 형식으로 변환합니다. 또한 선택적으로 기존 Recraft 색상 체인을 제공하면 새 색상으로 이를 확장할 수 있습니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `r` | INT | 예 | 0-255 | 색상의 빨강 값 (기본값: 0) |
| `g` | INT | 예 | 0-255 | 색상의 초록 값 (기본값: 0) |
| `b` | INT | 예 | 0-255 | 색상의 파랑 값 (기본값: 0) |
| `recraft_color` | COLOR | 아니요 | - | 기존 Recraft 색상 체인을 제공하면 새 RGB 색상으로 확장됩니다 |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-----------|-----------|-------------|
| `recraft_color` | COLOR | 지정된 RGB 값을 포함한 생성된 Recraft 색상 객체, 또는 기존 체인이 제공된 경우 이를 확장한 색상 체인 |
