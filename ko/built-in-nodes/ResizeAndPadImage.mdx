ResizeAndPadImage 노드는 이미지를 지정된 치수 안에 맞게 조정하면서 원래의 종횡비를 유지합니다. 이 노드는 이미지를 비례적으로 축소하여 목표 너비와 높이 안에 맞춘 후, 여유 공간을 채우기 위해 가장자리에 패딩을 추가합니다. 패딩 색상과 보간 방법은 맞춤 설정이 가능하며, 이를 통해 패딩 영역의 모양과 크기 조정의 품질을 제어할 수 있습니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `image` | IMAGE | 예 | - | 크기 조정 및 패딩할 입력 이미지 |
| `target_width` | INT | 예 | 1부터 MAX_RESOLUTION까지 | 출력 이미지의 원하는 너비 (기본값: 512) |
| `target_height` | INT | 예 | 1부터 MAX_RESOLUTION까지 | 출력 이미지의 원하는 높이 (기본값: 512) |
| `padding_color` | COMBO | 예 | "white"<br />"black" | 크기 조정된 이미지 주변의 패딩 영역에 사용할 색상 (기본값: "white") |
| `interpolation` | COMBO | 예 | "area"<br />"bicubic"<br />"nearest-exact"<br />"bilinear"<br />"lanczos" | 이미지 크기 조정에 사용되는 보간 방법 (기본값: "area") |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `image` | IMAGE | 크기 조정되고 패딩된 출력 이미지 |
