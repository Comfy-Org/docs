Kling Avatar 2.0 노드는 단일 참조 사진과 오디오 파일을 사용해 방송형 디지털 휴먼 동영상을 생성합니다. 선택적 텍스트 프롬프트를 통해 아바타의 행동, 감정, 카메라 움직임을 정의한 후 말하는 아바타 동영상을 만들 수 있습니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `image` | IMAGE | 예 | - | 아바타 참조 이미지. 너비와 높이는 최소 300px 이상이어야 합니다. 종횡비는 1:2.5에서 2.5:1 사이여야 합니다. |
| `sound_file` | AUDIO | 예 | - | 오디오 입력. 길이는 2초에서 300초 사이여야 합니다. |
| `mode` | COMBO | 예 | `"std"`<br />`"pro"` | 사용할 생성 모드입니다. |
| `prompt` | STRING | 아니요 | - | 아바타의 행동, 감정, 카메라 움직임을 정의하는 옵션 프롬프트입니다. (기본값: 빈 문자열) |
| `seed` | INT | 예 | 0부터 2147483647까지 | 시드는 노드가 다시 실행될지 여부를 제어합니다. 시드와 상관없이 결과는 비결정적입니다. (기본값: 0) |

**참고:** `image`와 `sound_file` 입력에는 특정 검증 요구사항이 있습니다. 이미지는 최소 300x300픽셀이며 종횡비는 1:2.5에서 2.5:1 사이여야 합니다. 오디오 파일은 길이가 2초에서 300초 사이여야 합니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `output` | VIDEO | 생성된 디지털 휴먼 동영상입니다. |
