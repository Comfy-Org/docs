## 개요

Select CLIP Device 노드를 사용하면 CLIP 텍스트 인코더를 실행할 장치(CPU 또는 특정 GPU)를 선택할 수 있습니다. 기본적으로 장치는 모델 로더에 의해 할당되지만, 이를 덮어쓰고 CPU 또는 특정 GPU를 사용하도록 지정할 수 있습니다. 요청한 장치가 사용자의 컴퓨터에 존재하지 않는 경우, 노드는 에러를 발생시키지 않고 그대로 CLIP을 전달하며 메시지를 로그에 기록합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `clip` | CLIP | 예 | | 특정 장치에 할당할 CLIP 텍스트 인코더입니다. |
| `device` | COMBO | 예 | `"default"`<br />`"cpu"`<br />`"gpu:0"`<br />`"gpu:1"`<br />`"gpu:2"`<br />`"gpu:3"`<br />`"gpu:4"`<br />`"gpu:5"`<br />`"gpu:6"`<br />`"gpu:7"` | CLIP 텍스트 인코더를 배치할 장치입니다. `"default"`는 로더가 할당한 장치를 복원합니다. `"cpu"`는 로드 및 언로드 장치를 모두 CPU에 고정합니다. `"gpu:N"`은 N번째로 사용 가능한 GPU에 로드 장치를 고정합니다(기본값: `"default"`). |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `clip` | CLIP | 선택한 장치에 할당된 CLIP 텍스트 인코더, 또는 요청한 장치가 사용 불가능한 경우 원본 CLIP을 그대로 전달합니다. |
