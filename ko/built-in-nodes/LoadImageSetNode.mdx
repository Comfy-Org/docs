LoadImageSetNode는 입력 디렉터리에서 여러 이미지를 불러와 일괄 처리 및 학습 목적으로 사용합니다. 다양한 이미지 형식을 지원하며, 선택적으로 다양한 방법을 사용해 이미지를 리사이즈할 수 있습니다. 이 노드는 선택된 모든 이미지를 한 번에 처리하여 하나의 텐서로 반환합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `images` | IMAGE | 예 | 여러 이미지 파일 | 입력 디렉터리에서 여러 이미지를 선택하세요. PNG, JPG, JPEG, WEBP, BMP, GIF, JPE, APNG, TIF, TIFF 형식을 지원합니다. 이미지를 일괄 선택할 수 있습니다. |
| `resize_method` | STRING | 아니요 | "None"<br />"Stretch"<br />"Crop"<br />"Pad" | 로드된 이미지를 리사이즈하는 옵션 방법(기본값: "None"). "None"을 선택하면 원본 크기를 유지하고, "Stretch"를 선택하면 강제로 리사이즈하며, "Crop"을 선택하면 비율을 유지하면서 잘라내고, "Pad"를 선택하면 비율을 유지하면서 여백을 추가합니다. |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `IMAGE` | IMAGE | 로드된 모든 이미지를 한 번에 담은 텐서로, 이후 처리를 위해 제공됩니다. |
