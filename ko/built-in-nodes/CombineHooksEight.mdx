Combine Hooks [8] 노드는 최대 여덟 개의 서로 다른 훅 그룹을 하나의 통합된 훅 그룹으로 병합합니다. 여러 개의 훅 입력을 받아 ComfyUI의 훅 조합 기능을 사용해 이를 결합합니다. 이를 통해 고급 워크플로우에서 간소화된 처리를 위해 여러 훅 구성들을 하나로 통합할 수 있습니다.

## 입력

| 매개변수 | 데이터 타입 | 입력 유형 | 기본값 | 범위 | 설명 |
|-----------|-----------|------------|---------|-------|-------------|
| `hooks_A` | HOOKS | 선택적 | 없음 | - | 결합할 첫 번째 훅 그룹 |
| `hooks_B` | HOOKS | 선택적 | 없음 | - | 결합할 두 번째 훅 그룹 |
| `hooks_C` | HOOKS | 선택적 | 없음 | - | 결합할 세 번째 훅 그룹 |
| `hooks_D` | HOOKS | 선택적 | 없음 | - | 결합할 네 번째 훅 그룹 |
| `hooks_E` | HOOKS | 선택적 | 없음 | - | 결합할 다섯 번째 훅 그룹 |
| `hooks_F` | HOOKS | 선택적 | 없음 | - | 결합할 여섯 번째 훅 그룹 |
| `hooks_G` | HOOKS | 선택적 | 없음 | - | 결합할 일곱 번째 훅 그룹 |
| `hooks_H` | HOOKS | 선택적 | 없음 | - | 결합할 여덟 번째 훅 그룹 |

**참고:** 모든 입력 매개변수는 선택적입니다. 노드는 제공된 훅 그룹만 결합하며, 비워진 항목은 무시됩니다. 결합에 사용할 수 있는 훅 그룹은 1개부터 8개까지 지정할 수 있습니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `HOOKS` | HOOKS | 제공된 모든 훅 구성이 포함된 단일 통합된 훅 그룹 |
