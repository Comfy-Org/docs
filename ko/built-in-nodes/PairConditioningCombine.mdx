PairConditioningCombine 노드는 두 개의 별도 컨디셔닝 쌍(각각 긍정적 및 부정적 컨디셔닝으로 구성됨)을 하나의 결합된 쌍으로 합칩니다. 두 개의 서로 다른 소스로부터 긍정적 및 부정적 컨디셔닝을 받아 ComfyUI의 내부 로직을 사용해 결합하고, 최종적으로 하나의 긍정적 컨디셔닝과 하나의 부정적 컨디셔닝을 출력합니다. 이 노드는 실험적인 기능이며 고급 컨디셔닝 조작 워크플로우를 위해 설계되었습니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `positive_A` | CONDITIONING | 예 | - | 첫 번째 긍정적 컨디셔닝 입력 |
| `negative_A` | CONDITIONING | 예 | - | 첫 번째 부정적 컨디셔닝 입력 |
| `positive_B` | CONDITIONING | 예 | - | 두 번째 긍정적 컨디셔닝 입력 |
| `negative_B` | CONDITIONING | 예 | - | 두 번째 부정적 컨디셔닝 입력 |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-----------|-----------|-------------|
| `positive` | CONDITIONING | 결합된 긍정적 컨디셔닝 출력 |
| `negative` | CONDITIONING | 결합된 부정적 컨디셔닝 출력 |
