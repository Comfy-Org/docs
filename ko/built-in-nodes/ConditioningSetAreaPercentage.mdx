ConditioningSetAreaPercentage 노드는 조건부 요소의 영향 범위를 백분율 값에 따라 조정하는 데 특화되어 있습니다. 이 노드는 전체 이미지 크기의 백분율로 영역의 치수와 위치를 지정할 수 있으며, 강도 매개변수를 통해 조건부 효과의 강도를 조절할 수 있습니다.

## 입력

| 매개변수 | 데이터 타입 | 설명 |
|-----------|-------------|-------------|
| `CONDITIONING` | CONDITIONING | 수정할 조건부 요소를 나타내며, 영역과 강도 조정을 적용하기 위한 기초가 됩니다. |
| `width`   | `FLOAT`     | 영역의 너비를 전체 이미지 너비의 백분율로 지정하며, 이미지의 어느 정도를 가로 방향으로 영향을 미칠지 결정합니다. |
| `height`  | `FLOAT`     | 영역의 높이를 전체 이미지 높이의 백분율로 지정하며, 조건부 효과의 세로 범위를 조정합니다. |
| `x`       | `FLOAT`     | 영역의 가로 시작점을 전체 이미지 너비의 백분율로 표시하며, 조건부 효과의 위치를 설정합니다. |
| `y`       | `FLOAT`     | 영역의 세로 시작점을 전체 이미지 높이의 백분율로 지정하며, 조건부 효과의 위치를 설정합니다. |
| `strength`| `FLOAT`     | 지정된 영역 내에서 조건부 효과의 강도를 제어하며, 효과의 영향력을 세밀하게 조정할 수 있습니다. |

## 출력

| 매개변수 | 데이터 타입 | 설명 |
|-----------|-------------|-------------|
| `CONDITIONING` | CONDITIONING | 업데이트된 영역과 강도 매개변수를 포함한 수정된 조건부 요소를 반환하며, 추가 처리 또는 적용을 위해 준비됩니다. |
