ImageQuantize 노드는 이미지의 색상을 지정된 수로 줄이고, 선택적으로 디더링 기법을 적용해 시각적 품질을 유지하도록 설계되었습니다. 이 과정은 팔레트 기반 이미지를 생성하거나 특정 애플리케이션에서 색상 복잡도를 줄이는 데 유용합니다.

## 입력

| 필드   | 데이터 타입 | 설명                                                                       |
|---------|-------------|-----------------------------------------------------------------------------------|
| `image` | `IMAGE`     | 양자화할 입력 이미지 텐서입니다. 색상 감소가 이루어지는 주요 데이터로서 노드의 실행에 영향을 미칩니다. |
| `colors`| `INT`       | 이미지를 몇 가지 색상으로 줄일지 지정합니다. 색상 팔레트 크기를 결정함으로써 양자화 과정에 직접적인 영향을 줍니다. |
| `dither`| COMBO[STRING] | 양자화 과정 중 적용할 디더링 기법을 결정하며, 출력 이미지의 시각적 품질과 모양에 영향을 줍니다. |

## 출력

| 필드 | 데이터 타입 | 설명                                                                   |
|-------|-------------|-------------------------------------------------------------------------------|
| `image`| `IMAGE`     | 입력 이미지의 양자화된 버전으로, 색상 복잡도가 줄어들고 선택적으로 디더링이 적용되어 시각적 품질을 유지합니다. |
