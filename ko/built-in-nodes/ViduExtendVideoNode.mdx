> 이 문서는 AI로 생성되었습니다. 오류를 발견하거나 개선할 점이 있다면 언제든지 기여해 주세요! [GitHub에서 편집하기](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/ViduExtendVideoNode/en.md)

ViduExtendVideoNode는 기존 동영상의 길이를 연장하기 위해 추가 프레임을 생성합니다. 지정된 AI 모델을 사용하여 소스 동영상과 선택적 텍스트 프롬프트를 기반으로 매끄러운 연속성을 만들어냅니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `model` | COMBO | 예 | `"viduq2-pro"`<br />`"viduq2-turbo"` | 동영상 확장에 사용할 AI 모델입니다. 모델을 선택하면 해당 모델의 구체적인 지속 시간과 해상도 설정이 표시됩니다. |
| `model.duration` | INT | 예 | 1~7 | 확장된 동영상의 지속 시간(초)(기본값: 4). 모델을 선택한 후 나타납니다. |
| `model.resolution` | COMBO | 예 | `"720p"`<br />`"1080p"` | 출력 동영상의 해상도입니다. 모델을 선택한 후 나타납니다. |
| `video` | VIDEO | 예 | - | 확장할 소스 동영상입니다. |
| `prompt` | STRING | 아니요 | - | 확장된 동영상의 내용을 안내하는 선택적 텍스트 프롬프트입니다(최대 2000자, 기본값: 비어 있음). |
| `seed` | INT | 아니요 | 0~2147483647 | 생성의 무작위성을 제어하는 시드 값입니다(기본값: 1). |
| `end_frame` | IMAGE | 아니요 | - | 확장에 사용할 대상 종료 프레임으로 사용할 선택적 이미지입니다. 제공될 경우 가로세로 비율은 1:4에서 4:1 사이여야 하며, 크기는 최소 128x128픽셀 이상이어야 합니다. |

**참고:** 소스 `video`의 지속 시간은 4초에서 55초 사이여야 합니다.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `output` | VIDEO | 확장된 영상을 포함한 새로 생성된 동영상 파일입니다. |
