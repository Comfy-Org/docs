> 이 문서는 AI가 생성했습니다. 오류를 발견하거나 개선할 점이 있으면 언제든지 기여해 주세요! [GitHub에서 편집하기](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/TextGenerateLTX2Prompt/en.md)

TextGenerateLTX2Prompt 노드는 텍스트 생성 노드의 특수 버전입니다. 사용자의 텍스트 프롬프트를 받아 특정 시스템 지침으로 자동 포맷한 후 언어 모델에 전송하여 향상 또는 완성 작업을 수행합니다. 이 노드는 두 가지 모드로 작동할 수 있으며, 텍스트만 사용하거나 이미지 참조와 함께 사용할 수 있으며 각 경우에 따라 다른 시스템 프롬프트를 사용합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `clip` | CLIP | 예 | | 텍스트 인코딩에 사용되는 CLIP 모델입니다. |
| `prompt` | 문자열 | 예 | | 사용자가 제공한 원시 텍스트 입력으로, 향상되거나 완성될 내용입니다. |
| `max_length` | 정수 | 예 | | 언어 모델이 생성할 수 있는 최대 토큰 수입니다. |
| `sampling_mode` | 콤보 | 예 | `"greedy"`<br />`"top_k"`<br />`"top_p"`<br />`"temperature"` | 텍스트 생성 중 다음 토큰을 선택하는 샘플링 전략입니다. |
| `image` | 이미지 | 아니요 | | 선택적 입력 이미지입니다. 제공된 경우 노드는 이미지 컨텍스트를 위한 자리표시자를 포함한 다른 시스템 프롬프트를 사용합니다. |
| `thinking` | 부울 | 아니요 | | 활성화 시, 모델은 최종 답변을 출력하기 전에 추론 과정을 출력합니다. |
| `use_default_template` | 부울 | 아니요 | | 활성화 시, 노드는 기본 챗 템플릿을 사용해 포맷팅을 진행합니다. |
| `video` | 비디오 | 아니요 | | 선택적 비디오 입력으로, 생성을 위한 추가적인 컨텍스트로 사용할 수 있습니다. |
| `audio` | 오디오 | 아니요 | | 선택적 오디오 입력으로, 생성을 위한 추가적인 컨텍스트로 사용할 수 있습니다. |

**참고:** 노드의 동작은 `image` 입력의 유무에 따라 달라집니다. 이미지를 제공하면 생성된 프롬프트는 이미지의 내용을 기반으로 프롬프트를 확장하는 방법을 설명하는 시스템 프롬프트를 사용해 이미지-비디오 작업용으로 포맷됩니다. 이미지를 제공하지 않으면 텍스트-비디오 작업용으로, 프롬프트를 상세한 비디오 생성 설명으로 확장하는 시스템 프롬프트를 사용해 포맷됩니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `output` | 문자열 | 언어 모델이 생성한 향상되거나 완성된 텍스트 문자열입니다. |
