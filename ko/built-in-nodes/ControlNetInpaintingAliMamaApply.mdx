ControlNetInpaintingAliMamaApply 노드는 컨트롤 이미지와 마스크를 사용하여 긍정적이고 부정적인 조건부를 결합함으로써 인페인팅 작업에 ControlNet 조건부를 적용합니다. 입력 이미지와 마스크를 처리하여 생성 과정을 안내하는 수정된 조건부를 만들어, 이미지의 어떤 영역을 인페인팅할지 정밀하게 제어할 수 있습니다. 이 노드는 강도 조정과 타이밍 제어를 지원하여 생성 과정의 각 단계에서 ControlNet의 영향력을 세밀하게 조정할 수 있습니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `positive` | CONDITIONING | 예 | - | 원하는 콘텐츠를 향해 생성을 유도하는 긍정적 조건부 |
| `negative` | CONDITIONING | 예 | - | 원치 않는 콘텐츠를 피하도록 생성을 유도하는 부정적 조건부 |
| `control_net` | CONTROL_NET | 예 | - | 생성 과정에 추가적인 제어를 제공하는 ControlNet 모델 |
| `vae` | VAE | 예 | - | 이미지를 인코딩하고 디코딩하는 데 사용되는 VAE(변분 오토인코더) |
| `image` | IMAGE | 예 | - | ControlNet의 제어 지침 역할을 하는 입력 이미지 |
| `mask` | MASK | 예 | - | 이미지의 어느 영역을 인페인팅할지 정의하는 마스크 |
| `strength` | FLOAT | 예 | 0.0부터 10.0까지 | ControlNet 효과의 강도 (기본값: 1.0) |
| `start_percent` | FLOAT | 예 | 0.0부터 1.0까지 | 생성 과정 중 ControlNet의 영향력이 시작되는 시점(백분율로 표시, 기본값: 0.0) |
| `end_percent` | FLOAT | 예 | 0.0부터 1.0까지 | 생성 과정 중 ControlNet의 영향력이 멈추는 시점(백분율로 표시, 기본값: 1.0) |

**참고:** ControlNet에 `concat_mask`가 활성화된 경우, 마스크는 반전되어 처리 전 이미지에 적용되며, 마스크는 ControlNet에 전송되는 추가 연결 데이터에 포함됩니다.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `positive` | CONDITIONING | ControlNet을 적용한 수정된 긍정적 조건부 (인페인팅용) |
| `negative` | CONDITIONING | ControlNet을 적용한 수정된 부정적 조건부 (인페인팅용) |
