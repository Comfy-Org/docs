Runway Text to Image 노드는 Runway의 Gen 4 모델을 사용하여 텍스트 프롬프트로부터 이미지를 생성합니다. 텍스트 설명을 제공하고, 선택적으로 참조 이미지를 포함하여 이미지 생성 과정을 안내할 수 있습니다. 이 노드는 API 통신을 처리하고 생성된 이미지를 반환합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `prompt` | 문자열 | 예 | - | 생성을 위한 텍스트 프롬프트 (기본값: "") |
| `ratio` | 콤보 | 예 | "16:9"<br />"1:1"<br />"21:9"<br />"2:3"<br />"3:2"<br />"4:5"<br />"5:4"<br />"9:16"<br />"9:21" | 생성될 이미지의 종횡비 |
| `reference_image` | 이미지 | 아니요 | - | 생성을 안내할 옵션 참조 이미지 |

**참고:** 참조 이미지는 가로세로 크기가 7999x7999 픽셀을 초과하지 않으며, 종횡비가 0.5에서 2.0 사이여야 합니다. 참조 이미지를 제공하면 이미지 생성 과정이 이 이미지를 기반으로 진행됩니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `output` | 이미지 | 텍스트 프롬프트와 선택적 참조 이미지를 기반으로 생성된 이미지 |
