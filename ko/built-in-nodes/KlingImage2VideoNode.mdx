> 이 문서는 AI로 생성되었습니다. 오류를 발견하거나 개선할 점이 있다면 언제든지 기여해 주세요! [GitHub에서 편집하기](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/KlingImage2VideoNode/en.md)

Kling 이미지 to 비디오 노드는 텍스트 프롬프트를 사용하여 시작 참조 이미지를 기반으로 비디오를 생성합니다. 첫 번째 프레임으로 이미지를 받아 긍정적이고 부정적인 텍스트 설명을 바탕으로 비디오 시퀀스를 만들며, 모델, 지속 시간, 종횡비 및 생성 모드를 구성 가능한 옵션으로 제공합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `start_frame` | IMAGE | 예 | - | 비디오를 생성하는 데 사용되는 참조 이미지입니다. 이미지는 최소 300x300 픽셀이어야 하며, 종횡비는 1:2.5에서 2.5:1 사이여야 합니다. |
| `prompt` | STRING | 예 | - | 긍정적 텍스트 프롬프트. 최대 500자. |
| `negative_prompt` | STRING | 예 | - | 부정적 텍스트 프롬프트. 최대 500자. |
| `model_name` | COMBO | 예 | `"kling-v2-master"`<br />`"kling-v2-1-master"`<br />`"kling-v2-5-turbo"`<br />`"kling-v2-1"`<br />`"kling-v1-6"`<br />`"kling-v1-5"`<br />`"kling-v1-4"`<br />`"kling-v1-0"` | 비디오 생성에 사용되는 모델(기본값: `"kling-v2-master"`). |
| `cfg_scale` | FLOAT | 예 | 0.0부터 1.0까지 | 비디오가 프롬프트를 얼마나 정확히 따르는지를 제어합니다. 값이 높을수록 더 강하게 준수됩니다(기본값: 0.8). |
| `mode` | COMBO | 예 | `"std"`<br />`"pro"` | 생성 모드입니다. `"std"`는 표준 품질, `"pro"`는 고화질입니다(기본값: `"std"`). 참고: `kling-v2-5-turbo` 모델을 사용할 경우, `"std"` 모드는 지원되지 않으며 자동으로 `"pro"`로 전환됩니다. |
| `aspect_ratio` | COMBO | 예 | `"16:9"`<br />`"9:16"`<br />`"1:1"` | 생성된 비디오의 종횡비(기본값: `"16:9"`). |
| `duration` | COMBO | 예 | `"5"`<br />`"10"` | 생성된 비디오의 지속 시간(초)(기본값: `"5"`). |

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `output` | VIDEO | 생성된 비디오 출력입니다. |
| `video_id` | STRING | 생성된 비디오의 고유 식별자입니다. |
| `duration` | STRING | 생성된 비디오의 지속 시간 정보입니다. |
