이 노드는 LoRA(저랭크 적응) 모델을 적용하여 신경망의 모델 구성 요소만 수정하는 후크를 생성합니다. 체크포인트 파일을 로드하고, 지정된 강도로 모델에 적용하며 CLIP 구성 요소는 그대로 유지합니다. 이 노드는 기본 CreateHookModelAsLora 클래스의 기능을 확장한 실험적 노드입니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `ckpt_name` | 문자열 | 예 | 여러 옵션 제공 | LoRA 모델로 로드할 체크포인트 파일입니다. 사용 가능한 옵션은 체크포인트 폴더 내용에 따라 달라집니다. |
| `strength_model` | 부동소수점 | 예 | -20.0부터 20.0까지 | 모델 구성 요소에 LoRA를 적용할 때의 강도 배율 (기본값: 1.0) |
| `prev_hooks` | 후크 | 아니요 | - | 이 후크와 연결할 수 있는 선택적 이전 후크 |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-----------|-----------|-------------|
| `hooks` | 후크 | LoRA 모델 수정을 포함한 생성된 후크 그룹 |
