제공된 프롬프트를 기반으로 이미지의 배경을 교체합니다. 이 노드는 Recraft API를 사용하여 텍스트 설명에 따라 이미지의 새 배경을 생성하므로, 주요 피사체는 그대로 유지하면서 배경을 완전히 변환할 수 있습니다.

## 입력

| 매개변수 | 설명 | 데이터 타입 | 필수 여부 | 범위 |
| --- | --- | --- | --- | --- |
| `이미지` | 처리할 입력 이미지 | IMAGE | 예 | - |
| `프롬프트` | 이미지 생성을 위한 프롬프트 (기본값: 비어 있음) | STRING | 예 | - |
| `개수` | 생성할 이미지 수 (기본값: 1) | INT | 예 | 1-6 |
| `시드` | 노드 재실행 여부를 결정하는 시드; 실제 결과는 시드와 관계없이 비결정적입니다 (기본값: 0) | INT | 예 | 0-18446744073709551615 |
| `recraft 스타일` | 생성된 배경에 대한 선택적 스타일 선택. 제공되지 않으면 기본적으로 "realistic_image" 스타일이 사용됩니다 | STYLEV3 | 아니요 | - |
| `부정 프롬프트` | 이미지에서 원하지 않는 요소에 대한 선택적 텍스트 설명 (기본값: 비어 있음) | STRING | 아니요 | - |

**참고:** `seed` 매개변수는 노드가 재실행되는 시점을 제어하지만, 외부 API의 특성상 결정적 결과를 보장하지는 않습니다.

## 출력

| 출력 이름 | 설명 | 데이터 타입 |
| --- | --- | --- |
| `IMAGE` | 배경이 교체된 생성된 이미지(들) | IMAGE |
