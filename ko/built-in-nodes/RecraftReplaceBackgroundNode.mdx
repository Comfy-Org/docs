> 이 문서는 AI로 생성되었습니다. 오류를 발견하거나 개선할 점이 있다면 언제든지 기여해 주세요! [GitHub에서 편집하기](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/RecraftReplaceBackgroundNode/en.md)

제공된 프롬프트를 기반으로 이미지의 배경을 교체합니다. 이 노드는 Recraft API를 사용하여 텍스트 설명에 따라 이미지의 새로운 배경을 생성하며, 주요 피사체는 그대로 유지한 채 배경을 완전히 변형할 수 있습니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `image` | IMAGE | 예 | - | 처리할 입력 이미지 |
| `prompt` | STRING | 예 | - | 이미지 생성을 위한 프롬프트 (기본값: 빈 문자열) |
| `n` | INT | 예 | 1-6 | 생성할 이미지의 수 (기본값: 1) |
| `seed` | INT | 예 | 0-18446744073709551615 | 노드가 다시 실행될지 여부를 결정하는 시드; 실제 결과는 시드와 관계없이 비결정적입니다 (기본값: 0) |
| `recraft_style` | STYLEV3 | 선택사항 | - | 생성된 배경에 대한 선택적 스타일 지정. 제공되지 않으면 기본적으로 "realistic_image" 스타일이 적용됩니다 |
| `negative_prompt` | STRING | 선택사항 | - | 이미지에서 원치 않는 요소에 대한 선택적 텍스트 설명 (기본값: 빈 문자열) |

**참고:** `seed` 매개변수는 노드가 재실행되는 시점을 제어하지만, 외부 API의 특성상 결정적인 결과를 보장하지 않습니다.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-----------|-----------|-------------|
| `IMAGE` | IMAGE | 배경이 교체된 생성된 이미지(들) |
