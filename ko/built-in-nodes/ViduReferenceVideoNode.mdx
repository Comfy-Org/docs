> 이 문서는 AI로 생성되었습니다. 오류를 발견하거나 개선할 점이 있다면 언제든지 기여해 주세요! [GitHub에서 편집하기](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/ViduReferenceVideoNode/en.md)

Vidu 참조 비디오 노드는 여러 참조 이미지와 텍스트 프롬프트를 사용해 비디오를 생성합니다. 이 노드는 제공된 이미지와 설명을 바탕으로 일관된 비디오 콘텐츠를 생성하기 위해 AI 모델을 사용합니다. 노드는 지속 시간, 종횡비, 해상도, 움직임 제어 등 다양한 비디오 설정을 지원합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `model` | COMBO | 예 | `"viduq1"` | 비디오 생성용 모델 이름 (기본값: "viduq1") |
| `images` | IMAGE | 예 | - | 비디오를 생성할 때 일관된 피사체를 위한 참조 이미지 (최대 7개) |
| `prompt` | STRING | 예 | - | 비디오 생성을 위한 텍스트 설명 |
| `duration` | INT | 아니요 | 5-5 | 출력 비디오의 지속 시간(초) (기본값: 5) |
| `seed` | INT | 아니요 | 0-2147483647 | 비디오 생성을 위한 시드 값 (랜덤일 경우 0) (기본값: 0) |
| `aspect_ratio` | COMBO | 아니요 | `"16:9"`<br />`"9:16"`<br />`"1:1"` | 출력 비디오의 종횡비 |
| `resolution` | COMBO | 아니요 | `"1080p"` | 지원되는 값은 모델 및 지속 시간에 따라 다를 수 있음 (기본값: "1080p") |
| `movement_amplitude` | COMBO | 아니요 | `"auto"`<br />`"small"`<br />`"medium"`<br />`"large"` | 프레임 내 객체의 움직임 진폭 (기본값: "auto") |

**제약사항 및 한계:**

- `prompt` 필드는 필수이며 비워둘 수 없습니다.
- 참조용 이미지는 최대 7개까지 허용됩니다.
- 각 이미지의 종횡비는 1:4에서 4:1 사이여야 합니다.
- 각 이미지의 최소 크기는 128x128픽셀이어야 합니다.
- 지속 시간은 고정되어 5초입니다.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `output` | VIDEO | 참조 이미지와 프롬프트를 기반으로 생성된 비디오 |
