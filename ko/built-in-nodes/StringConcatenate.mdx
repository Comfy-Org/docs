StringConcatenate 노드는 두 개의 텍스트 문자열을 지정된 구분자로 결합하여 하나의 문자열로 만듭니다. 두 개의 입력 문자열과 구분자 문자 또는 문자열을 받아, 두 입력 사이에 구분자를 삽입하여 연결된 단일 문자열을 출력합니다.

## 입력

| 매개변수 | 설명 | 데이터 타입 | 필수 여부 | 범위 |
| --- | --- | --- | --- | --- |
| `문자열_a` | 연결할 첫 번째 텍스트 문자열 | STRING | 예 | - |
| `문자열_b` | 연결할 두 번째 텍스트 문자열 | STRING | 예 | - |
| `구분자` | 두 입력 문자열 사이에 삽입할 문자 또는 문자열 (기본값: 빈 문자열) | STRING | 아니요 | - |

## 출력

| 출력 이름 | 설명 | 데이터 타입 |
| --- | --- | --- |
| `output` | string_a와 string_b 사이에 구분자가 삽입되어 결합된 문자열 | STRING |
