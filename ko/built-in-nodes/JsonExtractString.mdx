JsonExtractString 노드는 JSON 데이터가 포함된 텍스트 문자열을 읽고 특정 키와 연결된 값을 추출합니다. 추출한 값을 문자열로 변환합니다. JSON이 유효하지 않거나, 키를 찾을 수 없거나, 값이 null인 경우 노드는 빈 문자열을 반환합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `json_string` | STRING | 예 | N/A | 파싱할 JSON 데이터가 포함된 텍스트입니다. 이 필드는 다중행 입력을 지원합니다. |
| `key` | STRING | 예 | N/A | JSON 객체에서 추출하려는 특정 키입니다. 이 필드는 단일 행 입력만 지원합니다. |

**참고:** 이 노드는 JSON 객체(딕셔너리)에서만 값을 추출합니다. 만약 파싱된 JSON이 객체가 아니거나 지정된 키가 존재하지 않는다면 출력은 빈 문자열이 됩니다. 키와 연결된 값이 `null`인 경우에도 노드는 빈 문자열을 반환합니다.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `output` | STRING | 지정된 키에 대해 JSON에서 추출한 문자열 값이며, 추출에 실패하면 빈 문자열이 됩니다. |
