ElevenLabs Speech to Text 노드는 오디오 파일을 텍스트로 변환합니다. ElevenLabs의 API를 사용해 말한 내용을 글로 된 대본으로 변환하며, 자동 언어 감지, 다양한 화자 식별, 음악이나 웃음과 같은 비말음 태그 지정 등의 기능을 지원합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `audio` | AUDIO | 예 | - | 변환할 오디오. |
| `model` | COMBO | 예 | `"scribe_v2"` | 변환에 사용할 모델. 이 모델을 선택하면 추가 매개변수가 표시됩니다. |
| `tag_audio_events` | BOOLEAN | 아니요 | - | (웃음), (음악) 등과 같은 소리를 대본에 표시합니다. 이 매개변수는 `"scribe_v2"` 모델을 선택했을 때 표시됩니다. (기본값: False) |
| `diarize` | BOOLEAN | 아니요 | - | 어떤 화자가 말하고 있는지 표시합니다. 이 매개변수는 `"scribe_v2"` 모델을 선택했을 때 표시됩니다. (기본값: False) |
| `diarization_threshold` | FLOAT | 아니요 | 0.1 - 0.4 | 화자 분리 민감도. 값이 낮을수록 화자 변화에 더 민감합니다. 이 매개변수는 `"scribe_v2"` 모델을 선택하고 `diarize`가 활성화된 경우 표시됩니다. (기본값: 0.22) |
| `temperature` | FLOAT | 아니요 | 0.0 - 2.0 | 무작위성 조절. 0.0은 모델 기본값을 사용합니다. 값이 높을수록 무작위성이 증가합니다. 이 매개변수는 `"scribe_v2"` 모델을 선택했을 때 표시됩니다. (기본값: 0.0) |
| `timestamps_granularity` | COMBO | 아니요 | `"word"`<br />`"character"`<br />`"none"` | 대본 단어의 타이밍 정밀도. 이 매개변수는 `"scribe_v2"` 모델을 선택했을 때 표시됩니다. (기본값: "word") |
| `language_code` | STRING | 아니요 | - | ISO-639-1 또는 ISO-639-3 언어 코드 (예: 'en', 'es', 'fra'). 자동 감지를 위해 비워두세요. (기본값: "") |
| `num_speakers` | INT | 아니요 | 0 - 32 | 예측할 최대 화자 수. 자동 감지를 위해 0으로 설정하세요. (기본값: 0) |
| `seed` | INT | 아니요 | 0 - 2147483647 | 재현성을 위한 시드(결정론적 보장 없음). (기본값: 1) |

**참고:** `num_speakers` 매개변수는 `diarize` 옵션이 활성화된 경우 0보다 큰 값을 설정할 수 없습니다. `diarize`를 비활성화하거나 `num_speakers`를 0으로 설정해야 합니다.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `text` | STRING | 오디오에서 변환된 텍스트. |
| `language_code` | STRING | 오디오의 감지된 언어 코드. |
| `words_json` | STRING | 세부적인 단어 수준 정보를 포함한 JSON 형식 문자열. 타임스탬프와 화자 라벨이 활성화된 경우 포함됩니다. |
