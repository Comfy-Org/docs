입력 이미지로부터 SVG 벡터 그래픽을 생성합니다. 이 노드는 각 이미지를 Recraft API로 보내고 결과를 하나의 SVG 출력으로 결합하여 래스터 이미지를 확장 가능한 벡터 형식으로 변환합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `image` | IMAGE | 예 | - | SVG 형식으로 변환할 입력 이미지입니다. 여러 이미지의 일괄 처리를 지원합니다. |

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `SVG` | SVG | 처리된 모든 이미지를 하나의 SVG 문서로 결합한 벡터 그래픽 출력입니다. |
