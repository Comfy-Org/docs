InpaintModelConditioning 노드는 인페인팅 모델의 컨디셔닝 과정을 용이하게 하도록 설계되었으며, 다양한 컨디셔닝 입력을 통합하고 조작하여 인페인팅 결과를 맞춤화할 수 있도록 합니다. 이 노드는 특정 모델 체크포인트 로딩, 스타일 또는 제어 네트 모델 적용부터 컨디셔닝 요소의 인코딩과 결합까지 폭넓은 기능을 포함하며, 인페인팅 작업을 맞춤화하는 종합적인 도구로 작용합니다.

## 입력

| 매개변수 | Comfy 데이터형        | 설명 |
|-----------|--------------------|-------------|
| `positive`| `CONDITIONING`     | 인페인팅 모델에 적용될 긍정적 컨디셔닝 정보 또는 매개변수를 나타냅니다. 이 입력은 인페인팅 작업이 수행되어야 할 맥락이나 제약 조건을 정의하는 데 필수적이며, 최종 출력에 큰 영향을 미칩니다. |
| `negative`| `CONDITIONING`     | 인페인팅 모델에 적용될 부정적 컨디셔닝 정보 또는 매개변수를 나타냅니다. 이 입력은 인페인팅 과정에서 피해야 할 조건이나 맥락을 지정하는 데 필수적이며, 최종 출력에 영향을 줍니다. |
| `vae`     | `VAE`              | 컨디셔닝 과정에 사용될 VAE 모델을 지정합니다. 이 입력은 활용될 VAE 모델의 구체적인 아키텍처와 매개변수를 결정하는 데 중요합니다. |
| `pixels`  | `IMAGE`            | 인페인팅할 이미지의 픽셀 데이터를 나타냅니다. 이 입력은 인페인팅 작업에 필요한 시각적 맥락을 제공하는 데 필수적입니다. |
| `mask`    | `MASK`             | 이미지에 적용될 마스크를 지정하며, 인페인팅할 영역을 표시합니다. 이 입력은 이미지 내에서 인페인팅이 필요한 특정 영역을 정의하는 데 필수적입니다. |

## 출력

| 매개변수 | 데이터형 | 설명 |
|-----------|--------------|-------------|
| `positive`| `CONDITIONING` | 처리 후 수정된 긍정적 컨디셔닝 정보로, 인페인팅 모델에 적용할 준비가 된 상태입니다. 이 출력은 지정된 긍정적 조건에 따라 인페인팅 과정을 안내하는 데 필수적입니다. |
| `negative`| `CONDITIONING` | 처리 후 수정된 부정적 컨디셔닝 정보로, 인페인팅 모델에 적용할 준비가 된 상태입니다. 이 출력은 지정된 부정적 조건에 따라 인페인팅 과정을 안내하는 데 필수적입니다. |
| `latent`  | `LATENT`     | 컨디셔닝 과정에서 도출된 잠재 표현입니다. 이 출력은 인페인팅되는 이미지의 근본적인 특징과 속성을 이해하는 데 매우 중요합니다. |
