> 이 문서는 AI로 생성되었습니다. 오류를 발견하거나 개선할 점이 있다면 언제든지 기여해 주세요! [GitHub에서 편집하기](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/MinimaxImageToVideoNode/en.md)

MiniMax의 API를 사용하여 이미지와 프롬프트, 그리고 선택적 매개변수를 기반으로 동영상 시퀀스를 동기적으로 생성합니다. 이 노드는 입력 이미지와 텍스트 설명을 받아 동영상 시퀀스를 만들며, 다양한 모델 옵션과 구성 설정을 제공합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `image` | IMAGE | 예 | - | 동영상 생성의 첫 프레임으로 사용할 이미지 |
| `prompt_text` | STRING | 예 | - | 동영상 생성을 안내하는 텍스트 프롬프트 (기본값: 빈 문자열) |
| `model` | COMBO | 예 | "I2V-01-Director"<br />"I2V-01"<br />"I2V-01-live" | 동영상 생성에 사용할 모델 (기본값: "I2V-01") |
| `seed` | INT | 아니요 | 0부터 18446744073709551615까지 | 노이즈 생성에 사용되는 난수 시드 (기본값: 0) |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-----------|-----------|-------------|
| `output` | VIDEO | 생성된 동영상 출력 |
