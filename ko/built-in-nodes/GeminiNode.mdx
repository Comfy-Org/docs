> 이 문서는 AI가 생성했습니다. 오류를 발견하거나 개선할 점이 있다면 언제든지 기여해 주세요! [GitHub에서 편집하기](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/GeminiNode/en.md)

이 노드를 사용하면 사용자가 구글의 Gemini AI 모델과 상호작용하여 텍스트 응답을 생성할 수 있습니다. 텍스트, 이미지, 오디오, 비디오, 파일 등 다양한 유형의 입력을 제공하여 모델이 더 관련성 있고 의미 있는 응답을 생성하도록 할 수 있습니다. 이 노드는 모든 API 통신과 응답 파싱을 자동으로 처리합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `prompt` | 문자열 | 예 | - | 모델에 제공되는 텍스트 입력으로, 응답을 생성하는 데 사용됩니다. 세부 지침, 질문 또는 모델의 문맥을 포함할 수 있습니다. 기본값: 빈 문자열. |
| `model` | 콤보 | 예 | `gemini-2.5-pro-preview-05-06`<br />`gemini-2.5-flash-preview-04-17`<br />`gemini-2.5-pro`<br />`gemini-2.5-flash`<br />`gemini-3-pro-preview`<br />`gemini-3-1-pro`<br />`gemini-3-1-flash-lite` | 응답 생성에 사용할 Gemini 모델입니다. 기본값: gemini-3-1-pro. |
| `seed` | 정수 | 예 | 0부터 18446744073709551615까지 | 시드를 특정 값으로 고정하면, 동일한 요청에 대해 모델이 최대한 동일한 응답을 제공하려고 노력합니다. 결정론적 출력이 보장되지는 않습니다. 또한 모델이나 온도와 같은 매개변수 설정을 변경하면 동일한 시드 값을 사용하더라도 응답에 변동이 생길 수 있습니다. 기본적으로 임의의 시드 값이 사용됩니다. 기본값: 42. |
| `images` | 이미지 | 선택 | - | 모델의 문맥으로 사용할 선택적 이미지입니다. 여러 이미지를 포함하려면 배치 이미지 노드를 사용할 수 있습니다. 기본값: 없음. |
| `audio` | 오디오 | 선택 | - | 모델의 문맥으로 사용할 선택적 오디오입니다. 기본값: 없음. |
| `video` | 비디오 | 선택 | - | 모델의 문맥으로 사용할 선택적 비디오입니다. 기본값: 없음. |
| `files` | GEMINI_INPUT_FILES | 선택 | - | 모델의 문맥으로 사용할 선택적 파일입니다. Gemini 콘텐츠 생성 입력 파일 노드에서 입력을 받습니다. 기본값: 없음. |
| `system_prompt` | 문자열 | 선택 | - | AI의 행동을 규정하는 기초적인 지침입니다. 기본값: 빈 문자열. 이는 고급 매개변수입니다. |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `STRING` | 문자열 | Gemini 모델이 생성한 텍스트 응답입니다. |
