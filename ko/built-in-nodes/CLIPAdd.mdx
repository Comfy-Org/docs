CLIPAdd 노드는 두 개의 CLIP 모델을 결합하여 핵심 패치를 병합합니다. 첫 번째 CLIP 모델의 사본을 만든 후, 두 번째 모델의 핵심 패치 중 위치 ID와 로짓 스케일 파라미터를 제외한 대부분의 패치를 추가합니다. 이를 통해 첫 번째 모델의 구조를 유지하면서 서로 다른 CLIP 모델의 특징을 혼합할 수 있습니다.

## 입력

| 매개변수 | 데이터 유형 | 입력 유형 | 기본값 | 범위 | 설명 |
|-----------|-----------|------------|---------|-------|-------------|
| `clip1` | CLIP | 필수 | - | - | 병합의 기반으로 사용될 기본 CLIP 모델 |
| `clip2` | CLIP | 필수 | - | - | 추가 패치를 제공하는 보조 CLIP 모델 |

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `CLIP` | CLIP | 두 입력 모델의 특징을 결합한 병합된 CLIP 모델을 반환합니다 |
