ModelSamplingAuraFlow 노드는 특수한 샘플링 구성 설정을 확산 모델에 적용하며, 특히 AuraFlow 모델 아키텍처용으로 설계되었습니다. 이 노드는 모델의 샘플링 동작을 수정하며, 샘플링 분포를 조정하는 시프트 파라미터를 적용합니다. 이 노드는 SD3 모델 샘플링 프레임워크를 상속받으며, 샘플링 과정을 세밀하게 제어할 수 있습니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `model` | MODEL | 예 | - | AuraFlow 샘플링 구성 설정을 적용할 확산 모델 |
| `shift` | FLOAT | 예 | 0.0 - 100.0 | 샘플링 분포에 적용할 시프트 값 (기본값: 1.73) |

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `model` | MODEL | AuraFlow 샘플링 구성 설정이 적용된 수정된 모델 |
