## 개요

SAM3 추적 세션에서 특정 추적된 객체를 인덱스 번호로 선택하고 이를 하나의 출력 마스크로 결합합니다. 이를 통해 추적 결과 중 어떤 객체를 유지하고 어떤 객체를 무시할지 선택할 수 있습니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `track_data` | SAM3TRACKDATA | 예 | N/A | SAM3 추적자 노드에서 출력된 추적 데이터로, 압축된 마스크와 원본 이미지 크기를 포함합니다. |
| `object_indices` | STRING | 아니요 | 쉼표로 구분된 임의의 정수 목록 | 출력 마스크에 포함할 객체의 인덱스를 쉼표로 구분하여 입력합니다(예: '0,2,3'). 비워두면 모든 추적된 객체가 포함됩니다. |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `masks` | MASK | 각 프레임에 대한 단일 이진 마스크로, 선택된 객체들이 하나의 마스크로 결합됩니다. 객체를 선택하지 않았거나 추적 데이터가 없으면 0 마스크를 반환합니다. |
