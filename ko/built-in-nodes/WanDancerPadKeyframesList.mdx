## 개요

이 노드는 이미지 시퀀스와 선택적 오디오 트랙을 입력받아 지정된 수의 패딩된 세그먼트로 분할합니다. 비디오 생성을 위한 키프레임 시퀀스를 준비하는 데 사용되며, 각 세그먼트는 일정한 길이로 패딩되고, 해당 프레임이 유효한지를 나타내는 마스크가 함께 생성됩니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `images` | IMAGE | 예 | N/A | 세그먼트로 분할할 입력 이미지 시퀀스입니다. |
| `segment_length` | INT | 예 | 1부터 10000까지 | 각 세그먼트의 길이(프레임 단위, 기본값: 149) |
| `num_segments` | INT | 예 | 1부터 100까지 | 출력될 패딩된 세그먼트의 수(기본값: 1) |
| `audio` | AUDIO | 아니요 | N/A | 출력되는 각 세그먼트에 대해 슬라이스할 오디오입니다. |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `keyframes_sequence` | IMAGE | 패딩된 키프레임 시퀀스 목록으로, 각 세그먼트별로 하나씩 제공됩니다. |
| `keyframes_mask` | MASK | 각 세그먼트에 대한 유효한 프레임을 나타내는 마스크 목록입니다. |
| `audio_segment` | AUDIO | 각 비디오 세그먼트별로 하나씩 제공되는 오디오 세그먼트 목록입니다. |
