ConditioningSetAreaPercentageVideo 노드는 비디오 생성을 위해 특정 영역과 시간적 구역을 정의하여 조건부 데이터를 수정합니다. 전체 크기에 대한 백분율 값을 사용해 조건부가 적용될 영역의 위치, 크기 및 지속시간을 설정할 수 있습니다. 이는 비디오 시퀀스의 특정 부분에 집중해 생성하는 데 유용합니다.

## 입력

| 매개변수 | 데이터 타입 | 입력 유형 | 기본값 | 범위 | 설명 |
|-----------|-----------|------------|---------|-------|-------------|
| `conditioning` | CONDITIONING | 필수 | - | - | 수정할 조건부 데이터 |
| `width` | FLOAT | 필수 | 1.0 | 0.0 - 1.0 | 영역의 너비(전체 너비의 백분율) |
| `height` | FLOAT | 필수 | 1.0 | 0.0 - 1.0 | 영역의 높이(전체 높이의 백분율) |
| `temporal` | FLOAT | 필수 | 1.0 | 0.0 - 1.0 | 영역의 시간적 지속시간(전체 비디오 길이의 백분율) |
| `x` | FLOAT | 필수 | 0.0 | 0.0 - 1.0 | 영역의 가로 시작 위치(백분율) |
| `y` | FLOAT | 필수 | 0.0 | 0.0 - 1.0 | 영역의 세로 시작 위치(백분율) |
| `z` | FLOAT | 필수 | 0.0 | 0.0 - 1.0 | 영역의 시간적 시작 위치(비디오 타임라인의 백분율) |
| `strength` | FLOAT | 필수 | 1.0 | 0.0 - 10.0 | 정의된 영역 내 조건부에 적용되는 강도 승수 |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `conditioning` | CONDITIONING | 지정된 영역과 강도 설정이 적용된 수정된 조건부 데이터 |
