WanMoveTracksFromCoords 노드는 좌표를 JSON 형식으로 된 문자열로부터 모션 트랙을 생성합니다. 이 노드는 좌표 데이터를 다른 비디오 처리 노드에서 사용할 수 있는 텐서 형식으로 변환하며, 선택적으로 마스크를 적용하여 시간에 따른 트랙의 가시성을 제어할 수도 있습니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `track_coords` | STRING | 선택 | N/A | 트랙의 좌표 데이터를 포함한 JSON 형식의 문자열입니다. 기본값은 빈 리스트(`"[]"`)입니다. |
| `track_mask` | MASK | 선택 | N/A | 선택적 마스크입니다. 제공될 경우, 노드는 이를 사용해 각 프레임별 트랙의 가시성을 결정합니다. |

**참고:** `track_coords` 입력은 특정한 JSON 구조를 요구합니다. 트랙의 리스트이며, 각 트랙은 프레임의 리스트이고, 각 프레임은 `x`와 `y` 좌표를 가진 객체입니다. 모든 트랙에서 프레임의 수는 일정해야 합니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `tracks` | TRACKS | 생성된 트랙 데이터로, 각 트랙의 경로 좌표와 가시성 정보를 포함합니다. |
| `track_length` | INT | 생성된 트랙의 총 프레임 수입니다. |
