> 이 문서는 AI로 생성되었습니다. 오류를 발견하거나 개선할 점이 있다면 언제든지 기여해 주세요! [GitHub에서 편집하기](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/LumaConceptsNode/en.md)

Luma Text to Video 및 Luma Image to Video 노드와 함께 사용할 하나 이상의 카메라 개념을 보유합니다. 이 노드를 사용하면 최대 네 개의 카메라 개념을 선택하고, 선택적으로 기존 개념 체인과 결합할 수 있습니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `concept1` | STRING | 예 | 여러 옵션 제공<br />"없음" 옵션 포함 | 사용 가능한 Luma 개념 중 첫 번째 카메라 개념 선택 |
| `concept2` | STRING | 예 | 여러 옵션 제공<br />"없음" 옵션 포함 | 사용 가능한 Luma 개념 중 두 번째 카메라 개념 선택 |
| `concept3` | STRING | 예 | 여러 옵션 제공<br />"없음" 옵션 포함 | 사용 가능한 Luma 개념 중 세 번째 카메라 개념 선택 |
| `concept4` | STRING | 예 | 여러 옵션 제공<br />"없음" 옵션 포함 | 사용 가능한 Luma 개념 중 네 번째 카메라 개념 선택 |
| `luma_concepts` | LUMA_CONCEPTS | 아니요 | N/A | 여기서 선택한 개념에 추가할 옵션적인 카메라 개념 |

**참고:** 모든 개념 매개변수(`concept1`부터 `concept4`)는 네 개의 개념 슬롯 모두를 사용하지 않으려면 "없음"으로 설정할 수 있습니다. 노드는 제공된 `luma_concepts`를 선택한 개념들과 병합하여 결합된 개념 체인을 생성합니다.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `luma_concepts` | LUMA_CONCEPTS | 선택된 모든 개념을 포함한 결합된 카메라 개념 체인 |
