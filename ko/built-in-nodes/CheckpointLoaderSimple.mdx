## 개요

확산 모델 체크포인트 파일을 로드하고 이를 세 가지 핵심 구성 요소로 분해합니다: 잡음 제거용 라텐트에 사용되는 메인 모델, CLIP 텍스트 인코더, VAE 이미지 인코더/디코더입니다. 이 노드는 `ComfyUI/models/checkpoints` 폴더에 있는 모든 모델 파일과 `extra_model_paths.yaml` 파일에 구성된 추가 경로를 자동으로 탐지합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `ckpt_name` | 문자열 | 예 | 체크포인트 폴더에 있는 모든 모델 파일 | 로드할 체크포인트(모델)의 이름입니다. 이후 이미지 생성에 사용될 AI 모델을 결정하는 체크포인트 모델 파일명을 선택하세요. |

**참고:** ComfyUI가 실행 중일 때 새로운 모델 파일이 추가되면, 브라우저를 새로 고침(Ctrl+R)해야 드롭다운 목록에 새 파일이 표시됩니다.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `MODEL` | 모델 | 잡음 제거용 라텐트에 사용되는 모델입니다. 이미지 생성에 사용되는 핵심 확산 모델입니다. |
| `CLIP` | CLIP | 텍스트 프롬프트를 인코딩하는 데 사용되는 CLIP 모델로, 텍스트 설명을 AI가 이해할 수 있는 정보로 변환합니다. |
| `VAE` | VAE | 이미지를 라텐트 공간과 상호 변환하는 데 사용되는 VAE 모델입니다. |
