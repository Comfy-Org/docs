Wan Image to Video 노드는 단일 입력 이미지와 텍스트 프롬프트를 사용해 비디오를 생성합니다. 제공된 이미지를 첫 번째 프레임으로 사용하며, 설명에 따라 비디오 시퀀스를 생성합니다. 해상도, 지속 시간, 오디오 및 기타 고급 설정 옵션도 제공됩니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `model` | COMBO | 예 | "wan2.5-i2v-preview"<br />"wan2.6-i2v" | 사용할 모델 (기본값: "wan2.6-i2v") |
| `image` | IMAGE | 예 | - | 비디오 생성을 위한 첫 번째 프레임 역할을 하는 입력 이미지입니다. 정확히 하나의 이미지가 필요합니다. |
| `prompt` | STRING | 예 | - | 요소와 시각적 특징을 설명하는 프롬프트입니다. 영어와 중국어를 지원하며 기본값은 빈 문자열입니다. |
| `negative_prompt` | STRING | 아니요 | - | 피해야 할 사항을 설명하는 부정적 프롬프트 (기본값: 빈 문자열). |
| `resolution` | COMBO | 아니요 | "480P"<br />"720P"<br />"1080P" | 비디오 해상도 품질 (기본값: "720P"). Wan 2.6 모델은 480P를 지원하지 않습니다. |
| `duration` | INT | 아니요 | 5~15 (단계: 5) | 생성된 비디오의 지속 시간(초). 15초 지속 시간은 Wan 2.6 모델에서만 지원됩니다 (기본값: 5). |
| `audio` | AUDIO | 아니요 | - | 오디오는 명확하고 큰 목소리여야 하며, 잡음이나 배경 음악이 없어야 합니다. 제공될 경우 오디오 지속 시간은 3.0초에서 29.0초 사이여야 합니다. |
| `seed` | INT | 아니요 | 0~2147483647 | 생성에 사용할 시드 (기본값: 0). |
| `generate_audio` | BOOLEAN | 아니요 | - | 오디오 입력이 제공되지 않을 경우 자동으로 오디오를 생성합니다 (기본값: False). |
| `prompt_extend` | BOOLEAN | 아니요 | - | AI 도움을 통해 프롬프트를 강화할지 여부 (기본값: True). |
| `watermark` | BOOLEAN | 아니요 | - | 결과물에 AI가 생성한 워터마크를 추가할지 여부 (기본값: False). |
| `shot_type` | COMBO | 아니요 | "single"<br />"multi" | 생성된 비디오의 샷 유형을 지정합니다. 즉, 비디오가 단일 연속 샷인지, 커트가 있는 여러 샷인지 여부입니다. 이 매개변수는 prompt_extend가 True일 때만 적용됩니다 (기본값: "single"). |

**제약 조건:**

- 비디오 생성에는 정확히 하나의 입력 이미지가 필요합니다.
- Wan 2.6 모델(`wan2.6-i2v`)은 480P 해상도를 지원하지 않습니다.
- 15초 지속 시간은 Wan 2.6 모델(`wan2.6-i2v`)에서만 지원됩니다.
- 오디오를 제공할 경우, 지속 시간은 3.0초에서 29.0초 사이여야 합니다.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|------------|-----------|--------|
| `output` | VIDEO | 입력 이미지와 프롬프트를 기반으로 생성된 비디오입니다. |
