SamplerCustom 노드는 다양한 애플리케이션을 위한 유연하고 맞춤형 샘플링 메커니즘을 제공하도록 설계되었습니다. 사용자는 각자의 특정 요구에 맞게 다양한 샘플링 전략을 선택하고 구성할 수 있어 샘플링 과정의 적응성과 효율성을 높일 수 있습니다.

## 입력

| 매개변수 | 데이터 타입 | 설명 |
|-----------|--------------|-------------|
| `model`   | `MODEL`      | 'model' 입력 유형은 샘플링에 사용될 모델을 지정하며, 샘플링 동작과 출력을 결정하는 데 중요한 역할을 합니다. |
| `add_noise` | `BOOLEAN`    | 'add_noise' 입력 유형은 샘플링 과정에 노이즈를 추가할지 여부를 지정하며, 생성된 샘플의 다양성과 특성을 조정합니다. |
| `noise_seed` | `INT`        | 'noise_seed' 입력 유형은 노이즈 생성을 위한 시드를 제공하며, 노이즈를 추가할 때 샘플링 과정의 재현성과 일관성을 보장합니다. |
| `cfg`     | `FLOAT`      | 'cfg' 입력 유형은 샘플링 과정의 설정을 정의하며, 샘플링 매개변수와 동작을 세밀하게 조정할 수 있게 해줍니다. |
| `positive` | `CONDITIONING` | 'positive' 입력 유형은 긍정적인 조건 정보를 나타내며, 샘플링 과정이 지정된 긍정적 속성과 일치하는 샘플을 생성하도록 유도합니다. |
| `negative` | `CONDITIONING` | 'negative' 입력 유형은 부정적인 조건 정보를 나타내며, 샘플링 과정이 지정된 부정적 속성을 갖는 샘플을 생성하지 않도록 방향을 조정합니다. |
| `sampler` | `SAMPLER`    | 'sampler' 입력 유형은 사용할 특정 샘플링 전략을 선택하며, 생성된 샘플의 성격과 품질에 직접적인 영향을 미칩니다. |
| `sigmas`  | `SIGMAS`     | 'sigmas' 입력 유형은 샘플링 과정에서 사용할 노이즈 수준을 정의하며, 샘플 공간의 탐색과 출력의 다양성에 영향을 줍니다. |
| `latent_image` | `LATENT` | 'latent_image' 입력 유형은 샘플링 과정의 초기 잠재 이미지를 제공하며, 샘플 생성의 출발점 역할을 합니다. |

## 출력

| 매개변수 | 데이터 타입 | 설명 |
|-----------|--------------|-------------|
| `output`  | `LATENT`     | 'output'는 샘플링 과정의 주요 결과로, 생성된 샘플들을 포함합니다. |
| `denoised_output` | `LATENT` | 'denoised_output'는 노이즈 제거 과정이 적용된 후의 샘플들을 나타내며, 생성된 샘플의 선명도와 품질을 향상시킬 수 있습니다. |
