HunyuanImageToVideo 노드는 Hunyuan 비디오 모델을 사용하여 이미지를 비디오 잠재 표현으로 변환합니다. 조건 입력과 선택적 시작 이미지를 받아 비디오 생성 모델에서 추가 처리할 수 있는 비디오 잠재값을 생성합니다. 이 노드는 시작 이미지가 비디오 생성 과정에 어떻게 영향을 미치는지 제어하는 다양한 가이던스 유형을 지원합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `positive` | CONDITIONING | 예 | - | 비디오 생성을 안내하는 양성 조건 입력 |
| `vae` | VAE | 예 | - | 이미지를 잠재 공간으로 인코딩하는 데 사용되는 VAE 모델 |
| `width` | INT | 예 | 16부터 MAX_RESOLUTION까지 | 출력 비디오의 너비(픽셀)(기본값: 848, 단계: 16) |
| `height` | INT | 예 | 16부터 MAX_RESOLUTION까지 | 출력 비디오의 높이(픽셀)(기본값: 480, 단계: 16) |
| `length` | INT | 예 | 1부터 MAX_RESOLUTION까지 | 출력 비디오의 프레임 수(기본값: 53, 단계: 4) |
| `batch_size` | INT | 예 | 1부터 4096까지 | 동시에 생성할 비디오 수(기본값: 1) |
| `guidance_type` | COMBO | 예 | "v1 (concat)"<br />"v2 (replace)"<br />"custom" | 시작 이미지를 비디오 생성에 통합하는 방법(기본값: "v1 (concat)") |
| `start_image` | IMAGE | 아니요 | - | 비디오 생성을 초기화하는 선택적 시작 이미지 |

**참고:** `start_image`가 제공되면 노드는 선택한 `guidance_type`에 따라 다른 가이던스 방식을 사용합니다:

- "v1 (concat)": 이미지 잠재값을 비디오 잠재값과 연결하고 마스크를 적용해 이미지를 비디오에 혼합합니다.
- "v2 (replace)": 초기 비디오 프레임을 이미지 잠재값으로 교체하고 노이즈 마스크를 적용합니다.
- "custom": 이미지를 가이던스용 참조 잠재값으로 사용합니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `positive` | CONDITIONING | start_image가 제공된 경우 이미지 가이던스가 적용된 수정된 양성 조건 입력 |
| `latent` | LATENT | 비디오 생성 모델에서 추가 처리할 준비가 된 비디오 잠재 표현 |
