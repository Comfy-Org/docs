이 노드는 Recraft API 서비스를 사용하여 이미지의 배경을 제거합니다. 입력 배치의 각 이미지를 처리하며, 투명한 배경을 가진 처리된 이미지와, 제거된 배경 영역을 나타내는 해당 알파 마스크를 함께 반환합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `image` | IMAGE | 예 | - | 배경 제거를 위해 처리할 입력 이미지(들) |

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `image` | IMAGE | 투명한 배경을 가진 처리된 이미지 |
| `mask` | MASK | 제거된 배경 영역을 나타내는 알파 채널 마스크 |
