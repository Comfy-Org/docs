> 이 문서는 AI로 생성되었습니다. 오류를 발견하거나 개선할 점이 있다면 언제든지 기여해 주세요! [GitHub에서 편집하기](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/Wan2VideoEditApi/en.md)

Wan2VideoEditApi 노드는 Wan 2.7 모델을 사용하여 텍스트 지침, 참조 이미지 또는 스타일 전송에 따라 비디오를 편집합니다. 입력 비디오를 처리하고 해상도, 지속 시간, 종횡비와 같은 지정된 매개변수에 따라 새 비디오를 생성합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `model` | COMBO | 예 | `"wan2.7-videoedit"` | 비디오 편집에 사용할 모델입니다. |
| `model.prompt` | STRING | 예 | - | 편집 지침 또는 스타일 전송 요구사항입니다. (기본값: 빈 문자열) |
| `model.resolution` | COMBO | 예 | `"720P"`<br />`"1080P"` | 출력 비디오의 해상도입니다. |
| `model.ratio` | COMBO | 예 | `"16:9"`<br />`"9:16"`<br />`"1:1"`<br />`"4:3"`<br />`"3:4"` | 출력 비디오의 종횡비입니다. 변경하지 않으면 입력 비디오의 종횡비를 근사하게 유지합니다. |
| `model.duration` | COMBO | 예 | `"auto"`<br />`"2"`<br />`"3"`<br />`"4"`<br />`"5"`<br />`"6"`<br />`"7"`<br />`"8"`<br />`"9"`<br />`"10"` | 출력 지속 시간(초). 'auto'는 입력 비디오 지속시간과 일치합니다. 특정 값은 비디오 시작 부분부터 잘립니다. (기본값: "auto") |
| `model.reference_images` | IMAGE | 선택 | - | 편집을 안내할 최대 4개의 참조 이미지 목록입니다. |
| `video` | VIDEO | 예 | - | 편집할 비디오입니다. |
| `seed` | INT | 선택 | 0~2147483647 | 생성에 사용할 시드입니다. (기본값: 0) |
| `audio_setting` | COMBO | 선택 | `"auto"`<br />`"origin"` | 'auto': 모델이 프롬프트에 따라 오디오를 다시 생성할지 결정합니다. 'origin': 입력 비디오의 원래 오디오를 유지합니다. (기본값: "auto") |
| `watermark` | BOOLEAN | 선택 | - | 결과에 인공지능으로 생성된 워터마크를 추가할지 여부입니다. (기본값: False) |

**제약 조건:**
*   `model.prompt`은 최소 1자 이상이어야 합니다.
*   입력 `video`의 지속 시간은 2초에서 10초 사이여야 합니다.
*   `model.reference_images` 입력에는 최대 4개의 이미지를 제공할 수 있습니다.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-----------|-----------|-------------|
| `output` | VIDEO | 모델이 생성한 편집된 비디오입니다. |
