# COMFYUI MultiGPU CFG 분할 노드

## 개요:

MultiGPU CFG 분할 노드는 동일한 시스템에 설치된 여러 개의 GPU를 사용해 확산 처리를 수행할 수 있게 해줍니다. 성능 결과는 워크플로우에 따라 다르지만, 일반적인 워크플로우에서는 최대 1.95배의 속도 향상이 측정되었습니다.

## 주요 사항:

다양한 종류의 GPU를 혼합해서 사용하는 것은 지원되지 않으며, 설치된 GPU는 반드시 동일해야 합니다(예: 2x 5090 또는 2x 5080 등).

ComfyUI는 시작 시 자동으로 여러 개의 GPU를 인식합니다.

## 지원되는 GPU:

Ampere+ 아키텍처를 갖춘 동일한 듀얼 GPU 구성(예: 2 x 3090 또는 2 x RTX6000 Pro)

## 지원되는 모델:

* LTX-2.3  
* WAN 2.2  
* FLUX.2 Klein - 기본 버전  
* Z-Image  
* Stable Diffusion 3.5 Large  
* Hunyuan Video  
* Qwen-Image-Edit-2511  
* Hunyuan-3D-v2.1  
* SDXL

## 입력

| 매개변수 | 데이터 타입 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `model` | MODEL | 예 | N/A | 샘플링 전에 MultiGPU CFG 분할을 위해 준비할 모델입니다. |
| `max_gpus` | INT | 예 | 최소: 1<br />단계: 1<br />기본값: 2 | 작업량을 분할하는 데 사용할 동일한 GPU의 최대 개수입니다. 시스템에 설치된 일치하는 GPU의 수로 설정하세요. |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `MODEL` | MODEL | MultiGPU CFG 분할을 위해 준비된 모델로, 가속화된 샘플링을 위한 상태입니다. |

## 노드:

![image1.png](/images/built-in-nodes/MultiGPU_WorkUnits/image1.png)  
max_gpus 필드는 시스템에 설치된 동일한 GPU의 최대 개수로 설정해야 합니다.

**노드 배치:** MultiGPU CFG 분할은 모델 로드 노드와 샘플링 노드 사이에 배치해야 합니다. 모델 로더 노드의 모델 출력에 다른 노드가 연결되어 있다면, MultiGPU CFG는 샘플링 노드 이전 체인의 마지막 노드가 되어야 합니다.

![image2.png](/images/built-in-nodes/MultiGPU_WorkUnits/image2.png)

**워크플로우 요구사항:** 이 노드는 CFG 단계에서 확산 워크플로우를 분할하여 작동하므로, 워크플로우의 CFG는 반드시 CFG=1보다 커야 합니다. CFG=1을 필요로 하는 디스틸러 워크플로우는 MultiGPU CFG 분할 노드를 사용해 여러 GPU에서 실행해도 성능 향상 효과를 볼 수 없습니다.
