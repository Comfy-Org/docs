Hunyuan Video 15 Latent Upscale With Model 노드는 잠재 이미지 표현의 해상도를 높입니다. 먼저 선택한 보간 방법을 사용해 잠재 샘플을 지정된 크기로 확대한 다음, 특수한 Hunyuan Video 1.5 업스케일링 모델을 사용해 확대된 결과를 정밀하게 다듬어 품질을 향상시킵니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `model` | LATENT_UPSCALE_MODEL | 예 | N/A | 확대된 샘플을 다듬는 데 사용되는 Hunyuan Video 1.5 잠재 업스케일링 모델입니다. |
| `samples` | LATENT | 예 | N/A | 업스케일링할 잠재 이미지 표현입니다. |
| `upscale_method` | COMBO | 아니요 | `"nearest-exact"`<br />`"bilinear"`<br />`"area"`<br />`"bicubic"`<br />`"bislerp"` | 초기 업스케일링 단계에 사용되는 보간 알고리즘입니다(기본값: `"bilinear"`). |
| `width` | INT | 아니요 | 0부터 16384까지 | 업스케일링된 잠재의 목표 너비, 픽셀 단위입니다. 값이 0이면 목표 높이와 원래 종횡비를 기반으로 너비를 자동 계산합니다. 최종 출력 너비는 16의 배수로 설정됩니다(기본값: 1280). |
| `height` | INT | 아니요 | 0부터 16384까지 | 업스케일링된 잠재의 목표 높이, 픽셀 단위입니다. 값이 0이면 목표 너비와 원래 종횡비를 기반으로 높이를 자동 계산합니다. 최종 출력 높이는 16의 배수로 설정됩니다(기본값: 720). |
| `crop` | COMBO | 아니요 | `"disabled"`<br />`"center"` | 업스케일링된 잠재가 목표 치수에 맞도록 어떻게 자르는지 결정합니다. |

**치수 관련 참고사항:** `width`와 `height`가 모두 0으로 설정되면 노드는 입력 `samples`를 그대로 반환합니다. 한쪽 치수가 0으로 설정된 경우, 다른 쪽 치수는 원래 종횡비를 유지하도록 계산됩니다. 최종 치수는 항상 최소 64픽셀 이상이어야 하며, 16의 배수로 나누어져야 합니다.
