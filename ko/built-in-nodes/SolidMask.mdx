SolidMask 노드는 전체 영역에 지정된 값으로 균일한 마스크를 생성합니다. 특정 크기와 강도의 마스크를 만들도록 설계되었으며, 다양한 이미지 처리 및 마스크 작업에 유용합니다.

## 입력

| 매개변수 | 데이터 타입 | 설명 |
|-----------|-------------|-------------|
| `value`   | FLOAT       | 마스크의 강도 값을 지정하며, 이는 마스크의 전반적인 모양과 이후 작업에서의 유용성에 영향을 줍니다. |
| `width`   | INT         | 생성되는 마스크의 너비를 결정하며, 크기와 종횡비에 직접적인 영향을 줍니다. |
| `height`  | INT         | 생성되는 마스크의 높이를 설정하며, 크기와 종횡비에 영향을 줍니다. |

## 출력

| 매개변수 | 데이터 타입 | 설명 |
|-----------|-------------|-------------|
| `mask`    | MASK        | 지정된 크기와 값을 가진 균일한 마스크를 출력합니다. |
