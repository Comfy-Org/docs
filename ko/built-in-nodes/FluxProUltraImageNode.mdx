> 이 문서는 AI를 통해 생성되었습니다. 오류를 발견하거나 개선할 점이 있다면 언제든지 기여해 주세요! [GitHub에서 편집하기](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/FluxProUltraImageNode/en.md)

프롬프트와 해상도를 기반으로 Flux Pro 1.1 Ultra API를 사용해 이미지를 생성합니다. 이 노드는 외부 서비스와 연결되어 텍스트 설명과 지정된 크기에 따라 이미지를 생성합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `prompt` | 문자열 | 예 | - | 이미지 생성을 위한 프롬프트 (기본값: 빈 문자열) |
| `prompt_upsampling` | 부울 | 아니요 | - | 프롬프트에 업샘플링을 수행할지 여부. 활성화 시 더 창의적인 생성을 위해 자동으로 프롬프트를 수정하지만 결과는 비결정적입니다 (같은 시드로도 정확히 동일한 결과가 나오지 않습니다). (기본값: False) |
| `seed` | 정수 | 아니요 | 0부터 18446744073709551615까지 | 노이즈 생성에 사용되는 난수 시드. (기본값: 0) |
| `aspect_ratio` | 문자열 | 아니요 | - | 이미지의 종횡비; 1:4에서 4:1 사이여야 합니다. (기본값: "16:9") |
| `raw` | 부울 | 아니요 | - | True일 경우, 보다 덜 처리된 자연스러운 이미지를 생성합니다. (기본값: False) |
| `image_prompt` | 이미지 | 아니요 | - | 생성을 안내할 수 있는 선택적 참조 이미지 |
| `image_prompt_strength` | 실수 | 아니요 | 0.0부터 1.0까지 | 프롬프트와 이미지 프롬프트 간의 블렌딩 정도. (기본값: 0.1) |

**참고:** `aspect_ratio` 매개변수는 반드시 1:4에서 4:1 사이여야 합니다. `image_prompt`가 제공되면 `image_prompt_strength`가 활성화되어 참조 이미지가 최종 출력에 얼마나 영향을 미치는지 제어합니다. `image_prompt`가 제공되지 않을 경우, `prompt` 매개변수가 비어 있지 않은지 확인됩니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `output_image` | 이미지 | Flux Pro 1.1 Ultra에서 생성된 이미지 |
