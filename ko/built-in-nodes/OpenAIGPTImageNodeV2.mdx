## 개요

이 노드는 OpenAI의 GPT 이미지 API를 사용하여 이미지를 생성합니다. 여러 모델을 지원하며, 편집을 위해 입력 이미지를 제공할 수 있고, 마스크를 사용해 이미지의 특정 부분만 수정하도록 지정할 수도 있습니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `prompt` | 문자열 | 예 | 없음 | GPT 이미지용 텍스트 프롬프트 (기본값: ""). |
| `model` | 콤보 | 예 | `"gpt-image-2"`<br />`"gpt-image-1.5"`<br />`"gpt-image-1"` | 사용할 OpenAI GPT 이미지 모델입니다. 모델을 선택하면 해당 모델에 특화된 추가 매개변수가 표시됩니다. |
| `model.size` | 콤보 | 예 | `"auto"`<br />`"1024x1024"`<br />`"1024x1536"`<br />`"1536x1024"`<br />`"2048x2048"`<br />`"2048x1152"`<br />`"1152x2048"`<br />`"3840x2160"`<br />`"2160x3840"`<br />`"사용자 정의"` | 이미지 크기입니다. '사용자 정의'를 선택하면 맞춤 너비와 높이를 사용할 수 있습니다 (기본값: "auto"). `gpt-image-2`에서만 사용 가능합니다. |
| `model.custom_width` | 정수 | 아니요 | 1024부터 3840까지 | `size`가 '사용자 정의'일 때만 사용됩니다. 반드시 16의 배수여야 합니다 (기본값: 1024). `gpt-image-2`에서만 사용 가능합니다. |
| `model.custom_height` | 정수 | 아니요 | 1024부터 3840까지 | `size`가 '사용자 정의'일 때만 사용됩니다. 반드시 16의 배수여야 합니다 (기본값: 1024). `gpt-image-2`에서만 사용 가능합니다. |
| `model.background` | 콤보 | 예 | `"auto"`<br />`"불투명"` | 배경 포함 또는 제외된 이미지를 반환합니다 (기본값: "auto"). `gpt-image-2`에서만 사용 가능합니다. |
| `model.quality` | 콤보 | 예 | `"표준"`<br />`"HD"` | 생성된 이미지의 품질입니다. `gpt-image-2`에서만 사용 가능합니다. |
| `model.images` | 이미지 | 아니요 | 없음 | 편집을 위한 입력 이미지입니다. `gpt-image-2`에서만 사용 가능합니다. |
| `model.mask` | 마스크 | 아니요 | 없음 | 입력 이미지의 어느 부분을 편집할지 지정하는 마스크입니다. `gpt-image-2`에서만 사용 가능합니다. |
| `n` | 정수 | 예 | 1부터 8까지 | 몇 장의 이미지를 생성할지 (기본값: 1). |
| `seed` | 정수 | 예 | 0부터 2147483647까지 | 재현성을 위한 시드 값 (기본값: 0). 참고: 백엔드에서는 아직 구현되지 않았습니다. |

**매개변수 제약 및 한계:**

- `gpt-image-2`를 사용하고 `model.size`를 '사용자 정의'로 설정할 경우, `custom_width`와 `custom_height`는 반드시 16의 배수여야 하며, 최대 변 길이는 3840 이하이고, 종횡비는 3:1을 초과해서는 안 되며, 총 픽셀 수는 655,360에서 8,294,400 사이여야 합니다.
- `mask`를 제공할 경우, 입력 이미지(`model.images`)가 반드시 필요합니다. 입력 이미지 없이 마스크를 사용할 수 없습니다.
- 여러 입력 이미지와 함께 마스크를 사용할 수 없습니다.
- 마스크를 제공할 경우, 마스크의 크기는 입력 이미지의 크기와 일치해야 합니다.
- `seed` 매개변수는 현재 작동하지 않습니다.
