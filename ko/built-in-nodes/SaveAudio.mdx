> 이 문서는 AI로 생성되었습니다. 오류를 발견하거나 개선할 점이 있으면 언제든지 기여해 주세요! [GitHub에서 편집하기](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/SaveAudio/en.md)

SaveAudio 노드는 오디오 데이터를 FLAC 형식의 파일로 저장합니다. 오디오 입력을 받아 지정된 출력 디렉터리에 주어진 파일명 접두사와 함께 씁니다. 이 노드는 자동으로 파일명을 처리하며, 오디오가 이후에 사용할 수 있도록 올바르게 저장되도록 보장합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `audio` | AUDIO | 예 | - | 저장할 오디오 데이터 |
| `filename_prefix` | STRING | 아니요 | - | 출력 파일명의 접두사 (기본값: "audio/ComfyUI") |

*참고: `prompt` 및 `extra_pnginfo` 매개변수는 숨겨져 있으며 시스템에서 자동으로 처리됩니다.*

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| *없음* | - | 이 노드는 출력 데이터를 반환하지 않으며 오디오 파일을 출력 디렉터리에 저장합니다 |
