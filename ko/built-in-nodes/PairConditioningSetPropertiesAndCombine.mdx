PairConditioningSetPropertiesAndCombine 노드는 기존의 양성 및 음성 조건 입력에 새로운 조건 데이터를 적용하여 조건 쌍을 수정하고 결합합니다. 이를 통해 적용된 조건의 강도를 조정하고 조건 적용 영역을 제어할 수 있습니다. 이 노드는 여러 조건 소스를 함께 블렌딩해야 하는 고급 조건 조작 워크플로우에서 특히 유용합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `positive` | CONDITIONING | 예 | - | 원래의 양성 조건 입력 |
| `negative` | CONDITIONING | 예 | - | 원래의 음성 조건 입력 |
| `positive_NEW` | CONDITIONING | 예 | - | 적용할 새로운 양성 조건 |
| `negative_NEW` | CONDITIONING | 예 | - | 적용할 새로운 음성 조건 |
| `strength` | FLOAT | 예 | 0.0부터 10.0까지 | 새로운 조건을 적용하는 강도 요소 (기본값: 1.0) |
| `set_cond_area` | STRING | 예 | "default"<br />"mask bounds" | 조건 적용 영역을 제어하는 방법 (기본값: "default") |
| `mask` | MASK | 선택 | - | 조건 적용 영역을 제한하는 옵션 마스크 |
| `hooks` | HOOKS | 선택 | - | 고급 제어를 위한 옵션 후크 그룹 |
| `timesteps` | TIMESTEPS_RANGE | 선택 | - | 옵션 타임스텝 범위 지정 |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `positive` | CONDITIONING | 결합된 양성 조건 출력 |
| `negative` | CONDITIONING | 결합된 음성 조건 출력 |
