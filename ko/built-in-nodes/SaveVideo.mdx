> 이 문서는 AI로 생성되었습니다. 오류를 발견하거나 개선할 점이 있다면 언제든지 기여해 주세요! [GitHub에서 편집하기](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/SaveVideo/en.md)

SaveVideo 노드는 입력 동영상 콘텐츠를 ComfyUI 출력 디렉터리에 저장합니다. 저장할 파일의 파일명 접두사, 동영상 형식 및 코덱을 지정할 수 있습니다. 이 노드는 자동으로 카운터 증분을 이용한 파일명 처리를 지원하며, 저장된 동영상에 워크플로우 메타데이터를 포함시킬 수도 있습니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `video` | VIDEO | 예 | - | 저장할 동영상입니다. |
| `filename_prefix` | STRING | 아니요 | - | 저장할 파일의 접두사입니다. 여기에는 `%date:yyyy-MM-dd%`나 `%Empty Latent Image.width%`와 같은 포맷팅 정보를 포함하여 노드의 값들을 삽입할 수 있습니다(기본값: "video/ComfyUI"). |
| `format` | STRING | 아니요 | `"auto"`<br />`"mp4"`<br />`"webm"`<br />`"mkv"`<br />`"gif"` | 동영상을 저장할 형식입니다(기본값: "auto"). |
| `codec` | STRING | 아니요 | `"auto"`<br />`"h264"`<br />`"h265"`<br />`"vp9"`<br />`"av1"`<br />`"prores"` | 동영상에 사용할 코덱입니다(기본값: "auto"). |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `ui` | PREVIEW_VIDEO | 저장된 동영상 파일의 미리보기로, UI에 표시하기 위한 파일 경로 및 하위 폴더 정보를 포함합니다. |
