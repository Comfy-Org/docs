이 노드는 "ace step 1.5" 프로세스에서 사용할 기준 오디오 음색을 설정합니다. 컨디셔닝 입력과 선택적으로 오디오의 잠재 표현을 받아, 해당 잠재 데이터를 컨디셔닝에 첨부하여 워크플로우의 후속 노드에서 사용할 수 있도록 합니다.

## 입력

| 매개변수 | 설명 | 데이터 타입 | 필수 | 범위 |
| --- | --- | --- | --- | --- |
| `컨디셔닝` | 기준 오디오 정보가 첨부될 컨디셔닝 데이터입니다. | CONDITIONING | 예 |  |
| `latent` | 기준 오디오의 선택적 잠재 표현입니다. 제공된 경우 해당 샘플이 컨디셔닝에 추가됩니다. | LATENT | 아니요 |  |

## 출력

| 출력 이름 | 설명 | 데이터 타입 |
| --- | --- | --- |
| `컨디셔닝` | 선택적 `latent` 입력이 제공된 경우 기준 오디오 음색 잠재 정보를 포함하도록 수정된 컨디셔닝 데이터입니다. | CONDITIONING |
