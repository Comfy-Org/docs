SD_4XUpscale_Conditioning 노드는 확산 모델을 사용해 이미지를 업스케일링할 때 필요한 컨디셔닝 데이터를 준비합니다. 입력 이미지와 컨디셔닝 데이터를 받아 스케일링과 노이즈 증강을 적용하여 업스케일링 과정을 안내하는 수정된 컨디셔닝을 생성합니다. 이 노드는 업스케일링된 차원에 대한 잠재적 표현과 함께 양성 및 음성 컨디셔닝을 출력합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `images` | IMAGE | 예 | - | 업스케일링할 입력 이미지 |
| `positive` | CONDITIONING | 예 | - | 원하는 콘텐츠를 향해 생성을 유도하는 양성 컨디셔닝 데이터 |
| `negative` | CONDITIONING | 예 | - | 원치 않는 콘텐츠를 피하도록 생성을 조정하는 음성 컨디셔닝 데이터 |
| `scale_ratio` | FLOAT | 아니요 | 0.0 - 10.0 | 입력 이미지에 적용되는 스케일링 요인 (기본값: 4.0) |
| `noise_augmentation` | FLOAT | 아니요 | 0.0 - 1.0 | 업스케일링 과정 중 추가할 노이즈의 양 (기본값: 0.0) |

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-----------|-----------|-------------|
| `positive` | CONDITIONING | 업스케일링 정보가 적용된 수정된 양성 컨디셔닝 |
| `negative` | CONDITIONING | 업스케일링 정보가 적용된 수정된 음성 컨디셔닝 |
| `latent` | LATENT | 업스케일링된 차원과 일치하는 빈 잠재적 표현 |
