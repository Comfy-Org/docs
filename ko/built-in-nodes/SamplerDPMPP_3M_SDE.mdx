SamplerDPMPP_3M_SDE 노드는 샘플링 과정에서 사용할 DPM++ 3M SDE 샘플러를 생성합니다. 이 샘플러는 3차 다단계 확률 미분 방정식 방법을 사용하며, 노이즈 매개변수를 구성 가능하게 설계되었습니다. 이 노드를 통해 노이즈 계산을 GPU 또는 CPU에서 수행할지 선택할 수 있습니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `eta` | FLOAT | 예 | 0.0 - 100.0 | 샘플링 과정의 확률성을 제어합니다(기본값: 1.0) |
| `s_noise` | FLOAT | 예 | 0.0 - 100.0 | 샘플링 중 추가되는 노이즈의 양을 제어합니다(기본값: 1.0) |
| `noise_device` | COMBO | 예 | "gpu"<br />"cpu" | 노이즈 계산에 사용할 장치를 선택합니다. GPU 또는 CPU 중 하나를 선택하세요(기본값: "gpu") |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `sampler` | SAMPLER | 샘플링 워크플로우에서 사용할 구성된 샘플러 객체를 반환합니다 |
