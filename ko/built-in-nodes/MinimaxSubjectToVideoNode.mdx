> 이 문서는 AI로 생성되었습니다. 오류를 발견하거나 개선할 점이 있다면 언제든지 기여해 주세요! [GitHub에서 편집하기](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/MinimaxSubjectToVideoNode/en.md)

MiniMax의 API를 사용하여 피사체 이미지와 텍스트 프롬프트를 기반으로 동영상을 동기적으로 생성합니다. 이 노드는 피사체의 이미지와 설명을 받아 프롬프트에 따라 해당 피사체를 애니메이션하거나 강조한 동영상을 만듭니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `subject` | IMAGE | 예 | - | 동영상 생성에 참고할 피사체의 이미지 |
| `prompt_text` | STRING | 예 | - | 동영상 생성을 안내하는 텍스트 프롬프트 (기본값: 빈 문자열) |
| `model` | COMBO | 아니요 | `"S2V-01"` | 동영상 생성에 사용할 모델 (기본값: "S2V-01") |
| `seed` | INT | 아니요 | 0부터 18446744073709551615까지 | 노이즈 생성에 사용되는 난수 시드 (기본값: 0) |

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `output` | VIDEO | 입력된 피사체 이미지와 프롬프트를 기반으로 생성된 동영상 |
