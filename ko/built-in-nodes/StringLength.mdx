StringLength 노드는 텍스트 문자열의 글자 수를 계산합니다. 임의의 텍스트 입력을 받아 공백과 구두점까지 포함한 총 글자 수를 반환합니다. 이는 텍스트 길이를 측정하거나 문자열 크기 요구사항을 검증하는 데 유용합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `string` | STRING | 예 | N/A | 길이를 측정할 텍스트 문자열입니다. 다중 줄 입력도 지원됩니다. |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `length` | INT | 입력된 문자열의 공백과 특수 문자를 포함한 총 글자 수입니다. |
