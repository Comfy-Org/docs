> 이 문서는 AI로 생성되었습니다. 오류를 발견하거나 개선할 점이 있다면 언제든지 기여해 주세요! [GitHub에서 편집하기](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/Veo3VideoGenerationNode/en.md)

Google의 Veo 3 API를 사용하여 텍스트 프롬프트로부터 동영상을 생성합니다. 이 노드는 빠른 버전과 라이트 버전을 포함한 여러 Veo 3 모델을 지원하며, 동영상 해상도, 지속 시간 및 오디오 생성을 지정할 수 있습니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `prompt` | 문자열 | 예 | - | 동영상의 텍스트 설명 (기본값: "") |
| `aspect_ratio` | 콤보 | 예 | "16:9"<br />"9:16" | 출력 동영상의 종횡비 (기본값: "16:9") |
| `resolution` | 콤보 | 아니요 | "720p"<br />"1080p"<br />"4k" | 출력 동영상 해상도. 4K는 veo-3.1-lite 및 veo-3.0 모델에서는 사용할 수 없습니다. (기본값: "720p") |
| `negative_prompt` | 문자열 | 아니요 | - | 동영상에서 피하고자 하는 내용을 안내하는 부정적 텍스트 프롬프트 (기본값: "") |
| `duration_seconds` | 정수 | 아니요 | 4~8 | 출력 동영상의 지속 시간(초), 2단위로 설정 (기본값: 8) |
| `enhance_prompt` | 불리언 | 아니요 | - | 이 매개변수는 더 이상 사용되지 않으며 무시됩니다. (기본값: True) |
| `person_generation` | 콤보 | 아니요 | "ALLOW"<br />"BLOCK" | 동영상에 사람 생성을 허용할지 여부 (기본값: "ALLOW") |
| `seed` | 정수 | 아니요 | 0~4294967295 | 동영상 생성을 위한 시드 값 (랜덤일 경우 0) (기본값: 0) |
| `image` | 이미지 | 아니요 | - | 동영상 생성을 안내할 수 있는 선택적 참조 이미지 |
| `model` | 콤보 | 아니요 | "veo-3.1-generate"<br />"veo-3.1-fast-generate"<br />"veo-3.1-lite"<br />"veo-3.0-generate-001"<br />"veo-3.0-fast-generate-001" | 동영상 생성에 사용할 Veo 3 모델 (기본값: "veo-3.0-generate-001") |
| `generate_audio` | 불리언 | 아니요 | - | 동영상에 오디오를 생성할지 여부. 모든 Veo 3 모델에서 지원됩니다. (기본값: False) |

**참고:** `enhance_prompt` 매개변수는 더 이상 사용되지 않으며 그 값은 무시됩니다. 노드는 항상 내부적으로 프롬프트를 강화합니다. 또한 `resolution` 매개변수는 veo-3.1 모델을 사용할 때만 적용되며, veo-3.0 모델에서는 무시됩니다. 만약 veo-3.1-lite 또는 veo-3.0 모델로 "4k" 해상도를 선택하면 노드에서 오류가 발생합니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `output` | 동영상 | 생성된 동영상 파일 |
