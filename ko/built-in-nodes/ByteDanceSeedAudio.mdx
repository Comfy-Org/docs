ByteDance Seed Audio 1.0을 사용하여 단일 프롬프트로 음성, 음악, 음향 효과 및 다중 화자 대화를 생성합니다. 프롬프트에 음성, 감정, 분위기, 배경 음악 및 음향 효과를 설명하고, 말할 대사를 포함하세요. 선택적으로 내장된 사전 설정 음성을 선택하거나, 최대 3개의 참조 클립(프롬프트에 @Audio1-3으로 태그 지정)에서 음성을 복제하거나, 캐릭터 이미지에서 음성을 도출할 수 있습니다. 실행당 최대 2분의 오디오를 생성합니다.

## 입력

| 매개변수 | 설명 | 데이터 타입 | 필수 | 범위 |
|-----------|-------------|-----------|----------|-------|
| `text_prompt` | 음성, 감정, 속도, 분위기, 배경 음악 및 음향 효과를 설명하고, 말할 대사를 포함하세요(대화의 경우 캐릭터 이름을 인라인으로 지정). '오디오 참조' 모드에서는 연결된 클립을 @Audio1, @Audio2, @Audio3 순서로 참조합니다. 최대 3000자. | STRING | 예 | 1~3000자 |
| `reference_mode` | 음성을 조건화하는 방법: '텍스트 전용'(프롬프트에 모든 내용 설명), '오디오 참조'(최대 3개 음성 복제, @Audio1-3 태그 지정), '이미지 참조'(하나의 캐릭터 이미지에서 음성 도출), 또는 '사전 설정 음성'(프롬프트를 읽는 내장 명명된 음성 선택). | COMBO | 예 | `"text only"`<br />`"audio reference"`<br />`"image reference"`<br />`"preset voice"` |
| `reference_audio_1` | 음성 복제를 위한 참조 클립으로, 프롬프트에 @Audio1로 태그 지정됩니다. 최대 30초. | AUDIO | 아니요 | 최대 30초 |
| `reference_audio_2` | 프롬프트에 @Audio2로 태그 지정된 참조 클립입니다. 최대 30초. | AUDIO | 아니요 | 최대 30초 |
| `reference_audio_3` | 프롬프트에 @Audio3로 태그 지정된 참조 클립입니다. 최대 30초. | AUDIO | 아니요 | 최대 30초 |
| `reference_image` | 단일 캐릭터 이미지; 모델이 이 이미지에서 음성을 도출합니다. 참조 오디오와 함께 사용할 수 없습니다. | IMAGE | 아니요 | 단일 이미지 |
| `preset_voice` | 프롬프트를 읽는 내장 TTS 2.0 음성입니다. 참조 클립이 필요하지 않으며, 이 모드에서는 @AudioN 태그가 사용되지 않습니다. | COMBO | 아니요 | 여러 사전 설정 음성 옵션 사용 가능 |
| `sample_rate` | 출력 샘플 속도(Hz)입니다. (기본값: "24000") | COMBO | 예 | `"8000"`<br />`"16000"`<br />`"24000"`<br />`"32000"`<br />`"44100"`<br />`"48000"` |
| `speech_rate` | 말하기 속도입니다. 0 = 보통, 100 = 2.0배, -50 = 0.5배. (기본값: 0) | INT | 예 | -50~100 |
| `loudness_rate` | 음량입니다. 0 = 보통, 100 = 2.0배, -50 = 0.5배. (기본값: 0) | INT | 예 | -50~100 |
| `pitch_rate` | 반음 단위의 피치 이동(-12~12)입니다. (기본값: 0) | INT | 예 | -12~12 |
| `seed` | 시드는 노드 재실행 여부를 제어합니다. 시드와 관계없이 결과는 비결정적입니다. (기본값: 42) | INT | 예 | 0~2147483647 |

**매개변수 제약 조건 및 종속성:**

- `reference_mode` 선택에 따라 필요한 추가 매개변수가 결정됩니다:
  - **"text only"**: 참조 입력이 필요하지 않습니다. 프롬프트에 @AudioN 태그가 포함되어서는 안 됩니다.
  - **"audio reference"**: `reference_audio_1`, `reference_audio_2` 또는 `reference_audio_3` 중 하나 이상이 연결되어야 합니다. 참조 클립은 순서대로 연결되어야 하며, 중간에 빈 곳이 없어야 합니다(예: 두 개의 클립을 사용하는 경우 `reference_audio_1`과 `reference_audio_2`를 연결). 각 참조 오디오 클립은 최대 30초로 제한됩니다. 프롬프트는 연결된 클립을 @Audio1, @Audio2, @Audio3 태그를 사용하여 순서대로 참조해야 합니다.
  - **"image reference"**: `reference_image`가 연결되어야 합니다. 참조 오디오 입력과 함께 사용할 수 없습니다. 프롬프트에 @AudioN 태그가 포함되어서는 안 됩니다.
  - **"preset voice"**: `preset_voice`가 선택되어야 합니다. 참조 클립이 필요하지 않습니다. 프롬프트에 @Audio1을 제외한 @AudioN 태그가 포함되어서는 안 됩니다.

## 출력

| 출력 이름 | 설명 | 데이터 타입 |
|-------------|-------------|-----------|
| `audio` | 오디오 신호로 생성된 오디오 출력입니다. | AUDIO |
