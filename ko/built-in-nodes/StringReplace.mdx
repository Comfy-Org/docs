StringReplace 노드는 입력 문자열에서 텍스트 교체 작업을 수행합니다. 입력 텍스트 내에서 지정된 하위 문자열을 검색한 뒤, 모든 일치하는 항목을 다른 하위 문자열로 교체합니다. 이 노드는 교체가 적용된 수정된 문자열을 반환합니다.

## 입력값

| 매개변수 | 데이터 타입 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `string` | STRING | 예 | - | 교체가 이루어질 입력 텍스트 문자열 |
| `find` | STRING | 예 | - | 입력 텍스트 내에서 검색할 하위 문자열 |
| `replace` | STRING | 예 | - | 찾은 모든 일치 항목을 대체할 교체 텍스트 |

## 출력값

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `output` | STRING | find 텍스트의 모든 일치 항목이 replace 텍스트로 교체된 수정된 문자열 |
