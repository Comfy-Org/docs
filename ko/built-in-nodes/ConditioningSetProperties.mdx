ConditioningSetProperties 노드는 강도, 영역 설정을 조정하고 선택적 마스크, 후크 또는 타임스텝 범위를 적용하여 조건부 데이터의 속성을 수정합니다. 이미지 생성 과정에서 조건부 데이터의 적용 방식을 제어하려면 특정 매개변수를 설정하여 조건부 데이터가 어떻게 작용하는지 정밀하게 조정할 수 있습니다.

## 입력

| 매개변수 | 데이터 유형 | 입력 유형 | 기본값 | 범위 | 설명 |
|-----------|-----------|------------|---------|-------|-------------|
| `cond_NEW` | CONDITIONING | 필수 | - | - | 수정할 조건부 데이터 |
| `strength` | FLOAT | 필수 | 1.0 | 0.0 - 10.0 (단계: 0.01) | 조건부 효과의 강도를 제어합니다 |
| `set_cond_area` | STRING | 필수 | default | ["default", "mask bounds"] | 조건부 영역 적용 방식을 결정합니다. 표준 동작을 위해 "default"를 선택하거나, 마스크 영역으로 제한하려면 "mask bounds"를 선택하세요 |
| `mask` | MASK | 선택적 | - | - | 조건부 적용 영역을 제한하는 선택적 마스크 |
| `hooks` | HOOKS | 선택적 | - | - | 맞춤 처리를 위한 선택적 후크 함수 |
| `timesteps` | TIMESTEPS_RANGE | 선택적 | - | - | 조건부가 활성화되는 타임스텝 범위를 제한합니다 |

**참고:** `mask`가 제공될 경우, `set_cond_area` 매개변수를 "mask bounds"로 설정하면 조건부 적용이 마스크 영역으로만 제한됩니다. `hooks` 매개변수를 통해 후크 함수를 이용한 맞춤 처리가 가능하며, `timesteps`는 생성 과정 중 특정 타임스텝 범위로 조건부 효과를 제한합니다.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `CONDITIONING` | CONDITIONING | 속성이 업데이트된 수정된 조건부 데이터 |
