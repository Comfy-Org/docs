이 노드는 지정된 VAE 모델을 사용하여 이미지를 잠재 공간 표현으로 인코딩하도록 설계되었습니다. 인코딩 프로세스의 복잡성을 추상화하여 이미지를 잠재 표현으로 변환하는 간단한 방법을 제공합니다.

## 입력

| 매개변수 | 설명 | 데이터 타입 |
| --- | --- | --- |
| `픽셀 이미지` | 'pixels' 매개변수는 잠재 공간으로 인코딩할 이미지 데이터를 나타냅니다. 인코딩 프로세스의 직접적인 입력으로 작용하여 출력 잠재 표현을 결정하는 데 중요한 역할을 합니다. | `IMAGE` |
| `vae` | 'vae' 매개변수는 이미지 데이터를 잠재 공간으로 인코딩하는 데 사용할 변분 오토인코더(VAE) 모델을 지정합니다. 인코딩 메커니즘과 생성된 잠재 표현의 특성을 정의하는 데 필수적입니다. | VAE |

## 출력

| 매개변수 | 설명 | 데이터 타입 |
| --- | --- | --- |
| `latent` | 출력은 입력 이미지의 잠재 공간 표현으로, 압축된 형태로 필수 특징을 캡슐화합니다. | `LATENT` |
