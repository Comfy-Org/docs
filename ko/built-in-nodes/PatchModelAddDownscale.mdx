PatchModelAddDownscale 노드는 모델의 특정 블록에 다운스케일링과 업스케일링 작업을 적용하여 Kohya 딥 슈링크 기능을 구현합니다. 처리 과정 중에 중간 피처의 해상도를 줄였다가 원래 크기로 복원함으로써 품질을 유지하면서 성능을 향상시킬 수 있습니다. 이 노드를 통해 모델 실행 과정에서 이러한 스케일링 작업이 언제 어떻게 이루어지는지 정밀하게 제어할 수 있습니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `model` | MODEL | 예 | - | 다운스케일 패치를 적용할 모델 |
| `block_number` | INT | 아니요 | 1-32 | 다운스케일링을 적용할 특정 블록 번호 (기본값: 3) |
| `downscale_factor` | FLOAT | 아니요 | 0.1-9.0 | 피처를 다운스케일링하는 요인 (기본값: 2.0) |
| `start_percent` | FLOAT | 아니요 | 0.0-1.0 | 노이즈 제거 과정에서 다운스케일링이 시작되는 지점 (기본값: 0.0) |
| `end_percent` | FLOAT | 아니요 | 0.0-1.0 | 노이즈 제거 과정에서 다운스케일링이 멈추는 지점 (기본값: 0.35) |
| `downscale_after_skip` | BOOLEAN | 아니요 | - | 스킵 커넥션 이후에 다운스케일링을 적용할지 여부 (기본값: True) |
| `downscale_method` | COMBO | 아니요 | "bicubic"<br />"nearest-exact"<br />"bilinear"<br />"area"<br />"bislerp" | 다운스케일링 작업에 사용되는 보간 방법 |
| `upscale_method` | COMBO | 아니요 | "bicubic"<br />"nearest-exact"<br />"bilinear"<br />"area"<br />"bislerp" | 업스케일링 작업에 사용되는 보간 방법 |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `model` | MODEL | 다운스케일 패치가 적용된 수정된 모델 |
