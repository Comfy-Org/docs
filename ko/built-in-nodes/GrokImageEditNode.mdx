> 이 문서는 AI로 생성되었습니다. 오류를 발견하거나 개선할 점이 있다면 언제든지 기여해 주세요! [GitHub에서 편집하기](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/GrokImageEditNode/en.md)

Grok Image Edit 노드는 텍스트 프롬프트를 기반으로 기존 이미지를 수정합니다. Grok API를 사용해 입력된 이미지의 변형 버전 중 하나 이상을 생성하며, 이 과정은 사용자의 설명에 따라 이루어집니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `model` | COMBO | 예 | `"grok-imagine-image-quality"`<br />`"grok-imagine-image-pro"`<br />`"grok-imagine-image"`<br />`"grok-imagine-image-beta"` | 이미지 편집에 사용할 특정 AI 모델입니다. |
| `image` | IMAGE | 예 | | 편집할 입력 이미지입니다. 최대 3개의 입력 이미지를 지원하며, "pro" 모델의 경우 1개만 지원합니다. |
| `prompt` | STRING | 예 | | 편집된 이미지를 생성하는 데 사용되는 텍스트 프롬프트입니다. 공백을 제거한 후 최소 1자 이상이어야 합니다. |
| `resolution` | COMBO | 예 | `"1K"`<br />`"2K"` | 출력 이미지의 해상도입니다. |
| `number_of_images` | INT | 아니요 | 1~10 | 생성할 편집된 이미지의 수입니다(기본값: 1). |
| `seed` | INT | 아니요 | 0~2147483647 | 노드가 다시 실행될지 여부를 결정하는 시드입니다; 실제 결과는 시드와 관계없이 비결정적입니다(기본값: 0). |
| `aspect_ratio` | COMBO | 아니요 | `"auto"`<br />`"1:1"`<br />`"2:3"`<br />`"3:2"`<br />`"3:4"`<br />`"4:3"`<br />`"9:16"`<br />`"16:9"`<br />`"9:19.5"`<br />`"19.5:9"`<br />`"9:20"`<br />`"20:9"`<br />`"1:2"`<br />`"2:1"` | 출력 이미지의 종횡비입니다. 이미지 입력에 여러 이미지를 연결한 경우에만 설정 가능합니다. "auto"로 설정하면 종횡비가 자동으로 결정됩니다(기본값: "auto"). |

**중요한 제약사항:**
- `image` 입력은 최대 3개의 이미지를 지원하며, `grok-imagine-image-pro` 모델을 사용할 때는 1개의 입력 이미지만 지원합니다.
- `aspect_ratio` 매개변수는 이미지 입력에 여러 이미지를 연결한 경우에만 맞춤값("auto"가 아닌)으로 설정할 수 있습니다. 단일 입력 이미지에 맞춤 종횡비를 설정하면 오류가 발생합니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `output` | IMAGE | 노드가 생성한 편집된 이미지입니다. `number_of_images`가 1보다 크면 출력물이 배치로 연결됩니다. |
