## 입력

| 매개변수 | 데이터 유형 | 설명 |
|----------------|-----------|-------------|
| 조건부 | CONDITIONING | 입력 조건부 데이터, 일반적으로 이전 인코딩 또는 처리 단계에서 가져온 것 |
| 가이던스 | FLOAT | 이미지 생성 시 텍스트 프롬프트의 영향력을 제어하며, 조정 가능한 범위는 0.0부터 100.0까지 |

## 출력

| 매개변수 | 데이터 유형 | 설명 |
|----------------|-----------|-------------|
| CONDITIONING | CONDITIONING | 업데이트된 조건부 데이터로, 새로운 가이던스 값을 포함합니다 |
