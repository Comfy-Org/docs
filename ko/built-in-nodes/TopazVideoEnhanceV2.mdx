# Topaz Video Enhance V2

**Topaz Video Enhance V2** 노드를 사용하면 Topaz Labs의 AI 모델을 활용해 비디오를 업스케일하고 향상시킬 수 있습니다. 해상도를 높이고 보간을 통해 프레임 속도를 조정하며, 창의적 또는 사실적인 향상을 적용해 비디오 영상에 새로운 생명을 불어넣을 수 있습니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `video` | VIDEO | 예 | - | 처리할 입력 비디오입니다. MP4 컨테이너 형식이어야 합니다. |
| `upscaler_model` | COMBO | 예 | `"Astra 2"`<br />`"Starlight (Astra) Fast"`<br />`"Starlight (Astra) Creative"`<br />`"Starlight Precise 2.5"`<br />`"Disabled"` | 비디오를 업스케일링하는 데 사용되는 AI 모델입니다. "Disabled"를 선택하면 업스케일링이 적용되지 않습니다. |
| `upscaler_model.upscaler_resolution` | COMBO | 조건부 | `"FullHD (1080p)"`<br />`"4K (2160p)"` | 업스케일러의 목표 출력 해상도입니다. 업스케일러 모델을 선택한 경우(“Disabled”가 아님) 이 매개변수가 필요합니다. |
| `upscaler_model.creativity` | FLOAT / COMBO | 조건부 | Astra 2: 0.0부터 1.0까지(단계 0.1)<br />Starlight Creative: `"low"`<br />`"middle"`<br />`"high"` | 업스케일링의 창의성 강도입니다. "Astra 2"와 "Starlight (Astra) Creative" 모델에서만 사용 가능합니다. Astra 2의 경우 슬라이더이며 기본값은 0.5입니다. Starlight Creative의 경우 콤보형으로 기본값은 "low"입니다. |
| `upscaler_model.prompt` | STRING | 아니요 | - | 선택적 설명용(지시용이 아닌) 장면 프롬프트입니다. "Astra 2" 모델에서만 사용 가능합니다. 설정 시 최대 450개의 입력 프레임(~15초 @ 30fps)으로 제한됩니다. 기본값은 비어 있습니다. |
| `upscaler_model.sharp` | FLOAT | 아니요 | 0.0부터 1.0까지(단계 0.01) | 사전 향상된 선명도: 0.0=가우시안 블러, 0.5=통과(기본), 1.0=USM 샤프닝. "Astra 2" 모델에서만 사용 가능합니다. 기본값은 0.5입니다. |
| `upscaler_model.realism` | FLOAT | 아니요 | 0.0부터 1.0까지(단계 0.01) | 출력을 사진적 사실성에 가깝게 만듭니다. 모델 기본값인 0으로 두세요. "Astra 2" 모델에서만 사용 가능합니다. 기본값은 0.0입니다. |
| `interpolation_model` | COMBO | 예 | `"Disabled"`<br />`"apo-8"` | 프레임 보간에 사용되는 AI 모델입니다. "Disabled"를 선택하면 보간이 적용되지 않습니다. |
| `interpolation_model.interpolation_frame_rate` | INT | 조건부 | 15부터 240까지 | 출력 프레임 속도입니다. 보간 모델이 "apo-8"인 경우 필수입니다. 기본값은 60입니다. |
| `interpolation_model.interpolation_slowmo` | INT | 아니요 | 1부터 16까지 | 입력 비디오에 적용되는 슬로모션 요소입니다. 예를 들어 2를 선택하면 출력이 두 배 느려지고 지속시간이 두 배 늘어납니다. 기본값은 1입니다. |
| `interpolation_model.interpolation_duplicate` | BOOLEAN | 아니요 | True/False | 입력을 분석해 중복 프레임을 찾아 제거합니다. 기본값은 False입니다. |
| `interpolation_model.interpolation_duplicate_threshold` | FLOAT | 아니요 | 0.001부터 0.1까지(단계 0.001) | 중복 프레임 감지 민감도입니다. 기본값은 0.01입니다. |
| `dynamic_compression_level` | COMBO | 아니요 | `"Low"`<br />`"Mid"`<br />`"High"` | 비디오 압축을 위한 CQP 레벨입니다. 기본값은 "Low"입니다. |

**중요한 제약사항:**
- `upscaler_model` 또는 `interpolation_model` 중 적어도 하나는 활성화되어야 하며("Disabled"가 아님"), 그렇지 않으면 오류가 발생합니다.
- 입력 비디오는 MP4 컨테이너 형식이어야 합니다.
- 프롬프트가 있는 "Astra 2" 모델은 최대 450개의 입력 프레임(~15초 @ 30fps)으로 제한됩니다. 프롬프트가 없을 경우 최대 9000개의 입력 프레임으로 제한됩니다.
- `upscaler_model`이 "Disabled"가 아닐 때는 `upscaler_resolution` 하위 매개변수가 필요합니다.
- `interpolation_model`이 "Disabled"가 아닐 때는 `interpolation_frame_rate` 하위 매개변수가 필요합니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `video` | VIDEO | 선택된 업스케일링 및/또는 보간 필터를 적용한 향상된 비디오 출력입니다. |
