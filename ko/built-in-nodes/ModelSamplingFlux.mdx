ModelSamplingFlux 노드는 이미지 크기에 따라 시프트 파라미터를 계산하여 주어진 모델에 Flux 모델 샘플링을 적용합니다. 이 노드는 지정된 너비, 높이 및 시프트 파라미터에 따라 모델의 동작을 조정하는 특수한 샘플링 구성 설정을 만들고, 수정된 모델과 새로운 샘플링 설정을 반환합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `model` | MODEL | 예 | - | Flux 샘플링을 적용할 모델 |
| `max_shift` | FLOAT | 예 | 0.0 - 100.0 | 샘플링 계산에 사용되는 최대 시프트 값 (기본값: 1.15) |
| `base_shift` | FLOAT | 예 | 0.0 - 100.0 | 샘플링 계산에 사용되는 기본 시프트 값 (기본값: 0.5) |
| `width` | INT | 예 | 16 - MAX_RESOLUTION | 대상 이미지의 너비(픽셀 단위) (기본값: 1024) |
| `height` | INT | 예 | 16 - MAX_RESOLUTION | 대상 이미지의 높이(픽셀 단위) (기본값: 1024) |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `model` | MODEL | Flux 샘플링 구성이 적용된 수정된 모델 |
