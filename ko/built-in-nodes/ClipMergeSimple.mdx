`CLIPMergeSimple`은 지정된 비율에 따라 두 개의 CLIP 텍스트 인코더 모델을 결합하는 데 사용되는 고급 모델 병합 노드입니다.

이 노드는 지정된 비율에 따라 두 개의 CLIP 모델을 병합하여 각 모델의 특성을 효과적으로 혼합합니다. 특정 구성 요소(예: 위치 ID 및 로짓 스케일)를 제외하고 한 모델의 패치를 다른 모델에 선택적으로 적용하여 두 원본 모델의 특징을 결합한 하이브리드 모델을 만듭니다.

## 입력

| 매개변수 | 데이터 유형 | 입력 유형 | 기본값 | 범위 | 설명 |
|-----------|-----------|------------|---------|-------|-------------|
| `clip1` | CLIP | 필수 | - | - | 병합할 첫 번째 CLIP 모델입니다. 병합 과정의 기본 모델 역할을 합니다. |
| `clip2` | CLIP | 필수 | - | - | 병합할 두 번째 CLIP 모델입니다. 위치 ID와 로짓 스케일을 제외한 주요 패치가 지정된 비율에 따라 첫 번째 모델에 적용됩니다. |
| `ratio` | FLOAT | 필수 | 1.0 | 0.0 - 1.0 (단계: 0.01) | 두 번째 모델의 특징이 첫 번째 모델에 섞이는 비율을 결정합니다. 비율이 1.0이면 두 번째 모델의 특징을 완전히 채택하고, 0.0이면 첫 번째 모델의 특징만 유지됩니다. |

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `clip` | CLIP | 지정된 비율에 따라 두 입력 모델의 특징을 포함한 결과 병합 CLIP 모델입니다. |

## 병합 메커니즘 설명

### 병합 알고리즘

노드는 가중 평균 방식을 사용해 두 모델을 병합합니다:

1. **기본 모델 복제**: 먼저 `clip1`을 기본 모델로 복제합니다.
2. **패치 가져오기**: `clip2`에서 모든 주요 패치를 가져옵니다.
3. **특수 키 필터링**: `.position_ids`와 `.logit_scale`로 끝나는 키는 건너뜁니다.
4. **가중 병합 적용**: 수식 `(1.0 - ratio) * clip1 + ratio * clip2`를 사용합니다.

### 비율 매개변수 설명

- **비율 = 0.0**: `clip1`을 완전히 사용하고, `clip2`는 무시합니다.
- **비율 = 0.5**: 각 모델에서 50%씩 기여합니다.
- **비율 = 1.0**: `clip2`를 완전히 사용하고, `clip1`은 무시합니다.

## 활용 사례

1. **모델 스타일 융합**: 서로 다른 데이터로 학습된 CLIP 모델의 특성을 결합합니다.
2. **성능 최적화**: 서로 다른 모델의 강점과 약점을 균형 있게 조정합니다.
3. **실험적 연구**: 다양한 CLIP 인코더의 조합을 탐구합니다.
