ImageYUVToRGB 노드는 YUV 색 공간 이미지를 RGB 색 공간으로 변환합니다. Y(루마), U(블루 프로젝션), V(레드 프로젝션) 채널을 나타내는 세 개의 별도 입력 이미지를 받아 색 공간 변환을 통해 하나의 RGB 이미지로 결합합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `Y` | IMAGE | 예 | - | Y(루미넌스) 채널 입력 이미지 |
| `U` | IMAGE | 예 | - | U(블루 프로젝션) 채널 입력 이미지 |
| `V` | IMAGE | 예 | - | V(레드 프로젝션) 채널 입력 이미지 |

**참고:** 세 개의 입력 이미지(Y, U, V)는 모두 함께 제공되어야 하며, 적절한 변환을 위해 서로 호환 가능한 크기를 가져야 합니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `output` | IMAGE | 변환된 RGB 이미지 |
