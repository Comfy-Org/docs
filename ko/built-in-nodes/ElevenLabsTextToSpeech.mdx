> 이 문서는 AI로 생성되었습니다. 오류를 발견하거나 개선할 점이 있다면 언제든지 기여해 주세요! [GitHub에서 편집하기](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/ElevenLabsTextToSpeech/en.md)

ElevenLabs Text to Speech 노드는 ElevenLabs API를 사용해 작성된 텍스트를 음성으로 변환합니다. 특정 목소리를 선택하고 안정성, 속도, 스타일 등 다양한 말하기 특성을 세밀하게 조정하여 맞춤형 오디오 출력을 생성할 수 있습니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `voice` | CUSTOM | 예 | N/A | 음성 합성에 사용할 목소리입니다. Voice Selector 또는 Instant Voice Clone에서 연결하세요. |
| `text` | STRING | 예 | N/A | 음성으로 변환할 텍스트입니다. |
| `stability` | FLOAT | 아니요 | 0.0 - 1.0 | 목소리의 안정성입니다. 낮은 값은 더 폭넓은 감정 표현을 제공하며, 높은 값은 더 일정하지만 지루한 말하기를 만들어냅니다(기본값: 0.5). |
| `apply_text_normalization` | COMBO | 아니요 | `"auto"`<br />`"on"`<br />`"off"` | 텍스트 정규화 모드입니다. 'auto'는 시스템이 결정하도록 하며, 'on'은 항상 정규화를 적용하고, 'off'는 이를 건너뜁니다. |
| `model` | DYNAMICCOMBO | 아니요 | `"eleven_multilingual_v2"`<br />`"eleven_v3"` | 텍스트-음성 변환에 사용할 모델입니다. 모델을 선택하면 해당 모델의 구체적인 매개변수가 표시됩니다. |
| `language_code` | STRING | 아니요 | N/A | ISO-639-1 또는 ISO-639-3 언어 코드(예: 'en', 'es', 'fra'). 자동 감지하려면 비워두세요(기본값: ""). |
| `seed` | INT | 아니요 | 0 - 2147483647 | 재현성을 위한 시드값(결정성 보장되지 않음)(기본값: 1). |
| `output_format` | COMBO | 아니요 | `"mp3_44100_192"`<br />`"opus_48000_192"` | 오디오 출력 형식입니다. |

**모델별 매개변수:**
`model` 매개변수가 `"eleven_multilingual_v2"`로 설정된 경우, 다음과 같은 추가 매개변수가 사용 가능해집니다:

* `speed`: 말하기 속도입니다. 1.0은 일반 속도이며, <1.0은 느리고, >1.0은 빠릅니다(기본값: 1.0, 범위: 0.7 - 1.3).
* `similarity_boost`: 유사성 부스트입니다. 높은 값은 목소리를 원본과 더욱 유사하게 만듭니다(기본값: 0.75, 범위: 0.0 - 1.0).
* `use_speaker_boost`: 원래 화자의 목소리와의 유사성을 강화합니다(기본값: False).
* `style`: 스타일 과장입니다. 높은 값은 스타일적 표현을 증가시키지만 안정성을 떨어뜨릴 수 있습니다(기본값: 0.0, 범위: 0.0 - 0.2).

`model` 매개변수가 `"eleven_v3"`로 설정된 경우, 다음과 같은 추가 매개변수가 사용 가능해집니다:

* `speed`: 말하기 속도입니다. 1.0은 일반 속도이며, <1.0은 느리고, >1.0은 빠릅니다(기본값: 1.0, 범위: 0.7 - 1.3).
* `similarity_boost`: 유사성 부스트입니다. 높은 값은 목소리를 원본과 더욱 유사하게 만듭니다(기본값: 0.75, 범위: 0.0 - 1.0).
