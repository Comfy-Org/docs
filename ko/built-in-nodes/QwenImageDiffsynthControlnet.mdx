QwenImageDiffsynthControlnet 노드는 디퓨전 합성 제어 네트워크 패치를 적용하여 기본 모델의 동작을 수정합니다. 이미지 입력과 선택적 마스크를 사용해 모델의 생성 과정을 조정 가능한 강도로 유도하며, 제어 네트워크의 영향력을 반영한 패치된 모델을 생성합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `model` | MODEL | 예 | - | 제어 네트워크로 패치할 기본 모델 |
| `model_patch` | MODEL_PATCH | 예 | - | 기본 모델에 적용할 제어 네트워크 패치 모델 |
| `vae` | VAE | 예 | - | 디퓨전 과정에 사용되는 VAE (변분 오토인코더) |
| `image` | IMAGE | 예 | - | 제어 네트워크를 유도하는 입력 이미지 (RGB 채널만 사용됨) |
| `strength` | FLOAT | 예 | -10.0부터 10.0까지 | 제어 네트워크 영향력의 강도 (기본값: 1.0) |
| `mask` | MASK | 아니요 | - | 제어 네트워크를 적용할 영역을 정의하는 선택적 마스크 (내부적으로 역전됨) |

**참고:** 마스크가 제공되면 자동으로 역전(1.0 - 마스크)되고 제어 네트워크 처리에 필요한 크기에 맞춰 재조정됩니다. 노드는 모델 패치가 ZImage Control 유형인지 또는 일반 DiffSynth 제어 네트워크인지에 따라 서로 다른 내부 처리 방식을 사용합니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `model` | MODEL | 디퓨전 합성 제어 네트워크 패치가 적용된 수정된 모델 |
