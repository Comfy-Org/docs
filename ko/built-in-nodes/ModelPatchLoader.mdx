ModelPatchLoader 노드는 model_patches 폴더에서 전문적인 모델 패치를 로드합니다. 자동으로 패치 파일의 유형을 감지하고 적절한 모델 아키텍처를 로드한 후, 이를 ModelPatcher로 래핑하여 워크플로우에서 사용할 수 있도록 합니다. 이 노드는 컨트롤넷 블록, 피처 임베더 모델 및 기타 전문 아키텍처를 포함한 다양한 패치 유형을 지원합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `name` | 문자열 | 예 | model_patches 폴더에 있는 모든 사용 가능한 모델 패치 파일 | model_patches 디렉토리에서 로드할 모델 패치의 파일명 |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `MODEL_PATCH` | MODEL_PATCH | 워크플로우에서 사용할 수 있도록 ModelPatcher로 래핑된 로드된 모델 패치 |
