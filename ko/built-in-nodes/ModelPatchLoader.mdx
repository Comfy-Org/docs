ModelPatchLoader 노드는 model_patches 폴더에서 특화된 모델 패치를 로드합니다. 패치 파일의 유형을 자동으로 감지하여 적절한 모델 아키텍처를 로드한 후, 워크플로에서 사용할 수 있도록 ModelPatcher로 래핑합니다. 이 노드는 controlnet 블록, 특징 임베더 모델 및 기타 특수 아키텍처를 포함한 다양한 패치 유형을 지원합니다.

## 입력

| 매개변수 | 설명 | 데이터 유형 | 필수 | 범위 |
| --- | --- | --- | --- | --- |
| `이름` | model_patches 디렉터리에서 로드할 모델 패치의 파일 이름입니다. | STRING | 예 | model_patches 폴더의 모든 사용 가능한 모델 패치 파일 |

## 출력

| 출력 이름 | 설명 | 데이터 유형 |
| --- | --- | --- |
| `MODEL_PATCH` | 워크플로에서 사용할 수 있도록 ModelPatcher로 래핑된 로드된 모델 패치입니다. | MODEL_PATCH |
