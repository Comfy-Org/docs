> 이 문서는 AI로 생성되었습니다. 오류를 발견하거나 개선할 점이 있다면 언제든지 기여해 주세요! [GitHub에서 편집하기](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/ShuffleImageTextDataset/en.md)

이 노드는 이미지 목록과 텍스트 목록을 함께 섞으면서도 각 쌍의 짝은 그대로 유지합니다. 랜덤 시드를 사용해 섞는 순서를 결정하므로, 같은 시드를 재사용할 때마다 동일한 입력 목록이 동일한 방식으로 섞입니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `images` | IMAGE | 예 | - | 섞을 이미지 목록. |
| `texts` | STRING | 예 | - | 섞을 텍스트 목록. |
| `seed` | INT | 아니요 | 0부터 18446744073709551615까지 | 랜덤 시드. 이 값에 따라 섞는 순서가 결정됩니다(기본값: 0). |

**참고:** `images`와 `texts` 입력은 반드시 같은 길이의 리스트여야 합니다. 노드는 첫 번째 이미지를 첫 번째 텍스트와, 두 번째 이미지를 두 번째 텍스트와 등등 짝지은 후 이 쌍들을 함께 섞습니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `images` | IMAGE | 섞인 이미지 목록. |
| `texts` | STRING | 섞인 텍스트 목록이며, 이미지와의 원래 짝은 유지됩니다. |
