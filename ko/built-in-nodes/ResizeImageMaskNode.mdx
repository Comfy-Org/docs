Resize Image/Mask 노드는 입력 이미지 또는 마스크의 크기를 변경하는 여러 가지 방법을 제공합니다. 배율에 따라 스케일링하거나, 특정 크기를 설정하거나, 다른 입력의 크기에 맞추거나, 픽셀 수에 기반해 조정할 수 있으며, 다양한 보간 방법을 사용해 품질을 높일 수 있습니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `input` | IMAGE 또는 MASK | 예 | N/A | 크기를 조정할 이미지 또는 마스크입니다. |
| `resize_type` | COMBO | 예 | `SCALE_BY`<br />`SCALE_DIMENSIONS`<br />`SCALE_LONGER_DIMENSION`<br />`SCALE_SHORTER_DIMENSION`<br />`SCALE_WIDTH`<br />`SCALE_HEIGHT`<br />`SCALE_TOTAL_PIXELS`<br />`MATCH_SIZE` | 새 크기를 결정하는 방법입니다. 선택한 유형에 따라 필요한 매개변수가 달라집니다. |
| `multiplier` | FLOAT | 아니요 | 0.01부터 8.0까지 | 스케일링 요소입니다. `resize_type`가 `SCALE_BY`일 때 필요합니다(기본값: 1.00). |
| `width` | INT | 아니요 | 0부터 8192까지 | 픽셀 단위의 목표 너비입니다. `resize_type`가 `SCALE_DIMENSIONS` 또는 `SCALE_WIDTH`일 때 필요합니다(기본값: 512). |
| `height` | INT | 아니요 | 0부터 8192까지 | 픽셀 단위의 목표 높이입니다. `resize_type`가 `SCALE_DIMENSIONS` 또는 `SCALE_HEIGHT`일 때 필요합니다(기본값: 512). |
| `crop` | COMBO | 아니요 | `"disabled"`<br />`"center"` | 비율이 맞지 않을 때 적용할 자르기 방법입니다. `resize_type`가 `SCALE_DIMENSIONS` 또는 `MATCH_SIZE`일 때만 사용 가능합니다(기본값: "center"). |
| `longer_size` | INT | 아니요 | 0부터 8192까지 | 이미지의 더 긴 변의 목표 크기입니다. `resize_type`가 `SCALE_LONGER_DIMENSION`일 때 필요합니다(기본값: 512). |
| `shorter_size` | INT | 아니요 | 0부터 8192까지 | 이미지의 더 짧은 변의 목표 크기입니다. `resize_type`가 `SCALE_SHORTER_DIMENSION`일 때 필요합니다(기본값: 512). |
| `megapixels` | FLOAT | 아니요 | 0.01부터 16.0까지 | 목표 총 메가픽셀 수입니다. `resize_type`가 `SCALE_TOTAL_PIXELS`일 때 필요합니다(기본값: 1.0). |
| `match` | IMAGE 또는 MASK | 아니요 | N/A | 입력 이미지의 크기를 맞추기 위해 사용할 이미지 또는 마스크입니다. `resize_type`가 `MATCH_SIZE`일 때 필요합니다. |
| `scale_method` | COMBO | 예 | `"nearest-exact"`<br />`"bilinear"`<br />`"area"`<br />`"bicubic"`<br />`"lanczos"` | 스케일링에 사용되는 보간 알고리즘입니다(기본값: "area"). |

**참고:** `crop` 매개변수는 `resize_type`가 `SCALE_DIMENSIONS` 또는 `MATCH_SIZE`로 설정된 경우에만 사용 가능하며 관련성이 있습니다. `SCALE_WIDTH`나 `SCALE_HEIGHT`를 사용할 경우, 다른 한쪽 차원은 자동으로 원래의 종횡비를 유지하도록 스케일링됩니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `resized` | IMAGE 또는 MASK | 크기가 조정된 이미지 또는 마스크로, 입력과 동일한 데이터 타입을 갖습니다. |
