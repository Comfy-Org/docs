WanCameraEmbedding 노드는 카메라 모션 파라미터를 기반으로 플뤼커 임베딩을 사용하여 카메라 궤적 임베딩을 생성합니다. 이 노드는 다양한 카메라 움직임을 시뮬레이션하는 카메라 자세의 시퀀스를 만들어 비디오 생성 파이프라인에 적합한 임베딩 텐서로 변환합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `camera_pose` | COMBO | 예 | "정지"<br />"상향 팬"<br />"하향 팬"<br />"좌측 팬"<br />"우측 팬"<br />"줌 인"<br />"줌 아웃"<br />"반시계 방향 (ACW)"<br />"시계 방향 (CW)" | 시뮬레이션할 카메라 움직임의 유형 (기본값: "정지") |
| `width` | INT | 예 | 16부터 MAX_RESOLUTION까지 | 출력의 너비(픽셀 단위, 기본값: 832, 증분: 16) |
| `height` | INT | 예 | 16부터 MAX_RESOLUTION까지 | 출력의 높이(픽셀 단위, 기본값: 480, 증분: 16) |
| `length` | INT | 예 | 1부터 MAX_RESOLUTION까지 | 카메라 궤적 시퀀스의 길이(기본값: 81, 증분: 4) |
| `speed` | FLOAT | 아니요 | 0.0부터 10.0까지 | 카메라 움직임의 속도(기본값: 1.0, 증분: 0.1) |
| `fx` | FLOAT | 아니요 | 0.0부터 1.0까지 | 초점 거리 x 파라미터(기본값: 0.5, 증분: 0.000000001) |
| `fy` | FLOAT | 아니요 | 0.0부터 1.0까지 | 초점 거리 y 파라미터(기본값: 0.5, 증분: 0.000000001) |
| `cx` | FLOAT | 아니요 | 0.0부터 1.0까지 | 주점 x 좌표(기본값: 0.5, 증분: 0.01) |
| `cy` | FLOAT | 아니요 | 0.0부터 1.0까지 | 주점 y 좌표(기본값: 0.5, 증분: 0.01) |

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `camera_embedding` | TENSOR | 궤적 시퀀스를 포함한 생성된 카메라 임베딩 텐서 |
| `width` | INT | 처리에 사용된 너비 값 |
| `height` | INT | 처리에 사용된 높이 값 |
| `length` | INT | 처리에 사용된 길이 값 |
