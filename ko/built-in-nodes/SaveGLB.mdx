SaveGLB 노드는 3D 메시 데이터 또는 3D 파일을 출력 디렉터리에 저장합니다. 메시 데이터나 다양한 3D 파일 형식(GLB, GLTF, OBJ, FBX, STL, USDZ)을 받아 지정된 파일명 접두사와 함께 내보냅니다. 메시 데이터를 저장할 때는 여러 개의 메시를 처리할 수 있으며, 메타데이터가 활성화되면 파일에 자동으로 워크플로우 메타데이터를 추가합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `mesh` | MESH 또는 FILE3D | 예 | - | 저장할 메시 또는 3D 파일입니다. GLB, GLTF, OBJ, FBX, STL, USDZ를 포함한 메시 데이터 및 3D 파일 형식을 지원합니다 |
| `filename_prefix` | 문자열 | 아니요 | - | 출력 파일명의 접두사(기본값: "3d/ComfyUI") |

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `ui` | UI | 사용자 인터페이스에 저장된 3D 파일을 파일명, 하위 폴더 및 유형 정보와 함께 표시합니다 |
