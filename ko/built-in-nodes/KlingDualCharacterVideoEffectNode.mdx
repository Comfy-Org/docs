Kling 이중 캐릭터 비디오 효과 노드는 선택한 장면에 따라 특수 효과가 적용된 동영상을 생성합니다. 두 개의 이미지를 받아들여 첫 번째 이미지는 합성된 동영상의 왼쪽에, 두 번째 이미지는 오른쪽에 배치합니다. 선택한 효과 장면에 따라 다양한 시각적 효과가 적용됩니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `image_left` | IMAGE | 예 | - | 왼쪽 이미지 |
| `image_right` | IMAGE | 예 | - | 오른쪽 이미지 |
| `effect_scene` | COMBO | 예 | `"chat"`<br />`"dance"`<br />`"hug"`<br />`"kill"`<br />`"kiss"`<br />`"pat"`<br />`"punch"`<br />`"shrug"`<br />`"slap"`<br />`"tickle"` | 동영상 생성에 적용할 특수 효과 장면의 유형 |
| `model_name` | COMBO | 아니요 | `"kling-v1"`<br />`"kling-v1-5"`<br />`"kling-v1-6"` | 캐릭터 효과에 사용할 모델 (기본값: "kling-v1") |
| `mode` | COMBO | 아니요 | `"std"`<br />`"pro"` | 동영상 생성 모드 (기본값: "std") |
| `duration` | COMBO | 예 | `"5"`<br />`"10"` | 생성된 동영상의 지속 시간(초) |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `output` | VIDEO | 이중 캐릭터 효과가 적용된 생성된 동영상 |
| `duration` | STRING | 생성된 동영상의 지속 시간 정보 |
