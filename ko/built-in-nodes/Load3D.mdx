Load3D 노드는 3D 모델 파일을 로드하고 처리하는 핵심 노드입니다. 노드를 로드하면 `ComfyUI/input/3d/`에서 자동으로 사용 가능한 3D 리소스를 가져옵니다. 또한 업로드 기능을 사용해 지원되는 3D 파일을 업로드하여 미리보기할 수 있습니다.

**지원 형식**
현재 이 노드는 `.gltf`, `.glb`, `.obj`, `.fbx`, `.stl` 등 여러 3D 파일 형식을 지원합니다.

**3D 노드 환경 설정**
ComfyUI의 설정 메뉴에서 일부 관련 3D 노드 환경 설정을 구성할 수 있습니다. 해당 설정에 대한 자세한 내용은 다음 문서를 참조하세요:

[설정 메뉴](https://docs.comfy.org/interface/settings/3d)

일반적인 노드 출력 외에도 Load3D는 캔버스 메뉴에서 다양한 3D 뷰 관련 설정을 제공합니다.

## 입력

| 매개변수 이름 | 유형     | 설명                     | 기본값 | 범위        |
|---------------|----------|--------------------------|---------|--------------|
| model_file    | 파일 선택 | 3D 모델 파일 경로, 업로드 가능, 기본적으로 `ComfyUI/input/3d/`에서 모델 파일 읽기 | - | 지원 형식 |
| width         | 정수     | 캔버스 렌더링 너비       | 1024    | 1-4096      |
| height        | 정수     | 캔버스 렌더링 높이       | 1024    | 1-4096      |

## 출력

| 매개변수 이름   | 데이터 타입      | 설명                        |
|-----------------|----------------|------------------------------------|
| image           | 이미지          | 캔버스 렌더링 이미지              |
| mask            | 마스크           | 현재 모델 위치를 포함한 마스크 |
| mesh_path       | 문자열         | 모델 파일 경로                   |
| normal          | 이미지          | 노멀 맵                         |
| lineart         | 이미지          | 라인 아트 이미지 출력, 캔버스 모델 메뉴에서 `edge_threshold` 조정 가능 |
| camera_info     | LOAD3D 카메라  | 카메라 정보                     |
| recording_video | 비디오          | 녹화된 비디오 (녹화가 있는 경우만) |

모든 출력 미리보기:

## 캔버스 영역 설명

Load3D 노드의 캔버스 영역에는 다음과 같은 다양한 뷰 작업이 포함되어 있습니다:

- 미리보기 뷰 설정 (격자, 배경색, 미리보기 뷰)
- 카메라 제어: FOV, 카메라 유형 제어
- 전역 조명 강도: 조명 강도 조정
- 동영상 녹화: 비디오 녹화 및 내보내기
- 모델 내보내기: `GLB`, `OBJ`, `STL` 형식 지원
- 그 외 다수

![Load 3D 노드 UI](/images/built-in-nodes/Load3D/load3d_ui.jpg)

1. Load 3D 노드의 여러 메뉴와 숨겨진 메뉴 포함
2. 미리보기 창 크기 조정 및 캔버스 동영상 녹화 메뉴
3. 3D 뷰 작업 축
4. 미리보기 썸네일
5. 미리보기 크기 설정, 크기를 설정한 후 창 크기 조정으로 미리보기 뷰 표시 조정

### 1. 뷰 작업

뷰 제어 작업:

- 왼쪽 클릭 + 드래그: 뷰 회전
- 오른쪽 클릭 + 드래그: 뷰 이동
- 가운데 휠 스크롤 또는 가운데 클릭 + 드래그: 확대/축소
- 좌표축: 뷰 전환

### 2. 왼쪽 메뉴 기능

![메뉴](/images/built-in-nodes/Load3D/menu.webp)

캔버스에서는 일부 설정이 메뉴에 숨겨져 있습니다. 메뉴 버튼을 클릭해 다양한 메뉴를 펼치세요.

- 1. 장면: 미리보기 창 격자, 배경색, 미리보기 설정 포함
- 2. 모델: 모델 렌더링 모드, 텍스처 재료, 상향 방향 설정
- 3. 카메라: 직교 및 원근 뷰 전환, 원근 각도 크기 설정
- 4. 라이트: 장면 전역 조명 강도
- 5. 내보내기: 모델을 다른 형식(GLB, OBJ, STL)으로 내보내기

#### 장면

![scene menu](/images/built-in-nodes/Load3D/menu_scene.webp)

장면 메뉴는 몇 가지 기본적인 장면 설정 기능을 제공합니다.

1. 격자 표시/숨기기
2. 배경색 설정
3. 배경 이미지 업로드 클릭
4. 미리보기 숨기기

#### 모델

![Menu_Scene](/images/built-in-nodes/Load3D/menu_model.webp)

모델 메뉴는 모델 관련 기능을 제공합니다.

1. **상향 방향**: 모델의 상향 방향을 결정하는 축
2. **재료 모드**: 모델 렌더링 모드 전환 - 원본, 노말, 와이어프레임, 라인아트

#### 카메라

![menu_modelmenu_camera](/images/built-in-nodes/Load3D/menu_camera.webp)

이 메뉴는 직교 및 원근 뷰 전환과 원근 각도 크기 설정을 제공합니다.

1. **카메라**: 직교 및 원근 뷰 간 빠르게 전환
2. **FOV**: FOV 각도 조정

#### 라이트

![menu_modelmenu_camera](/images/built-in-nodes/Load3D/menu_light.webp)

이 메뉴를 통해 장면의 전역 조명 강도를 빠르게 조정할 수 있습니다.

#### 내보내기

![menu_export](/images/built-in-nodes/Load3D/menu_export.webp)

이 메뉴는 모델 형식을 빠르게 변환하고 내보낼 수 있는 기능을 제공합니다.

### 3. 오른쪽 메뉴 기능

<video controls width="640" height="360">
  <source src="https://raw.githubusercontent.com/Comfy-Org/embedded-docs/refs/heads/main/comfyui_embedded_docs/docs/Load3D/asset/view_operations.mp4" type="video/mp4" />
  브라우저가 비디오 재생을 지원하지 않습니다.
</video>

오른쪽 메뉴에는 두 가지 주요 기능이 있습니다:

1. **뷰 비율 초기화**: 버튼을 클릭한 후 뷰가 설정된 너비와 높이에 따라 캔버스 렌더링 영역 비율을 조정합니다.
2. **동영상 녹화**: 현재 3D 뷰 작업을 비디오로 녹화할 수 있으며, 불러오기 기능을 제공하며, 이후 노드로 `recording_video`로 출력할 수 있습니다.
