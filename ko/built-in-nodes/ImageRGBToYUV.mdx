ImageRGBToYUV 노드는 RGB 이미지를 YUV 색 공간으로 변환합니다. 이 노드는 이미지를 세 가지 채널로 분리하여 각각을 별도의 이미지로 반환합니다: Y(휘도 또는 밝기), U(파랑 차이), V(빨강 차이)입니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `image` | IMAGE | 예 | - | YUV 색 공간으로 변환할 입력 RGB 이미지 |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `Y` | IMAGE | YUV 색 공간의 휘도(밝기) 구성 요소 |
| `U` | IMAGE | YUV 색 공간의 파랑 차이 크로마 구성 요소 |
| `V` | IMAGE | YUV 색 공간의 빨강 차이 크로마 구성 요소 |
