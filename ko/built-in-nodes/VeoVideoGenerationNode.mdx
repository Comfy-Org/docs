> 이 문서는 AI로 생성되었습니다. 오류를 발견하거나 개선할 점이 있다면 언제든지 기여해 주세요! [GitHub에서 편집하기](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/VeoVideoGenerationNode/en.md)

Google의 Veo API를 사용하여 텍스트 프롬프트로부터 동영상을 생성합니다. 이 노드는 텍스트 설명과 선택적 이미지 입력을 바탕으로 동영상을 만들 수 있으며, 종횡비, 지속 시간 등 다양한 매개변수를 제어할 수 있습니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `prompt` | 문자열 | 예 | - | 동영상의 텍스트 설명 (기본값: 빈 문자열) |
| `aspect_ratio` | 콤보 | 예 | "16:9"<br />"9:16" | 출력 동영상의 종횡비 (기본값: "16:9") |
| `negative_prompt` | 문자열 | 아니요 | - | 동영상에서 피하고자 하는 내용을 안내하는 부정적인 텍스트 프롬프트 (기본값: 빈 문자열) |
| `duration_seconds` | 정수 | 아니요 | 5~8 | 출력 동영상의 지속 시간(초) (기본값: 5) |
| `enhance_prompt` | 불리언 | 아니요 | - | AI 도움을 받아 프롬프트를 강화할지 여부 (기본값: True). 이는 고급 매개변수입니다. |
| `person_generation` | 콤보 | 아니요 | "ALLOW"<br />"BLOCK" | 동영상에 사람을 생성하도록 허용할지 여부 (기본값: "ALLOW"). 이는 고급 매개변수입니다. |
| `seed` | 정수 | 아니요 | 0~4294967295 | 동영상 생성을 위한 시드 값 (랜덤일 경우 0) (기본값: 0). 이는 고급 매개변수입니다. |
| `image` | 이미지 | 아니요 | - | 동영상 생성을 안내할 수 있는 선택적 참조 이미지 |
| `model` | 콤보 | 아니요 | "veo-2.0-generate-001"<br />"veo-3.1-generate"<br />"veo-3.1-fast-generate"<br />"veo-3.1-lite"<br />"veo-3.0-generate-001"<br />"veo-3.0-fast-generate-001" | 동영상 생성에 사용할 Veo 모델 (기본값: "veo-2.0-generate-001") |

**참고:** `generate_audio` 매개변수는 Veo 3.0 및 Veo 3.1 모델에서만 사용 가능하며, 선택한 모델에 따라 노드에서 자동으로 처리됩니다. Veo 3.0 또는 Veo 3.1 모델을 사용할 경우 `enhance_prompt` 매개변수는 항상 True로 강제됩니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `output` | 동영상 | 생성된 동영상 파일 |
