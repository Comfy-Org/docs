`GenerateTracks` 노드는 비디오 생성을 위해 여러 개의 병렬 동작 경로를 생성합니다. 이 노드는 시작점에서 끝점까지의 기본 경로를 정의한 후, 이 경로와 평행하게 일정한 간격으로 배열된 트랙 세트를 생성합니다. 경로의 형태(직선 또는 베지어 곡선), 경로를 따라 움직이는 속도, 그리고 트랙이 표시되는 프레임을 제어할 수 있습니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `width` | INT | 예 | 16 - 4096 | 비디오 프레임의 너비(픽셀 단위). 기본값은 832입니다. |
| `height` | INT | 예 | 16 - 4096 | 비디오 프레임의 높이(픽셀 단위). 기본값은 480입니다. |
| `start_x` | FLOAT | 예 | 0.0 - 1.0 | 시작 위치의 정규화된 X 좌표(0~1). 기본값은 0.0입니다. |
| `start_y` | FLOAT | 예 | 0.0 - 1.0 | 시작 위치의 정규화된 Y 좌표(0~1). 기본값은 0.0입니다. |
| `end_x` | FLOAT | 예 | 0.0 - 1.0 | 끝 위치의 정규화된 X 좌표(0~1). 기본값은 1.0입니다. |
| `end_y` | FLOAT | 예 | 0.0 - 1.0 | 끝 위치의 정규화된 Y 좌표(0~1). 기본값은 1.0입니다. |
| `num_frames` | INT | 예 | 1 - 1024 | 트랙 위치를 생성할 총 프레임 수. 기본값은 81입니다. |
| `num_tracks` | INT | 예 | 1 - 100 | 생성할 병렬 트랙의 수. 기본값은 5입니다. |
| `track_spread` | FLOAT | 예 | 0.0 - 1.0 | 트랙 간의 정규화된 거리. 트랙은 움직임 방향과 수직으로 퍼져나갑니다. 기본값은 0.025입니다. |
| `bezier` | BOOLEAN | 예 | True / False | 중간점을 제어점으로 사용하는 베지어 곡선 경로를 활성화합니다. 기본값은 False입니다. |
| `mid_x` | FLOAT | 예 | 0.0 - 1.0 | 베지어 곡선의 정규화된 X 제어점. 'bezier'가 활성화된 경우에만 사용됩니다. 기본값은 0.5입니다. |
| `mid_y` | FLOAT | 예 | 0.0 - 1.0 | 베지어 곡선의 정규화된 Y 제어점. 'bezier'가 활성화된 경우에만 사용됩니다. 기본값은 0.5입니다. |
| `interpolation` | COMBO | 예 | `"linear"`<br />`"ease_in"`<br />`"ease_out"`<br />`"ease_in_out"`<br />`"constant"` | 경로를 따라 움직이는 시간/속도를 제어합니다. 기본값은 "linear"입니다. |
| `track_mask` | MASK | 선택 | - | 보이는 프레임을 지정하는 옵션 마스크입니다. |

**참고:** `mid_x`와 `mid_y` 매개변수는 'bezier' 매개변수가 True로 설정된 경우에만 사용됩니다. 'bezier'가 False일 때는 경로가 시작점에서 끝점까지 직선입니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-----------|-----------|-------------|
| `TRACKS` | TRACKS | 모든 프레임에 걸쳐 생성된 경로 좌표와 가시성 정보를 포함한 트랙 객체입니다. |
| `track_length` | INT | 트랙이 생성된 프레임 수로, 입력된 `num_frames`와 일치합니다. |
