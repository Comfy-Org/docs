RandomNoise 노드는 시드 값을 기반으로 무작위 노이즈 패턴을 생성합니다. 이 노드는 다양한 이미지 처리 및 생성 작업에 사용할 수 있는 재현 가능한 노이즈를 생성합니다. 동일한 시드는 항상 동일한 노이즈 패턴을 생성하므로 여러 번 실행해도 일관된 결과를 얻을 수 있습니다.

## 입력

| 매개변수 | 설명 | 데이터 타입 | 필수 | 범위 |
| --- | --- | --- | --- | --- |
| `노이즈 시드` | 무작위 노이즈 패턴을 생성하는 데 사용되는 시드 값입니다(기본값: 0). 동일한 시드는 항상 동일한 노이즈 출력을 생성합니다. | INT | 예 | 0 ~ 18446744073709551615 |

## 출력

| 출력 이름 | 설명 | 데이터 타입 |
| --- | --- | --- |
| `noise` | 제공된 시드 값을 기반으로 생성된 무작위 노이즈 패턴입니다. | NOISE |
