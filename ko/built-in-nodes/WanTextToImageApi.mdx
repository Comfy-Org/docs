> 이 문서는 AI로 생성되었습니다. 오류를 발견하거나 개선할 점이 있다면 언제든지 기여해 주세요! [GitHub에서 편집하기](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/WanTextToImageApi/en.md)

Wan Text to Image 노드는 텍스트 설명을 기반으로 이미지를 생성합니다. 이 노드는 AI 모델을 사용해 글로 쓰인 프롬프트로부터 시각적 콘텐츠를 만들어내며, 영어와 중국어 텍스트 입력을 모두 지원합니다. 노드는 출력 이미지 크기, 품질 및 스타일 선호도를 조정할 수 있는 다양한 컨트롤을 제공합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `model` | 문자열 | 예 | "wan2.5-t2i-preview" | 사용할 모델 (기본값: "wan2.5-t2i-preview") |
| `prompt` | 문자열 | 예 | - | 요소와 시각적 특징을 설명하는 프롬프트입니다. 영어와 중국어를 지원하며 기본값은 비어 있습니다. |
| `negative_prompt` | 문자열 | 아니요 | - | 피해야 할 내용을 설명하는 네거티브 프롬프트입니다. 기본값은 비어 있습니다. |
| `width` | 정수 | 아니요 | 768-1440 | 이미지 너비(픽셀 단위)입니다. 기본값은 1024이며, 증분은 32입니다. |
| `height` | 정수 | 아니요 | 768-1440 | 이미지 높이(픽셀 단위)입니다. 기본값은 1024이며, 증분은 32입니다. |
| `seed` | 정수 | 아니요 | 0-2147483647 | 생성에 사용할 시드입니다. 기본값은 0입니다. |
| `prompt_extend` | 부울 | 아니요 | - | AI의 도움을 받아 프롬프트를 강화할지 여부입니다. 기본값은 True입니다. |
| `watermark` | 부울 | 아니요 | - | 결과에 AI가 생성한 워터마크를 추가할지 여부입니다. 기본값은 False입니다. |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `output` | 이미지 | 텍스트 프롬프트를 기반으로 생성된 이미지 |
