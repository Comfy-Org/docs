> 이 문서는 AI로 생성되었습니다. 오류를 발견하거나 개선할 점이 있다면 언제든지 기여해 주세요! [GitHub에서 편집하기](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/TextEncodeZImageOmni/en.md)

TextEncodeZImageOmni 노드는 고급 컨디셔닝 노드로, 텍스트 프롬프트와 선택적 참조 이미지를 이미지 생성 모델에 적합한 컨디셔닝 형식으로 인코딩합니다. 최대 세 장의 이미지를 처리할 수 있으며, 선택적으로 비전 인코더 및/또는 VAE를 사용해 참조 래텐트를 생성하고 특정 템플릿 구조를 통해 텍스트 프롬프트와 통합합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `clip` | CLIP | 예 | | 텍스트 프롬프트를 토큰화하고 인코딩하는 데 사용되는 CLIP 모델입니다. |
| `image_encoder` | CLIPVision | 아니요 | | 선택적 비전 인코더 모델입니다. 제공되면 입력 이미지를 인코딩하는 데 사용되며, 결과 임베딩은 컨디셔닝에 추가됩니다. |
| `prompt` | STRING | 예 | | 인코딩할 텍스트 프롬프트입니다. 이 필드는 다중 줄 입력과 동적 프롬프트를 지원합니다. |
| `auto_resize_images` | BOOLEAN | 아니요 | | 활성화 시(기본값: True), 입력 이미지는 VAE에 전달되기 전에 픽셀 면적에 따라 자동으로 크기가 조정됩니다. |
| `vae` | VAE | 아니요 | | 선택적 VAE 모델입니다. 제공되면 입력 이미지를 래텐트 표현으로 인코딩하는 데 사용되며, 이 래텐트는 참조 래텐트로 컨디셔닝에 추가됩니다. |
| `image1` | IMAGE | 아니요 | | 첫 번째 선택적 참조 이미지입니다. |
| `image2` | IMAGE | 아니요 | | 두 번째 선택적 참조 이미지입니다. |
| `image3` | IMAGE | 아니요 | | 세 번째 선택적 참조 이미지입니다. |

**참고:** 노드는 최대 세 장의 이미지를 받을 수 있습니다(`image1`, `image2`, `image3`). `image_encoder`와 `vae` 입력은 이미지가 하나 이상 제공된 경우에만 사용됩니다. `auto_resize_images`가 True이고 `vae`가 연결된 경우, 이미지는 인코딩 전에 총 픽셀 면적이 약 1024x1024에 가깝도록 크기가 조정됩니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `CONDITIONING` | CONDITIONING | 최종 컨디셔닝 출력으로, 인코딩된 텍스트 프롬프트를 포함하며 이미지를 제공한 경우 이미지 임베딩 및/또는 참조 래텐트도 포함될 수 있습니다. |
