ModelSamplingContinuousV 노드는 연속 V-예측 샘플링 파라미터를 적용하여 모델의 샘플링 동작을 수정합니다. 입력 모델의 복제본을 생성하고, 고급 샘플링 제어를 위해 맞춤형 시그마 범위 설정으로 구성합니다. 이를 통해 사용자는 특정 최소 및 최대 시그마 값을 지정해 샘플링 과정을 세밀하게 조정할 수 있습니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `model` | MODEL | 예 | - | 연속 V-예측 샘플링으로 수정할 입력 모델 |
| `sampling` | STRING | 예 | `"v_prediction"` | 적용할 샘플링 방법 (현재 V-예측만 지원됨) |
| `sigma_max` | FLOAT | 예 | 0.0 - 1000.0 | 샘플링의 최대 시그마 값 (기본값: 500.0) |
| `sigma_min` | FLOAT | 예 | 0.0 - 1000.0 | 샘플링의 최소 시그마 값 (기본값: 0.03) |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-----------|-----------|-------------|
| `model` | MODEL | 연속 V-예측 샘플링이 적용된 수정된 모델 |
