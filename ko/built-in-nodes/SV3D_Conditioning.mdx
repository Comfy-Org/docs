SV3D_Conditioning 노드는 SV3D 모델을 사용하여 3D 비디오 생성을 위한 컨디셔닝 데이터를 준비합니다. 초기 이미지를 받아 CLIP 비전과 VAE 인코더를 통해 긍정적 및 부정적 컨디셔닝과 잠재 표현을 생성합니다. 이 노드는 지정된 비디오 프레임 수를 기반으로 다중 프레임 비디오 생성을 위한 카메라 고도와 방위각 시퀀스를 생성합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `clip_vision` | CLIP_VISION | 예 | - | 입력 이미지를 인코딩하는 데 사용되는 CLIP 비전 모델 |
| `init_image` | IMAGE | 예 | - | 3D 비디오 생성의 시작점이 되는 초기 이미지 |
| `vae` | VAE | 예 | - | 이미지를 잠재 공간으로 인코딩하는 데 사용되는 VAE 모델 |
| `width` | INT | 예 | 16부터 MAX_RESOLUTION까지 | 생성된 비디오 프레임의 출력 너비 (기본값: 576, 8로 나누어져야 함) |
| `height` | INT | 예 | 16부터 MAX_RESOLUTION까지 | 생성된 비디오 프레임의 출력 높이 (기본값: 576, 8로 나누어져야 함) |
| `video_frames` | INT | 예 | 1부터 4096까지 | 비디오 시퀀스에 생성할 프레임 수 (기본값: 21) |
| `elevation` | FLOAT | 예 | -90.0부터 90.0까지 | 3D 뷰의 카메라 고도 각도 (도 단위, 기본값: 0.0) |

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-----------|-----------|-------------|
| `positive` | CONDITIONING | 이미지 임베딩과 카메라 파라미터를 포함한 긍정적 컨디셔닝 데이터, 생성용 |
| `negative` | CONDITIONING | 대조적 생성을 위해 임베딩이 0으로 설정된 부정적 컨디셔닝 데이터 |
| `latent` | LATENT | 지정된 비디오 프레임과 해상도에 맞는 차원의 빈 잠재 텐서 |
