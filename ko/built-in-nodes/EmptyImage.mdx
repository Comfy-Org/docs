## 기능 설명

EmptyImage 노드는 지정된 크기와 색상을 가진 빈 이미지를 생성하는 데 사용됩니다. 이 노드는 단색 배경 이미지를 생성할 수 있으며, 이미지 처리 워크플로우의 시작점이나 배경 이미지로 자주 사용됩니다.

## 작동 원리

마치 화가가 작품을 시작하기 전에 빈 캔버스를 준비하듯이, EmptyImage 노드는 당신에게 '디지털 캔버스'를 제공합니다. 캔버스의 크기(너비와 높이)를 지정하고, 캔버스의 기본 색상을 선택할 수 있으며, 동일한 사양의 캔버스를 한 번에 여러 개 준비할 수도 있습니다. 이 노드는 마치 표준화된 캔버스를 완벽하게 맞춤형 크기와 색상으로 만들어주는 지능형 미술용품점과 같습니다.

## 입력

| 매개변수 이름 | 데이터 타입 | 설명 |
|----------------|-----------|-------------|
| `width` | INT | 생성할 이미지의 너비를 설정합니다(픽셀 단위), 캔버스의 가로 치수를 결정합니다 |
| `height` | INT | 생성할 이미지의 높이를 설정합니다(픽셀 단위), 캔버스의 세로 치수를 결정합니다 |
| `batch_size` | INT | 한 번에 생성할 이미지의 수를 설정하며, 동일한 사양의 이미지를 일괄 생성할 때 사용됩니다 |
| `color` | INT | 이미지의 배경색입니다. 16진수 색상 설정을 입력하면 자동으로 10진수로 변환됩니다 |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `image` | IMAGE | 생성된 빈 이미지 텐서로, [batch_size, height, width, 3] 형식이며 RGB 세 가지 색채 채널을 포함합니다 |

## 일반적인 색상 참조 값

현재 이 노드의 색상 입력은 사용자 친화적이지 않으며 모든 색상 값이 10진수로 변환되므로, 빠르게 적용할 수 있는 몇 가지 일반적인 색상 값을 아래에 정리했습니다.
