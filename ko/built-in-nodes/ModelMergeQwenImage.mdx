ModelMergeQwenImage 노드는 두 개의 AI 모델을 조정 가능한 가중치로 구성 요소를 결합하여 병합합니다. 이 노드를 통해 Qwen 이미지 모델의 특정 부분, 예를 들어 변환기 블록, 위치 임베딩, 텍스트 처리 구성 요소 등을 혼합할 수 있습니다. 각 모델이 병합된 결과의 서로 다른 부분에 얼마나 큰 영향을 미칠지 제어할 수 있습니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `model1` | MODEL | 예 | - | 병합할 첫 번째 모델 (기본값: 없음) |
| `model2` | MODEL | 예 | - | 병합할 두 번째 모델 (기본값: 없음) |
| `pos_embeds.` | FLOAT | 예 | 0.0부터 1.0까지 | 위치 임베딩 블렌딩 가중치 (기본값: 1.0) |
| `img_in.` | FLOAT | 예 | 0.0부터 1.0까지 | 이미지 입력 처리 블렌딩 가중치 (기본값: 1.0) |
| `txt_norm.` | FLOAT | 예 | 0.0부터 1.0까지 | 텍스트 정규화 블렌딩 가중치 (기본값: 1.0) |
| `txt_in.` | FLOAT | 예 | 0.0부터 1.0까지 | 텍스트 입력 처리 블렌딩 가중치 (기본값: 1.0) |
| `time_text_embed.` | FLOAT | 예 | 0.0부터 1.0까지 | 시간 및 텍스트 임베딩 블렌딩 가중치 (기본값: 1.0) |
| `transformer_blocks.0.`부터 `transformer_blocks.59.` | FLOAT | 예 | 0.0부터 1.0까지 | 각 변환기 블록 블렌딩 가중치 (기본값: 1.0) |
| `proj_out.` | FLOAT | 예 | 0.0부터 1.0까지 | 출력 프로젝션 블렌딩 가중치 (기본값: 1.0) |

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `model` | MODEL | 지정된 가중치로 두 입력 모델의 구성 요소를 결합한 병합된 모델 |
