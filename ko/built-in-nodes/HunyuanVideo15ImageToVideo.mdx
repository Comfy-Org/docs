HunyuanVideo15ImageToVideo 노드는 HunyuanVideo 1.5 모델을 기반으로 비디오 생성을 위한 조건부 및 잠재 공간 데이터를 준비합니다. 비디오 시퀀스의 초기 잠재 표현을 생성하며, 선택적으로 시작 이미지 또는 CLIP 비전 출력을 통합하여 생성 과정을 안내할 수 있습니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `positive` | CONDITIONING | 예 | - | 비디오에 포함되어야 할 내용을 설명하는 긍정적 조건부 프롬프트입니다. |
| `negative` | CONDITIONING | 예 | - | 비디오에 피해야 할 내용을 설명하는 부정적 조건부 프롬프트입니다. |
| `vae` | VAE | 예 | - | 시작 이미지를 잠재 공간으로 인코딩하는 데 사용되는 VAE(변분 오토인코더) 모델입니다. |
| `width` | INT | 아니요 | 16부터 MAX_RESOLUTION까지, 단계: 16 | 출력 비디오 프레임의 너비(픽셀). 16으로 나누어져야 합니다. (기본값: 848) |
| `height` | INT | 아니요 | 16부터 MAX_RESOLUTION까지, 단계: 16 | 출력 비디오 프레임의 높이(픽셀). 16으로 나누어져야 합니다. (기본값: 480) |
| `length` | INT | 아니요 | 1부터 MAX_RESOLUTION까지, 단계: 4 | 비디오 시퀀스의 총 프레임 수. 4의 배수여야 합니다. (기본값: 33) |
| `batch_size` | INT | 아니요 | 1부터 4096까지 | 한 번에 생성할 비디오 시퀀스의 수입니다. (기본값: 1) |
| `start_image` | IMAGE | 아니요 | - | 비디오 생성을 초기화하기 위한 선택적 시작 이미지입니다. 제공되면 인코딩되어 첫 번째 프레임의 조건부로 사용됩니다. 이미지의 처음 `length` 프레임만 사용됩니다. |
| `clip_vision_output` | CLIP_VISION_OUTPUT | 아니요 | - | 생성을 위한 추가적인 시각적 조건부를 제공하는 선택적 CLIP 비전 임베딩입니다. |

**참고:** `start_image`가 제공되면, 이 이미지는 자동으로 이중선형 보간법을 사용해 지정된 `width`와 `height`에 맞게 크기가 조정됩니다. 이미지의 처음 `length` 프레임만 사용됩니다. 그런 다음 인코딩된 이미지는 `concat_latent_image`로 `positive`와 `negative` 조건부 모두에 추가되며, 이에 대응하는 `concat_mask`도 함께 제공됩니다. 마스크는 시작 이미지가 적용된 프레임은 0.0으로, 나머지 프레임은 1.0으로 설정됩니다.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `positive` | CONDITIONING | 수정된 긍정적 조건부로, 이제 인코딩된 시작 이미지나 CLIP 비전 출력이 포함될 수 있습니다. |
| `negative` | CONDITIONING | 수정된 부정적 조건부로, 이제 인코딩된 시작 이미지나 CLIP 비전 출력이 포함될 수 있습니다. |
| `latent` | LATENT | 지정된 배치 크기, 비디오 길이, 너비 및 높이에 맞춰 구성된 빈 잠재 텐서입니다. |
