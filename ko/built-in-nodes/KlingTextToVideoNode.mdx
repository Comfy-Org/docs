Kling 텍스트 to 비디오 노드는 텍스트 설명을 비디오 콘텐츠로 변환합니다. 텍스트 프롬프트를 받아 지정된 구성 설정에 따라 해당하는 비디오 시퀀스를 생성합니다. 이 노드는 다양한 종횡비, 생성 모드 및 모델 버전을 지원하여 각기 다른 길이와 품질의 비디오를 생성할 수 있습니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `prompt` | 문자열 | 예 | 최대 2500자 | 원하는 비디오 콘텐츠를 설명하는 긍정적 텍스트 프롬프트 |
| `negative_prompt` | 문자열 | 예 | 최대 2500자 | 비디오에서 피해야 할 내용을 설명하는 부정적 텍스트 프롬프트 |
| `cfg_scale` | 실수 | 아니요 | 0.0~1.0 | 비디오가 프롬프트를 얼마나 정확히 따르는지를 제어하는 구성 스케일 값 (기본값: 1.0) |
| `aspect_ratio` | 콤보 | 아니요 | `"16:9"`<br />`"9:16"`<br />`"1:1"` | 비디오 종횡비 설정 (기본값: "16:9") |
| `mode` | 콤보 | 아니요 | `"표준 모드 / 5초 길이 / kling-v1-6"`<br />`"표준 모드 / 10초 길이 / kling-v1-6"`<br />`"프로 모드 / 5초 길이 / kling-v2-master"`<br />`"프로 모드 / 10초 길이 / kling-v2-master"`<br />`"표준 모드 / 5초 길이 / kling-v2-master"`<br />`"표준 모드 / 10초 길이 / kling-v2-master"`<br />`"프로 모드 / 5초 길이 / kling-v2-1-master"`<br />`"프로 모드 / 10초 길이 / kling-v2-1-master"`<br />`"프로 모드 / 5초 길이 / kling-v2-5-turbo"`<br />`"프로 모드 / 10초 길이 / kling-v2-5-turbo"` | 비디오 생성에 사용할 구성으로, 형식은 모드 / 길이 / 모델 이름입니다 (기본값: "프로 모드 / 5초 길이 / kling-v2-5-turbo") |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `output` | 비디오 | 생성된 비디오 출력 |
| `video_id` | 문자열 | 생성된 비디오의 고유 식별자 |
| `duration` | 문자열 | 생성된 비디오의 길이 정보 |
