이 노드는 최대 네 장의 이미지를 처리하여 Tripo의 API를 이용해 3D 모델을 동기적으로 생성합니다. 객체의 다양한 시점을 보여주는 앞면 이미지와 적어도 하나의 추가 시점(왼쪽, 뒷면 또는 오른쪽)이 필요하며, 이를 통해 질감과 재료 옵션을 포함한 완전한 3D 모델을 만들 수 있습니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `image` | IMAGE | 예 | - | 객체의 앞면 이미지 (필수) |
| `image_left` | IMAGE | 아니요 | - | 객체의 왼쪽 시점 이미지 |
| `image_back` | IMAGE | 아니요 | - | 객체의 뒷면 이미지 |
| `image_right` | IMAGE | 아니요 | - | 객체의 오른쪽 시점 이미지 |
| `model_version` | COMBO | 아니요 | 여러 옵션 제공 | 생성에 사용할 모델 버전 |
| `orientation` | COMBO | 아니요 | 여러 옵션 제공 | 3D 모델의 방향 설정 (기본값: "default") |
| `texture` | BOOLEAN | 아니요 | - | 모델에 질감을 생성할지 여부 (기본값: True) |
| `pbr` | BOOLEAN | 아니요 | - | PBR(물리 기반 렌더링) 재료를 생성할지 여부 (기본값: True) |
| `model_seed` | INT | 아니요 | - | 모델 생성용 난수 시드 (기본값: 42) |
| `texture_seed` | INT | 아니요 | - | 질감 생성용 난수 시드 (기본값: 42) |
| `texture_quality` | COMBO | 아니요 | `"standard"`<br />`"detailed"` | 질감 생성의 품질 수준 (기본값: "standard") |
| `texture_alignment` | COMBO | 아니요 | `"original_image"`<br />`"geometry"` | 질감을 모델에 맞추는 방법 (기본값: "original_image") |
| `face_limit` | INT | 아니요 | -1부터 500000까지 | 생성된 모델의 최대 면 수. 한계 없음으로 설정하려면 -1로 지정하세요 (기본값: -1) |
| `quad` | BOOLEAN | 아니요 | - | 이 매개변수는 더 이상 사용되지 않으며 아무런 효과가 없습니다 (기본값: False) |
| `geometry_quality` | COMBO | 아니요 | `"standard"`<br />`"detailed"` | 지오메트리 생성의 품질 수준 (기본값: "standard") |

**참고:** 앞면 이미지(`image`)는 항상 필수입니다. 다중 시점 처리를 위해서는 적어도 하나의 추가 시점 이미지(`image_left`, `image_back` 또는 `image_right`)가 제공되어야 합니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `model_file` | STRING | 생성된 3D 모델의 파일 경로 또는 식별자 (후방 호환성을 위해만 사용) |
| `model task_id` | MODEL_TASK_ID | 모델 생성 과정을 추적하는 작업 식별자 |
| `GLB` | FILE3DGLB | GLB 형식으로 생성된 3D 모델 파일 |
