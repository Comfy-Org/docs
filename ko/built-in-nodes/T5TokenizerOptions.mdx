T5TokenizerOptions 노드를 사용하면 다양한 T5 모델 유형에 대한 토크나이저 설정을 구성할 수 있습니다. t5xxl, pile_t5xl, t5base, mt5xl, umt5xxl 등 여러 T5 모델 변형에 대해 최소 패딩과 최소 길이 매개변수를 설정합니다. 이 노드는 CLIP 입력을 받아 지정된 토크나이저 옵션을 적용한 수정된 CLIP을 반환합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `clip` | CLIP | 예 | - | 토크나이저 옵션을 구성할 CLIP 모델 |
| `min_padding` | INT | 아니요 | 0부터 10000까지 | 모든 T5 모델 유형에 설정할 최소 패딩 값 (기본값: 0) |
| `min_length` | INT | 아니요 | 0부터 10000까지 | 모든 T5 모델 유형에 설정할 최소 길이 값 (기본값: 0) |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `output` | CLIP | 토크나이저 옵션이 업데이트된 모든 T5 변형에 적용된 수정된 CLIP 모델 |
