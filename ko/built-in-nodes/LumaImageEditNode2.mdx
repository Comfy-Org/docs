## 개요

이 노드는 Luma UNI-1 모델을 활용해 텍스트 프롬프트를 사용하여 기존 이미지를 편집합니다. 원본 이미지와 원하는 변경 사항에 대한 설명을 입력하면, 이미지의 새 버전을 생성합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `source` | IMAGE | 예 | - | 편집할 원본 이미지. |
| `prompt` | STRING | 예 | 1~6000자 | 원하는 편집 내용에 대한 설명. 기본값: "" (빈 문자열). |
| `model` | MODEL | 예 | `"uni-1"`<br />`"uni-1-max"` | 편집에 사용할 모델. |
| `seed` | INT | 예 | 0~2147483647 | 시드는 노드가 다시 실행될지 여부를 제어하며, 시드와 상관없이 결과는 비결정적입니다. 기본값: 0. |

**매개변수 제약사항:**
- `prompt`는 1자 이상 6000자 이내여야 합니다.
- `model` 매개변수는 동적 콤보로, `"uni-1"` 또는 `"uni-1-max"`로 설정하면 추가 하위 매개변수가 제공됩니다(예: `style`, `web_search`, `image_ref`). `image_ref` 하위 매개변수는 최대 8개의 이미지 참조를 허용합니다.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `image` | IMAGE | Luma UNI-1 모델이 생성한 편집된 이미지. |
