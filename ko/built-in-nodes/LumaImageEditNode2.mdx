이 노드는 Luma UNI-1 모델을 기반으로 텍스트 프롬프트를 사용하여 기존 이미지를 편집합니다. 소스 이미지와 원하는 변경 사항에 대한 설명을 입력받아 편집된 새 이미지 버전을 생성합니다.

# 입력

| 매개변수 | 설명 | 데이터 타입 | 필수 여부 | 범위 |
| --- | --- | --- | --- | --- |
| `source` | 편집할 소스 이미지입니다. | IMAGE | 예 | - |
| `prompt` | 원하는 편집 내용에 대한 설명입니다. 기본값: "" (빈 문자열). | STRING | 예 | 1–6000자 |
| `model` | 편집에 사용할 모델입니다. | MODEL | 예 | `"uni-1"`<br />`"uni-1-max"` |
| `seed` | 시드는 노드 재실행 여부를 제어합니다. 시드와 관계없이 결과는 비결정적입니다. 기본값: 0. | INT | 예 | 0 ~ 2147483647 |

**매개변수 제약 조건:**
- `prompt`는 1자에서 6000자 사이여야 합니다.
- `model` 매개변수는 동적 콤보 상자로, `"uni-1"` 또는 `"uni-1-max"`로 설정 시 추가 하위 매개변수(`style`, `web_search`, `image_ref`)를 제공합니다. `image_ref` 하위 매개변수는 최대 8개의 이미지 참조를 허용합니다.

# 출력

| 출력 이름 | 설명 | 데이터 타입 |
| --- | --- | --- |
| `image` | Luma UNI-1 모델이 생성한 편집된 이미지입니다. | IMAGE |
