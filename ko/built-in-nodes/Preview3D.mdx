Preview3D 노드는 주로 3D 모델 출력물을 미리 보는 데 사용됩니다. 이 노드는 두 가지 입력을 받습니다: 하나는 Load3D 노드에서 가져온 `camera_info`이고, 다른 하나는 3D 모델 파일의 경로입니다. 모델 파일 경로는 반드시 `ComfyUI/output` 폴더에 있어야 합니다.

**지원 형식**
현재 이 노드는 `.gltf`, `.glb`, `.obj`, `.fbx`, `.stl`를 포함한 여러 3D 파일 형식을 지원합니다.

**3D 노드 환경 설정**
ComfyUI의 설정 메뉴에서 일부 관련 3D 노드 환경 설정을 구성할 수 있습니다. 해당 설정에 대해서는 다음 문서를 참고하세요:
[설정 메뉴](https://docs.comfy.org/interface/settings/3d)

## 입력

| 매개변수 이름 | 유형           | 설명                                  |
| -------------- | -------------- | -------------------------------------------- |
| camera_info    | LOAD3D_CAMERA  | 카메라 정보                           |
| model_file     | LOAD3D_CAMERA  | `ComfyUI/output/` 아래의 모델 파일 경로      |

## 캔버스 영역 설명

현재 ComfyUI 프론트엔드의 3D 관련 노드들은 동일한 캔버스 컴포넌트를 공유하므로, 일부 기능적 차이를 제외하고 기본적인 작동 방식은 대부분 일치합니다.

> 다음 내용과 인터페이스는 주로 Load3D 노드를 기반으로 합니다. 구체적인 기능은 실제 노드 인터페이스를 참조해 주세요.

캔버스 영역에는 다음과 같은 다양한 뷰 조작 기능이 포함되어 있습니다:

- 미리 보기 뷰 설정 (격자, 배경색, 미리 보기 뷰)
- 카메라 제어: FOV, 카메라 유형
- 전역 조명 강도: 조명 조정
- 모델 내보내기: `GLB`, `OBJ`, `STL` 형식 지원
- 등등

뷰 제어 조작:

- 왼쪽 클릭 + 드래그: 뷰 회전
- 오른쪽 클릭 + 드래그: 뷰 이동
- 가운데 휠 스크롤 또는 가운데 클릭 + 드래그: 확대/축소
- 좌표축: 뷰 전환

### 2. 왼쪽 메뉴 기능

![메뉴](https://raw.githubusercontent.com/Comfy-Org/embedded-docs/refs/heads/main/comfyui_embedded_docs/docs/Load3d/asset/menu.webp)

미리 보기 영역에서는 일부 뷰 조작 메뉴가 메뉴에 숨겨져 있습니다. 메뉴 버튼을 클릭하면 다양한 메뉴를 펼칠 수 있습니다.

- 1. 장면: 미리 보기 창 격자, 배경색, 썸네일 설정 포함
- 2. 모델: 모델 렌더링 모드, 텍스처 재질, 상향 방향 설정
- 3. 카메라: 정사영과 원근감 뷰 전환, 원근각 설정
- 4. 라이트: 장면 전역 조명 강도
- 5. 내보내기: 모델을 다른 형식(GLB, OBJ, STL)으로 내보내기

#### 장면

![장면 메뉴](https://raw.githubusercontent.com/Comfy-Org/embedded-docs/refs/heads/main/comfyui_embedded_docs/docs/Load3d/asset/menu_scene.webp)

장면 메뉴는 몇 가지 기본적인 장면 설정 기능을 제공합니다:

1. 격자 표시/숨기기
2. 배경색 설정
3. 배경 이미지 업로드 클릭
4. 미리 보기 썸네일 숨기기

#### 모델

![메뉴_장면](https://raw.githubusercontent.com/Comfy-Org/embedded-docs/refs/heads/main/comfyui_embedded_docs/docs/Load3d/asset/menu_model.webp)

모델 메뉴는 모델 관련 기능을 제공합니다:

1. **상향 방향**: 모델의 상향 방향을 결정하는 축
2. **재질 모드**: 모델 렌더링 모드 전환 - 원본, 일반, 와이어프레임, 라인아트

#### 카메라

![메뉴_모델메뉴_카메라](https://raw.githubusercontent.com/Comfy-Org/embedded-docs/refs/heads/main/comfyui_embedded_docs/docs/Load3d/asset/menu_camera.webp)

이 메뉴는 정사영과 원근감 뷰 간 전환 및 원근각 크기 설정을 제공합니다:

1. **카메라**: 정사영과 원근감 뷰 간 빠르게 전환
2. **FOV**: FOV 각도 조정

#### 라이트

![메뉴_모델메뉴_카메라](https://raw.githubusercontent.com/Comfy-Org/embedded-docs/refs/heads/main/comfyui_embedded_docs/docs/Load3d/asset/menu_light.webp)

이 메뉴를 통해 장면의 전역 조명 강도를 빠르게 조정할 수 있습니다

#### 내보내기

![메뉴_내보내기](https://raw.githubusercontent.com/Comfy-Org/embedded-docs/refs/heads/main/comfyui_embedded_docs/docs/Load3d/asset/menu_export.webp)

이 메뉴는 모델 형식을 빠르게 변환하고 내보낼 수 있는 기능을 제공합니다
