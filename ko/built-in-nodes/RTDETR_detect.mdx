RT-DETR Detect 노드는 RT-DETR 모델을 사용해 입력 이미지에서 객체 검출을 수행합니다. 객체를 식별하고 그 주위에 경계 상자를 그려주며, COCO 데이터셋의 클래스에 따라 레이블을 붙입니다. 결과를 신뢰도 점수와 객체 클래스별로 필터링하고, 총 검출 수를 제한할 수 있습니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `model` | MODEL | 예 | N/A | 객체 검출에 사용되는 RT-DETR 모델입니다. |
| `image` | IMAGE | 예 | N/A | 객체를 검출할 입력 이미지입니다. 노드는 최대 32개씩 묶어서 이미지를 처리합니다. |
| `threshold` | FLOAT | 아니요 | N/A | 결과에 포함될 최소 신뢰도 점수입니다(기본값: 0.5). |

| `max_detections` | INT | 아니요 | N/A | 각 이미지당 반환할 최대 검출 수입니다. 신뢰도 점수가 높은 순서대로 정렬됩니다(기본값: 100). |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `bboxes` | BOUNDINGBOX | 각 입력 이미지에 대한 경계 상자 목록입니다. 각 상자는 좌표(x, y, 너비, 높이), 클래스 레이블, 신뢰도 점수를 포함합니다. |
