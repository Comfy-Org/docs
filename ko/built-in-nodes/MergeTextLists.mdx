이 노드는 여러 텍스트 리스트를 하나의 통합된 리스트로 병합합니다. 텍스트 입력을 리스트 형태로 받으며 이를 연결하여 하나로 합칩니다. 노드는 병합된 리스트에 포함된 총 텍스트 수를 로그로 기록합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `texts` | STRING | 예 | N/A | 병합할 텍스트 리스트입니다. 여러 리스트를 입력에 연결할 수 있으며, 이들은 하나로 연결됩니다. |

**참고:** 이 노드는 그룹 처리로 설정되어 있습니다(`is_group_process = True`), 즉, 메인 처리 함수가 실행되기 전에 자동으로 여러 리스트 입력을 연결하여 처리합니다.

**참고:** 이 노드는 더 이상 사용되지 않으며, 대신 Create List 노드로 대체되었습니다.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `texts` | STRING | 모든 입력 텍스트를 포함한 단일 병합된 리스트입니다. |
