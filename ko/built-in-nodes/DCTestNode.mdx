DCTestNode는 동적 콤보 박스에서 사용자가 선택한 내용에 따라 다양한 유형의 데이터를 반환하는 논리 노드입니다. 이 노드는 조건부 라우터 역할을 하며, 선택된 옵션에 따라 활성화되는 입력 필드와 노드가 출력하는 값의 유형이 결정됩니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `combo` | COMBO | 예 | `"option1"`<br />`"option2"`<br />`"option3"`<br />`"option4"` | 주요 선택으로, 어떤 입력 필드가 활성화되고 노드가 어떤 값을 출력할지 결정합니다. |
| `string` | STRING | 아니요 | - | 텍스트 입력 필드입니다. 이 필드는 `combo`가 `"option1"`로 설정된 경우에만 활성화되며 필수입니다. |
| `integer` | INT | 아니요 | - | 정수 입력 필드입니다. 이 필드는 `combo`가 `"option2"`로 설정된 경우에만 활성화되며 필수입니다. |
| `image` | IMAGE | 아니요 | - | 이미지 입력 필드입니다. 이 필드는 `combo`가 `"option3"`로 설정된 경우에만 활성화되며 필수입니다. |
| `subcombo` | COMBO | 아니요 | `"opt1"`<br />`"opt2"` | `combo`가 `"option4"`로 설정된 경우 나타나는 보조 선택입니다. 중첩된 입력 필드들이 활성화되는지를 결정합니다. |
| `float_x` | FLOAT | 아니요 | - | 소수점 숫자 입력입니다. 이 필드는 `combo`가 `"option4"`로 설정되고 `subcombo`가 `"opt1"`로 설정된 경우에만 활성화되며 필수입니다. |
| `float_y` | FLOAT | 아니요 | - | 소수점 숫자 입력입니다. 이 필드는 `combo`가 `"option4"`로 설정되고 `subcombo`가 `"opt1"`로 설정된 경우에만 활성화되며 필수입니다. |
| `mask1` | MASK | 아니요 | - | 마스크 입력 필드입니다. 이 필드는 `combo`가 `"option4"`로 설정되고 `subcombo`가 `"opt2"`로 설정된 경우에만 활성화됩니다. 선택사항입니다. |

**매개변수 제약사항:**

* `combo` 매개변수는 다른 모든 입력 필드의 표시 여부와 필수성을 제어합니다. 선택된 `combo` 옵션과 관련된 입력만 표시되며 필수입니다(단, `mask1`는 선택사항임).
* `combo`가 `"option4"`로 설정되면 `subcombo` 매개변수가 필수로 바뀌며, 두 번째 중첩 입력(`float_x`/`float_y` 또는 `mask1`)을 제어합니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `output` | ANYTYPE | 출력은 선택된 `combo` 옵션에 따라 달라집니다. 문자열(`"option1"`), 정수(`"option2"`), 이미지(`"option3"`), 또는 `subcombo` 딕셔너리의 문자열 표현(`"option4"`)일 수 있습니다. |
