Vidu 시작 끝에서 비디오 생성 노드는 시작 프레임과 끝 프레임 사이의 프레임을 생성하여 비디오를 만듭니다. 이 노드는 텍스트 프롬프트를 사용해 비디오 생성 과정을 안내하며, 다양한 해상도와 움직임 설정을 가진 여러 비디오 모델을 지원합니다. 노드는 처리 전에 시작 프레임과 끝 프레임의 종횡비가 호환 가능하도록 검증합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `model` | COMBO | 예 | `"viduq1"` | 모델 이름 |
| `first_frame` | IMAGE | 예 | - | 시작 프레임 |
| `end_frame` | IMAGE | 예 | - | 끝 프레임 |
| `prompt` | STRING | 아니요 | - | 비디오 생성을 위한 텍스트 설명 |
| `duration` | INT | 아니요 | 5-5 | 출력 비디오의 지속 시간(초)(기본값: 5, 고정: 5초) |
| `seed` | INT | 아니요 | 0-2147483647 | 비디오 생성을 위한 시드(랜덤일 경우 0)(기본값: 0) |
| `resolution` | COMBO | 아니요 | `"1080p"` | 지원되는 값은 모델 및 지속 시간에 따라 다를 수 있음(기본값: "1080p") |
| `movement_amplitude` | COMBO | 아니요 | `"auto"`<br />`"small"`<br />`"medium"`<br />`"large"` | 프레임 내 객체의 움직임 진폭(기본값: "auto") |

**참고:** 시작 프레임과 끝 프레임은 호환 가능한 종횡비를 가져야 합니다(최소 rel=0.8, 최대 rel=1.25의 비율 허용 오차로 검증됨).

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `output` | VIDEO | 생성된 비디오 파일 |
