Merge Image Lists 노드는 여러 개별 이미지 목록을 하나의 연속된 목록으로 합칩니다. 이 노드는 연결된 각 입력에서 모든 이미지를 받아들여 수신된 순서대로 차례로 연결합니다. 이는 서로 다른 소스의 이미지를 정리하거나 일괄 처리하여 추가 처리를 위해 활용할 때 유용합니다.

**참고:** 이 노드는 더 이상 사용되지 않으며, Create List 노드로 대체되었습니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `images` | IMAGE | 예 | - | 병합할 이미지 목록입니다. 이 입력은 여러 연결을 받을 수 있으며, 각 연결된 목록은 최종 출력에 연결됩니다. |

**참고:** 이 노드는 여러 입력을 받도록 설계되었습니다. 여러 이미지 목록을 단일 `images` 입력 소켓에 연결할 수 있습니다. 노드는 연결된 모든 목록의 이미지를 자동으로 하나의 출력 목록으로 연결합니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `images` | IMAGE | 연결된 모든 입력 목록의 이미지를 포함한 단일 병합된 목록입니다. |
