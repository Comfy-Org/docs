TrainLoraNode는 제공된 래텐트와 컨디셔닝 데이터를 사용하여 확산 모델에서 LoRA(저랭크 적응) 모델을 생성하고 학습합니다. 사용자 정의 학습 파라미터, 옵티마이저 및 손실 함수를 통해 모델을 세밀하게 조정할 수 있습니다. 이 노드는 학습된 LoRA 가중치, 손실 히스토리 맵 및 완료된 총 학습 단계를 출력합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `model` | MODEL | 예 | - | LoRA를 학습할 모델입니다. |
| `latents` | LATENT | 예 | - | 학습에 사용할 래텐트로, 모델의 데이터셋/입력으로 작용합니다. |
| `positive` | CONDITIONING | 예 | - | 학습에 사용할 양성 컨디셔닝입니다. |
| `batch_size` | INT | 예 | 1-10000 | 학습에 사용할 배치 크기입니다(기본값: 1). |
| `grad_accumulation_steps` | INT | 예 | 1-1024 | 학습에 사용할 그래디언트 누적 단계 수입니다(기본값: 1). |
| `steps` | INT | 예 | 1-100000 | LoRA를 학습할 단계 수입니다(기본값: 16). |
| `learning_rate` | FLOAT | 예 | 0.0000001-1.0 | 학습에 사용할 학습률입니다(기본값: 0.0005). |
| `rank` | INT | 예 | 1-128 | LoRA 레이어의 랭크입니다(기본값: 8). |
| `optimizer` | COMBO | 예 | "AdamW"<br />"Adam"<br />"SGD"<br />"RMSprop" | 학습에 사용할 옵티마이저입니다(기본값: "AdamW"). |
| `loss_function` | COMBO | 예 | "MSE"<br />"L1"<br />"Huber"<br />"SmoothL1" | 학습에 사용할 손실 함수입니다(기본값: "MSE"). |
| `seed` | INT | 예 | 0-18446744073709551615 | 학습에 사용할 시드입니다(LoRA 가중치 초기화 및 노이즈 샘플링을 위한 생성기에 사용됨)(기본값: 0). |
| `training_dtype` | COMBO | 예 | "bf16"<br />"fp32"<br />"none" | 학습에 사용할 dtype입니다. 'none'은 모델의 기본 컴퓨팅 dtype을 유지하며 덮어쓰지 않습니다. fp16 모델의 경우 GradScaler가 자동으로 활성화됩니다(기본값: "bf16"). |
| `lora_dtype` | COMBO | 예 | "bf16"<br />"fp32" | LoRA에 사용할 dtype입니다(기본값: "bf16"). |
| `quantized_backward` | BOOLEAN | 예 | - | training_dtype을 'none'으로 설정하고 양자화된 모델에서 학습할 때, 활성화되면 양자화된 행렬 곱셈을 통해 역전파를 수행합니다(기본값: False). |
| `algorithm` | COMBO | 예 | 여러 옵션 가능 | 학습에 사용할 알고리즘입니다. |
| `gradient_checkpointing` | BOOLEAN | 예 | - | 학습에 그래디언트 체크포인트를 사용합니다(기본값: True). |
| `checkpoint_depth` | INT | 예 | 1-5 | 그래디언트 체크포인트의 깊이 수준입니다(기본값: 1). |
| `offloading` | BOOLEAN | 예 | - | 학습 중 GPU 메모리를 절약하기 위해 모델 가중치를 CPU로 오프로드합니다(기본값: False). |
| `existing_lora` | COMBO | 예 | 여러 옵션 가능 | 추가할 기존 LoRA입니다. 새 LoRA를 만들려면 None으로 설정하세요(기본값: "[None]"). |
| `bucket_mode` | BOOLEAN | 예 | - | 해상도 버킷 모드를 활성화합니다. 활성화되면 ResolutionBucket 노드에서 사전 버킷화된 래텐트를 기대합니다(기본값: False). |
| `bypass_mode` | BOOLEAN | 예 | - | 학습에 바이패스 모드를 활성화합니다. 활성화되면 어댑터가 가중치 수정 대신 포워드 후크를 통해 적용됩니다. 양자화된 모델에서 직접 가중치를 수정할 수 없는 경우 유용합니다(기본값: False). |

**참고:** 양성 컨디셔닝 입력의 수는 래텐트 이미지의 수와 일치해야 합니다. 이미지가 여러 개인데 양성 컨디셔닝이 하나만 제공되면 모든 이미지에 대해 자동으로 반복됩니다.

**training_dtype에 대한 참고사항:** 'none'으로 설정하면 모델의 기본 컴퓨팅 dtype이 유지됩니다. fp16 모델의 경우 GradScaler가 자동으로 활성화되어 그래디언트 계산 중 언더플로우를 방지합니다. 또한 `fp16_accumulation`이 활성화된 경우(–fast 플래그를 통해), 이 조합은 수치적으로 불안정해 NaN 값이 발생할 수 있습니다.

**quantized_backward에 대한 참고사항:** 이 매개변수는 training_dtype을 'none'으로 설정하고 모델이 양자화된 모델일 때만 관련이 있습니다. 역전파 과정에서 양자화된 행렬 곱셈을 활성화합니다.

**bypass_mode에 대한 참고사항:** 활성화되면 어댑터가 모델 가중치를 직접 수정하는 대신 포워드 후크를 통해 적용됩니다. 이는 특히 가중치를 직접 수정할 수 없는 양자화된 모델에서 유용합니다.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `lora` | LORA_MODEL | 저장하거나 다른 모델에 적용할 수 있는 학습된 LoRA 가중치입니다. |
| `loss_map` | LOSS_MAP | 시간에 따른 학습 손실 값을 포함한 딕셔너리입니다. |
| `steps` | INT | 완료된 총 학습 단계 수입니다(기존 LoRA의 이전 단계 포함). |
