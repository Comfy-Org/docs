CropMask 노드는 주어진 마스크에서 지정된 영역을 자르기 위해 설계되었습니다. 사용자는 좌표와 크기를 지정하여 관심 영역을 정의할 수 있으며, 이를 통해 마스크의 일부를 추출해 추가 처리나 분석에 활용할 수 있습니다.

## 입력

| 매개변수 | 데이터 타입 | 설명 |
|-----------|-------------|-------------|
| `mask`    | MASK        | 마스크 입력은 자를 마스크 이미지를 나타냅니다. 지정된 좌표와 크기에 따라 추출할 영역을 정의하는 데 필수적입니다. |
| `x`       | INT         | x 좌표는 가로축에서 자를 시작점의 위치를 지정합니다. |
| `y`       | INT         | y 좌표는 세로축에서 자를 시작점의 위치를 결정합니다. |
| `width`   | INT         | 너비는 시작점에서부터 자를 영역의 가로 범위를 정의합니다. |
| `height`  | INT         | 높이는 시작점에서부터 자를 영역의 세로 범위를 지정합니다. |

## 출력

| 매개변수 | 데이터 타입 | 설명 |
|-----------|-------------|-------------|
| `mask`    | MASK        | 출력은 자른 마스크로, 지정된 좌표와 크기에 따라 원본 마스크의 일부를 잘라낸 것입니다. |
