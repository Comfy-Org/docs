이 노드는 Google의 Gemini Omni Flash 모델을 사용하여 텍스트 프롬프트로부터 오디오가 포함된 비디오를 생성합니다. 선택적으로 참조 이미지나 비디오를 제공하여 결과를 안내하거나 편집할 수 있습니다.

## 입력

| 매개변수 | 설명 | 데이터 타입 | 필수 여부 | 범위 |
|-----------|-------------|-----------|----------|-------|
| `model` | 비디오 생성에 사용되는 Gemini 비디오 모델입니다. | COMBO | 예 | `"Omni Flash"` |
| `seed` | 시드는 노드 재실행 여부를 제어합니다. 시드와 관계없이 결과는 비결정적입니다. (기본값: 42) | INT | 예 | 0 ~ 2147483647 |

**참고:** `model` 매개변수는 동적 콤보로, "Omni Flash"가 선택되면 추가 입력 항목이 확장되어 표시됩니다. 이러한 추가 입력 항목은 다음과 같습니다:
- `prompt` (STRING, 필수): 원하는 비디오를 설명하는 텍스트 프롬프트입니다. 프롬프트에 원하는 길이(3~10초)와 화면 비율(16:9 또는 9:16)을 직접 명시하십시오.
- `images` (IMAGE, 선택 사항): 비디오 생성을 안내하는 참조 이미지입니다. 최대 14개의 이미지를 사용할 수 있습니다.
- `videos` (VIDEO, 선택 사항): 비디오 생성을 안내하는 참조 비디오입니다. 최대 3개의 비디오를 사용할 수 있으며, 각 비디오의 최대 길이는 10초입니다.
- `temperature` (FLOAT, 선택 사항, 기본값: 1.0): 생성 과정의 무작위성을 제어합니다.
- `top_p` (FLOAT, 선택 사항, 기본값: 0.95): 핵심 샘플링을 제어합니다.

## 출력

| 출력 이름 | 설명 | 데이터 타입 |
|-------------|-------------|-----------|
| `video` | 오디오가 포함된 생성된 비디오입니다. | VIDEO |
| `text` | 모델의 텍스트 응답입니다(있는 경우). | STRING |
