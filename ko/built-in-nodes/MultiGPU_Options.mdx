## 개요

이 노드를 사용하면 속도가 다른 여러 그래픽 카드를 사용할 때 각 GPU의 상대적 성능을 지정할 수 있습니다. 이 노드는 GPU 옵션 그룹을 생성하여 장치 간에 작업을 분산하는 데 사용할 수 있지만, 현재 버전에서는 실제 속도 기반 작업 분산 기능은 아직 구현되지 않았습니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `device_index` | INT | 예 | 0부터 64까지 | 구성할 GPU 장치의 인덱스 번호 (기본값: 0) |
| `relative_speed` | FLOAT | 예 | 0.0부터 무한대까지 | 다른 GPU와 비교한 이 GPU의 상대적 속도로, 작업 분산에 사용됨 (기본값: 1.0, 단계: 0.01) |
| `gpu_options` | GPU_OPTIONS | 아니요 | - | 이 장치의 옵션을 추가할 기존 GPU 옵션 그룹. 제공되지 않으면 새 그룹이 생성됨 |

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `GPU_OPTIONS` | GPU_OPTIONS | 구성된 장치 설정을 포함한 GPU 옵션 그룹으로, 다중 GPU 작업을 위해 다른 노드에 전달될 수 있음 |

**참고:** `relative_speed` 매개변수는 정의되어 있지만, 현재 내부 스케줄러에서는 GPU 간 작업 분산에 아직 사용되지 않습니다. 현재 구현에서는 모든 장치의 상대적 속도와 관계없이 작업이 균등하게 분산됩니다.
