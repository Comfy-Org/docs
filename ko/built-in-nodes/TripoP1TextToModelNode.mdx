## 개요

이 노드는 Tripo P1 API를 사용하여 텍스트 설명을 기반으로 3D 모델을 생성합니다. 안정적인 토폴로지와 함께 저폴리고니 게임용 메시를 만드는 데 최적화되어 실시간 애플리케이션에 적합합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `prompt` | 문자열 | 예 | 최대 1024자 | 생성하고자 하는 3D 모델의 텍스트 설명입니다. |
| `negative_prompt` | 문자열 | 아니요 | 최대 255자 | 생성된 모델에 포함되지 않기를 원하는 내용을 설명한 텍스트입니다. |
| `output_mode` | 딕셔너리 | 예 | 설명 참조 | 출력 모델의 품질과 텍스처 설정을 제어합니다. 이 매개변수는 다음과 같은 키를 가진 딕셔너리입니다:<br /><br />`texture_quality`: 문자열, 범위: `"standard"`<br />`pbr`: 부울, 기본값: True<br />`texture`: 부울, 기본값: True<br />`subdivision`: 정수, 기본값: 0, 범위: 0부터 2까지<br />`texture_size`: 정수, 기본값: 2048, 범위: 512부터 4096까지 (2의 거듭제곱이어야 함)<br />`texture_format`: 문자열, 범위: `"png"`<br />`texture_clean`: 부울, 기본값: False<br />`texture_seamless`: 부울, 기본값: False<br /><br />기본값: `{"texture_quality": "standard", "pbr": True, "texture": True, "subdivision": 0, "texture_size": 2048, "texture_format": "png", "texture_clean": False, "texture_seamless": False}` |
| `image_seed` | 정수 | 아니요 |  | 이미지 생성을 위한 시드 값으로, 난수를 제어하는 데 사용됩니다. 기본값: 42. |
| `face_limit` | 정수 | 아니요 |  | 생성된 메시의 최대 면 수입니다. -1은 제한이 없음을 의미합니다. 기본값: -1. |
| `model_seed` | 정수 | 아니요 |  | 모델 생성을 위한 시드 값으로, 난수를 제어하는 데 사용됩니다. |
| `auto_size` | 부울 | 아니요 |  | 활성화되면 노드가 최적의 모델 크기를 자동으로 결정합니다. 기본값: False. |
| `export_uv` | 부울 | 아니요 |  | 활성화되면 모델에 텍스처 맵핑을 위한 UV 좌표가 포함됩니다. 기본값: True. |
| `compress_geometry` | 부울 | 아니요 |  | 활성화되면 지오메트리가 압축되어 파일 크기를 줄입니다. 기본값: False. |

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `model_file` | 문자열 | 생성된 3D 모델의 파일 경로(백ward compatibility용만). |
| `model task_id` | MODEL_TASK_ID | 모델 생성 요청의 고유한 작업 ID. |
| `GLB` | FILE3DGLB | GLB 형식으로 생성된 3D 모델. |
