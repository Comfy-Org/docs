EmptyLTXVLatentVideo 노드는 비디오 처리를 위한 빈 잠재 텐서를 생성합니다. 지정된 크기를 가진 빈 시작점을 생성하며, 이는 비디오 생성 워크플로우의 입력으로 사용될 수 있습니다. 이 노드는 구성된 너비, 높이, 길이 및 배치 크기를 가진 0으로 채워진 잠재 표현을 생성합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `width` | INT | 예 | 64부터 MAX_RESOLUTION까지 | 잠재 비디오 텐서의 너비 (기본값: 768, 단계: 32) |
| `height` | INT | 예 | 64부터 MAX_RESOLUTION까지 | 잠재 비디오 텐서의 높이 (기본값: 512, 단계: 32) |
| `length` | INT | 예 | 1부터 MAX_RESOLUTION까지 | 잠재 비디오의 프레임 수 (기본값: 97, 단계: 8) |
| `batch_size` | INT | 아니요 | 1부터 4096까지 | 한 번에 생성할 잠재 비디오의 수 (기본값: 1) |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `samples` | LATENT | 지정된 크기와 값으로 0으로 채워진 빈 잠재 텐서 |
