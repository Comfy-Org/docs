Random Crop Images 노드는 각 입력 이미지에서 임의의 직사각형 부분을 선택한 후, 지정된 너비와 높이로 자릅니다. 이는 일반적으로 학습용 이미지의 변형을 만들기 위한 데이터 증강에 사용됩니다. 자르기의 임의 위치는 시드 값에 의해 결정되며, 이를 통해 동일한 자르기를 재현할 수 있습니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `image` | IMAGE | 예 | - | 자를 이미지입니다. |
| `width` | INT | 아니요 | 1 - 8192 | 자르기 영역의 너비 (기본값: 512). |
| `height` | INT | 아니요 | 1 - 8192 | 자르기 영역의 높이 (기본값: 512). |
| `seed` | INT | 아니요 | 0 - 18446744073709551615 | 자르기의 임의 위치를 제어하는 데 사용되는 숫자 (기본값: 0). |

**참고:** `width`와 `height` 매개변수는 반드시 입력 이미지의 크기보다 작거나 같아야 합니다. 지정된 크기가 이미지보다 클 경우, 자르기는 이미지 경계 내로 제한됩니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `image` | IMAGE | 임의의 자르기가 적용된 결과 이미지입니다. |
