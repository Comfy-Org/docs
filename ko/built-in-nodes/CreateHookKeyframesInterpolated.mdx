시작점과 끝점 사이에 보간된 강도 값을 가진 후크 키프레임 시퀀스를 생성합니다. 이 노드는 다양한 보간 방법을 사용하여 생성 과정의 특정 백분율 범위 내에서 강도 파라미터가 부드럽게 전환되도록 여러 개의 키프레임을 생성하며, 전환 곡선을 제어합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `strength_start` | FLOAT | 예 | 0.0 - 10.0 | 보간 시퀀스의 시작 강도 값 (기본값: 1.0) |
| `strength_end` | FLOAT | 예 | 0.0 - 10.0 | 보간 시퀀스의 끝 강도 값 (기본값: 1.0) |
| `interpolation` | COMBO | 예 | `LINEAR`<br />`EASE_IN`<br />`EASE_OUT`<br />`EASE_IN_OUT`<br />`EASE_OUT_IN`<br />`SINE`<br />`CUBIC`<br />`QUARTIC`<br />`QUINTIC`<br />`EXPO`<br />`CIRC`<br />`BACK`<br />`BOUNCE`<br />`ELASTIC` | 강도 값 간 전환에 사용되는 보간 방법 (기본값: LINEAR) |
| `start_percent` | FLOAT | 예 | 0.0 - 1.0 | 생성 과정에서의 시작 백분율 위치 (기본값: 0.0) |
| `end_percent` | FLOAT | 예 | 0.0 - 1.0 | 생성 과정에서의 끝 백분율 위치 (기본값: 1.0) |
| `keyframes_count` | INT | 예 | 2 - 100 | 보간 시퀀스에서 생성할 키프레임의 수 (기본값: 5) |
| `print_keyframes` | BOOLEAN | 예 | True/False | 생성된 키프레임 정보를 로그에 출력할지 여부 (기본값: False) |
| `prev_hook_kf` | HOOK_KEYFRAMES | 아니요 | - | 추가할 수 있는 선택적 이전 후크 키프레임 그룹 |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `HOOK_KF` | HOOK_KEYFRAMES | 보간된 시퀀스를 포함한 생성된 후크 키프레임 그룹 |
