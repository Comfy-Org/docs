SamplerCustomAdvanced 노드는 맞춤형 노이즈, 가이던스 및 샘플링 구성으로 고급 잠재 공간 샘플링을 수행합니다. 이 노드는 맞춤형 노이즈 생성과 시그마 스케줄을 통해 안내된 샘플링 과정을 거쳐 잠재 이미지를 처리하며, 최종 샘플링 결과와 가능하다면 노이즈가 제거된 버전을 동시에 출력합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `noise` | NOISE | 예 | - | 샘플링 과정의 초기 노이즈 패턴과 시드를 제공하는 노이즈 생성기 |
| `guider` | GUIDER | 예 | - | 샘플링 과정을 원하는 출력으로 유도하는 가이던스 모델 |
| `sampler` | SAMPLER | 예 | - | 생성 중 잠재 공간을 어떻게 탐색할지 정의하는 샘플링 알고리즘 |
| `sigmas` | SIGMAS | 예 | - | 샘플링 단계 동안 노이즈 수준을 제어하는 시그마 스케줄 |
| `latent_image` | LATENT | 예 | - | 샘플링의 시작점이 되는 초기 잠재 표현. 선택적 `noise_mask`를 통해 선택적 노이즈 제거를 지원하며, 고급 잠재 처리를 위해 선택적 `downscale_ratio_spacial` 및 `downscale_ratio_temporal` 키도 지원 |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `output` | LATENT | 샘플링 과정을 마친 후의 최종 샘플링된 잠재 표현. 입력 잠재에서 제공된 `downscale_ratio_spacial` 또는 `downscale_ratio_temporal` 키는 이 출력에서 제거됩니다 |
| `denoised_output` | LATENT | 샘플링 과정에서 중간 청결한 예측(x0)을 생성한 경우, 이를 노이즈가 제거된 버전으로 출력합니다. 그렇지 않으면 출력과 동일하게 반환됩니다. 가능하다면 이 출력은 각 단계에서 모델이 추정한 가장 깨끗한 잠재 표현을 나타냅니다 |
