> 이 문서는 AI로 생성되었습니다. 오류를 발견하거나 개선할 점이 있다면 언제든지 기여해 주세요! [GitHub에서 편집하기](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/ByteDanceImageReferenceNode/en.md)

ByteDance 이미지 참조 노드는 텍스트 프롬프트와 하나에서 네 장의 참조 이미지를 사용해 동영상을 생성합니다. 이 노드는 이미지와 프롬프트를 외부 API 서비스로 전송하여, 참조 이미지의 시각적 스타일과 내용을 반영하면서 설명에 맞는 동영상을 만들어냅니다. 이 노드는 동영상 해상도, 종횡비, 지속 시간 및 기타 생성 매개변수를 조정하는 다양한 컨트롤을 제공합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `model` | 문자열 | 예 | `"seedance-1-0-pro-250528"`<br />`"seedance-1-0-lite-i2v-250428"` | 동영상 생성에 사용할 AI 모델 (기본값: `"seedance-1-0-lite-i2v-250428"`). |
| `prompt` | 문자열 | 예 | - | 동영상을 생성하는 데 사용되는 텍스트 프롬프트입니다. |
| `images` | 이미지 | 예 | - | 하나에서 네 장의 이미지입니다. 각 이미지는 300x300에서 6000x6000 픽셀 사이여야 하며, 종횡비는 0.4에서 2.5 사이여야 합니다. |
| `resolution` | 문자열 | 예 | `"480p"`<br />`"720p"` | 출력 동영상의 해상도입니다. |
| `aspect_ratio` | 문자열 | 예 | `"adaptive"`<br />`"16:9"`<br />`"4:3"`<br />`"1:1"`<br />`"3:4"`<br />`"9:16"`<br />`"21:9"` | 출력 동영상의 종횡비 (기본값: `"adaptive"`). |
| `duration` | 정수 | 예 | 3~12 | 출력 동영상의 지속 시간(초)입니다 (기본값: 5). |
| `seed` | 정수 | 아니요 | 0~2147483647 | 생성에 사용할 시드입니다 (기본값: 0). |
| `watermark` | 불리언 | 아니요 | - | 동영상에 "AI 생성" 워터마크를 추가할지 여부 (기본값: False). |

**참고:** 프롬프트 텍스트에는 다음 매개변수 문자열이 포함되어서는 안 됩니다: `--resolution`, `--ratio`, `--duration`, `--seed`, 또는 `--watermark`. 이러한 값은 전용 입력 위젯을 통해만 제어됩니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-----------|-----------|-------------|
| `output` | 동영상 | 입력 프롬프트와 참조 이미지를 기반으로 생성된 동영상 파일입니다. |
