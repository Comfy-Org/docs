이 노드는 후크 기반 시스템을 사용하여 기본 컨디셔닝 입력과 디폴트 컨디셔닝 입력을 결합합니다. 두 컨디셔닝 소스를 하나의 출력으로 병합하며, 기본 컨디셔닝은 주 컨디셔닝이 불완전한 경우 백업 또는 기본값 역할을 수행하도록 합니다.

## 입력

| 매개변수 | 데이터 유형 | 입력 유형 | 기본값 | 범위 | 설명 |
|-----------|-----------|------------|---------|-------|-------------|
| `cond` | CONDITIONING | 필수 | - | - | 처리하고 결합할 주 컨디셔닝 입력 |
| `cond_DEFAULT` | CONDITIONING | 필수 | - | - | 주 컨디셔닝과 결합할 디폴트 컨디셔닝 데이터 |
| `hooks` | HOOKS | 선택적 | - | - | 컨디셔닝 데이터의 처리 및 결합 방식을 제어하는 선택적 후크 구성 |

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `CONDITIONING` | CONDITIONING | 주 컨디셔닝과 디폴트 컨디셔닝 입력을 병합한 결과로 생성된 결합된 컨디셔닝 데이터 |
