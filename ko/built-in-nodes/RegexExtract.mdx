RegexExtract 노드는 정규 표현식을 사용해 텍스트에서 패턴을 검색합니다. 첫 번째 일치 항목, 모든 일치 항목, 일치 항목에서 특정 그룹 또는 여러 일치 항목의 모든 그룹을 찾을 수 있습니다. 이 노드는 대소문자 구분, 다중 줄 매칭, dotall 동작 등 다양한 정규 표현식 플래그를 지원합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `string` | STRING | 예 | - | 패턴을 검색할 입력 텍스트 |
| `regex_pattern` | STRING | 예 | - | 검색할 정규 표현식 패턴 |
| `mode` | COMBO | 예 | "첫 번째 일치"<br />"모든 일치"<br />"첫 번째 그룹"<br />"모든 그룹" | 추출 모드는 일치하는 부분 중 어떤 부분을 반환할지 결정합니다(기본값: "첫 번째 일치") |
| `case_insensitive` | BOOLEAN | 아니요 | - | 일치 시 대소문자를 무시할지 여부(기본값: True) |
| `multiline` | BOOLEAN | 아니요 | - | 문자열을 여러 줄로 처리할지 여부(기본값: False) |
| `dotall` | BOOLEAN | 아니요 | - | 점(.)이 줄 바꿈 문자와도 일치할지 여부(기본값: False) |
| `group_index` | INT | 아니요 | 0-100 | 그룹 모드를 사용할 때 추출할 캡처 그룹 인덱스(기본값: 1) |

**참고:** "첫 번째 그룹" 또는 "모든 그룹" 모드를 사용할 경우, `group_index` 매개변수는 어떤 캡처 그룹을 추출할지 지정합니다. 그룹 0은 전체 일치 항목을 나타내며, 그룹 1 이상은 정규 표현식 패턴의 번호가 붙은 캡처 그룹을 나타냅니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `output` | STRING | 선택한 모드와 매개변수에 따라 추출된 텍스트 |
