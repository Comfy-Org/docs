텍스트 저장 노드는 텍스트 콘텐츠를 출력 디렉터리의 파일에 기록합니다. 일반 텍스트(.txt), 마크다운(.md), JSON(.json)의 세 가지 파일 형식으로 저장을 지원합니다. JSON으로 저장할 경우, 노드는 입력 텍스트를 유효한 JSON으로 구문 분석하고 적절한 들여쓰기로 서식을 지정합니다.

## 입력

| 매개변수 | 설명 | 데이터 타입 | 필수 여부 | 범위 |
|-----------|-------------|-----------|----------|-------|
| `text` | 파일에 저장할 텍스트 콘텐츠 | STRING | 예 | |
| `filename_prefix` | 출력 파일 이름의 접두사(기본값: "ComfyUI") | STRING | 아니요 | |
| `format` | 텍스트를 저장할 파일 형식(기본값: "txt") | STRING | 아니요 | `"txt"`<br />`"md"`<br />`"json"` |

## 출력

| 출력 이름 | 설명 | 데이터 타입 |
|-------------|-------------|-----------|
| `text` | 파일에 저장된 원본 텍스트 콘텐츠 | STRING |
