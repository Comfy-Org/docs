> 이 문서는 AI로 생성되었습니다. 오류를 발견하거나 개선할 점이 있다면 언제든지 기여해 주세요! [GitHub에서 편집하기](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/WanFirstLastFrameToVideo/en.md)

WanFirstLastFrameToVideo 노드는 시작 프레임과 끝 프레임을 텍스트 프롬프트와 결합하여 비디오 조건부를 생성합니다. 첫 번째 및 마지막 프레임을 인코딩하여 잠재적 표현을 생성하고, 마스크를 적용해 생성 과정을 안내하며, CLIP 비전 특징이 제공되면 이를 포함합니다. 이 노드는 비디오 모델이 지정된 시작점과 끝점 사이에서 일관성 있는 시퀀스를 생성하도록 긍정적 및 부정적 조건부를 준비합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `positive` | CONDITIONING | 예 | - | 비디오 생성을 안내하는 긍정적 텍스트 조건부 |
| `negative` | CONDITIONING | 예 | - | 비디오 생성을 안내하는 부정적 텍스트 조건부 |
| `vae` | VAE | 예 | - | 이미지를 잠재 공간으로 인코딩하는 데 사용되는 VAE 모델 |
| `width` | INT | 예 | 16부터 MAX_RESOLUTION까지 | 출력 비디오 너비 (기본값: 832, 단계: 16) |
| `height` | INT | 예 | 16부터 MAX_RESOLUTION까지 | 출력 비디오 높이 (기본값: 480, 단계: 16) |
| `length` | INT | 예 | 1부터 MAX_RESOLUTION까지 | 비디오 시퀀스의 프레임 수 (기본값: 81, 단계: 4) |
| `batch_size` | INT | 예 | 1부터 4096까지 | 동시에 생성할 비디오 수 (기본값: 1) |
| `clip_vision_start_image` | CLIP_VISION_OUTPUT | 선택 | - | 시작 이미지에서 추출한 CLIP 비전 특징 |
| `clip_vision_end_image` | CLIP_VISION_OUTPUT | 선택 | - | 끝 이미지에서 추출한 CLIP 비전 특징 |
| `start_image` | IMAGE | 선택 | - | 비디오 시퀀스의 시작 프레임 이미지 |
| `end_image` | IMAGE | 선택 | - | 비디오 시퀀스의 끝 프레임 이미지 |

**참고:** `start_image`와 `end_image`가 모두 제공되면, 노드는 이 두 프레임 사이를 전환하는 비디오 시퀀스를 생성합니다. `clip_vision_start_image`와 `clip_vision_end_image` 매개변수는 선택사항이지만 제공될 경우, 해당 CLIP 비전 특징이 긍정적 및 부정적 조건부에 연결되어 적용됩니다. `start_image`는 처리 전에 처음 `length`개의 프레임으로 자르며, `end_image`는 마지막 `length`개의 프레임으로 자릅니다.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `positive` | CONDITIONING | 비디오 프레임 인코딩과 CLIP 비전 특징이 적용된 긍정적 조건부 |
| `negative` | CONDITIONING | 비디오 프레임 인코딩과 CLIP 비전 특징이 적용된 부정적 조건부 |
| `latent` | LATENT | 지정된 비디오 매개변수와 동일한 크기의 빈 잠재 텐서 |
