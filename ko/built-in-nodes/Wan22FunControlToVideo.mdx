Wan22FunControlToVideo 노드는 Wan 비디오 모델 아키텍처를 사용하여 비디오 생성을 위한 조건부 및 잠재적 표현을 준비합니다. 이 노드는 양성 및 음성 조건부 입력과 선택적 참조 이미지 및 제어 비디오를 처리하여 비디오 합성을 위한 필요한 잠재 공간 표현을 생성합니다. 노드는 공간 스케일링과 시간적 차원을 처리하여 비디오 모델에 적합한 조건부 데이터를 생성합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `positive` | CONDITIONING | 예 | - | 비디오 생성을 안내하는 양성 조건부 입력 |
| `negative` | CONDITIONING | 예 | - | 비디오 생성을 안내하는 음성 조건부 입력 |
| `vae` | VAE | 예 | - | 이미지를 잠재 공간으로 인코딩하는 데 사용되는 VAE 모델 |
| `width` | INT | 예 | 16부터 MAX_RESOLUTION까지 | 출력 비디오 너비(픽셀) (기본값: 832, 단계: 16) |
| `height` | INT | 예 | 16부터 MAX_RESOLUTION까지 | 출력 비디오 높이(픽셀) (기본값: 480, 단계: 16) |
| `length` | INT | 예 | 1부터 MAX_RESOLUTION까지 | 비디오 시퀀스의 프레임 수 (기본값: 81, 단계: 4) |
| `batch_size` | INT | 예 | 1부터 4096까지 | 생성할 비디오 시퀀스 수 (기본값: 1) |
| `ref_image` | IMAGE | 아니요 | - | 시각적 안내를 제공하기 위한 선택적 참조 이미지 |
| `control_video` | IMAGE | 아니요 | - | 생성 과정을 안내하기 위한 선택적 제어 비디오 |

**참고:** `length` 매개변수는 4프레임씩 조각별로 처리되며, 노드는 잠재 공간의 시간적 스케일링을 자동으로 처리합니다. `ref_image`가 제공되면 참조 잠재값을 통해 조건부에 영향을 줍니다. `control_video`가 제공되면 조건부에 사용되는 concat 잠재값에 직접적인 영향을 미칩니다. `start_image` 매개변수는 이 노드의 스키마에 입력으로 노출되지 않지만 실행 로직에서는 참조됩니다.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `positive` | CONDITIONING | 비디오 전용 잠재 데이터를 포함한 수정된 양성 조건부, 여기에는 concat 잠재값, 마스크 및 선택적 참조 잠재값 포함 |
| `negative` | CONDITIONING | 비디오 전용 잠재 데이터를 포함한 수정된 음성 조건부, 여기에는 concat 잠재값, 마스크 및 선택적 참조 잠재값 포함 |
| `latent` | LATENT | 배치 크기, 잠재 채널 및 공간/시간 스케일링에 따라 비디오 생성에 적합한 적절한 차원을 가진 빈 잠재 텐서 |
