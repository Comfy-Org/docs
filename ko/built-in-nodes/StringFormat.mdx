## 개요

이 노드는 파이썬의 문자열 포맷 메서드를 사용하여 텍스트를 서식화합니다. 이는 템플릿과 같으며, 여기서는 자리표시자를 포함한 텍스트 패턴을 정의하고, 해당 자리표시자를 채울 값을 제공합니다. 파이썬의 모든 포맷 옵션과 기능을 지원합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `f_string` | STRING | 예 | N/A | 자리표시자를 포함한 포맷 문자열 템플릿(기본값: `{a}`). 다중 줄 입력도 지원합니다. |
| `values` | STRING | 예 | N/A | 포맷 문자열의 자리표시자를 채울 값을 제공하는 동적 입력입니다. 필요에 따라 여러 값 입력을 추가할 수 있습니다. |

**`values` 입력에 대한 참고사항:** 이 입력은 동적이며, 여러 개의 명명된 값을 포함하도록 확장할 수 있습니다. 각 값 입력에는 알파벳(a, b, c 등)이 붙으며, 포맷 문자열의 자리표시자와 대응됩니다(예: `{a}`, `{b}`, `{c}`). 필요에 따라 값 입력을 추가하거나 삭제할 수 있습니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-----------|-----------|-------------|
| `STRING` | STRING | 자리표시자가 해당 값으로 교체된 서식화된 텍스트 문자열입니다. |
