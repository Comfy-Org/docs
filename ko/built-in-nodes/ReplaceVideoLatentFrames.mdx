ReplaceVideoLatentFrames 노드는 지정된 프레임 인덱스부터 소스 잠재 영상의 프레임을 대상 잠재 영상에 삽입합니다. 소스 잠재가 제공되지 않으면 대상 잠재는 변경되지 않은 채 그대로 반환됩니다. 이 노드는 음수 인덱싱을 처리하며, 소스 프레임이 대상에 맞지 않을 경우 경고를 발생시킵니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `destination` | LATENT | 예 | - | 프레임이 교체될 대상 잠재입니다. |
| `source` | LATENT | 아니요 | - | 대상 잠재에 삽입할 프레임을 제공하는 소스 잠재입니다. 제공되지 않으면 대상 잠재는 변경되지 않은 채 그대로 반환됩니다. |
| `index` | INT | 예 | -MAX_RESOLUTION부터 MAX_RESOLUTION까지 | 대상 잠재에서 소스 잠재 프레임이 배치될 시작 잠재 프레임 인덱스입니다. 음수 값은 끝에서부터 계산됩니다(기본값: 0). |

**제약사항:**

* `index`는 대상 잠재의 프레임 수 범위 내에 있어야 합니다. 그렇지 않으면 경고가 기록되고 대상은 변경되지 않은 채 그대로 반환됩니다.
* 소스 잠재 프레임은 지정된 `index`부터 대상 잠재 프레임에 맞아야 합니다. 그렇지 않으면 경고가 기록되고 대상은 변경되지 않은 채 그대로 반환됩니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-----------|-----------|-------------|
| `output` | LATENT | 프레임 교체 작업 후 결과 잠재 영상입니다. |
