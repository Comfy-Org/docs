Vidu Q3 이미지-비디오 생성 노드는 입력 이미지로부터 비디오 시퀀스를 생성합니다. Vidu Q3 모델을 사용해 이미지를 애니메이션화하며, 선택적으로 텍스트 프롬프트에 의해 안내되고, 비디오 파일을 출력합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `model` | COMBO | 예 | `"viduq3-pro"`<br />`"viduq3-turbo"` | 비디오 생성에 사용할 모델입니다. |
| `model.resolution` | COMBO | 예 | `"720p"`<br />`"1080p"`<br />`"2K"` (viduq3-pro만 해당) | 출력 비디오의 해상도입니다. 사용 가능한 옵션은 선택한 모델에 따라 다릅니다. |
| `model.duration` | INT | 예 | 1~16 | 출력 비디오의 지속 시간(초)(기본값: 5). |
| `model.audio` | BOOLEAN | 예 | `True` / `False` | 활성화 시, 대사와 효과음을 포함한 사운드가 있는 비디오를 출력합니다(기본값: False). |
| `image` | IMAGE | 예 | - | 생성된 비디오의 시작 프레임으로 사용될 이미지입니다. |
| `prompt` | STRING | 아니요 | - | 비디오 생성을 위한 선택적 텍스트 프롬프트(최대 2000자)(기본값: 빈 문자열). |
| `seed` | INT | 아니요 | 0~2147483647 | 생성의 무작위성을 제어하는 시드 값입니다(기본값: 1). |

**참고:** `image`의 종횡비는 1:4에서 4:1 사이여야 합니다(세로에서 가로). `prompt`는 선택사항이지만 최대 2000자를 초과할 수 없습니다. `model.resolution` 옵션은 선택한 `model`에 따라 달라집니다: `"viduq3-pro"`는 `"720p"`, `"1080p"`, `"2K"`를 지원하며, `"viduq3-turbo"`는 `"720p"`와 `"1080p"`를 지원합니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `output` | VIDEO | 생성된 비디오 파일입니다. |
