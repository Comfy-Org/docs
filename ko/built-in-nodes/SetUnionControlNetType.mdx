SetUnionControlNetType 노드를 사용하면 조건부로 사용할 제어 네트워크의 유형을 지정할 수 있습니다. 이 노드는 기존 제어 네트워크를 받아들여 선택한 유형에 따라 제어 유형을 설정하며, 지정된 유형 설정을 적용한 수정된 제어 네트워크 사본을 생성합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `control_net` | CONTROL_NET | 예 | - | 새로운 유형 설정으로 수정할 제어 네트워크 |
| `type` | STRING | 예 | `"auto"`<br />사용 가능한 모든 UNION_CONTROLNET_TYPES 키 | 적용할 제어 네트워크 유형입니다. 자동 유형 감지를 위해 "auto"를 사용하거나 사용 가능한 옵션 중에서 특정 제어 네트워크 유형을 선택하세요(기본값: "auto") |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `control_net` | CONTROL_NET | 지정된 유형 설정이 적용된 수정된 제어 네트워크 |
