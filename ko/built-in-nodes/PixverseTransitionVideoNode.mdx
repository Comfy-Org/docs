> 이 문서는 AI로 생성되었습니다. 오류를 발견하거나 개선할 점이 있다면 언제든지 기여해 주세요! [GitHub에서 편집하기](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/PixverseTransitionVideoNode/en.md)

PixVerse API를 사용하여 두 입력 이미지 간의 전환 동영상을 생성합니다. 시작 이미지와 끝 이미지를 제공하면, 노드가 텍스트 프롬프트와 선택한 설정에 따라 한 이미지에서 다른 이미지로 부드럽게 전환되는 동영상을 만들어냅니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `first_frame` | IMAGE | 예 | - | 동영상 전환의 시작 이미지 |
| `last_frame` | IMAGE | 예 | - | 동영상 전환의 끝 이미지 |
| `prompt` | STRING | 예 | - | 동영상 생성을 위한 프롬프트 (기본값: 빈 문자열) |
| `quality` | COMBO | 예 | `"360p"`<br />`"540p"`<br />`"720p"`<br />`"1080p"` | 동영상 품질 설정 (기본값: `"540p"`) |
| `duration_seconds` | COMBO | 예 | `5`<br />`8` | 동영상 지속 시간(초) |
| `motion_mode` | COMBO | 예 | `"normal"`<br />`"fast"` | 전환의 모션 스타일 (기본값: `"normal"`) |
| `seed` | INT | 예 | 0부터 2147483647까지 | 동영상 생성을 위한 시드 (기본값: 0) |
| `negative_prompt` | STRING | 아니요 | - | 이미지에서 원치 않는 요소에 대한 선택적 텍스트 설명 (기본값: 빈 문자열) |

**매개변수 제약 사항 참고:** 1080p 품질을 사용할 경우, 모션 모드는 자동으로 `"normal"`로 설정되고 지속 시간은 5초로 제한됩니다. 5초가 아닌 다른 지속 시간을 선택하면 모션 모드 역시 자동으로 `"normal"`로 설정됩니다.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `output` | VIDEO | 생성된 전환 동영상 |
