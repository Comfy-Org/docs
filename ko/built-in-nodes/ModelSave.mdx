> 이 문서는 AI로 생성되었습니다. 오류를 발견하거나 개선할 점이 있다면 언제든지 기여해 주세요! [GitHub에서 편집하기](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/ModelSave/en.md)

ModelSave 노드는 학습된 모델이나 수정된 모델을 컴퓨터의 저장소에 저장합니다. 모델을 입력으로 받아 사용자가 지정한 파일명으로 파일에 씁니다. 이를 통해 작업을 보존하고 미래 프로젝트에서도 모델을 재사용할 수 있습니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `model` | MODEL | 예 | - | 디스크에 저장할 모델 |
| `filename_prefix` | STRING | 예 | - | 저장된 모델 파일의 파일명 및 경로 접두사 (기본값: "diffusion_models/ComfyUI") |
| `prompt` | PROMPT | 아니요 | - | 워크플로우 프롬프트 정보 (자동 제공) |
| `extra_pnginfo` | EXTRA_PNGINFO | 아니요 | - | 추가 워크플로우 메타데이터 (자동 제공) |

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| *없음* | - | 이 노드는 출력 값을 반환하지 않습니다 |
