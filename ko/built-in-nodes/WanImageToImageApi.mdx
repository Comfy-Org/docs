> 이 문서는 AI로 생성되었습니다. 오류를 발견하거나 개선할 점이 있다면 언제든지 기여해 주세요! [GitHub에서 편집하기](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/WanImageToImageApi/en.md)

Wan Image to Image 노드는 하나 또는 두 개의 입력 이미지와 텍스트 프롬프트를 사용해 이미지를 생성합니다. 제공한 설명에 따라 입력 이미지를 변형하여 원본 입력 이미지의 종횡비를 유지한 새로운 이미지를 만들어냅니다. 출력 이미지는 입력 크기에 관계없이 160만 화소로 고정됩니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `model` | COMBO | 예 | "wan2.5-i2i-preview" | 사용할 모델 (기본값: "wan2.5-i2i-preview"). |
| `image` | IMAGE | 예 | - | 단일 이미지 편집 또는 다중 이미지 합성, 최대 2개 이미지. |
| `prompt` | STRING | 예 | - | 요소와 시각적 특징을 설명하는 프롬프트입니다. 영어와 중국어 지원(기본값: 비어 있음). |
| `negative_prompt` | STRING | 아니요 | - | 피해야 할 사항을 설명하는 네거티브 프롬프트 (기본값: 비어 있음). |
| `seed` | INT | 아니요 | 0부터 2147483647까지 | 생성에 사용할 시드 (기본값: 0). |
| `watermark` | BOOLEAN | 아니요 | - | 결과에 AI가 생성한 워터마크를 추가할지 여부 (기본값: false). |

**참고:** 이 노드는 정확히 1개 또는 2개의 입력 이미지를 받습니다. 2개 이상의 이미지를 제공하거나 전혀 이미지를 제공하지 않으면 노드는 오류를 반환합니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-----------|-----------|-------------|
| `image` | IMAGE | 입력 이미지와 텍스트 프롬프트를 기반으로 생성된 이미지입니다. |
