AutogrowPrefixTestNode는 자동 확장 입력 기능을 테스트하도록 설계된 논리 노드입니다. 이 노드는 동적 수의 부동소수점 입력을 받아들여 그 값들을 쉼표로 구분된 문자열로 결합한 뒤 해당 문자열을 출력합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `autogrow` | AUTOGROW | 예 | 1~10개의 입력 | 최대 10개의 부동소수점 값을 받을 수 있는 동적 입력 그룹입니다. 그룹 내 각 입력은 FLOAT 타입으로, 최소값은 1이고 최대값은 10입니다. |

**참고:** `autogrow` 입력은 특별한 동적 입력입니다. 최대 10개까지 여러 개의 부동소수점 입력을 이 그룹에 추가할 수 있습니다. 노드는 제공된 모든 값을 처리합니다. 각 개별 부동소수점 입력은 1부터 10까지의 범위로 제한됩니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `output` | STRING | 쉼표로 구분된 모든 입력 부동소수점 값을 포함한 단일 문자열입니다. |
