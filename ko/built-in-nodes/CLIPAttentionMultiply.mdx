CLIPAttentionMultiply 노드는 CLIP 모델의 어텐션 메커니즘을 조정할 수 있도록 하며, 자기 어텐션 레이어의 각 구성 요소에 곱셈 요인을 적용합니다. 이 노드는 CLIP 모델의 어텐션 메커니즘에서 쿼리, 키, 값 및 출력 프로젝션 가중치와 편향을 수정하여 작동합니다. 이 실험적 노드는 지정된 스케일링 요인을 적용한 입력 CLIP 모델의 수정된 복사본을 생성합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `clip` | CLIP | 예 | - | 수정할 CLIP 모델 |
| `q` | FLOAT | 예 | 0.0 - 10.0 | 쿼리 프로젝션 가중치와 편향에 적용되는 곱셈 요인 (기본값: 1.0) |
| `k` | FLOAT | 예 | 0.0 - 10.0 | 키 프로젝션 가중치와 편향에 적용되는 곱셈 요인 (기본값: 1.0) |
| `v` | FLOAT | 예 | 0.0 - 10.0 | 값 프로젝션 가중치와 편향에 적용되는 곱셈 요인 (기본값: 1.0) |
| `out` | FLOAT | 예 | 0.0 - 10.0 | 출력 프로젝션 가중치와 편향에 적용되는 곱셈 요인 (기본값: 1.0) |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-----------|-----------|-------------|
| `CLIP` | CLIP | 지정된 어텐션 스케일링 요인이 적용된 수정된 CLIP 모델을 반환합니다 |
