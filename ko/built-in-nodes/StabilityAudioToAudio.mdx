> 이 문서는 AI로 생성되었습니다. 오류를 발견하거나 개선할 점이 있다면 언제든지 기여해 주세요! [GitHub에서 편집하기](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/StabilityAudioToAudio/en.md)

텍스트 지침을 사용하여 기존 오디오 샘플을 새로운 고품질 구성으로 변환합니다. 이 노드는 입력 오디오 파일을 받아 텍스트 프롬프트에 따라 수정하여 새로운 오디오 콘텐츠를 생성합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `model` | COMBO | 예 | "stable-audio-2.5" | 오디오 변환에 사용할 AI 모델 |
| `prompt` | STRING | 예 |  | 오디오를 변환하는 방법을 설명하는 텍스트 지침 (기본값: 비어 있음, 최대 길이: 10000자) |
| `audio` | AUDIO | 예 |  | 오디오는 6초에서 190초 사이여야 합니다 |
| `duration` | INT | 아니요 | 1~190 | 생성된 오디오의 지속 시간(초)을 제어합니다 (기본값: 190) |
| `seed` | INT | 아니요 | 0~4294967294 | 생성에 사용되는 난수 시드입니다 (기본값: 0) |
| `steps` | INT | 아니요 | 4~8 | 샘플링 단계 수를 제어합니다 (기본값: 8) |
| `strength` | FLOAT | 아니요 | 0.01~1.0 | 오디오 매개변수가 생성된 오디오에 미치는 영향력을 제어하는 매개변수입니다 (기본값: 1.0) |

**참고:** 입력 오디오는 6초에서 190초 사이의 지속 시간이어야 합니다. 프롬프트 텍스트의 최대 길이는 10,000자입니다.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `audio` | AUDIO | 입력 오디오와 텍스트 프롬프트를 기반으로 생성된 변환된 오디오 |
