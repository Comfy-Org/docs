Load Video 노드는 입력 디렉터리에서 비디오 파일을 로드하고 워크플로우에서 처리할 수 있도록 제공합니다. 지정된 입력 폴더에서 비디오 파일을 읽어 다른 비디오 처리 노드와 연결 가능한 비디오 데이터로 출력합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `file` | 문자열 | 예 | 여러 옵션 가능 | 입력 디렉터리에서 로드할 비디오 파일입니다. 드롭다운 목록은 ComfyUI 입력 폴더에서 찾은 모든 비디오 파일로 동적으로 채워집니다. |

**참고:** `file` 매개변수의 사용 가능한 옵션은 입력 디렉터리에 존재하는 비디오 파일로부터 동적으로 채워집니다. 지원되는 콘텐츠 유형의 비디오 파일만 표시됩니다. 또한 노드의 파일 선택기 인터페이스를 통해 직접 새 비디오 파일을 업로드할 수도 있습니다.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `video` | 비디오 | 로드된 비디오 데이터로, 다른 비디오 처리 노드로 전달되어 추가 조작이나 분석을 수행할 수 있습니다. |
