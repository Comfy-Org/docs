ModelComputeDtype 노드는 처리 중 모델이 사용하는 연산 데이터 유형(정밀도)을 변경합니다. 입력 모델의 복사본을 만들어 선택한 정밀도 설정을 적용하며, 하드웨어에 따라 메모리 사용량과 성능을 최적화하는 데 도움을 줄 수 있습니다. 이는 다양한 정밀도 구성의 디버깅 및 테스트에 유용합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `model` | MODEL | 예 | - | 새로운 연산 데이터 유형으로 수정할 입력 모델 |
| `dtype` | STRING | 예 | "default"<br />"fp32"<br />"fp16"<br />"bf16" | 모델에 적용할 연산 데이터 유형 (기본값: "default") |

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `model` | MODEL | 새로운 연산 데이터 유형이 적용된 수정된 모델 |
