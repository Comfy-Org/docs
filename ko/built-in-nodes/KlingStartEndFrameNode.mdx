Kling 시작-끝 프레임 동영상 노드는 제공된 시작 이미지와 끝 이미지를 연결하는 동영상 시퀀스를 생성합니다. 첫 프레임부터 마지막 프레임까지 부드러운 전환을 위해 그 사이의 모든 프레임을 생성합니다. 이 노드는 이미지-동영상 API를 호출하지만, `image_tail` 요청 필드와 호환되는 입력 옵션만 지원합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `start_frame` | IMAGE | 예 | - | 참조 이미지 - URL 또는 Base64 인코딩 문자열, 크기는 10MB를 초과할 수 없으며 해상도는 300×300px 이상, 종횡비는 1:2.5~2.5:1 사이여야 합니다. Base64는 data:image 접두사를 포함해서는 안 됩니다. |
| `end_frame` | IMAGE | 예 | - | 참조 이미지 - 끝 프레임 제어. URL 또는 Base64 인코딩 문자열, 크기는 10MB를 초과할 수 없으며 해상도는 300×300px 이상이어야 합니다. Base64는 data:image 접두사를 포함해서는 안 됩니다. |
| `prompt` | STRING | 예 | - | 긍정적 텍스트 프롬프트 |
| `negative_prompt` | STRING | 예 | - | 부정적 텍스트 프롬프트 |
| `cfg_scale` | FLOAT | 아니요 | 0.0-1.0 | 프롬프트 가이드의 강도를 조절합니다(기본값: 0.5) |
| `aspect_ratio` | COMBO | 아니요 | "16:9"<br />"9:16"<br />"1:1" | 생성된 동영상의 종횡비(기본값: "16:9") |
| `mode` | COMBO | 아니요 | "pro mode / 5s duration / kling-v1-5"<br />"pro mode / 10s duration / kling-v1-5"<br />"pro mode / 5s duration / kling-v1-6"<br />"pro mode / 10s duration / kling-v1-6"<br />"pro mode / 5s duration / kling-v2-1"<br />"pro mode / 10s duration / kling-v2-1"<br />"pro mode / 5s duration / kling-v2-5-turbo"<br />"pro mode / 10s duration / kling-v2-5-turbo" | 동영상 생성에 사용할 구성으로, 형식은 모드 / 지속시간 / 모델명입니다(기본값: "pro mode / 5s duration / kling-v2-5-turbo"). |

**이미지 제약사항:**

- `start_frame`와 `end_frame` 모두 제공되어야 하며 파일 크기는 10MB를 초과할 수 없습니다.
- 최소 해상도: 두 이미지 모두 300×300픽셀 이상이어야 합니다.
- `start_frame`의 종횡비는 1:2.5에서 2.5:1 사이여야 합니다.
- Base64 인코딩된 이미지는 "data:image" 접두사를 포함해서는 안 됩니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `output` | VIDEO | 생성된 동영상 시퀀스 |
| `video_id` | STRING | 생성된 동영상의 고유 식별자 |
| `duration` | STRING | 생성된 동영상의 지속시간 |
