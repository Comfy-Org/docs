이 노드는 LoRA 폴더(하위 폴더 포함)에 위치한 모델을 자동으로 탐지하며, 해당 모델 경로는 `ComfyUI\models\loras`입니다. 더 자세한 정보는 LoRA 모델 설치를 참조해 주세요.

LoRA 로더 노드는 주로 LoRA 모델을 로드하는 데 사용됩니다. LoRA 모델은 이미지에 특정 스타일, 내용 및 디테일을 부여할 수 있는 필터라고 생각할 수 있습니다:

- 특정 예술적 스타일 적용하기 (예: 수묵화)
- 특정 캐릭터의 특징 추가하기 (예: 게임 캐릭터)
- 이미지에 특정 디테일 추가하기

이 모든 것이 LoRA를 통해 가능합니다.

여러 개의 LoRA 모델을 로드해야 하는 경우, 아래와 같이 여러 노드를 직접 연결하여 사용할 수 있습니다.

## 입력

| 매개변수 | 데이터 유형 | 설명 |
| --- | --- | --- |
| `model` | MODEL | 일반적으로 기본 모델과 연결하는 데 사용됨 |
| `clip` | CLIP | 일반적으로 CLIP 모델과 연결하는 데 사용됨 |
| `lora_name` | COMBO[STRING] | 사용할 LoRA 모델의 이름 선택 |
| `strength_model` | FLOAT | 값 범위는 -100.0에서 100.0이며, 일반적으로 일상적인 이미지 생성 시 0~1 사이에서 사용됩니다. 값이 높을수록 모델 조정 효과가 더욱 두드러집니다 |
| `strength_clip` | FLOAT | 값 범위는 -100.0에서 100.0이며, 일반적으로 일상적인 이미지 생성 시 0~1 사이에서 사용됩니다. 값이 높을수록 모델 조정 효과가 더욱 두드러집니다 |

## 출력

| 매개변수 | 데이터 유형 | 설명 |
| --- | --- | --- |
| `model` | MODEL | LoRA 조정이 적용된 모델 |
| `clip` | CLIP | LoRA 조정이 적용된 CLIP 인스턴스 |
