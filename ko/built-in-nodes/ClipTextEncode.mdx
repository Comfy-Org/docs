`CLIP 텍스트 인코딩(CLIPTextEncode)`는 당신의 텍스트 설명을 AI가 이해할 수 있는 형식으로 변환하는 번역기 역할을 합니다. 이를 통해 AI가 입력을 해석하고 원하는 이미지를 생성하도록 도와줍니다.

다른 언어를 쓰는 예술가와 소통한다고 생각해 보세요. 방대한 이미지-텍스트 쌍으로 학습된 CLIP 모델은 이러한 간극을 메워, 당신의 설명을 AI 모델이 따를 수 있는 "지침"으로 변환합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `text` | 문자열 | 예 | 임의의 텍스트 | 인코딩할 텍스트입니다. 다중 줄 입력과 동적 프롬프트를 지원합니다. |
| `clip` | CLIP | 예 | 로드된 CLIP 모델 | 텍스트를 인코딩하는 데 사용되는 CLIP 모델입니다. |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `CONDITIONING` | CONDITIONING | 확산 모델을 안내하는 데 사용되는 임베디드 텍스트를 포함한 조건부입니다. |

## 프롬프트 기능

### 임베딩 모델

임베딩 모델을 사용하면 특정 예술적 효과나 스타일을 적용할 수 있습니다. 지원되는 형식은 `.safetensors`, `.pt`, `.bin`입니다. 임베딩 모델을 사용하려면:

1. 해당 파일을 `ComfyUI/models/embeddings` 폴더에 넣으세요.
2. 텍스트에서 `embedding:model_name`을 사용해 참조하세요.

예시: `ComfyUI/models/embeddings` 폴더에 `EasyNegative.pt`라는 모델이 있다면 다음과 같이 사용할 수 있습니다.

```
worst quality, embedding:EasyNegative, bad quality
```

**중요**: 임베딩 모델을 사용할 때는 파일 이름이 정확히 일치하고 모델의 아키텍처와 호환되는지 확인하세요. 예를 들어 SD1.5용으로 설계된 임베딩 모델은 SDXL 모델에서는 제대로 작동하지 않습니다.

### 프롬프트 가중치 조정

괄호를 사용해 설명의 특정 부분의 중요도를 조정할 수 있습니다. 예를 들면:

- `(beautiful:1.2)`는 "beautiful"의 가중치를 높입니다.
- `(beautiful:0.8)`는 "beautiful"의 가중치를 낮춥니다.
- 일반 괄호 `(beautiful)`는 기본 가중치 1.1을 적용합니다.

키보드 단축키 `ctrl + 위/아래 화살표`를 사용해 가중치를 빠르게 조정할 수 있습니다. 설정에서 가중치 조정의 단계 크기를 변경할 수 있습니다.

프롬프트에 실제 괄호를 포함시키면서 가중치를 변경하지 않으려면 백슬래시를 사용해 이스케이프하세요. 예: `\{word\}`

### 와일드카드/동적 프롬프트

`{}`를 사용해 동적 프롬프트를 만들 수 있습니다. 예를 들어 `{day|night|morning}`는 프롬프트가 처리될 때마다 무작위로 하나의 옵션을 선택합니다.

프롬프트에 실제 중괄호를 포함시키면서 동적 행동을 트리거하지 않으려면 백슬래시를 사용해 이스케이프하세요. 예: `\{word\}`
