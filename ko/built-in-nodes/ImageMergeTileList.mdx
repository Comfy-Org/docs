이 노드는 이미지 타일 목록을 받아 하나의 더 큰 이미지로 병합합니다. 이 노드는 이전에 격자 형태로 나뉜 이미지를 재구성하도록 설계되었으며, 가중치 블렌딩 기법을 사용해 매끄러운 최종 결과를 만들어냅니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `image_list` | IMAGE | 예 | N/A | 병합할 이미지 타일 목록입니다. 목록의 첫 번째 타일은 전체 처리 과정의 타일 크기와 데이터 유형을 결정하는 데 사용됩니다. |
| `final_width` | INT | 예 | 64 - 32768 | 최종 병합된 이미지의 너비(픽셀 단위, 기본값: 1024). |
| `final_height` | INT | 예 | 64 - 32768 | 최종 병합된 이미지의 높이(픽셀 단위, 기본값: 1024). |
| `overlap` | INT | 예 | 0 - 4096 | 인접한 타일 간의 겹치는 부분의 크기(픽셀 단위). 0보다 큰 값은 타일 경계에서 부드러운 블렌딩 효과를 활성화합니다(기본값: 128). |

**참고:** `image_list`는 동적 입력 목록입니다. 노드는 제공된 순서대로 타일을 처리하며, `final_width`, `final_height` 및 첫 번째 타일의 크기에 따라 정해진 그리드를 채울 만큼의 타일을 처리합니다. 목록에 필요한 타일보다 많은 타일이 포함되어 있으면 추가 타일은 무시됩니다.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-----------|-----------|-------------|
| `image` | IMAGE | 입력 타일로부터 재구성된 최종 병합된 이미지입니다. |
