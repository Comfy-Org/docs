## 개요

Frame Interpolate 노드는 이미지 시퀀스에서 기존 프레임 사이에 새로운 프레임을 생성하여 프레임 속도를 효과적으로 높입니다. 이 노드는 AI 모델을 사용해 중간 프레임이 어떻게 보일지 예측하며, 이를 통해 부드러운 슬로우모션 효과를 만들거나 동영상의 부드러움을 향상시키는 데 활용할 수 있습니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `interp_model` | MODEL | 예 | - | 중간 프레임 생성에 사용할 프레임 보간 모델 |
| `images` | IMAGE | 예 | - | 보간할 연속된 이미지(프레임)의 배치입니다. 최소 2개의 이미지가 필요합니다. |
| `multiplier` | INT | 예 | 2~16 | 프레임 수를 몇 배로 늘릴지 지정합니다. 예를 들어, multiplier가 2이면 프레임 수가 두 배로 늘어납니다. (기본값: 2) |

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `IMAGE` | IMAGE | 원래 프레임 사이에 보간된 프레임이 삽입된 새로운 이미지 배치로, 더 부드러운 시퀀스를 얻을 수 있습니다. 총 출력 프레임 수는 `(입력 프레임 수 - 1) × multiplier + 1`입니다. |
