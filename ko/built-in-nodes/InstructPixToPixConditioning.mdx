InstructPixToPixConditioning 노드는 긍정적이고 부정적인 텍스트 프롬프트와 이미지 데이터를 결합하여 InstructPix2Pix 이미지 편집을 위한 조건부 데이터를 준비합니다. 입력 이미지를 VAE 인코더를 통해 잠재 표현으로 변환하고, 이 잠재 표현을 긍정적 및 부정적 조건부 데이터에 연결합니다. 노드는 VAE 인코딩 과정과의 호환성을 위해 이미지 크기를 자동으로 8픽셀의 배수로 자르며 처리합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `positive` | CONDITIONING | 예 | - | 원하는 이미지 특성에 대한 텍스트 프롬프트와 설정을 포함한 긍정적 조건부 데이터 |
| `negative` | CONDITIONING | 예 | - | 원하지 않는 이미지 특성에 대한 텍스트 프롬프트와 설정을 포함한 부정적 조건부 데이터 |
| `vae` | VAE | 예 | - | 입력 이미지를 잠재 표현으로 인코딩하는 데 사용되는 VAE 모델 |
| `pixels` | IMAGE | 예 | - | 처리하고 잠재 공간으로 인코딩할 입력 이미지 |

**참고:** 입력 이미지의 크기는 가로와 세로 모두 8픽셀의 가장 가까운 배수로 자르면서 자동으로 조정되어 VAE 인코딩 과정과의 호환성을 보장합니다.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `positive` | CONDITIONING | 잠재 이미지 표현이 연결된 긍정적 조건부 데이터 |
| `negative` | CONDITIONING | 잠재 이미지 표현이 연결된 부정적 조건부 데이터 |
| `latent` | LATENT | 인코딩된 이미지와 동일한 크기의 빈 잠재 텐서 |
