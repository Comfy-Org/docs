VAEEncodeTiled 노드는 이미지를 작은 타일로 나누어 변분 자동인코더를 사용해 인코딩합니다. 이 타일 방식은 메모리 한계를 초과할 수 있는 큰 이미지 처리를 가능하게 합니다. 이 노드는 이미지와 동영상 VAE 모두를 지원하며, 공간적 차원과 시간적 차원에 대해 별도의 타일링 컨트롤을 제공합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `pixels` | IMAGE | 예 | - | 인코딩할 입력 이미지 데이터 |
| `vae` | VAE | 예 | - | 인코딩에 사용되는 변분 자동인코더 모델 |
| `tile_size` | INT | 예 | 64-4096 (단계: 64) | 공간 처리를 위한 각 타일의 크기 (기본값: 512) |
| `overlap` | INT | 예 | 0-4096 (단계: 32) | 인접한 타일 간의 겹치는 부분의 크기 (기본값: 64) |
| `temporal_size` | INT | 예 | 8-4096 (단계: 4) | 동영상 VAE에만 적용: 한 번에 인코딩할 프레임 수 (기본값: 64) |
| `temporal_overlap` | INT | 예 | 4-4096 (단계: 4) | 동영상 VAE에만 적용: 겹치는 프레임 수 (기본값: 8) |

**참고:** `temporal_size`와 `temporal_overlap` 매개변수는 동영상 VAE를 사용할 때만 관련이 있으며, 일반 이미지 VAE에는 영향을 미치지 않습니다.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `LATENT` | LATENT | 입력 이미지의 인코딩된 잠재 표현 |
