PhotoMakerLoader 노드는 사용 가능한 모델 파일에서 PhotoMaker 모델을 로드합니다. 지정된 모델 파일을 읽고, ID 기반 이미지 생성 작업에 사용할 수 있도록 PhotoMaker ID 인코더를 준비합니다. 이 노드는 실험적이라고 표시되어 있으며 테스트용으로 설계되었습니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `photomaker_model_name` | 문자열 | 예 | 여러 옵션 제공 | 로드할 PhotoMaker 모델 파일의 이름입니다. 사용 가능한 옵션은 `photomaker` 폴더에 존재하는 모델 파일에 따라 결정됩니다. |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `photomaker_model` | PHOTOMAKER | ID 인코딩 작업에 바로 사용할 수 있도록 ID 인코더를 포함한 로드된 PhotoMaker 모델입니다. |
