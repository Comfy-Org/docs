RegexReplace 노드는 정규 표현식 패턴을 사용하여 문자열에서 텍스트를 찾아 교체합니다. 패턴 매칭 방식을 제어할 수 있는 옵션을 제공하며, 대소문자 구분 여부, 다중 줄 매칭, 교체 횟수 제한 등을 설정할 수 있습니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `string` | STRING | 예 | - | 검색하고 교체할 입력 텍스트 문자열 |
| `regex_pattern` | STRING | 예 | - | 입력 문자열에서 검색할 정규 표현식 패턴 |
| `replace` | STRING | 예 | - | 매칭된 패턴을 대체할 교체 텍스트 |
| `case_insensitive` | BOOLEAN | 아니요 | - | 활성화 시 패턴 매칭 시 대소문자 차이를 무시합니다(기본값: True) |
| `multiline` | BOOLEAN | 아니요 | - | 활성화 시 ^와 $의 동작을 전체 문자열의 시작/끝이 아닌 각 줄의 시작/끝에 맞추도록 변경합니다(기본값: False) |
| `dotall` | BOOLEAN | 아니요 | - | 활성화 시 점(.) 문자가 개행 문자를 포함한 모든 문자와 일치합니다. 비활성화 시 점은 개행 문자와 일치하지 않습니다(기본값: False) |
| `count` | INT | 아니요 | 0-100 | 최대 교체 횟수. 0으로 설정하면 모든 발생 위치를 교체합니다(기본값). 1로 설정하면 첫 번째 매칭만 교체, 2로 설정하면 처음 두 번째 매칭만 교체 등으로 설정할 수 있습니다(기본값: 0) |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-----------|-----------|-------------|
| `output` | STRING | 지정된 교체가 적용된 수정된 문자열 |
