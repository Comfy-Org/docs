**ColorToRGBInt** 노드는 16진수 형식으로 지정된 색상(예: `#FF5733`)을 하나의 정수 값으로 변환합니다. 이 노드는 색상 문자열에서 빨강, 초록, 파랑 성분을 가져와 이를 하나의 RGB 정수로 결합합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `color` | STRING | 예 | N/A | 16진수 형식의 색상 값 `#RRGGBB`. 반드시 정확히 7자리이며, 시작은 `#`로 해야 합니다. |

**참고:** 입력 `color` 문자열은 반드시 `#RRGGBB` 형식을 정확히 따라야 합니다. 문자열의 길이가 7자리가 아니거나 시작이 `#`로 되어 있지 않으면 노드에서 오류가 발생합니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `rgb_int` | INT | 계산된 RGB 정수 값입니다. 공식은 `(빨강 * 65536) + (초록 * 256) + 파랑`입니다. |
