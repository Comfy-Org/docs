ByteDance Text to Video 노드는 텍스트 프롬프트를 기반으로 API를 통해 ByteDance 모델을 사용하여 동영상을 생성합니다. 텍스트 설명과 다양한 동영상 설정을 입력받아 제공된 사양에 맞는 동영상을 만들어냅니다. 이 노드는 API 통신을 처리하고 생성된 동영상을 출력으로 반환합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `model` | 문자열 | 예 | `"seedance-1-5-pro-251215"`<br />`"seedance-1-0-pro-250528"`<br />`"seedance-1-0-lite-t2v-250428"`<br />`"seedance-1-0-pro-fast-251015"` | 생성에 사용할 ByteDance 모델 (기본값: `"seedance-1-0-pro-fast-251015"`). |
| `prompt` | 문자열 | 예 | - | 동영상을 생성하는 데 사용되는 텍스트 프롬프트입니다. |
| `resolution` | 문자열 | 예 | `"480p"`<br />`"720p"`<br />`"1080p"` | 출력 동영상의 해상도입니다. |
| `aspect_ratio` | 문자열 | 예 | `"16:9"`<br />`"4:3"`<br />`"1:1"`<br />`"3:4"`<br />`"9:16"`<br />`"21:9"` | 출력 동영상의 종횡비입니다. |
| `duration` | 정수 | 예 | 3~12 | 출력 동영상의 지속시간(초)입니다 (기본값: 5). |
| `seed` | 정수 | 아니요 | 0~2,147,483,647 | 생성에 사용할 시드입니다 (기본값: 0). |
| `camera_fixed` | 불리언 | 아니요 | - | 카메라를 고정할지 여부를 지정합니다. 플랫폼은 카메라를 고정하라는 지시를 프롬프트에 추가하지만, 실제로 효과가 보장되지는 않습니다 (기본값: False). |
| `watermark` | 불리언 | 아니요 | - | 동영상에 "AI 생성" 워터마크를 추가할지 여부입니다 (기본값: False). |
| `generate_audio` | 불리언 | 아니요 | - | 이 매개변수는 `seedance-1-5-pro-251215` 모델을 제외한 모든 모델에서는 무시됩니다 (기본값: False). |

**매개변수 제약사항:**

- `prompt` 매개변수는 공백 제거 후 최소 1자 이상 포함해야 합니다.
- `prompt` 매개변수에는 다음 텍스트 매개변수를 포함할 수 없습니다: "resolution", "ratio", "duration", "seed", "camerafixed", "watermark".
- `duration` 매개변수는 3초에서 12초 사이의 값으로 제한됩니다. `seedance-1-5-pro-251215` 모델의 경우, 지원되는 최소 지속시간은 4초입니다.
- `seed` 매개변수는 0부터 2,147,483,647까지의 값을 허용합니다.
- `generate_audio` 매개변수는 `model`이 `seedance-1-5-pro-251215`로 설정된 경우에만 영향을 미치며, 다른 모든 모델에서는 무시됩니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-----------|-----------|-------------|
| `output` | 동영상 | 생성된 동영상 파일 |
