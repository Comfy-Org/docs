PrimitiveInt 노드는 워크플로우에서 정수 값을 간편하게 다룰 수 있도록 해줍니다. 이 노드는 정수형 입력을 받아 동일한 값을 출력하므로, 노드 간에 정수형 매개변수를 전달하거나 다른 연산을 위해 특정 숫자 값을 설정하는 데 유용합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `value` | INT | 예 | -9223372036854775807부터 9223372036854775807까지 | 출력할 정수 값 (기본값: 0) |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `output` | INT | 변경되지 않은 채로 전달된 입력 정수 값 |
