DiffusersLoader 노드는 diffusers 형식의 사전 학습된 모델을 로드합니다. 이 노드는 `model_index.json` 파일이 포함된 유효한 diffusers 모델 디렉터리를 검색하고 이를 MODEL, CLIP, VAE 구성 요소로 로드하여 파이프라인에서 사용할 수 있도록 합니다. 이 노드는 폐기된 로더 카테고리에 속하며, Hugging Face diffusers 모델과의 호환성을 제공합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `model_path` | 문자열 | 예 | 여러 옵션 가능<br />(diffusers 폴더에서 자동으로 채워짐) | 로드할 diffusers 모델 디렉터리의 경로입니다. 노드는 구성된 diffusers 폴더에서 유효한 diffusers 모델을 자동으로 스캔하고 가능한 옵션을 나열합니다. |

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `MODEL` | MODEL | diffusers 형식에서 로드된 모델 구성 요소 |
| `CLIP` | CLIP | diffusers 형식에서 로드된 CLIP 모델 구성 요소 |
| `VAE` | VAE | diffusers 형식에서 로드된 VAE(변분 오토인코더) 구성 요소 |
