## 개요

파일에서 배경 제거 모델을 로드합니다. 이 노드는 이미지에서 배경을 제거하는 데 사용할 수 있도록 모델을 준비합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `bg_removal_name` | 문자열 | 예 | 사용 가능한 모델 파일 목록 | 이미지에서 배경을 제거하는 데 사용되는 모델입니다. 사용 가능한 배경 제거 모델 파일 목록에서 선택하세요. |

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `bg_model` | BACKGROUND_REMOVAL | 로드된 배경 제거 모델로, 다른 노드가 이미지를 처리하는 데 바로 사용할 수 있습니다. |
