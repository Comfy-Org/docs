TripoRefineNode는 Tripo v1.4 모델이 특별히 생성한 초안 3D 모델을 정밀하게 다듬습니다. 모델 작업 ID를 받아 Tripo API를 통해 처리하여 모델의 개선된 버전을 생성합니다. 이 노드는 오직 Tripo v1.4 모델이 생성한 초안 모델과만 함께 작동하도록 설계되었습니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `model_task_id` | MODEL_TASK_ID | 예 | - | 반드시 v1.4 Tripo 모델이어야 합니다 |

**참고:** 이 노드는 오직 Tripo v1.4 모델이 만든 초안 모델만 수용합니다. 다른 버전의 모델을 사용하면 오류가 발생할 수 있습니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `model_file` | STRING | 정밀화된 모델의 파일 경로 또는 참조 (후향 호환성을 위해만 제공) |
| `model task_id` | MODEL_TASK_ID | 정밀화된 모델 작업의 작업 식별자 |
| `GLB` | FILE3DGLB | GLB 형식으로 정밀화된 3D 모델 |
