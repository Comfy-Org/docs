Kling 텍스트를 비디오 카메라 제어 노드는 텍스트를 영화 같은 비디오로 변환하며, 실제 촬영을 모방한 전문적인 카메라 움직임을 제공합니다. 이 노드는 줌, 회전, 패닝, 틸트 및 1인칭 뷰와 같은 가상 카메라 동작을 제어할 수 있으며, 원본 텍스트에 초점을 유지합니다. 카메라 제어는 kling-v1-5 모델과 함께 프로 모드에서만 5초 길이로 지원되므로, 지속 시간, 모드 및 모델명은 하드코딩되어 있습니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `prompt` | 문자열 | 예 | - | 긍정적 텍스트 프롬프트 |
| `negative_prompt` | 문자열 | 예 | - | 부정적 텍스트 프롬프트 |
| `cfg_scale` | 부동소수점 | 아니요 | 0.0-1.0 | 출력물이 프롬프트를 얼마나 정확히 따르는지 제어합니다(기본값: 0.75) |
| `aspect_ratio` | 콤보 | 아니요 | "16:9"<br />"9:16"<br />"1:1"<br />"21:9"<br />"3:4"<br />"4:3" | 생성된 비디오의 종횡비(기본값: "16:9") |
| `camera_control` | 카메라 제어 | 아니요 | - | Kling 카메라 제어 노드를 사용해 생성할 수 있습니다. 비디오 생성 중 카메라 움직임과 모션을 제어합니다. |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `output` | 비디오 | 카메라 제어 효과가 적용된 생성된 비디오 |
| `video_id` | 문자열 | 생성된 비디오의 고유 식별자 |
| `duration` | 문자열 | 생성된 비디오의 지속 시간 |
