## 개요

이 노드는 모델 샘플링 시 사용되는 노이즈 스케일을 조정합니다. 특정 노이즈 스케일 값을 설정하여 모델의 샘플링 과정에 적용되는 노이즈의 양을 제어할 수 있습니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `model` | MODEL | 예 | - | 노이즈 스케일 조정을 적용할 모델입니다. |
| `noise_scale` | FLOAT | 예 | 0.0부터 64.0까지 (단계: 0.01) | 절대적인 학습용 노이즈 스케일입니다. 예: HiDream-O1 기본값: 8.0, 개발용: 7.5. (기본값: 1.0) |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `MODEL` | MODEL | 새로운 노이즈 스케일이 적용된 수정된 모델입니다. |
