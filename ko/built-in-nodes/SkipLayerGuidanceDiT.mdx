건너뛴 레이어를 사용하여 다른 CFG 음수 세트를 통해 상세한 구조로의 안내를 강화합니다. SkipLayerGuidance의 이 범용 버전은 모든 DiT 모델에 사용할 수 있으며, Perturbed Attention Guidance에서 영감을 받았습니다. 원래 실험적 구현은 SD3용으로 만들어졌습니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `model` | MODEL | 예 | - | 건너뛴 레이어 안내를 적용할 모델 |
| `double_layers` | STRING | 예 | - | 건너뛸 더블 블록의 레이어 번호를 쉼표로 구분해 입력 (기본값: "7, 8, 9") |
| `single_layers` | STRING | 예 | - | 건너뛸 싱글 블록의 레이어 번호를 쉼표로 구분해 입력 (기본값: "7, 8, 9") |
| `scale` | FLOAT | 예 | 0.0 - 10.0 | 안내 스케일 요인 (기본값: 3.0) |
| `start_percent` | FLOAT | 예 | 0.0 - 1.0 | 안내 적용 시작 비율 (기본값: 0.01) |
| `end_percent` | FLOAT | 예 | 0.0 - 1.0 | 안내 적용 종료 비율 (기본값: 0.15) |
| `rescaling_scale` | FLOAT | 예 | 0.0 - 10.0 | 출력 크기를 조정하는 재스케일링 스케일 요인 (기본값: 0.0, 즉 재스케일링 없음) |

**참고:** `double_layers`와 `single_layers`가 모두 비어 있거나(레이어 번호가 포함되지 않음) 경우, 노드는 아무런 안내를 적용하지 않고 원본 모델을 반환합니다.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `model` | MODEL | 건너뛴 레이어 안내가 적용된 수정된 모델 |
