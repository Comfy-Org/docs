LoraModelLoader 노드는 학습된 LoRA(저랭크 적응) 가중치를 확산 모델에 적용합니다. 이 노드는 학습된 LoRA 모델로부터 LoRA 가중치를 로드하고 그 영향력 강도를 조정함으로써 기본 모델을 수정합니다. 이를 통해 확산 모델을 처음부터 다시 학습하지 않고도 모델의 동작을 맞춤화할 수 있습니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `model` | MODEL | 예 | - | LoRA를 적용할 확산 모델입니다. |
| `lora` | LORA_MODEL | 예 | - | 확산 모델에 적용할 LoRA 모델입니다. |
| `strength_model` | FLOAT | 예 | -100.0부터 100.0까지 | 확산 모델을 얼마나 강하게 수정할지 설정합니다. 이 값은 음수일 수 있습니다(기본값: 1.0). |
| `bypass` | BOOLEAN | 예 | true 또는 false | 활성화 시, 기본 모델 가중치를 수정하지 않고 Bypass 모드로 LoRA를 적용합니다. 훈련 중이거나 모델 가중치가 오프로드된 경우 유용합니다(기본값: false). |

**참고:** `strength_model`을 0으로 설정하면 노드는 어떠한 LoRA 수정도 적용하지 않고 원래 모델을 반환합니다.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-----------|-----------|-------------|
| `model` | MODEL | LoRA 가중치가 적용된 수정된 확산 모델입니다. |
