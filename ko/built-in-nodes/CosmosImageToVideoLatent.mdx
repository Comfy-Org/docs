CosmosImageToVideoLatent 노드는 입력 이미지로부터 비디오 잠재 표현을 생성합니다. 빈 비디오 잠재를 생성하고, 선택적으로 시작 이미지와/또는 종료 이미지를 비디오 시퀀스의 첫 번째 프레임과/또는 마지막 프레임에 인코딩합니다. 이미지가 제공되면, 생성 과정에서 잠재의 어떤 부분을 보존해야 하는지 나타내는 해당 노이즈 마스크도 함께 생성합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `vae` | VAE | 예 | - | 이미지를 잠재 공간으로 인코딩하는 데 사용되는 VAE 모델 |
| `width` | INT | 예 | 16부터 MAX_RESOLUTION까지 | 출력 비디오의 너비(픽셀 단위) (기본값: 1280) |
| `height` | INT | 예 | 16부터 MAX_RESOLUTION까지 | 출력 비디오의 높이(픽셀 단위) (기본값: 704) |
| `length` | INT | 예 | 1부터 MAX_RESOLUTION까지 | 비디오 시퀀스의 프레임 수 (기본값: 121) |
| `batch_size` | INT | 예 | 1부터 4096까지 | 생성할 잠재 배치의 수 (기본값: 1) |
| `start_image` | IMAGE | 아니요 | - | 비디오 시퀀스의 시작 부분에 인코딩할 옵션 이미지 |
| `end_image` | IMAGE | 아니요 | - | 비디오 시퀀스의 끝 부분에 인코딩할 옵션 이미지 |

**참고:** `start_image`와 `end_image` 모두 제공되지 않을 경우, 노드는 아무런 노이즈 마스크 없이 빈 잠재를 반환합니다. 어느 한쪽 이미지라도 제공되면, 해당 부분의 잠재가 인코딩되고 그에 맞는 마스크가 생성됩니다.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `latent` | LATENT | 생성된 비디오 잠재 표현으로, 선택적 인코딩된 이미지와 해당 노이즈 마스크 포함 |
