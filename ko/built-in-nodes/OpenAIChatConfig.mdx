OpenAIChatConfig 노드는 OpenAI Chat 노드에 추가적인 구성 옵션을 설정할 수 있게 해줍니다. 모델이 응답을 생성하는 방식을 제어하는 고급 설정을 제공하며, 여기에는 잘림 동작, 출력 길이 제한, 맞춤형 지시사항 등이 포함됩니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `truncation` | COMBO | 예 | `"auto"`<br />`"disabled"` | 모델 응답에 사용할 잘림 전략입니다. auto: 이 응답과 이전 응답의 컨텍스트가 모델의 컨텍스트 창 크기를 초과하면, 모델은 대화 중간에 있는 입력 항목을 삭제하여 컨텍스트 창에 맞게 응답을 잘라냅니다. disabled: 모델 응답이 모델의 컨텍스트 창 크기를 초과할 경우, 요청은 400 오류로 실패합니다(기본값: "auto") |
| `max_output_tokens` | INT | 아니요 | 16부터 16384까지 | 응답에 생성할 수 있는 토큰 수의 상한값으로, 가시적 출력 토큰도 포함됩니다(기본값: 4096) |
| `instructions` | STRING | 아니요 | - | 모델에게 응답을 생성하는 방법에 대한 지시사항(다중 줄 입력 지원) |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `OPENAI_CHAT_CONFIG` | OPENAI_CHAT_CONFIG | OpenAI Chat 노드와 함께 사용하기 위해 지정된 설정을 포함한 구성 객체 |
