## 개요

이 노드는 파일에서 프레임 보간 모델을 로드하여 워크플로우에서 사용할 수 있도록 준비합니다. 모델 유형(FILM 또는 RIFE)을 자동으로 감지하고, 사용자의 하드웨어에 최적화된 성능을 위해 모델을 구성합니다.

## 입력

| 매개변수 | 설명 | 데이터 타입 | 필수 여부 | 범위 |
| --- | --- | --- | --- | --- |
| `model_name` | 로드할 프레임 보간 모델을 선택합니다. 모델은 'frame_interpolation' 폴더에 배치되어야 합니다. | STRING | 예 | `frame_interpolation` 폴더 내 모델 파일 목록 |

## 출력

| 출력 이름 | 설명 | 데이터 타입 |
| --- | --- | --- |
| `FRAME_INTERPOLATION_MODEL` | 로드 및 구성된 프레임 보간 모델로, 다른 노드에서 사용할 준비가 되었습니다. | MODEL |
