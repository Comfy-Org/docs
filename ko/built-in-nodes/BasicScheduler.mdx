`BasicScheduler` 노드는 제공된 스케줄러, 모델 및 노이즈 제거 매개변수를 기반으로 확산 모델에 대한 시그마 값의 시퀀스를 계산하도록 설계되었습니다. 노이즈 제거 요인에 따라 총 단계 수를 동적으로 조정하여 확산 과정을 세밀하게 조정하며, 다단계 샘플링과 같이 정밀한 제어가 필요한 고급 샘플링 과정의 각 단계에 맞는 정확한 "레시피"를 제공합니다.

## 입력

| 매개변수   | 데이터 유형     | 입력 유형 | 기본값 | 범위     | 비유적 설명           | 기술적 목적            |
| ----------- | ------------- | ---------- | ------- | --------- | ---------------------- | ------------------------ |
| `model`     | MODEL         | 입력      | -       | -         | **캔버스 유형**: 서로 다른 캔버스 재료에는 서로 다른 페인트 공식이 필요합니다 | 확산 모델 객체, 시그마 계산 기준 결정 |
| `scheduler` | COMBO[STRING] | 위젯     | -       | 9개 옵션 | **혼합 기법**: 페인트 농도 변화 방식 선택 | 스케줄링 알고리즘, 노이즈 감소 모드 제어 |
| `steps`     | INT           | 위젯     | 20      | 1-10000   | **혼합 횟수**: 20회 vs 50회 혼합의 정밀도 차이 | 샘플링 단계, 생성 품질과 속도에 영향 |
| `denoise`   | FLOAT         | 위젯     | 1.0     | 0.0-1.0   | **생성 강도**: 미세 조정부터 다시 칠하기까지 제어 | 노이즈 제거 강도, 부분적인 다시 칠하기 시나리오 지원 |

### 스케줄러 유형

소스 코드 `comfy.samplers.SCHEDULER_NAMES`에 기반해 다음 9가지 스케줄러를 지원합니다:

| 스케줄러 이름       | 특징      | 활용 사례                    | 노이즈 감소 패턴          |
| -------------------- | ---------- | ---------------------------- | -------------------------- |
| **normal**           | 표준 선형      | 일반적인 시나리오, 균형 잡힌 상태 | 균일한 감소               |
| **karras**           | 부드러운 전환    | 고품질, 디테일이 살아있는 상태    | 부드러운 비선형 감소      |
| **exponential**      | 지수 감소    | 빠른 생성, 효율성        | 지수 급격한 감소          |
| **sgm_uniform**      | SGM 균일          | 특정 모델 최적화        | SGM 최적화된 감소          |
| **simple**           | 간단한 스케줄링    | 빠른 테스트, 기본 사용     | 단순화된 감소             |
| **ddim_uniform**     | DDIM 균일         | DDIM 샘플링 최적화       | DDIM 특화 감소            |
| **beta**             | 베타 분포    | 특별한 분포 요구사항     | 베타 함수 감소            |
| **linear_quadratic** | 선형 이차     | 복잡한 시나리오 최적화    | 이차함수 감소             |
| **kl_optimal**       | KL 최적화           | 이론적 최적화     | KL 발산 최적화 감소      |

## 출력

| 매개변수 | 데이터 유형 | 출력 유형 | 비유적 설명   | 기술적 의미                |
| --------- | --------- | ---------- | -------------- | ---------------------------- |
| `sigmas`  | SIGMAS    | 출력      | **페인트 레시피 차트**: 단계별로 사용할 상세한 페인트 농도 목록 | 노이즈 수준 시퀀스, 확산 모델의 노이즈 제거 과정 안내 |

## 노드 역할: 아티스트의 색상 혼합 도우미

여러분이 혼란스러운 페인트(노이즈)의 혼합물에서 명확한 이미지를 만들어내는 아티스트라고 상상해 보세요. `BasicScheduler`는 마치 여러분의 **전문 색상 혼합 도우미**와 같으며, 일련의 정밀한 페인트 농도 레시피를 준비하는 것이 바로 그의 임무입니다:

### 워크플로우

- **1단계**: 90% 농도의 페인트 사용 (높은 노이즈 수준)
- **2단계**: 80% 농도의 페인트 사용  
- **3단계**: 70% 농도의 페인트 사용
- **...**
- **최종 단계**: 0% 농도의 페인트 사용 (깨끗한 캔버스, 노이즈 없음)

### 색상 도우미의 특별한 기술

**다양한 혼합 방법(스케줄러)**:

- **"karras" 혼합 방법**: 페인트 농도가 매우 부드럽게 변하며, 전문 아티스트의 그라데이션 기법과 같습니다
- **"exponential" 혼합 방법**: 페인트 농도가 빠르게 감소하며, 빠른 창작에 적합합니다
- **"linear" 혼합 방법**: 페인트 농도가 균일하게 감소하며, 안정적이고 제어 가능합니다

**정밀한 제어(단계)**:

- **20회 혼합**: 빠른 페인팅, 효율성 우선
- **50회 혼합**: 섬세한 페인팅, 품질 우선

**생성 강도(denoise)**:

- **1.0 = 완전한 새 창작**: 완전히 빈 캔버스에서 시작
- **0.5 = 절반 변형**: 원래 페인트의 절반은 유지하고 절반은 변형
- **0.2 = 미세 조정**: 원래 페인트에만 미세한 조정을 가합니다

### 다른 노드와의 협업

`BasicScheduler` (색상 도우미) → 레시피 준비 → `SamplerCustom` (아티스트) → 실제 페인팅 → 완성 작품
