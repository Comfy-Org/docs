> 이 문서는 AI로 생성되었습니다. 오류를 발견하거나 개선할 점이 있다면 언제든지 기여해 주세요! [GitHub에서 편집하기](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/TopazVideoEnhance/en.md)

Topaz Video Enhance 노드는 외부 API를 사용해 비디오 품질을 개선합니다. 비디오 해상도를 높이고 보간을 통해 프레임 속도를 증가시키며 압축을 적용할 수 있습니다. 이 노드는 입력 MP4 비디오를 처리하고 선택한 설정에 따라 향상된 버전을 반환합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `video` | VIDEO | 예 | - | 향상시킬 입력 비디오 파일입니다. |
| `upscaler_enabled` | BOOLEAN | 예 | - | 비디오 업스케일링 기능을 활성화하거나 비활성화합니다(기본값: True). |
| `upscaler_model` | COMBO | 예 | `"Starlight (Astra) Fast"`<br />`"Starlight (Astra) Creative"`<br />`"Starlight Precise 2.5"` | 비디오 업스케일링에 사용되는 AI 모델입니다. |
| `upscaler_resolution` | COMBO | 예 | `"FullHD (1080p)"`<br />`"4K (2160p)"` | 업스케일링된 비디오의 목표 해상도입니다. |
| `upscaler_creativity` | COMBO | 아니요 | `"low"`<br />`"middle"`<br />`"high"` | 창의성 수준(Starlight (Astra) Creative에만 적용됨). (기본값: "low") |
| `interpolation_enabled` | BOOLEAN | 아니요 | - | 프레임 보간 기능을 활성화하거나 비활성화합니다(기본값: False). |
| `interpolation_model` | COMBO | 아니요 | `"apo-8"` | 프레임 보간에 사용되는 모델입니다(기본값: "apo-8"). |
| `interpolation_slowmo` | INT | 아니요 | 1부터 16까지 | 입력 비디오에 적용되는 슬로모션 계수입니다. 예를 들어 2를 지정하면 출력이 두 배 느려지고 지속시간도 두 배가 됩니다. (기본값: 1) |
| `interpolation_frame_rate` | INT | 아니요 | 15부터 240까지 | 출력 프레임 속도입니다. (기본값: 60) |
| `interpolation_duplicate` | BOOLEAN | 아니요 | - | 입력을 분석해 중복 프레임을 찾아 제거합니다. (기본값: False) |
| `interpolation_duplicate_threshold` | FLOAT | 아니요 | 0.001부터 0.1까지 | 중복 프레임 탐지 민감도입니다. (기본값: 0.01) |
| `dynamic_compression_level` | COMBO | 아니요 | `"Low"`<br />`"Mid"`<br />`"High"` | CQP 수준입니다. (기본값: "Low") |

**참고:** 최소 하나 이상의 향상 기능을 활성화해야 합니다. `upscaler_enabled`와 `interpolation_enabled`가 모두 `False`로 설정되면 노드는 오류를 발생시킵니다. 입력 비디오는 MP4 형식이어야 합니다.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `video` | VIDEO | 향상된 출력 비디오 파일입니다. |
