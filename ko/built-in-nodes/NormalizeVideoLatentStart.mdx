> 이 문서는 AI로 생성되었습니다. 오류를 발견하거나 개선할 점이 있다면 언제든지 기여해 주세요! [GitHub에서 편집하기](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/NormalizeVideoLatentStart/en.md)

이 노드는 비디오 레이턴트의 처음 몇 프레임을 조정하여 이후에 나오는 프레임과 더 유사하게 만듭니다. 비디오 후반부의 참조 프레임 세트로부터 평균값과 변동성을 계산한 뒤, 시작 프레임에도 동일한 특성을 적용합니다. 이를 통해 비디오의 시작 부분에서 보다 부드럽고 일관된 시각적 전환을 만들 수 있습니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `latent` | LATENT | 예 | - | 처리할 비디오 레이턴트 표현입니다. |
| `start_frame_count` | INT | 예 | 1부터 16384까지 (최대 해상도) | 정규화할 레이턴트 프레임의 수이며, 시작점부터 계산됩니다(기본값: 4). |
| `reference_frame_count` | INT | 예 | 1부터 16384까지 (최대 해상도) | 시작 프레임 이후에 참조로 사용할 레이턴트 프레임의 수입니다(기본값: 5). |

**참고:** `reference_frame_count`는 자동으로 시작 프레임 이후에 사용 가능한 프레임 수로 제한됩니다. 만약 비디오 레이턴트가 단 한 프레임만 있다면, 정규화가 수행되지 않고 원래의 레이턴트가 그대로 반환됩니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `latent` | LATENT | 시작 프레임이 정규화된 처리된 비디오 레이턴트입니다. |
