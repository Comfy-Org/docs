이 노드는 두 개의 마스크 입력을 더하기, 빼기, 논리 연산 등 다양한 작업을 통해 결합하여 새롭고 수정된 마스크를 생성하는 데 특화되어 있습니다. 마스크 데이터의 조작을 추상적으로 처리하여 복잡한 마스크 효과를 구현하며, 마스크 기반 이미지 편집 및 처리 워크플로우에서 핵심적인 역할을 수행합니다.

## 입력

| 매개변수    | 데이터 유형 | 설명                                                                                                                                      |
| ------------ | ------------ | ------------------------------------------------------------------------------------------------------------------------------------------------ |
| `destination`| MASK        | 소스 마스크와의 연산에 따라 수정될 기본 마스크입니다. 합성 작업에서 중심적인 역할을 하며, 수정의 기준이 됩니다. |
| `source`     | MASK        | 목적 마스크와 함께 지정된 연산을 수행하는 데 사용되는 보조 마스크로, 최종 출력 마스크에 영향을 줍니다. |
| `x`          | INT         | 소스 마스크가 목적 마스크에 적용될 수평 오프셋으로, 합성 결과의 위치를 조정합니다.       |
| `y`          | INT         | 소스 마스크가 목적 마스크에 적용될 수직 오프셋으로, 합성 결과의 위치를 조정합니다.         |
| `operation`  | COMBO[STRING]| 목적 마스크와 소스 마스크 간에 적용할 연산 유형을 지정하며, 예를 들어 '더하기', '빼기' 또는 논리 연산 등을 통해 합성 효과의 성격을 결정합니다. |

## 출력

| 매개변수 | 데이터 유형 | 설명                                                                 |
| --------- | ------------ | ---------------------------------------------------------------------------- |
| `mask`    | MASK        | 목적 마스크와 소스 마스크 간에 지정된 연산을 적용한 후 생성된 결과 마스크로, 합성된 결과를 나타냅니다. |
