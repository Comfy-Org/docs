> 이 문서는 AI로 생성되었습니다. 오류를 발견하거나 개선할 점이 있다면 언제든지 기여해 주세요! [GitHub에서 편집하기](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/TencentTextToModelNode/en.md)

이 노드는 Tencent의 Hunyuan3D Pro API를 사용하여 텍스트 설명을 기반으로 3D 모델을 생성합니다. 생성 작업을 요청하고 결과를 폴링한 후, 최종 모델 파일을 GLB 및 OBJ 형식으로 다운로드합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `model` | COMBO | 예 | `"3.0"`<br />`"3.1"` | 사용할 Hunyuan3D 모델의 버전입니다. `3.1` 모델에서는 LowPoly 옵션이 사용 불가능합니다. |
| `prompt` | STRING | 예 | - | 생성할 3D 모델의 텍스트 설명입니다. 최대 1024자까지 지원됩니다. |
| `face_count` | INT | 예 | 3000 - 1500000 | 생성된 3D 모델의 목표 면 수입니다. 기본값: 500000. |
| `generate_type` | DYNAMICCOMBO | 예 | `"Normal"`<br />`"LowPoly"`<br />`"Geometry"` | 생성할 3D 모델의 유형입니다. 사용 가능한 옵션과 관련 매개변수는 다음과 같습니다:<br />- **Normal**: 표준 모델을 생성합니다. `pbr` 매개변수가 포함됩니다(기본값: `False`).<br />- **LowPoly**: 저폴리곤 모델을 생성합니다. `polygon_type`(`"triangle"` 또는 `"quadrilateral"`)와 `pbr` 매개변수(기본값: `False`)가 포함됩니다.<br />- **Geometry**: 기하학적 형태만을 가진 모델을 생성합니다. |
| `seed` | INT | 아니요 | 0 - 2147483647 | 생성을 위한 시드 값입니다. 시드와 상관없이 결과는 비결정적입니다. 새로운 시드를 설정하면 노드가 다시 실행될지 여부를 제어할 수 있습니다. 기본값: 0. |

**참고:** `generate_type` 매개변수는 동적입니다. `"LowPoly"`를 선택하면 `polygon_type` 및 `pbr`에 대한 추가 입력이 나타납니다. `"Normal"`을 선택하면 `pbr` 입력이 나타납니다. `"Geometry"`를 선택하면 추가 입력은 나타나지 않습니다.

**제약사항:** `"LowPoly"` 생성 유형은 `"3.1"` 모델과 함께 사용할 수 없습니다.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `model_file` | STRING | 역호환성을 위한 레거시 출력입니다. |
| `GLB` | FILE3DGLB | GLB 파일 형식으로 생성된 3D 모델입니다. |
| `OBJ` | FILE3DOBJ | OBJ 파일 형식으로 생성된 3D 모델입니다. |
| `texture_image` | IMAGE | 생성된 OBJ 파일에서 추출된 텍스처 이미지입니다. 가능하다면 제공됩니다. |
