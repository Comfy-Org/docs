GeminiNanoBanana2 노드는 Google의 Vertex AI Gemini 모델을 사용해 이미지를 생성하거나 편집합니다. 이 노드는 텍스트 프롬프트와 선택적 참조 이미지 또는 파일을 API로 전송한 후 생성된 이미지와 함께 제공되는 텍스트를 반환합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `prompt` | 문자열 | 예 | 없음 | 생성할 이미지나 적용할 편집 내용을 설명하는 텍스트 프롬프트입니다. 모델이 따라야 할 제약 조건, 스타일, 세부사항 등을 포함하세요. |
| `model` | 콤보 | 예 | "Nano Banana 2 (Gemini 3.1 Flash Image)" | 이미지 생성에 사용할 특정 Gemini 모델입니다. |
| `seed` | 정수 | 예 | 0부터 18446744073709551615까지 | 시드를 특정 값으로 고정하면 모델은 반복 요청 시 동일한 응답을 제공하려고 노력합니다. 결정론적 출력이 보장되지는 않습니다. 또한 모델이나 온도와 같은 매개변수 설정을 변경하면 동일한 시드 값을 사용하더라도 응답에 변동이 생길 수 있습니다. 기본적으로 임의의 시드 값이 사용됩니다. (기본값: 42) |
| `aspect_ratio` | 콤보 | 예 | "auto"<br />"1:1"<br />"2:3"<br />"3:2"<br />"3:4"<br />"4:3"<br />"4:5"<br />"5:4"<br />"9:16"<br />"16:9"<br />"21:9" | 'auto'로 설정하면 입력 이미지의 종횡비를 맞추며, 이미지를 제공하지 않으면 보통 16:9 정사각형이 생성됩니다. (기본값: "auto") |
| `resolution` | 콤보 | 예 | "1K"<br />"2K"<br />"4K" | 목표 출력 해상도입니다. 2K/4K의 경우 기본 Gemini 업스케일러가 사용됩니다. |
| `response_modalities` | 콤보 | 예 | "IMAGE"<br />"IMAGE+TEXT" | 모델이 반환할 콘텐츠 유형을 결정합니다. (고급) |
| `thinking_level` | 콤보 | 예 | "MINIMAL"<br />"HIGH" | 모델의 추론 과정의 깊이를 제어합니다. |
| `images` | 이미지 | 아니요 | 없음 | 선택적 참조 이미지입니다. 여러 이미지를 포함하려면 배치 이미지 노드를 사용하세요(최대 14개). |
| `files` | 커스텀 | 아니요 | 없음 | 모델의 컨텍스트로 사용할 선택적 파일입니다. Gemini Generate Content Input Files 노드에서 입력을 받습니다. |
| `system_prompt` | 문자열 | 아니요 | 없음 | AI의 행동을 지시하는 기본적인 지침입니다. (고급) |

**참고:** `images` 입력은 최대 14개의 이미지를 지원합니다. 더 많은 이미지를 제공하면 노드에서 오류가 발생합니다.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `image` | 이미지 | 모델이 생성하거나 편집한 주요 이미지입니다. |
| `string` | 문자열 | 모델이 반환한 텍스트 콘텐츠입니다. |
| `thought_image` | 이미지 | 모델의 사고 과정에서 나온 첫 번째 이미지입니다. thinking_level이 HIGH이고 response_modalities가 IMAGE+TEXT인 경우에만 사용 가능합니다. |
