> 이 문서는 AI로 생성되었습니다. 오류를 발견하거나 개선할 점이 있다면 언제든지 기여해 주세요! [GitHub에서 편집하기](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/OpenAIInputFiles/en.md)

OpenAI API용 입력 파일을 로드하고 형식화합니다. 이 노드는 텍스트(.txt) 및 PDF(.pdf) 파일을 준비하여 OpenAI 채팅 노드의 컨텍스트 입력으로 포함시킵니다. 모델은 응답을 생성할 때 해당 파일을 읽습니다. 여러 개의 OpenAI Input Files 노드를 연결하여 하나의 메시지에 여러 파일을 포함시킬 수 있습니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `file` | COMBO | 예 | 여러 옵션 가능 (입력 디렉터리 내 모든 .txt 및 .pdf 파일, 32MB 이하) | 모델의 컨텍스트로 포함할 입력 파일입니다. 현재 텍스트(.txt) 및 PDF(.pdf) 파일만 허용됩니다. 파일 크기는 32MB 미만이어야 합니다. |
| `OPENAI_INPUT_FILES` | OPENAI_INPUT_FILES | 아니요 | N/A | 이 노드에서 로드한 파일과 함께 일괄 처리할 추가 파일(들). 여러 입력 파일을 체인으로 연결하여 하나의 메시지에 여러 입력 파일을 포함할 수 있습니다. |

**파일 제약사항:**

- 지원되는 파일 형식은 .txt와 .pdf뿐입니다.
- 최대 파일 크기: 32MB
- 파일은 ComfyUI 입력 디렉터리에서 로드됩니다.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `OPENAI_INPUT_FILES` | OPENAI_INPUT_FILES | OpenAI API 호출의 컨텍스트로 사용할 수 있도록 형식화된 입력 파일입니다. |
