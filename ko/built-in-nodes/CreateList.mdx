Create List 노드는 여러 입력을 하나의 순차적 리스트로 결합합니다. 동일한 데이터 유형의 입력을 아무 수량이나 받아들여 연결된 순서대로 연결합니다. 이 노드는 이미지나 텍스트와 같은 데이터 배치를 워크플로우 내 다른 노드가 처리하도록 준비하는 데 유용합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `inputs` | 다양함 | 예 | 아무거나 | 가변 수의 입력 슬롯입니다. 더 많은 입력을 추가하려면 플러스(+), 아이콘을 클릭하세요. 모든 입력은 동일한 데이터 유형이어야 합니다(예: 모두 IMAGE 또는 모두 STRING). |

**참고:** 노드는 항목을 연결할 때 자동으로 새로운 입력 슬롯을 생성합니다. 연결된 모든 입력은 동일한 데이터 유형을 공유해야만 노드가 올바르게 작동합니다.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `list` | 다양함 | 연결된 모든 입력의 항목을 순서대로 연결한 단일 리스트입니다. 출력 데이터 유형은 입력 데이터 유형과 일치합니다. |
