Shuffle Dataset 노드는 이미지 목록을 받아들여 무작위로 순서를 변경합니다. 랜덤성을 제어하기 위해 시드 값을 사용하며, 동일한 섞기 순서를 재현할 수 있도록 보장합니다. 이는 데이터셋의 이미지 순서를 처리 전에 무작위로 섞는 데 유용합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `images` | IMAGE | 예 | - | 섞일 이미지 목록입니다. |
| `seed` | INT | 아니요 | 0부터 18446744073709551615까지 | 랜덤 시드입니다. (기본값: 0) |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `images` | IMAGE | 동일한 이미지 목록이지만, 새롭게 무작위로 섞인 순서로 출력됩니다. |
