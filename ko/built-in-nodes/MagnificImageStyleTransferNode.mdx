> 이 문서는 AI로 생성되었습니다. 오류를 발견하거나 개선할 점이 있으면 언제든지 기여해 주세요! [GitHub에서 편집하기](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/MagnificImageStyleTransferNode/en.md)

이 노드는 참조 이미지의 시각적 스타일을 입력 이미지에 적용합니다. 이미지를 처리하기 위해 외부 AI 서비스를 사용하며, 스타일 전환의 강도와 원본 이미지의 구조 보존 정도를 조정할 수 있습니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `image` | IMAGE | 예 | - | 스타일 전환을 적용할 이미지입니다. |
| `reference_image` | IMAGE | 예 | - | 스타일을 추출할 참조 이미지입니다. |
| `prompt` | STRING | 아니요 | - | 스타일 전환을 안내하는 선택적 텍스트 프롬프트입니다. |
| `style_strength` | INT | 아니요 | 0부터 100까지 | 스타일 강도의 백분율 (기본값: 100). |
| `structure_strength` | INT | 아니요 | 0부터 100까지 | 원본 이미지의 구조를 유지합니다 (기본값: 50). |
| `flavor` | COMBO | 아니요 | "faithful"<br />"gen_z"<br />"psychedelia"<br />"detaily"<br />"clear"<br />"donotstyle"<br />"donotstyle_sharp" | 스타일 전환의 플레이버입니다. |
| `engine` | COMBO | 아니요 | "balanced"<br />"definio"<br />"illusio"<br />"3d_cartoon"<br />"colorful_anime"<br />"caricature"<br />"real"<br />"super_real"<br />"softy" | 처리 엔진 선택입니다. |
| `portrait_mode` | COMBO | 아니요 | "disabled"<br />"enabled" | 얼굴 강화를 위한 세로 모드를 활성화합니다. |
| `portrait_style` | COMBO | 아니요 | "standard"<br />"pop"<br />"super_pop" | 세로 이미지에 적용되는 시각적 스타일입니다. 이 입력은 `portrait_mode`가 "enabled"로 설정된 경우에만 사용 가능합니다. |
| `portrait_beautifier` | COMBO | 아니요 | "none"<br />"beautify_face"<br />"beautify_face_max" | 세로 이미지의 얼굴 미화 강도입니다. 이 입력은 `portrait_mode`가 "enabled"로 설정된 경우에만 사용 가능합니다. |
| `fixed_generation` | BOOLEAN | 아니요 | - | 비활성화 시, 각 생성 과정마다 약간의 무작위성이 추가되어 더 다양한 결과를 얻을 수 있습니다 (기본값: True). |

**제약사항:**

* `image`와 `reference_image`는 정확히 하나씩 필요합니다.
* 두 이미지 모두 가로세로 비율이 1:3에서 3:1 사이여야 합니다.
* 두 이미지 모두 최소 높이와 너비가 160픽셀 이상이어야 합니다.
* `portrait_style`와 `portrait_beautifier` 매개변수는 `portrait_mode`가 "enabled"로 설정된 경우에만 활성화됩니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `image` | IMAGE | 스타일 전환이 적용된 결과 이미지입니다. |
