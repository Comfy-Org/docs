> 이 문서는 AI로 생성되었습니다. 오류를 발견하거나 개선할 점이 있다면 언제든지 기여해 주세요! [GitHub에서 편집하기](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/LoadImageTextDataSetFromFolder/en.md)

이 노드는 지정된 폴더에서 이미지와 해당 텍스트 캡션으로 구성된 데이터셋을 로드합니다. 이미지 파일을 검색한 뒤, 같은 기본 이름을 가진 `.txt` 파일을 자동으로 찾아 캡션으로 사용합니다. 또한 특정 폴더 구조를 지원하며, 하위 폴더의 이름에 숫자 접두사를 붙여 (예: `10_folder_name`) 해당 이미지가 출력물에서 여러 번 반복되도록 지정할 수 있습니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `folder` | COMBO | 예 | *`folder_paths.get_input_subfolders()`에서 동적으로 로드됨* | 이미지와 텍스트 캡션을 로드할 폴더입니다. 사용 가능한 옵션은 ComfyUI의 입력 디렉터리 내의 하위 디렉터리입니다. |

**참고:** 이 노드는 특정 파일 구조를 요구합니다. 각 이미지 파일(`.png`, `.jpg`, `.jpeg`, `.webp`)에 대해 동일한 이름을 가진 `.txt` 파일을 찾아 캡션으로 사용합니다. 캡션 파일이 없으면 빈 문자열이 사용됩니다. 또한 특별한 구조를 지원하며, 하위 폴더의 이름이 숫자와 밑줄로 시작하는 경우(예: `5_cats`), 해당 하위 폴더 내의 모든 이미지가 최종 출력 목록에서 그 숫자만큼 반복됩니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `images` | IMAGE | 로드된 이미지 텐서의 리스트입니다. |
| `texts` | STRING | 로드된 각 이미지에 해당하는 텍스트 캡션의 리스트입니다. |
