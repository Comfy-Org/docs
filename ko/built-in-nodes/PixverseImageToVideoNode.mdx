> 이 문서는 AI로 생성되었습니다. 오류를 발견하거나 개선할 점이 있다면 언제든지 기여해 주세요! [GitHub에서 편집하기](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/PixverseImageToVideoNode/en.md)

입력 이미지와 텍스트 프롬프트를 기반으로 동영상을 생성합니다. 이 노드는 이미지를 받아 지정된 모션 및 품질 설정을 적용하여 정적 이미지를 움직이는 시퀀스로 변환함으로써 애니메이션 동영상을 만듭니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `image` | IMAGE | 예 | - | 동영상으로 변환할 입력 이미지 |
| `prompt` | STRING | 예 | - | 동영상 생성을 위한 프롬프트 |
| `quality` | COMBO | 예 | `res_540p`<br />`res_1080p` | 동영상 품질 설정 (기본값: res_540p) |
| `duration_seconds` | COMBO | 예 | `dur_2`<br />`dur_5`<br />`dur_10` | 생성된 동영상의 지속 시간(초) |
| `motion_mode` | COMBO | 예 | `normal`<br />`fast`<br />`slow`<br />`zoom_in`<br />`zoom_out`<br />`pan_left`<br />`pan_right`<br />`pan_up`<br />`pan_down`<br />`tilt_up`<br />`tilt_down`<br />`roll_clockwise`<br />`roll_counterclockwise` | 동영상 생성에 적용되는 모션 스타일 |
| `seed` | INT | 예 | 0-2147483647 | 동영상 생성을 위한 시드 (기본값: 0) |
| `negative_prompt` | STRING | 선택 | - | 이미지에서 원치 않는 요소에 대한 선택적 텍스트 설명 |
| `pixverse_template` | CUSTOM | 선택 | - | 생성 스타일에 영향을 미치는 옵션 템플릿, PixVerse Template 노드에서 생성 |

**참고:** 1080p 품질을 사용할 경우 모션 모드는 자동으로 normal로 설정되고 지속 시간은 5초로 제한됩니다. 5초가 아닌 다른 지속 시간을 선택하면 모션 모드도 자동으로 normal로 설정됩니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `output` | VIDEO | 입력 이미지와 매개변수를 기반으로 생성된 동영상 |
