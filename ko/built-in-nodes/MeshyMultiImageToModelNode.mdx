> 이 문서는 AI로 생성되었습니다. 오류를 발견하거나 개선할 점이 있다면 언제든지 기여해 주세요! [GitHub에서 편집하기](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/MeshyMultiImageToModelNode/en.md)

이 노드는 Meshy API를 사용하여 여러 입력 이미지로부터 3D 모델을 생성합니다. 제공된 이미지를 업로드하고 처리 작업을 제출한 후, 결과 3D 모델 파일(GLB 및 FBX)과 참조용 작업 ID를 반환합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
| :--- | :--- | :--- | :--- | :--- |
| `model` | COMBO | 예 | `"latest"` | 사용할 AI 모델 버전을 지정합니다. |
| `images` | IMAGE | 예 | 2~4개의 이미지 | 3D 모델 생성에 사용되는 이미지 세트입니다. 2~4개의 이미지를 제공해야 합니다. |
| `should_remesh` | COMBO | 예 | `"true"`<br />`"false"` | 생성된 메시를 처리할지 여부를 결정합니다. `"false"`로 설정하면 노드는 처리되지 않은 삼각형 메시를 반환합니다. |
| `topology` | COMBO | 선택 | `"triangle"`<br />`"quad"` | 리메시된 출력의 대상 다각형 유형입니다. 이 매개변수는 `should_remesh`가 `"true"`로 설정된 경우에만 사용 가능하며 필수입니다. |
| `target_polycount` | INT | 선택 | 100~300000 | 리메시된 모델의 목표 다각형 수(기본값: 300000). 이 매개변수는 `should_remesh`가 `"true"`로 설정된 경우에만 사용 가능합니다. |
| `symmetry_mode` | COMBO | 예 | `"auto"`<br />`"on"`<br />`"off"` | 생성된 모델에 대칭 적용 여부를 제어합니다. |
| `should_texture` | COMBO | 예 | `"true"`<br />`"false"` | 텍스처 생성 여부를 결정합니다. `"false"`로 설정하면 텍스처 단계를 건너뛰고 텍스처 없는 메시를 반환합니다. |
| `enable_pbr` | BOOLEAN | 선택 | True / False | `should_texture`가 `"true"`일 때, 이 옵션은 기본 색상 외에도 PBR 맵(금속성, 거칠기, 법선)을 생성합니다(기본값: False). |
| `texture_prompt` | STRING | 선택 | - | 텍스처링 과정을 안내하는 텍스트 프롬프트(최대 600자). `texture_image`와 동시에 사용할 수 없습니다. 이 매개변수는 `should_texture`가 `"true"`로 설정된 경우에만 사용 가능합니다. |
| `texture_image` | IMAGE | 선택 | - | 텍스처링 과정을 안내하는 이미지입니다. `texture_image`와 `texture_prompt` 중 하나만 동시에 사용할 수 있습니다. 이 매개변수는 `should_texture`가 `"true"`로 설정된 경우에만 사용 가능합니다. |
| `pose_mode` | COMBO | 예 | `""`(비어 있음)<br />`"A-pose"`<br />`"T-pose"` | 생성된 모델의 포즈 모드를 지정합니다. |
| `seed` | INT | 예 | 0~2147483647 | 생성 과정의 시드 값(기본값: 0). 시드와 상관없이 결과는 비결정적이나, 시드를 변경하면 노드가 다시 실행될 수 있습니다. |

**매개변수 제약사항:**

* `images` 입력에는 반드시 2~4개의 이미지를 제공해야 합니다.
* `topology`와 `target_polycount` 매개변수는 `should_remesh`가 `"true"`로 설정된 경우에만 활성화됩니다.
* `enable_pbr`, `texture_prompt`, `texture_image` 매개변수는 `should_texture`가 `"true"`로 설정된 경우에만 활성화됩니다.
* `texture_prompt`와 `texture_image`는 동시에 사용할 수 없으며 서로 배타적입니다.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
| :--- | :--- | :--- |
| `model_file` | STRING | 생성된 GLB 모델의 파일명입니다. 이 출력은 하위 호환성을 위해 제공됩니다. |
| `meshy_task_id` | MESHY_TASK_ID | Meshy API 작업의 고유 식별자입니다. |
| `GLB` | FILE3DGLB | GLB 형식으로 생성된 3D 모델입니다. |
| `FBX` | FILE3DFBX | FBX 형식으로 생성된 3D 모델입니다. |
