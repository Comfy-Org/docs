EmptyMochiLatentVideo 노드는 지정된 크기의 빈 잠재적 비디오 텐서를 생성합니다. 이 노드는 0으로 채워진 잠재적 표현을 생성하며, 이를 비디오 생성 워크플로우의 시작점으로 사용할 수 있습니다. 노드를 통해 잠재적 비디오 텐서의 너비, 높이, 길이 및 배치 크기를 정의할 수 있습니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `width` | INT | 예 | 16부터 MAX_RESOLUTION까지 | 잠재적 비디오의 너비(픽셀) (기본값: 848, 16의 배수여야 함) |
| `height` | INT | 예 | 16부터 MAX_RESOLUTION까지 | 잠재적 비디오의 높이(픽셀) (기본값: 480, 16의 배수여야 함) |
| `length` | INT | 예 | 7부터 MAX_RESOLUTION까지 | 잠재적 비디오의 프레임 수 (기본값: 25, `(length - 1)`가 6의 배수가 되어야 함) |
| `batch_size` | INT | 아니요 | 1부터 4096까지 | 한 번에 생성할 잠재적 비디오의 수 (기본값: 1) |

**참고:** 실제 잠재적 차원은 너비/8과 높이/8로 계산되며, 시간적 차원은 `((length - 1) // 6) + 1`로 계산됩니다. `length` 매개변수는 `(length - 1)`가 6의 배수가 되어야 하며, 유효한 값은 7, 13, 19, 25 등입니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `samples` | LATENT | 지정된 크기의 빈 잠재적 비디오 텐서로, 모든 값이 0으로 채워져 있습니다 |
