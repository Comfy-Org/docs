## 개요

이 노드는 등방형 파노라마 이미지에 대한 깊이 추정을 수행합니다. 파노라마를 12개의 원근 보기로 분할한 뒤 각 보기마다 MoGe 깊이 추정 모델을 실행하고, 그 결과를 다시 하나의 완전한 깊이 맵으로 합쳐서 원래 파노라마의 깊이 맵을 생성합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `moge_model` | MOGE_MODEL | 예 | | 추론에 사용할 MoGe 모델입니다. |
| `image` | IMAGE | 예 | | 등방형 파노라마 이미지(임의의 종횡비). |
| `resolution_level` | INT | 예 | 0부터 9까지 | 각 보기의 세부 수준입니다. 값이 높을수록 더 상세한 깊이 맵이 생성됩니다(기본값: 9). |
| `split_resolution` | INT | 예 | 256부터 1024까지 | 파노라마를 분할한 후 각 원근 보기의 해상도입니다(기본값: 512). |
| `merge_resolution` | INT | 예 | 256부터 8192까지 | 최종 병합된 등방형 깊이 맵의 장측 해상도입니다(기본값: 1920). |
| `batch_size` | INT | 예 | 1부터 12까지 | 각 추론 배치에서 처리할 원근 보기의 수입니다. 전체 보기 수는 12개입니다(기본값: 4). |

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `moge_geometry` | MOGE_GEOMETRY | 추정된 지오메트리를 포함하는 사전으로, `points`(3D 포인트 클라우드), `depth`(깊이 맵), `mask`(유효 영역 마스크), `image`(입력 이미지)가 포함됩니다. |
