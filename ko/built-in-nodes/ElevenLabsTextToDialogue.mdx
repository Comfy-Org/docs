ElevenLabs Text to Dialogue 노드는 텍스트를 기반으로 다중 스피커 오디오 대화를 생성합니다. 각 참여자별로 서로 다른 텍스트 줄과 고유한 목소리를 지정하여 대화를 만들 수 있습니다. 이 노드는 대화 요청을 ElevenLabs API로 보내고 생성된 오디오를 반환합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `stability` | FLOAT | 선택적 | 0.0 - 1.0 | 목소리 안정성. 낮은 값은 더 넓은 감정 범위를 제공하며, 높은 값은 더 일관되지만 다소 단조로운 말투를 만들어냅니다. (기본값: 0.5) |
| `apply_text_normalization` | COMBO | 선택적 | `"auto"`<br />`"on"`<br />`"off"` | 텍스트 정규화 모드. 'auto'는 시스템이 결정하도록 하고, 'on'은 항상 정규화를 적용하며, 'off'는 이를 건너뜁니다. |
| `model` | COMBO | 선택적 | `"eleven_v3"` | 대화 생성에 사용할 모델. |
| `inputs` | DYNAMICCOMBO | 필수 | `"1"`<br />`"2"`<br />`"3"`<br />`"4"`<br />`"5"`<br />`"6"`<br />`"7"`<br />`"8"`<br />`"9"`<br />`"10"` | 대화 항목의 수. 숫자를 선택하면 해당 수만큼 텍스트 및 목소리 입력 필드가 생성됩니다. |
| `language_code` | STRING | 선택적 | - | ISO-639-1 또는 ISO-639-3 언어 코드 (예: 'en', 'es', 'fra'). 자동 감지하려면 비워 두세요. (기본값: 비워둠) |
| `seed` | INT | 선택적 | 0 - 4294967295 | 재현성을 위한 시드. (기본값: 1) |
| `output_format` | COMBO | 선택적 | `"mp3_44100_192"`<br />`"opus_48000_192"` | 오디오 출력 형식. |

**참고:** `inputs` 매개변수는 동적입니다. 숫자를 선택하면(예: "3"), 노드는 해당하는 `text` 및 `voice` 입력 필드 세 개를 표시합니다(예: `text1`, `voice1`, `text2`, `voice2`, `text3`, `voice3`). 각 `text` 필드에는 최소 한 글자 이상이 포함되어야 합니다.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `audio` | AUDIO | 선택된 출력 형식의 다중 스피커 대화 오디오. |
