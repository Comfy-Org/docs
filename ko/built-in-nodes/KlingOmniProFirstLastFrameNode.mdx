> 이 문서는 AI로 생성되었습니다. 오류를 발견하거나 개선할 점이 있다면 언제든지 기여해 주세요! [GitHub에서 편집하기](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/KlingOmniProFirstLastFrameNode/en.md)

이 노드는 최신 Kling AI 모델을 사용하여 시작 프레임, 선택적 종료 프레임 또는 참조 이미지로부터 비디오를 생성합니다. 단일 비디오나 여러 샷의 스토리보드를 만들 수 있으며, 각 구간마다 개별적인 프롬프트와 지속 시간을 설정할 수 있습니다. 이 노드는 이러한 입력을 처리해 지정된 길이와 해상도의 비디오를 생성하며, 선택적으로 오디오까지 생성할 수 있습니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `model_name` | COMBO | 예 | `"kling-v3-omni"`<br />`"kling-video-o1"` | 비디오 생성에 사용할 특정 Kling AI 모델입니다. |
| `prompt` | STRING | 예 | - | 비디오 내용을 설명하는 텍스트 프롬프트입니다. 긍정적 및 부정적 설명을 모두 포함할 수 있습니다. 스토리보드가 활성화된 경우 무시됩니다. |
| `duration` | INT | 예 | 3~15 | 생성할 비디오의 원하는 길이(초)(기본값: 5). |
| `first_frame` | IMAGE | 예 | - | 비디오 시퀀스의 시작 이미지입니다. |
| `end_frame` | IMAGE | 아니요 | - | 비디오의 선택적 종료 프레임입니다. `reference_images`와 동시에 사용할 수 없습니다. 스토리보드에서는 작동하지 않습니다. |
| `reference_images` | IMAGE | 아니요 | - | 최대 6개의 추가 참조 이미지입니다. |
| `resolution` | COMBO | 아니요 | `"4k"`<br />`"1080p"`<br />`"720p"` | 생성할 비디오의 출력 해상도입니다(기본값: "1080p"). |
| `storyboards` | DYNAMIC_COMBO | 아니요 | `"비활성화"`<br />`"1개 스토리보드"`<br />`"2개 스토리보드"`<br />`"3개 스토리보드"`<br />`"4개 스토리보드"`<br />`"5개 스토리보드"`<br />`"6개 스토리보드"` | 개별적인 프롬프트와 지속 시간을 가진 일련의 비디오 구간을 생성합니다. `kling-v3-omni`에서만 지원됩니다. 활성화 시 각 스토리보드마다 프롬프트와 지속 시간 입력이 필요합니다. |
| `generate_audio` | BOOLEAN | 아니요 | True / False | 비디오에 오디오를 생성할지 여부(기본값: False). `kling-v3-omni`에서만 지원됩니다. |
| `seed` | INT | 아니요 | 0~2147483647 | 시드는 노드가 다시 실행될지 여부를 제어합니다; 시드와 상관없이 결과는 비결정적입니다(기본값: 0). |

**중요한 제약사항:**

* `end_frame` 입력은 `reference_images` 입력과 동시에 사용할 수 없습니다.
* `end_frame` 입력은 스토리보드와 동시에 사용할 수 없습니다.
* `kling-video-o1` 모델은 10초 이상의 지속 시간, 오디오 생성, 4k 해상도, 스토리보드를 지원하지 않습니다.
* `kling-video-o1` 모델로 `end_frame`이나 `reference_images`를 제공하지 않을 경우, `duration`은 5초 또는 10초로만 설정할 수 있습니다.
* 모든 입력 이미지(`first_frame`, `end_frame`, 그리고 모든 `reference_images`)는 가로와 세로 최소 300픽셀 이상이어야 합니다.
* 모든 입력 이미지의 종횡비는 1:2.5에서 2.5:1 사이여야 합니다.
* `reference_images` 입력을 통해 제공할 수 있는 이미지는 최대 6개입니다.
* `prompt` 텍스트는 1~2500자 사이여야 합니다(스토리보드가 활성화된 경우 0자도 허용됨).
* 스토리보드가 활성화된 경우, 모든 스토리보드 구간의 총 지속 시간은 전체 `duration` 값과 같아야 합니다.
* 각 스토리보드 프롬프트는 1~512자 사이여야 합니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-----------|-----------|-------------|
| `output` | VIDEO | 생성된 비디오 파일입니다. |
