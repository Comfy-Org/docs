Vidu2 텍스트-영상 생성 노드는 텍스트 설명을 기반으로 동영상을 생성합니다. 외부 API와 연결하여 프롬프트에 따라 동영상 콘텐츠를 생성하며, 동영상의 길이, 시각적 스타일 및 형식을 제어할 수 있습니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `model` | COMBO | 예 | `"viduq2"` | 동영상 생성에 사용할 AI 모델입니다. 현재 사용 가능한 모델은 하나뿐입니다. |
| `prompt` | STRING | 예 | - | 동영상 생성을 위한 텍스트 설명으로, 최대 길이는 2000자입니다. |
| `duration` | INT | 아니요 | 1~10 | 생성된 동영상의 길이(초)입니다. 슬라이더를 통해 값을 조정할 수 있습니다(기본값: 5). |
| `seed` | INT | 아니요 | 0~2147483647 | 생성의 무작위성을 제어하는 데 사용되는 숫자로, 재현 가능한 결과를 얻을 수 있습니다. 생성 후에도 제어 가능합니다(기본값: 1). |
| `aspect_ratio` | COMBO | 아니요 | `"16:9"`<br />`"9:16"`<br />`"3:4"`<br />`"4:3"`<br />`"1:1"` | 동영상의 가로와 세로 비율입니다. |
| `resolution` | COMBO | 아니요 | `"720p"`<br />`"1080p"` | 생성된 동영상의 픽셀 크기입니다. 이는 고급 매개변수입니다. |
| `background_music` | BOOLEAN | 아니요 | - | 생성된 동영상에 배경 음악을 추가할지 여부(기본값: False). 이는 고급 매개변수입니다. |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `output` | VIDEO | 생성된 동영상 파일입니다. |
