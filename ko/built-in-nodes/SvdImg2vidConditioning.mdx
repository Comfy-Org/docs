이 노드는 비디오 생성 작업을 위한 컨디셔닝 데이터를 생성하도록 설계되었으며, 특히 SVD_img2vid 모델과 함께 사용하도록 맞춰져 있습니다. 초기 이미지, 비디오 매개변수, VAE 모델 등 다양한 입력을 받아 비디오 프레임 생성을 안내하는 데 사용할 수 있는 컨디셔닝 데이터를 생성합니다.

## 입력

| 매개변수             | Comfy 데이터형        | 설명 |
|----------------------|--------------------|-------------|
| `clip_vision`         | `CLIP_VISION`      | 초기 이미지에서 시각적 특징을 인코딩하는 데 사용되는 CLIP 비전 모델을 나타냅니다. 비디오 생성을 위해 이미지의 내용과 맥락을 이해하는 데 중요한 역할을 합니다. |
| `init_image`          | `IMAGE`            | 비디오 생성이 시작될 초기 이미지로, 비디오 생성 과정의 출발점 역할을 합니다. |
| `vae`                 | `VAE`              | 초기 이미지를 잠재 공간으로 인코딩하는 데 사용되는 변분 오토인코더(VAE) 모델로, 일관성 있고 연속적인 비디오 프레임 생성을 지원합니다. |
| `width`               | `INT`              | 생성할 비디오 프레임의 원하는 너비로, 비디오의 해상도를 맞춤화할 수 있습니다. |
| `height`              | `INT`              | 비디오 프레임의 원하는 높이로, 비디오의 종횡비와 해상도를 제어할 수 있습니다. |
| `video_frames`        | `INT`              | 비디오에 생성할 프레임의 수를 지정하며, 비디오의 길이를 결정합니다. |
| `motion_bucket_id`    | `INT`              | 비디오 생성 시 적용할 동작 유형을 분류하기 위한 식별자로, 다이나믹하고 흥미로운 비디오 제작에 도움을 줍니다. |
| `fps`                 | `INT`              | 비디오의 초당 프레임 수(fps)로, 생성된 비디오의 부드러움과 사실성을 좌우합니다. |
| `augmentation_level`  | `FLOAT`            | 초기 이미지에 적용되는 증강 수준을 제어하는 매개변수로, 생성된 비디오 프레임의 다양성과 변동성을 조절합니다. |

## 출력

| 매개변수     | Comfy 데이터형        | 설명 |
|---------------|--------------------|-------------|
| `positive`    | `CONDITIONING`     | 비디오 생성 과정을 원하는 방향으로 이끌기 위한 인코딩된 특징과 매개변수로 구성된 양성 컨디셔닝 데이터입니다. |
| `negative`    | `CONDITIONING`     | 양성 컨디셔닝과 대비되는 음성 컨디셔닝 데이터로, 생성된 비디오에서 특정 패턴이나 특징을 피하는 데 사용할 수 있습니다. |
| `latent`      | `LATENT`           | 비디오 각 프레임에 대해 생성된 잠재 표현으로, 비디오 생성 과정의 기초적인 구성 요소 역할을 합니다. |
