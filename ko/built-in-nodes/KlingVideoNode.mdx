> 이 문서는 AI로 생성되었습니다. 오류를 발견하거나 개선할 점이 있다면 언제든지 기여해 주세요! [GitHub에서 편집하기](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/KlingVideoNode/en.md)

이 노드는 Kling V3 모델을 사용해 동영상을 생성합니다. 두 가지 주요 모드를 지원합니다: 텍스트를 입력하면 동영상이 생성되는 텍스트-투-비디오 모드와, 기존 이미지를 애니메이션화하는 이미지-투-비디오 모드입니다. 또한 각 부분마다 서로 다른 프롬프트를 사용한 다중 세그먼트 동영상 생성과, 선택적으로 함께 제공할 오디오 생성과 같은 고급 기능도 제공합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `multi_shot` | COMBO | 예 | `"disabled"`<br />`"1 storyboard"`<br />`"2 storyboards"`<br />`"3 storyboards"`<br />`"4 storyboards"`<br />`"5 storyboards"`<br />`"6 storyboards"` | 단일 동영상 생성 여부 또는 각 스토리보드별 프롬프트와 지속시간을 설정한 일련의 세그먼트 생성 여부를 제어합니다. "disabled"가 아닐 경우, 각 스토리보드의 프롬프트와 지속시간을 설정하는 추가 입력 항목이 나타납니다. |
| `generate_audio` | BOOLEAN | 예 | `True` / `False` | 활성화 시, 노드는 동영상용 오디오를 생성합니다. 기본값은 `True`입니다. |
| `model` | COMBO | 예 | `"kling-v3"` | 모델 및 관련 설정입니다. 이 옵션을 선택하면 `resolution`과 `aspect_ratio` 하위 매개변수가 표시됩니다. |
| `model.resolution` | COMBO | 예 | `"4k"`<br />`"1080p"`<br />`"720p"` | 생성될 동영상의 해상도입니다. 이 설정은 `model`이 "kling-v3"로 설정된 경우에만 사용 가능합니다. |
| `model.aspect_ratio` | COMBO | 예 | `"16:9"`<br />`"9:16"`<br />`"1:1"` | 생성될 동영상의 종횡비입니다. 이미지가 `start_frame`에 제공된 경우(이미지-투-비디오 모드) 이 설정은 무시됩니다. `model`이 "kling-v3"로 설정된 경우에만 사용 가능합니다. |
| `seed` | INT | 예 | 0부터 2147483647까지 | 생성을 위한 시드 값입니다. 이 값을 변경하면 노드가 다시 실행되지만 결과는 비결정적입니다. 기본값은 `0`입니다. |
| `start_frame` | IMAGE | 아니요 | - | 선택적 시작 이미지입니다. 연결되면 노드는 텍스트-투-비디오 모드에서 이미지-투-비디오 모드로 전환되어 제공된 이미지를 애니메이션화합니다. |

**`multi_shot` 모드의 입력:**

* `multi_shot`이 **"disabled"**로 설정된 경우, 다음 입력 항목이 나타납니다:
  * `prompt` (STRING): 동영상의 주요 텍스트 설명입니다. 필수입니다. 1~2500자 사이여야 합니다.
  * `negative_prompt` (STRING): 동영상에 나타나지 말아야 할 내용을 설명하는 텍스트입니다. 선택적입니다.
  * `duration` (INT): 동영상의 길이(초). 3~15초 사이여야 합니다. 기본값은 `5`입니다.
* `multi_shot`이 스토리보드 옵션으로 설정된 경우(예: `"3 storyboards"`), 각 스토리보드 세그먼트에 대한 입력 항목이 나타납니다(예: `storyboard_1_prompt`, `storyboard_1_duration`). 각 프롬프트는 1~512자 사이여야 합니다. **모든 스토리보드 지속시간의 합계**는 3~15초 사이여야 합니다.

**제약사항:**

* `start_frame`이 연결되지 않은 경우 노드는 **텍스트-투-비디오** 모드로 작동하며, 이 모드에서는 `model.aspect_ratio` 설정이 사용됩니다.
* `start_frame`이 연결된 경우 노드는 **이미지-투-비디오** 모드로 작동하며, `model.aspect_ratio` 설정은 무시됩니다. 입력 이미지는 최소 300x300 픽셀이어야 하고, 종횡비는 1:2.5에서 2.5:1 사이여야 합니다.
* 스토리보드 모드(`multi_shot`이 "disabled"가 아닐 때)에서는 주요 `prompt`와 `negative_prompt` 입력 항목이 숨겨져 사용되지 않습니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `video` | VIDEO | 생성된 동영상 파일입니다. |
