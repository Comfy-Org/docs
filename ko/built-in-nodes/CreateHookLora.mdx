Create Hook LoRA 노드는 모델에 LoRA(Low-Rank Adaptation) 수정을 적용하기 위한 훅 객체를 생성합니다. 지정된 LoRA 파일을 로드하고 모델과 CLIP의 강도를 조정할 수 있는 훅을 생성한 다음, 이 노드에 전달된 기존 훅들과 결합합니다. 이 노드는 이전에 로드된 LoRA 파일을 캐싱하여 중복 작업을 피함으로써 LoRA 로딩을 효율적으로 관리합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `lora_name` | 문자열 | 예 | 여러 옵션 제공 | loras 디렉터리에서 로드할 LoRA 파일의 이름 |
| `strength_model` | 부동소수점 | 예 | -20.0부터 20.0까지 | 모델 조정을 위한 강도 승수 (기본값: 1.0) |
| `strength_clip` | 부동소수점 | 예 | -20.0부터 20.0까지 | CLIP 조정을 위한 강도 승수 (기본값: 1.0) |
| `prev_hooks` | HOOKS | 아니요 | N/A | 새 LoRA 훅들과 결합할 기존 훅 그룹 (선택사항) |

**매개변수 제약사항:**

- `strength_model`와 `strength_clip` 모두 0으로 설정된 경우, 노드는 새 LoRA 훅 생성을 건너뛰고 기존 훅을 그대로 반환합니다.
- 노드는 마지막에 로드된 LoRA 파일을 캐싱하여 동일한 LoRA가 반복적으로 사용될 때 성능을 최적화합니다.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `HOOKS` | HOOKS | 결합된 LoRA 훅들과 기존 훅들을 포함하는 훅 그룹 |
