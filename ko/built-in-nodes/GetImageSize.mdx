GetImageSize 노드는 입력 이미지에서 크기와 배치 정보를 추출합니다. 노드 인터페이스에 진행률 텍스트로 해당 정보를 표시하며, 이미지의 너비, 높이, 배치 크기를 반환합니다. 원본 이미지 데이터는 변경되지 않은 채 그대로 전달됩니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `image` | IMAGE | 예 | - | 크기 정보를 추출할 입력 이미지 |

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `width` | INT | 입력 이미지의 너비(픽셀 단위) |
| `height` | INT | 입력 이미지의 높이(픽셀 단위) |
| `batch_size` | INT | 배치 내 이미지 수 |
