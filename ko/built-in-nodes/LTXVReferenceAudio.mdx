LTXV 참조 오디오 노드는 오디오 생성 시 화자 식별 전송에 사용됩니다. 이 노드는 참조 오디오 클립을 모델의 조건부 입력으로 인코딩하여 생성된 오디오가 화자의 음성 특성을 갖도록 합니다. 또한 식별 안내를 적용할 수도 있는데, 이는 추가 처리 단계를 실행하여 화자 식별 효과를 강화합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `model` | MODEL | 예 | - | 식별 안내로 패치할 모델입니다. |
| `positive` | CONDITIONING | 예 | - | 양의 조건부 입력입니다. |
| `negative` | CONDITIONING | 예 | - | 음의 조건부 입력입니다. |
| `reference_audio` | AUDIO | 예 | - | 화자 식별을 전송할 참조 오디오 클립입니다. 약 5초 권장(훈련 기간). 더 짧거나 긴 클립은 음성 식별 전송을 저하시킬 수 있습니다. |
| `audio_vae` | VAE | 예 | - | 참조 오디오를 인코딩하는 LTXV 오디오 VAE입니다. |
| `identity_guidance_scale` | FLOAT | 아니요 | 0.0 - 100.0 | 식별 안내의 강도입니다. 각 단계마다 참조 없이 추가 전방 통과를 실행하여 화자 식별을 강화합니다. 0으로 설정하면 비활성화됩니다(추가 통과 없음). (기본값: 3.0) |
| `start_percent` | FLOAT | 아니요 | 0.0 - 1.0 | 식별 안내가 활성화되는 시그마 범위의 시작점입니다. (기본값: 0.0) |
| `end_percent` | FLOAT | 아니요 | 0.0 - 1.0 | 식별 안내가 활성화되는 시그마 범위의 끝점입니다. (기본값: 1.0) |

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `model` | MODEL | 식별 안내 기능으로 패치된 모델입니다. |
| `positive` | CONDITIONING | 이제 인코딩된 참조 오디오 데이터를 포함한 양의 조건부 입력입니다. |
| `negative` | CONDITIONING | 이제 인코딩된 참조 오디오 데이터를 포함한 음의 조건부 입력입니다. |
