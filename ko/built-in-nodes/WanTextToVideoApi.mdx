Wan Text to Video 노드는 텍스트 설명을 기반으로 비디오 콘텐츠를 생성합니다. AI 모델을 사용해 프롬프트로부터 비디오를 만들며, 다양한 비디오 크기와 지속 시간, 선택적 오디오 입력을 지원합니다. 필요 시 자동으로 오디오를 생성하며, 프롬프트 향상과 워터마크 추가 옵션도 제공합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `model` | COMBO | 예 | "wan2.5-t2v-preview"<br />"wan2.6-t2v" | 사용할 모델 (기본값: "wan2.6-t2v") |
| `prompt` | STRING | 예 | - | 요소와 시각적 특징을 설명하는 프롬프트. 영어와 중국어 지원 (기본값: "") |
| `negative_prompt` | STRING | 아니요 | - | 피해야 할 내용을 설명하는 부정적 프롬프트 (기본값: "") |
| `size` | COMBO | 아니요 | "480p: 1:1 (624x624)"<br />"480p: 16:9 (832x480)"<br />"480p: 9:16 (480x832)"<br />"720p: 1:1 (960x960)"<br />"720p: 16:9 (1280x720)"<br />"720p: 9:16 (720x1280)"<br />"720p: 4:3 (1088x832)"<br />"720p: 3:4 (832x1088)"<br />"1080p: 1:1 (1440x1440)"<br />"1080p: 16:9 (1920x1080)"<br />"1080p: 9:16 (1080x1920)"<br />"1080p: 4:3 (1632x1248)"<br />"1080p: 3:4 (1248x1632)" | 비디오 해상도와 종횡비 (기본값: "720p: 1:1 (960x960)") |
| `duration` | INT | 아니요 | 5-15 (5 단위) | 비디오 지속 시간(초). 15초 지속 시간은 Wan 2.6 모델에서만 가능 (기본값: 5) |
| `audio` | AUDIO | 아니요 | - | 오디오는 명확하고 큰 목소리가 포함되어야 하며, 잡음이나 배경 음악이 없어야 합니다 |
| `seed` | INT | 아니요 | 0-2147483647 | 생성에 사용할 시드 (기본값: 0) |
| `generate_audio` | BOOLEAN | 아니요 | - | 오디오 입력이 제공되지 않으면 자동으로 오디오를 생성합니다 (기본값: False) |
| `prompt_extend` | BOOLEAN | 아니요 | - | AI 도움을 받아 프롬프트를 향상시킬지 여부 (기본값: True) |
| `watermark` | BOOLEAN | 아니요 | - | 결과에 AI가 생성한 워터마크를 추가할지 여부 (기본값: False) |
| `shot_type` | COMBO | 아니요 | "single"<br />"multi" | 생성된 비디오의 샷 유형을 지정합니다. 즉, 비디오가 하나의 연속 샷인지, 컷이 있는 여러 샷인지입니다. 이 매개변수는 prompt_extend가 True일 때만 적용됩니다 (기본값: "single") |

**참고:** Wan 2.6 모델은 480p 해상도를 지원하지 않습니다. 15초 지속 시간은 Wan 2.6 모델에서만 지원됩니다. 오디오 입력을 제공할 경우, 지속 시간은 3.0초에서 29.0초 사이여야 하며, 배경 잡음이나 음악이 없는 명확한 목소리를 포함해야 합니다.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `output` | VIDEO | 입력 매개변수를 기반으로 생성된 비디오 |
