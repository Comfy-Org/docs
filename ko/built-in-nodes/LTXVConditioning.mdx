LTXVConditioning 노드는 비디오 생성 모델을 위한 양성 및 음성 조건 입력에 프레임레이트 정보를 추가합니다. 기존의 조건 데이터를 받아들여 지정된 프레임레이트 값을 두 조건 세트 모두에 적용하여 비디오 모델 처리에 적합하게 만듭니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `positive` | CONDITIONING | 예 | - | 프레임레이트 정보를 받게 될 양성 조건 입력 |
| `negative` | CONDITIONING | 예 | - | 프레임레이트 정보를 받게 될 음성 조건 입력 |
| `frame_rate` | FLOAT | 예 | 0.0 - 1000.0 | 두 조건 세트에 적용할 프레임레이트 값 (기본값: 25.0) |

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `positive` | CONDITIONING | 프레임레이트 정보가 적용된 양성 조건 |
| `negative` | CONDITIONING | 프레임레이트 정보가 적용된 음성 조건 |
