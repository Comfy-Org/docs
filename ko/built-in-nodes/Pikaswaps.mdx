Pika Swaps 노드는 동영상에서 객체나 영역을 새로운 이미지로 교체합니다. 마스크를 사용해 교체할 영역을 정의하며, 노드는 동영상 시퀀스 전반에 걸쳐 지정된 내용을 매끄럽게 교체합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `video` | VIDEO | 예 | - | 객체를 교체할 동영상입니다. |
| `image` | IMAGE | 예 | - | 동영상에서 마스크로 교체할 객체를 대체하는 데 사용되는 이미지입니다. |
| `mask` | MASK | 예 | - | 마스크를 사용해 동영상에서 교체할 영역을 정의하세요. |
| `prompt_text` | STRING | 예 | - | 교체하고자 하는 내용을 설명하는 텍스트 프롬프트입니다. |
| `negative_prompt` | STRING | 예 | - | 교체 과정에서 피해야 할 내용을 설명하는 텍스트 프롬프트입니다. |
| `seed` | INT | 예 | 0부터 4294967295까지 | 일관된 결과를 위해 사용하는 난수 시드 값입니다. |

**참고:** 이 노드는 모든 입력 매개변수가 제공되어야 합니다. `video`, `image`, `mask`는 함께 작동하여 교체 작업을 정의하며, 마스크는 동영상의 어느 영역을 제공된 이미지로 교체할지 지정합니다.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `output` | VIDEO | 지정된 객체나 영역이 교체된 처리된 동영상입니다. |
