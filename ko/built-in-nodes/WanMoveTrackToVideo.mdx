WanMoveTrackToVideo 노드는 동영상 생성을 위해 조건부 및 잠재 공간 데이터를 준비하며, 선택적 모션 트래킹 정보를 포함합니다. 이 노드는 시작 이미지 시퀀스를 잠재 표현으로 인코딩하고, 객체 트랙의 위치 데이터를 혼합하여 생성된 동영상의 움직임을 안내할 수 있습니다. 노드는 수정된 긍정적 및 부정적 조건부와 비디오 모델을 위한 빈 잠재 텐서를 출력합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `positive` | CONDITIONING | 예 | - | 수정될 긍정적 조건부 입력입니다. |
| `negative` | CONDITIONING | 예 | - | 수정될 부정적 조건부 입력입니다. |
| `vae` | VAE | 예 | - | 시작 이미지를 잠재 공간으로 인코딩하는 데 사용되는 VAE 모델입니다. |
| `tracks` | TRACKS | 아니요 | - | 객체 경로를 포함한 선택적 모션 트래킹 데이터입니다. |
| `strength` | FLOAT | 아니요 | 0.0 - 100.0 | 트랙 조건부의 강도입니다. (기본값: 1.0) |
| `width` | INT | 아니요 | 16 - MAX_RESOLUTION | 출력 동영상의 너비입니다. 16으로 나누어져야 합니다. (기본값: 832) |
| `height` | INT | 아니요 | 16 - MAX_RESOLUTION | 출력 동영상의 높이입니다. 16으로 나누어져야 합니다. (기본값: 480) |
| `length` | INT | 아니요 | 1 - MAX_RESOLUTION | 동영상 시퀀스의 프레임 수입니다. (기본값: 81) |
| `batch_size` | INT | 아니요 | 1 - 4096 | 잠재 출력의 배치 크기입니다. (기본값: 1) |
| `start_image` | IMAGE | 예 | - | 인코딩할 시작 이미지 또는 이미지 시퀀스입니다. |
| `clip_vision_output` | CLIPVISIONOUTPUT | 아니요 | - | 조건부에 추가할 선택적 CLIP 비전 모델 출력입니다. |

**참고:** `strength` 매개변수는 `tracks`가 제공된 경우에만 효과를 발휘합니다. `tracks`가 제공되지 않거나 `strength`가 0.0인 경우, 트랙 조건부는 적용되지 않습니다. `start_image`는 조건부를 위한 잠재 이미지와 마스크를 생성하는 데 사용되며, 제공되지 않을 경우 노드는 조건부를 그대로 전달하고 빈 잠재를 출력합니다.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `positive` | CONDITIONING | 수정된 긍정적 조건부로, 잠재 이미지 합치기(`concat_latent_image`), 마스크 합치기(`concat_mask`), CLIP 비전 출력(`clip_vision_output`)을 포함할 수 있습니다. |
| `negative` | CONDITIONING | 수정된 부정적 조건부로, 잠재 이미지 합치기(`concat_latent_image`), 마스크 합치기(`concat_mask`), CLIP 비전 출력(`clip_vision_output`)을 포함할 수 있습니다. |
| `latent` | LATENT | `batch_size`, `length`, `height`, `width` 입력에 따라 형상이 정해진 빈 잠재 텐서입니다. |
