GITSScheduler 노드는 GITS(Generative Iterative Time Steps) 샘플링 방식을 위한 노이즈 스케줄 시그마를 생성합니다. 이 노드는 계수 매개변수와 단계 수를 기반으로 시그마 값을 계산하며, 선택적으로 덴오이징 요소를 추가해 사용하는 총 단계 수를 줄일 수 있습니다. 노드는 사전 정의된 노이즈 레벨과 보간법을 사용해 최종 시그마 스케줄을 생성합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `coeff` | FLOAT | 예 | 0.80 - 1.50 | 노이즈 스케줄 곡선을 제어하는 계수 값 (기본값: 1.20) |
| `steps` | INT | 예 | 2 - 1000 | 시그마를 생성할 총 샘플링 단계 수 (기본값: 10) |
| `denoise` | FLOAT | 예 | 0.0 - 1.0 | 사용하는 단계 수를 줄이는 덴오이징 요소 (기본값: 1.0) |

**참고:** `denoise`를 0.0으로 설정하면 노드는 빈 텐서를 반환합니다. `denoise`가 1.0보다 작을 경우, 실제 사용되는 단계 수는 `round(steps * denoise)`로 계산됩니다. 단계 수가 20 이상인 경우, 노드는 로그-선형 보간법을 사용해 사전 정의된 노이즈 레벨을 원하는 단계 수까지 확장합니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-----------|-----------|-------------|
| `sigmas` | SIGMAS | 노이즈 스케줄을 위한 생성된 시그마 값 |
