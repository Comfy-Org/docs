StringContains 노드는 주어진 문자열에 특정 하위 문자열이 포함되어 있는지 확인합니다. 이 노드는 대소문자 구분 또는 대소문자 무시 방식으로 검사를 수행하며, 하위 문자열이 본문 문자열 내에 존재하는지 여부를 나타내는 불리언 결과를 반환합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `string` | STRING | 예 | - | 검색할 본문 문자열 |
| `substring` | STRING | 예 | - | 본문 문자열 내에서 검색할 텍스트 |
| `case_sensitive` | BOOLEAN | 아니요 | - | 검색이 대소문자 구분인지 여부를 결정합니다(기본값: true) |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `contains` | BOOLEAN | 하위 문자열이 문자열에 포함되어 있으면 true, 그렇지 않으면 false를 반환합니다 |
