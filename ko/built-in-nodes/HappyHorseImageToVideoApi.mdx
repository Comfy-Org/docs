## 개요

이 노드는 HappyHorse 모델을 사용하여 단일 시작 이미지로부터 짧은 동영상을 생성합니다. 첫 프레임 이미지와 원하는 움직임과 장면을 설명하는 텍스트 프롬프트를 제공하면, 해당 이미지에서 이어지는 동영상을 만들어냅니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `model` | COMBO | 예 | `"happyhorse-1.0-i2v"` | 동영상 생성에 사용할 HappyHorse 모델입니다. |
| `model.prompt` | STRING | 아니요 | N/A | 요소와 시각적 특징을 설명하는 프롬프트입니다. 영어와 중국어를 지원합니다. (기본값: "") |
| `model.resolution` | COMBO | 예 | `"720P"`<br />`"1080P"` | 출력 동영상의 해상도입니다. (기본값: "720P") |
| `model.duration` | INT | 예 | 3~15 | 생성된 동영상의 지속 시간(초)입니다. (기본값: 5) |
| `first_frame` | IMAGE | 예 | N/A | 첫 프레임 이미지입니다. 출력 비율은 이 이미지에서 파생됩니다. |
| `seed` | INT | 아니요 | 0~2147483647 | 생성에 사용할 시드입니다. (기본값: 0) |
| `watermark` | BOOLEAN | 아니요 | True / False | 결과에 AI가 생성한 워터마크를 추가할지 여부입니다. (기본값: False) |

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `video` | VIDEO | 생성된 동영상 파일입니다. |
