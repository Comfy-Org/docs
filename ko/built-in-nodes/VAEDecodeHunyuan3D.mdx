VAEDecodeHunyuan3D 노드는 VAE 디코더를 사용하여 잠재 표현을 3D 복셀 데이터로 변환합니다. 이 노드는 구성 가능한 청크 처리와 해상도 설정을 통해 잠재 샘플을 VAE 모델을 거쳐 3D 애플리케이션에 적합한 볼륨 데이터로 생성합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `samples` | LATENT | 예 | - | 3D 복셀 데이터로 디코딩할 잠재 표현 |
| `vae` | VAE | 예 | - | 잠재 샘플을 디코딩하는 데 사용되는 VAE 모델 |
| `num_chunks` | INT | 예 | 1000-500000 | 메모리 관리를 위해 처리를 나누는 청크 수 (기본값: 8000) |
| `octree_resolution` | INT | 예 | 16-512 | 3D 복셀 생성에 사용되는 옥트리 구조의 해상도 (기본값: 256) |

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `voxels` | VOXEL | 디코딩된 잠재 표현에서 생성된 3D 복셀 데이터 |
