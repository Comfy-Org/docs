| 매개변수 | 데이터 타입 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `positive` | CONDITIONING | 예 | - | 처리할 주요 양수 조건 입력 |
| `negative` | CONDITIONING | 예 | - | 처리할 주요 음수 조건 입력 |
| `positive_DEFAULT` | CONDITIONING | 예 | - | 백업용으로 사용될 기본 양수 조건값 |
| `negative_DEFAULT` | CONDITIONING | 예 | - | 백업용으로 사용될 기본 음수 조건값 |
