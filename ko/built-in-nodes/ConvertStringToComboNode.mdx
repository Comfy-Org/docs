문자열을 콤보로 변환하는 노드는 텍스트 문자열을 입력으로 받아 콤보 데이터 유형으로 변환합니다. 이를 통해 텍스트 값을 콤보 입력을 요구하는 다른 노드의 선택지로 사용할 수 있습니다. 이 노드는 문자열 값을 그대로 전달하되 데이터 유형만 변경합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `string` | STRING | 예 | N/A | 콤보 유형으로 변환할 텍스트 문자열. |

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `output` | COMBO | 입력된 문자열이 이제 콤보 데이터 유형으로 포맷된 상태. |
