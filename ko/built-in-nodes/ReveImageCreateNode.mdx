> 이 문서는 AI로 생성되었습니다. 오류를 발견하거나 개선할 점이 있다면 언제든지 기여해 주세요! [GitHub에서 편집하기](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/ReveImageCreateNode/en.md)

Reve Image Create 노드는 Reve AI 모델을 사용해 텍스트 설명으로 이미지를 생성합니다. 이 노드는 텍스트 프롬프트를 Reve API에 전송하고 생성된 이미지를 반환합니다. 이미지의 종횡비를 제어하고, 업스케일링 및 배경 제거와 같은 선택적 후처리 효과를 적용할 수 있습니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `prompt` | 문자열 | 예 | 없음 | 원하는 이미지의 텍스트 설명. 최대 2560자. |
| `model` | 콤보 | 예 | `"reve-create@20250915"` | 생성에 사용할 모델 버전. 이 매개변수 내에서 제공되는 옵션 중에서 종횡비를 선택합니다. |
| `aspect_ratio` | 콤보 | 예 | `"3:2"`<br />`"16:9"`<br />`"9:16"`<br />`"2:3"`<br />`"4:3"`<br />`"3:4"`<br />`"1:1"` | 생성할 이미지의 원하는 종횡비. |
| `test_time_scaling` | 콤보 | 예 | `"disabled"`<br />`"enabled"` | 테스트 시간 스케일링을 활성화하거나 비활성화합니다. 이 기능은 이미지 품질을 높이지만 생성 시간이 길어질 수 있습니다. |
| `upscale` | 콤보 | 아니요 | `"disabled"`<br />`"enabled"` | 업스케일링 후처리 단계를 활성화하거나 비활성화합니다. 활성화 시 업스케일링 요인을 선택해야 합니다. |
| `upscale_factor` | 콤보 | 아니요 | `2`<br />`3`<br />`4` | 이미지 해상도를 증가시키는 요인. 이 매개변수는 `upscale`이 `"enabled"`로 설정된 경우에만 활성화됩니다. |
| `remove_background` | 부울 | 아니요 | 없음 | 활성화 시 생성된 이미지에 배경 제거 후처리 단계를 적용합니다. |
| `seed` | 정수 | 아니요 | 0부터 2147483647까지 | 노드가 다시 실행될지 여부를 제어하는 시드 값입니다. 참고: 시드 값과 관계없이 결과는 비결정적입니다. 기본값: 0. |

**참고:** `upscale_factor` 매개변수는 `upscale` 매개변수가 `"enabled"`로 설정된 경우에만 작동합니다. `seed` 매개변수는 결정적 출력을 보장하지 않습니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `image` | 이미지 | 입력 프롬프트를 기반으로 Reve 모델이 생성한 이미지. |
