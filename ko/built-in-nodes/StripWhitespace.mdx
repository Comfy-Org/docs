이 노드는 텍스트 문자열의 시작과 끝에서 추가적인 공백, 탭 또는 줄 바꿈을 모두 제거합니다. 텍스트 입력을 받아 앞뒤 공백이 제거된 정제된 버전을 반환합니다.

**참고: 이 노드는 더 이상 사용되지 않으며, Trim Text 노드로 대체되었습니다.**

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `text` | STRING | 예 | N/A | 앞뒤 공백을 제거할 텍스트 문자열입니다. |

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `text` | STRING | 처리된 텍스트로, 앞뒤 공백 문자가 모두 제거된 상태입니다. |
