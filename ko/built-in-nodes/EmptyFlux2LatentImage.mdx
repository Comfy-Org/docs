EmptyFlux2LatentImage 노드는 빈, 비어 있는 잠재 표현을 생성합니다. 이 노드는 0으로 채워진 텐서를 생성하며, 이는 Flux 모델의 노이즈 제거 과정의 시작점 역할을 합니다. 잠재 표현의 차원은 입력되는 너비와 높이에 따라 결정되며, 16으로 나누어 축소됩니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `width` | INT | 예 | 16부터 8192까지 | 생성할 최종 이미지의 너비입니다. 잠재 표현의 너비는 이 값에서 16으로 나눈 결과입니다. 기본값은 1024입니다. |
| `height` | INT | 예 | 16부터 8192까지 | 생성할 최종 이미지의 높이입니다. 잠재 표현의 높이는 이 값에서 16으로 나눈 결과입니다. 기본값은 1024입니다. |
| `batch_size` | INT | 아니요 | 1부터 4096까지 | 한 번에 생성할 잠재 표현 샘플의 수입니다. 기본값은 1입니다. |

**참고:** `width`와 `height` 입력값은 반드시 16으로 나누어 떨어져야 하며, 노드가 내부적으로 이를 16으로 나누어 잠재 표현의 차원을 생성하기 때문입니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `samples` | LATENT | 0으로 채워진 잠재 텐서입니다. 형태는 `[batch_size, 128, height // 16, width // 16]`입니다. |
