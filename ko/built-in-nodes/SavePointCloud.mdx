Save Point Cloud 노드는 3D 포인트 클라우드 파일을 출력 디렉토리에 저장하고, 선택적으로 3D 뷰어용 미리보기 데이터를 제공합니다. 파일 이름 지정 및 저장을 처리하면서, 표시 목적으로 카메라 및 모델 정보를 전달합니다.

## 입력

| 매개변수 | 설명 | 데이터 타입 | 필수 | 범위 |
|-----------|-------------|-----------|----------|-------|
| `model_3d` | 포인트 클라우드 파일(.ply) | FILE3D_POINT_CLOUD_ANY 또는 FILE3D_PLY | 예 | - |
| `filename_prefix` | 저장된 파일 이름의 접두사(기본값: "3d/ComfyUI") | STRING | 예 | - |
| `viewport_state` | 카메라 및 모델 정보를 포함한 현재 뷰포트 상태 | LOAD3D | 예 | - |
| `model_3d_info` | 3D 뷰어용 추가 모델 정보 | LOAD3D_MODEL_INFO | 아니요 | - |
| `camera_info` | 3D 뷰어용 카메라 정보 | LOAD3D_CAMERA | 아니요 | - |
| `width` | 미리보기 표시의 너비(픽셀 단위, 기본값: 1024) | INT | 예 | 1 ~ 4096 |
| `height` | 미리보기 표시의 높이(픽셀 단위, 기본값: 1024) | INT | 예 | 1 ~ 4096 |

## 출력

| 출력 이름 | 설명 | 데이터 타입 |
|-------------|-------------|-----------|
| `model_3d` | 저장된 포인트 클라우드 파일 | FILE3D_POINT_CLOUD_ANY |
| `model_3d_info` | 3D 뷰어용 모델 정보 | LOAD3D_MODEL_INFO |
| `camera_info` | 3D 뷰어용 카메라 정보 | LOAD3D_CAMERA |
| `width` | 미리보기 표시의 너비 | INT |
| `height` | 미리보기 표시의 높이 | INT |
