Save Point Cloud 노드는 3D 포인트 클라우드 파일을 ComfyUI 출력 디렉토리에 저장합니다. 또한 포인트 클라우드 데이터와 선택적 카메라 및 모델 정보를 워크플로우의 다른 노드에서 사용할 수 있도록 전달합니다.

## 입력

| 매개변수 | 설명 | 데이터 타입 | 필수 여부 | 범위 |
|-----------|-------------|-----------|----------|-------|
| `model_3d` | 포인트 클라우드 파일(.ply) | FILE3D | 예 | - |
| `filename_prefix` | 저장될 파일 이름의 접두사(기본값: "3d/ComfyUI") | STRING | 예 | - |
| `viewport_state` | 3D 뷰포트 노드의 상태 | LOAD3D | 예 | - |
| `model_3d_info` | 고급 사용을 위한 선택적 3D 모델 정보 | LOAD3DMODELINFO | 아니요 | - |
| `camera_info` | 고급 사용을 위한 선택적 카메라 정보 | LOAD3DCAMERA | 아니요 | - |
| `width` | 미리보기 이미지의 너비(픽셀 단위, 기본값: 1024) | INT | 예 | 최소: 1, 최대: 4096, 단계: 1 |
| `height` | 미리보기 이미지의 높이(픽셀 단위, 기본값: 1024) | INT | 예 | 최소: 1, 최대: 4096, 단계: 1 |

## 출력

| 출력 이름 | 설명 | 데이터 타입 |
|-------------|-------------|-----------|
| `model_3d` | 저장된 포인트 클라우드 파일 | FILE3D |
| `model_3d_info` | 3D 모델에 대한 정보 | LOAD3DMODELINFO |
| `camera_info` | 뷰포트용 카메라 정보 | LOAD3DCAMERA |
| `width` | 미리보기 이미지의 너비 | INT |
| `height` | 미리보기 이미지의 높이 | INT |
