## 개요

Rodin Gen-2.5 API를 사용하여 텍스트 프롬프트로부터 3D 모델을 생성합니다. 생성 속도와 출력 품질 간의 균형을 맞추기 위해 다양한 품질 모드(빠른, 일반, 극고) 중에서 선택할 수 있습니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `prompt` | 문자열 | 예 | 최대 2500자 | 생성하고자 하는 3D 모델을 설명하는 텍스트 프롬프트입니다. |
| `mode` | 콤보 | 예 | `"빠른"`<br />`"일반"`<br />`"극고"` | 생성 품질과 속도 모드입니다. "빠른"은 가장 빠르며, "극고"는 가장 높은 품질을 제공하지만 시간이 더 걸립니다. |
| `material` | 콤보 | 예 | `"PBR"`<br />`"무광"`<br />`"유광"` | 생성된 3D 모델의 재질 스타일입니다. |
| `geometry_file_format` | 콤보 | 예 | `"glb"`<br />`"obj"`<br />`"stl"`<br />`"usdz"` | 출력 3D 모델의 파일 형식입니다. |
| `texture_mode` | 콤보 | 예 | `"없음"`<br />`"생성됨"`<br />`"생성됨+HD"` | 텍스처 생성 모드입니다. "없음"은 텍스처를 생성하지 않으며, "생성됨"은 표준 텍스처를 생성하고, "생성됨+HD"는 고화질 텍스처를 생성합니다. |
| `seed` | 정수 | 예 | 0부터 2147483647까지 | 재현 가능한 결과를 위한 난수 시드입니다. 동일한 입력과 동일한 시드를 사용하면 동일한 출력이 생성됩니다. |
| `TAPose` | 부울 | 예 | true / false | 생성된 모델에 T자세(팔을 벌린 상태)를 적용할지 여부입니다. |
| `hd_texture` | 부울 | 예 | true / false | 모델에 고화질 텍스처를 생성할지 여부입니다. |
| `texture_delight` | 부울 | 예 | true / false | 모델에 텍스처 딜라이트(강화된 텍스처 품질)를 적용할지 여부입니다. |
| `addon_highpack` | 부울 | 예 | true / false | 표준 버전 외에도 모델의 고다각형 버전을 생성할지 여부입니다. |
| `bbox_width` | 정수 | 예 | 1부터 1000까지 | 경계 박스의 너비(세계 단위). |
| `bbox_height` | 정수 | 예 | 1부터 1000까지 | 경계 박스의 높이(세계 단위). |
| `bbox_length` | 정수 | 예 | 1부터 1000까지 | 경계 박스의 길이(세계 단위). |
| `height_cm` | 정수 | 예 | 1부터 300까지 | 생성된 모델의 높이(센티미터). |

**참고:** `prompt` 매개변수는 1자에서 2500자 사이여야 합니다. `seed` 매개변수는 지정되지 않은 경우 기본값으로 0(랜덤)이 됩니다.

|-------------|-----------|-------------|
