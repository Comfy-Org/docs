LTXVConcatAVLatent 노드는 비디오 잠재 표현과 오디오 잠재 표현을 하나의 연결된 잠재 출력으로 결합합니다. 두 입력의 `samples` 텐서를 합치고, 존재하는 경우 `noise_mask` 텐서도 함께 합쳐 비디오 생성 파이프라인에서 추가 처리를 위해 준비합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `video_latent` | LATENT | 예 | | 비디오 데이터의 잠재 표현입니다. |
| `audio_latent` | LATENT | 예 | | 오디오 데이터의 잠재 표현입니다. |

**참고:** `video_latent`와 `audio_latent` 입력의 `samples` 텐서가 연결됩니다. 두 입력 중 하나에 `noise_mask`가 포함되어 있으면 이를 사용하고, 하나가 없으면 해당 `samples`와 동일한 형태의 1로 채워진 마스크가 생성됩니다. 생성된 마스크들도 함께 연결됩니다.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `latent` | LATENT | 연결된 `samples`와, 필요하다면 비디오와 오디오 입력의 연결된 `noise_mask`를 포함한 단일 잠재 사전입니다. |
