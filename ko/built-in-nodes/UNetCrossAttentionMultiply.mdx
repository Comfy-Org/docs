UNetCrossAttentionMultiply 노드는 UNet 모델의 교차 주의 메커니즘에 곱셈 요인을 적용합니다. 이를 통해 교차 주의 레이어의 쿼리, 키, 값 및 출력 구성 요소를 스케일링하여 다양한 주의 행태와 효과를 실험할 수 있습니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `model` | MODEL | 예 | - | 주의 스케일링 요인으로 수정할 UNet 모델 |
| `q` | FLOAT | 아니요 | 0.0 - 10.0 | 교차 주의에서 쿼리 구성 요소의 스케일링 요인 (기본값: 1.0) |
| `k` | FLOAT | 아니요 | 0.0 - 10.0 | 교차 주의에서 키 구성 요소의 스케일링 요인 (기본값: 1.0) |
| `v` | FLOAT | 아니요 | 0.0 - 10.0 | 교차 주의에서 값 구성 요소의 스케일링 요인 (기본값: 1.0) |
| `out` | FLOAT | 아니요 | 0.0 - 10.0 | 교차 주의에서 출력 구성 요소의 스케일링 요인 (기본값: 1.0) |

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `model` | MODEL | 교차 주의 구성 요소가 스케일링된 수정된 UNet 모델 |
