LoraLoaderBypass 노드는 특별한 "바이패스" 모드로 디퓨전 모델과 CLIP 모델에 LoRA(저랭크 적응)를 적용합니다. 일반적인 LoRA 로더와 달리, 이 방법은 기본 모델의 가중치를 영구적으로 수정하지 않습니다. 대신, 모델의 정상적인 순방향 전파 과정에 LoRA의 효과를 더하여 출력을 계산하며, 이는 학습 과정이나 가중치가 오프로드된 모델을 다룰 때 유용합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `model` | MODEL | 예 | - | LoRA를 적용할 디퓨전 모델입니다. |
| `clip` | CLIP | 예 | - | LoRA를 적용할 CLIP 모델입니다. |
| `lora_name` | COMBO | 예 | *사용 가능한 LoRA 파일 목록* | 적용할 LoRA 파일의 이름입니다. 옵션은 `loras` 폴더에서 불러옵니다. |
| `strength_model` | FLOAT | 예 | -100.0부터 100.0까지 | 디퓨전 모델을 얼마나 강하게 수정할지 설정합니다. 이 값은 음수도 가능합니다 (기본값: 1.0). |
| `strength_clip` | FLOAT | 예 | -100.0부터 100.0까지 | CLIP 모델을 얼마나 강하게 수정할지 설정합니다. 이 값은 음수도 가능합니다 (기본값: 1.0). |

**참고:** `strength_model`과 `strength_clip` 모두 0으로 설정된 경우, 노드는 처리 없이 원래의 수정되지 않은 `model`과 `clip` 입력을 그대로 반환합니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-----------|-----------|-------------|
| `MODEL` | MODEL | 바이패스 모드로 LoRA가 적용된 디퓨전 모델입니다. |
| `CLIP` | CLIP | 바이패스 모드로 LoRA가 적용된 CLIP 모델입니다. |
