> 이 문서는 AI로 생성되었습니다. 오류를 발견하거나 개선할 점이 있다면 언제든지 기여해 주세요! [GitHub에서 편집하기](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/TencentModelTo3DUVNode/en.md)

이 노드는 Tencent Hunyuan3D API를 사용하여 3D 모델의 UV 전개 작업을 수행합니다. 3D 모델 파일을 입력으로 받아 API로 전송하여 처리한 후, 처리된 모델을 OBJ 및 FBX 형식으로 반환하고 생성된 UV 텍스처 이미지도 함께 제공합니다. 입력 모델은 면수가 30,000개 미만이어야 합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `model_3d` | FILE3D | 예 | GLB<br />OBJ<br />FBX | 입력 3D 모델(GLB, OBJ 또는 FBX). 모델의 면수는 30,000개 미만이어야 합니다. |
| `seed` | INT | 아니요 | 0부터 2147483647까지 | 시드 값(기본값: 1). 이 값은 노드가 다시 실행될지 여부를 제어하지만, 시드 값과 상관없이 결과는 비결정적입니다. |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `OBJ` | FILE3D | 처리된 3D 모델 파일(형식: OBJ). |
| `FBX` | FILE3D | 처리된 3D 모델 파일(형식: FBX). |
| `uv_image` | IMAGE | 생성된 UV 텍스처 이미지. |
