이미지를 타일 목록으로 분할하는 노드는 단일 입력 이미지를 여러 개의 작은 겹치는 직사각형 섹션인 타일로 나눕니다. 이 타일들을 일괄 처리된 목록으로 만들어 다른 노드에서 개별적으로 처리할 수 있습니다. 각 타일의 크기와 타일 간의 겹침 정도는 지정할 수 있습니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `image` | IMAGE | 예 | - | 타일로 분할할 입력 이미지입니다. |
| `tile_width` | INT | 예 | 64부터 1048576까지 | 각 출력 타일의 너비(픽셀 단위, 기본값: 1024). |
| `tile_height` | INT | 예 | 64부터 1048576까지 | 각 출력 타일의 높이(픽셀 단위, 기본값: 1024). |
| `overlap` | INT | 예 | 0부터 4096까지 | 인접한 타일들이 겹치는 픽셀 수(기본값: 128). |

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `image` | IMAGE | 모든 개별 이미지 타일을 포함한 일괄 처리된 목록입니다. |
