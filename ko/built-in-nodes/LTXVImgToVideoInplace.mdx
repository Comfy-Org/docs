LTXVImgToVideoInplace 노드는 입력 이미지를 초기 프레임으로 인코딩하여 비디오 잠재 표현을 조건부로 만듭니다. 이 노드는 VAE를 사용해 이미지를 잠재 공간으로 인코딩한 뒤, 지정된 강도에 따라 기존의 잠재 샘플과 블렌딩합니다. 이를 통해 이미지가 비디오 생성의 시작점이나 조건 신호로 활용될 수 있습니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `vae` | VAE | 예 | - | 입력 이미지를 잠재 공간으로 인코딩하는 데 사용되는 VAE 모델입니다. |
| `image` | IMAGE | 예 | - | 인코딩되어 비디오 잠재를 조건부로 사용할 입력 이미지입니다. |
| `latent` | LATENT | 예 | - | 수정할 대상 비디오 잠재 표현입니다. |
| `strength` | FLOAT | 아니요 | 0.0 - 1.0 | 인코딩된 이미지를 잠재에 블렌딩하는 강도를 제어합니다. 값이 1.0이면 초기 프레임을 완전히 교체하고, 낮은 값일수록 블렌딩됩니다. (기본값: 1.0) |
| `bypass` | BOOLEAN | 아니요 | - | 조건부를 우회합니다. 활성화 시 노드는 입력 잠재를 그대로 반환합니다. (기본값: False) |

**참고:** `image`는 `latent` 입력의 너비와 높이에 따라 `vae`가 요구하는 공간적 크기에 맞게 자동으로 리사이즈됩니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-----------|-----------|-------------|
| `latent` | LATENT | 수정된 비디오 잠재 표현입니다. 여기에는 업데이트된 샘플과 초기 프레임에 조건부 강도를 적용하는 `noise_mask`가 포함됩니다. |
