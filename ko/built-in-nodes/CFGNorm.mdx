CFGNorm 노드는 확산 모델의 분류자 없는 안내(CFG) 과정에 정규화 기법을 적용합니다. 이 노드는 조건부 출력과 무조건적 출력의 노름을 비교하여 덴오즈된 예측의 스케일을 조정한 후, 강도 승수를 적용해 효과를 제어합니다. 이를 통해 안내 스케일링에서 극단적인 값이 발생하는 것을 방지함으로써 생성 과정을 안정화시킵니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `model` | MODEL | 예 | - | CFG 정규화를 적용할 확산 모델 |
| `strength` | FLOAT | 예 | 0.0부터 100.0까지 | CFG 스케일링에 적용되는 정규화 효과의 강도를 제어합니다(기본값: 1.0) |

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `patched_model` | MODEL | CFG 정규화가 샘플링 과정에 적용된 수정된 모델을 반환합니다 |
