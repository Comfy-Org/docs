SaveLatent 노드는 잠재 텐서를 디스크에 파일로 저장하여 나중에 사용하거나 공유할 수 있도록 합니다. 이 노드는 잠재 샘플을 받아 출력 디렉터리에 저장하며, 옵션으로 프롬프트 정보를 포함한 메타데이터도 함께 저장합니다. 노드는 잠재 데이터 구조를 유지하면서 파일 이름 지정과 정리를 자동으로 처리합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `samples` | LATENT | 예 | - | 디스크에 저장할 잠재 샘플 |
| `filename_prefix` | STRING | 아니요 | - | 출력 파일명의 접두어 (기본값: "latents/ComfyUI") |
| `prompt` | PROMPT | 아니요 | - | 메타데이터에 포함할 프롬프트 정보 (숨겨진 매개변수) |
| `extra_pnginfo` | EXTRA_PNGINFO | 아니요 | - | 메타데이터에 포함할 추가 PNG 정보 (숨겨진 매개변수) |

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `ui` | UI | ComfyUI 인터페이스에서 저장된 잠재 데이터의 파일 위치 정보 제공 |
