ExtendIntermediateSigmas 노드는 기존의 시그마 값 시퀀스를 받아들여 그 사이에 추가적인 중간 시그마 값을 삽입합니다. 얼마나 많은 추가 단계를 넣을지, 보간을 위한 간격 방식, 그리고 선택적으로 시작 및 끝 시그마 경계를 지정하여 시그마 시퀀스 내에서 확장이 이루어지는 위치를 조정할 수 있습니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `sigmas` | SIGMAS | 예 | - | 중간 값으로 확장할 입력 시그마 시퀀스 |
| `steps` | INT | 예 | 1~100 | 기존 시그마 사이에 삽입할 중간 단계의 수 (기본값: 2) |
| `start_at_sigma` | FLOAT | 예 | -1.0~20000.0 | 확장의 상한 시그마 경계 - 이 값 아래의 시그마만 확장됩니다 (기본값: -1.0, 무한대 의미) |
| `end_at_sigma` | FLOAT | 예 | 0.0~20000.0 | 확장의 하한 시그마 경계 - 이 값 위의 시그마만 확장됩니다 (기본값: 12.0) |
| `spacing` | COMBO | 예 | `"linear"`<br />`"cosine"`<br />`"sine"` | 중간 시그마 값을 간격 지정하는 보간 방식 (기본값: "linear") |

**참고:** 노드는 현재 시그마가 `start_at_sigma`보다 작거나 같고 `end_at_sigma`보다 크거나 같은 기존 시그마 쌍 사이에만 중간 시그마를 삽입합니다. `start_at_sigma`를 -1.0으로 설정하면 무한대로 취급되며, 이 경우 `end_at_sigma` 하한 경계만 적용됩니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `sigmas` | SIGMAS | 추가된 중간 값이 삽입된 확장된 시그마 시퀀스 |
