LTXVImgToVideo 노드는 입력 이미지를 비디오 생성 모델용 비디오 잠재 표현으로 변환합니다. 단일 이미지를 입력으로 받아 VAE 인코더를 사용해 프레임 시퀀스로 확장한 뒤, 강도 조절을 통해 원본 이미지의 내용이 비디오 생성 과정에서 얼마나 보존되는지 또는 수정되는지를 결정합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `positive` | CONDITIONING | 예 | - | 비디오 생성을 안내하는 긍정적 조건부 프롬프트 |
| `negative` | CONDITIONING | 예 | - | 비디오에서 특정 요소를 피하도록 하는 부정적 조건부 프롬프트 |
| `vae` | VAE | 예 | - | 입력 이미지를 잠재 공간으로 인코딩하는 데 사용되는 VAE 모델 |
| `image` | IMAGE | 예 | - | 비디오 프레임으로 변환할 입력 이미지 |
| `width` | INT | 아니요 | 64부터 MAX_RESOLUTION까지 | 출력 비디오의 너비(픽셀)(기본값: 768, 단계: 32) |
| `height` | INT | 아니요 | 64부터 MAX_RESOLUTION까지 | 출력 비디오의 높이(픽셀)(기본값: 512, 단계: 32) |
| `length` | INT | 아니요 | 9부터 MAX_RESOLUTION까지 | 생성된 비디오의 프레임 수(기본값: 97, 단계: 8) |
| `batch_size` | INT | 아니요 | 1부터 4096까지 | 동시에 생성할 비디오 수(기본값: 1) |
| `strength` | FLOAT | 아니요 | 0.0부터 1.0까지 | 생성된 비디오의 첫 번째 프레임에서 원본 이미지 내용이 얼마나 보존될지 제어합니다. 값이 1.0이면 원본 이미지를 완전히 유지하고, 0.0이면 최대한 수정됩니다(기본값: 1.0) |

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `positive` | CONDITIONING | 비디오 프레임 마스크가 적용된 처리된 긍정적 조건부 |
| `negative` | CONDITIONING | 비디오 프레임 마스크가 적용된 처리된 부정적 조건부 |
| `latent` | LATENT | 인코딩된 프레임과 비디오 생성용 노이즈 마스크를 포함한 비디오 잠재 표현 |
