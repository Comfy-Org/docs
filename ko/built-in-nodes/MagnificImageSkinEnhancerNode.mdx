Magnific Image Skin Enhancer 노드는 인물 사진에 특화된 AI 처리를 적용하여 피부 표현을 개선합니다. 세 가지 다른 강화 목표를 위한 세 가지 모드를 제공합니다: 예술적 효과를 위한 '크리에이티브', 원래 모습을 유지하는 '피드백', 조명이나 사실감과 같은 특정한 개선을 위한 '플렉시블'입니다. 이 노드는 이미지를 외부 API로 업로드하여 처리하고, 개선된 결과를 반환합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `image` | IMAGE | 예 | - | 강화할 인물 사진입니다. |
| `sharpen` | INT | 아니요 | 0부터 100까지 | 선명화 강도 수준 (기본값: 0). |
| `smart_grain` | INT | 아니요 | 0부터 100까지 | 스마트 그레인 강도 수준 (기본값: 2). |
| `mode` | COMBO | 예 | `"creative"`<br />`"faithful"`<br />`"flexible"` | 사용할 처리 모드입니다. `"creative"`는 예술적 강화용, `"faithful"`는 원래 모습을 유지하는 데, `"flexible"`는 특정 최적화를 위한 것입니다. |
| `skin_detail` | INT | 아니요 | 0부터 100까지 | 피부 디테일 강화 수준입니다. 이 입력은 `mode`가 `"faithful"`로 설정된 경우에만 사용 가능합니다 (기본값: 80). |
| `optimized_for` | COMBO | 아니요 | `"enhance_skin"`<br />`"improve_lighting"`<br />`"enhance_everything"`<br />`"transform_to_real"`<br />`"no_make_up"` | 강화 최적화 대상입니다. 이 입력은 `mode`가 `"flexible"`로 설정된 경우에만 사용 가능합니다. |

**제약사항:**

* 노드는 정확히 하나의 입력 이미지만 받습니다.
* 입력 이미지의 최소 높이와 너비는 160픽셀 이상이어야 합니다.
* 입력 이미지의 종횡비는 1:3에서 3:1 사이여야 합니다 (비엄격 검증).
* `skin_detail` 매개변수는 `mode`가 `"faithful"`로 설정된 경우에만 활성화됩니다.
* `optimized_for` 매개변수는 `mode`가 `"flexible"`로 설정된 경우에만 활성화됩니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `image` | IMAGE | 강화된 인물 사진입니다. |
