이 노드는 StableZero123 모델에서 사용하기 위해 데이터를 처리하고 조건부로 설정하도록 설계되었으며, 이러한 모델과 호환되고 최적화된 특정 형식으로 입력을 준비하는 데 중점을 둡니다.

## 입력

| 매개변수             | Comfy 데이터 유형        | 설명 |
|-----------------------|--------------------|-------------|
| `clip_vision`         | `CLIP_VISION`      | 모델의 요구사항에 맞게 시각 데이터를 처리하여, 모델의 시각적 맥락 이해도를 향상시킵니다. |
| `init_image`          | `IMAGE`            | 모델의 초기 이미지 입력으로 작용하며, 이후 이미지 기반 작업의 기본값을 설정합니다. |
| `vae`                 | `VAE`              | 변분 오토인코더 출력을 통합하여, 모델이 이미지를 생성하거나 수정하는 능력을 촉진합니다. |
| `width`               | `INT`              | 출력 이미지의 너비를 지정하며, 모델의 필요에 따라 동적으로 크기를 조정할 수 있습니다. |
| `height`              | `INT`              | 출력 이미지의 높이를 결정하며, 출력 크기의 맞춤화가 가능합니다. |
| `batch_size`          | `INT`              | 한 번에 처리되는 이미지의 수를 제어하며, 연산 효율성을 최적화합니다. |
| `elevation`           | `FLOAT`            | 3D 모델 렌더링을 위한 고도 각도를 조정하며, 모델의 공간적 이해도를 향상시킵니다. |
| `azimuth`             | `FLOAT`            | 3D 모델 시각화를 위한 방위 각도를 수정하며, 모델의 방향 인식을 개선합니다. |

## 출력

| 매개변수     | 데이터 유형 | 설명 |
|---------------|--------------|-------------|
| `positive`    | `CONDITIONING` | 긍정적인 조건부 벡터를 생성하며, 모델의 긍정적 특성 강화에 도움을 줍니다. |
| `negative`    | `CONDITIONING` | 부정적인 조건부 벡터를 생성하며, 모델이 특정 특성을 피하도록 지원합니다. |
| `latent`      | `LATENT`     | 잠재적 표현을 생성하며, 모델이 데이터에 대해 더 깊은 통찰을 얻도록 돕습니다. |
