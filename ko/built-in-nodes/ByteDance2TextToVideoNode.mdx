이 노드는 ByteDance의 Seedance 2.0 API를 사용하여 텍스트 설명을 기반으로 비디오를 생성합니다. 선택한 모델에 프롬프트를 전송하고 비디오 처리가 완료될 때까지 기다린 후 최종 결과를 반환합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `model` | COMBO | 예 | `"Seedance 2.0"`<br />`"Seedance 2.0 Fast"` | 비디오 생성에 사용할 모델입니다. 모델을 선택하면 프롬프트, 해상도, 종횡비, 지속 시간 및 오디오 생성과 관련된 추가 필수 입력 항목이 표시됩니다. "Seedance 2.0"은 최고 품질을 위해, "Seedance 2.0 Fast"는 속도 최적화를 위해 선택하세요. |
| `seed` | INT | 아니요 | 0부터 2147483647까지 | 시드 값(기본값: 0). 이 값이 변경되면 노드가 다시 실행되지만, 시드와 관계없이 결과는 비결정적입니다. |
| `watermark` | BOOLEAN | 아니요 | True / False | 비디오에 워터마크를 추가할지 여부(기본값: False). 이는 고급 설정입니다. |

**참고:** `model` 매개변수는 동적 콤보입니다. 모델을 선택하면 텍스트 프롬프트, 해상도, 종횡비, 지속 시간 및 오디오 생성 여부를 포함한 여러 필수 하위 매개변수가 표시됩니다. 프롬프트 텍스트는 공백을 제거한 뒤 최소 1자 이상이어야 합니다.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `video` | VIDEO | 생성된 비디오 파일입니다. |
