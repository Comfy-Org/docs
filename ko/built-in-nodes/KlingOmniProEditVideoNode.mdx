> 이 문서는 AI로 생성되었습니다. 오류를 발견하거나 개선할 점이 있다면 언제든지 기여해 주세요! [GitHub에서 편집하기](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/KlingOmniProEditVideoNode/en.md)

Kling Omni Edit Video (Pro) 노드는 AI 모델을 사용해 텍스트 설명을 바탕으로 기존 비디오를 편집합니다. 소스 비디오와 프롬프트를 제공하면 노드는 요청된 변경 사항을 반영한 동일한 길이의 새 비디오를 생성합니다. 선택적으로 참조 이미지를 사용해 스타일을 안내하고 소스 비디오의 원래 오디오를 유지할 수도 있습니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `model_name` | COMBO | 예 | `"kling-v3-omni"`<br />`"kling-video-o1"` | 비디오 편집에 사용할 AI 모델 (기본값: `"kling-v3-omni"`). |
| `prompt` | STRING | 예 | | 비디오 내용을 설명하는 텍스트 프롬프트입니다. 여기에는 긍정적 및 부정적 설명 모두 포함될 수 있습니다. |
| `video` | VIDEO | 예 | | 편집할 비디오입니다. 출력 비디오의 길이는 소스 비디오와 동일합니다. |
| `keep_original_sound` | BOOLEAN | 예 | | 입력 비디오의 원래 오디오를 출력에도 유지할지 여부를 결정합니다 (기본값: True). |
| `reference_images` | IMAGE | 아니요 | | 최대 4개의 추가 참조 이미지입니다. |
| `resolution` | COMBO | 아니요 | `"1080p"`<br />`"720p"` | 출력 비디오의 해상도 (기본값: `"1080p"`). |
| `seed` | INT | 아니요 | 0부터 2147483647까지 | 시드는 노드가 다시 실행될지 여부를 제어합니다. 시드와 상관없이 결과는 비결정적입니다 (기본값: 0). |

**제약사항 및 한계:**

* `prompt`는 1자에서 2500자 사이여야 합니다.
* 입력 `video`의 길이는 3.0초에서 10.05초 사이여야 합니다.
* 입력 `video`의 크기는 720x720픽셀에서 2160x2160픽셀 사이여야 합니다.
* 비디오를 사용할 경우 최대 4개의 `reference_images`를 제공할 수 있습니다.
* 각 `reference_image`는 최소 300x300픽셀 이상이어야 합니다.
* 각 `reference_image`의 종횡비는 1:2.5에서 2.5:1 사이여야 합니다.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `video` | VIDEO | AI 모델이 생성한 편집된 비디오입니다. |
