## 개요

HappyHorse 모델을 사용하여 텍스트 프롬프트를 기반으로 비디오를 생성합니다. 이 노드는 프롬프트와 설정을 HappyHorse API로 전송한 후, 비디오가 생성될 때까지 기다렸다가 결과를 다운로드합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `model` | DICT | 예 | 설명 참조 | 모델 선택과 관련된 파라미터를 포함한 딕셔너리입니다. 모델은 반드시 `"happyhorse-1.0-t2v"`여야 합니다. 이 딕셔너리는 다음과 같은 하위 매개변수를 포함합니다:<br /><br />**`prompt`** (STRING): 생성하고자 하는 비디오의 텍스트 설명입니다. 영어와 중국어를 지원합니다. (기본값: "").<br />**`resolution`** (COMBO): 출력 비디오의 해상도입니다. 옵션: `"720P"`<br />`"1080P"`.<br />**`ratio`** (COMBO): 출력 비디오의 종횡비입니다. 옵션: `"16:9"`<br />`"9:16"`<br />`"1:1"`<br />`"4:3"`<br />`"3:4"`.<br />**`duration`** (INT): 비디오의 길이(초)입니다. (기본값: 5, 최소: 3, 최대: 15, 단계: 1). |
| `seed` | INT | 예 | 0부터 2147483647까지 | 생성에 사용할 시드입니다. 동일한 입력과 동일한 시드를 사용하면 동일한 결과가 생성됩니다. (기본값: 0). |
| `watermark` | BOOLEAN | 아니요 | True / False | 결과에 AI가 생성한 워터마크를 추가할지 여부입니다. (기본값: False). |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `VIDEO` | VIDEO | 생성된 비디오 파일입니다. |
