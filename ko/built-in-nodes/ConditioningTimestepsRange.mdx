ConditioningTimestepsRange 노드는 생성 과정 중 조건부 효과가 적용되는 시점을 제어하기 위해 세 가지 서로 다른 텀스텝 범위를 생성합니다. 시작 및 끝 비율 값을 입력받아 전체 텀스텝 범위(0.0부터 1.0까지)를 세 구간으로 나눕니다: 지정된 비율 사이의 메인 범위, 시작 비율 이전의 범위, 그리고 끝 비율 이후의 범위입니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `start_percent` | FLOAT | 예 | 0.0 - 1.0 | 텀스텝 범위의 시작 비율 (기본값: 0.0) |
| `end_percent` | FLOAT | 예 | 0.0 - 1.0 | 텀스텝 범위의 끝 비율 (기본값: 1.0) |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `TIMESTEPS_RANGE` | TIMESTEPS_RANGE | start_percent와 end_percent로 정의된 메인 텀스텝 범위 |
| `BEFORE_RANGE` | TIMESTEPS_RANGE | 0.0부터 start_percent까지의 텀스텝 범위 |
| `AFTER_RANGE` | TIMESTEPS_RANGE | end_percent부터 1.0까지의 텀스텝 범위 |
