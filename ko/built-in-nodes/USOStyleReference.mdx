USOStyleReference 노드는 CLIP 비전 출력에서 인코딩된 이미지 특징을 사용하여 모델에 스타일 참조 패치를 적용합니다. 시각적 입력에서 추출한 스타일 정보를 포함함으로써 입력 모델의 수정된 버전을 생성하며, 이를 통해 스타일 전송 또는 참조 기반 생성 기능을 가능하게 합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `model` | MODEL | 예 | - | 스타일 참조 패치를 적용할 기본 모델 |
| `model_patch` | MODEL_PATCH | 예 | - | 스타일 참조 정보를 포함한 모델 패치 |
| `clip_vision_output` | CLIP_VISION_OUTPUT | 예 | - | CLIP 비전 처리 과정에서 추출한 인코딩된 시각적 특징 |

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `model` | MODEL | 스타일 참조 패치가 적용된 수정된 모델 |
