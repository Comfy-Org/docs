> 이 문서는 AI로 생성되었습니다. 오류를 발견하거나 개선할 점이 있다면 언제든지 기여해 주세요! [GitHub에서 편집하기](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/QuiverImageToSVGNode/en.md)

이 노드는 Quiver AI의 벡터화 모델을 사용하여 래스터 이미지를 스케일러블한 벡터 그래픽(SVG)으로 변환합니다. 이미지를 외부 API로 전송하여 처리하고 벡터화된 결과를 반환합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `image` | IMAGE | 예 | N/A | 벡터화할 입력 이미지. |
| `auto_crop` | BOOLEAN | 아니요 | `True`<br />`False` | 주요 피사체를 자동으로 자르기. 이는 고급 매개변수입니다(기본값: `False`). |
| `model` | DYNAMICCOMBO | 예 | 여러 옵션 제공 | SVG 벡터화에 사용할 모델입니다. 모델을 선택하면 해당 모델에 특화된 추가 매개변수가 표시됩니다: `target_size`(픽셀 단위의 정사각형 크기 조정 목표, 기본값: 1024, 범위: 128~4096), `temperature`, `top_p`, 그리고 `presence_penalty`. |
| `seed` | INT | 아니요 | 0부터 2147483647까지 | 노드가 다시 실행될지 여부를 결정하는 시드. 시드 값과 관계없이 실제 결과는 비결정적입니다. 이 매개변수는 "생성 후 제어" 기능을 갖습니다(기본값: 0). |

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `SVG` | SVG | 벡터화된 SVG 출력. |
