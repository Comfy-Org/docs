Pika Text2Video v2.2 노드는 Pika API 버전 2.2에 텍스트 프롬프트를 보내 동영상을 생성합니다. 이 노드는 Pika의 AI 동영상 생성 서비스를 이용해 텍스트 설명을 동영상으로 변환합니다. 노드를 통해 비율, 지속 시간, 해상도 등 동영상 생성 과정의 다양한 요소를 맞춤 설정할 수 있습니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `prompt_text` | 문자열 | 예 | - | 동영상에 생성하고자 하는 내용을 설명하는 주요 텍스트 설명 |
| `negative_prompt` | 문자열 | 예 | - | 생성된 동영상에 나타나지 않기를 원하는 내용을 설명하는 텍스트 |
| `seed` | 정수 | 예 | - | 재현 가능한 결과를 위해 생성의 무작위성을 제어하는 숫자 |
| `resolution` | 문자열 | 예 | - | 출력 동영상의 해상도 설정 |
| `duration` | 정수 | 예 | - | 동영상의 길이(초) |
| `aspect_ratio` | 부동소수점 | 아니요 | 0.4~2.5 | 종횡비(너비 / 높이) (기본값: 1.7777777777777777) |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `output` | 동영상 | Pika API에서 반환된 생성된 동영상 파일 |
