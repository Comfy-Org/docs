`CLIPTextEncodeFlux`는 Flux 아키텍처를 위해 설계된 고급 텍스트 인코딩 노드입니다. 두 개의 별도 텍스트 입력을 각각 다른 인코더—CLIP-L과 T5XXL—를 통해 처리한 뒤, 가이던스 스케일과 결합하여 이미지 생성을 위한 통합된 조건부 출력을 생성합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `clip` | CLIP | 예 | - | Flux 아키텍처를 지원하는 CLIP 모델로, CLIP-L 및 T5XXL 인코더를 모두 포함합니다. |
| `clip_l` | STRING | 예 | - | CLIP-L 인코더를 통해 처리된 텍스트 입력입니다. 스타일이나 주제와 같은 간결한 키워드 설명에 적합합니다. 다중 줄 입력 및 동적 프롬프트를 지원합니다. |
| `t5xxl` | STRING | 예 | - | T5XXL 인코더를 통해 처리된 텍스트 입력입니다. 복잡한 장면과 세부 사항을 표현하는 상세한 자연어 설명에 적합합니다. 다중 줄 입력 및 동적 프롬프트를 지원합니다. |
| `guidance` | FLOAT | 예 | 0.0 - 100.0 | 생성 과정에서 텍스트 조건이 미치는 영향을 제어합니다. 값이 높을수록 텍스트에 더욱 엄격하게 맞춰집니다. 기본값: 3.5. |

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `CONDITIONING` | CONDITIONING | 두 인코더의 융합된 임베딩과 가이던스 파라미터를 포함하며, 조건부 이미지 생성에 사용됩니다. |
