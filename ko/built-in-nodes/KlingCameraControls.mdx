Kling Camera Controls 노드를 사용하면 비디오 생성 시 모션 컨트롤 효과를 만들기 위해 다양한 카메라 이동 및 회전 매개변수를 구성할 수 있습니다. 카메라의 위치 조정, 회전, 줌 기능을 제공하여 다양한 카메라 움직임을 시뮬레이션할 수 있습니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `camera_control_type` | COMBO | 예 | `"simple"`<br />`"advanced"` | 사용할 카메라 컨트롤 구성 유형을 지정합니다 |
| `horizontal_movement` | FLOAT | 아니요 | -10.0부터 10.0까지 | 카메라의 가로축(x축) 방향 이동을 제어합니다. 음수는 왼쪽, 양수는 오른쪽을 나타냅니다(기본값: 0.0) |
| `vertical_movement` | FLOAT | 아니요 | -10.0부터 10.0까지 | 카메라의 세로축(y축) 방향 이동을 제어합니다. 음수는 아래쪽, 양수는 위쪽을 나타냅니다(기본값: 0.0) |
| `pan` | FLOAT | 아니요 | -10.0부터 10.0까지 | 카메라의 수직 평면(x축) 회전을 제어합니다. 음수는 아래쪽 회전, 양수는 위쪽 회전을 나타냅니다(기본값: 0.5) |
| `tilt` | FLOAT | 아니요 | -10.0부터 10.0까지 | 카메라의 수평 평면(y축) 회전을 제어합니다. 음수는 왼쪽 회전, 양수는 오른쪽 회전을 나타냅니다(기본값: 0.0) |
| `roll` | FLOAT | 아니요 | -10.0부터 10.0까지 | 카메라의 롤링 양(z축)을 제어합니다. 음수는 반시계방향, 양수는 시계방향을 나타냅니다(기본값: 0.0) |
| `zoom` | FLOAT | 아니요 | -10.0부터 10.0까지 | 카메라의 초점 거리 변경을 제어합니다. 음수는 좁은 화각, 양수는 넓은 화각을 나타냅니다(기본값: 0.0) |

**참고:** 카메라 컨트롤 매개변수 중 적어도 하나(`horizontal_movement`, `vertical_movement`, `pan`, `tilt`, `roll`, 또는 `zoom`)가 0이 아닌 값이어야만 유효한 구성이 됩니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `camera_control` | CAMERA_CONTROL | 비디오 생성에 사용할 구성된 카메라 컨트롤 설정을 반환합니다 |
