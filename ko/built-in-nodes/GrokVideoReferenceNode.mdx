Grok Reference-to-Video 노드는 텍스트 프롬프트를 기반으로 비디오를 생성하며, 최대 7개의 참조 이미지를 사용해 출력물의 스타일과 내용을 안내합니다. 이 노드는 외부 API와 연결되어 비디오를 생성한 후 다운로드하여 반환합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `prompt` | 문자열 | 예 | 없음 | 원하는 비디오에 대한 텍스트 설명. |
| `model` | 콤보 | 예 | `"grok-imagine-video"` | 비디오 생성에 사용할 모델. |
| `model.reference_images` | 이미지 | 예 | 1~7개의 이미지 | 비디오 생성을 안내할 최대 7개의 참조 이미지. |
| `model.resolution` | 콤보 | 예 | `"480p"`<br />`"720p"` | 출력 비디오의 해상도. |
| `model.aspect_ratio` | 콤보 | 예 | `"16:9"`<br />`"4:3"`<br />`"3:2"`<br />`"1:1"`<br />`"2:3"`<br />`"3:4"`<br />`"9:16"` | 출력 비디오의 종횡비. |
| `model.duration` | 정수 | 예 | 2~10 | 출력 비디오의 지속 시간(초)(기본값: 6). |
| `seed` | 정수 | 아니요 | 0~2147483647 | 노드가 다시 실행될지 여부를 결정하는 시드; 실제 결과는 시드와 관계없이 비결정적임(기본값: 0). |

**참고:** `model` 매개변수는 `reference_images`, `resolution`, `aspect_ratio`, `duration`을 포함하는 그룹입니다. 참조 이미지를 최소 하나 이상 제공해야 하며, 최대 일곱 개까지 제공할 수 있습니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `video` | 비디오 | 생성된 비디오 파일. |
