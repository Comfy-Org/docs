PrimitiveBoundingBox 노드는 위치와 크기로 정의된 간단한 직사각형 영역을 생성합니다. 이 노드는 좌상단 모서리의 X 및 Y 좌표와 너비 및 높이 값을 입력받으며, 워크플로우 내 다른 노드에서 사용할 수 있는 경계 상자 데이터 구조를 출력합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `x` | INT | 예 | 0부터 8192까지 | 경계 상자의 좌상단 모서리의 X좌표 (기본값: 0). |
| `y` | INT | 예 | 0부터 8192까지 | 경계 상자의 좌상단 모서리의 Y좌표 (기본값: 0). |
| `width` | INT | 예 | 1부터 8192까지 | 경계 상자의 너비 (기본값: 512). |
| `height` | INT | 예 | 1부터 8192까지 | 경계 상자의 높이 (기본값: 512). |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `bounding_box` | BOUNDING_BOX | 정의된 직사각형의 `x`, `y`, `width`, `height` 속성을 포함하는 데이터 구조입니다. |
