Magnific Image Relight 노드는 입력 이미지의 조명을 조정합니다. 텍스트 프롬프트를 기반으로 스타일리시한 조명을 적용하거나, 선택적 참조 이미지로부터 조명 특성을 전송할 수 있습니다. 이 노드는 최종 출력의 밝기, 대비 및 전반적인 분위기를 세밀하게 조정할 수 있는 다양한 컨트롤을 제공합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `image` | IMAGE | 예 | N/A | 조명을 다시 비추고자 하는 이미지입니다. 이미지는 정확히 하나만 필요하며, 최소 크기는 160x160픽셀입니다. 종횡비는 1:3에서 3:1 사이여야 합니다. |
| `prompt` | STRING | 아니요 | N/A | 조명에 대한 설명적 안내입니다. 강조 표기(1~1.4)를 지원합니다. 기본값은 빈 문자열입니다. |
| `light_transfer_strength` | INT | 예 | 0부터 100까지 | 조명 전송의 강도입니다. 기본값: 100. |
| `style` | COMBO | 예 | `"standard"`<br />`"darker_but_realistic"`<br />`"clean"`<br />`"smooth"`<br />`"brighter"`<br />`"contrasted_n_hdr"`<br />`"just_composition"` | 스타일리시한 출력 선호도입니다. |
| `interpolate_from_original` | BOOLEAN | 예 | N/A | 원본과 더 유사하도록 생성 자유도를 제한합니다. 기본값: False. |
| `change_background` | BOOLEAN | 예 | N/A | 프롬프트/참조 이미지를 기반으로 배경을 수정합니다. 기본값: True. |
| `preserve_details` | BOOLEAN | 예 | N/A | 원본의 질감과 세부 사항을 유지합니다. 기본값: True. |
| `advanced_settings` | DYNAMICCOMBO | 예 | `"disabled"`<br />`"enabled"` | 고급 조명 제어를 위한 세밀한 설정 옵션입니다. `"enabled"`로 설정하면 추가 매개변수가 사용 가능해집니다. |
| `reference_image` | IMAGE | 아니요 | N/A | 조명을 전송할 선택적 참조 이미지입니다. 제공된 경우 이미지는 정확히 하나만 필요하며, 최소 크기는 160x160픽셀입니다. 종횡비는 1:3에서 3:1 사이여야 합니다. |

**고급 설정 참고:** `advanced_settings`를 `"enabled"`로 설정하면 다음 중첩 매개변수가 활성화됩니다:

* `whites`: 이미지의 가장 밝은 색조를 조정합니다. 범위: 0부터 100까지. 기본값: 50.
* `blacks`: 이미지의 가장 어두운 색조를 조정합니다. 범위: 0부터 100까지. 기본값: 50.
* `brightness`: 전체 밝기 조정입니다. 범위: 0부터 100까지. 기본값: 50.
* `contrast`: 대비 조정입니다. 범위: 0부터 100까지. 기본값: 50.
* `saturation`: 색상 채도 조정입니다. 범위: 0부터 100까지. 기본값: 50.
* `engine`: 처리 엔진 선택입니다. 옵션: `"automatic"`, `"balanced"`, `"cool"`, `"real"`, `"illusio"`, `"fairy"`, `"colorful_anime"`, `"hard_transform"`, `"softy"`.
* `transfer_light_a`: 조명 전송의 강도입니다. 옵션: `"automatic"`, `"low"`, `"medium"`, `"normal"`, `"high"`, `"high_on_faces"`.
* `transfer_light_b`: 조명 전송 강도를 추가로 조정합니다. 이전 컨트롤과 결합하여 다양한 효과를 낼 수 있습니다. 옵션: `"automatic"`, `"composition"`, `"straight"`, `"smooth_in"`, `"smooth_out"`, `"smooth_both"`, `"reverse_both"`, `"soft_in"`, `"soft_out"`, `"soft_mid"`, `"style_shift"`, `"strong_shift"`.
* `fixed_generation`: 동일한 설정으로 일관된 출력을 보장합니다. 기본값: True.
