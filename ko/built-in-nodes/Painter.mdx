페인터 노드는 ComfyUI 내에서 이미지와 마스크를 직접 만들거나 편집할 수 있는 대화형 캔버스를 제공합니다. 빈 캔버스나 기존 이미지를 사용해 시작할 수 있으며, 브러시 도구를 이용해 그림을 그릴 수 있고, 결과 이미지와 해당 알파 마스크를 함께 출력합니다. 마스크는 그려진 영역을 정의하며, 이 영역은 기본 이미지나 배경색 위에 합성됩니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `image` | IMAGE | 아니요 | - | 그림을 그릴 기본 이미지입니다. 제공되지 않으면 지정된 배경색, 너비, 높이를 사용해 빈 캔버스가 생성됩니다. |
| `mask` | STRING | 예 | - | 그림 데이터로, 일반적으로 노드의 내장 대화형 위젯에서 생성됩니다. 이 매개변수는 UI의 페인터 도구에 의해 관리되며, 일반 소켓에 연결해서는 안 됩니다. |
| `width` | INT | 예 | 64~4096 | 캔버스의 너비(픽셀), 기본 이미지가 제공되지 않을 경우 사용됩니다. 값은 64의 배수여야 합니다. 기본값은 512입니다. |
| `height` | INT | 예 | 64~4096 | 캔버스의 높이(픽셀), 기본 이미지가 제공되지 않을 경우 사용됩니다. 값은 64의 배수여야 합니다. 기본값은 512입니다. |
| `bg_color` | COLOR | 예 | - | 캔버스의 배경색으로, 16진수 코드로 지정됩니다(예: #000000). 기본 이미지가 제공되지 않을 때만 사용됩니다. 기본값은 검정(#000000)입니다. |

**참고:** `mask` 입력은 노드의 특수한 UI 위젯과 함께 작동하도록 설계되었습니다. 캔버스에 그림을 그리면 위젯이 자동으로 이 값을 채워줍니다. `width`와 `height` 입력은 표준 UI에서는 숨겨져 있지만, 새 이미지를 만들 때 캔버스의 크기를 정의합니다. 기본 이미지가 제공되면, 해당 이미지의 크기가 `width`와 `height` 설정을 덮어씁니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-----------|-----------|-------------|
| `IMAGE` | IMAGE | 최종 합성된 이미지입니다. 이는 그려진 영역(마스크에서 가져온)을 제공된 기본 이미지 또는 색상 배경 위에 블렌딩한 결과입니다. |
| `MASK` | MASK | 그림에서 추출된 알파 채널(투명도) 마스크입니다. 흰색 영역은 그려진 영역을 나타내며, 검은색 영역은 손대지 않은 배경을 나타냅니다. |
