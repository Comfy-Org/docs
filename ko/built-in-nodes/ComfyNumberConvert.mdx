숫자 변환 노드는 다양한 입력 데이터 유형을 숫자 값으로 변환합니다. 정수, 부동소수점, 문자열 또는 불리언 형식의 단일 입력을 받아 두 가지 출력을 생성합니다: 부동소수점 숫자와 정수입니다. 이는 텍스트나 논리값을 워크플로우 내 다른 수학적 또는 처리 노드에서 사용할 수 있는 형식으로 변환하는 데 유용합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `value` | INT, FLOAT, STRING, BOOLEAN | 예 | N/A | 숫자 출력으로 변환할 값입니다. 정수, 부동소수점 숫자, 텍스트 문자열 또는 참/거짓 불리언을 받습니다. |

**참고:** 입력이 문자열인 경우, 비어 있어서는 안 되며 유효한 숫자 표현을 포함해야 합니다(예: `"123"`, `"3.14"`). 빈 문자열이나 숫자로 파싱할 수 없는 텍스트, 또는 무한대(`"inf"`), 비정상값(`"nan"`)과 같은 값은 노드에서 오류를 발생시킵니다. 불리언 입력의 경우, `true`는 1.0(FLOAT)과 1(INT)로 변환되고, `false`는 0.0(FLOAT)과 0(INT)로 변환됩니다. 부동소수점 입력의 경우, 정수 출력은 소수점 이하 부분을 잘라내어 얻습니다.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `FLOAT` | FLOAT | 입력 값을 부동소수점 숫자로 변환한 결과입니다. |
| `INT` | INT | 입력 값을 정수로 변환한 결과입니다. 부동소수점 입력의 경우, 소수점 이하 부분을 잘라냅니다. |
