ImageHistogram 노드는 입력 이미지의 색상 분포를 분석합니다. 이미지 내 각 가능한 강도 값을 가진 픽셀의 수를 나타내는 그래프인 여러 히스토그램을 계산하여 출력합니다. 빨강, 초록, 파랑 색상 채널별로 별도의 히스토그램을 생성하며, 복합 RGB 히스토그램과 표준 밝기 공식을 기반으로 한 휘도 히스토그램도 생성합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `image` | IMAGE | 예 | N/A | 분석할 입력 이미지입니다. 노드는 배치에서 첫 번째 이미지를 처리합니다. |

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `rgb` | HISTOGRAM | 빨강, 초록, 파랑 채널의 평균 픽셀 강도를 나타내는 복합 히스토그램입니다. |
| `luminance` | HISTOGRAM | ITU-R BT.709 표준 휘도 공식을 사용해 계산한 이미지의 인지된 밝기 히스토그램입니다. |
| `red` | HISTOGRAM | 빨강 색상 채널의 픽셀 강도 분포를 보여주는 히스토그램입니다. |
| `green` | HISTOGRAM | 초록 색상 채널의 픽셀 강도 분포를 보여주는 히스토그램입니다. |
| `blue` | HISTOGRAM | 파랑 색상 채널의 픽셀 강도 분포를 보여주는 히스토그램입니다. |
