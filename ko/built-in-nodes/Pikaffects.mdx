Pikaffects 노드는 입력 이미지에 다양한 시각적 효과를 적용하여 동영상으로 생성합니다. Pika의 동영상 생성 API를 사용해 정적인 이미지를 녹는, 폭발하는, 공중부양하는 등 특정 효과가 적용된 애니메이션 동영상으로 변환합니다. 이 노드를 사용하려면 Pika 서비스에 접근하기 위한 API 키와 인증 토큰이 필요합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `image` | IMAGE | 예 | - | Pikaffect을 적용할 참조 이미지입니다. |
| `pikaffect` | COMBO | 예 | "Cake-ify"<br />"Crumble"<br />"Crush"<br />"Decapitate"<br />"Deflate"<br />"Dissolve"<br />"Explode"<br />"Eye-pop"<br />"Inflate"<br />"Levitate"<br />"Melt"<br />"Peel"<br />"Poke"<br />"Squish"<br />"Ta-da"<br />"Tear" | 이미지에 적용할 특정 시각적 효과입니다(기본값: "Cake-ify"). |
| `prompt_text` | STRING | 예 | - | 동영상 생성을 안내하는 텍스트 설명입니다. |
| `negative_prompt` | STRING | 예 | - | 생성된 동영상에서 피해야 할 내용을 설명하는 텍스트입니다. |
| `seed` | INT | 예 | 0부터 4294967295까지 | 재현 가능한 결과를 위해 사용하는 임의의 시드 값입니다. |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|------------|-----------|--------|
| `output` | VIDEO | Pikaffect이 적용된 생성된 동영상입니다. |
