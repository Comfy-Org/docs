Kling 이미지 생성 노드는 텍스트 프롬프트를 기반으로 이미지를 생성하며, 안내를 위해 참조 이미지를 사용하는 옵션도 제공합니다. 사용자의 텍스트 설명과 참조 설정에 따라 하나 이상의 이미지를 생성한 후, 생성된 이미지를 출력으로 반환합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `prompt` | 문자열 | 예 | - | 긍정적인 텍스트 프롬프트 |
| `negative_prompt` | 문자열 | 예 | - | 부정적인 텍스트 프롬프트 |
| `image_type` | 콤보 | 예 | `"subject_reference"`<br />`"style_reference"` | 이미지 참조 유형 선택(고급). 참조 이미지를 제공할 경우 필수입니다. |
| `image_fidelity` | 실수 | 예 | 0.0 - 1.0 | 사용자가 업로드한 이미지에 대한 참조 강도 (기본값: 0.5, 고급) |
| `human_fidelity` | 실수 | 예 | 0.0 - 1.0 | 피사체 참조 유사도 (기본값: 0.45, 고급) |
| `model_name` | 콤보 | 예 | `"kling-v3"`<br />`"kling-v2"`<br />`"kling-v1-5"` | 이미지 생성용 모델 선택 (기본값: "kling-v3") |
| `aspect_ratio` | 콤보 | 예 | `"16:9"`<br />`"9:16"`<br />`"1:1"`<br />`"4:3"`<br />`"3:4"`<br />`"21:9"`<br />`"9:21"` | 생성된 이미지의 종횡비 (기본값: "16:9") |
| `n` | 정수 | 예 | 1 - 9 | 생성된 이미지의 수 (기본값: 1) |
| `image` | 이미지 | 아니요 | - | 선택적 참조 이미지 |
| `seed` | 정수 | 아니요 | 0 - 2147483647 | 시드는 노드가 다시 실행될지 여부를 제어하며, 시드와 관계없이 결과는 비결정적입니다 (기본값: 0) |

**매개변수 제약사항:**

- `image` 매개변수는 선택적입니다. 참조 이미지를 제공할 경우 `image_type` 매개변수는 `"subject_reference"` 또는 `"style_reference"`로 설정해야 합니다.
- 참조 이미지를 제공하지 않을 경우 `image_type`, `image_fidelity`, `human_fidelity` 매개변수는 사용되지 않습니다.
- 프롬프트와 부정적 프롬프트의 최대 길이는 500자입니다.
- `seed` 매개변수는 선택적이며, 결정적 결과를 보장하지 않습니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-----------|-----------|-------------|
| `output` | 이미지 | 입력 매개변수를 기반으로 생성된 이미지 |
