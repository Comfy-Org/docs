SamplerLCMUpscale 노드는 잠재적 일관성 모델(LCM) 샘플링과 이미지 업스케일링 기능을 결합한 특수한 샘플링 방법을 제공합니다. 다양한 보간 방법을 사용해 샘플링 과정 중에 이미지를 업스케일링할 수 있어, 이미지 품질을 유지하면서 더 높은 해상도의 출력을 생성하는 데 유용합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `scale_ratio` | FLOAT | 선택 | 0.1 - 20.0 | 업스케일링 시 적용할 스케일링 요소 (기본값: 1.0) |
| `scale_steps` | INT | 선택 | -1 - 1000 | 업스케일링 과정에 사용할 단계 수. 자동 계산을 위해 -1을 사용하세요 (기본값: -1) |
| `upscale_method` | COMBO | 필수 | "bislerp"<br />"nearest-exact"<br />"bilinear"<br />"area"<br />"bicubic" | 이미지를 업스케일링하는 데 사용되는 보간 방법 |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `sampler` | SAMPLER | 샘플링 파이프라인에서 사용할 수 있는 구성된 샘플러 객체를 반환합니다 |
