Pikadditions 노드를 사용하면 어떤 객체나 이미지를 동영상에 추가할 수 있습니다. 동영상을 업로드하고 추가하고자 하는 내용을 지정하면 자연스럽게 통합된 결과를 얻을 수 있습니다. 이 노드는 Pika API를 사용하여 동영상에 이미지를 삽입하며, 자연스러운 모습으로 통합됩니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `video` | VIDEO | 예 | - | 이미지를 추가할 동영상입니다. |
| `image` | IMAGE | 예 | - | 동영상에 추가할 이미지입니다. |
| `prompt_text` | STRING | 예 | - | 동영상에 추가할 내용에 대한 텍스트 설명입니다. |
| `negative_prompt` | STRING | 예 | - | 동영상에서 피해야 할 내용에 대한 텍스트 설명입니다. |
| `seed` | INT | 예 | 0부터 4294967295까지 | 재현 가능한 결과를 위한 임의의 시드 값입니다. |

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `output` | VIDEO | 이미지가 삽입된 처리된 동영상입니다. |
