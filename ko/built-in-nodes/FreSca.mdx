FreSca 노드는 샘플링 과정 중 가이던스에 주파수별 스케일링을 적용합니다. Fourier 필터링을 통해 가이던스 신호를 저주파와 고주파 성분으로 분리한 뒤, 각 주파수 대역에 서로 다른 스케일링 요소를 적용한 후 다시 합칩니다. 이를 통해 생성된 출력의 다양한 측면에 가이던스가 미치는 영향을 더욱 세밀하게 조정할 수 있습니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `model` | MODEL | 예 | - | 주파수 스케일링을 적용할 모델 |
| `scale_low` | FLOAT | 아니요 | 0 - 10 | 저주파 성분에 적용할 스케일링 요소 (기본값: 1.0) |
| `scale_high` | FLOAT | 아니요 | 0 - 10 | 고주파 성분에 적용할 스케일링 요소 (기본값: 1.25) |
| `freq_cutoff` | INT | 아니요 | 1 - 10000 | 중심 주파수를 기준으로 저주파로 간주할 주파수 인덱스의 수 (기본값: 20) |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `model` | MODEL | 주파수별 스케일링이 적용된 수정된 모델 |
