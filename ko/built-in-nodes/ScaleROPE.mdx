ScaleROPE 노드를 사용하면 모델의 로터리 위치 임베딩(ROPE)을 수정할 수 있으며, X, Y 및 T(시간) 구성 요소에 각각 별도의 스케일링 및 시프팅 요소를 적용합니다. 이 노드는 모델의 위치 인코딩 동작을 조정하는 데 사용되는 고급 실험적 노드입니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `model` | MODEL | 예 | - | ROPE 매개변수가 수정될 모델입니다. |
| `scale_x` | FLOAT | 아니요 | 0.0 - 100.0 | ROPE의 X 구성 요소에 적용할 스케일링 요소입니다(기본값: 1.0). |
| `shift_x` | FLOAT | 아니요 | -256.0 - 256.0 | ROPE의 X 구성 요소에 적용할 시프트 값입니다(기본값: 0.0). |
| `scale_y` | FLOAT | 아니요 | 0.0 - 100.0 | ROPE의 Y 구성 요소에 적용할 스케일링 요소입니다(기본값: 1.0). |
| `shift_y` | FLOAT | 아니요 | -256.0 - 256.0 | ROPE의 Y 구성 요소에 적용할 시프트 값입니다(기본값: 0.0). |
| `scale_t` | FLOAT | 아니요 | 0.0 - 100.0 | ROPE의 T(시간) 구성 요소에 적용할 스케일링 요소입니다(기본값: 1.0). |
| `shift_t` | FLOAT | 아니요 | -256.0 - 256.0 | ROPE의 T(시간) 구성 요소에 적용할 시프트 값입니다(기본값: 0.0). |

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `model` | MODEL | 새로운 ROPE 스케일링 및 시프팅 매개변수가 적용된 모델입니다. |
