> 이 문서는 AI로 생성되었습니다. 오류를 발견하거나 개선할 점이 있다면 언제든지 기여해 주세요! [GitHub에서 편집하기](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/ImageOnlyCheckpointSave/en.md)

ImageOnlyCheckpointSave 노드는 모델, CLIP 비전 인코더, VAE를 포함한 체크포인트 파일을 저장합니다. 지정된 파일명 접두사로 safetensors 파일을 생성하여 출력 디렉토리에 저장합니다. 이 노드는 이미지 관련 모델 구성 요소를 하나의 체크포인트 파일에 함께 저장하도록 특별히 설계되었습니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `model` | MODEL | 예 | - | 체크포인트에 저장할 모델 |
| `clip_vision` | CLIP_VISION | 예 | - | 체크포인트에 저장할 CLIP 비전 인코더 |
| `vae` | VAE | 예 | - | 체크포인트에 저장할 VAE(변분 오토인코더) |
| `filename_prefix` | STRING | 예 | - | 출력 파일명의 접두사 (기본값: "checkpoints/ComfyUI") |
| `prompt` | PROMPT | 아니요 | - | 워크플로우 프롬프트 데이터용 숨겨진 매개변수 |
| `extra_pnginfo` | EXTRA_PNGINFO | 아니요 | - | 추가 PNG 메타데이터용 숨겨진 매개변수 |

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| - | - | 이 노드는 출력을 반환하지 않습니다 |
