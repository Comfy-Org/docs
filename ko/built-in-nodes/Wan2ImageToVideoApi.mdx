> 이 문서는 AI가 생성했습니다. 오류를 발견하거나 개선할 점이 있으면 언제든지 기여해 주세요! [GitHub에서 편집하기](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/Wan2ImageToVideoApi/en.md)

Wan 2.7 이미지 to 비디오 노드는 첫 프레임 이미지를 기반으로 비디오를 생성합니다. 선택적으로 마지막 프레임 이미지를 제공하여 두 이미지 간의 전환을 만들거나, 오디오 파일을 제공해 비디오의 움직임과 타이밍을 안내할 수 있습니다. 이 노드는 사용자의 텍스트 설명을 바탕으로 AI 모델을 이용해 장면을 애니메이션화합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `model` | COMBO | 예 | `"wan2.7-i2v"` | 비디오 생성에 사용할 AI 모델입니다. |
| `model.prompt` | STRING | 예 | - | 비디오에 포함하고자 하는 요소와 시각적 특성에 대한 텍스트 설명입니다. 영어와 중국어를 지원합니다. |
| `model.negative_prompt` | STRING | 예 | - | 모델이 피해야 할 요소나 특성에 대한 텍스트 설명입니다. |
| `model.resolution` | COMBO | 예 | `"720P"`<br />`"1080P"` | 출력 비디오의 해상도입니다. |
| `model.duration` | INT | 예 | 2~15 | 생성된 비디오의 길이(초 단위, 기본값: 5). |
| `first_frame` | IMAGE | 예 | - | 비디오의 첫 프레임으로 사용할 이미지입니다. 출력 비디오의 종횡비는 이 이미지에서 파생됩니다. |
| `last_frame` | IMAGE | 선택 | - | 선택적으로 마지막 프레임으로 사용할 이미지입니다. 제공될 경우, 모델은 첫 프레임에서 이 마지막 프레임으로 전환되는 비디오를 생성합니다. |
| `audio` | AUDIO | 선택 | - | 비디오 생성을 유도하는 선택적 오디오 파일로, 입술 동기화나 비트에 맞춘 움직임에 유용합니다. 길이는 2~30초 사이여야 합니다. 제공되지 않을 경우, 모델은 적합한 배경 음악이나 효과음을 생성합니다. |
| `seed` | INT | 예 | 0~2147483647 | 생성의 무작위성을 제어하는 시드 값입니다(기본값: 0). |
| `prompt_extend` | BOOLEAN | 예 | - | 활성화 시, 노드는 AI의 도움을 받아 텍스트 프롬프트를 강화합니다(기본값: True). 이는 고급 설정입니다. |
| `watermark` | BOOLEAN | 예 | - | 활성화 시, 최종 비디오에 AI가 생성한 워터마크가 추가됩니다(기본값: False). 이는 고급 설정입니다. |

**참고:** `audio` 입력에는 길이 제약이 있습니다. 제공될 경우, 오디오 파일의 길이는 2~30초 사이여야 합니다.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `output` | VIDEO | 생성된 비디오 파일입니다. |
