> 이 문서는 AI가 생성했습니다. 오류를 발견하거나 개선할 점이 있으면 언제든지 기여해 주세요! [GitHub에서 편집하기](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/SaveLoRANode/en.md)

SaveLoRA 노드는 LoRA(Low-Rank Adaptation) 모델을 출력 디렉터리에 저장합니다. 이 노드는 LoRA 모델을 입력으로 받아 자동으로 생성된 파일명을 가진 safetensors 파일을 만듭니다. 파일명 접두사를 맞춤 설정할 수 있으며, 더 나은 정리를 위해 선택적으로 학습 단계 수를 파일명에 포함시킬 수도 있습니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `lora` | LORA_MODEL | 예 | - | 저장할 LoRA 모델입니다. LoRA 레이어가 포함된 모델은 사용하지 마세요. |
| `prefix` | STRING | 예 | - | 저장할 LoRA 파일에 사용할 접두어 (기본값: "loras/ComfyUI_trained_lora") |
| `steps` | INT | 아니요 | - | 선택사항: LoRA가 학습된 단계 수로, 저장된 파일의 이름에 사용됩니다. |

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-----------|-----------|-------------|
| *없음* | - | 이 노드는 출력을 반환하지 않으며, LoRA 모델을 출력 디렉터리에 저장합니다. |
