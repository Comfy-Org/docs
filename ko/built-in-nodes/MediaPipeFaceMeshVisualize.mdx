## 개요

입력 이미지 위에 얼굴 랜드마크 점과 연결선(얼굴 메쉬)을 그립니다. 이 노드는 얼굴 감지 노드가 생성한 랜드마크 데이터를 사용하여 눈, 코, 입, 얼굴 윤곽 등 감지된 얼굴 특징을 시각화합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `face_landmarks` | FACE_LANDMARKS | 예 | | 감지 노드에서 가져온 얼굴 랜드마크 데이터입니다. |
| `image` | IMAGE | 아니요 | | 메쉬를 그릴 이미지입니다. 연결되지 않은 경우, 감지 결과와 동일한 크기의 검은색 캔버스가 사용됩니다. |
| `connections` | COMBO | 예 | `"all"`<br />`"fill"`<br />`"custom"` | 얼굴 메쉬의 어떤 부분을 그릴지 결정합니다. `"all"`은 전체 메쉬(타원, 눈, 눈썹, 입술, 홍채, 코)를 그립니다. `"fill"`은 얼굴 타원의 실체 다각형(실루엣 마스크)을 그립니다. `"custom"`은 각 특징을 개별적으로 켜고 끌 수 있게 합니다. (기본값: `"all"`) |
| `color` | COLOR | 예 | | 메쉬 선과 점의 색상입니다. (기본값: `#00ff00`) |
| `thickness` | INT | 예 | 0부터 8까지 | 메쉬 선의 두께(픽셀 단위). 이 값을 0으로 설정하면 선 그리기가 비활성화됩니다. (기본값: 1) |
| `point_size` | INT | 예 | 0부터 16까지 | 랜드마크 점의 반경(픽셀 단위). 이 값을 0으로 설정하면 점 그리기가 비활성화됩니다. (기본값: 2) |

**`connections` 매개변수 참고:** `"custom"`을 선택하면 각 얼굴 특징별 추가적인 부울 입력이 나타납니다(예: `face_oval`, `lips`, `left_eye`, `right_eye`, `left_eyebrow`, `right_eyebrow`, `left_iris`, `right_iris`, `nose`, `tesselation`). 활성화한 특징만 그려집니다.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `IMAGE` | IMAGE | 얼굴 랜드마크 메쉬가 그려진 입력 이미지입니다. |
