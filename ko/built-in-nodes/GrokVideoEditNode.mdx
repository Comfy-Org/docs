> 이 문서는 AI로 생성되었습니다. 오류를 발견하거나 개선할 점이 있다면 언제든지 기여해 주세요! [GitHub에서 편집하기](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/GrokVideoEditNode/en.md)

이 노드는 Grok API를 사용하여 텍스트 프롬프트를 기반으로 기존 동영상을 편집합니다. 사용자의 동영상을 업로드하고, AI 모델에 설명에 따라 수정하도록 요청한 뒤, 새로 생성된 동영상을 반환합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `model` | COMBO | 예 | `"grok-imagine-video"`<br />`"grok-imagine-video-beta"` | 동영상 편집에 사용할 AI 모델 (기본값: `"grok-imagine-video"`). |
| `prompt` | STRING | 예 | N/A | 원하는 동영상에 대한 텍스트 설명. |
| `video` | VIDEO | 예 | N/A | 편집할 입력 동영상. 최대 지원 시간은 8.7초이며, 파일 크기는 50MB입니다. |
| `seed` | INT | 아니요 | 0부터 2147483647까지 | 노드가 다시 실행될지 여부를 결정하는 시드 값입니다. 실제 결과는 시드 값과 관계없이 비결정적입니다 (기본값: 0). |

**제약 조건:**

* 입력 `video`의 길이는 1초에서 8.7초 사이여야 합니다.
* 입력 `video`의 파일 크기는 50MB를 초과해서는 안 됩니다.
* `prompt`는 비어 있어서는 안 됩니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `video` | VIDEO | AI 모델이 생성한 편집된 동영상. |
