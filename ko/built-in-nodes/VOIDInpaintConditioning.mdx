VOIDInpaintConditioning 노드는 CogVideoX 모델을 이용한 인페인팅에 필요한 컨디셔닝 데이터를 준비합니다. 소스 비디오와 사전 처리된 쿼드마스크를 받아 VAE를 통해 인코딩한 후, 이를 합쳐서 모델이 마스크 영역을 채우는 데 사용하는 32채널 컨디셔닝 신호로 만듭니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `positive` | CONDITIONING | 예 | - | 인페인팅 래티언트 정보로 보강될 긍정적 컨디셔닝 |
| `negative` | CONDITIONING | 예 | - | 인페인팅 래티언트 정보로 보강될 부정적 컨디셔닝 |
| `vae` | VAE | 예 | - | 마스크와 마스크된 비디오를 래티언트 공간으로 인코딩하는 데 사용되는 VAE 모델 |
| `video` | IMAGE | 예 | - | 소스 비디오 프레임 [T, H, W, 3] |
| `quadmask` | MASK | 예 | - | VOIDQuadmaskPreprocess에서 생성된 사전 처리된 쿼드마스크 [T, H, W] |
| `width` | INT | 예 | 16부터 MAX_RESOLUTION까지 (단계: 8) | 비디오와 마스크를 리사이즈할 너비 (기본값: 672) |
| `height` | INT | 예 | 16부터 MAX_RESOLUTION까지 (단계: 8) | 비디오와 마스크를 리사이즈할 높이 (기본값: 384) |
| `length` | INT | 예 | 1부터 MAX_RESOLUTION까지 (단계: 1) | 처리할 픽셀 프레임 수. CogVideoX-Fun-V1.5(패치 크기_t=2)의 경우 latent_t는 반드시 짝수여야 하며, 홀수인 length는 내림됩니다(예: 49 → 45) (기본값: 45) |
| `batch_size` | INT | 예 | 1부터 64까지 | 출력 노이즈 래티언트의 배치 크기 (기본값: 1) |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `positive` | CONDITIONING | 인페인팅 래티언트 정보가 추가된 긍정적 컨디셔닝 |
| `negative` | CONDITIONING | 인페인팅 래티언트 정보가 추가된 부정적 컨디셔닝 |
| `latent` | LATENT | [batch_size, 16, latent_t, latent_h, latent_w] 형태의 0으로 채워진 노이즈 래티언트 텐서 |
