Rodin 3D Detail 노드는 Rodin API를 사용해 상세한 3D 자산을 생성합니다. 입력 이미지를 받아 Rodin 서비스를 통해 처리하여 세밀한 지오메트리와 재료를 갖춘 고품질 3D 모델을 만들어냅니다. 이 노드는 작업 생성부터 최종 3D 모델 파일 다운로드까지 전체 워크플로우를 처리합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `Images` | IMAGE | 예 | - | 3D 모델 생성에 사용되는 입력 이미지입니다. 최소 1개 이상의 이미지가 필요하며, 최대 5개까지 제공할 수 있습니다. |
| `Seed` | INT | 아니요 | 0~65535 | 재현 가능한 결과를 위한 임의의 시드 값 (기본값: 0) |
| `Material_Type` | STRING | 아니요 | `"PBR"`<br />`"Shaded"` | 3D 모델에 적용할 재료의 유형 (기본값: "PBR") |
| `Polygon_count` | STRING | 아니요 | `"4K-Quad"`<br />`"8K-Quad"`<br />`"18K-Quad"`<br />`"50K-Quad"`<br />`"200K-Triangle"` | 생성될 3D 모델의 목표 폴리곤 수입니다. 메쉬의 품질 수준을 결정합니다 (기본값: "18K-Quad") |

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `3D Model Path` | STRING | 생성된 3D 모델의 파일 경로 (후향적 호환성을 위해만 제공됨) |
| `GLB` | FILE3DGLB | GLB 형식으로 생성된 3D 모델 |
