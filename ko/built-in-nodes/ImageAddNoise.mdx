ImageAddNoise 노드는 입력 이미지에 무작위 노이즈를 추가합니다. 지정된 난수 시드를 사용하여 일관된 노이즈 패턴을 생성하며, 노이즈 효과의 강도를 조절할 수 있습니다. 결과 이미지는 입력 이미지와 동일한 크기를 유지하되, 시각적 질감이 추가됩니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `image` | IMAGE | 예 | - | 노이즈가 추가될 입력 이미지 |
| `seed` | INT | 예 | 0부터 18446744073709551615까지 | 노이즈 생성에 사용되는 난수 시드(기본값: 0). 이 매개변수는 "생성 후 제어" 기능을 지원합니다. |
| `strength` | FLOAT | 예 | 0.0부터 1.0까지 | 노이즈 효과의 강도를 조절합니다(기본값: 0.5, 단계: 0.01) |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-----------|-----------|-------------|
| `image` | IMAGE | 노이즈가 적용된 출력 이미지 |
