ImageAddNoise 노드는 입력 이미지에 무작위 노이즈를 추가합니다. 지정된 난수 시드를 사용하여 일관된 노이즈 패턴을 생성하고, 노이즈 효과의 강도를 제어할 수 있습니다. 결과 이미지는 입력 이미지와 동일한 크기를 유지하지만 시각적 텍스처가 추가됩니다.

## 입력

| 매개변수 | 설명 | 데이터 타입 | 필수 여부 | 범위 |
| --- | --- | --- | --- | --- |
| `이미지` | 노이즈를 추가할 입력 이미지 | IMAGE | 예 | - |
| `시드` | 노이즈 생성에 사용되는 난수 시드 (기본값: 0) | INT | 예 | 0 ~ 18446744073709551615 |
| `강도` | 노이즈 효과의 강도를 제어합니다 (기본값: 0.5) | FLOAT | 예 | 0.0 ~ 1.0 |

## 출력

| 출력 이름 | 설명 | 데이터 타입 |
| --- | --- | --- |
| `이미지` | 노이즈가 적용된 출력 이미지 | IMAGE |
