LoadLatent 노드는 입력 디렉터리에 있는 .latent 파일에서 이전에 저장된 잠재 표현을 로드합니다. 이 노드는 파일에서 잠재 텐서 데이터를 읽고 필요한 스케일링 조정을 적용한 후, 다른 노드에서 사용할 수 있도록 잠재 데이터를 반환합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `latent` | 문자열 | 예 | 입력 디렉터리에 있는 모든 .latent 파일 | 입력 디렉터리에 있는 사용 가능한 파일 중에서 어떤 .latent 파일을 로드할지 선택합니다 |

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `LATENT` | LATENT | 선택한 파일에서 로드된 잠재 표현 데이터를 반환합니다 |
