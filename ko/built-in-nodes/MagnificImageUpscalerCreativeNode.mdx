> 이 문서는 AI로 생성되었습니다. 오류를 발견하거나 개선할 점이 있다면 언제든지 기여해 주세요! [GitHub에서 편집하기](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/MagnificImageUpscalerCreativeNode/en.md)

이 노드는 Magnific AI 서비스를 사용하여 이미지를 업스케일하고 창의적으로 향상시킵니다. 텍스트 프롬프트를 통해 향상 과정을 안내하고, 특정 스타일을 선택하여 최적화하며, 디테일 수준, 원본과의 유사성, 스타일화 강도 등 창의적 과정의 다양한 측면을 제어할 수 있습니다. 노드는 선택한 배율(2x, 4x, 8x 또는 16x)로 업스케일된 이미지를 출력하며, 최대 출력 크기는 2530만 화소입니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `image` | IMAGE | 예 | - | 업스케일하고 향상시킬 입력 이미지입니다. |
| `prompt` | STRING | 아니요 | - | 이미지의 창의적 향상을 안내하는 텍스트 설명입니다. 선택사항(기본값: 비어 있음). |
| `scale_factor` | COMBO | 예 | `"2x"`<br />`"4x"`<br />`"8x"`<br />`"16x"` | 이미지의 크기를 확대할 배율입니다. |
| `optimized_for` | COMBO | 예 | `"standard"`<br />`"soft_portraits"`<br />`"hard_portraits"`<br />`"art_n_illustration"`<br />`"videogame_assets"`<br />`"nature_n_landscapes"`<br />`"films_n_photography"`<br />`"3d_renders"`<br />`"science_fiction_n_horror"` | 향상 과정을 최적화할 스타일이나 콘텐츠 유형입니다. |
| `creativity` | INT | 아니요 | -10부터 10까지 | 이미지에 적용되는 창의적 해석의 수준을 제어합니다(기본값: 0). |
| `hdr` | INT | 아니요 | -10부터 10까지 | 선명도와 디테일 수준을 제어합니다(기본값: 0). |
| `resemblance` | INT | 아니요 | -10부터 10까지 | 원본 이미지와의 유사성 수준을 제어합니다(기본값: 0). |
| `fractality` | INT | 아니요 | -10부터 10까지 | 프롬프트의 강도와 각 픽셀당 복잡성을 제어합니다(기본값: 0). |
| `engine` | COMBO | 예 | `"automatic"`<br />`"magnific_illusio"`<br />`"magnific_sharpy"`<br />`"magnific_sparkle"` | 처리에 사용할 특정 AI 엔진입니다. 이는 고급 매개변수입니다. |
| `auto_downscale` | BOOLEAN | 아니요 | - | 활성화 시, 요청된 업스케일이 최대 허용 출력 크기인 2530만 화소를 초과할 경우 노드가 자동으로 입력 이미지를 다운스케일합니다. 이는 고급 매개변수입니다(기본값: False). |

**제약 조건:**

* 입력 `image`는 정확히 하나의 이미지여야 합니다.
* 입력 이미지는 최소 높이와 너비가 160픽셀 이상이어야 합니다.
* 입력 이미지의 종횡비는 1:3에서 3:1 사이여야 합니다.
* 최종 출력 크기(입력 크기에 `scale_factor`를 곱한 값)는 2530만 화소를 초과해서는 안 됩니다. 만약 `auto_downscale`가 비활성화되어 있고 이 한계를 초과할 경우 노드는 오류를 발생시킵니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-----------|-----------|-------------|
| `image` | IMAGE | 창의적으로 향상되고 업스케일된 출력 이미지입니다. |
