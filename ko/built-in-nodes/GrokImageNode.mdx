Grok Image 노드는 Grok AI 모델을 사용해 텍스트 설명을 바탕으로 하나 이상의 이미지를 생성합니다. 이 노드는 프롬프트를 외부 서비스로 전송한 뒤, 생성된 이미지를 워크플로우에서 사용할 수 있는 텐서로 반환합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `model` | COMBO | 예 | `"grok-imagine-image-quality"`<br />`"grok-imagine-image-pro"`<br />`"grok-imagine-image"`<br />`"grok-imagine-image-beta"` | 이미지 생성에 사용할 특정 Grok 모델입니다. 모델마다 품질, 속도 또는 기능이 다를 수 있습니다. |
| `prompt` | STRING | 예 | 없음 | 이미지를 생성하는 데 사용되는 텍스트 프롬프트입니다. 이 설명은 AI에게 무엇을 만들어야 하는지 안내합니다. 최소 1자 이상이어야 합니다. |
| `aspect_ratio` | COMBO | 예 | `"1:1"`<br />`"2:3"`<br />`"3:2"`<br />`"3:4"`<br />`"4:3"`<br />`"9:16"`<br />`"16:9"`<br />`"9:19.5"`<br />`"19.5:9"`<br />`"9:20"`<br />`"20:9"`<br />`"1:2"`<br />`"2:1"` | 생성할 이미지의 원하는 가로세로 비율입니다. |
| `number_of_images` | INT | 아니요 | 1~10 | 생성할 이미지의 수 (기본값: 1). |
| `seed` | INT | 아니요 | 0~2147483647 | 노드가 다시 실행될지 여부를 결정하는 시드 값입니다. 실제 이미지 결과는 비결정적이고 동일한 시드라도 다르게 나타날 수 있습니다 (기본값: 0). |
| `resolution` | COMBO | 아니요 | `"1K"`<br />`"2K"` | 생성할 이미지의 원하는 출력 해상도 (기본값: "1K"). |

**참고:** `seed` 매개변수는 주로 워크플로우 내에서 노드가 다시 실행될 때를 제어하는 데 사용됩니다. 외부 AI 서비스의 특성상, 동일한 시드를 사용하더라도 생성된 이미지는 재현되거나 동일하게 나오지 않습니다.

**가격 관련 참고사항:** 이미지 생성 비용은 선택한 `model`, `resolution`, 그리고 `number_of_images`에 따라 달라집니다. 예를 들어, "grok-imagine-image-quality" 모델의 "1K" 해상도는 이미지당 $0.05이며, "2K" 해상도는 이미지당 $0.07입니다. "grok-imagine-image-pro" 모델은 이미지당 $0.07이고, 다른 모델들은 이미지당 $0.02입니다.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `output` | IMAGE | 생성된 이미지 또는 이미지 배치입니다. `number_of_images`가 1이면 단일 이미지 텐서가 반환되고, 1보다 크면 이미지 텐서 배치가 반환됩니다. |
