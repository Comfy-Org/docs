SetClipHooks 노드를 사용하면 CLIP 모델에 맞춤형 후크를 적용하여 그 동작을 고급으로 수정할 수 있습니다. 이 노드는 조건부 출력에 후크를 적용하고, 선택적으로 클립 스케줄링 기능을 활성화할 수도 있습니다. 이 노드는 입력된 CLIP 모델의 복제본을 만들어 지정된 후크 설정을 적용합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `clip` | CLIP | 예 | - | 후크를 적용할 CLIP 모델 |
| `apply_to_conds` | BOOLEAN | 예 | - | 조건부 출력에 후크를 적용할지 여부 (기본값: True) |
| `schedule_clip` | BOOLEAN | 예 | - | 클립 스케줄링을 활성화할지 여부 (기본값: False) |
| `hooks` | HOOKS | 아니요 | - | CLIP 모델에 적용할 옵션 후크 그룹 |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-----------|-----------|-------------|
| `clip` | CLIP | 지정된 후크가 적용된 복제된 CLIP 모델 |
