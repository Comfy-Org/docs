SUPIRApply 노드는 SUPIR 모델 패치를 확산 모델에 적용합니다. 이 패치를 사용해 모델의 동작을 수정함으로써 샘플링 과정 중에 입력 이미지로부터의 가이드라인을 포함하도록 합니다. 또한 이 노드는 시간에 따라 가이드라인의 강도를 조정하는 컨트롤을 제공하며, 원본 입력과의 충실도를 유지하는 옵션 기능도 포함하고 있습니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `model` | MODEL | 예 | - | SUPIR 패치가 적용될 기본 확산 모델입니다. |
| `model_patch` | MODELPATCH | 예 | - | 모델을 수정하기 위한 가중치와 구성 정보를 포함한 SUPIR 모델 패치입니다. |
| `vae` | VAE | 예 | - | 입력 이미지를 잠재 표현으로 인코딩하는 데 사용되는 VAE(변분 오토인코더)입니다. |
| `image` | IMAGE | 예 | - | 생성 과정을 안내하는 데 사용되는 입력 이미지입니다. 첫 번째 세 개의 색상 채널(RGB)만 사용됩니다. |
| `strength_start` | FLOAT | 아니요 | 0.0 - 10.0 | 샘플링 시작 시 제어 강도(높은 시그마). 이미지 가이드라인의 영향력은 이 값에서부터 시작됩니다. (기본값: 1.0) |
| `strength_end` | FLOAT | 아니요 | 0.0 - 10.0 | 샘플링 종료 시 제어 강도(낮은 시그마). 시작값에서 선형 보간됩니다. 이미지 가이드라인의 영향력은 이 값에서 종료됩니다. (기본값: 1.0) |
| `restore_cfg` | FLOAT | 아니요 | 0.0 - 20.0 | 덴오즈된 출력을 입력 잠재 표현 쪽으로 당깁니다. 값이 높을수록 입력과의 충실도가 강해집니다. 0으로 설정하면 비활성화됩니다. (기본값: 4.0) |
| `restore_cfg_s_tmin` | FLOAT | 아니요 | 0.0 - 1.0 | restore_cfg가 비활성화되는 시그마 임계값입니다. (기본값: 0.05) |

*참고:* `image` 입력은 RGB 채널만 추출하여 처리됩니다. 알파 채널이 있는 이미지를 제공하면 알파 채널은 무시됩니다.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `model` | MODEL | SUPIR 패치가 적용되고 추가적인 후처리 CFG 함수가 구성된 확산 모델입니다. |
