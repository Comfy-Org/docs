CropByBBoxes 노드는 입력 이미지 배치에서 특정 직사각형 영역을 추출하고 크기를 조정합니다. 제공된 바운딩박스 좌표를 사용하여 각 이미지에서 자를 영역을 정의합니다. 추출된 영역은 지정된 출력 크기로 리사이즈되며, 자르기를 늘리거나 원래의 종횡비를 유지하도록 패딩을 추가하는 옵션도 제공됩니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `image` | IMAGE | 예 | - | 자를 입력 이미지 배치입니다. |
| `bboxes` | BOUNDINGBOX | 예 | - | 자를 영역을 정의하는 바운딩박스 목록입니다. 이 입력은 강제로 연결되어야 합니다. |
| `output_width` | INT | 아니요 | 64 - 4096 | 각 자른 이미지의 너비(기본값: 512)입니다. |
| `output_height` | INT | 아니요 | 64 - 4096 | 각 자른 이미지의 높이(기본값: 512)입니다. |
| `padding` | INT | 아니요 | 0 - 1024 | 자르기 전에 바운딩박스 각 면에 추가되는 픽셀 수준의 여유 공간(기본값: 0)입니다. |
| `keep_aspect` | COMBO | 아니요 | `"stretch"`<br />`"pad"` | 자르기를 출력 크기에 맞춰 늘릴지, 아니면 검은색 픽셀로 패딩해 종횡비를 유지할지 선택합니다(기본값: "stretch"). |

**참고:** 이 노드는 한 번에 하나의 이미지 프레임씩 처리합니다. 만약 한 프레임에 여러 개의 바운딩박스가 제공되면, 제공된 모든 박스를 합친 최소 직사각형인 단일 자르기 영역을 계산합니다. 계산된 자르기 영역이 유효하지 않으면(예: 너비나 높이가 0), 노드는 이미지의 중앙 상단에서 기본 자르기를 생성합니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `image` | IMAGE | 모든 자른 후 리사이즈된 영역들이 하나의 이미지 배치로 쌓여 있습니다. |
