EmptySD3LatentImage 노드는 Stable Diffusion 3 모델 전용으로 특별히 포맷된 빈 잠재 이미지 텐서를 생성합니다. 이 노드는 SD3 파이프라인에서 요구하는 올바른 크기와 구조를 가진, 값이 모두 0인 텐서를 생성합니다. 이는 일반적으로 이미지 생성 워크플로우의 시작점으로 사용됩니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `width` | INT | 예 | 16부터 MAX_RESOLUTION까지 (단계: 16) | 출력 잠재 이미지의 너비(픽셀 단위, 기본값: 1024) |
| `height` | INT | 예 | 16부터 MAX_RESOLUTION까지 (단계: 16) | 출력 잠재 이미지의 높이(픽셀 단위, 기본값: 1024) |
| `batch_size` | INT | 예 | 1부터 4096까지 | 한 번에 생성할 잠재 이미지의 수(기본값: 1) |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-----------|-----------|-------------|
| `LATENT` | LATENT | SD3 호환 크기를 가진 빈 샘플을 포함한 잠재 텐서입니다. 이 텐서는 16개의 채널을 가지며, 입력 너비와 높이에 비해 공간적으로 8배 축소됩니다. |
