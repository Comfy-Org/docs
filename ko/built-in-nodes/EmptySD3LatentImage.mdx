EmptySD3LatentImage 노드는 Stable Diffusion 3 모델용으로 특별히 형식화된 빈 잠재 이미지 텐서를 생성합니다. SD3 파이프라인에서 요구하는 올바른 차원과 구조를 가진 0으로 채워진 텐서를 생성합니다. 이는 일반적으로 이미지 생성 워크플로우의 시작점으로 사용됩니다.

## 입력

| 매개변수 | 설명 | 데이터 타입 | 필수 여부 | 범위 |
| --- | --- | --- | --- | --- |
| `너비` | 출력 잠재 이미지의 픽셀 단위 너비 (기본값: 1024) | INT | 예 | 16 ~ MAX_RESOLUTION (단위: 16) |
| `높이` | 출력 잠재 이미지의 픽셀 단위 높이 (기본값: 1024) | INT | 예 | 16 ~ MAX_RESOLUTION (단위: 16) |
| `배치 크기` | 배치로 생성할 잠재 이미지의 개수 (기본값: 1) | INT | 예 | 1 ~ 4096 |

## 출력

| 출력 이름 | 설명 | 데이터 타입 |
| --- | --- | --- |
| `LATENT` | SD3 호환 차원을 가진 빈 샘플이 포함된 잠재 텐서입니다. 이 텐서는 16개 채널을 가지며, 입력 너비와 높이에 비해 8배 공간적으로 다운스케일링됩니다. | LATENT |
