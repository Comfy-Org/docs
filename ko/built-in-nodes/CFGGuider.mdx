CFGGuider 노드는 이미지 생성 과정의 샘플링 프로세스를 제어하기 위한 가이던스 시스템을 생성합니다. 모델과 긍정적이고 부정적인 조건 입력을 받아 분류기 없는 가이던스 스케일을 적용하여 원하는 콘텐츠를 향해 생성되도록 유도하고 원치 않는 요소는 피하도록 합니다. 이 노드는 샘플링 노드가 이미지 생성 방향을 제어하는 데 사용할 수 있는 가이던스 객체를 출력합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `model` | MODEL | 예 | - | 가이던스에 사용될 모델 |
| `positive` | CONDITIONING | 예 | - | 원하는 콘텐츠를 향해 생성되도록 유도하는 긍정적 조건 |
| `negative` | CONDITIONING | 예 | - | 원치 않는 콘텐츠를 멀리하도록 유도하는 부정적 조건 |
| `cfg` | FLOAT | 예 | 0.0부터 100.0까지 | 조건이 생성에 얼마나 강하게 영향을 미치는지를 제어하는 분류기 없는 가이던스 스케일 (기본값: 8.0) |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `GUIDER` | GUIDER | 샘플링 노드에 전달되어 생성 프로세스를 제어할 수 있는 가이던스 객체 |
