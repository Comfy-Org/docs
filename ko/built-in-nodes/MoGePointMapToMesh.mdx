## 개요

이 노드는 MoGe 포인트 맵을 3D 메시로 변환합니다. MoGe 심도 추정 노드가 생성한 지오메트리 데이터를 받아 삼각형으로 분할하여 UV 좌표와 선택적 텍스처를 포함한 메시로 만듭니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `moge_geometry` | MOGE_GEOMETRY | 예 | N/A | 포인트 맵, 심도 및 선택적으로 원본 이미지를 포함한 MoGe 지오메트리 데이터입니다. |
| `batch_index` | INT | 예 | 0부터 4096까지 | 배치된 MoGe 지오메트리 중 어떤 이미지의 메시를 생성할지 지정합니다. 이미지별 정점 수가 다르므로, 여러 이미지를 하나의 MESH로 쌓을 수 없습니다(기본값: 0). |
| `decimation` | INT | 예 | 1부터 8까지 | 정점 간격; 1 = 전체 해상도(기본값: 1). |
| `discontinuity_threshold` | FLOAT | 예 | 0.0부터 1.0까지 | 3x3 심도 범위가 이 값보다 큰 픽셀은 제거합니다. 0 = 비활성화(기본값: 0.04). |
| `texture` | BOOLEAN | 예 | true/false | 원본 이미지를 baseColor 텍스처로 전달합니다(기본값: true). |

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `MESH` | MESH | 정점, 면, UV 좌표 및 선택적 원본 이미지 텍스처를 포함한 3D 메시입니다. |
