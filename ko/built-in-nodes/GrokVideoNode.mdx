Grok Video 노드는 텍스트 설명을 바탕으로 짧은 동영상을 생성합니다. 프롬프트를 사용해 처음부터 동영상을 만들거나, 하나의 입력 이미지를 프롬프트에 따라 애니메이션화할 수 있습니다. 이 노드는 외부 API로 요청을 보내고 생성된 동영상을 반환합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `model` | COMBO | 예 | `"grok-imagine-video"`<br />`"grok-imagine-video-beta"` | 동영상 생성에 사용할 모델입니다. |
| `prompt` | STRING | 예 | - | 원하는 동영상의 텍스트 설명입니다. |
| `resolution` | COMBO | 예 | `"480p"`<br />`"720p"` | 출력 동영상의 해상도입니다. |
| `aspect_ratio` | COMBO | 예 | `"auto"`<br />`"16:9"`<br />`"4:3"`<br />`"3:2"`<br />`"1:1"`<br />`"2:3"`<br />`"3:4"`<br />`"9:16"` | 출력 동영상의 종횡비입니다(기본값: "auto"). |
| `duration` | INT | 예 | 1~15 | 출력 동영상의 지속 시간(초)입니다(기본값: 6). |
| `seed` | INT | 예 | 0~2147483647 | 노드가 다시 실행될지 여부를 결정하는 시드입니다; 실제 결과는 시드와 관계없이 비결정적입니다(기본값: 0). |
| `image` | IMAGE | 선택 | - | 애니메이션화할 옵션 입력 이미지입니다. |

**참고:** `image`가 제공되면 단일 이미지만 지원됩니다. 여러 이미지를 제공하면 오류가 발생합니다. `prompt`는 공백을 제거한 후 최소 1자 이상이어야 합니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `output` | VIDEO | 생성된 동영상입니다. |
