이 노드는 제공된 시작 프레임과 끝 프레임 사이를 보간하여 비디오를 생성하며, 텍스트 프롬프트에 따라 안내됩니다. Vidu Q3 모델을 사용해 두 이미지 간의 매끄러운 전환을 만들어 지정된 지속 시간과 해상도의 비디오를 생성합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `model` | COMBO | 예 | `"viduq3-pro"`<br />`"viduq3-turbo"` | 비디오 생성에 사용할 모델입니다. 옵션을 선택하면 `resolution`, `duration`, `audio`에 대한 추가 구성 매개변수가 표시됩니다. |
| `model.resolution` | COMBO | 예 | `"720p"`<br />`"1080p"` | 출력 비디오의 해상도입니다. 이 매개변수는 `model`을 선택한 후 표시됩니다. |
| `model.duration` | INT | 예 | 1부터 16까지 | 출력 비디오의 지속 시간(초)(기본값: 5). 이 매개변수는 `model`을 선택한 후 표시됩니다. |
| `model.audio` | BOOLEAN | 예 | `True` / `False` | 활성화 시 대사와 효과음을 포함한 사운드가 있는 비디오를 출력합니다(기본값: False). 이 매개변수는 `model`을 선택한 후 표시됩니다. |
| `first_frame` | IMAGE | 예 | - | 비디오 시퀀스의 시작 이미지입니다. |
| `end_frame` | IMAGE | 예 | - | 비디오 시퀀스의 끝 이미지입니다. |
| `prompt` | STRING | 예 | - | 비디오 생성을 안내하는 텍스트 설명(최대 2000자). |
| `seed` | INT | 아니요 | 0부터 2147483647까지 | 생성의 무작위성을 제어하는 시드 값입니다(기본값: 1). |

**참고:** `first_frame`과 `end_frame` 이미지는 최적의 결과를 위해 비율이 비슷해야 합니다. 두 이미지의 종횡비는 서로 80%에서 125% 사이여야 합니다(상대적인 근접성은 0.8에서 1.25 사이).

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `video` | VIDEO | 생성된 비디오 파일입니다. |
