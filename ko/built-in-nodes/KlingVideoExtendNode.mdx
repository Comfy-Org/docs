Kling 비디오 확장 노드는 다른 Kling 노드가 생성한 비디오를 확장할 수 있게 해줍니다. 비디오 ID로 식별된 기존 비디오를 받아 텍스트 프롬프트에 따라 추가 콘텐츠를 생성합니다. 이 노드는 확장 요청을 Kling API로 전송하고, 확장된 비디오와 함께 새로운 ID와 지속시간을 반환합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `prompt` | 문자열 | 선택 | - | 비디오 확장을 안내하는 긍정적 텍스트 프롬프트 |
| `negative_prompt` | 문자열 | 선택 | - | 확장된 비디오에서 피해야 할 요소에 대한 부정적 텍스트 프롬프트 |
| `cfg_scale` | 부동소수점 | 선택 | 0.0 - 1.0 | 프롬프트 안내의 강도를 조절합니다(기본값: 0.5) |
| `video_id` | 문자열 | 필수 | - | 확장할 비디오의 ID입니다. 텍스트-비디오, 이미지-비디오 및 이전 비디오 확장 작업으로 생성된 비디오를 지원합니다. 확장 후 총 지속시간은 3분을 초과할 수 없습니다. |

**참고:** `video_id`는 반드시 다른 Kling 노드가 생성한 비디오를 참조해야 하며, 확장 후 총 지속시간은 3분을 넘어서는 안 됩니다. 긍정적 프롬프트는 비워둘 수 없으며, 두 프롬프트 모두 2500자 미만이어야 합니다.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `output` | 비디오 | Kling API가 생성한 확장된 비디오 |
| `video_id` | 문자열 | 확장된 비디오의 고유 식별자 |
| `duration` | 문자열 | 확장된 비디오의 지속시간 |
