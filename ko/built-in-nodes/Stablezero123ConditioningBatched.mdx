이 노드는 StableZero123 모델에 특화된 방식으로 조건부 정보를 배치 처리하도록 설계되었습니다. 여러 세트의 조건부 데이터를 동시에 효율적으로 처리하며, 배치 처리가 중요한 시나리오에서 워크플로우를 최적화합니다.

## 입력

| 매개변수             | 데이터 타입 | 설명 |
|----------------------|--------------|-------------|
| `clip_vision`         | `CLIP_VISION` | 조건부 처리를 위한 시각적 맥락을 제공하는 CLIP 비전 임베딩입니다. |
| `init_image`          | `IMAGE`      | 조건부 처리의 시작점이 되는 초기 이미지로, 생성 프로세스의 기준이 됩니다. |
| `vae`                 | `VAE`        | 조건부 처리 과정에서 이미지를 인코딩하고 디코딩하는 데 사용되는 변분 오토인코더입니다. |
| `width`               | `INT`        | 출력 이미지의 너비입니다. |
| `height`              | `INT`        | 출력 이미지의 높이입니다. |
| `batch_size`          | `INT`        | 한 번에 처리할 조건부 세트의 수입니다. |
| `elevation`           | `FLOAT`      | 3D 모델 조건부 처리를 위한 고도 각도로, 생성된 이미지의 원근감에 영향을 줍니다. |
| `azimuth`             | `FLOAT`      | 3D 모델 조건부 처리를 위한 방위 각도로, 생성된 이미지의 방향성에 영향을 줍니다. |
| `elevation_batch_increment` | `FLOAT` | 배치 내 고도 각도의 증분 변화량으로, 다양한 원근감을 제공합니다. |
| `azimuth_batch_increment` | `FLOAT` | 배치 내 방위 각도의 증분 변화량으로, 다양한 방향성을 제공합니다. |

## 출력

| 매개변수     | 데이터 타입 | 설명 |
|---------------|--------------|-------------|
| `positive`    | `CONDITIONING` | 특정한 특징이나 측면을 강조하기 위해 조정된 긍정적 조건부 출력입니다. |
| `negative`    | `CONDITIONING` | 특정한 특징이나 측면을 약화하기 위해 조정된 부정적 조건부 출력입니다. |
| `latent`      | `LATENT`     | 조건부 처리 과정에서 도출된 잠재 표현으로, 추가 처리나 생성 단계에 바로 사용할 수 있습니다. |
