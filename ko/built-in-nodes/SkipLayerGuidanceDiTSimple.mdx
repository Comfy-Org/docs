SkipLayerGuidanceDiT 노드의 간단한 버전으로, 노이즈 제거 과정 중 무조건적 패스만 수정합니다. 이 노드는 DiT(Diffusion Transformer) 모델의 특정 트랜스포머 레이어에 스킵 레이어 가이던스를 적용하며, 지정된 타이밍과 레이어 매개변수에 따라 무조건적 패스 중 특정 레이어를 선택적으로 건너뜁니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `model` | MODEL | 예 | - | 스킵 레이어 가이던스를 적용할 모델 |
| `double_layers` | STRING | 아니요 | - | 건너뛸 더블 블록 레이어 인덱스의 쉼표로 구분된 리스트 (기본값: "7, 8, 9") |
| `single_layers` | STRING | 아니요 | - | 건너뛸 싱글 블록 레이어 인덱스의 쉼표로 구분된 리스트 (기본값: "7, 8, 9") |
| `start_percent` | FLOAT | 아니요 | 0.0 - 1.0 | 스킵 레이어 가이던스가 시작되는 노이즈 제거 과정의 시작 비율 (기본값: 0.0) |
| `end_percent` | FLOAT | 아니요 | 0.0 - 1.0 | 스킵 레이어 가이던스가 멈추는 노이즈 제거 과정의 종료 비율 (기본값: 1.0) |

**참고:** 스킵 레이어 가이던스는 `double_layers`와 `single_layers` 모두 유효한 레이어 인덱스를 포함하는 경우에만 적용됩니다. 둘 다 비어 있으면 노드는 원래 모델을 그대로 반환합니다. 스킵 레이어 가이던스는 현재 노이즈 제거 단계의 시그마 값이 `start_percent`와 `end_percent` 사이에 있을 때만 활성화됩니다(내부적으로 시그마 값으로 변환됨).

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `model` | MODEL | 지정된 레이어에 스킵 레이어 가이던스가 적용된 수정된 모델 |
