최소한의 변경으로 이미지를 4K 해상도로 업스케일합니다. 이 노드는 Stability AI의 보수적 업스케일링 기술을 사용하여 이미지 해상도를 높이면서 원본 내용을 유지하고 미세한 변화만 가합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `image` | IMAGE | 예 | - | 업스케일할 입력 이미지 |
| `prompt` | STRING | 예 | - | 출력 이미지에서 보고 싶은 내용입니다. 요소, 색상, 주제를 명확히 정의하는 강력하고 구체적인 프롬프트를 사용하면 더 나은 결과를 얻을 수 있습니다. (기본값: 빈 문자열) |
| `creativity` | FLOAT | 예 | 0.2-0.5 | 초기 이미지에 크게 의존하지 않고 추가적인 디테일을 생성할 가능성을 조정합니다. (기본값: 0.35) |
| `seed` | INT | 예 | 0-4294967294 | 노이즈 생성에 사용되는 난수 시드입니다. (기본값: 0) |
| `negative_prompt` | STRING | 아니요 | - | 출력 이미지에서 보고 싶지 않은 키워드입니다. 이는 고급 기능입니다. (기본값: 빈 문자열) |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `image` | IMAGE | 4K 해상도로 업스케일된 이미지 |
