# ComfyOrNode

ComfyOrNode는 일련의 입력 값들에 논리적 OR 연산을 수행합니다. 제공된 값 중 하나라도 파이썬의 표준 참 거짓 규칙에 따라 참으로 간주되면 `true`를 반환합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `value` | ANY | 예 | 여러 값 허용 | 참 거짓 여부를 평가할 값입니다. 더 많은 입력을 추가하여 여러 값을 제공할 수 있습니다. 노드는 이러한 값 중 하나라도 참이면 `true`를 반환합니다. |

**참고:** 노드는 최소 1개의 입력 값을 받습니다. 필요에 따라 자동 확장 기능을 사용해 더 많은 입력을 추가할 수 있습니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `BOOLEAN` | BOOLEAN | 입력 값 중 하나라도 참이면 `true`를 반환하고, 모든 입력 값이 거짓이면 `false`를 반환합니다. |
