이 노드는 ComfyUI 출력 디렉토리에 3D 모델을 파일로 저장하며, 출력 크기와 카메라/뷰포트 설정을 고급 제어할 수 있습니다. 또한 다운스트림 노드를 위해 3D 모델, 모델 정보, 카메라 정보 및 크기를 전달합니다.

## 입력

| 매개변수 | 설명 | 데이터 타입 | 필수 | 범위 |
|-----------|-------------|-----------|----------|-------|
| `model_3d` | 상위 3D 노드의 3D 모델 파일입니다. | FILE3D | 예 | GLB, GLTF, FBX, OBJ, STL, USDZ, Any |
| `filename_prefix` | 저장된 파일 이름의 접두사입니다(기본값: "3d/ComfyUI"). | STRING | 예 | 자유 텍스트 |
| `viewport_state` | 3D 로드 노드의 뷰포트 상태로, 카메라 및 모델 정보를 포함합니다. | LOAD3D | 예 | - |
| `model_3d_info` | 뷰포트 상태를 재정의하는 선택적 3D 모델 정보입니다. | LOAD3DMODELINFO | 아니요 | - |
| `camera_info` | 뷰포트 상태를 재정의하는 선택적 카메라 정보입니다. | LOAD3DCAMERA | 아니요 | - |
| `width` | 출력 미리보기의 픽셀 너비입니다(기본값: 1024). | INT | 예 | 1 ~ 4096 |
| `height` | 출력 미리보기의 픽셀 높이입니다(기본값: 1024). | INT | 예 | 1 ~ 4096 |

## 출력

| 출력 이름 | 설명 | 데이터 타입 |
|-------------|-------------|-----------|
| `model_3d` | 입력에서 전달된 3D 모델 파일입니다. | FILE3D |
| `model_3d_info` | 뷰포트 상태 또는 선택적 입력에서 가져온 모델 정보입니다. | LOAD3DMODELINFO |
| `camera_info` | 뷰포트 상태 또는 선택적 입력에서 가져온 카메라 정보입니다. | LOAD3DCAMERA |
| `width` | 입력에서 전달된 너비 값입니다. | INT |
| `height` | 입력에서 전달된 높이 값입니다. | INT |
