이 노드는 3D 모델을 ComfyUI 출력 디렉토리에 저장하고 고급 미리보기 기능을 제공합니다. 출력 파일 이름, 미리보기 크기를 지정하고, 선택적으로 카메라 및 모델 정보를 전달하여 향상된 3D 뷰어 환경을 구성할 수 있습니다.

## 입력

| 매개변수 | 설명 | 데이터 타입 | 필수 | 범위 |
|-----------|-------------|-----------|----------|-------|
| `model_3d` | 상위 3D 노드에서 전달된 3D 모델 파일입니다. | FILE3D | 예 | GLB, GLTF, FBX, OBJ, STL, USDZ 또는 모든 3D 형식 |
| `filename_prefix` | 저장될 파일 이름의 접두사입니다. 카운터와 파일 확장자가 자동으로 추가됩니다. (기본값: "3d/ComfyUI") | STRING | 예 | 모든 유효한 파일 이름 문자열 |
| `viewport_state` | 일반적으로 3D 뷰어 노드에서 전달된 현재 뷰포트 상태입니다. | LOAD3D | 예 | - |
| `model_3d_info` | 미리보기를 위한 선택적 3D 모델 정보입니다. | LOAD3DMODELINFO | 아니요 | - |
| `camera_info` | 미리보기를 위한 선택적 카메라 정보입니다. | LOAD3DCAMERA | 아니요 | - |
| `width` | 미리보기의 가로 픽셀 크기입니다. (기본값: 1024) | INT | 예 | 1 ~ 4096 |
| `height` | 미리보기의 세로 픽셀 크기입니다. (기본값: 1024) | INT | 예 | 1 ~ 4096 |

## 출력

| 출력 이름 | 설명 | 데이터 타입 |
|-------------|-------------|-----------|
| `model_3d` | 저장된 3D 모델 파일입니다. | FILE3D |
| `model_3d_info` | 하위 노드로 전달되는 3D 모델 정보입니다. | LOAD3DMODELINFO |
| `camera_info` | 하위 노드로 전달되는 카메라 정보입니다. | LOAD3DCAMERA |
| `width` | 하위 노드로 전달되는 가로 크기 값입니다. | INT |
| `height` | 하위 노드로 전달되는 세로 크기 값입니다. | INT |
