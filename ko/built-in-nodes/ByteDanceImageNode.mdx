ByteDance Image 노드는 텍스트 프롬프트를 기반으로 API를 통해 ByteDance 모델을 사용하여 이미지를 생성합니다. 모델을 선택하고 이미지 크기를 지정하며 시드와 가이던스 스케일과 같은 다양한 생성 매개변수를 제어할 수 있습니다. 이 노드는 ByteDance의 이미지 생성 서비스에 연결하여 생성된 이미지를 반환합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `model` | 문자열 | 예 | `"seedream-3-0-t2i-250415"` | 이미지 생성에 사용할 ByteDance 모델입니다. 현재 사용 가능한 모델 옵션은 하나뿐입니다. |
| `prompt` | 문자열 | 예 | - | 이미지를 생성하는 데 사용되는 텍스트 프롬프트입니다. 공백을 제거한 후 최소 1자 이상이어야 합니다. |
| `size_preset` | 문자열 | 예 | 설명 참조 | 권장 크기를 선택하세요. 맞춤형을 선택하면 아래의 너비와 높이를 사용합니다. 사용 가능한 사전 설정은 `RECOMMENDED_PRESETS` 목록에 정의되어 있습니다. |
| `width` | 정수 | 예 | 512부터 2048까지 (64 단계) | 이미지의 맞춤형 너비입니다. 이 값은 `size_preset`가 `맞춤형`으로 설정된 경우에만 사용됩니다. 기본값: 1024. |
| `height` | 정수 | 예 | 512부터 2048까지 (64 단계) | 이미지의 맞춤형 높이입니다. 이 값은 `size_preset`가 `맞춤형`으로 설정된 경우에만 사용됩니다. 기본값: 1024. |
| `seed` | 정수 | 아니요 | 0부터 2147483647까지 (1 단계) | 생성에 사용할 시드입니다. 기본값: 0. |
| `guidance_scale` | 부동소수점 | 아니요 | 1.0부터 10.0까지 (0.01 단계) | 값이 높을수록 이미지가 프롬프트를 더 잘 따르게 됩니다. 기본값: 2.5. |
| `watermark` | 불리언 | 아니요 | true / false | 이미지에 "AI 생성" 워터마크를 추가할지 여부입니다. 기본값: false. 이는 고급 매개변수입니다. |

**크기 매개변수에 대한 참고사항:** `width`와 `height` 매개변수는 `size_preset`가 `맞춤형`으로 설정된 경우에만 사용됩니다. 사전 설정된 크기를 선택하면 해당 사전 설정의 치수가 맞춤형 너비와 높이 값을 무시하게 됩니다. 맞춤형 치수를 사용할 때는 너비와 높이 모두 512~2048픽셀 사이여야 합니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-----------|-----------|-------------|
| `IMAGE` | 이미지 | ByteDance API에서 텐서 형태로 반환된 생성된 이미지입니다. |
