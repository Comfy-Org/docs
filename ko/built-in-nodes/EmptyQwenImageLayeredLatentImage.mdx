Empty Qwen Image Layered Latent 노드는 Qwen 이미지 모델과 함께 사용하기 위한 빈 다중 레이어 잠재 표현을 생성합니다. 이 노드는 지정된 레이어 수, 배치 크기 및 공간적 차원으로 구성된, 모두 0으로 채워진 텐서를 생성합니다. 이 빈 잠재 표현은 이후 이미지 생성 또는 조작 워크플로우의 시작점 역할을 합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `width` | INT | 예 | 16부터 MAX_RESOLUTION까지 | 생성할 잠재 이미지의 너비입니다. 값은 16으로 나누어져야 합니다. (기본값: 640) |
| `height` | INT | 예 | 16부터 MAX_RESOLUTION까지 | 생성할 잠재 이미지의 높이입니다. 값은 16으로 나누어져야 합니다. (기본값: 640) |
| `layers` | INT | 예 | 0부터 MAX_RESOLUTION까지 | 잠재 구조에 추가할 레이어 수입니다. 이는 잠재 표현의 깊이를 정의합니다. (기본값: 3) |
| `batch_size` | INT | 아니요 | 1부터 4096까지 | 한 배치에서 생성할 잠재 샘플의 수입니다. (기본값: 1) |

**참고:** `width`와 `height` 매개변수는 내부적으로 8로 나누어 출력 잠재 텐서의 공간적 차원을 결정합니다.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `samples` | LATENT | 모두 0으로 채워진 잠재 텐서입니다. 그 형태는 `[batch_size, 16, layers + 1, height // 8, width // 8]`입니다. |
