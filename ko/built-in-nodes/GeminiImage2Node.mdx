> 이 문서는 AI로 생성되었습니다. 오류를 발견하거나 개선할 점이 있다면 언제든지 기여해 주세요! [GitHub에서 편집하기](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/GeminiImage2Node/en.md)

GeminiImage2Node는 Google의 Vertex AI Gemini 모델을 사용해 이미지를 생성하거나 편집합니다. 텍스트 프롬프트와 선택적 참조 이미지 또는 파일을 API로 전송하고, 생성된 이미지 및/또는 텍스트 설명을 반환합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `prompt` | 문자열 | 예 | 없음 | 생성할 이미지나 적용할 편집 내용을 설명하는 텍스트 프롬프트입니다. 모델이 따라야 할 제약 조건, 스타일, 세부사항 등을 포함하세요. |
| `model` | 콤보 | 예 | `"gemini-3-pro-image-preview"`<br />`"Nano Banana 2 (Gemini 3.1 Flash Image)"` | 생성에 사용할 특정 Gemini 모델입니다. "Nano Banana 2" 옵션은 내부적으로 `gemini-3.1-flash-image-preview` 모델과 매핑됩니다. |
| `seed` | 정수 | 예 | 0부터 18446744073709551615까지 | 특정 값으로 고정하면, 모델은 반복 요청 시 동일한 응답을 제공하려고 노력합니다. 결정론적 출력이 보장되지는 않습니다. 모델이나 다른 설정을 변경하면 같은 시드라도 결과가 달라질 수 있습니다. 기본값: 42. |
| `aspect_ratio` | 콤보 | 예 | `"auto"`<br />`"1:1"`<br />`"2:3"`<br />`"3:2"`<br />`"3:4"`<br />`"4:3"`<br />`"4:5"`<br />`"5:4"`<br />`"9:16"`<br />`"16:9"`<br />`"21:9"` | 출력 이미지의 원하는 종횡비입니다. 'auto'로 설정하면 입력 이미지의 종횡비와 일치하며, 이미지를 제공하지 않으면 보통 16:9 정사각형이 생성됩니다. 기본값: "auto". |
| `resolution` | 콤보 | 예 | `"1K"`<br />`"2K"`<br />`"4K"` | 목표 출력 해상도입니다. 2K/4K의 경우 기본 Gemini 업스케일러가 사용됩니다. |
| `response_modalities` | 콤보 | 예 | `"IMAGE+TEXT"`<br />`"IMAGE"` | 이미지만 출력하려면 'IMAGE'를 선택하고, 생성된 이미지와 텍스트 응답을 모두 반환하려면 'IMAGE+TEXT'를 선택하세요. |
| `images` | 이미지 | 아니요 | 없음 | 선택적 참조 이미지입니다. 여러 이미지를 포함하려면 배치 이미지 노드를 사용하세요(최대 14개). |
| `files` | 맞춤형 | 아니요 | 없음 | 모델의 컨텍스트로 사용할 선택적 파일입니다. Gemini 생성 콘텐츠 입력 파일 노드에서 출력되는 입력을 받습니다. |
| `system_prompt` | 문자열 | 아니요 | 없음 | AI의 행동을 지시하는 기본적인 지침입니다. 기본값: 이미지 생성을 위한 사전 정의된 시스템 프롬프트입니다. |

**제약 조건:**

* `images` 입력은 최대 14개의 이미지를 지원합니다. 더 많은 이미지를 제공하면 오류가 발생합니다.
* `files` 입력은 `GEMINI_INPUT_FILES` 데이터 유형을 출력하는 노드에 연결되어야 합니다.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|------------|-----------|--------|
| `image` | 이미지 | Gemini 모델이 생성하거나 편집한 이미지입니다. |
| `string` | 문자열 | 모델에서 나온 텍스트 응답입니다. `response_modalities`를 "IMAGE"로 설정한 경우 이 출력은 비어 있습니다. |
