Adjust Contrast 노드는 입력 이미지의 대비 수준을 조정합니다. 이미지의 밝은 영역과 어두운 영역 간의 차이를 조정하여 작동합니다. 1.0의 값은 이미지를 변경하지 않고, 1.0 미만의 값은 대비를 줄이고, 1.0 초과의 값은 대비를 높입니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `image` | IMAGE | 예 | - | 대비를 조정할 입력 이미지입니다. |
| `factor` | FLOAT | 아니요 | 0.0 - 2.0 | 대비 요인. 1.0 = 변화 없음, <1.0 = 더 낮은 대비, >1.0 = 더 높은 대비. (기본값: 1.0) |

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `image` | IMAGE | 대비가 조정된 결과 이미지입니다. |
