이 노드는 고급 모델 병합 작업을 위해 설계되었으며, 특히 지정된 승수를 기반으로 한 두 모델 간의 매개변수를 뺄셈하는 데 사용됩니다. 이는 한 모델의 매개변수들이 다른 모델에 미치는 영향력을 조정함으로써 모델의 동작을 맞춤화할 수 있게 하며, 새로운 하이브리드 모델을 생성하는 데 유용합니다.

## 입력

| 매개변수     | 데이터 타입 | 설명 |
|---------------|--------------|-------------|
| `model1`      | `MODEL`     | 매개변수를 뺄 기본 모델입니다. |
| `model2`      | `MODEL`     | 기본 모델에서 뺄매개변수를 가진 모델입니다. |
| `multiplier`  | `FLOAT`     | 기본 모델의 매개변수에 대한 뺄셈 효과를 조정하는 부동소수점 값입니다. |

## 출력

| 매개변수 | 데이터 타입 | 설명 |
|-----------|-------------|-------------|
| `model`   | MODEL     | 한 모델의 매개변수를 다른 모델에서 뺀 후, 승수에 의해 조정된 결과 모델입니다. |
