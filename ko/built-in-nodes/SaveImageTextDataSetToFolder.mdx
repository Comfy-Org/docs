Save Image and Text Dataset to Folder 노드는 ComfyUI의 출력 디렉터리 내 특정 폴더에 이미지 목록과 해당하는 텍스트 캡션을 저장합니다. PNG 파일로 저장된 각 이미지에 대해 동일한 기본 이름을 가진 텍스트 파일이 생성되어 캡션을 저장합니다. 이는 생성된 이미지와 그에 대한 설명을 체계적으로 정리한 데이터셋을 만드는 데 유용합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `images` | IMAGE | 예 | - | 저장할 이미지 목록. |
| `texts` | STRING | 아니요 | - | 저장할 텍스트 캡션 목록. 이 입력은 선택적입니다. |
| `folder_name` | STRING | 아니요 | - | 이미지를 저장할 폴더의 이름(출력 디렉터리 내). (기본값: "dataset") |
| `filename_prefix` | STRING | 아니요 | - | 저장된 이미지 파일명의 접두사. (기본값: "image") |

**참고:** `images` 입력은 리스트입니다. `texts` 입력은 선택적이며, 제공될 경우 텍스트 캡션 목록이어야 합니다. 노드는 제공된 이미지 수와 일치하는 텍스트 캡션 수를 기대합니다. 각 캡션은 짝지어진 이미지에 해당하는 `.txt` 파일로 저장됩니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| - | - | 이 노드는 출력을 제공하지 않습니다. 파일을 직접 파일 시스템에 저장합니다. |
