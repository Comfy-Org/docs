> 이 문서는 AI로 생성되었습니다. 오류를 발견하거나 개선할 점이 있으면 언제든지 기여해 주세요! [GitHub에서 편집하기](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/TopazImageEnhance/en.md)

Topaz Image Enhance 노드는 업계 표준의 업스케일링과 이미지 향상을 제공합니다. 클라우드 기반 AI 모델을 사용해 단일 입력 이미지를 처리하여 품질, 디테일 및 해상도를 향상시킵니다. 이 노드는 향상 과정을 세밀하게 제어할 수 있으며, 창의적 가이드라인, 피사체 집중, 얼굴 보존 옵션 등을 제공합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `model` | COMBO | 예 | `"Reimagine"` | 이미지 향상에 사용할 AI 모델입니다. |
| `image` | IMAGE | 예 | - | 향상할 입력 이미지입니다. 단일 이미지만 지원됩니다. |
| `prompt` | STRING | 아니요 | - | 창의적 업스케일링 가이드를 위한 선택적 텍스트 프롬프트 (기본값: 비어 있음) |
| `subject_detection` | COMBO | 아니요 | `"All"`<br />`"Foreground"`<br />`"Background"` | 이미지의 어느 부분을 중심으로 향상할지 제어합니다 (기본값: "All"). |
| `face_enhancement` | BOOLEAN | 아니요 | - | 이미지에 얼굴이 포함된 경우 얼굴을 향상시키도록 설정합니다 (기본값: True). |
| `face_enhancement_creativity` | FLOAT | 아니요 | 0.0 - 1.0 | 얼굴 향상의 창의성 수준을 설정합니다 (기본값: 0.0). |
| `face_enhancement_strength` | FLOAT | 아니요 | 0.0 - 1.0 | 강조된 얼굴이 배경과 비교해 얼마나 선명하게 나타날지 제어합니다 (기본값: 1.0). |
| `crop_to_fill` | BOOLEAN | 아니요 | - | 기본적으로 출력 종횡비가 다르면 이미지는 레터박스 처리됩니다. 대신 출력 크기에 맞게 이미지를 자르도록 설정하려면 활성화하세요 (기본값: False). |
| `output_width` | INT | 아니요 | 0 - 32000 | 출력 이미지의 원하는 너비입니다. 값이 0이면 자동으로 계산되며, 일반적으로 원본 크기 또는 `output_height`가 지정된 경우 이를 기반으로 합니다 (기본값: 0). |
| `output_height` | INT | 아니요 | 0 - 32000 | 출력 이미지의 원하는 높이입니다. 값이 0이면 자동으로 계산되며, 일반적으로 원본 크기 또는 `output_width`가 지정된 경우 이를 기반으로 합니다 (기본값: 0). |
| `creativity` | INT | 아니요 | 1 - 9 | 향상의 전반적인 창의성 수준을 제어합니다 (기본값: 3). |
| `face_preservation` | BOOLEAN | 아니요 | - | 이미지 속 피사체의 얼굴 정체성을 보존합니다 (기본값: True). |
| `color_preservation` | BOOLEAN | 아니요 | - | 입력 이미지의 원래 색상을 보존합니다 (기본값: True). |

**참고:** 이 노드는 단일 입력 이미지만 처리할 수 있습니다. 여러 이미지의 배치를 제공하면 오류가 발생합니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `image` | IMAGE | 향상된 출력 이미지입니다. |
