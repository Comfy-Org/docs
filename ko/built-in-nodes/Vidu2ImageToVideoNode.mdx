Vidu2 이미지-비디오 생성 노드는 단일 입력 이미지를 기반으로 비디오 시퀀스를 생성합니다. 선택적 텍스트 프롬프트를 활용해 지정된 Vidu2 모델을 사용해 장면을 애니메이션화하며, 비디오의 길이, 해상도 및 움직임의 강도를 제어합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `model` | COMBO | 예 | `"viduq2-pro-fast"`<br />`"viduq2-pro"`<br />`"viduq2-turbo"` | 비디오 생성에 사용할 Vidu2 모델입니다. 모델마다 속도와 품질 간의 균형이 다릅니다. |
| `image` | IMAGE | 예 | - | 생성될 비디오의 시작 프레임으로 사용할 이미지입니다. 이미지는 하나만 허용됩니다. |
| `prompt` | STRING | 아니요 | - | 비디오 생성을 위한 선택적 텍스트 프롬프트(최대 2000자). 기본값은 빈 문자열입니다. |
| `duration` | INT | 예 | 1~10 | 생성될 비디오의 길이(초). 기본값은 5입니다. |
| `seed` | INT | 아니요 | 0~2147483647 | 재현 가능한 결과를 보장하기 위한 난수 생성용 시드 값입니다. 기본값은 1입니다. |
| `resolution` | COMBO | 예 | `"720p"`<br />`"1080p"` | 생성될 비디오의 출력 해상도입니다. 이 매개변수는 고급 옵션입니다. |
| `movement_amplitude` | COMBO | 예 | `"auto"`<br />`"small"`<br />`"medium"`<br />`"large"` | 프레임 내 객체의 움직임 진폭입니다. 이 매개변수는 고급 옵션입니다. |

**제약사항:**

* `image` 입력에는 정확히 하나의 이미지만 포함되어야 합니다.
* 입력 이미지의 종횡비는 1:4에서 4:1 사이여야 합니다.
* `prompt` 텍스트는 최대 2000자로 제한됩니다.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `output` | VIDEO | 생성된 비디오 파일입니다. |
