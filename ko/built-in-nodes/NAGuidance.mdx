NAGuidance 노드는 모델에 정규화된 주의력 가이던스를 적용합니다. 이 기술은 샘플링 과정 중 모델의 주의 메커니즘을 수정하여 원치 않는 개념으로부터 생성을 멀리하도록 함으로써, 농축된 모델이나 schnell 모델과 함께 부정적 프롬프트를 사용할 수 있게 합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `model` | MODEL | 예 | - | 정규화된 주의력 가이던스를 적용할 모델입니다. |
| `nag_scale` | FLOAT | 예 | 0.0 - 50.0 | 가이던스 스케일 요인입니다. 값이 높을수록 생성 결과가 부정적 프롬프트에서 더 멀어집니다. (기본값: 5.0) |
| `nag_alpha` | FLOAT | 예 | 0.0 - 1.0 | 정규화된 주의력의 블렌딩 요인입니다. 값이 1.0이면 원래의 주의력을 완전히 대체하고, 0.0이면 아무런 영향을 미치지 않습니다. (기본값: 0.5) |
| `nag_tau` | FLOAT | 예 | 1.0 - 10.0 | 정규화 비율을 제한하는 데 사용되는 스케일링 요인입니다. (기본값: 1.5) |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-----------|-----------|-------------|
| `model` | MODEL | 정규화된 주의력 가이던스가 활성화된 패치된 모델입니다. |
