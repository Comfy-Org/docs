> 이 문서는 AI로 생성되었습니다. 오류를 발견하거나 개선할 점이 있다면 언제든지 기여해 주세요! [GitHub에서 편집하기](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/Wan2VideoContinuationApi/en.md)

Wan 2.7 비디오 연속성 노드는 입력 동영상 클립의 끝에서 부드럽게 이어지는 새로운 비디오 세그먼트를 생성합니다. Wan 2.7 모델을 사용하여 텍스트 프롬프트를 기반으로 연속성을 합성하며, 선택적으로 특정 목표 프레임으로 끝을 유도할 수도 있습니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
| :--- | :--- | :--- | :--- | :--- |
| `model` | COMBO | 예 | `"wan2.7-i2v"` | 사용할 비디오 생성 모델입니다. |
| `model.prompt` | STRING | 예 | - | 요소와 시각적 특징을 설명하는 프롬프트입니다. 영어와 중국어를 지원합니다. (기본값: 빈 문자열) |
| `model.negative_prompt` | STRING | 예 | - | 피해야 할 사항을 설명하는 네거티브 프롬프트입니다. (기본값: 빈 문자열) |
| `model.resolution` | COMBO | 예 | `"720P"`<br />`"1080P"` | 출력 비디오의 해상도입니다. |
| `model.duration` | INT | 예 | 2~15 | 총 출력 지속시간(초). 모델은 입력 클립 이후 남은 시간을 채우기 위해 연속성을 생성합니다. (기본값: 5) |
| `first_clip` | VIDEO | 예 | - | 연속성을 이어갈 입력 비디오입니다. 지속시간: 2~10초. 출력의 종횡비는 이 비디오에서 파생됩니다. |
| `last_frame` | IMAGE | 아니요 | - | 마지막 프레임 이미지입니다. 연속성은 이 프레임을 향해 전환됩니다. |
| `seed` | INT | 예 | 0~2147483647 | 생성에 사용할 시드입니다. (기본값: 0) |
| `prompt_extend` | BOOLEAN | 예 | - | AI 도움을 통해 프롬프트를 강화할지 여부입니다. (기본값: True) |
| `watermark` | BOOLEAN | 예 | - | 결과에 AI가 생성한 워터마크를 추가할지 여부입니다. (기본값: False) |

**참고:** `first_clip` 입력 비디오는 지속시간이 2초에서 10초 사이여야 합니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
| :--- | :--- | :--- |
| `output` | VIDEO | 생성된 비디오 연속성입니다. |
