> 이 문서는 AI로 생성되었습니다. 오류를 발견하거나 개선할 점이 있다면 언제든지 기여해 주세요! [GitHub에서 편집하기](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/FluxProImageNode/en.md)

프롬프트와 해상도를 기반으로 이미지를 동기식으로 생성합니다. 이 노드는 Flux 1.1 Pro 모델을 사용하여 API 엔드포인트에 요청을 보내고, 생성된 이미지가 완전히 반환될 때까지 기다린 후 이미지를 생성합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `prompt` | 문자열 | 예 | - | 이미지 생성을 위한 프롬프트 (기본값: 빈 문자열) |
| `prompt_upsampling` | 부울 | 예 | - | 프롬프트에 대해 업스케일링을 수행할지 여부입니다. 활성화 시 자동으로 프롬프트를 수정하여 더 창의적인 생성을 가능하게 하지만 결과는 비결정적입니다(동일한 시드로도 정확히 같은 결과가 나오지 않습니다). (기본값: False) |
| `width` | 정수 | 예 | 256-1440 | 이미지 너비(픽셀 단위) (기본값: 1024, 단계: 32) |
| `height` | 정수 | 예 | 256-1440 | 이미지 높이(픽셀 단위) (기본값: 768, 단계: 32) |
| `seed` | 정수 | 예 | 0-18446744073709551615 | 노이즈 생성에 사용되는 난수 시드입니다. (기본값: 0) |
| `image_prompt` | 이미지 | 선택 | - | 생성을 안내하는 선택적 참조 이미지 |

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-----------|-----------|-------------|
| `output` | 이미지 | API에서 반환된 생성된 이미지 |
