ZImageFunControlnet 노드는 특수한 제어 네트워크를 적용하여 이미지 생성 또는 편집 과정에 영향을 줍니다. 기본 모델, 모델 패치 및 VAE를 사용하며, 이를 통해 제어 효과의 강도를 조정할 수 있습니다. 이 노드는 기본 이미지, 인페인팅 이미지 및 마스크와 함께 작동하여 보다 정밀한 편집이 가능합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `model` | MODEL | 예 | - | 생성 과정에 사용되는 기본 모델입니다. |
| `model_patch` | MODEL_PATCH | 예 | - | 제어 네트워크의 지침을 적용하는 특수한 패치 모델입니다. |
| `vae` | VAE | 예 | - | 이미지를 인코딩하고 디코딩하는 데 사용되는 변분 오토인코더입니다. |
| `strength` | FLOAT | 예 | -10.0부터 10.0까지 | 제어 네트워크의 영향 강도입니다. 양수 값은 효과를 적용하고, 음수 값은 이를 반전시킬 수 있습니다(기본값: 1.0). |
| `image` | IMAGE | 아니요 | - | 생성 과정을 안내하는 선택적 기본 이미지입니다. |
| `inpaint_image` | IMAGE | 아니요 | - | 마스크로 정의된 영역을 인페인팅하는 데 특별히 사용되는 선택적 이미지입니다. |
| `mask` | MASK | 아니요 | - | 이미지의 어떤 영역을 편집하거나 인페인팅할지 정의하는 선택적 마스크입니다. |

**참고:** `inpaint_image` 매개변수는 일반적으로 `mask`와 함께 사용되어 인페인팅할 내용을 지정합니다. 노드의 동작은 제공된 선택적 입력에 따라 달라질 수 있습니다(예: `image`를 사용해 안내하거나, `image`, `mask`, `inpaint_image`를 모두 사용해 인페인팅하는 경우).

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `model` | MODEL | 제어 네트워크 패치가 적용된 모델로, 샘플링 파이프라인에서 바로 사용할 수 있습니다. |
| `positive` | CONDITIONING | 제어 네트워크 입력에 의해 잠재적으로 수정된 긍정적 조건화입니다. |
| `negative` | CONDITIONING | 제어 네트워크 입력에 의해 잠재적으로 수정된 부정적 조건화입니다. |
