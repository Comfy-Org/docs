> 이 문서는 AI로 생성되었습니다. 오류를 발견하거나 개선할 점이 있다면 언제든지 기여해 주세요! [GitHub에서 편집하기](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/QuiverTextToSVGNode/en.md)

Quiver Text to SVG 노드는 Quiver AI의 모델을 사용하여 텍스트 설명을 기반으로 벡터 그래픽(SVG) 이미지를 생성합니다. 선택적으로 참조 이미지와 스타일 지침을 제공하여 생성 과정을 안내할 수 있습니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `prompt` | 문자열 | 예 | 없음 | 원하는 SVG 출력에 대한 텍스트 설명입니다. 생성할 내용에 대한 주요 지침입니다. |
| `instructions` | 문자열 | 아니요 | 없음 | 추가적인 스타일 또는 서식 지침입니다. 이는 선택적이고 고급 매개변수입니다. |
| `reference_images` | 이미지 | 아니요 | 0~4개의 이미지 | 생성을 안내할 최대 4개의 참조 이미지를 제공합니다. 이는 선택적 입력입니다. |
| `model` | 콤보 | 예 | `"Quiver SVG v1"`<br />`"Quiver SVG v1 Max"`<br />`"Quiver SVG v1 Preview"` | SVG 생성에 사용할 모델입니다. 사용 가능한 옵션은 Quiver API에 의해 결정됩니다. |
| `seed` | 정수 | 예 | 0~2147483647 | 노드가 다시 실행될지 여부를 결정하는 시드입니다. 실제 결과는 시드와 관계없이 비결정적입니다. 기본값: 0. |

**참고:** `reference_images` 입력은 최대 4개의 이미지를 받습니다. 더 많은 이미지를 제공하면 노드에서 오류가 발생합니다.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `SVG` | SVG | 생성된 벡터 그래픽(SVG) 이미지입니다. |
