TripleCLIPLoader 노드는 세 가지 서로 다른 텍스트 인코더 모델을 동시에 로드한 뒤 이를 하나의 CLIP 모델로 결합합니다. 이는 SD3 워크플로우와 같이 clip-l, clip-g, t5 모델이 함께 작동해야 하는 고급 텍스트 인코딩 시나리오에서 유용합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `clip_name1` | 문자열 | 예 | 여러 옵션 제공 | 사용 가능한 텍스트 인코더 중 첫 번째로 로드할 텍스트 인코더 모델 |
| `clip_name2` | 문자열 | 예 | 여러 옵션 제공 | 사용 가능한 텍스트 인코더 중 두 번째로 로드할 텍스트 인코더 모델 |
| `clip_name3` | 문자열 | 예 | 여러 옵션 제공 | 사용 가능한 텍스트 인코더 중 세 번째로 로드할 텍스트 인코더 모델 |

**참고:** 세 텍스트 인코더 매개변수 모두 시스템에 있는 사용 가능한 텍스트 인코더 모델에서 선택해야 합니다. 노드는 세 모델을 모두 로드한 뒤 이를 하나의 CLIP 모델로 결합하여 처리합니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `CLIP` | CLIP | 로드된 세 텍스트 인코더를 모두 포함하는 결합된 CLIP 모델 |
