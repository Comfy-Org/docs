Center Crop Images 노드는 이미지를 중앙에서 지정된 너비와 높이로 자릅니다. 입력 이미지의 중심 영역을 계산하고 정의된 크기의 직사각형 영역을 추출합니다. 요청한 자르기 크기가 이미지보다 클 경우, 자르기는 이미지의 경계 내로 제한됩니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `image` | IMAGE | 예 | - | 자를 입력 이미지입니다. |
| `width` | INT | 예 | 1~8192 | 자르기 영역의 너비(기본값: 512). |
| `height` | INT | 예 | 1~8192 | 자르기 영역의 높이(기본값: 512). |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `image` | IMAGE | 중앙 자르기 작업 후 결과 이미지입니다. |
