> 이 문서는 AI로 생성되었습니다. 오류를 발견하거나 개선할 점이 있다면 언제든지 기여해 주세요! [GitHub에서 편집하기](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/GeminiImage/en.md)

GeminiImage 노드는 Google의 Gemini AI 모델로부터 텍스트와 이미지 응답을 생성합니다. 텍스트 프롬프트, 이미지, 파일 등 다중 모달리티 입력을 제공하여 일관된 텍스트와 이미지 출력을 만들 수 있습니다. 이 노드는 최신 Gemini 모델을 이용해 모든 API 통신과 응답 파싱을 처리합니다.

## 입력

| 매개변수 | 데이터 타입 | 입력 유형 | 기본값 | 범위 | 설명 |
|-----------|-----------|------------|---------|-------|-------------|
| `prompt` | 문자열 | 필수 | "" | - | 생성을 위한 텍스트 프롬프트 |
| `model` | 콤보 | 필수 | gemini_2_5_flash_image_preview | 사용 가능한 Gemini 모델<br />옵션은 GeminiImageModel 열거형에서 추출됨 | 응답 생성에 사용할 Gemini 모델 |
| `seed` | 정수 | 필수 | 42 | 0부터 18446744073709551615까지 | 시드를 특정 값으로 고정하면 모델은 반복된 요청에서도 동일한 응답을 제공하려고 노력합니다. 결정론적 출력이 보장되지는 않습니다. 또한 모델이나 온도와 같은 매개변수 설정을 변경하면 동일한 시드 값을 사용하더라도 응답에 변동이 생길 수 있습니다. 기본적으로 임의의 시드 값이 사용됩니다 |
| `images` | 이미지 | 선택적 | 없음 | - | 모델의 컨텍스트로 사용할 선택적 이미지들입니다. 여러 이미지를 포함하려면 Batch Images 노드를 사용할 수 있습니다 |
| `files` | GEMINI_INPUT_FILES | 선택적 | 없음 | - | 모델의 컨텍스트로 사용할 선택적 파일들입니다. Gemini Generate Content Input Files 노드에서 입력을 받습니다 |

*참고: 이 노드에는 숨겨진 매개변수(`auth_token`, `comfy_api_key`, `unique_id`)가 포함되어 있으며, 시스템에서 자동으로 처리되므로 사용자가 직접 입력할 필요가 없습니다.*

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `IMAGE` | 이미지 | Gemini 모델에서 생성한 이미지 응답 |
| `STRING` | 문자열 | Gemini 모델에서 생성한 텍스트 응답 |
