ImageToMask 노드는 지정된 색상 채널을 기반으로 이미지를 마스크로 변환하도록 설계되었습니다. 이 노드를 통해 이미지의 빨강, 녹색, 파랑 또는 알파 채널에 해당하는 마스크 레이어를 추출할 수 있으며, 채널별 마스크 처리나 특정 작업을 필요로 하는 경우 유용합니다.

## 입력

| 매개변수   | 데이터 타입 | 설명                                                                                                          |
|-------------|-------------|----------------------------------------------------------------------------------------------------------------------|
| `image`     | `IMAGE`     | 'image' 매개변수는 지정된 색상 채널을 기반으로 마스크를 생성할 입력 이미지를 나타냅니다. 이는 결과 마스크의 내용과 특성을 결정하는 데 중요한 역할을 합니다. |
| `channel`   | COMBO[STRING] | 'channel' 매개변수는 입력 이미지의 어떤 색상 채널(빨강, 녹색, 파랑 또는 알파)을 사용해 마스크를 생성할지 지정합니다. 이 선택은 마스크의 모양과 이미지의 어느 부분이 강조되거나 마스크 처리되는지에 직접적인 영향을 미칩니다. |

## 출력

| 매개변수 | 데이터 타입 | 설명 |
|-----------|-------------|-------------|
| `mask`    | `MASK`      | 출력 'mask'는 입력 이미지의 지정된 색상 채널을 이진 또는 회색조로 표현한 것으로, 추가적인 이미지 처리나 마스크 작업에 유용합니다. |
