CosmosPredict2ImageToVideoLatent 노드는 이미지를 기반으로 비디오 잠재 표현을 생성하여 비디오를 생성합니다. 빈 비디오 잠재를 생성하거나 시작 및 종료 이미지를 포함해 지정된 크기와 지속 시간의 비디오 시퀀스를 만들 수 있습니다. 이 노드는 이미지를 비디오 처리에 적합한 잠재 공간 형식으로 인코딩합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `vae` | VAE | 예 | - | 이미지를 잠재 공간으로 인코딩하는 데 사용되는 VAE 모델 |
| `width` | INT | 예 | 16부터 MAX_RESOLUTION까지 | 출력 비디오의 너비(픽셀)(기본값: 848, 16으로 나누어져야 함) |
| `height` | INT | 예 | 16부터 MAX_RESOLUTION까지 | 출력 비디오의 높이(픽셀)(기본값: 480, 16으로 나누어져야 함) |
| `length` | INT | 예 | 1부터 MAX_RESOLUTION까지 | 비디오 시퀀스의 프레임 수(기본값: 93, 단계: 4) |
| `batch_size` | INT | 예 | 1부터 4096까지 | 생성할 비디오 시퀀스의 수(기본값: 1) |
| `start_image` | IMAGE | 아니요 | - | 비디오 시퀀스의 선택적 시작 이미지 |
| `end_image` | IMAGE | 아니요 | - | 비디오 시퀀스의 선택적 종료 이미지 |

**참고:** `start_image`와 `end_image`가 모두 제공되지 않을 경우, 노드는 빈 비디오 잠재를 생성합니다. 이미지가 제공될 경우, 해당 이미지는 인코딩되어 비디오 시퀀스의 시작과/또는 끝에 적절한 마스크와 함께 배치됩니다.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `samples` | LATENT | 인코딩된 비디오 시퀀스를 포함한 생성된 비디오 잠재 표현 |
| `noise_mask` | LATENT | 생성 과정 중 잠재의 어떤 부분을 보존해야 하는지 나타내는 마스크 |
