이 노드는 SDXL Refiner 모델 전용으로 설계되었으며, 미적 점수와 치수 정보를 포함해 텍스트 프롬프트를 조건부 정보로 변환함으로써 생성 작업의 조건을 강화하고 최종 정밀도를 높여줍니다. 이는 마치 전문 아트 디렉터와 같아서, 여러분의 창작 의도를 전달할 뿐 아니라 정확한 미적 기준과 구체적인 요구 사항을 작품에 반영합니다.

## SDXL Refiner 소개

SDXL Refiner는 SDXL 기본 모델을 기반으로 이미지의 세부사항과 품질을 향상시키는 특화된 정밀화 모델입니다. 이 과정은 마치 아트 리터처가 작업하는 것과 같습니다:

1. 먼저 기본 모델이 생성한 예비 이미지나 텍스트 설명을 받습니다.
2. 그다음, 정밀한 미적 점수와 치수 파라미터를 통해 정밀화 과정을 안내합니다.
3. 마지막으로 고주파 이미지 세부사항을 처리해 전체 품질을 개선합니다.

Refiner는 두 가지 방식으로 사용할 수 있습니다:

- 기본 모델이 생성한 이미지를 후처리하기 위한 독립적인 정밀화 단계로 활용
- 전문 통합 시스템의 일부로, 생성 과정의 저노이즈 단계에서 처리를 맡기는 방식

## 입력

| 매개변수 이름 | 데이터 타입 | 입력 유형 | 기본값 | 값 범위 | 설명 |
|----------------|-----------|------------|---------------|-------------|-------------|
| `clip` | CLIP | 필수 | - | - | 텍스트 토큰화 및 인코딩에 사용되는 CLIP 모델 인스턴스로, 텍스트를 모델이 이해할 수 있는 형식으로 변환하는 핵심 구성요소 |
| `ascore` | FLOAT | 선택 | 6.0 | 0.0-1000.0 | 생성된 이미지의 시각적 품질과 미학을 제어하며, 마치 작품의 품질 기준을 설정하는 것과 같습니다:<br/>- 높은 점수(7.5-8.5): 더욱 정교하고 디테일이 풍부한 효과 추구<br/>- 중간 점수(6.0-7.0): 균형 잡힌 품질 관리<br/>- 낮은 점수(2.0-3.0): 부정적 프롬프트에 적합 |
| `width` | INT | 필수 | 1024 | 64-16384 | 출력 이미지 너비(픽셀)를 지정하며, 반드시 8의 배수여야 합니다. SDXL은 총 픽셀수가 1024×1024(약 1M 픽셀)에 가까울 때 가장 우수한 성능을 발휘합니다 |
| `height` | INT | 필수 | 1024 | 64-16384 | 출력 이미지 높이(픽셀)를 지정하며, 반드시 8의 배수여야 합니다. SDXL은 총 픽셀수가 1024×1024(약 1M 픽셀)에 가까울 때 가장 우수한 성능을 발휘합니다 |
| `text` | STRING | 필수 | - | - | 텍스트 프롬프트 설명으로, 다중 줄 입력과 동적 프롬프트 문법을 지원합니다. Refiner에서는 텍스트 프롬프트가 원하는 시각적 품질과 디테일 특성을 더 중점적으로 설명해야 합니다 |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `CONDITIONING` | CONDITIONING | 텍스트 의미론, 미적 기준, 치수 정보를 통합한 정밀화된 조건부 출력으로, SDXL Refiner 모델이 이미지를 정밀하게 수정하도록 안내합니다 |

## 참고사항

1. 이 노드는 SDXL Refiner 모델에 특화되어 있으며 일반 CLIPTextEncode 노드와 차이가 있습니다.
2. 미적 점수 7.5를 기본값으로 권장하며, 이는 SDXL 학습에 사용된 표준 설정입니다.
3. 모든 치수 파라미터는 반드시 8의 배수여야 하며, 총 픽셀수가 1024×1024(약 1M 픽셀)에 가까운 것이 권장됩니다.
4. Refiner 모델은 이미지의 세부사항과 품질을 강화하는 데 중점을 두므로, 텍스트 프롬프트는 장면 내용보다 원하는 시각적 효과를 강조해야 합니다.
5. 실제 사용 시 Refiner는 보통 생성 과정의 후반 단계(약 마지막 20% 단계)에서 사용되며, 세부사항 최적화에 집중합니다.
