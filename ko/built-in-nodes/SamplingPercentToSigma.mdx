SamplingPercentToSigma 노드는 모델의 샘플링 매개변수를 사용하여 샘플링 백분율 값을 해당하는 시그마 값으로 변환합니다. 0.0과 1.0 사이의 백분율 값을 입력받아 모델의 노이즈 스케줄에서 적절한 시그마 값으로 매핑하며, 계산된 시그마 값 또는 경계에서의 실제 최대/최소 시그마 값을 반환하는 옵션도 제공됩니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `model` | MODEL | 예 | - | 변환에 사용되는 샘플링 매개변수를 포함한 모델 |
| `sampling_percent` | FLOAT | 예 | 0.0부터 1.0까지 (단계: 0.0001) | 시그마로 변환할 샘플링 백분율 (기본값: 0.0) |
| `return_actual_sigma` | BOOLEAN | 예 | - | 구간 검사에 사용된 값 대신 실제 시그마 값을 반환합니다. 이는 0.0과 1.0에서만 결과에 영향을 줍니다. (기본값: False) |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `sigma_value` | FLOAT | 입력된 샘플링 백분율에 해당하는 변환된 시그마 값 |
