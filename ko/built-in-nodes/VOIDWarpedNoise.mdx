## 개요

VOID 비디오 정밀화 과정의 두 번째 패스를 위해 시간적으로 상관된 노이즈를 생성합니다. 첫 번째 패스의 출력 비디오를 받아 광학 흐름 벡터를 따라 가우시안 노이즈를 왜곡하여, 비디오 내용과 일관되게 움직이는 노이즈를 만듭니다. 이 왜곡된 노이즈는 두 번째 패스의 초기 잠재값으로 사용되어 최종 출력물의 시간적 일관성을 높입니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `optical_flow` | OPTICAL_FLOW | 예 | - | OpticalFlowLoader (RAFT-large)에서 가져온 광학 흐름 모델. |
| `video` | IMAGE | 예 | - | 첫 번째 패스 출력 비디오 프레임 [T, H, W, 3]. |
| `width` | INT | 예 | 16부터 MAX_RESOLUTION까지 (단계 8) | 출력 잠재값의 너비 (기본값: 672). |
| `height` | INT | 예 | 16부터 MAX_RESOLUTION까지 (단계 8) | 출력 잠재값의 높이 (기본값: 384). |
| `length` | INT | 예 | 1부터 MAX_RESOLUTION까지 (단계 1) | 픽셀 프레임 수. 잠재값 t가 짝수로 맞춰지도록 내림 처리됨 (patch_size_t=2 요구사항), 예: 49 → 45 (기본값: 45). |
| `batch_size` | INT | 예 | 1부터 64까지 | 생성할 동일한 노이즈 시퀀스 수 (기본값: 1). |

**`length` 매개변수 참고:** `length` 값은 자동으로 유효한 가장 가까운 값으로 내려서 잠재값 t 차원을 짝수로 만듭니다. 이는 CogVideoX-Fun-V1.5 모델의 `patch_size_t=2` 제약 조건에 필요합니다. 반올림이 발생하면 경고가 기록됩니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `warped_noise` | LATENT | 광학 흐름 왜곡된 가우시안 노이즈를 담은 5D 텐서 (B, C, T, H, W)로, VOID 두 번째 패스의 초기 잠재값으로 바로 사용 가능합니다. |
