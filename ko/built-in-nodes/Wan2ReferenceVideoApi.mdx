> 이 문서는 AI로 생성되었습니다. 오류를 발견하거나 개선할 점이 있다면 언제든지 기여해 주세요! [GitHub에서 편집하기](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/Wan2ReferenceVideoApi/en.md)

이 노드는 제공된 참조 자료를 바탕으로 사람이나 객체가 등장하는 동영상을 생성합니다. Wan 2.7 모델을 사용하여 텍스트 프롬프트로부터 동영상을 만들며, 단일 문자 연기와 다중 문자 간 상호작용을 지원합니다. 생성을 위해서는 최소한 하나의 참조 동영상 또는 이미지를 제공해야 합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `model` | COMBO | 예 | `"wan2.7-r2v"` | 동영상 생성에 사용할 특정 모델입니다. |
| `model.prompt` | STRING | 예 | - | 동영상을 설명하는 프롬프트입니다. 'character1' 및 'character2'와 같은 식별자를 사용해 참조 캐릭터를 지정하세요. |
| `model.negative_prompt` | STRING | 아니요 | - | 생성된 동영상에서 피해야 할 내용을 설명하는 네거티브 프롬프트입니다(기본값: 비어 있음). |
| `model.resolution` | COMBO | 예 | `"720P"`<br />`"1080P"` | 출력 동영상의 해상도입니다. |
| `model.ratio` | COMBO | 예 | `"16:9"`<br />`"9:16"`<br />`"1:1"`<br />`"4:3"`<br />`"3:4"` | 출력 동영상의 종횡비입니다. |
| `model.duration` | INT | 예 | 2~10 | 생성된 동영상의 길이(초)입니다(기본값: 5). |
| `model.reference_videos` | VIDEO | 아니요 | - | 참조 동영상 목록입니다. 최대 3개의 동영상을 추가할 수 있습니다. |
| `model.reference_images` | IMAGE | 아니요 | - | 참조 이미지 목록입니다. 최대 5개의 이미지를 추가할 수 있습니다. |
| `seed` | INT | 아니요 | 0~2147483647 | 생성에 사용할 시드로, 출력의 무작위성을 조절하는 데 도움을 줍니다(기본값: 0). |
| `watermark` | BOOLEAN | 아니요 | - | 결과물에 AI가 생성한 워터마크를 추가할지 여부입니다(기본값: False). 이는 고급 설정입니다. |

**중요한 제약사항:**
*   `model.reference_videos` 또는 `model.reference_images` 입력에 최소 한 개의 참조 동영상 또는 참조 이미지를 제공해야 합니다.
*   참조 동영상과 이미지의 총 합계는 5개를 초과해서는 안 됩니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-----------|-----------|-------------|
| `output` | VIDEO | 생성된 동영상 파일입니다. |
