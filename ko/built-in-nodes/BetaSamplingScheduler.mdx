BetaSamplingScheduler 노드는 베타 스케줄링 알고리즘을 사용하여 샘플링 과정에 필요한 노이즈 레벨(시그마) 시퀀스를 생성합니다. 모델과 구성 매개변수를 입력받아 이미지 생성 중 디노이징 과정을 제어하는 맞춤형 노이즈 스케줄을 생성합니다. 이 스케줄러는 알파와 베타 매개변수를 통해 노이즈 감소 궤적을 세밀하게 조정할 수 있습니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `model` | MODEL | 예 | - | 샘플링에 사용되는 모델로, 모델 샘플링 객체를 제공합니다 |
| `steps` | INT | 예 | 1~10000 | 시그마를 생성할 샘플링 단계 수 (기본값: 20) |
| `alpha` | FLOAT | 예 | 0.0~50.0 | 베타 스케줄러의 알파 매개변수로, 스케줄링 곡선을 제어합니다 (기본값: 0.6) |
| `beta` | FLOAT | 예 | 0.0~50.0 | 베타 스케줄러의 베타 매개변수로, 스케줄링 곡선을 제어합니다 (기본값: 0.6) |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-----------|-----------|-------------|
| `SIGMAS` | SIGMAS | 샘플링 과정에 사용되는 노이즈 레벨(시그마) 시퀀스 |
