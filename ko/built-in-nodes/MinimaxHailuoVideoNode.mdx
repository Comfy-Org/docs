MiniMax Hailuo-02 모델을 사용하여 텍스트 프롬프트로부터 동영상을 생성합니다. 선택적으로 첫 번째 프레임으로 시작 이미지를 제공하면 해당 이미지부터 이어지는 동영상을 만들 수 있습니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `prompt_text` | 문자열 | 예 | - | 동영상 생성을 안내하는 텍스트 프롬프트입니다. |
| `seed` | 정수 | 아니요 | 0~18446744073709551615 | 노이즈 생성에 사용되는 난수 시드입니다(기본값: 0). |
| `first_frame_image` | 이미지 | 아니요 | - | 동영상을 생성할 때 첫 번째 프레임으로 사용할 옵션 이미지입니다. |
| `prompt_optimizer` | 불리언 | 아니요 | - | 필요시 프롬프트를 최적화하여 생성 품질을 높입니다(기본값: True). |
| `duration` | 콤보 | 아니요 | `6`<br />`10` | 출력 동영상의 길이(초 단위, 기본값: 6). |
| `resolution` | 콤보 | 아니요 | `"768P"`<br />`"1080P"` | 동영상 디스플레이의 크기입니다. 1080p는 1920x1080이고, 768p는 1366x768입니다(기본값: "768P"). |

**참고:** MiniMax-Hailuo-02 모델을 1080P 해상도로 사용할 경우, 동영상 길이는 6초로 제한됩니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `output` | 동영상 | 생성된 동영상 파일입니다. |
