Vidu 텍스트 to 비디오 생성 노드는 텍스트 설명을 기반으로 동영상을 생성합니다. Vidu 비디오 생성 모델을 사용하여 텍스트 프롬프트를 지정된 지속 시간, 종횡비 및 시각적 스타일 설정으로 비디오 콘텐츠로 변환합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `model` | COMBO | 예 | `viduq1` | 모델 이름 |
| `prompt` | STRING | 예 | - | 비디오 생성을 위한 텍스트 설명 |
| `duration` | INT | 아니요 | 5-5 | 출력 동영상의 지속 시간(초)(기본값: 5) |
| `seed` | INT | 아니요 | 0-2147483647 | 비디오 생성을 위한 시드(랜덤은 0)(기본값: 0) |
| `aspect_ratio` | COMBO | 아니요 | `16:9`<br />`9:16`<br />`1:1` | 출력 동영상의 종횡비 |
| `resolution` | COMBO | 아니요 | `1080p` | 지원되는 값은 모델 및 지속 시간에 따라 다를 수 있음 |
| `movement_amplitude` | COMBO | 아니요 | `auto`<br />`small`<br />`medium`<br />`large` | 프레임 내 객체의 움직임 진폭 |

**참고:** `prompt` 필드는 필수이며 비워둘 수 없습니다. `duration` 매개변수는 현재 5초로 고정되어 있습니다.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `output` | VIDEO | 텍스트 프롬프트를 기반으로 생성된 동영상 |
