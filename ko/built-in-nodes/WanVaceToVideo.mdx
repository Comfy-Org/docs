WanVaceToVideo 노드는 비디오 생성 모델을 위한 비디오 조건부 데이터를 처리합니다. 이 노드는 양성 및 음성 조건부 입력과 비디오 제어 데이터를 받아 비디오 생성을 위한 잠재 표현을 준비합니다. 노드는 비디오 상향 스케일링, 마스크 처리 및 VAE 인코딩을 통해 비디오 모델에 적합한 조건부 구조를 생성합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `positive` | CONDITIONING | 예 | - | 생성을 안내하는 양성 조건부 입력 |
| `negative` | CONDITIONING | 예 | - | 생성을 안내하는 음성 조건부 입력 |
| `vae` | VAE | 예 | - | 이미지와 비디오 프레임을 인코딩하는 데 사용되는 VAE 모델 |
| `width` | INT | 예 | 16부터 MAX_RESOLUTION까지 | 출력 비디오 너비(픽셀 단위, 기본값: 832, 단계: 16) |
| `height` | INT | 예 | 16부터 MAX_RESOLUTION까지 | 출력 비디오 높이(픽셀 단위, 기본값: 480, 단계: 16) |
| `length` | INT | 예 | 1부터 MAX_RESOLUTION까지 | 비디오의 프레임 수(기본값: 81, 단계: 4) |
| `batch_size` | INT | 예 | 1부터 4096까지 | 동시에 생성할 비디오 수(기본값: 1) |
| `strength` | FLOAT | 예 | 0.0부터 1000.0까지 | 비디오 조건부 제어 강도(기본값: 1.0, 단계: 0.01) |
| `control_video` | IMAGE | 아니요 | - | 제어 조건부를 위한 선택적 입력 비디오입니다. 제공되지 않으면 중립적인 회색 비디오가 자동으로 생성됩니다. |
| `control_masks` | MASK | 아니요 | - | 비디오의 특정 부분만 수정하도록 제어하는 선택적 마스크입니다. 제공되지 않으면 전체 흰색 마스크가 사용됩니다. |
| `reference_image` | IMAGE | 아니요 | - | 추가 조건부를 위한 선택적 참조 이미지입니다. 제공되면 인코딩되어 잠재 시퀀스 앞에 추가됩니다. |

**참고:** `control_video`가 제공되면 지정된 너비와 높이에 맞게 상향 스케일링됩니다. `control_masks`가 제공되면 그 크기가 제어 비디오의 크기와 일치해야 합니다. `reference_image`는 VAE를 통해 인코딩되어 제공될 경우 잠재 시퀀스 앞에 추가됩니다. `length` 매개변수는 프레임 수를 결정하며, 잠재 길이는 `((length - 1) // 4) + 1`로 계산됩니다.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `positive` | CONDITIONING | 비디오 제어 데이터(vace_frames, vace_mask, vace_strength)가 적용된 양성 조건부 |
| `negative` | CONDITIONING | 비디오 제어 데이터(vace_frames, vace_mask, vace_strength)가 적용된 음성 조건부 |
| `latent` | LATENT | 비디오 생성을 위해 준비된 빈 잠재 텐서로, 형태는 [batch_size, 16, latent_length, height/8, width/8] |
| `trim_latent` | INT | 참조 이미지를 사용할 경우 트리밍할 잠재 프레임 수(참조 이미지가 제공되지 않으면 0) |
