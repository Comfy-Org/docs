> 이 문서는 AI로 생성되었습니다. 오류를 발견하거나 개선할 점이 있으면 언제든지 기여해 주세요! [GitHub에서 편집하기](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/StabilityStableImageSD_3_5Node/en.md)

이 노드는 Stability AI의 Stable Diffusion 3.5 모델을 사용하여 동기식으로 이미지를 생성합니다. 텍스트 프롬프트를 기반으로 이미지를 생성하며, 입력으로 기존 이미지를 제공하면 이를 수정할 수도 있습니다. 이 노드는 다양한 종횡비와 스타일 사전 설정을 지원하여 출력물을 맞춤화할 수 있습니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `prompt` | 문자열 | 예 | - | 출력 이미지에서 보고 싶은 내용입니다. 요소, 색상, 주제를 명확히 정의하는 강력하고 구체적인 프롬프트를 사용하면 더 나은 결과를 얻을 수 있습니다. (기본값: 빈 문자열) |
| `model` | 콤보 | 예 | `sd3.5-large`<br />`sd3.5-large-turbo`<br />`sd3.5-medium` | 생성에 사용할 Stable Diffusion 3.5 모델입니다. |
| `aspect_ratio` | 콤보 | 예 | `16:9`<br />`1:1`<br />`21:9`<br />`2:3`<br />`3:2`<br />`4:5`<br />`5:4`<br />`9:16`<br />`9:21` | 생성된 이미지의 종횡비입니다. (기본값: 1:1) |
| `style_preset` | 콤보 | 아니요 | `3d-model`<br />`analog-film`<br />`anime`<br />`cinematic`<br />`comic-book`<br />`digital-art`<br />`enhance`<br />`fantasy-art`<br />`isometric`<br />`line-art`<br />`low-poly`<br />`modeling-compound`<br />`neon-punk`<br />`origami`<br />`photographic`<br />`pixel-art`<br />`tile-texture`<br />`None` | 생성된 이미지의 원하는 스타일입니다. 스타일 사전 설정 없음을 선택하려면 "None"을 선택하세요. |
| `cfg_scale` | 부동소수점 | 예 | 1.0부터 10.0까지 | 확산 과정이 프롬프트 텍스트를 얼마나 엄격하게 따르는지(높은 값일수록 이미지가 프롬프트에 더 가깝게 유지됩니다). (기본값: 4.0) |
| `seed` | 정수 | 예 | 0부터 4294967294까지 | 노이즈 생성에 사용되는 난수 시드입니다. (기본값: 0) |
| `image` | 이미지 | 아니요 | - | 이미지 대 이미지 생성을 위한 선택적 입력 이미지입니다. 제공되면 노드는 이미지 대 이미지 모드로 전환되며, `aspect_ratio` 매개변수는 무시됩니다. |
| `negative_prompt` | 문자열 | 아니요 | - | 출력 이미지에서 보고 싶지 않은 키워드입니다. 이는 고급 기능입니다. (기본값: 빈 문자열) |
| `image_denoise` | 부동소수점 | 아니요 | 0.0부터 1.0까지 | 입력 이미지의 노이즈 제거 정도; 0.0은 입력 이미지와 동일한 이미지를 생성하며, 1.0은 이미지를 전혀 제공하지 않은 것과 같습니다. (기본값: 0.5) 이 매개변수는 `image`가 제공된 경우에만 사용됩니다. |

**참고:** `image`가 제공되면 노드는 이미지 대 이미지 생성 모드로 전환되며, `aspect_ratio` 매개변수는 자동으로 입력 이미지에서 결정됩니다. `image`가 제공되지 않으면 `image_denoise` 매개변수는 무시됩니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `image` | 이미지 | 생성된 또는 수정된 이미지입니다. |
