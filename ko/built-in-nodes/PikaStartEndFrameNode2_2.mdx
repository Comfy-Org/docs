> 이 문서는 AI를 통해 생성되었습니다. 오류를 발견하거나 개선할 점이 있다면 언제든지 기여해 주세요! [GitHub에서 편집하기](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/PikaStartEndFrameNode2_2/en.md)

PikaFrames v2.2 노드는 시작 프레임과 끝 프레임을 결합하여 동영상을 생성합니다. 시작점과 끝점을 정의하기 위해 두 장의 이미지를 업로드하면 AI가 그 사이를 부드럽게 전환하여 완성된 동영상을 만들어냅니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `image_start` | IMAGE | 예 | - | 결합할 첫 번째 이미지입니다. |
| `image_end` | IMAGE | 예 | - | 결합할 마지막 이미지입니다. |
| `prompt_text` | STRING | 예 | - | 원하는 동영상 내용을 설명하는 텍스트 프롬프트입니다. |
| `negative_prompt` | STRING | 예 | - | 동영상에서 피해야 할 내용을 설명하는 텍스트입니다. |
| `seed` | INT | 예 | - | 생성의 일관성을 유지하기 위한 난수 시드 값입니다. |
| `resolution` | STRING | 예 | - | 출력 동영상의 해상도입니다. |
| `duration` | INT | 예 | - | 생성된 동영상의 지속시간입니다. |

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `output` | VIDEO | 시작 프레임과 끝 프레임을 AI 전환으로 결합한 생성된 동영상입니다. |
