CLIPSubtract 노드는 두 개의 CLIP 모델 간에 뺄셈 연산을 수행합니다. 첫 번째 CLIP 모델을 기본 모델로 취하고, 두 번째 CLIP 모델의 핵심 패치를 빼내며, 선택적으로 곱셈자를 사용해 뺄셈 강도를 조절할 수 있습니다. 이를 통해 한 모델에서 특정 특성을 다른 모델을 이용해 제거함으로써 세밀하게 모델을 혼합할 수 있습니다.

## 입력

| 매개변수 | 데이터 타입 | 입력 유형 | 기본값 | 범위 | 설명 |
|-----------|-----------|------------|---------|-------|-------------|
| `clip1` | CLIP | 필수 | - | - | 수정될 기본 CLIP 모델 |
| `clip2` | CLIP | 필수 | - | - | 기본 모델에서 빼낼 핵심 패치를 가진 CLIP 모델 |
| `multiplier` | FLOAT | 필수 | 1.0 | -10.0부터 10.0까지, 단계 0.01 | 뺄셈 연산의 강도를 조절합니다 |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `CLIP` | CLIP | 뺄셈 연산 후 생성된 CLIP 모델 |
