이 노드는 LoRA(Low-Rank Adaptation)를 모델에 적용하여 동작을 수정하지만, 영향을 미치는 것은 모델 구성 요소 자체에만 국한됩니다. 지정된 LoRA 파일을 로드하고 주어진 강도만큼 모델의 가중치를 조정하며, CLIP 텍스트 인코더와 같은 다른 구성 요소는 그대로 유지합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `model` | MODEL | 예 | - | LoRA 조정이 적용될 기본 모델입니다. |
| `lora_name` | STRING | 예 | (사용 가능한 LoRA 파일 목록) | 로드하고 적용할 LoRA 파일의 이름입니다. 옵션은 `loras` 디렉토리의 파일들로부터 채워집니다. |
| `strength_model` | FLOAT | 예 | -100.0부터 100.0까지 | LoRA가 모델의 가중치에 미치는 영향의 강도입니다. 양수 값은 LoRA를 적용하고, 음수 값은 역방향으로 적용하며, 0값은 아무런 효과를 주지 않습니다(기본값: 1.0). |

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `model` | MODEL | LoRA 조정이 가중치에 적용된 수정된 모델입니다. |
