## 개요

이 노드는 샘플링 과정 후반부에 모델의 출력을 여러 번 시프트된 패치 격자 위치에 걸쳐 평균화하여 HiDream-O1 모델이 생성한 이미지의 눈에 띄는 이음매를 줄여줍니다. 모델을 약간 다른 이미지 정렬로 여러 번 실행한 뒤 결과를 블렌딩함으로써 패치 경계에 나타날 수 있는 격자 형태의 아티팩트를 상쇄하는 데 도움을 줍니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `model` | MODEL | 예 | - | 이음매 부드럽게하기를 적용할 HiDream-O1 모델입니다. |
| `start_percent` | FLOAT | 예 | 0.0부터 1.0까지 (단계: 0.01) | 부드럽게하기 효과가 켜지는 샘플링 진행률(0=시작, 1=종료) (기본값: 0.8). |
| `end_percent` | FLOAT | 예 | 0.0부터 1.0까지 (단계: 0.01) | 부드럽게하기 효과가 꺼지는 샘플링 진행률 (기본값: 1.0). |
| `pattern` | COMBO | 예 | `"single_shift"`<br />`"symmetric"` | 시프트된 격자 위치의 레이아웃입니다. `single_shift`: 자연 패치 격자에서 한 번 실행하고 나머지는 오프셋으로 배치합니다. `symmetric`: 모든 실행이 격자를 벗어나며, 원점을 중심으로 시프트가 분할됩니다 (기본값: `"single_shift"`). |
| `passes` | COMBO | 예 | `"2"`<br />`"4"`<br />`"ramp_2_4"`<br />`"ramp_2_4_8"` | 각 게이트 단계당 실행 횟수(모델 실행 수)입니다. `2` 또는 `4`는 고정된 수입니다. `ramp_2_4`와 `ramp_2_4_8`는 샘플링이 끝날 때쯤 실행 횟수가 증가하며, 이음매가 가장 두드러지는 부분에서 더 많은 부드러움을 제공합니다 (기본값: `"2"`). |
| `blend` | COMBO | 예 | `"average"`<br />`"window"`<br />`"median"` | 각 실행의 결과를 결합하는 방법입니다. `average`: 모든 실행의 균등 가중 평균. `window`: 한스 윈도우를 사용해 각 실행의 중앙부에 더 큰 가중치를 부여하여 경계 아티팩트를 줄입니다. `median`: 픽셀별 중앙값을 취하며, 래핑으로 인한 이상치 실행을 거부할 수 있습니다 (기본값: `"average"`). |
| `strength` | FLOAT | 예 | 0.0부터 1.0까지 (단계: 0.01) | 원래 모델 출력(0.0)과 완전히 부드러워진 결과(1.0) 사이의 보간을 제어합니다 (기본값: 1.0). |

**매개변수 제약 사항 참고:**
- `strength`가 0.0 이하이거나, `end_percent`가 `start_percent` 이하인 경우 부드럽게하기 효과가 적용되지 않습니다.
- `passes` 매개변수의 램프 옵션(`ramp_2_4`, `ramp_2_4_8`)은 `start_percent`와 `end_percent`가 범위를 정의할 때만 의미가 있으며, 샘플링이 해당 범위를 지나면서 실행 횟수가 증가합니다.
