> 이 문서는 AI로 생성되었습니다. 오류를 발견하거나 개선할 점이 있다면 언제든지 기여해 주세요! [GitHub에서 편집하기](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/RecraftImageToImageNode/en.md)

이 노드는 텍스트 프롬프트와 강도 매개변수를 기반으로 기존 이미지를 수정합니다. Recraft API를 사용하여 제공된 설명에 따라 입력 이미지를 변환하며, 강도 설정에 따라 원본 이미지와 어느 정도 유사성을 유지합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `image` | IMAGE | 예 | - | 수정할 입력 이미지 |
| `prompt` | STRING | 예 | - | 이미지 생성을 위한 프롬프트 (기본값: "", 최대 길이: 1000자) |
| `n` | INT | 예 | 1-6 | 생성할 이미지 수 (기본값: 1) |
| `strength` | FLOAT | 예 | 0.0-1.0 | 원본 이미지와의 차이를 정의하며, [0, 1] 범위 내에 있어야 합니다. 0은 거의 동일한 것을 의미하고, 1은 형편없는 유사성을 의미합니다 (기본값: 0.5) |
| `seed` | INT | 예 | 0-18446744073709551615 | 노드가 다시 실행될지 여부를 결정하는 시드. 실제 결과는 시드와 관계없이 비결정적입니다 (기본값: 0) |
| `recraft_style` | STYLEV3 | 선택 | - | 이미지 생성을 위한 옵션 스타일 선택. 제공되지 않으면 기본적으로 `realistic_image`로 설정됩니다 |
| `negative_prompt` | STRING | 선택 | - | 이미지에서 원하지 않는 요소에 대한 선택적 텍스트 설명 (기본값: "") |
| `recraft_controls` | CONTROLS | 선택 | - | Recraft Controls 노드를 통해 생성 과정에 추가적인 제어 옵션 제공 |

**참고:** `seed` 매개변수는 노드의 재실행을 유발할 뿐, 결정적인 결과를 보장하지 않습니다. 강도 매개변수는 내부적으로 소수점 둘째 자리까지 반올림됩니다. 프롬프트는 유효성 검사를 거쳐 1000자를 초과해서는 안 됩니다. `recraft_style`이 제공되지 않으면 노드는 기본적으로 `realistic_image` 스타일을 사용합니다. Infinite Style Library의 `style_id`를 사용할 경우, 벡터 아트 스타일이 아닌지 확인하세요. 그렇지 않으면 노드가 이미지 대신 SVG 데이터를 받게 되어 오류가 발생할 수 있습니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `image` | IMAGE | 입력 이미지와 프롬프트를 기반으로 생성된 이미지 |
