오디오 이퀄라이저(3밴드) 노드를 사용하면 오디오 파형의 베이스, 미드, 트레블 주파수를 조정할 수 있습니다. 이 노드는 세 가지 별도의 필터를 적용합니다: 베이스용 로우 셰프, 미드용 피킹 필터, 트레블용 하이 셰프입니다. 각 밴드는 이득, 주파수, 대역폭 설정을 통해 독립적으로 제어할 수 있습니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `audio` | AUDIO | 예 | - | 파형과 샘플레이트를 포함한 입력 오디오 데이터입니다. |
| `low_gain_dB` | FLOAT | 아니요 | -24.0부터 24.0까지 | 저주파(베이스) 이득입니다. 양수 값은 부스트하고 음수 값은 커트합니다. (기본값: 0.0) |
| `low_freq` | INT | 아니요 | 20부터 500까지 | 로우 셰프 필터의 커트오프 주파수(헤르츠, Hz). (기본값: 100) |
| `mid_gain_dB` | FLOAT | 아니요 | -24.0부터 24.0까지 | 미드 주파수 이득입니다. 양수 값은 부스트하고 음수 값은 커트합니다. (기본값: 0.0) |
| `mid_freq` | INT | 아니요 | 200부터 4000까지 | 미드 피킹 필터의 중심 주파수(헤르츠, Hz). (기본값: 1000) |
| `mid_q` | FLOAT | 아니요 | 0.1부터 10.0까지 | 미드 피킹 필터의 Q 요소(대역폭). 낮은 값은 더 넓은 대역을 만들고 높은 값은 더 좁은 대역을 만듭니다. (기본값: 0.707) |
| `high_gain_dB` | FLOAT | 아니요 | -24.0부터 24.0까지 | 고주파(트레블) 이득입니다. 양수 값은 부스트하고 음수 값은 커트합니다. (기본값: 0.0) |
| `high_freq` | INT | 아니요 | 1000부터 15000까지 | 하이 셰프 필터의 커트오프 주파수(헤르츠, Hz). (기본값: 5000) |

**참고:** `low_gain_dB`, `mid_gain_dB`, `high_gain_dB` 매개변수는 해당 값이 0이 아닐 때만 적용됩니다. 이득을 0.0으로 설정하면 해당 필터 단계는 건너뜁니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `audio` | AUDIO | 이퀄라이징이 적용된 처리된 오디오 데이터로, 수정된 파형과 원래 샘플레이트를 포함합니다. |
