> 이 문서는 AI로 생성되었습니다. 오류를 발견하거나 개선할 점이 있다면 언제든지 기여해 주세요! [GitHub에서 편집하기](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/Tencent3DTextureEditNode/en.md)

이 노드는 Tencent Hunyuan3D API를 사용하여 3D 모델의 텍스처를 편집합니다. 3D 모델과 원하는 변경 사항에 대한 텍스트 설명을 제공하면, 노드는 귀하의 프롬프트에 따라 텍스처가 다시 그려진 새로운 버전의 모델을 반환합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `model_3d` | FILE3D | 예 | FBX, 임의 | FBX 형식의 3D 모델. 모델의 면 수는 100,000개 미만이어야 합니다. |
| `prompt` | STRING | 예 | | 텍스처 편집을 설명합니다. 최대 1024 UTF-8 문자를 지원합니다. |
| `seed` | INT | 아니요 | 0부터 2,147,483,647까지 | 시드는 노드가 다시 실행될지 여부를 제어합니다. 시드와 관계없이 결과는 비결정적입니다. (기본값: 0) |

**참고:** `model_3d` 입력은 반드시 FBX 형식의 파일이어야 합니다. 다른 3D 파일 형식은 이 노드에서 지원되지 않습니다.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `GLB` | FILE3D | 처리된 3D 모델을 GLB 형식으로 출력합니다. |
| `OBJ` | FILE3D | 처리된 3D 모델을 OBJ 형식으로 출력합니다. |
| `texture_image` | IMAGE | 3D 모델용 새로 생성된 텍스처 이미지입니다. |
