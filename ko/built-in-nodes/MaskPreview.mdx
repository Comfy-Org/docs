MaskPreview 노드는 마스크 데이터를 미리보기 이미지로 ComfyUI 출력 디렉토리에 저장하여 워크플로 실행 중에 마스크 데이터를 시각적으로 확인할 수 있도록 합니다. 입력 마스크를 이미지 표시에 적합한 형식으로 변환하고, 구성 가능한 파일명 접두사와 함께 저장합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `mask` | MASK | 예 | - | 미리보기 및 이미지로 저장할 마스크 데이터 |
| `filename_prefix` | STRING | 아니요 | - | 출력 파일명의 접두사 (기본값: "ComfyUI") |
| `prompt` | PROMPT | 아니요 | - | 메타데이터용 프롬프트 정보 (자동 제공) |
| `extra_pnginfo` | EXTRA_PNGINFO | 아니요 | - | 메타데이터용 추가 PNG 정보 (자동 제공) |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `ui` | DICT | UI에 표시할 미리보기 이미지 정보와 메타데이터를 포함합니다 |
