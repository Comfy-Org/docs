Vidu Q3 텍스트-to-비디오 생성 노드는 텍스트 설명을 바탕으로 비디오를 생성합니다. Vidu Q3 Pro 또는 Q3 Turbo 모델을 사용하여 프롬프트에 따라 비디오 콘텐츠를 생성하며, 비디오의 길이, 해상도, 종횡비 및 오디오 포함 여부를 제어할 수 있습니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `model` | COMBO | 예 | `"viduq3-pro"`<br />`"viduq3-turbo"` | 비디오 생성에 사용할 모델입니다. 모델을 선택하면 종횡비, 해상도, 지속 시간 및 오디오와 관련된 추가 구성 매개변수가 표시됩니다. |
| `model.aspect_ratio` | COMBO | 예* | `"16:9"`<br />`"9:16"`<br />`"3:4"`<br />`"4:3"`<br />`"1:1"` | 출력 비디오의 종횡비입니다. 이 매개변수는 `model`을 선택하면 표시됩니다. |
| `model.resolution` | COMBO | 예* | `"720p"`<br />`"1080p"` | 출력 비디오의 해상도입니다. 이 매개변수는 `model`을 선택하면 표시됩니다. |
| `model.duration` | INT | 예* | 1~16 | 출력 비디오의 지속 시간(초)(기본값: 5). 이 매개변수는 `model`을 선택하면 표시됩니다. |
| `model.audio` | BOOLEAN | 예* | True/False | 활성화 시 대사와 효과음을 포함한 사운드가 있는 비디오를 출력합니다(기본값: False). 이 매개변수는 `model`을 선택하면 표시됩니다. |
| `prompt` | STRING | 예 | N/A | 비디오 생성을 위한 텍스트 설명으로, 최대 길이는 2000자입니다. |
| `seed` | INT | 아니요 | 0~2147483647 | 생성의 무작위성을 제어하는 시드 값입니다(기본값: 1). |

*참고: `aspect_ratio`, `resolution`, `duration`, `audio` 매개변수는 `model`을 선택하면 해당 구성의 일부로서 필수적으로 설정됩니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `video` | VIDEO | 생성된 비디오 파일입니다. |
