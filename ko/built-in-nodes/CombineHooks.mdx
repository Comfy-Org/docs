Combine Hooks [2] 노드는 두 개의 후크 그룹을 하나의 통합된 후크 그룹으로 병합합니다. 두 개의 선택적 후크 입력을 받아 ComfyUI의 후크 조합 기능을 사용하여 이를 결합합니다. 이를 통해 여러 후크 구성들을 하나로 통합하여 처리 과정을 간소화할 수 있습니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `hooks_A` | HOOKS | 선택 | - | 결합할 첫 번째 후크 그룹 |
| `hooks_B` | HOOKS | 선택 | - | 결합할 두 번째 후크 그룹 |

**참고:** 두 입력 모두 선택적이지만, 노드가 작동하려면 적어도 하나의 후크 그룹은 제공되어야 합니다. 만약 단 한 개의 후크 그룹만 제공된다면, 그대로 반환됩니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `hooks` | HOOKS | 두 입력 그룹의 모든 후크를 포함한 통합된 후크 그룹 |
