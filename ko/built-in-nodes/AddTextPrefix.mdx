텍스트 접두사 추가 노드는 각 입력 텍스트의 앞부분에 지정된 문자열을 추가하여 텍스트를 수정합니다. 이 노드는 텍스트와 접두사를 입력으로 받아 결합된 결과를 반환합니다.

**참고:** 이 노드는 더 이상 사용되지 않으며, 텍스트 연결 노드로 대체되었습니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `text` | STRING | 예 | | 접두사를 추가할 텍스트입니다. |
| `prefix` | STRING | 아니요 | | 텍스트의 앞부분에 추가할 문자열입니다(기본값: ""). |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `text` | STRING | 접두사를 앞에 추가한 결과 텍스트입니다. |
