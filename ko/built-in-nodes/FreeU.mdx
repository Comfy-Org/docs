FreeU 노드는 모델의 출력 블록에 주파수 도메인 수정을 적용하여 이미지 생성 품질을 향상시킵니다. 이 노드는 서로 다른 채널 그룹을 스케일링하고 특정 피처 맵에 푸리에 필터링을 적용함으로써 생성 과정 중 모델의 동작을 세밀하게 조정할 수 있게 해줍니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `model` | MODEL | 예 | - | FreeU 수정을 적용할 모델 |
| `b1` | FLOAT | 예 | 0.0 - 10.0 | model_channels × 4 피처에 대한 백본 스케일링 요소 (기본값: 1.1) |
| `b2` | FLOAT | 예 | 0.0 - 10.0 | model_channels × 2 피처에 대한 백본 스케일링 요소 (기본값: 1.2) |
| `s1` | FLOAT | 예 | 0.0 - 10.0 | model_channels × 4 피처에 대한 스킵 커넥션 스케일링 요소 (기본값: 0.9) |
| `s2` | FLOAT | 예 | 0.0 - 10.0 | model_channels × 2 피처에 대한 스킵 커넥션 스케일링 요소 (기본값: 0.2) |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `model` | MODEL | FreeU 패치가 적용된 수정된 모델 |
