Kling 단일 이미지 비디오 효과 노드는 단일 참조 이미지를 기반으로 다양한 특수 효과를 적용한 동영상을 생성합니다. 다양한 시각적 효과와 장면을 적용하여 정적인 이미지를 역동적인 동영상 콘텐츠로 변환합니다. 이 노드는 원하는 시각적 결과를 얻기 위해 다양한 효과 장면, 모델 옵션 및 동영상 길이를 지원합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `image` | IMAGE | 예 | - | 참조 이미지. URL 또는 Base64 인코딩된 문자열(데이터:image 접두사 없음). 파일 크기는 10MB를 초과하지 않으며, 해상도는 300x300px 이상이고, 종횡비는 1:2.5에서 2.5:1 사이여야 합니다. |
| `effect_scene` | COMBO | 예 | `"dizzydizzy"`<br />`"bloombloom"`<br />`"neon"`<br />`"cartoon"`<br />`"sketch"`<br />`"oil"`<br />`"watercolor"`<br />`"3d"` | 동영상 생성에 적용할 특수 효과 장면의 유형입니다. 일부 효과는 가격이 다를 수 있습니다. |
| `model_name` | COMBO | 예 | `"kling-v1-5"`<br />`"kling-v1-6"` | 동영상 효과 생성에 사용할 특정 모델 버전입니다. |
| `duration` | COMBO | 예 | `"5"`<br />`"10"` | 생성된 동영상의 길이(초 단위). |

**참고:** `effect_scene` 매개변수는 노드의 가격에 영향을 줍니다. `dizzydizzy`와 `bloombloom` 효과는 한 번 생성당 0.49 USD이며, 다른 모든 효과는 한 번 생성당 0.28 USD입니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `output` | VIDEO | 적용된 효과가 포함된 생성된 동영상 |
| `video_id` | STRING | 생성된 동영상의 고유 식별자 |
| `duration` | STRING | 생성된 동영상의 길이 |
