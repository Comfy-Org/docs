## 개요

HappyHorse 모델을 사용하여 텍스트 지침이나 참조 이미지를 통해 동영상을 편집하세요. 출력 길이는 3~15초이며 입력 동영상과 일치합니다. 15초 이상의 입력은 잘려집니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `model` | DICT | 예 | 아래 참조 | 모델 선택, 프롬프트, 해상도, 종횡비 및 선택적 참조 이미지를 포함한 모델 구성. |
| `video` | VIDEO | 예 | - | 편집할 동영상. |
| `seed` | INT | 예 | 0부터 2147483647까지 | 생성에 사용할 시드 (기본값: 0). |
| `watermark` | BOOLEAN | 아니요 | True / False | 결과에 AI가 생성한 워터마크를 추가할지 여부 (기본값: False). |

### `model` 매개변수 세부사항

`model` 매개변수는 다음과 같은 필드를 가진 딕셔너리입니다:

| 필드 | 데이터 타입 | 필수 | 범위 | 설명 |
|-------|-----------|------|-------|-------------|
| `model` | STRING | 예 | `"happyhorse-1.0-video-edit"` | 사용할 HappyHorse 동영상 편집 모델. |
| `prompt` | STRING | 예 | - | 편집 지침 또는 스타일 전송 요구사항. 최소 1자 이상이어야 합니다. |
| `resolution` | STRING | 예 | `"720P"`<br />`"1080P"` | 출력 해상도. |
| `ratio` | STRING | 예 | `"16:9"`<br />`"9:16"`<br />`"1:1"`<br />`"4:3"`<br />`"3:4"` | 종횡비. 변경하지 않으면 입력 동영상의 종횡비를 근사하게 유지합니다. |
| `reference_images` | DICT | 아니요 | 이미지 1~5개 | 편집을 안내하는 선택적 참조 이미지 (image1, image2, image3, image4, image5). |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `video` | VIDEO | 편집된 동영상 출력. |
