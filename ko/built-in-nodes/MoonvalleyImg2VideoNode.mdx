Moonvalley Marey 이미지 to 비디오 노드는 Moonvalley API를 사용하여 참조 이미지를 비디오로 변환합니다. 입력 이미지와 텍스트 프롬프트를 받아 지정된 해상도, 품질 설정 및 창의적 제어를 통해 비디오를 생성합니다. 이 노드는 이미지 업로드부터 비디오 생성 및 다운로드까지 전 과정을 처리합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `image` | IMAGE | 예 | - | 비디오 생성에 사용되는 참조 이미지 |
| `prompt` | STRING | 예 | - | 비디오 생성을 위한 텍스트 설명 (다중 줄 입력) |
| `negative_prompt` | STRING | 아니요 | - | 원치 않는 요소를 제외하기 위한 부정적 프롬프트 텍스트 (기본값: 광범위한 부정적 프롬프트 목록) |
| `resolution` | COMBO | 아니요 | "16:9 (1920 x 1080)"<br />"9:16 (1080 x 1920)"<br />"1:1 (1152 x 1152)"<br />"4:3 (1536 x 1152)"<br />"3:4 (1152 x 1536)" | 출력 비디오의 해상도 (기본값: "16:9 (1920 x 1080)") |
| `prompt_adherence` | FLOAT | 아니요 | 1.0 - 20.0 | 생성 제어를 위한 가이드라인 스케일 (기본값: 4.5, 단계: 1.0) |
| `seed` | INT | 아니요 | 0 - 4294967295 | 난수 시드 값 (기본값: 9, 생성 후 제어 활성화) |
| `steps` | INT | 아니요 | 1 - 100 | 노이즈 제거 단계 수 (기본값: 33, 단계: 1) |

**제약사항:**

- 입력 이미지는 가로세로 각각 300x300 픽셀부터 최대 허용 크기까지여야 합니다.
- 프롬프트 및 부정적 프롬프트 텍스트 길이는 Moonvalley Marey의 최대 프롬프트 길이로 제한됩니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `output` | VIDEO | 생성된 비디오 출력 |
