![ComfyUI 네이티브 Luma Concepts 노드](/images/built-in-nodes/api_nodes/luma/luma-concepts.jpg)

Luma Concepts 노드는 미리 정의된 카메라 개념을 Luma 생성 과정에 적용할 수 있게 해주며, 복잡한 프롬프트 설명 없이도 카메라 앵글과 원근감을 정밀하게 제어할 수 있습니다.

## 노드 기능

이 노드는 Luma 생성 노드를 위한 도우미 도구로, 사용자가 미리 정의된 카메라 개념을 선택해 적용할 수 있도록 합니다. 이러한 개념에는 다양한 촬영 앵글(예: 위에서 내려다보기 또는 로우 앵글), 카메라 거리(예: 클로즈업 또는 롱 샷), 움직임 스타일(예: 푸시인 또는 팔로우) 등이 포함됩니다. 이를 통해 생성된 결과물에서 카메라 효과를 직관적으로 제어할 수 있어 창작 워크플로우를 간소화합니다.

## 매개변수

### 기본 매개변수

| 매개변수 | 유형   | 설명                                                |
| --------- | ------ | ---------------------------------------------------------- |
| concept1  | 선택   | 첫 번째 카메라 개념 선택, 다양한 사전 설정 및 '없음' 포함 |
| concept2  | 선택   | 두 번째 카메라 개념 선택, 다양한 사전 설정 및 '없음' 포함 |
| concept3  | 선택   | 세 번째 카메라 개념 선택, 다양한 사전 설정 및 '없음' 포함 |
| concept4  | 선택   | 네 번째 카메라 개념 선택, 다양한 사전 설정 및 '없음' 포함 |

### 옵션 매개변수

| 매개변수     | 유형          | 설명                                               |
| ------------ | ------------- | --------------------------------------------------------- |
| luma_concepts| LUMA_CONCEPTS | 선택한 개념들과 병합할 수 있는 옵션 카메라 개념들   |

### 출력

| 출력        | 유형          | 설명                                    |
| ------------- | ------------- | ---------------------------------------------- |
| luma_concepts | LUMA_CONCEPT  | 선택한 모든 개념을 포함한 결합된 객체|

## 작동 방식

Luma Concepts 노드는 다음과 같은 다양한 미리 정의된 카메라 개념을 제공합니다:
- 카메라 거리 (클로즈업, 미디엄 샷, 롱 샷)
- 시야각 (눈높이, 위에서 내려다보기, 로우 앵글)
- 움직임 유형 (푸시인, 팔로우, 오비트)
- 특수 효과 (핸드헬드, 안정화, 플로팅)

사용자는 최대 4가지 개념을 선택해 함께 사용할 수 있습니다. 이 노드는 선택한 카메라 개념들을 포함한 객체를 생성하며, 이를 Luma 생성 노드로 전달합니다. 생성 과정에서 Luma AI는 이 카메라 개념들을 활용해 출력물의 시점과 구성을 조정하여, 선택한 사진적 효과를 반영하도록 합니다.

여러 카메라 개념을 조합함으로써 사용자는 상세한 프롬프트 설명을 작성하지 않고도 복잡한 카메라 가이드라인을 만들 수 있습니다. 특정 카메라 앵글이나 구도가 필요할 때 특히 유용합니다.

## 소스 코드

[노드 소스 코드 (2025-05-03 업데이트)]

```python

class LumaConceptsNode(ComfyNodeABC):
    """
    Luma 텍스트 동영상 및 Luma 이미지 동영상 노드와 함께 사용하기 위한 하나 이상의 카메라 개념을 보유합니다.
    """

    RETURN_TYPES = (LumaIO.LUMA_CONCEPTS,)
    RETURN_NAMES = ("luma_concepts",)
    DESCRIPTION = cleandoc(__doc__ or "")  # None 값 처리
    FUNCTION = "create_concepts"
    CATEGORY = "api node/image/Luma"

    @classmethod
    def INPUT_TYPES(s):
        return {
            "required": {
                "concept1": (get_luma_concepts(include_none=True),),
                "concept2": (get_luma_concepts(include_none=True),),
                "concept3": (get_luma_concepts(include_none=True),),
                "concept4": (get_luma_concepts(include_none=True),),
            },
            "optional": {
                "luma_concepts": (
                    LumaIO.LUMA_CONCEPTS,
                    {
                        "tooltip": "여기에 선택한 개념들에 추가할 수 있는 옵션 카메라 개념들."
                    },
                ),
            },
        }

    def create_concepts(
        self,
        concept1: str,
        concept2: str,
        concept3: str,
        concept4: str,
        luma_concepts: LumaConceptChain = None,
    ):
        chain = LumaConceptChain(str_list=[concept1, concept2, concept3, concept4])
        if luma_concepts is not None:
            chain = luma_concepts.clone_and_merge(chain)
        return (chain,)


```
