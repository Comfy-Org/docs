Luma 이미지에서 비디오로 노드는 Luma AI의 기술을 활용해 정적 이미지를 부드럽고 역동적인 비디오로 변환하여 이미지를 생생하게 만들어줍니다.

## 노드 기능

이 노드는 Luma AI의 이미지에서 비디오로 변환 API와 연결되어 사용자가 입력 이미지로부터 동적 비디오를 생성할 수 있도록 합니다. 노드는 이미지의 내용을 이해하고 원래의 시각적 스타일을 유지하면서 자연스럽고 일관된 움직임을 생성합니다. 텍스트 프롬프트와 결합하면 사용자는 비디오의 동적 효과를 정밀하게 제어할 수 있습니다.

## 매개변수

### 기본 매개변수

| 매개변수 | 유형 | 기본값 | 설명                                             |
| ---------- | ---- | ------- | -------------------------------------------------- |
| 프롬프트     | 문자열 | ""      | 비디오의 움직임과 내용을 설명하는 텍스트 프롬프트   |
| 모델       | 선택    | -       | 사용할 비디오 생성 모델                           |
| 해상도     | 선택    | "540p"  | 출력 비디오 해상도                                 |
| 지속시간   | 선택    | -       | 비디오 길이 옵션                                   |
| 반복       | 불리언 | False   | 비디오를 반복할지 여부                             |
| 시드       | 정수    | 0       | 노드 재실행을 위한 시드 값, 결과는 비결정적입니다 |

### 선택적 매개변수

| 매개변수     | 유형          | 설명                                           |
| ------------- | ------------- | ------------------------------------------------- |
| 첫 번째 이미지   | 이미지         | 비디오의 첫 프레임 (마지막 이미지가 없을 경우 필수) |
| 마지막 이미지    | 이미지         | 비디오의 마지막 프레임 (첫 번째 이미지가 없을 경우 필수) |
| luma 개념들    | LUMA_CONCEPTS | 카메라 움직임과 샷 스타일을 제어하는 개념들      |

### 요구사항

- **첫 번째 이미지** 또는 **마지막 이미지** 중 하나는 반드시 제공해야 합니다.
- 각 이미지 입력(첫 번째 이미지 및 마지막 이미지)은 1개의 이미지만 허용됩니다.

### 출력

| 출력 | 유형  | 설명     |
| ---- | ----- | ---------- |
| 비디오 | 비디오 | 생성된 비디오 |

class LumaImageToVideoGenerationNode(ComfyNodeABC):
    """
    프롬프트, 입력 이미지, 출력 크기를 기반으로 동기적으로 비디오를 생성합니다.
    """

    RETURN_TYPES = (IO.VIDEO,)
    DESCRIPTION = cleandoc(__doc__ or "")  # None 값 처리
    FUNCTION = "api_call"
    API_NODE = True
    CATEGORY = "api node/video/Luma"

    @classmethod
    def INPUT_TYPES(s):
        return {
            "required": {
                "프롬프트": (
                    IO.STRING,
                    {
                        "multiline": True,
                        "default": "",
                        "tooltip": "비디오 생성을 위한 프롬프트",
                    },
                ),
                "모델": ([model.value for model in LumaVideoModel],),
                # "종횡비": ([ratio.value for ratio in LumaAspectRatio], {
                #     "default": LumaAspectRatio.ratio_16_9,
                # }),
                "해상도": (
                    [resolution.value for resolution in LumaVideoOutputResolution],
                    {
                        "default": LumaVideoOutputResolution.res_540p,
                    },
                ),
                "지속시간": ([dur.value for dur in LumaVideoModelOutputDuration],),
                "반복": (
                    IO.BOOLEAN,
                    {
                        "default": False,
                    },
                ),
                "시드": (
                    IO.INT,
                    {
                        "default": 0,
                        "min": 0,
                        "max": 0xFFFFFFFFFFFFFFFF,
                        "control_after_generate": True,
                        "tooltip": "노드가 다시 실행될지 결정하는 시드; 실제 결과는 시드에 관계없이 비결정적입니다.",
                    },
                ),
            },
            "optional": {
                "첫 번째 이미지": (
                    IO.IMAGE,
                    {"tooltip": "생성된 비디오의 첫 프레임."},
                ),
                "마지막 이미지": (IO.IMAGE, {"tooltip": "생성된 비디오의 마지막 프레임."}),
                "luma 개념들": (
                    LumaIO.LUMA_CONCEPTS,
                    {
                        "tooltip": "선택적 카메라 개념들로 Luma Concepts 노드를 통해 카메라 움직임을 지정합니다."
                    },
                ),
            },
            "hidden": {
                "auth_token": "AUTH_TOKEN_COMFY_ORG",
            },
        }

    def api_call(
        self,
        프롬프트: str,
        모델: str,
        해상도: str,
        지속시간: str,
        반복: bool,
        시드,
        첫 번째 이미지: torch.Tensor = None,
        마지막 이미지: torch.Tensor = None,
        luma_concepts: LumaConceptChain = None,
        auth_token=None,
        **kwargs,
    ):
        if 첫 번째 이미지 is None and 마지막 이미지 is None:
            raise Exception(
                "첫 번째 이미지와 마지막 이미지 중 적어도 하나는 입력이 필요합니다."
            )
        키프레임 = self._convert_to_keyframes(첫 번째 이미지, 마지막 이미지, auth_token)
        지속시간 = 지속시간 if 모델 != LumaVideoModel.ray_1_6 else None
        해상도 = 해상도 if 모델 != LumaVideoModel.ray_1_6 else None

        operation = SynchronousOperation(
            endpoint=ApiEndpoint(
                path="/proxy/luma/generations",
                method=HttpMethod.POST,
                request_model=LumaGenerationRequest,
                response_model=LumaGeneration,
            ),
            request=LumaGenerationRequest(
                프롬프트=프롬프트,
                모델=모델,
                종횡비=LumaAspectRatio.ratio_16_9,  # 무시되지만 API에서는 일부 이유로 필요함
                해상도=해상도,
                지속시간=지속시간,
                반복=반복,
                키프레임=키프레임,
                개념들=luma_concepts.create_api_model() if luma_concepts else None,
            ),
            auth_token=auth_token,
        )
        response_api: LumaGeneration = operation.execute()

        operation = PollingOperation(
            poll_endpoint=ApiEndpoint(
                path=f"/proxy/luma/generations/{response_api.id}",
                method=HttpMethod.GET,
                request_model=EmptyRequest,
                response_model=LumaGeneration,
            ),
            completed_statuses=[LumaState.completed],
            failed_statuses=[LumaState.failed],
            status_extractor=lambda x: x.state,
            auth_token=auth_token,
        )
        response_poll = operation.execute()

        vid_response = requests.get(response_poll.assets.video)
        return (VideoFromFile(BytesIO(vid_response.content)),)

    def _convert_to_keyframes(
        self,
        첫 번째 이미지: torch.Tensor = None,
        마지막 이미지: torch.Tensor = None,
        auth_token=None,
    ):
        if 첫 번째 이미지 is None and 마지막 이미지 is None:
            return None
        frame0 = None
        frame1 = None
        if 첫 번째 이미지 is not None:
            download_urls = upload_images_to_comfyapi(
                첫 번째 이미지, max_images=1, auth_token=auth_token
            )
            frame0 = LumaImageReference(type="image", url=download_urls[0])
        if 마지막 이미지 is not None:
            download_urls = upload_images_to_comfyapi(
                마지막 이미지, max_images=1, auth_token=auth_token
            )
            frame1 = LumaImageReference(type="image", url=download_urls[0])
        return LumaKeyframes(frame0=frame0, frame1=frame1)
