Kling 시작-끝 프레임을 동영상으로 노드를 사용하면 두 이미지 간에 부드러운 비디오 전환을 만들 수 있습니다. 이 노드는 모든 중간 프레임을 자동으로 생성하여 유연한 변환을 만들어냅니다.

## 매개변수

### 필수 매개변수

| 매개변수       | 유형   | 설명                                     |
| --------------- | ------ | ----------------------------------------------- |
| start_frame     | 이미지  | 비디오의 시작 이미지                    |
| end_frame       | 이미지  | 비디오의 끝 이미지                      |
| prompt          | 문자열 | 비디오 내용 및 전환을 설명하는 텍스트    |
| negative_prompt | 문자열 | 비디오에서 피해야 할 요소들              |
| cfg_scale       | 부동소수점 | 프롬프트를 얼마나 정확히 따르는지 제어    |
| aspect_ratio    | 선택   | 출력 비디오 종횡비                       |
| mode            | 선택   | 비디오 생성 설정 (모드/지속시간/모델)      |

### 모드 옵션

사용 가능한 모드 조합:
- 표준 모드 / 5초 지속시간 / kling-v1
- 표준 모드 / 5초 지속시간 / kling-v1-5  
- 프로 모드 / 5초 지속시간 / kling-v1
- 프로 모드 / 5초 지속시간 / kling-v1-5
- 프로 모드 / 5초 지속시간 / kling-v1-6
- 프로 모드 / 10초 지속시간 / kling-v1-5
- 프로 모드 / 10초 지속시간 / kling-v1-6

기본값: "프로 모드 / 5초 지속시간 / kling-v1"

## 출력

| 출력 | 유형  | 설명     |
| ------ | ----- | --------------- |
| VIDEO  | 비디오 | 생성된 비디오 |

## 작동 방식

노드는 시작 프레임과 끝 프레임을 분석해 그 사이의 부드러운 전환 시퀀스를 만듭니다. 이미지와 매개변수를 Kling의 API 서버에 전송하면, 서버는 유연한 변환을 위해 필요한 모든 중간 프레임을 생성합니다.

전환 스타일과 내용은 프롬프트를 통해 안내할 수 있으며, 네거티브 프롬프트는 원치 않는 요소를 피하도록 도와줍니다.

```python


class KlingStartEndFrameNode(KlingImage2VideoNode):
    """
    Kling 첫 번째 마지막 프레임 노드. 이 노드는 첫 번째와 마지막 프레임으로 비디오를 생성할 수 있게 해줍니다. 일반 이미지에서 비디오로의 엔드포인트를 호출하지만, `image_tail` 요청 필드를 지원하는 입력 옵션의 하위 집합만 허용합니다.
    """

    @staticmethod
    def get_mode_string_mapping() -> dict[str, tuple[str, str, str]]:
        """
        모드 문자열과 해당하는 (모드, 지속시간, 모델명) 튜플의 매핑을 반환합니다.
        'image_tail' 요청 필드를 지원하는 구성 조합만 포함합니다.
        """
        return {
            "standard mode / 5s duration / kling-v1": ("std", "5", "kling-v1"),
            "standard mode / 5s duration / kling-v1-5": ("std", "5", "kling-v1-5"),
            "pro mode / 5s duration / kling-v1": ("pro", "5", "kling-v1"),
            "pro mode / 5s duration / kling-v1-5": ("pro", "5", "kling-v1-5"),
            "pro mode / 5s duration / kling-v1-6": ("pro", "5", "kling-v1-6"),
            "pro mode / 10s duration / kling-v1-5": ("pro", "10", "kling-v1-5"),
            "pro mode / 10s duration / kling-v1-6": ("pro", "10", "kling-v1-6"),
        }

    @classmethod
    def INPUT_TYPES(s):
        modes = list(KlingStartEndFrameNode.get_mode_string_mapping().keys())
        return {
            "required": {
                "start_frame": model_field_to_node_input(
                    IO.IMAGE, KlingImage2VideoRequest, "image"
                ),
                "end_frame": model_field_to_node_input(
                    IO.IMAGE, KlingImage2VideoRequest, "image_tail"
                ),
                "prompt": model_field_to_node_input(
                    IO.STRING, KlingImage2VideoRequest, "prompt", multiline=True
                ),
                "negative_prompt": model_field_to_node_input(
                    IO.STRING,
                    KlingImage2VideoRequest,
                    "negative_prompt",
                    multiline=True,
                ),
                "cfg_scale": model_field_to_node_input(
                    IO.FLOAT, KlingImage2VideoRequest, "cfg_scale"
                ),
                "aspect_ratio": model_field_to_node_input(
                    IO.COMBO,
                    KlingImage2VideoRequest,
                    "aspect_ratio",
                    enum_type=AspectRatio,
                ),
                "mode": (
                    modes,
                    {
                        "default": modes[2],
                        "tooltip": "비디오 생성에 사용할 구성으로, 형식: 모드 / 지속시간 / 모델명입니다.",
                    },
                ),
            },
            "hidden": {"auth_token": "AUTH_TOKEN_COMFY_ORG"},
        }

    DESCRIPTION = "제공된 시작 이미지와 끝 이미지 간에 전환되는 비디오 시퀀스를 생성합니다. 노드는 그 사이의 모든 프레임을 생성해 첫 번째 프레임에서 마지막 프레임까지 부드러운 변환을 만들어냅니다."

    def parse_inputs_from_mode(self, mode: str) -> tuple[str, str, str]:
        """모드 입력을 (모델명, 지속시간, 모드) 튜플로 파싱합니다."""
        return KlingStartEndFrameNode.get_mode_string_mapping()[mode]

    def api_call(
        self,
        start_frame: torch.Tensor,
        end_frame: torch.Tensor,
        prompt: str,
        negative_prompt: str,
        cfg_scale: float,
        aspect_ratio: str,
        mode: str,
        auth_token: Optional[str] = None,
    ):
        mode, duration, model_name = self.parse_inputs_from_mode(mode)
        return super().api_call(
            prompt=prompt,
            negative_prompt=negative_prompt,
            model_name=model_name,
            start_frame=start_frame,
            cfg_scale=cfg_scale,
            mode=mode,
            aspect_ratio=aspect_ratio,
            duration=duration,
            end_frame=end_frame,
            auth_token=auth_token,
        )
