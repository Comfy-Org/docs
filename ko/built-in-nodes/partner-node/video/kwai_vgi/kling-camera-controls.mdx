Kling 카메라 컨트롤 노드는 가상 카메라의 동작 매개변수를 정의하여 Kling 비디오 생성 중 카메라 움직임과 뷰 변경을 제어합니다.

## 매개변수

| 매개변수           | 유형   | 기본값  | 설명                                                                                                                                                                                                                                    |
| ------------------- | ------ | -------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 카메라_컨트롤_타입 | 선택 | "simple" | 미리 설정된 카메라 모션 유형. simple: 사용자 지정 카메라 움직임; down_back: 카메라가 아래로 이동하고 뒤로 이동함; forward_up: 카메라가 앞으로 이동하고 위로 기울어짐; right_turn_forward: 오른쪽으로 회전하며 앞으로 이동함; left_turn_forward: 왼쪽으로 회전하며 앞으로 이동함 |
| 수평_움직임       | 부동소수점 | 0        | 카메라의 수평축(x축) 움직임을 제어합니다. 음수 값은 왼쪽으로, 양수 값은 오른쪽으로 이동합니다                                                                                                                                    |
| 수직_움직임       | 부동소수점 | 0        | 카메라의 수직축(y축) 움직임을 제어합니다. 음수 값은 아래로, 양수 값은 위로 이동합니다                                                                                                                                         |
| 팬                | 부동소수점 | 0.5      | 카메라의 수직면(x축) 회전을 제어합니다. 음수 값은 아래로 회전하고, 양수 값은 위로 회전합니다                                                                                                                                    |
| 틸트              | 부동소수점 | 0        | 카메라의 수평면(y축) 회전을 제어합니다. 음수 값은 왼쪽으로 회전하고, 양수 값은 오른쪽으로 회전합니다                                                                                                                               |
| 롤                | 부동소수점 | 0        | 카메라의 롤링 양(z축)을 제어합니다. 음수 값은 시계 반대 방향으로 회전하고, 양수 값은 시계 방향으로 회전합니다                                                                                                                                |
| 줌                | 부동소수점 | 0        | 카메라의 초점 거리를 제어합니다. 음수 값은 화각을 좁히고, 양수 값은 화각을 넓힙니다                                                                                                                                                   |

**참고**: 효과가 작동하려면 적어도 하나 이상의 카메라 제어 매개변수가 0이 아니어야 합니다.

### 출력

| 출력         | 유형           | 설명                               |
| -------------- | -------------- | ----------------------------------------- |
| 카메라_컨트롤 | CAMERA_CONTROL | 카메라 설정이 포함된 구성 객체 |

**참고**: 모든 모델 및 모드 조합이 카메라 제어를 지원하는 것은 아닙니다. 자세한 내용은 Kling API 문서를 확인해 주세요.

## 소스 코드

[노드 소스 코드 (2025-05-03 업데이트)]

```python

class KlingCameraControls(KlingNodeBase):
    """Kling 카메라 컨트롤 노드"""

    @classmethod
    def INPUT_TYPES(cls):
        return {
            "required": {
                "카메라_컨트롤_타입": (
                    IO.COMBO,
                    {
                        "options": [
                            camera_control_type.value
                            for camera_control_type in CameraType
                        ],
                        "default": "simple",
                        "tooltip": "미리 설정된 카메라 움직임 유형. simple: 사용자 지정 가능한 카메라 움직임. down_back: 카메라가 아래로 내려가며 뒤로 이동함. forward_up: 카메라가 앞으로 이동하며 위로 기울어짐. right_turn_forward: 오른쪽으로 회전하며 앞으로 이동함. left_turn_forward: 왼쪽으로 회전하며 앞으로 이동함.",
                    },
                ),
                "수평_움직임": get_camera_control_input_config(
                    "카메라의 수평축(x축)을 따라 움직임을 제어합니다. 음수는 왼쪽, 양수는 오른쪽을 의미함"
                ),
                "수직_움직임": get_camera_control_input_config(
                    "카메라의 수직축(y축)을 따라 움직임을 제어합니다. 음수는 아래로, 양수는 위로를 의미함."
                ),
                "팬": get_camera_control_input_config(
                    "카메라의 수직면(x축)에서 회전을 제어합니다. 음수는 아래로 회전, 양수는 위로 회전을 의미함.",
                    default=0.5,
                ),
                "틸트": get_camera_control_input_config(
                    "카메라의 수평면(y축)에서 회전을 제어합니다. 음수는 왼쪽 회전, 양수는 오른쪽 회전을 의미함.",
                ),
                "롤": get_camera_control_input_config(
                    "카메라의 롤링 양(z축)을 제어합니다. 음수는 시계 반대 방향, 양수는 시계 방향을 의미함.",
                ),
                "줌": get_camera_control_input_config(
                    "카메라의 초점 거리 변화를 제어합니다. 음수는 화각을 좁히고, 양수는 화각을 넓힘.",
                ),
            }
        }

    DESCRIPTION = "Kling 카메라 컨트롤 노드. 모든 모델 및 모드 조합이 카메라 제어를 지원하는 것은 아닙니다. 자세한 정보는 Kling API 문서를 참고하세요."
    RETURN_TYPES = ("CAMERA_CONTROL",)
    RETURN_NAMES = ("camera_control",)
    FUNCTION = "main"

    @classmethod
    def VALIDATE_INPUTS(
        cls,
        수평_움직임: float,
        수직_움직임: float,
        팬: float,
        틸트: float,
        롤: float,
        줌: float,
    ) -> bool | str:
        if not is_valid_camera_control_configs(
            [
                수평_움직임,
                수직_움직임,
                팬,
                틸트,
                롤,
                줌,
            ]
        ):
            return "잘못된 카메라 제어 설정: 값 중 적어도 하나는 0이 아니어야 합니다"
        return True

    def main(
        self,
        카메라_컨트롤_타입: str,
        수평_움직임: float,
        수직_움직임: float,
        팬: float,
        틸트: float,
        롤: float,
        줌: float,
    ) -> tuple[CameraControl]:
        return (
            CameraControl(
                type=CameraType(카메라_컨트롤_타입),
                config=CameraConfig(
                    horizontal=수평_움직임,
                    vertical=수직_움직임,
                    pan=팬,
                    roll=롤,
                    tilt=틸트,
                    zoom=줌,
                ),
            ),
        )
```
