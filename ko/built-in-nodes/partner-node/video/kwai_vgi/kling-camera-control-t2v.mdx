클링 텍스트 to 비디오 (카메라 제어) 노드는 텍스트를 전문적인 카메라 움직임과 함께 비디오로 변환합니다. 이 노드는 표준 클링 텍스트 to 비디오 노드에 카메라 제어 기능을 추가하여 확장한 것입니다.

## 매개변수

### 기본 매개변수

| 매개변수       | 유형           | 기본값 | 설명                                     |
| --------------- | -------------- | ------- | ----------------------------------------------- |
| 프롬프트          | 문자열         | ""      | 비디오 내용을 설명하는 텍스트 프롬프트            |
| 네거티브 프롬프트 | 문자열         | ""      | 비디오에서 피하고자 하는 요소                  |
| cfg_scale       | 부동소수점     | 7.0     | 프롬프트를 얼마나 정확히 따르도록 할지 제어    |
| 종횡비          | 선택           | "16:9"  | 출력 비디오 종횡비                       |
| 카메라 제어      | CAMERA_CONTROL | -       | 클링 카메라 제어 노드의 카메라 설정 |

### 고정 매개변수

참고: 다음 매개변수는 고정되어 변경할 수 없습니다:
- 모델: kling-v1-5  
- 모드: pro  
- 지속시간: 5초

### 출력

| 출력 | 유형  | 설명     |
| ------ | ----- | --------------- |
| 비디오  | 비디오 | 생성된 비디오 |

## 소스 코드

[노드 소스 코드 (2025-05-03 업데이트)]

```python

class KlingCameraControlT2VNode(KlingTextToVideoNode):
    """
    클링 텍스트 to 비디오 카메라 제어 노드. 이 노드는 텍스트 to 비디오 노드이지만, 카메라 제어가 가능합니다.
    지속시간, 모드 및 모델명 요청 필드는 하드코딩되어 있습니다. 카메라 제어는 2025-05-02 기준으로 kling-v1-5 모델의 pro 모드에서만 5초 지속시간으로 지원됩니다.
    """

    @classmethod
    def INPUT_TYPES(s):
        return {
            "required": {
                "prompt": model_field_to_node_input(
                    IO.STRING, KlingText2VideoRequest, "prompt", multiline=True
                ),
                "negative_prompt": model_field_to_node_input(
                    IO.STRING,
                    KlingText2VideoRequest,
                    "negative_prompt",
                    multiline=True,
                ),
                "cfg_scale": model_field_to_node_input(
                    IO.FLOAT, KlingText2VideoRequest, "cfg_scale"
                ),
                "aspect_ratio": model_field_to_node_input(
                    IO.COMBO,
                    KlingText2VideoRequest,
                    "aspect_ratio",
                    enum_type=AspectRatio,
                ),
                "camera_control": (
                    "CAMERA_CONTROL",
                    {
                        "tooltip": "클링 카메라 제어 노드를 사용해 생성할 수 있습니다. 비디오 생성 중 카메라의 움직임과 동작을 제어합니다.",
                    },
                ),
            },
            "hidden": {"auth_token": "AUTH_TOKEN_COMFY_ORG"},
        }

    DESCRIPTION = "텍스트를 영화 같은 비디오로 변환하며, 실제 촬영을 모방한 전문적인 카메라 움직임을 제공합니다. 줌, 회전, 패닝, 틸트, 1인칭 시점 등 가상 카메라 액션을 제어하면서도 원본 텍스트에 초점을 유지합니다."

    def api_call(
        self,
        prompt: str,
        negative_prompt: str,
        cfg_scale: float,
        aspect_ratio: str,
        camera_control: Optional[CameraControl] = None,
        auth_token: Optional[str] = None,
    ):
        return super().api_call(
            model_name="kling-v1-5",
            cfg_scale=cfg_scale,
            mode="pro",
            aspect_ratio=aspect_ratio,
            duration="5",
            prompt=prompt,
            negative_prompt=negative_prompt,
            camera_control=camera_control,
            auth_token=auth_token,
        )



```
