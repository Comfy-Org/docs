클링 이미지에서 비디오로 변환(카메라 제어) 노드는 정적인 이미지를 전문적인 카메라 움직임과 함께 비디오로 변환합니다. 줌, 회전, 패닝, 틸트 및 1인칭 시점과 같은 카메라 제어를 지원하며, 원본 이미지의 내용에 초점을 유지합니다.

## 매개변수

### 기본 매개변수

| 매개변수       | 유형   | 기본값 | 설명                                     |
| --------------- | ------ | ------- | ----------------------------------------------- |
| start_frame     | 이미지  | -       | 비디오로 변환할 입력 이미지                 |
| prompt          | 문자열 | ""      | 비디오 동작과 내용을 설명하는 텍스트 프롬프트 |
| negative_prompt | 문자열 | ""      | 비디오에서 피하고자 하는 요소                  |
| cfg_scale       | 부동소수점 | 7.0     | 프롬프트를 얼마나 충실히 따르는지 제어         |
| aspect_ratio    | 선택   | 16:9    | 출력 비디오의 종횡비                       |

### 카메라 제어 매개변수

| 매개변수      | 유형          | 설명                                           |
| -------------- | ------------- | ------------------------------------------------- |
| camera_control | 카메라제어    | 클링 카메라 제어 노드에서 가져온 카메라 제어 구성 |

### 출력

| 출력 | 유형  | 설명     |
| ------ | ----- | ---------- |
| VIDEO  | 비디오 | 생성된 비디오 |

## 소스 코드

[노드 소스 코드 (2025-05-03 업데이트)]
```python

class KlingCameraControlI2VNode(KlingImage2VideoNode):
    """
    클링 이미지에서 비디오로 변환 카메라 제어 노드. 이 노드는 이미지에서 비디오로 변환하는 노드이지만, 카메라 제어 기능을 지원합니다.
    지속시간, 모드 및 모델명 요청 필드는 하드코딩되어 있습니다. 카메라 제어는 2025-05-02 기준으로 kling-v1-5 모델의 프로 모드에서만 5초 지속시간으로 지원됩니다.
    """

    @classmethod
    def INPUT_TYPES(s):
        return {
            "required": {
                "start_frame": model_field_to_node_input(
                    IO.IMAGE, KlingImage2VideoRequest, "image"
                ),
                "prompt": model_field_to_node_input(
                    IO.STRING, KlingImage2VideoRequest, "prompt", multiline=True
                ),
                "negative_prompt": model_field_to_node_input(
                    IO.STRING,
                    KlingImage2VideoRequest,
                    "negative_prompt",
                    multiline=True,
                ),
                "cfg_scale": model_field_to_node_input(
                    IO.FLOAT, KlingImage2VideoRequest, "cfg_scale"
                ),
                "aspect_ratio": model_field_to_node_input(
                    IO.COMBO,
                    KlingImage2VideoRequest,
                    "aspect_ratio",
                    enum_type=AspectRatio,
                ),
                "camera_control": (
                    "CAMERA_CONTROL",
                    {
                        "tooltip": "클링 카메라 제어 노드를 사용해 생성할 수 있습니다. 비디오 생성 중 카메라 움직임과 동작을 제어합니다.",
                    },
                ),
            },
            "hidden": {"auth_token": "AUTH_TOKEN_COMFY_ORG"},
        }

    DESCRIPTION = "정지된 이미지를 영화 같은 비디오로 변환하며, 실제 촬영을 모방한 전문적인 카메라 움직임을 제공합니다. 줌, 회전, 패닝, 틸트 및 1인칭 시점과 같은 가상 카메라 동작을 제어하면서 원본 이미지에 초점을 유지합니다."

    def api_call(
        self,
        start_frame: torch.Tensor,
        prompt: str,
        negative_prompt: str,
        cfg_scale: float,
        aspect_ratio: str,
        camera_control: CameraControl,
        auth_token: Optional[str] = None,
    ):
        return super().api_call(
            model_name="kling-v1-5",
            start_frame=start_frame,
            cfg_scale=cfg_scale,
            mode="pro",
            aspect_ratio=aspect_ratio,
            duration="5",
            prompt=prompt,
            negative_prompt=negative_prompt,
            camera_control=camera_control,
            auth_token=auth_token,
        )
```
