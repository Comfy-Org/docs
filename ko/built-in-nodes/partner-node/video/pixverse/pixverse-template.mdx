![ComfyUI 기본 PixVerse 템플릿 노드](/images/built-in-nodes/api_nodes/pixverse/pixverse-template.jpg)

PixVerse 템플릿 노드를 사용하면 미리 정의된 동영상 생성 템플릿 중에서 선택하여 PixVerse 동영상 생성 노드의 스타일과 효과를 제어할 수 있습니다.  
이 도우미 노드는 PixVerse 동영상 생성 노드와 연결되어, 사용자가 복잡한 파라미터 조합을 직접 조정하지 않고도 빠르게 사전 설정된 동영상 스타일을 적용할 수 있도록 해줍니다.

## 매개변수

### 필수 매개변수

| 매개변수 | 유형   | 설명                                    |
| --------- | ------ | ---------------------------------------------- |
| template  | 선택   | 사용 가능한 동영상 사전 설정 중에서 템플릿을 선택하세요 |

### 출력

| 출력            | 유형                | 설명                                              |
| ----------------- | ------------------- | -------------------------------------------------------- |
| pixverse_template | PixverseIO.TEMPLATE | 선택한 템플릿 ID를 포함한 구성 객체 |

## 소스 코드

[노드 소스 코드 (2025-05-05 업데이트)]

```python

class PixverseTemplateNode:
    """
    PixVerse 동영상 생성을 위한 템플릿을 선택합니다.
    """

    RETURN_TYPES = (PixverseIO.TEMPLATE,)
    RETURN_NAMES = ("pixverse_template",)
    FUNCTION = "create_template"
    CATEGORY = "api node/video/Pixverse"

    @classmethod
    def INPUT_TYPES(s):
        return {
            "required": {
                "template": (list(pixverse_templates.keys()), ),
            }
        }

    def create_template(self, template: str):
        template_id = pixverse_templates.get(template, None)
        if template_id is None:
            raise Exception(f"'{template}' 템플릿은 인식되지 않습니다.")
        # 그냥 정수를 반환합니다.
        return (template_id,)

```
