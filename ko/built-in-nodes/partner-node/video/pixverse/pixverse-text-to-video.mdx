PixVerse 텍스트에서 비디오로 노드는 PixVerse의 텍스트-비디오 API에 연결되어 사용자가 텍스트 설명을 통해 고품질 비디오를 생성할 수 있도록 합니다. 사용자는 비디오 품질, 지속 시간, 모션 모드와 같은 다양한 매개변수를 조정하여 창작물을 맞춤화할 수 있습니다.

## 매개변수

### 필수 매개변수

| 매개변수        | 유형    | 기본값                  | 설명                                   |
| ---------------- | ------- | ------------------------ | --------------------------------------------- |
| 프롬프트           | 문자열  | ""                       | 비디오 내용을 설명하는 텍스트 프롬프트      |
| 종횡비             | 선택    | -                        | 출력 비디오 종횡비                     |
| 품질               | 선택    | PixverseQuality.res_540p | 비디오 품질 레벨                           |
| 지속 시간 초       | 선택    | -                        | 비디오 지속 시간                                |
| 모션 모드           | 선택    | -                        | 비디오 모션 모드                             |
| 시드               | 정수    | 0                        | 일관된 생성 결과를 위한 난수 시드 |

### 선택적 매개변수

| 매개변수         | 유형              | 기본값 | 설명                          |
| ----------------- | ----------------- | ------- | ------------------------------------ |
| 네거티브 프롬프트   | 문자열            | ""      | 비디오에서 제외할 요소들   |
| pixverse_template | PIXVERSE_TEMPLATE | None    | 스타일 설정을 위한 옵션 템플릿 |

### 제한사항

- 1080p 품질은 5초 지속 시간과 함께 일반 모션 모드만 지원합니다.
- 5초가 아닌 지속 시간은 일반 모션 모드만 지원합니다.

### 출력

| 출력 | 유형  | 설명     |
| ------ | ----- | --------------- |
| 비디오  | 비디오 | 생성된 비디오 |


## 소스 코드

[노드 소스 코드 (2025-05-05 업데이트)]

```python

class PixverseTextToVideoNode(ComfyNodeABC):
    """
    프롬프트와 출력 크기에 따라 동기적으로 비디오를 생성합니다.
    """

    RETURN_TYPES = (IO.VIDEO,)
    DESCRIPTION = cleandoc(__doc__ or "")  # 잠재적인 None 값 처리
    FUNCTION = "api_call"
    API_NODE = True
    CATEGORY = "api node/video/Pixverse"

    @classmethod
    def INPUT_TYPES(s):
        return {
            "required": {
                "프롬프트": (
                    IO.STRING,
                    {
                        "다중행": True,
                        "기본값": "",
                        "툴팁": "비디오 생성을 위한 프롬프트",
                    },
                ),
                "종횡비": (
                    [ratio.value for ratio in PixverseAspectRatio],
                ),
                "품질": (
                    [resolution.value for resolution in PixverseQuality],
                    {
                        "기본값": PixverseQuality.res_540p,
                    },
                ),
                "지속 시간 초": ([dur.value for dur in PixverseDuration],),
                "모션 모드": ([mode.value for mode in PixverseMotionMode],),
                "시드": (
                    IO.INT,
                    {
                        "기본값": 0,
                        "최소": 0,
                        "최대": 2147483647,
                        "생성 후 제어": True,
                        "툴팁": "비디오 생성을 위한 시드.",
                    },
                ),
            },
            "optional": {
                "네거티브 프롬프트": (
                    IO.STRING,
                    {
                        "기본값": "",
                        "강제 입력": True,
                        "툴팁": "이미지에서 원치 않는 요소를 설명하는 선택적 텍스트.",
                    },
                ),
                "pixverse_template": (
                    PixverseIO.TEMPLATE,
                    {
                        "툴팁": "PixVerse Template 노드로 생성된 스타일 영향을 주는 옵션 템플릿."
                    }
                )
            },
            "hidden": {
                "auth_token": "AUTH_TOKEN_COMFY_ORG",
            },
        }

    def api_call(
        self,
        프롬프트: str,
        종횡비: str,
        품질: str,
        지속 시간 초: int,
        모션 모드: str,
        시드,
        네거티브 프롬프트: str=None,
        pixverse_template: int=None,
        auth_token=None,
        **kwargs,
    ):
        # 1080p는 5초 지속 시간으로 제한됩니다.
        # 1080p나 5초가 아닌 지속 시간에서는 일반 모션 모드만 지원합니다.
        if 품질 == PixverseQuality.res_1080p:
            모션 모드 = PixverseMotionMode.normal
            지속 시간 초 = PixverseDuration.dur_5
        elif 지속 시간 초 != PixverseDuration.dur_5:
            모션 모드 = PixverseMotionMode.normal

        operation = SynchronousOperation(
            endpoint=ApiEndpoint(
                path="/proxy/pixverse/video/text/generate",
                method=HttpMethod.POST,
                request_model=PixverseTextVideoRequest,
                response_model=PixverseVideoResponse,
            ),
            request=PixverseTextVideoRequest(
                프롬프트=프롬프트,
                종횡비=종횡비,
                품질=품질,
                지속 시간=지속 시간 초,
                모션 모드=모션 모드,
                네거티브 프롬프트=네거티브 프롬프트 if 네거티브 프롬프트 else None,
                template_id=pixverse_template,
                시드=시드,
            ),
            auth_token=auth_token,
        )
        response_api = operation.execute()

        if response_api.Resp is None:
            raise Exception(f"Pixverse 요청 실패: '{response_api.ErrMsg}'")

        operation = PollingOperation(
            poll_endpoint=ApiEndpoint(
                path=f"/proxy/pixverse/video/result/{response_api.Resp.video_id}",
                method=HttpMethod.GET,
                request_model=EmptyRequest,
                response_model=PixverseGenerationStatusResponse,
            ),
            completed_statuses=[PixverseStatus.successful],
            failed_statuses=[PixverseStatus.contents_moderation, PixverseStatus.failed, PixverseStatus.deleted],
            status_extractor=lambda x: x.Resp.status,
            auth_token=auth_token,
        )
        response_poll = operation.execute()

        vid_response = requests.get(response_poll.Resp.url)
        return (VideoFromFile(BytesIO(vid_response.content)),)
``` 
