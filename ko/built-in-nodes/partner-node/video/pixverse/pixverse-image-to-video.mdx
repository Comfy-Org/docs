PixVerse 이미지에서 비디오로 노드는 PixVerse의 API를 사용해 정적 이미지를 동적 비디오로 변환합니다. 원본 이미지의 시각적 특성을 그대로 유지하면서 텍스트 프롬프트에 기반한 자연스러운 움직임을 추가합니다.

## 매개변수

### 필수 매개변수

| 매개변수       | 유형    | 기본값     | 설명                                 |
| --------------- | ------- | ----------- | ------------------------------------------- |
| image           | 이미지   | -           | 비디오로 변환할 입력 이미지             |
| prompt          | 문자열  | ""          | 비디오의 움직임/내용을 설명하는 텍스트 프롬프트 |
| negative_prompt | 문자열  | ""          | 비디오에서 피해야 할 요소              |
| seed            | 정수    | -1          | 난수 시드 (-1일 경우 랜덤)                 |
| quality         | 선택    | "high"      | 출력 비디오의 품질 수준                  |
| aspect_ratio    | 선택    | "r16_9"     | 출력 비디오의 종횡비                   |
| duration        | 선택    | "seconds_4" | 생성된 비디오의 길이                   |
| motion_mode     | 선택    | "standard"  | 비디오의 움직임 스타일                          |

### 선택적 매개변수

| 매개변수         | 유형              | 기본값 | 설명                |
| ----------------- | ----------------- | ------- | -------------------------- |
| pixverse_template | PIXVERSE_TEMPLATE | None    | 선택적 PixVerse 템플릿 |

### 출력

| 출력 | 유형  | 설명     |
| ------ | ----- | --------------- |
| VIDEO  | 비디오 | 생성된 비디오 |

## 소스 코드

[노드 소스 코드 (2025-05-05 업데이트)]

```python
class PixverseImageToVideoNode(ComfyNodeABC):
    """
    Pixverse 이미지에서 비디오로

    이미지와 프롬프트를 이용해 비디오를 생성합니다.
    """

    @classmethod
    def INPUT_TYPES(s):
        return {
            "required": {
                "image": ("IMAGE",),
                "prompt": ("STRING", {"multiline": True, "default": ""}),
                "negative_prompt": ("STRING", {"multiline": True, "default": ""}),
                "seed": ("INT", {"default": -1, "min": -1, "max": 0xffffffffffffffff}),
                "quality": (list(PixverseQuality.__members__.keys()), {"default": "high"}),
                "aspect_ratio": (list(PixverseAspectRatio.__members__.keys()), {"default": "r16_9"}),
                "duration": (list(PixverseDuration.__members__.keys()), {"default": "seconds_4"}),
                "motion_mode": (list(PixverseMotionMode.__members__.keys()), {"default": "standard"}),
            },
            "optional": {
                "pixverse_template": ("PIXVERSE_TEMPLATE",),
            },
            "hidden": {
                "auth_token": "AUTH_TOKEN_COMFY_ORG",
            },
        }

    RETURN_TYPES = ("VIDEO",)
    DESCRIPTION = "PixVerse의 API를 사용해 이미지와 프롬프트를 이용해 비디오를 생성합니다"
    FUNCTION = "generate_video"
    CATEGORY = "api node/video/Pixverse"
    API_NODE = True
    OUTPUT_NODE = True
```
