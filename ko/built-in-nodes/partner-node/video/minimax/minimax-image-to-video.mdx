![ComfyUI 기본 MiniMax 이미지에서 비디오로 노드](/images/built-in-nodes/api_nodes/minimax/minimax-image-to-video.jpg)

MiniMax 이미지에서 비디오로 노드는 MiniMax의 API를 사용해 입력 이미지와 텍스트 프롬프트를 기반으로 비디오를 생성합니다.

## 매개변수

### 필수 매개변수

| 매개변수   | 유형   | 기본값  | 설명                                                  |
| ----------- | ------ | -------- | ------------------------------------------------------------ |
| 이미지       | 이미지  | -        | 비디오의 첫 번째 프레임으로 사용할 입력 이미지                 |
| 프롬프트 텍스트 | 문자열 | ""       | 비디오 생성을 안내하는 텍스트 프롬프트                        |
| 모델       | 선택   | "I2V-01" | 사용 가능한 모델: "I2V-01-Director", "I2V-01", "I2V-01-live" |

### 선택적 매개변수

| 매개변수 | 유형    | 설명                      |
| --------- | ------- | -------------------------------- |
| 시드      | 정수    | 노이즈 생성을 위한 임의의 시드 |

### 출력

| 출력 | 유형  | 설명     |
| ------ | ----- | --------------- |
| 비디오  | 비디오 | 생성된 비디오 |

## 소스 코드

[노드 소스 코드 (2025-05-03 업데이트)]

```python

class MinimaxImageToVideoNode(MinimaxTextToVideoNode):
    """
    MiniMax의 API를 사용해 이미지와 프롬프트, 그리고 선택적 매개변수를 기반으로 동기식으로 비디오를 생성합니다.
    """

    @classmethod
    def INPUT_TYPES(s):
        return {
            "required": {
                "image": (
                    IO.IMAGE,
                    {
                        "tooltip": "비디오 생성의 첫 번째 프레임으로 사용할 이미지"
                    },
                ),
                "prompt_text": (
                    "STRING",
                    {
                        "multiline": True,
                        "default": "",
                        "tooltip": "비디오 생성을 안내하는 텍스트 프롬프트",
                    },
                ),
                "model": (
                    [
                        "I2V-01-Director",
                        "I2V-01",
                        "I2V-01-live",
                    ],
                    {
                        "default": "I2V-01",
                        "tooltip": "비디오 생성에 사용할 모델",
                    },
                ),
            },
            "optional": {
                "seed": (
                    IO.INT,
                    {
                        "default": 0,
                        "min": 0,
                        "max": 0xFFFFFFFFFFFFFFFF,
                        "control_after_generate": True,
                        "tooltip": "노이즈 생성에 사용되는 임의의 시드.",
                    },
                ),
            },
            "hidden": {
                "auth_token": "AUTH_TOKEN_COMFY_ORG",
            },
        }

    RETURN_TYPES = ("VIDEO",)
    DESCRIPTION = "MiniMax의 API를 사용해 이미지와 프롬프트를 기반으로 비디오를 생성합니다"
    FUNCTION = "generate_video"
    CATEGORY = "api node/video/Minimax"
    API_NODE = True
    OUTPUT_NODE = True
```
