Pika 2.2 이미지에서 비디오로 노드는 Pika의 최신 2.2 API와 연결되어 정적 이미지를 동적 비디오로 변환합니다. 원본 이미지의 시각적 특성을 유지하면서 텍스트 프롬프트에 기반한 자연스러운 움직임을 추가합니다.

## 매개변수

### 필수 매개변수

| 매개변수       | 유형    | 기본값 | 설명                                     |
| --------------- | ------- | ------- | ----------------------------------------------- |
| 이미지           | 이미지   | -       | 비디오로 변환할 입력 이미지                 |
| 프롬프트 텍스트     | 문자열  | ""      | 비디오의 움직임과 내용을 설명하는 텍스트 프롬프트 |
| 네거티브 프롬프트 | 문자열  | ""      | 비디오에서 피해야 할 요소                   |
| 시드            | 정수    | 0       | 생성을 위한 난수 시드                      |
| 해상도          | 선택    | "1080p" | 출력 비디오 해상도                         |
| 지속시간        | 선택    | "5초"   | 생성된 비디오의 길이                       |

### 출력

| 출력 | 유형  | 설명     |
| ------ | ----- | ---------- |
| 비디오  | 비디오 | 생성된 비디오 |

## 작동 방식

노드는 입력 이미지와 매개변수(프롬프트, 해상도, 지속시간 등)를 멀티파트 폼 데이터로 Pika의 API 서버에 전송합니다. API는 이를 처리한 후 생성된 비디오를 반환합니다. 사용자는 프롬프트, 네거티브 프롬프트, 난수 시드 및 기타 매개변수를 조정하여 출력을 제어할 수 있습니다.

## 소스 코드

[노드 소스 코드 (2025-05-05 업데이트)]

```python

class PikaImageToVideoV2_2(PikaNodeBase):
    """Pika 2.2 이미지에서 비디오로 노드."""

    @classmethod
    def INPUT_TYPES(cls):
        return {
            "required": {
                "image": (
                    IO.IMAGE,
                    {"tooltip": "비디오로 변환할 이미지"},
                ),
                **cls.get_base_inputs_types(PikaBodyGenerate22I2vGenerate22I2vPost),
            },
            "hidden": {
                "auth_token": "AUTH_TOKEN_COMFY_ORG",
            },
        }

    DESCRIPTION = "Pika API v2.2에 이미지와 프롬프트를 전송해 비디오를 생성합니다."
    RETURN_TYPES = ("VIDEO",)

    def api_call(
        self,
        image: torch.Tensor,
        prompt_text: str,
        negative_prompt: str,
        seed: int,
        resolution: str,
        duration: int,
        auth_token: Optional[str] = None,
    ) -> tuple[VideoFromFile]:
        """Pika 2.2 이미지에서 비디오로 API 호출."""
        # 이미지를 BytesIO로 변환
        image_bytes_io = tensor_to_bytesio(image)
        image_bytes_io.seek(0)  # 스트림 위치 재설정

        # 멀티파트 업로드를 위한 파일 데이터 준비
        pika_files = {"image": ("image.png", image_bytes_io, "image/png")}

        # Pydantic 모델을 사용해 파일 외부 데이터 준비
        pika_request_data = PikaBodyGenerate22I2vGenerate22I2vPost(
            promptText=prompt_text,
            negativePrompt=negative_prompt,
            seed=seed,
            resolution=resolution,
            duration=duration,
        )

        initial_operation = SynchronousOperation(
            endpoint=ApiEndpoint(
                path=PATH_IMAGE_TO_VIDEO,
                method=HttpMethod.POST,
                request_model=PikaBodyGenerate22I2vGenerate22I2vPost,
                response_model=PikaGenerateResponse,
            ),
            request=pika_request_data,
            files=pika_files,
            content_type="multipart/form-data",
            auth_token=auth_token,
        )

        return self.execute_task(initial_operation, auth_token)

```
