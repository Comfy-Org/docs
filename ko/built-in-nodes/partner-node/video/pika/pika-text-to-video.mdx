Pika 2.2 텍스트에서 비디오로 노드는 Pika의 2.2 API를 사용해 텍스트 설명을 비디오로 생성합니다. 이 노드는 Pika의 텍스트에서 비디오로 변환 API와 연결되어, 사용자가 다양한 제어 매개변수를 통해 텍스트 프롬프트를 이용해 비디오를 생성할 수 있도록 합니다.

## 매개변수

### 필수 매개변수
| 매개변수       | 유형    | 기본값            | 설명                                   |
| --------------- | ------- | ------------------ | --------------------------------------------- |
| prompt_text     | 문자열  | ""                 | 비디오 내용을 설명하는 텍스트 프롬프트      |
| negative_prompt | 문자열  | ""                 | 비디오에서 제외할 요소들            |
| seed            | 정수    | 0                  | 생성을 위한 난수 시드                    |
| resolution      | 선택    | "1080p"            | 출력 비디오 해상도                       |
| duration        | 선택    | "5s"               | 생성된 비디오의 길이                     |
| aspect_ratio    | 부동소수점 | 1.7777777777777777 | 비디오 종횡비, 범위 0.4~2.5, 단계 0.001 |

### 출력

| 출력 | 유형  | 설명     |
| ------ | ----- | --------------- |
| VIDEO  | 비디오 | 생성된 비디오 |

            content_type="application/x-www-form-urlencoded",
