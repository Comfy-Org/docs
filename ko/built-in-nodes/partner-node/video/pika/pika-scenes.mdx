![ComfyUI 내장 Pika 2.2 장면 노드](/images/built-in-nodes/api_nodes/pika/pika-2-2-scenes.jpg)

Pika 2.2 장면 노드를 사용하면 여러 이미지를 업로드하고, 이러한 요소들을 포함한 고품질의 동영상을 생성할 수 있습니다. 이 노드는 Pika의 2.2 API를 사용해 이미지 간에 부드러운 장면 전환을 만들어냅니다.

## 매개변수

### 필수 매개변수

| 매개변수        | 유형    | 기본값                   | 설명                                     |
| ---------------- | ------- | ------------------------- | ----------------------------------------------- |
| prompt_text      | 문자열  | ""                        | 동영상 내용과 장면을 설명하는 텍스트 프롬프트 |
| negative_prompt  | 문자열  | ""                        | 동영상에서 제외할 요소들              |
| seed             | 정수    | 0                         | 생성을 위한 랜덤 시드                      |
| ingredients_mode | 선택    | "creative"                | 이미지 조합 모드                          |
| resolution       | 선택    | API 기본값               | 출력 동영상 해상도                         |
| duration         | 선택    | API 기본값               | 출력 동영상 길이                             |
| aspect_ratio     | 실수    | 1.7777777777777777 (16:9) | 동영상 종횡비, 범위 0.4~2.5               |

### 선택적 매개변수

| 매개변수          | 유형  | 설명        |
| ------------------ | ----- | ------------------ |
| image_ingredient_1 | 이미지 | 첫 번째 장면 이미지  |
| image_ingredient_2 | 이미지 | 두 번째 장면 이미지 |
| image_ingredient_3 | 이미지 | 세 번째 장면 이미지  |
| image_ingredient_4 | 이미지 | 네 번째 장면 이미지 |
| image_ingredient_5 | 이미지 | 다섯 번째 장면 이미지 |

## 출력

| 출력 | 유형  | 설명     |
| ------ | ----- | --------------- |
| VIDEO  | 동영상 | 생성된 동영상 |

## 작동 방식

Pika 2.2 장면 노드는 입력된 모든 이미지를 분석해 해당 이미지 요소들을 포함한 동영상을 만듭니다. 노드는 이미지와 매개변수를 Pika의 API 서버로 전송하며, 서버는 이를 처리한 후 생성된 동영상을 반환합니다.

사용자는 프롬프트를 통해 동영상 스타일과 내용을 조정할 수 있으며, 음모 프롬프트를 사용해 원치 않는 요소를 제외할 수 있습니다. 노드는 최대 5개의 입력 이미지를 재료로 지원하며, 지정된 조합 모드, 해상도, 길이, 종횡비에 따라 최종 동영상을 생성합니다.

## 소스 코드

```python

class PikaScenesV2_2(PikaNodeBase):
    """Pika 2.2 장면 노드."""

    @classmethod
    def INPUT_TYPES(cls):
        image_ingredient_input = (
            IO.IMAGE,
            {"tooltip": "동영상 생성에 재료로 사용될 이미지."},
        )
        return {
            "required": {
                **cls.get_base_inputs_types(
                    PikaBodyGenerate22C2vGenerate22PikascenesPost,
                ),
                "ingredients_mode": model_field_to_node_input(
                    IO.COMBO,
                    PikaBodyGenerate22C2vGenerate22PikascenesPost,
                    "ingredientsMode",
                    enum_type=IngredientsMode,
                    default="creative",
                ),
                "aspect_ratio": model_field_to_node_input(
                    IO.FLOAT,
                    PikaBodyGenerate22C2vGenerate22PikascenesPost,
                    "aspectRatio",
                    step=0.001,
                    min=0.4,
                    max=2.5,
                    default=1.7777777777777777,
                ),
            },
            "optional": {
                "image_ingredient_1": image_ingredient_input,
                "image_ingredient_2": image_ingredient_input,
                "image_ingredient_3": image_ingredient_input,
                "image_ingredient_4": image_ingredient_input,
                "image_ingredient_5": image_ingredient_input,
            },
            "hidden": {
                "auth_token": "AUTH_TOKEN_COMFY_ORG",
            },
        }

    DESCRIPTION = "이미지를 결합해 그 안의 객체를 포함한 동영상을 만들 수 있습니다. 여러 이미지를 재료로 업로드하고, 이를 모두 포함한 고품질의 동영상을 생성하세요."
    RETURN_TYPES = ("VIDEO",)

    def api_call(
        self,
        prompt_text: str,
        negative_prompt: str,
        seed: int,
        resolution: str,
        duration: int,
        ingredients_mode: str,
        aspect_ratio: float,
        image_ingredient_1: Optional[torch.Tensor] = None,
        image_ingredient_2: Optional[torch.Tensor] = None,
        image_ingredient_3: Optional[torch.Tensor] = None,
        image_ingredient_4: Optional[torch.Tensor] = None,
        image_ingredient_5: Optional[torch.Tensor] = None,
        auth_token: Optional[str] = None,
    ) -> tuple[VideoFromFile]:
        """Pika Scenes 2.2 API 호출."""
        all_image_bytes_io = []
        for image in [
            image_ingredient_1,
            image_ingredient_2,
            image_ingredient_3,
            image_ingredient_4,
            image_ingredient_5,
        ]:
            if image is not None:
                image_bytes_io = tensor_to_bytesio(image)
                image_bytes_io.seek(0)
                all_image_bytes_io.append(image_bytes_io)

        # 멀티파트 업로드를 위한 파일 데이터 준비
        pika_files = [
            ("images", (f"image_{i}.png", image_bytes_io, "image/png"))
            for i, image_bytes_io in enumerate(all_image_bytes_io)
        ]

        # Pydantic 모델을 사용해 파일이 아닌 데이터 준비
        pika_request_data = PikaBodyGenerate22C2vGenerate22PikascenesPost(
            ingredientsMode=ingredients_mode,
            promptText=prompt_text,
            negativePrompt=negative_prompt,
            seed=seed,
            resolution=resolution,
            duration=duration,
            aspectRatio=aspect_ratio,
        )

        initial_operation = SynchronousOperation(
            endpoint=ApiEndpoint(
                path=PATH_PIKASCENES,
                method=HttpMethod.POST,
                request_model=PikaBodyGenerate22C2vGenerate22PikascenesPost,
                response_model=PikaGenerateResponse,
            ),
            request=pika_request_data,
            files=pika_files,
            content_type="multipart/form-data",
            auth_token=auth_token,
        )

        return self.execute_task(initial_operation, auth_token)
``` 
