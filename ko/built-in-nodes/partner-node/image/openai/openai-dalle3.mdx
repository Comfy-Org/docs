![ComfyUI 네이티브 OpenAI DALL·E 3 노드](/images/built-in-nodes/api_nodes/openai/openai-dall-e-3.jpg)

이 노드는 OpenAI의 DALL·E 3 API에 연결되어, 사용자가 상세한 텍스트 프롬프트를 통해 고품질 이미지를 생성할 수 있도록 합니다. DALL·E 3는 OpenAI의 이미지 생성 모델로, 이전 버전에 비해 이미지 품질이 크게 향상되고 프롬프트 이해도가 더 정확하며 세부 묘사 능력이 뛰어납니다.

## 매개변수

### 입력 매개변수
| 매개변수 | 유형 | 기본값 | 설명 |
|-----|-----|-------|------|
| prompt | 문자열 | "" | 생성할 내용을 설명하는 상세한 텍스트 프롬프트 |
| seed | 정수 | 0 | 최종 결과는 시드와 관련이 없으며, 이 매개변수는 재실행 여부만 결정합니다 |
| quality | 선택 | "standard" | 이미지 품질, 옵션: "standard" 또는 "hd" |
| style | 선택 | "natural" | 시각적 스타일, 옵션: "natural" 또는 "vivid". "Vivid"는 모델이 보다 초현실적이고 극적인 이미지를 생성하도록 유도하며, "natural"은 보다 사실적이고 덜 초현실적인 이미지를 생성합니다 |
| size | 선택 | "1024x1024" | 출력 이미지 크기, 옵션: "1024x1024", "1024x1792" 또는 "1792x1024" |

### 출력 매개변수
| 출력 | 유형 | 설명 |
|-----|-----|------|
| IMAGE | 이미지 | 생성된 이미지 결과 |

## 소스 코드

[노드 소스 코드 (2025-05-03 업데이트)]

```python

class OpenAIDalle3(ComfyNodeABC):
    """
    OpenAI의 DALL·E 3 엔드포인트를 통해 동기식으로 이미지를 생성합니다.

    /proxy/openai/images/generations 경유 프록시를 사용합니다. 반환된 URL은 단기 유효하므로, 결과를 유지해야 한다면 다운로드하거나 캐시하세요.
    """

    def __init__(self):
        pass

    @classmethod
    def INPUT_TYPES(cls) -> InputTypeDict:
        return {
            "required": {
                "prompt": (
                    IO.STRING,
                    {
                        "multiline": True,
                        "default": "",
                        "tooltip": "DALL·E용 텍스트 프롬프트",
                    },
                ),
            },
            "optional": {
                "seed": (
                    IO.INT,
                    {
                        "default": 0,
                        "min": 0,
                        "max": 2**31 - 1,
                        "step": 1,
                        "display": "number",
                        "control_after_generate": True,
                        "tooltip": "백엔드에서 아직 구현되지 않았습니다",
                    },
                ),
                "quality": (
                    IO.COMBO,
                    {
                        "options": ["standard", "hd"],
                        "default": "standard",
                        "tooltip": "이미지 품질",
                    },
                ),
                "style": (
                    IO.COMBO,
                    {
                        "options": ["natural", "vivid"],
                        "default": "natural",
                        "tooltip": "Vivid는 모델이 초현실적이고 극적인 이미지를 생성하도록 유도합니다. Natural은 보다 자연스럽고 덜 초현실적으로 보이는 이미지를 생성합니다.",
                    },
                ),
                "size": (
                    IO.COMBO,
                    {
                        "options": ["1024x1024", "1024x1792", "1792x1024"],
                        "default": "1024x1024",
                        "tooltip": "이미지 크기",
                    },
                ),
            },
            "hidden": {"auth_token": "AUTH_TOKEN_COMFY_ORG"},
        }

    RETURN_TYPES = (IO.IMAGE,)
    FUNCTION = "api_call"
    CATEGORY = "api node/image/openai"
    DESCRIPTION = cleandoc(__doc__ or "")
    API_NODE = True

    def api_call(
        self,
        prompt,
        seed=0,
        style="natural",
        quality="standard",
        size="1024x1024",
        auth_token=None,
    ):
        model = "dall-e-3"

        # 작업 구성
        operation = SynchronousOperation(
            endpoint=ApiEndpoint(
                path="/proxy/openai/images/generations",
                method=HttpMethod.POST,
                request_model=OpenAIImageGenerationRequest,
                response_model=OpenAIImageGenerationResponse,
            ),
            request=OpenAIImageGenerationRequest(
                model=model,
                prompt=prompt,
                quality=quality,
                size=size,
                style=style,
                seed=seed,
            ),
            auth_token=auth_token,
        )

        response = operation.execute()

        img_tensor = validate_and_cast_response(response)
        return (img_tensor,)
``` 
