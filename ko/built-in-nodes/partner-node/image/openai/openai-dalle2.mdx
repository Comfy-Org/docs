OpenAI DALL·E 2 노드를 사용하면 OpenAI의 DALL·E 2 API를 통해 텍스트 설명을 기반으로 창의적인 이미지를 생성할 수 있습니다.

## 매개변수

### 기본 매개변수

| 매개변수 | 유형   | 기본값    | 설명                                         |
| --------- | ------ | ---------- | --------------------------------------------------- |
| 프롬프트    | 문자열 | ""         | DALL·E가 이미지를 생성하기 위한 텍스트 프롬프트, 다중 줄 입력 지원 |
| 시드      | 정수   | 0          | 결과는 실제로 시드와 관련이 없으며, 이 매개변수는 재실행 여부만 결정합니다 |
| 크기      | 선택   | "1024x1024"| 출력 이미지 크기, 옵션: 256x256, 512x512, 1024x1024 |
| n         | 정수   | 1          | 생성할 이미지 개수, 범위 1~8             |

### 선택적 매개변수

| 매개변수 | 유형  | 기본값 | 설명                                   |
| --------- | ----- | ------- | --------------------------------------------- |
| 이미지     | 이미지 | 없음    | 이미지 편집을 위한 선택적 참조 이미지    |
| 마스크      | 마스크  | 없음    | 인페인팅을 위한 선택적 마스크 (흰색 영역은 교체됨) |

### 출력

| 출력 | 유형  | 설명        |
| ------ | ----- | ------------------ |
| 이미지  | 이미지 | 생성된 이미지(들) |

## 특징

- 기본 기능: 텍스트 프롬프트로부터 이미지 생성
- 이미지 편집: 이미지와 마스크 매개변수가 모두 제공되면 이미지 편집 수행 (흰색 마스크 영역은 교체됨)

## 소스 코드

[노드 소스 코드 (2025-05-03 업데이트)]

```python

class OpenAIDalle2(ComfyNodeABC):
    """
    OpenAI의 DALL·E 2 엔드포인트를 통해 동기적으로 이미지를 생성합니다.

    /proxy/openai/images/generations 경유로 프록시를 사용합니다. 반환된 URL은 단기 유효하므로, 결과를 유지해야 한다면 다운로드하거나 캐시하세요.
    """

    def __init__(self):
        pass

    @classmethod
    def INPUT_TYPES(cls) -> InputTypeDict:
        return {
            "required": {
                "prompt": (
                    IO.STRING,
                    {
                        "multiline": True,
                        "default": "",
                        "tooltip": "DALL·E를 위한 텍스트 프롬프트",
                    },
                ),
            },
            "optional": {
                "seed": (
                    IO.INT,
                    {
                        "default": 0,
                        "min": 0,
                        "max": 2**31 - 1,
                        "step": 1,
                        "display": "number",
                        "control_after_generate": True,
                        "tooltip": "백엔드에서 아직 구현되지 않았습니다",
                    },
                ),
                "size": (
                    IO.COMBO,
                    {
                        "options": ["256x256", "512x512", "1024x1024"],
                        "default": "1024x1024",
                        "tooltip": "이미지 크기",
                    },
                ),
                "n": (
                    IO.INT,
                    {
                        "default": 1,
                        "min": 1,
                        "max": 8,
                        "step": 1,
                        "display": "number",
                        "tooltip": "생성할 이미지 개수",
                    },
                ),
                "image": (
                    IO.IMAGE,
                    {
                        "default": None,
                        "tooltip": "이미지 편집을 위한 선택적 참조 이미지.",
                    },
                ),
                "mask": (
                    IO.MASK,
                    {
                        "default": None,
                        "tooltip": "인페인팅을 위한 선택적 마스크 (흰색 영역은 교체됨)",
                    },
                ),
            },
            "hidden": {"auth_token": "AUTH_TOKEN_COMFY_ORG"},
        }

    RETURN_TYPES = (IO.IMAGE,)
    FUNCTION = "api_call"
    CATEGORY = "api node/image/openai"
    DESCRIPTION = cleandoc(__doc__ or "")
    API_NODE = True

    def api_call(
        self,
        prompt,
        seed=0,
        image=None,
        mask=None,
        n=1,
        size="1024x1024",
        auth_token=None,
    ):
        model = "dall-e-2"
        path = "/proxy/openai/images/generations"
        content_type = "application/json"
        request_class = OpenAIImageGenerationRequest
        img_binary = None

        if image is not None and mask is not None:
            path = "/proxy/openai/images/edits"
            content_type = "multipart/form-data"
            request_class = OpenAIImageEditRequest

            input_tensor = image.squeeze().cpu()
            height, width, channels = input_tensor.shape
            rgba_tensor = torch.ones(height, width, 4, device="cpu")
            rgba_tensor[:, :, :channels] = input_tensor

            if mask.shape[1:] != image.shape[1:-1]:
                raise Exception("마스크와 이미지는 반드시 같은 크기여야 합니다")
            rgba_tensor[:, :, 3] = 1 - mask.squeeze().cpu()

            rgba_tensor = downscale_image_tensor(rgba_tensor.unsqueeze(0)).squeeze()

            image_np = (rgba_tensor.numpy() * 255).astype(np.uint8)
            img = Image.fromarray(image_np)
            img_byte_arr = io.BytesIO()
            img.save(img_byte_arr, format="PNG")
            img_byte_arr.seek(0)
            img_binary = img_byte_arr  # .getvalue()
            img_binary.name = "image.png"
        elif image is not None or mask is not None:
            raise Exception("Dall-E 2 이미지 편집을 위해서는 이미지와 마스크가 모두 필요합니다")

        # 작업 구축
        operation = SynchronousOperation(
            endpoint=ApiEndpoint(
                path=path,
                method=HttpMethod.POST,
                request_model=request_class,
                response_model=OpenAIImageGenerationResponse,
            ),
            request=request_class(
                model=model,
                prompt=prompt,
                n=n,
                size=size,
                seed=seed,
            ),
            files=(
                {
                    "image": img_binary,
                }
                if img_binary
                else None
            ),
            content_type=content_type,
            auth_token=auth_token,
        )

        response = operation.execute()

        img_tensor = validate_and_cast_response(response)
        return (img_tensor,)
``` 
