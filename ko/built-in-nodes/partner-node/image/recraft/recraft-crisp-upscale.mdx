Recraft 크리스프 업스케일 노드는 Recraft의 API를 사용해 이미지의 해상도와 선명도를 개선합니다.

## 매개변수

### 기본 매개변수

| 매개변수 | 유형  | 기본값 | 설명                |
| --------- | ----- | ------- | -------------------------- |
| image     | 이미지 | -       | 업스케일할 입력 이미지 |

### 출력

| 출력 | 유형  | 설명                        |
| ------ | ----- | ---------------------------------- |
| IMAGE  | 이미지 | 업스케일되고 향상된 출력 이미지 |

## 소스 코드

[노드 소스 코드 (2025-05-03 업데이트)]

```python
class RecraftCrispUpscaleNode:
    """
    이미지를 동기적으로 업스케일합니다.
    주어진 래스터 이미지를 '크리스프 업스케일' 도구를 사용해 향상시켜 이미지 해상도를 높이고, 더 선명하고 깨끗하게 만듭니다.
    """

    RETURN_TYPES = (IO.IMAGE,)
    DESCRIPTION = cleandoc(__doc__ or "")  # 잠재적인 None 값 처리
    FUNCTION = "api_call"
    API_NODE = True
    CATEGORY = "api node/image/Recraft"

    RECRAFT_PATH = "/proxy/recraft/images/crispUpscale"

    @classmethod
    def INPUT_TYPES(s):
        return {
            "required": {
                "image": (IO.IMAGE, ),
            },
            "optional": {
            },
            "hidden": {
                "auth_token": "AUTH_TOKEN_COMFY_ORG",
            },
        }

    def api_call(
        self,
        image: torch.Tensor,
        auth_token=None,
        **kwargs,
    ):
        images = []
        total = image.shape[0]
        pbar = ProgressBar(total)
        for i in range(total):
            sub_bytes = handle_recraft_file_request(
                image=image[i],
                path=self.RECRAFT_PATH,
                auth_token=auth_token,
            )
            images.append(torch.cat([bytesio_to_image_tensor(x) for x in sub_bytes], dim=0))
            pbar.update(1)

        images_tensor = torch.cat(images, dim=0)
        return (images_tensor,)
```
