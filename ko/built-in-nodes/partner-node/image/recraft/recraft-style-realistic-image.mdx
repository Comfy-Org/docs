Recraft 스타일 - 사실적인 이미지 노드는 Recraft 이미지 생성에 사실적인 사진 스타일을 설정하도록 돕습니다. 생성된 이미지의 시각적 특성을 제어할 수 있는 다양한 하위 스타일 옵션이 제공됩니다.

## 노드 기능

이 노드는 Recraft의 이미지 생성 과정을 사실적인 사진 효과로 유도하는 스타일 구성 객체를 생성합니다.

## 매개변수

### 기본 매개변수

| 매개변수 | 타입   | 기본값 | 설명                                     |
| --------- | ------ | ------- | ----------------------------------------------- |
| substyle  | 선택   | 없음    | 특정한 사실적인 사진의 하위 스타일 (필수) |

### 출력

| 출력        | 타입          | 설명                                                |
| ------------- | ------------- | ---------------------------------------------------------- |
| recraft_style | Recraft 스타일 | Recraft 생성 노드에 연결할 스타일 구성 객체 |

```python

class RecraftStyleV3RealisticImageNode:
    """
    사실적인 이미지 스타일과 선택적 하위 스타일을 선택하세요.
    """

    RETURN_TYPES = (RecraftIO.STYLEV3,)
    RETURN_NAMES = ("recraft_style",)
    DESCRIPTION = cleandoc(__doc__ or "")  # 잠재적인 None 값 처리
    FUNCTION = "create_style"
    CATEGORY = "api node/image/Recraft"

    RECRAFT_STYLE = RecraftStyleV3.realistic_image

    @classmethod
    def INPUT_TYPES(s):
        return {
            "required": {
                "substyle": (get_v3_substyles(s.RECRAFT_STYLE),),
            }
        }

    def create_style(self, substyle: str):
        if substyle == "None":
            substyle = None
        return (RecraftStyle(self.RECRAFT_STYLE, substyle),)

```
