![ComfyUI 내장 Recraft 이미지 벡터화 노드](/images/built-in-nodes/api_nodes/recraft/recraft-vectorize-image.jpg)

Recraft 이미지 벡터화 노드는 Recraft의 API를 사용해 래스터 이미지(사진, PNG 또는 JPEG 등)를 벡터 SVG 형식으로 변환합니다.

## 매개변수

### 기본 매개변수

| 매개변수 | 유형  | 기본값 | 설명                           |
| --------- | ----- | ------- | ------------------------------------- |
| image     | 이미지 | -       | 벡터로 변환할 입력 이미지 |

### 출력

| 출력 | 유형   | 설명                                                                 |
| ------ | ------ | --------------------------------------------------------------------------- |
| SVG    | 벡터 | 변환된 SVG 벡터 그래픽, SaveSVG 노드에 연결하여 저장해야 합니다 |

[노드 소스 코드 (2025-05-03 업데이트)]

```python

class RecraftVectorizeImageNode:
    """
    입력 이미지로부터 SVG를 동기적으로 생성합니다.
    """

    RETURN_TYPES = (RecraftIO.SVG,)
    DESCRIPTION = cleandoc(__doc__ or "")  # 잠재적인 None 값 처리
    FUNCTION = "api_call"
    API_NODE = True
    CATEGORY = "api node/image/Recraft"

    @classmethod
    def INPUT_TYPES(s):
        return {
            "required": {
                "image": (IO.IMAGE, ),
            },
            "optional": {
            },
            "hidden": {
                "auth_token": "AUTH_TOKEN_COMFY_ORG",
            },
        }

    def api_call(
        self,
        image: torch.Tensor,
        auth_token=None,
        **kwargs,
    ):
        svgs = []
        total = image.shape[0]
        pbar = ProgressBar(total)
        for i in range(total):
            sub_bytes = handle_recraft_file_request(
                image=image[i],
                path="/proxy/recraft/images/vectorize",
                auth_token=auth_token,
            )
            svgs.append(SVG(sub_bytes))
            pbar.update(1)

        return (SVG.combine_all(svgs), )
```
