![ComfyUI 기본 Recraft Color RGB 노드](/images/built-in-nodes/api_nodes/recraft/recraft-color-rgb.jpg)
Recraft Color RGB 노드를 사용하면 정확한 RGB 색상 값을 정의하여 Recraft 이미지 생성에서 색상을 제어할 수 있습니다.

## 노드 기능

이 노드는 Recraft Controls 노드에 연결되는 색상 구성 객체를 생성하며, 생성된 이미지에 사용될 색상을 지정합니다.

## 매개변수

### 기본 매개변수

| 매개변수 | 타입    | 기본값 | 설명           |
| --------- | ------- | ------- | --------------------- |
| r         | 정수    | 0       | 빨간색 채널 (0-255)   |
| g         | 정수    | 0       | 초록색 채널 (0-255)   |
| b         | 정수    | 0       | 파란색 채널 (0-255)   |

### 출력

| 출력        | 타입          | 설명                                        |
| ------------- | ------------- | -------------------------------------------------- |
| recraft_color | Recraft Color | Recraft Controls에 연결할 색상 구성 객체 |

## 소스 코드

[노드 소스 코드 (2025-05-03 업데이트)]

```python
class RecraftColorRGBNode:
    """
    특정 RGB 값을 선택하여 Recraft Color를 생성합니다.
    """

    RETURN_TYPES = (RecraftIO.COLOR,)
    DESCRIPTION = cleandoc(__doc__ or "")  # None 값 처리
    RETURN_NAMES = ("recraft_color",)
    FUNCTION = "create_color"
    CATEGORY = "api node/image/Recraft"

    @classmethod
    def INPUT_TYPES(s):
        return {
            "required": {
                "r": (IO.INT, {
                    "default": 0,
                    "min": 0,
                    "max": 255,
                    "tooltip": "색상의 빨간색 값."
                }),
                "g": (IO.INT, {
                    "default": 0,
                    "min": 0,
                    "max": 255,
                    "tooltip": "색상의 초록색 값."
                }),
                "b": (IO.INT, {
                    "default": 0,
                    "min": 0,
                    "max": 255,
                    "tooltip": "색상의 파란색 값."
                }),
            },
            "optional": {
                "recraft_color": (RecraftIO.COLOR,),
            }
        }

    def create_color(self, r: int, g: int, b: int, recraft_color: RecraftColorChain=None):
        recraft_color = recraft_color.clone() if recraft_color else RecraftColorChain()
        recraft_color.add(RecraftColor(r, g, b))
        return (recraft_color, )
```
