Recraft 이미지에서 이미지로 다시 만들기 노드는 Recraft의 API를 사용해 참조 이미지와 텍스트 프롬프트를 기반으로 새로운 이미지를 생성합니다.

## 매개변수

### 기본 매개변수

| 매개변수 | 유형    | 기본값 | 설명                              |
| --------- | ------- | ------- | ---------------------------------------- |
| image     | 이미지   | -       | 참조 이미지 입력                    |
| prompt    | 문자열  | ""      | 생성된 이미지에 대한 텍스트 설명 |
| n         | 정수    | 1       | 생성할 이미지 수 (1~6)               |
| seed      | 정수    | 0       | 난수 시드 값                        |

### 선택적 매개변수

| 매개변수        | 유형             | 설명                           |
| ---------------- | ---------------- | ------------------------------------- |
| recraft_style    | Recraft 스타일    | 생성된 이미지의 스타일 설정   |
| negative_prompt  | 문자열           | 생성된 이미지에서 피해야 할 요소 |
| recraft_controls | Recraft 컨트롤    | 색상과 같은 추가 컨트롤       |

### 출력

| 출력 | 유형  | 설명            |
| ------ | ----- | ---------------------- |
| IMAGE  | 이미지 | 생성된 이미지 결과 |


## 소스 코드

[노드 소스 코드 (2025-05-03 업데이트)]
```python

class RecraftImageToImageNode:
    """
    프롬프트와 강도를 기반으로 이미지를 수정합니다.
    """

    RETURN_TYPES = (IO.IMAGE,)
    DESCRIPTION = cleandoc(__doc__ or "")  # 잠재적인 None 값 처리
    FUNCTION = "api_call"
    API_NODE = True
    CATEGORY = "api node/image/Recraft"

    @classmethod
    def INPUT_TYPES(s):
        return {
            "required": {
                "image": (IO.IMAGE, ),
                "prompt": (
                    IO.STRING,
                    {
                        "multiline": True,
                        "default": "",
                        "tooltip": "이미지 생성을 위한 프롬프트.",
                    },
                ),
                "n": (
                    IO.INT,
                    {
                        "default": 1,
                        "min": 1,
                        "max": 6,
                        "tooltip": "생성할 이미지의 수.",
                    },
                ),
                "strength": (
                    IO.FLOAT,
                    {
                        "default": 0.5,
                        "min": 0.0,
                        "max": 1.0,
                        "step": 0.01,
                        "tooltip": "원본 이미지와의 차이를 정의하며, [0, 1] 범위 내에 있어야 합니다. 0은 거의 동일한 것을 의미하고, 1은 매우 비슷하지 않은 것을 의미합니다."
                    }
                ),
                "seed": (
                    IO.INT,
                    {
                        "default": 0,
                        "min": 0,
                        "max": 0xFFFFFFFFFFFFFFFF,
                        "control_after_generate": True,
                        "tooltip": "노드가 다시 실행될지 여부를 결정하는 시드; 실제 결과는 시드와 상관없이 비결정적입니다.",
                    },
                ),
            },
            "optional": {
                "recraft_style": (RecraftIO.STYLEV3,),
                "negative_prompt": (
                    IO.STRING,
                    {
                        "default": "",
                        "forceInput": True,
                        "tooltip": "이미지에서 원치 않는 요소에 대한 선택적 텍스트 설명.",
                    },
                ),
                "recraft_controls": (
                    RecraftIO.CONTROLS,
                    {
                        "tooltip": "Recraft Controls 노드를 통해 생성 과정에 추가적인 컨트롤 옵션 제공."
                    },
                ),
            },
            "hidden": {
                "auth_token": "AUTH_TOKEN_COMFY_ORG",
            },
        }

    def api_call(
        self,
        image: torch.Tensor,
        prompt: str,
        n: int,
        strength: float,
        seed,
        auth_token=None,
        recraft_style: RecraftStyle = None,
        negative_prompt: str = None,
        recraft_controls: RecraftControls = None,
        **kwargs,
    ):
        default_style = RecraftStyle(RecraftStyleV3.realistic_image)
        if recraft_style is None:
            recraft_style = default_style

        controls_api = None
        if recraft_controls:
            controls_api = recraft_controls.create_api_model()

        if not negative_prompt:
            negative_prompt = None

        request = RecraftImageGenerationRequest(
            prompt=prompt,
            negative_prompt=negative_prompt,
            model=RecraftModel.recraftv3,
            n=n,
            strength=round(strength, 2),
            style=recraft_style.style,
            substyle=recraft_style.substyle,
            style_id=recraft_style.style_id,
            controls=controls_api,
            random_seed=seed,
        )

        images = []
        total = image.shape[0]
        pbar = ProgressBar(total)
        for i in range(total):
            sub_bytes = handle_recraft_file_request(
                image=image[i],
                path="/proxy/recraft/images/imageToImage",
                request=request,
                auth_token=auth_token,
            )
            images.append(torch.cat([bytesio_to_image_tensor(x) for x in sub_bytes], dim=0))
            pbar.update(1)

        images_tensor = torch.cat(images, dim=0)
        return (images_tensor, )
```
