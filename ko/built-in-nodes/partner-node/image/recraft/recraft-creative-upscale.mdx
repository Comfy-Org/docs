![ComfyUI 네이티브 Recraft Creative Upscale 노드](/images/built-in-nodes/api_nodes/recraft/recraft-creative-upscale-image.jpg)

Recraft Creative Upscale 노드는 Recraft의 API를 사용해 이미지의 해상도를 높이는 동시에 이미지의 디테일을 창의적으로 향상하고 풍부하게 만듭니다.

## 매개변수

### 기본 매개변수

| 매개변수 | 유형  | 기본값 | 설명                           |
| --------- | ----- | ------- | ------------------------------------- |
| image     | 이미지 | -       | 창의적으로 업스케일링할 입력 이미지 |

### 출력

| 출력 | 유형  | 설명                                    |
| ------ | ----- | ---------------------------------------------- |
| IMAGE  | 이미지 | 창의적 업스케일링 후 고해상도 이미지 |

## 소스 코드

[노드 소스 코드 (2025-05-03 업데이트)]

```python
class RecraftCreativeUpscaleNode(RecraftCrispUpscaleNode):
    """
    이미지를 동기식으로 업스케일링합니다.
    주어진 래스터 이미지를 '창의적 업스케일링' 도구를 사용해 향상시키며, 작은 디테일과 얼굴을 중점적으로 정교화하여 해상도를 높입니다.
    """

    RETURN_TYPES = (IO.IMAGE,)
    DESCRIPTION = cleandoc(__doc__ or "")  # None 값 처리
    FUNCTION = "api_call"
    API_NODE = True
    CATEGORY = "api node/image/Recraft"

    RECRAFT_PATH = "/proxy/recraft/images/creativeUpscale"
```
