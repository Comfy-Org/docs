Recraft 컨트롤 노드를 사용하면 컨트롤 매개변수(색상 및 배경색 등)를 정의하여 Recraft의 이미지 생성 과정을 안내할 수 있습니다. 이 노드는 여러 제어 입력을 통합된 제어 객체로 결합합니다.

## 매개변수

### 선택적 매개변수

| 매개변수        | 유형          | 설명                         |
| ---------------- | ------------- | ----------------------------------- |
| colors           | Recraft Color | 이미지 생성을 위한 색상 제어 |
| background_color | Recraft Color | 배경색 제어            |

### 출력

| 출력           | 유형             | 설명                                        |
| ---------------- | ---------------- | -------------------------------------------------- |
| recraft_controls | Recraft Controls | Recraft 생성 노드용 제어 구성 객체 |

## 작동 방식

노드 처리 과정:

1. 입력 제어 매개변수(색상 및 배경색) 수집
2. 이러한 매개변수를 구조화된 제어 객체로 결합
3. Recraft 생성 노드에 연결하기 위한 이 제어 객체 출력

Recraft 생성 노드에 연결되면, 이 제어 매개변수들이 AI 생성 과정에 영향을 미칩니다. AI는 텍스트 프롬프트의 의미적 내용 외에도 다양한 요소를 고려합니다. 색상 입력이 설정된 경우, AI는 생성된 이미지에 적절히 해당 색상을 사용하려고 합니다.

```python
class RecraftControlsNode:
    """
    Recraft 컨트롤을 만들어 Recraft 생성을 맞춤화합니다.
    """

    RETURN_TYPES = (RecraftIO.CONTROLS,)
    RETURN_NAMES = ("recraft_controls",)
    DESCRIPTION = cleandoc(__doc__ or "")  # None 값 처리
    FUNCTION = "create_controls"
    CATEGORY = "api node/image/Recraft"

    @classmethod
    def INPUT_TYPES(s):
        return {
            "required": {
            },
            "optional": {
                "colors": (RecraftIO.COLOR,),
                "background_color": (RecraftIO.COLOR,),
            }
        }

    def create_controls(self, colors: RecraftColorChain=None, background_color: RecraftColorChain=None):
        return (RecraftControls(colors=colors, background_color=background_color), )
```
