이 노드는 Recraft의 이미지 생성 과정을 전문적인 로고 디자인 효과로 이끌어주는 스타일 구성 객체를 생성합니다. 다양한 하위 스타일을 선택함으로써, 생성되는 로고의 디자인 스타일, 복잡성 및 사용 사례를 정의할 수 있습니다.

## 매개변수

### 기본 매개변수

| 매개변수 | 유형      | 기본값 | 설명                                  |
| --------- | --------- | ------- | -------------------------------------------- |
| substyle  | 선택     | -       | 로고 래스터의 특정 하위 스타일 (필수) |

### 출력

| 출력        | 유형          | 설명                                                     |
| ------------- | ------------- | --------------------------------------------------------------- |
| recraft_style | Recraft 스타일 | 스타일 구성 객체, Recraft 생성 노드에 연결됨 |

## 소스 코드

[노드 소스 코드 (2025-05-03 업데이트)]

```python
class RecraftStyleV3LogoRasterNode(RecraftStyleV3RealisticImageNode):
    """
    벡터 일러스트레이션 스타일과 선택적 하위 스타일을 선택하세요.
    """

    @classmethod
    def INPUT_TYPES(s):
        return {
            "required": {
                "substyle": (get_v3_substyles(s.RECRAFT_STYLE, include_none=False),),
            }
        }

    RECRAFT_STYLE = RecraftStyleV3.logo_raster
```
