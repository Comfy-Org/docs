![ComfyUI 기본 Recraft 배경 제거 노드](/images/built-in-nodes/api_nodes/recraft/recraft-remove-background.jpg)

Recraft 배경 제거 노드는 Recraft의 API를 사용해 이미지의 배경을 지능적으로 감지하고 제거하며, 투명한 배경과 해당 알파 마스크를 포함한 이미지를 생성합니다.

## 매개변수

### 기본 매개변수

| 매개변수 | 유형  | 기본값 | 설명                           |
| --------- | ----- | ------- | ------------------------------------- |
| image     | image | -       | 배경을 제거할 입력 이미지 |

### 출력

| 출력 | 유형  | 설명                                          |
| ------ | ----- | ---------------------------------------------------- |
| IMAGE  | image | 배경이 제거된 이미지 (알파 채널 포함)   |
| MASK   | mask  | 주요 피사체의 마스크 (흰색 영역은 보존됨) |

## 소스 코드

[노드 소스 코드 (2025-05-03 업데이트)]

```python
class RecraftRemoveBackgroundNode:
    """
    이미지에서 배경을 제거하고 처리된 이미지와 마스크를 반환합니다.
    """

    RETURN_TYPES = (IO.IMAGE, IO.MASK)
    DESCRIPTION = cleandoc(__doc__ or "")  # 잠재적인 None 값 처리
    FUNCTION = "api_call"
    API_NODE = True
    CATEGORY = "api node/image/Recraft"

    @classmethod
    def INPUT_TYPES(s):
        return {
            "required": {
                "image": (IO.IMAGE, ),
            },
            "optional": {
            },
            "hidden": {
                "auth_token": "AUTH_TOKEN_COMFY_ORG",
            },
        }

    def api_call(
        self,
        image: torch.Tensor,
        auth_token=None,
        **kwargs,
    ):
        images = []
        total = image.shape[0]
        pbar = ProgressBar(total)
        for i in range(total):
            sub_bytes = handle_recraft_file_request(
                image=image[i],
                path="/proxy/recraft/images/removeBackground",
                auth_token=auth_token,
            )
            images.append(torch.cat([bytesio_to_image_tensor(x) for x in sub_bytes], dim=0))
            pbar.update(1)

        images_tensor = torch.cat(images, dim=0)
        # 알파 채널을 마스크로 사용하며, B,H,W 형식으로 반환합니다.
        masks_tensor = images_tensor[:,:,:,-1:].squeeze(-1)
        return (images_tensor, masks_tensor)

```
