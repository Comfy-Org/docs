![ComfyUI 기본 Luma 텍스트 이미지 노드](/images/built-in-nodes/api_nodes/luma/luma-text-to-image.jpg)

Luma 텍스트 이미지 노드는 Luma AI의 고급 이미지 생성 기능을 활용해 텍스트 설명을 통해 매우 사실적이고 예술적인 이미지를 생성할 수 있게 해줍니다.

## 노드 기능

이 노드는 Luma AI의 텍스트-이미지 API와 연결되어 사용자가 상세한 텍스트 프롬프트를 통해 이미지를 생성하도록 합니다. Luma AI는 뛰어난 사실성과 디테일 표현으로 유명하며, 특히 포토리얼리즘 콘텐츠와 아트 스타일 이미지를 잘 만들어냅니다.

## 매개변수

### 기본 매개변수

| 매개변수          | 타입    | 기본값 | 설명                                                                                                         |
| ------------------ | ------- | ------- | ------------------------------------------------------------------------------------------------------------------- |
| prompt             | 문자열  | ""      | 생성할 내용을 설명하는 텍스트 프롬프트                                                                      |
| model              | 선택    | -       | 사용할 생성 모델을 선택하세요                                                                                |
| 종횡비           | 선택    | 16:9    | 출력 이미지의 종횡비 설정                                                                                 |
| 시드               | 정수    | 0       | 노드가 다시 실행될지 여부를 결정하는 시드 값이며, 실제 결과는 시드에 의존하지 않습니다                   |
| 스타일 이미지 가중치 | 부동소수점 | 1.0     | 스타일 이미지 가중치, 범위 0.0~1.0, 스타일 이미지가 제공된 경우에만 적용되며, 값이 높을수록 스타일 참조 강도가 높아집니다 |

### 옵션 매개변수

| 매개변수       | 타입     | 설명                                                                              |
| --------------- | -------- | ---------------------------------------------------------------------------------------- |
| image_luma_ref  | LUMA_REF | Luma 참조 노드 연결을 통해 입력 이미지로 생성 과정에 영향을 줌; 최대 4개 이미지 가능 |
| 스타일 이미지     | 이미지    | 스타일 참조 이미지; 단 1개 이미지만 사용됩니다                                         |
| 캐릭터 이미지     | 이미지    | 캐릭터 참조 이미지; 여러 개의 배치로 가능하며 최대 4개 이미지 가능                   |

### 출력

| 출력 | 타입  | 설명            |
| ------ | ----- | ---------------------- |
| IMAGE  | 이미지 | 생성된 이미지 결과 |

## 작동 방식

Luma 텍스트 이미지 노드는 사용자가 제공한 텍스트 프롬프트를 분석하고, Luma AI의 생성 모델을 통해 해당 이미지를 생성합니다. 이 과정은 딥러닝 기술을 사용해 텍스트 설명을 이해하고 이를 시각적 표현으로 변환합니다. 사용자는 해상도, 가이던스 스케일, 네거티브 프롬프트 등 다양한 매개변수를 조정해 생성 과정을 세밀하게 조정할 수 있습니다.

또한, 노드는 참조 이미지와 컨셉 가이던스를 사용해 생성 결과에 추가적인 영향을 미칠 수 있어 창작자들이 더욱 정확히 자신들의 창의적 비전을 구현할 수 있도록 돕습니다.

```python

class LumaImageGenerationNode(ComfyNodeABC):
    """
    프롬프트와 종횡비를 기반으로 동기적으로 이미지를 생성합니다.
    """

    RETURN_TYPES = (IO.IMAGE,)
    DESCRIPTION = cleandoc(__doc__ or "")  # 잠재적인 None 값 처리
    FUNCTION = "api_call"
    API_NODE = True
    CATEGORY = "api node/image/Luma"

    @classmethod
    def INPUT_TYPES(s):
        return {
            "required": {
                "prompt": (
                    IO.STRING,
                    {
                        "multiline": True,
                        "default": "",
                        "tooltip": "이미지 생성을 위한 프롬프트",
                    },
                ),
                "model": ([model.value for model in LumaImageModel],),
                "aspect_ratio": (
                    [ratio.value for ratio in LumaAspectRatio],
                    {
                        "default": LumaAspectRatio.ratio_16_9,
                    },
                ),
                "seed": (
                    IO.INT,
                    {
                        "default": 0,
                        "min": 0,
                        "max": 0xFFFFFFFFFFFFFFFF,
                        "control_after_generate": True,
                        "tooltip": "노드가 다시 실행될지 여부를 결정하는 시드; 실제 결과는 시드에 관계없이 비결정적입니다.",
                    },
                ),
                "style_image_weight": (
                    IO.FLOAT,
                    {
                        "default": 1.0,
                        "min": 0.0,
                        "max": 1.0,
                        "step": 0.01,
                        "tooltip": "스타일 이미지의 가중치. 스타일 이미지가 제공되지 않으면 무시됩니다.",
                    },
                ),
            },
            "optional": {
                "image_luma_ref": (
                    LumaIO.LUMA_REF,
                    {
                        "tooltip": "Luma 참조 노드 연결을 통해 입력 이미지로 생성 과정에 영향을 줌; 최대 4개 이미지 고려 가능."
                    },
                ),
                "style_image": (
                    IO.IMAGE,
                    {"tooltip": "스타일 참조 이미지; 단 1개 이미지만 사용됩니다."},
                ),
                "character_image": (
                    IO.IMAGE,
                    {
                        "tooltip": "캐릭터 참조 이미지; 여러 개의 배치로 가능하며 최대 4개 이미지 고려 가능."
                    },
                ),
            },
            "hidden": {
                "auth_token": "AUTH_TOKEN_COMFY_ORG",
            },
        }

    def api_call(
        self,
        prompt: str,
        model: str,
        aspect_ratio: str,
        seed,
        style_image_weight: float,
        image_luma_ref: LumaReferenceChain = None,
        style_image: torch.Tensor = None,
        character_image: torch.Tensor = None,
        auth_token=None,
        **kwargs,
    ):
        # image_luma_ref 처리
        api_image_ref = None
        if image_luma_ref is not None:
            api_image_ref = self._convert_luma_refs(
                image_luma_ref, max_refs=4, auth_token=auth_token
            )
        # style_luma_ref 처리
        api_style_ref = None
        if style_image is not None:
            api_style_ref = self._convert_style_image(
                style_image, weight=style_image_weight, auth_token=auth_token
            )
        # character_ref 이미지 처리
        character_ref = None
        if character_image is not None:
            download_urls = upload_images_to_comfyapi(
                character_image, max_images=4, auth_token=auth_token
            )
            character_ref = LumaCharacterRef(
                identity0=LumaImageIdentity(images=download_urls)
            )

        operation = SynchronousOperation(
            endpoint=ApiEndpoint(
                path="/proxy/luma/generations/image",
                method=HttpMethod.POST,
                request_model=LumaImageGenerationRequest,
                response_model=LumaGeneration,
            ),
            request=LumaImageGenerationRequest(
                prompt=prompt,
                model=model,
                aspect_ratio=aspect_ratio,
                image_ref=api_image_ref,
                style_ref=api_style_ref,
                character_ref=character_ref,
            ),
            auth_token=auth_token,
        )
        response_api: LumaGeneration = operation.execute()

        operation = PollingOperation(
            poll_endpoint=ApiEndpoint(
                path=f"/proxy/luma/generations/{response_api.id}",
                method=HttpMethod.GET,
                request_model=EmptyRequest,
                response_model=LumaGeneration,
            ),
            completed_statuses=[LumaState.completed],
            failed_statuses=[LumaState.failed],
            status_extractor=lambda x: x.state,
            auth_token=auth_token,
        )
        response_poll = operation.execute()

        img_response = requests.get(response_poll.assets.image)
        img = process_image_response(img_response)
        return (img,)

    def _convert_luma_refs(
        self, luma_ref: LumaReferenceChain, max_refs: int, auth_token=None
    ):
        luma_urls = []
        ref_count = 0
        for ref in luma_ref.refs:
            download_urls = upload_images_to_comfyapi(
                ref.image, max_images=1, auth_token=auth_token
            )
            luma_urls.append(download_urls[0])
            ref_count += 1
            if ref_count >= max_refs:
                break
        return luma_ref.create_api_model(download_urls=luma_urls, max_refs=max_refs)

    def _convert_style_image(
        self, style_image: torch.Tensor, weight: float, auth_token=None
    ):
        chain = LumaReferenceChain(
            first_ref=LumaReference(image=style_image, weight=weight)
        )
        return self._convert_luma_refs(chain, max_refs=1, auth_token=auth_token)
```
