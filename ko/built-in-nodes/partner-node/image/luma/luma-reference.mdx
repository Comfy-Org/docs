![ComfyUI 내장 Luma 참조 노드](/images/built-in-nodes/api_nodes/luma/luma-reference.jpg)

Luma 참조 노드는 참조 이미지와 가중치를 설정하여 Luma 이미지 생성 노드의 생성 과정을 안내하며, 생성된 이미지를 참조 이미지의 특정 특징에 더욱 가깝게 만듭니다.

## 노드 기능

이 노드는 Luma 생성 노드를 위한 도우미 도구로 작동하며, 사용자가 참조 이미지를 제공해 생성 결과에 영향을 줄 수 있도록 합니다. 또한 참조 이미지의 가중치를 설정해 최종 결과에 얼마나 영향을 미칠지 조절할 수 있습니다.

여러 개의 Luma 참조 노드를 체인으로 연결할 수 있으며, API 요구 사항에 따라 최대 4개가 동시에 작동합니다.

## 매개변수

### 기본 매개변수

| 매개변수 | 유형 | 기본값 | 설명 |
| ------ | ------ | ------ | ---------------------------- |
| image | 이미지 | - | 참조용으로 사용할 입력 이미지 |
| weight | 부동소수점 | 1.0 | 참조 이미지의 영향 강도를 제어하는 값 (0-1) |

### 출력

| 출력 | 유형 | 설명 |
| -------- | -------- | ------------------------ |
| luma_ref | LUMA_REF | 이미지와 가중치를 포함한 참조 객체 |

## 작동 방식

Luma 참조 노드는 이미지 입력을 받고 가중치 값을 설정할 수 있습니다. 이 노드는 직접 이미지를 생성하거나 수정하지 않으며, 이미지 데이터와 가중치 정보를 포함한 참조 객체를 생성해 Luma 생성 노드에 전달합니다.

생성 과정에서 Luma AI는 참조 이미지의 특징을 분석하고, 설정된 가중치에 따라 이러한 특징을 생성 결과에 반영합니다. 가중치 값이 높을수록 생성된 이미지가 참조 이미지의 특징에 더욱 가까워지고, 낮을수록 참조 이미지가 최종 결과에 약간만 영향을 미칩니다.

## 소스 코드

```python

class LumaReferenceNode(ComfyNodeABC):
    """
    Luma Generate Image 노드와 함께 사용하기 위한 이미지와 가중치를 보관합니다.
    """

    RETURN_TYPES = (LumaIO.LUMA_REF,)
    RETURN_NAMES = ("luma_ref",)
    DESCRIPTION = cleandoc(__doc__ or "")  # 잠재적인 None 값 처리
    FUNCTION = "create_luma_reference"
    CATEGORY = "api node/image/Luma"

    @classmethod
    def INPUT_TYPES(s):
        return {
            "required": {
                "image": (
                    IO.IMAGE,
                    {
                        "tooltip": "참조용으로 사용할 이미지.",
                    },
                ),
                "weight": (
                    IO.FLOAT,
                    {
                        "default": 1.0,
                        "min": 0.0,
                        "max": 1.0,
                        "step": 0.01,
                        "tooltip": "이미지 참조의 가중치.",
                    },
                ),
            },
            "optional": {"luma_ref": (LumaIO.LUMA_REF,)},
        }

    def create_luma_reference(
        self, image: torch.Tensor, weight: float, luma_ref: LumaReferenceChain = None
    ):
        if luma_ref is not None:
            luma_ref = luma_ref.clone()
        else:
            luma_ref = LumaReferenceChain()
        luma_ref.add(LumaReference(image=image, weight=round(weight, 2)))
        return (luma_ref,)

```
