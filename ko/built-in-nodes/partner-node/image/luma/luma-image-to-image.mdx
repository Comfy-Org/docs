![ComfyUI 내장 Luma 이미지에서 이미지로 노드](/images/built-in-nodes/api_nodes/luma/luma-image-to-image.jpg)

Luma 이미지에서 이미지로 노드는 텍스트 프롬프트를 기반으로 Luma AI 기술을 사용해 기존 이미지를 수정할 수 있게 해줍니다. 이 과정에서 원본 이미지의 특정 특징과 구조는 유지됩니다.

## 노드 기능

이 노드는 Luma AI의 텍스트-이미지 API에 연결되어 사용자가 상세한 텍스트 프롬프트를 통해 이미지를 생성할 수 있도록 합니다. Luma AI는 뛰어난 사실성과 디테일로 유명하며, 특히 포토리얼리즘 콘텐츠와 아트 스타일 이미지 생성에 뛰어납니다.

## 매개변수

### 기본 매개변수

| 매개변수          | 타입    | 기본값 | 설명                                                                            |
| ------------------ | ------- | ------- | -------------------------------------------------------------------------------------- |
| prompt             | 문자열  | ""      | 생성할 내용을 설명하는 텍스트 프롬프트                                         |
| model              | 선택    | -       | 사용할 생성 모델을 선택하세요                                                   |
| 종횡비           | 선택    | 16:9    | 출력 이미지의 종횡비 설정                                                       |
| 시드               | 정수    | 0       | 노드가 다시 실행될지 여부를 결정하는 시드 값, 실제 결과는 시드에 의존하지 않음  |
| 스타일 이미지 가중치 | 실수    | 1.0     | 스타일 이미지의 가중치, 범위는 0.02~1.0이며, style_image가 제공된 경우에만 유효합니다 |

### 옵션 매개변수
다음 매개변수 입력이 없으면 노드는 텍스트-이미지 모드로 작동합니다.

| 매개변수       | 타입     | 설명                                                                                                               |
| --------------- | -------- | ------------------------------------------------------------------------------------------------------------------------- |
| image_luma_ref  | LUMA_REF | Luma 참조 노드 연결, 입력 이미지를 통해 생성 결과에 영향을 미치며 최대 4개의 이미지를 고려할 수 있습니다           |
| 스타일 이미지     | 이미지    | 스타일 참조 이미지, 단 1개의 이미지만 사용하며 생성된 이미지의 스타일에 영향을 주며 `style_image_weight`를 통해 조정됩니다 |
| 캐릭터 이미지    | 이미지    | 생성된 결과에 캐릭터 특징을 추가하며, 여러 이미지의 배치도 가능하며 최대 4개의 이미지를 지원합니다                       |

### 출력

| 출력 | 타입  | 설명         |
| ------ | ----- | ------------------- |
| IMAGE  | 이미지 | 생성된 이미지 |

## 작동 방식

Luma 이미지에서 이미지로 노드는 입력 이미지를 분석하고 이를 텍스트 프롬프트와 결합해 수정 과정을 안내합니다. Luma AI의 생성 모델을 사용해 프롬프트에 따라 이미지에 창의적인 변화를 줍니다.

노드 처리 과정:

1. 먼저 입력 이미지를 ComfyAPI에 업로드합니다.
2. 그 후 이미지 URL과 프롬프트를 Luma API에 전송합니다.
3. Luma AI가 처리를 완료할 때까지 기다립니다.
4. 생성된 이미지를 다운로드하여 반환합니다.

image_weight 매개변수는 원본 이미지의 영향 정도를 제어하며, 0에 가까울수록 원본 이미지의 특징이 더 많이 유지되고, 1에 가까울수록 보다 큰 수정이 가능합니다.

## 소스 코드

```python

class LumaImageModifyNode(ComfyNodeABC):
    """
    프롬프트와 종횡비를 기반으로 이미지를 동기적으로 수정합니다.
    """

    RETURN_TYPES = (IO.IMAGE,)
    DESCRIPTION = cleandoc(__doc__ or "")  # None 값 처리
    FUNCTION = "api_call"
    API_NODE = True
    CATEGORY = "api node/image/Luma"

    @classmethod
    def INPUT_TYPES(s):
        return {
            "required": {
                "image": (IO.IMAGE,),
                "prompt": (
                    IO.STRING,
                    {
                        "multiline": True,
                        "default": "",
                        "tooltip": "이미지 생성을 위한 프롬프트",
                    },
                ),
                "image_weight": (
                    IO.FLOAT,
                    {
                        "default": 1.0,
                        "min": 0.02,
                        "max": 1.0,
                        "step": 0.01,
                        "tooltip": "이미지의 가중치; 0.0에 가까울수록 이미지가 덜 수정됩니다.",
                    },
                ),
                "model": ([model.value for model in LumaImageModel],),
                "seed": (
                    IO.INT,
                    {
                        "default": 0,
                        "min": 0,
                        "max": 0xFFFFFFFFFFFFFFFF,
                        "control_after_generate": True,
                        "tooltip": "노드가 다시 실행될지 여부를 결정하는 시드; 실제 결과는 시드에 관계없이 비결정적입니다.",
                    },
                ),
            },
            "optional": {},
            "hidden": {
                "auth_token": "AUTH_TOKEN_COMFY_ORG",
            },
        }

    def api_call(
        self,
        prompt: str,
        model: str,
        image: torch.Tensor,
        image_weight: float,
        seed,
        auth_token=None,
        **kwargs,
    ):
        # 첫째, 이미지를 업로드합니다.
        download_urls = upload_images_to_comfyapi(
            image, max_images=1, auth_token=auth_token
        )
        image_url = download_urls[0]
        # 다음으로 다운로드 URL을 제공해 Luma 호출을 수행합니다.
        operation = SynchronousOperation(
            endpoint=ApiEndpoint(
                path="/proxy/luma/generations/image",
                method=HttpMethod.POST,
                request_model=LumaImageGenerationRequest,
                response_model=LumaGeneration,
            ),
            request=LumaImageGenerationRequest(
                prompt=prompt,
                model=model,
                modify_image_ref=LumaModifyImageRef(
                    url=image_url, weight=round(image_weight, 2)
                ),
            ),
            auth_token=auth_token,
        )
        response_api: LumaGeneration = operation.execute()

        operation = PollingOperation(
            poll_endpoint=ApiEndpoint(
                path=f"/proxy/luma/generations/{response_api.id}",
                method=HttpMethod.GET,
                request_model=EmptyRequest,
                response_model=LumaGeneration,
            ),
            completed_statuses=[LumaState.completed],
            failed_statuses=[LumaState.failed],
            status_extractor=lambda x: x.state,
            auth_token=auth_token,
        )
        response_poll = operation.execute()

        img_response = requests.get(response_poll.assets.image)
        img = process_image_response(img_response)
        return (img,)
```
