![ComfyUI 네이티브 Ideogram V3 노드](/images/built-in-nodes/api_nodes/ideogram/ideogram-v3.jpg)

이 노드는 Ideogram V3 API에 연결하여 이미지 생성 작업을 수행합니다.

현재 이 노드는 두 가지 이미지 생성 모드를 지원합니다:
- **텍스트-to-이미지 모드** - 텍스트 프롬프트로부터 새로운 이미지를 생성
- **인페인팅 모드** - 원본 이미지와 마스크를 제공해 특정 영역을 재생성

### 텍스트-to-이미지 모드
이 모드는 기본 모드로, 이미지나 마스크 입력이 제공되지 않을 때 활성화됩니다. 간단히 프롬프트와 원하는 매개변수를 제공하세요:

1. 프롬프트 필드에 원하는 이미지 내용을 설명하세요.
2. 적절한 가로세로 비율이나 해상도를 선택하세요.
3. 매직 프롬프트, 시드, 렌더링 품질과 같은 다른 매개변수를 조정하세요.
4. 노드를 실행해 이미지를 생성하세요.

### 인페인팅 모드
**중요 참고사항**: 이 모드는 이미지와 마스크 모두가 필요합니다. 둘 중 하나만 제공되면 노드는 오류를 발생시킵니다.

1. 원본 이미지를 `image` 입력 포트에 연결하세요.
2. 원본 이미지와 동일한 크기의 마스크를 생성하세요. 여기서 흰색 부분은 재생성할 영역을 나타냅니다.
3. 마스크를 `mask` 입력 포트에 연결하세요.
4. 마스크로 덮인 영역에서 무엇을 생성하고 싶은지 프롬프트에 설명하세요.
5. 노드를 실행해 로컬 편집을 수행하세요.

## 매개변수 설명

### 기본 매개변수

| 매개변수           | 유형   | 기본값    | 설명                                 |
|---------------------|--------|------------|---------------------------------------------|
| prompt              | 문자열 | ""         | 생성할 내용을 설명하는 텍스트 프롬프트 |
| aspect_ratio        | 콤보   | "1:1"      | 이미지 가로세로 비율 (텍스트-to-이미지 모드 전용) |
| resolution          | 콤보   | "자동"     | 이미지 해상도, 설정 시 가로세로 비율 무시됨 |
| magic_prompt_option | 콤보   | "AUTO"     | 매직 프롬프트 강화: AUTO, ON 또는 OFF  |
| seed                | 정수   | 0          | 난수 시드 값, 0은 임의 생성 |
| num_images          | 정수   | 1          | 생성할 이미지 수 (1~8)          |
| rendering_speed     | 콤보   | "밸런스"   | 렌더링 속도: 밸런스, 터보 또는 품질 |

### 옵션 매개변수

| 매개변수 | 유형  | 설명                                                     |
|-----------|-------|-----------------------------------------------------------------|
| image     | 이미지 | 인페인팅 모드용 입력 이미지 (**마스크와 함께 제공해야 함**) |
| mask      | 마스크  | 인페인팅용 마스크, 흰색 영역은 교체됩니다 (**이미지와 함께 제공해야 함**) |

### 출력

| 출력 | 유형  | 설명       |
|------|-------|------------|
| IMAGE | 이미지 | 생성된 이미지   |

## 작동 방식

Ideogram V3 노드는 최첨단 AI 모델을 사용해 사용자 입력을 처리하며, 복잡한 디자인 의도와 텍스트 레이아웃 요구사항을 이해할 수 있습니다. 두 가지 주요 모드를 지원합니다:

1. **생성 모드**: 텍스트 프롬프트로부터 새로운 이미지를 생성
2. **편집 모드**: 원본 이미지 + 마스크 조합을 사용해 마스크로 지정된 영역만 교체

## 소스 코드

[노드 소스 코드 (2025-05-03 업데이트)]

```python

class IdeogramV3(ComfyNodeABC):
    """
    Ideogram V3 모델을 사용해 동기적으로 이미지를 생성합니다.

    텍스트 프롬프트로부터 일반적인 이미지 생성과 마스크를 이용한 이미지 편집을 모두 지원합니다.
    이미지 링크는 일정 기간 동안만 사용 가능하며, 이미지를 보관하려면 다운로드해야 합니다.
    """

    def __init__(self):
        pass

    @classmethod
    def INPUT_TYPES(cls) -> InputTypeDict:
        return {
            "required": {
                "prompt": (
                    IO.STRING,
                    {
                        "multiline": True,
                        "default": "",
                        "tooltip": "이미지 생성 또는 편집을 위한 프롬프트",
                    },
                ),
            },
            "optional": {
                "image": (
                    IO.IMAGE,
                    {
                        "default": None,
                        "tooltip": "이미지 편집용 선택적 참조 이미지.",
                    },
                ),
                "mask": (
                    IO.MASK,
                    {
                        "default": None,
                        "tooltip": "인페인팅용 선택적 마스크 (흰색 영역은 교체됨)",
                    },
                ),
                "aspect_ratio": (
                    IO.COMBO,
                    {
                        "options": list(V3_RATIO_MAP.keys()),
                        "default": "1:1",
                        "tooltip": "이미지 생성 시 가로세로 비율. 해상도를 자동으로 설정하지 않으면 무시됨.",
                    },
                ),
                "resolution": (
                    IO.COMBO,
                    {
                        "options": V3_RESOLUTIONS,
                        "default": "자동",
                        "tooltip": "이미지 생성 시 해상도. 자동으로 설정하지 않으면 가로세로 비율 설정을 무시함.",
                    },
                ),
                "magic_prompt_option": (
                    IO.COMBO,
                    {
                        "options": ["AUTO", "ON", "OFF"],
                        "default": "AUTO",
                        "tooltip": "생성 시 매직프롬프트 사용 여부 결정",
                    },
                ),
                "seed": (
                    IO.INT,
                    {
                        "default": 0,
                        "min": 0,
                        "max": 2147483647,
                        "step": 1,
                        "control_after_generate": True,
                        "display": "number",
                    },
                ),
                "num_images": (
                    IO.INT,
                    {"default": 1, "min": 1, "max": 8, "step": 1, "display": "number"},
                ),
                "rendering_speed": (
                    IO.COMBO,
                    {
                        "options": ["BALANCED", "TURBO", "QUALITY"],
                        "default": "BALANCED",
                        "tooltip": "생성 속도와 품질 간의 균형 조절",
                    },
                ),
            },
            "hidden": {"auth_token": "AUTH_TOKEN_COMFY_ORG"},
        }

    RETURN_TYPES = (IO.IMAGE,)
    FUNCTION = "api_call"
    CATEGORY = "api node/image/ideogram/v3"
    DESCRIPTION = cleandoc(__doc__ or "")
    API_NODE = True

    def api_call(
        self,
        prompt,
        image=None,
        mask=None,
        resolution="자동",
        aspect_ratio="1:1",
        magic_prompt_option="AUTO",
        seed=0,
        num_images=1,
        rendering_speed="BALANCED",
        auth_token=None,
    ):
        # 편집 모드에서 이미지와 마스크가 모두 제공되었는지 확인
        if image is not None and mask is not None:
            # 편집 모드
            path = "/proxy/ideogram/ideogram-v3/edit"

            # 이미지와 마스크 처리
            input_tensor = image.squeeze().cpu()

            # 마스크 크기가 이미지와 일치하는지 검증
            if mask.shape[1:] != image.shape[1:-1]:
                raise Exception("마스크와 이미지는 반드시 동일한 크기여야 합니다")

            # 이미지 처리
            img_np = (input_tensor.numpy() * 255).astype(np.uint8)
            img = Image.fromarray(img_np)
            img_byte_arr = io.BytesIO()
            img.save(img_byte_arr, format="PNG")
            img_byte_arr.seek(0)
            img_binary = img_byte_arr
            img_binary.name = "image.png"

            # 마스크 처리 - 흰색 영역은 교체됨
            mask_np = (mask.squeeze().cpu().numpy() * 255).astype(np.uint8)
            mask_img = Image.fromarray(mask_np)
            mask_byte_arr = io.BytesIO()
            mask_img.save(mask_byte_arr, format="PNG")
            mask_byte_arr.seek(0)
            mask_binary = mask_byte_arr
            mask_binary.name = "mask.png"

            # 편집 요청 생성
            edit_request = IdeogramV3EditRequest(
                prompt=prompt,
                rendering_speed=rendering_speed,
            )

            # 옵션 매개변수 추가
            if magic_prompt_option != "AUTO":
                edit_request.magic_prompt = magic_prompt_option
            if seed != 0:
                edit_request.seed = seed
            if num_images > 1:
                edit_request.num_images = num_images

            # 편집 모드의 작업 실행
            operation = SynchronousOperation(
                endpoint=ApiEndpoint(
                    path=path,
                    method=HttpMethod.POST,
                    request_model=IdeogramV3EditRequest,
                    response_model=IdeogramGenerateResponse,
                ),
                request=edit_request,
                files={
                    "image": img_binary,
                    "mask": mask_binary,
                },
                content_type="multipart/form-data",
                auth_token=auth_token,
            )

        elif image is not None or mask is not None:
            # 이미지나 마스크 중 하나만 제공됐다면 오류 발생
            raise Exception("Ideogram V3 이미지 편집은 이미지와 마스크가 모두 필요합니다")
        else:
            # 생성 모드
            path = "/proxy/ideogram/ideogram-v3/generate"

            # 생성 요청 생성
            gen_request = IdeogramV3Request(
                prompt=prompt,
                rendering_speed=rendering_speed,
            )

            # 해상도 vs 가로세로 비율 처리
            if resolution != "자동":
                gen_request.resolution = resolution
            elif aspect_ratio != "1:1":
                v3_aspect = V3_RATIO_MAP.get(aspect_ratio)
                if v3_aspect:
                    gen_request.aspect_ratio = v3_aspect

            # 옵션 매개변수 추가
            if magic_prompt_option != "AUTO":
                gen_request.magic_prompt = magic_prompt_option
            if seed != 0:
                gen_request.seed = seed
            if num_images > 1:
                gen_request.num_images = num_images

            # 생성 모드의 작업 실행
            operation = SynchronousOperation(
                endpoint=ApiEndpoint(
                    path=path,
                    method=HttpMethod.POST,
                    request_model=IdeogramV3Request,
                    response_model=IdeogramGenerateResponse,
                ),
                request=gen_request,
                auth_token=auth_token,
            )

        # 작업 실행 및 응답 처리
        response = operation.execute()

        if not response.data or len(response.data) == 0:
            raise Exception("응답에서 이미지가 생성되지 않았습니다")

        image_urls = [image_data.url for image_data in response.data if image_data.url]

        if not image_urls:
            raise Exception("응답에서 이미지 URL이 생성되지 않았습니다")

        return (download_and_process_images(image_urls),)
```
