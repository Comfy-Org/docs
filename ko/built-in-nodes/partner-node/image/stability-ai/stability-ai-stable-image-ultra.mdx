![ComfyUI 네이티브 Stability Stable Image Ultra 노드](/images/built-in-nodes/api_nodes/stability-ai/stability-ai-stable-image-ultra.jpg)

Stability Stable Image Ultra 노드는 Stability AI의 Stable Diffusion Ultra API를 사용해 고품질 이미지를 생성합니다. 텍스트에서 이미지로, 이미지에서 이미지로의 생성을 모두 지원하며, 텍스트 프롬프트로부터 세밀하고 예술적인 비주얼을 만들어냅니다.

## 매개변수

### 필수 매개변수

| 매개변수    | 유형    | 기본값 | 설명                                                                                                                                                                                                                                                                                                               |
| ------------ | ------- | ------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| prompt       | 문자열  | ""      | 생성하고자 하는 내용에 대한 텍스트 설명입니다. 명확하고 상세한 프롬프트를 사용하면 요소, 색상 및 주제를 명확히 지정하여 더 나은 결과를 얻을 수 있습니다. `(단어:가중치)` 형식을 사용해 단어의 중요도를 조절할 수 있으며, 여기서 가중치는 0~1입니다. 예: `The sky was (blue:0.3) and (green:0.8)`는 하늘을 파란색보다 녹색으로 더 강조합니다. |
| aspect_ratio | 선택   | "1:1"   | 출력 이미지의 가로세로 비율                                                                                                                                                                                                                                                                                     |
| style_preset | 선택   | "None"  | 생성된 이미지에 적용할 옵션 스타일                                                                                                                                                                                                                                                                                 |
| seed         | 정수    | 0       | 노이즈 생성을 위한 난수 시드 (0~4294967294)                                                                                                                                                                                                                                                                           |

### 선택적 매개변수

| 매개변수       | 유형   | 기본값 | 설명                                                                                                      |
| --------------- | ------ | ------- | ---------------------------------------------------------------------------------------------------------------- |
| image           | 이미지  | -       | 이미지에서 이미지로 생성할 때 입력 이미지                                                                 |
| negative_prompt | 문자열 | ""      | 출력 이미지에 포함되지 않기를 원하는 내용을 설명합니다. 이는 고급 기능입니다                                   |
| image_denoise   | 실수  | 0.5     | 입력 이미지의 노이즈 제거 강도 (0.0~1.0). 0.0은 입력 이미지를 그대로 유지하며, 1.0은 입력 이미지가 없는 것과 같습니다 |

### 출력

| 출력 | 유형  | 설명     |
| ------ | ----- | ---------- |
| IMAGE  | 이미지 | 생성된 이미지 |

## 참고사항

- image_denoise는 입력 이미지가 제공되지 않을 경우 아무런 효과가 없습니다.
- style_preset이 "None"일 경우 어떤 스타일도 적용되지 않습니다.
- 이미지에서 이미지로 생성할 때는 입력 이미지를 API 제출 전에 적합한 형식으로 변환합니다.

```python

class StabilityStableImageUltraNode:
    """
    프롬프트와 해상도를 기반으로 동기적으로 이미지를 생성합니다.
    """

    RETURN_TYPES = (IO.IMAGE,)
    DESCRIPTION = cleandoc(__doc__ or "")  # None 값 처리
    FUNCTION = "api_call"
    API_NODE = True
    CATEGORY = "api node/image/stability"

    @classmethod
    def INPUT_TYPES(s):
        return {
            "required": {
                "prompt": (
                    IO.STRING,
                    {
                        "multiline": True,
                        "default": "",
                        "tooltip": "출력 이미지에서 보고 싶은 내용입니다. 강력하고 상세한 프롬프트를 사용해" +
                                    "출력 이미지에서 보고 싶은 내용을 명확히 정의하세요. 강력하고 상세한 프롬프트를 사용해" +
                                    "요소, 색상 및 주제를 명확히 정의하면 더 나은 결과를 얻을 수 있습니다. " +
                                    "특정 단어의 가중치를 조절하려면 `(단어:가중치)` 형식을 사용하세요," +
                                    "여기서 `단어`는 가중치를 조절하고자 하는 단어이고 `가중치`는 0과 1 사이의 값입니다. 예: `The sky was a crisp (blue:0.3) and (green:0.8)`" +
                                    "는 하늘이 파랗고 녹색이지만, 녹색이 파란색보다 더 강조된다는 의미입니다."
                    },
                ),
                "aspect_ratio": ([x.value for x in StabilityAspectRatio],
                    {
                        "default": StabilityAspectRatio.ratio_1_1,
                        "tooltip": "생성된 이미지의 가로세로 비율.",
                    },
                ),
                "style_preset": (get_stability_style_presets(),
                    {
                        "tooltip": "생성된 이미지에 원하는 스타일을 선택하세요.",
                    },
                ),
                "seed": (
                    IO.INT,
                    {
                        "default": 0,
                        "min": 0,
                        "max": 4294967294,
                        "control_after_generate": True,
                        "tooltip": "노이즈 생성에 사용되는 난수 시드.",
                    },
                ),
            },
            "optional": {
                "image": (IO.IMAGE,),
                "negative_prompt": (
                    IO.STRING,
                    {
                        "default": "",
                        "forceInput": True,
                        "tooltip": "출력 이미지에서 보고 싶지 않은 내용을 설명하는 텍스트 블러브입니다. 이는 고급 기능입니다."
                    },
                ),
                "image_denoise": (
                    IO.FLOAT,
                    {
                        "default": 0.5,
                        "min": 0.0,
                        "max": 1.0,
                        "step": 0.01,
                        "tooltip": "입력 이미지의 노이즈 제거 강도; 0.0은 입력 이미지와 동일하게, 1.0은 입력 이미지가 전혀 없을 때와 같습니다.",
                    },
                ),
            },
            "hidden": {
                "auth_token": "AUTH_TOKEN_COMFY_ORG",
            },
        }

    def api_call(self, prompt: str, aspect_ratio: str, style_preset: str, seed: int,
                 negative_prompt: str=None, image: torch.Tensor = None, image_denoise: float=None,
                 auth_token=None):
        # 이미지가 있을 경우 이미지 바이너리를 준비
        image_binary = None
        if image is not None:
            image_binary = tensor_to_bytesio(image, 1504 * 1504).read()
        else:
            image_denoise = None

        if not negative_prompt:
            negative_prompt = None
        if style_preset == "None":
            style_preset = None

        files = {
            "image": image_binary
        }

        operation = SynchronousOperation(
            endpoint=ApiEndpoint(
                path="/proxy/stability/v2beta/stable-image/generate/ultra",
                method=HttpMethod.POST,
                request_model=StabilityStableUltraRequest,
                response_model=StabilityStableUltraResponse,
            ),
            request=StabilityStableUltraRequest(
                prompt=prompt,
                negative_prompt=negative_prompt,
                aspect_ratio=aspect_ratio,
                seed=seed,
                strength=image_denoise,
                style_preset=style_preset,
            ),
            files=files,
            content_type="multipart/form-data",
            auth_token=auth_token,
        )
        response_api = operation.execute()

        if response_api.finish_reason != "SUCCESS":
            raise Exception(f"Stable Image Ultra 생성 실패: {response_api.finish_reason}.")

        image_data = base64.b64decode(response_api.image)
        returned_image = bytesio_to_image_tensor(BytesIO(image_data))

        return (returned_image,)
```
