Stability AI Stable Diffusion 3.5 이미지 노드는 Stability AI의 Stable Diffusion 3.5 API를 사용해 고품질 이미지를 생성합니다. 텍스트-to-image 및 이미지-to-image 생성을 모두 지원하며, 텍스트 프롬프트로부터 세밀한 시각적 콘텐츠를 만들어낼 수 있습니다.

## 매개변수

### 필수 매개변수

| 매개변수    | 유형    | 기본값 | 설명                                                                                                                                       |
| ------------ | ------- | ------- | ------------------------------------------------------------------------------------------------------------------------------------------------- |
| prompt       | 문자열  | ""      | 출력 이미지에서 보고 싶은 내용입니다. 요소, 색상, 주제를 명확히 정의하는 강력하고 구체적인 프롬프트가 더 나은 결과를 가져옵니다.   |
| model        | 선택    | -       | 사용할 Stability SD 3.5 모델을 선택하세요                                                                                                        |
| aspect_ratio | 선택    | "1:1"   | 생성된 이미지의 가로세로 비율                                                                                                          |
| style_preset | 선택    | "None"  | 원하는 이미지에 대한 옵션 스타일 사전 설정                                                                                                       |
| cfg_scale    | 부동소수점 | 4.0     | 확산 과정이 프롬프트 텍스트를 얼마나 엄격하게 따르는지 (높은 값일수록 이미지가 프롬프트에 더 가깝게 유지됩니다). 범위: 1.0 - 10.0, 단계: 0.1 |
| seed         | 정수    | 0       | 노이즈 생성을 위한 임의의 시드(0~4294967294)                                                                                                   |

### 선택적 매개변수

| 매개변수       | 유형   | 기본값 | 설명                                                                                                                                                                             |
| --------------- | ------ | ------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| image           | 이미지  | -       | 입력 이미지. 제공될 경우 노드는 이미지-to-image 모드로 전환됩니다                                                                                                                    |
| negative_prompt | 문자열 | ""      | 출력 이미지에서 보고 싶지 않은 키워드입니다. 이는 고급 기능입니다                                                                                                 |
| image_denoise   | 부동소수점 | 0.5     | 입력 이미지의 노이즈 제거 강도입니다. 0.0은 입력 이미지와 동일한 이미지를 생성하며, 1.0은 이미지를 전혀 제공하지 않은 것과 같습니다. 범위: 0.0 - 1.0, 단계: 0.01. 이미지가 제공될 때만 유효합니다 |

### 출력

| 출력 | 유형  | 설명     |
| ------ | ----- | ---------- |
| IMAGE  | 이미지 | 생성된 이미지 |

## 참고사항

- 입력 이미지를 제공하면 노드는 텍스트-to-image 모드에서 이미지-to-image 모드로 전환됩니다.
- 이미지-to-image 모드에서는 가로세로 비율 매개변수가 무시됩니다.
- 모드 선택은 이미지 제공 여부에 따라 자동으로 전환됩니다:
  - 이미지를 제공하지 않으면 텍스트-to-image 모드
  - 이미지를 제공하면 이미지-to-image 모드
- style_preset을 "None"으로 설정하면 사전 설정된 스타일이 적용되지 않습니다.

```python
class StabilityStableImageSD_3_5Node:
    """
    프롬프트와 해상도를 기반으로 동기적으로 이미지를 생성합니다.
    """

    RETURN_TYPES = (IO.IMAGE,)
    DESCRIPTION = cleandoc(__doc__ or "")  # 잠재적인 None 값 처리
    FUNCTION = "api_call"
    API_NODE = True
    CATEGORY = "api node/image/Stability AI"

    @classmethod
    def INPUT_TYPES(s):
        return {
            "required": {
                "prompt": (
                    IO.STRING,
                    {
                        "multiline": True,
                        "default": "",
                        "tooltip": "출력 이미지에서 보고 싶은 내용입니다. 요소, 색상, 주제를 명확히 정의하는 강력하고 구체적인 프롬프트가 더 나은 결과를 가져옵니다."
                    },
                ),
                "model": ([x.value for x in Stability_SD3_5_Model],),
                "aspect_ratio": ([x.value for x in StabilityAspectRatio],
                    {
                        "default": StabilityAspectRatio.ratio_1_1,
                        "tooltip": "생성된 이미지의 가로세로 비율.",
                    },
                ),
                "style_preset": (get_stability_style_presets(),
                    {
                        "tooltip": "생성된 이미지의 원하는 스타일 (선택사항).",
                    },
                ),
                "cfg_scale": (
                    IO.FLOAT,
                    {
                        "default": 4.0,
                        "min": 1.0,
                        "max": 10.0,
                        "step": 0.1,
                        "tooltip": "확산 과정이 프롬프트 텍스트를 얼마나 엄격하게 따르는지 (높은 값일수록 이미지가 프롬프트에 더 가깝게 유지됩니다)",
                    },
                ),
                "seed": (
                    IO.INT,
                    {
                        "default": 0,
                        "min": 0,
                        "max": 4294967294,
                        "control_after_generate": True,
                        "tooltip": "노이즈 생성에 사용되는 임의의 시드.",
                    },
                ),
            },
            "optional": {
                "image": (IO.IMAGE,),
                "negative_prompt": (
                    IO.STRING,
                    {
                        "default": "",
                        "forceInput": True,
                        "tooltip": "출력 이미지에서 보고 싶지 않은 키워드입니다. 이는 고급 기능입니다."
                    },
                ),
                "image_denoise": (
                    IO.FLOAT,
                    {
                        "default": 0.5,
                        "min": 0.0,
                        "max": 1.0,
                        "step": 0.01,
                        "tooltip": "입력 이미지의 노이즈 제거; 0.0은 입력 이미지와 동일한 이미지를 생성하며, 1.0은 이미지를 전혀 제공하지 않은 것과 같습니다.",
                    },
                ),
            },
            "hidden": {
                "auth_token": "AUTH_TOKEN_COMFY_ORG",
            },
        }

    def api_call(self, model: str, prompt: str, aspect_ratio: str, style_preset: str, seed: int, cfg_scale: float,
                 negative_prompt: str=None, image: torch.Tensor = None, image_denoise: float=None,
                 auth_token=None):
        validate_string(prompt, strip_whitespace=False)
        # 이미지가 있을 경우 이미지 바이너리 준비
        image_binary = None
        mode = Stability_SD3_5_GenerationMode.text_to_image
        if image is not None:
            image_binary = tensor_to_bytesio(image, total_pixels=1504*1504).read()
            mode = Stability_SD3_5_GenerationMode.image_to_image
            aspect_ratio = None
        else:
            image_denoise = None

        if not negative_prompt:
            negative_prompt = None
        if style_preset == "None":
            style_preset = None

        files = {
            "image": image_binary
        }

        operation = SynchronousOperation(
            endpoint=ApiEndpoint(
                path="/proxy/stability/v2beta/stable-image/generate/sd3",
                method=HttpMethod.POST,
                request_model=StabilityStable3_5Request,
                response_model=StabilityStableUltraResponse,
            ),
            request=StabilityStable3_5Request(
                prompt=prompt,
                negative_prompt=negative_prompt,
                aspect_ratio=aspect_ratio,
                seed=seed,
                strength=image_denoise,
                style_preset=style_preset,
                cfg_scale=cfg_scale,
                model=model,
                mode=mode,
            ),
            files=files,
            content_type="multipart/form-data",
            auth_token=auth_token,
        )
        response_api = operation.execute()

        if response_api.finish_reason != "SUCCESS":
            raise Exception(f"Stable Diffusion 3.5 이미지 생성 실패: {response_api.finish_reason}.")

        image_data = base64.b64decode(response_api.image)
        returned_image = bytesio_to_image_tensor(BytesIO(image_data))

        return (returned_image,)
```
