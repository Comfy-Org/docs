Flux 1.1 [pro] 울트라 이미지 노드는 텍스트 프롬프트를 통해 초고해상도 이미지를 생성할 수 있게 해주며, Black Forest Labs의 최신 이미지 생성 API와 직접 연결됩니다.

이 노드는 두 가지 주요 사용 모드를 지원합니다:
1. **텍스트-to-이미지**: 이미지 입력을 사용하지 않을 때 텍스트 프롬프트로부터 고품질 이미지를 생성합니다.
2. **이미지-to-이미지**: 기존 이미지와 프롬프트를 결합해 두 이미지의 특징을 혼합한 새로운 이미지를 만듭니다(리믹스 모드).

이 노드는 API 호출을 통해 울트라 모드를 지원하며, 표준 Flux 1.1 [pro]보다 4배 높은 해상도(최대 4MP)로 이미지를 생성할 수 있습니다. 이 과정에서 프롬프트의 정확성은 전혀 손상되지 않으며, 생성 시간은 단 10초로 매우 빠릅니다. 다른 고해상도 모델과 비교하면 2.5배 이상 빠릅니다.

## 매개변수 설명

### 기본 매개변수

| 매개변수         | 유형    | 기본값 | 설명                                                                                                                                                                                                                      |
| ----------------- | ------- | ------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| prompt            | 문자열  | ""      | 이미지를 생성하기 위한 텍스트 설명                                                                                                                                                                                        |
| prompt_upsampling | 부울    | False   | 세부 사항을 강화하기 위해 프롬프트 업샘플링 기법을 사용할지 여부입니다. 활성화 시 더 창의적인 생성을 위해 자동으로 프롬프트를 수정하지만 결과는 비결정적입니다(같은 시드로도 완벽히 동일한 결과가 나오지 않습니다) |
| seed              | 정수    | 0       | 난수 시드 값으로, 생성의 무작위성을 제어합니다                                                                                                                                                                                |
| aspect_ratio      | 문자열  | "16:9"  | 이미지의 가로세로 비율로, 반드시 1:4에서 4:1 사이여야 합니다                                                                                                                                                                  |
| raw               | 부울    | False   | True로 설정하면 보다 덜 처리된 자연스러운 이미지를 생성합니다                                                                                                                                                          |

### 선택적 매개변수 

| 매개변수             | 유형  | 기본값 | 설명                                                                                                                                               |
| --------------------- | ----- | ------- | --------------------------------------------------------------------------------------------------------------------------------------------------------- |
| image_prompt          | 이미지 | None    | 선택적 입력으로, 이미지-to-이미지(리믹스) 모드에 사용됩니다                                                                                                      |
| image_prompt_strength | 실수  | 0.1     | `image_prompt`가 입력될 경우 활성화되며, 프롬프트와 이미지 프롬프트 간의 혼합 정도를 조정합니다. 값이 높을수록 출력물이 입력 이미지에 더 가까워지며, 범위는 0.0~1.0입니다 |

### 출력

| 출력 | 유형  | 설명                            |
| ------ | ----- | -------------------------------------- |
| IMAGE  | 이미지 | 생성된 고해상도 이미지 결과 |

## 작동 방식

Flux 1.1 [pro] 울트라 모드는 최적화된 딥러닝 아키텍처와 효율적인 GPU 가속 기술을 사용해 속도를 저하시키지 않고 고해상도 이미지를 생성합니다. API로 요청이 전송되면 시스템은 프롬프트를 파싱하고 적절한 매개변수를 적용한 후 병렬로 이미지를 계산한 뒤, 최종적으로 고해상도 결과를 생성해 반환합니다.

일반 모델과 비교했을 때, 울트라 모드는 특히 큰 규모에서도 세부 사항의 보존과 일관성을 중점적으로 다뤄 4MP 고해상도에서도 인상적인 품질을 보장합니다.

```python
class FluxProUltraImageNode(ComfyNodeABC):
    """
    프롬프트와 해상도를 기반으로 동기식으로 이미지를 생성합니다.
    """

    MINIMUM_RATIO = 1 / 4
    MAXIMUM_RATIO = 4 / 1
    MINIMUM_RATIO_STR = "1:4"
    MAXIMUM_RATIO_STR = "4:1"

    @classmethod
    def INPUT_TYPES(s):
        return {
            "required": {
                "prompt": (
                    IO.STRING,
                    {
                        "multiline": True,
                        "default": "",
                        "tooltip": "이미지 생성을 위한 프롬프트",
                    },
                ),
                "prompt_upsampling": (
                    IO.BOOLEAN,
                    {
                        "default": False,
                        "tooltip": "프롬프트를 업샘플링할지 여부입니다. 활성화 시 더 창의적인 생성을 위해 자동으로 프롬프트를 수정하지만 결과는 비결정적입니다(같은 시드로도 완벽히 동일한 결과가 나오지 않습니다).",
                    },
                ),
                "seed": (
                    IO.INT,
                    {
                        "default": 0,
                        "min": 0,
                        "max": 0xFFFFFFFFFFFFFFFF,
                        "control_after_generate": True,
                        "tooltip": "노이즈 생성에 사용되는 난수 시드",
                    },
                ),
                "aspect_ratio": (
                    IO.STRING,
                    {
                        "default": "16:9",
                        "tooltip": "이미지의 가로세로 비율; 반드시 1:4에서 4:1 사이여야 합니다.",
                    },
                ),
                "raw": (
                    IO.BOOLEAN,
                    {
                        "default": False,
                        "tooltip": "True일 경우, 덜 처리된 자연스러운 이미지를 생성합니다.",
                    },
                ),
            },
            "optional": {
                "image_prompt": (IO.IMAGE,),
                "image_prompt_strength": (
                    IO.FLOAT,
                    {
                        "default": 0.1,
                        "min": 0.0,
                        "max": 1.0,
                        "step": 0.01,
                        "tooltip": "프롬프트와 이미지 프롬프트 간의 혼합 정도",
                    },
                ),
            },
            "hidden": {
                "auth_token": "AUTH_TOKEN_COMFY_ORG",
            },
        }

    @classmethod
    def VALIDATE_INPUTS(cls, aspect_ratio: str):
        try:
            validate_aspect_ratio(
                aspect_ratio,
                minimum_ratio=cls.MINIMUM_RATIO,
                maximum_ratio=cls.MAXIMUM_RATIO,
                minimum_ratio_str=cls.MINIMUM_RATIO_STR,
                maximum_ratio_str=cls.MAXIMUM_RATIO_STR,
            )
        except Exception as e:
            return str(e)
        return True

    RETURN_TYPES = (IO.IMAGE,)
    DESCRIPTION = cleandoc(__doc__ or "")  # 잠재적인 None 값 처리
    FUNCTION = "api_call"
    API_NODE = True
    CATEGORY = "api node/image/bfl"

    def api_call(
        self,
        prompt: str,
        aspect_ratio: str,
        prompt_upsampling=False,
        raw=False,
        seed=0,
        image_prompt=None,
        image_prompt_strength=0.1,
        auth_token=None,
        **kwargs,
    ):
        operation = SynchronousOperation(
            endpoint=ApiEndpoint(
                path="/proxy/bfl/flux-pro-1.1-ultra/generate",
                method=HttpMethod.POST,
                request_model=BFLFluxProUltraGenerateRequest,
                response_model=BFLFluxProGenerateResponse,
            ),
            request=BFLFluxProUltraGenerateRequest(
                prompt=prompt,
                prompt_upsampling=prompt_upsampling,
                seed=seed,
                aspect_ratio=validate_aspect_ratio(
                    aspect_ratio,
                    minimum_ratio=self.MINIMUM_RATIO,
                    maximum_ratio=self.MAXIMUM_RATIO,
                    minimum_ratio_str=self.MINIMUM_RATIO_STR,
                    maximum_ratio_str=self.MAXIMUM_RATIO_STR,
                ),
                raw=raw,
                image_prompt=(
                    image_prompt
                    if image_prompt is None
                    else convert_image_to_base64(image_prompt)
                ),
                image_prompt_strength=(
                    None if image_prompt is None else round(image_prompt_strength, 2)
                ),
            ),
            auth_token=auth_token,
        )
        output_image = handle_bfl_synchronous_operation(operation)
        return (output_image,)
```
