> 이 문서는 AI로 생성되었습니다. 오류를 발견하거나 개선할 점이 있다면 언제든지 기여해 주세요! [GitHub에서 편집하기](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/SaveImageDataSetToFolder/en.md)

이 노드는 ComfyUI의 출력 디렉터리 내 지정된 폴더에 이미지 목록을 저장합니다. 여러 이미지를 입력으로 받아 사용자 정의 가능한 파일명 접두사와 함께 디스크에 쓰게 됩니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `images` | IMAGE | 예 | N/A | 저장할 이미지 목록. |
| `folder_name` | STRING | 아니요 | N/A | 이미지를 저장할 폴더 이름 (출력 디렉터리 내). 기본값은 "dataset"입니다. |
| `filename_prefix` | STRING | 아니요 | N/A | 저장된 이미지 파일명의 접두사. 기본값은 "image"입니다. |

**참고:** `images` 입력은 리스트 형식으로, 한번에 여러 이미지를 받고 처리할 수 있습니다. `folder_name`과 `filename_prefix` 매개변수는 스칼라 값이며, 리스트가 연결되면 해당 리스트의 첫 번째 값만 사용됩니다.

## 출력

이 노드에는 출력이 없습니다. 파일 시스템에 저장 작업을 수행하는 출력 노드입니다.
