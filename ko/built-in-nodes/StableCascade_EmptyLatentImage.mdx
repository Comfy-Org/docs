StableCascade_EmptyLatentImage 노드는 Stable Cascade 모델용 빈 잠재 텐서를 생성합니다. 입력 해상도와 압축 설정에 따라 적절한 크기의 두 개의 별도 잠재 표현 — 하나는 단계 C용이고 다른 하나는 단계 B용 — 을 생성합니다. 이 노드는 Stable Cascade 생성 파이프라인의 시작점을 제공합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `width` | INT | 예 | 256부터 MAX_RESOLUTION까지 | 출력 이미지의 너비(픽셀) (기본값: 1024, 증분: 8) |
| `height` | INT | 예 | 256부터 MAX_RESOLUTION까지 | 출력 이미지의 높이(픽셀) (기본값: 1024, 증분: 8) |
| `compression` | INT | 예 | 4부터 128까지 | 단계 C의 잠재 차원을 결정하는 압축 계수 (기본값: 42, 증분: 1) |
| `batch_size` | INT | 아니요 | 1부터 4096까지 | 한 번에 생성할 잠재 샘플의 수 (기본값: 1) |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `stage_c` | LATENT | 차원이 [batch_size, 16, height//compression, width//compression]인 단계 C 잠재 텐서 |
| `stage_b` | LATENT | 차원이 [batch_size, 4, height//4, width//4]인 단계 B 잠재 텐서 |
