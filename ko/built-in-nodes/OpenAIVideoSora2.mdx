OpenAIVideoSora2 노드는 OpenAI의 Sora 모델을 사용하여 동영상을 생성합니다. 텍스트 프롬프트와 선택적 입력 이미지를 기반으로 동영상 콘텐츠를 만들고, 생성된 동영상 출력을 반환합니다. 노드는 선택한 모델에 따라 다양한 동영상 지속시간과 해상도를 지원합니다.

**폐기 안내:** OpenAI는 2026년 9월에 Sora v2 API의 서비스를 중단할 예정입니다. 이 노드는 그 시점에 ComfyUI에서 삭제됩니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `model` | COMBO | 예 | "sora-2"<br />"sora-2-pro" | 동영상 생성에 사용할 OpenAI Sora 모델 (기본값: "sora-2") |
| `prompt` | STRING | 예 | - | 안내용 텍스트; 입력 이미지가 있는 경우 비워둘 수 있음 (기본값: 비워둠) |
| `size` | COMBO | 예 | "720x1280"<br />"1280x720"<br />"1024x1792"<br />"1792x1024" | 생성될 동영상의 해상도 (기본값: "1280x720") |
| `duration` | COMBO | 예 | 4<br />8<br />12 | 생성될 동영상의 지속시간(초) (기본값: 8) |
| `image` | IMAGE | 아니요 | - | 동영상 생성에 사용할 선택적 입력 이미지 |
| `seed` | INT | 아니요 | 0부터 2147483647까지 | 노드가 다시 실행될지 여부를 결정하는 시드; 실제 결과는 시드와 관계없이 비결정적임 (기본값: 0) |

**제약사항 및 한계:**

- "sora-2" 모델은 "720x1280" 및 "1280x720" 해상도만 지원합니다.
- 이미지 매개변수를 사용할 때는 단 하나의 입력 이미지만 지원됩니다.
- 시드 값과 관계없이 결과는 비결정적입니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-----------|-----------|-------------|
| `output` | VIDEO | 생성된 동영상 출력 |
