SplitSigmasDenoise 노드는 노이즈 제거 강도 매개변수를 기반으로 시그마 값의 시퀀스를 두 부분으로 나눕니다. 입력된 시그마 값을 높은 시그마와 낮은 시그마 시퀀스로 분할하며, 분할 지점은 총 단계 수에 노이즈 제거 계수를 곱한 값으로 결정됩니다. 이를 통해 노이즈 스케줄을 서로 다른 강도 범위로 분리하여 특화된 처리를 수행할 수 있습니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `sigmas` | SIGMAS | 예 | - | 노이즈 스케줄을 나타내는 입력 시그마 값의 시퀀스 |
| `denoise` | FLOAT | 예 | 0.0 - 1.0 | 노이즈 제거 강도 계수로, 시그마 시퀀스를 어디서 분할할지 결정합니다(기본값: 1.0) |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-----------|-----------|-------------|
| `high_sigmas` | SIGMAS | 높은 시그마 값을 포함하는 시그마 시퀀스의 첫 번째 부분 |
| `low_sigmas` | SIGMAS | 낮은 시그마 값을 포함하는 시그마 시퀀스의 두 번째 부분 |
