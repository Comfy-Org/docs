> 이 문서는 AI로 생성되었습니다. 오류를 발견하거나 개선할 점이 있다면 언제든지 기여해 주세요! [GitHub에서 편집하기](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/LumaImageToVideoNode/en.md)

텍스트 프롬프트와 선택적 시작/끝 이미지를 기반으로 동기식으로 비디오를 생성합니다. 이 노드는 Luma API를 사용하여 비디오를 생성하며, 프롬프트를 통해 비디오의 내용을 정의하고 선택적으로 첫 번째 및/또는 마지막 프레임을 지정하여 비디오의 구조를 제어할 수 있습니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `prompt` | STRING | 예 | - | 비디오 생성을 위한 프롬프트 (기본값: "") |
| `model` | COMBO | 예 | `"ray-1-6"`<br />`"ray-2"` | 사용 가능한 Luma 모델 중에서 비디오 생성 모델을 선택합니다 |
| `resolution` | COMBO | 예 | `"540p"`<br />`"720p"<br />`"1080p"`<br />`"4k"` | 생성된 비디오의 출력 해상도 (기본값: "540p"). `ray-1-6` 모델을 사용할 경우 이 매개변수는 무시됩니다. |
| `duration` | COMBO | 예 | `"5s"`<br />`"9s"` | 생성된 비디오의 지속 시간. `ray-1-6` 모델을 사용할 경우 이 매개변수는 무시됩니다. |
| `loop` | BOOLEAN | 예 | - | 생성된 비디오가 반복될지 여부 (기본값: False) |
| `seed` | INT | 예 | 0부터 18446744073709551615까지 | 노드가 다시 실행될지 여부를 결정하는 시드. 실제 결과는 시드와 관계없이 비결정적입니다. (기본값: 0) |
| `first_image` | IMAGE | 아니요 | - | 생성된 비디오의 첫 번째 프레임. (선택적) |
| `last_image` | IMAGE | 아니요 | - | 생성된 비디오의 마지막 프레임. (선택적) |
| `luma_concepts` | CUSTOM | 아니요 | - | Luma Concepts 노드를 통해 카메라 움직임을 지정하는 옵션 카메라 개념들. (선택적) |

**참고:** `first_image` 또는 `last_image` 중 적어도 하나는 제공되어야 합니다. 두 매개변수가 모두 없으면 노드는 예외를 발생시킵니다. `model`이 `ray-1-6`로 설정된 경우 `resolution`과 `duration` 매개변수는 무시됩니다.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `output` | VIDEO | 생성된 비디오 파일 |
