ConditioningSetPropertiesAndCombine 노드는 새로운 컨디셔닝 입력에서 제공되는 속성을 기존 컨디셔닝 입력에 적용하여 컨디셔닝 데이터를 수정합니다. 이 노드는 두 컨디셔닝 세트를 결합하면서 새 컨디셔닝의 강도를 조절하고 컨디셔닝 영역이 어떻게 적용될지 지정합니다.

## 입력

| 매개변수 | 데이터 유형 | 입력 유형 | 기본값 | 범위 | 설명 |
|-----------|-----------|------------|---------|-------|-------------|
| `cond` | CONDITIONING | 필수 | - | - | 수정할 원래 컨디셔닝 데이터 |
| `cond_NEW` | CONDITIONING | 필수 | - | - | 적용할 속성을 제공하는 새로운 컨디셔닝 데이터 |
| `strength` | FLOAT | 필수 | 1.0 | 0.0 - 10.0 | 새 컨디셔닝 속성의 강도를 조절합니다 |
| `set_cond_area` | STRING | 필수 | default | ["default", "mask bounds"] | 컨디셔닝 영역이 어떻게 적용될지 결정합니다 |
| `mask` | MASK | 선택적 | - | - | 컨디셔닝을 위한 특정 영역을 정의하는 선택적 마스크 |
| `hooks` | HOOKS | 선택적 | - | - | 맞춤 처리를 위한 선택적 후크 함수 |
| `timesteps` | TIMESTEPS_RANGE | 선택적 | - | - | 컨디셔닝이 적용되는 시점을 제어하는 선택적 타임스텝 범위 |

**참고:** `mask`가 제공될 경우, `set_cond_area` 매개변수는 "mask bounds"를 사용하여 마스크된 영역으로 컨디셔닝 적용을 제한할 수 있습니다.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `CONDITIONING` | CONDITIONING | 속성이 수정된 결합된 컨디셔닝 데이터 |
