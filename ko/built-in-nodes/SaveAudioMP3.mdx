> 이 문서는 AI로 생성되었습니다. 오류를 발견하거나 개선할 점이 있다면 언제든지 기여해 주세요! [GitHub에서 편집하기](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/SaveAudioMP3/en.md)

SaveAudioMP3 노드는 오디오 데이터를 MP3 파일로 저장합니다. 오디오 입력을 받아 지정된 출력 디렉터리에 사용자 정의 가능한 파일명과 품질 설정으로 내보냅니다. 노드는 파일명 지정과 형식 변환을 자동으로 처리하여 재생 가능한 MP3 파일을 생성합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `audio` | AUDIO | 예 | - | MP3 파일로 저장할 오디오 데이터 |
| `filename_prefix` | STRING | 아니요 | - | 출력 파일명의 접두어 (기본값: "audio/ComfyUI") |
| `quality` | STRING | 아니요 | "V0"<br />"128k"<br />"320k" | MP3 파일의 오디오 품질 설정 (기본값: "V0") |
| `prompt` | PROMPT | 아니요 | - | 내부 프롬프트 데이터 (시스템에서 자동으로 제공) |
| `extra_pnginfo` | EXTRA_PNGINFO | 아니요 | - | 추가 PNG 정보 (시스템에서 자동으로 제공) |

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| *없음* | - | 이 노드는 출력 데이터를 반환하지 않으며, 오디오 파일을 출력 디렉터리에 저장합니다 |
