ControlNet을 사용하려면 입력 이미지의 전처리가 필요합니다. ComfyUI 초기 노드에는 전처리기와 ControlNet 모델이 포함되어 있지 않으므로, 먼저 ControlNet 전처리기를 설치해 주세요 [전처리기 다운로드 여기에서](https://github.com/Fannovel16/comfy_controlnet_preprocessors) 그리고 해당하는 ControlNet 모델도 설치해 주세요.

## 입력

| 매개변수 | 데이터 타입 | 기능 |
| --- | --- | --- |
| `positive` | `CONDITIONING` | 긍정적 조건 데이터, CLIP 텍스트 인코더 또는 기타 조건 입력에서 가져옴 |
| `negative` | `CONDITIONING` | 부정적 조건 데이터, CLIP 텍스트 인코더 또는 기타 조건 입력에서 가져옴 |
| `control_net` | `CONTROL_NET` | 적용할 ControlNet 모델, 일반적으로 ControlNet 로더에서 입력됨 |
| `image` | `IMAGE` | ControlNet 적용용 이미지, 전처리기를 통해 처리해야 함 |
| `vae` | `VAE` | VAE 모델 입력 |
| `strength` | `FLOAT` | 네트워크 조정의 강도를 제어하며, 값 범위는 0~10입니다. 권장값은 0.5~1.5 사이로 적당합니다. 낮은 값은 모델의 자유도를 높이고, 높은 값은 더 엄격한 제약을 가합니다. 너무 높은 값은 이상한 이미지를 생성할 수 있습니다. 이 값을 테스트하고 조정하여 ControlNet의 영향력을 세밀하게 조정할 수 있습니다. |
| `start_percent` | `FLOAT` | 값 0.000~1.000, ControlNet 적용을 시작할 시점을 백분율로 지정합니다. 예를 들어 0.2는 ControlNet 가이드라인이 확산 과정의 20%에서 이미지 생성에 영향을 미치기 시작한다는 의미입니다. |
| `end_percent` | `FLOAT` | 값 0.000~1.000, ControlNet 적용을 중단할 시점을 백분율로 지정합니다. 예를 들어 0.8은 ControlNet 가이드라인이 확산 과정의 80%에서 이미지 생성에 더 이상 영향을 미치지 않는다는 의미입니다. |

### 출력

| 매개변수 | 데이터 타입 | 기능 |
| --- | --- | --- |
| `positive` | `CONDITIONING` | ControlNet으로 처리된 긍정적 조건 데이터, 다음 ControlNet 또는 K 샘플러 노드로 출력 가능 |
| `negative` | `CONDITIONING` | ControlNet으로 처리된 부정적 조건 데이터, 다음 ControlNet 또는 K 샘플러 노드로 출력 가능 |
