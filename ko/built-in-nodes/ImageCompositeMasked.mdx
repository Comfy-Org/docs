`ImageCompositeMasked` 노드는 이미지를 합성하는 데 사용되며, 지정된 좌표에 소스 이미지를 대상 이미지 위에 오버레이하고, 선택적으로 크기 조정과 마스크 적용이 가능합니다.

## 입력

| 매개변수 | 데이터 타입 | 설명 |
|-----------|-------------|-------------|
| `destination` | `IMAGE` | 소스 이미지를 합성할 대상 이미지입니다. 합성 작업의 배경이 됩니다. |
| `source` | `IMAGE` | 대상 이미지 위에 합성될 소스 이미지입니다. 선택적으로 소스 이미지를 대상 이미지의 크기에 맞게 크기를 조정할 수 있습니다. |
| `x` | `INT` | 소스 이미지의 왼쪽 상단 모서리가 배치될 대상 이미지의 x좌표입니다. |
| `y` | `INT` | 소스 이미지의 왼쪽 상단 모서리가 배치될 대상 이미지의 y좌표입니다. |
| `resize_source` | `BOOLEAN` | 소스 이미지를 대상 이미지의 크기에 맞게 크기를 조정할지 여부를 나타내는 부울 플래그입니다. |
| `mask` | `MASK` | 선택적으로 제공되는 마스크로, 소스 이미지의 어느 부분을 대상 이미지에 합성할지 지정합니다. 이를 통해 블렌딩이나 부분 오버레이와 같은 보다 복잡한 합성 작업이 가능합니다. |

## 출력

| 매개변수 | 데이터 타입 | 설명 |
|-----------|-------------|-------------|
| `image` | `IMAGE` | 합성 작업 후 생성된 결과 이미지로, 두 이미지의 요소를 결합한 것입니다.
