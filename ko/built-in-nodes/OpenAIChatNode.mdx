> 이 문서는 AI가 생성했습니다. 오류를 발견하거나 개선할 점이 있다면 언제든지 기여해 주세요! [GitHub에서 편집하기](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/OpenAIChatNode/en.md)

이 노드는 OpenAI 모델로부터 텍스트 응답을 생성합니다. 사용자의 텍스트 프롬프트(및 선택적으로 이미지나 파일)를 OpenAI 모델에 전송하고 생성된 텍스트 응답을 반환합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `prompt` | 문자열 | 예 | - | 모델에 입력되는 텍스트로, 응답을 생성하는 데 사용됩니다 (기본값: 비어 있음) |
| `persist_context` | 부울 | 예 | - | 이 매개변수는 폐기되었으며 아무런 효과도 없습니다 (기본값: False) |
| `model` | 콤보 | 예 | `gpt-5.5-pro`<br />`gpt-5.5`<br />`gpt-5`<br />`gpt-5-mini`<br />`gpt-5-nano`<br />`gpt-4.1`<br />`gpt-4.1-mini`<br />`gpt-4.1-nano`<br />`o4-mini`<br />`o3`<br />`o1-pro`<br />`o1` | 응답을 생성하는 데 사용되는 모델입니다 (기본값: `gpt-5`) |
| `images` | 이미지 | 아니요 | - | 모델의 컨텍스트로 사용할 옵션 이미지입니다. 여러 이미지를 포함하려면 Batch Images 노드를 사용할 수 있습니다 |
| `files` | OPENAI_INPUT_FILES | 아니요 | - | 모델의 컨텍스트로 사용할 옵션 파일입니다. OpenAI Chat Input Files 노드에서 입력을 받습니다 |
| `advanced_options` | OPENAI_CHAT_CONFIG | 아니요 | - | 모델에 대한 옵션 구성입니다. OpenAI Chat Advanced Options 노드에서 입력을 받습니다 |

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `output_text` | 문자열 | OpenAI 모델이 생성한 텍스트 응답 |
