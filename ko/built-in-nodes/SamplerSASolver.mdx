SamplerSASolver 노드는 확산 모델용 맞춤형 샘플링 알고리즘을 구현합니다. 구성 가능한 순서 설정과 확률 미분 방정식(SDE) 매개변수를 사용한 예측-보정 접근 방식을 통해 입력 모델로부터 샘플을 생성합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `model` | MODEL | 예 | - | 샘플링에 사용할 확산 모델 |
| `eta` | FLOAT | 아니요 | 0.0 - 10.0 | 단계 크기 스케일링 요인을 제어합니다(기본값: 1.0) |
| `sde_start_percent` | FLOAT | 아니요 | 0.0 - 1.0 | SDE 샘플링의 시작 비율을 설정합니다(기본값: 0.2) |
| `sde_end_percent` | FLOAT | 아니요 | 0.0 - 1.0 | SDE 샘플링의 종료 비율을 설정합니다(기본값: 0.8) |
| `s_noise` | FLOAT | 아니요 | 0.0 - 100.0 | 샘플링 중 추가되는 노이즈의 양을 제어합니다(기본값: 1.0) |
| `predictor_order` | INT | 아니요 | 1 - 6 | 솔버에서 예측자 구성 요소의 순서를 설정합니다(기본값: 3) |
| `corrector_order` | INT | 아니요 | 0 - 6 | 솔버에서 보정자 구성 요소의 순서를 설정합니다(기본값: 4) |
| `use_pece` | BOOLEAN | 아니요 | - | PECE(예측-평가-보정-평가) 방법을 활성화하거나 비활성화합니다 |
| `simple_order_2` | BOOLEAN | 아니요 | - | 간편한 2차 계산을 활성화하거나 비활성화합니다 |

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `sampler` | SAMPLER | 확산 모델과 함께 사용할 수 있는 구성된 샘플러 객체 |
