> 이 문서는 AI를 통해 생성되었습니다. 오류를 발견하거나 개선할 점이 있다면 언제든지 기여해 주세요! [GitHub에서 편집하기](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/Rodin3D_Sketch/en.md)

이 노드는 Rodin API를 사용하여 3D 자산을 생성합니다. 입력 이미지를 받아 외부 서비스를 통해 3D 모델로 변환합니다. 노드는 작업 생성부터 최종 3D 모델 파일 다운로드까지 전 과정을 처리합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `Images` | IMAGE | 예 | 1~5개 이미지 | 3D 모델로 변환할 입력 이미지입니다. 1개에서 5개까지 제공할 수 있습니다. |
| `Seed` | INT | 아니요 | 0~65535 | 생성을 위한 난수 시드 값 (기본값: 0). 0으로 설정하면 임의의 시드가 사용됩니다. |

**참고:** 노드는 최소 1개의 이미지를 필요로 하며, 최대 5개의 이미지를 지원합니다. 이미지를 제공하지 않으면 노드에서 오류가 발생합니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `3D Model Path` | STRING | 생성된 3D 모델의 파일 경로 (후향적 호환성을 위해만 제공) |
| `GLB` | FILE3DGLB | GLB 형식으로 생성된 3D 모델 |
