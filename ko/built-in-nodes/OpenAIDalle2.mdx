# OpenAIDalle2

OpenAI의 DALL·E 2 엔드포인트를 통해 이미지를 동기적으로 생성합니다.

## 작동 방식

이 노드는 OpenAI의 DALL·E 2 API와 연결되어 텍스트 설명을 기반으로 이미지를 생성합니다. 텍스트 프롬프트를 제공하면 노드는 이를 OpenAI 서버로 전송하고, 서버는 해당하는 이미지를 생성한 후 ComfyUI로 반환합니다. 노드는 두 가지 모드로 작동할 수 있습니다: 텍스트 프롬프트만을 이용한 표준 이미지 생성 모드와 이미지와 마스크를 모두 제공할 경우 이미지를 편집하는 모드입니다. 편집 모드에서는 마스크를 사용해 원본 이미지의 특정 부분만 수정하고 나머지는 그대로 유지하도록 합니다.

## 입력

| 매개변수 | 데이터 타입 | 입력 유형 | 기본값 | 범위 | 설명 |
|-----------|-----------|------------|---------|-------|-------------|
| `prompt` | 문자열 | 필수 | "" | - | DALL·E용 텍스트 프롬프트 |
| `seed` | 정수 | 선택적 | 0 | 0부터 2147483647까지 | 아직 백엔드에서 구현되지 않음 |
| `size` | 콤보 | 선택적 | "1024x1024" | "256x256", "512x512", "1024x1024" | 이미지 크기 |
| `n` | 정수 | 선택적 | 1 | 1부터 8까지 | 생성할 이미지 수 |
| `image` | 이미지 | 선택적 | 없음 | - | 이미지 편집용 선택적 참조 이미지. |
| `mask` | 마스크 | 선택적 | 없음 | - | 인페인팅용 선택적 마스크 (흰색 영역은 교체됨) |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `IMAGE` | 이미지 | DALL·E 2에서 생성되거나 편집된 이미지 |
