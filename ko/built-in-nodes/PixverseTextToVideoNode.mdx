> 이 문서는 AI로 생성되었습니다. 오류를 발견하거나 개선할 점이 있다면 언제든지 기여해 주세요! [GitHub에서 편집하기](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/PixverseTextToVideoNode/en.md)

텍스트 프롬프트와 다양한 생성 매개변수를 기반으로 동영상을 생성합니다. 이 노드는 PixVerse API를 사용해 비디오 콘텐츠를 생성하며, 종횡비, 품질, 지속 시간, 모션 스타일 등을 제어할 수 있습니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `prompt` | 문자열 | 예 | - | 비디오 생성을 위한 프롬프트 (기본값: "") |
| `aspect_ratio` | 콤보 | 예 | `"16:9"`<br />`"9:16"`<br />`"1:1"` | 생성된 비디오의 종횡비 |
| `quality` | 콤보 | 예 | `"540p"`<br />`"1080p"` | 비디오 품질 설정 (기본값: "540p") |
| `duration_seconds` | 콤보 | 예 | `"5"`<br />`"10"` | 생성된 비디오의 지속 시간(초) |
| `motion_mode` | 콤보 | 예 | `"normal"`<br />`"fast"` | 비디오 생성 시 모션 스타일 |
| `seed` | 정수 | 예 | 0부터 2147483647까지 | 비디오 생성을 위한 시드 (기본값: 0) |
| `negative_prompt` | 문자열 | 아니요 | - | 이미지에서 원치 않는 요소에 대한 선택적 텍스트 설명 (기본값: "") |
| `pixverse_template` | 맞춤형 | 아니요 | - | 스타일에 영향을 미치도록 설계된 옵션 템플릿, PixVerse Template 노드에서 생성 |

**참고:** 1080p 품질을 사용할 경우 모션 모드는 자동으로 일반으로 설정되고 지속 시간은 5초로 제한됩니다. 5초가 아닌 지속 시간에서는 모션 모드도 자동으로 일반으로 설정됩니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `output` | 비디오 | 생성된 비디오 파일 |
