## 개요

VOIDQuadmaskPreprocess 노드는 마스크를 특수한 4단계 '쿼드마스크'로 변환하여 VOID 인페인팅용으로 준비합니다. 입력 마스크를 받아들여 선택적으로 주요 영역을 팽창시킨 후, 마스크 값을 4개의 서로 다른 수준으로 양자화하여 각각의 세미틱 영역(주요 객체, 오버랩, 영향받은 영역, 배경)을 나타냅니다. 마지막으로 마스크를 반전하고 정규화하여 출력 값이 [0, 1] 범위 내에 있도록 합니다. 여기서 1.0은 제거할 영역을, 0.0은 유지할 영역을 나타냅니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `mask` | MASK | 예 | N/A | 전처리할 입력 마스크입니다. |
| `dilate_width` | INT | 아니요 | 0부터 50까지 (단계: 1) | 주요 마스크 영역의 팽창 반경입니다. 값이 0이면 팽창이 적용되지 않습니다. (기본값: 0) |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `quadmask` | MASK | 전처리된 쿼드마스크로, 값이 [0, 1] 범위 내에 있으며 4개의 구분된 레벨을 나타냅니다: 1.0 (제거할 주요 객체), ~0.75 (주요 객체와 영향받은 영역의 오버랩), ~0.50 (영향받은 영역), 0.0 (유지할 배경). |
