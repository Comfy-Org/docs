CLIPTextEncodeControlnet 노드는 CLIP 모델을 사용해 텍스트 입력을 처리하고, 기존 컨디셔닝 데이터와 결합하여 ControlNet 애플리케이션을 위한 강화된 컨디셔닝 출력을 생성합니다. 입력 텍스트를 토큰화한 뒤 CLIP 모델을 통해 인코딩하고, 그 결과로 생성된 임베딩을 제공된 컨디셔닝 데이터에 교차 주의력 ControlNet 파라미터로 추가합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `clip` | CLIP | 예 | - | 텍스트 토큰화 및 인코딩에 사용되는 CLIP 모델 |
| `conditioning` | CONDITIONING | 예 | - | ControlNet 파라미터로 강화될 기존 컨디셔닝 데이터 |
| `text` | STRING | 예 | - | CLIP 모델로 처리할 텍스트 입력입니다. 다중행 텍스트 및 동적 프롬프트 지원 |

**참고:** 이 노드는 `clip`, `conditioning`, `text` 세 가지 입력 모두가 있어야 제대로 작동합니다. `text` 입력은 동적 프롬프트와 다중행 텍스트를 지원하여 유연한 텍스트 처리가 가능합니다.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `CONDITIONING` | CONDITIONING | CLIP 텍스트 인코딩에서 유래한 ControlNet 교차 주의력 파라미터(`cross_attn_controlnet` 및 `pooled_output_controlnet`)가 추가된 강화된 컨디셔닝 데이터 |
