Moonvalley Marey 비디오 투 비디오 노드는 입력 비디오를 텍스트 설명을 기반으로 새로운 비디오로 변환합니다. Moonvalley API를 사용해 원본 비디오의 움직임이나 자세 특성을 유지하면서 프롬프트와 일치하는 비디오를 생성합니다. 텍스트 프롬프트와 다양한 생성 매개변수를 통해 출력 비디오의 스타일과 내용을 제어할 수 있습니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `prompt` | 문자열 | 예 | - | 생성할 비디오를 설명합니다 (다중 줄 입력) |
| `negative_prompt` | 문자열 | 아니요 | - | 부정적 프롬프트 텍스트 (기본값: 광범위한 부정적 설명 목록) |
| `seed` | 정수 | 예 | 0부터 4294967295까지 | 난수 시드 값 (기본값: 9) |
| `video` | 비디오 | 예 | - | 출력 비디오 생성에 사용되는 참조 비디오입니다. 최소 5초 이상이어야 합니다. 5초 초과 비디오는 자동으로 잘려집니다. MP4 형식만 지원됩니다. |
| `control_type` | 콤보 | 아니요 | "움직임 전송"<br />"자세 전송" | 제어 유형 선택 (기본값: "움직임 전송") |
| `motion_intensity` | 정수 | 아니요 | 0부터 100까지 | 제어 유형이 "움직임 전송"일 때만 사용됩니다 (기본값: 100) |
| `steps` | 정수 | 예 | 1부터 100까지 | 추론 단계 수 (기본값: 33) |

**참고:** `motion_intensity` 매개변수는 `control_type`이 "움직임 전송"으로 설정된 경우에만 적용됩니다. "자세 전송"을 사용할 경우 이 매개변수는 무시됩니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `output` | 비디오 | 생성된 비디오 출력 |
