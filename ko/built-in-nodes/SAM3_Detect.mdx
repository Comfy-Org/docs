# SAM3 Detect 노드

## 개요

SAM3 Detect 노드는 텍스트 설명, 경계 상자 또는 포인트 프롬프트를 사용하여 오픈 빈도 사전 검출 및 세분화를 수행합니다. 이미지 내 객체를 텍스트로 설명하거나 상자를 그려 넣거나 포인트를 클릭한 위치에 따라 식별하고 세분화할 수 있습니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `model` | MODEL | 예 | - | 검출 및 세분화에 사용할 SAM3 모델 |
| `image` | IMAGE | 예 | - | 처리할 입력 이미지 |
| `conditioning` | CONDITIONING | 아니요 | - | CLIPTextEncode의 텍스트 조건부. 검출에 텍스트 프롬프트를 사용할 경우 필요 |
| `bboxes` | BOUNDING_BOX | 아니요 | - | 세분화할 경계 상자. 단일 상자(모든 프레임에 적용), 상자 목록(모든 프레임에 적용), 리스트의 리스트(프레임별 상자) 형태로 제공 가능. 텍스트 조건부 없이 제공될 경우 노드는 각 상자 내부를 세분화합니다 |
| `positive_coords` | STRING | 아니요 | - | 픽셀 좌표를 사용한 JSON 형식의 양성 포인트 프롬프트 `[{"x": int, "y": int}, ...]`. 세분화에 포함하고자 하는 포인트입니다 |
| `negative_coords` | STRING | 아니요 | - | 픽셀 좌표를 사용한 JSON 형식의 음성 포인트 프롬프트 `[{"x": int, "y": int}, ...]`. 세분화에서 제외하고자 하는 포인트입니다 |
| `threshold` | FLOAT | 아니요 | 0.0~1.0 | 텍스트 기반 검출의 신뢰도 임계값. 이 값보다 높은 점수를 가진 검출만 유지됩니다(기본값: 0.5) |
| `refine_iterations` | INT | 아니요 | 0~5 | SAM 디코더 정밀화 반복 횟수. 값이 높을수록 마스크 품질이 향상됩니다. 0으로 설정하면 정밀화 없이 원시 검출 마스크를 사용합니다(기본값: 2) |
| `individual_masks` | BOOLEAN | 아니요 | true/false | 활성화 시, 검출된 각 객체별로 별도의 마스크를 출력하며 하나의 마스크로 합치지 않습니다(기본값: false) |

### 매개변수 제약 및 참고사항

- **텍스트 프롬프트**: 텍스트 기반 검출을 사용하려면 반드시 `conditioning` 입력을 제공해야 합니다. 텍스트 조건부가 제공되면 노드는 이미지에서 텍스트 기반 검출을 실행합니다.
- **상자 프롬프트**: 텍스트 조건부 없이 `bboxes`가 제공될 경우 노드는 각 경계 상자 내부를 세분화합니다.
- **포인트 프롬프트**: `positive_coords` 또는 `negative_coords`가 제공될 경우 노드는 포인트 기반 세분화를 사용합니다. 포인트는 자동으로 모델의 내부 해상도로 스케일링됩니다.
- **다양한 프롬프트 유형 혼합**: 서로 다른 프롬프트 유형을 조합할 수 있습니다. 예를 들어, 텍스트 조건부와 경계 상자를 함께 제공해 특정 영역에 한정된 텍스트 검출을 수행할 수 있습니다.
- **배치 처리**: 노드는 배치 처리된 이미지를 지원합니다. 여러 프레임을 처리할 때는 리스트의 리스트 형식을 사용해 프레임별로 경계 상자를 제공할 수 있습니다.
- **포인트의 JSON 형식**: 포인트 좌표는 `[{"x": 100, "y": 200}, {"x": 150, "y": 250}]`와 같은 유효한 JSON 문자열로 제공되어야 합니다.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `masks` | MASK | 세분화 마스크. `individual_masks`가 False(기본값)일 경우, 프레임당 하나의 통합 마스크를 반환합니다. True일 경우, 검출된 각 객체별로 개별 마스크를 반환합니다 |
| `bboxes` | BOUNDING_BOX | 검출된 경계 상자와 좌표 및 신뢰도 점수. 각 상자는 `x`, `y`, `width`, `height`, `score` 값을 포함합니다 |
