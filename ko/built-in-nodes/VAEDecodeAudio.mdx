VAEDecodeAudio 노드는 변분 오토인코더를 사용하여 잠재 표현을 다시 오디오 파형으로 변환합니다. 인코딩된 오디오 샘플을 받아 변분 오토인코더를 통해 원래의 오디오로 재구성하며, 일정한 출력 레벨을 보장하기 위해 정규화를 적용합니다. 결과 오디오는 표준 샘플레이트 44100 Hz로 반환되며, 입력 샘플에 샘플레이트가 제공된 경우 해당 샘플레이트를 사용합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `samples` | LATENT | 예 | - | 잠재 공간에 인코딩된 오디오 샘플로, 오디오 파형으로 다시 디코딩됩니다 |
| `vae` | VAE | 예 | - | 잠재 샘플을 오디오로 디코딩하는 데 사용되는 변분 오토인코더 모델 |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `AUDIO` | AUDIO | 정규화된 볼륨과 샘플레이트를 갖춘 디코딩된 오디오 파형 (기본값: 44100 Hz, 또는 입력 `samples`에 샘플레이트가 제공된 경우 해당 샘플레이트) |
