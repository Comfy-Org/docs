StringSubstring 노드는 더 큰 문자열에서 일부 텍스트를 추출합니다. 추출하려는 부분을 정의하기 위해 시작 위치와 끝 위치를 지정하면, 해당 두 위치 사이의 텍스트를 반환합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `string` | STRING | 예 | - | 추출할 입력 텍스트 문자열입니다. 다중 줄 텍스트도 지원됩니다. |
| `start` | INT | 예 | - | 하위 문자열의 시작 위치 인덱스입니다. 첫 번째 문자는 인덱스 0에 있습니다. |
| `end` | INT | 예 | - | 하위 문자열의 끝 위치 인덱스입니다. 이 인덱스의 문자는 결과에 포함되지 않습니다. |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `output` | STRING | 입력 텍스트에서 추출된 하위 문자열로, `start` 위치부터 (포함하지 않음) `end` 위치까지의 모든 문자를 포함합니다. |
