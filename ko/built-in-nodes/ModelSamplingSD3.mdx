ModelSamplingSD3 노드는 안정적인 디퓨전 3 샘플링 파라미터를 모델에 적용합니다. 이 노드는 샘플링 분포 특성을 제어하는 시프트 파라미터를 조정하여 모델의 샘플링 동작을 수정합니다. 노드는 지정된 샘플링 구성이 적용된 입력 모델의 수정된 복사본을 생성합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `model` | MODEL | 예 | - | SD3 샘플링 파라미터를 적용할 입력 모델 |
| `shift` | FLOAT | 예 | 0.0 - 100.0 | 샘플링 시프트 파라미터를 제어합니다 (기본값: 3.0) |

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `model` | MODEL | SD3 샘플링 파라미터가 적용된 수정된 모델 |
