이 노드는 가우시안 스플랫 3D 파일을 ComfyUI 출력 디렉토리에 저장하고 3D 뷰어용 미리보기 데이터를 제공합니다.

## 입력

| 매개변수 | 설명 | 데이터 타입 | 필수 | 범위 |
|-----------|-------------|-----------|----------|-------|
| `model_3d` | 가우시안 스플랫 3D 파일입니다. | FILE3D | 예 | - |
| `filename_prefix` | 저장된 파일 이름의 접두사입니다 (기본값: "3d/ComfyUI") | STRING | 예 | - |
| `viewport_state` | 3D 로더 노드의 현재 뷰포트 상태입니다. | LOAD3D | 예 | - |
| `model_3d_info` | 미리보기를 위한 추가 3D 모델 정보입니다. | LOAD3DMODELINFO | 아니요 | - |
| `camera_info` | 미리보기를 위한 카메라 정보입니다. | LOAD3DCAMERA | 아니요 | - |
| `width` | 미리보기의 픽셀 너비입니다 (기본값: 1024) | INT | 예 | 최소: 1, 최대: 4096, 단계: 1 |
| `height` | 미리보기의 픽셀 높이입니다 (기본값: 1024) | INT | 예 | 최소: 1, 최대: 4096, 단계: 1 |

## 출력

| 출력 이름 | 설명 | 데이터 타입 |
|-------------|-------------|-----------|
| `model_3d` | 저장된 가우시안 스플랫 3D 파일입니다. | FILE3D |
| `model_3d_info` | 미리보기를 위한 3D 모델 정보입니다. | LOAD3DMODELINFO |
| `camera_info` | 미리보기를 위한 카메라 정보입니다. | LOAD3DCAMERA |
| `width` | 미리보기의 너비입니다. | INT |
| `height` | 미리보기의 높이입니다. | INT |
