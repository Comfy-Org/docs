WanTrackToVideo 노드는 모션 트래킹 데이터를 처리하여 트랙 포인트를 비디오 프레임으로 변환하고, 이를 통해 비디오 시퀀스를 생성합니다. 이 노드는 트래킹 좌표를 입력으로 받아 비디오 조건부와 잠재 표현을 생성하며, 이를 통해 비디오 생성에 활용할 수 있습니다. 트랙이 제공되지 않을 경우, 표준 이미지-비디오 변환 방식으로 전환됩니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `positive` | CONDITIONING | 예 | - | 비디오 생성을 위한 양성 조건부 |
| `negative` | CONDITIONING | 예 | - | 비디오 생성을 위한 음성 조건부 |
| `vae` | VAE | 예 | - | 인코딩 및 디코딩을 위한 VAE 모델 |
| `tracks` | STRING | 예 | - | JSON 형식의 트래킹 데이터, 다중 줄 문자열로 제공됨 (기본값: "[]") |
| `width` | INT | 예 | 16부터 MAX_RESOLUTION까지 | 출력 비디오의 너비(픽셀), 기본값: 832, 단계: 16 |
| `height` | INT | 예 | 16부터 MAX_RESOLUTION까지 | 출력 비디오의 높이(픽셀), 기본값: 480, 단계: 16 |
| `length` | INT | 예 | 1부터 MAX_RESOLUTION까지 | 출력 비디오의 프레임 수, 기본값: 81, 단계: 4 |
| `batch_size` | INT | 예 | 1부터 4096까지 | 동시에 생성할 비디오 수, 기본값: 1 |
| `temperature` | FLOAT | 예 | 1.0부터 1000.0까지 | 모션 패치를 위한 온도 파라미터, 기본값: 220.0, 단계: 0.1 |
| `topk` | INT | 예 | 1부터 10까지 | 모션 패치의 top-k 값, 기본값: 2 |
| `start_image` | IMAGE | 선택 | - | 비디오 생성을 위한 시작 이미지 |
| `clip_vision_output` | CLIPVISIONOUTPUT | 선택 | - | 추가 조건부를 위한 CLIP 비전 출력 |

**참고:** `tracks`에 유효한 트래킹 데이터가 포함되어 있을 경우, 노드는 모션 트랙을 처리하여 비디오를 생성합니다. `tracks`가 비어 있을 경우, 표준 이미지-비디오 모드로 전환됩니다. `start_image`가 제공되면, 비디오 시퀀스의 첫 번째 프레임이 초기화됩니다.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-----------|-----------|-------------|
| `positive` | CONDITIONING | 모션 트랙 정보가 적용된 양성 조건부 |
| `negative` | CONDITIONING | 모션 트랙 정보가 적용된 음성 조건부 |
| `latent` | LATENT | 생성된 비디오 잠재 표현 |
