Rodin 3D Smooth 노드는 입력 이미지를 처리하여 부드러운 3D 모델로 변환함으로써 Rodin API를 이용해 3D 자산을 생성합니다. 이 노드는 여러 장의 이미지를 입력받아 다운로드 가능한 3D 모델 파일을 생성합니다. 노드는 작업 생성, 상태 폴링, 파일 다운로드까지 전 과정을 자동으로 처리합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `Images` | IMAGE | 예 | - | 3D 모델 생성에 사용할 입력 이미지입니다. 최대 5장의 이미지를 제공할 수 있습니다. |
| `Seed` | INT | 아니요 | 0부터 65535까지 | 생성의 일관성을 유지하기 위한 랜덤 시드 값 (기본값: 0). |
| `Material_Type` | STRING | 아니요 | "PBR"<br />"Shaded" | 3D 모델에 적용할 재질 유형 (기본값: "PBR"). |
| `Polygon_count` | STRING | 아니요 | "4K-Quad"<br />"8K-Quad"<br />"18K-Quad"<br />"50K-Quad"<br />"200K-Triangle" | 생성된 3D 모델의 목표 폴리곤 수입니다. 메쉬의 품질과 세밀도를 결정합니다 (기본값: "18K-Quad"). |

**참고:** 이 노드는 최대 5장의 입력 이미지를 받습니다. 이미지를 제공하지 않으면 오류가 발생합니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `3D Model Path` | STRING | 다운로드한 3D 모델의 파일 경로 (후방 호환성용만 제공됨). |
| `GLB` | FILE3DGLB | GLB 형식으로 생성된 3D 모델입니다. |
