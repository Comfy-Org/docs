Magnific Image Upscale (Precise V2) 노드는 선명도, 입자감, 디테일 강화를 정밀하게 제어하며 고화질 이미지 업스케일링을 수행합니다. 이 노드는 외부 API를 통해 이미지를 처리하며 최대 출력 해상도는 10060×10060픽셀까지 지원합니다. 노드는 다양한 처리 스타일을 제공하며, 요청된 출력 크기가 허용된 최대 크기를 초과할 경우 자동으로 입력 이미지를 다운스케일링할 수 있습니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `image` | IMAGE | 예 | - | 업스케일링할 입력 이미지입니다. 이미지는 반드시 하나만 필요합니다. 최소 크기는 160x160픽셀입니다. 종횡비는 1:3에서 3:1 사이여야 합니다. |
| `scale_factor` | STRING | 예 | `"2x"`<br />`"4x"`<br />`"8x"`<br />`"16x"` | 원하는 업스케일링 배율입니다. |
| `flavor` | STRING | 예 | `"sublime"`<br />`"photo"`<br />`"photo_denoiser"` | 처리 스타일입니다. "sublime"는 일반적인 용도에 적합하고, "photo"는 사진에 최적화되었으며, "photo_denoiser"는 노이즈가 많은 사진에 적합합니다. |
| `sharpen` | INT | 아니요 | 0부터 100까지 | 이미지의 선명도를 조정하여 가장자리의 정의와 명확성을 높입니다. 값이 높을수록 더 선명한 결과를 얻을 수 있습니다. 기본값: 7. |
| `smart_grain` | INT | 아니요 | 0부터 100까지 | 지능형 입자감이나 질감 강화를 추가하여 업스케일링된 이미지가 너무 매끄럽거나 인공적으로 보이는 것을 방지합니다. 기본값: 7. |
| `ultra_detail` | INT | 아니요 | 0부터 100까지 | 업스케일링 과정에서 추가되는 미세한 디테일, 질감, 마이크로 디테일의 양을 조정합니다. 기본값: 30. |
| `auto_downscale` | BOOLEAN | 아니요 | - | 활성화 시, 계산된 출력 크기가 최대 허용 해상도인 10060x10060픽셀을 초과할 경우 노드가 자동으로 입력 이미지를 다운스케일링합니다. 이는 오류를 방지하는 데 도움이 되지만 품질에 영향을 줄 수 있습니다. 기본값: False. |

**참고:** `auto_downscale`가 비활성화된 상태에서 요청된 출력 크기(입력 이미지 크기 × `scale_factor`)가 10060x10060픽셀을 초과하면 노드는 오류를 발생시킵니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-----------|-----------|-------------|
| `image` | IMAGE | 결과로 생성된 업스케일링된 이미지입니다. |
