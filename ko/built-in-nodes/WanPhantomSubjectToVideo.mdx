WanPhantomSubjectToVideo 노드는 조건 입력과 선택적 참조 이미지를 처리하여 비디오 콘텐츠를 생성합니다. 이 노드는 비디오 생성을 위한 잠재 표현을 만들며, 입력 이미지가 제공되면 시각적 가이드라인을 포함할 수 있습니다. 노드는 비디오 모델을 위해 시간 차원의 연결을 통해 조건 데이터를 준비하고, 생성된 잠재 비디오 데이터와 함께 수정된 조건 데이터를 출력합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `positive` | CONDITIONING | 예 | - | 비디오 생성을 안내하는 긍정적 조건 입력 |
| `negative` | CONDITIONING | 예 | - | 특정 특성을 피하도록 하는 부정적 조건 입력 |
| `vae` | VAE | 예 | - | 이미지를 인코딩할 때 제공되는 VAE 모델 |
| `width` | INT | 예 | 16부터 MAX_RESOLUTION까지 | 출력 비디오 너비(픽셀) (기본값: 832, 16으로 나누어져야 함) |
| `height` | INT | 예 | 16부터 MAX_RESOLUTION까지 | 출력 비디오 높이(픽셀) (기본값: 480, 16으로 나누어져야 함) |
| `length` | INT | 예 | 1부터 MAX_RESOLUTION까지 | 생성된 비디오의 프레임 수 (기본값: 81, 4로 나누어져야 함) |
| `batch_size` | INT | 예 | 1부터 4096까지 | 동시에 생성할 비디오 수 (기본값: 1) |
| `images` | IMAGE | 아니요 | - | 시간 차원 조건을 위한 선택적 참조 이미지 |

**참고:** `images`가 제공되면 자동으로 지정된 `width`와 `height`에 맞게 업스케일링되며, 처리에 사용되는 프레임은 처음 `length`개만입니다.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `positive` | CONDITIONING | 이미지가 제공되면 시간 차원 연결을 통해 수정된 긍정적 조건 입력 |
| `negative_text` | CONDITIONING | 이미지가 제공되면 시간 차원 연결을 통해 수정된 부정적 조건 입력 |
| `negative_img_text` | CONDITIONING | 이미지가 제공되면 시간 차원 연결이 제거된 부정적 조건 입력 |
| `latent` | LATENT | 지정된 크기와 길이를 가진 생성된 잠재 비디오 표현 |
