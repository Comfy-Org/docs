Mahiro 노드는 가이던스 함수를 수정하여 긍정적 프롬프트와 부정적 프롬프트 간의 차이보다는 긍정적 프롬프트의 방향에 더 중점을 둡니다. 정규화된 조건부 및 무조건적 디노이즈 출력 간의 코사인 유사도를 사용해 맞춤형 가이던스 스케일링 방식을 적용한 패치 모델을 생성합니다. 이 실험적인 노드는 생성 과정을 더욱 강력하게 긍정적 프롬프트가 의도한 방향으로 유도하는 데 도움을 줍니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `model` | MODEL | 예 | | 수정된 가이던스 함수로 패치될 모델 |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `patched_model` | MODEL | Mahiro 가이던스 함수가 적용된 수정된 모델 |
