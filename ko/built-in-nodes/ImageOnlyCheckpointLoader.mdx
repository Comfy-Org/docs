이 노드는 `ComfyUI/models/checkpoints` 폴더에 위치한 모델을 탐지하며, extra_model_paths.yaml 파일에 구성된 추가 경로에서도 모델을 읽습니다. 때때로, 해당 폴더의 모델 파일을 읽도록 하려면 **ComfyUI 인터페이스를 새로 고침**해야 할 수 있습니다.

이 노드는 비디오 생성 워크플로우에서 이미지 기반 모델 전용 체크포인트를 로드하는 데 특화되어 있습니다. 주어진 체크포인트에서 필요한 구성 요소들을 효율적으로 가져와 설정하며, 모델의 이미지 관련 부분에 중점을 둡니다.

## 입력

| 필드      | 데이터 유형 | 설명                                                                       |
|------------|-------------|-----------------------------------------------------------------------------------|
| `ckpt_name`| COMBO[STRING] | 로드할 체크포인트의 이름을 지정하며, 미리 정해진 목록에서 올바른 체크포인트 파일을 식별하고 가져오는 데 필수적입니다. |

## 출력

| 필드     | 데이터 유형 | 설명                                                                                   |
|-----------|-------------|-----------------------------------------------------------------------------------------------|
| `model`   | MODEL     | 체크포인트에서 로드된 메인 모델을 반환하며, 비디오 생성 컨텍스트에서 이미지 처리를 위해 구성됩니다. |
| `clip_vision` | `CLIP_VISION` | 체크포인트의 CLIP 비전 구성 요소를 제공하며, 이미지 이해 및 특징 추출에 맞춰져 있습니다. |
| `vae`     | VAE       | 변분 오토인코더(VAE) 구성 요소를 제공하며, 이미지 조작 및 생성 작업에 필수적입니다. |
