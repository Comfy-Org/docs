StableCascade_SuperResolutionControlnet 노드는 Stable Cascade 초해상도 처리를 위한 입력을 준비합니다. 입력 이미지를 VAE를 사용해 인코딩하여 ControlNet 입력을 생성하며, 동시에 Stable Cascade 파이프라인의 단계 C와 단계 B를 위한 자리표시용 잠재 표현도 생성합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `image` | IMAGE | 예 | - | 초해상도 처리를 위해 처리할 입력 이미지 |
| `vae` | VAE | 예 | - | 입력 이미지를 인코딩하는 데 사용되는 VAE 모델 |

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `controlnet_input` | IMAGE | ControlNet 입력에 적합한 인코딩된 이미지 표현 |
| `stage_c` | LATENT | Stable Cascade 처리의 단계 C를 위한 자리표시용 잠재 표현으로, 입력 이미지 크기를 16으로 나눈 크기로 설정됨 |
| `stage_b` | LATENT | Stable Cascade 처리의 단계 B를 위한 자리표시용 잠재 표현으로, 입력 이미지 크기를 2로 나눈 크기로 설정됨 |
