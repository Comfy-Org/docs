`ImageCrop` 노드는 주어진 x와 y 좌표를 기준으로 이미지를 지정된 너비와 높이로 자르기 위해 설계되었습니다. 이 기능은 이미지의 특정 영역에 집중하거나 이미지 크기를 특정 요구 사항에 맞게 조정하는 데 필수적입니다.

## 입력

| 필드 | 데이터 타입 | 설명                                                                                   |
|-------|-------------|-----------------------------------------------------------------------------------------------|
| `image` | `IMAGE` | 자를 대상인 입력 이미지입니다. 이 매개변수는 매우 중요하며, 지정된 치수와 좌표를 기반으로 원본 이미지에서 영역을 추출할 수 있도록 합니다. |
| `width` | `INT` | 자른 이미지의 너비를 지정합니다. 이 매개변수는 결과적으로 얻어지는 자른 이미지의 가로 크기를 결정합니다. |
| `height` | `INT` | 자른 이미지의 높이를 지정합니다. 이 매개변수는 결과적으로 얻어지는 자른 이미지의 세로 크기를 결정합니다. |
| `x` | `INT` | 자르기 영역의 왼쪽 상단 모서리의 x좌표입니다. 이 매개변수는 자르기의 가로 치수의 시작점을 설정합니다. |
| `y` | `INT` | 자르기 영역의 왼쪽 상단 모서리의 y좌표입니다. 이 매개변수는 자르기의 세로 치수의 시작점을 설정합니다. |

## 출력

| 필드 | 데이터 타입 | 설명                                                                   |
|-------|-------------|-------------------------------------------------------------------------------|
| `image` | `IMAGE` | 자르기 작업의 결과로 얻어진 자른 이미지입니다. 이 출력은 지정된 이미지 영역을 추가로 처리하거나 분석하는 데 중요한 역할을 합니다. |
