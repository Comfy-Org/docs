ElevenLabs 음성 변환 노드는 입력 오디오 파일을 한 목소리에서 다른 목소리로 변환합니다. ElevenLabs API를 사용해 음성 변환 시 원래의 내용과 감정적 어조를 그대로 유지합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `voice` | CUSTOM | 예 | - | 변환할 대상 목소리입니다. Voice Selector 또는 Instant Voice Clone에서 연결하세요. |
| `audio` | AUDIO | 예 | - | 변환할 소스 오디오입니다. |
| `stability` | FLOAT | 아니요 | 0.0 - 1.0 | 목소리 안정성입니다. 낮은 값은 더 넓은 감정 범위를 제공하며, 높은 값은 더 일정하지만 다소 단조로운 말투를 만들어냅니다(기본값: 0.5). |
| `model` | DYNAMICCOMBO | 아니요 | `eleven_multilingual_sts_v2`<br />`eleven_english_sts_v2` | 음성 변환에 사용할 모델입니다. 각 옵션은 특정한 목소리 설정(유사도 강화, 스타일, 스피커 강화 사용, 속도)을 제공합니다. |
| `output_format` | COMBO | 아니요 | `"mp3_44100_192"`<br />`"opus_48000_192"` | 오디오 출력 형식(기본값: "mp3_44100_192"). |
| `seed` | INT | 아니요 | 0 - 4294967295 | 재현성을 위한 시드값(기본값: 0). |
| `remove_background_noise` | BOOLEAN | 아니요 | - | 오디오 격리를 통해 입력 오디오의 배경 잡음을 제거합니다(기본값: False). |

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `audio` | AUDIO | 지정된 출력 형식으로 변환된 오디오 파일입니다. |
