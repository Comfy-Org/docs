> 이 문서는 AI로 생성되었습니다. 오류를 발견하거나 개선할 점이 있다면 언제든지 기여해 주세요! [GitHub에서 편집하기](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/ByteDanceImageToVideoNode/en.md)

ByteDance 이미지 비디오 노드는 입력 이미지와 텍스트 프롬프트를 기반으로 API를 통해 ByteDance 모델을 사용하여 비디오를 생성합니다. 시작 프레임 이미지를 받아 제공된 설명을 따라 비디오 시퀀스를 만듭니다. 이 노드는 비디오 해상도, 종횡비, 지속 시간 및 기타 생성 매개변수에 대한 다양한 맞춤 옵션을 제공합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `model` | 문자열 | 예 | `"seedance-1-5-pro-251215"`<br />`"seedance-1-0-pro-250528"`<br />`"seedance-1-0-lite-i2v-250428"`<br />`"seedance-1-0-pro-fast-251015"` | 비디오 생성에 사용할 ByteDance 모델 (기본값: `"seedance-1-0-pro-fast-251015"`). |
| `prompt` | 문자열 | 예 | - | 비디오 생성에 사용되는 텍스트 프롬프트입니다. 공백을 제거한 후 최소 1자 이상이어야 합니다. |
| `image` | 이미지 | 예 | - | 비디오에 사용될 첫 번째 프레임입니다. 가로세로 비율은 0.4~2.5 사이이며, 크기는 300x300픽셀에서 6000x6000픽셀 사이여야 합니다. |
| `resolution` | 문자열 | 예 | `"480p"`<br />`"720p"`<br />`"1080p"` | 출력 비디오의 해상도입니다. |
| `aspect_ratio` | 문자열 | 예 | `"adaptive"`<br />`"16:9"`<br />`"4:3"`<br />`"1:1"`<br />`"3:4"`<br />`"9:16"`<br />`"21:9"` | 출력 비디오의 종횡비입니다. |
| `duration` | 정수 | 예 | 3~12 | 출력 비디오의 지속 시간(초)입니다 (기본값: 5). `seedance-1-5-pro-251215` 모델의 경우 최소 지원 지속 시간은 4초입니다. |
| `seed` | 정수 | 아니요 | 0~2147483647 | 생성에 사용할 시드입니다 (기본값: 0). |
| `camera_fixed` | 부울 | 아니요 | - | 카메라를 고정할지 여부를 지정합니다. 플랫폼은 카메라를 고정하라는 지시를 프롬프트에 추가하지만, 실제로 효과가 보장되지는 않습니다 (기본값: False). |
| `watermark` | 부울 | 아니요 | - | 비디오에 "AI 생성" 워터마크를 추가할지 여부입니다 (기본값: False). |
| `generate_audio` | 부울 | 아니요 | - | 이 매개변수는 `seedance-1-5-pro-251215` 모델을 제외한 모든 모델에서는 무시됩니다 (기본값: False). |

**참고:** 프롬프트에는 다음 단어가 포함되어서는 안 됩니다 (대소문자 구분 없음): `resolution`, `ratio`, `duration`, `seed`, `camerafixed`, `watermark`. 이러한 매개변수는 각각 전용 입력을 통해 설정됩니다.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-----------|-----------|-------------|
| `output` | 비디오 | 입력 이미지와 프롬프트 매개변수를 기반으로 생성된 비디오 파일입니다. |
