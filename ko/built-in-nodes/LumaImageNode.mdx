텍스트 프롬프트와 종횡비를 기반으로 이미지를 동기식으로 생성합니다. 이 노드는 텍스트 설명을 이용해 이미지를 생성하며, 문자 및 스타일 이미지를 포함한 다양한 참조 입력을 통해 이미지의 크기와 스타일을 제어할 수 있습니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `prompt` | STRING | 예 | - | 이미지 생성을 위한 프롬프트 (기본값: 빈 문자열). 최소 3자 이상이어야 합니다. |
| `model` | COMBO | 예 | `photon-flash-1`<br />`photon-1`<br />`photon` | 이미지 생성을 위한 모델 선택. 모델마다 비용이 다릅니다. |
| `aspect_ratio` | COMBO | 예 | `16:9`<br />`1:1`<br />`4:3`<br />`3:2`<br />`21:9`<br />`9:16`<br />`3:4`<br />`2:3`<br />`9:21` | 생성된 이미지의 종횡비 (기본값: `16:9`) |
| `seed` | INT | 예 | 0부터 18446744073709551615까지 | 노드가 다시 실행될지 여부를 결정하는 시드; 실제 결과는 시드와 상관없이 비결정적입니다 (기본값: 0) |
| `style_image_weight` | FLOAT | 아니요 | 0.0부터 1.0까지 | 스타일 이미지의 가중치. `style_image`가 제공되지 않으면 무시됩니다 (기본값: 1.0) |
| `image_luma_ref` | LUMA_REF | 아니요 | - | 생성 과정에 영향을 미치는 Luma 참조 노드 연결; 최대 4개의 이미지를 고려할 수 있습니다. |
| `style_image` | IMAGE | 아니요 | - | 스타일 참조 이미지; 단 1개의 이미지만 사용됩니다. |
| `character_image` | IMAGE | 아니요 | - | 캐릭터 참조 이미지; 여러 개의 배치로 제공 가능하며 최대 4개의 이미지를 고려할 수 있습니다. |

**매개변수 제약사항:**

- `prompt`는 공백을 제거한 후 최소 3자 이상이어야 합니다.
- `image_luma_ref` 매개변수는 최대 4개의 참조 이미지를 받을 수 있습니다.
- `character_image` 매개변수는 최대 4개의 캐릭터 참조 이미지를 받을 수 있습니다.
- `style_image` 매개변수는 단 1개의 스타일 참조 이미지만 받습니다.
- `style_image_weight` 매개변수는 `style_image`가 제공되었을 때만 사용됩니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `output` | IMAGE | 입력 매개변수를 기반으로 생성된 이미지입니다. |
