PerturbedAttentionGuidance 노드는 확산 모델에 교란된 주의 지도를 적용하여 생성 품질을 향상시킵니다. 이 노드는 샘플링 과정 중 모델의 자기 주의 메커니즘을 간소화된 버전으로 대체하며, 여기서는 값 프로젝션에 집중합니다. 이러한 기법은 조건부 노이즈 제거 과정을 조정함으로써 생성 이미지의 일관성과 품질을 개선하는 데 도움을 줍니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `model` | MODEL | 예 | - | 교란된 주의 지도를 적용할 확산 모델 |
| `scale` | FLOAT | 아니요 | 0.0 - 100.0 | 교란된 주의 지도 효과의 강도 (기본값: 3.0). 0으로 설정하면 노드가 아무런 영향을 미치지 않고 원래의 노이즈 제거 결과를 반환합니다. |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `model` | MODEL | 교란된 주의 지도가 적용된 수정된 모델 |
