Kling 가상 피팅 노드. 사람 이미지와 옷 이미지를 입력하여 사람에게 옷을 입혀 보세요. 여러 개의 의류 이미지를 하나의 흰색 배경 이미지로 합칠 수 있습니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `human_image` | IMAGE | 예 | - | 옷을 입혀 볼 사람 이미지 |
| `cloth_image` | IMAGE | 예 | - | 사람에게 입혀 볼 의류 이미지 |
| `model_name` | STRING | 예 | `"kolors-virtual-try-on-v1"` | 사용할 가상 피팅 모델 (기본값: "kolors-virtual-try-on-v1") |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `output` | IMAGE | 옷을 입힌 사람을 보여주는 결과 이미지 |
