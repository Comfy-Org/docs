PreviewAny 노드는 모든 입력 데이터 유형을 텍스트 형식으로 미리 보여줍니다. 이 노드는 어떤 데이터 유형이든 입력받아 읽기 쉬운 문자열 표현으로 변환하여 표시합니다. 노드는 문자열, 숫자, 불린 값, 복합 객체 등 다양한 데이터 유형을 자동으로 처리하며, 이를 JSON 형식으로 직렬화하려고 시도합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `source` | ANY | 예 | 어떤 데이터 유형이든 | 미리 보기 표시를 위해 어떤 입력 데이터 유형이든 수락합니다 |

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `UI 텍스트 표시` | STRING | 사용자 인터페이스에 입력 데이터를 텍스트 형식으로 변환해 표시합니다. 또한 추가 처리를 위해 텍스트를 문자열 출력으로 반환합니다. |
