SaveAudioOpus 노드는 오디오 데이터를 Opus 형식 파일로 저장합니다. 오디오 입력을 받아 구성 가능한 품질 설정으로 압축된 Opus 파일로 내보냅니다. 노드는 자동으로 파일 이름을 처리하고 지정된 출력 디렉토리에 결과물을 저장합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `audio` | AUDIO | 예 | - | Opus 파일로 저장할 오디오 데이터 |
| `filename_prefix` | STRING | 아니요 | - | 출력 파일명의 접두어 (기본값: "audio/ComfyUI") |
| `quality` | COMBO | 아니요 | "64k"<br />"96k"<br />"128k"<br />"192k"<br />"320k" | Opus 파일의 오디오 품질 설정 (기본값: "128k") |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| - | - | 이 노드는 출력 값을 반환하지 않습니다. 오디오 파일을 디스크에 저장하는 것이 주된 기능입니다. |
