Flux KV Cache 노드는 Flux 계열 모델에 대한 키-값(KV) 캐시 최적화를 가능하게 합니다. 이 최적화는 특정 연산을 캐싱함으로써 참조 이미지를 사용할 때 성능을 향상시키며, 이는 생성 과정을 더욱 빠르게 만듭니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `model` | MODEL | 예 | | KV 캐시 최적화를 적용할 모델입니다. |

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `model` | MODEL | KV 캐시 최적화가 활성화된 패치된 모델입니다. |
