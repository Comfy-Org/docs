케이스 변환기 노드는 텍스트 문자열을 다양한 대소문자 형식으로 변환합니다. 입력 문자열을 받아 선택된 모드에 따라 변환하며, 지정된 대소문자 형식이 적용된 출력 문자열을 생성합니다. 이 노드는 텍스트의 대소문자를 수정하는 네 가지 변환 옵션을 지원합니다.

## 입력

| 매개변수 | 설명 | 데이터 타입 | 필수 여부 | 범위 |
| --- | --- | --- | --- | --- |
| `string` | 다른 대소문자 형식으로 변환할 텍스트 문자열 | STRING | 예 | - |
| `mode` | 적용할 대소문자 변환 모드 (기본값: `"UPPERCASE"`) | STRING | 예 | `"UPPERCASE"`<br />`"lowercase"`<br />`"Capitalize"`<br />`"Title Case"` |

## 출력

| 출력 이름 | 설명 | 데이터 타입 |
| --- | --- | --- |
| `output` | 지정된 대소문자 형식으로 변환된 입력 문자열 | STRING |
