Case Converter 노드는 텍스트 문자열을 다양한 대소문자 형식으로 변환합니다. 입력 문자열을 선택한 모드에 따라 변환하여 지정된 대소문자 형식이 적용된 출력 문자열을 생성합니다. 이 노드는 텍스트의 대소문자를 변경하는 네 가지 서로 다른 대소문자 변환 옵션을 지원합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `string` | STRING | 예 | - | 다른 대소문자 형식으로 변환할 텍스트 문자열 |
| `mode` | STRING | 예 | `"UPPERCASE"`<br />`"lowercase"`<br />`"Capitalize"`<br />`"Title Case"` | 적용할 대소문자 변환 모드 (기본값: `"UPPERCASE"`) |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `output` | STRING | 지정된 대소문자 형식으로 변환된 입력 문자열 |
