> 이 문서는 AI로 생성되었습니다. 오류를 발견하거나 개선할 점이 있다면 언제든지 기여해 주세요! [GitHub에서 편집하기](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/StabilityTextToAudio/en.md)

텍스트 설명을 기반으로 고품질 음악과 사운드 효과를 생성합니다. 이 노드는 Stability AI의 오디오 생성 기술을 사용하여 텍스트 프롬프트를 바탕으로 오디오 콘텐츠를 생성합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `model` | COMBO | 예 | `"stable-audio-2.5"` | 사용할 오디오 생성 모델 (기본값: "stable-audio-2.5") |
| `prompt` | STRING | 예 | - | 오디오 콘텐츠 생성에 사용되는 텍스트 설명 (기본값: 빈 문자열) |
| `duration` | INT | 아니요 | 1~190 | 생성된 오디오의 지속 시간(초)을 제어합니다 (기본값: 190) |
| `seed` | INT | 아니요 | 0~4294967294 | 생성에 사용되는 난수 시드입니다 (기본값: 0) |
| `steps` | INT | 아니요 | 4~8 | 샘플링 단계 수를 제어합니다 (기본값: 8) |

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `audio` | AUDIO | 텍스트 프롬프트를 기반으로 생성된 오디오 파일 |
