ByteDance Seedream 4.5 및 5.0 노드는 최대 4K 해상도에서 통합된 텍스트-to-이미지 생성과 정밀한 단일 문장 편집 기능을 제공합니다. 텍스트 프롬프트를 통해 새로운 이미지를 생성하거나 텍스트 지침을 사용해 기존 이미지를 편집할 수 있습니다. 이 노드는 단일 이미지 생성과 관련된 여러 이미지의 순차적 생성을 모두 지원합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `model` | 문자열 | 예 | "seedream 5.0 lite"<br />"seedream-4-5-251128"<br />"seedream-4-0-250828" | 생성에 사용할 Seedream 모델입니다. |
| `prompt` | 문자열 | 예 | - | 이미지를 생성하거나 편집하는 데 사용할 텍스트 프롬프트입니다. 최소 1자 이상이어야 합니다. |
| `image` | 이미지 | 아니요 | - | 이미지 대 이미지 생성에 사용할 입력 이미지입니다. 단일 또는 다중 참조 생성 시 참조 이미지를 지정합니다. seedream-5-0-260128의 경우 최대 14개의 참조 이미지를, 다른 모델의 경우 최대 10개의 참조 이미지를 사용할 수 있습니다. |
| `size_preset` | 문자열 | 아니요 | 여러 옵션 제공 | 권장 크기를 선택하세요. 'Custom'을 선택하면 아래의 너비와 높이를 사용합니다. 기본값: RECOMMENDED_PRESETS_SEEDREAM_4의 첫 번째 사전 설정. |
| `width` | 정수 | 아니요 | 1024부터 6240까지 (단계 2) | 이미지의 맞춤 너비입니다. 값은 `size_preset`이 'Custom'으로 설정된 경우에만 작동합니다. 기본값: 2048. |
| `height` | 정수 | 아니요 | 1024부터 4992까지 (단계 2) | 이미지의 맞춤 높이입니다. 값은 `size_preset`이 'Custom'으로 설정된 경우에만 작동합니다. 기본값: 2048. |
| `sequential_image_generation` | 문자열 | 아니요 | "disabled"<br />"auto" | 이미지 생성 모드입니다. "disabled"는 단일 이미지를 생성합니다. "auto"는 모델이 관련된 여러 이미지를 생성할지 여부를 결정하도록 합니다(예: 스토리 장면, 캐릭터 변형). 기본값: "disabled". |
| `max_images` | 정수 | 아니요 | 1부터 15까지 (단계 1) | sequential_image_generation='auto'일 때 생성할 최대 이미지 수입니다. 입력 이미지와 생성된 이미지의 총합은 15개를 초과할 수 없습니다. 기본값: 1. |
| `seed` | 정수 | 아니요 | 0부터 2147483647까지 (단계 1) | 생성에 사용할 시드입니다. 기본값: 0. |
| `watermark` | 부울 | 아니요 | - | 이미지에 "AI generated" 워터마크를 추가할지 여부입니다. 기본값: False. |
| `fail_on_partial` | 부울 | 아니요 | - | 활성화된 경우 요청된 이미지 중 일부가 없으면 실행을 중단하거나 오류를 반환합니다. 기본값: True. |

**매개변수 제약사항 참고:**
- 최소 이미지 해상도는 선택한 모델에 따라 달라집니다: seedream-4-5 및 seedream-5-0 모델의 경우 3.68MP, seedream-4-0 모델의 경우 0.92MP입니다.
- 최대 이미지 해상도는 seedream-5-0 모델의 경우 10.4MP이고, 다른 모델의 경우 16.78MP입니다.
- 참조 이미지는 가로세로 비율이 1:3에서 3:1 사이여야 합니다.
- `sequential_image_generation`이 "auto"로 설정된 경우, 입력 이미지 수와 `max_images`의 합은 15개를 초과할 수 없습니다.
- `width`와 `height` 매개변수는 `size_preset`이 'Custom'으로 설정된 경우에만 사용됩니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-----------|-----------|-------------|
| `IMAGE` | 이미지 | 입력 매개변수와 프롬프트를 기반으로 생성된 이미지입니다. 단일 이미지 텐서를 반환하거나, 여러 이미지를 생성한 경우 이미지 텐서의 배치를 반환합니다. |
