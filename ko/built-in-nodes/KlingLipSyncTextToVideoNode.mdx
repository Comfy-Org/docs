Kling Lip Sync Text to Video Node는 비디오 파일의 입술 움직임을 텍스트 프롬프트와 일치하도록 동기화합니다. 입력 비디오를 받아 캐릭터의 입술 움직임이 제공된 텍스트와 정확히 맞춰진 새 비디오를 생성합니다. 이 노드는 음성 합성을 사용해 자연스러운 말소리 동기화를 만들어냅니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `video` | VIDEO | 예 | - | 입술 동기화용 입력 비디오 파일. 비디오는 높이/너비가 720px에서 1920px 사이, 길이는 2초에서 10초 사이여야 하며, 크기는 100MB를 초과해서는 안 됩니다. |
| `text` | STRING | 예 | - | 입술 동기화 비디오 생성용 텍스트 내용. 모드가 text2video일 때 필수입니다. 최대 길이는 120자입니다. |
| `voice` | COMBO | 아니요 | "Melody"<br />"Sunny"<br />"Sage"<br />"Ace"<br />"Blossom"<br />"Peppy"<br />"Dove"<br />"Shine"<br />"Anchor"<br />"Lyric"<br />"Tender"<br />"Siren"<br />"Zippy"<br />"Bud"<br />"Sprite"<br />"Candy"<br />"Beacon"<br />"Rock"<br />"Titan"<br />"Grace"<br />"Helen"<br />"Lore"<br />"Crag"<br />"Prattle"<br />"Hearth"<br />"The Reader"<br />"Commercial Lady"<br />"양광소년"<br />"철부지동생"<br />"운동소년"<br />"청춘소녀"<br />"상냥한 여동생"<br />"활기찬 소녀"<br />"양광남자"<br />"유머러스한 형"<br />"문예형"<br />"달콤한 이웃집"<br />"상냥한 언니"<br />"직장인 여성청년"<br />"발랄한 남아"<br />"귀여운 여아"<br />"중후한 아빠"<br />"상냥한 엄마"<br />"진지한 상사"<br />"우아한 귀부인"<br />"자상한 할아버지"<br />"잔소리하는 할아버지"<br />"잔소리하는 할머니"<br />"친절한 할머니"<br />"둥북로테"<br />"충칭남자"<br />"쓰촨여자"<br />"차오산아저씨"<br />"대만남자"<br />"시안주인"<br />"톈진언니"<br />"뉴스앵커남자"<br />"더빙남자"<br />"애교있는 여자친구"<br />"칼날같은 목소리"<br />"착한 정태" | 입술 동기화 음성 선택 (기본값: "Melody"). 영어와 중국어 음성 옵션 포함. |
| `voice_speed` | FLOAT | 아니요 | 0.8-2.0 | 말 속도. 유효 범위: 0.8~2.0, 소수점 한 자리까지 정확함. (기본값: 1) |

**비디오 요구사항:**

- 비디오 파일 크기는 100MB를 초과해서는 안 됩니다.
- 높이/너비는 720px에서 1920px 사이여야 합니다.
- 길이는 2초에서 10초 사이여야 합니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `output` | VIDEO | 입술 동기화된 음성이 포함된 생성된 비디오 |
| `video_id` | STRING | 생성된 비디오의 고유 식별자 |
| `duration` | STRING | 생성된 비디오의 길이 정보 |
