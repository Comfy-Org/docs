텍스트 프롬프트와 출력 설정에 따라 동기적으로 비디오를 생성합니다. 이 노드는 텍스트 설명과 다양한 생성 매개변수를 사용해 비디오 콘텐츠를 만들며, 생성 과정이 완료되면 최종 비디오 출력을 생성합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `prompt` | 문자열 | 예 | - | 비디오 생성을 위한 프롬프트 (기본값: 빈 문자열). 최소 3자 이상이어야 합니다. |
| `model` | 콤보 | 예 | `"ray_1_6"`<br />`"ray_2"` | 사용할 비디오 생성 모델입니다. |
| `aspect_ratio` | 콤보 | 예 | `"16:9"`<br />`"9:16"`<br />`"1:1"`<br />`"4:3"`<br />`"3:4"`<br />`"21:9"`<br />`"9:21"` | 생성될 비디오의 종횡비 (기본값: "16:9"). |
| `resolution` | 콤보 | 예 | `"540p"`<br />`"720p"`<br />`"1080p"` | 비디오의 출력 해상도 (기본값: "540p"). `ray_1_6` 모델을 사용할 경우 이 매개변수는 무시됩니다. |
| `duration` | 콤보 | 예 | `"5s"`<br />`"9s"` | 생성될 비디오의 지속시간. `ray_1_6` 모델을 사용할 경우 이 매개변수는 무시됩니다. |
| `loop` | 부울 | 예 | - | 비디오를 반복 재생할지 여부 (기본값: False). |
| `seed` | 정수 | 예 | 0부터 18446744073709551615까지 | 노드가 다시 실행될지 여부를 결정하는 시드입니다. 실제 결과는 시드와 관계없이 비결정적입니다 (기본값: 0). |
| `luma_concepts` | 맞춤형 | 아니요 | - | Luma Concepts 노드를 통해 카메라 움직임을 지시하는 선택적 카메라 개념입니다. |

**참고:** `ray_1_6` 모델을 사용할 경우 `duration` 및 `resolution` 매개변수는 자동으로 무시되며, 생성에 영향을 미치지 않습니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `output` | 비디오 | 생성된 비디오 파일입니다. |
