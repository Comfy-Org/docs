SDPoseDrawKeypoints 노드는 자세 추정 데이터(키포인트)를 받아 빈 캔버스에 시각적 골격으로 그립니다. 신체, 손, 얼굴, 발 등 자세의 다양한 부분을 선택적으로 그릴 수 있으며, 선 폭과 점 크기를 맞춤 설정할 수 있습니다. 결과 이미지는 시각화용으로 사용하거나, 자세 이미지를 필요로 하는 다른 노드의 입력으로 활용할 수 있습니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `keypoints` | POSE_KEYPOINT | 예 | - | 그려야 할 자세 키포인트 데이터입니다. 일반적으로 자세 검출 노드에서 출력된 데이터입니다. |
| `draw_body` | BOOLEAN | 아니요 | - | 메인 신체 골격을 그릴지 여부를 제어합니다 (기본값: True). |
| `draw_hands` | BOOLEAN | 아니요 | - | 손 키포인트를 그릴지 여부를 제어합니다 (기본값: True). |
| `draw_face` | BOOLEAN | 아니요 | - | 얼굴 키포인트를 그릴지 여부를 제어합니다 (기본값: True). |
| `draw_feet` | BOOLEAN | 아니요 | - | 발 키포인트를 그릴지 여부를 제어합니다 (기본값: False). |
| `stick_width` | INT | 아니요 | 1~10 | 신체 골격을 그리는 라인의 폭입니다 (기본값: 4). |
| `face_point_size` | INT | 아니요 | 1~10 | 얼굴 키포인트를 그리는 점의 크기입니다 (기본값: 3). |
| `score_threshold` | FLOAT | 아니요 | 0.0~1.0 | 그려야 할 키포인트가 가져야 하는 최소 신뢰도 점수입니다. 이 값보다 낮은 점수를 가진 키포인트는 무시됩니다 (기본값: 0.3). |

**참고:** `keypoints` 입력이 비어 있거나 `None`인 경우, 노드는 빈 64x64 이미지를 출력합니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `output` | IMAGE | 그려진 자세 키포인트가 포함된 이미지입니다. 이미지의 크기는 입력 키포인트 데이터에 지정된 `canvas_height`와 `canvas_width`와 일치합니다. |
