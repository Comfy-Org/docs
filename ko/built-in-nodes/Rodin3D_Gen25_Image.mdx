## 개요

이 노드는 Rodin Gen-2.5 API를 사용하여 하나에서 다섯 장의 참조 이미지로부터 3D 모델을 생성합니다. 빠른 속도, 일반, 극고화질 모드 중에서 선택하여 생성 속도와 비용 간의 균형을 맞출 수 있습니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `images` | IMAGE | 예 | 1~5개 이미지 | 하나에서 다섯 장의 입력 이미지입니다. 여러 이미지를 제공할 경우 첫 번째 이미지는 재료로 사용됩니다. |
| `mode` | COMBO | 예 | `"빠른"`<br />`"일반"`<br />`"극고화질"` | 생성 품질 모드입니다. 고화질 모드일수록 더 나은 결과를 얻을 수 있지만 비용이 더 높습니다. |
| `material` | COMBO | 예 | `"PBR"`<br />`"무광"` | 생성된 3D 모델의 재료 유형입니다. |
| `geometry_file_format` | COMBO | 예 | `"glb"`<br />`"obj"`<br />`"stl"`<br />`"usdz"` | 3D 모델 지오메트리의 출력 파일 형식입니다. |
| `texture_mode` | COMBO | 예 | `"원본"`<br />`"클린"`<br />`"스타일"` | 텍스처 생성 모드입니다. "원본"은 입력 텍스처를 그대로 유지하고, "클린"은 이를 제거하며, "스타일"은 스타일화된 텍스처를 적용합니다. |
| `seed` | INT | 예 | 0~2147483647 | 재현 가능한 결과를 위해 사용하는 임의의 시드값입니다. 동일한 시드를 사용하면 동일한 출력을 얻을 수 있습니다. |
| `TAPose` | BOOLEAN | 예 | True / False | 생성된 모델에 T자세를 적용할지 여부입니다. |
| `hd_texture` | BOOLEAN | 예 | True / False | 고화질 텍스처 맵을 생성할지 여부입니다. |
| `texture_delight` | BOOLEAN | 예 | True / False | 텍스처 생성 전에 입력 이미지의 조명을 제거할지 여부입니다. |
| `use_original_alpha` | BOOLEAN | 예 | True / False | 입력 이미지의 원본 알파 채널을 사용할지 여부입니다. |
| `addon_highpack` | BOOLEAN | 예 | True / False | 표준 버전과 함께 고다각형 버전의 모델을 생성할지 여부입니다. |
| `bbox_width` | INT | 예 | 1~1000 | 생성된 모델의 경계상자의 너비(센티미터 단위)입니다. |
| `bbox_height` | INT | 예 | 1~1000 | 생성된 모델의 경계상자의 높이(센티미터 단위)입니다. |
| `bbox_length` | INT | 예 | 1~1000 | 생성된 모델의 경 boundary상자의 길이(센티미터 단위)입니다. |
| `height_cm` | INT | 예 | 1~300 | 생성된 모델의 높이(센티미터 단위)입니다. |

**이미지 개수 관련 참고사항:** 노드는 1~5개의 이미지를 받습니다. 이미지 일괄 처리(예: 4장짜리 일괄 처리)를 제공할 경우, 일괄 처리 내 각 이미지는 별도의 입력 이미지로 취급됩니다. 5개 이상의 이미지를 제공하면 오류가 발생합니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-----------|-----------|-------------|
| `model_file` | FILE3D | 선택한 지오메트리 형식으로 생성된 3D 모델 파일입니다. |
