> 이 문서는 AI로 생성되었습니다. 오류를 발견하거나 개선할 점이 있다면 언제든지 기여해 주세요! [GitHub에서 편집하기](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/MakeTrainingDataset/en.md)

이 노드는 이미지와 텍스트를 인코딩하여 학습용 데이터를 준비합니다. 이미지 목록과 해당하는 텍스트 캡션 목록을 입력받아 VAE 모델을 사용해 이미지를 잠재 표현으로 변환하고, CLIP 모델을 사용해 텍스트를 조건부 데이터로 변환합니다. 결과로 생성된 짝지어진 잠재 표현과 조건부 데이터는 리스트 형태로 출력되며, 학습 워크플로우에서 바로 사용할 수 있습니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `images` | IMAGE | 예 | N/A | 인코딩할 이미지 목록입니다. |
| `vae` | VAE | 예 | N/A | 이미지를 잠재 표현으로 인코딩하는 VAE 모델입니다. |
| `clip` | CLIP | 예 | N/A | 텍스트를 조건부 데이터로 인코딩하는 CLIP 모델입니다. |
| `texts` | STRING | 아니요 | N/A | 텍스트 캡션 목록입니다. 길이는 n(이미지와 일치), 1(모든 이미지에 동일 적용), 또는 생략 가능하며 이 경우 빈 문자열이 사용됩니다. |

**매개변수 제약사항:**

* `texts` 목록의 항목 수는 0, 1이거나 `images` 목록의 항목 수와 정확히 일치해야 합니다. 0인 경우 모든 이미지에 빈 문자열이 사용됩니다. 1인 경우 그 하나의 텍스트가 모든 이미지에 반복 적용됩니다.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `latents` | LATENT | 잠재 표현 딕트 목록입니다. |
| `conditioning` | CONDITIONING | 조건부 리스트 목록입니다. |
