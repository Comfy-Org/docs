> 이 문서는 AI로 생성되었습니다. 오류를 발견하거나 개선할 점이 있다면 언제든지 기여해 주세요! [GitHub에서 편집하기](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/ViduImageToVideoNode/en.md)

Vidu 이미지 비디오 생성 노드는 시작 이미지와 선택적 텍스트 설명을 통해 짧은 동영상을 생성합니다. 이 노드는 제공된 이미지 프레임에서 이어지는 비디오 콘텐츠를 생성하는 AI 모델을 사용하며, 결과 비디오를 반환합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `model` | COMBO | 예 | `viduq1` | 모델 이름 (기본값: viduq1) |
| `image` | IMAGE | 예 | - | 생성될 비디오의 시작 프레임으로 사용할 이미지 |
| `prompt` | STRING | 아니요 | - | 비디오 생성을 위한 텍스트 설명 (기본값: 빈 문자열) |
| `duration` | INT | 아니요 | 5-5 | 출력 비디오의 지속 시간(초) (기본값: 5, 고정 5초) |
| `seed` | INT | 아니요 | 0-2147483647 | 비디오 생성을 위한 시드 값 (랜덤일 경우 0) (기본값: 0) |
| `resolution` | COMBO | 아니요 | `1080p` | 지원되는 값은 모델 및 지속 시간에 따라 다를 수 있습니다 (기본값: 1080p) |
| `movement_amplitude` | COMBO | 아니요 | `auto`<br />`small`<br />`medium`<br />`large` | 프레임 내 객체의 움직임 진폭 (기본값: auto) |

**제약사항:**

- 입력 이미지는 하나만 허용됩니다(여러 이미지를 처리할 수 없습니다).
- 입력 이미지의 종횡비는 1:4에서 4:1 사이여야 합니다.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `output` | VIDEO | 생성된 비디오 출력 |
