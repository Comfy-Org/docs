> 이 문서는 AI로 생성되었습니다. 오류를 발견하거나 개선할 점이 있다면 언제든지 기여해 주세요! [GitHub에서 편집하기](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/RecraftImageInpaintingNode/en.md)

이 노드는 텍스트 프롬프트와 마스크를 기반으로 이미지의 특정 영역을 수정합니다. Recraft API를 사용하여 마스크로 지정된 영역만 지능적으로 편집하고, 이미지의 나머지 부분은 변경되지 않도록 유지합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `image` | IMAGE | 예 | - | 수정할 입력 이미지 |
| `mask` | MASK | 예 | - | 이미지의 어느 영역을 수정할지 정의하는 마스크 |
| `prompt` | STRING | 예 | - | 이미지 생성을 위한 프롬프트 (기본값: 빈 문자열, 최대 길이: 1000자) |
| `n` | INT | 예 | 1-6 | 생성할 이미지 수 (기본값: 1, 최소: 1, 최대: 6) |
| `seed` | INT | 예 | 0-18446744073709551615 | 노드가 다시 실행될지 여부를 결정하는 시드. 실제 결과는 시드와 관계없이 비결정적임 (기본값: 0) |
| `recraft_style` | STYLEV3 | 선택 | - | Recraft API의 선택적 스타일 매개변수. 제공되지 않을 경우 기본값은 "realistic_image" 스타일입니다 |
| `negative_prompt` | STRING | 선택 | - | 이미지에서 원하지 않는 요소에 대한 선택적 텍스트 설명 (기본값: 빈 문자열) |

*참고: 인페인팅 작업을 위해 `image`와 `mask`는 함께 제공되어야 합니다. 마스크는 이미지 크기에 맞게 자동으로 조정됩니다. `prompt`는 유효성 검사가 이루어지며 최대 길이는 1000자입니다. Infinite Style Library의 스타일 ID를 사용할 경우, 벡터 아트 스타일이 아닌지 확인하세요. 그렇지 않으면 API가 이미지 대신 SVG 데이터를 반환할 수 있습니다.*

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `image` | IMAGE | 프롬프트와 마스크를 기반으로 생성된 수정된 이미지들입니다. 입력 이미지 수에 `n` 매개변수를 곱한 만큼의 이미지를 반환합니다 |
