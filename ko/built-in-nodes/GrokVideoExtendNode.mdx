Grok Video Extend 노드는 AI 모델을 사용하여 기존 비디오의 끊김 없는 연속 영상을 생성합니다. 짧은 비디오와 다음에 일어나야 할 일을 설명하는 텍스트 프롬프트를 제공하면, 노드는 원본 비디오에 이어지는 새로운 비디오 클립을 생성합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `prompt` | 문자열 | 예 | 없음 | 비디오에서 다음에 일어나야 할 일을 설명하는 텍스트입니다. |
| `video` | 비디오 | 예 | 없음 | 확장할 소스 비디오입니다. MP4 형식, 2~15초 길이. |
| `model` | 콤보 | 예 | `"grok-imagine-video"` | 비디오 확장을 위해 사용할 모델입니다. 선택 시 중첩된 `duration` 매개변수가 나타납니다. |
| `seed` | 정수 | 아니요 | 0~2147483647 | 노드가 다시 실행될지 여부를 결정하는 시드입니다. 실제 결과는 시드와 관계없이 비결정적입니다(기본값: 0). |

**매개변수 제약사항:**
*   `video` 입력은 길이가 2~15초 사이이고 파일 크기가 50MB를 초과하지 않는 MP4 파일이어야 합니다.
*   `prompt`에는 최소 한 글자 이상이 포함되어야 합니다(공백은 제거됨).
*   `model` 매개변수는 동적 콤보입니다. "grok-imagine-video" 옵션을 선택하면 중첩된 `duration` 매개변수가 나타나며, 이는 확장 길이를 초 단위로 제어합니다(기본값: 8, 범위: 2~10).

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-----------|-----------|-------------|
| `output` | 비디오 | 새로 생성된 비디오 확장판입니다. |
