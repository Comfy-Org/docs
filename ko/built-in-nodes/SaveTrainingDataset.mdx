> 이 문서는 AI로 생성되었습니다. 오류를 발견하거나 개선할 점이 있다면 언제든지 기여해 주세요! [GitHub에서 편집하기](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/SaveTrainingDataset/en.md)

이 노드는 준비된 트레이닝 데이터셋을 컴퓨터의 하드 드라이브에 저장합니다. 이미지 래티언트와 그에 해당하는 텍스트 컨디셔닝을 포함한 인코딩된 데이터를 받아들여, 관리가 용이하도록 샤드라고 불리는 여러 개의 작은 파일로 정리합니다. 노드는 자동으로 출력 디렉토리에 폴더를 생성하고, 데이터 파일과 데이터셋을 설명하는 메타데이터 파일을 함께 저장합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `latents` | LATENT | 예 | N/A | MakeTrainingDataset에서 가져온 래티언트 딕트 리스트. |
| `conditioning` | CONDITIONING | 예 | N/A | MakeTrainingDataset에서 가져온 컨디셔닝 리스트 리스트. |
| `folder_name` | STRING | 아니요 | N/A | 데이터셋을 저장할 폴더 이름 (출력 디렉토리 내). (기본값: "training_dataset") |
| `shard_size` | INT | 아니요 | 1부터 100000까지 | 샤드 파일당 샘플 수. (기본값: 1000) |

**참고:** `latents` 리스트의 항목 수와 `conditioning` 리스트의 항목 수가 정확히 일치해야 합니다. 이 두 개의 카운트가 맞지 않으면 노드는 오류를 발생시킵니다.

## 출력

이 노드는 출력 데이터를 생성하지 않습니다. 노드의 역할은 파일을 디스크에 저장하는 것입니다.
