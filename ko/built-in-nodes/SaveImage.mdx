SaveImage 노드는 수신한 이미지를 `ComfyUI/output` 디렉터리에 저장합니다. 각 이미지를 PNG 파일로 저장하며, 저장된 파일에 프롬프트와 같은 워크플로우 메타데이터를 포함시켜 추후 참조할 수 있도록 합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `images` | IMAGE | 예 | - | 저장할 이미지입니다. |
| `filename_prefix` | STRING | 예 | - | 저장할 파일의 접두어입니다. 여기에는 `%date:yyyy-MM-dd%`나 `%Empty Latent Image.width%`와 같은 형식 정보를 포함하여 노드의 값도 포함할 수 있습니다(기본값: "ComfyUI"). |

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `ui` | UI_RESULT | 이 노드는 저장된 이미지의 파일명과 하위 폴더를 포함한 목록을 담은 UI 결과를 출력합니다. 다른 노드와 연결하기 위한 데이터는 출력되지 않습니다. |
