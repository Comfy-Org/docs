> 이 문서는 AI로 생성되었습니다. 오류를 발견하거나 개선할 점이 있다면 언제든지 기여해 주세요! [GitHub에서 편집하기](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/KlingOmniProTextToVideoNode/en.md)

이 노드는 최신 Kling AI 모델을 사용하여 텍스트 설명으로 비디오를 생성합니다. 프롬프트를 원격 API로 전송한 후 생성된 비디오를 반환합니다. 이 노드를 통해 비디오의 길이, 형태, 품질 등을 제어하고 멀티샷 스토리보드를 만들 수도 있습니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `model_name` | COMBO | 예 | `"kling-v3-omni"`<br />`"kling-video-o1"` | 비디오 생성에 사용할 특정 Kling 모델 (기본값: `"kling-v3-omni"`). |
| `prompt` | STRING | 예 | 0~2500자 | 비디오 내용을 설명하는 텍스트 프롬프트입니다. 긍정적·부정적 설명을 모두 포함할 수 있습니다. 스토리보드가 활성화된 경우 무시됩니다. |
| `aspect_ratio` | COMBO | 예 | `"16:9"`<br />`"9:16"`<br />`"1:1"` | 생성할 비디오의 형태 또는 크기입니다. |
| `duration` | INT | 예 | 3~15초 | 비디오의 길이(초 단위, 기본값: 5). |
| `resolution` | COMBO | 아니요 | `"4k"`<br />`"1080p"`<br />`"720p"` | 비디오의 품질 또는 화소 해상도 (기본값: `"1080p"`). |
| `storyboards` | DYNAMIC_COMBO | 아니요 | `"비활성화"`<br />`"1개 스토리보드"`<br />`"2개 스토리보드"`<br />`"3개 스토리보드"`<br />`"4개 스토리보드"`<br />`"5개 스토리보드"`<br />`"6개 스토리보드"` | 각각의 프롬프트와 지속시간을 가진 일련의 비디오 세그먼트를 생성합니다. o1 모델에서는 무시됩니다. |
| `generate_audio` | BOOLEAN | 아니요 | True / False | 비디오에 오디오를 생성할지 여부 (기본값: False). |
| `seed` | INT | 아니요 | 0~2147483647 | 시드는 노드가 다시 실행될지 여부를 제어하며, 시드와 상관없이 결과는 비결정적입니다 (기본값: 0). |

### 매개변수 제약사항 및 한계

- **모델별 제한사항:**
  - `kling-video-o1` 모델은 **5초 또는 10초**의 지속시간만 지원합니다.
  - `kling-video-o1` 모델은 **오디오 생성을 지원하지 않습니다.**
  - `kling-video-o1` 모델은 **4k 해상도를 지원하지 않습니다.**
  - `kling-video-o1` 모델은 **스토리보드를 지원하지 않습니다.**
- **스토리보드 제약사항:**
  - 스토리보드가 활성화된 경우 `prompt` 필드는 무시됩니다.
  - 각 스토리보드는 고유한 프롬프트(1~512자)와 지속시간을 필요로 합니다.
  - 모든 스토리보드의 총 지속시간은 반드시 전체 `duration` 매개변수와 정확히 일치해야 합니다.
- **프롬프트 요구사항:**
  - 스토리보드가 **비활성화된 경우**, `prompt` 필드는 필수입니다(최소 1자).
  - 스토리보드가 **활성화된 경우**, `prompt` 필드는 비워둘 수 있습니다(0자).

| `output` | VIDEO | 제공된 텍스트 프롬프트와 설정에 따라 생성된 비디오입니다. |
