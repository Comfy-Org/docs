Ideogram V3 노드는 Ideogram V3 모델을 사용해 이미지를 생성합니다. 텍스트 프롬프트로부터 일반적인 이미지 생성과 이미지와 마스크가 모두 제공될 경우 이미지 편집을 지원합니다. 이 노드는 종횡비, 해상도, 생성 속도 및 선택적 문자 참조 이미지에 대한 다양한 제어 옵션을 제공합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `prompt` | STRING | 예 | - | 이미지 생성 또는 편집을 위한 프롬프트 (기본값: 비어 있음) |
| `image` | IMAGE | 아니요 | - | 이미지 편집을 위한 선택적 참조 이미지 |
| `mask` | MASK | 아니요 | - | 인페인팅을 위한 선택적 마스크 (흰색 영역은 교체됨) |
| `aspect_ratio` | COMBO | 아니요 | "1:1"<br />"1:3"<br />"3:1"<br />"1:2"<br />"2:1"<br />"9:16"<br />"16:9"<br />"10:16"<br />"16:10"<br />"2:3"<br />"3:2"<br />"3:4"<br />"4:3"<br />"4:5"<br />"5:4" | 이미지 생성 시 종횡비입니다. 해상도가 Auto로 설정되지 않은 경우 무시됩니다 (기본값: "1:1") |
| `resolution` | COMBO | 아니요 | "Auto"<br />"512x1536"<br />"576x1408"<br />"576x1472"<br />"576x1536"<br />"640x1344"<br />"640x1408"<br />"640x1472"<br />"640x1536"<br />"704x1152"<br />"704x1216"<br />"704x1280"<br />"704x1344"<br />"704x1408"<br />"704x1472"<br />"736x1312"<br />"768x1088"<br />"768x1216"<br />"768x1280"<br />"768x1344"<br />"800x1280"<br />"832x960"<br />"832x1024"<br />"832x1088"<br />"832x1152"<br />"832x1216"<br />"832x1248"<br />"864x1152"<br />"896x960"<br />"896x1024"<br />"896x1088"<br />"896x1120"<br />"896x1152"<br />"960x832"<br />"960x896"<br />"960x1024"<br />"960x1088"<br />"1024x832"<br />"1024x896"<br />"1024x960"<br />"1024x1024"<br />"1088x768"<br />"1088x832"<br />"1088x896"<br />"1088x960"<br />"1120x896"<br />"1152x704"<br />"1152x832"<br />"1152x864"<br />"1152x896"<br />"1216x704"<br />"1216x768"<br />"1216x832"<br />"1248x832"<br />"1280x704"<br />"1280x768"<br />"1280x800"<br />"1312x736"<br />"1344x640"<br />"1344x704"<br />"1344x768"<br />"1408x576"<br />"1408x640"<br />"1408x704"<br />"1472x576"<br />"1472x640"<br />"1472x704"<br />"1536x512"<br />"1536x576"<br />"1536x640" | 이미지 생성 시 해상도입니다. Auto로 설정되지 않은 경우, aspect_ratio 설정을 무시합니다 (기본값: "Auto") |
| `magic_prompt_option` | COMBO | 아니요 | "AUTO"<br />"ON"<br />"OFF" | 생성 시 MagicPrompt 사용 여부 결정 (기본값: "AUTO") |
| `seed` | INT | 아니요 | 0-2147483647 | 생성을 위한 랜덤 시드 (기본값: 0) |
| `num_images` | INT | 아니요 | 1-8 | 생성할 이미지 수 (기본값: 1) |
| `rendering_speed` | COMBO | 아니요 | "DEFAULT"<br />"TURBO"<br />"QUALITY" | 생성 속도와 품질 간의 균형 조절 (기본값: "DEFAULT") |
| `character_image` | IMAGE | 아니요 | - | 문자 참조용 이미지 |
| `character_mask` | MASK | 아니요 | - | 문자 참조 이미지용 선택적 마스크 |

**매개변수 제약사항:**

- `image`와 `mask`가 모두 제공되면 노드는 편집 모드로 전환됩니다.
- `image`나 `mask` 중 하나만 제공된 경우 오류가 발생합니다.
- `character_mask`는 `character_image`가 존재해야 합니다.
- `resolution`이 "Auto"로 설정되지 않은 경우 `aspect_ratio` 매개변수는 무시됩니다.
- 마스크의 흰색 영역은 인페인팅 시 교체됩니다.
- 문자 마스크와 문자 이미지는 반드시 같은 크기여야 합니다.
