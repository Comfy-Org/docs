UNetTemporalAttentionMultiply 노드는 시계열 UNet 모델에서 다양한 유형의 주의 메커니즘에 곱셈 요인을 적용합니다. 이 노드는 자기 주의와 교차 주의 레이어의 가중치를 조정하여 모델을 수정하며, 구조적 구성과 시간적 구성 간의 차이를 구분합니다. 이를 통해 각 주의 유형이 모델 출력에 미치는 영향력을 세밀하게 조정할 수 있습니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `model` | MODEL | 예 | - | 주의 곱셈자로 수정할 입력 모델 |
| `self_structural` | FLOAT | 아니요 | 0.0 - 10.0 | 자기 주의 구조적 구성 요소에 대한 곱셈자 (기본값: 1.0) |
| `self_temporal` | FLOAT | 아니요 | 0.0 - 10.0 | 자기 주의 시간적 구성 요소에 대한 곱셈자 (기본값: 1.0) |
| `cross_structural` | FLOAT | 아니요 | 0.0 - 10.0 | 교차 주의 구조적 구성 요소에 대한 곱셈자 (기본값: 1.0) |
| `cross_temporal` | FLOAT | 아니요 | 0.0 - 10.0 | 교차 주의 시간적 구성 요소에 대한 곱셈자 (기본값: 1.0) |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `model` | MODEL | 주의 가중치가 조정된 수정된 모델 |
