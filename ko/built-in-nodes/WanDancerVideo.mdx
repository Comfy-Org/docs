WanDancerVideo 노드는 WanDancer 모델을 이용한 비디오 생성을 위해 조건부 데이터와 빈 잠재 텐서를 준비합니다. 시작 이미지, 마스크, CLIP 비전 임베딩, 오디오 특징과 같은 선택적 입력을 통해 긍정적 및 부정적 조건부를 결합하여 생성된 비디오를 제어합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `positive` | CONDITIONING | 예 | | 비디오 생성을 안내하는 긍정적 조건부입니다. |
| `negative` | CONDITIONING | 예 | | 비디오 생성을 안내하는 부정적 조건부입니다. |
| `vae` | VAE | 예 | | 시작 이미지를 잠재 공간으로 인코딩하는 데 사용되는 VAE입니다. |
| `width` | INT | 예 | 16부터 MAX_RESOLUTION까지 (단계: 16) | 생성된 비디오의 너비(픽셀 단위, 기본값: 480). |
| `height` | INT | 예 | 16부터 MAX_RESOLUTION까지 (단계: 16) | 생성된 비디오의 높이(픽셀 단위, 기본값: 832). |
| `length` | INT | 예 | 1부터 MAX_RESOLUTION까지 (단계: 4) | 생성된 비디오의 프레임 수입니다. WanDancer에서는 149로 유지해야 합니다(기본값: 149). |
| `clip_vision_output` | CLIP_VISION_OUTPUT | 아니요 | | 첫 번째 프레임의 CLIP 비전 임베딩입니다. |
| `clip_vision_output_ref` | CLIP_VISION_OUTPUT | 아니요 | | 참조 이미지의 CLIP 비전 임베딩입니다. |
| `start_image` | IMAGE | 아니요 | | 인코딩할 초기 이미지입니다. 지정된 `length`까지 최대 프레임 수로 제공할 수 있습니다. |
| `mask` | MASK | 아니요 | | 시작 이미지의 이미지 조건부 마스크입니다. 흰색 영역은 유지되고, 검은색 영역은 재생성됩니다. 지역 생성에 사용됩니다. |
| `audio_encoder_output` | AUDIO_ENCODER_OUTPUT | 아니요 | | 오디오 인코더의 출력으로, 오디오 특징, fps 및 오디오 조건부 생성을 위한 주입 스케일을 제공합니다. |

**매개변수 제약 사항 참고:**
- `start_image`와 `mask` 입력은 선택적이지만 함께 사용할 수 있습니다. `start_image`가 제공되면 인코딩되어 잠재 텐서와 연결됩니다. 또한 `mask`가 제공되면 시작 이미지의 어느 부분을 유지하고(흰색), 어느 부분을 재생성할지(검은색) 제어합니다. `mask`가 제공되지 않으면 시작 이미지 전체 영역이 조건부 안내로 사용됩니다.
- `clip_vision_output`와 `clip_vision_output_ref` 입력은 선택적이며, 첫 번째 프레임과 참조 이미지에 대한 시각적 맥락을 제공하기 위해 함께 사용할 수 있습니다.
- `audio_encoder_output` 입력은 선택적이며, 오디오 조건부 생성을 위한 오디오 특징을 제공합니다.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `positive` | CONDITIONING | 추가 데이터(잠재 텐서 연결, CLIP 비전, 오디오)가 첨부된 긍정적 조건부입니다. |
| `negative` | CONDITIONING | 추가 데이터(잠재 텐서 연결, CLIP 비전, 오디오)가 첨부된 부정적 조건부입니다. |
| `latent` | LATENT | 지정된 비디오 길이, 높이 및 너비와 일치하는 차원을 가진 빈 잠재 텐서입니다. |
