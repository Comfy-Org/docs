## 개요

이 노드는 Google의 Vertex AI API에 텍스트 프롬프트를 보내 이미지를 생성하거나 편집합니다. 특정 Gemini 모델을 사용하여 새로운 이미지를 만들거나 기존 이미지를 지시사항에 따라 수정합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `prompt` | 문자열 | 예 | 없음 | 생성할 이미지 또는 적용할 편집 내용을 설명하는 텍스트 프롬프트입니다. 모델이 따르야 할 제약 조건, 스타일, 세부사항 등을 포함하세요. |
| `model` | 콤보 | 예 | `"Nano Banana 2 (Gemini 3.1 Flash Image)"` | 이미지 생성에 사용할 Gemini 모델을 선택합니다. 현재는 하나의 옵션만 제공됩니다. 이 매개변수에는 해상도, 종횡비, 사고 수준 및 이미지 입력과 관련된 추가 하위 매개변수가 포함됩니다. |
| `seed` | 정수 | 예 | 0부터 18446744073709551615까지 | 시드를 특정 값으로 고정하면 모델은 반복 요청 시 동일한 응답을 제공하려고 노력합니다. 결정론적 출력이 보장되지는 않습니다. 또한 모델이나 온도와 같은 매개변수 설정을 변경하면 동일한 시드 값을 사용하더라도 응답에 변동이 생길 수 있습니다. 기본적으로 임의의 시드 값이 사용됩니다. (기본값: 42) |
| `response_modalities` | 콤보 | 예 | `"IMAGE"`<br />`"IMAGE+TEXT"` | 응답 형식을 결정합니다. "IMAGE"를 선택하면 이미지만 받고, "IMAGE+TEXT"를 선택하면 이미지와 텍스트 설명을 모두 받습니다. (기본값: "IMAGE") |
| `system_prompt` | 문자열 | 아니요 | 없음 | AI의 행동을 규정하는 기본적인 지침입니다. 이는 고급 매개변수입니다. (기본값: 모델에게 항상 이미지를 생성하도록 지시하는 사전 정의된 시스템 프롬프트) |

**`model` 매개변수에 대한 참고사항:** `model` 매개변수는 동적 콤보로, 해상도, 종횡비, 사고 수준 및 이미지 입력과 관련된 추가 하위 매개변수를 포함합니다. 이러한 하위 매개변수는 모델 선택 내에서 정의되며, 이 표에는 별도의 입력으로 나열되지 않습니다.

**이미지 입력에 대한 참고사항:** 모델에 최대 14개의 이미지를 입력할 수 있습니다. 이 이미지는 `model` 매개변수의 이미지 하위 필드를 통해 전달되며, 편집용으로 사용되거나 생성 시 시각적 맥락으로 활용됩니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `IMAGE` | 이미지 | 생성되거나 편집된 이미지입니다. |
| `STRING` | 문자열 | 모델이 생성한 텍스트 설명 또는 캡션입니다. |
| `thought_image` | 이미지 | 모델의 사고 과정에서 나온 첫 번째 이미지입니다. 사고 수준이 HIGH이고 응답 모달리티가 IMAGE+TEXT인 경우에만 사용 가능합니다. |
