Hunyuan3Dv2ConditioningMultiView 노드는 3D 비디오 생성을 위해 다중 뷰 CLIP 비전 임베딩을 처리합니다. 선택적 전방, 좌측, 후방 및 우측 뷰 임베딩을 받아 위치 인코딩과 결합하여 비디오 모델용 조건 데이터를 생성합니다. 이 노드는 결합된 임베딩에서 얻은 긍정적 조건과 값이 0인 부정적 조건을 모두 출력합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `front` | CLIP_VISION_OUTPUT | 선택 | - | 전방 뷰의 CLIP 비전 출력 |
| `left` | CLIP_VISION_OUTPUT | 선택 | - | 좌측 뷰의 CLIP 비전 출력 |
| `back` | CLIP_VISION_OUTPUT | 선택 | - | 후방 뷰의 CLIP 비전 출력 |
| `right` | CLIP_VISION_OUTPUT | 선택 | - | 우측 뷰의 CLIP 비전 출력 |

**참고:** 노드가 작동하려면 최소한 하나의 뷰 입력이 제공되어야 합니다. 노드는 유효한 CLIP 비전 출력 데이터를 포함한 뷰만 처리합니다.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `positive` | CONDITIONING | 위치 인코딩과 결합된 다중 뷰 임베딩을 포함한 긍정적 조건 |
| `negative` | CONDITIONING | 대비 학습을 위한 값이 0인 부정적 조건 |
