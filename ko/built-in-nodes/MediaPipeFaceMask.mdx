## 개요

이 노드는 MediaPipe가 감지한 얼굴 랜드마크를 기반으로 이진 마스크(흑백 이미지)를 생성합니다. 각 감지된 얼굴 영역에 대해 채워진 다각형 모양을 그려, 일괄 처리된 프레임마다 하나의 마스크를 생성합니다. 동일한 프레임에서 여러 얼굴이 감지되면, 각 얼굴의 마스크가 하나의 마스크로 합쳐집니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `face_landmarks` | FACE_LANDMARKS | 예 | - | MediaPipe 얼굴 감지 노드에서 제공하는 얼굴 랜드마크 데이터입니다. |
| `regions` | COMBO | 예 | `"all"`<br />`"custom"` | 마스크에 포함할 얼굴 영역을 선택합니다. `"all"`은 모든 얼굴 영역(얼굴 타원, 입술, 눈, 홍채)의 합집합으로 마스크를 생성합니다. `"custom"`은 각 영역을 개별적으로 활성화하거나 비활성화할 수 있게 합니다. 기본값: `"all"` |

`regions`를 `"custom"`로 설정하면 다음과 같은 추가 부울 매개변수가 사용 가능해집니다:

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `face_oval` | BOOLEAN | 아니요 | True/False | 마스크에 얼굴 타원 영역을 포함합니다. 기본값: True |
| `lips` | BOOLEAN | 아니요 | True/False | 마스크에 입술 영역을 포함합니다. 기본값: True |
| `eyes` | BOOLEAN | 아니요 | True/False | 마스크에 눈 영역을 포함합니다. 기본값: True |
| `irises` | BOOLEAN | 아니요 | True/False | 마스크에 홍채 영역을 포함합니다. 기본값: True |

**참고:** `"all"` 모드를 사용할 경우, 마스크에는 모든 영역이 합쳐져 포함됩니다. 얼굴 타원이 다른 영역을 둘러싸고 있으므로, `"all"`을 선택하면 사실상 얼굴 타원만 선택한 것과 동일한 결과를 얻게 됩니다.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `MASK` | MASK | 얼굴 영역은 흰색(값 1.0), 배경은 검은색(값 0.0)인 이진 마스크 텐서입니다. 마스크는 입력 이미지와 동일한 크기를 가지며, 일괄 처리된 프레임마다 하나의 마스크를 포함합니다. |
