BasicGuider 노드는 샘플링 과정을 위한 간단한 안내 메커니즘을 생성합니다. 모델과 조건 데이터를 입력으로 받아, 샘플링 중에 생성 과정을 안내하는 데 사용할 수 있는 안내자 객체를 출력합니다. 이 노드는 제어된 생성에 필요한 기본적인 안내 기능을 제공합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `model` | MODEL | 예 | - | 안내에 사용할 모델 |
| `conditioning` | CONDITIONING | 예 | - | 생성 과정을 안내하는 조건 데이터 |

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `GUIDER` | GUIDER | 샘플링 과정 중에 생성을 안내하는 데 사용할 수 있는 안내자 객체 |
