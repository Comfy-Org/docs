SaveWEBM 노드는 이미지 시퀀스를 WEBM 비디오 파일로 저장합니다. 여러 입력 이미지를 받아들여, 구성 가능한 품질 설정과 프레임 속도를 사용해 VP9 또는 AV1 코덱으로 비디오로 인코딩합니다. 결과 비디오 파일은 출력 디렉터리에 저장되며, 여기에는 프롬프트 정보를 포함한 메타데이터가 추가됩니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `images` | IMAGE | 예 | - | 비디오 프레임으로 인코딩할 입력 이미지 시퀀스 |
| `filename_prefix` | STRING | 아니요 | - | 출력 파일명의 접두사 (기본값: "ComfyUI") |
| `codec` | COMBO | 예 | "vp9"<br />"av1" | 인코딩에 사용할 비디오 코덱 |
| `fps` | FLOAT | 아니요 | 0.01-1000.0 | 출력 비디오의 프레임 속도 (기본값: 24.0) |
| `crf` | FLOAT | 아니요 | 0-63.0 | 품질 설정으로, 높은 crf는 낮은 품질과 작은 파일 크기를 의미하며, 낮은 crf는 높은 품질과 큰 파일 크기를 의미합니다 (기본값: 32.0) |

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-------------|-----------|-------------|
| `ui` | PREVIEW | 저장된 WEBM 파일을 보여주는 비디오 미리보기 |
