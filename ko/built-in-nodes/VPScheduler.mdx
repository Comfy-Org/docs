VPScheduler 노드는 분산 보존(VP) 스케줄링 방식에 기반해 노이즈 수준(시그마)의 시퀀스를 생성하도록 설계되었습니다. 이 시퀀스는 확산 모델에서 디노이징 과정을 안내하는 데 매우 중요하며, 이미지나 기타 데이터 유형을 제어된 방식으로 생성할 수 있게 해줍니다.

## 입력

| 매개변수   | 데이터 타입 | 설명                                                                                                                                      |
|-------------|-------------|--------------------------------------------------------------------------------------------------------------------------------------------------|
| `steps`     | INT         | 확산 과정의 단계 수를 지정하며, 생성되는 노이즈 수준의 세밀도에 영향을 줍니다.                              |
| `beta_d`    | FLOAT       | 전체 노이즈 수준 분포를 결정하며, 생성되는 노이즈 수준의 분산에 영향을 줍니다.                                 |
| `beta_min`  | FLOAT       | 노이즈 수준의 최소 한계를 설정하여, 노이즈가 특정 임계값 아래로 떨어지지 않도록 합니다.                              |
| `eps_s`     | FLOAT       | 시작 epsilon 값을 조정하며, 확산 과정의 초기 노이즈 수준을 세밀하게 조정합니다.                                    |

## 출력

| 매개변수   | 데이터 타입 | 설명                                                                                   |
|-------------|-------------|-----------------------------------------------------------------------------------------------|
| `sigmas`    | SIGMAS      | VP 스케줄링 방식에 따라 생성된 노이즈 수준(시그마)의 시퀀스로, 확산 모델의 디노이징 과정을 안내하는 데 사용됩니다. |
