Meshy: Rig Model 노드는 이전 Meshy 작업에서 가져온 3D 모델을 받아 자동으로 뼈대를 생성하며, 이를 통해 리깅된 캐릭터를 만들어 포즈와 애니메이션을 적용할 수 있습니다. 이 노드는 리깅된 모델을 GLB 및 FBX 파일 형식으로 출력합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `meshy_task_id` | 문자열 | 예 | 없음 | 이전 Meshy 작업(예: 텍스트-3D 또는 이미지-3D)에서 생성된 모델의 고유한 작업 ID로, 리깅할 모델을 지정합니다. |
| `height_meters` | 부동소수점 | 예 | 0.1~15.0 | 캐릭터 모델의 대략적인 높이(미터 단위). 스케일링과 리깅 정확도를 높이는 데 도움이 됩니다(기본값: 1.7). |
| `texture_image` | 이미지 | 아니요 | 없음 | 모델의 UV 언랩된 기본 색상 텍스처 이미지입니다. |

**참고:** 현재 자동 리깅 과정은 무텍스처 메시, 비인간형 아셋, 또는 팔다리와 신체 구조가 명확하지 않은 인체형 아셋에는 적합하지 않습니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `model_file` | 문자열 | 후향 호환성을 위한 레거시 출력으로, GLB 모델의 파일명이 포함됩니다. |
| `rig_task_id` | 문자열 | 이 리깅 작업의 고유한 작업 ID로, 결과를 참조하는 데 사용할 수 있습니다. |
| `GLB` | FILE3DGLB | GLB 파일 형식으로 저장된 리깅된 3D 캐릭터 모델입니다. |
| `FBX` | FILE3DFBX | FBX 파일 형식으로 저장된 리깅된 3D 캐릭터 모델입니다. |
