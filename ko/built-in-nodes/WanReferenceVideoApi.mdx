> 이 문서는 AI로 생성되었습니다. 오류를 발견하거나 개선할 점이 있다면 언제든지 기여해 주세요! [GitHub에서 편집하기](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/WanReferenceVideoApi/en.md)

Wan Reference to Video 노드는 하나 이상의 입력 참조 동영상의 시각적 모습과 목소리, 그리고 텍스트 프롬프트를 사용해 새로운 동영상을 생성합니다. 참조 자료의 캐릭터와 일관성을 유지하면서도 사용자의 설명에 따라 새로운 콘텐츠를 만들어냅니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `model` | COMBO | 예 | `"wan2.6-r2v"` | 동영상 생성에 사용할 특정 AI 모델입니다. |
| `prompt` | STRING | 예 | - | 새 동영상의 요소와 시각적 특징에 대한 설명입니다. 영어와 중국어를 지원하며, `character1`, `character2`와 같은 식별자를 사용해 참조 동영상의 캐릭터를 지칭할 수 있습니다. |
| `negative_prompt` | STRING | 아니요 | - | 생성된 동영상에서 피해야 할 요소나 특징에 대한 설명입니다. |
| `reference_videos` | AUTOGROW | 예 | - | 캐릭터의 외모와 목소리를 위한 참조 동영상 목록입니다. 최소 한 개의 동영상을 제공해야 합니다. 각 동영상에는 `character1`, `character2`, `character3`와 같은 이름을 지정할 수 있습니다. |
| `size` | COMBO | 예 | `"720p: 1:1 (960x960)"`<br />`"720p: 16:9 (1280x720)"`<br />`"720p: 9:16 (720x1280)"`<br />`"720p: 4:3 (1088x832)"`<br />`"720p: 3:4 (832x1088)"`<br />`"1080p: 1:1 (1440x1440)"`<br />`"1080p: 16:9 (1920x1080)"`<br />`"1080p: 9:16 (1080x1920)"`<br />`"1080p: 4:3 (1632x1248)"`<br />`"1080p: 3:4 (1248x1632)"` | 출력 동영상의 해상도와 종횡비입니다. |
| `duration` | INT | 예 | 5~10 | 생성된 동영상의 길이(초)입니다. 값은 5의 배수여야 합니다(기본값: 5). |
| `seed` | INT | 아니요 | 0~2147483647 | 재현 가능한 결과를 위해 사용하는 랜덤 시드 값입니다. 0을 입력하면 랜덤 시드가 생성됩니다. |
| `shot_type` | COMBO | 예 | `"single"`<br />`"multi"` | 생성된 동영상이 단일 연속 샷인지, 컷이 포함된 여러 샷인지 여부를 지정합니다. |
| `watermark` | BOOLEAN | 아니요 | - | 활성화 시, 최종 동영상에 AI로 생성된 워터마크가 추가됩니다(기본값: False). |

**제약사항:**

* `reference_videos`에 제공된 각 동영상의 길이는 2~30초 사이여야 합니다.
* `duration` 매개변수는 특정 값(5초 또는 10초)으로 제한됩니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-----------|-----------|-------------|
| `output` | VIDEO | 새로 생성된 동영상 파일입니다. |
