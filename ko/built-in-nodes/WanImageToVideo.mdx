WanImageToVideo 노드는 비디오 생성 작업을 위한 조건부 및 잠재 표현을 준비합니다. 이 노드는 비디오 생성을 위한 빈 잠재 공간을 생성하며, 선택적으로 시작 이미지와 CLIP 비전 출력을 포함하여 비디오 생성 과정을 안내할 수 있습니다. 노드는 제공된 이미지와 비전 데이터를 기반으로 긍정적 및 부정적 조건부 입력을 모두 수정합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 | 범위 | 설명 |
|-----------|-----------|------|-------|-------------|
| `positive` | CONDITIONING | 예 | - | 생성을 안내하는 긍정적 조건부 입력 |
| `negative` | CONDITIONING | 예 | - | 생성을 안내하는 부정적 조건부 입력 |
| `vae` | VAE | 예 | - | 이미지를 잠재 공간으로 인코딩하는 VAE 모델 |
| `width` | INT | 예 | 16부터 MAX_RESOLUTION까지 | 출력 비디오의 너비 (기본값: 832, 단계: 16) |
| `height` | INT | 예 | 16부터 MAX_RESOLUTION까지 | 출력 비디오의 높이 (기본값: 480, 단계: 16) |
| `length` | INT | 예 | 1부터 MAX_RESOLUTION까지 | 비디오의 프레임 수 (기본값: 81, 단계: 4) |
| `batch_size` | INT | 예 | 1부터 4096까지 | 한 번에 생성할 비디오 수 (기본값: 1) |
| `clip_vision_output` | CLIP_VISION_OUTPUT | 아니요 | - | 추가적인 조건부를 위한 선택적 CLIP 비전 출력 |
| `start_image` | IMAGE | 아니요 | - | 비디오 생성을 초기화하는 선택적 시작 이미지입니다. 제공될 경우 이미지는 지정된 너비와 높이에 맞게 리사이즈되며, 비디오의 첫 프레임은 이 이미지로부터 초기화됩니다. 나머지 프레임은 중립적인 회색(0.5) 값으로 채워집니다. |

**참고:** `start_image`가 제공되면 노드는 VAE를 사용해 이미지 시퀀스를 인코딩하고 조건부 입력에 마스크를 적용합니다. 이 마스크는 시작 이미지로 초기화된 프레임을 제외한 모든 프레임을 덮으며, 생성 과정이 제공된 이미지를 기반으로 진행되도록 합니다. `clip_vision_output` 매개변수가 제공되면 긍정적 및 부정적 입력 모두에 비전 기반 조건부가 추가됩니다.

## 출력

| 출력 이름 | 데이터 유형 | 설명 |
|-----------|-----------|-------------|
| `positive` | CONDITIONING | 이미지와 비전 데이터가 통합된 수정된 긍정적 조건부 |
| `negative` | CONDITIONING | 이미지와 비전 데이터가 통합된 수정된 부정적 조건부 |
| `latent` | LATENT | 비디오 생성을 위해 준비된 빈 잠재 공간 텐서로, 형태는 [batch_size, 16, ((length-1)//4)+1, height//8, width//8] 입니다 |
