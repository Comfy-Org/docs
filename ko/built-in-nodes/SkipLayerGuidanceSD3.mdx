SkipLayerGuidanceSD3 노드는 건너뛴 레이어를 사용하여 추가적인 분류기 없는 가이던스를 적용함으로써 세부 구조를 향상시킵니다. 이 실험적 구현은 Perturbed Attention Guidance에서 영감을 받았으며, 음성 조건부 과정 중 특정 레이어를 선택적으로 우회하여 생성된 출력물의 구조적 세부사항을 개선합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `model` | MODEL | 예 | - | 스킵 레이어 가이던스를 적용할 모델 |
| `layers` | STRING | 예 | - | 건너뛸 레이어 인덱스의 쉼표로 구분된 리스트 (기본값: "7, 8, 9") |
| `scale` | FLOAT | 예 | 0.0 - 10.0 | 스킵 레이어 가이던스 효과의 강도 (기본값: 3.0) |
| `start_percent` | FLOAT | 예 | 0.0 - 1.0 | 가이던스 적용의 시작 지점 (총 단계 수의 백분율로 표시, 기본값: 0.01) |
| `end_percent` | FLOAT | 예 | 0.0 - 1.0 | 가이던스 적용의 종료 지점 (총 단계 수의 백분율로 표시, 기본값: 0.15) |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `model` | MODEL | 스킵 레이어 가이던스가 적용된 수정된 모델 |
