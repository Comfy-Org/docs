이 노드는 입력 이미지의 종횡비를 기반으로 Lanczos 알고리즘을 사용해 Flux Kontext 모델 학습 시 사용되는 최적 크기로 입력 이미지를 스케일링합니다. 특히 큰 크기의 이미지를 입력할 때 유용한데, 너무 큰 입력 이미지는 모델의 출력 품질 저하나 출력물에 여러 개체가 나타나는 등의 문제를 야기할 수 있습니다.

## 입력

| 매개변수 이름 | 데이터 타입 | 입력 유형 | 기본값 | 값 범위 | 설명 |
|----------------|-----------|------------|---------------|-------------|-------------|
| `image` | IMAGE | 필수 | - | - | 리사이즈할 입력 이미지 |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `image` | IMAGE | 리사이즈된 이미지 |
