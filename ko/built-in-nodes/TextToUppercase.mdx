Text to Uppercase 노드는 텍스트 입력을 받아 모든 문자를 대문자로 변환합니다. 이 노드는 제공된 문자열의 대소문자를 변경하는 간단한 텍스트 처리 유틸리티입니다.

**참고:** 이 노드는 더 이상 사용되지 않으며, 대신 텍스트 대소문자 변환 노드가 대체되었습니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `text` | STRING | 예 | N/A | 대문자로 변환할 텍스트 문자열입니다. |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `text` | STRING | 모든 문자가 대문자로 변환된 결과 텍스트입니다. |
