Image Compare 노드는 드래그 가능한 슬라이더를 사용해 두 이미지를 나란히 비교할 수 있는 시각적 인터페이스를 제공합니다. 이 노드는 출력 노드로 설계되었으며, 다른 노드로 데이터를 전달하지 않고 사용자 인터페이스에 직접 이미지를 표시해 검사하도록 합니다.

## 입력

| 매개변수 | 데이터 유형 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `image_a` | IMAGE | 선택 | - | 비교할 첫 번째 이미지입니다. |
| `image_b` | IMAGE | 선택 | - | 비교할 두 번째 이미지입니다. |
| `compare_view` | IMAGECOMPARE | 필수 | - | UI에서 슬라이더 비교 뷰를 활성화하는 컨트롤입니다. |

**참고:** 이 노드는 출력 노드입니다. `image_a`와 `image_b`는 선택적이지만, 노드가 눈에 보이는 효과를 내려면 적어도 하나의 이미지는 반드시 제공되어야 합니다. 연결되지 않은 이미지 입력에 대해서는 노드가 빈 영역을 표시합니다.

## 출력

이 노드는 출력 노드이며, 다른 노드에서 사용할 데이터 출력을 생성하지 않습니다. 노드의 역할은 제공된 이미지를 ComfyUI 인터페이스에 표시하는 것입니다.
