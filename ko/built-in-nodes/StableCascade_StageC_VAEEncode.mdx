StableCascade_StageC_VAEEncode 노드는 입력 이미지를 VAE 인코더를 통해 처리하여 Stable Cascade 모델용 잠재 표현을 생성합니다. 먼저 압축 계수와 VAE의 다운스케일 비율에 따라 이미지를 리사이즈한 다음, 리사이즈된 이미지를 인코딩합니다. 이 노드는 두 개의 잠재 텐서를 출력합니다: 하나는 스테이지 C용(실제 인코딩된 결과)이고, 다른 하나는 스테이지 B용(0으로 채워진 자리표시)입니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `image` | IMAGE | 예 | - | 잠재 공간으로 인코딩할 입력 이미지 |
| `vae` | VAE | 예 | - | 이미지를 인코딩하는 데 사용되는 VAE 모델 |
| `compression` | INT | 아니요 | 4-128 | 인코딩 전 이미지에 적용되는 압축 계수입니다. 이미지 크기는 이 값으로 나눈 후 VAE의 다운스케일 비율을 곱합니다. (기본값: 42) |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `stage_c` | LATENT | Stable Cascade 모델의 스테이지 C용 인코딩된 잠재 표현 |
| `stage_b` | LATENT | 스테이지 B용 자리표시 잠재 표현입니다. 현재 입력 이미지 크기에 따라 계산된 크기를 가진 0으로 채워진 텐서를 반환합니다. |
