> 이 문서는 AI로 생성되었습니다. 오류를 발견하거나 개선할 점이 있다면 언제든지 기여해 주세요! [GitHub에서 편집하기](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/RecraftV4TextToVectorNode/en.md)

Recraft V4 텍스트 to 벡터 노드는 텍스트 설명을 기반으로 스케일러블 벡터 그래픽스(SVG) 일러스트레이션을 생성합니다. 이미지 생성을 위해 외부 API와 연결되어 Recraft V4 또는 Recraft V4 Pro 모델을 사용합니다. 노드는 프롬프트에 따라 하나 이상의 SVG 이미지를 출력합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `prompt` | 문자열 | 예 | N/A | 이미지 생성을 위한 프롬프트입니다. 최대 10,000자까지 가능합니다. |
| `negative_prompt` | 문자열 | 아니요 | N/A | 이미지에서 원치 않는 요소에 대한 선택적 텍스트 설명입니다. |
| `model` | 콤보 | 예 | `"recraftv4"`<br />`"recraftv4_pro"` | 생성에 사용할 모델입니다. 모델을 선택하면 사용 가능한 `size` 옵션이 변경됩니다. |
| `size` | 콤보 | 예 | `recraftv4`: `"1024x1024"`, `"1152x896"`, `"896x1152"`, `"1216x832"`, `"832x1216"`, `"1344x768"`, `"768x1344"`, `"1536x640"`, `"640x1536"`<br />`recraftv4_pro`: `"2048x2048"`, `"2304x1792"`, `"1792x2304"`, `"2432x1664"`, `"1664x2432"`, `"2688x1536"`, `"1536x2688"`, `"3072x1280"`, `"1280x3072"` | 생성된 이미지의 크기입니다. 사용 가능한 옵션은 선택한 `model`에 따라 달라집니다. 기본값은 `recraftv4`의 `"1024x1024"`이며, `recraftv4_pro`의 경우 `"2048x2048"`입니다. |
| `n` | 정수 | 예 | 1~6 | 생성할 이미지 수입니다(기본값: 1). |
| `seed` | 정수 | 예 | 0~18446744073709551615 | 노드가 다시 실행될지 여부를 결정하는 시드입니다. 실제 결과는 시드와 관계없이 비결정적입니다. |
| `recraft_controls` | 맞춤형 | 아니요 | N/A | Recraft Controls 노드를 통해 생성 과정을 추가로 제어할 수 있는 옵션입니다. |

**참고:** `size` 매개변수는 동적 입력으로, 선택한 `model`에 따라 사용 가능한 옵션이 달라집니다. `seed` 값은 외부 API에서 재현 가능한 결과를 보장하지 않습니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `output` | SVG | 생성된 스케일러블 벡터 그래픽스(SVG) 이미지입니다. |
