확장 프로그램 설정 패널은 ComfyUI 프론트엔드 설정 시스템의 특수 관리 패널로, 프론트엔드 확장 플러그인의 활성화/비활성화 상태를 관리하는 데 사용됩니다. 커스텀 노드와 달리, 이 패널은 커스텀 노드 자체를 비활성화하는 것이 아니라 커스텀 노드에 의해 등록된 프론트엔드 확장 프로그램만을 관리하는 데 사용됩니다.

이러한 프론트엔드 확장 플러그인은 단축키, 설정, UI 구성 요소, 메뉴 항목 등 다양한 기능을 제공하여 ComfyUI 경험을 향상시키는 데 사용됩니다.

확장 프로그램 상태 변경은 페이지를 다시 로드해야 적용됩니다:

## 확장 프로그램 설정 패널 기능

### 1. 확장 프로그램 필터
상단의 SelectButton 필터를 사용하면 표시되는 확장 프로그램을 좁힐 수 있습니다:
- **All** — 등록된 모든 확장 프로그램 표시
- **Core** — 핵심 확장 프로그램만 표시
- **Custom** — 서드파티 확장 프로그램만 표시

### 2. 확장 프로그램 목록 관리
등록된 모든 확장 프로그램을 표시하며, 다음 정보를 포함합니다:
- 확장 프로그램 이름
- 확장 프로그램 레이블 — 핵심 확장 프로그램은 **"Core"** 태그, 서드파티(비핵심) 확장 프로그램은 **"Custom"** (severity="info") 태그 표시
- 활성화/비활성화 상태 토글
- 일괄 작업을 위한 다중 선택 체크박스 열

### 3. 검색 기능
특정 확장 프로그램을 빠르게 찾을 수 있도록 검색창을 제공합니다.

### 4. 활성화/비활성화 제어
각 확장 프로그램마다 독립적인 토글 스위치가 있습니다.

### 5. 일괄 작업
마우스 오른쪽 버튼 클릭 컨텍스트 메뉴를 통해 일괄 작업을 수행할 수 있습니다:
- **선택 항목 활성화** — 체크된 확장 프로그램 활성화
- **선택 항목 비활성화** — 체크된 확장 프로그램 비활성화
- **────**
- **모두 활성화** — 모든 확장 프로그램 활성화
- **모두 비활성화** — 모든 확장 프로그램 비활성화
- **서드파티 비활성화** — 모든 서드파티 확장 프로그램 비활성화 (핵심 확장 프로그램 유지)

### 6. 변경 알림
토글 스위치가 변경되면 패널 하단에 변경된 모든 확장 프로그램을 나열하는 정보 메시지가 표시됩니다:
- **켜진** 확장 프로그램(비활성화 → 활성화)은 **[+]** 접두사로 표시됩니다.
- **꺼진** 확장 프로그램(활성화 → 비활성화)은 **[-]** 접두사로 표시됩니다.
- **"변경 사항 적용을 위해 새로고침"** 버튼으로 페이지를 즉시 다시 로드할 수 있습니다.

## 참고사항

- 확장 프로그램 상태 변경은 페이지를 다시 로드해야 적용됩니다.
- 일부 핵심 확장 프로그램은 비활성화할 수 없습니다.
- 시스템은 알려진 문제가 있는 확장 프로그램을 자동으로 비활성화합니다.
- 확장 프로그램 설정은 사용자 구성 파일에 자동으로 저장됩니다.

이 확장 프로그램 설정 패널은 사실상 "프론트엔드 플러그인 관리자"로서, 사용자가 ComfyUI의 기능 모듈을 유연하게 제어할 수 있게 해줍니다.
