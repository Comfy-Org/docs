확장 프로그램 설정 패널은 ComfyUI 프론트엔드 설정 시스템의 특수 관리 패널로, 특히 프론트엔드 확장 플러그인의 활성화/비활성화 상태를 관리하는 데 사용됩니다. 커스텀 노드와 달리, 이 패널은 커스텀 노드 자체를 비활성화하는 것이 아니라, 커스텀 노드에 의해 등록된 프론트엔드 확장 프로그램만을 관리하는 데 사용됩니다.

![확장 프로그램](/images/interface/setting/extension/extension.jpg)

이러한 프론트엔드 확장 플러그인들은 단축키, 설정, UI 구성 요소, 메뉴 항목 등 다양한 기능을 제공하여 ComfyUI의 경험을 향상시키는 데 사용됩니다.

확장 프로그램 상태 변경은 페이지를 다시 로드해야만 적용됩니다:

## 확장 프로그램 설정 패널의 기능

### 1. 확장 프로그램 목록 관리
등록된 모든 확장 프로그램을 표시하며, 다음 정보를 포함합니다:
- 확장 프로그램 이름
- 핵심 확장 프로그램 식별자 (핵심이라고 표시됨)
- 활성화/비활성화 상태

### 2. 검색 기능
특정 확장 프로그램을 빠르게 찾을 수 있도록 검색창을 제공합니다:

### 3. 활성화/비활성화 제어
각 확장 프로그램마다 독립적인 토글 스위치가 있습니다:

### 4. 일괄 작업
마우스 오른쪽 버튼 클릭 메뉴를 통해 일괄 작업을 수행할 수 있습니다:
- 모든 확장 프로그램 활성화
- 모든 확장 프로그램 비활성화
- 서드파티 확장 프로그램 비활성화 (핵심 확장 프로그램은 유지)

## 참고사항

- 확장 프로그램 상태 변경은 페이지를 다시 로드해야만 적용됩니다.
- 일부 핵심 확장 프로그램은 비활성화할 수 없습니다.
- 시스템은 이미 알려진 문제 있는 확장 프로그램을 자동으로 비활성화합니다.
- 확장 프로그램 설정은 사용자 구성 파일에 자동으로 저장됩니다.

이 확장 프로그램 설정 패널은 사실상 '프론트엔드 플러그인 관리자'로서, 사용자가 ComfyUI의 기능 모듈을 유연하게 제어할 수 있게 해줍니다.
