이 섹션에서는 ComfyUI 프론트엔드 설정 메뉴의 상세한 설정 설명을 다룹니다. 모든 사용자 설정은 자동으로 `ComfyUI/user/default/comfy.settings.json` 파일에 저장됩니다.

`Ctrl + ,` 키보드 단축키를 사용해 설정 패널을 열고, 해당 설정 옵션을 클릭하여 구성할 수 있습니다.

커스텀 노드 역시 메뉴에 해당 설정 카테고리를 등록할 수 있으므로, 현재 공식 문서에는 기본 ComfyUI 설정 내용만 포함되어 있습니다. 또한 일부 설정 옵션은 **ComfyUI Desktop에서만 유효**합니다.

## 설정 메뉴 구조

설정 패널은 세 가지 주요 그룹이 있는 사이드바로 구성됩니다. 구조는 팀 워크스페이스가 활성화되었는지 여부에 따라 다릅니다.

### 팀 워크스페이스가 활성화된 경우:

1. **Workspace**
   - 워크스페이스 설정 (해당하는 경우)
   - Credits

2. **General**
   - User / Profile
   - **Comfy** — 핵심 ComfyUI 설정 (인터페이스, 큐, 노드, 워크플로우 등)
   - Secrets
   - **Lite Graph** — 캔버스 렌더링, 노드, 링크, 그룹 등
   - **Appearance** — 테마, 사이드바 위치, 배경 등
   - **3D** — 3D 노드 초기화 설정
   - **Mask Editor** — 마스크 편집기 기본 설정
   - Keybinding
   - Extension
   - About
   - Server Config (Desktop 전용)

3. **Other** — 서드파티 커스텀 노드가 등록한 설정 (있는 경우에만 표시)

### 팀 워크스페이스가 비활성화된 경우 (레거시):

1. **Account**
   - User / Profile
   - Subscription
   - Secrets
   - Credits

2. **Application Settings** — 모든 핵심 설정 카테고리:
   - **Comfy**
   - **Lite Graph**
   - **Appearance**
   - **3D**
   - **Mask Editor**
   - **Other** (분류되지 않은 설정)

3. **Special Settings**
   - Keybinding
   - Extension
   - About
   - Server Config (Desktop 전용)

## Desktop 전용 설정

다음 설정은 ComfyUI Desktop에서만 사용할 수 있습니다:

- **Server Config** — 서버 설정 구성에 액세스 및 수정
- **Comfy Desktop 일반 설정** — 업데이트 기본 설정, 창 스타일, 미러 소스

## Cloud 전용 패널

일부 패널은 ComfyUI 계정에 로그인한 경우에만 나타납니다:

- **Credits** — 크레딧 구매 및 크레딧 잔액 내역 보기
- **Secrets** — API 키 및 시크릿 관리
- **Workspace** — 팀 워크스페이스 설정
- **Subscription** — 구독 요금제 관리
