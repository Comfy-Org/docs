LiteGraph는 ComfyUI의 기본 그래픽 렌더링 엔진입니다. 이 카테고리의 설정은 주로 캔버스, 노드, 링크와 같은 그래픽 인터페이스의 동작과 모양을 제어합니다.

## 캔버스

### 선택 도구 상자 표시
- **기본값**: 활성화됨
- **기능**: 선택 도구 상자는 노드를 선택한 후 나타나는 떠 있는 빠른 작업 도구 모음으로, 부분 실행, 고정, 삭제, 색상 수정 등의 일반적인 빠른 작업을 제공합니다.

![선택 도구 상자 표시](/images/interface/setting/lite-graph/selection-toolbox.jpg)

### 확대/축소 노드 상세 수준 - 글꼴 크기 임계값
- **기본값**: 8
- **범위**: 0 - 24
- **기능**: 노드가 저품질 LOD(Level of Detail) 렌더링으로 전환되는 시점을 제어합니다. 전환 시점을 결정하는 데 글꼴 크기(픽셀)를 사용합니다. 0으로 설정하면 비활성화됩니다. 값이 높을수록(24px) 축소 시 노드가 더 빨리 단순화된 렌더링으로 전환됩니다. 값이 낮을수록(1px) 전체 노드 품질을 더 오래 유지합니다.
- **대체**: 더 이상 사용되지 않는 "저품질 렌더링 확대 비율 임계값" 설정을 대체합니다.

![확대/축소 노드 상세 수준](/images/interface/setting/lite-graph/render-mode.jpg)

### 최대 FPS
- **기본값**: 0 (화면 새로고침률 사용)
- **범위**: 0 - 120
- **기능**: 캔버스의 렌더링 프레임 속도를 제한합니다. 0은 화면 새로고침률을 사용하는 것을 의미합니다. FPS가 높을수록 캔버스 렌더링이 더 부드러워지지만 성능 소모도 커집니다. 너무 낮은 값은 더 뚜렷한 끊김 현상을 일으킬 수 있습니다.

### 항상 그리드에 스냅
- **기본값**: 비활성화됨
- **기능**: 이 옵션이 활성화되지 않은 경우 `Shift` 키를 누르고 있으면 노드 가장자리를 그리드에 맞출 수 있습니다. 활성화된 경우 `Shift` 키를 누르지 않아도 노드 가장자리가 자동으로 그리드에 맞춰집니다.

### 그리드 크기에 스냅
- **기본값**: 10
- **범위**: 1 - 500
- **기능**: 자동 스냅이 활성화되었거나 `Shift` 키를 누른 채 노드를 이동할 때 이 매개변수는 스냅할 그리드 크기를 결정합니다.

### 빠른 확대 단축키 활성화 (Ctrl + Shift + 드래그)
- **기본값**: 활성화됨
- **기능**: `Ctrl + Shift + 마우스 왼쪽 버튼 드래그` 빠른 확대 기능을 활성화하여 더 빠른 확대 작업 방법을 제공합니다.

### 그래프 캔버스 메뉴 표시
- **기본값**: 활성화됨
- **기능**: 오른쪽 하단에 캔버스 메뉴를 표시할지 여부를 제어합니다.

캔버스 메뉴는 ComfyUI 인터페이스의 오른쪽 하단에 위치하며, 캔버스 확대/축소, 모든 연결 일시 숨기기, 워크플로우를 캔버스에 맞게 빠르게 조정 등의 작업을 포함합니다. 아래 이미지와 같습니다.

![그래프 캔버스 메뉴 표시](/images/interface/setting/lite-graph/canvas_menu.jpg)

### 캔버스 확대 속도
- **기본값**: 1.1
- **범위**: 1.01 - 2.5
- **기능**: 캔버스 확대 속도를 제어하며, 마우스 휠 확대의 민감도를 조정합니다.

### 자동 팬 속도
- **기본값**: 15
- **범위**: 0 - 30
- **기능**: 캔버스 가장자리로 드래그하여 자동 팬할 때의 최대 속도입니다. 0으로 설정하면 자동 팬이 비활성화됩니다.

### 왼쪽 하단에 캔버스 정보 표시 (FPS 등)
- **기본값**: 활성화됨
- **기능**: 왼쪽 하단에 캔버스 정보를 표시할지 여부를 제어하며, FPS와 같은 성능 지표를 표시합니다.

![캔버스 정보](/images/interface/setting/lite-graph/canvas-info.jpg)

### 탐색 모드
- **기본값**: legacy
- **옵션**:
  - **표준 (신규)**: 왼쪽 클릭으로 노드를 선택하고, 마우스 휠로 캔버스를 스크롤합니다.
  - **드래그 탐색**: 왼쪽 클릭으로 캔버스를 드래그합니다(레거시 동작).
  - **사용자 정의**: 아래에서 왼쪽 클릭 동작과 마우스 휠 동작을 수동으로 구성합니다.
- **기능**: 캔버스의 전체 탐색 모드를 제어합니다. 이 설정을 변경하면 [왼쪽 마우스 클릭 동작](#left-mouse-click-behavior) 및 [마우스 휠 스크롤](#mouse-wheel-scroll) 설정이 자동으로 조정됩니다.

### 왼쪽 마우스 클릭 동작
- **기본값**: panning
- **옵션**: Panning, Select
- **기능**: 캔버스 배경에서 왼쪽 클릭 동작을 제어합니다.
  - **Panning**: 캔버스 배경을 왼쪽 클릭하고 드래그하여 보기를 이동합니다.
  - **Select**: 왼쪽 클릭으로 노드를 선택하고, 드래그하여 선택 영역을 만듭니다. 이동하려면 가운데 마우스 버튼이나 오른쪽 클릭을 사용하세요.

### 마우스 휠 스크롤
- **기본값**: zoom
- **옵션**: Panning, Zoom in/out
- **기능**: 캔버스에서 마우스 휠의 동작을 제어합니다.
  - **Panning**: 마우스 휠로 캔버스를 세로로 스크롤합니다.
  - **Zoom in/out**: 마우스 휠로 캔버스를 확대/축소합니다.

### 트랙패드 제스처 사용
- **기본값**: 활성화됨 (표준 탐색 모드에서)
- **기능**: 탐색 모드가 **표준(신규)**으로 설정된 경우 트랙패드 제스처(두 손가락으로 확대/축소 및 드래그)가 자동으로 활성화됩니다. 이는 독립적인 설정이 아니라 탐색 모드 선택에서 파생됩니다.

![트랙패드 제스처 활성화](/images/interface/setting/lite-graph/canvas-menu.jpg)

### 실시간 선택
- **기준값**: 활성화됨
- **기능**: 활성화하면 선택 사각형을 드래그할 때 노드가 실시간으로 선택/해제되어 다른 디자인 도구와 유사하게 작동합니다.

## 컨텍스트 메뉴

### 확대 시 노드 콤보 위젯 메뉴(목록) 크기 조정
- **기본값**: 비활성화됨
- **기능**: 확대 시 노드 콤보 위젯 메뉴(목록)의 크기를 조정할지 여부를 제어하여, 사용자가 노드 콤보 위젯을 선택할 수 있도록 합니다.

## 그래프

### 링크 렌더링 모드
- **기본값**: 2 (Spline)
- **옵션**: Straight, Linear, Spline, Hidden
- **기능**: 연결의 렌더링 스타일을 설정하여 노드 간 링크의 시각적 스타일을 제어합니다.

## 그룹

이 설정 섹션은 주로 노드 그룹 기능과 관련이 있습니다.

![노드 그룹](/images/interface/setting/lite-graph/node-group.png)

### 그룹 제목을 더블클릭하여 편집
- **기본값**: 활성화됨
- **기능**: 노드 제목을 더블클릭하여 편집할 수 있도록 제어하며, 사용자가 노드 이름을 바꿀 수 있게 합니다. 이미지에서 `1`로 표시된 부분입니다.

### 선택된 노드 그룹의 패딩
- **기준값**: 10
- **범위**: 0 - 100
- **기능**: 선택된 노드를 그룹화할 때 내부 패딩을 설정하여 그룹 프레임과 노드 사이의 간격을 제어합니다. 이미지에서 화살표 주석 `2`로 표시된 부분입니다.

### 클릭 시 그룹 자식 선택
- **기본값**: 비활성화됨
- **기능**: 활성화하면 그룹을 클릭할 때 그룹 내 모든 노드와 항목이 선택됩니다.

## 노드

### 새 노드를 항상 축소
- **기본값**: 비활성화됨
- **기능**: 새 노드를 생성할 때 자동으로 축소할지 여부를 제어합니다. 노드가 항상 최소 크기로 표시되지만 추가 시 일부 텍스트 표시가 잘려 보일 수 있으므로 수동으로 노드 크기를 조정해야 합니다.

### DOM 요소 클리핑 활성화 (활성화 시 성능 저하 가능)
- **기본값**: 활성화됨
- **기능**: DOM 요소 클리핑을 활성화합니다(성능에 영향을 줄 수 있음). 렌더링을 최적화하지만 성능이 저하될 수 있습니다.

### 가운데 클릭 시 새 Reroute 노드 생성
- **기본값**: 활성화됨
- **기능**: 가운데 클릭 시 새 리루트 노드를 생성하여 연결을 정리하기 위한 리루트 노드를 빠르게 생성합니다.

### 노드 삭제 시 모든 링크 유지
- **기본값**: 활성화됨
- **기능**: 중간 노드를 삭제할 때 자동으로 연결을 우회하며, 노드를 삭제할 때 입력 및 출력 링크를 다시 연결하려고 시도합니다.

### 스냅 시 노드 강조 표시
- **기본값**: 활성화됨
- **기능**: 링크를 노드로 드래그할 때 노드를 강조 표시하여 시각적 피드백을 제공하고 연결 가능한 노드를 표시합니다. 활성화 시 아래 이미지와 같이 링크의 해당 쪽이 강조된 스타일로 표시됩니다.

![스냅 시 노드 강조 표시](/images/interface/setting/lite-graph/highlights-node.jpg)

### 링크를 노드 슬롯에 자동 스냅
- **기본값**: 활성화됨
- **기능**: 링크를 노드로 드래그할 때 자동으로 사용 가능한 슬롯에 스냅하여 연결 작업을 간소화하고 적합한 입력 슬롯을 자동으로 찾습니다.

### 툴팁 활성화
- **기본값**: 활성화됨
- **기능**: 일부 노드 정보에는 툴팁이 포함되며(매개변수 설명 등), 활성화 시 마우스를 가져다 대면 이러한 툴팁이 표시됩니다. 아래 이미지와 같습니다.

![툴팁 활성화](/images/interface/setting/lite-graph/tooltips.jpg)

### 툴팁 지연 시간
- **기본값**: 500
- **범위**: 100 - 3000
- **기능**: 툴팁의 지연 시간을 밀리초 단위로 제어합니다. 0으로 설정하면 즉시 툴팁이 표시됩니다.

### 노드 수명 주기 배지 모드
- **기본값**: Show all
- **옵션**: None, Show all
- **기능**: 노드 수명 주기 표시의 표시를 제어하여 노드 상태 정보(폐기됨, 실험적 등)를 표시합니다.

### 노드 ID 배지 모드
- **기본값**: None
- **옵션**: None, Show all
- **기능**: 노드 ID 표시의 표시를 제어하여 노드 고유 식별자를 표시합니다.

![노드 ID 배지 모드](/images/interface/setting/lite-graph/node-id-badge.jpg)

### 노드 소스 배지 모드
- **옵션**:
  - None: 모든 소스 배지 숨기기
  - Hide built-in: 사용자 정의 노드에만 배지 표시
  - Show all: 내장 노드와 사용자 정의 노드 모두에 배지 표시
- **기능**: 노드 소스 표시의 표시 모드를 제어하여 노드 소스 정보를 표시합니다. 여우 로고는 ComfyUI 내장 노드를 나타냅니다.

![노드 소스 배지 모드](/images/interface/setting/lite-graph/node-source-badge.jpg)

### 노드 제목을 더블클릭하여 편집
- **기본값**: 활성화됨
- **기능**: 노드 제목을 더블클릭하여 편집할 수 있도록 제어하며, 사용자가 노드 이름을 바꿀 수 있게 합니다.

## 노드 위젯

### 플로트 위젯 반올림 소수 자릿수 [0 = 자동]
- **기본값**: 0 (자동)
- **범위**: 0 - 6
- **기능**: 플로트 위젯 반올림의 소수 자릿수를 설정합니다. 0은 자동을 의미하며 페이지 재로드가 필요합니다.

### 기본 플로트 위젯 반올림 비활성화
- **기본값**: 비활성화됨
- **기능**: 기본 플로트 위젯 반올림을 비활성화할지 여부를 제어합니다. 페이지 재로드가 필요하며 노드 백엔드에서 반올림을 설정한 경우 비활성화할 수 없습니다.

### 노드 위젯 슬라이더 비활성화
- **기본값**: 비활성화됨
- **기능**: 노드 위젯의 슬라이더 컨트롤을 비활성화할지 여부를 제어하여 슬라이더 대신 텍스트 입력을 강제합니다.

### 미리보기 이미지 형식
- **기본값**: 빈 문자열 (원본 형식 사용)
- **기능**: 이미지 위젯의 미리보기 이미지 형식을 설정하여 webp, jpeg 등 경량 형식으로 변환합니다.

### 이미지 미리보기 아래에 가로×세로 표시
- **기본값**: 활성화됨
- **기능**: 이미지 미리보기 아래에 가로×세로 정보를 표시하여 이미지의 치수 정보를 보여줍니다.

## 포인터

### 더블클릭 간격 (최대)
- **기본값**: 300
- **범위**: 100 - 1000
- **기능**: 더블클릭의 두 클릭 사이 최대 시간(밀리초)입니다. 이 값을 늘리면 더블클릭이 때때로 인식되지 않는 문제를 해결하는 데 도움이 됩니다.

### 포인터 클릭 드리프트 지연 시간
- **기본값**: 32
- **범위**: 0 - 1000
- **실험적**: 예
- **기능**: 포인터 버튼을 누른 후 포인터 움직임을 무시할 수 있는 최대 시간(밀리초)입니다. 클릭 시 실수로 마우스가 움직이는 것을 방지하는 데 도움이 됩니다.

### 포인터 클릭 드리프트 (최대 거리)
- **기본값**: 6
- **범위**: 0 - 20
- **실험적**: 예
- **기능**: 포인터가 버튼을 누른 채 이 거리보다 더 많이 움직이면 드래그로 간주됩니다(클릭이 아닌). 클릭 시 실수로 마우스가 움직이는 것을 방지하는 데 도움이 됩니다.

## Reroute

### Reroute 스플라인 오프셋
- **기본값**: 20
- **범위**: 0 - 400
- **기능**: 리루트 노드 양쪽 곡선의 부드러움을 결정하는 데 사용됩니다. 값이 클수록 곡선이 더 부드러워지고, 작을수록 곡선이 더 날카로워집니다.
