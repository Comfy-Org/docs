이 설정 섹션은 주로 ComfyUI의 3D 관련 구성 요소, 예를 들어 카메라, 조명, 장면 등의 초기화 설정을 제어하는 데 사용됩니다. 새로운 3D 구성 요소를 생성할 때는 이 설정에 따라 초기화됩니다. 생성된 후에도 각 구성 요소별로 개별적으로 설정을 조정할 수 있습니다.

## 카메라
  
### 초기 카메라 유형
- **옵션**:  
  - `perspective`  
  - `orthographic` 
- **기능**: 새 3D 구성 요소를 생성할 때 기본 카메라가 원근법인지 직교법인지를 제어합니다. 이 기본 설정은 생성된 후에도 각 구성 요소별로 개별적으로 전환할 수 있습니다.

![카메라 유형](/images/interface/setting/3d/camera_type.jpg)


## 라이트

이 섹션의 라이트 설정은 3D 구성 요소의 기본 조명 설정을 지정하는 데 사용됩니다. ComfyUI의 3D 설정에서 해당 설정을 수정할 수도 있습니다.

![라이트](/images/interface/setting/3d/light.jpg)

### 라이트 조정 증분
- **기본값**: 0.5  
- **기능**: 3D 장면에서 라이트 강도를 조정할 때 단계 크기를 제어합니다. 작은 값은 더 세밀한 라이트 조정을 가능하게 하고, 큰 값은 각 조정이 더 두드러지게 나타납니다.

### 라이트 강도 최소값
- **기본값**: 1  
- **기능**: 3D 장면에서 허용되는 최소 라이트 강도 값을 설정합니다. 이는 어떤 3D 컨트롤의 조명을 조정할 때 설정할 수 있는 가장 낮은 밝기를 정의합니다.

### 라이트 강도 최대값
- **기본값**: 10  
- **기능**: 3D 장면에서 허용되는 최대 라이트 강도 값을 설정합니다. 이는 어떤 3D 컨트롤의 조명을 조정할 때 설정할 수 있는 가장 높은 밝기를 정의합니다.

### 초기 라이트 강도
- **기본값**: 3  
- **기능**: 3D 장면에서 라이트의 기본 밝기 수준을 설정합니다. 이 값은 새 3D 컨트롤을 생성할 때 라이트가 객체를 비추는 강도를 결정하지만, 생성된 후에는 각 컨트롤별로 개별적으로 조정할 수 있습니다.

## 장면
  
### 초기 배경색    
- **기능**: 3D 장면의 기본 배경색을 제어합니다. 이 설정은 새 3D 구성 요소를 생성할 때 배경의 모습을 결정하지만, 생성된 후에는 각 구성 요소별로 개별적으로 조정할 수 있습니다.
- **기본값**: `282828` (진회색)  

배경색을 변경하면 캔버스에서도 조정할 수 있습니다.
![초기 배경색 vs 수정된 배경색](/images/interface/setting/3d/background_color.jpg)

### 초기 미리보기 표시 여부
- **기능**: 새 3D 구성 요소를 생성할 때 기본적으로 미리보기 화면이 표시되는지 여부를 제어합니다. 생성된 후에도 각 구성 요소별로 개별적으로 이 기본 설정을 전환할 수 있습니다.
- **기본값**: true (활성화)  

![미리보기 vs 미리보기 숨김](/images/interface/setting/3d/hide_preview.jpg)

  
### 초기 그리드 표시 여부
- **기능**: 새 3D 구성 요소를 생성할 때 기본적으로 그리드가 표시되는지 여부를 제어합니다. 생성된 후에도 각 구성 요소별로 개별적으로 이 기본 설정을 전환할 수 있습니다.
- **기본값**: true (활성화)
