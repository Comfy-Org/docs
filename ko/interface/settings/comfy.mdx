## 파트너 노드

### 파트너 노드 가격 배지 표시
- **기본값**: 활성화됨
- **기능**: 파트너 노드에 가격 배지를 표시할지 여부를 제어하여 사용자가 파트너 노드의 사용 비용을 쉽게 식별할 수 있도록 합니다.

![파트너 노드 가격 배지 표시](/images/interface/setting/comfy/api_node_pricing_badge.jpg)

> 파트너 노드에 대한 자세한 내용은 [파트너 노드](/ko/tutorials/partner-nodes/overview)를 참조하세요.

## 모양

### 애니메이션 비활성화
- **기본값**: 시스템의 `prefers-reduced-motion` 설정을 따름
- **기능**: 대부분의 CSS 애니메이션과 전환을 끕니다. 디스플레이 GPU가 생성에도 사용될 때 추론 속도를 높입니다.

### 탭 바 레이아웃
- **옵션**: 기본, 레거시
- **기본값**: 기본
- **기능**: 통합 탭 바에 포함된 요소를 제어합니다.

## 개발자 모드

### 개발자 모드 옵션 활성화 (API 저장 등)
- **기본값**: 비활성화됨
- **기능**: 개발자 모드 옵션(예: API 저장 등)을 활성화합니다.

## 토큰 가중치 편집

### Ctrl+위/아래 정밀도
- **기본값**: 0.05
- **범위**: 0.01 - 0.50
- **기능**: CLIPTextEncode 유형 노드 또는 텍스트 입력 노드 위젯을 사용할 때 Ctrl+위/아래를 사용해 가중치를 빠르게 조정할 수 있습니다. 이 옵션은 각 조정 시 가중치 값을 변경합니다.

## 오류 시스템

### 사이드 패널에 오류 탭 표시
- **기본값**: 활성화됨
- **실험적**: 예
- **기능**: 활성화되면 오른쪽 사이드 패널에 오류 탭이 표시되어 워크플로우 실행 오류를 한눈에 확인할 수 있습니다.

## 실행

### 실시간 미리보기 방법
- **기본값**: default
- **옵션**: default, none, auto, latent2rgb, taesd
- **기능**: 이미지 생성 중 실시간 미리보기 방법입니다. "default"는 서버 CLI 설정을 사용합니다.
  - `none`: 미리보기 이미지가 표시되지 않으며, 생성 중에는 진행 표시줄만 표시됩니다.
  - `auto`: 가장 적합한 미리보기 방법을 자동으로 선택합니다.
  - `latent2rgb`: 잠재 공간 데이터를 RGB 이미지로 직접 변환하여 미리봅니다.
  - `taesd`: 경량 TAESD 모델을 사용하여 미리봅니다.

## 모델 라이브러리

모델 라이브러리는 ComfyUI 사이드바 메뉴의 모델 관리 기능을 말합니다. 이 기능을 사용해 `ComfyUI/models` 및 추가로 구성된 모델 폴더의 모델을 볼 수 있습니다.

![모델 라이브러리](/images/interface/sidepanel/model_library.jpg)

### 모델 라이브러리 트리 보기에서 표시할 이름
- **기본값**: title
- **옵션**: filename, title
- **기능**: 모델 라이브러리 트리 보기에서 표시할 이름 형식을 선택합니다.

### 모든 모델 폴더 자동 로드
- **기본값**: 비활성화됨
- **기능**: 모델 라이브러리를 클릭할 때 모든 폴더의 모델 파일을 자동으로 탐지할지 여부입니다. 활성화하면 로딩 지연이 발생할 수 있습니다(모든 폴더를 순회해야 함). 비활성화되면 해당 폴더 이름을 클릭할 때만 폴더 내 파일이 로드됩니다.

## 노드

ComfyUI의 반복 과정에서 일부 노드를 조정하고 일부 노드를 활성화합니다. 이러한 노드는 향후 버전에서 주요 변경이 있거나 제거될 수 있습니다. 그러나 호환성을 보장하기 위해 폐기된 노드는 제거되지 않았습니다. 아래 설정을 사용하여 **실험적 노드**와 **폐기된 노드**를 표시할지 여부를 활성화할 수 있습니다.

### 검색에서 폐기된 노드 표시
- **기본값**: 비활성화됨
- **기능**: 검색에서 폐기된 노드를 표시할지 여부를 제어합니다. 폐기된 노드는 기본적으로 UI에서 숨겨져 있지만, 기존 워크플로우에서는 계속 유효합니다.

![검색에서 폐기된 노드 표시](/images/interface/setting/comfy/depr_node.jpg)

### 검색에서 실험적 노드 표시
- **기본값**: 활성화됨
- **기능**: 검색에서 실험적 노드를 표시할지 여부를 제어합니다. 실험적 노드는 일부 새로운 기능을 지원하지만 완전히 안정적이지 않으며, 향후 버전에서 변경되거나 제거될 수 있습니다.

![검색에서 실험적 노드 표시](/images/interface/setting/comfy/beta_node.jpg)

### 노드 교체 제안 활성화
- **기본값**: 활성화됨
- **기능**: 활성화하면 알려진 교체 노드가 있는 누락된 노드가 누락 노드 대화상자에서 교체 가능한 것으로 표시되어, 교체를 검토하고 적용할 수 있습니다.

### 모든 노드에서 고급 위젯 항상 표시
- **기본값**: 비활성화됨
- **기능**: 활성화하면 고급 위젯이 모든 노드에서 개별적으로 확장할 필요 없이 항상 표시됩니다.

## 노드 라이브러리

### 새 노드 라이브러리 디자인
- **기본값**: 활성화됨
- **실험적**: 예
- **기능**: 탭(기본, 전체, 사용자 정의), 향상된 검색, 호버 미리보기가 포함된 재설계된 노드 라이브러리 사이드바를 활성화합니다.

## 노드 검색창

### 노드 검색창 구현
- **기본값**: default
- **옵션**: default, v1 (legacy), litegraph (legacy)
- **실험적**: 예
- **기능**: 노드 검색창의 구현 방식을 선택합니다. `default`가 최신 구현입니다. `v1 (legacy)` 및 `litegraph (legacy)`는 이전 버전의 ComfyUI 검색창으로 전환합니다.

### 검색 결과에 노드 카테고리 표시
- **기본값**: 활성화됨
- **기능**: 검색 결과에 노드 카테고리를 표시할지 여부를 제어합니다. v1 (legacy) 구현에만 적용됩니다.

### 검색 결과에 노드 ID 이름 표시
- **기본값**: 비활성화됨
- **기능**: 검색 결과에 노드 ID 이름을 표시할지 여부를 제어합니다. litegraph (legacy)에는 적용되지 않습니다.

### 검색 결과에 노드 빈도 표시
- **기본값**: 비활성화됨
- **기능**: 검색 결과에 노드 사용 빈도를 표시할지 여부를 제어합니다. v1 (legacy)에만 적용됩니다.

### 노드 미리보기
- **기본값**: 활성화됨
- **기능**: 검색 결과에 노드 미리보기를 표시할지 여부를 제어합니다. 기본 구현에만 적용됩니다.

### 추가된 노드가 커서를 따름
- **기본값**: 활성화됨
- **기능**: 활성화하면 검색창에서 추가된 노드가 클릭하여 놓을 때까지 커서를 따라다닙니다. 기본 구현에만 적용됩니다.

## 알림 기본 설정

### 버전 업데이트 표시
- **기본값**: 활성화됨
- **기능**: 새 모델 및 주요 새 기능에 대한 업데이트를 표시합니다.

## 노드 위젯

### 위젯 제어 모드
- **옵션**: before, after
- **기본값**: after
- **기능**: 노드 위젯 값 업데이트 시점이 워크플로우 실행 전인지 후인지(예: 시드 값 업데이트)를 제어합니다.

### 텍스트 영역 위젯 맞춤법 검사
- **기본값**: 비활성화됨
- **기능**: 텍스트 영역 위젯이 맞춤법 검사를 활성화할지 여부를 제어하며, 텍스트 입력 중 맞춤법 검사 기능을 제공합니다. 이 기능은 브라우저의 spellcheck 속성을 통해 구현됩니다.

### 텍스트 영역 위젯 글꼴 크기
- **기본값**: 10
- **범위**: 8 - 24
- **기능**: 텍스트 영역 위젯의 글꼴 크기를 설정합니다. 텍스트 입력 상자의 텍스트 표시 크기를 조정하여 가독성을 향상시킵니다.

## 큐

### 큐 기록 크기
- **기본값**: 64
- **범위**: 2 - 256
- **기능**: 사이드바 큐 기록 패널에 기록되는 큐 기록 크기를 제어합니다. 값이 클수록 더 많은 큐 기록이 기록됩니다.

### 고정된 작업 기록/큐 패널
- **기본값**: 비활성화됨 (나이틀리 빌드에서는 활성화됨)
- **실험적**: 예
- **기능**: 떠 있는 작업 큐 패널을 작업 기록 사이드 패널에 내장된 동등한 작업 큐로 대체합니다. 비활성화하면 떠 있는 패널 레이아웃으로 되돌릴 수 있습니다.

## 큐 버튼

### 일괄 처리 한도
- **기본값**: 100
- **기능**: 한 번의 클릭으로 큐에 추가할 수 있는 최대 작업 수를 설정하여, 실수로 너무 많은 작업을 큐에 추가하는 것을 방지합니다.

## 검증

### 워크플로우 검증
- **기본값**: 비활성화됨
- **기능**: 워크플로우 로드 시 구조적 및 연결적 정확성을 보장합니다.
- **검증 과정**: 검증 과정은 두 단계로 이루어집니다:
    - 스키마 검증: Zod 스키마를 사용해 워크플로우 구조를 검증합니다.
    - 링크 복구: 노드 간 연결 문제를 확인하고 복구합니다.
- **오류 처리**: 검증 실패 시 오류 메시지가 표시되지만, 워크플로우 로딩이 차단되지는 않습니다.

## 창

### 창 닫을 때 확인 표시
- **기본값**: 활성화됨
- **기능**: 수정되었으나 저장되지 않은 워크플로우가 있을 때 브라우저 창이나 탭을 닫을 때 확인 대화상자를 표시할지 여부를 제어하여, 실수로 창을 닫아 저장되지 않은 워크플로우가 손실되는 것을 방지합니다.

## 워크플로우

### 워크플로우 상태 유지 및 페이지 (재)로드 시 복원
- **기본값**: 활성화됨
- **기능**: 페이지 (재)로드 시 워크플로우 상태를 복원하여 페이지 새로고침 후에도 워크플로우 내용을 유지합니다.

### 자동 저장
- **옵션**: off, after delay
- **기본값**: off
- **기능**: 워크플로우의 자동 저장 동작을 제어하여, 워크플로우 변경 사항을 자동으로 저장해 데이터 손실을 방지합니다.

### 자동 저장 지연 시간 (ms)
- **기본값**: 1000
- **기능**: 자동 저장의 지연 시간을 설정하며, 자동 저장이 "after delay"로 설정된 경우에만 적용됩니다.

### 워크플로우 삭제 시 확인 표시
- **기본값**: 활성화됨
- **기능**: 사이드바에서 워크플로우를 삭제할 때 확인 대화상자를 표시할지 여부를 제어하여, 중요한 워크플로우를 실수로 삭제하지 않도록 방지합니다.

### 워크플로우에서 캔버스 위치와 확대/축소 수준 저장 및 복원
- **기본값**: 활성화됨
- **기능**: 워크플로우에서 캔버스 위치와 확대/축소 수준을 저장하고 복원하여, 워크플로우를 다시 열 때 이전 보기 상태를 복원합니다.

### 열린 워크플로우 위치
- **옵션**: Sidebar, Topbar
- **기본값**: Topbar
- **기능**: 열린 워크플로우 탭의 표시 위치를 제어합니다.

### 워크플로우 저장 시 파일명 입력 요청
- **기본값**: 활성화됨
- **기능**: 워크플로우를 저장할 때 파일명 입력을 요청할지 여부를 제어하여, 사용자가 워크플로우 파일명을 사용자 지정할 수 있도록 합니다.

### 워크플로우 저장 시 노드 ID 정렬
- **기본값**: 비활성화됨
- **기능**: 워크플로우 저장 시 노드 ID를 정렬할지 여부를 결정하여, 워크플로우 파일 형식을 더욱 표준화하고 버전 관리에 편리하게 만듭니다.

### 누락된 모델 경고 표시
- **기본값**: 활성화됨 (Comfy Cloud에서는 비활성화됨)
- **실험적**: 예
- **기능**: 워크플로 파일의 위젯 값에 모델 링크 정보를 추가하여 모델 파일 로드 시 프롬프트를 지원합니다. 활성화하면 로컬에 해당 모델 파일이 없는 경우 워크플로에서 누락된 모델에 대한 경고가 표시됩니다.

### 누락된 노드 경고 표시 (노드 교체 제안으로 대체됨)
- **기본값**: 활성화됨
- **기능**: 워크플로우에서 누락된 노드에 대한 경고를 표시할지 여부를 제어하여, 사용자가 워크플로우에서 사용할 수 없는 노드를 식별하도록 돕습니다. 이 기능은 [노드 교체 제안 활성화](#enable-node-replacement-suggestions) 설정으로 대체되었으며, 이 설정은 단순한 경고 대신 더 유용한 교체 가능한 제안을 제공합니다.

### 워크플로우 지우기 시 확인 요구
- **기본값**: 활성화됨
- **기능**: 워크플로우를 지울 때 확인 대화상자를 표시할지 여부를 제어하여, 실수로 워크플로우 내용이 지워지는 것을 방지합니다.

### 기존 서브그래프 블루프린트 덮어쓰기 시 확인 요구
- **기본값**: 활성화됨
- **기능**: 기존 서브그래프 블루프린트를 새 것으로 덮어쓰려고 할 때 확인 대화상자를 표시할지 여부를 제어합니다.
