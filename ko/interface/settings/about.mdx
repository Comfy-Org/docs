About 페이지는 ComfyUI 설정 시스템의 정보 표시 패널로, 애플리케이션 버전 정보, 관련 링크 및 시스템 통계를 표시하는 데 사용됩니다. 이러한 설정은 피드백을 제출하거나 문제를 보고할 때 중요한 정보를 제공할 수 있습니다.

![about](/images/interface/setting/settings-about.jpg)

### 버전 정보 배지

About 페이지에는 다음과 같은 핵심 버전 정보가 표시됩니다:

- **ComfyUI 버전**: 백엔드 ComfyUI 버전을 표시합니다. 형식은 배포판에 따라 다릅니다:
  - **OSS (오픈 소스)**: `ComfyUI <커밋 해시>`로 표시되며, 주요 [GitHub 저장소](https://github.com/Comfy-Org/ComfyUI)로 연결됩니다.
  - **Desktop**: `ComfyUI v<버전>`으로 표시되며, 동일한 GitHub 저장소로 연결됩니다.
  - **Cloud**: `ComfyUI <커밋 해시>`로 표시되며, 클라우드 아이콘과 함께 [comfy.org](https://www.comfy.org)로 연결됩니다.
- **ComfyUI_frontend 버전**: 프론트엔드 인터페이스 버전 번호를 표시하며, 프론트엔드 GitHub 저장소로 연결됩니다.
- **Templates 버전**: `systemStats.system.installed_templates_version`을 사용할 수 있는 경우 Templates 배지가 표시됩니다:
  - 레이블: `Templates v<버전>`
  - 링크: [https://pypi.org/project/comfyui-workflow-templates/](https://pypi.org/project/comfyui-workflow-templates/)
  - 아이콘: `pi pi-book`
  - 설치된 버전이 필요한 버전과 다른 경우 배지가 **danger** 심각도 수준(빨간색)으로 표시됩니다.
- **Discord 커뮤니티**: ComfyOrg Discord 서버로 연결되는 링크를 제공합니다.
- **ComfyOrg**: ComfyOrg 공식 웹사이트로 연결됩니다.

배지는 다음 순서로 표시됩니다: ComfyUI → Frontend → Templates (설치된 경우) → Discord → ComfyOrg.

<Tip>
여기 표시된 버전 정보는 주로 안정적인 버전 정보에 해당하므로, 나이틀리 버전을 사용 중인 경우 해당 커밋 해시가 여기에 표시되지 않습니다. 나이틀리 버전을 사용하는 경우 해당 ComfyUI 메인 디렉토리에서 `git log` 명령어를 사용해 해당 커밋 해시 및 기타 정보를 확인할 수 있습니다.
또 다른 일반적인 문제는 업데이트 중 서로 다른 종속성 패키지가 실패하거나 롤백될 수 있다는 점입니다.
</Tip>

### 커스텀 노드 배지

커스텀 노드가 설치된 경우 About 페이지는 커스텀 노드가 제공하는 추가 배지 정보도 표시합니다. 이러한 배지는 각 커스텀 노드가 `aboutPageBadges` 속성을 통해 등록합니다.

### 시스템 정보

페이지 하단에는 배포판에 따라 다른 상세한 시스템 통계가 표시됩니다:

**OSS / 일반 버전:**
- **OS**: 운영 체제 식별자
- **Python 버전**: 설치된 Python 버전
- **Embedded Python**: 내장 Python 세부 정보 (해당하는 경우)
- **Pytorch 버전**: PyTorch 프레임워크 버전
- **인수 (argv)**: ComfyUI를 시작하는 데 사용된 명령줄 인수
- **RAM 전체**: 사용 가능한 전체 시스템 RAM
- **RAM 사용 가능**: 현재 사용 가능한 시스템 RAM
- **Templates 버전**: 설치된 버전이 오래된 경우 danger 스타일로 표시됩니다.

**Cloud 버전:**
- **Cloud 버전**: 클라우드 플랫폼 버전
- **ComfyUI 버전**: 백엔드 ComfyUI 버전
- **Frontend 버전**: 프론트엔드 인터페이스 버전
- **Templates 버전**: 워크플로우 템플릿 버전

**장치** (호환되는 하드웨어가 감지된 경우 표시):
- 장치 이름 및 유형 (예: GPU 모델)
- **VRAM 전체 / VRAM 사용 가능**: 그래픽 메모리 통계
- **Torch VRAM 전체 / Torch VRAM 사용 가능**: PyTorch 관리 VRAM 통계
- 여러 장치는 **TabView**로 표시되며, 장치당 하나의 탭이 있습니다.

하단에는 **시스템 정보 복사** 버튼이 제공되어, 모든 시스템 정보를 형식화된 텍스트 블록으로 클립보드에 복사하여 문제 보고 시 쉽게 공유할 수 있습니다.

### 버전 업데이트 알림

`ShowVersionUpdates` 설정 (Comfy → 알림 기본 설정에서 찾을 수 있음)은 About 페이지 및 인터페이스의 다른 곳에서 버전 업데이트 알림을 표시할지 여부를 제어합니다.

## 확장 개발자 가이드

확장 개발자는 확장 구성에 `aboutPageBadges` 속성을 추가하여 About 페이지에 사용자 정의 배지를 추가할 수 있습니다:

```javascript
app.registerExtension({
  name: 'MyExtension',
  aboutPageBadges: [
    {
      label: 'My Extension v1.0.0',
      url: 'https://github.com/myuser/myextension',
      icon: 'pi pi-github'
    }
  ]
})
```
