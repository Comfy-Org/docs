About 페이지는 ComfyUI 설정 시스템의 정보 표시 패널로, 애플리케이션 버전 정보, 관련 링크, 시스템 통계를 보여주는 데 사용됩니다. 이러한 설정은 피드백을 제출하거나 문제를 보고할 때 우리에게 중요한 정보를 제공할 수 있습니다.

![about](/images/interface/setting/settings-about.jpg)

### 버전 정보 배지

About 페이지에는 다음과 같은 핵심 버전 정보가 표시됩니다:

- **ComfyUI 버전**: 백엔드 ComfyUI 버전 번호를 표시하며, 공식 GitHub 저장소와 연결되어 있습니다.
- **ComfyUI_frontend 버전**: 프론트엔드 인터페이스 버전 번호를 표시하며, 프론트엔드 GitHub 저장소와 연결되어 있습니다.  
- **Discord 커뮤니티**: ComfyOrg Discord 서버로 연결되는 링크를 제공합니다.
- **공식 웹사이트**: ComfyOrg 공식 웹사이트로 연결됩니다.

<Tip>
여기 표시된 버전 정보는 주로 안정적인 버전 정보에 해당하므로, 만약 야간 버전을 사용하고 있다면 여기서 해당 커밋 해시는 표시되지 않습니다. 야간 버전을 사용하는 경우, 해당 ComfyUI 메인 디렉토리에서 `git log` 명령어를 사용해 해당 커밋 해시 및 기타 정보를 확인할 수 있습니다.
또 다른 흔한 문제는 업데이트 과정에서 서로 다른 종속성 패키지가 실패하거나 롤백될 수 있다는 점입니다.
</Tip>

### 커스텀 노드 배지

커스텀 노드가 설치된 경우, About 페이지는 커스텀 노드가 제공하는 추가 배지 정보도 표시합니다. 이 배지는 각 커스텀 노드가 `aboutPageBadges` 속성을 통해 등록한 것입니다.

### 시스템 정보

페이지 하단에는 다음을 포함한 상세한 시스템 통계가 표시됩니다:
- 하드웨어 구성 정보
- 소프트웨어 환경 정보
- 시스템 성능 데이터

## 확장 개발자 가이드

확장 개발자는 자신의 확장 구성에 `aboutPageBadges` 속성을 추가함으로써 About 페이지에 맞춤형 배지를 추가할 수 있습니다:

```javascript
app.registerExtension({
  name: 'MyExtension',
  aboutPageBadges: [
    {
      label: 'My Extension v1.0.0',
      url: 'https://github.com/myuser/myextension',
      icon: 'pi pi-github'
    }
  ]
})
```
