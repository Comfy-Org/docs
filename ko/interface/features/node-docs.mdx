노드 문서는 워크플로우 내 어떤 노드든 빠르게 문서를 확인할 수 있도록 해줍니다. 노드의 기능, 매개변수 및 사용 예시에 대한 상세한 정보를 ComfyUI 내에서 바로 확인할 수 있습니다.

## 노드 문서 보기 방법

![노드 문서](/images/interface/features/nod-docs/node-docs-1.jpg)

1. **노드 선택:**  
   워크플로우의 어떤 노드나 클릭하여 선택하세요.

2. **노드 문서 열기:**  
   선택 도구 모음에서 노드 정보 아이콘을 클릭해 노드 문서 페이지를 엽니다.

3. **노드 문서 종료:**  
   노드 문서를 종료하려면 왼쪽 상단의 화살표 아이콘을 클릭하세요.

또한 노드 라이브러리를 통해 노드 문서에 접근할 수도 있으며, 노드 위에 마우스를 올린 후 "?" 아이콘을 클릭하면 노드 문서가 열립니다.

![노드 문서](/images/interface/features/nod-docs/node-docs-2.jpg)


## 내장 노드 문서

ComfyUI의 내장 노드 문서는 [embedded-docs 리포지토리](https://github.com/Comfy-Org/embedded-docs)에서 관리됩니다.

**내장 노드 문서 기여하기:**  
- 사용자는 내장 노드 문서의 개선점과 추가 사항을 기여할 수 있습니다.  
- [embedded-docs 리포지토리](https://github.com/Comfy-Org/embedded-docs)에 풀 리퀘스트를 제출하세요.  
- ComfyUI 커뮤니티 전체의 문서 품질 향상에 기여해 주세요.

## 사용자 정의 노드 문서

사용자 정의 노드 역시 자체 문서를 포함할 수 있습니다. 사용자 정의 노드 작성자가 문서를 제공하면 해당 노드를 선택할 때 자동으로 노드 문서 패널에 표시됩니다.
