부분 실행 기능은 ComfyUI 노드 선택 도구 상자에 위치해 있습니다. 이 기능을 사용하면 모든 노드를 실행하는 대신 워크플로우의 일부만 실행할 수 있습니다. 이 기능은 선택한 노드가 출력 노드일 때만 사용 가능하며, 사용 가능한 경우 녹색 삼각형 아이콘으로 표시됩니다.

## 부분 실행이란 무엇인가요?

이름에서 알 수 있듯이, 부분 실행은 워크플로우의 모든 노드를 실행하는 대신 워크플로우의 일부만 실행할 수 있게 해줍니다.

![부분 실행](/images/interface/features/partial-execution/partial-execution-vs-run-workflow.jpg)

위 다이agram은 부분 실행과 전체 워크플로우 실행을 비교한 것입니다:

1. 부분 실행(왼쪽): 시작 노드부터 출력 노드까지의 워크플로우 브랜치만 실행합니다.
2. 전체 워크플로우 실행(오른쪽): 워크플로우의 모든 노드를 실행합니다.

이 기능을 통해 매번 전체 워크플로우를 실행하는 대신 워크플로우의 특정 부분만 유연하게 실행할 수 있습니다.

## 부분 실행 기능을 사용하는 방법은 무엇인가요?

![부분 실행 요구사항](/images/interface/features/partial-execution/requirement.jpg)

부분 실행 기능을 사용하려면 현재 선택된 노드가 미리보기 노드나 저장 노드와 같은 출력 노드여야 합니다. 해당 노드가 조건을 충족하면, 노드를 선택한 후 도구 상자의 버튼이 녹색 삼각형 아이콘으로 표시됩니다. 이 아이콘을 클릭하면 부분 워크플로우를 실행할 수 있습니다.

## 일반적인 문제

Q: 이 기능을 사용할 때 왜 모든 노드가 실행되나요?
A: ComfyUI 프론트엔드 버전이 v1.23.4 이후인지 확인해 주세요. 또는 v1.24.x 버전이 필요할 수도 있습니다. 해당 버그는 v1.24.x 버전 부근에서 수정되었으므로, 프론트엔드 버전이 요구 사항을 충족하도록 ComfyUI를 최신 버전으로 업데이트해 주세요.
