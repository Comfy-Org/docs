ComfyUI는 서버로 실행되고 API를 통해 접근 가능하며, 커스텀 노드로 확장할 수 있고 명령줄에서 관리 가능한 모듈식 GenAI 추론 엔진입니다. 아래에서 원하는 방식을 선택하세요.

## ComfyUI를 서버로 배포하기

자신의 환경에서 ComfyUI를 실행하고 이를 API 엔드포인트로 노출하세요. WebSocket 메시지 프로토콜, 사용 가능한 라우트 및 실행 모드에 대해 알아보세요.

</Card>

## 클라우드 API

자체 하드웨어를 관리하지 않고도 Comfy Cloud에서 워크플로를 프로그래밍 방식으로 실행하세요. REST API를 사용하고 OpenAPI 스펙을 둘러보며, API 키를 통해 파트너 노드를 통합하세요.

</Card>

## 에이전트 도구 / MCP

모델 컨텍스트 프로토콜(MCP)을 통해 AI 에이전트를 ComfyUI와 연결하세요. 호스팅된 클라우드 MCP를 사용하거나 현지에서 파트너 MCP 서버를 실행하여 30개 이상의 제공업체에서 생성하세요.

<Card title="MCP 개요" icon="robot" href="/ko/agent-tools">
  클라우드 MCP와 파트너 MCP를 비교하고, AI 에이전트 통합에 적합한 설정을 찾아보세요.
</Card>

참고: [Comfy Cloud MCP](/ko/mcp) · [Comfy 파트너 MCP](/ko/agent-tools/partner-mcp)

## 레지스트리

Comfy 레지스트리를 통해 커스텀 노드를 게시하고 관리하세요. 레지스트리는 버전 관리, 보안 스캔 및 커뮤니티를 위한 검색 기능을 제공합니다.

<Card title="게시 개요" icon="puzzle-piece" href="/ko/registry/overview">
  게시자로 등록하고 첫 번째 노드를 게시하며, 기존 노드를 인수하고 자동 릴리스를 위한 CI/CD를 설정하세요.
</Card>
