## API 키 관리하기

로그인 후 [platform.comfy.org](https://platform.comfy.org)에서 모든 API 키를 확인하고 관리할 수 있습니다.

- **기존 키 보기**: 각 키의 이름과 생성 날짜 확인
- **키 삭제**: 키가 유출되었거나 더 이상 필요하지 않다면 즉시 삭제하세요.
- **추가 키 생성**: 서로 다른 애플리케이션용으로 여러 개의 API 키를 가질 수 있습니다.

<Warning>
  API 키는 생성 시 한 번만 표시됩니다. 즉시 안전한 위치에 저장하세요. 만약 키를 분실했다면 삭제하고 새 키를 생성하세요.
</Warning>

## API 키 사용하기

API 키를 갖게 되면 다음을 수행할 수 있습니다:

1. **로컬에서 ComfyUI에 로그인** — **설정 → 사용자**로 이동해 'Comfy API 키' 로그인 선택
2. **파트너 노드 API 호출** — 워크플로우를 전송할 때 `extra_data.api_key_comfy_org`에 키 포함
3. **클라우드 API 사용** — 클라우드 API 엔드포인트 요청 헤더에 키 포함

보안을 위해 API 키를 절대 공유하거나 버전 관리에 커밋하지 마세요.
