[PR #8041](https://github.com/Comfy-Org/ComfyUI/pull/8041)부터 ComfyUI는 특정 프론트엔드 인터페이스가 필요 없이 ComfyUI 계정 API 키를 통해 내장된 유료 파트너 노드를 직접 사용할 수 있도록 지원합니다(프론트엔드 없이 실행할 수도 있습니다).

이는 다음과 같은 요소를 결합한 워크플로우를 생성할 수 있음을 의미합니다:
- 로컬 OS 모델
- 맞춤형 노드 커뮤니티의 도구
- 인기 있는 유료 모델

그런 다음 Comfy 웹서버 API에 단순히 프롬프트를 보내 모든 작업을 함께 실행하며, 모든 오케스트레이션을 처리하도록 할 수 있습니다.

이는 Comfy를 백엔드 서비스로 사용하고자 하는 사용자나 명령줄을 통해 자신의 프론트엔드와 함께 사용하는 경우 등에 유용합니다.

## 사전 요구사항

유료 파트너 노드를 호출하려면 ComfyUI 계정 API 키가 필요합니다:
- [ComfyUI 계정 API 키](/development/api-development/getting-an-api-key)
- 충분한 계정 크레딧

<Note>
  **중요:** 이 페이지는 워크플로우에서 유료 파트너 노드에 접근하는 데 사용되는 **ComfyUI 계정 API 키**를 설명합니다. 만약 맞춤형 노드를 레지스트리에 게시하려는 경우, [노드 게시하기](/registry/publishing)를 참조하세요.
</Note>

ComfyUI 계정 API 키를 사용해 유료 파트너 노드를 호출하려면 먼저 [ComfyUI 플랫폼](https://platform.comfy.org/login)에서 계정을 등록하고 [API 키를 생성](/development/api-development/getting-an-api-key)해야 합니다.

<Card title="API 키로 로그인" icon="link" href="/ko/interface/user#logging-in-with-an-api-key">
API 키로 로그인하는 방법은 사용자 인터페이스 섹션을 참고하세요.
</Card>

ComfyUI 계정에 해당 기능을 테스트할 만큼 충분한 크레딧이 있는지 확인해야 합니다.

</Card>


## 파이썬 예제

다음은 파이썬 코드를 사용해 파트너 노드를 포함한 워크플로우를 ComfyUI API로 전송하는 예시입니다:

```python
"""ComfyUI 헤드리스 또는 대체 프론트엔드에서 파트너 노드 사용하기

API 키를 프롬프트에 포함해 파트너 노드를 포함한 ComfyUI 워크플로우를 실행할 수 있습니다.
API 키는 페이로드의 `extra_data` 필드에 추가해야 합니다.
아래는 이를 수행하는 방법의 예시입니다.

더 알아보기:

- 파트너 노드 개요: https://docs.comfy.org/tutorials/partner-nodes/overview
- API 키를 생성하려면 여기에 로그인하세요: https://platform.comfy.org/login
"""

import json
from urllib import request

SERVER_URL = "http://127.0.0.1:8188"

# 파트너 노드를 포함한 프롬프트/작업(“API 형식” 워크플로우)이 있습니다.
workflow_with_api_nodes = """{
  "11": {
    "inputs": {
      "prompt": "꿈꾸는 듯한 초현실적인 젊은 여성의 반신 초상화, 명상 중입니다. 그녀는 파스텔 핑크로 염색된 짧고 곧은 보브컷 머리를 하고 있으며, 부드러운 앞머리가 이마를 덮고 있습니다. 눈은 부드럽게 감겨 있고, 손은 차분하게 열린 손바닥 자세로 들어 올려져 있으며, 손가락은 살짝 구부러져 마치 공중에 떠 있는 듯하거나 깊은 집중 상태에 있는 듯합니다. 그녀는 패치워크처럼 파스텔 색조의 타일로 이루어진 화려한 드레스를 입고 있으며, 구름과 별, 무지개가 장식되어 있습니다. 주변에는 무지개 빛깔을 반사하는 반투명하고 무지갯빛 비누방울들이 떠다닙니다. 배경은 솜사탕 구름과 선명한 무지개 물결로 가득한 환상적인 하늘로, 전체 장면은 마법같고 꿈같은 분위기를 연출합니다. 젊음의 평온함과 기발한 분위기, 생동감 넘치는 부드러운 조명에 중점을 두었습니다.",
      "prompt_upsampling": false,
      "seed": 589991183902375,
      "aspect_ratio": "1:1",
      "raw": false,
      "image_prompt_strength": 0.4000000000000001,
      "image_prompt": [
        "14",
        0
      ]
    },
    "class_type": "FluxProUltraImageNode",
    "_meta": {
      "title": "Flux 1.1 [pro] Ultra Image"
    }
  },
  "12": {
    "inputs": {
      "filename_prefix": "ComfyUI",
      "images": [
        "11",
        0
      ]
    },
    "class_type": "SaveImage",
    "_meta": {
      "title": "이미지 저장"
    }
  },
  "14": {
    "inputs": {
      "image": "example.png"
    },
    "class_type": "LoadImage",
    "_meta": {
      "title": "이미지 불러오기"
    }
  }
}"""


prompt = json.loads(workflow_with_api_nodes)
payload = {
    "prompt": prompt,
    # 페이로드에 `api_key_comfy_org`를 추가하세요.
    # 여러 클라이언트를 관리하는 경우 먼저 관련 사용자로부터 키를 가져올 수 있습니다.
    "extra_data": {
        "api_key_comfy_org": "comfyui-87d01e28d*******************************************************"  # 실제 키로 교체하세요
    },
}
data = json.dumps(payload).encode("utf-8")
req = request.Request(f"{SERVER_URL}/prompt", data=data)
request.urlopen(req)

```
