</Note>

## 새로운 UI 개요

![새로운 UI](/images/manager/new_ui_labeled.jpg)

1. **왼쪽 사이드바(필터)**: 설치된 노드, 워크플로우 내 노드, 누락된 노드, 업데이트 가능한 노드 등을 필터링합니다.
2. **상단 검색창**: 노드 팩(Node Pack) 또는 개별 노드(Node)를 검색하고, 필터 드롭다운을 사용해 검색 유형을 전환합니다.
3. **오른쪽 세부 정보 패널**: 노드를 클릭하면 설명, 활성화 상태, 버전 정보 등 상세한 정보가 표시됩니다. 설명 탭에는 리포지토리 정보가 포함되며, 노드 탭에서는 모든 노드의 미리보기가 제공됩니다.

## 노드 관리

새로운 ComfyUI Manager를 사용해 커스텀 노드를 관리하는 방법을 알아보세요

### 노드 검색
![필터](/images/manager/new_ui_filter.jpg)

Manager는 노드 팩과 개별 노드를 각각 별도로 검색할 수 있도록 지원합니다. 위와 같이 필터를 토글하여 검색 유형을 전환할 수 있습니다.
- 노드 팩: 완전한 커스텀 노드 팩
- 개별 노드: 노드 팩 내 개별 노드 검색

### 노드 설치

![노드 설치](/images/manager/new_ui_install_pack.jpg)
1. 해당 노드 카드를 선택하세요.
2. 확장된 노드 정보에서 **설치**를 클릭하세요.
3. 또는 **버전**에서 특정 버전을 선택해 설치하세요.

### 노드 업데이트
**업데이트 가능** 필터 아래에서는 현재 노드 목록에서 업데이트가 가능한 노드를 필터링할 수 있습니다.
![노드 업데이트](/images/manager/new_ui_update_pack.jpg)
1. 업데이트 가능한 노드에는 업데이트 화살표 표시가 나타납니다.
2. **버전**에서 특정 버전을 선택하세요.
3. 버전을 선택한 후 **업데이트** 버튼을 클릭해 업데이트하세요.

### 누락된 노드 찾기

![누락된 노드 찾기](/images/manager/new_ui_missing_packs_prompt.jpg)
ComfyUI Manager가 올바르게 설치된 경우, 누락된 노드가 포함된 워크플로우를 로드할 때 알림이 표시됩니다.
1. **모두 설치**를 선택해 한 번에 모든 노드를 설치할 수 있습니다.
2. 또는 **Manager 열기**를 선택해 Manager를 열고 설치하기 전에 세부 정보를 살펴볼 수 있습니다.

![누락된 노드 찾기](/images/manager/new_ui_missing_pack.jpg)
워크플로우에서 누락된 노드를 찾으려면 해당 노드를 선택한 후 미리보기 패널의 **누락됨** 버튼을 클릭해 누락된 노드를 찾으세요.

### 노드 제거

![노드 제거](/images/manager/new_ui_uninstall_pack.jpg)
설치된 노드를 제거하려면 해당 노드를 선택한 후 미리보기 패널의 **제거** 버튼을 클릭해 제거하세요.

## 자주 묻는 질문
1. 필요한 노드를 찾을 수 없는 이유는 무엇인가요?
새로운 Manager는 오직 [레지스트리](/ko/registry/overview)에서 노드를 설치하는 것을 지원합니다. 만약 귀하의 노드가 레지스트리에 등록되지 않았다면 먼저 Manager에 등록해 주세요.

2. git을 통해 설치하는 옵션이 보이지 않는 이유는 무엇인가요?
ComfyUI 사용자 시스템의 보안과 안정성을 위해 새로운 UI에서는 git을 통한 노드 설치를 지원하지 않습니다. 자세한 내용은 [커스텀 노드 수동 설치](/ko/installation/install_custom_node)를 참고해 주세요.
