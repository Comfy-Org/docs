Comfy Organization Inc는 글로벌 사용자 기반을 지원하기 위해 여러 통화로 결제를 처리합니다.

## 기본 통화

모든 구독 가격은 기본적으로 **미국 달러(USD)**로 표시됩니다.

## 결제 수단 이용 가능 여부

### Alipay 및 WeChat Pay

- **Alipay**는 현재 현지 API 크레딧과 Comfy Cloud 구독 모두에서 지원됩니다.
- **WeChat Pay**는 현재 현지 API 크레딧에서만 이용 가능합니다.

## 통화 선택

귀하의 결제 통화는 일반적으로 다음에 따라 결정됩니다:

- 계정 위치
- 청구 주소
- 결제 수단의 기본 통화

## 통화 변환

### 자동 변환

카드의 통화가 청구 통화와 다를 경우:

- 은행 또는 카드 발급사가 자동으로 금액을 환전합니다.
- 환율은 귀하의 금융기관에서 설정합니다.
- 추가적인 통화 변환 수수료가 부과될 수 있습니다.

### 변환 수수료

- 통화 변환 수수료는 Comfy Organization Inc가 아닌 귀하의 은행에서 부과합니다.
- 이 수수료는 보통 거래 금액의 1~3% 정도입니다.
- 구체적인 환전 수수료율은 은행에 문의하세요.

## 귀하의 통화로 요금 확인하기

현지 통화로 대략적인 요금을 확인하려면:

1. 은행이나 카드 명세서를 확인하세요.
2. 은행의 통화 환산기를 사용하세요.
3. 환율은 매일 변동됩니다.

## 청구 통화 변경하기

청구 통화를 변경하려면:

1. 저희 [지원팀](https://support.comfy.org/)에 연락하세요.
2. 선호하는 통화를 알려주세요.
3. 통화 변경 시 환율에 따라 구독 가격이 달라질 수 있습니다.

## 지역별 가격 차이

구독 가격은 다음과 같은 이유로 지역별로 약간씩 차이가 날 수 있습니다:

- 현지 세금 규정
- 통화 환율
- 지역별 가격 조정

## 세금 및 부가가치세

귀하의 위치에 따라 추가 요금이 부과될 수 있습니다:

- 판매세 (미국 고객)
- 부가가치세 (유럽 고객)
- 소비세 (호주, 캐나다 및 기타 해당 지역)

이러한 세금은 귀하의 청구 주소를 기준으로 계산되며 전체 금액에 추가됩니다.
