Ideogram 4.0은 Ideogram에서 출시한 최신 텍스트 기반 이미지 생성 모델로, 완전히 자체 하드웨어에서 실행되는 오픈소스 모델입니다. 뛰어난 포토리얼리즘 품질, 정확한 텍스트 렌더링, 정밀한 스타일 제어를 제공합니다. 일반 자연어 또는 **구조화된 JSON 프롬프트**를 사용해 레이아웃, 색상, 이미지 내 텍스트까지 세밀하게 제어할 수 있습니다.

### 프롬프트 형식

오픈소스 워크플로는 두 가지 프롬프트 모드를 지원합니다:

1. **자연어** — 빠르고 간편하며, 단순한 아이디어에 가장 적합
2. **구조화된 JSON** — 레이아웃, 색상, 스타일을 정밀하게 제어하기 위한 방식

이 워크플로에는 어떤 LLM과 함께 사용해도 일치하는 JSON 프롬프트를 생성할 수 있는 프롬프트 구성 템플릿이 포함되어 있습니다.

워크플로 내부 주석에는 다음과 같이 설명되어 있습니다:
> 이 모델은 구조화된 JSON 캡션(장면 요약, 스타일 블록, 배경, 선택적 개체별 설명(경계 상자 및 16진수 색상 팔레트 포함))으로 학습되었습니다. 공식 추론에서는 해당 스키마에 맞게 프롬프트를 검증합니다.

### 워크플로 단계

1. 모델을 다운로드해 올바른 디렉터리에 저장하세요 (위 표 참조)
2. 워크플로 파일을 다운로드해 ComfyUI로 드래그하세요
3. Ideogram4 서브그래프 노드에 프롬프트(자연어 또는 구조화된 JSON)를 입력하세요
4. (선택) `ResolutionSelector` 노드를 사용해 해상도를 조정하세요
5. `Run`을 클릭하거나 단축키 `Ctrl(cmd) + Enter`를 사용해 이미지를 생성하세요
6. 결과는 `Save Image` 노드에서 확인하세요: 출력 파일은 `ComfyUI/output/`에 저장됩니다.

### 안전 필터 참고사항

"이미지가 안전 필터에 의해 차단되었습니다"라는 메시지가 표시되면 이는 Ideogram 4.0의 내장 안전 필터 때문입니다. JSON이 아닌(일반 텍스트) 프롬프트는 오탐률이 더 높습니다. 구조화된 JSON 프롬프트를 사용하면 프롬프트가 차단될 가능성이 줄어듭니다. 자세한 내용은 [Ideogram 4 공식 프롬프트 가이드](https://github.com/ideogram-oss/ideogram4/blob/main/docs/prompting.md#safety-filter)를 참조하세요.
