**Boogu-Image-0.1**은 Apache-2.0 오픈소스 통합 이미지 생성 및 편집 모델 제품군입니다. 이해와 생성을 통합한 시스템을 사용하여 사진, 텍스트 렌더링, 스타일화 및 이미지 편집 작업 전반에서 경쟁력 있는 성능을 제공합니다.

**모델 변형**:
- **Boogu-Image-0.1-Turbo** – 고품질 생성과 사실적인 사진 표현에 초점을 맞춘 4단계 증류 텍스트 기반 이미지 생성 변형으로, LoRA rank-128 모듈이 포함되어 있습니다. 3~4단계의 샘플링만 필요합니다. 대부분의 텍스트 기반 이미지 생성 용도로 기본값으로 권장됩니다.
- **Boogu-Image-0.1-Base** – 뛰어난 다양성과 제어력을 갖춘 기본 모델로, 미세 조정 및 2K 해상도에서 초고밀도 텍스트 렌더링에 이상적입니다.
- **Boogu-Image-0.1-Edit** – 이미지 편집 및 변환 작업을 위해 지시 기반 제어로 미세 조정된 변형입니다.

**모델 주요 특징**:
- **10B 파라미터**로 소비자 GPU에서 효율적인 추론
- **Apache-2.0 라이선스**로 개방적 사용 및 다운스트림 개발 허용
- **이중 언어 텍스트 렌더링**(중국어 및 영어)으로 안정적인 타이포그래피
- **다양한 기능**: 사실적인 사진, 스타일화, 포스터 디자인 및 정밀한 텍스트 렌더링

**관련 링크**:
