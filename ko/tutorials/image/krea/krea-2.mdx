**Krea 2**는 [Krea AI](https://www.krea.ai)가 처음부터 학습시킨 이미지 생성 모델로, 창의적이고 스타일적인 탐구에 중점을 둡니다.

Krea 2는 함께 작동하도록 설계된 두 가지 모델로 제공됩니다: **LoRA는 RAW에서 학습하고, 추론은 Turbo에서 실행**하세요.
- **Krea 2 RAW**: 전체 스텝 샘플링(52스텝)을 사용하는 베이스 모델입니다. 증류되지 않았으며 다양하고 가소성이 뛰어나 파인튜닝 및 LoRA 학습에 이상적입니다.
- **Krea 2 Turbo**: 빠르고 고품질 생성을 위한 8스텝 증류 체크포인트입니다. RAW에서 학습한 LoRA를 Turbo에 원활하게 적용할 수 있습니다.

**라이선스**: [Krea AI 커뮤니티 라이선스](https://www.krea.ai/krea-2-licensing)

## 사용 가능한 모델 가중치

Krea 2는 두 가지 변형으로 제공됩니다:
- **Krea 2 RAW**: 전체 스텝 샘플링(52스텝)을 사용하는 베이스 모델입니다. 다양성과 가소성을 위해 설계되어 파인튜닝 및 LoRA 학습에 가장 적합합니다.
- **Krea 2 Turbo**: 빠르고 고품질 생성을 위해 구축된 8스텝 증류 체크포인트입니다. RAW에서 학습한 LoRA를 Turbo에 원활하게 적용할 수 있습니다.

워크플로는 다음과 같은 부분으로 구성됩니다:

1. **Text to Image (Krea-2 Turbo) 서브그래프**: 모델 로딩, 프롬프트 처리, 샘플링, VAE 디코드를 포함한 핵심 생성 파이프라인
2. **ResolutionSelector**: 원하는 출력 해상도를 선택하세요. Krea 2는 1K에서 2K까지 출력을 지원합니다. 메가픽셀 값을 2.0으로 설정하면 2K 해상도를 얻을 수 있습니다.
3. **CustomCombo (LoRA 선택기)**: 사용 가능한 스타일 LoRA에 대해 사전 구성된 트리거 단어 선택기입니다. 추가 LoRA를 다운로드한 경우 이 선택기를 사용자 정의하고 해당 LoRA 파일과 페어링할 수 있습니다.
4. **SaveImage**: 생성된 이미지 저장

### 원클릭 생성

가장 간단한 사용법: 서브그래프에 텍스트 프롬프트를 입력하고, 해상도를 선택한 후 **Queue Prompt**를 클릭하기만 하면 됩니다. 기본 설정(8스텝, 프롬프트 향상 켜짐, LoRA 없음)으로 최소한의 설정만으로 고품질 이미지를 생성할 수 있습니다.

### 워크플로 컨트롤

**Text to Image (Krea-2 Turbo)** 서브그래프는 다음과 같은 컨트롤을 제공합니다:

| 컨트롤 | 설명 |
|-------|------|
| **Text String (User Prompt)** | 생성할 이미지를 설명하는 텍스트 프롬프트 |
| **prompt_enhance** | LLM 기반 프롬프트 확장 켜기/끄기 |
| **LLM_max_token** | 프롬프트 확장의 최대 토큰 수 |
| **Width / Height** | 출력 해상도 (ResolutionSelector로 제어) |
| **Seed** | 재현성을 위한 랜덤 시드 |
| **enable_lora?** | 스타일 LoRA 활성화/비활성화 |
| **LoRA Strength** | 스타일 LoRA 적용 강도 |
| **LoRA Trigger Word** | 선택한 LoRA의 트리거 단어 (자동 입력) |

### 스타일 LoRA

Krea는 또한 Krea 2용 스타일 LoRA 컬렉션을 출시했습니다. **CustomCombo** 노드에서 하나를 선택하고 서브그래프에서 `enable_lora?`를 활성화하면 스타일이 적용됩니다:

| LoRA | 트리거 단어 | 권장 강도 | 다운로드 |
|------|-----------|:--------:|---------|
| krea2_coolblue | teal watercolor illustration style | 0.8 | [다운로드](https://huggingface.co/Comfy-Org/Krea-2/blob/main/loras/krea2_coolblue.safetensors) |
| krea2_darkbrush | monochrome ink wash style | 1.0 | [다운로드](https://huggingface.co/Comfy-Org/Krea-2/blob/main/loras/krea2_darkbrush.safetensors) |
| krea2_plasmoid | ethereal shimmering light style | 0.8 | [다운로드](https://huggingface.co/Comfy-Org/Krea-2/blob/main/loras/krea2_plasmoid.safetensors) |
| krea2_warmpastel | muted minimalist sketch style | 0.8 | [다운로드](https://huggingface.co/Comfy-Org/Krea-2/blob/main/loras/krea2_warmpastel.safetensors) |

`.safetensors` 파일을 `ComfyUI/models/loras/` 디렉토리에 넣으세요.

스타일 참조 워크플로는 Krea-2 Turbo 파이프라인에 참조 이미지 조건화를 추가하여 구축됩니다. 하나 이상의 참조 이미지를 업로드하여 생성된 출력물의 미적 스타일, 분위기 및 시각적 방향에 영향을 줍니다.

워크플로는 몇 가지 부분으로 구성됩니다:

1. **이미지 스타일 참조 (Krea-2 Turbo) 서브그래프**: 스타일 참조 지원이 포함된 핵심 생성 파이프라인으로, 모델 로딩, 참조 이미지 조건화, 프롬프트 처리 및 샘플링을 포함합니다.
2. **LoadImage**: 스타일 참조 이미지를 업로드합니다.
3. **ResolutionSelector**: 원하는 출력 해상도를 선택합니다. Krea 2는 1K에서 2K까지의 출력을 지원합니다.
4. **SaveImage**: 생성된 이미지를 저장합니다.

### 스타일 참조 사용

1. **참조 이미지 업로드**: **LoadImage** 노드를 사용하여 하나 이상의 스타일 참조 이미지를 로드합니다.
2. **프롬프트 입력**: 서브그래프의 사용자 프롬프트 입력에 텍스트 설명을 입력합니다.
3. **모델 변형 선택**: 서브그래프의 모델 선택기에서 모델 체크포인트를 선택합니다.
4. **설정 조정**: 해상도, 시드 및 선택적으로 프롬프트 향상 기능을 구성합니다.
5. **실행 대기열 클릭**: `Ctrl+Enter`를 눌러 생성합니다.

이 스타일 참조 워크플로는 전용 확산 모델과 LoRA를 사용합니다. 텍스트 인코더와 VAE는 텍스트-이미지 워크플로와 공유됩니다.
