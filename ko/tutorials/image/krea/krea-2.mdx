**Krea 2**는 [Krea AI](https://www.krea.ai)가 처음부터 학습시킨 이미지 생성 모델로, 창의적이고 스타일적인 탐구에 중점을 둡니다.

Krea 2는 함께 작동하도록 설계된 두 가지 모델로 제공됩니다: **LoRA는 RAW에서 학습하고, 추론은 Turbo에서 실행**하세요.
- **Krea 2 RAW**: 전체 스텝 샘플링(52스텝)을 사용하는 베이스 모델입니다. 증류되지 않았으며 다양하고 가소성이 뛰어나 파인튜닝 및 LoRA 학습에 이상적입니다.
- **Krea 2 Turbo**: 빠르고 고품질 생성을 위한 8스텝 증류 체크포인트입니다. RAW에서 학습한 LoRA를 Turbo에 원활하게 적용할 수 있습니다.

**라이선스**: [Krea AI 커뮤니티 라이선스](https://www.krea.ai/krea-2-licensing)

Krea 2는 두 가지 변형으로 제공됩니다:

The workflow is organized into a few parts:

1. **Text to Image (Krea-2 Turbo) subgraph**: the core generation pipeline, containing model loading, prompt handling, sampling, and VAE decode
2. **ResolutionSelector**: choose your desired output resolution. Krea 2 supports outputs from 1K to 2K — set the megapixels value to 2.0 to get 2K resolution.
3. **CustomCombo (LoRA selector)**: a pre-built trigger word selector for the available style LoRAs. If you download additional LoRAs, you can customize this selector and pair them with the corresponding LoRA files accordingly.
4. **SaveImage**: saves the generated image

### 원클릭 생성

가장 간단한 사용법: 서브그래프에 텍스트 프롬프트를 입력하고, 해상도를 선택한 후 **Queue Prompt**를 클릭하기만 하면 됩니다. 기본 설정(8스텝, 프롬프트 향상 켜짐, LoRA 없음)으로 최소한의 설정만으로 고품질 이미지를 생성할 수 있습니다.

### 워크플로 컨트롤

**Text to Image (Krea-2 Turbo)** 서브그래프는 다음과 같은 컨트롤을 제공합니다:

| 컨트롤 | 설명 |
|-------|------|
| **Text String (User Prompt)** | 생성할 이미지를 설명하는 텍스트 프롬프트 |
| **prompt_enhance** | LLM 기반 프롬프트 확장 켜기/끄기 |
| **LLM_max_token** | 프롬프트 확장의 최대 토큰 수 |
| **Width / Height** | 출력 해상도 (ResolutionSelector로 제어) |
| **Seed** | 재현성을 위한 랜덤 시드 |
| **enable_lora?** | 스타일 LoRA 활성화/비활성화 |
| **LoRA Strength** | 스타일 LoRA 적용 강도 |
| **LoRA Trigger Word** | 선택한 LoRA의 트리거 단어 (자동 입력) |

The style reference workflow builds on the Krea-2 Turbo pipeline by adding reference image conditioning. Upload one or more reference images to influence the aesthetic style, mood, and visual direction of the generated output.

The workflow is organized into a few parts:

1. **Image Style Reference (Krea-2 Turbo) subgraph**: the core generation pipeline with style reference support, containing model loading, reference image conditioning, prompt handling, and sampling
2. **LoadImage**: upload your style reference images
3. **ResolutionSelector**: choose your desired output resolution. Krea 2 supports outputs from 1K to 2K.
4. **SaveImage**: saves the generated image

### 스타일 참조 사용

1. **참조 이미지 업로드**: **LoadImage** 노드를 사용하여 하나 이상의 스타일 참조 이미지를 로드합니다.
2. **프롬프트 입력**: 서브그래프의 사용자 프롬프트 입력에 텍스트 설명을 입력합니다.
3. **모델 변형 선택**: 서브그래프의 모델 선택기에서 모델 체크포인트를 선택합니다.
4. **설정 조정**: 해상도, 시드 및 선택적으로 프롬프트 향상 기능을 구성합니다.
5. **실행 대기열 클릭**: `Ctrl+Enter`를 눌러 생성합니다.

이 스타일 참조 워크플로는 전용 확산 모델과 LoRA를 사용합니다. 텍스트 인코더와 VAE는 텍스트-이미지 워크플로와 공유됩니다.
