Recraft V4는 전문 디자인 작업을 위해 개발된 새로운 이미지 생성 모델입니다. 디자이너들과 협력해 개발되었으며, 구성, 조명, 색상 관계, 재질의 사실성 등을 최적화했습니다.

<UpdateReminder/>

## V4의 새로운 기능

- **래스터 + 벡터를 하나의 모델로** — 동일한 모델에서 포토리얼리즘 이미지, 스타일리시한 일러스트레이션, 제작 준비가 완료된 SVG까지 생성 가능
- **더 나은 디테일** — 더 깨끗한 텍스처, 더 정확한 조명, 더 타이트한 지오메트리. 복잡한 장면에서도 뛰어난 성능을 발휘합니다
- **잘 맞는 얼굴과 해부학** — 여러 인물이나 액션 포즈에서도 신뢰할 수 있는 비율과 표정을 제공합니다
- **텍스트 렌더링** — 간판, 패키징, 인포그래픽용 가독성 높은 텍스트. 짧고 중간 길이의 문구도 잘 처리합니다
- **SVG 출력** — 깔끔한 선과 적절한 색상 분할을 갖춘 편집 가능한 벡터. 제작 수준의 벡터 출력
- **프롬프트 정확성** — 공간 관계, 재질, 반사 등 세부적인 창작 방향을 정확히 따릅니다
- **V4 Pro** — 동일한 모델이지만 더 높은 해상도. 인쇄 및 대형 작업용으로 2048×2048 해상도로 출력됩니다

### 워크플로우 실행 단계

1. (선택사항) `Recraft Style` 노드를 수정해 시각적 스타일을 조정하세요
2. (선택사항) `Recraft Text to Image` 노드의 `prompt` 파라미터를 편집하세요
3. (선택사항) 다른 출력 크기를 위해 `size` 파라미터를 변경하세요
4. `Run` 버튼을 클릭하거나 단축키 `Ctrl(cmd) + Enter`를 사용해 이미지를 생성하세요
5. API가 결과를 반환한 후, `Save Image` 노드에서 생성된 이미지를 확인하세요

## 추가 참고사항

- **Recraft Style**: 사실적인 사진, 디지털 아트, 로고 디자인 등 다양한 사전 설정 스타일 제공
- **시드 파라미터**: 노드가 다시 실행될지 여부를 결정하는 데만 사용되며, 실제 생성 결과에는 시드 값이 영향을 미치지 않습니다
