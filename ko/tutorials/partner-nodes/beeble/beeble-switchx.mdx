[Beeble SwitchX](/ko/built-in-nodes/BeebleSwitchXImageEdit) 파트너 노드를 사용하면 기존 이미지와 비디오의 환경, 조명, 스타일을 변형하면서도 원래의 피사체는 그대로 유지할 수 있습니다. 캐릭터나 피사체 이미지와 참조 환경을 업로드하면 SwitchX가 실제와 같은 조명과 그림자를 적용해 새로운 장면에 피사체를 합성합니다.

이 가이드에서는 이미지 편집과 비디오 편집 워크플로우를 사용하는 방법을 보여드립니다.

## Beeble SwitchX: 이미지 편집

배경 환경을 교체하고 참조 이미지를 통해 피사체의 조명을 다시 설정하세요.

### 작동 방식

이 워크플로우는 두 개의 입력 이미지를 사용합니다:
- **피사체 이미지** — 배경이 포함된 캐릭터나 주요 피사체
- **참조 환경** — 적용하려는 새로운 장면, 조명 또는 스타일

노드는 피사체가 새 환경에 자연스럽게 합성되고 적절한 조명, 그림자, 원근감이 적용된 출력을 생성합니다.

### 실행 단계

1. **피사체 이미지 업로드** — 첫 번째 **LoadImage** 노드를 사용해 피사체 이미지를 로드하세요(예: 헤드폰을 착용한 사람)
2. **참조 이미지 업로드** — 두 번째 **LoadImage** 노드를 사용해 목표 환경이나 스타일을 로드하세요
3. **프롬프트 입력** — **BeebleSwitchXImageEdit** 노드에서 장면을 설명하세요
4. **해상도 설정** — 출력 해상도를 선택하세요
5. **시드 조정** — 다른 결과를 위해 시드 값을 변경하세요
6. **큐 클릭** — `Ctrl+Enter`를 눌러 생성하세요
7. **결과 확인** — 편집된 이미지는 **SaveImage**에 나타나며, 마스크 처리된 컷아웃은 별도로 저장됩니다

### 노드 매개변수

| 매개변수 | 설명 |
|---|---|
| `prompt` | 장면과 환경에 대한 설명 |
| `resolution` | 출력 해상도 (예: 720p) |
| `seed` | 재현 가능한 결과를 위한 임의의 시드 |

### 작동 방식

이 워크플로우는 소스 비디오와 참조 이미지를 받아들인 후, 참조 조명과 환경을 비디오의 마스크 처리된 피사체 영역에 적용합니다. 마스크 해제된 영역은 원래 픽셀과 모션을 그대로 유지합니다. 또한 워크플로우는 간편한 합성을 위해 알파 마스크를 생성합니다.

### 실행 단계

1. **소스 비디오 업로드** — **LoadVideo** 노드를 사용해 입력 비디오를 로드하세요
2. **참조 이미지 업로드** — **LoadImage** 노드를 사용해 목표 환경/스타일을 로드하세요
3. **비디오 범위 설정** — **Video Slice** 노드를 사용해 처리할 프레임을 지정하세요(시작, 끝, 포함 여부)
4. **프롬프트 입력** — **BeebleSwitchXVideoEdit** 노드에서 원하는 환경과 조명을 설명하세요
5. **해상도 설정** — 출력 해상도를 선택하세요
6. **시드 조정** — 다른 결과를 위해 시드 값을 변경하세요
7. **큐 클릭** — `Ctrl+Enter`를 누르세요
8. **결과 확인** — 편집된 비디오는 저장되며, 별도의 알파 마스크 비디오는 합성을 위해 생성됩니다

### 출력물

| 출력물 | 설명 |
|---|---|
| **편집된 비디오** (`SaveVideo`) | 마스크 처리된 영역에 새로운 환경/조명이 적용된 최종 비디오 |
| **알파 마스크** (`SaveVideo`) | 편집된 영역을 합성하기 위한 별도의 알파 비디오 |
