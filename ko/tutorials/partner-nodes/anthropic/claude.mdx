Anthropic Claude는 강력한 추론 능력, 안전성 및 긴 컨텍스트 처리 기능으로 유명한 고성능 AI 모델군입니다. ComfyUI는 파트너 노드를 통해 Anthropic Claude API를 통합하여, 대화 및 이미지 분석 작업에 Claude를 워크플로우에서 바로 사용할 수 있도록 지원합니다.

이 가이드에서는 ComfyUI에서 Anthropic Claude 노드를 사용하는 방법을 안내하겠습니다.

<UpdateReminder/>

## 지원되는 모델

| 모델 | 강점 |
|-------|-----------|
| **Opus 4.7** | Anthropic의 가장 뛰어난 성능을 자랑하는 모델입니다. 에이전트형 코딩, 복잡한 추론 및 미세한 분석에서 획기적인 개선을 이루었습니다. 1M 컨텍스트 창. 지식 최신화일자: 2026년 1월 |
| **Opus 4.6** | 전 세대의 주력 모델로, 전반적인 성능이 뛰어납니다. 1M 컨텍스트 창 |
| **Sonnet 4.6** | 대부분의 작업 부하에 적합한 속도와 지능의 최적 조합입니다. 1M 컨텍스트 창. 빠른 지연 시간 |
| **Sonnet 4.5** | 빠르고 성능이 뛰어나며 일상적인 작업에 적합합니다. 200k 컨텍스트 창 |
| **Haiku 4.5** | 간단한 질의와 높은 처리량의 사용 사례에 가장 빠르고 비용 효율적인 옵션입니다. 200k 컨텍스트 창 |

모든 모델은 텍스트 전용 입력과 멀티모달(텍스트 + 이미지) 입력을 모두 지원하며, 요청당 최대 20개의 이미지를 처리할 수 있습니다.

<Note>
해당 템플릿에서는 역할 프롬프트를 분석하고 생성하는 프롬프트를 구축했습니다. 이를 통해 사용자의 이미지를 해당 드로잉 프롬프트로 해석합니다.
</Note>

워크플로우 이미지의 번호가 표시된 단계를 참고해 기본 워크플로우 실행을 완료하세요:
1. `Load Image` 노드에서 AI에게 해석하도록 할 이미지를 불러옵니다(선택사항 — 텍스트 전용 질의는 건너뛰세요)
2. `Anthropic Claude` 노드에서 `prompt`를 수정해 대화 프롬프트를 설정하거나, `model`을 변경해 다른 Claude 모델을 선택하세요
3. (선택사항) `system_prompt`을 설정해 모델의 행동 지침을 정의하세요
4. `Run`을 클릭하거나 `Ctrl(cmd) + Enter`를 눌러 대화를 실행하세요
5. API가 결과를 반환한 후, `Preview Any` 노드에서 AI 응답을 확인하세요

### 추가 참고사항

- 노드의 고급 옵션에서 `max_tokens`(최대 32000) 및 `temperature`(0.0~1.0)를 조정하세요
- `seed` 매개변수는 노드가 다시 실행되는지 여부를 제어합니다—seed 값과 관계없이 결과는 비결정적입니다
