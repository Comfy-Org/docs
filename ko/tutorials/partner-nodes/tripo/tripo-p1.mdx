[Tripo P1](/ko/built-in-nodes/TripoP1ImageToModelNode) Partner Node는 텍스트 프롬프트, 단일 이미지 또는 멀티뷰 참조에서 빠르고 고품질의 3D 모델 생성을 가능하게 합니다. 생성된 모델은 깔끔한 토폴로지, 최적화된 폴리곤 예산을 가지며 텍스처링, 리깅 및 리타겟팅을 지원하여 게임 엔진 및 3D 애플리케이션에서 즉시 사용할 수 있습니다.

이 가이드에서는 Tripo P1의 세 가지 워크플로우를 살펴보겠습니다: 텍스트 투 모델, 이미지 투 모델, 멀티뷰 투 모델.

### 실행 단계

1. **프롬프트 입력** — **TripoP1TextToModelNode**에서 3D 모델 설명
2. **텍스처 모드 선택** — Textured, None 또는 PBR 중 선택
3. **리파인먼트 활성화** — 쿼드 리메싱을 위한 quad remeshing 토글
4. **시드 설정** — 재현성 제어
5. **큐 실행** — `Ctrl+Enter`를 눌러 생성
6. **미리보기 및 내보내기** — **Preview3D** 노드로 검사, **SaveGLB**로 내보내기

### 주요 처리 노드

초기 생성 후, 워크플로우는 자동으로 다음 노드를 통해 모델을 처리합니다:
- **TripoTextureNode** — 재질 및 텍스처 적용
- **TripoConversionNode** — GLTF 형식(또는 기타 내보내기 형식)으로 변환
- **TripoRefineNode** — 토폴로지 및 지오메트리 리파인
- **TripoRigNode** — 애니메이션용 스켈레톤 추가
- **TripoRetargetNode** — 사전 설정 애니메이션 적용 (기본값: idle)

2. **텍스처 모드 선택** — Textured, None 또는 PBR 중 선택

4. **텍스처 유형 선택** — `original_image` 또는 `standard` 옵션 중 선택
5. **큐 실행** — `Ctrl+Enter`를 눌러 생성

1. **멀티뷰 이미지 업로드** — 정면, 좌측, 후면 등 여러 각도의 **LoadImage** 노드 사용
2. **텍스처 모드 선택** — Textured, None 또는 PBR 중 선택
3. **디테일 수준 선택** — `standard` 및 `detailed` 옵션 중 선택
4. **텍스처 참조 설정** — 텍스처를 결정할 이미지 선택
5. **큐 실행** — `Ctrl+Enter`를 눌러 생성

### 입력 팁

- 최상의 결과를 위해 **정면 및 측면 뷰**를 모두 제공하세요
- 모든 참조 이미지에서 일관된 조명과 포즈를 유지하세요
- 더 높은 품질의 입력 이미지가 더 나은 텍스처 출력을 생성합니다
