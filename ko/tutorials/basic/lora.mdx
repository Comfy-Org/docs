**LoRA(저랭크 적응)**는 Stable Diffusion와 같은 대규모 생성 모델을 미세 조정하는 효율적인 기법입니다.  
이는 사전 훈련된 모델에 학습 가능한 저랭크 행렬을 도입하여, 전체 모델을 다시 훈련하는 대신 일부 파라미터만 조정함으로써 더 낮은 계산 비용으로 특정 작업에 대한 최적화를 달성합니다.  
SD1.5와 같은 기본 모델과 비교하면 LoRA 모델은 크기가 작고 훈련하기도 쉽습니다.

![LoRA 모델 vs 기본 모델 비교](/images/tutorial/basic/lora/compare.png)

위 그림은 [dreamshaper_8](https://civitai.com/models/4384?modelVersionId=128713)을 직접 사용한 경우와 [blindbox_V1Mix](https://civitai.com/models/25995/blindbox) LoRA 모델을 사용한 경우의 동일한 파라미터로 생성한 결과를 비교한 것입니다.  
보시다시피, LoRA 모델을 사용하면 기본 모델을 조정하지 않고도 다양한 스타일의 이미지를 생성할 수 있습니다.

다음은 LoRA 모델을 사용하는 방법을 보여드리겠습니다. 모든 LoRA 변형—Lycoris, loha, lokr, locon 등—은 동일한 방식으로 사용됩니다.

이 예제에서는 [ComfyUI](https://github.com/Comfy-Org/ComfyUI)에서 LoRA 모델을 로드하고 사용하는 방법을 배우며, 다음 주제를 다룹니다:

1. LoRA 모델 설치하기
2. LoRA 모델을 사용해 이미지 생성하기
3. `Load LoRA` 노드 간단 소개

## 필수 모델 설치

[dreamshaper_8.safetensors](https://civitai.com/api/download/models/128713?type=Model&format=SafeTensor&size=pruned&fp=fp16) 파일을 다운로드해 `ComfyUI/models/checkpoints` 폴더에 넣으세요.

[blindbox_V1Mix.safetensors](https://civitai.com/api/download/models/32988?type=Model&format=SafeTensor&size=full&fp=fp16) 파일을 다운로드해 `ComfyUI/models/loras` 폴더에 넣으세요.

## LoRA 워크플로우 파일

아래 이미지를 다운로드해 **ComfyUI로 드래그**하면 워크플로우가 로드됩니다.  
![ComfyUI 워크플로우 - LoRA](/images/tutorial/basic/lora/lora.png)

## 워크플로우를 단계별로 완료하세요

아래 다이어그램의 단계를 따라 워크플로우가 올바르게 실행되도록 하세요.

![ComfyUI 워크플로우 - LoRA 흐름 다이어그램](https://raw.githubusercontent.com/Comfy-Org/example_workflows/refs/heads/main/lora.png)

1. `Load Checkpoint`가 `dreamshaper_8.safetensors`를 로드하도록 확인하세요.
2. `Load LoRA`가 `blindbox_V1Mix.safetensors`를 로드하도록 확인하세요.
3. `Queue` 버튼을 클릭하거나 단축키 `Ctrl(cmd) + Enter`를 사용해 이미지를 생성하세요.

## Load LoRA 노드 소개

![Load LoRA 노드](/images/comfy_core/loaders/load_lora.jpg)

`ComfyUI\models\loras` 폴더에 있는 모델들은 ComfyUI에서 자동으로 감지되며, 이 노드를 사용해 로드할 수 있습니다.

### 입력 유형

| 파라미터 이름   | 기능                                                                                               |
|------------------|----------------------------------------------------------------------------------------------------|
| `model`          | 기본 모델에 연결하세요                                                                            |
| `clip`           | CLIP 모델에 연결하세요                                                                            |
| `lora_name`      | 로드하고 사용할 LoRA 모델을 선택하세요                                                             |
| `strength_model` | LoRA가 모델 가중치에 얼마나 강하게 영향을 미치는지 결정하며, 값이 높을수록 LoRA 스타일이 더 강해집니다 |
| `strength_clip`  | LoRA가 CLIP 텍스트 임베딩에 얼마나 강하게 영향을 미치는지 결정합니다                                |

### 출력 유형

| 파라미터 이름 | 기능                                             |
|----------------|--------------------------------------------------|
| `model`        | LoRA 조정이 적용된 모델을 출력합니다              |
| `clip`         | LoRA 조정이 적용된 CLIP 모델을 출력합니다          |

이 노드는 체인 연결을 지원하므로 여러 개의 `Load LoRA` 노드를 연속적으로 연결해 여러 LoRA 모델을 적용할 수 있습니다. 자세한 내용은 [ComfyUI 다중 LoRA 예제](/ko/tutorials/basic/multiple-loras)를 참고하세요.

![LoRA 노드 체인 연결](/images/tutorial/basic/lora/chain_link.png)

## 직접 해보세요

1. 프롬프트를 수정하거나 `Load LoRA` 노드의 `strength_model` 등 다른 파라미터를 조정해 생성된 이미지의 변화를 관찰하며 `Load LoRA` 노드에 익숙해져 보세요.
2. [CivitAI](https://civitai.com/models)를 방문해 다른 종류의 LoRA 모델을 다운로드하고 사용해 보세요.
