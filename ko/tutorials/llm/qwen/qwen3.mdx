**Qwen 3.0**(또는 Qwen3라고도 함)은 Alibaba Cloud가 개발한 강력한 오픈소스 LLM으로, 텍스트 생성, 추론 및 구조적 분석을 위해 설계되었습니다. 이 모델은 내장된 `TextGenerate` 노드를 통해 ComfyUI에서 기본적으로 지원됩니다.

**모델 주요 특징**:

- **강력한 추론** — Qwen3-4B-Thinking은 단계별 사고 체인 추론을 지원합니다.
- **텍스트 생성 중심** — 구조화된 텍스트 출력, 분석 및 코딩 작업에 최적화되어 있습니다.
- **ComfyUI 기본 지원** — 내장된 `TextGenerate` 노드와 함께 작동하며, 맞춤형 노드가 필요하지 않습니다.
- **경량성** — Qwen3.5와 동일한 텍스트 인코더 형식을 공유하며, 2B, 4B 및 9B 변형으로 제공되어 다양한 하드웨어에 맞게 사용할 수 있습니다.


## 활용 사례

Qwen 3.0은 ComfyUI 워크플로우 내에서 구조화된 텍스트 생성과 지능형 추론이 필요한 작업에 적합합니다:

- **프롬프트 최적화** — 기존 프롬프트를 Qwen 3.0에 입력하고, 이미지 생성 결과를 더 좋게 만들기 위해 다시 작성하거나 확장하거나 정교하게 다듬도록 요청하세요. 예를 들어, 짧은 키워드 프롬프트를 스타일, 조명, 구성 등의 세부 정보를 포함한 상세하고 구조화된 설명으로 변환할 수 있습니다.
- **워크플로우 로직 추론** — 모델을 사용해 워크플로우 파라미터를 분석하고 처리 단계를 제안하거나 다른 노드로 전달되는 구조화된 지침을 생성할 수 있습니다.
- **텍스트 기반 분석** — 텍스트 입력에서 정보를 추출하고, 콘텐츠를 분류하거나 구조화된 보고서를 생성할 수 있습니다.
- **사고 체인 추론** — 복잡한 다단계 작업에서 중간 과정의 추론을 거친 후 최종 결과를 생성하는 데 유리한 사고 모드를 활성화하세요.

![Qwen 3.0 텍스트 생성 워크플로우](https://raw.githubusercontent.com/Comfy-Org/workflow_templates/main/templates/llm_qwen3_text_gen-1.webp)

이 워크플로우는 Qwen 3.0의 핵심 **텍스트 생성** 기능을 보여줍니다. 텍스트 프롬프트를 받아 모델의 내장된 추론 기능을 사용해 상세하고 구조화된 응답을 생성합니다.

**입력**:
- **텍스트 프롬프트** — 질문, 지시사항 또는 작업 설명

**주요 컨트롤**:
- **최대 길이** — 생성할 토큰의 최대 수 (기본값 256)
- **샘플링 모드** — 샘플링 켜기/끄기 및 온도, top-k, top-p, 반복 페널티, 시드 조정
- **사고 모드** — 최종 답변 전에 단계별 추론을 활성화
- **기본 템플릿 사용** — 모델용 내장 시스템 프롬프트 적용

**출력**:
- **생성된 텍스트** — 모델의 응답을 일반 텍스트 문자열로 반환

## 모델 다운로드

Qwen 3.0 모델은 ComfyUI에서 텍스트 인코더로 로드됩니다. 모델 파일은 Qwen3.5와 공유되며, 하드웨어에 가장 적합한 변형을 다운로드하세요:

<Card title="Qwen3.5 2B (bf16)" icon="download" href="https://huggingface.co/Comfy-Org/Qwen3.5/resolve/main/text_encoders/qwen3.5_2b_bf16.safetensors">
    경량형, 약 4.5GB. 낮은 VRAM 환경과 빠른 다운로드에 최적입니다.
  </Card>

<Card title="Qwen3.5 4B (bf16)" icon="download" href="https://huggingface.co/Comfy-Org/Qwen3.5/resolve/main/text_encoders/qwen3.5_4b_bf16.safetensors">
    균형 잡힌 크기와 품질. 대부분의 소비자용 GPU에 권장됩니다.
  </Card>

<Card title="Qwen3.5 9B (bf16)" icon="download" href="https://huggingface.co/Comfy-Org/Qwen3.5/resolve/main/text_encoders/qwen3.5_9b_bf16.safetensors">
    가장 큰 변형, 약 19GB. 더 높은 품질의 출력을 제공하며, 더 많은 VRAM이 필요합니다.
  </Card>

다운로드한 `.safetensors` 파일을 다음 위치에 저장하세요:

```

├── 📂 models/
│   └── 📂 text_encoders/
│       └── qwen3.5_4b_bf16.safetensors   # 또는 2B / 9B 변형
