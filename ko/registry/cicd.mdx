## 소개

맞춤형 노드를 변경할 때 Comfy나 다른 맞춤형 노드에서 문제가 발생하는 경우가 드물지 않습니다. 모든 운영체제와 다양한 PyTorch 구성에서 테스트를 수행하는 것은 종종 현실적이지 않습니다.

### GitHub Actions를 사용한 Comfy 워크플로우 실행

[Comfy-Action](https://github.com/Comfy-Org/comfy-action)을 사용하면 GitHub Actions에서 Comfy workflow.json 파일을 실행할 수 있습니다. 모델 다운로드, 맞춤형 노드 지원 및 Linux/Mac/Windows에서 실행됩니다.

### 결과

출력 파일은 [CI/CD 대시보드](https://ci.comfy.org/)에 업로드되며, 새로운 변경사항을 커밋하거나 맞춤형 노드의 새 버전을 출시하기 전 마지막 단계에서 확인할 수 있습니다.
