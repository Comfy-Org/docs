[comfy-cli](https://github.com/Comfy-Org/comfy-cli)는 Comfy의 명령줄 도구입니다. [MCP 서버](/ko/agent-tools)와 함께 에이전트와 개발자에게 UI 외부에서 Comfy를 조작하는 또 다른 방법을 제공합니다.

**터미널 또는 스크립트 접근**이 필요할 때 사용하세요: 배치 작업, CI 파이프라인, 빠른 일회성 생성, 셸 자동화에 적합합니다. `comfy generate` 하위 명령(베타)은 워크플로에 연결하는 것과 동일한 호스팅된 [파트너 노드](/ko/tutorials/partner-nodes/overview)를 단일 CLI 호출로 실행합니다.

<Info>
  **`comfy generate`는 베타 버전입니다.** 플래그 이름, 모델 별칭, 출력 형식이 변경될 수 있습니다. 기본 파트너 엔드포인트는 안정적입니다. 피드백은 [comfy-cli GitHub 저장소](https://github.com/Comfy-Org/comfy-cli/issues)에 제출해 주세요.
</Info>

## MCP와의 연동

| | Comfy Cloud MCP | Comfy CLI (`comfy generate`) |
|---|---|---|
| **인터페이스** | MCP 클라이언트(Claude Code, Claude Desktop 등)와 채팅 | 터미널 명령 및 스크립트 |
| **최적 사용** | 대화형 에이전트 워크플로, 템플릿 검색, 전체 Cloud 워크플로 | 자동화, 배치 실행, CI, 셸 파이프라인 |
| **인증** | OAuth(클라이언트) 또는 API 키(헤드리스) | [Comfy API 키](/ko/development/api-development/getting-an-api-key) |
| **출력** | 채팅 세션에 반환 | 디스크에 저장(`--download`) 또는 JSON(`--json`) |

MCP 설정, 설치, 도구 목록 및 인증 방법은 [Comfy Cloud MCP](/ko/agent-tools/cloud)를 참조하세요.

## `comfy generate` 필수 조건

1. [Comfy API 키 생성](/ko/development/api-development/getting-an-api-key)
2. 계정에 [크레딧 추가](/ko/interface/credits)
3. 선택 사항: [파트너 노드 및 호출별 가격 확인](/ko/tutorials/partner-nodes/pricing)

키를 한 번 설정하세요:

```bash
export COMFY_API_KEY=comfyui-...   # 또는 각 호출에 --api-key 전달
```

## 첫 번째 생성

```bash
comfy generate flux-pro \
    --prompt "a cat on the moon, cinematic lighting" \
    --width 1024 --height 1024 \
    --download cat.png
```

CLI는 로컬 파일 입력을 업로드하고, 작업을 제출하며, 준비될 때까지 폴링한 후 결과를 `cat.png`로 저장합니다.

## 사용 가능한 모델

`comfy generate`는 이미지 및 비디오 파트너 모델을 지원합니다. CLI 버전에 설치된 모델을 확인하려면:

```bash
comfy generate list                            # 모든 모델
comfy generate list --category text-to-video   # 카테고리별 필터
comfy generate list --partner kling            # 파트너별 필터
comfy generate schema flux-kontext             # 특정 모델의 매개변수
```

일반적인 별칭은 다음과 같습니다:

| 카테고리 | 예시 |
|---|---|
| **이미지** | `flux-pro`, `flux-kontext`, `nano-banana` (Gemini Flash Image), `grok`, `grok-edit`, `ideogram-edit`, `dalle`, `recraft`, `stability` |
| **비디오** | `seedance`, `kling`, `luma`, `runway`, `pika`, `vidu`, `hailuo`, `moonvalley`, `grok-video` |

전체 목록은 환경에서 `comfy generate list`를 실행하세요. 베타 기간 동안 별칭과 플래그가 변경될 수 있습니다.

## 사용 예시

### 텍스트 기반 이미지 생성

```bash
comfy generate nano-banana \
    --prompt "a watercolor of a sleeping fox" \
    --download fox.png
```

### 이미지 편집 (참조 이미지)

로컬 경로를 직접 전달하세요. CLI는 Comfy 저장소를 통해 업로드하거나(파트너에 따라 base64 인라인 인코딩):

```bash
comfy generate nano-banana \
    --prompt "add a top hat" \
    --image ./cat.png \
    --download edited.png

comfy generate flux-kontext \
    --prompt "add a top hat and a monocle" \
    --input_image ./photo.jpg \
    --download out.png
```

한 번 업로드하고 서명된 URL을 재사용:

```bash
comfy generate upload ./photo.jpg
# --input_image에 사용할 서명된 URL 출력
```

<Note>
  업로드된 참조 에셋은 **24시간** 후 자동 삭제됩니다. 장기 실행 파이프라인의 경우 각 작업 전에 다시 업로드하세요. [CLI 참조](/ko/comfy-cli/reference#upload)를 참조하세요.
</Note>

### 텍스트 기반 비디오 생성

비디오 작업은 비동기입니다. CLI는 기본적으로 준비될 때까지 차단 및 폴링합니다:

```bash
comfy generate seedance \
    --prompt "a hummingbird hovering over a flower" \
    --resolution 1080p --duration 5 \
    --download hummingbird.mp4

comfy generate kling \
    --prompt "a paper boat drifting on a river at dusk" \
    --duration 5 \
    --download boat.mp4
```

`--async`를 전달하면 작업 ID와 함께 즉시 반환되고, 나중에 재개:

```bash
comfy generate luma --prompt "neon koi swimming through clouds" --aspect_ratio 16:9 --async
comfy generate resume luma <job_id> --download out.mp4
```

### JSON으로 스크립팅

파이프라인의 경우 `--json`은 원시 API 응답을 출력합니다:

```bash
comfy generate dalle --prompt "a watercolor whale" --json | jq '.data[0].url'
```
