ComfyUI는 **MCP 서버**와 **Comfy CLI**를 제공하여 AI 에이전트와 개발자가 항상 ComfyUI 캔버스를 열지 않고도 이미지, 비디오, 오디오 및 3D 콘텐츠를 생성할 수 있도록 합니다.

<Note>
  **Comfy Cloud MCP는 퍼블릭 베타입니다.** 설정, 피드백 채널 및 관련 도구(Comfy Skills, Comfy CLI)에 대한 내용은 [Comfy Cloud MCP](/ko/agent-tools/cloud) 페이지에서 확인하세요.
</Note>

## 어떤 것을 사용해야 할까요?

| | Comfy Cloud MCP | Comfy Partner MCP | Comfy CLI |
|---|---|---|---|
| **유형** | 원격(호스팅) | 로컬 MCP 서버 | 로컬 CLI |
| **필요사항** | Comfy Cloud 구독 | [Comfy API 키](/ko/development/api-development/getting-an-api-key) | [Comfy API 키](/ko/development/api-development/getting-an-api-key) |
| **모델** | Comfy Cloud 모델(사전 설치됨) | 30개 이상의 파트너 제공업체 (BFL, Ideogram, Kling, Runway, Veo, ElevenLabs 등) | `comfy generate`를 통한 파트너 노드 |
| **워크플로** | 전체 ComfyUI 워크플로 실행 | 파트너 API 생성 (맞춤형 워크플로 없음) | 일회성 파트너 API 호출 (워크플로 JSON 불필요) |
| **GPU** | 클라우드 GPU (로컬 GPU 필요 없음) | GPU 불필요 (API 기반) | GPU 불필요 (API 기반) |
| **최적 사용처** | 채팅 기반 에이전트, 템플릿 검색 | 다양한 제공업체를 위한 통합 MCP 도구 | 스크립트, CI, 배치 작업 |

## 로컬 ComfyUI는 어떻게 하나요?

많은 분들이 Comfy Cloud MCP 대신 (또는 함께) **로컬** ComfyUI 설치를 에이전트로 구동하는 방법에 대해 질문하십니다.

**왜 Comfy Cloud MCP가 먼저 출시되었나요?** `cloud.comfy.org`의 호스팅 서버는 Comfy 팀만이 구축하고 운영할 수 있습니다. 따라서 당사의 자체 MCP 릴리스는 클라우드 기반입니다.

**로컬 에이전트는 새로운 것이 아닙니다.** ComfyUI가 오픈소스로 공개된 이후로 자신의 머신에서 ComfyUI에 에이전트를 연결하는 것이 가능했습니다. 실행 중인 로컬 서버는 MCP 브리지 및 기타 도구가 호출할 수 있는 워크플로와 API를 노출합니다.

**더 나은 로컬 에이전트 사용자 경험은 로드맵에 있습니다.** 자체 로컬 에이전트 경험을 개선하는 것은 당사의 비전의 일부입니다. Cloud MCP는 먼저 출시할 수 있었던 부분이며, 로컬 개선 사항은 추후에 추가될 예정입니다.

**지금 시도해볼 수 있는 커뮤니티 MCP 서버.** 여러 커뮤니티 프로젝트가 이미 MCP 클라이언트를 로컬 ComfyUI 인스턴스에 연결하고 있습니다:

| 프로젝트 | 저장소 |
| --- | --- |
| comfyui-mcp | [artokun/comfyui-mcp](https://github.com/artokun/comfyui-mcp) |
| comfyui-mcp | [shawnrushefsky/comfyui-mcp](https://github.com/shawnrushefsky/comfyui-mcp) |
| comfyui-mcp-server | [joenorton/comfyui-mcp-server](https://github.com/joenorton/comfyui-mcp-server) |
| comfy-mcp-server | [lalanikarim/comfy-mcp-server](https://github.com/lalanikarim/comfy-mcp-server) |

이들은 **커뮤니티 유지보수** 프로젝트입니다. 편의를 위해 링크를 제공합니다; 지원, 기능 및 호환성은 프로젝트마다 다릅니다.

**채팅 대신 터미널 워크플로를 원하시나요?** 스크립트 및 자동화를 위해 [Comfy CLI](/ko/agent-tools/comfy-cli)(`comfy generate`, 베타)를 참조하세요. 터미널에서 로컬 ComfyUI를 설치하고 시작하려면 [Comfy CLI 시작하기](/ko/comfy-cli/getting-started)부터 시작하세요.

## MCP란 무엇인가요?

**모델 컨텍스트 프로토콜(MCP)**은 AI 비서가 표준화된 인터페이스를 통해 외부 도구 및 서비스와 상호작용할 수 있도록 하는 오픈 표준입니다. AI 에이전트가 모든 서비스의 맞춤형 API 형식을 알아야 하는 대신, MCP는 도구를 공통적으로 노출하는 방법을 제공합니다.

MCP 서버를 Claude Desktop, Claude Code, Cursor 또는 Amp와 연결하면 AI 비서는 다음과 같은 작업을 수행할 수 있습니다:

- 텍스트 설명으로 **이미지 생성**
- 텍스트 또는 이미지로 **비디오 제작**
- 텍스트 또는 이미지로 **3D 모델 생성**
- AI 모델을 사용해 **오디오 및 음악 생성**
- 작업에 적합한 도구를 찾기 위해 **모델 및 템플릿 검색**

...이 모든 것을 자연어 대화를 통해 수행하며, API 호출을 직접 작성할 필요가 없습니다.
