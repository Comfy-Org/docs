## Quick Start

1. Download a checkpoint file. Maybe [Stable Diffusion v1.5](https://huggingface.co/Comfy-Org/stable-diffusion-v1-5-archive/resolve/main/v1-5-pruned-emaonly-fp16.safetensors?download=true). Place the file under `ComfyUI/models/checkpoints`.

1. Refresh the ComfyUI.

1. Click `Load Default` button to use the default workflow.

1. In the `Load Checkpoint` node, select the checkpoint file you just downloaded.

1. Click `Queue Prompt` and watch your image generated. Play around with the prompts to generate different images.
