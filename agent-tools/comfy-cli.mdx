[comfy-cli](https://github.com/Comfy-Org/comfy-cli) is Comfy's command-line tool. Alongside [MCP servers](/agent-tools), it gives agents and developers another way to drive Comfy from outside the UI.

Use it when you want **terminal or script access**: batch jobs, CI pipelines, quick one-off generations, and shell automation. The `comfy generate` subcommand (beta) calls the same hosted [partner nodes](/tutorials/partner-nodes/overview) you would wire into a workflow, but as one-shot CLI calls.

<Info>
  **`comfy generate` is in beta.** Flag names, model aliases, and output formats may change. The underlying partner endpoints are stable. File feedback on the [comfy-cli GitHub repo](https://github.com/Comfy-Org/comfy-cli/issues).
</Info>

## How it fits with MCP

| | Comfy Cloud MCP | Comfy CLI (`comfy generate`) |
|---|---|---|
| **Interface** | Chat with an MCP client (Claude Code, Claude Desktop, …) | Terminal commands and scripts |
| **Best for** | Interactive agent workflows, template search, full Cloud workflows | Automation, batch runs, CI, shell pipelines |
| **Auth** | OAuth (clients) or API key (headless) | [Comfy API key](/development/api-development/getting-an-api-key) |
| **Output** | Returned in the chat session | Saved to disk (`--download`) or JSON (`--json`) |

For MCP setup, installation, tool lists, and auth methods, see [Comfy Cloud MCP](/agent-tools/cloud).

```bash
comfy setup
```

Pass `-y` for non-interactive installs in CI or scripts.

## Prerequisites for `comfy generate`

1. [Create a Comfy API key](/development/api-development/getting-an-api-key)
2. [Add credits](/interface/credits) to your account
3. Optional: [Browse partner nodes and per-call pricing](/tutorials/partner-nodes/pricing)

Set the key once:

```bash
export COMFY_API_KEY=comfyui-...   # or pass --api-key on each call
```

## Your first generation

```bash
comfy generate flux-pro \
    --prompt "a cat on the moon, cinematic lighting" \
    --width 1024 --height 1024 \
    --download cat.png
```

The CLI uploads local file inputs, submits the job, polls until ready, and saves the result to `cat.png`.

## Available models

`comfy generate` covers image and video partner models. Discover what's installed in your CLI version:

```bash
comfy generate list                            # all models
comfy generate list --category text-to-video   # filter by category
comfy generate list --partner kling            # filter by partner
comfy generate schema flux-kontext             # params for one model
```

Common aliases include:

| Category | Examples |
|---|---|
| **Image** | `flux-pro`, `flux-kontext`, `nano-banana` (Gemini Flash Image), `grok`, `grok-edit`, `ideogram-edit`, `dalle`, `recraft`, `stability` |
| **Video** | `seedance`, `kling`, `luma`, `runway`, `pika`, `vidu`, `hailuo`, `moonvalley`, `grok-video` |

Run `comfy generate list` for the full set in your environment. Aliases and flags may change during beta.

## Usage examples

### Text-to-image

```bash
comfy generate nano-banana \
    --prompt "a watercolor of a sleeping fox" \
    --download fox.png
```

### Image edit (reference image)

Pass local paths directly. The CLI uploads through Comfy storage (or base64-encodes inline, depending on the partner):

```bash
comfy generate nano-banana \
    --prompt "add a top hat" \
    --image ./cat.png \
    --download edited.png

comfy generate flux-kontext \
    --prompt "add a top hat and a monocle" \
    --input_image ./photo.jpg \
    --download out.png
```

Upload once and reuse a signed URL:

```bash
comfy generate upload ./photo.jpg
# prints a signed URL for --input_image
```

<Note>
  Uploaded reference assets are auto-deleted after **24 hours**. For long-running pipelines, re-upload before each job. See [CLI reference](/comfy-cli/reference#upload).
</Note>

### Text-to-video

Video jobs are async. The CLI blocks and polls until ready by default:

```bash
comfy generate seedance \
    --prompt "a hummingbird hovering over a flower" \
    --resolution 1080p --duration 5 \
    --download hummingbird.mp4

comfy generate kling \
    --prompt "a paper boat drifting on a river at dusk" \
    --duration 5 \
    --download boat.mp4
```

Pass `--async` to return immediately with a job id, then resume:

```bash
comfy generate luma --prompt "neon koi swimming through clouds" --aspect_ratio 16:9 --async
comfy generate resume luma <job_id> --download out.mp4
```

### Scripting with JSON

For pipelines, `--json` emits the raw API response:

```bash
comfy generate dalle --prompt "a watercolor whale" --json | jq '.data[0].url'
```
