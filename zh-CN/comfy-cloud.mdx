Comfy Cloud 是 ComfyUI 的官方云托管版本,完全在浏览器中运行。它提供 ComfyUI 的全部功能,无需本地安装、GPU 硬件或依赖管理。

## 概述

Comfy Cloud 通过提供以下功能消除了使用 ComfyUI 的技术障碍:

- **零设置** - 在浏览器中立即开始创作
- **强大的 GPU** - 访问高性能服务器 GPU
- **预配置模型** - 流行模型已安装并可立即使用
- **自定义节点库** - 不断增长的社区扩展集合
- **始终保持最新** - 自动与最新的 ComfyUI 版本保持同步
- **跨设备兼容** - 在任何带有网络浏览器的设备上工作

## 主要功能

### 基于浏览器的界面

直接在浏览器中访问完整的 ComfyUI 节点式界面。无需下载、无需安装、无需配置。

### 专业性能

Comfy Cloud 专为专业用途设计:

- 在强大的服务器 GPU 上快速执行工作流
- 稳定可靠的基础设施
- 与开源版本功能对等

### 模型可用性

所有流行的生成式 AI 模型都已预安装并正确配置,无需:

- 下载大型模型文件
- 管理存储空间
- 配置模型路径
- 更新依赖项

### 自定义节点支持

访问来自 ComfyUI 社区不断增长的自定义节点和扩展库,无需手动安装。

## 开始使用

### 加入测试版

Comfy Cloud 目前处于私有测试阶段。您可以在 [cloud.comfy.org](https://cloud.comfy.org) 加入等待列表。

测试版参与者将获得:

- 测试期间的免费访问
- 提供反馈以塑造产品的机会
- 提前访问新功能

### 未来定价

测试期结束后,Comfy Cloud 将使用基于 GPU 使用量的订阅模式。您只需为工作流在 GPU 上实际运行的时间付费,而不是为构建工作流时 GPU 空闲的时间付费。

## Comfy Cloud 与本地安装对比

| 功能 | Comfy Cloud | 本地安装 |
|---------|-------------|----------------------|
| 需要设置 | 无 | 手动安装 |
| 需要 GPU | 否(使用云 GPU) | 是(为了良好性能) |
| 模型管理 | 预配置 | 手动下载和设置 |
| 更新 | 自动 | 手动更新 |
| 自定义节点 | 预安装库 | 手动安装 |
| 成本 | 订阅(GPU 使用量) | 免费(硬件成本) |
| 需要互联网 | 是 | 否(设置后) |
| 隐私 | 云处理 | 完全本地 |

## 对开源的承诺

ComfyUI 将始终保持免费和开源。Comfy Cloud 是支持开源项目长期发展的收入模式。

主要承诺:

- 开源版本将始终可以免费在本地运行
- 云版本和本地版本功能对等
- 社区自定义节点开发者将通过收入分享计划受益

## 支持和反馈
