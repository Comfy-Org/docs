## Comfy Cloud

<Note>
Comfy Cloud 目前不支持安装自定义节点。如果您需要特定的自定义节点或功能，请在[支持](/zh-CN/support/contact-support)中留言或提交反馈，让我们了解您想使用的功能。
</Note>

## 访问管理器

点击主菜单（顶部栏）上的 **Manager** 按钮。

![Manager 按钮](https://raw.githubusercontent.com/ltdrdata/ComfyUI-extension-tutorials/Main/ComfyUI-Manager/images/topbar.jpg)

## 安装自定义节点

1. 从 Manager 菜单点击 **Install Custom Nodes**
2. 浏览或搜索您要安装的节点
3. 点击 **Install** 或 **Try Install**

![自定义节点对话框](https://raw.githubusercontent.com/ltdrdata/ComfyUI-extension-tutorials/Main/ComfyUI-Manager/images/custom-nodes.jpg)

### 安装状态指示器

- **Installed**：此项目已安装
- **Install**：点击安装该项目
- **Try Install**：无法确认安装信息；点击尝试安装

### 数据库模式

节点列表有三种数据库模式：

| 模式 | 描述 |
|------|------|
| `DB: Channel (1day cache)` | 使用缓存的频道信息（1天有效期）快速显示。这是**默认**模式。 |
| `DB: Local` | 使用本地存储在 ComfyUI-Manager 中的信息。仅在更新 ComfyUI-Manager 时更新。 |
| `DB: Channel (remote)` | 从远程频道获取信息，始终显示最新列表。 |

<Note>
如果顶部出现红色背景的 `Channel` 指示器，表示您没有使用默认频道。在此状态下，许多自定义节点可能不会显示。
</Note>

## 安装模型

1. 从 Manager 菜单点击 **Install Models**
2. 浏览或搜索您想要的模型
3. 点击 **Install**

![模型对话框](https://raw.githubusercontent.com/ltdrdata/ComfyUI-extension-tutorials/Main/ComfyUI-Manager/images/models.jpg)

## 更新自定义节点

1. 点击 **Fetch Updates** 在本地获取自定义节点的更新数据
2. 在 **Install Custom Nodes** 菜单中点击 **Update** 按钮应用更新

## 安装缺失节点

当您加载包含缺失节点的工作流时：

1. 从 Manager 菜单点击 **Install Missing Custom Nodes**
2. 将显示包含缺失节点的扩展节点列表
3. 安装所需的扩展

![缺失节点](https://raw.githubusercontent.com/ltdrdata/ComfyUI-extension-tutorials/Main/ComfyUI-Manager/images/missing-list.jpg)

## 快照管理器

快照保存您当前的安装状态，允许您恢复到之前的状态。

![快照管理器](https://raw.githubusercontent.com/ltdrdata/ComfyUI-extension-tutorials/Main/ComfyUI-Manager/images/snapshot.jpg)

当您按下 **Save snapshot** 或使用 **Update All** 时，快照会自动保存。快照文件存储在：`<USER_DIRECTORY>/default/ComfyUI-Manager/snapshots`

要恢复：打开快照管理器，选择一个快照，然后按 **Restore**。

<Note>
恢复在下次 ComfyUI 启动时生效。对于不由 Git 管理的自定义节点，快照支持不完整。
</Note>

## 修复节点（重新创建）

右键点击节点并选择 **Fix node (recreate)** 来重新创建节点。小部件值会重置，而同名连接会保持。这对于修正旧工作流中与较新自定义节点版本不兼容的节点错误很有用。
