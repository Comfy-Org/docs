## 什么是 Comfy Cloud？

ComfyUI Cloud 是 ComfyUI 的云端版本，具有与本地版本相同的功能。一切都已预装并可立即使用。

## 云端版本 vs 本地版本

| 功能 | Comfy Cloud | 本地 ComfyUI |
|------|-------------|--------------|
| **设置** | 无需安装 - 立即可用 | 需要 Python 环境设置和配置 |
| **GPU 要求** | 访问强大的服务器 GPU | 需要您自己的 GPU 硬件 |
| **模型和节点** | 预装流行模型和自定义节点 | 需要手动安装和管理 |
| **更新** | 始终保持最新版本 | 需要手动更新和故障排除 |
| **依赖关系** | 无依赖冲突 | 可能存在版本不匹配和包冲突 |
| **维护** | 零维护开销 | 持续的维护和故障排除 |
| **成本** | 基于订阅 | 免费（硬件成本另计） |
| **访问** | 从任何设备在线使用 | 仅限于安装了本地版本的设备 |
| **功能对等** | 与本地版本功能相同 | 与云版本功能相同 |

两个版本将始终具有相同的功能。我们同样致力于开发两者。ComfyUI 将始终免费在本地运行，开源 ComfyUI 和云版本将始终保持功能对等。Cloud 是我们创造收入和维持 ComfyUI 长期发展的方式。

## 账户和登录

要使用 Comfy Cloud，您需要一个 Comfy 账户。访问 [cloud.comfy.org](https://cloud.comfy.org) 并使用您的 Comfy 账户凭据登录。登录后，您可以立即开始使用 ComfyUI。

## 管理您的工作流

### 检查工作流状态

点击左侧边栏的队列图标以打开面板并查看所有工作流的状态。

### 保存您的输出

目前，如果输出超出队列历史长度，Cloud 不支持访问所有输出。（这将在后续更新中得到支持）。请手动保存输出以避免丢失它们。

### 增加队列历史大小

1. 点击徽标打开下拉菜单
2. 点击设置打开设置面板
3. 搜索 `history` 设置
4. 增加 `Queue history size`
