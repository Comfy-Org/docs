## 什么是 Comfy Cloud？

ComfyUI Cloud 是 ComfyUI 的云端版本，具有与本地版本相同的功能。一切都已预装并可立即使用。

## 云端版本 vs 本地版本

| 类别 | Comfy Cloud | 自托管（本地 ComfyUI） |
|------|-------------|----------------------|
| **成本** | 月度订阅 | 免费 |
| **GPU** | 强大的 A100，40GB+ 显存 | 使用您自己的 GPU |
| **技术知识** | 无需技术知识。 | 虽然桌面版和便携版为您提供了简单的入门方式，但您需要解决自定义节点安装和本地安装问题。 |
| **自定义节点** | 使用预装的自定义节点，永远不用担心兼容性问题。 | 安装任何您想要的自定义节点，但您需要自己管理。 |
| **模型** | 使用预装模型。上传您自己的模型（即将推出）。 | 使用任何您想要的模型，但您需要先下载它们。 |
| **显著差异** | 易于团队入门 | 离线工作，无限可定制 |
| **开始使用** | [运行 ComfyUI Cloud](https://comfy.org/cloud) | [本地安装 ComfyUI](/zh-CN/installation/system_requirements) |

两个版本将始终具有相同的功能。我们同样致力于开发两者。ComfyUI 将始终免费在本地运行，开源 ComfyUI 和云版本将始终保持功能对等。Cloud 是我们创造收入和维持 ComfyUI 长期发展的方式。

## 管理您的工作流

### 检查工作流状态

点击左侧边栏的队列图标以打开面板并查看所有工作流的状态。

### 保存您的输出

目前，如果输出超出队列历史长度，Cloud 不支持访问所有输出。（这将在后续更新中得到支持）。请手动保存输出以避免丢失它们。

### 增加队列历史大小

1. 点击徽标打开下拉菜单
2. 点击设置打开设置面板
3. 搜索 `history` 设置
4. 增加 `Queue history size`
