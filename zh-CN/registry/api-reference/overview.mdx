## 概览

自定义节点注册表遵循以下结构：

```mermaid
erDiagram
    PUBLISHER {
        string id PK
    }

    USER {
        string id PK
        string name
        string publisher_id FK
    }

    CUSTOM_NODE {
        string id PK
        string name
        string publisher_id FK
    }

    NODE_VERSION {
        string id PK
        string version
        string node_id FK
    }

    PUBLISHER ||--o{ USER: "has many"
    PUBLISHER ||--o{ CUSTOM_NODE: "has many"
    CUSTOM_NODE ||--o{ NODE_VERSION: "has many"
```
