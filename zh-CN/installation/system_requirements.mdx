在本篇我们将介绍安装 ComfyUI 的系统要求, 由于 ComfyUI 的更新频繁，本篇文档未必能够及时更新，请参考[ComfyUI](https://github.com/comfyanonymous/ComfyUI)中的相关说明。

无论是哪个版本的 ComfyUI，都是运行在一个独立的 Python 环境中。

### 操作系统要求

目前我们支持以下操作系统：
- Windows
- Linux
- macOS（支持 Apple Silicon M1/M2）

请参考[ComfyUI Windows 和 Linux 手动安装章节](https://github.com/comfyanonymous/ComfyUI?tab=readme-ov-file#manual-install-windows-linux)了解详细的安装步骤。

你可以参考下面的章节来了解不同系统和版本 ComfyUI 的安装方式，在不同版本的安装中我们简单对安装的系统要求进行了说明。

<InstallLink/>

### Python 版本
- 推荐 Python 3.12
- 支持 Python 3.13（部分自定义节点可能不兼容）

### 支持的硬件
- NVIDIA 显卡
- AMD 显卡
- Intel 显卡（包括 Arc 系列，支持 IPEX）
- Apple Silicon（M1/M2）
- Ascend NPU
- Cambricon MLU
- CPU（可用 --cpu 参数，速度较慢）

<Note>
目前稳定版本的 PyTorch 2.7 已经支持 Blackwell 架构（CUDA 12.8），ComfyUI 的核心版本和桌面版本已经采用了这个版本。
</Note>

### 依赖

- 需安装 PyTorch（不同硬件需不同版本，详见下方）
- 需安装 ComfyUI 的 requirements.txt 中所有依赖

### 各硬件 PyTorch 安装命令示例

#### NVIDIA 显卡
```bash
pip install torch torchvision torchaudio --extra-index-url https://download.pytorch.org/whl/cu128
```

#### AMD 显卡（Linux）
```bash
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/rocm6.2.4
```

#### Intel 显卡
```bash
pip install --pre torch torchvision torchaudio --index-url https://download.pytorch.org/whl/nightly/xpu
```

#### Apple Silicon
- 需安装 PyTorch nightly，参考 [Apple 官方 Metal PyTorch 指南](https://developer.apple.com/metal/pytorch/)

#### Ascend NPU
- 需安装 Ascend Basekit、CANN、torch-npu，详见 [官方安装文档](https://ascend.github.io/docs/sources/ascend/quick_install.html)

#### Cambricon MLU
- 需安装 CNToolkit、torch_mlu，详见 [官方安装文档](https://www.cambricon.com/docs/sdk_1.15.0/cntoolkit_3.7.2/cntoolkit_install_3.7.2/index.html)

---

如需详细安装步骤，请参考官方文档和各硬件平台的安装说明。
