Vidu Q3 文生视频节点可根据文本描述生成视频。它使用 Vidu Q3 Pro 模型，基于您的提示词生成视频内容，并允许您控制视频的长度、分辨率和宽高比。

## 输入参数

| 参数 | 数据类型 | 必填 | 取值范围 | 描述 |
|-----------|-----------|----------|-------|-------------|
| `model` | COMBO | 是 | `"viduq3-pro"` | 用于视频生成的模型。选择此选项将显示宽高比、分辨率、时长和音频等额外配置参数。 |
| `model.aspect_ratio` | COMBO | 是* | `"16:9"`<br />`"9:16"`<br />`"3:4"`<br />`"4:3"`<br />`"1:1"` | 输出视频的宽高比。此参数在选择 `model` 后显示。 |
| `model.resolution` | COMBO | 是* | `"720p"`<br />`"1080p"` | 输出视频的分辨率。此参数在选择 `model` 后显示。 |
| `model.duration` | INT | 是* | 1 到 16 | 输出视频的时长（单位：秒，默认值：5）。此参数在选择 `model` 后显示。 |
| `model.audio` | BOOLEAN | 是* | True/False | 启用后，输出带声音（包括对话和音效）的视频（默认值：False）。此参数在选择 `model` 后显示。 |
| `prompt` | STRING | 是 | N/A | 用于视频生成的文本描述，最大长度为 2000 个字符。 |
| `seed` | INT | 否 | 0 到 2147483647 | 用于控制生成随机性的种子值（默认值：1）。 |

*注意：一旦选择了 `model`，参数 `aspect_ratio`、`resolution`、`duration` 和 `audio` 即为必填项，因为它们是模型配置的一部分。

## 输出结果

| 输出名称 | 数据类型 | 描述 |
