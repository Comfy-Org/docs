音频均衡器（3段）节点允许您调整音频波形的低频、中频和高频。它应用三个独立的滤波器：用于低频的低架滤波器、用于中频的峰值滤波器以及用于高频的高架滤波器。每个频段都可以通过增益、频率和带宽设置进行独立控制。

## 输入参数

| 参数 | 数据类型 | 必填 | 范围 | 描述 |
|-----------|-----------|----------|-------|-------------|
| `audio` | AUDIO | 是 | - | 包含波形和采样率的输入音频数据。 |
| `low_gain_dB` | FLOAT | 否 | -24.0 到 24.0 | 低频（低音）增益。正值提升，负值削减。（默认值：0.0） |
| `low_freq` | INT | 否 | 20 到 500 | 低架滤波器的截止频率，单位为赫兹（Hz）。（默认值：100） |
| `mid_gain_dB` | FLOAT | 否 | -24.0 到 24.0 | 中频增益。正值提升，负值削减。（默认值：0.0） |
| `mid_freq` | INT | 否 | 200 到 4000 | 中频峰值滤波器的中心频率，单位为赫兹（Hz）。（默认值：1000） |
| `mid_q` | FLOAT | 否 | 0.1 到 10.0 | 中频峰值滤波器的 Q 因子（带宽）。值越低频带越宽，值越高频带越窄。（默认值：0.707） |
| `high_gain_dB` | FLOAT | 否 | -24.0 到 24.0 | 高频（高音）增益。正值提升，负值削减。（默认值：0.0） |
| `high_freq` | INT | 否 | 1000 到 15000 | 高架滤波器的截止频率，单位为赫兹（Hz）。（默认值：5000） |

**注意：** 仅当 `low_gain_dB`、`mid_gain_dB` 和 `high_gain_dB` 参数的值不为零时，才会应用它们。如果增益设置为 0.0，则会跳过相应的滤波器阶段。

## 输出参数

| 输出名称 | 数据类型 | 描述 |
|-------------|-----------|-------------|
| `audio` | AUDIO | 应用了均衡处理后的音频数据，包含修改后的波形和原始采样率。 |
