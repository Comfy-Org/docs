| `vae` | VAE | 是 | - | 用于解码潜在样本的 VAE 模型 |
| `块数` | INT | 是 | 1000-500000 | 为内存管理将处理过程分割的块数（默认：8000） |
| `八叉树分辨率` | INT | 是 | 16-512 | 用于 3D 体素生成的八叉树结构分辨率（默认：256） |
