Recraft Controls 节点允许你定义一系列高级控制参数，用于精确指导Recraft的图像生成过程。这个节点将多种控制输入整合为一个统一的控制对象。

## 节点功能

此节点收集和组织各种控制参数（如颜色和风格指导），创建一个综合控制对象，用于微调Recraft图像生成的行为。这为用户提供了对生成过程的精细控制，远超基本文本提示的能力。

## 参数说明

### 可选参数

| 参数 | 类型 | 说明 |
|-----|-----|------|
| recraft_color_1 | Recraft Color | 主要颜色控制 |
| recraft_color_2 | Recraft Color | 次要颜色控制 |
| recraft_color_3 | Recraft Color | 辅助颜色控制 |
| recraft_color_4 | Recraft Color | 第四颜色控制 |
| recraft_color_5 | Recraft Color | 第五颜色控制 |

### 输出

| 输出 | 类型 | 说明 |
|-----|-----|------|
| recraft_controls | Recraft Controls | 控制配置对象，连接到Recraft生成节点 |

## 使用示例

此节点适用于需要精确控制生成图像视觉特性的场景：

- 品牌内容创建：确保生成图像遵循品牌颜色方案
- 多色协调：控制图像中多种颜色的比例和关系
- 精确设计：为产品设计、广告或UI元素提供精确的颜色指导

## 工作原理

节点处理流程:

1. 收集所有输入的控制参数（如来自Recraft Color RGB节点的颜色值）
2. 将这些参数整合到一个结构化的控制对象中
3. 输出此控制对象，可连接到各种Recraft生成节点

当连接到Recraft生成节点后，这些控制参数会影响AI的生成过程，使AI能够考虑多种因素，而不仅仅是文本提示的语义内容。如果配置了多个颜色输入，AI将尝试在生成的图像中合理地使用这些颜色。
