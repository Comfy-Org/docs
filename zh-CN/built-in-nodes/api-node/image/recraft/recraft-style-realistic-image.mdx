# Recraft Style - Realistic Image 节点

Recraft Style - Realistic Image 节点用于设置Recraft图像生成的真实照片风格，提供多种子风格选项，以控制生成图像的视觉特性。

## 节点功能

此节点创建一个风格配置对象，用于指导Recraft的图像生成过程朝向真实照片的视觉效果。通过选择不同的子风格，可以细化生成图像的表现形式，如摄影风格、照明效果和色调处理。

## 参数说明

### 基本参数

| 参数 | 类型 | 默认值 | 说明 |
|-----|-----|-------|------|
| substyle | 选择项 | None | 真实照片风格的具体子风格 |

### 输出

| 输出 | 类型 | 说明 |
|-----|-----|------|
| recraft_style | Recraft Style | 风格配置对象，连接到Recraft生成节点 |

## 使用示例

此节点通常与以下节点组合使用:

- `Recraft Text to Image` 节点: 控制生成图像的真实照片风格
- `Recraft Image to Image` 节点: 引导图像转换朝向真实照片效果
- `Recraft Replace Background` 节点: 设置生成背景的风格

## 工作原理

节点创建一个包含风格信息的配置对象，指定主风格为"realistic_image"，并可选择性地应用子风格。当连接到Recraft生成节点时，这些风格设置会影响AI模型的行为，使其生成具有真实照片特性的图像，如自然光照、真实质感和适当的透视效果。

选择"None"子风格时，模型将应用标准的真实照片风格，而选择特定子风格可以进一步细化输出图像的视觉效果。
