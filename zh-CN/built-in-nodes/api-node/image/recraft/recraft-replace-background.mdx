# Recraft Replace Background 节点

Recraft Replace Background 节点能够智能识别图像中的主体对象，并根据文本描述生成新的背景场景，创建无缝融合的合成图像。

## 节点功能

此节点将自动分析图像内容，识别前景主体，然后移除原始背景并根据提示词生成新的背景场景。生成的背景与主体对象自然融合，包括光照、阴影和透视效果的匹配。

## 参数说明

### 基本参数

| 参数 | 类型 | 默认值 | 说明 |
|-----|-----|-------|------|
| image | 图像 | - | 包含需要保留主体的输入图像 |
| prompt | 字符串 | "" | 描述要生成的新背景 |
| n | 整数 | 1 | 生成结果的数量(1-6) |
| seed | 整数 | 0 | 随机种子值 |

### 可选参数

| 参数 | 类型 | 说明 |
|-----|-----|------|
| recraft_style | Recraft Style | 设置生成背景的风格 |
| negative_prompt | 字符串 | 指定不希望在背景中出现的元素 |

### 输出

| 输出 | 类型 | 说明 |
|-----|-----|------|
| IMAGE | 图像 | 替换背景后的完整图像 |

## 使用示例

适用场景:

- 产品展示：将产品放置在不同场景和环境中
- 人像摄影：更改人物背景场景
- 创意合成：将对象放入幻想或现实环境中
- 营销材料：创建情境展示图像

## 工作原理

节点处理流程:

1. 接收输入图像并上传到Recraft服务器
2. AI技术识别并分离前景主体
3. 根据提示词和风格参数生成新的背景场景
4. 智能调整光照、色调和透视，确保主体与新背景融合自然
5. 返回合成的完整图像

该节点通过复杂的AI算法确保前景与背景之间的自然过渡，包括边缘处理、色彩调和和光影匹配。即使对于复杂边缘(如头发)和半透明元素也能实现良好的融合效果。
