Recraft Creative Upscale 节点使用先进的AI技术不仅增加图像分辨率，还创造性地增强和丰富图像细节，特别适合照片、人像和艺术作品。

## 节点功能

此节点将输入图像放大的同时智能增强视觉细节，为照片添加自然和富有艺术性的细节，如改善皮肤质感、发丝细节、材质表现和环境质感。与常规放大不同，Creative Upscale不仅是简单的分辨率提升，还会增加合理的视觉信息。

## 参数说明

### 基本参数

| 参数 | 类型 | 默认值 | 说明 |
|-----|-----|-------|------|
| image | 图像 | - | 需要创意放大的输入图像 |

### 输出

| 输出 | 类型 | 说明 |
|-----|-----|------|
| IMAGE | 图像 | 创意放大后的高分辨率图像 |

## 工作原理

节点处理流程:

1. 接收输入图像并上传到Recraft服务器
2. AI模型分析图像内容，识别主体类型和现有细节
3. 应用专门设计的creative放大算法，注重自然细节的生成
4. 智能增强特定区域(如面部特征、材质和纹理)的细节
5. 返回既有更高分辨率又有更丰富细节的图像

此节点特别优化了对面部特征的处理，能够自然地增强眼睛、唇部和肤质细节，同时保持人物的真实感。与"Crisp Upscale"节点相比，Creative Upscale更专注于自然细节的丰富而非锐度提升。
