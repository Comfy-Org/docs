Recraft Style - Logo Raster 节点用于设置Recraft图像生成的标志栅格风格，提供多种专业标志设计的子风格选项，以创建适合品牌和企业使用的图像。

## 节点功能

此节点创建一个风格配置对象，用于指导Recraft的图像生成过程朝向专业标志设计的视觉效果。通过选择不同的子风格，可以定义生成标志的设计风格、复杂度和适用场景。

## 参数说明

### 基本参数

| 参数 | 类型 | 默认值 | 说明 |
|-----|-----|-------|------|
| substyle | 选择项 | - | Logo栅格风格的具体子风格(必选) |

### 输出

| 输出 | 类型 | 说明 |
|-----|-----|------|
| recraft_style | Recraft Style | 风格配置对象，连接到Recraft生成节点 |

## 使用示例

此节点通常与以下节点组合使用:

- `Recraft Text to Image` 节点: 生成品牌标志和图标
- `Recraft Image to Image` 节点: 将概念图转换为专业标志
- `Recraft Controls` 节点: 精确控制标志的颜色和其他视觉元素

## 工作原理

节点创建一个包含风格信息的配置对象，指定主风格为"logo_raster"，并必须选择一个子风格。与其他风格节点不同，Logo Raster风格必须指定子风格，不支持"None"选项，这确保了生成的标志有明确的设计方向。

当连接到Recraft生成节点时，这些风格设置会影响AI模型的行为，使其生成具有专业标志特性的图像，如简洁的形式、品牌识别度和适当的复杂度。不同的子风格可以产生现代、极简、复古或行业特定的标志设计。

class RecraftStyleV3LogoRasterNode(RecraftStyleV3RealisticImageNode):
    """
    Select vector_illustration style and optional substyle.
    """

    @classmethod
    def INPUT_TYPES(s):
        return {
            "required": {
                "substyle": (get_v3_substyles(s.RECRAFT_STYLE, include_none=False),),
            }
        }

    RECRAFT_STYLE = RecraftStyleV3.logo_raster
