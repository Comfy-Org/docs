Luma Reference 节点允许你设置参考图像和权重，用于指导Luma图像生成节点的创作过程，使得生成的图像更接近参考图像的特定特征。

## 节点功能

此节点作为Luma生成节点的辅助工具，允许用户提供参考图像来影响生成结果。它让用户能够设置参考图像的权重，以控制参考图像对最终结果的影响程度。

## 参数说明

### 基本参数

| 参数 | 类型 | 默认值 | 说明 |
|-----|-----|-------|------|
| image | 图像 | - | 作为参考的输入图像 |
| weight | 浮点数 | 1.0 | 控制参考图像的影响强度 (0-1) |

### 输出

| 输出 | 类型 | 说明 |
|-----|-----|------|
| luma_ref | LUMA_REF | 包含图像和权重的参考对象 |

## 使用示例

Luma Reference 特别适合：

- 向生成的图像添加特定风格元素
- 保持生成图像中的特定颜色方案或构图
- 引导生成结果更接近特定的视觉风格
- 融合多个参考图像的特征(通过使用多个参考节点)

## 特点和优势

参考功能的主要特点：

- **精确的风格控制**：让生成图像更接近特定的视觉风格或美学
- **灵活的权重设置**：通过权重参数微调参考图像的影响程度
- **与多种生成节点兼容**：可用于文本到图像和图像到图像节点
- **可组合性**：支持多个参考节点的链接以混合不同参考的特征

## 工作原理

Luma Reference 节点接收图像输入并允许设置权重值。该节点不直接生成或修改图像，而是创建一个包含图像数据和权重信息的参考对象，后续传递给Luma生成节点。

在生成过程中，Luma AI会分析参考图像的特征，并根据设定的权重将这些特征融入到生成结果中。较高的权重值意味着生成的图像将更接近参考图像的特征，而较低的权重值则表示参考图像只会轻微影响最终结果。
