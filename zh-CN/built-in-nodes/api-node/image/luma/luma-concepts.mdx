Luma Concepts 节点允许你将预定义的概念应用到Luma生成过程中，提供更精确的风格和内容控制，无需复杂的提示词描述。

## 节点功能

此节点作为Luma生成节点的辅助工具，让用户能够选择和应用预定义的概念，这些概念代表特定的视觉风格、内容类型或艺术表现。它简化了创作工作流程，提供了一种直观的方式来控制生成结果的风格和特性。

## 参数说明

### 基本参数

| 参数 | 类型 | 默认值 | 说明 |
|-----|-----|-------|------|
| concept_1 | 选择项 | "none" | 第一个概念选择 |
| weight_1 | 浮点数 | 1.0 | 第一个概念的权重 (0-1) |

### 可选参数

| 参数 | 类型 | 说明 |
|-----|-----|------|
| concept_2 | 选择项 | 可选的第二个概念 |
| weight_2 | 浮点数 | 第二个概念的权重 |
| concept_3 | 选择项 | 可选的第三个概念 |
| weight_3 | 浮点数 | 第三个概念的权重 |

### 输出

| 输出 | 类型 | 说明 |
|-----|-----|------|
| luma_concepts | LUMA_CONCEPT | 概念和权重的组合对象 |

## 使用示例

Luma Concepts 特别适合：

- 应用特定的艺术风格而不需要详细描述
- 确保生成内容遵循特定的视觉语言
- 混合多个视觉风格创建独特的美学效果
- 简化创作工作流程，减少提示词工程的复杂性

## 特点和优势

概念功能的主要特点：

- **预定义的视觉风格**：快速访问Luma AI支持的多种风格和概念
- **灵活的权重控制**：通过权重参数精确控制每个概念的影响程度
- **概念组合**：支持多个概念的混合以创建复杂的视觉风格
- **与多种生成节点兼容**：可用于所有Luma生成节点

## 工作原理

Luma Concepts 节点让用户从预定义的概念列表中选择，并为每个所选概念设置权重。这些概念代表了Luma AI理解的特定视觉风格、内容类型或艺术表现。

节点创建一个包含所选概念和权重的对象，该对象随后传递给Luma生成节点。在生成过程中，Luma AI会根据这些概念和相应的权重来影响生成结果，确保输出图像体现所选概念的视觉特性。

通过组合多个概念并调整它们的权重，用户可以创建复杂的视觉风格指导，而无需撰写详细的提示词描述。

class LumaConceptsNode(ComfyNodeABC):
    """
    Holds one or more Camera Concepts for use with Luma Text to Video and Luma Image to Video nodes.
    """

    RETURN_TYPES = (LumaIO.LUMA_CONCEPTS,)
    RETURN_NAMES = ("luma_concepts",)
    DESCRIPTION = cleandoc(__doc__ or "")  # Handle potential None value
    FUNCTION = "create_concepts"
    CATEGORY = "api node/image/Luma"

    @classmethod
    def INPUT_TYPES(s):
        return {
            "required": {
                "concept1": (get_luma_concepts(include_none=True),),
                "concept2": (get_luma_concepts(include_none=True),),
                "concept3": (get_luma_concepts(include_none=True),),
                "concept4": (get_luma_concepts(include_none=True),),
            },
            "optional": {
                "luma_concepts": (
                    LumaIO.LUMA_CONCEPTS,
                    {
                        "tooltip": "Optional Camera Concepts to add to the ones chosen here."
                    },
                ),
            },
        }

    def create_concepts(
        self,
        concept1: str,
        concept2: str,
        concept3: str,
        concept4: str,
        luma_concepts: LumaConceptChain = None,
    ):
        chain = LumaConceptChain(str_list=[concept1, concept2, concept3, concept4])
        if luma_concepts is not None:
            chain = luma_concepts.clone_and_merge(chain)
        return (chain,)
