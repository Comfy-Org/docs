Pika 2.2 Image to Video 节点允许你使用Pika最新的2.2版本API，将静态图像转变为动态、流畅的视频内容。

## 节点功能

此节点连接到Pika的图像到视频API，让用户能够基于输入图像创建动态视频。它保留原始图像的视觉特征，同时根据文本提示词添加自然的动态效果。该节点支持各种参数控制，包括提示词引导、视频持续时间和分辨率设置。

## 参数说明

### 基本参数

| 参数             | 类型     | 默认值               | 说明                           |
|-----------------|----------|---------------------|-------------------------------|
| image           | 图像      | -                   | 要转换为视频的输入图像            |
| prompt_text     | 字符串    | ""                  | 描述视频动作和内容的文本提示词     |
| negative_prompt | 字符串    | ""                  | 指定不希望在视频中出现的元素       |
| seed            | 整数      | 0                   | 生成过程的随机种子               |
| resolution      | 选择项    | "Resolution.1152_2048" | 生成视频的分辨率              |
| duration        | 选择项    | "Duration.four_seconds" | 生成视频的持续时间           |
| aspect_ratio    | 选择项    | "9:16"              | 输出视频的宽高比                 |

### 输出

| 输出   | 类型   | 说明          |
|--------|--------|--------------|
| VIDEO  | 视频   | 生成的视频结果 |

## 工作原理

Pika 2.2 Image to Video 节点首先分析输入图像的内容和结构，然后结合用户提供的文本提示词，为图像添加动态效果。节点将请求（包含图像数据和参数）发送到Pika的API服务器，处理完成后返回生成的视频结果。

用户可以通过调整提示词、负面提示词、随机种子等参数来精确控制生成结果。不同的分辨率和持续时间选项适合不同类型的图像内容和所需的动态效果。

class PikaImageToVideoV2_2(PikaNodeBase):
    """Pika 2.2 Image to Video Node."""

    @classmethod
    def INPUT_TYPES(cls):
        return {
            "required": {
                "image": (
                    IO.IMAGE,
                    {"tooltip": "The image to convert to video"},
                ),
                **cls.get_base_inputs_types(PikaBodyGenerate22I2vGenerate22I2vPost),
            },
            "hidden": {
                "auth_token": "AUTH_TOKEN_COMFY_ORG",
            },
        }

    DESCRIPTION = "Sends an image and prompt to the Pika API v2.2 to generate a video."
    RETURN_TYPES = ("VIDEO",)

    def api_call(
        self,
        image: torch.Tensor,
        prompt_text: str,
        negative_prompt: str,
        seed: int,
        resolution: str,
        duration: int,
        auth_token: Optional[str] = None,
    ) -> tuple[VideoFromFile]:
        """API call for Pika 2.2 Image to Video."""
        # Convert image to BytesIO
        image_bytes_io = tensor_to_bytesio(image)
        image_bytes_io.seek(0)  # Reset stream position

        # Prepare file data for multipart upload
        pika_files = {"image": ("image.png", image_bytes_io, "image/png")}

        # Prepare non-file data using the Pydantic model
        pika_request_data = PikaBodyGenerate22I2vGenerate22I2vPost(
            promptText=prompt_text,
            negativePrompt=negative_prompt,
            seed=seed,
            resolution=resolution,
            duration=duration,
        )

        initial_operation = SynchronousOperation(
            endpoint=ApiEndpoint(
                path=PATH_IMAGE_TO_VIDEO,
                method=HttpMethod.POST,
                request_model=PikaBodyGenerate22I2vGenerate22I2vPost,
                response_model=PikaGenerateResponse,
            ),
            request=pika_request_data,
            files=pika_files,
            content_type="multipart/form-data",
            auth_token=auth_token,
        )

        return self.execute_task(initial_operation, auth_token)
