Pixverse Template 节点允许你选择预定义的视频生成模板，用于控制Pixverse视频生成节点的输出风格和效果。

## 节点功能

此节点作为辅助节点，用于创建模板配置，可连接到Pixverse的视频生成节点。它让用户能够快速应用预设的视频风格和效果，而无需手动调整复杂的参数组合。

## 参数说明

### 基本参数

| 参数             | 类型     | 默认值  | 说明                      |
|-----------------|----------|--------|--------------------------|
| template        | 选择项    | -      | 预设的视频生成模板         |

### 输出

| 输出                | 类型                 | 说明                      |
|--------------------|---------------------|--------------------------|
| PIXVERSE_TEMPLATE  | PixverseIO.TEMPLATE | 包含模板设置的配置对象     |

## 工作原理

Pixverse Template 节点提供一系列预定义的模板选项，这些模板包含了对视频生成过程的各种设置组合。当用户选择一个模板后，节点创建一个包含模板ID的配置对象，可以传递给Pixverse的视频生成节点。

这些模板涵盖了不同的视频风格、效果和动作特性，使用户能够快速获得专业级的视频效果，而无需了解底层的复杂参数设置。这对于希望快速获得高质量结果的用户特别有用。

            raise Exception(f"Template '{template}' is not recognized.")
