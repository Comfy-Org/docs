Luma Image to Video 节点允许你使用Luma AI的先进技术，将静态图像转换为流畅、动态的视频内容，为图像赋予生命力和动态特性。

## 节点功能

此节点连接到Luma AI的图像到视频API，让用户能够基于输入图像创建动态视频。它可以理解图像中的内容并生成自然、连贯的动作，同时保持原始图像的视觉风格和特性。结合文本提示词，用户可以精确控制生成视频的动态效果。

## 参数说明

### 基本参数

| 参数 | 类型 | 默认值 | 说明 |
|-----|-----|-------|------|
| image | 图像 | - | 要转换为视频的输入图像 |
| prompt | 字符串 | "" | 描述视频动作和内容的文本提示词 |
| model | 选择项 | "luma_imagination" | 使用的视频生成模型 |
| output_resolution | 选择项 | "1080p" | 输出视频分辨率 |
| output_duration | 选择项 | "normal" | 视频时长选项 |

### 可选参数

| 参数 | 类型 | 说明 |
|-----|-----|------|
| negative_prompt | 字符串 | 指定不希望在视频中出现的元素 |
| last_image | 图像 | 可选的视频最后帧参考图像 |
| luma_concepts | LUMA_CONCEPT | 用于调整生成风格的概念引导 |

### 输出

| 输出 | 类型 | 说明 |
|-----|-----|------|
| VIDEO | 视频 | 生成的视频结果 |

## 使用示例

Luma Image to Video 特别适合：

- 将静态照片转变为生动的短视频
- 为产品图像创建动态展示
- 赋予艺术作品动态效果
- 创建角色和场景的简单动画

## 特点和优势

Luma AI图像到视频转换的主要特点：

- **自然流畅的动作**：创建与图像内容匹配的自然动态效果
- **保留原始图像特质**：维持原始图像的视觉风格和特征
- **高质量输出**：生成高分辨率、流畅的视频内容
- **文本引导控制**：通过提示词精确控制动作类型和方向

## 工作原理

Luma Image to Video 节点分析输入图像的内容和结构，然后结合文本提示词来确定如何为图像添加动态效果。它使用Luma AI的生成模型理解图像中的对象、人物或场景，并创建合理、连贯的动作序列。

用户可以通过提示词描述期望的动作类型、方向和强度，节点将据此生成相应的视频效果。通过设置不同的参数，如分辨率和时长，用户可以进一步定制输出视频的特性。

此外，通过提供最后帧参考图像，用户可以指定视频的结束状态，使动作朝特定方向发展。概念引导功能则允许用户进一步控制视频的整体风格和美学效果。
