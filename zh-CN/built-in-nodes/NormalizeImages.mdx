## 输入参数

| 参数 | 数据类型 | 必填 | 取值范围 | 描述 |
|-----------|-----------|----------|-------|-------------|
| `image` | IMAGE | 是 | - | 待归一化的输入图像。 |
| `mean` | FLOAT | 否 | 0.0 - 1.0 | 从图像像素中减去的均值（默认值：0.5）。 |
| `std` | FLOAT | 否 | 0.001 - 1.0 | 用于除以图像像素的标准差值（默认值：0.5）。 |
