| `string_b` | STRING | 是 | - | 要连接的第二个文本字符串 |
| `delimiter` | STRING | 否 | - | 在两个输入字符串之间插入的字符或字符串（默认为空字符串） |

| 输出名称 | 数据类型 | 描述 |
