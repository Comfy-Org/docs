# ComfyUI About 页面说明

## 概述

About 页面是 ComfyUI 设置系统中的一个信息展示面板，用于显示应用程序版本信息、相关链接和系统统计数据。该页面位于设置对话框的"特殊设置"分组下。

## 访问方式

### 通过设置菜单
1. 打开 ComfyUI 设置对话框
2. 在左侧导航栏中找到"特殊设置"分组
3. 点击"About"选项

### 通过帮助菜单
- 在顶部菜单栏中选择"帮助" > "About ComfyUI"

## 页面内容

### 版本信息徽章

About 页面显示以下核心版本信息：

- **ComfyUI 版本**：显示后端 ComfyUI 的版本号，链接到官方 GitHub 仓库
- **ComfyUI_frontend 版本**：显示前端界面的版本号，链接到前端 GitHub 仓库  
- **Discord 社区**：提供 ComfyOrg Discord 服务器的链接
- **官方网站**：链接到 ComfyOrg 官方网站

### 扩展徽章

如果安装了扩展，About 页面还会显示扩展提供的额外徽章信息。这些徽章由各个扩展通过 `aboutPageBadges` 属性注册。

### 系统统计信息

页面底部显示详细的系统统计信息，包括：
- 硬件配置信息
- 软件环境信息
- 系统性能数据

## 技术实现

About 页面通过以下组件实现：

- **AboutPanel.vue**：主要的 Vue 组件，负责页面渲染
- **useAboutPanelStore**：Pinia 状态管理，管理徽章数据
- **SystemStatsPanel**：系统统计信息组件
## 扩展开发者指南

扩展开发者可以通过在扩展配置中添加 `aboutPageBadges` 属性来向 About 页面添加自定义徽章：

```javascript
app.registerExtension({
  name: 'MyExtension',
  aboutPageBadges: [
    {
      label: 'My Extension v1.0.0',
      url: 'https://github.com/myuser/myextension',
      icon: 'pi pi-github'
    }
  ]
})
```

## 多语言支持

About 页面支持多语言显示，所有文本内容都通过 vue-i18n 国际化系统进行翻译。

## 注意事项

- About 页面是只读的信息展示页面，不包含可配置的设置项
- 系统统计信息会在页面挂载时自动获取最新数据
- 所有外部链接都会在新标签页中打开，确保用户不会丢失当前工作状态
