## API 节点

**显示 API 节点定价徽章**
- **作用**：控制是否在 API 节点上显示定价徽章
- **默认值**：启用
- **功能**：帮助用户识别 API 节点的使用成本

![启用效果](/images/interface/setting/comfy/api_node_pricing_badge.jpg)

> 更多关于 API 节点请参考 [API 节点](/zh-CN/tutorials/api-nodes/overview)

## 开发者模式

- **作用**：启用开发者模式选项（如API保存等）
- **默认值**：禁用

## 编辑令牌权重

**作用**：使用 Ctrl+上/下 可以快速调整权重，这个选项会改变每次调整的权重值

当你在使用类型 CLIPTextEncode 节点时，使用 Ctrl+上/下 可以快速调整权重，这个选项会改变每次调整的权重值

## 区域设置（本地化）

目前 ComfyUI 针对以下语言进行了本地化支持：英语、中文、俄语、日语、韩语、法语、西班牙语
- **作用**：修改 ComfyUI 界面显示语言
- **选项**：英语、中文、俄语、日语、韩语、法语、西班牙语
- **默认值**：自动检测浏览器语言

## 菜单

**使用新菜单**
- **作用**：控制菜单系统的使用和位置
- **选项**：禁用、顶部、底部
- **默认值**：顶部
- **功能**：选择菜单界面和位置

## 模型库

模型库指的是 ComfyUI 侧边菜单栏中的模型管理功能，你可以通过这个功能来查看你在 `ComfyUI/models` 及额外配置的模型文件夹中的模型

![模型库](/images/interface/sidepanel/model_library.jpg)

**模型库名称格式** 
- **作用**：选择在模型库树视图中显示的名称格式
- **选项**：文件名、标题
- **默认值**：标题
- **功能**：选择显示简化文件名或配置的元数据标题

**自动加载所有模型文件夹** 
- **作用**：控制是否在打开模型库时自动加载所有文件夹
- **默认值**：禁用
- **功能**：启用可能导致加载延迟，禁用时只有点击文件夹名称才会加载对应文件夹下的文件

## 节点

在 ComfyUI 的迭代过程中，我们会对一些节点进行调整，也会启用一些节点，这些节点可能在未来版本中发生重大变化或被移除，但是为了保证兼容性类似弃用的节点并没有被移除，你可以通过下面的设置来启用是否显示 **实验性节点** 和 **已弃用节点**


**显示已弃用节点**
- **作用**：控制是否在搜索中显示已弃用的节点
- **默认值**：禁用
- **功能**：已弃用节点在UI中默认隐藏，但在现有工作流中仍然有效

![已弃用节点](/images/interface/setting/comfy/depr_node.jpg)


**显示实验性节点**
- **作用**：控制是否在搜索中显示实验性节点
- **默认值**：启用
- **功能**：实验性节点可能在未来版本中发生重大变化或被移除

![已弃用节点](/images/interface/setting/comfy/beta_node.jpg)


## 节点搜索框

**节点建议数量**
- **作用**：用于修改相关节点上下文菜单中推荐的节点数量，数值越大显示的相关推荐节点数量越多
- **默认值**：5

![节点建议数量](/images/interface/setting/comfy/node_suggestions.jpg)

**显示节点频率**
- **作用**：控制是否在搜索结果中显示节点使用频率
- **默认值**：禁用

![节点频率](/images/interface/setting/comfy/node_frequency.jpg)

**显示节点ID名称**
- **作用**：控制是否在搜索结果中显示节点ID名称
- **默认值**：禁用

![节点ID名称](/images/interface/setting/comfy/node_id_name.jpg)

**节点搜索框实现**
- **作用**：选择节点搜索框的实现方式（实验性功能）
- **选项**：默认、litegraph（旧版）
- **默认值**：默认
- **功能**：控制搜索框界面和功能

**节点预览**
- **作用**：控制是否在搜索结果中显示节点预览
- **默认值**：启用
- **功能**：仅适用于默认搜索框实现

**显示节点类别**
- **作用**：控制是否在搜索结果中显示节点类别
- **默认值**：启用
- **功能**：帮助用户了解节点的分类信息



## 界面和菜单设置

**已打开工作流位置**
- **作用**：控制打开的工作流标签的显示位置
- **选项**：侧边栏、顶部栏、顶部栏（第二行）
- **默认值**：根据窗口宽度自动选择
- **功能**：优化工作流标签的布局显示

**关闭窗口时显示确认**
- **作用**：控制关闭浏览器窗口或标签页时是否显示确认
- **默认值**：启用
- **功能**：防止意外关闭窗口导致工作丢失

## 工作流相关设置

**工作流验证**
- **作用**：控制是否对工作流进行验证检查
- **默认值**：启用
- **功能**：确保工作流的结构和连接正确性

**保存工作流时排序节点ID**
- **作用**：决定保存工作流时是否对节点ID进行排序
- **默认值**：禁用 
- **功能**：使工作流文件格式更规范，便于版本控制

**显示缺失节点警告**
- **作用**：控制是否显示工作流中缺失节点的警告
- **默认值**：启用
- **功能**：帮助用户识别工作流中不可用的节点

**显示缺失模型警告**
- **作用**：控制是否显示工作流中缺失模型的警告（实验性功能）
- **默认值**：启用
- **功能**：帮助用户识别工作流中不可用的模型文件

**清除工作流时需要确认**
- **作用**：控制清除工作流时是否显示确认对话框
- **默认值**：启用
- **功能**：防止意外清除工作流内容

**保存工作流时提示文件名**
- **作用**：控制保存工作流时是否提示输入文件名
- **默认值**：启用
- **功能**：允许用户自定义工作流文件名

**删除工作流时显示确认**
- **作用**：控制删除工作流时是否显示确认对话框
- **默认值**：启用
- **功能**：防止意外删除重要工作流

**自动保存**
- **作用**：控制工作流的自动保存行为
- **选项**：关闭、延迟后
- **默认值**：关闭
- **功能**：自动保存工作流更改，避免数据丢失

**自动保存延迟**
- **作用**：设置自动保存的延迟时间（毫秒）
- **默认值**：1000毫秒
- **功能**：仅在自动保存设置为"延迟后"时生效

**工作流状态持久化**
- **作用**：控制是否在页面重新加载时恢复工作流状态
- **默认值**：启用 
- **功能**：在页面刷新后保持工作流内容


## 验证和开发者设置

**验证节点定义**
- **作用**：控制是否在启动时验证所有节点定义（慢）
- **默认值**：禁用
- **功能**：推荐给节点开发者使用，会在启动时验证所有节点定义


## 小部件和组件设置

**文本框组件拼写检查**
- **作用**：控制文本域小部件是否启用拼写检查
- **默认值**：禁用
- **功能**：在文本输入时提供拼写检查功能

**预览图像格式**
- **作用**：设置图像小部件中预览图像的格式
- **默认值**：空（使用原格式）
- **功能**：转换为轻量级格式如webp、jpeg等以优化性能

**禁用节点小部件滑块**
- **作用**：控制是否禁用节点小部件中的滑块
- **默认值**：禁用
- **功能**：可以强制使用文本输入而非滑块

**禁用浮点数四舍五入**
- **作用**：控制是否禁用默认浮点小部件的四舍五入
- **默认值**：禁用
- **功能**：需要页面重新加载，当节点后端设置了

**浮点数四舍五入精度**
- **作用**：设置浮点小部件四舍五入的小数位数
- **默认值**：0（自动）
- **功能**：0表示自动，需要页面重新加载

**小部件控制模式**
- **作用**：控制小部件值更新的时机（随机/增加/减少）
- **选项**：之前、之后
- **默认值**：之后
- **功能**：决定在执行工作流之前还是之后更新小部件值

## 系统和其他设置

**启用工具提示**
- **作用**：控制是否显示界面元素的工具提示
- **默认值**：启用
- **功能**：提供界面元素的说明信息

**批处理计数限制**
- **作用**：设置单次点击添加到队列的最大任务数量
- **默认值**：100
- **功能**：防止意外添加过多任务到队列

**启用DOM元素裁剪**
- **作用**：控制是否启用DOM元素裁剪（可能影响性能）
- **默认值**：启用
- **功能**：优化界面渲染，但可能降低性能

**保存和恢复工作流视图**
- **作用**：控制是否在工作流中保存和恢复画布位置和缩放级别
- **默认值**：启用
- **功能**：在重新打开工作流时恢复之前的视图状态
