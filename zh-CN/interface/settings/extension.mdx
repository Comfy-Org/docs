# ComfyUI Extension 设置完整文档

## 概述

Extension 设置面板是 ComfyUI 前端设置系统中的一个特殊管理面板，专门用于管理前端扩展插件的启用/禁用状态。它位于设置对话框的"特殊设置"分组中。

## Extension vs Custom Node 的区别

### Extension（前端扩展）
- **定义**: 前端 JavaScript/TypeScript 模块，遵循 `ComfyExtension` 接口
- **功能范围**: 可提供命令、快捷键、设置、UI 组件、菜单项等多种功能
- **注册方式**: 通过 `app.registerExtension()` 或 `useExtensionService().registerExtension()` 注册

### Custom Node（自定义节点）
- **定义**: 主要指后端 Python 实现的节点类型
- **功能范围**: 专注于提供新的节点功能
- **获取方式**: 通过 API 从后端获取节点定义

## 使用方法

### 打开 Extension 设置
1. 使用快捷键 `Ctrl+,` 打开设置对话框
2. 在左侧导航中找到"Special Settings"分组
3. 点击"Extension"进入扩展管理面板

### 管理扩展状态
1. **启用/禁用单个扩展**: 点击扩展右侧的切换开关
2. **搜索扩展**: 在顶部搜索框输入扩展名称
3. **批量操作**: 点击右上角的三点菜单选择批量操作

## 扩展类型识别

### 核心扩展
- 显示 "Core" 标签
- 通常不能被禁用
- 提供 ComfyUI 的基础功能

### 第三方扩展
- 来自自定义节点包或独立扩展
- 可以自由启用/禁用
- 某些扩展可能被标记为只读

## 使用场景

1. **性能优化**: 禁用不需要的扩展以提高应用性能
2. **功能定制**: 根据工作流程启用特定功能
3. **故障排除**: 在出现问题时快速禁用可能有问题的扩展
4. **开发调试**: 开发者可以方便地测试扩展的启用/禁用效果

## 注意事项

- 扩展状态更改需要重新加载页面才能生效
- 某些核心扩展无法被禁用
- 系统会自动禁用已知有问题的扩展
- 扩展设置会自动保存到用户配置文件中

这个 Extension 设置面板本质上是一个"前端插件管理器"，让用户可以灵活控制 ComfyUI 的功能模块，而不仅仅是传统意义上的配置选项。
