## 基本界面

可视化界面是目前绝大数用户使用 ComfyUI 来调用 [ComfyUI Server](/essentials/comfyui-server/comms_overview) 进行相应媒体资源生成的方式，它提供了一个可供用户操作和组织工作流的可视化界面，用于组织和调试工作流，并生成令人惊叹的作品。

通常，当你启用 ComfyUI server 后你可以看到下面这样的一个界面：

![ComfyUI 基础界面](/images/interface/overview/comfyui_new_interface.jpg)

如果你是较为早期的用户，你应该还见过之前的这样的菜单界面：

![ComfyUI 旧界面](/images/interface/overview/comfyui_old_interface.jpg)

目前 [ComfyUI前端](https://github.com/Comfy-Org/ComfyUI_frontend) 是一个独立的项目，作为一个独立的 pip 包进行发布和更新维护，如果你想要参与贡献，可以 fork 这个[仓库](https://github.com/Comfy-Org/ComfyUI_frontend)，并进行 pull request。

## 工作区（Workspace）

工作区是 ComfyUI 的默认界面，它提供了基本的界面分区，以及工作流、节点、快捷键、设置、自定义节点、模型等功能的入口。
