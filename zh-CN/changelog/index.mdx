<Update label="v0.3.41" description="January 17, 2025">
**重大更新内容**

**🎯 新模型支持**
- **Cosmos Predict2 模型支持**：新增对 Cosmos predict2 文本到图像模型（包括 2B 和 14B 版本）的完整支持，为用户提供更强大的文本生成图像能力
- **Cosmos 图像到视频**：现在支持 Cosmos predict2 图像到视频模型，扩展了视频生成工作流的可能性
- **ModelSamplingContinuousEDM 节点增强**：新增 `cosmos_rflow` 选项，优化 Cosmos 模型的采样过程

**🔧 工作流优化改进**
- **Flux 模型兼容性**：Chroma TE 现在可以在常规 Flux 模型上正常工作，提升了模型兼容性
- **LoRA 训练器**：新增 LoRA 训练节点，采用权重适配器方案，让用户能够直接在 ComfyUI 中进行模型微调
- **SaveLora 文件命名**：统一了 SaveLora 节点的文件命名格式，与其他节点保持一致

**⚡ 性能与稳定性提升**
- **AMD GPU 优化**：在支持的 AMD GPU 上默认启用 fp8 运算和 PyTorch attention，显著提升性能
- **Flux 模型修复**：解决了某些 Flux 模型在 fp16 模式下产生黑色图像的问题
- **内存估算优化**：改进了 Cosmos 模型的内存使用估算，提升系统稳定性

**🛠️ 采样器扩展**
- **SEEDS 采样器泛化**：扩展了 SEEDS 采样器对 RF（Rectified Flow）的支持
- **多步 DPM++ SDE 采样器**：为 RF 模型新增多步 DPM++ SDE 采样器，提供更多采样选择

**🔧 开发者功能**
- **SQLite 数据库支持**：新增对 SQLite 数据库的支持，改善数据管理
- **自动 Web 文件夹注册**：从 pyproject.toml 自动注册 web 文件夹，简化扩展开发
- **前端版本控制**：支持指定任何前端 semver 后缀，增加 '@prerelease' 选项使用最新测试前端

**🐛 错误修复**
- 修复了 WebcamCapture IS_CHANGED 签名问题
- 解决了 Kontext 宽高比仅作为小部件工作的问题
- 修复了 LTXV RMSNorm 的 eps 值设置
- 改进了 PyTorch 版本比较逻辑

此版本为 ComfyUI 用户带来了更强大的模型支持、更稳定的性能表现和更丰富的工作流创建选项。特别是 Cosmos 模型系列的支持和 LoRA 训练功能，将大大扩展用户的创作可能性。
</Update>
