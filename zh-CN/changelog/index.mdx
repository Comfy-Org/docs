<Update label="v0.21.0" description="2026年5月11日">

**主要媒体加载改进**
* **统一音频/视频加载：** 视频加载器节点现在可以同时读取音频和视频，简化媒体工作流程
* **PyAV 集成：** 使用 PyAV 替换 Pillow 进行图像加载，减少 JPEG 格式的内存使用并支持 tRNS PNG 透明度
* **元数据处理：** 在 PyAV 代码中添加了自动元数据旋转处理，确保正确的图像方向

**新模型支持**
* **CogVideoX：** 添加了对 CogVideoX 视频生成模型的支持
* **Gemma4：** 引入了对 Gemma4 模型的文本生成支持
* **Void 模型：** 新增对具有第1和第2阶段功能的 Void 模型的支持
* **BiRefNet：** 添加了背景移除模型支持，用于高级遮罩工作流程
* **Wan-Dancer：** 实现了对 Wan-Dancer 模型的支持

**性能增强**
* **块预取和 LoRA 异步加载：** 显著的速度提升改进，特别是对 LTX 模型
* **动态 VRAM 优化：** 增强内存管理，支持 --cache-ram 2 并减少视频处理的峰值 VRAM 使用
* **Triton ComfyUI kitchen：** 添加 CLI 参数以启用高级优化功能

**视频生成功能**
* **自回归视频生成：** 用于扩展视频序列创建的新功能
* **因果强制模型的 I2V：** 为因果模型架构提供图像到视频支持
* **上下文窗口：** 添加 causal_window_fix 以改进长视频序列中的混合效果
* **帧插值改进：** 更好的内存估算和减少的处理开销

**合作伙伴节点新增**
* **Luma UNI-1：** 新增对 Luma Uni-1 统一图像模型（图像创建与编辑工作流）的合作伙伴节点支持
* **Topaz Astra 2：** 集成 Topaz 视频增强（Astra 2 创意放大模型）
* **Tripo3D 3.1：** 更新了 3D 生成能力
* **GPT-5.5 模型：** 支持 GPT-5.5 和 GPT-5.5-pro LLM 变体
* **NanoBanana2：** 具有 DynamicCombo/Autogrow 功能的新节点

**节点改进**
* **Flux2 潜在预览：** 为 Flux2 模型潜在变量提供高质量预览支持
* **图像混合修复：** 解决了 --gpu-only 标志的设备不匹配问题并改进了通道数处理
* **批处理广播：** 增强 JoinImageWithAlpha 节点的批处理功能
* **数学表达式：** 添加了布尔支持以进行更复杂的计算
* **瓦片处理：** 修复了 SplitImageToTileList 和 ImageMergeTileList 节点中的垂直步幅计算

**开发者和 API 更新**
* **功能标志系统：** 添加了通用的 --feature-flag CLI 参数和注册表以提供扩展性
* **OpenAPI 增强：** 在规范中添加了云运行时操作和实验端点
* **蓝图支持：** 为蓝图子图添加了描述字段以改进文档
* **ControlNet 确定性：** 使 ControlNet 加载顺序确定化以获得一致的结果

**错误修复和稳定性**
* 通过处理未初始化的元张量修复了 CPU TE 崩溃
* 解决了 SolidMask 和 MaskComposite 设备不匹配问题
* 修正了 VAEDecodeAudio 与 LTX-2.x 生成的音频潜在变量的兼容性
* 改进了服务器启动时端口占用错误的优雅处理
* 修复了不带尾随下划线的保存文件名支持

</Update>
