<Update label="v0.3.39" description="2025年6月26日">

**增强的 LoRA 支持和 API 集成**

此版本为模型兼容性和工作流灵活性带来了重大改进：

- **HiDream SimpleTuner LoRA 支持**：添加了对 HiDream SimpleTuner LoRA 的原生支持，扩展了您的微调选项和模型自定义功能，专为专业 AI 艺术生成而设计
- **Pika API 节点改进**：重构了 Pika API 节点导入，修复了 unique_id 处理问题，确保视频生成工作流更加可靠
- **WAN Phantom 集成**：对 WAN Phantom 模型的实验性支持（未完全测试）- 为富有冒险精神的工作流创建者提供早期体验
- **BFL Kontext API 节点**：新增用于 Black Forest Labs Kontext 集成的 API 节点，开启更多高质量生成选项
- **模板更新**：更新至模板版本 0.1.22，改进了工作流启动模板
- **代码库清理**：移除了不必要的文件，代码库更加整洁、易于维护

**对工作流创建者的影响：**
- 扩展的 LoRA 兼容性意味着您的创意项目拥有更多样化的模型选择
- 增强的 API 集成提供了更多专业级生成工具
- 改进的节点可靠性减少了工作流中断和调试时间

</Update>
