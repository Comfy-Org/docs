# ComfyUI v0.3.40

增强工作流能力的新功能和改进

• 新增 ImageStitch 节点用于拼接图像，GetImageSize 节点用于获取图像尺寸
• 引入正则表达式替换操作节点，支持高级文本处理工作流
• 增强前端功能，在节点库中添加帮助菜单，修复多项稳定性问题
• 改进 ROCm 支持和自定义节点与 pyproject.toml 配置的兼容性
• 修复安全漏洞（XSS）和弃用的 Pillow API 使用，提升稳定性
</Update>
