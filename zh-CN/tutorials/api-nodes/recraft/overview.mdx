# Recraft AI 图像生成与编辑 API

![Recraft x ComfyUI](/images/tutorial/api_nodes/brands/Recraft_blue.png)

[Recraft AI](https://www.recraft.ai/) 是一款功能强大的人工智能设计工具，专注于生成和编辑矢量艺术、图标、3D图像和各种风格的插画，适用于网站设计、印刷品和网络营销等多种场景。
在 ComfyUI 中，我们已经集成了 Recraft API，让你可以直接在工作流程中使用这些强大的图像生成和编辑功能。

Recraft AI 支持多种风格的图像生成，包括线稿、扁平风、卡通风、涂鸦风、手绘、发光、迷幻风、像素风等，同时还能生成写实和3D插图。您可以通过简单的文字描述一键生成图像，无需设计经验。

## 功能概述

### 风格类型

Recraft 支持多种图像风格类型：

- **真实图像 (realistic_image)** - 看起来像普通的数码相机或智能手机拍摄的照片
- **数字插画 (digital_illustration)** - 手绘或使用电脑绘制的图片，具有简化的纹理
- **矢量插画 (vector_illustration)** - 使用矢量图形绘制的图像，通常只使用少量颜色
- **图标 (icon)** - 用于图形用户界面的小型数字图像或符号

每种风格都可以通过添加子风格进一步细化定义。

### 模型

Recraft 开发了两个强大的模型：
- **Recraft V2** (也称为 Recraft 20B) - 于2024年2月发布
- **Recraft V3** - 于2024年10月发布，在 Hugging Face 的文本到图像模型排行榜上排名第一

## 在 ComfyUI 中使用 Recraft API

### 准备工作

1. 首先，您需要从 Recraft 网站获取 API 密钥。登录 Recraft，进入个人资料并点击"生成"以获取 API 密钥（仅当 API 单位余额大于零时可用）。

2. 在 ComfyUI 中，找到 Recraft API 节点并将您的 API 密钥输入到相应字段中。

### 主要功能及使用方法

#### 生成图像

1. 将 Recraft 文本到图像节点添加到您的工作流程中
2. 在提示文本框中输入您想要生成的图像描述
3. 选择所需的风格类型（真实图像、数字插画、矢量插画或图标）
4. 选择所需的图像尺寸
5. 如果需要，添加负面提示以避免某些元素出现在结果中
6. 运行工作流程，查看生成的图像结果

#### 图像到图像转换

1. 将 Recraft 图像到图像节点添加到您的工作流程中
2. 连接输入图像
3. 在提示文本框中描述您希望如何修改输入图像
4. 调整强度滑块以控制转换的程度
5. 运行工作流程，查看转换后的图像结果

#### 背景替换

1. 将 Recraft 背景替换节点添加到您的工作流程中
2. 连接输入图像
3. 在提示文本框中描述您想要的新背景
4. 运行工作流程，查看更改背景后的图像结果

#### 图像修复

1. 将 Recraft 图像修复节点添加到您的工作流程中
2. 连接输入图像
3. 创建并连接一个掩码图像（白色区域表示要修复的部分）
4. 在提示文本框中描述您希望在掩码区域生成的内容
5. 运行工作流程，查看修复后的图像结果

#### 图像矢量化

1. 将 Recraft 图像矢量化节点添加到您的工作流程中
2. 连接输入图像
3. 运行工作流程，获取 SVG 格式的矢量图像结果

#### 背景移除

1. 将 Recraft 背景移除节点添加到您的工作流程中
2. 连接输入图像
3. 运行工作流程，获取透明背景的图像结果

#### 图像放大

1. 将 Recraft 清晰放大或创意放大节点添加到您的工作流程中
2. 连接输入图像
3. 运行工作流程，获取放大后的高分辨率图像结果

## 注意事项


## 支持的图像尺寸

- 1024x1024
- 1365x1024
- 1024x1365
- 1536x1024
- 1024x1536
- 1820x1024
- 1024x1820
等多种尺寸
