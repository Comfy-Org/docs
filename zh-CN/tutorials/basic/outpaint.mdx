## 关于扩图

在 AI 图像生成过程中，我们经常会遇到这样的需求：已有的图片构图很好，但是画面范围太小，需要扩展画布来获得更大的场景。这时候就需要用到扩图功能。

这就像让 **画家(AI 绘图模型)** 在已有的画作基础上，向外延伸绘制更大的场景。我们需要告诉画家 **需要扩展的方向和范围**，画家会根据已有的画面内容，合理地延伸和扩展场景，基本上它要求的内容与[局部重绘](/zh-CN/tutorials/basic/inpaint)相似，只不过我们用来**构建遮罩（Mask）的节点不同**

扩图的应用场景包括：
- **场景扩展：** 扩大原有画面的场景范围，展现更完整的环境
- **构图调整：** 通过扩展画布来优化整体构图
- **内容补充：** 为原有画面添加更多相关的场景元素

### 扩图工作流讲解

![ComfyUI 扩图工作流示意图](/images/tutorial/basic/outpaint/outpaint_workflow_detail.jpg)

扩图工作流的关键步骤如下：

1. **加载原始图片：** 使用 `Load Image` 节点加载需要扩展的图片
2. **设置扩展参数：** 在 `Image Resize with Crop` 节点中设置：
   - `width`：目标宽度
   - `height`：目标高度
   - `crop_position`：裁剪位置，决定扩展方向
3. **生成扩展区域：** 使用 `KSampler` 节点生成扩展部分的内容
4. **查看结果：** 通过 `Preview Image` 节点预览生成结果

### 扩图参数调整

为了获得更好的扩图效果，你可以调整以下参数：

1. **提示词优化：**
   - 正向提示词要尽可能描述完整的场景
   - 负向提示词要避免生成的扩展部分出现不协调的元素

2. **采样参数：**
   - 适当增加采样步数可以提高生成质量
   - 调整 CFG Scale 来控制对提示词的遵循程度

3. **扩展尺寸：**
   - 建议每次扩展不要过大，可以分步扩展
   - 保持适当的重叠区域以确保生成内容的连续性

## 扩图相关节点说明

### Image Resize with Crop 节点

这是扩图工作流中的核心节点，用于控制图片的扩展方向和范围。

![Image Resize with Crop 节点](/images/tutorial/basic/outpaint/image_resize_with_crop.jpg)

**输入参数**

| 参数名称 | 作用 |
|----------|------|
| `image` | 需要进行扩展的输入图像 |
| `width` | 目标图像宽度 |
| `height` | 目标图像高度 |
| `crop_position` | 裁剪位置，决定扩展方向（center、top、bottom、left、right） |

**输出参数**

| 参数名称 | 作用 |
|----------|------|
| `IMAGE` | 调整后的图像 |
| `MASK` | 生成的扩展区域遮罩 |

### 使用技巧

1. **渐进式扩展：**
   - 对于大幅扩展，建议分多次进行
   - 每次扩展后检查结果，必要时进行局部调整

2. **保持一致性：**
   - 使用合适的提示词来保持场景风格的一致
   - 可以结合局部重绘来优化过渡区域

3. **提示词技巧：**
   - 在提示词中包含原始图片的场景描述
   - 添加你希望在扩展区域出现的元素描述

通过合理使用这些参数和技巧，你可以创建出自然、连贯的扩展效果。记住，好的扩图效果往往需要多次尝试和细节调整。
