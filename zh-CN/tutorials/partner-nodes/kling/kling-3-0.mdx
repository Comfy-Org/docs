Kling 3.0 是目前最先进的多模态生成系统之一，现已通过合作伙伴节点在 ComfyUI 中提供。此版本包括 **Kling Video 3.0**、**Kling Video 3.0 Omni**、**Kling Image 3.0** 和 **Kling Image 3.0 Omni**，将视频、图像、视听和叙事生成能力直接集成到你的节点工作流中。

## 产品亮点

### 多镜头与时长控制

全新的 **多镜头** 功能可以直接从提示词理解场景覆盖，并自动规划相机角度、镜头构图、对话正反打和交叉剪辑结构。

无需手动编辑和剪切，你可以告诉模型在一次生成中需要 **多少个镜头**，并明确为 **每个镜头分配时长**。

**示例提示词：**
```
镜头 1：快节奏跟踪镜头，摩托车翘起前轮，马匹奔腾，微微扬起尘土
镜头 2：低角度拍摄马蹄和摩托车轮
镜头 3：过肩镜头拍摄骑摩托车的主角
镜头 4：马匹跳跃中的极致慢动作
```

### 图生视频与锁定主体一致性

Kling 3.0 大幅改进了 **参考驱动生成**：

- 支持 **多图像和视频参考** 作为场景元素
- 为 **角色、产品和环境** 建立稳定的身份特征
- 在 **相机运动和场景演变** 中保持一致性

这为 ComfyUI 中的故事叙述、品牌推广和角色工作流提供了 **专业级连续性**。

**带参考的示例提示词：**
```
镜头 1：推镜头拍摄来自 @image 的剪影主角
镜头 2：从 @character 背后弧形运镜到正面，他很紧张，可见胸部随深呼吸起伏，缓慢推向他的脸部
镜头 3：@character 手持 @device，设备发出微弱光芒，他按下按钮
镜头 4：拍摄来自 @image 的巨大黑色方尖碑，方尖碑爆炸，火焰冲天
```

### 原生音频与角色感知

音频生成现已与场景理解深度集成：

- 在多角色场景中精确控制 **哪个角色在说话**
- 支持 **中文、英文、日文、韩文和西班牙文**
- 真实的 **方言、口音和多语言切换**
- 自然的 **口型同步和面部表情对齐**

**示例提示词：**
```
女人（轻声，担忧的语气）："你确定这是合法的吗？" 男人回应，他带着狡黠的微笑，（用冷静自信的低沉南方口音）："这些未公开的付费广告永远不会见光。" 他关上后备箱，一片漆黑。
```

### 视频中的原生级文字渲染

Kling 3.0 现在可以可靠地生成 **清晰、结构化的屏幕文字**：

- 保留原始 **标识和字幕**
- 生成新的 **布局感知排版**
- 提升 **广告、UI 场景和品牌内容** 的真实感

这是迈向 **可用商业视频生成** 的重要一步。

## 获得更好效果的技巧

- **使用清晰的镜头描述**：使用多镜头生成时，用具体的相机运动和动作描述每个镜头
- **使用 @ 引用图像**：在提示词中使用 `@image`、`@character` 或 `@device` 语法引用上传的图像
- **指定音频细节**：对于对话，在括号中包含语气、口音和情感提示
- **规划镜头时长**：明确为每个镜头分配时长以获得更好的节奏控制
- **匹配参考构图**：确保参考图像具有适合预期相机运动的构图
