Reve 是一个通过合作伙伴节点在 ComfyUI 中提供的图像生成模型。它通过三个专用节点支持最高 **4K** 分辨率的文生图、图像编辑和图像混合。

## 关于 Reve

**核心功能：**
- **最高 4K 分辨率**：生成清晰、细节丰富的高分辨率图像
- **内置美学训练**：在时尚编辑、电影感和自然光照风格方面表现出色
- **图像编辑**：修改光照条件、色彩分级，并为现有图像添加动态效果
- **图像混合**：将参考图像中的元素组合到目标图像中，同时保持一致性

## Reve Create

Reve Image Create 节点根据文本提示词生成图像。该模型在描述时尚编辑风格、特定光照条件、自然纹理和动态效果的提示词方面表现良好。

## Reve Remix

Reve Image Remix 节点将参考图像中的元素作为图层应用到目标图像上。你可以使用它在图像之间迁移特定的视觉属性，如阴影、色彩分级或美学风格。

## 开始使用

1. **更新** ComfyUI 到最新版本，或访问 [Comfy Cloud](https://cloud.comfy.org/?utm_source=docs)。
2. **找到** 左侧边栏节点库或模板中的 Reve Image Create、Reve Image Remix 和 Reve Image Edit 节点。
3. **拖放节点** 到画布上即可开始使用。

## 可用节点

| 节点 | 类型 | 描述 |
|------|------|------|
| **Reve Image Create** | 文生图 | 根据文本提示词生成最高 4K 分辨率的图像 |
| **Reve Image Edit** | 图像编辑 | 修改现有图像的光照、色彩分级和动态效果 |
| **Reve Image Remix** | 图像混合 | 将参考图像的视觉属性迁移到目标图像 |
