Reve 现已通过合作伙伴节点在 ComfyUI 中提供。该模型具有强大的内置美学风格，能够生成动态、高质量的图像，分辨率最高可达 **4K**。Create 节点具有快速的生成速度，让你可以快速迭代，并使用 Remix 和 Edit 节点调整输出效果。

## 关于 Reve

**核心功能：**
- **强大的内置美学**：训练内置的美学风格，在时尚编辑、电影感和自然光照风格方面产出高质量结果
- **最高 4K 分辨率**：以高分辨率生成清晰、锐利、细节丰富的图像
- **快速生成**：Create 节点提供快速的迭代速度
- **图像编辑**：使用 Edit 节点添加动态效果、更改光照条件和调整色彩分级
- **图像混合**：使用参考图像中的元素作为图层混合到目标图像中，同时保持一致性

## 开始使用

1. **更新** ComfyUI 到最新版本，或访问 [Comfy Cloud](https://cloud.comfy.org/?utm_source=docs)。
2. **找到** 左侧边栏节点库或模板中的 Reve Image Create、Reve Image Remix 和 Reve Image Edit 节点。
3. **拖放节点** 到画布上，即可开始创作。

## 可用节点

| 节点 | 类型 | 描述 |
|------|------|------|
| **Reve Image Create** | 文生图 | 以强大的内置美学风格生成最高 4K 分辨率的高质量图像 |
| **Reve Image Edit** | 图像编辑 | 编辑光照、色彩分级，并为现有图像添加动态效果 |
| **Reve Image Remix** | 图像混合 | 使用参考图像作为图层混合到目标图像中，同时保持一致性 |
