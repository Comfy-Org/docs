title: 登录您的账户
description: 访问您的 ComfyUI 账户以使用所有平台功能和服务。
---

您的 ComfyUI 账户可让您访问[合作伙伴节点（API 节点）](/zh-CN/tutorials/partner-nodes/overview)和云订阅，使您能够在整个平台上使用高级功能和服务。

## 支持的登录方式

ComfyUI 支持以下登录方式：
- **电子邮件**：使用您的电子邮件地址和密码登录
- **Google**：使用您的 Google 账户登录
- **GitHub**：使用您的 GitHub 账户登录

## 在 Comfy Cloud 上登录

要在 Comfy Cloud 上访问您的 ComfyUI 账户：

1. 访问 [Comfy Cloud](https://www.comfy.org)
2. 点击**登录**或**登入**
3. 选择您的登录方式：
   - **电子邮件**：输入您的电子邮件地址和密码，然后点击**登录**
   - **Google**：点击 Google 登录按钮并进行身份验证
   - **GitHub**：点击 GitHub 登录按钮并进行身份验证

## 在本地登录

如果您已在本地安装 ComfyUI：

1. 在本地计算机上打开 ComfyUI
2. 在界面中导航到**设置**
3. 进入**用户**部分（详见[用户设置](/interface/user)）
4. 选择您的登录方式：
   - **电子邮件**：输入您的电子邮件地址和密码
   - **Google**：点击 Google 登录按钮并进行身份验证
   - **GitHub**：点击 GitHub 登录按钮并进行身份验证

## 忘记密码

如果您忘记了密码（针对电子邮件登录）：

1. 点击登录页面上的**忘记密码？**链接
2. 输入您注册的电子邮件地址
3. 检查您的电子邮件以获取密码重置说明
4. 点击邮件中的重置链接
5. 创建新密码
6. 使用新密码登录

## 登录问题故障排除

如果您在登录时遇到问题：

- 验证您的电子邮件地址是否正确
- 检查输入密码时是否启用了大写锁定
- 清除浏览器 Cookie 和缓存
- 尝试使用其他浏览器或无痕模式
- 确保您的账户电子邮件已验证
- 如果问题仍然存在，请联系[支持](/zh-CN/support/contact-support)

## 安全提示

为了保护您的账户安全：
- 使用强大且唯一的密码
- 启用双因素身份验证
- 不要分享您的登录凭据
- 使用共享设备时请注销
- 定期更新您的密码
