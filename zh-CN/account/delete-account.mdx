<Warning>
账户删除是永久性的，无法撤销。您的所有数据、工作流和设置将被永久删除。
</Warning>

## 删除前须知

在继续删除账户之前，请考虑：

- **备份您的数据**：对于 Comfy Cloud 用户，您的资产存储在您的账户下，因此请备份您想要保留的任何数据。如果您不使用 Comfy Cloud，您的 API 使用情况可以在 https://platform.comfy.org/ 找到
- **取消订阅**：确保所有活动订阅已取消，以避免未来的费用
- **下载发票**：保存您可能需要的任何付款历史记录或发票副本
- **替代选项**：考虑暂时停用您的账户，而不是永久删除

## 删除您的 Comfy 账户

要永久删除您的 Comfy 账户以及 ComfyUI 相关数据，请发送删除请求至 [support@comfy.org](mailto:support@comfy.org)。

请在邮件中包含以下信息：
- 与您账户关联的电子邮件地址
- 明确的账户删除请求声明（例如："我请求删除我的账户及所有相关个人数据"）

根据 GDPR 及类似数据保护法规，您有权请求删除您的个人数据。我们将在 30 天内处理您的请求，并在您的账户和数据被永久删除后向您发送确认邮件。

<Note>
旧版本的应用程序可能仍会在账户设置中显示"删除账户"按钮。该按钮已不再有效。请使用上述电子邮件流程。
</Note>

## 将被删除的内容

当您删除账户时，以下数据将被永久删除：

- 您的用户资料和账户信息
- 所有保存的工作流和项目
- 生成的图像和输出
- 自定义设置和偏好
- 付款历史记录和账单信息
- API 密钥和访问令牌
