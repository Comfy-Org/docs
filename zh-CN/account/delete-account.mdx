<Warning>
账户删除是永久性的，无法撤销。您的所有数据、工作流和设置将被永久删除。
</Warning>

## 删除前须知

在继续删除账户之前，请考虑：

- **备份您的数据**：对于 Comfy Cloud 用户，您的资产存储在您的账户下，因此请备份您想要保留的任何数据。如果您不使用 Comfy Cloud，您的 API 使用情况可以在 https://platform.comfy.org/ 找到
- **取消订阅**：确保所有活动订阅已取消，以避免未来的费用
- **下载发票**：保存您可能需要的任何付款历史记录或发票副本
- **替代选项**：考虑暂时停用您的账户，而不是永久删除

## 删除您的 Comfy 账户

要永久删除您的 Comfy 账户以及 ComfyUI 相关数据：

1. 登录您的账户
2. 导航至**设置** > **账户**

<Frame>
  <img src="/images/interface/setting/menu-user-logged.jpg" alt="显示删除账户选项的账户设置菜单" />
</Frame>

3. 滚动到页面底部
4. 点击**删除账户**
5. 仔细阅读警告消息
6. 输入您的密码以确认
7. 在确认字段中输入"DELETE"
8. 点击**永久删除我的账户**

## 将被删除的内容

当您删除账户时，以下数据将被永久删除：

- 您的用户资料和账户信息
- 所有保存的工作流和项目
- 生成的图像和输出
- 自定义设置和偏好
- 付款历史记录和账单信息
- API 密钥和访问令牌
