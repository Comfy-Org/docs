### コントリビューションの方法

あらゆる種類のコントリビューションを歓迎しています。当社の [GitHub 組織](https://github.com/Comfy-Org) でサポートしているさまざまなリポジトリをご確認ください。

また、ワークフローの共有や、[カスタムノード](/custom-nodes/overview) の開発を通じてコントリビューションすることもできます。

### ステージング環境でのフロントエンド実行

`http://localhost:5173/` でパートナー/APIノードをローカル開発・テストする場合、ComfyUIをステージング環境に接続する必要があります。

**オプション1: ステージングAPIでComfyUIを起動**

```bash
python main.py --comfy-api-base https://stagingapi.comfy.org --verbose
```

これにより、ステージング環境が有効になり、パートナー/APIノードをローカルでテストする際の権限の問題を回避できます。

**オプション2: `DEV_SERVER_COMFYUI_URL` を使用**

フロントエンドリポジトリで `DEV_SERVER_COMFYUI_URL` 環境変数を変更することもできます:

```bash
"dev:cloud": "cross-env DEV_SERVER_COMFYUI_URL='https://cloud.comfy.org/' nx serve"
```

この方法では、特別な権限が不要になります。
