---

Chooser ビューで左上の **+** カードをクリックして新しい ComfyUI インストールを追加します。

<Steps>
  <Step title="Comfy Desktop の設定">

<Tabs>
  <Tab title="Windows">
    ![Windows で Comfy Desktop を設定](/images/desktop/usage/new_install_01_settings_windows.png)
  </Tab>
  <Tab title="macOS">
    ![macOS で Comfy Desktop を設定](/images/desktop/usage/new_install_01_settings_mac.png)
  </Tab>
</Tabs>

1. **① インスタンス名** — インスタンスの名前。デフォルトは "ComfyUI" です。
2. **② インストール場所** — ComfyUI のインストール先を選択します。Windows の場合は**SSD** を使用し、C ドライブのルートは避けることを強く推奨します。
3. **③ 詳細（Advanced）** — クリックして展開します。ここで選択できます：
   - **④ スタンドアロン（Standalone）** — 推奨オプション。Python バンドル、自動更新、スナップショット対応の自己完結型インストール。
   - **⑤ リモート接続（Remote Connection）** — 別のマシンで動作している ComfyUI に接続。

  </Step>
  <Step title="インストール中">

![ComfyUI をインストール中](/images/desktop/usage/new_install_02_installing.png)

Comfy Desktop が残りの作業（Python のダウンロード、環境構築、ComfyUI のインストール）を自動的に行います。進捗画面に現在のステップが表示されます。

  </Step>
  <Step title="テンプレートを選択">

![インストール完了後のテンプレートライブラリ](/images/desktop/usage/new_install_03_interface_template.png)

インストールが完了すると、Comfy Desktop がテンプレートライブラリを開きます。ここからワークフローテンプレートを選んで使い始められます。

  </Step>
</Steps>
