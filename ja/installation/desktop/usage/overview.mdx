このガイドでは、Comfy Desktop インストール後の使用方法を説明します。

![Comfy Desktop ホーム画面](/images/desktop/usage/01-homepage.png)

Comfy Desktop を開くと表示されるのが **Chooser** ビューです。すべての ComfyUI インストールがカードのグリッドとして表示されます。

## Chooser ビュー

**Chooser** は Comfy Desktop を開いたときに表示されるメイン画面です。すべての ComfyUI インストールがカードのグリッドとして表示されます。

### インストールカード

各カードには以下が表示されます：
- **名前** — インストールに付けた名前
- **タイプアイコン** — ソースタイプ（Standalone、Cloud、Git など）
- **最終起動** — 最後に使用してからの経過時間
- **ステータス** — 実行中、エラー、更新利用可能などの表示

### フィルタリング

グリッド上部のフィルターチップでインストールを絞り込めます：

- **All（すべて）** — すべてのインストール
- **Local（ローカル）** — Standalone および Legacy Desktop インストールのみ
- **Cloud（クラウド）** — Comfy Cloud 接続のみ
- **Remote（リモート）** — リモートサーバー接続のみ
