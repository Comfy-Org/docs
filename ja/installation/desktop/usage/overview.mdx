Comfy Desktop は **ComfyUI**（ノードベースの生成 AI エンジン）の公式デスクトップアプリケーションです。ComfyUI を自己完結型の GPU 対応環境にインストールし、複数の独立したセットアップを同時に実行・管理できるようにします。

このガイドでは、Comfy Desktop を使って ComfyUI インストールを管理する方法を説明します。
