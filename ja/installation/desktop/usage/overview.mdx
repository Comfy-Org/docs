Comfy Desktop は **ComfyUI** の公式デスクトップアプリケーションです。ComfyUI のインストール、更新、複数の独立したセットアップの並行管理を一手に引き受けます。

関係を簡単に言うと：**Comfy Desktop** が **ComfyUI** インスタンスを管理・起動するアプリです。ComfyUI を別途インストールする必要はありません — Comfy Desktop が各インスタンスのインストール、Python 環境、依存関係を自動的に処理します。

### 主な特長

- **[マルチインスタンス管理](/ja/installation/desktop/usage/instance-management)** — 複数の ComfyUI インスタンスを並行実行。それぞれが独自のバージョン、カスタムノード、設定を持てます
- **[スナップショット](/ja/installation/desktop/usage/manage#snapshots-tab)** — インスタンスの完全な状態（ComfyUI バージョン、カスタムノード、Python パッケージ）を保存・復元。ロールバックや環境の再現が容易
- **[共有ストレージ](/ja/installation/desktop/usage/settings#storage)** — モデルはデフォルトですべてのインスタンス間で共有され、ディスク容量を節約。外部モデルフォルダの追加も可能
- **[自動更新](/ja/installation/desktop/usage/manage#update-tab)** — Manage パネルから Comfy Desktop 自体と各 ComfyUI インスタンスを個別に更新
- **クラウド連携** — Chooser ビューから直接 Comfy Cloud インスタンスを起動・接続
- **[既存インストールの追跡](/ja/installation/desktop/usage/instance-management#tracking-an-existing-installation)** — 既存の portable や git clone の ComfyUI インストールを Comfy Desktop に追加して一元管理

このガイドではこれらの機能の使い方を説明します。

## Dashboard 概要

Comfy Desktop ウィンドウはいくつかの主要エリアに分かれています。下図は Dashboard の各部を示しています：

### アプリケーションメニュー

左上のハンバーガーメニュー（☰）からアプリケーションメニュー（File / Edit / View / Window / Help）を開きます。新しいウィンドウの表示、グローバル設定、アプリの終了など、デスクトップレベルの操作を行えます。

### インスタンス切替

中央のピルボタンは現在アクティブな ComfyUI インスタンス名を表示します。クリックすると**インスタンスピッカー**が開き、全インスタンスの一覧と動作状態を確認し、切り替えや起動ができます。

  <Card title="インストール管理" icon="sliders" href="/ja/installation/desktop/usage/manage">
    Manage パネルでの起動、更新、スナップショット、設定。
  </Card>
</CardGroup>

### 右側操作ボタン

タイトルバー右側のボタン群からクイックアクセス：

- **Desktop Update**：Comfy Desktop に新しいバージョンがある場合、ここに更新ボタンが表示されます。クリックしてダウンロード・適用します。
- **Feedback**：Comfy Desktop チームにフィードバックを送信します。
- **Downloads**：ダウンロードトレイを開き、進行中・最近のダウンロードを確認します。

### 検索ボックス

検索ボックスでインストール済みインスタンスを名前でフィルタリングできます。インスタンスが多い場合に便利です。

### Chooser（インスタンス一覧）

Dashboard のメインエリアは **Chooser** と呼ばれ、すべての ComfyUI インスタンスがカード形式で表示されます。**カードをクリックするとそのインスタンスが起動・切り替わります。**

ここで以下の操作が可能です：

- **＋ New Install**：新しいローカルの ComfyUI インスタンスを作成。
- **Comfy Cloud**：クラウドインスタンスに接続または起動。
- **既存のインスタンスカード**：各カードにはインスタンス名、ステータス、タイプが表示されます。右クリック（または ⋮ アイコン）で管理、更新、スナップショット、削除が行えます。
