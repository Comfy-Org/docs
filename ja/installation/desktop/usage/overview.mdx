このガイドでは Comfy Desktop について説明します。Comfy Desktop は ComfyUI のインストールを管理するデスクトップアプリケーションです。ワークフローの構築やノードの使い方については [ComfyUI エディター](/comfyui-editor)のセクションをご覧ください。

Comfy Desktop を使うと、複数の ComfyUI インスタンスを同じ場所でインストール、起動、管理できます。各インスタンスは独自の Python 環境、モデル、カスタムノードを持ちます。
