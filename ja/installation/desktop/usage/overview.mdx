このガイドでは、Comfy Desktop インストール後の使い方を説明します。

![Comfy Desktop ホーム画面](/images/desktop/usage/01-homepage.png)

Comfy Desktop を開くと **Chooser** が表示されます。すべての ComfyUI インストールがカードグリッドに並び、上部のフィルターチップでローカル、クラウド、リモートに絞り込めます。カードをクリックして起動するか、**+** をクリックして新しく追加します。
