**Comfy Desktop** は、複数の ComfyUI インスタンスを一つのランチャーから管理できるマルチインストレーションマネージャーです。このガイドでは、macOS への Comfy Desktop のインストールとアンインストールについて説明します。

## システム要件

- **OS:** macOS 13 (Ventura) 以降
- **ハードウェア:** Apple Silicon（M1 以降）
- **ディスク容量:** 各インストールにつき最低 15 GB 推奨

## インストール

1. ダウンロードした `.dmg` ファイルを開く
2. **Comfy Desktop** を **Applications** フォルダにドラッグ
3. Launchpad または Spotlight から起動

初回起動時にセキュリティ警告が表示される場合があります。**システム設定 → プライバシーとセキュリティ** から **このまま開く** をクリックしてください。

アプリを開くと **Welcome** 画面が表示され、最初のインストールを作成できます。

## アンインストール

### Comfy Desktop のアンインストール

**Comfy Desktop** を Applications フォルダからゴミ箱にドラッグします。

### インストールデータの削除

Comfy Desktop とそのインストールはディスク上で分離されています。完全にクリーンアップするには：

1. **アプリ内から個別のインストールを削除**するか、インストールフォルダを手動で削除
2. **アプリデータフォルダ** `~/Library/Application Support/Comfy Desktop` を削除

## データ保存場所

| データ | macOS パス |
|------|-----------|
| アプリデータ（設定、インストール一覧） | `~/Library/Application Support/Comfy Desktop` |
| ログ | `~/Library/Application Support/Comfy Desktop/logs` |
| インストールディレクトリ | セットアップ時にユーザーが選択 |

## トラブルシューティング

### macOS セキュリティ警告

macOS がアプリをブロックした場合は、**システム設定 → プライバシーとセキュリティ** で「このまま開く」をクリックしてください。

### ミラーエラー

ミラーが失敗した場合は、アプリ内のミラー設定を展開して、Python、PyPI、PyTorch 用のカスタムミラーを設定してください。
