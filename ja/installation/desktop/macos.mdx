**Comfy Desktop** は、複数の ComfyUI インスタンスを一つのランチャーから管理できるマルチインストレーションマネージャーです。このガイドでは、macOS への Comfy Desktop のインストールとアンインストールについて説明します。

## システム要件

- **OS:** macOS 13 (Ventura) 以降
- **ハードウェア:** Apple Silicon（M1 以降）
- **ディスク容量:** 各インストールにつき最低 15 GB 推奨

## インストール

1. ダウンロードした `.dmg` ファイルを開く
2. **Comfy Desktop** を **Applications** フォルダにドラッグ
3. Launchpad または Spotlight から起動

初回起動時にセキュリティ警告が表示される場合があります。**システム設定 → プライバシーとセキュリティ** から **このまま開く** をクリックしてください。

アプリを開くと **Welcome** 画面が表示され、最初のインストールを作成できます。

## アンインストール

### Comfy Desktop のアンインストール

**Comfy Desktop** を Applications フォルダからゴミ箱にドラッグします。

### インストールデータの削除

Comfy Desktop とそのインストールはディスク上で分離されています。完全にクリーンアップするには：

1. **アプリ内から個別のインストールを削除**するか、インストールフォルダを手動で削除
2. **アプリデータフォルダ** `~/Library/Application Support/Comfy Desktop` を削除

## データ保存場所

| データ | macOS パス |
|------|-----------|
| アプリデータ（設定、インストール一覧） | `~/Library/Application Support/Comfy Desktop` |
| ログ | `~/Library/Application Support/Comfy Desktop/logs` |
| インストールディレクトリ | セットアップ時にユーザーが選択 |

## トラブルシューティング

### macOS セキュリティ警告

macOS がアプリをブロックした場合は、**システム設定 → プライバシーとセキュリティ** で「このまま開く」をクリックしてください。

### ミラーエラー

ミラーが失敗した場合は、アプリ内のミラー設定を展開して、Python、PyPI、PyTorch 用のカスタムミラーを設定してください。

### エラーの特定

インストールに失敗した場合、以下の画面が表示されます

![Comfy Desktop インストール失敗](/images/desktop/win-comfyui-desktop-7.jpg)

以下の手順でエラーの原因を特定してください：
1. `ターミナルを表示` をクリックしてエラー出力を確認
2. `ログを開く` をクリックしてインストールログを確認
3. [Comfy 公式フォーラム](https://forum.comfy.org/) で類似のエラーレポートを検索
4. `再インストール` をクリックして再試行

フィードバックを送信する前に、**エラー出力** と **ログファイル** を GPT などの AI ツールに提供することをお勧めします

![Comfy Desktop インストール失敗 - エラーログ](/images/desktop/win-comfyui-desktop-8.jpg)
![Comfy Desktop インストール失敗 - GPT フィードバック](/images/desktop/win-comfyui-desktop-9.jpg)

上記のように、GPT にエラーの原因を分析させるか、Comfy Desktop を完全にアンインストールして再インストールしてください。

### インストール失敗の報告

インストール中にエラーが発生した場合は、類似のエラーレポートがないか確認するか、以下の方法で報告してください：

- [GitHub Issues](https://github.com/Comfy-Org/desktop/issues)
- [Comfy 公式フォーラム](https://forum.comfy.org/)

エラーレポートを送信する際は、以下のログと設定ファイルを必ず含めてください：

1. **ログファイル**

| ファイル名 | 説明 | 場所 |
|-----------|------|------|
| main.log | Electron プロセスのデスクトップアプリケーションとサーバー起動に関するログ | ~/Library/Application Support/Comfy Desktop/logs |
| comfyui.log | ComfyUI の通常動作に関するログ（コア ComfyUI プロセスの端末出力を含む） | ~/Library/Application Support/Comfy Desktop/logs |

![Comfy Desktop ログファイルの場所](/images/desktop/win-comfyui-desktop-10-logs.jpg)

2. **設定ファイル**

| ファイル名 | 説明 | 場所 |
|-----------|------|------|
| extra_model_paths.yaml | ComfyUI がモデルとカスタムノードを検索する追加パスを含む | ~/Library/Application Support/Comfy Desktop |
| config.json | アプリケーション設定を含む。このファイルは直接編集しないでください | ~/Library/Application Support/Comfy Desktop |

![Comfy Desktop 設定ファイルの場所](/images/desktop/win-comfyui-desktop-11-config.jpg)
