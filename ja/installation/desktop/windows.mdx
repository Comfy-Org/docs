**Comfy Desktop** は複数の ComfyUI インスタンスを1つのランチャーから実行できるマルチインストールマネージャーです。このガイドでは、Windows に Comfy Desktop をインストールおよびアンインストールする方法を説明します。

## システム要件

- **OS:** Windows 10 以降
- **アーキテクチャ:** x64 または ARM64
- **GPU:** 良好なパフォーマンスのために、専用 GPU（NVIDIA / AMD）を推奨しますが、必須ではありません
- **ディスク容量:** インストールごとに最低 4.85 GB を推奨

<Card title="Comfy Desktop をダウンロード" icon="download" href="https://comfy.org/download">
    Windows 用の最新インストーラーをダウンロードしてください。ページが自動的にプラットフォームを検出します。
</Card>

## インストール

ダウンロードした `.exe` インストーラー（NSIS インストーラー）を実行し、画面の指示に従ってインストールします。インストール後、スタートメニューまたはデスクトップのショートカットから Comfy Desktop を起動します。

![Windows setup](/images/desktop/installation/windows_1_install.png)
初めてアプリを開くと、最初のインストールを作成するための **Welcome** 画面が表示されます。

## アンインストール

### Comfy Desktop の削除

1. **設定 → アプリ → インストールされているアプリ** に移動
2. **Comfy Desktop** を見つけて **アンインストール** をクリック

![Windows で Comfy Desktop をアンインストール](/images/desktop/installation/windows_2_uninstall.png)

### アンインストールで削除されないもの

Comfy Desktop のアンインストールはランチャーアプリ自体のみを削除します。以下のデータは**削除されません**。完全にクリーンアップするには手動で削除が必要です：

| データ | 場所 | 説明 |
|------|------|------|
| インストール | `%USERPROFILE%\ComfyUI-Installs` | 作成した各 ComfyUI インストール（アプリ内からも削除可能）。カスタムインストールパスも削除されません。 |
| 共有モデルと出力 | `%USERPROFILE%\ComfyUI-Shared` | 共有モデルライブラリ、入出力ディレクトリ |
| アプリ設定とレジストリ | `%APPDATA%\Comfy Desktop` | 設定、インストール記録、ログ |

<Warning>上記のディレクトリには、あなたの**個人のComfyUIデータ**（モデル、出力、設定、インストール）が含まれています。これらはComfy Desktopアプリケーションファイルでは**ありません**。削除するとデータが完全に失われます。</Warning>

完全にすべてを削除するには：

1. **アプリデータフォルダを削除** – `%APPDATA%\Comfy Desktop`
2. **アプリ内から個別のインストールを削除**、または `%USERPROFILE%\ComfyUI-Installs` を手動で削除
3. **共有モデルライブラリ** `%USERPROFILE%\ComfyUI-Shared` を削除（ファイルが不要な場合のみ）
