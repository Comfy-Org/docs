**Comfy Desktop** は複数の ComfyUI インスタンスを1つのランチャーから実行できるマルチインストールマネージャーです。このガイドでは、Windows に Comfy Desktop をインストールおよびアンインストールする方法を説明します。

## システム要件

- **OS:** Windows 10 以降
- **アーキテクチャ:** x64 または ARM64
- **GPU:** 専用 GPU（NVIDIA / AMD）を推奨しますが、必須ではありません
- **ディスク容量:** インストールごとに最低 15 GB を推奨

<Card title="Comfy Desktop をダウンロード" icon="download" href="https://comfy.org/download">
    Windows 用の最新インストーラーをダウンロードしてください。ページが自動的にプラットフォームを検出します。
</Card>

## インストール

ダウンロードした `.exe` インストーラー（NSIS インストーラー）を実行し、画面の指示に従ってインストールします。インストール後、スタートメニューまたはデスクトップのショートカットから Comfy Desktop を起動します。

![Windows setup](/images/desktop/installation/windows_1_install.png)

初めてアプリを開くと、最初のインストールを作成するための **Welcome** 画面が表示されます。

## アンインストール

### Comfy Desktop の削除

**設定 → アプリ → アプリと機能** から Comfy Desktop を選択してアンインストールするか、スタートメニューで Comfy Desktop を右クリック → **アンインストール** を選択します。

### インストールデータの削除

Comfy Desktop とそのインストールはディスク上で別々に保存されています。完全にクリーンアップするには：

1. **アプリ内から個別のインストールを削除**（インストールカードを右クリック → 削除）、またはインストールフォルダを手動で削除
2. **アプリデータフォルダを削除** `%APPDATA%\Comfy Desktop`
