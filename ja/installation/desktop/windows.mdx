**ComfyUI Desktop** は、複数の ComfyUI インスタンスを一つのランチャーから実行できるマルチインストレーションマネージャーです。このガイドでは、Windows での ComfyUI Desktop のインストールと使用方法を説明します。

ComfyUI Desktop はオープンソースプロジェクトです。[GitHub](https://github.com/Comfy-Org/Comfy-Desktop) でソースコードを閲覧できます。

## システム要件

- **OS:** Windows 10 以降
- **GPU:** CUDA 対応 NVIDIA GPU
- **アーキテクチャ:** x64 または ARM64
- **ディスク容量:** 各インストールにつき最低 15 GB 推奨

## インストール

### 手順 1 — インストーラーを実行

ダウンロードした `.exe` インストーラーをダブルクリックします。インストーラーが ComfyUI Desktop をセットアップし、スタートメニューにショートカットを作成します。

### 手順 2 — 初回起動

スタートメニューから ComfyUI Desktop を開きます。初回起動時に**ようこそ**画面が表示されます。

**Get Started** をクリックして開始します。

## データ保存場所

| データ | Windows パス |
|------|-------------|
| アプリデータ（設定、インストール一覧） | `%APPDATA%\ComfyUI Desktop` |
| ログ | `%APPDATA%\ComfyUI Desktop\logs` |
| デフォルトインストールディレクトリ | セットアップ時にユーザーが選択 |

## トラブルシューティング

### アンチウイルスが `uv.exe` を警告

ComfyUI Desktop は Python 環境管理に [`uv`](https://github.com/astral-sh/uv) をバンドルしています。一部のアンチウイルス製品が `uv.exe` を誤検出することがあります。インストールが失敗する場合は、ComfyUI インストールディレクトリをアンチウイルスのホワイトリストに追加して再インストールしてください。
