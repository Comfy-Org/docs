**ComfyUI Desktop** は、複数の ComfyUI インスタンスを一つのランチャーから管理できるマルチインストレーションマネージャーです。このガイドでは、Windows への ComfyUI Desktop のインストールとアンインストールについて説明します。

## システム要件

- **OS:** Windows 10 以降
- **アーキテクチャ:** x64 または ARM64
- **GPU:** 専用 GPU（NVIDIA / AMD）を推奨しますが、必須ではありません
- **ディスク容量:** 各インストールにつき最低 15 GB 推奨

## インストール

ダウンロードした `.exe` インストーラー（NSIS インストーラー）を実行し、指示に従ってインストールします。インストール後、スタートメニューまたはデスクトップのショートカットから ComfyUI Desktop を起動します。

初回起動時に **Welcome** 画面が表示され、最初のインストールを作成できます。

## アンインストール

### ComfyUI Desktop のアンインストール

**設定 → アプリ → アプリと機能** から ComfyUI Desktop を見つけてアンインストールするか、スタートメニューで ComfyUI Desktop を右クリック → **アンインストール** を選択します。

### インストールデータの削除

ComfyUI Desktop とそのインストールはディスク上で分離されています。完全にクリーンアップするには：

1. **アプリ内から個別のインストールを削除**するか、インストールフォルダを手動で削除
2. **アプリデータフォルダ** `%APPDATA%\ComfyUI Desktop` を削除

## データ保存場所

| データ | Windows パス |
|------|-------------|
| アプリデータ（設定、インストール一覧） | `%APPDATA%\ComfyUI Desktop` |
| ログ | `%APPDATA%\ComfyUI Desktop\logs` |
| インストールディレクトリ | セットアップ時にユーザーが選択 |

## トラブルシューティング

### アンチウイルスによるブロック

ComfyUI Desktop は Python 環境管理に [`uv`](https://github.com/astral-sh/uv) を使用しています。一部のアンチウイルス製品が `uv.exe` を誤検出することがあります — これは **既知の誤検出** です。インストールに失敗した場合は、ComfyUI インストールディレクトリをアンチウイルスのホワイトリストに追加してください。

### ミラーエラー

ミラーが失敗した場合は、アプリ内のミラー設定を展開して、Python、PyPI、PyTorch 用のカスタムミラーを設定してください。
