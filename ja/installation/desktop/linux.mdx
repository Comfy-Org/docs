**Comfy Desktop** は、複数の ComfyUI インスタンスを一つのランチャーから管理できるマルチインストレーションマネージャーです。このガイドでは、Linux への Comfy Desktop のインストールとアンインストールについて説明します。

## システム要件

- **OS:** Debian 系ディストリビューション（Ubuntu 22.04+ 推奨）
- **GPU:** 専用 GPU（NVIDIA / AMD）を推奨しますが、必須ではありません
- **ディスク容量:** 各インストールにつき最低 15 GB 推奨

## インストール

### .deb パッケージ（Debian/Ubuntu）

```bash
sudo apt install ./comfyui-desktop_*.deb
```

### AppImage（その他のディストリビューション）

```bash
chmod +x ./comfyui-desktop-*.AppImage
./comfyui-desktop-*.AppImage --no-sandbox
```

インストール後、アプリケーションメニューから Comfy Desktop を起動します。初回起動時に **Welcome** 画面が表示され、最初のインストールを作成できます。

## アンインストール

### Comfy Desktop のアンインストール

**Debian/Ubuntu：**
```bash
sudo apt remove comfyui-desktop
```

**AppImage：** AppImage ファイルを削除するだけです。

### インストールデータの削除

Comfy Desktop とそのインストールはディスク上で分離されています。完全にクリーンアップするには：

1. **アプリ内から個別のインストールを削除**するか、インストールフォルダを手動で削除
2. **アプリデータと設定ディレクトリを削除：**

| データ | パス |
|------|------|
| 設定 | `~/.config/Comfy Desktop` |
| データ | `~/.local/share/Comfy Desktop` |
| キャッシュ | `~/.cache/Comfy Desktop` |
| 状態 | `~/.local/state/Comfy Desktop` |

## データ保存場所

Comfy Desktop は XDG Base Directory 規約に従います：

| データ | Linux パス |
|------|-----------|
| 設定 | `~/.config/Comfy Desktop` |
| データ | `~/.local/share/Comfy Desktop` |
| キャッシュ | `~/.cache/Comfy Desktop` |
| 状態 | `~/.local/state/Comfy Desktop` |
| インストールディレクトリ | セットアップ時にユーザーが選択 |

## トラブルシューティング

### AppImage には fuse2 が必要

AppImage が実行できない場合は、`fuse2` をインストールします：

```bash
sudo apt install fuse2
```

### ミラーエラー

ミラーが失敗した場合は、アプリ内のミラー設定を展開して、Python、PyPI、PyTorch 用のカスタムミラーを設定してください。
