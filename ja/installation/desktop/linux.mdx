**Comfy Desktop** は複数の ComfyUI インスタンスを1つのランチャーから実行できるマルチインストールマネージャーです。このガイドでは、Linux に Comfy Desktop をインストールおよびアンインストールする方法を説明します。

## システム要件

- **OS:** Debian 系ディストリビューション（Ubuntu 22.04+ 推奨）
- **GPU:** 専用 GPU（NVIDIA / AMD）を推奨しますが、必須ではありません
- **ディスク容量:** インストールごとに最低 15 GB を推奨

<Card title="Comfy Desktop をダウンロード" icon="download" href="https://comfy.org/download">
    Linux 用の最新パッケージをダウンロードしてください。ページが自動的にプラットフォームを検出し、適切なパッケージを提供します — Debian/Ubuntu には .deb、その他のディストリビューションには AppImage。
</Card>

## インストール

### .deb パッケージ (Debian/Ubuntu)

```bash
sudo apt install ./comfyui-desktop_*.deb
```

### AppImage (その他のディストリビューション)

```bash
chmod +x ./comfyui-desktop-*.AppImage
./comfyui-desktop-*.AppImage --no-sandbox
```

AppImage が実行できない場合は、`fuse2` をインストールしてください：

```bash
sudo apt install fuse2
```

インストール後、アプリケーションメニューから Comfy Desktop を起動します。アプリを開くと、最初のインストールを作成するための **Welcome** 画面が表示されます。

## アンインストール

### Comfy Desktop の削除

**Debian/Ubuntu：**
```bash
sudo apt remove comfyui-desktop
```

**AppImage：** AppImage ファイルを削除するだけです。

### インストールデータの削除

Comfy Desktop とそのインストールはディスク上で別々に保存されています。完全にクリーンアップするには：

1. **アプリ内から個別のインストールを削除**、またはインストールフォルダを手動で削除
2. **アプリデータと設定ディレクトリを削除：**

| データ | パス |
|------|------|
| 設定 | `~/.config/Comfy Desktop` |
| データ | `~/.local/share/Comfy Desktop` |
| キャッシュ | `~/.cache/Comfy Desktop` |
| 状態 | `~/.local/state/Comfy Desktop` |
