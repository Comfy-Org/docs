**Comfy Desktop** は、複数の ComfyUI インスタンスを一つのランチャーから管理できるマルチインストレーションマネージャーです。このガイドでは、Linux への Comfy Desktop のインストールとアンインストールについて説明します。

## システム要件

- **OS:** Debian 系ディストリビューション（Ubuntu 22.04+ 推奨）
- **GPU:** 専用 GPU（NVIDIA / AMD）を推奨しますが、必須ではありません
- **ディスク容量:** 各インストールにつき最低 15 GB 推奨

## インストール

### .deb パッケージ（Debian/Ubuntu）

```bash
sudo apt install ./comfyui-desktop_*.deb
```

### AppImage（その他のディストリビューション）

```bash
chmod +x ./comfyui-desktop-*.AppImage
./comfyui-desktop-*.AppImage --no-sandbox
```

インストール後、アプリケーションメニューから Comfy Desktop を起動します。初回起動時に **Welcome** 画面が表示され、最初のインストールを作成できます。

## アンインストール

### Comfy Desktop のアンインストール

**Debian/Ubuntu：**
```bash
sudo apt remove comfyui-desktop
```

**AppImage：** AppImage ファイルを削除するだけです。

### インストールデータの削除

Comfy Desktop とそのインストールはディスク上で分離されています。完全にクリーンアップするには：

1. **アプリ内から個別のインストールを削除**するか、インストールフォルダを手動で削除
2. **アプリデータと設定ディレクトリを削除：**

| データ | パス |
|------|------|
| 設定 | `~/.config/Comfy Desktop` |
| データ | `~/.local/share/Comfy Desktop` |
| キャッシュ | `~/.cache/Comfy Desktop` |
| 状態 | `~/.local/state/Comfy Desktop` |

## データ保存場所

Comfy Desktop は XDG Base Directory 規約に従います：

| データ | Linux パス |
|------|-----------|
| 設定 | `~/.config/Comfy Desktop` |
| データ | `~/.local/share/Comfy Desktop` |
| キャッシュ | `~/.cache/Comfy Desktop` |
| 状態 | `~/.local/state/Comfy Desktop` |
| インストールディレクトリ | セットアップ時にユーザーが選択 |

## トラブルシューティング

### AppImage には fuse2 が必要

AppImage が実行できない場合は、`fuse2` をインストールします：

```bash
sudo apt install fuse2
```

### ミラーエラー

ミラーが失敗した場合は、アプリ内のミラー設定を展開して、Python、PyPI、PyTorch 用のカスタムミラーを設定してください。

### エラーの特定

インストールに失敗した場合、以下の画面が表示されます

![Comfy Desktop インストール失敗](/images/desktop/win-comfyui-desktop-7.jpg)

以下の手順でエラーの原因を特定してください：
1. `ターミナルを表示` をクリックしてエラー出力を確認
2. `ログを開く` をクリックしてインストールログを確認
3. [Comfy 公式フォーラム](https://forum.comfy.org/) で類似のエラーレポートを検索
4. `再インストール` をクリックして再試行

フィードバックを送信する前に、**エラー出力** と **ログファイル** を GPT などの AI ツールに提供することをお勧めします

![Comfy Desktop インストール失敗 - エラーログ](/images/desktop/win-comfyui-desktop-8.jpg)
![Comfy Desktop インストール失敗 - GPT フィードバック](/images/desktop/win-comfyui-desktop-9.jpg)

上記のように、GPT にエラーの原因を分析させるか、Comfy Desktop を完全にアンインストールして再インストールしてください。

### インストール失敗の報告

インストール中にエラーが発生した場合は、類似のエラーレポートがないか確認するか、以下の方法で報告してください：

- [GitHub Issues](https://github.com/Comfy-Org/desktop/issues)
- [Comfy 公式フォーラム](https://forum.comfy.org/)

エラーレポートを送信する際は、以下のログと設定ファイルを必ず含めてください：

1. **ログファイル**

| ファイル名 | 説明 | 場所 |
|-----------|------|------|
| main.log | Electron プロセスのデスクトップアプリケーションとサーバー起動に関するログ | ~/.config/Comfy Desktop/logs |
| comfyui.log | ComfyUI の通常動作に関するログ（コア ComfyUI プロセスの端末出力を含む） | ~/.config/Comfy Desktop/logs |

![Comfy Desktop ログファイルの場所](/images/desktop/win-comfyui-desktop-10-logs.jpg)

2. **設定ファイル**

| ファイル名 | 説明 | 場所 |
|-----------|------|------|
| extra_model_paths.yaml | ComfyUI がモデルとカスタムノードを検索する追加パスを含む | ~/.config/Comfy Desktop |
| config.json | アプリケーション設定を含む。このファイルは直接編集しないでください | ~/.config/Comfy Desktop |

![Comfy Desktop 設定ファイルの場所](/images/desktop/win-comfyui-desktop-11-config.jpg)
