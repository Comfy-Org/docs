**ComfyUI Desktop** は、複数の ComfyUI インスタンスを一つのランチャーから実行できるマルチインストレーションマネージャーです。このガイドでは、Linux での ComfyUI Desktop のインストール方法を説明します。

ComfyUI Desktop はオープンソースプロジェクトです。[GitHub](https://github.com/Comfy-Org/Comfy-Desktop) でソースコードを閲覧できます。

## システム要件

- **OS:** Ubuntu 22.04+（Debian 系ディストリビューション）
- **GPU:** CUDA 対応 NVIDIA GPU（推奨）
- **ディスク容量:** 各インストールにつき最低 15 GB 推奨

## ダウンロード

[公式ダウンロードページ](https://comfy.org/download)から ComfyUI Desktop をダウンロードしてください。ページがプラットフォームを検出し、適切なパッケージを提供します。Debian/Ubuntu では .deb、その他のディストリビューションでは AppImage です。

### .deb でのインストール

```bash
sudo dpkg -i comfyui-desktop_*.deb
```

### AppImage でのインストール

```bash
chmod +x comfyui-desktop-*.AppImage
./comfyui-desktop-*.AppImage
```

## データ保存場所

ComfyUI Desktop は Linux で XDG Base Directory 規約に従います：

| データ | Linux パス |
|------|-----------|
| 設定 | `~/.config/comfyui-desktop` |
| データ | `~/.local/share/comfyui-desktop` |
| キャッシュ | `~/.cache/comfyui-desktop` |
| 状態 | `~/.local/state/comfyui-desktop` |
| デフォルトインストールディレクトリ | セットアップ時にユーザーが選択 |
