このガイドでは、ComfyUI Desktop インストール後の使用方法を説明します。

## Chooser ビュー

**Chooser** は ComfyUI Desktop を開いたときに表示されるメイン画面です。すべての ComfyUI インストールがカードのグリッドとして表示されます。

### インストールカード

各カードには以下が表示されます：
- **名前** — インストールに付けた名前
- **タイプアイコン** — ソースタイプ（Standalone、Cloud、Git など）
- **最終起動** — 最後に使用してからの経過時間
- **ステータス** — 実行中、エラー、更新利用可能などの表示

### フィルタリング

グリッド上部のフィルターチップでインストールを絞り込めます：

- **All（すべて）** — すべてのインストール
- **Local（ローカル）** — Standalone および Legacy Desktop インストールのみ
- **Cloud（クラウド）** — Comfy Cloud 接続のみ
- **Remote（リモート）** — リモートサーバー接続のみ

## スナップショット

ComfyUI Desktop は更新前に自動的に**スナップショット**を作成します。スナップショットは以下を記録します：
- ComfyUI バージョン（Git commit hash）
- インストール済みのカスタムノード（バージョン付き）
- Python パッケージ一覧

いつでも任意のスナップショットを復元でき、ComfyUI とカスタムノードを以前の状態にロールバックできます。

## アプリ内ターミナル

ComfyUI Desktop には ComfyUI Python 環境にアクセスするための内蔵ターミナルがあります：

1. タイトルバーのターミナルアイコン（⊞）をクリックして下部パネルを開く
2. **Terminal** タブをクリック
3. アクティブなインストールの Python 仮想環境が開きます

## キーボードショートカット

| ショートカット | 操作 |
|-------------|------|
| `Ctrl+R` / `Cmd+R` | 現在のビューを更新 |
| `Ctrl+W` / `Cmd+W` | 現在のパネルを閉じる |
| `Escape` | モーダルまたはオーバーレイを閉じる |
| `Ctrl+,` / `Cmd+,` | グローバル設定を開く |
