このガイドでは、Comfy Desktop インストール後の使用方法を説明します。

## Chooser ビュー

**Chooser** は Comfy Desktop を開いたときに表示されるメイン画面です。すべての ComfyUI インストールがカードのグリッドとして表示されます。

### インストールカード

各カードには以下が表示されます：
- **名前** — インストールに付けた名前
- **タイプアイコン** — ソースタイプ（Standalone、Cloud、Git など）
- **最終起動** — 最後に使用してからの経過時間
- **ステータス** — 実行中、エラー、更新利用可能などの表示

### フィルタリング

グリッド上部のフィルターチップでインストールを絞り込めます：

- **All（すべて）** — すべてのインストール
- **Local（ローカル）** — Standalone および Legacy Desktop インストールのみ
- **Cloud（クラウド）** — Comfy Cloud 接続のみ
- **Remote（リモート）** — リモートサーバー接続のみ

## 新規インストール

左上の **+** カードをクリックして新しい ComfyUI インストールを追加します。このガイドでは全手順を説明します。

### ステップ 1 — Comfy Desktop の設定

<Tabs>
  <Tab title="Windows">
    ![Windows で Comfy Desktop を設定](/images/desktop/usage/new_install_01_settings_windows.png)
  </Tab>
  <Tab title="macOS">
    ![macOS で Comfy Desktop を設定](/images/desktop/usage/new_install_01_settings_mac.png)
  </Tab>
</Tabs>

1. **① インスタンス名** — インスタンスの名前。デフォルトは "ComfyUI" です。
2. **② インストール場所** — ComfyUI のインストール先を選択します。Windows の場合は**SSD** を使用し、C ドライブのルートは避けることを強く推奨します。
3. **③ 詳細（Advanced）** — クリックして展開します。ここで選択できます：
   - **④ スタンドアロン（Standalone）** — 推奨オプション。Python バンドル、自動更新、スナップショット対応の自己完結型インストール。
   - **⑤ リモート接続（Remote Connection）** — 別のマシンで動作している ComfyUI に接続。

### ステップ 2 — インストール中

![ComfyUI をインストール中](/images/desktop/usage/new_install_02_installing.png)

Comfy Desktop が残りの作業（Python のダウンロード、環境構築、ComfyUI のインストール）を自動的に行います。進捗画面に現在のステップが表示されます。

### ステップ 3 — テンプレートを選択

![インストール完了後のテンプレートライブラリ](/images/desktop/usage/new_install_03_interface_template.png)

インストールが完了すると、Comfy Desktop がテンプレートライブラリを開きます。ここからワークフローテンプレートを選んで使い始められます。

## スナップショット

Comfy Desktop は更新前に自動的に**スナップショット**を作成します。スナップショットは以下を記録します：
- ComfyUI バージョン（Git commit hash）
- インストール済みのカスタムノード（バージョン付き）
- Python パッケージ一覧

いつでも任意のスナップショットを復元でき、ComfyUI とカスタムノードを以前の状態にロールバックできます。

## キーボードショートカット

| ショートカット | 操作 |
|-------------|------|
| `Ctrl+R` / `Cmd+R` | 現在のビューを更新 |
| `Ctrl+W` / `Cmd+W` | 現在のパネルを閉じる |
| `Escape` | モーダルまたはオーバーレイを閉じる |
| `Ctrl+,` / `Cmd+,` | グローバル設定を開く |
