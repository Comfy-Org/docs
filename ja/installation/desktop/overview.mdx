**ComfyUI Desktop** は、一箇所から複数の ComfyUI インスタンスをインストール、管理、起動できる次世代デスクトップアプリケーションです。従来の Desktop（シングルインストール）とは異なり、ComfyUI Desktop はマルチインストレーションマネージャーです。

## 主な機能

### マルチインストール管理

ComfyUI Desktop は以下の ComfyUI インストールタイプをサポートしています：

| ソースタイプ | 説明 |
|-------------|------|
| **Standalone** | バンドルされた Python と依存関係を含む自己完結型インストール |
| **Cloud** | Comfy Cloud に接続してリモート GPU を使用 |
| **Git Clone** | 既存の ComfyUI Git リポジトリを追跡 |
| **Portable** | 既存の ComfyUI ポータブルインストールをインポート (Windows) |
| **Legacy Desktop** | 既存の Desktop Legacy インストールを検出して移行 |
| **Remote Connection** | 別のマシンで実行中の ComfyUI インスタンスに接続 |

### できること

- **クイックインストール** — ワンクリックで新しい ComfyUI をインストール、Python 環境を自動設定
- **複数インスタンスの実行** — 異なるバージョンの ComfyUI を同時に起動・切り替え
- **スナップショット** — 更新前にカスタムノード、モデル、設定を自動保存、いつでも復元可能
- **ワンクリックアップデート** — クリック一つで ComfyUI を更新
- **モデルダウンロード管理** — 内蔵ダウンロードマネージャー
- **アプリ内ターミナル** — ComfyUI Python 環境に直接アクセス
- **グローバル設定** — テーマ、プロキシミラー、使用データを設定

## システム要件

### Windows
- **OS:** Windows 10 以降
- **GPU:** CUDA 対応 NVIDIA GPU
- **アーキテクチャ:** x64 または ARM64

### macOS
- **OS:** macOS 13 (Ventura) 以降
- **ハードウェア:** Apple Silicon (M1 以降)

### Linux
- **OS:** Ubuntu 22.04+（Debian 系ディストリビューション）
- **GPU:** CUDA 対応 NVIDIA GPU（推奨）

### 共通
- **ディスク容量:** 各インストールにつき最低 15 GB
- **RAM:** 最低 8 GB、推奨 16 GB
- **インターネット:** インストールと更新に必要

## オープンソース

ComfyUI Desktop は完全にオープンソースです。[GitHub](https://github.com/Comfy-Org/Comfy-Desktop) でソースコードを閲覧できます。
