アカウントシステムは `API Nodes` をサポートするために追加されました。`API Nodes` はクローズドソースモデル API の呼び出しを可能にし、ComfyUI の可能性を大幅に拡張します。これらの API 呼び出しはトークンを消費するため、対応するユーザーシステムを追加しました。

現在、以下のログイン方法をサポートしています：

- メールアドレスログイン
- Google ログイン
- Github ログイン
- API Key ログイン（ホワイトリストに登録されていないサイトの認証用）

ログイン要件及相关び説明については、このドキュメントで説明します。

## ComfyUI バージョン要件

アカウントシステムを使用するには、少なくとも [ComfyUI v0.3.0](https://github.com/comfyanonymous/ComfyUI/releases/tag/v0.3.30) が必要になる場合があります。対応するフロントエンドバージョンが少なくとも `1.17.11` であることを確認してください。場合によってはフロントエンドのインストールに失敗し、古いバージョンにロールバックすることがあるため、`設定` -> `バージョン情報` でフロントエンドバージョンが `1.17.11` より大きいか確認してください。

地域によっては、ネットワーク制限によりログイン API に正常にアクセスできず、タイムアウトや失敗が発生する可能性があります。ログイン前に、**ネットワーク環境が対応する API へのアクセスを制限していないことを確認し**、Google や Github などのサイトに正常にアクセスできることを確認してください。

<Tip>
私たちは依然として迅速な反復更新を行っているため、関連機能は変更される可能性があります。特別な事情がない限り、関連機能のサポートを取得するために最新バージョンに更新することをお勧めします。
</Tip>

## ネットワーク要件

ComfyUI アカウントにログインするには、安全なネットワーク環境にいる必要があります：

- `127.0.0.1` または `localhost` からのアクセスのみを許可します。
- `--listen` パラメータを使用してローカルネットワーク経由で API ノードにアクセスすることはサポートされていません。
- SSL 証明書を使用していない、または `https` で始まらないサイトでは、正常にログインできない可能性があります。
- ホワイトリストに登録されていないサイトではログインできない可能性があります（ただし、現在は API Key を使用してログインできます）。
- 当社のサービスに正常に接続できることを確認してください（地域によってはプロキシが必要な場合があります）。

![user-login](/images/interface/setting/user-login.jpg)

初めてログインする場合は、まずアカウントを作成してください。

<Card title="API Key でのログイン" icon="key" href="/ja/account/login#logging-in-with-an-api-key">
  ホワイトリストに登録されていないサイトや LAN 環境では、API Key ログインを使用できます。詳細なガイドはアカウントログインドキュメントをご覧ください。

## ログイン後の状態

ログイン後、ComfyUI インターフェースのトップメニューバーにログインボタンが表示されます。このボタンから対応するログインインターフェースを開き、設定メニューから対応するアカウントをログアウトできます。

## よくある質問

<AccordionGroup>
  <Accordion title="ログインデバイスに制限はありますか？">
   ログインデバイスは制限されていません。どのデバイスでもアカウントにログインできますが、アカウント情報が他のデバイスからアクセスされる可能性があるため、公共のデバイスではアカウントにログインしないでください。
  </Accordion>

  <Accordion title="LAN 環境ではどのようにログインしますか？">
  現在、LAN 環境では API Key ログインのみサポートされています。LAN 経由で ComfyUI サービスにアクセスしている場合は、API Key を使用してログインしてください。
  </Accordion>
   <Accordion title="一部のサイトでログインできないのはなぜですか？">
  ログインサービスにはホワイトリストがあるため、一部のサーバーにデプロイされた ComfyUI では正常にログインできない可能性があります。この場合、API Key ログインを使用して解決できます。
  </Accordion>
</AccordionGroup>
