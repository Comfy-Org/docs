マスクエディターは ComfyUI に組み込まれたツールで、外部画像編集ソフトを使わずに画像上で直接マスクを作成・編集できます。複数の描画ツール、レイヤー管理、アンドゥ/リドゥ、キャンバス変換、GPU アクセラレーションによるブラシ描画をサポートしています。

---

## 開き方

マスクエディターを開くには 3 つの方法があります：

1. **Selection Toolbox** — Load Image ノードを選択し、ノード上部の選択ツールバーにある Mask アイコンボタンをクリックします。
2. **Image Overlay** — 画像プレビュー（ギャラリーモード）にマウスを乗せ、左上に表示される **Edit or mask image** ボタン（mask アイコン）をクリックします。
3. **右クリックメニュー** — Load Image ノードを右クリックし、メニューから `Open in Mask Editor` を選択します。

> マスクエディターは **Graph Mode** と **App Mode** の両方で使用できます。

![マスクエディターを開く 3 つの方法：Selection Toolbox、Image Overlay、右クリックメニュー](/images/interface/maskeditor/maskeditor_3ways_to_open_maskeditor.png)

## インターフェース概要

![マスクエディターのメインインターフェース：ツールパネル、キャンバス、設定パネル](/images/interface/maskeditor/maskeditor_main_editor.jpg)

マスクエディターはモーダルダイアログとして開き、4 つの主要エリアで構成されています：

| # | エリア | 位置 | 説明 |
|---|-------|------|------|
| 1 | **Top Bar** | 上部 | アンドゥ/リドゥ、キャンバス変換、反転/クリア、保存/キャンセル |
| 2 | **Tool Panel** | 左側 | ツール選択とズーム情報 |
| — | **Pointer Zone** | 中央 | 描画・編集を行うメインキャンバスエリア |
| 3 | **Side Panel** | 右側 | ツール固有の設定とレイヤーコントロール |

---

## ツール

左側のツールパネルには 5 つのツールがあります。ツールを選択すると、右側パネルに対応する設定が表示されます。

<img src="/images/interface/maskeditor/maskeditor_tool_panel.jpg" alt="マスクエディターのツールパネル（5 つのツール番号付き）" style={{maxWidth: '300px'}} />

| # | ツール | レイヤー | 説明 |
|---|-------|---------|------|
| 1 | **Mask Pen** | Mask | マスク描画の主要ツール。左クリック＋ドラッグで画像上にマスクを描きます。 |
| 2 | **Paint Pen** | RGB (Paint) | 画像の RGB レイヤーに直接描画します。インペイントやベース画像の修正に便利です。 |
| 3 | **Eraser** | Mask / RGB | 既存のマスクやペイントレイヤーの一部を消去します。 |
| 4 | **Paint Bucket** | Mask | 色の類似性に基づく塗りつぶしツール。ピクセルをクリックすると接続領域にマスクを塗りつぶします。マスク済み領域をクリックするとマスクを消去します。Tolerance（許容値）で塗りつぶしの広がりを制御します。 |
| 5 | **Color Select** | Mask | ターゲット色に一致する全ピクセルを選択的にマスクします。高度なマッチングアルゴリズム（Simple HSL / LAB）に対応。 |

## TopBar 操作

![マスクエディター TopBar 操作（1–10 の番号付き）](/images/interface/maskeditor/maskeditor_topbar_actions.jpg)

上部バーではキャンバス全体に対する操作が行えます：

| # | 操作 | 説明 |
|---|------|------|
| 1 | **Undo** (Ctrl+Z) | 最後の操作を元に戻す |
| 2 | **Redo** (Ctrl+Shift+Z / Ctrl+Y) | 元に戻した操作をやり直す |
| 3 | **Rotate Left** | 全レイヤーを反時計回りに 90° 回転 |
| 4 | **Rotate Right** | 全レイヤーを時計回りに 90° 回転 |
| 5 | **Mirror Horizontal** | 全レイヤーを水平反転 |
| 6 | **Mirror Vertical** | 全レイヤーを垂直反転 |
| 7 | **Invert** | マスクを反転（黒 ↔ 白） |
| 8 | **Clear** | マスク全体をクリア |
| 9 | **Save** | マスクを保存してエディターを閉じる |
| 10 | **Cancel** | 保存せずにエディターを閉じる |

> **注**：回転と反転操作は 3 つのレイヤー（マスク、ペイント、ベース画像）すべてに同時に適用されます。変換後に GPU テクスチャが再作成されます。

## レイヤー

右側パネル下部の **Layers** セクションでは、各レイヤーの表示/非表示を個別に切り替えられます：

- **Mask Layer**（マスクレイヤー）：マスクオーバーレイ。**不透明度**と**ブレンドモード**を調整可能：
  - `Black` — マスクを黒のオーバーレイとして表示
  - `White` — マスクを白のオーバーレイとして表示
  - `Negative` — マスクを反転表示
- **Paint Layer**（ペイントレイヤー）：RGB ペイントレイヤー（Paint Pen 使用時に表示）
- **Base Image Layer**（ベース画像レイヤー）：元のソース画像

レイヤーの **Activate Layer** をクリックすると、そのレイヤーに描画フォーカスが切り替わります。

上のメインエディターのスクリーンショットを参照 — 右側パネルにはブラシ設定と下部のレイヤーセクションが含まれています。

## ブラシ設定

ブラシベースのツール（Mask Pen、Paint Pen、Eraser）がアクティブな場合、サイドパネルにブラシ設定が表示されます：

| 設定 | 範囲 | 説明 |
|------|------|------|
| **Shape** | Arc / Rect | 丸ブラシまたは四角ブラシ |
| **Color** | 任意の 16 進色 | ペイント色（Paint Pen 用）またはマスク色 |
| **Thickness** | 1–250 | ブラシ直径（ピクセル） |
| **Opacity** | 0–1 | ブラシストロークの透明度 |
| **Hardness** | 0–1 | エッジの柔らかさ。1 = 硬いエッジ、0 = 非常に柔らかい |
| **Step Size** | 1–100 | ブラシの打点間隔。値が低いほど滑らかなストローク |

**Reset to Default** でブラシパラメーターをデフォルトに戻せます。

## キャンバス操作とショートカット

| ショートカット | 操作 |
|--------------|------|
| **Space + ドラッグ** | キャンバスのパン |
| **Ctrl + スクロール** | ズームイン/アウト |
| **ズーム％をクリック** | ズームを 100% にリセット |
| **Ctrl + Z** | アンドゥ |
| **Ctrl + Shift + Z** または **Ctrl + Y** | リドゥ |
| **Alt + 右クリック + ドラッグ** | ブラシ調整：左右で**サイズ**、上下で**硬度**を変更 |

---

## 設定

ComfyUI の設定パネルで、以下のマスクエディターオプションを設定できます：

| 設定 | カテゴリー | デフォルト | 説明 |
|------|-----------|-----------|------|
| **Brush adjustment speed multiplier** | Mask Editor → Brush Adjustment → Sensitivity | 1.0 (0.1–2.0) | Alt+右クリックドラッグ時のブラシサイズ・硬度変更速度。値が大きいほど速い |
| **Lock brush adjustment to dominant axis** | Mask Editor → Brush Adjustment → Use Dominant Axis | オン | 有効にすると、移動の主軸方向に応じてサイズ**または**硬度のみが調整される |
