LiteGraph は ComfyUI の基盤となるグラフィックレンダリングエンジンです。このカテゴリの設定は、キャンバス、ノード、リンクなどのグラフィカルインターフェースの動作と外観を主に制御します。

## キャンバス

### 選択ツールボックスを表示
- **デフォルト値**: 有効
- **機能**: 選択ツールボックスは、ノードを選択した後にノード上に浮動表示されるクイックアクションツールバーで、部分的な実行、ピン留め、削除、色の変更などの一般的なクイック操作を提供します。

![選択ツールボックスを表示](/images/interface/setting/lite-graph/selection-toolbox.jpg)

### ズームノード詳細度 - フォントサイズ閾値
- **デフォルト値**: 8
- **範囲**: 0 - 24
- **機能**: ノードが低品質 LOD（詳細度）レンダリングに切り替わるタイミングを制御します。フォントサイズ（ピクセル）を使用して切り替えタイミングを決定します。0 に設定すると無効になります。値が大きい（24px）ほど、ズームアウト時にノードがより早く簡略化レンダリングに切り替わります。値が小さい（1px）ほど、完全なノード品質をより長く維持します。
- **置き換え**: 非推奨の「低品質レンダリングズーム閾値」設定

![ズームノード詳細度](/images/interface/setting/lite-graph/render-mode.jpg)

### 最大 FPS
- **デフォルト値**: 0（画面リフレッシュレートを使用）
- **範囲**: 0 - 120
- **機能**: キャンバスのレンダリングフレームレートを制限します。0 は画面のリフレッシュレートを使用することを意味します。FPS が高いほどキャンバスのレンダリングは滑らかになりますが、より多くのパフォーマンスを消費します。値が低すぎると、より顕著なスタッターが発生します。

### 常にグリッドにスナップ
- **デフォルト値**: 無効
- **機能**: このオプションが有効になっていない場合、`Shift` キーを押しながらノードの縁をグリッドに揃えることができます。有効にすると、`Shift` キーを押さなくてもノードの縁が自動的にグリッドに揃います。

</video>

### グリッドスナップサイズ
- **デフォルト値**: 10
- **範囲**: 1 - 500
- **機能**: 自動スナップが有効な場合、または `Shift` キーを押しながらノードを移動する場合、このパラメータはスナップ用のグリッドサイズを決定します。

### 高速ズームショートカットを有効化（Ctrl + Shift + ドラッグ）
- **デフォルト値**: 有効
- **機能**: `Ctrl + Shift + 左マウスボタンドラッグ` による高速ズーム機能を有効にし、より高速なズーム操作方法を提供します。

<video controls>
  <source src="/images/interface/setting/lite-graph/fast-zoom.mp4" type="video/mp4" />
</video>

### グラフキャンバスメニューを表示
- **デフォルト値**: 有効
- **機能**: 右下隅にキャンバスメニューを表示するかどうかを制御します。

キャンバスメニューは ComfyUI インターフェース全体の右下隅にあり、キャンバスのズーム、すべての接続の一時的な非表示、ワークフローをキャンバスに合わせるための高速スケーリングなどの操作が含まれています。

![グラフキャンバスメニューを表示](/images/interface/setting/lite-graph/canvas_menu.jpg)

### キャンバスズーム速度
- **デフォルト値**: 1.1
- **範囲**: 1.01 - 2.5
- **機能**: キャンバスのズーム速度を制御し、マウスホイールズームの感度を調整します。

### 自動パン速度
- **デフォルト値**: 15
- **範囲**: 0 - 30
- **機能**: キャンバス端へのドラッグによる自動パンの最大速度。0 に設定すると自動パンを無効にします。

### 左下隅にキャンバス情報を表示（FPS など）
- **デフォルト値**: 有効
- **機能**: 左下隅にキャンバス情報を表示するかどうかを制御し、FPS などのパフォーマンス指標を表示します。

![キャンバス情報](/images/interface/setting/lite-graph/canvas-info.jpg)

### ナビゲーションモード
- **デフォルト値**: レガシー
- **オプション**:
  - **標準（新）**: 左クリックでノードを選択、マウスホイールでキャンバスをスクロール
  - **ドラッグナビゲーション**: 左クリックでキャンバスをドラッグ（レガシー動作）
  - **カスタム**: 左クリック動作とマウスホイール動作を手動で設定
- **機能**: キャンバスの全体的なナビゲーションモードを制御します。この設定を変更すると、[左クリック動作](#左クリック動作)と[マウスホイールスクロール](#マウスホイールスクロール)の設定が自動的に調整されます。

### 左クリック動作
- **デフォルト値**: パン
- **オプション**: パン、選択
- **機能**: キャンバス背景での左クリックの動作を制御します。
  - **パン**: キャンバス背景で左クリック＆ドラッグするとビューを移動します。
  - **選択**: 左クリックでノードを選択、ドラッグで選択矩形を作成します。パンには中ボタンまたは右クリックを使用します。

### マウスホイールスクロール
- **デフォルト値**: ズーム
- **オプション**: パン、ズーム
- **機能**: キャンバス上でのマウスホイールの動作を制御します。
  - **パン**: マウスホイールでキャンバスを垂直にスクロールします。
  - **ズーム**: マウスホイールでキャンバスを拡大/縮小します。

### トラックパッドジェスチャー
- **デフォルト値**: 有効（標準ナビゲーションモード時）
- **機能**: ナビゲーションモードが**標準（新）**に設定されている場合、トラックパッドジェスチャー（2本指ピンチズームとドラッグ）が自動的に有効になります。これは独立した設定ではなく、ナビゲーションモードの選択から派生します。

### ライブ選択
- **デフォルト値**: 有効
- **機能**: 有効にすると、選択範囲をドラッグする際にノードがリアルタイムで選択/解除され、他のデザインツールと同様の操作感を提供します。

## コンテキストメニュー

### ズーム時にノードコンボウィジェットメニュー（リスト）を拡大
- **デフォルト値**: 無効
- **機能**: ズーム時にノードコンボウィジェットメニュー（リスト）を拡大するかどうかを制御します。

## グラフ

### リンクレンダリングモード
- **デフォルト値**: 2（スプライン）
- **オプション**: 直線、リニア、スプライン、非表示
- **機能**: 接続のレンダリングスタイルを設定し、ノード間のリンクの視覚スタイルを制御します。

![リンクレンダリングモード](/images/interface/setting/lite-graph/link-render-mode.jpg)

## グループ

このセクションの設定は主にノードグループ機能に関連しています。

![ノードグループ](/images/interface/setting/lite-graph/node-group.png)

### グループタイトルをダブルクリックして編集
- **デフォルト値**: 有効
- **機能**: ノードタイトルをダブルクリックして編集できるかどうかを制御し、ユーザーがノードの名前を変更できるようにします。画像の `1` としてマークされた部分です。

### 選択したノードのグループ化パディング
- **デフォルト値**: 10
- **範囲**: 0 - 100
- **機能**: 選択したノードをグループ化する際の内側パディングを設定し、グループフレームとノード間の間隔を制御します。画像の矢印注釈部分 `2` としてマークされています。

### クリック時にグループ子ノードを選択
- **デフォルト値**: 無効
- **機能**: 有効にすると、グループをクリックしたときに内部のすべてのノードとアイテムが選択されます。

## リンク

### リンク中点マーカー
- **デフォルト値**: 円
- **オプション**: なし、円、矢印
- **機能**: リンク中点のマーカーのスタイルを設定し、リンク中点に方向指示子を表示します。

![リンク中点マーカー](/images/interface/setting/lite-graph/link-midpoint.jpg)

### リンクリリース時のアクション（Shift）

## ノード

### 新しいノードを常に縮小
- **デフォルト値**: 無効
- **機能**: 新しいノードを作成する時に自動的に縮小するかどうかを制御します。これにより、ノードは常に最小サイズを表示できますが、追加時に一部のテキスト表示が切り捨てられる可能性があり、ノードサイズを手動で調整する必要があります。

### DOM 要素のクリッピングを有効化（有効化するとパフォーマンスが低下する可能性があります）
- **デフォルト値**: 有効
- **機能**: DOM 要素のクリッピングを有効化します（パフォーマンスに影響する可能性があります）。レンダリングを最適化しますが、パフォーマンスが低下する可能性があります。

### ミドルクリックで新しいリルートノードを作成
- **デフォルト値**: 有効
- **機能**: ミドルクリック時に新しいリルートノードを作成します。接続を整理するためのリルートノードを高速に作成します。

### ノード削除時にすべてのリンクを保持
- **デフォルト値**: 有効
- **機能**: 中間ノードを削除する時に接続を自動的にバイパスし、ノードを削除する時に入力および出力リンクの再接続を試みます。

### スナップ時にノードをハイライト
- **デフォルト値**: 有効
- **機能**: リンクをノードにドラッグする時にノードをハイライト表示し、視覚的なフィードバックを提供し、接続可能なノードを示します。

![スナップ時にノードをハイライト](/images/interface/setting/lite-graph/highlights-node.jpg)

### リンクをノードスロットに自動スナップ
- **デフォルト値**: 有効
- **機能**: リンクをノードにドラッグする時に利用可能なスロットに自動的にスナップし、接続操作を簡素化し、適切な入力スロットを自動的に見つけます。

<video controls>
  <source src="/images/interface/setting/lite-graph/snap-link-to-node-slot.mp4" type="video/mp4" />
</video>

### ツールチップを有効化
- **デフォルト値**: 有効
- **機能**: 一部のノード情報にはツールチップが含まれます（パラメータの説明など）。有効にすると、マウスをホバーした時にこれらのツールチップが表示されます。

![ツールチップを有効化](/images/interface/setting/lite-graph/tooltips.jpg)

### ツールチップ遅延
- **デフォルト値**: 500
- **範囲**: 100 - 3000
- **機能**: ツールチップの遅延時間を制御します（ミリ秒単位）。0 に設定すると、ツールチップが即座に表示されます。

### ノードライフサイクルバッジモード
- **デフォルト値**: すべて表示
- **オプション**: なし、すべて表示
- **機能**: ノードライフサイクルマーカーの表示を制御し、ノードのステータス情報（非推奨、実験的など）を表示します。

### ノード ID バッジモード
- **デフォルト値**: なし
- **オプション**: なし、すべて表示
- **機能**: ノード ID マーカーの表示を制御し、ノードの一意の識別子を表示します。

![ノード ID バッジモード](/images/interface/setting/lite-graph/node-id-badge.jpg)

### ノードソースバッジモード
- **オプション**:
  - なし：すべてのソースバッジを非表示
  - 内蔵を非表示：カスタムノードのバッジのみ表示
  - すべて表示：内蔵ノードとカスタムノードの両方のバッジを表示
- **デフォルト値**: 内蔵を非表示
- **機能**: ノードソースマーカーの表示モードを制御します。キツネのロゴは ComfyUI 内蔵ノードを表します。

![ノードソースバッジモード](/images/interface/setting/lite-graph/node-source-badge.jpg)

### ノードタイトルをダブルクリックして編集
- **デフォルト値**: 有効
- **機能**: ノードタイトルをダブルクリックして編集できるかどうかを制御し、ユーザーがノードの名前を変更できるようにします。

## ノードウィジェット

### 浮動ウィジェット丸め小数桁数 [0 = 自動]
- **デフォルト値**: 0（自動）
- **範囲**: 0 - 6
- **機能**: 浮動ウィジェット丸めの小数桁数を設定します。0 は自動を意味し、ページの再読み込みが必要です。

### デフォルトの浮動ウィジェット丸めを無効化
- **デフォルト値**: 無効
- **機能**: デフォルトの浮動ウィジェット丸めを無効にするかどうかを制御します。ページの再読み込みが必要であり、ノードバックエンドで丸めが設定されている場合は無効化できません。

### ノードウィジェットスライダーを無効化
- **デフォルト値**: 無効
- **機能**: ノードウィジェット内のスライダーコントロールを無効にするかどうかを制御し、スライダーの代わりにテキスト入力を強制します。

### プレビュー画像形式
- **デフォルト値**: 空文字列（元の形式を使用）
- **機能**: 画像ウィジェット内のプレビュー画像の形式を設定し、webp、jpeg などの軽量形式に変換します。

### 画像プレビュー下に幅×高さを表示
- **デフォルト値**: 有効
- **機能**: 画像プレビューの下に幅×高さの情報を表示し、画像の寸法情報を示します。

![画像プレビュー下に幅×高さを表示](/images/interface/setting/lite-graph/show-size.jpg)

## ポインター

### ダブルクリック間隔（最大）
- **デフォルト値**: 300
- **範囲**: 100 - 1000
- **機能**: ダブルクリックの 2 回クリック間の最大時間（ミリ秒）。この値を増やすと、ダブルクリックが時々認識されない問題を解決するのに役立ちます。

### ポインタークリックドリフト遅延
- **デフォルト値**: 32
- **範囲**: 0 - 1000
- **実験的**: はい
- **機能**: ポインターボタンを押した後、ポインターの移動を無視する最大時間（ミリ秒）。クリック時に誤ってマウスが移動するのを防ぐのに役立ちます。

### ポインタークリックドリフト（最大距離）
- **デフォルト値**: 6
- **範囲**: 0 - 20
- **実験的**: はい
- **機能**: ボタンを押している間にポインターがこの距離以上移動した場合、クリックではなくドラッグとみなされます。クリック時に誤ってマウスが移動するのを防ぐのに役立ちます。

## リルート

### リルートスプラインオフセット
- **デフォルト値**: 20
- **範囲**: 0 - 400
- **機能**: リルートノードの両側の曲線の滑らかさを決定するために使用されます。値が大きいほど曲線は滑らかになり、値が小さいほど曲線は鋭くなります。

![リルートスプラインオフセット](/images/interface/setting/lite-graph/reroute-spline-offset.jpg)

## Nodes 2.0（VueNodes）

### モダンノードデザイン（Nodes 2.0）
- **デフォルト値**: 無効（Comfy Cloud および Desktop の新規インストールではデフォルト有効）
- **実験的**: はい
- **機能**: モダン：DOM ベースのレンダリングで、強化されたインタラクティブ性、ネイティブブラウザ機能、更新されたビジュアルデザインを提供。クラシック：従来のキャンバスレンダリング。
