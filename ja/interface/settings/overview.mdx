このセクションでは、ComfyUI フロントエンド設定メニュー内の詳細な設定説明について解説します。すべてのユーザー設定は自動的に `ComfyUI/user/default/comfy.settings.json` ファイルに保存されます。

`Ctrl + ,` キーボードショートカットを使用して設定パネルを開き、対応する設定オプションをクリックして構成できます。

カスタムノードもメニューに対応する設定カテゴリを登録できるため、当公式ドキュメントでは現時点でネイティブ設定コンテンツのみを含んでいます。また、一部の設定オプションは **ComfyUI Desktop でのみ有効** ですので、該当ページで注記しています。

## ComfyUI 設定メニュー構造

設定パネルはサイドバー形式で 3 つの主要グループに整理されています。構造はチームワークスペースが有効かどうかによって異なります。

### チームワークスペースが有効な場合：

1. **ワークスペース**
   - ワークスペース設定（利用可能な場合）
   - クレジット

2. **一般**
   - ユーザー / プロフィール
   - **Comfy** — ComfyUI のコア設定（インターフェース、キュー、ノード、ワークフローなど）
   - シークレット
   - **キャンバス (Lite Graph)** — キャンバスレンダリング、ノード、リンク、グループなど
   - **外観** — テーマ、サイドバー位置、背景など
   - **3D** — 3D ノード初期化設定
   - **マスクエディター** — マスクエディターの設定
   - キーバインディング
   - 拡張機能
   - バージョン情報
   - サーバー設定（Desktop のみ）

3. **その他** — サードパーティのカスタムノードによって登録された設定（存在する場合のみ表示）

### チームワークスペースが無効な場合（レガシー）：

1. **アカウント**
   - ユーザー / プロフィール
   - サブスクリプション
   - シークレット
   - クレジット

2. **アプリケーション設定** — すべてのコア設定カテゴリー：
   - **Comfy**
   - **キャンバス (Lite Graph)**
   - **外観**
   - **3D**
   - **マスクエディター**
   - **その他**（未分類の設定）

3. **特別設定**
   - キーバインディング
   - 拡張機能
   - バージョン情報
   - サーバー設定（Desktop のみ）

## Desktop 限定設定

以下の設定は ComfyUI Desktop でのみ利用可能です：

- **サーバー設定** — サーバー構成設定へのアクセスと変更
- **Comfy Desktop 一般設定** — 更新設定、ウィンドウスタイル、ミラーソース

## Cloud 限定パネル

一部のパネルは ComfyUI アカウントにログインした場合のみ表示されます：

- **クレジット** — クレジットの購入と残高履歴の表示
- **シークレット** — API キーと機密情報の管理
- **ワークスペース** — チームワークスペースの設定
- **サブスクリプション** — サブスクリプションプランの管理
