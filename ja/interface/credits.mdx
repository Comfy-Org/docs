クレジットシステムは `Partner Nodes` をサポートするために追加されました。クローズドソースの AI モデルを呼び出すにはトークンの消費が必要なため、適切なクレジット管理が必要です。デフォルトでは、クレジットインターフェースは表示されません。まず `設定` -> `ユーザー` で ComfyUI アカウントにログインしてください。その後、`設定` -> `クレジット` で関連アカウントのクレジット情報を確認できます。

![ComfyUI クレジットインターフェース](/images/interface/setting/menu-credits.jpg)

<Note>
ComfyUI は常に完全オープンソースであり、ローカルユーザーにとっては無料であり続けます。
</Note>

## クレジットの購入方法

以下はクレジット購入のデモ動画です：

詳細な手順は以下の通りです：

<Steps>

<Step title="ComfyUI アカウントにログインする">
`設定` -> `ユーザー` で ComfyUI アカウントにログインします
![ログインインターフェース](/images/interface/setting/user.jpg)
</Step>

<Step title="`設定` -> `クレジット` に移動してクレジットを購入する">
ログイン後、メニューに `クレジット` オプションが追加されているのが確認できます

`設定` -> `クレジット` に移動してクレジットを購入します
![クレジットインターフェース](/images/interface/setting/purchase-1.jpg)
</Step>

<Step title="購入するクレジット量を設定する">
![購入金額を設定](/images/interface/setting/buy.jpg)
ポップアップで購入金額を設定し、`購入` ボタンをクリックします
</Step>

<Step title="Stripe で支払いを行う">
![Stripe 支払いページ](/images/interface/setting/stripe_payment.jpg)
支払いページで、以下の手順に従ってください：
1. 支払いに使用する通貨を選択する
2. メールアドレスが ComfyUI 登録時のメールアドレスと同じであることを確認する
3. 支払い方法を選択する
  - クレジットカード
  - WeChat（Comfy Credits での支払いを選択した場合のみサポート）
  - Alipay（Comfy Credits での支払いを選択した場合のみサポート）
4. `支払い` ボタンまたは `QR コードを生成` ボタンをクリックして支払いプロセスを完了する
</Step>

<Step title="支払いを完了し、クレジット残高を確認する">
![支払い成功](/images/interface/setting/purchase-2.jpg)
支払い完了後、`メニュー` -> `クレジット` に戻り、残高が更新されているか確認してください。必要に応じてインターフェースを更新するか、再起動してください
</Step>

</Steps>

## よくある質問

<AccordionGroup>
  <Accordion title="クレジットが負になることはありますか？">
    いいえ、クレジット残高が負の場合、Partner Nodes を実行することはできません
  </Accordion>

  <Accordion title="未使用のクレジットの返金は可能ですか？">
  現時点では、返金はサポートしていません
  </Accordion>
    <Accordion title="現在の残高と使用量を確認するにはどうすればよいですか？">
  `設定` -> `クレジット` をクリックすると、現在の残高を確認し、`クレジット履歴` エントリにアクセスできます
  </Accordion>

<Accordion title="クレジットを他のユーザーと共有できますか？">
  複数のデバイスで同じアカウントにログインすることはできますが、他のユーザーとクレジットを共有することはサポートしていません
  </Accordion>

  <Accordion title="毎回どのくらいのクレジットを消費したかを知るにはどうすればよいですか？">
  画像のサイズや生成数によって、毎回消費される `Tokens` と `Credits` は異なります。`設定` -> `クレジット` で、毎回消費されたクレジットと対応するクレジット履歴を確認できます
  </Accordion>

  <Accordion title="WeChat や Alipay の支払いオプションが表示されないのはなぜですか？">
  Comfy Credits で支払いを選択していることを確認してください。WeChat や Alipay は、Comfy Credits で支払いを行う場合のみサポートされています
  </Accordion>

  <Accordion title="クレジットは有効期限がありますか？">
  はい、有効期限はクレジットのタイプによって異なります：
  - **月度クレジット**：請求期間の終了時に有効期限が切れます
  - **チャージクレジット**：購入日から 1 年後に有効期限が切れます
  </Accordion>
</AccordionGroup>
