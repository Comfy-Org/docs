フレーム補間は既存のフレーム間に中間フレームを生成し、動きを滑らかにして高フレームレートの動画出力を実現します。ComfyUI は **FILM**（Google）と **RIFE**（リアルタイム中間フロー推定）の両モデルをネイティブでサポートしています。

<Img src="https://raw.githubusercontent.com/Comfy-Org/workflow_templates/main/templates/utility_video_frame_interpolation-1.webp" alt="フレーム補間ワークフローの比較" />

## ワークフロー概要

このワークフローは 3 つの主要ステップで構成されています：

1. **動画の読み込み** — 標準の LoadVideo ノードで動画ファイルをアップロード
2. **補間モデルの選択** — モデルサブグラフ設定で FILM または RIFE を選択
3. **補間結果の出力** — 滑らかな高フレームレート動画として保存

<Img src="https://raw.githubusercontent.com/Comfy-Org/workflow_templates/main/templates/utility_video_frame_interpolation-2.webp" alt="フレーム補間ワークフローノード" />

### サブグラフパラメータ

ワークフローはサブグラフを通じて以下の制御パラメータを公開しています：

- **model_name** — FILM または RIFE フレーム補間モデルを選択
- **multiplier** — フレームレート倍率（例：2倍でフレームレートが 2 倍に）
- **enable_fps_multiplier** — 元の FPS に基づいて倍率を適用するか切り替え

## モデルのダウンロード

モデルファイルをダウンロードし、`ComfyUI/models/frame_interpolation/` フォルダに配置します：

ダウンロードした .safetensors ファイルを以下の場所に配置：

```
📂 ComfyUI/
├── 📂 models/
│   └── 📂 frame_interpolation/
│       └── film_net_fp16.safetensors / rife_v4.25.safetensors
```

## 使用方法

1. **ワークフローを開く** — 上の「Comfy Cloud で実行」ボタンをクリックするか、JSON をダウンロードして ComfyUI にドラッグ
2. **動画を読み込む** — LoadVideo で入力動画を選択
3. **モデルを選択** — サブグラフのドロップダウンで FILM または RIFE を選択
4. **倍率を設定** — ほとんどの用途では 2 倍を推奨；より高い倍率は滑らかですが処理に時間がかかります
5. **生成** — ワークフローを実行し、出力動画をダウンロード
