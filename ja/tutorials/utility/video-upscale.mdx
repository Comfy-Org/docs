## 要約（TL;DR）

- **品質重視モード**：SeedVR2 および HitPaw が、最も正確かつ一貫性のあるリアリズムを実現します。
- **速度重視モード**：FlashVSR および HitPaw が最も高速なオプションです。
- **AI 動画修復**：Topaz Astra Creative は、映画的な仕上げや AI 生成動画の修復において最も優れた性能を発揮します。
- **アーカイブ修復**：SeedVR2 は元の構造を保ちつつ、明瞭度を向上させます。
- **CGI／3D レンダリング**：Topaz、SeedVR2、HitPaw のいずれも、クリーンでシャープな結果を生成します。

## 主な概念

### アップスケーリング vs. エンハンスメント

- **アップスケーリング**：解像度を高めながら、ディテールを再構築します。
- **エンハンスメント**：知覚される品質（ノイズ除去、シャープ化、修復、時間的整合性など）を向上させます。

動画アップスケーリングでは、フレーム間の一貫性を維持しつつ解像度を向上させるという「時間的要素」が加わります。

### クリエイティブアップスケーリング vs. コンサバティブアップスケーリング

| | コンサバティブ | クリエイティブ |
|---|---|---|
| **アプローチ** | 元の内容を忠実に保持 | 再構成・強化を試みる |
| **強み** | 高精度、時間的整合性、制作現場向けの信頼性 | 豊かなディテール、シャープさ、視覚的にインパクトのある表現 |
| **限界** | 平坦に見える場合があり、低品質な入力に対する改善効果が限定的 | ホールシネーション（幻覚）や時間的フリッカー（ちらつき）が発生する可能性あり |
| **モデル例** | Topaz Starlight Fast、FlashVSR、SeedVR2、HitPaw | Topaz Starlight Creative（Astra）、Wan 2.2、HitPaw |

## ユースケース別解説

### 品質重視モード

**SeedVR2** および **HitPaw** は、最も正確な結果を提供し、高い画質と優れたキャラクターの一貫性を実現します。

**Topaz** も効果的にアップスケーリングを行いますが、予想以上にクリエイティブな出力を生成することがあります。これは、スタイライズされた見た目が許容される映画系映像素材に特に適しています。

### 速度重視モード

**FlashVSR** は、処理速度を最優先する場合に非常に優れた性能を発揮します。720p から 1080p へのアップスケーリングは、10 秒間の動画で約 **41 秒** かかり、現在利用可能な中でも最速クラスの選択肢です。

**HitPaw** も比較的高速です。**Topaz** では、Fast モードと Creative モードのレンダリング速度はほぼ同等であるため、視覚的品質を損なわずより良い結果を得るために、通常は **Creative モード** の使用が推奨されます。

### AI 動画修復

**Topaz Astra Creative** は、非常に滑らかで視覚的に一貫性の高い出力を生成するため、AI 生成動画の修復に最適な選択肢です。

有用なワークフローのヒントとして、まず **修復またはクリーンアップ処理** を行い、次に **1080p へアップスケール**、最後に段階的に **4K へアップスケール** するという手順が挙げられます。このステップバイステップ方式は、安定性とディテールの両方を向上させる傾向があります。

## アップスケーリング速度比較（10 秒間の 720p 動画）

| モデル | 解像度 | 処理時間 |
|---|---|---|
| FlashVSR（ローカル） | 1080p | 約 41 秒 |
| FlashVSR（ローカル） | 4K | 約 52 秒 |
| HitPaw | 2K | 約 80 秒 |
| HitPaw | 4K | 約 175 秒 |
| SeedVR2（ローカル、RTX 5090） | 1080p | 約 312 秒 |
| Topaz Astra | 1080p | 約 374 秒 |
| Topaz Astra | 4K | 約 560 秒 |

## 利用可能なモデル

### パートナーノード

パートナーノードは、API 経由で高度な動画アップスケーリングモデルを利用するためのインターフェースを提供します。

| モデル | タイプ | 最大解像度 | 特長 |
|---|---|---|---|
| Topaz Starlight Fast | コンサバティブ | 4K | 高速かつ高精度なアップスケーリング |
| Topaz Starlight Creative（Astra） | クリエイティブ | 4K | 拡散モデルベースのクリエイティブアップスケーリング |
| Topaz Apollo | - | - | フレーム補間 |
| WaveSpeed SeedVR2 | コンサバティブ | - | 高忠実度再生 |
| WaveSpeed FlashVSR | コンサバティブ | - | 高速動画超解像 |
| HitPaw | 両方 | - | オールインワン動画エンハンスメント |

### Topaz Video Enhance の機能

- **アップスケーリング**：Starlight（Astra）は、FullHD（1080p）または 4K への高速動画アップスケーリングをサポートし、明瞭度を維持します。
- **補間**：Apollo モデルによるフレーム補間機能。スローモーション設定およびカスタムフレームレート（例：60fps）に対応。
- **ダイナミック圧縮処理**：圧縮アーティファクトを自動的に検出し、画像の純度を向上させるスマートな処理。

## Topaz Video Enhance の使い始め方

1. ComfyUI を最新バージョンに更新します。
2. ノードリスト内で `Topaz Video Enhance` ノードを検索します。
3. 動画入力を接続し、設定を構成します。
4. ワークフローを実行します。

## 便利なヒント

- アップスケーリングと[フレーム補間](/tutorials/utility/frame-interpolation)を組み合わせることで、より滑らかで高解像度の出力を得られます。
- AI 生成動画の場合、クリエイティブアップスケーリングに修復を依存するのではなく、アップスケーリング前にアーティファクトを修正することを検討してください。
- 実際のフル動画を処理する前に、サンプルクリップで複数のモデルをテストしてください。
- 元のコンテンツへの忠実度が厳密に求められる場合は、コンサバティブアップスケーリングをご利用ください。
