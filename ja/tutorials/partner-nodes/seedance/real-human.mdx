**Seedance 2.0 実人物モード** は ComfyUI 内で直接利用可能です。ComfyUI ワークフローを通じて肖像をアップロードし、一度だけの検証を完了すると、その実人物をフィーチャーした一貫性のある同一性と自然な表情の動画を生成できます — ComfyUI から離れることなく。

ComfyUI の実人物サポートにより、バーチャルキャラクターに限定されることはもうありません。Seedance 2.0 の制御可能性とネイティブ音声同期が実人物にも拡張され、同一性とパフォーマンスがクリップ全体で維持されます。

## 従来との違い

- **実人物、映画的制御** — カメラワーク、照明、シーン変更を指示しながら、被写体の信頼性を維持
- **同一性の一貫性** — 顔の特徴と全体的な外観がモーション、カメラ移動、トランジション全体で安定
- **ネイティブ音声同期** — 動画とオーディオ（会話、環境音、音楽）が一緒に生成され、リップシンクと表情がより信頼性向上
- **マルチリファレンスディレクティング** — テキストと最大 **9枚の画像**、**3つのビデオ**、**3つのオーディオ**を組み合わせて外観、モーション、リズムを固定

## ComfyUI での検証の仕組み

Seedance 2.0 の生成に実人物が登場する前に検証が必要です。これはなりすましや無断使用を防止し、ByteDance が処理し、進化する AI 透明性規制に準拠します。全体の流れは **ByteDance Create Image/Video Asset** ノードを通じて ComfyUI から駆動されます。

### 初回実行（検証必須）

1. **肖像をアップロード** — ComfyUI で **ByteDance Create Image/Video Asset** ノードを追加し、肖像画像またはビデオをアップロード
2. **ワークフローを実行** — ComfyUI がアセットを処理のために送信し、ByteDance からの検証リンクを返す
3. **ライブネスチェックを完了** — スマートフォンまたはブラウザでリンクを開き、ライブネスチェックを完了（通常1分以内）
4. **ID を保存** — 検証通過後、ノードが2つの ID を出力：
   - **Group ID** — 検証済みの本人を表す。後で同じ本人を再検証なしで再利用するために保存
   - **Asset ID** — この特定の画像/ビデオに紐づく。Seedance 2.0 動画生成ノードに接続して、検証済み本人の動画を生成

### 再アップロード（検証不要）

1. **ByteDance Create Image/Video Asset** ノードで同じ人の新しい写真またはビデオをアップロード
2. ノードに保存済みの **Group ID** を入力
3. 実行時に ByteDance が新しい顔データを初回検証記録と比較
4. 同一人物と確認されると、新しいアセットが自動的に有効化

## 利用可能なワークフロー

### リファレンスから動画 実人物モード

検証済みの肖像（オプションで追加の参照画像、ビデオ、オーディオ）を使用して、一貫した実人物同一性を持つ Seedance 2.0 動画を生成。

`ByteDance2ReferenceNode` の `asset_1` 入力は Asset ID 文字列を受け付けます。実人物 Asset ID を使用する場合、直接の `image` および `video` 入力は実人物をサポートしないため、Asset ID 入力を使用してください。

### 先頭・最終フレームから動画 実人物モード

検証済みの先頭フレームと最終フレームを提供し、その間の動画を生成しながら実人物の同一性を維持。

`ByteDance2FirstLastFrameNode` の `first_frame_asset_id` および `last_frame_asset_id` 入力は Asset ID 文字列を受け付けます。Asset ID を使用する場合、対応するフレームに直接画像入力を同時に使用することはできません。

## 重要な注意事項

- **Group ID** は単一アカウントで検証された場合、そのアカウントのみで使用可能。クロスアカウント使用はサポートされていません
- アセット作成に使用する画像/ビデオには**1人の人物のみ**が含まれていることを確認
- Asset ID は画像入力1つ分に相当します。先頭フレームに Asset ID を使用する場合、同時に画像入力を使用することはできません。最終フレームも同様
- アセット制限：画像 9 枚まで、ビデオ 3 つまで、オーディオ 3 つまで
