Seedance 2.0 リファレンスから動画ノード（`ByteDance2ReferenceNode`）は、参照画像、ビデオクリップ、オーディオを使用して、被写体の外観、モーション、リズムをガイドしながら動画を生成します。1つのワークフローで最大 **9枚の参照画像**、**3つの参照ビデオ**、**3つの参照オーディオ**を提供できます。

<UpdateReminder/>

## ワークフロー概要

リファレンスから動画ワークフローは、複数のロードノードから Seedance 2.0 リファレンスノードに入力します：

1. **LoadImage（×4）** — 参照画像を読み込み
2. **ByteDance2ReferenceNode** — 画像、オプションのビデオ/オーディオ入力、テキストプロンプトを受け取り VIDEO を出力
3. **SaveVideo** — 生成された動画をディスクに保存

## ノードパラメータ

| パラメータ | 説明 |
| :--- | :--- |
| `model` | モデル名。デフォルト：`Seedance 2.0` |
| `prompt` | テキスト説明。ショットタイミング、カメラ指示、服装説明を含む構造化プロンプトを推奨 |
| `resolution` | 出力解像度。オプション：`480p`、`720p`、`1080p` |
| `aspect_ratio` | アスペクト比。オプション：`16:9`、`9:16`、`1:1`、`4:5` |
| `duration` | 動画の長さ（秒） |
| `generate_audio` | 動画と同期したオーディオを生成するかどうか |
| `seed` | ランダムシード |
| `cfg` | Classifier-free guidance scale |

## 入力

| 入力 | タイプ | 最大 | 説明 |
| :--- | :--- | :-: | :--- |
| `image_1` 〜 `image_5` | IMAGE | 5 | 参照外観用の直接画像入力 |
| `video_1` | VIDEO | 1 | モーションガイド用の参照ビデオ入力 |
| `audio_1` | AUDIO | 1 | リズムと同期用の参照オーディオ入力 |
| `asset_1` | STRING | 1 | 検証済み実人物肖像の Asset ID |

## ワークフローの実行

1. **参照画像を読み込み** — LoadImage ノードで被写体の外観を定義する画像を読み込み。複数衣装のショーケースでは衣装ごとに1枚の画像を読み込み
2. **構造化プロンプトを記述** — 各セグメントのカメラアングル、トランジション、被写体のアクションを記述。`[00:00-00:02]` 形式のタイミング分割で精密な制御が可能
3. **パラメータを設定** — 解像度、アスペクト比、長さ、オーディオ生成を選択
4. **ワークフローを実行** — 「Queue」ボタンをクリックするか `Ctrl(Cmd) + Enter` を使用
5. **出力を確認** — SaveVideo ノードに生成された動画が表示

## ヒント

- **複数リファレンスを組み合わせる** — 画像で外観、ビデオでモーションパターン、オーディオでリズムを制御
- **構造化プロンプト** — 複雑なマルチシーン動画では、タイムスタンプ付きのショットごとのフォーマットが効果的
- **衣装の一貫性** — 製品ショーケースでは、プロンプトで各参照衣装と各時間セグメントでの可視性を明示的に記述
- **実人物の Asset ID** — 検証済みの実人物肖像には、直接画像入力の代わりに `asset_1` 文字列入力を使用
