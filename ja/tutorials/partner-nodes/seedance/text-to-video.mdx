Seedance 2.0 テキストから動画ノード（`ByteDance2TextToVideoNode`）は、テキスト説明から直接動画を生成します。Seedance 2.0 がプロンプトに基づいてシーン構成、モーション、カメラワーク、テンポを処理します。

<ReqHint/>
<UpdateReminder/>

## ワークフロー概要

テキストから動画ワークフローは最もシンプルな Seedance 2.0 ワークフローで、2つのノードのみで構成されます：

1. **ByteDance2TextToVideoNode** — テキストプロンプトを受け取り VIDEO を出力
2. **SaveVideo** — 生成された動画をディスクに保存

## ノードパラメータ

| パラメータ | 説明 |
| :--- | :--- |
| `model` | モデル名。デフォルト：`Seedance 2.0` |
| `prompt` | 動画のテキスト説明。カメラ指示、シーン詳細、タイミングを含む構造化プロンプトに対応 |
| `resolution` | 出力解像度。オプション：`480p`、`720p`、`1080p` |
| `aspect_ratio` | アスペクト比。オプション：`16:9`、`9:16`、`1:1`、`4:5` |
| `duration` | 動画の長さ（秒） |
| `generate_audio` | 動画と同期したオーディオを生成するかどうか |
| `seed` | 再現性のためのランダムシード |
| `cfg` | Classifier-free guidance scale |

## ワークフローの実行

1. **プロンプトを記述** — シーン、被写体、アクション、カメラワーク、雰囲気を詳細に記述。タイミングとカメラ指示を含む構造化フォーマットが効果的
2. **パラメータを設定** — 解像度、アスペクト比、長さ、オーディオ生成を選択
3. **ワークフローを実行** — 「Queue」ボタンをクリックするか `Ctrl(Cmd) + Enter` を使用
4. **出力を確認** — SaveVideo ノードに生成された動画が表示され、`ComfyUI/output/` に保存

## ヒント

- **詳細なプロンプト** ほど良い結果が得られます。被写体の外観、環境、照明、カメラワークを含めましょう
- マルチシーン動画では**ショットブレイクダウン形式**（各時間セグメントのカメラ、アクション、エフェクト）を使用
- **音声同期**はネイティブ — `generate_audio` を有効にすると同期音声付きの動画が生成されます
- **Seed** はノードの再実行のみに影響。再現可能な結果を得るには `fixed` を使用
