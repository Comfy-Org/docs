Seedance 2.0 は ByteDance の次世代マルチモーダル動画生成モデルで、ComfyUI で利用可能です。テキスト、画像、ビデオ、オーディオを統合入力として受け付け、同期オーディオ、一貫性のあるキャラクター、映画的なカメラモーションを備えた高品質な動画をワンパスで生成します。

<ReqHint/>
<UpdateReminder/>

## 主な機能

- **マルチモーダル入力** — テキストに画像、ビデオ、オーディオを組み合わせたプロンプト
- **音声同期** — 動画とオーディオを同時生成
- **ディレクティング制御** — カメラワークとショットのテンポを制御可能
- **一貫性** — キャラクターとシーンをクリップ全体で安定維持
- **編集と拡張** — 既存映像の編集や延長が可能

これらの機能により、Seedance 2.0 は広告、短編ドラマ、映画のプリビジュアライゼーション、製品紹介、バーチャルキャラクターコンテンツ、ゲーム・アニメーション用アセットなど幅広いユースケースに対応します。

## ノードリファレンス

| ノード | 説明 |
| :--- | :--- |
| `ByteDance2TextToVideoNode` | テキストからの動画生成 |
| `ByteDance2ReferenceNode` | 参照ベースの動画生成（画像、ビデオ、オーディオ） |
| `ByteDance2FirstLastFrameNode` | 先頭・最終フレーム補間 |
| `ByteDanceCreateImageAsset` | 実人物アセットとして画像をアップロード（検証用） |
| `ByteDanceCreateVideoAsset` | 実人物アセットとしてビデオをアップロード（検証用） |
