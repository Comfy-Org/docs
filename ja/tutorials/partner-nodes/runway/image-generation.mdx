Runway は、生成AIに特化した企業であり、強力な画像生成機能を提供しています。同社のモデルは、スタイル変換、画像の拡張、ディテール制御などの機能をサポートしています。現在、ComfyUI には Runway API が統合されており、関連ノードを ComfyUI 上で直接利用して画像生成を行うことができます。

本ガイドでは、以下のワークフローについて順に解説します：
- テキスト→画像
- 参照→画像

## Runway 画像：テキスト→画像 ワークフロー

### 1. ワークフローファイルのダウンロード

下記の画像には、`metadata` 内にワークフロー情報が含まれています。この画像をダウンロードし、ComfyUI へドラッグ＆ドロップすることで、対応するワークフローを読み込むことができます。

![ComfyUI Runway Image Text to Image](https://raw.githubusercontent.com/Comfy-Org/example_workflows/refs/heads/main/api_nodes/runway/image/text_to_image.png)

### 2. ワークフロー実行手順（ステップ・バイ・ステップ）

![ComfyUI Runway Image Text to Image Step Guide](/images/tutorial/api_nodes/runway/runway_text_to_image_step_guide.jpg)

画像中の番号を参考に、基本的なテキスト→画像ワークフローの実行を行ってください：  
1. `Runway Text to Image` ノードの `prompt` フィールドにプロンプトを入力します。
2. （任意）`ratio` 設定を調整して、出力画像のアスペクト比を変更できます。
3. `Run` ボタンをクリックするか、ショートカット `Ctrl（macOS の場合 Cmd）+ Enter` を使用して画像生成を実行します。
4. API からの結果を待った後、`Save Image` ノードで生成された画像を確認できます（右クリックで保存可能）。また、対応する画像は `ComfyUI/output/` ディレクトリにも自動保存されます。

### 2. ワークフロー実行手順（ステップ・バイ・ステップ）

![ComfyUI Runway Image Reference to Image Step Guide](/images/tutorial/api_nodes/runway/runway_reference_to_image_step_guide.jpg)

画像中の番号を参考に、基本的な参照→画像ワークフローの実行を行ってください：  
1. `Load Image` ノードで、提供された入力画像を読み込みます。
2. `Runway Text to Image` ノードの `prompt` フィールドにプロンプトを入力し、必要に応じて出力サイズを調整します。
3. `Run` ボタンをクリックするか、ショートカット `Ctrl（macOS の場合 Cmd）+ Enter` を使用して画像生成を実行します。
4. API からの結果を待った後、`Save Image` ノードで生成された画像を確認できます（右クリックで保存可能）。また、対応する画像は `ComfyUI/output/` ディレクトリにも自動保存されます。
