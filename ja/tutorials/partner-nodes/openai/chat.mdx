### 2. ワークフロー実行手順（ステップごと）

![OpenAI Chat Step Guide](/images/tutorial/api_nodes/openai/openai_chat_step_guide.jpg)

<Note>
対応するテンプレートでは、プロンプト生成の分析を目的としたロール設定を構築しています。
</Note>

画像中の番号を参考に、基本的なテキストから画像を生成するワークフローの実行を行ってください：  
1. `Load Image` ノードで、AI に解釈させたい画像を読み込みます  
2. （任意）必要に応じて、`OpenAI Chat Advanced Options` の設定を変更することで、AI に特定のタスクを実行させることができます  
3. `OpenAI Chat` ノードでは、対話のプロンプトを設定するため `Prompt` を編集したり、異なるモデルを選択するために `model` を変更したりできます  
4. `Run` ボタンをクリックするか、ショートカットキー `Ctrl（macOS の場合は Cmd）+ Enter` を押して対話を実行します  
5. API からの結果が返信されるのを待った後、`Preview Any` ノードで AI が返答した内容を確認できます  

### 3. 補足事項

- 現在、ファイル入力ノード `OpenAI Chat Input Files` では、ファイルを事前に `ComfyUI/input/` ディレクトリへアップロードする必要があります。このノードは現在改良中であり、更新後にテンプレートも修正する予定です  
- ワークフローには、入力として `Batch Images` を使用するサンプルが含まれています。複数の画像を AI に解釈させる必要がある場合、手順図を参考に、対象ノードを右クリックしてモードを `Always（常に）` に設定することで、その機能を有効化できます
