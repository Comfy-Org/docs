**Magnific プレサイス画像アップスケール (V2)** ノードは、忠実度を重視して画像を拡大します。元の特徴、テクスチャ、ディテールを保持しながら解像度を向上させるため、写真や写実的な画像に最適です。

### 実行手順

1. **Load Image** でアップスケールする画像を読み込む
2. **Magnific Image Upscale Precise V2** で **upscale_factor**（例：`2x`）を選択
3. 画像内容に合わせて **type**（`photo` / `art` / `general`）を選択
4. **resemblance**（元画像との類似度）と **resolution**（ディテール精細度）を調整
5. **sharpen** でエッジのシャープネスを設定
6. 必要に応じて **add_grain** を有効にし自然なフィルム調テクスチャを追加
7. **Run** をクリック、または `Ctrl(cmd) + Enter` で実行

### ウィジェット一覧

| ウィジェット | 型 | デフォルト | 説明 |
| :---------- | :- | :--------- | :--- |
| upscale_factor | ドロップダウン | `2x` | 拡大倍率（例：`2x`、`4x`） |
| type | ドロップダウン | `photo` | 処理を最適化する画像タイプ：`photo` / `art` / `general` |
| resemblance | 整数 | 7 | 元画像への忠実度（0–10） |
| resolution | 整数 | 7 | ディテール解像度の精細度（0–10） |
| sharpen | 整数 | 30 | シャープネス強度（0–100） |
| add_grain | ブール | true | 出力に自然なフィルムグレインを追加 |

### 入力

| 入力 | 型 | 説明 |
| :--- | :- | :--- |
| image | IMAGE | アップスケールする元画像 |

### 出力

| 出力 | 型 | 説明 |
| :--- | :- | :--- |
| IMAGE | IMAGE | プレサイスアップスケール後の出力画像 |

### 補足

- **type** — 写真は `photo`、イラスト/絵画は `art`、混合コンテンツは `general`
- **resemblance** — 高い値ほど元画像に忠実；低い値ほど AI の解釈を許容
- **add_grain** — 微妙なグレインテクスチャを追加し、アップスケール画像をより自然に見せる。特に写真に有用
- テンプレートの **ImageCompare** ノードで前後比較を並べてプレビュー可能
