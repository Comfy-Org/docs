**Magnific クリエイティブ画像アップスケール**ノードは、AI 生成のディテールを追加して画像を拡大します。単純な補間とは異なり、妥当なテクスチャ、シャープネス、ディテールを追加し、より自然な高解像度結果を生成します。

### 実行手順

1. **Load Image** でアップスケールする画像を読み込む
2. **Magnific Image Upscale Creative** で画像内容を説明する **prompt** を設定（ディテール生成のガイド用）
3. **upscale_factor**（最大 16 倍）を選択
4. **creativity** レベルを選択し、新たに追加するディテール量を制御
5. **resemblance**（元画像との類似度）、**hdr**、**sharpen**、**noise** を調整
6. 詳細設定の **preset** でシナリオに合ったプリセットを選択（`automatic` 推奨）
7. 必要に応じて **tune_color** を有効にし色補正
8. **Run** をクリック、または `Ctrl(cmd) + Enter` で実行

### ウィジェット一覧

| ウィジェット | 型 | デフォルト | 説明 |
| :---------- | :- | :--------- | :--- |
| prompt | 文字列 | — | AI のディテール生成をガイドする画像内容の説明 |
| upscale_factor | ドロップダウン | `4x` | 拡大倍率（例：`2x`、`4x`、`8x`、`16x`） |
| creativity | ドロップダウン | `standard` | 追加するクリエイティブディテール量（`low` / `standard` / `high`） |
| resemblance | 整数 | 2 | 元画像との類似度（0–10） |
| creativity_scale | 整数 | 3 | クリエイティブディテール注入の規模（0–10） |
| detail_boost | 整数 | 1 | ディテール強調度（0–10） |
| hdr | 整数 | 0 | HDR 効果の強度（0–10） |
| sharpen | 整数 | 0 | シャープネス強度（0–10） |
| noise | ドロップダウン | `automatic` | ノイズ低減モード |
| tune_color | ブール | true | 色調整を有効化 |

### 補足

- **prompt** — モデルが画像内容を理解し、コンテキストに合ったディテールを生成するのに役立つ
- **creativity** — `standard` は忠実度とディテール強調のバランス；`high` はより推測的なディテールを追加、スタイライズ画像に効果的
- **resemblance** — 高い値ほど元画像に近い結果；低い値ほどクリエイティブな再解釈を許容
- テンプレートの **ImageCompare** ノードで前後比較を並べてプレビュー可能
