**Magnific 画像スタイル転送**ノードは、参照画像のアートスタイル、カラーパレット、テクスチャ、構図の雰囲気をソース画像に転送します。テキストプロンプトで結果をガイドし、スタイル影響とコンテンツ保存のバランスを微調整できます。

### 実行手順

1. 最初の **Load Image** でスタイルを適用するソース画像を読み込む
2. 2 つ目の **Load Image** で参照画像（スタイルの元）を読み込む
3. **Magnific Image Style Transfer** で出力を説明する **prompt** を設定
4. **content_weight**（ソース画像の内容保持率）と **style_weight**（参照スタイルの適用強度）を調整
5. **style_fidelity**（`faithful` 忠実 / `creative` クリエイティブ）で参照スタイルの追従度を選択
6. **text_fidelity**（`balanced` バランス / `strict` 厳格）でプロンプト追従度を選択
7. 必要に応じて **preserve_faces** を有効にし顔の特徴を保持
8. **Run** をクリック、または `Ctrl(cmd) + Enter` で実行

### ウィジェット一覧

| ウィジェット | 型 | デフォルト | 説明 |
| :---------- | :- | :--------- | :--- |
| prompt | 文字列 | — | スタイル転送出力をガイドするテキスト説明 |
| content_weight | 整数 | 70 | ソース画像の内容を保持する度合い（0–100） |
| style_weight | 整数 | 41 | 参照画像のスタイルを適用する強度（0–100） |
| style_fidelity | ドロップダウン | `faithful` | `faithful` — 参照スタイルに忠実；`creative` — より芸術的な解釈 |
| text_fidelity | ドロップダウン | `balanced` | `balanced` — 適度なプロンプト追従；`strict` — より厳格な追従 |
| preserve_faces | ブール | false | 顔の特徴を変更せず保持するかどうか |
| use_original_color | ブール | true | ソース画像の元の色を維持 |

### 入力

| 入力 | 型 | 説明 |
| :--- | :- | :--- |
| image | IMAGE | スタイルを適用するソース画像 |
| reference_image | IMAGE | スタイルが転送される参照画像 |

### 出力

| 出力 | 型 | 説明 |
| :--- | :- | :--- |
| IMAGE | IMAGE | スタイル適用後の出力画像 |

### 補足

- **prompt** — アートの方向性を記述。例：「Fantasy portrait, retro anime hand-painted style, muted warm color palette, visible brushstroke texture」
- **content_weight vs style_weight** — content_weight が高いほど元の構造を保持；style_weight が高いほど参照スタイルが強く適用
- **style_fidelity** — 特定のアート参照に正確に合わせたい場合は `faithful`、より表現的な結果を得たい場合は `creative`
- **use_original_color** — 有効にすると、スタイルは形状とテクスチャに適用され、元の配色が保持される
