Magnific AI は、最先端のアップスケーリング、リライティング、スタイル転送、スキン補正機能で知られるプロフェッショナルな画像処理プラットフォームです。ComfyUI の Magnific パートナーノードを使用すると、API キーを管理したり ComfyUI を離れることなく、これらの機能をワークフローに直接統合できます。

<ReqHint/>
<UpdateReminder/>

## ノード

| ノード | 説明 |
| :--- | :--- |
| **画像リライティング** | テキストプロンプトで照明を調整。顔を保持。オプションの参照画像で光源方向/スタイルを適合可能。 |
| **スタイル転送** | 参照画像の視覚スタイルをソース画像に転送。スタイル影響度とプロンプトガイダンスの細かな制御が可能。 |
| **クリエイティブアップスケール** | 単純な補間以上のディテールを追加する AI アップスケーリング。スタイル/テクスチャガイダンス付きで最大 16 倍対応。 |
| **プレサイスアップスケール** | 忠実度重視の AI アップスケーリング。元の特徴、テクスチャ、ディテールを保存し、予期せぬアーティファクトを防止。 |
| **スキンエンハンサー** | 肌のキズ、シワ、不完全箇所を除去し顔の肌質を改善。クリエイティブ、フェイスフル、フレキシブルの 3 モード。 |

## 料金

Magnific ノードは **Freepik** 製品の一部として実行ごとに課金されます。詳細は[パートナーノード料金](/tutorials/partner-nodes/pricing#freepik)をご覧ください。
