**Magnific 画像リライティング**ノードは、テキストプロンプトに基づいて入力画像の照明を調整します。全体の照明方向、雰囲気、ムードを変更しながら、顔の特徴を保持します。

### 実行手順

1. **Load Image** でリライティングする画像を読み込む
2. （オプション）**参照画像**を `reference_image` 入力に接続して照明方向/スタイルを一致させる
3. **Magnific Image Relight** で希望の照明を説明する **prompt** を設定
4. **strength**（0–100）でリライティングの強さを調整
5. **mode**（`standard` 標準 / `creative` クリエイティブ）でリライティング方法を選択
6. **Run** をクリック、または `Ctrl(cmd) + Enter` で実行

### ウィジェット一覧

| ウィジェット | 型 | デフォルト | 説明 |
| :---------- | :- | :--------- | :--- |
| prompt | 文字列 | — | 希望する照明のテキスト説明（例：「暖かい自然光を左上から、柔らかな夕方のゴールデンアワーの照明」） |
| strength | 整数 | 75 | リライティング効果の強度（0–100） |
| mode | ドロップダウン | `standard` | `standard` — バランスの取れたリライティング；`creative` — より劇的な照明変化 |
| preserve_faces | ブール | false | 元の顔の外観を保持するかどうか |
| reference_strength | ブール | true | 参照画像の照明影響を有効にする |
| preserve_reference_colors | ブール | true | 参照画像の色を維持する |
| auto_crop | ドロップダウン | `disabled` | 入力画像のアスペクト比調整の自動クロップモード |

### 入力

| 入力 | 型 | 説明 |
| :--- | :- | :--- |
| image | IMAGE | リライティングする元画像 |
| reference_image | IMAGE（任意） | 照明スタイル/方向が結果に影響を与える画像 |

### 出力

| 出力 | 型 | 説明 |
| :--- | :- | :--- |
| IMAGE | IMAGE | リライティング後の出力画像 |

### 補足

- **prompt** — 光源の位置、色温度、影の柔らかさ、ムードを具体的に記述
- **reference_image** — 接続時、`reference_strength` と `preserve_reference_colors` で参照の影響を制御
- **preserve_faces** — 有効にすると顔の特徴を保持しつつ、画像の他の部分の照明のみを調整
