**Magnific スキンエンハンサー**ノードは、キズ、小ジワ、シワ、肌の不完全箇所を低減しながら、自然な外観を維持して顔の肌質を改善します。3 つの動作モード（クリエイティブ / フェイスフル / フレキシブル）を提供し、様々なレタッチニーズに対応します。

### 実行手順

1. **Load Image** で補正するポートレートまたはクローズアップ画像を読み込む
2. **Magnific Skin Enhancer** で **strength**（0–100）を調整し処理の強さを制御
3. **detail**（0–100）で保持する肌テクスチャのディテール量を設定
4. **mode**（`creative` / `faithful` / `flexible`）を選択
5. **Run** をクリック、または `Ctrl(cmd) + Enter` で実行
6. **ImageCompare** ノードで前後比較を確認

### ウィジェット一覧

| ウィジェット | 型 | デフォルト | 説明 |
| :---------- | :- | :--------- | :--- |
| strength | 整数 | 0 | スキン補正の適用強度（0–100）。低い値から始めて微妙な効果を試す |
| detail | 整数 | 15 | 保持する元の肌テクスチャディテール量（0–100）。高いほど自然な質感を維持 |
| mode | ドロップダウン | `creative` | `creative` — より積極的なスムージング；`faithful` — 最小限の自然な補正；`flexible` — 両者の中間 |

### 入力

| 入力 | 型 | 説明 |
| :--- | :- | :--- |
| image | IMAGE | 補正するポートレートまたはクローズアップ画像 |

### 出力

| 出力 | 型 | 説明 |
| :--- | :- | :--- |
| IMAGE | IMAGE | スキン補正後の出力画像 |

### 補足

- **strength** — 最初は低め（10–30）から始めて微妙なレタッチ効果を得る。強めの補正には値を上げる
- **detail** — 高い値ほど元の肌テクスチャを保持し、リアルな仕上がりに。低い値ほどスムーズでエアブラシ風に
- **mode**：
  - `creative` — ある程度のアーティスティックなスムージングが必要なポートレート写真に最適（最もコスト効率が良い）
  - `faithful` — 自然な肌構造の維持が重要なエディトリアルや商業作品に最適
  - `flexible` — クリエイティブとフェイスフルの中間
- 各モードの料金：`creative` 61.19 クレジット、`faithful` 78.07 クレジット、`flexible` 94.95 クレジット
