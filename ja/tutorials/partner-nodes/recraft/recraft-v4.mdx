Recraft V4 は、プロフェッショナルなデザイン作業向けに開発された新しい画像生成モデルです。デザイナーとの共同開発により、構図、ライティング、色の関係性、素材のリアリズムに最適化されています。

<ReqHint/>
<UpdateReminder/>

## V4 の新機能

- **ラスターグラフィックスとベクターグラフィックスを統合したモデル** — 同一モデルから、写真のような高精細画像、スタイライズされたイラスト、そして実用可能な SVG ベクター画像を生成可能
- **より優れたディテール表現** — よりクリアなテクスチャ、より正確なライティング、よりシャープなジオメトリ。複雑なシーンでも品質が維持されます
- **信頼性の高い顔・人体表現** — 多数の人物やアクションポーズを含む場合でも、比例や表情が自然で安定しています
- **テキストのレンダリング機能** — 看板、パッケージ、インフォグラフィックなどに適した、読みやすい文字を生成可能。短文から中程度の長さのフレーズまで対応します
- **SVG 出力機能** — 編集可能なベクター画像で、ラインがクリーンかつ色分割が適切です。実用レベルの高品質ベクター出力が可能です
- **プロンプトの忠実な再現** — 空間的関係性、素材、反射といった詳細なクリエイティブ指示を、正確に反映します
- **V4 Pro** — 同じモデルですが、解像度が向上。印刷物や大判出力向けの 2048×2048 で出力します

### ワークフローの実行手順

1. （任意）`Recraft Style` ノードを編集し、ビジュアルスタイルを制御します  
2. （任意）`Recraft Text to Image` ノード内の `prompt` パラメーターを編集します  
3. （任意）出力サイズを変更するには、`size` パラメーターを変更します  
4. `Run` ボタンをクリックするか、ショートカット `Ctrl（Cmd）+ Enter` を使用して画像を生成します  
5. API から結果が返された後、`Save Image` ノードで生成された画像を確認できます

## 補足情報

- **Recraft Style**：写真風、デジタルアート、ロゴデザインなど、さまざまなプリセットスタイルを提供します  
- **Seed パラメーター**：ノードを再実行するかどうかを判断するためだけに使用されます。実際の生成結果はシード値の影響を受けません
