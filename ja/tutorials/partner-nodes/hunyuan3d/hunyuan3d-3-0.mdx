Hunyuan 3D 3.0 は、テンセント社が開発した最先端の 3D 生成モデルであり、数分単位での高品質な 3D コンテンツ作成を実現します。テキストプロンプト、画像、またはスケッチから、実用レベルの 3D アセットを生成可能で、従来の 3D モデリングパイプラインに比べて、はるかに低い導入ハードルを実現しています。

ComfyUI は現在、対応する Hunyuan 3D API をネイティブ統合しており、関連ノードを ComfyUI 上で簡単に利用してモデル生成を行うことができます。

現在、ComfyUI のパートナーノードは以下の Hunyuan 3D モデル生成功能をサポートしています：
- テキストから 3D への生成（Text-to-3D）
- 画像から 3D への生成（Image-to-3D）
- 複数視点画像から 3D への生成（Multi-view-to-3D）

## 主なユースケース

- **ゲーム開発におけるアイデア創出**：リギングおよびリアルタイムレンダリングに最適化されたトポロジーを備えた、キャラクターやプロップ、環境の高速生成
- **製品・EC（電子商取引）用途**：製品写真をインタラクティブな 3D モデルに変換し、オンライン上で表示可能に
- **工業デザイン**：スケッチから 3D プロトタイピングを実施、および 3D プリンター対応の部品分解を実行
- **VFX（視覚効果）およびアニメーション**：プロフェッショナルな UV マップおよび PBR 材質を備えた高解像度アセットの生成

## クイックスタート

### 1. ワークフローテンプレートへのアクセス

1. サイドバー内の「テンプレートライブラリ」を開きます  
2. **3D** → **Hunyuan 3D** へとナビゲートします  
3. Hunyuan 3D 用のワークフローテンプレートを選択します  

### 2. ワークフローの実行

1. 使用するワークフローの種類に応じて、テキストプロンプト、画像、またはスケッチを入力します  
2. `Run` ボタンをクリックするか、ショートカットキー `Ctrl（Windows）／Cmd（macOS） + Enter` を使用してモデル生成を実行します  
3. ワークフローの実行が完了したら、生成された 3D モデルをプレビューおよびエクスポートできます  

## テキストから 3D へのワークフロー（Text-to-3D）

テキストによる説明を入力するだけで、ワンクリックで 3D モデルを生成します。スタイル、形状、マテリアルなどの詳細を正確に再現し、一貫性の高い結果を提供します。

**例としてのプロンプト**：「ギア付きのメカニカル・ドルフィン、ステampunk 風」

## 画像から 3D へのワークフロー（Image-to-3D）

1 枚以上の画像をアップロードすることで、高品質な 3D モデルを生成します。2～4 枚の多視点画像を入力すると、ジオメトリおよびマテリアルの忠実度が向上します。

## 複数視点画像から 3D へのワークフロー（Multi-view-to-3D）

正面・背面・側面など、複数の視点画像を提供することで、精度とディテールが向上した 3D モデルを生成します。この機能は「画像から 3D へのワークフロー」と同一のワークフローを使用しており、異なる角度から撮影した 2～4 枚の画像をアップロードするだけで利用可能です。

## 高度な機能

[初期の HY 3D 3.0 統合](https://blog.comfy.org/p/hunyuan-3d-30-in-comfyui-state-of) に続いて、Hunyuan 3D の高度な処理機能がパートナーノード経由で利用可能になりました。これらのワークフローは、重要なポストプロセッシング工程を ComfyUI 内に取り込むことで、生成から実際の制作までのギャップを埋め、実用的な制作フローを支援します。

### 3D パーツ分解

完成した 3D モデルを、装甲パーツ、アクセサリー、車輪など、意味のある構造的部品に分割します。これにより、アセットの特定領域を編集したり、バリエーション作成のために部品を交換したり、モジュラーなワークフロー、アニメーション、または下流の組立工程向けにモデルを準備することが容易になります。

### UV アンラップ

対応する 3D モデルに対して自動的に UV レイアウトを生成し、生のジオメトリをテクスチャリングが容易なアセットへと変換します。手動でシームをカットしたり UV アイランドを配置したりする必要がなく、クリーンな初期状態からペイント、ベイク、マテリアル作成といった作業へと迅速に移行できます。

### スマートトポロジー

高密度のジオメトリを、より整理されたエッジフローを持つクリーンなメッシュへと変換します。これにより、生成されたモデルを実際の制作パイプラインで最適化・再利用しやすくなります。特に、ゲームエンジンやリアルタイムレンダリング、あるいは低密度かつ構造的に優れたジオメトリを必要とするあらゆるワークフローにおいて有効です。
