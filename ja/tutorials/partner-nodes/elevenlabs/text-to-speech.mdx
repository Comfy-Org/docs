**ElevenLabsTextToSpeech** ノードは、ElevenLabs 音声モデルを使用してテキストを自然な音声に変換します。多言語音声と設定可能な出力フォーマットをサポートしています。

<ReqHint/>
<UpdateReminder/>

## 対応モデル

| モデル | 言語 |
| :---- | :-------- |
| `eleven_multilingual_v2` | 29言語対応の多言語対応 |
| `eleven_v3` | 英語最適化、高品質 |

### テキスト読み上げパラメーター

| パラメーター | 説明 |
| :-------- | :---------- |
| **text** | 音声に変換するテキスト内容 |
| **similarity** | 音声の類似度（0.0–1.0）、デフォルト 0.5 |
| **stability** | 音声の安定性（0.0–1.0）、デフォルト auto |
| **style_exaggeration** | スタイルの誇張度（auto / 0.0–1.0） |
| **speed** | 再生速度倍率 |
| **optimize_streaming_latency** | レイテンシ最適化レベル |
| **output_format** | オーディオ出力フォーマット（例: mp3_44100_192） |
| **seed** | 再現可能な出力のためのランダムシード |
| **seed_mode** | シードモード: randomize / fixed |

このワークフローには2つのオプションがあります：

1. **プリセット音声を使用** — **ElevenLabsVoiceSelector** ノードから音声を選択（例: "Roger (male, american)"）
2. **音声クローンを作成** — **LoadAudio** で音声ファイルを読み込み、**ElevenLabsInstantVoiceClone** に接続してクローン音声を使用

テンプレートを参考にする場合の一般的な実行手順：

1. **ElevenLabsVoiceSelector** でプリセット音声を選択するか、音声サンプルを提供して **ElevenLabsInstantVoiceClone** を有効化
2. **ElevenLabsTextToSpeech** ノードにテキストを入力
3. モデルを選択（`eleven_multilingual_v2` または `eleven_v3`）
4. 必要に応じて音声パラメーター（類似度、安定性、速度）を調整
5. **実行** をクリックするか `Ctrl(cmd) + Enter` を押す
6. 生成された音声は **SaveAudioMP3** ノードで保存

### 補足事項

- **ElevenLabsInstantVoiceClone** ノードはデフォルトでバイパスされています。音声クローンを使用する場合は `Ctrl-B` で有効化
- 出力フォーマットはデフォルトで `mp3_44100_192`（高品質 MP3）
- **seed** を `0` に設定するとシードを無視します。ほとんどのモデルではヒントとしてのみ扱われます
