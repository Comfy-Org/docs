[ElevenLabs](https://elevenlabs.io) は、主要な AI 音声・オーディオプラットフォームです。ComfyUI は **ElevenLabs** パートナーノードセットを通じて接続し、テキスト読み上げ、音声変換、対話生成、効果音生成、音声分離、音声認識機能をノードベースのワークフローで提供します。

<ReqHint/>
<UpdateReminder/>

## できること

- **テキスト読み上げ** — プリセット音声またはカスタム音声クローンを使用してテキストを自然な音声に変換
- **音声変換** — コンテンツを維持しながら既存の音声の声質やスタイルを変換
- **テキスト対話生成** — 各参加者に異なる音声を割り当ててマルチスピーカー対話を生成
- **テキスト効果音生成** — テキスト説明からカスタム効果音を生成
- **音声分離** — オーディオファイルから前景音声と背景ノイズを分離
- **音声認識** — 単語レベルのタイムスタンプと言語検出付きで音声をテキストに変換

## 主要ノード

| ノード | 説明 |
| :--- | :---------- |
| **ElevenLabsTextToSpeech** | ElevenLabs 音声モデルを使用してテキストを音声に変換 |
| **ElevenLabsSpeechToSpeech** | 入力音声の声質/スタイルを変換 |
| **ElevenLabsTextToDialogue** | 複数話者間の対話を生成 |
| **ElevenLabsTextToSoundEffects** | テキストプロンプトから効果音を作成 |
| **ElevenLabsAudioIsolation** | 背景ノイズを除去し音声トラックを分離 |
| **ElevenLabsSpeechToText** | 音声を単語レベルの詳細付きでテキストに変換 |
| **ElevenLabsVoiceSelector** | プリセット ElevenLabs 音声を選択 |
| **ElevenLabsInstantVoiceClone** | アップロードした音声から音声クローンを作成 |

料金については [パートナーノード料金](/ja/tutorials/partner-nodes/pricing#elevenlabs) をご覧ください。
