**ElevenLabsTextToDialogue** ノードは、各参加者に異なる音声を割り当ててマルチスピーカー対話オーディオを生成します。各話者に独自のテキスト入力と音声選択があります。

<UpdateReminder/>

## 対応モデル

| モデル | 説明 |
| :---- | :---------- |
| `eleven_v3` | 英語最適化の高品質対話生成 |

### テキスト対話生成パラメーター

| パラメーター | 説明 |
| :-------- | :---------- |
| **similarity** | 音声の類似度（0.0–1.0）、デフォルト 0.5 |
| **stability** | 音声の安定性（0.0–1.0）、デフォルト auto |
| **model** | 音声モデル選択 |
| **turn_count** | 対話のターン数（話者切り替え回数） |
| **text_1 / text_2** | 各話者のテキスト内容 |
| **output_format** | オーディオ出力フォーマット（例: mp3_44100_192） |
| **seed** | 再現可能な出力のためのランダムシード |
| **seed_mode** | シードモード: randomize / fixed |

このワークフローには2つのオプションがあります：

1. **プリセット音声を使用** — 2つの **ElevenLabsVoiceSelector** ノードから異なる音声を選択（例: "Roger (male, american)" と "Laura (female, american)"）
2. **音声クローンを作成** — 音声サンプルを読み込み **ElevenLabsInstantVoiceClone** を有効化

テンプレートを参考にする場合の一般的な実行手順：

1. **ElevenLabsVoiceSelector** ノードを使用して各話者の音声を選択
2. （オプション）**ElevenLabsInstantVoiceClone** を有効化してクローン音声を使用
3. **ElevenLabsTextToDialogue** ノードに各話者の対話テキストを入力
4. **turn_count** を設定して話者の切り替え回数を制御
5. **実行** をクリックするか `Ctrl(cmd) + Enter` を押す
6. 生成された対話音声は **SaveAudioMP3** で保存

### 補足事項

- 音声クローン部分はデフォルトでバイパスされています。`Ctrl-B` で有効化
- マルチターン対話では、ノードが自動的に2つの音声を交互に使用
- 各話者のテキストフィールドは独立した内容をサポートし、自然な会話の流れを実現
