**ElevenLabsTextToSoundEffects** ノードは、テキスト説明からカスタム効果音を生成します。ゲーム、動画、インタラクティブ体験向けのオーディオアセットを ComfyUI ワークフロー内で直接作成できます。

<ReqHint/>
<UpdateReminder/>

## 対応モデル

| モデル | 説明 |
| :---- | :---------- |
| `eleven_sfx_v2` | 効果音生成モデル |

### テキスト効果音生成パラメーター

| パラメーター | 説明 |
| :-------- | :---------- |
| **text** | 生成する効果音の説明 |
| **model** | 音生成モデル選択 |
| **duration_seconds** | 生成する効果音の長さ |
| **prompt_influence** | プロンプトへの追従度（0.0–1.0） |
| **output_format** | オーディオ出力フォーマット（例: mp3_44100_192） |

このワークフローはシンプルで焦点が絞られています：

1. **ElevenLabsTextToSoundEffects** に効果音の説明を入力（例: "Traffic noise with honking horns, busy highway"）
2. モデルを選択（`eleven_sfx_v2`）
3. 効果音の **duration_seconds** を設定
4. **prompt_influence** を調整して創造性とプロンプト追従のバランスを調整
5. **実行** をクリックするか `Ctrl(cmd) + Enter` を押す
6. 生成された音声は **SaveAudioMP3** で保存

### プロンプト例

| プロンプト | 主な用途 |
| :----- | :------- |
| "Traffic noise with honking horns, busy highway" | 都市環境のアンビエンス |
| "Rain on a window with distant thunder" | 雰囲気背景 |
| "Footsteps on wooden floor, slow and deliberate" | ゲーム/サウンドデザイン |
| "Door creaking open, suspenseful atmosphere" | 映画/動画制作 |

### 補足事項

- 長い時間の効果音はより多くのクレジットを消費します。1分あたりの料金は [ElevenLabs 料金ページ](/ja/tutorials/partner-nodes/pricing#elevenlabs) をご覧ください
- **prompt_influence** を 0.3 に設定すると、忠実度と多様性のバランスが取れます。低い値はより創造的な解釈を可能にします
- 効果音は生成されたオーディオの分数で課金されます
