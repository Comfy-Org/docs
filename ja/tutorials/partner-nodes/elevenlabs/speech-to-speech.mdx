**ElevenLabsSpeechToSpeech** ノードは、元の音声コンテンツを保持しながら、入力音声の声質や話し方を変換します。アクセント、音声特性、感情表現を変更できます。

<UpdateReminder/>

## 対応モデル

| モデル | 説明 |
| :---- | :---------- |
| `eleven_multilingual_sts_v2` | 多言語音声変換 |

### 音声変換パラメーター

| パラメーター | 説明 |
| :-------- | :---------- |
| **similarity** | 音声の類似度（0.0–1.0）、デフォルト 0.5 |
| **stability** | 音声の安定性（0.0–1.0）、デフォルト auto |
| **style_exaggeration** | スタイルの誇張度（auto / 0.0–1.0） |
| **speed** | 再生速度倍率 |
| **optimize_streaming_latency** | レイテンシ最適化レベル |
| **output_format** | オーディオ出力フォーマット（例: mp3_44100_192） |
| **seed** | 再現可能な出力のためのランダムシード |
| **seed_mode** | シードモード: randomize / fixed |
| **enable_logging** | ログの有効/無効 |

このワークフローには3つのセクションがあります：

1. **ソース音声** — **LoadAudio** で入力音声ファイルを読み込み
2. **プリセット音声を使用** — **ElevenLabsVoiceSelector** からターゲット音声を選択
3. **音声クローンを作成（オプション）** — 音声サンプルを読み込み **ElevenLabsInstantVoiceClone** を有効化

テンプレートを参考にする場合の一般的な実行手順：

1. **LoadAudio** でソース音声を読み込み（例: `api_elevenlabs_speech_to_speech.mp3`）
2. **ElevenLabsVoiceSelector** からターゲット音声を選択（例: "Jessica (female, american)"）
3. （オプション）**ElevenLabsInstantVoiceClone** を有効化してクローン音声を使用
4. ソース音声とターゲット音声を **ElevenLabsSpeechToSpeech** に接続
5. 必要に応じて音声パラメーター（類似度、安定性、速度）を調整
6. **実行** をクリックするか `Ctrl(cmd) + Enter` を押す
7. 変換された音声は **SaveAudioMP3** で保存

### 補足事項

- **ElevenLabsInstantVoiceClone** と関連する **LoadAudio** ノードはデフォルトでバイパスされています。`Ctrl-B` で有効化
