**Qwen 3.0**（Qwen3 とも呼ばれる）は、Alibaba Cloud の強力なオープンソース大規模言語モデルで、テキスト生成、推論、構造化分析に特化しています。ComfyUI の組み込み `TextGenerate` ノードでネイティブにサポートされています。

**モデルのハイライト**：

- **強力な推論能力** — Qwen3-4B-Thinking はステップバイステップの連鎖推論に対応
- **テキスト生成に特化** — 構造化テキスト出力、分析、コーディングタスクに最適化
- **ComfyUI ネイティブ** — 組み込みの `TextGenerate` ノードを使用、カスタムノード不要
- **軽量** — 4B パラメータモデル、コンシューマー GPU に適切

**関連リンク**：
- [Hugging Face (Qwen/Qwen3-4B)](https://huggingface.co/Qwen/Qwen3-4B)
- [Qwen 公式ブログ](https://qwenlm.github.io/blog/)
- [ComfyUI ソースコード (nodes_textgen.py)](https://github.com/Comfy-Org/ComfyUI/blob/master/comfy_extras/nodes_textgen.py)

![Qwen 3.0 テキスト生成ワークフロー](https://raw.githubusercontent.com/Comfy-Org/workflow_templates/main/templates/llm_qwen3_text_gen-1.webp)

このワークフローは Qwen 3.0 のコア**テキスト生成**機能を示します。テキストプロンプトを受け取り、モデルの推論能力を使用して詳細な構造化レスポンスを生成します。

**入力**：
- **テキストプロンプト** — 質問、指示、タスクの説明

**主要なコントロール**：
- **最大長** — 生成する最大トークン数（デフォルト 256）
- **サンプリングモード** — サンプリングのオン/オフ切り替え、温度、top-k、top-p、繰り返しペナルティ、シードの調整
- **思考モード** — 最終回答の前にステップバイステップの推論を有効化
- **デフォルトテンプレートを使用** — モデル内蔵のシステムプロンプトを適用

**出力**：
- **生成されたテキスト** — プレーンテキスト文字列

## モデルのダウンロード

Qwen 3.0 モデルは ComfyUI でテキストエンコーダーとして読み込まれます。該当するモデルファイルをダウンロードし、適切なディレクトリに配置してください：

<Card title="Qwen3 4B (bf16)" icon="download" href="https://huggingface.co/Comfy-Org/Qwen3.5/resolve/main/text_encoders/qwen3.5_4b_bf16.safetensors">
  Qwen3-4B-Thinking モデル。バランスの取れたサイズと品質で、コンシューマー GPU に適切。
</Card>

ダウンロードした `.safetensors` ファイルを以下の場所に配置：

```
📂 ComfyUI/
├── 📂 models/
│   └── 📂 text_encoders/
│       └── qwen3.5_4b_bf16.safetensors
```
