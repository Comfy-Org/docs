**Krea 2** は、[Krea AI](https://www.krea.ai) がゼロから学習したテキストから画像への基盤モデルで、高い美的品質とクリエイティブな制御を実現します。表現力豊かで芸術的なスタイルのレンダリングに優れており、コンセプトアート、ビジュアルブレインストーミング、スタイライズされたデザインに最適です。

Krea 2には2つのバリアントがあります：
- **Krea 2 RAW** — フルステップサンプリング（52ステップ）のベースモデル。多様性と可塑性に優れ、ファインチューニングやLoRAトレーニングに最適です。
- **Krea 2 Turbo** — 高速・高品質な生成のための8ステップ蒸留モデル。RAWでトレーニングしたLoRAをTurboにそのまま適用できます。

**モデルのハイライト**：
- **Artificial Analysis テキストから画像リーダーボードで第1位のオープンソースモデル**
- **2つのバリアント** — RAW（ファインチューニング可能なベース）とTurbo（高速推論）
- **内蔵スタイルLoRA** — warmpastel、coolblue、darkbrush、plasmoid
- **プロンプト拡張** — オプションのLLMによるプロンプト拡張機能
- **オプションのLoRAサポート** — 自動トリガーワード挿入機能

ワークフローは以下の部分で構成されています：

1. **Text to Image (Krea-2 Turbo) サブグラフ** — モデル読み込み、プロンプト処理、サンプリング、VAEデコードを含むコア生成パイプライン
2. **ResolutionSelector** — 出力解像度の選択
3. **CustomCombo (LoRAセレクター)** — 4つの内蔵スタイルLoRAから選択
4. **SaveImage** — 生成画像の保存

<Card title="サブグラフについて" icon="book-open" href="/ja/interface/features/subgraph">
  このワークフローはサブグラフノードを使用してモジュール化されています。サブグラフのドキュメントを確認して、ワークフローをカスタマイズおよび拡張する方法を学びましょう。
</Card>

### ワンクリック生成

最もシンプルな使い方：サブグラフにテキストプロンプトを入力し、解像度を選択して **Queue Prompt** をクリックするだけです。デフォルト設定（8ステップ、プロンプト拡張オン、LoRAなし）で高品質な画像が生成されます。

### ワークフローコントロール

**Text to Image (Krea-2 Turbo)** サブグラフは以下のコントロールを公開しています：

| コントロール | 説明 |
|------------|------|
| **Text String (User Prompt)** | 生成したい画像を説明するテキストプロンプト |
| **prompt_enhance** | LLMによるプロンプト拡張のオン/オフ |
| **LLM_max_token** | プロンプト拡張の最大トークン数 |
| **Width / Height** | 出力解像度（ResolutionSelectorで制御） |
| **Seed** | 再現性のためのランダムシード |
| **enable_lora?** | スタイルLoRAの有効/無効 |
| **LoRA Strength** | スタイルLoRAの適用強度 |
| **LoRA Trigger Word** | 選択したLoRAのトリガーワード（自動入力） |

### スタイル LoRA

Krea 2には4つの内蔵スタイルLoRAが含まれています。**CustomCombo** ノードで一つを選択し、サブグラフで `enable_lora?` を有効にするとスタイルが適用されます：

| LoRA | トリガーワード | 推奨強度 | ダウンロード |
|------|--------------|:--------:|------------|
| krea2_coolblue | teal watercolor illustration style | 0.8 | [ダウンロード](https://huggingface.co/Comfy-Org/Krea-2/blob/main/loras/krea2_coolblue.safetensors) |
| krea2_darkbrush | monochrome ink wash style | 1.0 | [ダウンロード](https://huggingface.co/Comfy-Org/Krea-2/blob/main/loras/krea2_darkbrush.safetensors) |
| krea2_plasmoid | ethereal shimmering light style | 0.8 | [ダウンロード](https://huggingface.co/Comfy-Org/Krea-2/blob/main/loras/krea2_plasmoid.safetensors) |
| krea2_warmpastel | muted minimalist sketch style | 0.8 | [ダウンロード](https://huggingface.co/Comfy-Org/Krea-2/blob/main/loras/krea2_warmpastel.safetensors) |

`.safetensors` ファイルを `ComfyUI/models/loras/` に配置してください。
