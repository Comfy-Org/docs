StringSubstring ノードは、長い文字列から一部のテキストを抽出します。抽出したい範囲を開始位置と終了位置で指定し、その2つの位置の間にあるテキストを返します。

## 入力

| パラメーター | データ型 | 必須 | 範囲 | 説明 |
|------------|----------|------|------|------|
| `string` | STRING | はい | - | 子文字列を抽出する対象となる入力テキスト文字列 |
| `start` | INT | はい | - | 子文字列の開始位置インデックス |
| `end` | INT | はい | - | 子文字列の終了位置インデックス |

## 出力

| 出力名 | データ型 | 説明 |
|--------|----------|------|
| `output` | STRING | 入力テキストから抽出された子文字列 |
