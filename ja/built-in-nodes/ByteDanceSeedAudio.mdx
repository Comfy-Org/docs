ByteDance Seed Audio 1.0 を使用して、単一のプロンプトから音声、音楽、効果音、および複数話者の対話を生成します。プロンプト内で音声、感情、雰囲気、背景音楽、効果音を説明し、発話する台詞を含めてください。オプションで、内蔵のプリセット音声を選択したり、最大3つの参照クリップ（プロンプト内で @Audio1-3 とタグ付け）から音声をクローンしたり、キャラクター画像から音声を派生させることができます。1回の実行で最大2分間の音声を生成できます。

## 入力

| パラメータ | 説明 | データ型 | 必須 | 範囲 |
|-----------|-------------|-----------|----------|-------|
| `text_prompt` | 音声、感情、ペース、雰囲気、背景音楽、効果音を説明し、発話する台詞を含めてください（対話の場合はキャラクター名をインラインで記述）。「音声参照」モードでは、接続されたクリップを @Audio1、@Audio2、@Audio3 の順序で参照します。最大3000文字。 | STRING | はい | 1～3000文字 |
| `reference_mode` | 音声を条件付ける方法：「text only」（すべてをプロンプトで説明）、「audio reference」（最大3つの音声をクローン、@Audio1-3 でタグ付け）、「image reference」（1つのキャラクター画像から音声を派生）、または「preset voice」（プロンプトを読み上げる内蔵の名前付き音声を選択）。 | COMBO | はい | `"text only"`<br />`"audio reference"`<br />`"image reference"`<br />`"preset voice"` |
| `reference_audio_1` | 音声クローン用の参照クリップ。プロンプト内で @Audio1 とタグ付け。最大30秒。 | AUDIO | いいえ | 最大30秒 |
| `reference_audio_2` | プロンプト内で @Audio2 とタグ付けされた参照クリップ。最大30秒。 | AUDIO | いいえ | 最大30秒 |
| `reference_audio_3` | プロンプト内で @Audio3 とタグ付けされた参照クリップ。最大30秒。 | AUDIO | いいえ | 最大30秒 |
| `reference_image` | 単一のキャラクター画像。モデルがそこから音声を派生させます。参照音声と組み合わせることはできません。 | IMAGE | いいえ | 単一画像 |
| `preset_voice` | プロンプトを読み上げる内蔵の TTS 2.0 音声。参照クリップは不要で、このモードでは @AudioN タグは使用されません。 | COMBO | いいえ | 複数のプリセット音声オプションから選択可能 |
| `sample_rate` | 出力サンプルレート（Hz）。（デフォルト："24000"） | COMBO | はい | `"8000"`<br />`"16000"`<br />`"24000"`<br />`"32000"`<br />`"44100"`<br />`"48000"` |
| `speech_rate` | 話速。0 = 通常、100 = 2.0倍、-50 = 0.5倍。（デフォルト：0） | INT | はい | -50～100 |
| `loudness_rate` | 音量。0 = 通常、100 = 2.0倍、-50 = 0.5倍。（デフォルト：0） | INT | はい | -50～100 |
| `pitch_rate` | ピッチシフト（半音単位、-12～12）。（デフォルト：0） | INT | はい | -12～12 |
| `seed` | シードはノードを再実行するかどうかを制御します。シードに関係なく結果は非決定的です。（デフォルト：42） | INT | はい | 0～2147483647 |

**パラメータの制約と依存関係：**

- `reference_mode` の選択によって、追加で必要なパラメータが決まります。
  - **"text only"**：参照入力は不要です。プロンプトに @AudioN タグを含めてはいけません。
  - **"audio reference"**：`reference_audio_1`、`reference_audio_2`、`reference_audio_3` のうち少なくとも1つを接続する必要があります。参照クリップは順序通りに、かつ隙間なく接続する必要があります（例：2つのクリップを使用する場合は、`reference_audio_1` と `reference_audio_2` を接続します）。各参照音声クリップの長さは最大30秒です。プロンプトでは、接続されたクリップを @Audio1、@Audio2、@Audio3 タグで順序通りに参照する必要があります。
  - **"image reference"**：`reference_image` を接続する必要があります。参照音声入力と組み合わせることはできません。プロンプトに @AudioN タグを含めてはいけません。
  - **"preset voice"**：`preset_voice` を選択する必要があります。参照クリップは不要です。プロンプトに @Audio1 以外の @AudioN タグを含めてはいけません。
