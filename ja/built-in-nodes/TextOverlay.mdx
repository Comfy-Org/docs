このノードは、画像または画像バッチの上にテキストを描画します。フォントサイズ、色、位置、配置、およびオプションの輪郭をカスタマイズ可能なテキストオーバーレイを作成し、元の画像にテキストを合成します。

## 入力

| パラメータ | 説明 | データ型 | 必須 | 範囲 |
|-----------|-------------|-----------|----------|-------|
| `画像` | テキストを描画する入力画像または画像バッチ | IMAGE | はい | |
| `テキスト` | 画像に重ねるテキスト（デフォルト：""） | STRING | はい | |
| `フォントサイズ` | 画像の高さに対するパーセンテージで指定するフォントサイズ（デフォルト：5.0） | FLOAT | はい | 0.5 ～ 50.0 |
| `色` | テキストの色（デフォルト："#ffffff"） | STRING | はい | |
| `位置` | 画像上のテキストの垂直位置（デフォルト："top"） | COMBO | はい | `"top"`<br />`"bottom"` |
| `整列` | テキストの水平配置（デフォルト："left"） | COMBO | はい | `"left"`<br />`"center"`<br />`"right"` |
| `アウトライン` | テキストの周りに黒い輪郭を描画するかどうか（デフォルト：True） | BOOLEAN | はい | |

## 出力

| 出力名 | 説明 | データ型 |
|-------------|-------------|-----------|
| `画像` | テキストオーバーレイが合成された入力画像 | IMAGE |
