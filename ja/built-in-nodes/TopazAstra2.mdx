Astra 2 は、GenAI 動画向けの拡散モデルベースのクリエイティブアップスケーリングを進化させたモデルで、Astra 1 をベースにディテール生成とスタイライズの制御を強化しています。AI 生成映像に新しいダイナミックなディテールやテクスチャを追加し、調整可能な創造性・シャープネスのコントロールとプロンプトによるガイドにより、より精密なクリエイティブ表現が可能です。

Astra 2 はスタイライズ素材や GenAI 素材、ワイドショットや大規模なシーンなど、追加のディテールを創造的に補完できる映像で最も性能を発揮します。クリアさ、豊かさ、視覚的な複雑さを向上させ、表現力豊かなスタイライズワークフローに最適化されています。

## 主な機能

- GenAI 素材にクリエイティブな新しいディテールとテクスチャを追加。
- 創造性・シャープネスの調整、プロンプトによるガイドなど拡張されたコントロール。
- スタイライズコンテンツ、ワイドショット、ディテールの少ないシーンで強力な性能。

- **モデル識別子**: `"ast-2"`

## モデル固有のパラメーター

| パラメーター | 型 | 必須 | 説明 |
|------------|-----|------|------|
| `creativity` | string | はい | `"low"`、`"middle"`、`"high"` のいずれか。クリエイティブなディテール生成の強度を制御します。 |
| `prompt` | string | いいえ | クリエイティブ方向を導くためのプロンプト。 |
| `input_frame_count` | integer | いいえ | プロンプトなしの場合は最大 9000、プロンプトありの場合は最大 450。 |
| `input_frame_rate` | integer | いいえ | 入力動画のフレームレート。 |
| `input_height` | integer | いいえ | 入力動画の高さ。 |
| `input_width` | integer | いいえ | 入力動画の幅。 |
| `output_height` | integer | いいえ | 出力高さ。 |
| `output_width` | integer | いいえ | 出力幅。 |
| `realism` | decimal | いいえ | `0.0` から `1.0` の値。 |
| `sharp` | decimal | いいえ | プリエンハンスのシャープネス。`0.0` から `1.0`、デフォルト `0.5`。`0.0` はガウスぼかし、`0.5` はパススルー、`1.0` は USM シャープニング。 |
