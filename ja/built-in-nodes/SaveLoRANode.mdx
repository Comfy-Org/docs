SaveLoRAノードは、LoRA（Low-Rank Adaptation）モデルを出力ディレクトリに保存します。入力としてLoRAモデルを受け取り、自動生成されたファイル名でsafetensorsファイルを作成します。ファイル名のプレフィックスをカスタマイズしたり、オプションでトレーニングステップ数をファイル名に含めて整理しやすくすることができます。

## 入力

| パラメータ | データ型 | 必須 | 範囲 | 説明 |
|-----------|-----------|----------|-------|-------------|
| `lora` | LORA_MODEL | はい | - | 保存するLoRAモデルです。LoRAレイヤーが適用されたモデルは使用しないでください。 |
| `prefix` | STRING | はい | - | 保存されるLoRAファイルに使用するプレフィックスです（デフォルト："loras/ComfyUI_trained_lora"）。 |
| `steps` | INT | いいえ | - | オプション：LoRAがトレーニングされたステップ数です。保存ファイルの命名に使用されます。 |

## 出力

| 出力名 | データ型 | 説明 |
|-------------|-----------|-------------|
| *なし* | - | このノードは出力を返しませんが、LoRAモデルを出力ディレクトリに保存します。 |
