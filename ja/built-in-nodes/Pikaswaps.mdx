Pika Swaps ノードを使用すると、動画内のオブジェクトまたは領域を新しい画像に置き換えることができます。置き換え対象の領域は、マスクまたは座標を用いて定義でき、このノードは指定されたコンテンツを動画全体にわたりシームレスに置き換えます。

## 入力

| パラメーター | データ型 | 必須 | 範囲 | 説明 |
|-----------|-----------|----------|-------|-------------|
| `video` | VIDEO | はい | - | オブジェクトを置き換える対象の動画です。 |
| `image` | IMAGE | はい | - | 動画内のマスク適用領域を置き換えるために使用する画像です。 |
| `mask` | MASK | はい | - | マスクを用いて、動画内で置き換える領域を定義します。 |
| `prompt_text` | STRING | はい | - | 望ましい置き換え効果を記述するテキストプロンプトです。 |
| `negative_prompt` | STRING | はい | - | 置き換え時に回避すべき内容を記述するテキストプロンプトです。 |
| `seed` | INT | はい | 0 ～ 4294967295 | 結果の一貫性を保つための乱数シード値です。 |

**注意:** このノードでは、すべての入力パラメーターの指定が必須です。`video`、`image`、および `mask` は、置き換え操作を共同で定義するために使用され、そのうち `mask` は、提供された画像で置き換える動画内の領域を特定します。

## 出力

| 出力名 | データ型 | 説明 |
|-------------|-----------|-------------|
| `output` | VIDEO | 指定されたオブジェクトまたは領域が置き換えられた、処理済みの動画です。 |
