Save Point Cloudノードは、3D点群ファイルをComfyUIの出力ディレクトリに保存します。また、点群データとオプションのカメラ情報およびモデル情報をワークフロー内の他のノードで使用できるように通過させます。

## 入力

| パラメータ | 説明 | データ型 | 必須 | 範囲 |
|-----------|-------------|-----------|----------|-------|
| `model_3d` | 点群ファイル (.ply) | FILE3D | はい | - |
| `filename_prefix` | 保存ファイル名のプレフィックス（デフォルト："3d/ComfyUI"） | STRING | はい | - |
| `viewport_state` | 3Dビューポートノードからの状態 | LOAD3D | はい | - |
| `model_3d_info` | 高度な使用のためのオプションの3Dモデル情報 | LOAD3DMODELINFO | いいえ | - |
| `camera_info` | 高度な使用のためのオプションのカメラ情報 | LOAD3DCAMERA | いいえ | - |
| `width` | プレビュー画像の幅（ピクセル単位、デフォルト：1024） | INT | はい | 最小：1、最大：4096、ステップ：1 |
| `height` | プレビュー画像の高さ（ピクセル単位、デフォルト：1024） | INT | はい | 最小：1、最大：4096、ステップ：1 |

## 出力

| 出力名 | 説明 | データ型 |
|-------------|-------------|-----------|
| `model_3d` | 保存された点群ファイル | FILE3D |
| `model_3d_info` | 3Dモデルに関する情報 | LOAD3DMODELINFO |
| `camera_info` | ビューポートのカメラ情報 | LOAD3DCAMERA |
| `width` | プレビュー画像の幅 | INT |
| `height` | プレビュー画像の高さ | INT |
