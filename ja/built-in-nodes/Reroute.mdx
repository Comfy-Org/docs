**ノード名：** ルートノード
**ノードの目的：** 主にComfyUIワークフロー内で、長すぎる接続線のロジックを整理するために使用されます。

## ルートノードの使用方法

| メニューオプション | 説明 |
| --- | --- |
| Show Type | ノードの型プロパティを表示します |
| Hide Type By Default | デフォルトでノードの型プロパティを非表示にします |
| Set Vertical | ノードの配線方向を垂直に設定します |
| Set Horizontal | ノードの配線方向を水平に設定します |

配線ロジックが長く複雑になり、インターフェースを整理したい場合、2つの接続ポイントの間に```Reroute```ノードを挿入できます。このノードの入力と出力は型に制限がなく、デフォルトのスタイルは水平です。右クリックメニューから配線方向を垂直に変更することもできます。
