このノードは、ガウシアンスプラット3Dファイルを出力ディレクトリに保存します。ファイル保存処理を実行し、3Dビューポート用のプレビューデータを提供します。

## 入力

| パラメータ | 説明 | データ型 | 必須 | 範囲 |
|-----------|-------------|-----------|----------|-------|
| `model_3d` | ガウシアンスプラット3Dファイルです。 | FILE3D | はい | SplatAny<br />PLY<br />SPLAT<br />SPZ<br />KSPLAT |
| `filename_prefix` | 保存ファイル名のプレフィックスです（デフォルト："3d/ComfyUI"）。 | STRING | はい | 任意の有効なファイル名プレフィックス |
| `viewport_state` | カメラとモデル情報を含む現在のビューポート状態です。 | LOAD3D | はい | - |
| `model_3d_info` | ビューポート用の追加のモデル3D情報です。 | LOAD3DMODELINFO | いいえ | - |
| `camera_info` | ビューポートプレビュー用のカメラ情報です。 | LOAD3DCAMERA | いいえ | - |
| `width` | プレビューの幅です（デフォルト：1024）。 | INT | はい | 1～4096 |
| `height` | プレビューの高さです（デフォルト：1024）。 | INT | はい | 1～4096 |

## 出力
