このノードは、Gaussian splat 3DファイルをComfyUIの出力ディレクトリに保存し、3Dビューア用のプレビューデータを提供します。

## 入力

| パラメータ | 説明 | データ型 | 必須 | 範囲 |
|-----------|-------------|-----------|----------|-------|
| `model_3d` | Gaussian splat 3Dファイルです。 | FILE3D | はい | - |
| `filename_prefix` | 保存されるファイル名のプレフィックス（デフォルト："3d/ComfyUI"） | STRING | はい | - |
| `viewport_state` | 3Dローダーノードからの現在のビューポート状態です。 | LOAD3D | はい | - |
| `model_3d_info` | プレビュー用の追加の3Dモデル情報です。 | LOAD3DMODELINFO | いいえ | - |
| `camera_info` | プレビュー用のカメラ情報です。 | LOAD3DCAMERA | いいえ | - |
| `width` | プレビューの幅（ピクセル単位、デフォルト：1024） | INT | はい | 最小：1、最大：4096、ステップ：1 |
| `height` | プレビューの高さ（ピクセル単位、デフォルト：1024） | INT | はい | 最小：1、最大：4096、ステップ：1 |

## 出力

| 出力名 | 説明 | データ型 |
|-------------|-------------|-----------|
| `model_3d` | 保存されたGaussian splat 3Dファイルです。 | FILE3D |
| `model_3d_info` | プレビュー用の3Dモデル情報です。 | LOAD3DMODELINFO |
| `camera_info` | プレビュー用のカメラ情報です。 | LOAD3DCAMERA |
| `width` | プレビューの幅です。 | INT |
| `height` | プレビューの高さです。 | INT |
