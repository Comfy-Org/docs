このノードは、3DモデルをComfyUIの出力ディレクトリに保存し、高度なプレビュー機能を提供します。出力ファイル名、プレビューサイズを指定でき、オプションでカメラ情報やモデル情報を渡すことで、拡張された3Dビューア体験が可能です。

## 入力

| パラメータ | 説明 | データ型 | 必須 | 範囲 |
|-----------|-------------|-----------|----------|-------|
| `model_3d` | 上流の3Dノードからの3Dモデルファイルです。 | FILE3D | はい | GLB、GLTF、FBX、OBJ、STL、USDZ、または任意の3D形式 |
| `filename_prefix` | 保存されるファイル名の接頭辞です。カウンターとファイル拡張子が自動的に追加されます。（デフォルト："3d/ComfyUI"） | STRING | はい | 任意の有効なファイル名文字列 |
| `viewport_state` | 現在のビューポート状態です。通常は3Dビューアノードから取得されます。 | LOAD3D | はい | - |
| `model_3d_info` | プレビュー用のオプションの3Dモデル情報です。 | LOAD3DMODELINFO | いいえ | - |
| `camera_info` | プレビュー用のオプションのカメラ情報です。 | LOAD3DCAMERA | いいえ | - |
| `width` | プレビューの幅（ピクセル単位）です。（デフォルト：1024） | INT | はい | 1 ～ 4096 |
| `height` | プレビューの高さ（ピクセル単位）です。（デフォルト：1024） | INT | はい | 1 ～ 4096 |

## 出力

| 出力名 | 説明 | データ型 |
|-------------|-------------|-----------|
| `model_3d` | 保存された3Dモデルファイルです。 | FILE3D |
| `model_3d_info` | 下流ノードに渡される3Dモデル情報です。 | LOAD3DMODELINFO |
| `camera_info` | 下流ノードに渡されるカメラ情報です。 | LOAD3DCAMERA |
| `width` | 下流ノードに渡される幅の値です。 | INT |
| `height` | 下流ノードに渡される高さの値です。 | INT |
