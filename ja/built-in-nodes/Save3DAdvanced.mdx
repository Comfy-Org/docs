このノードは、ComfyUIの出力ディレクトリに3Dモデルをファイルとして保存します。出力の寸法やカメラ/ビューポート設定を高度に制御できます。また、下流のノードで使用するために、3Dモデル、モデル情報、カメラ情報、寸法をそのまま通過させます。

## 入力

| パラメータ | 説明 | データ型 | 必須 | 範囲 |
|-----------|-------------|-----------|----------|-------|
| `model_3d` | 上流の3Dノードからの3Dモデルファイルです。 | FILE3D | はい | GLB, GLTF, FBX, OBJ, STL, USDZ, Any |
| `filename_prefix` | 保存するファイル名のプレフィックスです（デフォルト："3d/ComfyUI"）。 | STRING | はい | 自由テキスト |
| `viewport_state` | Load 3Dノードからのビューポート状態で、カメラとモデルの情報を含みます。 | LOAD3D | はい | - |
| `model_3d_info` | ビューポート状態を上書きするためのオプションの3Dモデル情報です。 | LOAD3DMODELINFO | いいえ | - |
| `camera_info` | ビューポート状態を上書きするためのオプションのカメラ情報です。 | LOAD3DCAMERA | いいえ | - |
| `width` | 出力プレビューの幅（ピクセル単位）です（デフォルト：1024）。 | INT | はい | 1 ～ 4096 |
| `height` | 出力プレビューの高さ（ピクセル単位）です（デフォルト：1024）。 | INT | はい | 1 ～ 4096 |

## 出力

| 出力名 | 説明 | データ型 |
|-------------|-------------|-----------|
| `model_3d` | 入力からそのまま通過した3Dモデルファイルです。 | FILE3D |
| `model_3d_info` | ビューポート状態またはオプションの入力からのモデル情報です。 | LOAD3DMODELINFO |
| `camera_info` | ビューポート状態またはオプションの入力からのカメラ情報です。 | LOAD3DCAMERA |
| `width` | 入力からそのまま通過した幅の値です。 | INT |
| `height` | 入力からそのまま通過した高さの値です。 | INT |
