ByteDance 画像編集ノード（ByteDance Image Edit Node）では、API を通じて字節跳動（ByteDance）の AI モデルを用いて画像を編集できます。入力画像と、望ましい変更内容を記述したテキストプロンプトを指定すると、ノードがその指示に従って画像を処理します。このノードは API との通信を自動的に処理し、編集後の画像を返します。

## 入力

| パラメーター | データ型 | 入力タイプ | デフォルト値 | 範囲 | 説明 |
|-----------|-----------|------------|---------|-------|-------------|
| `model` | MODEL | COMBO | seededit_3 | Image2ImageModelName の選択肢 | モデル名 |
| `image` | IMAGE | IMAGE | - | - | 編集対象となるベース画像 |
| `prompt` | STRING | STRING | "" | - | 画像編集のための指示文 |
| `seed` | INT | INT | 0 | 0–2147483647 | 生成時に使用するシード値 |
| `guidance_scale` | FLOAT | FLOAT | 5.5 | 1.0–10.0 | 数値が高いほど、画像がプロンプトに忠実に従うようになります |
| `watermark` | BOOLEAN | BOOLEAN | True | - | 画像に「AI 生成」の透かしを追加するかどうか |

## 出力

| 出力名 | データ型 | 説明 |
|-------------|-----------|-------------|
| `IMAGE` | IMAGE | 字節跳動（ByteDance）API から返される編集済み画像 |
