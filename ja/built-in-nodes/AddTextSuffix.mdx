このノードは、指定された接尾辞（サフィックス）を入力テキスト文字列の末尾に追加します。元のテキストと接尾辞を入力として受け取り、結合された結果を返します。

## 入力

| パラメーター | データ型 | 必須 | 範囲 | 説明 |
|-----------|-----------|----------|-------|-------------|
| `text` | STRING | はい | | 接尾辞を追加する元のテキストです。 |
| `suffix` | STRING | いいえ | | テキストに追加する接尾辞（デフォルト値：""）。 |

## 出力

| 出力名 | データ型 | 説明 |
|-------------|-----------|-------------|
| `text` | STRING | 接尾辞が追加された後の最終的なテキストです。 |
