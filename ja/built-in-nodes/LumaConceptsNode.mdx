Luma Text to Video および Luma Image to Video ノードで使用するための、1つ以上のカメラコンセプトを保持します。このノードでは、最大4つのカメラコンセプトを選択し、必要に応じて既存のコンセプトチェーンと組み合わせることができます。

## 入力

| パラメータ | データ型 | 必須 | 範囲 | 説明 |
|-----------|-----------|----------|-------|-------------|
| `concept1` | STRING | はい | 複数のオプションが利用可能<br />「None」オプションを含む | 利用可能なLumaコンセプトから最初のカメラコンセプトを選択します |
| `concept2` | STRING | はい | 複数のオプションが利用可能<br />「None」オプションを含む | 利用可能なLumaコンセプトから2番目のカメラコンセプトを選択します |
| `concept3` | STRING | はい | 複数のオプションが利用可能<br />「None」オプションを含む | 利用可能なLumaコンセプトから3番目のカメラコンセプトを選択します |
| `concept4` | STRING | はい | 複数のオプションが利用可能<br />「None」オプションを含む | 利用可能なLumaコンセプトから4番目のカメラコンセプトを選択します |
| `luma_concepts` | LUMA_CONCEPTS | いいえ | N/A | ここで選択したコンセプトに追加するオプションのカメラコンセプトです |

**注記：** すべてのコンセプトパラメータ（`concept1` から `concept4`）は、4つすべてのコンセプトスロットを使用したくない場合は「None」に設定できます。このノードは、提供された `luma_concepts` と選択されたコンセプトをマージして、結合されたコンセプトチェーンを作成します。

## 出力

| 出力名 | データ型 | 説明 |
|-------------|-----------|-------------|
| `luma_concepts` | LUMA_CONCEPTS | 選択されたすべてのコンセプトを含む、結合されたカメラコンセプトチェーンです |
