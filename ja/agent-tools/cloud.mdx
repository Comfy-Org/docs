## 概要

<Note>
  Comfy Cloud MCP は**クローズドベータ版です — 招待制**です。アクセスはユーザー単位の機能フラグによって制御されています。まだアクセス権がない場合は、[ウェイトリストに登録](https://form.typeform.com/to/hHmvw1UH)してください。
</Note>

**Comfy Cloud MCP サーバー**は、[モデルコンテキストプロトコル（MCP）](https://modelcontextprotocol.io)を通じて AI エージェントを [Comfy Cloud](https://cloud.comfy.org) に接続します。接続後、チャットから画像・動画・音声・3D の生成、モデル・ノード・テンプレートの検索、ComfyUI ワークフローの実行ができます。

現在は **Claude Code** と **Claude Desktop** に対応しており、どちらも **OAuth** でサインインします — ブラウザでの一度きりのログインで完了します。他のクライアントへの対応も予定されています。

### <span style={{ marginRight: "0.6rem" }}><Icon icon="terminal" size={22} /></span> Claude Code

**comfy-cloud** プラグインをインストール — MCP 接続とスラッシュコマンドを一度に追加します。

<Steps>
  <Step title="マーケットプレイスを追加">
    ```
    /plugin marketplace add Comfy-Org/comfy-skills
    ```
  </Step>
  <Step title="プラグインをインストール">
    ```
    /plugin install comfy-cloud@comfy-skills
    ```
  </Step>
  <Step title="サインイン">
    `/mcp` を実行し、**comfy-cloud** → **Authenticate** を選択。ブラウザでサインインし、トークンは自動更新されます。
  </Step>
</Steps>

プラグインは次のスラッシュコマンドを追加します：

| コマンド | 機能 |
| --- | --- |
| `/comfy-cloud:generate-image` | 画像の生成・編集・修正 |
| `/comfy-cloud:generate-video` | 動画の生成・編集・延長 |
| `/comfy-cloud:generate-audio` | 音声の生成 |
| `/comfy-cloud:generate-3d` | 3D モデルの生成 |
| `/comfy-cloud:remove-background` | 画像の背景を除去 |
| `/comfy-cloud:upscale-image` | 画像をアップスケール |
| `/comfy-cloud:search-models` | 利用可能なモデルを検索 |
| `/comfy-cloud:search-nodes` | ノードを検索 |
| `/comfy-cloud:search-templates` | プリビルトワークフローを検索 |
| `/comfy-cloud:help` | 利用可能な機能を確認 |

<Accordion title="プラグインなしで接続だけしたい場合">
  サーバーを直接追加（スラッシュコマンドなし）：

  ```bash
  claude mcp add --transport http comfy-cloud https://cloud.comfy.org/mcp
  ```

  次に `/mcp` を実行し、**comfy-cloud** → **Authenticate** を選択。`-s user` を付けると全プロジェクトで利用できます。
</Accordion>

### <span style={{ marginRight: "0.6rem" }}><Icon icon="desktop" size={22} /></span> Claude Desktop

Claude Desktop は UI から Comfy Cloud を**カスタムコネクタ**として追加し、OAuth サインインを行います。

<Steps>
  <Step title="Connectors を開く">
    Claude Desktop で **Customize** を開き、**Connectors** を選択。

    ![Claude Desktop Connectors](/images/cloud/mcp/desktop-connectors.png)
  </Step>
  <Step title="カスタムコネクタを追加">
    **+** ボタンをクリックし、**Add custom connector** を選択。

    ![Add custom connector](/images/cloud/mcp/desktop-add-custom-connector.png)
  </Step>
  <Step title="サーバー URL を入力">
    名前は任意（例：*Comfy Cloud*）で、**Remote MCP server URL** を次のように設定：

    ```
    https://cloud.comfy.org/mcp
    ```

    ![Connector details](/images/cloud/mcp/desktop-connector-url.png)
  </Step>
  <Step title="サインイン">
    **Add** をクリックし、Claude Desktop のプロンプトに従ってサインイン。接続完了。

    ![Sign in through Claude Desktop](/images/cloud/mcp/desktop-signin.png)
  </Step>
</Steps>

### ヘッドレス / CI（API キー）

Claude Code と Claude Desktop は OAuth を使用するため API キーは不要です。**ブラウザが使えないヘッドレスまたは CI** 環境では、API キーで認証してください：

<Steps>
  <Step title="キーを作成">
    [platform.comfy.org/profile/api-keys](https://platform.comfy.org/profile/api-keys) で **New API Key** をクリックし、キーをコピー（`comfyui-` で始まる）。
  </Step>
  <Step title="キーヘッダー付きでサーバーを追加">
    ```bash
    claude mcp add --transport http comfy-cloud https://cloud.comfy.org/mcp -H "X-API-Key: comfyui-…"
    ```
  </Step>
</Steps>

## FAQ

<AccordionGroup>
  <Accordion title="どのクライアントがサポートされていますか？">
    現在は Claude Code と Claude Desktop で、どちらも OAuth でサインインします。他のクライアントへの対応も予定されています。
  </Accordion>
  <Accordion title="API キーは必要ですか？">
    Claude Code と Claude Desktop では不要 — OAuth を使用します。ブラウザが使えないヘッドレスまたは CI 環境でのみ API キーが必要です。
  </Accordion>
  <Accordion title="スラッシュコマンドは Claude Desktop で使えますか？">
    いいえ。スラッシュコマンドは Claude Code プラグインに含まれます。Claude Desktop は同じ MCP サーバーに接続するためツールは利用できます（自然言語で依頼するだけ）が、Claude Code プラグインやスラッシュコマンドはサポートしていません。
  </Accordion>
  <Accordion title="サインイン時にブラウザが開きません。">
    Claude Code では `/mcp` を実行し、**comfy-cloud** を選択して **Authenticate** を選んでください。Claude Desktop では **Customize → Connectors** からコネクタを再度開き、サインインをトリガーしてください。
  </Accordion>
</AccordionGroup>
