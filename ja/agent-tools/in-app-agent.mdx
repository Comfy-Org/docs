**Comfy In-App Agent** は Comfy Cloud 内のエージェント体験です：チャットでプロンプトを入力すると、エージェントがグラフ上でワークフローを構築または編集できます（生成前に確認あり）。

このページはウェイトリストのランディングページのみです。完全な製品ドキュメントはまだ公開されていません。
