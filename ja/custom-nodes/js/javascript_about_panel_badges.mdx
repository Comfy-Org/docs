关于パネルバッジ API により、拡張機能は ComfyUI の关于ページにカスタムバッジを追加できます。これらのバッジには拡張機能に関する情報を表示でき、ドキュメント、ソースコード、または他のリソースへのリンクを含めることができます。

## 基本的な使い方

```javascript
app.registerExtension({
  name: "MyExtension",
  aboutPageBadges: [
    {
      label: "Documentation",
      url: "https://example.com/docs",
      icon: "pi pi-file"
    },
    {
      label: "GitHub",
      url: "https://github.com/username/repo",
      icon: "pi pi-github"
    }
  ]
});
```

## バッジ設定

各バッジには、以下のすべてのプロパティが必要です：

```javascript
{
  label: string,           // バッジに表示されるテキスト
  url: string,             // バッジがクリックされたときに開く URL
  icon: string             // アイコンクラス（例：PrimeVue アイコン）
}
```

## アイコンオプション

バッジのアイコンには PrimeVue のアイコンセットが使用されます。以下はよく使用されるアイコンの一部です：

- ドキュメント：`pi pi-file` または `pi pi-book`
- GitHub: `pi pi-github`
- 外部リンク：`pi pi-external-link`
- 情報：`pi pi-info-circle`
- ダウンロード：`pi pi-download`
- ウェブサイト：`pi pi-globe`
- Discord: `pi pi-discord`

利用可能なアイコンの完全なリストについては、[PrimeVue アイコンドキュメント](https://primevue.org/icons/) を参照してください。

## 例

```javascript
app.registerExtension({
  name: "BadgeExample",
  aboutPageBadges: [
    {
      label: "Website",
      url: "https://example.com",
      icon: "pi pi-home"
    },
    {
      label: "Donate",
      url: "https://example.com/donate",
      icon: "pi pi-heart"
    },
    {
      label: "Documentation",
      url: "https://example.com/docs",
      icon: "pi pi-book"
    }
  ]
});
```

バッジは設定ダイアログの关于パネルに表示され、ComfyUI インターフェースの右上隅にあるギアアイコンからアクセスできます。
