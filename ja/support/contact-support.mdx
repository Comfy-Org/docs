サポートチャンネルを通じて ComfyUI のヘルプを取得し、適切なリポジトリで問題を報告してください。

## サポートチャンネル

<CardGroup cols={2}>
  <Card title="Zendesk サポート" icon="life-ring" href="https://support.comfy.org/">
    サポートチームからヘルプを取得できます。
  </Card>
</CardGroup>

## 問題を報告する

バグおよび問題は、対応する GitHub リポジトリで報告してください：

<CardGroup cols={2}>
  <Card title="UI に関する問題" icon="browser" href="https://github.com/Comfy-Org/ComfyUI_frontend/issues">
    ComfyUI_frontend の UI 関連問題を報告できます。
  </Card>

  <Card title="コア機能に関する問題" icon="code" href="https://github.com/comfyanonymous/ComfyUI/issues">
    コア機能に関する問題を報告できます。
  </Card>

  <Card title="デスクトップ版に関する問題" icon="desktop" href="https://github.com/Comfy-Org/desktop/issues">
    デスクトップ版に関する問題を報告できます。
  </Card>
</CardGroup>
