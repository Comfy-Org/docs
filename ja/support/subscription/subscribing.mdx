## サブスクリプション登録方法

サブスクリプションに登録していなくても、Comfy Cloud を通常通り閲覧・使用することはできますが、ワークフローを実行するにはサブスクリプション登録が必要です。

Comfy Cloud にサブスクリプション登録するには：

1. [cloud.comfy.org](https://cloud.comfy.org/?utm_source=docs) にアクセスし、アカウントにログインします。
2. **サブスクリプション登録** ボタンをクリックします。

<img src="/images/support/subscription/subscribe.jpg" alt="サブスクリプションボタンの場所" />

3. 希望のサブスクリプションプランを選択します。

Comfy Cloud のサブスクリプションプランと価格の詳細については、[comfy.org/cloud/pricing](https://www.comfy.org/cloud/pricing) をご覧ください。

Comfy Cloud のサブスクリプションは、決済処理業者である Stripe を通じて管理されます。

## プロモーションコードの適用

プロモーションコードをお持ちの場合、チェックアウト時に適用することでサブスクリプションの割引を受けられます：

### ステップ 1: プロモーションコードフィールドを有効化する

チェックアウト時に、**プロモーションコードを追加** ボタンをクリックして入力フィールドを表示します。

<img src="/images/support/subscription/apply-coupon-1.jpg" alt="プロモーションコードを追加ボタンをクリック" />

### ステップ 2: コードを入力して適用する

フィールドにプロモーションコードを入力し、**適用** をクリックします。

<img src="/images/support/subscription/apply-coupon-2.jpg" alt="プロモーションコードを入力し、適用をクリック" />

### ステップ 3: 割引を確認する

コードが有効な場合、割引が反映された更新後の価格が表示されます。

<img src="/images/support/subscription/apply-coupon-3.jpg" alt="割引が正常に適用されました" />

支払い手続きを完了して、割引適用後のサブスクリプションを有効化します。

<Note>
プロモーションコードには、有効期限や利用制限など、特定の利用規約が適用される場合があります。適用前に詳細を確認してください。
</Note>

## サブスクリプションの更新

サブスクリプションは、初めて登録した日時の毎月同じ日に自動的に更新されます。

キャンセル設定を行わない限り、すべての Comfy Cloud サブスクリプションプランは自動的に更新されます。
