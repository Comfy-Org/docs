## はじめに

ComfyUI の歴史の大部分において、カスタムノードは Git を使用してインストールされてきました。これは入門には最適な方法でしたが、いくつかの課題がありました。これらについては [概要](/registry/overview) ページで説明されています。このページでは、Git によるインストール方法からレジストリへの移行方法について説明します。
