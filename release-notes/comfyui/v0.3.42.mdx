# ComfyUI v0.3.42

Release date: December 2023

<Note>
This release brings significant improvements to model support, training capabilities, and system stability.
</Note>

## Major Updates

<CardGroup>
  <Card title="SQLite Database Support" icon="database">
    Added native support for SQLite database integration
  </Card>
  <Card title="Cosmos Model Support" icon="stars">
    Comprehensive support for Cosmos Predict2 models including text-to-image and image-to-video
  </Card>
  <Card title="Enhanced LoRA Training" icon="graduation-cap">
    New LoRA training implementation using weight adapter scheme
  </Card>
</CardGroup>

## New Features

<AccordionGroup>
  <Accordion title="Cosmos Integration" icon="stars">
    * Added support for Cosmos Predict2 text-to-image models (2B and 14B)
    * Added support for Cosmos Predict2 image-to-video models
    * Introduced cosmos_rflow option in ModelSamplingContinuousEDM node
  </Accordion>

  <Accordion title="Advanced Sampling" icon="wand-magic-sparkles">
    * Added SEEDS samplers for RF
    * Introduced Multistep DPM++ SDE samplers for RF
  </Accordion>
</AccordionGroup>

## Improvements

<Check>
### AMD GPU Optimizations
* Enabled FP8 operations by default on GFX1201
* Enabled AMD FP8 and PyTorch attention on selected GPUs
</Check>

<Check>
### Frontend Updates
* Added support for frontend semver suffixes
* Introduced '@prerelease' option for testing latest versions
* Updated embedded documentation to version 0.2.2
</Check>

<Check>
### Configuration Enhancements
* Added support for extra settings from pyproject.toml
* Improved web folder registration system
</Check>

## Bug Fixes

<Warning>
* Fixed black image generation issues with flux models in FP16
* Resolved WebcamCapture IS_CHANGED signature issue
* Fixed Kontext aspect ratio functionality
* Corrected eps values in ltxv rmsnorm
* Standardized SaveLora filename format to match other nodes
</Warning>

## Other Changes

<Info>
* Added warning system for outdated Python versions
* Implemented cleaner torch version comparisons
* Refactored Cosmos attention code for better performance
* Added support for minimum length setting with tokenizer_data
</Info>

## Known Issues

<Error>
Apple FP16 attention bug remains unresolved
</Error>

## Documentation

Various documentation and template updates have been implemented to improve user experience and clarity.

For detailed implementation guides and API references, please refer to our [documentation](https://docs.comfyui.org).
