<Steps>
  <Step title="https://platform.comfy.org/login にアクセスしてログイン">
    https://platform.comfy.org/login にアクセスし、対応するアカウントでログインしてください。
    ![プラットフォームのログインページにアクセス](/images/interface/setting/user/user-login-api-key-1.jpg)
  </Step>
  <Step title="API キーの一覧で `+ New` をクリックして API キーを作成">
    API キーの一覧で `+ New` をクリックし、API キーを作成します。
    ![API キーを作成](/images/interface/setting/user/user-login-api-key-2.jpg)
  </Step>
  <Step title="API キーの名前を入力">
    ![API キーの名前を入力](/images/interface/setting/user/user-login-api-key-3.jpg)
    1. （必須）API キーの名前を入力してください。
    2. `Generate` をクリックして作成します。
  </Step>
  <Step title="取得した API キーを保存">
    ![API キーを取得](/images/interface/setting/user/user-login-api-key-4.jpg)
    <Warning>
      API キーは初回作成時のみ表示されるため、作成直後に必ず保存してください。その後は再表示されませんので、安全に保管してください。また、API キーを第三者と共有しないようご注意ください。万一漏洩した場合は、該当のキーを削除し、新しいキーを作成してください。
    </Warning>
  </Step>
</Steps>
