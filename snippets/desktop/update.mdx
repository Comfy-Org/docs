Comfy Desktop automatically checks for updates and lets you manage them in two ways.

## Quick Update

When an update is available, a **Desktop Update Ready** button appears at the top of the window:

![Desktop Update Ready button](/images/desktop/installation/install_update_1.png)

Just click the **① Desktop Update Ready** button and Comfy Desktop will restart and install the update automatically.

## Manual Update via Settings

You can also check for and manage updates from the Settings panel:

<Tabs>
  <Tab title="Step 1 — Open Desktop Settings">

![Open Desktop Settings menu](/images/desktop/installation/install_update_2-1.png)

1. **①** Click the hamburger menu (top-left)
2. **②** Select **Desktop Settings**

  </Tab>
  <Tab title="Step 2 — Updates Panel">

![Updates panel in Desktop Settings](/images/desktop/installation/install_update_2-2.png)

In the **Updates** panel:

- **① Updates** — Click the sidebar Updates tab
- **② Check for updates** — Manually check for available updates
- **③ Restart & Update** — Appears when a new version is available; click to restart and update
- **④ Automatically install Desktop updates** — Toggle this on/off to control automatic updates

  </Tab>
</Tabs>
