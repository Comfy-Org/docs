使用任意高于 3.9 的 Python 版本创建一个虚拟环境。

<CodeGroup>

```bash conda
conda create -n comfy-env python=3.11
conda activate comfy-env
```

```bash venv
python3 -m venv comfy-env
source comfy-env/bin/activate
```

</CodeGroup>

安装 ComfyUI

```bash
comfy install
```

<Warning>您仍然需要根据您的 GPU 安装 CUDA 或 ROCm。</Warning>
