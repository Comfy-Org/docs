Comfy Desktop 会自动检查更新，你可以选择以下两种方式管理更新。

## 快速更新

当有可用更新时，窗口顶部会显示 **Desktop Update Ready** 按钮：

![Desktop Update Ready 按钮](/images/desktop/installation/install_update_1.png)

直接点击 **<Icon icon="arrows-rotate" /> Desktop Update Ready** 按钮，Comfy Desktop 就会重启并自动安装更新。

## 通过设置手动更新

你也可以从设置面板检查和管理更新：

<Steps>
  <Step title="打开 Desktop Settings">

![打开 Desktop Settings 菜单](/images/desktop/installation/install_update_2-1.png)

1. **①** 点击左上角的汉堡菜单
2. **②** 选择 **Desktop Settings**

  </Step>
  <Step title="Updates 面板">

![Desktop Settings 的 Updates 面板](/images/desktop/installation/install_update_2-2.png)

在 **Updates** 面板中：

1. **① Updates** — 点击侧边栏的 Updates 标签
2. **② Check for updates** — 手动检查是否有可用更新
3. **③ Restart &amp; Update** — 当有新版本时出现，点击重启并更新
4. **④ Automatically install Desktop updates** — 开关自动安装桌面版更新

  </Step>
</Steps>
