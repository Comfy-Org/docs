## 데스크톱 파이썬 환경

데스크톱 설치 과정에서는 선택한 설치 디렉터리에 파이썬 가상환경을 생성하며, 일반적으로 숨겨진 `.venv` 폴더가 생성됩니다.

ComfyUI 플러그인의 의존성 관리가 필요한 경우, 이 환경 내에서 수행해야 합니다. 시스템 명령줄을 직접 사용하면 의존성을 시스템 환경에 설치할 위험이 있으니, 아래 지침에 따라 적절한 환경을 활성화해 주세요.

### 데스크톱 파이썬 환경 사용 방법은?

<Tabs>
  <Tab title="데스크톱 (권장)">
    데스크톱 앱에 내장된 터미널을 사용하여 파이썬 환경에 접근할 수 있습니다.
    ![Comfy 데스크톱 터미널](/images/desktop/desktop_terminal.png)
    1. 메뉴바의 아이콘을 클릭하여 하단 패널을 열어주세요.
    2. `터미널`을 클릭하여 터미널을 열어주세요.
    3. 해당 환경의 파이썬 설치 위치를 확인하려면 다음 명령어를 사용할 수 있습니다.
    <Tabs>
      <Tab title="Windows">
      ```
        python -c "import sys; print(sys.executable)"
      ```
      </Tab>
      <Tab title="macOS">
        ```
        which python
        ```
      </Tab>
    </Tabs>

    <Warning>
      현재 수행하는 작업의 의미를 잘 이해하지 못한다면, 해당 환경에서 의존성 문제가 발생할 수 있습니다. 이 방법을 사용하실 때는 각별히 주의해 주세요.
    </Warning>
  </Tab>
  <Tab title="터미널">
    선호하는 터미널을 사용해 파이썬 환경에 접근할 수도 있지만, 먼저 가상환경을 활성화해야 합니다.

    <Warning>
    다른 터미널을 사용할 경우, 작업에 익숙하지 않다면 실수로 의존성을 시스템 환경에 설치할 수 있습니다. 이 방법을 사용하실 때는 각별히 주의해 주세요.
    </Warning>

    ![시스템을 이용한 환경 활성화](/images/desktop/terminal.jpg)

    <Warning>
    스크린샷은 macOS 터미널을 보여줍니다. Windows를 사용하시는 경우, 다음 단계를 참고해 시스템에서 가상환경을 활성화해 주세요.
    </Warning>
    <Steps>
        <Step title="터미널 열기">
            선호하는 터미널을 열고 `cd` 명령어를 사용해 ComfyUI 설치 디렉터리로 이동하세요. 예를 들어:
            ```
            cd <your comfyui install directory>/ComfyUI
            ```
        </Step>
        <Step title="가상환경 활성화">
            터미널에 다음 명령어를 입력해 가상환경을 활성화하세요.

            <Tabs>
              <Tab title="Windows">
                ```
                .venv/Scripts/activate
                ```
              </Tab>
              <Tab title="macOS">
                ```
                source .venv/bin/activate
                ```
              </Tab>
            </Tabs>
           
            활성화 후, 터미널에 `(ComfyUI)`와 같은 프롬프트가 표시되면 가상환경이 성공적으로 활성화된 것입니다.
        </Step>
        <Step title="현재 파이썬 환경 확인">
            `which python` 명령어를 사용해 현재 파이썬 설치 위치를 확인하고, 이것이 시스템 환경이 아님을 확인하세요.

            이 단계를 완료하면, 해당 파이썬 환경이 활성화되었으며, 이 환경에서 의존성 설치 작업을 계속 진행할 수 있습니다.
        </Step>
    </Steps>
  </Tab>
</Tabs>
