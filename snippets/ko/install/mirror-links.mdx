다양한 미러 오류의 경우, 다음 내용을 참조하여 수동으로 다양한 미러를 찾아 교체해 보세요.

다음 사례는 주로 중국 사용자에게 적용됩니다.

#### 파이썬 설치 미러
    
기본 미러를 이용할 수 없는 경우, 아래의 미러를 사용해 보세요.

```
https://python-standalone.org/mirror/astral-sh/python-build-standalone
```

다른 대체 GitHub 미러 주소를 찾으려면 `python-build-standalone` 리포지토리의 릴리스를 가리키는 미러 주소를 찾아서 구성하세요.

```
https://github.com/astral-sh/python-build-standalone/releases/download
```

다음과 같은 형식으로 링크를 구성하세요.

```
https://xxx/astral-sh/python-build-standalone/releases/download
```
