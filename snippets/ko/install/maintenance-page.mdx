ComfyUI를 설치하는 데 문제가 생길 수 있는 이유는 여러 가지입니다. 예를 들어, PyTorch(15GB)를 설치할 때 네트워크 연결이 실패했을 수 있습니다. 또는 git이 설치되어 있지 않을 수도 있습니다. 문제를 감지하면 유지보수 페이지가 자동으로 열리며, 이를 통해 문제를 해결할 수 있는 방법을 제공합니다.

대부분의 문제를 해결하려면 다음을 수행할 수 있습니다:

- 파이썬 가상환경 생성하기
- Desktop에서 관리하는 파이썬 가상환경에 누락된 핵심 종속성 모두 다시 설치하기
- git, VC Redis 설치하기
- 새로운 설치 위치 선택하기

기본 유지보수 페이지에는 현재 발생한 오류 내용이 표시됩니다.

![ComfyUI 유지보수 페이지](/images/desktop/maintenance-1.jpg)

'모두'를 클릭하면 현재 작업 가능한 모든 내용을 확인할 수 있습니다.

![ComfyUI 유지보수 페이지](/images/desktop/maintenance-2.jpg)
