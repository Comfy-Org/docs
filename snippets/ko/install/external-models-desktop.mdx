## 외부 모델 경로 추가하기

컴퓨터의 ComfyUI 설치 디렉터리 외부에 저장된 모델이 있는 경우, `extra_models_config.yaml` 파일을 구성하여 이를 ComfyUI에 추가할 수 있습니다.

ComfyUI 데스크톱 버전의 경우, 이 파일은 다음과 같은 위치에 있습니다:
- Windows: `C:\Users\<YOUR_USERNAME>\AppData\Roaming\ComfyUI\extra_models_config.yaml`
- macOS: `~/Library/Application Support/ComfyUI/extra_models_config.yaml`
- Linux: `~/.config/ComfyUI/extra_models_config.yaml`

자세한 지침은 [모델 문서](/ko/development/core-concepts/models#adding-external-model-paths)를 참조하십시오.
