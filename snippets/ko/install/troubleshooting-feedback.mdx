### 피드백 설치 실패

설치 중에 오류가 발생한 경우, 유사한 오류 보고서가 있는지 확인하거나 다음을 통해 오류를 제출해 주세요:

- GitHub 이슈: https://github.com/Comfy-Org/desktop/issues  
- Comfy 공식 포럼: https://forum.comfy.org/

오류 보고서를 제출할 때는 다음 로그 및 구성 파일을 포함하여 문제를 신속히 파악하고 조사할 수 있도록 해주세요:

1. 로그 파일

| 파일명    | 설명                                                                                     | 위치         |
|-------------|-------------------------------------------------------------------------------------------------|------------------|
| main.log    | Electron 프로세스에서 데스크톱 애플리케이션 및 서버 시작과 관련된 로그 포함       | {log_path}       |
| comfyui.log | 코어 ComfyUI 프로세스의 터미널 출력 등 ComfyUI 정상 작동과 관련된 로그 포함     | {log_path}       |

![ComfyUI 로그 파일 위치](/images/desktop/win-comfyui-desktop-10-logs.jpg)

2. 구성 파일

| 파일명                 | 설명                                                                     | 위치          |
|--------------------------|---------------------------------------------------------------------------------|-------------------|
| extra_model_paths.yaml | ComfyUI가 모델 및 맞춤형 노드를 검색할 추가 경로 포함                   | {config_path}     |
| config.json              | 애플리케이션 설정을 포함합니다. 이 파일은 직접 수정하지 마세요           | {config_path}     |

![ComfyUI 구성 파일 위치](/images/desktop/win-comfyui-desktop-11-config.jpg)
