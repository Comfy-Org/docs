Comfy Desktop은 자동으로 업데이트를 확인하며, 다음 두 가지 방법으로 관리할 수 있습니다.

## 빠른 업데이트

업데이트가 있으면 창 상단에 **Desktop Update Ready** 버튼이 표시됩니다:

![Desktop Update Ready 버튼](/images/desktop/installation/install_update_1.png)

**①** <Icon icon="arrows-rotate" /> **Desktop Update Ready** 버튼을 클릭하면 Comfy Desktop이 재시작되며 업데이트가 자동으로 설치됩니다.

## 설정에서 수동 업데이트

설정 패널에서도 업데이트를 확인하고 관리할 수 있습니다:

<Steps>
  <Step title="Desktop Settings 열기">

![Desktop Settings 메뉴 열기](/images/desktop/installation/install_update_2-1.png)

1. **①** 왼쪽 상단 햄버거 메뉴 클릭
2. **②** **Desktop Settings** 선택

  </Step>
  <Step title="Updates 패널">

![Desktop Settings의 Updates 패널](/images/desktop/installation/install_update_2-2.png)

**Updates** 패널에서:

1. **① Updates** — 사이드바의 Updates 탭 클릭
2. **② Check for updates** — 수동으로 업데이트 확인
3. **③ Restart &amp; Update** — 새 버전이 있을 때 표시되며, 클릭하면 재시작 후 업데이트
4. **④ Automatically install Desktop updates** — 자동 업데이트 설치 켜기/끄기

  </Step>
</Steps>
