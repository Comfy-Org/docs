**Comfy Desktop** is a multi-installation manager that lets you run multiple ComfyUI instances from a single launcher. This guide covers how to install and uninstall Comfy Desktop on macOS.

## System Requirements

- **OS:** macOS 13 (Ventura) or later
- **Hardware:** Apple Silicon (M1 or later)
- **Disk Space:** At least 15 GB recommended per installation

## Installation

1. Open the downloaded `.dmg` file
2. Drag **Comfy Desktop** into the **Applications** folder
3. Launch it from Launchpad or Spotlight

The first time you launch, macOS may show a security warning. Go to **System Settings → Privacy & Security** and click **Open Anyway** to allow it.

When the app opens, you'll see the **Welcome** screen where you can create your first installation.

## Uninstalling

### Remove Comfy Desktop

Drag **Comfy Desktop** from the Applications folder to the Trash.

### Remove Installation Data

Comfy Desktop and its installations are stored separately on disk. To fully clean up:

1. **Delete individual installations** from within the app, or delete the installation folders manually
2. **Delete the app data folder** at `~/Library/Application Support/Comfy Desktop`

## Data Locations

| Data | macOS Path |
|------|-----------|
| App data (settings, installation list) | `~/Library/Application Support/Comfy Desktop` |
| Logs | `~/Library/Application Support/Comfy Desktop/logs` |
| Installation directories | User-chosen during setup |

## Troubleshooting

### macOS security warning

If macOS blocks the app, go to **System Settings → Privacy & Security**, scroll to the bottom, and click **Open Anyway** next to the ComfyUI entry.

### Mirror errors during installation

If mirrors fail, expand the mirror settings in the app to configure custom mirrors for Python, PyPI, and PyTorch.

### Error identification

If installation fails, you should see the following screen

![Comfy Desktop Installation Failed](/images/desktop/win-comfyui-desktop-7.jpg)

It is recommended to take these steps to find the error cause:
1. Click `Show Terminal` to view error output
2. Click `Open Logs` to view installation logs
3. Visit the [Comfy Official Forum](https://forum.comfy.org/) to search for error reports
4. Click `Reinstall` to try reinstalling

Before submitting feedback, it's recommended to provide the **error output** and **log files** to tools like **GPT**

![Comfy Desktop Installation Failed - Error Log](/images/desktop/win-comfyui-desktop-8.jpg)
![Comfy Desktop Installation Failed - GPT Feedback](/images/desktop/win-comfyui-desktop-9.jpg)

As shown above, ask the GPT for the cause of the corresponding error, or remove Comfy Desktop completely and retry the installation.

### Feedback Installation Failure

If you encounter any errors during installation, please check if there are similar error reports or submit errors to us through:

- [GitHub Issues](https://github.com/Comfy-Org/desktop/issues)
- [Comfy Official Forum](https://forum.comfy.org/)

When submitting error reports, please ensure you include the following logs and configuration files to help us locate and investigate the issue:

1. **Log Files**

| Filename    | Description                                                                                     | Location         |
|-------------|-------------------------------------------------------------------------------------------------|------------------|
| main.log    | Contains logs related to desktop application and server startup from the Electron process       | ~/Library/Application Support/Comfy Desktop/logs |
| comfyui.log | Contains logs related to ComfyUI normal operation, such as core ComfyUI process terminal output | ~/Library/Application Support/Comfy Desktop/logs |

![Comfy Desktop Log Files Location](/images/desktop/win-comfyui-desktop-10-logs.jpg)

2. **Configuration Files**

| Filename                 | Description                                                                     | Location          |
|--------------------------|---------------------------------------------------------------------------------|-------------------|
| extra_model_paths.yaml   | Contains additional paths where ComfyUI will search for models and custom nodes | ~/Library/Application Support/Comfy Desktop |
| config.json              | Contains application configuration. This file should not be edited directly     | ~/Library/Application Support/Comfy Desktop |
