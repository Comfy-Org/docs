**ComfyUI Desktop** is a multi-installation manager that lets you run multiple ComfyUI instances from a single launcher. This guide walks you through installing ComfyUI Desktop on Linux.

ComfyUI Desktop is an open source project. View the source code on [GitHub](https://github.com/Comfy-Org/Comfy-Desktop).

## System Requirements

- **OS:** Ubuntu 22.04+ (Debian-based distributions)
- **GPU:** NVIDIA GPU with CUDA support (recommended)
- **Disk Space:** At least 15 GB recommended per installation

## Download

Download ComfyUI Desktop from the [official download page](https://comfy.org/download). The page will detect your platform and provide the correct package — .deb for Debian/Ubuntu or AppImage for other distributions.

### Installation via .deb

```bash
sudo dpkg -i comfyui-desktop_*.deb
```

### Installation via AppImage

```bash
chmod +x comfyui-desktop-*.AppImage
./comfyui-desktop-*.AppImage
```

### Build from Source (alternative)

You can also build ComfyUI Desktop for Linux from source. The project is fully open source.

#### Prerequisites

- [Node.js](https://nodejs.org/) **v22 LTS** or later
- [pnpm](https://pnpm.io/) **v10** or later (via [Corepack](https://nodejs.org/api/corepack.html) recommended)

#### Build steps

```bash
git clone https://github.com/Comfy-Org/Comfy-Desktop.git
cd Comfy-Desktop
pnpm run init
pnpm run make
```

Once built, you'll find the packaged app in the `out/make/` directory as a `.deb` or AppImage.

## First Launch

After installation, launch ComfyUI Desktop. The first time you run it, you'll see the **Welcome** screen. Click **Get Started** to begin.

## Data Locations

ComfyUI Desktop follows XDG Base Directory conventions on Linux:

| Data | Linux Path |
|------|-----------|
| Config | `~/.config/comfyui-desktop` |
| Data | `~/.local/share/comfyui-desktop` |
| Cache | `~/.cache/comfyui-desktop` |
| State | `~/.local/state/comfyui-desktop` |
| Default install directory | User-chosen during setup |
