**ComfyUI Desktop** is a multi-installation manager that lets you run multiple ComfyUI instances from a single launcher. This guide covers how to install and uninstall ComfyUI Desktop on Linux.

## System Requirements

- **OS:** Debian-based distributions (Ubuntu 22.04+ recommended)
- **GPU:** Dedicated GPU (NVIDIA / AMD) recommended for good performance, but not required
- **Disk Space:** At least 15 GB recommended per installation

## Installation

### .deb Package (Debian/Ubuntu)

```bash
sudo apt install ./comfyui-desktop_*.deb
```

### AppImage (Other Distributions)

```bash
chmod +x ./comfyui-desktop-*.AppImage
./comfyui-desktop-*.AppImage --no-sandbox
```

After installation, launch ComfyUI Desktop from your application menu. When the app opens, you'll see the **Welcome** screen where you can create your first installation.

## Uninstalling

### Remove ComfyUI Desktop

**Debian/Ubuntu:**
```bash
sudo apt remove comfyui-desktop
```

**AppImage:** Simply delete the AppImage file.

### Remove Installation Data

ComfyUI Desktop and its installations are stored separately on disk. To fully clean up:

1. **Delete individual installations** from within the app, or delete the installation folders manually
2. **Delete the app data and config directories:**

| Data | Path |
|------|------|
| Config | `~/.config/comfyui-desktop` |
| Data | `~/.local/share/comfyui-desktop` |
| Cache | `~/.cache/comfyui-desktop` |
| State | `~/.local/state/comfyui-desktop` |

## Data Locations

ComfyUI Desktop follows XDG Base Directory conventions on Linux:

| Data | Linux Path |
|------|-----------|
| Config | `~/.config/comfyui-desktop` |
| Data | `~/.local/share/comfyui-desktop` |
| Cache | `~/.cache/comfyui-desktop` |
| State | `~/.local/state/comfyui-desktop` |
| Installation directories | User-chosen during setup |

## Troubleshooting

### AppImage requires fuse2

If the AppImage doesn't run, install `fuse2`:

```bash
sudo apt install fuse2
```

### Mirror errors during installation

If mirrors fail, expand the mirror settings in the app to configure custom mirrors for Python, PyPI, and PyTorch.
