**Comfy Desktop** is a multi-installation manager that lets you run multiple ComfyUI instances from a single launcher. This guide covers how to install and uninstall Comfy Desktop on Linux.

Comfy Desktop is available as a **.deb** package (Debian/Ubuntu) or **AppImage** (other distributions). You can find download links for both on the [Comfy Desktop GitHub releases page](https://github.com/Comfy-Org/Comfy-Desktop/releases).

## System Requirements

- **OS:** Debian-based distributions (Ubuntu 22.04+ recommended)
- **GPU:** Dedicated GPU (NVIDIA / AMD) recommended for good performance, but not required
- **Disk Space:** At least 15 GB recommended per installation

## Installation

### .deb Package (Debian/Ubuntu)

```bash
sudo apt install ./comfyui-desktop_*.deb
```

### AppImage (Other Distributions)

```bash
chmod +x ./comfyui-desktop-*.AppImage
./comfyui-desktop-*.AppImage --no-sandbox
```

If the AppImage doesn't run, install `fuse2`:

```bash
sudo apt install fuse2
```

After installation, launch Comfy Desktop from your application menu. When the app opens, you'll see the **Welcome** screen where you can create your first installation.

## Uninstalling

### Remove Comfy Desktop

**Debian/Ubuntu:**
```bash
sudo apt remove comfyui-desktop
```

**AppImage:** Simply delete the AppImage file.

### Remove Installation Data

Comfy Desktop and its installations are stored separately on disk. To fully clean up:

1. **Delete individual installations** from within the app, or delete the installation folders manually
2. **Delete the app data and config directories:**

| Data | Path |
|------|------|
| Config | `~/.config/Comfy Desktop` |
| Data | `~/.local/share/Comfy Desktop` |
| Cache | `~/.cache/Comfy Desktop` |
| State | `~/.local/state/Comfy Desktop` |
