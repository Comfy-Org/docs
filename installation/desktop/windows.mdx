**Comfy Desktop** is a multi-installation manager that lets you run multiple ComfyUI instances from a single launcher. This guide covers how to install and uninstall Comfy Desktop on Windows.

## System Requirements

- **OS:** Windows 10 or later
- **Architecture:** x64 or ARM64
- **GPU:** Dedicated GPU (NVIDIA / AMD) recommended for good performance, but not required
- **Disk Space:** At least 15 GB recommended per installation

## Installation

Run the downloaded `.exe` installer (NSIS installer) and follow the prompts. After installation, launch Comfy Desktop from the Start Menu or desktop shortcut.

When you first open the app, you'll see the **Welcome** screen where you can create your first installation.

## Uninstalling

### Remove Comfy Desktop

Uninstall Comfy Desktop from **Settings → Apps → Apps & features**, or find **Comfy Desktop** in the Start Menu and select **Uninstall**.

### Remove Installation Data

Comfy Desktop and its installations are stored separately on disk. To fully clean up:

1. **Delete individual installations** from within the app by right-clicking an installation card → **Delete**, or delete the installation folders manually
2. **Delete the app data folder** at `%APPDATA%\Comfy Desktop`

## Data Locations

| Data | Windows Path |
|------|-------------|
| App data (settings, installation list) | `%APPDATA%\Comfy Desktop` |
| Logs | `%APPDATA%\Comfy Desktop\logs` |
| Installation directories | User-chosen during setup |

## Troubleshooting

### Installation fails with antivirus warnings

Comfy Desktop bundles [`uv`](https://github.com/astral-sh/uv) for Python environment management. Some antivirus products may flag `uv.exe` as malicious — this is a **known false positive**. If installation fails, add the ComfyUI installation directory to your antivirus whitelist.

### Mirror errors during installation

If mirrors fail, you'll see a red ❌ icon. Expand the mirror settings in the app to configure custom mirrors for Python, PyPI, and PyTorch — especially useful if you're in a region with restricted GitHub access.
