**Comfy Desktop** is a multi-installation manager that lets you run multiple ComfyUI instances from a single launcher. This guide covers how to install and uninstall Comfy Desktop on Windows.

## System Requirements

- **OS:** Windows 10 or later
- **Architecture:** x64 or ARM64
- **GPU:** Dedicated GPU (NVIDIA / AMD) recommended for good performance, but not required
- **Disk Space:** At least 15 GB recommended per installation

## Installation

Run the downloaded `.exe` installer (NSIS installer) and follow the prompts. After installation, launch Comfy Desktop from the Start Menu or desktop shortcut.

When you first open the app, you'll see the **Welcome** screen where you can create your first installation.

## Uninstalling

### Remove Comfy Desktop

Uninstall Comfy Desktop from **Settings → Apps → Apps & features**, or find **Comfy Desktop** in the Start Menu and select **Uninstall**.

### Remove Installation Data

Comfy Desktop and its installations are stored separately on disk. To fully clean up:

1. **Delete individual installations** from within the app by right-clicking an installation card → **Delete**, or delete the installation folders manually
2. **Delete the app data folder** at `%APPDATA%\Comfy Desktop`

## Data Locations

| Data | Windows Path |
|------|-------------|
| App data (settings, installation list) | `%APPDATA%\Comfy Desktop` |
| Logs | `%APPDATA%\Comfy Desktop\logs` |
| Installation directories | User-chosen during setup |

## Troubleshooting

### Installation fails with antivirus warnings

Comfy Desktop bundles [`uv`](https://github.com/astral-sh/uv) for Python environment management. Some antivirus products may flag `uv.exe` as malicious — this is a **known false positive**. If installation fails, add the ComfyUI installation directory to your antivirus whitelist.

### Mirror errors during installation

If mirrors fail, you'll see a red ❌ icon. Expand the mirror settings in the app to configure custom mirrors for Python, PyPI, and PyTorch — especially useful if you're in a region with restricted GitHub access.

### Error identification

If installation fails, you should see the following screen

![Comfy Desktop Installation Failed](/images/desktop/win-comfyui-desktop-7.jpg)

It is recommended to take these steps to find the error cause:
1. Click `Show Terminal` to view error output
2. Click `Open Logs` to view installation logs
3. Visit the [Comfy Official Forum](https://forum.comfy.org/) to search for error reports
4. Click `Reinstall` to try reinstalling

Before submitting feedback, it's recommended to provide the **error output** and **log files** to tools like **GPT**

![Comfy Desktop Installation Failed - Error Log](/images/desktop/win-comfyui-desktop-8.jpg)
![Comfy Desktop Installation Failed - GPT Feedback](/images/desktop/win-comfyui-desktop-9.jpg)

As shown above, ask the GPT for the cause of the corresponding error, or remove Comfy Desktop completely and retry the installation.

### Feedback Installation Failure

If you encounter any errors during installation, please check if there are similar error reports or submit errors to us through:

- [GitHub Issues](https://github.com/Comfy-Org/desktop/issues)
- [Comfy Official Forum](https://forum.comfy.org/)

When submitting error reports, please ensure you include the following logs and configuration files to help us locate and investigate the issue:

1. **Log Files**

| Filename    | Description                                                                                     | Location         |
|-------------|-------------------------------------------------------------------------------------------------|------------------|
| main.log    | Contains logs related to desktop application and server startup from the Electron process       | %APPDATA%\Comfy Desktop\logs |
| comfyui.log | Contains logs related to ComfyUI normal operation, such as core ComfyUI process terminal output | %APPDATA%\Comfy Desktop\logs |

![Comfy Desktop Log Files Location](/images/desktop/win-comfyui-desktop-10-logs.jpg)

2. **Configuration Files**

| Filename                 | Description                                                                     | Location          |
|--------------------------|---------------------------------------------------------------------------------|-------------------|
| extra_model_paths.yaml   | Contains additional paths where ComfyUI will search for models and custom nodes | %APPDATA%\Comfy Desktop |
| config.json              | Contains application configuration. This file should not be edited directly     | %APPDATA%\Comfy Desktop |
