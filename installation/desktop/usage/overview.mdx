This guide covers **Comfy Desktop** — the desktop application that installs and manages your ComfyUI instances. It is not a tutorial on how to build workflows or use ComfyUI nodes.

With Comfy Desktop you can install, launch, and manage multiple ComfyUI instances side by side, each with its own Python environment, models, and custom nodes.
