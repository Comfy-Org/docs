This guide covers how to use Comfy Desktop after installation.

![Comfy Desktop homepage](/images/desktop/usage/01-homepage.png)

After installing and launching Comfy Desktop, you'll see all your ComfyUI instances displayed as cards. From here you can launch any instance, add new ones, manage settings, and more.

  </Card>
  <Card title="Managing Installations" icon="sliders" href="/installation/desktop/usage/manage">

  </Card>
