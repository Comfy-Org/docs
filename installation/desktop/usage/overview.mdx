Comfy Desktop is the official desktop application for **ComfyUI**, the node-based engine for generative AI. It installs ComfyUI into a self-contained, GPU-ready environment, lets you run multiple independent setups side by side, and keeps them updated.

This guide covers how to use Comfy Desktop to manage your ComfyUI installations.
