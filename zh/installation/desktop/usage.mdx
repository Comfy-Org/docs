本指南介绍 Comfy Desktop 安装后的使用方法。

## Chooser 视图

**Chooser** 是打开 Comfy Desktop 时看到的主屏幕。它以网格形式显示所有 ComfyUI 安装。

### 安装卡片

每张卡片显示：
- **名称** — 你为安装指定的名称
- **类型图标** — 表示来源类型（独立、云、Git、旧版 Desktop、远程）
- **上次启动时间** — 距离上次使用多久
- **状态** — 运行中、错误或有可用更新

### 筛选

使用网格顶部的筛选标签来缩小范围：

- **All（全部）** — 显示所有安装
- **Local（本地）** — 仅独立安装和旧版 Desktop 安装
- **Cloud（云）** — 仅 Comfy Cloud 连接
- **Remote（远程）** — 仅远程服务器连接

## 添加新安装

点击左上角的 **+** 卡片添加新的 ComfyUI 安装。下面带你一步步完成。

### 第 1 步 — 配置 Comfy Desktop

<Tabs>
  <Tab title="Windows">
    ![在 Windows 上配置 Comfy Desktop](/images/desktop/usage/new_install_01_settings_windows.png)
  </Tab>
  <Tab title="macOS">
    ![在 macOS 上配置 Comfy Desktop](/images/desktop/usage/new_install_01_settings_mac.png)
  </Tab>
</Tabs>

1. **① 实例名称** — 你的实例名称。默认是 "ComfyUI"。
2. **② 安装位置** — 选择 ComfyUI 的安装位置。Windows 用户强烈建议使用**固态硬盘**，避免安装在 C 盘根目录。
3. **③ 高级（Advanced）** — 点击展开。在这里可以选择：
   - **④ 独立安装（Standalone）** — 推荐选项。自带 Python、自动更新、支持快照。
   - **⑤ 远程连接（Remote Connection）** — 连接到另一台机器上运行的 ComfyUI。

### 第 2 步 — 安装中

![正在安装 ComfyUI](/images/desktop/usage/new_install_02_installing.png)

Comfy Desktop 会自动完成后续工作——下载 Python、搭建环境、安装 ComfyUI。进度界面会显示当前步骤。

### 第 3 步 — 选择模板

![安装完成后的模板库](/images/desktop/usage/new_install_03_interface_template.png)

安装完成后，Comfy Desktop 会打开模板库，你可以选择工作流模板开始使用。

## 启动 ComfyUI

点击任意安装卡片即可启动。Comfy Desktop 将：
1. 启动 ComfyUI 后端进程
2. 等待其就绪
3. 打开 ComfyUI 编辑器窗口

如果安装已在运行，点击卡片会将现有窗口置于前台，而不是启动新实例。

## 安装操作

右键点击安装卡片（或点击 ⋮ 图标）访问操作：

| 操作 | 说明 |
|------|------|
| **Launch / Restart** | 启动或重启 ComfyUI 实例 |
| **Manage** | 打开此安装的完整设置面板 |
| **Update** | 将 ComfyUI 更新到最新版本 |
| **Restore Snapshot** | 恢复到已保存的快照 |
| **Open Folder** | 在文件管理器中打开安装目录 |
| **Delete** | 删除此安装（不会删除模型文件） |

## 快照

Comfy Desktop 在每次更新前自动创建**快照**。快照记录：
- ComfyUI 版本（Git commit hash）
- 所有已安装的自定义节点（含版本）
- Python 包列表

你可以随时恢复任何快照，有效回滚 ComfyUI 及其自定义节点到之前的状态。

## 键盘快捷键

| 快捷键 | 操作 |
|--------|------|
| `Ctrl+R` / `Cmd+R` | 刷新当前视图 |
| `Ctrl+W` / `Cmd+W` | 关闭当前面板 |
| `Escape` | 关闭模态框或覆盖层 |
| `Ctrl+,` / `Cmd+,` | 打开全局设置 |
