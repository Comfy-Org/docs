本指南介绍 ComfyUI Desktop 安装后的使用方法。

## Chooser 视图

**Chooser** 是打开 ComfyUI Desktop 时看到的主屏幕。它以网格形式显示所有 ComfyUI 安装。

### 安装卡片

每张卡片显示：
- **名称** — 你为安装指定的名称
- **类型图标** — 表示来源类型（独立、云、Git、旧版 Desktop、远程）
- **上次启动时间** — 距离上次使用多久
- **状态** — 运行中、错误或有可用更新

### 筛选

使用网格顶部的筛选标签来缩小范围：

- **All（全部）** — 显示所有安装
- **Local（本地）** — 仅独立安装和旧版 Desktop 安装
- **Cloud（云）** — 仅 Comfy Cloud 连接
- **Remote（远程）** — 仅远程服务器连接

## 添加新安装

点击左上角的 **+** 卡片添加新的 ComfyUI 安装：

1. **选择来源类型**：独立、云、Git Clone、Portable、旧版 Desktop 或远程连接
2. **配置**：填写必要字段（名称、路径、版本等）
3. **安装**：ComfyUI Desktop 处理其余工作——下载 Python、设置环境、安装 ComfyUI

## 启动 ComfyUI

点击任意安装卡片即可启动。ComfyUI Desktop 将：
1. 启动 ComfyUI 后端进程
2. 等待其就绪
3. 打开 ComfyUI 编辑器窗口

如果安装已在运行，点击卡片会将现有窗口置于前台，而不是启动新实例。

## 安装操作

右键点击安装卡片（或点击 ⋮ 图标）访问操作：

| 操作 | 说明 |
|------|------|
| **Launch / Restart** | 启动或重启 ComfyUI 实例 |
| **Manage** | 打开此安装的完整设置面板 |
| **Update** | 将 ComfyUI 更新到最新版本 |
| **Restore Snapshot** | 恢复到已保存的快照 |
| **Open Folder** | 在文件管理器中打开安装目录 |
| **Delete** | 删除此安装（不会删除模型文件） |

## 快照

ComfyUI Desktop 在每次更新前自动创建**快照**。快照记录：
- ComfyUI 版本（Git commit hash）
- 所有已安装的自定义节点（含版本）
- Python 包列表

你可以随时恢复任何快照，有效回滚 ComfyUI 及其自定义节点到之前的状态。

## 应用内终端

ComfyUI Desktop 提供内置终端，用于访问 ComfyUI Python 环境：

1. 点击标题栏中的终端图标（⊞）打开底部面板
2. 点击 **Terminal** 标签
3. 你现在位于活动安装的 Python 虚拟环境中

## 键盘快捷键

| 快捷键 | 操作 |
|--------|------|
| `Ctrl+R` / `Cmd+R` | 刷新当前视图 |
| `Ctrl+W` / `Cmd+W` | 关闭当前面板 |
| `Escape` | 关闭模态框或覆盖层 |
| `Ctrl+,` / `Cmd+,` | 打开全局设置 |
