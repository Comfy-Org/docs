从左侧栏的齿轮图标（⚙️）进入全局设置。

设置面板分为四个标签页，点击侧边栏标签即可切换。

## 偏好设置

![偏好设置](/images/desktop/usage/comfy_desktop_settings_01_preferences.png)

### 语言

从下拉菜单选择显示语言，支持以下语言：
- **English**
- **中文**

### 应用行为

- **启动时自动重启上一个实例** — 开启后，Comfy Desktop 启动时自动运行上次使用的实例。
- **关闭最后一个窗口时退出 Comfy Desktop（不返回 Dashboard）** — 开启后，关闭最后一个 ComfyUI 窗口将完全退出应用，而不是返回 Dashboard。
- **在实例选择器中隐藏云实例** — 开启后，Chooser 视图中不再显示云实例（适用于不使用云服务的用户）。

### 隐私

- **分享匿名使用遥测数据** — 开启后，Comfy Desktop 会发送匿名使用数据以帮助改进产品，包括基本功能使用统计和崩溃报告。

### 社区

- **Comfy Desktop** — 链接到 ComfyDesktop GitHub 仓库的按钮。点击在浏览器中打开。

## 更新

![更新](/images/desktop/usage/comfy_desktop_settings_02_updates.png)

更新标签页显示当前 Comfy Desktop 版本，并可管理应用更新。

状态区域显示 Comfy Desktop 是否为最新版本、当前安装版本以及上次检查更新时间。

- **检查更新** — 点击手动检查新版本。如果有可用更新，按钮会变为**下载**再变为**安装**。
- **自动安装桌面端更新** — 开启后，Comfy Desktop 在后台自动下载并安装更新，无需手动操作。

## 存储

![存储](/images/desktop/usage/comfy_desktop_settings_03_storage.png)

存储标签页管理 Comfy Desktop 查找模型的位置以及输入/输出的存储目录。

### 共享模型

添加 Comfy Desktop 扫描模型的目录。列表中的第一个目录标记为**主要**——新模型默认保存到此目录。
- 点击文件夹图标在文件管理器中打开目录。
- 点击目录上的 **...** 菜单可更改排序或设为主要目录。
- 点击 **+ 添加目录** 浏览并添加其他模型目录。

### 共享目录

- **输入目录** — ComfyUI 查找输入文件（图片、视频等）的目录。可自定义路径以在多个实例间共享输入。
- **输出目录** — ComfyUI 保存生成结果（图片、视频等）的目录。自定义路径可将所有输出集中到一处。

点击文件夹图标浏览选择新路径。每个标签旁的 **?** 工具提示可查看详细信息。

## 高级

![高级](/images/desktop/usage/comfy_desktop_settings_04_advanced.png)

### 默认安装位置

设置新建独立 ComfyUI 实例的默认目录。点击文件夹图标浏览其他位置。

### 高级

- **PyPI 镜像 URL** — 输入自定义 pip 镜像地址（如 `https://mirrors.aliyun.com/pypi/simple/`），适用于默认 PyPI 访问慢或被屏蔽的环境。
- **使用中国镜像（Git & PyPI）** — 开启后，Comfy Desktop 自动使用中国镜像服务器访问 Git 仓库和 PyPI 包。适用于中国大陆用户。

### 缓存

- **缓存目录** — 显示当前缓存路径。点击文件夹图标可选择新目录。Comfy Desktop 在此存储临时和缓存数据。

## 数据存储位置

Comfy Desktop 将数据存储在平台特定的目录中：

### Windows
```
应用数据:  %APPDATA%\Comfy Desktop
日志:      %APPDATA%\Comfy Desktop\logs
```

### macOS
```
应用数据:  ~/Library/Application Support/Comfy Desktop
日志:      ~/Library/Application Support/Comfy Desktop/logs
```

### Linux（XDG 基础目录）
```
配置:  ~/.config/Comfy Desktop
数据:  ~/.local/share/Comfy Desktop
缓存:  ~/.cache/Comfy Desktop
状态:  ~/.local/state/Comfy Desktop
```

## 键盘快捷键

| 快捷键 | 操作 |
|--------|------|
| `Ctrl+R` / `Cmd+R` | 刷新当前视图 |
| `Ctrl+W` / `Cmd+W` | 关闭当前面板 |
| `Escape` | 关闭模态框或覆盖层 |
| `Ctrl+,` / `Cmd+,` | 打开全局设置 |

## Deep Links (comfy://)

Comfy Desktop 支持 `comfy://` 自定义协议，可直接从浏览器或终端打开本地工作流文件。

### 通过 Deep Link 打开工作流

```
comfy://open?file=/path/to/workflow.json
```

点击 `comfy://` 链接或从命令行运行将：
1. 启动 Comfy Desktop（如果尚未运行）
2. 在 ComfyUI 编辑器中打开指定的工作流文件
3. 聚焦现有窗口（如果已在运行）

> **注意：** Deep links 仅支持本地文件路径。此功能需要 Comfy Desktop 已注册为协议处理器——安装时会自动注册。
