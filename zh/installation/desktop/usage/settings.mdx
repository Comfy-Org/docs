点击左上角 **☰ 菜单**（1）并选择 **Desktop Settings**（2）进入全局设置。

![打开桌面设置](/images/desktop/usage/comfy_desktop_settings_00_menu.png)

设置面板分为四个标签页，点击侧边栏标签即可切换。

## 偏好设置

![偏好设置](/images/desktop/usage/comfy_desktop_settings_01_preferences.png)

- **语言** — 选择显示语言：**English** 或 **中文**。
- **启动时自动重启上一个实例** — 启动时自动运行上次使用的实例。
- **关闭最后一个窗口时退出 Comfy Desktop（不返回 Dashboard）** — 关闭最后一个窗口时完全退出应用。
- **在实例选择器中隐藏云实例** — 在 Chooser 视图中隐藏云实例。
- **分享匿名使用遥测数据** — 发送匿名使用数据帮助改进 Comfy Desktop。
- **Comfy Desktop** — 链接到 ComfyDesktop GitHub 仓库。

## 更新

![更新](/images/desktop/usage/comfy_desktop_settings_02_updates.png)

- **状态** — 显示是否为最新版本、当前安装版本和上次检查时间。
- **检查更新** — 手动检查新版本。有更新时按钮变为**下载** → **安装**。
- **自动安装桌面端更新** — 后台自动下载并安装更新。

## 存储

![存储](/images/desktop/usage/comfy_desktop_settings_03_storage.png)

- **共享模型** — 模型扫描目录。首个目录为**主要**（新模型默认保存于此）。
  - 点击文件夹图标在文件管理器中打开目录。
  - 使用 **...** 菜单更改排序或设为主要目录。
  - 点击 **+ 添加目录** 添加更多模型目录。
- **输入目录** — ComfyUI 查找输入文件（图片、视频等）的目录。
- **输出目录** — ComfyUI 保存生成结果的目录。可自定义路径集中管理输出。
- 点击文件夹图标浏览选择新路径。**?** 工具提示提供更多信息。

## 高级

![高级](/images/desktop/usage/comfy_desktop_settings_04_advanced.png)

- **默认安装位置** — 新实例的创建目录。点击文件夹图标更改。
- **PyPI 镜像 URL** — 自定义 pip 镜像地址（如 `https://mirrors.aliyun.com/pypi/simple/`）。
- **使用中国镜像（Git & PyPI）** — 对 Git 和 PyPI 使用中国镜像服务器。
- **缓存目录** — Comfy Desktop 存储临时和缓存数据的目录。点击文件夹图标更改。

## 数据存储位置

Comfy Desktop 将数据存储在平台特定的目录中：

### Windows
```
应用数据:  %APPDATA%\Comfy Desktop
日志:      %APPDATA%\Comfy Desktop\logs
```

### macOS
```
应用数据:  ~/Library/Application Support/Comfy Desktop
日志:      ~/Library/Application Support/Comfy Desktop/logs
```

### Linux（XDG 基础目录）
```
配置:  ~/.config/Comfy Desktop
数据:  ~/.local/share/Comfy Desktop
缓存:  ~/.cache/Comfy Desktop
状态:  ~/.local/state/Comfy Desktop
```

## 键盘快捷键

| 快捷键 | 操作 |
|--------|------|
| `Ctrl+R` / `Cmd+R` | 刷新当前视图 |
| `Ctrl+W` / `Cmd+W` | 关闭当前面板 |
| `Escape` | 关闭模态框或覆盖层 |
| `Ctrl+,` / `Cmd+,` | 打开全局设置 |

## Deep Links (comfy://)

Comfy Desktop 支持 `comfy://` 自定义协议，可直接从浏览器或终端打开本地工作流文件。

### 通过 Deep Link 打开工作流

```
comfy://open?file=/path/to/workflow.json
```

点击 `comfy://` 链接或从命令行运行将：
1. 启动 Comfy Desktop（如果尚未运行）
2. 在 ComfyUI 编辑器中打开指定的工作流文件
3. 聚焦现有窗口（如果已在运行）

> **注意：** Deep links 仅支持本地文件路径。此功能需要 Comfy Desktop 已注册为协议处理器——安装时会自动注册。
