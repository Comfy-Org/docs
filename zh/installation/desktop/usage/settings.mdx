从左侧栏的齿轮图标（⚙️）进入全局设置。

## 通用

- **主题** — 选择系统跟随、浅色或深色
- **自动更新** — 启用/禁用自动更新检查
- **使用数据** — 选择是否参与匿名使用数据收集

## 代理 / 镜像设置

适用于网络访问受限地区的用户：
- **Python Install Mirror** — 替换 Astral python-build-standalone 下载地址
- **Pypi Install Mirror** — 设置自定义 pip 镜像
- **Torch Install Mirror** — 设置自定义 PyTorch 下载镜像

## 数据存储位置

Comfy Desktop 将数据存储在平台特定的目录中：

### Windows
```
应用数据:  %APPDATA%\Comfy Desktop
日志:      %APPDATA%\Comfy Desktop\logs
```

### macOS
```
应用数据:  ~/Library/Application Support/Comfy Desktop
日志:      ~/Library/Application Support/Comfy Desktop/logs
```

### Linux（XDG 基础目录）
```
配置:  ~/.config/Comfy Desktop
数据:  ~/.local/share/Comfy Desktop
缓存:  ~/.cache/Comfy Desktop
状态:  ~/.local/state/Comfy Desktop
```

## 键盘快捷键

| 快捷键 | 操作 |
|--------|------|
| `Ctrl+R` / `Cmd+R` | 刷新当前视图 |
| `Ctrl+W` / `Cmd+W` | 关闭当前面板 |
| `Escape` | 关闭模态框或覆盖层 |
| `Ctrl+,` / `Cmd+,` | 打开全局设置 |

## Deep Links (comfy://)

Comfy Desktop 支持 `comfy://` 自定义协议，可直接从浏览器或终端打开本地工作流文件。

### 通过 Deep Link 打开工作流

```
comfy://open?file=/path/to/workflow.json
```

点击 `comfy://` 链接或从命令行运行将：
1. 启动 Comfy Desktop（如果尚未运行）
2. 在 ComfyUI 编辑器中打开指定的工作流文件
3. 聚焦现有窗口（如果已在运行）

> **注意：** Deep links 仅支持本地文件路径。此功能需要 Comfy Desktop 已注册为协议处理器——安装时会自动注册。
