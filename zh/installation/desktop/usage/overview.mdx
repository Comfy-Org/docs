Comfy Desktop 是 **ComfyUI** 的官方桌面应用。它替你打理一切——安装 ComfyUI、保持更新，并让你并行运行多个互不干扰的实例。

两者之间的关系是：**Comfy Desktop** 是管理并启动 **ComfyUI** 实例的应用。你不用单独安装 ComfyUI——Comfy Desktop 会自动处理每个实例的安装、Python 环境和依赖。

### 主要特性

- **[多实例管理](/zh/installation/desktop/usage/instance-management)** — 并行运行多个 ComfyUI 实例，各自拥有独立的版本、自定义节点和配置
- **[快照](/zh/installation/desktop/usage/manage#snapshots-tab)** — 保存和恢复实例的完整状态（ComfyUI 版本、自定义节点、Python 包），方便回滚或复现环境
- **[共享存储](/zh/installation/desktop/usage/settings#storage)** — 模型默认在所有实例间共享，节省磁盘空间。也可以添加外部模型文件夹
- **[自动更新](/zh/installation/desktop/usage/manage#update-tab)** — 通过 Manage 面板分别更新 Comfy Desktop 本身和各个 ComfyUI 实例
- **云集成** — 直接从 Chooser 视图启动或连接 Comfy Cloud 实例
- **[追踪现有安装](/zh/installation/desktop/usage/instance-management#tracking-an-existing-installation)** — 将已有的 portable 或 git clone 的 ComfyUI 安装加入 Comfy Desktop 统一管理

本篇指南介绍如何使用这些功能。

## Dashboard 总览

Comfy Desktop 窗口分为几个关键区域。下图标出了 Dashboard 的主要组成部分：

![Comfy Desktop Dashboard](/images/desktop/interface/interface_desktop_dashboard.png)

### 应用菜单

左上角的汉堡菜单（☰）打开应用菜单（File / Edit / View / Window / Help）。可以新开窗口、访问全局偏好设置、退出应用等桌面级操作。

  <Card title="设置" icon="cog" href="/zh/installation/desktop/usage/settings">
    全局设置：主题、自动更新、代理/镜像配置和数据位置。
  </Card>
</CardGroup>

### 实例切换器

中间胶囊按钮显示当前激活的 ComfyUI 实例名称。点击它打开**实例选择器**弹窗，列出所有已安装的实例及其运行状态，方便切换或启动新实例。

<CardGroup cols={1}>
  <Card title="管理安装" icon="sliders" href="/zh/installation/desktop/usage/manage">
    通过 Manage 面板启动、更新、快照和配置安装。
  </Card>
</CardGroup>

### 右侧操作栏

标题栏右侧的按钮组提供快捷入口：

- **Desktop Update**：当 Comfy Desktop 有新版本时，这里出现更新按钮。点击即可下载并应用更新。
- **Feedback**：向 Comfy Desktop 团队提交反馈。
- **Downloads**：打开下载管理托盘，查看进行中和近期的下载任务。

### 搜索框

搜索框可按名称筛选已安装的实例，在实例较多时快速找到目标。

### Chooser（实例列表）

Dashboard 的主区域称为 **Chooser**，以卡片形式展示所有 ComfyUI 实例。**点击任意实例卡片即可启动或切换到该实例。**

你可以在这里：

- **＋ New Install**：创建新的本地独立 ComfyUI 实例。
- **Comfy Cloud**：连接或启动云实例。
- **已有实例卡片**：每张卡片显示实例名称、状态和类型。右键（或点击 ⋮ 图标）可管理、更新、快照或删除实例。
