本篇指南介绍 Comfy Desktop —— 负责管理 ComfyUI 安装的桌面应用。如果你在找如何搭建工作流或使用 ComfyUI 节点的教程，请查看 [ComfyUI 编辑器](/comfyui-editor)部分。

Comfy Desktop 让你在同一个地方安装、启动和管理多个 ComfyUI 实例，每个实例拥有独立的 Python 环境、模型和自定义节点。
