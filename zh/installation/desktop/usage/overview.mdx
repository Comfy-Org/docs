本指南介绍 Comfy Desktop 安装后的使用方法。

![Comfy Desktop 首页](/images/desktop/usage/01-homepage.png)

打开 Comfy Desktop 后看到的画面就是 **Chooser** 视图，所有 ComfyUI 安装以卡片网格展示。

## Chooser 视图

**Chooser** 是打开 Comfy Desktop 时看到的主屏幕。它以网格形式显示所有 ComfyUI 安装。

### 安装卡片

每张卡片显示：
- **名称** — 你为安装指定的名称
- **类型图标** — 表示来源类型（独立、云、Git、旧版 Desktop、远程）
- **上次启动时间** — 距离上次使用多久
- **状态** — 运行中、错误或有可用更新

### 筛选

使用网格顶部的筛选标签来缩小范围：

- **All（全部）** — 显示所有安装
- **Local（本地）** — 仅独立安装和旧版 Desktop 安装
- **Cloud（云）** — 仅 Comfy Cloud 连接
- **Remote（远程）** — 仅远程服务器连接
