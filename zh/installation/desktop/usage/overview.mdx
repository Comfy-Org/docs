本指南介绍 Comfy Desktop 安装完成后的使用方法。

![Comfy Desktop 首页](/images/desktop/usage/01-homepage.png)

打开 Comfy Desktop 后你会看到 **Chooser**——一个展示所有 ComfyUI 安装的卡片网格。顶部的过滤标签可以按本地、云端或远程筛选。点击任意卡片启动它，或点击 **+** 添加新安装。
