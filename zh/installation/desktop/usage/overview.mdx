本指南介绍 Comfy Desktop 安装完成后的使用方法。

![Comfy Desktop 首页](/images/desktop/usage/01-homepage.png)

安装并启动 Comfy Desktop 后，你会看到所有 ComfyUI 实例以卡片形式展示。从这里你可以启动任意实例、添加新实例、管理设置等。

本指南包含以下部分：

- **[实例管理](/zh/installation/desktop/usage/instance-management)** — 创建、重命名、编辑和删除 ComfyUI 实例。
- **[管理安装](/zh/installation/desktop/usage/manage)** — 通过 Manage 面板启动、更新、快照和配置安装。
- **[设置](/zh/installation/desktop/usage/settings)** — 全局设置——主题、自动更新、代理/镜像配置和数据位置。
- **[从旧版迁移](/zh/installation/desktop/usage/migrate)** — 从 Desktop Legacy 升级到 Comfy Desktop。
