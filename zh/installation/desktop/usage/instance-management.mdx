Comfy Desktop 允许你在 Chooser 视图中管理多个 ComfyUI 实例。下图标出了每个实例卡片的常用操作入口：

![实例管理操作](/images/desktop/interface/interface_instance_menu.png)

1. **New Install** — 创建全新的 ComfyUI 实例。

<Card title="创建新实例" icon="plus" href="#创建新实例">
  New Install
</Card>

2. **⋮ 菜单** — 点击三点图标打开实例操作菜单。
3. **Manage** — 打开管理面板配置实例（路径、启动参数、模型目录、环境变量）。

<Card title="管理" icon="sliders" href="/zh/installation/desktop/usage/manage">
  Manage
</Card>

4. **Restore Snapshot** — 将实例恢复到之前的快照。

<Card title="快照管理" icon="camera" href="/zh/installation/desktop/usage/snapshots">
  Restore Snapshot
</Card>

5. **Share** — 导出实例快照分享给他人。
6. **Copy Installation** — 复制现有独立安装，包含独立的 Python 环境和自定义节点。
7. **Forget** — 从 Chooser 中移除实例但不删除文件（等同于 Untrack）。
8. **Uninstall** — 从磁盘永久删除实例及其所有文件。

## 创建新实例

在 Chooser 视图中点击左上角的 **+** 卡片添加新的 ComfyUI 实例。

<Steps>
  <Step title="配置 Comfy Desktop">

<Tabs>
  <Tab title="Windows">
    ![在 Windows 上配置 Comfy Desktop](/images/desktop/usage/new_install_01_settings_windows.png)
  </Tab>
  <Tab title="macOS">
    ![在 macOS 上配置 Comfy Desktop](/images/desktop/usage/new_install_01_settings_mac.png)
  </Tab>
</Tabs>

1. **实例名称** — 你的实例名称。默认是 "ComfyUI"。
2. **安装位置** — 选择 ComfyUI 的安装位置。Windows 用户强烈建议使用**固态硬盘**，避免安装在 C 盘根目录。
3. **高级（Advanced）** — 点击展开以访问更多选项。
4. **独立安装（Standalone）** — 推荐选项。自带 Python、自动更新、支持快照。
5. **远程连接（Remote Connection）** — 连接到另一台机器上运行的 ComfyUI。

## 删除实例

要删除实例，右键点击实例卡片（或点击 ⋮ 图标）并选择 **Delete**。

- **Delete** 会从磁盘上永久删除安装及所有文件。共享目录中的模型文件不受影响。
- **Untrack**（在右键菜单中可用）仅将安装从 Chooser 中移除，不会删除任何文件——适合只想清理列表而不想丢失数据的情况。

## 重命名实例

点击 Chooser 中实例名称旁的铅笔图标（✏️）即可重命名。

## 复制安装

右键点击安装卡片并选择 **Copy Installation**，可以复制现有的独立安装。复制后的安装包含：
- 新的 Python 环境
- 相同的 ComfyUI 版本
- 独立的自定义节点配置

开始复制前会提示输入新安装的名称。

## 编辑实例

在实例的 **Manage** 面板中，你可以：

- 更改安装路径
- 调整启动参数
- 修改模型目录
- 更新环境变量

右键点击实例卡片并选择 **Manage** 打开管理面板。

## 错误状态

如果安装遇到问题，卡片上会显示警告标识。右键点击卡片并选择 **Dismiss Error** 清除错误状态。如果问题持续存在，请查看[数据位置](/zh/installation/desktop/usage/settings#data-locations)中的日志。
