## 管理面板

在实例卡片的 ⋮ 菜单中选择 **Manage** 打开设置面板。面板包含六个标签页：

### 更新

![更新](/images/desktop/usage/instance_settings_01_update.png)

截图显示 Update 标签页：当前版本信息、「检查更新」按钮和更新频道下拉菜单（选择 Stable — Current）。

- **版本** — 显示当前已安装版本及是否为最新。
- **上次检查** — 显示上次检查更新时间。
- **检查更新** — 手动检查新版本。
- **更新频道** — 从下拉菜单选择：
  - **Stable — Current** — 经过测试的版本，推荐大多数用户使用。

### 启动参数

![启动参数](/images/desktop/usage/instance_settings_02_startup_args.png)

截图显示启动参数配置：启动参数框（`--enable-manager`）、启动模式选「应用窗口」、浏览器缓存选「此安装独有」、以及已添加的一个环境变量（`COMFYUI_VER`）。

- **启动参数** — 自定义 CLI 参数（如 `--enable-manager`）。点击齿轮图标查看选项。
- **启动模式** — 选择 **应用窗口** 或 **浏览器标签页**。
- **浏览器缓存** — **此安装独有** 或 **跨实例共享**。
- **端口被占用时** — **询问如何处理**、**自动选择不同端口** 或 **启动失败**。
- **环境变量** — 点击 **+ 添加变量** 添加自定义环境变量。

### 快照

![快照](/images/desktop/usage/instance_settings_03_snapshots.png)

截图显示快照时间线：两条快照记录。最新一条高亮显示（v0.20.1 → v0.24.1），附带导出/删除按钮和包统计。

- **时间线** — 按时间顺序列出所有快照，显示版本变更和统计。
- **最新** — 显示最近快照的创建时间。
- **导入** — 从文件导入快照。
- **导出全部** — 备份导出所有快照。
- **创建快照** — 随时手动创建新快照。
- 每个快照条目显示：
  - 版本变更（如 `v0.20.1 → v0.24.1`）
  - 包变更数
  - 节点和包数量统计
  - **导出** 和 **删除** 按钮

### 存储

![存储](/images/desktop/usage/instance_settings_04_storage.png)

截图显示两个开关（使用共享模型、使用共享输入/输出文件夹）、共享模型列表（一个 Primary 目录）以及输入/输出目录路径。

- **使用共享模型** — 切换此实例是否使用共享模型目录。
- **使用共享输入/输出文件夹** — 切换此实例是否使用共享输入输出路径。
- **共享模型** — 列出所有共享模型目录。**主要**目录为新模型默认保存位置。详见[设置存储](/zh/installation/desktop/usage/settings#storage)。
- **输入目录** — 此实例查找输入文件的目录。
- **输出目录** — 此实例保存生成结果的目录。

### 控制台

![控制台](/images/desktop/usage/instance_settings_05_console.png)

截图显示实时终端，输出 ComfyUI 后端启动日志，包括 Python 版本、硬件检测和加载的节点数。

实时终端，显示 ComfyUI 后端进程输出。
- 可用于调试——实时查看启动日志、错误和 Python 输出。

### 关于

![关于](/images/desktop/usage/instance_settings_05_about.png)

截图显示当前版本、硬件信息（Apple Silicon MPS）、Python 和 PyTorch 版本、安装路径以及磁盘使用量。

- **当前版本** — 已安装的 ComfyUI 版本。
- **硬件** — 检测到的硬件（如 `Apple Silicon (MPS)`）。
- **Python** — 捆绑的 Python 版本。
- **PyTorch** — 包含的 PyTorch 版本。
- **位置** — 磁盘上的完整安装路径。点击复制按钮复制。
- **磁盘使用量** — 安装占用的磁盘空间。
