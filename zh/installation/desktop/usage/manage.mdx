## 启动 ComfyUI

点击任意安装卡片即可启动。Comfy Desktop 将：
1. 启动 ComfyUI 后端进程
2. 等待其就绪
3. 打开 ComfyUI 编辑器窗口

如果安装已在运行，点击卡片会将现有窗口置于前台，而不是启动新实例。

### 运行多个实例

你可以同时启动多个安装。每个独立运行：
- 各自拥有独立的 Python 环境
- 各自拥有独立的 ComfyUI 版本和自定义节点
- 各自使用不同的端口
- 各自打开独立的窗口

## 安装操作

右键点击安装卡片（或点击 ⋮ 图标）访问操作：

| 操作 | 说明 |
|------|------|
| **Launch / Restart** | 启动或重启 ComfyUI 实例 |
| **Manage** | 打开此安装的完整设置面板 |
| **Update** | 将 ComfyUI 更新到最新版本 |
| **Restore Snapshot** | 恢复到已保存的快照 |
| **Open Folder** | 在文件管理器中打开安装目录 |
| **Share** | 将最新快照导出为文件分享给他人 |
| **Copy Installation** | 复制一个独立安装，包含新的 Python 环境和 ComfyUI 版本 |
| **Untrack** | 从 Chooser 中移除安装，不删除任何文件 |
| **Delete** | 永久删除此安装及其所有文件（共享目录中的模型文件不受影响） |
| **Dismiss Error** | 清除安装的错误状态，移除警告标识 |

## 管理面板

选择 **Manage** 后会打开设置面板，包含以下标签页：

### Status（状态）
显示：
- 当前 ComfyUI 版本
- 磁盘空间使用情况
- 上次启动时间
- Launch / Stop / Restart 控制

### Update（更新）
- 检查新版本
- 一键更新安装
- 显示更新日志

### Snapshots（快照）
- 查看所有保存的快照
- 恢复到任意历史状态
- 手动创建快照
- 快照对比（显示版本间差异）

### Settings（设置）
配置每个安装的 ComfyUI 参数：
- **ComfyUI arguments** — 自定义 CLI 参数（如 `--gpu-only`、`--highvram`）
- **Environment variables** — 设置环境变量
- **Model directories** — 添加外部模型目录
- **Launch settings** — 配置额外端口、队列大小等

## 编辑安装

创建后仍可编辑安装的名称、路径和配置。点击安装名称旁的铅笔图标（✏️）即可重命名。

## 快照

Comfy Desktop 在每次更新前自动创建**快照**。快照记录 ComfyUI 版本、所有已安装的自定义节点和 Python 包列表。

> 快照仅包含 ComfyUI 版本信息和自定义节点元数据，不包含模型文件。

关于创建、恢复、导出和导入快照的完整说明，请查看[快照管理](/zh/installation/desktop/usage/snapshots)页面。
