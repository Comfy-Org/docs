**ComfyUI Desktop** 是一个多实例管理器，让您从一个启动器运行多个 ComfyUI 实例。本指南介绍如何在 macOS 上安装和卸载 ComfyUI Desktop。

## 系统要求

- **系统：** macOS 13 (Ventura) 或更新
- **硬件：** Apple Silicon（M1 或更新）
- **磁盘空间：** 每个安装至少推荐 15 GB

## 安装

1. 打开下载的 `.dmg` 文件
2. 将 **ComfyUI Desktop** 拖入 **Applications** 文件夹
3. 从 Launchpad 或 Spotlight 启动

首次启动时，macOS 可能会显示安全警告。前往 **系统设置 → 隐私与安全性**，点击 **仍要打开**。

应用打开后会显示 **欢迎** 页面，在此创建您的第一个安装。

## 卸载

### 卸载 ComfyUI Desktop

将 **ComfyUI Desktop** 从 Applications 文件夹拖到废纸篓。

### 删除安装数据

ComfyUI Desktop 和它的安装在磁盘上是分开存储的。要完全清理：

1. **从应用内删除单个安装**，或手动删除安装文件夹
2. **删除应用数据文件夹** `~/Library/Application Support/ComfyUI Desktop`

## 数据路径

| 数据 | macOS 路径 |
|------|-----------|
| 应用数据（设置、安装列表） | `~/Library/Application Support/ComfyUI Desktop` |
| 日志 | `~/Library/Application Support/ComfyUI Desktop/logs` |
| 安装目录 | 用户在设置时选择 |

## 故障排除

### macOS 安全警告

如果 macOS 阻止应用启动，前往 **系统设置 → 隐私与安全性**，滚动到底部，点击 ComfyUI 旁的 **仍要打开**。

### 镜像错误

如果镜像下载失败，展开镜像设置配置自定义的 Python、PyPI 和 PyTorch 镜像源。
