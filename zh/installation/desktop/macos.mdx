**ComfyUI Desktop** 是一个多实例管理器，让你从一个启动器运行多个 ComfyUI 实例。本指南将引导你在 macOS 上安装和使用 ComfyUI Desktop。

ComfyUI Desktop 是一个开源项目。在 [GitHub](https://github.com/Comfy-Org/Comfy-Desktop) 上查看完整代码。

## 系统要求

- **系统：** macOS 13 (Ventura) 或更新
- **硬件：** Apple Silicon（M1 或更新）
- **磁盘空间：** 每个安装建议至少 15 GB

## 下载

从[官方下载页面](https://comfy.org/download)下载 ComfyUI Desktop——安装程序会自动检测你的平台。

## 安装

### 第一步 — 安装应用

1. 打开下载的 `.dmg` 文件
2. 将 **ComfyUI Desktop** 拖入 **Applications** 文件夹
3. 从 Launchpad 或 Spotlight 启动

### 第二步 — 首次启动

首次启动时，macOS 可能会显示安全警告（应用未经公证）。前往 **系统设置 → 隐私与安全性**，点击 **仍要打开** 以允许。

启动后你会看到**欢迎**屏幕。点击 **Get Started** 开始。

## 数据位置

| 数据类型 | macOS 路径 |
|---------|-----------|
| 应用数据（设置、安装列表） | `~/Library/Application Support/ComfyUI Desktop` |
| 日志 | `~/Library/Application Support/ComfyUI Desktop/logs` |
| 默认安装目录 | 用户在设置时选择 |

## 卸载

要卸载 ComfyUI Desktop：
1. 从 Applications 文件夹中删除 **ComfyUI Desktop**
2. 要删除所有数据，删除 `~/Library/Application Support/ComfyUI Desktop`
3. 删除安装过程中创建的任何安装目录

## 故障排除

### macOS 安全警告

如果 macOS 阻止应用运行，前往 **系统设置 → 隐私与安全性**，滚动到底部，点击 ComfyUI 条目旁的**仍要打开**。
