**Comfy Desktop** 是一个多实例管理器，让您从一个启动器运行多个 ComfyUI 实例。本指南介绍如何在 macOS 上安装和卸载 Comfy Desktop。

## 系统要求

- **系统：** macOS 13 (Ventura) 或更新版本
- **硬件：** Apple Silicon（M1 或更新）
- **磁盘空间：** 每个安装至少推荐 4.85 GB

<Card title="下载 Comfy Desktop" icon="download" href="https://comfy.org/download">
    下载 macOS 最新安装程序。页面会自动检测您的平台。
</Card>

## 安装

1. 打开下载的 `.dmg` 文件
2. 将 **Comfy Desktop** 拖入 **Applications** 文件夹
3. 从 Launchpad 或 Spotlight 启动

首次启动时，macOS 可能显示安全警告。前往 **系统设置 → 隐私与安全性**，点击 **仍要打开** 以允许运行。

应用打开后，您将看到 **欢迎** 页面，在此创建您的第一个安装。

## 卸载

### 卸载 Comfy Desktop

1. 在访达中打开 **Applications** 文件夹
2. 右键点击 **Comfy Desktop**，选择 **移到废纸篓**

![在 macOS 上卸载 Comfy Desktop](/images/desktop/installation/mac_2_uninstall.png)

### 卸载应用不会删除的内容

卸载 Comfy Desktop 只会移除启动器本身。以下数据**不会被删除**，如需完全清理请手动删除：

| 数据 | 位置 | 说明 |
|------|------|------|
| 安装实例 | `~/ComfyUI-Installs` | 每个 ComfyUI 安装（也可在应用内删除） |
| 共享模型和输出 | `~/ComfyUI-Shared` | 共享模型库、输入/输出目录 |
| 应用设置和记录 | `~/Library/Application Support/Comfy Desktop` | 设置、安装记录、日志 |

要完全清除所有数据：

1. **删除应用数据文件夹** – `~/Library/Application Support/Comfy Desktop`
2. **从应用内删除单个安装**，或手动删除 `~/ComfyUI-Installs`
3. **删除共享模型库** `~/ComfyUI-Shared`（仅当不再需要这些文件时）
