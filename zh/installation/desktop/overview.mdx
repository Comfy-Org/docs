**ComfyUI Desktop** 是一款新一代桌面应用，让你从一个地方安装、管理和启动多个 ComfyUI 实例。不同于旧的 Desktop 版本（单一实例），ComfyUI Desktop 是一个多实例管理器——可以理解为一个启动所有 ComfyUI 环境的启动器。

## 主要功能

### 多实例管理

ComfyUI Desktop 支持多种类型的 ComfyUI 安装：

| 来源类型 | 说明 |
|---------|------|
| **Standalone（独立安装）** | 自包含的安装，带有自己的 Python 和依赖——干净、隔离、易于管理 |
| **Cloud（云）** | 连接到 Comfy Cloud，使用远程 GPU |
| **Git Clone** | 跟踪现有的 ComfyUI Git 仓库——适合开发者 |
| **Portable** | 导入现有的 ComfyUI 便携版安装（Windows） |
| **Legacy Desktop** | 检测并迁移旧的 Desktop Legacy 安装 |
| **远程连接** | 连接到其他机器上运行的 ComfyUI 实例 |

### 你可以做什么

- **快速安装** — 一键安装全新的 ComfyUI，自动配置 Python 环境
- **运行多个实例** — 同时启动和切换不同版本的 ComfyUI
- **快照（Snapshots）** — 每次更新前自动保存自定义节点、模型和设置；随时恢复到任意快照
- **一键更新** — 点击一次即可更新 ComfyUI
- **模型下载管理** — 内置模型下载管理器，带进度跟踪
- **应用内终端** — 直接在应用内访问 ComfyUI Python 环境
- **全局设置** — 配置主题、代理镜像、匿名使用数据

## 工作原理

ComfyUI Desktop 将**启动器**与**工作流编辑器**分开。应用管理你的安装；每个安装运行自己的 ComfyUI 后端（和自己的 Python 环境）。当你启动一个安装时，它会在独立的窗口中打开完整的 ComfyUI 编辑器。

## 系统要求

### Windows
- **系统：** Windows 10 或更新
- **GPU：** 支持 CUDA 的 NVIDIA GPU
- **架构：** x64 或 ARM64

### macOS
- **系统：** macOS 13 (Ventura) 或更新
- **硬件：** Apple Silicon（M1 或更新）

### Linux
- **系统：** Ubuntu 22.04+（基于 Debian 的发行版）
- **GPU：** 推荐使用支持 CUDA 的 NVIDIA GPU

### 通用
- **磁盘空间：** 每个独立安装至少 15 GB
- **内存：** 最低 8 GB，推荐 16 GB
- **网络：** 安装和更新需要联网

## 开源

ComfyUI Desktop 完全开源。在 [GitHub](https://github.com/Comfy-Org/Comfy-Desktop) 上查看源代码。
