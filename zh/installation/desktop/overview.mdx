**Comfy Desktop** 是一个桌面应用，让你从一个启动器中安装、管理和启动多个 ComfyUI 实例。

![Desktop 截图](/images/desktop/usage/01-homepage.png)

## 工作原理

Comfy Desktop 将**启动器**与**工作流编辑器**分开。每个安装独立运行自己的 ComfyUI 后端，拥有自己的 Python 环境、自定义节点和设置。启动一个安装时，它会在独立的窗口中打开完整的 ComfyUI 编辑器。

```mermaid
flowchart LR
    A[Comfy Desktop 启动器] --> B[安装 1<br/>最新版]
    A --> C[安装 2<br/>v0.2.4]
    B --> F[ComfyUI 编辑窗口]
    C --> G[ComfyUI 编辑窗口]
```

## 系统要求

<CardGroup cols={3}>
  <Card title="Windows" icon="windows">
    - **系统：** Windows 10 或更新
    - **架构：** x64 或 ARM64
    - **GPU：** 推荐使用独立 GPU，但不是必须
  </Card>

  <Card title="macOS" icon="app-store">
    - **系统：** macOS 13 (Ventura) 或更新
    - **硬件：** Apple Silicon（M1 或更新）
  </Card>

  <Card title="Linux" icon="linux">
    - **系统：** 基于 Debian（推荐 Ubuntu 22.04+）
    - **GPU：** 推荐使用独立 GPU，但不是必须
  </Card>
</CardGroup>

- **磁盘空间：** 每个独立安装至少 4.85 GB
- **内存：** 最低 8 GB，推荐 16 GB
- **网络：** 安装和更新需要联网
