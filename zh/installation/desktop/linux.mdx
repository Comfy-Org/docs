**ComfyUI Desktop** 是一个多实例管理器，让你从一个启动器运行多个 ComfyUI 实例。本指南将引导你在 Linux 上安装 ComfyUI Desktop。

ComfyUI Desktop 是一个开源项目。在 [GitHub](https://github.com/Comfy-Org/Comfy-Desktop) 上查看完整代码。

## 系统要求

- **系统：** Ubuntu 22.04+（基于 Debian 的发行版）
- **GPU：** 推荐使用支持 CUDA 的 NVIDIA GPU
- **磁盘空间：** 每个安装建议至少 15 GB

### 通过 .deb 安装

```bash
sudo dpkg -i comfyui-desktop_*.deb
```

### 通过 AppImage 安装

```bash
chmod +x comfyui-desktop-*.AppImage
./comfyui-desktop-*.AppImage
```

## 数据位置

ComfyUI Desktop 在 Linux 上遵循 XDG 基础目录规范：

| 数据类型 | Linux 路径 |
|---------|-----------|
| 配置 | `~/.config/comfyui-desktop` |
| 数据 | `~/.local/share/comfyui-desktop` |
| 缓存 | `~/.cache/comfyui-desktop` |
| 状态 | `~/.local/state/comfyui-desktop` |
| 默认安装目录 | 用户在设置时选择 |
