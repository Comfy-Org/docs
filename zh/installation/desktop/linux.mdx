**ComfyUI Desktop** 是一个多实例管理器，让您从一个启动器运行多个 ComfyUI 实例。本指南介绍如何在 Linux 上安装和卸载 ComfyUI Desktop。

## 系统要求

- **系统：** 基于 Debian 的发行版（推荐 Ubuntu 22.04+）
- **GPU：** 推荐使用独立 GPU（NVIDIA / AMD）以获得更好性能，但不是必须
- **磁盘空间：** 每个安装至少推荐 15 GB

## 安装

### .deb 包（Debian/Ubuntu）

```bash
sudo apt install ./comfyui-desktop_*.deb
```

### AppImage（其他发行版）

```bash
chmod +x ./comfyui-desktop-*.AppImage
./comfyui-desktop-*.AppImage --no-sandbox
```

安装后从应用菜单启动 ComfyUI Desktop。首次打开会显示 **欢迎** 页面，在此创建您的第一个安装。

## 卸载

### 卸载 ComfyUI Desktop

**Debian/Ubuntu：**
```bash
sudo apt remove comfyui-desktop
```

**AppImage：** 直接删除 AppImage 文件即可。

### 删除安装数据

ComfyUI Desktop 和它的安装在磁盘上是分开存储的。要完全清理：

1. **从应用内删除单个安装**，或手动删除安装文件夹
2. **删除应用数据和配置目录：**

| 数据 | 路径 |
|------|------|
| 配置 | `~/.config/comfyui-desktop` |
| 数据 | `~/.local/share/comfyui-desktop` |
| 缓存 | `~/.cache/comfyui-desktop` |
| 状态 | `~/.local/state/comfyui-desktop` |

## 数据路径

ComfyUI Desktop 遵循 XDG Base Directory 规范：

| 数据 | Linux 路径 |
|------|-----------|
| 配置 | `~/.config/comfyui-desktop` |
| 数据 | `~/.local/share/comfyui-desktop` |
| 缓存 | `~/.cache/comfyui-desktop` |
| 状态 | `~/.local/state/comfyui-desktop` |
| 安装目录 | 用户在设置时选择 |

## 故障排除

### AppImage 需要 fuse2

如果 AppImage 无法运行，安装 `fuse2`：

```bash
sudo apt install fuse2
```

### 镜像错误

如果镜像下载失败，展开镜像设置配置自定义的 Python、PyPI 和 PyTorch 镜像源。
