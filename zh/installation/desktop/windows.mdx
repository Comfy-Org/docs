**ComfyUI Desktop** 是一个多实例管理器，让你从一个启动器运行多个 ComfyUI 实例。本指南将引导你在 Windows 上安装和使用 ComfyUI Desktop。

ComfyUI Desktop 是一个开源项目。在 [GitHub](https://github.com/Comfy-Org/Comfy-Desktop) 上查看完整代码。

## 系统要求

- **系统：** Windows 10 或更新
- **GPU：** 支持 CUDA 的 NVIDIA GPU
- **架构：** x64 或 ARM64
- **磁盘空间：** 每个安装建议至少 15 GB

## 下载

从[官方下载页面](https://comfy.org/download)下载 ComfyUI Desktop——安装程序会自动检测你的平台。

## 安装

### 第一步 — 运行安装程序

双击下载的 `.exe` 安装文件。安装程序将设置 ComfyUI Desktop 并在开始菜单创建快捷方式。

### 第二步 — 首次启动

从开始菜单打开 ComfyUI Desktop。首次启动时，你会看到**欢迎**屏幕。这是你设置第一个 ComfyUI 安装的地方。

点击 **Get Started** 开始。

### 第三步 — 选择安装方式

你有两种主要方式：

<AccordionGroup>
  <Accordion title="快速安装（推荐新手使用）">
    选择 **Quick Install** — 这将创建一个全新的独立 ComfyUI 安装：

    1. 选择你的 **GPU**：选择 **NVIDIA GPU（推荐）**
    2. 选择**安装目录**（一个单独的空文件夹）
    3. 配置可选设置：
       - **自动更新**：有新版本时自动更新 ComfyUI
       - **使用数据**：发送匿名使用数据以帮助改进应用
       - **镜像设置**：如果你在访问 GitHub 较慢的地区，展开此项设置自定义 Python 和 PyTorch 镜像
    4. 点击 **Install** — ComfyUI Desktop 将下载 Python、设置环境、安装 ComfyUI 并启动
  </Accordion>
  <Accordion title="跟踪现有安装">
    如果你已有 ComfyUI 安装（Git 克隆、便携版或旧版 Desktop），选择 **Add Existing Install**：

    - **Git Clone** — 指向现有的 ComfyUI Git 仓库文件夹；ComfyUI Desktop 将检测其 Python 虚拟环境
    - **Portable** — 选择现有的 ComfyUI 便携版目录
    - **Legacy Desktop** — ComfyUI Desktop 自动检测现有的旧版 Desktop 安装并提供迁移
  </Accordion>
</AccordionGroup>

## 首次启动

安装后，ComfyUI Desktop 打开 **Chooser** 视图——一个所有 ComfyUI 安装的网格。点击任意安装卡片即可启动。

每张卡片显示：
- 安装名称和类型（独立、云、Git 等）
- 上次启动时间
- 状态指示器（运行中、错误、有更新）

## 设置面板

点击左侧边栏的设置齿轮图标（⚙️）访问三个面板：

### 1. ComfyUI 设置（按安装配置）

在安装卡片上选择 **Manage** 打开。标签包括：

- **Status（状态）** — 查看安装版本、磁盘使用情况，启动/停止 ComfyUI
- **Update（更新）** — 检查并应用 ComfyUI 更新
- **Snapshots（快照）** — 查看、创建和恢复快照
- **Settings（设置）** — 配置 ComfyUI 参数、环境变量和模型目录

### 2. 共享目录

管理所有安装共享的模型和媒体目录：

- **Models（模型）** — 添加外部模型目录
- **Media（媒体）** — 查看输入和输出目录

### 3. 全局设置

配置 ComfyUI Desktop 的全局行为：

- **General（通用）** — 主题（系统/浅色/深色）、自动更新检查、匿名使用数据
- **Proxy / Mirror Settings（代理/镜像设置）** — 自定义 Python 安装、PyPI 和 Torch 镜像（尤其适合中国用户）

## 数据位置

| 数据类型 | Windows 路径 |
|---------|-------------|
| 应用数据（设置、安装列表） | `%APPDATA%\ComfyUI Desktop` |
| 日志 | `%APPDATA%\ComfyUI Desktop\logs` |
| 默认安装目录 | 用户在设置时选择 |

## 更新 ComfyUI Desktop

ComfyUI Desktop 会自动检查更新。你也可以在 **全局设置 → General → Check for Updates** 手动检查。

## 故障排除

### 杀毒软件标记 `uv.exe`

ComfyUI Desktop 使用 [`uv`](https://github.com/astral-sh/uv)（来自 Astral）管理 Python 环境。一些杀毒软件可能将 `uv.exe` 标记为恶意——这是一个**已知的误报**。如果安装因杀毒软件警告而失败，请将 ComfyUI 安装目录加入杀毒软件白名单后重新安装。

### 镜像错误导致安装失败

如果镜像在安装过程中失败，你会看到红色 ❌ 图标。展开镜像设置并配置可用的镜像：
- **Python 安装镜像** — 用于 Astral python-build-standalone 项目
- **Pypi 安装镜像** — 用于 pip 包
- **Torch 安装镜像** — 用于 PyTorch
