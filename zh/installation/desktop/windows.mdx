**Comfy Desktop** 是一个多实例管理器，让您从一个启动器运行多个 ComfyUI 实例。本指南介绍如何在 Windows 上安装和卸载 Comfy Desktop。

## 系统要求

- **系统：** Windows 10 或更新版本
- **架构：** x64 或 ARM64
- **GPU：** 推荐使用独立 GPU（NVIDIA / AMD）以获得更好性能，但不是必须
- **磁盘空间：** 每个安装至少推荐 15 GB

## 安装

运行下载的 `.exe` 安装程序（NSIS 安装器），按照提示完成安装。安装后从开始菜单或桌面快捷方式启动 Comfy Desktop。

首次打开应用时，您将看到 **欢迎** 页面，在此创建您的第一个安装。

## 卸载

### 卸载 Comfy Desktop

进入 **设置 → 应用 → 应用和功能**，找到 Comfy Desktop 并选择卸载；或从开始菜单右键 Comfy Desktop → 卸载。

### 删除安装数据

Comfy Desktop 和它的安装在磁盘上是分开存储的。要完全清理：

1. **从应用内删除单个安装**（右键安装卡片 → 删除），或手动删除安装文件夹
2. **删除应用数据文件夹** `%APPDATA%\Comfy Desktop`

## 数据路径

| 数据 | Windows 路径 |
|------|-------------|
| 应用数据（设置、安装列表） | `%APPDATA%\Comfy Desktop` |
| 日志 | `%APPDATA%\Comfy Desktop\logs` |
| 安装目录 | 用户在设置时选择 |

## 故障排除

### 安装被杀毒软件拦截

Comfy Desktop 使用 [`uv`](https://github.com/astral-sh/uv) 管理 Python 环境。某些杀毒软件可能将 `uv.exe` 误报为威胁——这是**已知的误报**。如果安装失败，请将 ComfyUI 安装目录加入杀毒软件白名单。

### 镜像错误

如果镜像下载失败，界面上会显示红色 ❌ 图标。展开镜像设置，配置自定义的 Python、PyPI 和 PyTorch 镜像源 — 这在某些网络受限地区尤其有用。

### 识别错误

如果安装失败，您会看到以下界面

![Comfy Desktop 安装失败](/images/desktop/win-comfyui-desktop-7.jpg)

建议按照以下步骤查找错误原因：
1. 点击 `显示终端` 查看错误输出
2. 点击 `打开日志` 查看安装日志
3. 访问 [Comfy 官方论坛](https://forum.comfy.org/) 搜索类似错误报告
4. 点击 `重新安装` 重试

在提交反馈之前，建议将 **错误输出** 和 **日志文件** 提供给 AI 工具（如 GPT）进行分析

![Comfy Desktop 安装失败 - 错误日志](/images/desktop/win-comfyui-desktop-8.jpg)
![Comfy Desktop 安装失败 - GPT 反馈](/images/desktop/win-comfyui-desktop-9.jpg)

如上图所示，让 GPT 分析对应错误的原因，或完全卸载 Comfy Desktop 后重新安装。

### 反馈安装失败

如果在安装过程中遇到任何错误，请检查是否有类似的错误报告，或通过以下方式向我们提交错误：

- [GitHub Issues](https://github.com/Comfy-Org/desktop/issues)
- [Comfy 官方论坛](https://forum.comfy.org/)

提交错误报告时，请确保包含以下日志和配置文件，以帮助我们定位和排查问题：

1. **日志文件**

| 文件名 | 描述 | 位置 |
|--------|------|------|
| main.log | 包含 Electron 进程的桌面应用和服务器启动日志 | %APPDATA%\Comfy Desktop\logs |
| comfyui.log | 包含 ComfyUI 正常运行的相关日志，如核心 ComfyUI 进程终端输出 | %APPDATA%\Comfy Desktop\logs |

![Comfy Desktop 日志文件位置](/images/desktop/win-comfyui-desktop-10-logs.jpg)

2. **配置文件**

| 文件名 | 描述 | 位置 |
|--------|------|------|
| extra_model_paths.yaml | 包含 ComfyUI 搜索模型和自定义节点的额外路径 | %APPDATA%\Comfy Desktop |
| config.json | 包含应用配置。此文件不应直接编辑 | %APPDATA%\Comfy Desktop |

![Comfy Desktop 配置文件位置](/images/desktop/win-comfyui-desktop-11-config.jpg)
