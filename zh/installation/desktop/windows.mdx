**Comfy Desktop** 是一个多实例管理器，让您从一个启动器运行多个 ComfyUI 实例。本指南介绍如何在 Windows 上安装和卸载 Comfy Desktop。

## 系统要求

- **系统：** Windows 10 或更新版本
- **架构：** x64 或 ARM64
- **GPU：** 推荐使用独立 GPU（NVIDIA / AMD）以获得更好性能，但不是必须
- **磁盘空间：** 每个安装至少推荐 15 GB

<Card title="下载 Comfy Desktop" icon="download" href="https://comfy.org/download">
    下载 Windows 最新安装程序。页面会自动检测您的平台。
</Card>

## 安装

运行下载的 `.exe` 安装程序（NSIS 安装器），按照提示完成安装。安装后从开始菜单或桌面快捷方式启动 Comfy Desktop。

![Windows setup](/images/desktop/installation/windows_1_install.png)

首次打开应用时，您将看到 **欢迎** 页面，在此创建您的第一个安装。

## 卸载

### 卸载 Comfy Desktop

1. 前往 **设置 → 应用 → 已安装的应用**
2. 找到 **Comfy Desktop** 并点击 **卸载**

![在 Windows 上卸载 Comfy Desktop](/images/desktop/installation/windows_2_uninstall.png)

### 删除安装数据

Comfy Desktop 和它的安装在磁盘上是分开存储的。要完全清理：

1. **从应用内删除单个安装**（右键安装卡片 → 删除），或手动删除安装文件夹
2. **删除应用数据文件夹** `%APPDATA%\Comfy Desktop`
