# 预览 3D（高级）

此节点提供高级 3D 模型预览功能，并输出相机和模型信息。它会将 3D 模型保存到临时文件并在界面中显示，同时将模型数据、相机信息和视口尺寸传递到下游进行进一步处理。

## ## 输入

| 参数 | 描述 | 数据类型 | 是否必需 | 范围 |
|-----------|-------------|-----------|----------|-------|
| `model_3d` | 来自上游 3D 节点的 3D 模型文件。 | FILE3D | 是 | GLB、GLTF、FBX、OBJ、STL、USDZ 或任何支持的 3D 格式 |
| `3D 模型信息` | 可选的模型信息元数据。 | LOAD3DMODELINFO | 否 | - |
| `viewport_state` | 包含相机和模型信息的当前视口状态。 | LOAD3D | 是 | - |
| `相机信息` | 3D 视图的可选相机配置。 | LOAD3DCAMERA | 否 | - |
| `宽度` | 预览的宽度（像素）。 | INT | 是 | 1 到 4096（默认值：1024） |
| `高度` | 预览的高度（像素）。 | INT | 是 | 1 到 4096（默认值：1024） |

## ## 输出

| 输出名称 | 描述 | 数据类型 |
|-------------|-------------|-----------|
| `model_3d` | 从输入传递的 3D 模型文件。 | FILE3D |
| `3D 模型信息` | 模型信息元数据，来自输入或视口状态。 | LOAD3DMODELINFO |
| `相机信息` | 相机配置，来自输入或视口状态。 | LOAD3DCAMERA |
| `宽度` | 预览的宽度（像素）。 | INT |
| `高度` | 预览的高度（像素）。 | INT |
