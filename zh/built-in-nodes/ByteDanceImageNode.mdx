ByteDance 图像节点通过 API 基于文本提示使用 ByteDance 模型生成图像。它允许您选择模型、指定图像尺寸，并控制种子和引导比例等各种生成参数。该节点连接到 ByteDance 的图像生成服务，并返回创建的图像。

## 输入

| 参数 | 描述 | 数据类型 | 是否必需 | 范围 |
| --- | --- | --- | --- | --- |
| `模型` | 用于图像生成的 ByteDance 模型。目前仅有一个模型选项可用。 | STRING | 是 | `"seedream-3-0-t2i-250415"` |
| `提示` | 用于生成图像的文本提示。去除首尾空格后长度至少为 1 个字符。 | STRING | 是 | - |
| `尺寸预设` | 选择推荐尺寸。选择自定义可使用下方的宽度和高度。可用预设由 `RECOMMENDED_PRESETS` 列表定义。 | STRING | 是 | 见描述 |
| `宽度` | 图像的自定义宽度。仅当 `尺寸预设` 设置为 `Custom` 时使用此值。默认值：1024。 | INT | 是 | 512 至 2048（步长 64） |
| `高度` | 图像的自定义高度。仅当 `尺寸预设` 设置为 `Custom` 时使用此值。默认值：1024。 | INT | 是 | 512 至 2048（步长 64） |
| `种子` | 用于生成的种子。默认值：0。 | INT | 否 | 0 至 2147483647（步长 1） |
| `引导尺度` | 数值越高，图像越贴近提示。默认值：2.5。 | FLOAT | 否 | 1.0 至 10.0（步长 0.01） |
| `水印` | 是否在图像上添加"AI 生成"水印。默认值：False。这是一个高级参数。 | BOOLEAN | 否 | True / False |

**关于尺寸参数的说明：** `width` 和 `height` 参数仅在 `size_preset` 设置为 `Custom` 时使用。如果选择了预设尺寸，预设的尺寸将覆盖自定义的宽度和高度值。使用自定义尺寸时，宽度和高度均必须在 512 至 2048 像素之间。

## 输出

| 输出名称 | 描述 | 数据类型 |
| --- | --- | --- |
| `IMAGE` | 从 ByteDance API 返回的生成图像，以张量形式呈现。 | IMAGE |
