# 获取 Splat 计数

获取 Splat 计数节点返回 splat 批次中的 splat（高斯点）总数，该计数是对批次中所有项目求和得出的。它会在提供 splat 数量的同时，将原始 splat 数据原样传递出去。

## ## 输入

| 参数 | 描述 | 数据类型 | 是否必需 | 范围 |
|-----------|-------------|-----------|----------|-------|
| `splat` | 要统计 splat 数量的 splat 数据 | SPLAT | 是 | - |

## ## 输出

| 输出名称 | 描述 | 数据类型 |
|-------------|-------------|-----------|
| `splat` | 原始 splat 数据，原样传递 | SPLAT |
| `count` | 批次中所有 splat 的总数 | INT |
