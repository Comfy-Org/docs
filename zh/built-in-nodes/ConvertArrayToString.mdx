## ## 输入

| 参数 | 描述 | 数据类型 | 是否必填 | 范围 |
|------|------|----------|----------|------|
| `数组` | 要转换为 JSON 字符串的数组 | ARRAY | 是 | - |
| `缩进` | 每级缩进的空格数。0 表示生成紧凑的单行字符串（默认值：2） | INT | 否 | 0 到 8 |

## ## 输出

|----------|------|----------|
| `STRING` | 转换为 JSON 格式字符串的数组 | STRING |
