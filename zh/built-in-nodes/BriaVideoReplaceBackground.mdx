# Bria 视频背景替换

此节点使用 Bria 的 API，将视频背景替换为提供的图像或视频。输出内容保持前景视频的分辨率和帧率；若背景素材的宽高比不同，则会被拉伸以适配，因此匹配宽高比可产生无失真的结果。

## ## 输入

| 参数 | 描述 | 数据类型 | 是否必需 | 取值范围 |
|-----------|-------------|-----------|----------|-------|
| `video` | 需要替换背景的前景视频。 | VIDEO | 是 | - |
| `background_image` | 合成到前景后的背景图像。请提供背景图像或背景视频，不可同时提供两者。 | IMAGE | 否 | - |
| `background_video` | 合成到前景后的背景视频。请提供背景图像或背景视频，不可同时提供两者。 | VIDEO | 否 | - |
| `seed` | 种子值控制节点是否重新运行；无论种子值如何，结果均具有非确定性。（默认值：0） | INT | 是 | 0 至 2147483647 |

**注意：** 您必须且仅能提供 `background_image` 或 `background_video` 中的一项——不可同时提供，也不可两者均不提供。前景视频时长不得超过 60 秒。

## ## 输出

| 输出名称 | 描述 | 数据类型 |
|-------------|-------------|-----------|
| `video` | 背景替换后的最终视频。 | VIDEO |
