# ## 概述

VPScheduler 节点旨在基于方差保持（VP）调度方法生成一系列噪声水平（sigma 值）。该序列对于指导扩散模型中的去噪过程至关重要，可实现图像或其他数据类型的受控生成。

# ## 输入

| 参数 | 数据类型 | 描述 |
|------|----------|------|
| `步数` | INT | 指定扩散过程中的步数，影响生成噪声水平的粒度。 |
| `beta_d` | FLOAT | 决定整体噪声水平分布，影响生成噪声水平的方差。 |
| `beta_min` | FLOAT | 设置噪声水平的最小边界，确保噪声不低于特定阈值。 |
| `eps_s` | FLOAT | 调整起始 epsilon 值，微调扩散过程中的初始噪声水平。 |

# ## 输出

| 参数 | 数据类型 | 描述 |
|------|----------|------|
| `sigmas` | SIGMAS | 基于 VP 调度方法生成的一系列噪声水平（sigma 值），用于指导扩散模型中的去噪过程。 |
