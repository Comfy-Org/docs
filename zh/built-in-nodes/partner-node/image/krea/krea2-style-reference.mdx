Krea 2 Style Reference 节点为 Krea 2 生成添加图像风格参考。风格迁移内置在 Krea 2 模型中——传入参考图像后，模型会提取其美学本质（色彩搭配、光影、质感、构图），在生成过程中自然应用。可链式连接最多 10 个风格参考，实现多风格融合。

## 输入

| 输入            | 类型   | 描述                                |
| --------------- | ------ | ----------------------------------- |
| image           | image  | 影响生成效果的风格参考图像           |
| strength        | float  | -2.0 至 2.0（默认 1.0）             |
| style_reference | chain  | 可选的传入风格参考链                |

## 输出

| 输出            | 类型  | 描述                               |
| --------------- | ----- | ---------------------------------- |
| style_reference | chain | 用于 Krea 2 Image 的链式风格参考    |
