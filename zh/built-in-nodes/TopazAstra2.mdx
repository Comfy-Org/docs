Astra 2 是基于扩散模型的创意视频放大方案的全新版本，在 Astra 1 的基础上扩展了对细节生成与风格化的控制能力。它能够为 AI 生成的视频添加新的动态细节与纹理，提供可调的创意强度与锐化控制，并支持基于提示词的引导，使创意方向更加精确。

Astra 2 在风格化与 GenAI 素材、广角镜头以及大场景上的表现最佳，能够创造性地补充新的细节，从而提升画面清晰度、丰富度与视觉复杂度，特别适合富有表现力的风格化工作流。

## 主要能力

- 为 GenAI 素材创造性地添加新的细节与纹理。
- 提供可调的创意强度、锐化等级以及基于提示词的引导。
- 在风格化内容、广角镜头与细节稀疏的场景中表现出色。

## 接口

- **URL**：`https://api.topazlabs.com/video/`
- **模型标识**：`"ast-2"`

## 模型参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `creativity` | string | 是 | 取值 `"low"`、`"middle"`、`"high"` 之一，控制创意细节生成强度。 |
| `prompt` | string | 否 | 用于引导创意方向的提示词。 |
| `input_frame_count` | integer | 否 | 不带 prompt 时最大 9000；带 prompt 时最大 450。 |
| `input_frame_rate` | integer | 否 | 输入视频的帧率。 |
| `input_height` | integer | 否 | 输入视频高度。 |
| `input_width` | integer | 否 | 输入视频宽度。 |
| `output_height` | integer | 否 | 目标输出高度。 |
| `output_width` | integer | 否 | 目标输出宽度。 |
| `realism` | decimal | 否 | 取值范围 `0.0` 到 `1.0`。 |
| `sharp` | decimal | 否 | 预增强锐化，取值 `0.0` 到 `1.0`，默认 `0.5`。`0.0` 为高斯模糊，`0.5` 为原样透传，`1.0` 为 USM 锐化。 |
