TextGenerateLTX2Prompt 节点是文本生成节点的一个专用版本。它接收用户的文本提示，自动使用特定的系统指令进行格式化，然后发送给语言模型进行增强或补全。该节点支持两种运行模式：纯文本模式或附带图像参考模式，每种模式使用不同的系统提示。

## 输入

| 参数 | 描述 | 数据类型 | 必填 | 范围 |
| --- | --- | --- | --- | --- |
| `clip` | 用于文本编码的 CLIP 模型。 | CLIP | 是 |  |
| `提示词` | 用户输入的原始文本，将被增强或补全。 | STRING | 是 |  |
| `最大长度` | 语言模型允许生成的最大 token 数量。 | INT | 是 |  |
| `采样模式` | 文本生成过程中用于选择下一个 token 的采样策略。 | COMBO | 是 | `"greedy"`<br />`"top_k"`<br />`"top_p"`<br />`"temperature"` |
| `图像` | 可选的输入图像。当提供时，节点会使用包含图像上下文占位符的不同系统提示。 | IMAGE | 否 |  |
| `思考模式` | 启用后，模型将在最终答案之前输出其推理过程。 | BOOLEAN | 否 |  |
| `use_default_template` | 启用后，节点将使用默认的聊天模板进行格式化。 | BOOLEAN | 否 |  |
| `视频` | 可选的视频输入，可作为生成的额外上下文。 | VIDEO | 否 |  |
| `音频` | 可选的音频输入，可作为生成的额外上下文。 | AUDIO | 否 |  |

**注意：** 节点的行为会根据 `image` 输入是否存在而变化。如果提供了图像，生成的提示将格式化为图像到视频任务。如果未提供图像，则格式化为文本到视频任务。

## 输出

| 输出名称 | 描述 | 数据类型 |
| --- | --- | --- |
| `生成文本` | 由语言模型生成的增强或补全后的文本字符串。 | STRING |
