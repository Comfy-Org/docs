WanMoveVisualizeTracks 节点可将运动跟踪数据叠加到图像序列或视频帧上。它会绘制跟踪点的可视化表示，包括其运动路径和当前位置，使运动数据可见且更易于分析。

## 输入

| 参数 | 描述 | 数据类型 | 是否必填 | 范围 |
| --- | --- | --- | --- | --- |
| `图像` | 用于可视化跟踪轨迹的输入图像或视频帧序列。 | IMAGE | 是 | - |
| `轨道` | 包含点路径和可见性信息的运动跟踪数据。如果未提供，输入图像将原样传递。 | TRACKS | 否 | - |
| `线条分辨率` | 绘制每个跟踪点尾迹路径时使用的先前帧数（默认值：24）。 | INT | 是 | 1 - 1024 |
| `圆环尺寸` | 在每个跟踪点当前位置绘制的圆圈大小（默认值：12）。 | INT | 是 | 1 - 128 |
| `不透明度` | 绘制的跟踪叠加层的透明度（默认值：0.75）。 | FLOAT | 是 | 0.0 - 1.0 |
| `线条宽度` | 用于绘制跟踪路径的线条宽度（默认值：16）。 | INT | 是 | 1 - 128 |

**注意：** 如果输入图像数量与提供的 `tracks` 数据中的帧数不匹配，图像序列将重复以匹配跟踪长度。

## 输出

| 输出名称 | 描述 | 数据类型 |
| --- | --- | --- |
| `IMAGE` | 已叠加运动跟踪数据可视化的图像序列。如果未提供 `轨道`，则返回原始输入图像。 | IMAGE |
