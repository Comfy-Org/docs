# CFG 覆盖

CFG 覆盖节点允许您为采样过程的特定范围（以总步数的百分比定义）设置固定的 CFG（无分类器引导）缩放值。当连接多个 CFG 覆盖节点时，链中离采样器最近的节点将优先处理重叠范围。

## ## 输入

| 参数 | 描述 | 数据类型 | 必填 | 范围 |
|-----------|-------------|-----------|----------|-------|
| `模型` | 要应用 CFG 覆盖的模型 | MODEL | 是 | |
| `cfg` | 在覆盖范围内使用的固定 CFG 缩放值（默认值：1.0） | FLOAT | 是 | 0.0 至 100.0 |
| `起始百分比` | 覆盖范围的起始点，以采样过程的百分比表示（默认值：0.0） | FLOAT | 是 | 0.0 至 1.0 |
| `结束百分比` | 覆盖范围的结束点，以采样过程的百分比表示（默认值：1.0） | FLOAT | 是 | 0.0 至 1.0 |

## ## 输出

| 输出名称 | 描述 | 数据类型 |
|-------------|-------------|-----------|
| `MODEL` | 已应用 CFG 覆盖包装器的模型 | MODEL |
