## ## 概述

此节点使用 Rodin Gen-2.5 API 从一到五张参考图像生成 3D 模型。您可以在快速、常规或极致高质量模式之间进行选择，以平衡生成速度与成本。

## ## 输入

| 参数 | 描述 | 数据类型 | 是否必需 | 范围 |
| --- | --- | --- | --- | --- |
| `图像` | 一到五张输入图像。当提供多张图像时，第一张图像用于材质。 | IMAGE | 是 | 1 到 5 张图像 |
| `模式` | 生成质量模式。更高质量的模式可产生更好的结果，但成本更高。 | COMBO | 是 | `"Fast"`<br />`"Regular"`<br />`"Extreme-High"` |
| `材质` | 生成 3D 模型的材质类型。 | COMBO | 是 | `"PBR"`<br />`"Matte"` |
| `几何文件格式` | 3D 模型几何体的输出文件格式。 | COMBO | 是 | `"glb"`<br />`"obj"`<br />`"stl"`<br />`"usdz"` |
| `贴图模式` | 纹理生成模式。“Original”保留输入纹理，“Clean”移除纹理，“Style”应用风格化纹理。 | COMBO | 是 | `"Original"`<br />`"Clean"`<br />`"Style"` |
| `种子` | 用于可重现结果的随机种子。使用相同的种子可获得相同的输出。 | INT | 是 | 0 到 2147483647 |
| `T/A 姿势` | 是否对生成的模型应用 T 形姿势。 | BOOLEAN | 是 | True / False |
| `高清贴图` | 是否生成高清晰度纹理贴图。 | BOOLEAN | 是 | True / False |
| `贴图去光照` | 是否在纹理生成前从输入图像中移除光照。 | BOOLEAN | 是 | True / False |
| `保留原始透明度` | 是否使用输入图像中的原始 Alpha 通道。 | BOOLEAN | 是 | True / False |
| `HighPack 插件` | 是否在标准模型之外额外生成高多边形版本。 | BOOLEAN | 是 | True / False |
| `包围盒宽度` | 生成模型边界框的宽度，单位为厘米。 | INT | 是 | 1 到 1000 |
| `包围盒高度` | 生成模型边界框的高度，单位为厘米。 | INT | 是 | 1 到 1000 |
| `包围盒长度` | 生成模型边界框的长度，单位为厘米。 | INT | 是 | 1 到 1000 |
| `模型高度（厘米）` | 生成模型的高度，单位为厘米。 | INT | 是 | 1 到 300 |

**关于图像数量的说明：** 此节点接受 1 到 5 张图像。如果您提供一批图像（例如，一个包含 4 张图像的批次），批次中的每张图像都将被视为单独的输入图像。提供超过 5 张图像将导致错误。
