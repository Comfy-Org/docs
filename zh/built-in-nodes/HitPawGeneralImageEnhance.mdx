此节点通过将低分辨率图像放大至超分辨率，并去除伪影和噪点来增强图像质量。它使用外部 API 处理图像，并可自动调整输入尺寸以保持在处理限制范围内。允许的最大输出尺寸为 4 百万像素。

| `图像` | IMAGE | 是 | - | 要增强的输入图像。 |
| `放大倍数` | INT | 是 | `1`<br />`2`<br />`4` | 图像尺寸的放大倍数。倍数为 1 表示不放大，2 表示尺寸加倍，4 表示尺寸变为四倍。 |
