Vidu Q3 文本转视频生成节点可根据文本描述创建视频。它使用 Vidu Q3 Pro 或 Q3 Turbo 模型，基于您的提示词生成视频内容，并允许您控制视频长度、分辨率、宽高比以及是否包含音频。

## 输入

| 参数 | 描述 | 数据类型 | 是否必填 | 取值范围 |
| --- | --- | --- | --- | --- |
| `model` | 用于视频生成的模型。选择模型后会显示额外的配置参数，包括宽高比、分辨率、时长和音频。 | COMBO | 是 | `"viduq3-pro"`<br />`"viduq3-turbo"` |
| `model.aspect_ratio` | 输出视频的宽高比。选择 `model` 后显示此参数。 | COMBO | 是* | `"16:9"`<br />`"9:16"`<br />`"3:4"`<br />`"4:3"`<br />`"1:1"` |
| `model.resolution` | 输出视频的分辨率。选择 `model` 后显示此参数。 | COMBO | 是* | `"720p"`<br />`"1080p"` |
| `model.duration` | 输出视频的时长（秒），默认值为 5。选择 `model` 后显示此参数。 | INT | 是* | 1 到 16 |
| `model.audio` | 启用后，输出带声音的视频（包括对话和音效），默认值为 False。选择 `model` 后显示此参数。 | BOOLEAN | 是* | True/False |
| `prompt` | 用于视频生成的文本描述，最大长度为 2000 个字符。 | STRING | 是 | 不适用 |
| `seed` | 用于控制生成随机性的种子值，默认值为 1。 | INT | 否 | 0 到 2147483647 |

*注意：选择 `model` 后，参数 `aspect_ratio`、`resolution`、`duration` 和 `audio` 为必填项，因为它们属于模型配置的一部分。

## 输出

| 输出名称 | 描述 | 数据类型 |
| --- | --- | --- |
| `video` | 生成的视频文件。 | VIDEO |
