此节点根据提供的参考素材生成包含人物或对象的视频。它使用 Wan 2.7 模型根据文本提示创建视频，支持单人表演和多人互动。您必须至少提供一个参考视频或图像才能进行生成。

## 输入

| 参数 | 描述 | 数据类型 | 是否必填 | 范围 |
| --- | --- | --- | --- | --- |
| `model` | 用于视频生成的特定模型。 | COMBO | 是 | `"wan2.7-r2v"` |
| `model.prompt` | 描述视频的提示词。使用诸如 'character1' 和 'character2' 之类的标识符来指代参考角色。 | STRING | 是 | - |
| `model.negative_prompt` | 描述生成视频中应避免内容的负面提示词（默认：空）。 | STRING | 否 | - |
| `model.resolution` | 输出视频的分辨率。 | COMBO | 是 | `"720P"`<br />`"1080P"` |
| `model.ratio` | 输出视频的宽高比。 | COMBO | 是 | `"16:9"`<br />`"9:16"`<br />`"1:1"`<br />`"4:3"`<br />`"3:4"` |
| `model.duration` | 生成视频的时长（秒）（默认：5）。 | INT | 是 | 2 到 10 |
| `model.reference_videos` | 参考视频列表。最多可添加 3 个视频。 | VIDEO | 否 | - |
| `model.reference_images` | 参考图像列表。最多可添加 5 张图像。 | IMAGE | 否 | - |
| `seed` | 用于生成的随机种子，有助于控制输出的随机性（默认：0）。 | INT | 否 | 0 到 2147483647 |
| `watermark` | 是否在结果中添加 AI 生成的水印（默认：False）。这是一个高级设置。 | BOOLEAN | 否 | - |

**重要限制：**
*   您必须在 `model.reference_videos` 或 `model.reference_images` 输入中至少提供一个参考视频或参考图像。
*   参考视频和图像的总数不能超过 5 个。

## 输出

| 输出名称 | 描述 | 数据类型 |
| --- | --- | --- |
| `output` | 生成的视频文件。 | VIDEO |
