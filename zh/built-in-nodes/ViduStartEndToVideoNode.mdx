# Vidu 起始帧到结束帧视频生成节点

Vidu 起始帧到结束帧视频生成节点通过在起始帧和结束帧之间生成帧来创建视频。它使用文本提示来引导视频生成过程，并支持具有不同分辨率和运动设置的各种视频模型。该节点在处理前会验证起始帧和结束帧具有兼容的宽高比。
