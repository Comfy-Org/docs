节点名称：Reroute 节点
节点用途：主要用于整理 ComfyUI 工作流中过长的连接线逻辑。

## 如何使用 Reroute 节点

| 菜单选项 | 说明 |
| --- | --- |
| Show Type | 显示节点的类型属性 |
| Hide Type By Default | 默认隐藏节点的类型属性 |
| Set Vertical | 将节点的连线方向设置为垂直 |
| Set Horizontal | 将节点的连线方向设置为水平 |

当您的连线逻辑过长且复杂，希望整理界面时，可以在两个连接点之间插入一个 ```Reroute``` 节点。该节点的输入和输出不受类型限制，默认样式为水平。您可以通过右键菜单将连线方向更改为垂直。
