| 参数 | 数据类型 | 必填 | 范围 | 描述 |

| `宽度` | INT | 是 | 256 至 MAX_RESOLUTION | 输出图像的宽度（像素），默认值：1024，步长：8 |
| `高度` | INT | 是 | 256 至 MAX_RESOLUTION | 输出图像的高度（像素），默认值：1024，步长：8 |
| `压缩` | INT | 是 | 4 至 128 | 决定阶段 C 潜空间维度的压缩因子，默认值：42，步长：1 |
| `批量大小` | INT | 否 | 1 至 4096 | 每批生成的潜空间样本数量，默认值：1 |
