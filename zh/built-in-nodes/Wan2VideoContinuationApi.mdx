## 概述

Wan 2.7 视频续接节点用于生成一段新的视频片段，该片段能够从输入视频片段的结尾无缝衔接。它利用 Wan 2.7 模型，根据文本提示合成续接内容，并可选择性地引导结尾朝向特定的目标帧。

## 输入

| 参数 | 描述 | 数据类型 | 是否必填 | 取值范围 |
| --- | --- | --- | --- | --- |
| `model` | 要使用的视频生成模型。 | COMBO | 是 | `"wan2.7-i2v"` |
| `model.prompt` | 描述元素和视觉特征的提示词。支持英文和中文。（默认值：空字符串） | STRING | 是 | - |
| `model.negative_prompt` | 描述需要避免内容的负面提示词。（默认值：空字符串） | STRING | 是 | - |
| `model.resolution` | 输出视频的分辨率。 | COMBO | 是 | `"720P"`<br />`"1080P"` |
| `model.duration` | 输出视频的总时长（秒）。模型会生成续接内容，以填充输入片段之后剩余的时间。（默认值：5） | INT | 是 | 2 到 15 |
| `first_clip` | 需要续接的输入视频。时长：2秒-10秒。输出视频的宽高比由此视频决定。 | VIDEO | 是 | - |
| `last_frame` | 最后一帧图像。续接内容将向此帧过渡。 | IMAGE | 否 | - |
| `seed` | 用于生成的随机种子。（默认值：0） | INT | 是 | 0 到 2147483647 |
| `prompt_extend` | 是否使用 AI 辅助增强提示词。（默认值：True） | BOOLEAN | 是 | - |
| `watermark` | 是否在结果中添加 AI 生成水印。（默认值：False） | BOOLEAN | 是 | - |

**注意：** `first_clip` 输入视频的时长必须在 2 到 10 秒之间。

## 输出
