此节点将字典（键值对集合）转换为文本字符串，通常采用 JSON 格式。您可以控制缩进级别，使输出更易读或更紧凑。

## 输入

| 参数 | 描述 | 数据类型 | 是否必填 | 取值范围 |
|-----------|-------------|-----------|----------|-------|
| `字典` | 要转换为字符串的字典 | DICT | 是 | - |
| `缩进` | 每级缩进的空格数。0 生成紧凑的单行字符串（默认值：2） | INT | 否 | 0 到 8 |
