# 概述

VAESave 节点用于将 VAE 模型及其元数据（包括提示词和额外的 PNG 信息）保存到指定的输出目录。它封装了将模型状态和相关关联信息序列化到文件中的功能，便于保存和共享训练好的模型。

## 输入

| 参数 | 数据类型 | 描述 |
|-----------|-------------|-------------|
| `vae`     | VAE       | 要保存的 VAE 模型。此参数至关重要，因为它代表需要序列化和存储其状态的模型。 |
| `文件名前缀` | STRING   | 保存模型及其元数据的文件名前缀。这有助于实现有序存储和便捷的模型检索。 |
