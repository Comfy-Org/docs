PixVerse 模板节点允许您从可用的模板中选择用于 PixVerse 视频生成的模板。它会将您选择的模板名称转换为 PixVerse API 创建视频所需的对应模板 ID。

## 输入

| 参数 | 数据类型 | 是否必填 | 范围 | 描述 |
|-----------|-----------|----------|-------|-------------|
| `模板` | STRING | 是 | 多个可用选项 | 用于 PixVerse 视频生成的模板。可用选项对应 PixVerse 系统中的预定义模板。 |

## 输出

| 输出名称 | 数据类型 | 描述 |
|-------------|-----------|-------------|
| `pixverse_template` | STRING | 与所选模板名称对应的模板 ID，可供其他 PixVerse 节点用于视频生成。 |
