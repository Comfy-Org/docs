## 输入
| 参数 | 描述 | 数据类型 | 是否必需 | 范围 |
|------|------|----------|----------|------|
| `latents` | 按顺序排列的采样时间块。 | LATENT | 是 | 潜在变量列表 |
| `temporal_overlap` | Split SeedVR2 Latent 节点的 `temporal_overlap` 输出。0 表示简单拼接。 | INT | 是 | 0 至 16384（默认值：0） |

**注意：** `temporal_overlap` 参数必须从 Split SeedVR2 Latent 节点的 `temporal_overlap` 输出连接。所有输入的潜在变量必须具有相同的批次大小、通道数、高度和宽度。只有最后一个时间块的时序维度可能比其他块短。
