ChromaRadianceOptions 节点允许您为 Chroma Radiance 模型配置高级设置。它会封装现有模型，并根据 sigma 值在去噪过程中应用特定选项，从而实现对 NeRF 瓦片大小及其他辐射相关参数的精细控制。

## 输入

| 参数 | 描述 | 数据类型 | 是否必填 | 范围 |
| --- | --- | --- | --- | --- |
| `模型` | 要应用 Chroma Radiance 选项的模型 | MODEL | 是 | - |
| `保留包装器` | 启用后，若存在现有模型函数封装器，将委托给该封装器。通常应保持启用状态。（默认值：True） | BOOLEAN | 否 | - |
| `起始sigma` | 这些选项生效的起始 sigma 值。（默认值：1.0） | FLOAT | 否 | 0.0 至 1.0 |
| `结束sigma` | 这些选项生效的结束 sigma 值。（默认值：0.0） | FLOAT | 否 | 0.0 至 1.0 |
| `NeRF瓦片大小` | 允许覆盖默认的 NeRF 瓦片大小。-1 表示使用默认值（32）。0 表示使用非瓦片模式（可能需要大量显存）。（默认值：-1） | INT | 否 | -1 及以上 |

**注意：** Chroma Radiance 选项仅在当前 sigma 值介于 `end_sigma` 和 `start_sigma`（含边界值）之间时生效。`nerf_tile_size` 参数仅在设置为 0 或更高值时才会被应用。

## 输出

| 输出名称 | 描述 | 数据类型 |
| --- | --- | --- |
| `模型` | 已应用 Chroma Radiance 选项的修改后模型 | MODEL |
