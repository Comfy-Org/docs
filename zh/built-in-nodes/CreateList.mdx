# 创建列表节点

创建列表节点将多个输入合并为一个有序列表。它接收任意数量的相同数据类型输入，并按照连接顺序将它们连接起来。此节点适用于准备批处理数据（如图像或文本），以便工作流中的其他节点进行处理。

| `input_*` | 可变 | 是 | 任意 | 可变数量的输入插槽。您可以通过点击加号（+）图标添加更多输入。所有输入必须为相同数据类型（例如，全部为 IMAGE 或全部为 STRING）。 |
