保存一个或多个相机概念，用于 Luma 文本转视频和 Luma 图像转视频节点。此节点允许您选择最多四个相机概念，并可选择将其与现有概念链组合。

| 参数 | 数据类型 | 必填 | 范围 | 描述 |

| `概念1` | STRING | 是 | 提供多个选项<br />包含"None"选项 | 从可用 Luma 概念中选择第一个相机概念 |
| `概念2` | STRING | 是 | 提供多个选项<br />包含"None"选项 | 从可用 Luma 概念中选择第二个相机概念 |
| `概念3` | STRING | 是 | 提供多个选项<br />包含"None"选项 | 从可用 Luma 概念中选择第三个相机概念 |
| `概念4` | STRING | 是 | 提供多个选项<br />包含"None"选项 | 从可用 Luma 概念中选择第四个相机概念 |
