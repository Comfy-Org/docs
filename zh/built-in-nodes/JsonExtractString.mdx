## 概述

JsonExtractString 节点用于读取包含 JSON 数据的文本字符串，并提取与特定键关联的值。它会将提取的值转换为字符串。如果 JSON 无效、未找到该键或值为 null，节点将返回空字符串。

## 输入

| 参数 | 数据类型 | 是否必填 | 范围 | 描述 |
|-----------|-----------|----------|-------|-------------|
| `json_string` | STRING | 是 | 不适用 | 包含待解析 JSON 数据的文本。 |
| `key` | STRING | 是 | 不适用 | 要从 JSON 对象中提取字符串值的特定键。 |

**注意：** 该节点仅从 JSON 对象（字典）中提取值。如果解析后的 JSON 不是对象，或者对象中不存在指定的键，则输出将为空字符串。

## 输出

| 输出名称 | 数据类型 | 描述 |
|-------------|-----------|-------------|
| `output` | STRING | 从 JSON 中为指定键提取的字符串值，如果提取失败则返回空字符串。 |
