此节点使用 Grok API 基于文本提示编辑现有视频。它会上传您的视频，向 AI 模型发送请求以根据您的描述进行修改，并返回新生成的视频。

## 输入

| 参数 | 描述 | 数据类型 | 是否必填 | 范围 |
| --- | --- | --- | --- | --- |
| `model` | 用于视频编辑的 AI 模型（默认值：`"grok-imagine-video"`）。 | COMBO | 是 | `"grok-imagine-video"`<br />`"grok-imagine-video-beta"` |
| `prompt` | 对所需视频的文本描述。 | STRING | 是 | 无 |
| `video` | 待编辑的输入视频。最长支持 8.7 秒，文件大小不超过 50MB。 | VIDEO | 是 | 无 |
| `seed` | 用于确定节点是否应重新运行的种子值。无论种子值如何，实际结果均具有非确定性（默认值：0）。 | INT | 否 | 0 到 2147483647 |

**约束条件：**

* 输入 `video` 的时长必须在 1 到 8.7 秒之间。
* 输入 `video` 的文件大小不得超过 50MB。
* `prompt` 不能为空。

## 输出

| 输出名称 | 描述 | 数据类型 |
| --- | --- | --- |
| `video` | 由 AI 模型生成的编辑后视频。 | VIDEO |
