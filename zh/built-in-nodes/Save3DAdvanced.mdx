此节点将 3D 模型保存到 ComfyUI 输出目录中的文件，并可对输出尺寸和相机/视口设置进行高级控制。它还会将 3D 模型、模型信息、相机信息和尺寸传递到下游节点。

## ## 输入

| 参数 | 描述 | 数据类型 | 是否必需 | 范围 |
|-----------|-------------|-----------|----------|-------|
| `model_3d` | 来自上游 3D 节点的 3D 模型文件。 | FILE3D | 是 | GLB, GLTF, FBX, OBJ, STL, USDZ, 任意 |
| `filename_prefix` | 保存文件名的前缀（默认值："3d/ComfyUI"）。 | STRING | 是 | 自由文本 |
| `viewport_state` | 来自加载 3D 节点的视口状态，包含相机和模型信息。 | LOAD3D | 是 | - |
| `model_3d_info` | 可选的 3D 模型信息，用于覆盖视口状态。 | LOAD3DMODELINFO | 否 | - |
| `camera_info` | 可选的相机信息，用于覆盖视口状态。 | LOAD3DCAMERA | 否 | - |
| `width` | 输出预览的宽度（像素，默认值：1024）。 | INT | 是 | 1 到 4096 |
| `height` | 输出预览的高度（像素，默认值：1024）。 | INT | 是 | 1 到 4096 |

## ## 输出

| 输出名 | 描述 | 数据类型 |
|-------------|-------------|-----------|
| `model_3d` | 从输入传递过来的 3D 模型文件。 | FILE3D |
| `model_3d_info` | 模型信息，来自视口状态或可选输入。 | LOAD3DMODELINFO |
| `camera_info` | 相机信息，来自视口状态或可选输入。 | LOAD3DCAMERA |
| `width` | 从输入传递过来的宽度值。 | INT |
| `height` | 从输入传递过来的高度值。 | INT |
