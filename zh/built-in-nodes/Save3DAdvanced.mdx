此节点将 3D 模型保存到您的 ComfyUI 输出目录，并提供高级预览功能。您可以指定输出文件名、预览尺寸，并可选择传递相机和模型信息，以获得增强的 3D 查看器体验。

## 输入
| 参数 | 描述 | 数据类型 | 是否必需 | 范围 |
|-----------|-------------|-----------|----------|-------|
| `model_3d` | 来自上游 3D 节点的 3D 模型文件。 | FILE3D | 是 | GLB、GLTF、FBX、OBJ、STL、USDZ 或任何 3D 格式 |
| `filename_prefix` | 保存文件名的前缀。计数器和文件扩展名将自动附加。（默认值："3d/ComfyUI"） | STRING | 是 | 任何有效的文件名字符串 |
| `viewport_state` | 当前视口状态，通常来自 3D 查看器节点。 | LOAD3D | 是 | - |
| `model_3d_info` | 可选的预览用 3D 模型信息。 | LOAD3DMODELINFO | 否 | - |
| `camera_info` | 可选的预览用相机信息。 | LOAD3DCAMERA | 否 | - |
| `width` | 预览的宽度（像素）。（默认值：1024） | INT | 是 | 1 到 4096 |
| `height` | 预览的高度（像素）。（默认值：1024） | INT | 是 | 1 到 4096 |

## 输出
| 输出名 | 描述 | 数据类型 |
|-------------|-------------|-----------|
| `model_3d` | 已保存的 3D 模型文件。 | FILE3D |
| `model_3d_info` | 传递至下游节点的 3D 模型信息。 | LOAD3DMODELINFO |
| `camera_info` | 传递至下游节点的相机信息。 | LOAD3DCAMERA |
| `width` | 传递至下游节点的宽度值。 | INT |
| `height` | 传递至下游节点的高度值。 | INT |
