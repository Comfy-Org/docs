此节点通过处理最多四张展示物体不同视角的图像，使用 Tripo API 同步生成 3D 模型。它需要一张正面图像以及至少一个额外视角（左、后或右），以创建带有纹理和材质选项的完整 3D 模型。

## 输入

| 参数 | 描述 | 数据类型 | 是否必填 | 范围 |
| --- | --- | --- | --- | --- |
| `图像` | 物体的正面视角图像（必填） | IMAGE | 是 | - |
| `左侧图像` | 物体的左侧视角图像 | IMAGE | 否 | - |
| `背面图像` | 物体的背面视角图像 | IMAGE | 否 | - |
| `右侧图像` | 物体的右侧视角图像 | IMAGE | 否 | - |
| `模型版本` | 用于生成的模型版本 | COMBO | 否 | 提供多个选项 |
| `朝向` | 3D 模型的方向设置（默认值："default"） | COMBO | 否 | 提供多个选项 |
| `纹理` | 是否为模型生成纹理（默认值：True） | BOOLEAN | 否 | - |
| `PBR` | 是否生成 PBR（基于物理的渲染）材质（默认值：True） | BOOLEAN | 否 | - |
| `模型种子` | 模型生成的随机种子（默认值：42） | INT | 否 | - |
| `纹理种子` | 纹理生成的随机种子（默认值：42） | INT | 否 | - |
| `纹理质量` | 纹理生成的质量级别（默认值："standard"） | COMBO | 否 | `"standard"`<br />`"detailed"` |
| `纹理对齐` | 纹理与模型对齐的方法（默认值："original_image"） | COMBO | 否 | `"original_image"`<br />`"geometry"` |
| `面数限制` | 生成模型的最大面数。设置为 -1 表示无限制（默认值：-1） | INT | 否 | -1 到 500000 |
| `四边形` | 此参数已弃用，不执行任何操作（默认值：False） | BOOLEAN | 否 | - |
| `几何质量` | 几何体生成的质量级别（默认值："standard"） | COMBO | 否 | `"standard"`<br />`"detailed"` |

**注意：** 正面图像（`image`）始终为必填项。必须至少提供一张额外视角图像（`image_left`、`image_back` 或 `image_right`）以进行多视角处理。
