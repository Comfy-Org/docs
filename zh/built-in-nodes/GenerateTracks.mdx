`GenerateTracks` 节点用于为视频生成创建多条平行运动路径。它定义一条从起点到终点的主路径，然后生成一组与该路径平行且等距分布的轨道。您可以控制路径的形状（直线或贝塞尔曲线）、沿路径移动的速度以及轨道可见的帧范围。

## 输入

| 参数 | 描述 | 数据类型 | 是否必填 | 范围 |
| --- | --- | --- | --- | --- |
| `宽度` | 视频帧的宽度（像素）。默认值为 832。 | INT | 是 | 16 - 4096 |
| `高度` | 视频帧的高度（像素）。默认值为 480。 | INT | 是 | 16 - 4096 |
| `起始点X` | 起始位置的归一化 X 坐标（0-1）。默认值为 0.0。 | FLOAT | 是 | 0.0 - 1.0 |
| `起始点Y` | 起始位置的归一化 Y 坐标（0-1）。默认值为 0.0。 | FLOAT | 是 | 0.0 - 1.0 |
| `结束点X` | 结束位置的归一化 X 坐标（0-1）。默认值为 1.0。 | FLOAT | 是 | 0.0 - 1.0 |
| `结束点Y` | 结束位置的归一化 Y 坐标（0-1）。默认值为 1.0。 | FLOAT | 是 | 0.0 - 1.0 |
| `帧数` | 需要生成轨道位置的总帧数。默认值为 81。 | INT | 是 | 1 - 1024 |
| `轨迹数` | 要生成的平行轨道数量。默认值为 5。 | INT | 是 | 1 - 100 |
| `轨道分布` | 轨道之间的归一化距离。轨道沿运动方向的垂直方向分布。默认值为 0.025。 | FLOAT | 是 | 0.0 - 1.0 |
| `贝塞尔曲线` | 启用贝塞尔曲线路径，使用中点作为控制点。默认值为 False。 | BOOLEAN | 是 | True / False |
| `中间点Y` | 贝塞尔曲线的归一化 X 控制点。仅在启用 `贝塞尔曲线` 时使用。默认值为 0.5。 | FLOAT | 是 | 0.0 - 1.0 |
| `中间点Y` | 贝塞尔曲线的归一化 Y 控制点。仅在启用 `贝塞尔曲线` 时使用。默认值为 0.5。 | FLOAT | 是 | 0.0 - 1.0 |
| `插值` | 控制沿路径移动的时间/速度。默认值为 "linear"。 | COMBO | 是 | `"linear"`<br />`"ease_in"`<br />`"ease_out"`<br />`"ease_in_out"`<br />`"constant"` |
| `轨道遮罩` | 可选的遮罩，用于指示可见帧。 | MASK | 否 | - |

**注意：** `mid_x` 和 `mid_y` 参数仅在 `bezier` 参数设置为 `True` 时使用。当 `bezier` 为 `False` 时，路径为从起点到终点的直线。

## 输出

| 输出名称 | 描述 | 数据类型 |
| --- | --- | --- |
| `轨迹长度` | 一个轨道对象，包含所有帧中所有轨道的生成路径坐标和可见性信息。 | TRACKS |
| `track_length` | 生成轨道的帧数，与输入的 `帧数` 一致。 | INT |
