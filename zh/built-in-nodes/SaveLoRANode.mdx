SaveLoRA 节点将 LoRA（低秩适配）模型保存到您的输出目录。它接收一个 LoRA 模型作为输入，并创建一个带有自动生成文件名的 safetensors 文件。您可以自定义文件名前缀，并可选择在文件名中包含训练步数，以便更好地组织文件。

## 输入

| 参数 | 数据类型 | 是否必填 | 范围 | 描述 |
|-----------|-----------|----------|-------|-------------|
| `lora` | LORA_MODEL | 是 | - | 要保存的 LoRA 模型。请勿使用带有 LoRA 层的模型。 |
| `prefix` | STRING | 是 | - | 用于保存 LoRA 文件的前缀（默认值："loras/ComfyUI_trained_lora"）。 |
| `steps` | INT | 否 | - | 可选：LoRA 已训练的步数，用于命名保存的文件。 |

## 输出

| 输出名称 | 数据类型 | 描述 |
|-------------|-----------|-------------|
| *无* | - | 此节点不返回任何输出，但会将 LoRA 模型保存到输出目录。 |
