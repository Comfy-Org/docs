## 输入

| 参数 | 描述 | 数据类型 | 是否必填 | 范围 |
| --- | --- | --- | --- | --- |

| `模型补丁` | 包含样式参考信息的模型补丁 | MODEL_PATCH | 是 | - |
| `CLIP视觉输出` | 从 CLIP 视觉处理中提取的编码视觉特征 | CLIP_VISION_OUTPUT | 是 | - |
