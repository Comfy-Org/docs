根据文本描述，使用 Tripo 的 API 同步生成 3D 模型。此节点接收文本描述，并创建具有可选纹理和材质属性的 3D 模型。

## 输入

| 参数 | 描述 | 数据类型 | 是否必需 | 取值范围 |
| --- | --- | --- | --- | --- |
| `提示词` | 用于生成 3D 模型的文本描述（支持多行输入） | STRING | 是 | - |
| `负面提示词` | 描述生成模型中应避免的内容的文本（支持多行输入） | STRING | 否 | - |
| `模型版本` | 用于生成的 Tripo 模型版本（默认值：v2.5-20250123） | COMBO | 否 | 提供多个选项 |
| `风格` | 生成模型的样式设置（默认值："None"） | COMBO | 否 | 提供多个选项 |
| `纹理` | 是否为模型生成纹理（默认值：True） | BOOLEAN | 否 | - |
| `PBR` | 是否生成 PBR（基于物理的渲染）材质（默认值：True） | BOOLEAN | 否 | - |
| `图像种子` | 图像生成的随机种子（默认值：42） | INT | 否 | - |
| `模型种子` | 模型生成的随机种子（默认值：42） | INT | 否 | - |
| `纹理种子` | 纹理生成的随机种子（默认值：42） | INT | 否 | - |
| `纹理质量` | 纹理生成的质量等级（默认值："standard"） | COMBO | 否 | "standard"<br />"detailed" |
| `面数限制` | 生成模型中的最大面数，-1 表示无限制（默认值：-1） | INT | 否 | -1 到 2000000 |
| `四边形` | 是否生成基于四边形的几何体而非三角形（默认值：False） | BOOLEAN | 否 | - |
| `几何质量` | 几何体生成的质量等级（默认值："standard"） | COMBO | 否 | "standard"<br />"detailed" |

**注意：** `prompt` 参数是必需的，且不能为空。如果未提供提示词，此节点将引发错误。

## 输出

| 输出名称 | 描述 | 数据类型 |
| --- | --- | --- |
| `模型任务ID` | 生成的 3D 模型文件（仅用于向后兼容） | STRING |
| `GLB` | 模型生成过程的唯一任务标识符 | MODEL_TASK_ID |
| `GLB` | 以 GLB 格式生成的 3D 模型 | FILE3DGLB |
