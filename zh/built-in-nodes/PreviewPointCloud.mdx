预览点云节点允许您在 ComfyUI 界面中查看 3D 点云文件。该节点将点云保存到临时文件，并在 3D 预览窗口中显示，同时传递模型数据和视口设置以供进一步处理。

## 输入

| 参数 | 描述 | 数据类型 | 必需 | 范围 |
|-----------|-------------|-----------|----------|-------|
| `model_3d` | 点云文件 (.ply) | FILE3D | 是 | - |
| `model_3d_info` | 3D 模型的信息 | LOAD3DMODELINFO | 否 | - |
| `viewport_state` | 当前视口状态 | LOAD3D | 是 | - |
| `camera_info` | 3D 视图的相机信息 | LOAD3DCAMERA | 否 | - |
| `width` | 预览窗口的宽度（默认：1024） | INT | 是 | 1 到 4096 |
| `height` | 预览窗口的高度（默认：1024） | INT | 是 | 1 到 4096 |

## 输出

| 输出名称 | 描述 | 数据类型 |
|-------------|-------------|-----------|
| `model_3d` | 点云模型数据 | FILE3D |
| `model_3d_info` | 3D 模型的信息 | LOAD3DMODELINFO |
| `camera_info` | 3D 视图的相机信息 | LOAD3DCAMERA |
| `width` | 预览窗口的宽度 | INT |
| `height` | 预览窗口的高度 | INT |
