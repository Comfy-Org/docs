此节点在单张图像或一批图像上绘制文本。支持自定义文本、字号、颜色、位置、对齐方式以及可选描边，将文本合成到原始图像之上。

## 输入
| 参数 | 描述 | 数据类型 | 是否必填 | 范围 |
|------|------|----------|----------|------|
| `图像` | 要绘制文本的输入图像或图像批次。 | IMAGE | 是 | |
| `文字` | 要叠加的文本字符串。支持换行符（`\n`）和制表符（`\t`）。默认值："" | STRING | 是 | |
| `字体大小` | 字号，以图像高度的百分比表示。默认值：5.0 | FLOAT | 是 | 0.5 至 50.0 |
| `颜色` | 文本颜色。默认值："#ffffff"（白色） | STRING | 是 | |
| `位置` | 文本在图像上的垂直位置。默认值："top" | COMBO | 是 | "top"<br />"bottom" |
| `对齐` | 文本的水平对齐方式。默认值："left" | COMBO | 是 | "left"<br />"center"<br />"right" |
| `描边` | 在文本周围绘制黑色描边。默认值：True | BOOLEAN | 是 | |

## 输出
| 输出名 | 描述 | 数据类型 |
|--------|------|----------|
| `图像` | 叠加了文本合成效果的图像或图像批次。 | IMAGE |
