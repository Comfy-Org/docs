Meshy：细化草稿模型节点接收先前生成的 3D 草稿模型，提升其质量，并可选择添加纹理。该节点向 Meshy API 提交细化任务，并在处理完成后返回最终的 3D 模型文件。

## 输入

| 参数 | 描述 | 数据类型 | 是否必填 | 取值范围 |
| --- | --- | --- | --- | --- |
| `model` | 指定用于细化的 AI 模型。目前仅提供 "latest" 模型。 | COMBO | 是 | `"latest"` |
| `meshy_task_id` | 要细化的草稿模型的唯一任务 ID。 | MESHY_TASK_ID | 是 | - |
| `enable_pbr` | 在基础颜色之外生成 PBR 贴图（金属度、粗糙度、法线）。注意：使用雕塑风格时应设为 false，因为雕塑风格会生成自己的一套 PBR 贴图。（默认值：`False`） | BOOLEAN | 否 | - |
| `texture_prompt` | 提供文本提示以指导纹理生成过程。最多 600 个字符。不能与 `texture_image` 同时使用。（默认值：空字符串） | STRING | 否 | - |
| `texture_image` | `texture_image` 和 `texture_prompt` 中每次只能使用一个。 | IMAGE | 否 | - |

**注意：** `texture_prompt` 和 `texture_image` 输入互斥。在同一操作中，不能同时提供文本提示和图像用于纹理生成。

## 输出

| 输出名称 | 描述 | 数据类型 |
| --- | --- | --- |
| `模型文件` | 生成的 GLB 模型文件名。（仅用于向后兼容） | STRING |
| `meshy_task_id` | 提交的细化作业的唯一任务 ID。 | MESHY_TASK_ID |
| `GLB` | 最终细化的 3D 模型，格式为 GLB。 | FILE3DGLB |
| `FBX` | 最终细化的 3D 模型，格式为 FBX。 | FILE3DFBX |
