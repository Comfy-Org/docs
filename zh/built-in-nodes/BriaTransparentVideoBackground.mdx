此节点使用 Bria 的 AI 服务移除视频背景，并输出去除背景后的帧画面及对应的 Alpha 遮罩。将两个输出连接到合成节点，或将其输入到保存 WEBM 节点以生成透明视频。

## ## 输入

| 参数 | 描述 | 数据类型 | 是否必填 | 范围 |
|------|------|----------|----------|------|
| `视频` | 待处理的输入视频 | VIDEO | 是 | - |
| `种子` | 种子控制节点是否重新运行；无论种子值如何，结果均非确定性（默认值：0） | INT | 是 | 0 至 2147483647 |

## ## 输出

| 输出名称 | 描述 | 数据类型 |
|----------|------|----------|
| `images` | 移除背景后的视频帧画面 | IMAGE |
| `mask` | 视频帧的 Alpha 遮罩 | MASK |
