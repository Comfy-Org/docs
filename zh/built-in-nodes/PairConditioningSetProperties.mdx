### 概述

**PairConditioningSetProperties** 节点允许您同时修改正向和负向条件对（conditioning pairs）的属性。它对两个条件输入应用强度调整、条件区域设置以及可选的遮罩或时序控制，并返回修改后的正向和负向条件数据。

### 输入

| 参数 | 描述 | 数据类型 | 是否必填 | 范围 |
| --- | --- | --- | --- | --- |
| `新正面条件` | 要修改的正向条件输入 | CONDITIONING | 是 | - |
| `新负面条件` | 要修改的负向条件输入 | CONDITIONING | 是 | - |
| `强度` | 应用于条件的强度乘数（默认值：1.0） | FLOAT | 是 | 0.0 至 10.0 |
| `设置条件区域` | 决定条件区域的计算方式（默认值："default"） | STRING | 是 | "default"<br />"mask bounds" |
| `遮罩` | 可选遮罩，用于约束条件区域 | MASK | 否 | - |
| `约束` | 可选钩子组，用于高级条件修改 | HOOKS | 否 | - |
| `间隔` | 可选时间步范围，用于限制条件生效的时间段 | TIMESTEPS_RANGE | 否 | - |

### 输出

| 输出名称 | 描述 | 数据类型 |
| --- | --- | --- |
| `positive` | 应用属性后的修改版正向条件 | CONDITIONING |
| `negative` | 应用属性后的修改版负向条件 | CONDITIONING |
