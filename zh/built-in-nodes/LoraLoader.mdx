此节点会自动检测 LoRA 文件夹（包括子文件夹）中的模型，对应的模型路径为 `ComfyUI\models\loras`。更多信息请参考《安装 LoRA 模型》。

LoRA 加载器节点主要用于加载 LoRA 模型。你可以将 LoRA 模型视为滤镜，能够为图像赋予特定的风格、内容和细节：

- 应用特定的艺术风格（如水墨画）
- 添加特定角色的特征（如游戏角色）
- 为图像添加特定细节
所有这些都可以通过 LoRA 实现。

## 输入

| 参数 | 数据类型 | 描述 |
| --- | --- | --- |
| `模型` | MODEL | 通常用于连接基础模型 |
| `CLIP` | CLIP | 通常用于连接 CLIP 模型 |
| `LoRA名称` | COMBO[STRING] | 选择要使用的 LoRA 模型名称 |
| `模型强度` | FLOAT | 取值范围为 -100.0 到 100.0，日常图像生成通常在 0~1 之间使用。数值越高，模型调整效果越明显 |
| `CLIP强度` | FLOAT | 取值范围为 -100.0 到 100.0，日常图像生成通常在 0~1 之间使用。数值越高，模型调整效果越明显 |

## 输出

| 参数 | 数据类型 | 描述 |
| --- | --- | --- |
| `模型` | MODEL | 应用了 LoRA 调整后的模型 |
| `CLIP` | CLIP | 应用了 LoRA 调整后的 CLIP 实例 |
