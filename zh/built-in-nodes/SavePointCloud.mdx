保存点云节点将 3D 点云文件保存到您的 ComfyUI 输出目录。它还会将点云数据以及可选的相机和模型信息传递给工作流中的其他节点。

## 输入

| 参数 | 描述 | 数据类型 | 是否必填 | 范围 |
|-----------|-------------|-----------|----------|-------|
| `model_3d` | 点云文件 (.ply) | FILE3D | 是 | - |
| `filename_prefix` | 保存文件名的前缀（默认："3d/ComfyUI"） | STRING | 是 | - |
| `viewport_state` | 来自 3D 视口节点的状态 | LOAD3D | 是 | - |
| `model_3d_info` | 可选的高级用途 3D 模型信息 | LOAD3DMODELINFO | 否 | - |
| `camera_info` | 可选的高级用途相机信息 | LOAD3DCAMERA | 否 | - |
| `width` | 预览图像的宽度（像素，默认：1024） | INT | 是 | 最小值：1，最大值：4096，步长：1 |
| `height` | 预览图像的高度（像素，默认：1024） | INT | 是 | 最小值：1，最大值：4096，步长：1 |

## 输出

| 输出名 | 描述 | 数据类型 |
|-------------|-------------|-----------|
| `model_3d` | 已保存的点云文件 | FILE3D |
| `model_3d_info` | 3D 模型的信息 | LOAD3DMODELINFO |
| `camera_info` | 视口的相机信息 | LOAD3DCAMERA |
| `width` | 预览图像的宽度 | INT |
| `height` | 预览图像的高度 | INT |
