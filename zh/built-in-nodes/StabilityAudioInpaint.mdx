根据文本指令转换现有音频样本的一部分。此节点允许您通过提供描述性提示来修改音频的特定部分，有效地“修复”或重新生成选定部分，同时保留音频的其余部分。

## 输入

| 参数 | 描述 | 数据类型 | 是否必填 | 范围 |
| --- | --- | --- | --- | --- |
| `模型` | 用于音频修复的 AI 模型。 | COMBO | 是 | "stable-audio-2.5" |
| `提示词` | 指导音频如何转换的文本描述（默认：空）。 | STRING | 是 |  |
| `音频` | 要转换的输入音频文件。音频时长必须在 6 到 190 秒之间。 | AUDIO | 是 |  |
| `时长` | 控制生成音频的时长（以秒为单位）（默认：190）。 | INT | 否 | 1-190 |
| `随机种` | 用于生成的随机种子（默认：0）。 | INT | 否 | 0-4294967294 |
| `步数` | 控制采样步数（默认：8）。 | INT | 否 | 4-8 |
| `开始遮罩` | 要转换的音频部分的起始位置（以秒为单位）（默认：30）。 | INT | 否 | 0-190 |
| `结束遮罩` | 要转换的音频部分的结束位置（以秒为单位）（默认：190）。 | INT | 否 | 0-190 |

**注意：** `mask_end` 值必须大于 `mask_start` 值。输入音频的时长必须在 6 到 190 秒之间。
