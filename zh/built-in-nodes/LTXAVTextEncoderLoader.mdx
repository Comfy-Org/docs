| `ckpt名称` | STRING | 是 | 多个选项可用 | 要加载的检查点文件名。可用选项从 `checkpoints` 文件夹中加载。 |
| `设备` | STRING | 否 | `"default"`<br />`"cpu"` | 指定加载模型的设备。使用 `"cpu"` 强制加载到 CPU。默认行为（`"default"`）使用系统的自动设备分配。 |
