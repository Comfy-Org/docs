通过 OpenAI 的 DALL·E 3 端点同步生成图像。此节点接收文本提示，并使用 OpenAI 的 DALL·E 3 模型创建相应图像，允许您指定图像质量、风格和尺寸。

| 参数 | 数据类型 | 是否必填 | 范围 | 描述 |
|-----------|-----------|----------|-------|-------------|
| `提示词` | STRING | 是 | - | DALL·E 的文本提示（默认值：""） |
| `种子` | INT | 否 | 0 到 2147483647 | 后端尚未实现（默认值：0） |
| `质量` | COMBO | 否 | "standard"<br />"hd" | 图像质量（默认值："standard"） |
| `风格` | COMBO | 否 | "natural"<br />"vivid" | 生动风格会使模型倾向于生成超现实且富有戏剧性的图像。自然风格则使模型生成更自然、超现实感较弱的图像。（默认值："natural"） |
| `尺寸` | COMBO | 否 | "1024x1024"<br />"1024x1792"<br />"1792x1024" | 图像尺寸（默认值："1024x1024"） |
