# LTXVImgToVideo 节点

LTXVImgToVideo 节点将输入图像转换为视频生成模型所需的视频潜在表示。它接收单张图像，使用 VAE 编码器将其扩展为帧序列，然后通过强度控制应用条件处理，以确定视频生成过程中原始图像内容被保留或修改的程度。

## 输入

| 参数 | 数据类型 | 必填 | 范围 | 描述 |
|-----------|-----------|----------|-------|-------------|
| `正面条件` | CONDITIONING | 是 | - | 用于引导视频生成的正面条件提示 |
| `负面条件` | CONDITIONING | 是 | - | 用于避免视频中出现特定元素的负面条件提示 |
| `vae` | VAE | 是 | - | 用于将输入图像编码到潜在空间的 VAE 模型 |
| `图像` | IMAGE | 是 | - | 要转换为视频帧的输入图像 |
| `宽度` | INT | 否 | 64 至 MAX_RESOLUTION | 输出视频的宽度（像素）（默认值：768，步长：32） |
| `高度` | INT | 否 | 64 至 MAX_RESOLUTION | 输出视频的高度（像素）（默认值：512，步长：32） |
| `长度` | INT | 否 | 9 至 MAX_RESOLUTION | 生成视频的帧数（默认值：97，步长：8） |
| `批量大小` | INT | 否 | 1 至 4096 | 同时生成的视频数量（默认值：1） |
| `强度` | FLOAT | 否 | 0.0 至 1.0 | 控制生成视频的第一帧中保留原始图像内容的程度。值为 1.0 时完全保留原始图像，值为 0.0 时允许最大程度的修改（默认值：1.0） |

## 输出

| 输出名称 | 数据类型 | 描述 |
|-------------|-----------|-------------|
| `负面条件` | CONDITIONING | 已应用视频帧遮罩处理后的正面条件 |

| `latent` | LATENT | 包含编码帧和噪声遮罩的视频潜在表示，用于视频生成 |
