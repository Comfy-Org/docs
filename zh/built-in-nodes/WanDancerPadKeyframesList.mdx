## 概述

此节点接收一个图像序列和可选的音轨，并将其分割成指定数量的填充片段。它旨在为视频生成准备关键帧序列，其中每个片段都被填充为一致的长度，并创建相应的遮罩来指示哪些帧是有效的。

## 输入

| 参数 | 数据类型 | 是否必需 | 范围 | 描述 |
|-----------|-----------|----------|-------|-------------|
| `图像` | IMAGE | 是 | 不适用 | 要分割成片段的输入图像序列。 |
| `片段长度` | INT | 是 | 1 到 10000 | 每个片段的帧长度（默认值：149）。 |
| `片段数量` | INT | 是 | 1 到 100 | 作为列表输出的填充片段数量（默认值：1）。 |
| `音频` | AUDIO | 否 | 不适用 | 为每个输出片段切分的音频。 |

## 输出

| 输出名称 | 数据类型 | 描述 |
|-------------|-----------|-------------|
| `关键帧掩码` | IMAGE | 填充后的关键帧序列列表，每个片段对应一个序列。 |
