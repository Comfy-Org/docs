# ElevenLabs 即时语音克隆节点

此节点通过分析1到8段人声音频录音，创建全新的独特语音模型。它将样本发送至ElevenLabs API，由API处理生成可用于文本转语音合成的语音克隆。

## 输入

| 参数 | 数据类型 | 必填 | 范围 | 描述 |
|-----------|-----------|----------|-------|-------------|
| `audio_*` | AUDIO | 是 | 1至8个文件 | 用于语音克隆的音频录音。您必须提供1到8个音频文件。 |
| `去除背景噪音` | BOOLEAN | 否 | True / False | 使用音频隔离技术从语音样本中移除背景噪音。（默认值：False） |

**注意：** 您必须至少提供一个音频文件，最多可提供八个。节点会自动为您添加的音频文件创建输入插槽。

## 输出

| 输出名称 | 数据类型 | 描述 |
|-------------|-----------|-------------|
| `voice` | ELEVENLABS_VOICE | 新创建的克隆语音模型的唯一标识符。此输出可连接至其他ElevenLabs文本转语音节点。 |
