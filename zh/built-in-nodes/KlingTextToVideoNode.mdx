Kling 文本转视频节点可将文本描述转换为视频内容。它接收文本提示词，并根据指定的配置设置生成相应的视频序列。该节点支持不同的宽高比和生成模式，以生成不同时长和质量的视频。

## 输入

| 参数 | 数据类型 | 是否必填 | 取值范围 | 描述 |
|-----------|-----------|----------|-------|-------------|
| `提示词` | STRING | 是 | - | 正向文本提示词 |
| `负面提示词` | STRING | 是 | - | 反向文本提示词 |
| `CFG` | FLOAT | 否 | 0.0 至 1.0 | 配置缩放值（默认值：1.0） |
| `宽高比` | COMBO | 否 | 来自 KlingVideoGenAspectRatio 的选项 | 视频宽高比设置（默认值："16:9"） |
| `模式` | COMBO | 否 | 提供多个选项 | 用于视频生成的配置，格式为：模式 / 时长 / 模型名称。（默认值：modes[8]） |

## 输出
