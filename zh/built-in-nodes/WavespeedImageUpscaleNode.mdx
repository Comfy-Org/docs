WaveSpeed 图像放大节点使用外部 AI 服务来提高图像的分辨率和质量。它接收一张输入照片，并将其放大到更高的目标分辨率（例如 2K、4K 或 8K），从而生成更清晰、更精细的结果。

## 输入

| 参数 | 数据类型 | 是否必填 | 取值范围 | 描述 |
|-----------|-----------|----------|-------|-------------|
| `model` | STRING | 是 | `"SeedVR2"`<br />`"Ultimate"` | 用于放大的 AI 模型。"SeedVR2" 和 "Ultimate" 提供不同的质量和价格等级。 |
| `image` | IMAGE | 是 | | 待放大的输入图像。 |
| `target_resolution` | STRING | 是 | `"2K"`<br />`"4K"`<br />`"8K"` | 放大后图像的期望输出分辨率。 |

**注意：** 此节点仅接受单张输入图像。提供一批图像将导致错误。

## 输出

| 输出名称 | 数据类型 | 描述 |
|-------------|-----------|-------------|
| `image` | IMAGE | 放大后的高分辨率输出图像。 |
