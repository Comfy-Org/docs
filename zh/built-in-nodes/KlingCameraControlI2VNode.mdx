# Kling 图像转视频摄像机控制节点

此节点可将静态图像转换为具有专业摄像机运镜的电影级视频。这款专门的图像转视频节点允许您控制虚拟摄像机动作，包括变焦、旋转、平移、俯仰和第一人称视角，同时保持对原始图像的关注。摄像机控制目前仅在专业模式下支持，使用 kling-v1-5 模型，时长为5秒。

## 输入

| 参数 | 描述 | 数据类型 | 必填 | 范围 |
| --- | --- | --- | --- | --- |
| `起始帧` | 参考图像 - URL或Base64编码字符串，不能超过10MB，分辨率不低于300x300px，宽高比在1:2.5到2.5:1之间。Base64不应包含data:image前缀。 | IMAGE | 是 | - |
| `提示词` | 正向文本提示词，描述期望的视频内容 | STRING | 是 | - |
| `负面提示词` | 负向文本提示词，描述生成视频中需要避免的内容 | STRING | 是 | - |
| `CFG` | 控制文本引导的强度。数值越高，输出越严格遵循提示词（默认值：0.75） | FLOAT | 否 | 0.0 到 1.0 |
| `宽高比` | 生成视频的宽高比（默认值："16:9"） | COMBO | 否 | `"16:9"`<br />`"9:16"`<br />`"1:1"` |
