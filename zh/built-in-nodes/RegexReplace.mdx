RegexReplace 节点使用正则表达式模式在字符串中查找和替换文本。它允许您搜索文本模式并将其替换为新文本，并提供选项来控制模式匹配的工作方式，包括大小写敏感性、多行匹配以及限制替换次数。

## 输入

| 参数 | 说明 | 数据类型 | 是否必填 | 取值范围 |
| --- | --- | --- | --- | --- |
| `字符串` | 要搜索和替换的输入文本字符串 | STRING | 是 | - |
| `正则表达式模式` | 在输入字符串中搜索的正则表达式模式 | STRING | 是 | - |
| `替换内容` | 用于替换匹配模式的替换文本 | STRING | 是 | - |
| `忽略大小写` | 启用后，模式匹配将忽略大小写差异（默认值：True） | BOOLEAN | 否 | - |
| `多行模式` | 启用后，^ 和 $ 的行为将变为匹配每行的开头/结尾，而不仅仅是整个字符串的开头/结尾（默认值：False） | BOOLEAN | 否 | - |
| `点号匹配所有` | 启用后，点号 (.) 字符将匹配包括换行符在内的任何字符。禁用时，点号不匹配换行符（默认值：False） | BOOLEAN | 否 | - |
| `计数` | 最大替换次数。设置为 0 可替换所有匹配项（默认值）。设置为 1 仅替换第一个匹配项，设置为 2 替换前两个匹配项，以此类推。（默认值：0） | INT | 否 | 0-100 |

## 输出

| 输出名称 | 说明 | 数据类型 |
| --- | --- | --- |
| `output` | 应用了指定替换后的修改字符串 | STRING |
