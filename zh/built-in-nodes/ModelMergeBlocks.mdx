ModelMergeBlocks 专为高级模型合并操作而设计，允许将两个模型进行整合，并可针对模型的不同部分自定义混合比例。此节点通过根据指定参数有选择地合并两个源模型的组件，从而便于创建混合模型。

## 输入

| 参数 | 描述 | 数据类型 |
| --- | --- | --- |
| `模型1` | 要合并的第一个模型。它作为基础模型，第二个模型的补丁将应用于此模型。 | `MODEL` |
| `模型2` | 从中提取补丁并应用于第一个模型的第二个模型，具体应用方式取决于指定的混合比例。 | `MODEL` |
| `输入` | 指定模型输入层的混合比例。它决定了第二个模型的输入层有多少被合并到第一个模型中。 | `FLOAT` |
| `中间` | 定义模型中间层的混合比例。此参数控制模型中间层的整合程度。 | `FLOAT` |
| `输出` | 确定模型输出层的混合比例。它通过调整第二个模型输出层的贡献来影响最终输出。 | `FLOAT` |

## 输出

| 参数 | 描述 | 数据类型 |
| --- | --- | --- |
| `model` | 合并后得到的模型，它是两个输入模型的混合体，并根据指定的混合比例应用了补丁。 | MODEL |
