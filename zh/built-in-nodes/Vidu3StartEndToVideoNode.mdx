此节点通过在提供的起始帧和结束帧之间进行插值，并在文本提示的引导下生成视频。它使用 Vidu Q3 模型在两幅图像之间创建无缝过渡，生成指定时长和分辨率的视频。

## 输入

| 参数 | 描述 | 数据类型 | 是否必填 | 范围 |

| `model.resolution` | 输出视频的分辨率。选择 `模型` 后显示此参数。 | COMBO | 是 | `"720p"`<br />`"1080p"` |
| `model.duration` | 输出视频的时长（秒），默认值为 5。选择 `模型` 后显示此参数。 | INT | 是 | 1 到 16 |
| `model.audio` | 启用后，输出带声音的视频（包括对话和音效），默认值为 False。选择 `模型` 后显示此参数。 | BOOLEAN | 是 | `True` / `False` |

| `提示词` | 指导视频生成的文本描述（最多 2000 个字符）。 | STRING | 是 | - |
