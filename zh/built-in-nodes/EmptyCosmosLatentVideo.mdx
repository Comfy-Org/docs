EmptyCosmosLatentVideo 节点用于创建具有指定维度的空潜在视频张量。它会生成一个填充为零的潜在表示，可作为视频生成工作流的起点，并支持可配置的宽度、高度、长度和批次大小参数。

## 输入

| 参数 | 数据类型 | 是否必填 | 范围 | 描述 |
|-----------|-----------|----------|-------|-------------|
| `宽度` | INT | 是 | 16 至 MAX_RESOLUTION | 潜在视频的宽度（像素），默认值：1280，必须能被 16 整除 |
| `高度` | INT | 是 | 16 至 MAX_RESOLUTION | 潜在视频的高度（像素），默认值：704，必须能被 16 整除 |
| `长度` | INT | 是 | 1 至 MAX_RESOLUTION | 潜在视频的帧数，默认值：121，必须能被 8 整除 |
| `批量大小` | INT | 否 | 1 至 4096 | 每批次生成的潜在视频数量，默认值：1 |

## 输出

| 输出名称 | 数据类型 | 描述 |
|-------------|-----------|-------------|
| `samples` | LATENT | 生成的值为零的空潜在视频张量 |
