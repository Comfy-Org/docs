Runway 文本转图像节点使用 Runway 的 Gen 4 模型，根据文本提示生成图像。您可以提供文本描述，并可选择包含参考图像来引导图像生成过程。该节点负责处理 API 通信并返回生成的图像。

## 输入

| 参数 | 数据类型 | 是否必填 | 范围 | 描述 |
|-----------|-----------|----------|-------|-------------|
| `提示词` | STRING | 是 | - | 用于生成的文本提示（默认值：""） |
| `比例` | COMBO | 是 | "16:9"<br />"1:1"<br />"21:9"<br />"2:3"<br />"3:2"<br />"4:5"<br />"5:4"<br />"9:16"<br />"9:21" | 生成图像的宽高比 |
| `参考图像` | IMAGE | 否 | - | 用于引导生成的可选参考图像 |

**注意：** 参考图像的尺寸不得超过 7999x7999 像素，且宽高比必须在 0.5 到 2.0 之间。当提供参考图像时，它将引导图像生成过程。

## 输出

| 输出名称 | 数据类型 | 描述 |
|-------------|-----------|-------------|
| `output` | IMAGE | 基于文本提示和可选参考图像生成的图像 |
