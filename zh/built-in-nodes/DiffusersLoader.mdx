DiffusersLoader 节点用于加载 diffusers 格式的预训练模型。它会搜索包含 `model_index.json` 文件的有效 diffusers 模型目录，并将其作为 MODEL、CLIP 和 VAE 组件加载到流程中。此节点属于已弃用的加载器类别，提供与 Hugging Face diffusers 模型的兼容性。

## 输入

| 参数 | 数据类型 | 是否必填 | 范围 | 描述 |
|-----------|-----------|----------|-------|-------------|
| `模型路径` | STRING | 是 | 提供多个选项<br />（从 diffusers 文件夹自动填充） | 要加载的 diffusers 模型目录路径。该节点会自动扫描配置的 diffusers 文件夹中的有效 diffusers 模型，并列出可用选项。 |

## 输出

| 输出名称 | 数据类型 | 描述 |
|-------------|-----------|-------------|
| `MODEL` | MODEL | 从 diffusers 格式加载的模型组件 |
| `CLIP` | CLIP | 从 diffusers 格式加载的 CLIP 模型组件 |
| `VAE` | VAE | 从 diffusers 格式加载的 VAE（变分自编码器）组件 |
