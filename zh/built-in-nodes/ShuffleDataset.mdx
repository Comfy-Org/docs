Shuffle Dataset（数据集打乱）节点接收一个图像列表并随机改变其顺序。它使用种子值来控制随机性，确保可以重现相同的打乱顺序。这在处理数据集前随机化图像序列时非常有用。

## 输入

| 参数 | 数据类型 | 是否必填 | 范围 | 描述 |
|-----------|-----------|----------|-------|-------------|
| `图像` | IMAGE | 是 | - | 需要打乱的图像列表。 |
| `随机种` | INT | 否 | 0 到 18446744073709551615 | 随机种子。值为 0 时每次将产生不同的打乱结果。（默认值：0） |

## 输出

| 输出名称 | 数据类型 | 描述 |
|-------------|-----------|-------------|
| `图像` | IMAGE | 与输入相同的图像列表，但顺序已随机打乱。 |
