此节点将高斯泼溅 3D 文件保存到 ComfyUI 输出目录，并为 3D 查看器提供预览数据。

## 输入
| 参数 | 描述 | 数据类型 | 是否必需 | 范围 |
|-----------|-------------|-----------|----------|-------|
| `model_3d` | 一个高斯泼溅 3D 文件。 | FILE3D | 是 | - |
| `filename_prefix` | 保存文件名的前缀（默认值："3d/ComfyUI"） | STRING | 是 | - |
| `viewport_state` | 来自 3D 加载器节点的当前视口状态。 | LOAD3D | 是 | - |
| `model_3d_info` | 用于预览的附加 3D 模型信息。 | LOAD3DMODELINFO | 否 | - |
| `camera_info` | 用于预览的相机信息。 | LOAD3DCAMERA | 否 | - |
| `width` | 预览的宽度（像素，默认值：1024） | INT | 是 | 最小值：1，最大值：4096，步长：1 |
| `height` | 预览的高度（像素，默认值：1024） | INT | 是 | 最小值：1，最大值：4096，步长：1 |

## 输出
| 输出名 | 描述 | 数据类型 |
|-------------|-------------|-----------|
| `model_3d` | 已保存的高斯泼溅 3D 文件。 | FILE3D |
| `model_3d_info` | 用于预览的 3D 模型信息。 | LOAD3DMODELINFO |
| `camera_info` | 用于预览的相机信息。 | LOAD3DCAMERA |
| `width` | 预览的宽度。 | INT |
| `height` | 预览的高度。 | INT |
