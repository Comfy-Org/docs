此节点将高斯泼溅 3D 文件保存到输出目录。它处理文件保存过程，并为 3D 视口提供预览数据。

## ## 输入

| 参数 | 描述 | 数据类型 | 是否必填 | 范围 |
|-----------|-------------|-----------|----------|-------|
| `model_3d` | 一个高斯泼溅 3D 文件。 | FILE3D | 是 | SplatAny<br />PLY<br />SPLAT<br />SPZ<br />KSPLAT |
| `filename_prefix` | 保存文件名的前缀（默认值："3d/ComfyUI"）。 | STRING | 是 | 任意有效文件名前缀 |
| `viewport_state` | 包含相机和模型信息的当前视口状态。 | LOAD3D | 是 | - |
| `model_3d_info` | 视口的附加模型 3D 信息。 | LOAD3DMODELINFO | 否 | - |
| `camera_info` | 视口预览的相机信息。 | LOAD3DCAMERA | 否 | - |
| `width` | 预览的宽度（默认值：1024）。 | INT | 是 | 1 到 4096 |
| `height` | 预览的高度（默认值：1024）。 | INT | 是 | 1 到 4096 |

## ## 输出

| 输出名 | 描述 | 数据类型 |
|-------------|-------------|-----------|
| `model_3d` | 已保存的高斯泼溅 3D 文件。 | FILE3D |
| `model_3d_info` | 视口的模型 3D 信息。 | LOAD3DMODELINFO |
| `camera_info` | 视口预览的相机信息。 | LOAD3DCAMERA |
| `width` | 预览的宽度。 | INT |
| `height` | 预览的高度。 | INT |
