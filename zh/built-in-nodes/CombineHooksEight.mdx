组合钩子 [8] 节点将最多八个不同的钩子组合合并为一个统一的组合钩子组合。它接收多个钩子输入，并利用 ComfyUI 的钩子组合功能将其合并。这使得您可以整合多个钩子配置，从而在高级工作流中实现精简处理。

## 输入

| 参数 | 描述 | 数据类型 | 输入类型 | 默认值 | 范围 |
| --- | --- | --- | --- | --- | --- |
| `hooks_A` | 要合并的第一个钩子组合 | HOOKS | 可选 | 无 | - |
| `hooks_B` | 要合并的第二个钩子组合 | HOOKS | 可选 | 无 | - |
| `hooks_C` | 要合并的第三个钩子组合 | HOOKS | 可选 | 无 | - |
| `hooks_D` | 要合并的第四个钩子组合 | HOOKS | 可选 | 无 | - |
| `hooks_E` | 要合并的第五个钩子组合 | HOOKS | 可选 | 无 | - |
| `hooks_F` | 要合并的第六个钩子组合 | HOOKS | 可选 | 无 | - |
| `hooks_G` | 要合并的第七个钩子组合 | HOOKS | 可选 | 无 | - |
| `hooks_H` | 要合并的第八个钩子组合 | HOOKS | 可选 | 无 | - |

**注意：** 所有输入参数均为可选。该节点仅合并已提供的钩子组合，忽略任何留空的输入。您可以提供一至八个钩子组合进行合并。
