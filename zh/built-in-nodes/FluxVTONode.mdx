# Flux 虚拟试穿

此节点通过将提供的服装图像穿在人物身上来执行虚拟试穿。它使用 BFL Flux VTO API 生成人物穿着指定服装的真实感图像。

## ## 输入

| 参数 | 描述 | 数据类型 | 是否必需 | 范围 |
|-----------|-------------|-----------|----------|-------|
| `人物` | 待穿衣的人物图像。 | IMAGE | 是 | - |
| `服装` | 待应用的服装图像。 | IMAGE | 是 | - |
| `提示词` | 可选的自然语言风格指令（例如服装应如何合身）。 | STRING | 否 | - |
| `随机种子` | 用于生成噪声的随机种子。 | INT | 否 | 0 到 18446744073709551615 |

## ## 输出

| 输出名称 | 描述 | 数据类型 |
|-------------|-------------|-----------|
| `image` | 显示人物穿着所提供服装的结果图像。 | IMAGE |
