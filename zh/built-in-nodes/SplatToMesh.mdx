# 从 Splat 提取网格

此节点将 3D 高斯泼溅转换为彩色网格表面。其工作原理是将高斯体光栅化到密度网格上，在选定的密度水平提取等值面，然后应用可选的平滑和清理操作，生成干净的彩色三角网格。

## 输入

| 参数 | 描述 | 数据类型 | 必填 | 范围 |
|-----------|-------------|-----------|----------|-------|
| `splat` | 要转换为网格的输入高斯泼溅 | SPLAT | 是 | - |
| `分辨率` | 沿最长轴的密度网格分辨率。值越高，表面细节越精细，但需要更多显存和处理时间（随 resolution^3 增长）。默认值：384 | INT | 是 | 64 - 768（步长 16） |
| `核宽` | 体素中最大泼溅半宽。每个高斯体在其自身 3-sigma 范围内进行光栅化，窗口大小受此值限制。小表面保持轻量，大表面不会被截断。如果稀疏泼溅出现间隙，可增大此值。默认值：5 | INT | 是 | 1 - 8 |
| `平滑` | Taubin 网格平滑迭代次数。与模糊密度不同，此方法在平滑表面的同时保持体积不变（不收缩）。0 表示不平滑。默认值：0 | INT | 是 | 0 - 60 |
| `等值面级别` | 等值面水平。通过 Otsu 阈值自动选取；此值会偏置自动选取（1.0 = 自动，较低值产生更饱满/连接更紧密的表面，较高值产生更薄/更紧凑的表面）。默认值：0.4 | FLOAT | 是 | 0.0 - 2.0（步长 0.01） |
| `最小组件` | 丢弃顶点数小于此值的连通组件。可移除分离的漂浮块和双壁内壳。0 表示保留所有组件。默认值：500 | INT | 是 | 0 - 100000（步长 50） |
| `最小不透明度` | 在网格化前忽略透明度低于此值的高斯体。默认值：0.02 | FLOAT | 是 | 0.0 - 1.0（步长 0.01） |
| `颜色锐化` | 锐化顶点纹理。1.0 为物理正确混合；较高值使每个体素的颜色偏向其主导高斯体而非平均相邻颜色（减少纹理模糊）。仅影响颜色，不影响几何形状。默认值：2.0 | FLOAT | 是 | 1.0 - 8.0（步长 0.5） |

## 输出

| 输出名称 | 描述 | 数据类型 |
|-------------|-------------|-----------|
| `mesh` | 提取的彩色网格，采用无光照渲染（类似自发光）以匹配泼溅外观 | MESH |
