## 概述

Vidu 文本转视频生成节点可根据文本描述创建视频。它利用 Vidu 视频生成模型，将您的文本提示转换为视频内容，并支持自定义时长、宽高比和视觉风格等设置。

## 输入

| 参数 | 说明 | 数据类型 | 是否必填 | 取值范围 |
| --- | --- | --- | --- | --- |
| `模型` | 模型名称 | COMBO | 是 | `viduq1` |
| `提示词` | 用于视频生成的文本描述 | STRING | 是 | - |
| `时长` | 输出视频的时长（秒），默认值：5 | INT | 否 | 5-5 |
| `种子` | 视频生成的随机种子（0 表示随机），默认值：0 | INT | 否 | 0-2147483647 |
| `宽高比` | 输出视频的宽高比 | COMBO | 否 | `16:9`<br />`9:16`<br />`1:1` |
| `分辨率` | 支持的分辨率可能因模型和时长而异 | COMBO | 否 | `1080p` |
| `运动幅度` | 画面中物体的运动幅度 | COMBO | 否 | `auto`<br />`small`<br />`medium`<br />`large` |

**注意：** `prompt` 字段为必填项，不能为空。`duration` 参数当前固定为 5 秒。

## 输出

| 输出名称 | 说明 | 数据类型 |
| --- | --- | --- |
| `output` | 根据文本提示生成的视频 | VIDEO |
