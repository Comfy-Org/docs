Grok Video 节点可根据文本描述生成短视频。它可以通过提示词从零创建视频，或基于提示词对单张输入图像进行动画处理。该节点向外部 API 发送请求并返回生成的视频。

## 输入

| 参数 | 数据类型 | 是否必填 | 取值范围 | 描述 |
|-----------|-----------|----------|-------|-------------|
| `model` | COMBO | 是 | `"grok-imagine-video"`<br />`"grok-imagine-video-beta"` | 用于视频生成的模型。 |
| `prompt` | STRING | 是 | - | 所需视频的文本描述。 |
| `resolution` | COMBO | 是 | `"480p"`<br />`"720p"` | 输出视频的分辨率。 |
| `aspect_ratio` | COMBO | 是 | `"auto"`<br />`"16:9"`<br />`"4:3"`<br />`"3:2"`<br />`"1:1"`<br />`"2:3"`<br />`"3:4"`<br />`"9:16"` | 输出视频的宽高比（默认值："auto"）。 |
| `duration` | INT | 是 | 1 至 15 | 输出视频的时长（秒）（默认值：6）。 |
| `seed` | INT | 是 | 0 至 2147483647 | 用于确定节点是否应重新运行的种子；实际结果与种子无关，具有不确定性（默认值：0）。 |
| `image` | IMAGE | 否 | - | 可选的输入图像，用于生成动画。 |

**注意：** 如果提供了 `image`，则仅支持单张图像。提供多张图像将导致错误。`prompt` 在去除首尾空格后至少需要包含 1 个字符。

## 输出

| 输出名称 | 数据类型 | 描述 |
|-------------|-----------|-------------|
| `output` | VIDEO | 生成的视频。 |
