## 概述

使用 Rodin Gen-2.5 API 从文本提示生成 3D 模型。您可以在不同的质量模式（快速、常规或极致高清）之间进行选择，以平衡生成速度和输出质量。

## 输入

| 参数 | 描述 | 数据类型 | 是否必需 | 范围 |
| --- | --- | --- | --- | --- |
| `提示词` | 描述您想要生成的 3D 模型的文本提示。 | STRING | 是 | 最多 2500 个字符 |
| `模式` | 生成质量和速度模式。"Fast" 速度最快，"Extreme-High" 质量最高但耗时更长。 | COMBO | 是 | `"Fast"`<br />`"Regular"`<br />`"Extreme-High"` |
| `材质` | 生成 3D 模型的材质风格。 | COMBO | 是 | `"PBR"`<br />`"Matte"`<br />`"Shiny"` |
| `几何文件格式` | 输出 3D 模型的文件格式。 | COMBO | 是 | `"glb"`<br />`"obj"`<br />`"stl"`<br />`"usdz"` |
| `贴图模式` | 纹理生成模式。"None" 不生成纹理，"Generated" 创建标准纹理，"Generated+HD" 创建高清纹理。 | COMBO | 是 | `"None"`<br />`"Generated"`<br />`"Generated+HD"` |
| `随机种子` | 用于可重现结果的随机种子。使用相同的种子和相同的输入将产生相同的输出。 | INT | 是 | 0 到 2147483647 |
| `T/A 姿势` | 是否对生成的模型应用 T 姿势（双臂伸展）。 | BOOLEAN | 是 | True / False |
| `高清贴图` | 是否为模型生成高清纹理。 | BOOLEAN | 是 | True / False |
| `贴图去光照` | 是否对模型应用纹理增强（提升纹理质量）。 | BOOLEAN | 是 | True / False |
| `HighPack 插件` | 是否在标准模型之外额外生成高多边形版本。 | BOOLEAN | 是 | True / False |
| `包围盒宽度` | 边界框的宽度（世界单位）。 | INT | 是 | 1 到 1000 |
| `包围盒高度` | 边界框的高度（世界单位）。 | INT | 是 | 1 到 1000 |
| `包围盒长度` | 边界框的长度（世界单位）。 | INT | 是 | 1 到 1000 |
| `模型高度（厘米）` | 生成模型的高度（厘米）。 | INT | 是 | 1 到 300 |

**注意：** `prompt` 参数长度必须在 1 到 2500 个字符之间。如果未指定，`seed` 参数默认为 0（随机）。
