## 概述

使用 HappyHorse 模型，通过文本指令或参考图像编辑视频。输出时长为 3-15 秒，与输入视频匹配；超过 15 秒的输入将被截断。

## 输入

| 参数 | 描述 | 数据类型 | 是否必需 | 范围 |
| --- | --- | --- | --- | --- |
| `model` | 模型配置，包含模型选择、提示词、分辨率、宽高比以及可选的参考图像。 | DICT | 是 | 见下文 |
| `video` | 待编辑的视频。 | VIDEO | 是 | - |
| `seed` | 用于生成的随机种子（默认值：0）。 | INT | 是 | 0 至 2147483647 |
| `watermark` | 是否在结果中添加 AI 生成水印（默认值：False）。 | BOOLEAN | 否 | True / False |

### `model` 参数详情

`model` 参数是一个字典，包含以下字段：

| 字段 | 描述 | 数据类型 | 是否必需 | 范围 |
| --- | --- | --- | --- | --- |
| `model` | 要使用的 HappyHorse 视频编辑模型。 | STRING | 是 | `"happyhorse-1.0-video-edit"` |
| `prompt` | 编辑指令或风格迁移要求。长度至少为 1 个字符。 | STRING | 是 | - |
| `resolution` | 输出分辨率。 | STRING | 是 | `"720P"`<br />`"1080P"` |
| `ratio` | 宽高比。若未更改，则近似于输入视频的宽高比。 | STRING | 是 | `"16:9"`<br />`"9:16"`<br />`"1:1"`<br />`"4:3"`<br />`"3:4"` |
| `reference_images` | 可选的参考图像（image1、image2、image3、image4、image5），用于指导编辑。 | DICT | 否 | 0 至 5 张图像 |
