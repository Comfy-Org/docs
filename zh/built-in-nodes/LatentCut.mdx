LatentCut 节点用于从潜在样本中沿指定维度提取特定部分。通过指定维度（x、y 或 t）、起始位置和提取数量，您可以裁剪出潜在表示的一部分。该节点支持正负索引，并会自动调整提取数量，确保其不超过可用范围。

## 输入

| 参数 | 数据类型 | 是否必填 | 取值范围 | 描述 |
|-----------|-----------|----------|-------|-------------|
| `样本` | LATENT | 是 | - | 用于提取的输入潜在样本 |
| `维度` | COMBO | 是 | "x"<br />"y"<br />"t" | 裁剪潜在样本时沿用的维度 |
| `索引` | INT | 是 | -16384 到 16384 | 裁剪的起始位置（默认值：0）。正值从起始位置开始计数，负值从末尾开始计数。节点会自动将索引限制在潜在样本的有效范围内 |
| `数量` | INT | 是 | 1 到 16384 | 沿指定维度提取的元素数量（默认值：1）。如果该值超出起始索引后的可用数据范围，节点会自动减小该值 |

## 输出

| 输出名称 | 数据类型 | 描述 |
|-------------|-----------|-------------|
| `output` | LATENT | 从潜在样本中提取出的部分 |
