### 概述

Rodin 3D Regular 节点使用 Rodin API 生成 3D 资产。它接收输入图像，并通过 Rodin 服务进行处理，以创建 3D 模型。该节点处理从任务创建到下载最终 3D 模型文件的整个工作流程。

### 输入

| 参数 | 数据类型 | 是否必填 | 范围 | 描述 |
|-----------|-----------|----------|-------|-------------|
| `图像` | IMAGE | 是 | - | 用于 3D 模型生成的输入图像。可提供多张图像。 |
| `种子` | INT | 是 | - | 用于可重复结果的随机种子值。 |
| `材质类型` | STRING | 是 | - | 应用于 3D 模型的材质类型。 |
| `多边形数量` | STRING | 是 | - | 生成 3D 模型的目标多边形数量。此参数决定质量模式和网格复杂度。 |

### 输出

| 输出名称 | 数据类型 | 描述 |
|-------------|-----------|-------------|
| `3D Model Path` | STRING | 生成的 3D 模型文件路径（为保持向后兼容性而保留）。 |
| `GLB` | FILE3DGLB | 以 GLB 格式生成的 3D 模型。 |
