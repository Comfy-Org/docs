Vidu2 文本转视频生成节点可根据文本描述创建视频。该节点通过连接外部 API，基于您的提示词生成视频内容，并允许您控制视频长度、视觉风格和格式。

## 输入

| 参数 | 描述 | 数据类型 | 是否必填 | 取值范围 |
| --- | --- | --- | --- | --- |
| `模型` | 用于视频生成的AI模型。目前仅支持一个可用模型。 | COMBO | 是 | `"viduq2"` |
| `提示词` | 用于视频生成的文本描述，最大长度为2000个字符。 | STRING | 是 | - |
| `时长` | 生成视频的时长（秒）。可通过滑块调整数值（默认值：5）。 | INT | 否 | 1 至 10 |
| `种子` | 用于控制生成随机性的数值，可实现可重复的结果。生成后可进行控制（默认值：1）。 | INT | 否 | 0 至 2147483647 |
| `宽高比` | 视频宽度与高度的比例关系。 | COMBO | 否 | `"16:9"`<br />`"9:16"`<br />`"3:4"`<br />`"4:3"`<br />`"1:1"` |
| `分辨率` | 生成视频的像素尺寸。这是一个高级参数。 | COMBO | 否 | `"720p"`<br />`"1080p"` |
| `背景音乐` | 是否为生成的视频添加背景音乐（默认值：False）。这是一个高级参数。 | BOOLEAN | 否 | - |

## 输出

| 输出名称 | 描述 | 数据类型 |
| --- | --- | --- |
| `output` | 生成的视频文件。 | VIDEO |
