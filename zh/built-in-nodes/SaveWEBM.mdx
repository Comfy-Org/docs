SaveWEBM 节点将图像序列保存为 WEBM 视频文件。该节点接收多张输入图像，并使用 VP9 或 AV1 编码器将其编码为视频，支持可配置的质量设置和帧率。生成的视频文件将保存到输出目录，并包含提示信息等元数据。

## 输入

| 参数 | 数据类型 | 是否必填 | 取值范围 | 说明 |
|-----------|-----------|----------|-------|-------------|
| `图像` | IMAGE | 是 | - | 要编码为视频帧的输入图像序列 |
| `文件名前缀` | STRING | 否 | - | 输出文件名的前缀（默认值："ComfyUI"） |
| `编解码器` | COMBO | 是 | "vp9"<br />"av1" | 用于编码的视频编解码器 |
| `帧率` | FLOAT | 否 | 0.01-1000.0 | 输出视频的帧率（默认值：24.0） |
| `crf` | FLOAT | 否 | 0-63.0 | 质量设置，crf 值越高，质量越低但文件越小；crf 值越低，质量越高但文件越大（默认值：32.0） |

## 输出

| 输出名称 | 数据类型 | 说明 |
|-------------|-----------|-------------|
| `ui` | PREVIEW | 显示已保存 WEBM 文件的视频预览 |
