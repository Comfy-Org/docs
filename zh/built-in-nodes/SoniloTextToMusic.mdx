Sonilo 文本转音乐节点使用 Sonilo 的 AI 模型，根据文本描述生成音乐。您提供描述所需音乐的提示词，节点会向 Sonilo 服务发送请求以创建音频文件。您可以指定目标时长，或让模型根据提示词自行推断。

## 输入

| 参数 | 数据类型 | 是否必填 | 范围 | 描述 |
|-----------|-----------|----------|-------|-------------|
| `prompt` | STRING | 是 | 不适用 | 描述要生成音乐的文本提示词。此为必填字段。 |
| `duration` | INT | 否 | 0 到 360 | 目标时长（秒）。设置为 0 可让模型根据提示词推断时长。最长：6 分钟（360 秒）。默认值：0。 |
| `seed` | INT | 否 | 0 到 18446744073709551615 | 用于可复现性的随机种子。目前 Sonilo 服务会忽略此参数，但保留以保持工作流一致性。默认值：0。 |

**注意：** `seed` 输入是为保持工作流一致性而提供的，但目前不会影响 Sonilo 服务的输出。
