使用 Google 的 Veo 3 API 根据文本提示生成视频。此节点支持多种 Veo 3 模型，包括快速版和精简版，并允许您指定视频分辨率、时长和音频生成。

## 输入

| 参数 | 描述 | 数据类型 | 是否必填 | 取值范围 |
| --- | --- | --- | --- | --- |
| `提示词` | 视频的文本描述（默认值：""） | STRING | 是 | - |
| `宽高比` | 输出视频的宽高比（默认值："16:9"） | COMBO | 是 | "16:9"<br />"9:16" |
| `分辨率` | 输出视频的分辨率。veo-3.1-lite 和 veo-3.0 模型不支持 4K 分辨率。（默认值："720p"） | COMBO | 否 | "720p"<br />"1080p"<br />"4k" |
| `负面提示词` | 负面文本提示，用于指导视频中应避免的内容（默认值：""） | STRING | 否 | - |
| `时长` | 输出视频的时长（秒），步长为 2（默认值：8） | INT | 否 | 4-8 |
| `优化提示词` | 此参数已弃用，其值将被忽略。（默认值：True） | BOOLEAN | 否 | - |

| `随机种` | 视频生成的随机种子（0 表示随机）（默认值：0） | INT | 否 | 0-4294967295 |
| `图像` | 可选的参考图像，用于指导视频生成 | IMAGE | 否 | - |
