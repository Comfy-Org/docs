基于单张图像，使用 Tripo 的 API 同步生成 3D 模型。此节点接收输入图像，并通过多种自定义选项（纹理、质量、模型属性）将其转换为 3D 模型。

## 输入

| 参数 | 描述 | 数据类型 | 是否必填 | 取值范围 |
| --- | --- | --- | --- | --- |
| `图像` | 用于生成 3D 模型的输入图像 | IMAGE | 是 | - |
| `模型版本` | 用于生成的 Tripo 模型版本 | COMBO | 否 | 提供多个选项 |
| `风格` | 生成模型的样式设置（默认值："None"） | COMBO | 否 | 提供多个选项 |
| `纹理` | 是否为模型生成纹理（默认值：True） | BOOLEAN | 否 | - |
| `PBR` | 是否使用基于物理的渲染（默认值：True） | BOOLEAN | 否 | - |
| `模型种子` | 模型生成的随机种子（默认值：42） | INT | 否 | - |
| `朝向` | 生成模型的朝向设置 | COMBO | 否 | 提供多个选项 |
| `纹理种子` | 纹理生成的随机种子（默认值：42） | INT | 否 | - |
| `纹理质量` | 纹理生成的质量等级（默认值："standard"） | COMBO | 否 | "standard"<br />"detailed" |
| `纹理对齐` | 纹理贴图的对齐方式（默认值："original_image"） | COMBO | 否 | "original_image"<br />"geometry" |
| `面数限制` | 生成模型的最大面数，-1 表示无限制（默认值：-1） | INT | 否 | -1 到 500000 |
| `四边形` | 是否使用四边形面而非三角形面（默认值：False） | BOOLEAN | 否 | - |
| `几何质量` | 几何体生成的质量等级（默认值："standard"） | COMBO | 否 | "standard"<br />"detailed" |

**注意：** `image` 参数为必填项，必须提供该参数节点才能正常运行。若未提供图像，节点将引发 RuntimeError。

## 输出

| 输出名称 | 描述 | 数据类型 |
| --- | --- | --- |
| `模型文件` | 生成的 3D 模型文件（仅用于向后兼容） | STRING |
| `模型任务ID` | 用于追踪模型生成过程的任务 ID | MODEL_TASK_ID |
| `GLB` | 以 GLB 格式生成的 3D 模型 | FILE3DGLB |
