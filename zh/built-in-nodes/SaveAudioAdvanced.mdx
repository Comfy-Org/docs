将输入的音频保存到您的 ComfyUI 输出目录。此节点支持以多种格式导出音频，包括 FLAC、MP3 和 Opus，并具有可配置的质量设置。

## ## 输入

| 参数 | 描述 | 数据类型 | 必填 | 范围 |
|-----------|-------------|-----------|----------|-------|
| `audio` | 要保存的音频。 | AUDIO | 是 | - |
| `filename_prefix` | 要保存的文件前缀。可包含格式化标记，例如 %date:yyyy-MM-dd%。（默认值："audio/ComfyUI"） | STRING | 是 | - |
| `format` | 保存音频的文件格式。 | COMBO | 是 | "flac"<br />"mp3"<br />"opus" |

当选择 "mp3" 作为格式时，将出现一个 `quality` 子参数，其选项如下："V0"、"128k"、"320k"（默认值："V0"）。

当选择 "opus" 作为格式时，将出现一个 `quality` 子参数，其选项如下："64k"、"96k"、"128k"、"192k"、"320k"（默认值："128k"）。

## ## 输出

| 输出名称 | 描述 | 数据类型 |
|-------------|-------------|-----------|
| `ui` | 包含已保存音频文件信息的 UI 输出。 | UI |
