此节点创建一个仅应用于模型组件的 LoRA（低秩适配）钩子，完全保持 CLIP 组件不变。它加载一个 LoRA 文件，并以指定的强度应用于模型，同时将 CLIP 强度设置为零。该节点可与之前的钩子链接，构建复杂的修改流程。

## 输入

| 参数 | 数据类型 | 是否必填 | 范围 | 描述 |
|-----------|-----------|----------|-------|-------------|
| `LoRA名称` | STRING | 是 | 提供多个选项 | 从 loras 文件夹中加载的 LoRA 文件名 |
| `模型强度` | FLOAT | 是 | -20.0 到 20.0 | 将 LoRA 应用于模型组件的强度乘数（默认值：1.0） |
| `前一个约束` | HOOKS | 否 | - | 可选的前置钩子，用于与此钩子链接 |

## 输出

| 输出名称 | 数据类型 | 描述 |
|-------------|-----------|-------------|
| `hooks` | HOOKS | 创建的 LoRA 钩子，可应用于模型处理流程 |
