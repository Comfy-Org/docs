SolidMask 节点用于生成一个在整个区域内具有指定值的均匀遮罩。它旨在创建具有特定尺寸和强度的遮罩，适用于各种图像处理和遮罩任务。

## 输入

| 参数 | 数据类型 | 描述 |
|-----------|-------------|-------------|
| `明度`   | FLOAT       | 指定遮罩的强度值，影响其在后续操作中的整体外观和用途。 |
| `宽度`   | INT         | 确定生成遮罩的宽度，直接影响其尺寸和宽高比。 |
| `高度`  | INT         | 设置生成遮罩的高度，影响其尺寸和宽高比。 |

## 输出

| 参数 | 数据类型 | 描述 |
|-----------|-------------|-------------|
| `mask`    | MASK        | 输出一个具有指定尺寸和值的均匀遮罩。 |
