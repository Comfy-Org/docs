SolidMask 节点用于生成一个在整个区域内具有指定值的均匀遮罩。它旨在创建具有特定尺寸和强度的遮罩，适用于各种图像处理和遮罩任务。

## 输入

| 参数 | 描述 | 数据类型 |
| --- | --- | --- |
| `明度` | 指定遮罩的强度值，影响其在后续操作中的整体外观和用途。 | FLOAT |
| `宽度` | 确定生成遮罩的宽度，直接影响其尺寸和宽高比。 | INT |
| `高度` | 设置生成遮罩的高度，影响其尺寸和宽高比。 | INT |

## 输出

| 参数 | 描述 | 数据类型 |
| --- | --- | --- |
| `mask` | 输出一个具有指定尺寸和值的均匀遮罩。 | MASK |
