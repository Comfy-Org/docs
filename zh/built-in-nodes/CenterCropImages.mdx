中心裁剪图像节点从图像中心裁剪出指定宽度和高度的区域。它会计算输入图像的中心区域，并提取出定义尺寸的矩形区域。如果请求的裁剪尺寸大于图像本身，裁剪将被限制在图像的边界内。

## 输入

| 参数 | 数据类型 | 是否必填 | 范围 | 描述 |
|-----------|-----------|----------|-------|-------------|
| `image` | IMAGE | 是 | - | 待裁剪的输入图像。 |
| `宽度` | INT | 是 | 1 至 8192 | 裁剪区域的宽度（默认值：512）。 |
| `高度` | INT | 是 | 1 至 8192 | 裁剪区域的高度（默认值：512）。 |

## 输出

| 输出名称 | 数据类型 | 描述 |
|-------------|-----------|-------------|
| `image` | IMAGE | 中心裁剪操作后生成的图像。 |
