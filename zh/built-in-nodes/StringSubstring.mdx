StringSubstring 节点用于从较长的字符串中提取一部分文本。它通过指定起始位置和结束位置来定义要提取的区间，并返回这两个位置之间的文本。

## 输入

| 参数 | 数据类型 | 是否必填 | 范围 | 描述 |
|-----------|-----------|----------|-------|-------------|
| `字符串` | STRING | 是 | - | 要从中提取文本的输入字符串。支持多行文本。 |
| `起始` | INT | 是 | - | 子字符串的起始位置索引。第一个字符的索引为 0。 |
| `结束` | INT | 是 | - | 子字符串的结束位置索引。此索引处的字符不包含在结果中。 |

## 输出

| 输出名称 | 数据类型 | 描述 |
|-------------|-----------|-------------|
| `output` | STRING | 从输入文本中提取的子字符串，包含从 `起始` 位置开始到（但不包括）`结束` 位置的所有字符。 |
