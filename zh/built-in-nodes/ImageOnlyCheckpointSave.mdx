ImageOnlyCheckpointSave 节点用于保存包含模型、CLIP 视觉编码器和 VAE 的检查点文件。它会创建一个带有指定文件名前缀的 safetensors 文件，并将其存储在输出目录中。此节点专为将图像相关模型组件一起保存到单个检查点文件而设计。

## 输入

| 参数 | 描述 | 数据类型 | 必填 | 范围 |
| --- | --- | --- | --- | --- |
| `模型` | 要保存在检查点中的模型 | MODEL | 是 | - |
| `clip视觉` | 要保存在检查点中的 CLIP 视觉编码器 | CLIP_VISION | 是 | - |
| `vae` | 要保存在检查点中的 VAE（变分自编码器） | VAE | 是 | - |
| `文件名前缀` | 输出文件名的前缀（默认值："checkpoints/ComfyUI"） | STRING | 是 | - |
| `prompt` | 工作流提示数据的隐藏参数 | PROMPT | 否 | - |
| `extra_pnginfo` | 附加 PNG 元数据的隐藏参数 | EXTRA_PNGINFO | 否 | - |

## 输出

| 输出名称 | 描述 | 数据类型 |
| --- | --- | --- |
| - | 此节点不返回任何输出 | - |
