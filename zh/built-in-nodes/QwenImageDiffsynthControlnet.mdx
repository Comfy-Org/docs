QwenImageDiffsynthControlnet 节点应用扩散合成控制网络补丁来修改基础模型的行为。它使用图像输入和可选的遮罩，以可调节的强度引导模型的生成过程，创建一个融合了控制网络影响的补丁模型，从而实现更可控的图像合成。

## 输入

| 参数 | 数据类型 | 是否必填 | 范围 | 描述 |
|-----------|-----------|----------|-------|-------------|
| `模型` | MODEL | 是 | - | 待应用控制网络补丁的基础模型 |
| `模型补丁` | MODEL_PATCH | 是 | - | 要应用于基础模型的控制网络补丁模型 |
| `vae` | VAE | 是 | - | 扩散过程中使用的 VAE（变分自编码器） |
| `图像` | IMAGE | 是 | - | 用于引导控制网络的输入图像（仅使用 RGB 通道） |
| `强度` | FLOAT | 是 | -10.0 到 10.0 | 控制网络影响的强度（默认值：1.0） |
| `遮罩` | MASK | 否 | - | 可选遮罩，用于定义控制网络应应用的区域（内部会进行反转处理） |

**注意：** 当提供遮罩时，会自动进行反转处理（1.0 - 遮罩）并调整形状以匹配控制网络处理所需的尺寸。节点会根据模型补丁是 ZImage 控制类型还是标准 DiffSynth 控制网络，采用不同的内部处理方法。
