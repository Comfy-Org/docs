# OpenAIDalle2

通过 OpenAI 的 DALL·E 2 端点同步生成图像。

## 工作原理

此节点连接到 OpenAI 的 DALL·E 2 API，根据文本描述创建图像。当您提供文本提示时，节点会将其发送到 OpenAI 的服务器，服务器生成相应的图像并将其返回给 ComfyUI。该节点可在两种模式下运行：仅使用文本提示的标准图像生成模式，或同时提供图像和遮罩时的图像编辑模式。在编辑模式下，节点使用遮罩确定原始图像中应修改的部分，同时保持其他区域不变。
