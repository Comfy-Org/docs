此节点用于根据指定的批次索引和长度，从给定批次中提取特定的潜在样本子集。它允许对潜在样本进行选择性处理，便于对批次中的较小片段进行操作，以提高效率或实现针对性处理。

## 输入

| 参数 | 数据类型 | 描述 |
|---------------|-------------|-------------|
| `Latent` | `LATENT` | 从中提取子集的潜在样本集合。此参数对于确定待处理样本的源批次至关重要。 |
| `批次索引` | `INT` | 指定批次中开始提取样本子集的起始索引。此参数支持从批次中的特定位置进行针对性提取。 |
| `长度` | `INT` | 定义从指定起始索引开始提取的样本数量。此参数控制待处理子集的大小，允许灵活操作批次片段。 |

## 输出

| 参数 | 数据类型 | 描述 |
|-----------|-------------|-------------|
