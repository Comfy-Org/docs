SD_4XUpscale_Conditioning 节点用于准备扩散模型放大图像所需的 conditioning 数据。该节点接收输入图像和 conditioning 数据，然后应用缩放和噪声增强，生成修改后的 conditioning 以指导放大过程。节点输出正向和负向 conditioning，以及对应放大尺寸的潜空间表示。

## 输入

| 参数 | 数据类型 | 是否必填 | 范围 | 描述 |
|-----------|-----------|----------|-------|-------------|
| `图片` | IMAGE | 是 | - | 待放大的输入图像 |
| `正面条件` | CONDITIONING | 是 | - | 正向 conditioning 数据，引导生成朝向期望内容 |
| `负面条件` | CONDITIONING | 是 | - | 负向 conditioning 数据，引导生成远离不期望内容 |
| `缩放比例` | FLOAT | 否 | 0.0 - 10.0 | 应用于输入图像的缩放因子（默认值：4.0） |
| `噪波增强` | FLOAT | 否 | 0.0 - 1.0 | 放大过程中添加的噪声量（默认值：0.0） |

## 输出

| 输出名称 | 数据类型 | 描述 |

| `latent` | LATENT | 与放大后尺寸匹配的空潜空间表示 |
