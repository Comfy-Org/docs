# Ideogram 4 调度器

Ideogram 4 调度器节点基于 Ideogram 4 参考调度表生成扩散采样过程中的 sigma 值序列（噪声水平）。它会创建一个自定义噪声调度表，该表会根据图像尺寸进行调整，并允许通过统计参数进行微调。

## ## 输入

| 参数 | 描述 | 数据类型 | 是否必需 | 范围 |
|------|------|----------|----------|------|
| `步数` | 生成调度表所需的采样步数（默认值：20） | INT | 是 | 1 至 200 |
| `宽度` | 图像的宽度（像素）（默认值：1024） | INT | 是 | 256 至 8192（步长：16） |
| `高度` | 图像的高度（像素）（默认值：1024） | INT | 是 | 256 至 8192（步长：16） |
| `均值` | logit-正态分布的均值参数，控制中心噪声水平（默认值：0.0） | FLOAT | 是 | -10.0 至 10.0（步长：0.05） |
| `标准差` | logit-正态分布的标准差参数，控制噪声水平的分布范围（默认值：1.75） | FLOAT | 是 | 0.1 至 5.0（步长：0.05） |

## ## 输出

| 输出名称 | 描述 | 数据类型 |
|----------|------|----------|
| `SIGMAS` | 表示噪声调度表的 sigma 值张量，长度等于 `steps + 1`。这些值从高噪声递减至低噪声，最终值设为 0.0 以实现完全去噪。 | SIGMAS |
