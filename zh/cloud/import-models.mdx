## 概述

Comfy Cloud 允许你直接从 Civitai 和 Hugging Face 导入模型。此功能仅对 **Creator** 及以上订阅用户开放。

## 要求

- **订阅等级**: Creator 或更高
- **支持来源**: Civitai 和 Hugging Face

## 如何导入模型

### 1. 导入功能入口
![Import](/images/cloud/import_model/import_model.png)
1. 点击左侧边栏的 **Models** 按钮打开模型库。
2. 在模型库弹窗中，点击 **Import** 按钮。

### 2. 粘贴链接
1. 粘贴你从 huggingface 或者 civitai 上获取的模型链接
![Import Modal](/images/cloud/import_model/import_model_modal-1.png)

2. 如果链接有效，则你可以看到对应的通过标识，并可以点击 `Continue` 按钮进行下一步
![Import Modal](/images/cloud/import_model/import_model_modal-2.png)

3. 选择导入的模型类型和对应的文件夹
![Import Modal](/images/cloud/import_model/import_model_modal-3.png)

4. 等待模型下载完成 
![Import Modal](/images/cloud/import_model/import_model_modal-4.png)

5. 下载完成后即可在模型库中找到对应的模型

## 如何查看已导入的模型

在模型库中，在下拉筛选中，选择 “My Models” 即可筛选出你导入的模型

![Imported Models](/images/cloud/import_model/imported_model.png)

## 如何获取导入模型链接

### 1. 从 Civitai 获取链接

1. 前往 [Civitai](https://civitai.com/) 找到你想导入的模型。
2. 使用鼠标右键点击下载按钮，选择"复制链接地址"，即可获取对应的模型下载链接

![Copy Civitai Link](/images/cloud/import_model/copy_civitai_link.png)

### 2. 从 Hugging Face 获取链接

1. 前往 [Hugging Face](https://huggingface.co/) 找到你想导入的模型仓库。
2. 进入模型文件页面通常受支持的格式是 “.safetensor”, ".sft" 的模型，你可以节点具体的模型名称进入对应详情页面

![Copy Huggingface model link](/images/cloud/import_model/copy_huggingface_link-1.png)

3. 在详情页面点击 `Copy Download Link` 按钮即可获取到模型下载链接

![Copy Huggingface model link](/images/cloud/import_model/copy_huggingface_link-2.png)

## 导入私有模型

如果你想从 Hugging Face 或 Civitai 导入私有模型，需要先配置你的 API 密钥。

### 1. 生成 API 密钥

首先，你需要从对应平台生成 API 密钥：

<Tabs>
  <Tab title="Civitai">
    1. 登录 Civitai 后访问 [https://civitai.com/user/account](https://civitai.com/user/account)。
    2. 在账户设置中生成 API 密钥。

    ![Civitai API Key](/images/cloud/import_model/civitai_key-1.png)
  </Tab>
  <Tab title="Hugging Face">
    1. 访问 [https://huggingface.co/settings/tokens](https://huggingface.co/settings/tokens)。
    2. 生成你的 API token。

    ![Hugging Face API Key](/images/cloud/import_model/huggingface_key-1.png)

    3. 确保你的 token 具有对应私有模型仓库的读取权限。

    ![Hugging Face Token 权限](/images/cloud/import_model/huggingface_key-2.png)
  </Tab>
</Tabs>

### 2. 保存 API 密钥

1. 前往 [Comfy Cloud 设置](https://cloud.comfy.org)，进入 **Secrets** 部分。

![Settings Secrets](/images/cloud/import_model/setting_secerts-1.png)

2. 点击添加你的 API 密钥（密钥名称可以自由命名）：

![添加 Secret Key](/images/cloud/import_model/setting_secerts-2.png)

3. 确保你的 API 密钥具有访问你想导入的私有模型的权限。

### 3. 导入私有模型

配置好 API 密钥后，你可以使用与公开模型相同的流程导入私有模型：

1. 从 Hugging Face 或 Civitai 获取模型链接。
2. 在导入弹窗中粘贴链接。
3. 系统会自动使用你保存的 API 密钥来访问私有模型。

<Note>
请确保你的 API 密钥具有访问特定私有模型所需的权限。
</Note>

## 常见问题

<AccordionGroup>
  <Accordion title="上传的模型有什么限制？">
    有。仅支持 **safetensor** 文件格式。文件大小基本没有限制（最大 100GB）。
  </Accordion>

  <Accordion title="我上传的模型是私有的还是公开的？">
    你导入的模型是**私有的**。只有你自己可以在模型库中看到这些模型。
  </Accordion>

  <Accordion title="我可以从本地硬盘上传自定义模型吗？">
    目前，你需要先将模型上传到**公开的 Hugging Face 或 Civitai 仓库**，然后再从这些平台导入。
  </Accordion>

  <Accordion title="我可以上传分块的模型文件吗？">
    不可以。你需要先将分块的模型文件合并成一个文件后再导入。
  </Accordion>

  <Accordion title="这个功能包含在我当前的云订阅中吗？">
    此功能仅适用于 **Creator** 和 **Pro** 订阅计划。
  </Accordion>

  <Accordion title="这个功能在桌面版或便携版 Comfy 上可用吗？">
    不可以。这是**仅限云端**的功能。
  </Accordion>

  <Accordion title="我可以删除已上传的模型吗？">
    可以。你可以删除自己导入的模型。请注意，Comfy 内部团队上传的模型无法删除。
  </Accordion>

  <Accordion title="BYOM 可以导入哪些模型类型？">
    不限于下列示例。BYOM 支持 Comfy Cloud 在导入时**类型选择器**中列出的任意模型文件夹。该列表来自 Comfy Cloud 支持的 ComfyUI 模型文件夹，以此为准。

    实际使用中，只要 ComfyUI 将文件视为模型权重（checkpoint、LoRA、VAE、ControlNet、embedding、upscaler、IP-Adapter、SAM、audio encoder 等），且该文件夹出现在选择器中，就可以导入。

    常见示例：

    - **Checkpoint 与 UNet**：`checkpoints`、`diffusion_models`
    - **LoRA 与适配器**：`loras`、`animatediff_motion_lora`、`cogvideox_loras` 及类似适配器文件夹
    - **VAE**：`vae`、`vae_approx`
    - **ControlNet**：`controlnet`
    - **CLIP 与 text encoder**：`clip_vision`、`text_encoders`
    - **Embedding / textual inversion**：`embeddings`（在 `CLIPTextEncode` 提示词中使用 `embedding:my-embedding-name` 引用）
    - **Upscaler**：`upscale_models`、`latent_upscale_models`
    - **视觉 / 分割 / 抠图**：`ipadapter`、`gligen`、`sams`、`sam3`、`rmbg`、`ultralytics_bbox`、`ultralytics_segm`
    - **选择器中的其他文件夹**：`audio_encoders`、`model_patches`、`style_models`、`photomaker`、`LLM`、`CogVideo` 等

    真正限制导入的不是模型类别，而是以下规则：

    - **文件格式**：仅支持 `.safetensors`（即使 ComfyUI 本地对该文件夹支持 `.ckpt`、`.pth` 或 `.onnx`）
    - **来源**：仅支持 Civitai 或 Hugging Face 下载链接（不支持本地上传）
    - **禁止的文件夹**：不支持导入 `nlf` 模型
  </Accordion>
</AccordionGroup>
