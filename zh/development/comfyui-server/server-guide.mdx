ComfyUI 默认作为 HTTP 服务器在本地运行。你可以通过编程方式调用它来提交工作流、上传文件、下载输出和监控进度 — 无需打开浏览器。

## 启动服务器

启动 ComfyUI 时，它会自动启动一个 HTTP 服务器：

```bash
# 默认服务器地址
http://127.0.0.1:8188
```

以无界面模式运行 ComfyUI（不打开浏览器）：

<CodeGroup>
```bash
# 不打开浏览器运行
python main.py --headless
```

```bash
# 指定自定义端口
python main.py --port 8288
```

```bash
# 监听所有网络接口（用于远程访问）
python main.py --listen 0.0.0.0
```
</CodeGroup>

<Tip>
  查看完整启动参数列表请运行 `python main.py --help`，或参阅[安装指南](/zh/installation/manual_install)。
</Tip>
