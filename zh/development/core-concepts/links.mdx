## 连线连接节点

在 ComfyUI 中，节点之间绘制的线条或曲线称为 **_连线_**（有时也称为 **_连接_** 或导线）。它们将数据从一个节点的输出传递到另一个节点的输入，从而定义工作流的流程。

连线可以以多种视觉样式显示：曲线、直角线、直线或完全隐藏。

![连线样式](/images/interface/link/link_styles.jpg)

要更改连线样式，请转到 **设置菜单** → **显示（Lite Graph）** → **画面（Graph）** → **连线渲染模式（Link Render Mode）**。

![渲染模式设置](/images/interface/link/render_mode.jpg)

你也可以从 **画布菜单（Canvas Menu）** 临时隐藏所有连线。

![画布菜单](/images/interface/link/canvas_menu.jpg)

<Tip>
显示连线在学习、分享或调试工作流时特别有用——它们能让数据流一目了然。对于不打算编辑的已完成工作流，隐藏连线可以减少视觉干扰。
</Tip>
### 重新路由节点

当工作流变得复杂时，连接线可能会重叠或被节点遮挡，导致难以阅读。**重新路由**节点允许你在二维图空间内手动重定向连接线，从而保持布局的整洁和清晰。

![ComfyUI 重新路由节点](/images/interface/link/reroute.jpg)

ComfyUI 在图画布中也内置了原生的重新路由功能。对于新的工作流，我们建议使用此功能。

![ComfyUI 原生重新路由](/images/interface/link/native_reroute.jpg)
## 颜色编码

ComfyUI 中的每种数据类型都有独特的颜色。节点的输入/输出端口以及连接它们的链接线都遵循此配色方案。**只能连接相同颜色的端口**——这确保了工作流中的类型安全。

![ComfyUI 节点数据类型](/images/concepts/node/data_type.jpg)

| 数据类型                 | 颜色       |
|---------------------------|-------------|
| 扩散模型                  | 薰衣草色    |
| CLIP 模型                 | 黄色        |
| VAE 模型                  | 玫瑰色      |
| 条件化                    | 橙色        |
| 潜在图像                  | 粉色        |
| 像素图像                  | 蓝色        |
| 蒙版                      | 绿色        |
| 数字（整数或浮点数）      | 浅绿色      |
| 网格                      | 亮绿色      |
