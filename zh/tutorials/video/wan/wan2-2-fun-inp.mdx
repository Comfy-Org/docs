## Wan2.2 Fun Inp 首尾帧视频生成工作流示例

<UpdateReminder/>

此工作流提供两个版本：
1. 使用 lightx2v 的 [Wan2.2-Lightning](https://huggingface.co/lightx2v/Wan2.2-Lightning) 4步 LoRA 加速视频生成
2. 不带加速 LoRA 的 fp8_scaled 版本

下面是使用 RTX4090D 24GB 显存 GPU 测试的结果 640×640 分辨率，81 帧

| 模型类型               | 显存占用 | 首次生成时长 | 第二次生成时长 |
| ---------------------- | -------- | ------------ | -------------- |
| fp8_scaled             | 83%      | ≈ 524秒      | ≈ 520秒        |
| fp8_scaled + 4步 LoRA  | 89%      | ≈ 138秒      | ≈ 79秒         |

由于使用 LoRA 加速后速度提升明显，但视频动态有所损失，因此提供的工作流默认启用加速 LoRA 版本。如需启用另一版本，选择后使用 **Ctrl+B** 激活。

### 3. 按步骤完成工作流

请参考上一步下载的工作流。该工作流已预配置为使用加速 LoRA 版本。所有必需节点已连接以完成基本工作流执行：

- `Load Diffusion Model`: 加载扩散模型
- `Load CLIP Vision` 和 `Load CLIP`: 加载文本编码器
- `Load VAE`: 加载 VAE 模型
- `Load Start Image` 和 `Load End Image`: 加载首尾帧
- `WanFunInpaintToVideo`: 预览并运行推理
- `WanVideoReverse`: 将输出视频翻转以获得无缝循环
- `VHS VideoCombine`: 保存视频

### 4. 工作流说明

<Tip>
请注意使用正确的模型。`wan2.2_fun_inpaint_high_noise_14B_fp8_scaled.safetensors` 和 `wan2.2_fun_inpaint_low_noise_14B_fp8_scaled.safetensors` 都保存在同一文件夹中，名称相似。激活和未加速的 LoRA 工作流可能需要使用不同的模型。
</Tip>
