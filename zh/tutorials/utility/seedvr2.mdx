[SeedVR2](https://iceclear.github.io/projects/seedvr2/)（arXiv:2506.05301，来自字节跳动 Seed）是一种一步式扩散视频修复模型。它通过对真实数据进行对抗训练来高保真地修复和放大图像与视频。SeedVR2 已在 ComfyUI 中原生支持（PR [#14424](https://github.com/Comfy-Org/ComfyUI/pull/14424)）。

主要功能：

- **保守放大**：在提升清晰度的同时保留原始结构和细节
- **一步推理**：单次前向传播即可完成图像和视频放大
- **多种模型规模**：提供 3B 和 7B 两个版本，支持 FP16、FP8、INT8、NVFP4 和 MXFP8 量化
- **图像与视频兼顾**：支持单张图像放大和视频分辨率增强

SeedVR2 有两种规模。[3B 模型](https://huggingface.co/ByteDance-Seed/SeedVR2-3B) 是较小的版本，显存需求较低，适合硬件资源有限的用户。[7B 模型](https://huggingface.co/ByteDance-Seed/SeedVR2-7B) 是较大的版本，能提供更高质量的结果，但需要更多显存和处理时间。两者均采用 Apache 2.0 许可证。

## 模型安装

从 [Comfy-Org SeedVR2 仓库](https://huggingface.co/Comfy-Org/SeedVR2) 下载 SeedVR2 检查点并保存到对应的 ComfyUI 文件夹：

- [seedvr2_3b_fp16.safetensors](https://huggingface.co/Comfy-Org/SeedVR2/resolve/main/diffusion_models/seedvr2_3b_fp16.safetensors): 3B FP16 基础模型，显存需求较低
- [seedvr2_3b_fp8_e4m3fn.safetensors](https://huggingface.co/Comfy-Org/SeedVR2/resolve/main/diffusion_models/seedvr2_3b_fp8_e4m3fn.safetensors): 3B FP8 量化版，进一步降低显存占用
- [seedvr2_7b_fp16.safetensors](https://huggingface.co/Comfy-Org/SeedVR2/resolve/main/diffusion_models/seedvr2_7b_fp16.safetensors): 7B FP16 基础模型，画质更优
- [seedvr2_7b_fp8_e4m3fn.safetensors](https://huggingface.co/Comfy-Org/SeedVR2/resolve/main/diffusion_models/seedvr2_7b_fp8_e4m3fn.safetensors): 7B FP8 量化版，兼顾画质与显存
- [seedvr2_7b_sharp_fp16.safetensors](https://huggingface.co/Comfy-Org/SeedVR2/resolve/main/diffusion_models/seedvr2_7b_sharp_fp16.safetensors): 7B FP16 锐化版，细节更丰富
- [seedvr2_ema_vae_fp16.safetensors](https://huggingface.co/Comfy-Org/SeedVR2/resolve/main/vae/seedvr2_ema_vae_fp16.safetensors): VAE 检查点（所有模型变体共用）

将文件放入以下目录：

```
ComfyUI/
├── models/
│   ├── diffusion_models/
│   │   ├── seedvr2_3b_fp16.safetensors
│   │   ├── seedvr2_3b_fp8_e4m3fn.safetensors
│   │   ├── seedvr2_7b_fp16.safetensors
│   │   ├── seedvr2_7b_fp8_e4m3fn.safetensors
│   │   ├── seedvr2_7b_sharp_fp16.safetensors
│   └── vae/
│       └── seedvr2_ema_vae_fp16.safetensors
```

## 1. 图像放大

**功能说明：** 接受单张图像，使用 SeedVR2 模型进行放大。工作流包含之前/之后对比，便于直观验证放大结果。

<Card title="下载工作流" icon="download" href="https://github.com/Comfy-Org/workflow_templates/blob/main/templates/utility_seedvr2_image_upscale.json">
    下载 JSON 或在模板库中搜索 "SeedVR2 Image Upscale"
  </Card>

<Card title="下载示例图像" icon="image" href="https://raw.githubusercontent.com/Comfy-Org/workflow_templates/main/input/vr_ad.png">
    获取此工作流的示例输入图像
  </Card>

### 1.1 运行步骤

1. 将图像放入 `ComfyUI/input/` 文件夹中，然后在 `加载图像` 节点中选择它
2. 在 SeedVR2 放大子图中选择您的 SeedVR2 模型检查点
3. 点击 `队列` 或使用 `Ctrl(cmd) + 回车` 运行

输出图像将保存到 `ComfyUI/output/` 文件夹，前缀为 `ComfyUI-upscaled`。

### 提示

为获得更好效果，在将输入图像送入 SeedVR2 放大器之前，先使用 `ImageScaleToTotalPixels` 节点将其缩小至约 0.35 百万像素。对于高分辨率来源图像尤其有帮助。

## 2. 视频放大

**功能：** 使用 SeedVR2 模型对视频进行放大。该工作流支持可配置的输出分辨率和模型选择。

<Card title="下载工作流" icon="download" href="https://github.com/Comfy-Org/workflow_templates/blob/main/templates/utility_seedvr2_video_upscale.json">
    下载 JSON，或在模板库中搜索“SeedVR2 Video Upscale”
  </Card>

<Card title="下载示例视频" icon="download" href="https://raw.githubusercontent.com/Comfy-Org/workflow_templates/main/input/input_3d.mp4">
    获取此工作流的示例输入视频
  </Card>

### 2.1 运行步骤

1. 将视频放入 `ComfyUI/input/` 文件夹，并在 `Load Video` 节点中选择该视频
2. 设置**分辨率**参数（基于输出高度）：`1080` 对应 1080p，`1440` 对应 1440p，`2160` 对应 4K
3. 选择您的 SeedVR2 模型检查点。默认值为 `seedvr2_7b_fp16.safetensors`
4. `ImageFromBatch` 节点默认限制处理 96 帧。选中该节点并按 `Ctrl-B` 禁用以实现全长视频处理
5. 点击 `Queue` 或使用 `Ctrl(Cmd) + Enter` 运行

### 分辨率工作原理

分辨率设置基于输出高度。工作流会自动调整宽度以保持源纵横比：

| 分辨率设置 | 示例输出 (16:9) |
|:----------:|:--------------:|
| 1080 | 1920x1080 (1080p) |
| 1440 | 2560x1440 (1440p) |
| 2160 | 3840x2160 (4K) |

如果源视频是 1280x720，并且您将分辨率设置为 1080，则输出将为 1920x1080。在放大之前使用 `Image Scale` 节点将输入缩小到正确的起始大小。放大器会对输入进行中心裁剪，以匹配目标纵横比。

### 性能

更高的目标分辨率和更大的模型（7B）需要显著更长的处理时间。从较低的分辨率开始，并在设置迭代时使用帧批处理限制，然后再进行完整渲染。
