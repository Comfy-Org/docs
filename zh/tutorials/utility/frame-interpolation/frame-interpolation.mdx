帧插值在现有帧之间生成中间帧，使运动更平滑，输出更高帧率的视频。ComfyUI 原生支持 **FILM**（Google）和 **RIFE**（实时中间流估计）两种模型。

<Img src="https://raw.githubusercontent.com/Comfy-Org/workflow_templates/main/templates/utility_video_frame_interpolation-1.webp" alt="帧插值工作流对比效果" />

## 工作流概述

本工作流包含三个主要步骤：

1. **加载视频** — 使用标准的 LoadVideo 节点上传视频文件
2. **选择插值模型** — 通过模型子图配置选择 FILM 或 RIFE
3. **输出插值后的视频** — 结果以平滑的高帧率视频保存

<Img src="https://raw.githubusercontent.com/Comfy-Org/workflow_templates/main/templates/utility_video_frame_interpolation-2.webp" alt="帧插值工作流节点" />

### 子图参数

工作流通过子图暴露以下控制参数：

- **model_name** — 选择 FILM 或 RIFE 帧插值模型
- **multiplier** — 帧率倍数（如 2 倍使帧率翻倍）
- **enable_fps_multiplier** — 切换是否基于原始 FPS 应用倍数

下载模型文件并放置到 `ComfyUI/models/frame_interpolation/` 文件夹：

```
📂 ComfyUI/

│   └── 📂 frame_interpolation/

```

## 使用方法

1. **打开工作流** — 点击上方在 Comfy Cloud 中运行的按钮，或下载 JSON 拖入 ComfyUI
2. **加载视频** — 使用 LoadVideo 选择输入视频
3. **选择模型** — 在子图下拉菜单中选择 FILM 或 RIFE
4. **设置倍率** — 大多数场景推荐 2 倍；更高倍率效果更平滑但耗时更长
5. **生成** — 运行工作流并下载输出视频
