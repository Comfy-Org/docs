Seedance 2.0 参考生成视频节点（`ByteDance2ReferenceNode`）使用参考图片、视频片段和音频来引导主体的外观、运动和节奏。你可以在一个工作流中提供最多 **9 张参考图片**、**3 个参考视频**和 **3 个参考音频片段**。

<ReqHint/>
<UpdateReminder/>

## 工作流概览

参考生成视频工作流使用多个加载节点输入到 Seedance 2.0 参考节点：

1. **LoadImage（×4）** — 加载参考图片
2. **ByteDance2ReferenceNode** — 接收图片、可选视频/音频输入和文本提示，输出 VIDEO
3. **SaveVideo** — 将生成的视频保存到磁盘

## 节点参数

| 参数 | 说明 |
| :--- | :--- |
| `model` | 模型名称。默认：`Seedance 2.0` |
| `prompt` | 文字描述。建议使用包含时序、镜头方向和服装描述的结构化提示 |
| `resolution` | 输出分辨率。可选：`480p`、`720p`、`1080p` |
| `aspect_ratio` | 画面比例。可选：`16:9`、`9:16`、`1:1`、`4:5` |
| `duration` | 视频时长（秒） |
| `generate_audio` | 是否生成与视频同步的音频 |
| `seed` | 随机种子 |
| `cfg` | 无分类器引导尺度 |

## 输入

| 输入 | 类型 | 最多 | 说明 |
| :--- | :--- | :-: | :--- |
| `image_1` 至 `image_5` | IMAGE | 5 | 直接图片输入，用于参考外观 |
| `video_1` | VIDEO | 1 | 参考视频输入，用于运动引导 |
| `audio_1` | AUDIO | 1 | 参考音频输入，用于节奏和同步 |
| `asset_1` | STRING | 1 | 已验证真人肖像的 Asset ID |

## 运行工作流

1. **加载参考图片** — 使用 LoadImage 节点加载定义主体外观的图片。对于多套服装展示，可以为每套服装加载一张图片
2. **编写结构化提示词** — 描述每个片段，包括镜头角度、转场和主体动作。使用 `[00:00-00:02]` 格式进行时序拆分解获取更精确的控制
3. **设置参数** — 选择分辨率、画面比例、时长及音频生成
4. **运行工作流** — 点击"加入队列"或使用 `Ctrl(Cmd) + Enter`
5. **查看结果** — 生成的视频出现在 SaveVideo 节点中

## 提示

- **混合多种参考** — 使用图片控制外观，视频控制运动模式，音频控制节奏
- **结构化提示词** — 对于复杂多场景视频，使用带时间戳的逐镜头格式
- **服装一致性** — 在产品展示中，提示词应明确描述每套参考服装及其在每个时间段的可视性
- **真人 Asset ID** — 对于已验证的真人肖像，使用 `asset_1` 字符串输入而非直接图片输入
