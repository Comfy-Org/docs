Seedance 2.0 文本生成视频节点（`ByteDance2TextToVideoNode`）可直接从文字描述生成视频。Seedance 2.0 根据你的提示词负责场景构图、运动、镜头和节奏的完整呈现。

<ReqHint/>
<UpdateReminder/>

## 工作流概览

文本生成视频工作流是最简单的 Seedance 2.0 工作流，仅包含两个节点：

1. **ByteDance2TextToVideoNode** — 接收文本提示并输出 VIDEO
2. **SaveVideo** — 将生成的视频保存到磁盘

## 节点参数

| 参数 | 说明 |
| :--- | :--- |
| `model` | 模型名称。默认：`Seedance 2.0` |
| `prompt` | 视频的文字描述。支持包含镜头指导、场景细节和时序拆分的结构化提示 |
| `resolution` | 输出分辨率。可选：`480p`、`720p`、`1080p` |
| `aspect_ratio` | 画面比例。可选：`16:9`、`9:16`、`1:1`、`4:5` |
| `duration` | 视频时长（秒） |
| `generate_audio` | 是否生成与视频同步的音频 |
| `seed` | 随机种子。可选：`randomize`、`fixed` 或具体数值 |
| `cfg` | 无分类器引导尺度 |

## 运行工作流

1. **编写提示词** — 详细描述场景、主体、动作、镜头运动和氛围。建议使用结构化格式，包含时序拆分和镜头指导
2. **设置参数** — 选择分辨率、画面比例、时长及是否生成音频
3. **运行工作流** — 点击"加入队列"或使用 `Ctrl(Cmd) + Enter`
4. **查看结果** — 生成的视频出现在 SaveVideo 节点中，并保存至 `ComfyUI/output/` 目录

## 提示

- **详细提示词** 能获得更好的效果。包含主体外观、环境、照明和镜头运动
- 多场景视频建议使用**镜头时序拆分**格式：为每个时间段指定镜头、动作和效果
- **音频同步**是原生的 — 开启 `generate_audio` 即可获得带同步音效的视频
- **Seed** 仅决定节点是否重新运行；使用 `fixed` 可保持结果可复现
