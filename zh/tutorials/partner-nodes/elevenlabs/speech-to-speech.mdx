**ElevenLabsSpeechToSpeech** 节点在保留原始语音内容的同时，转换输入音频的语音或说话风格。它可以改变口音、语音特征和情感表达。

<UpdateReminder/>

## 支持模型

| 模型 | 描述 |
| :---- | :---------- |
| `eleven_multilingual_sts_v2` | 多语言语音转语音转换 |

### 语音转语音参数

| 参数 | 描述 |
| :-------- | :---------- |
| **similarity** | 语音相似度（0.0–1.0），默认 0.5 |
| **stability** | 语音稳定性（0.0–1.0），默认自动 |
| **style_exaggeration** | 风格夸张度（auto / 0.0–1.0） |
| **speed** | 播放速度倍数 |
| **optimize_streaming_latency** | 延迟优化级别 |
| **output_format** | 音频输出格式（如 mp3_44100_192） |
| **seed** | 随机种子，用于可重现输出 |
| **seed_mode** | 种子模式：randomize / fixed |
| **enable_logging** | 启用/禁用日志 |

该工作流包含三个部分：

1. **源音频** — 通过 **LoadAudio** 加载输入音频文件
2. **使用预设语音** — 从 **ElevenLabsVoiceSelector** 选择目标语音
3. **创建语音克隆（可选）** — 加载音频样本并启用 **ElevenLabsInstantVoiceClone**

使用模板作为参考时的操作步骤：

1. 通过 **LoadAudio** 加载源音频（如 `api_elevenlabs_speech_to_speech.mp3`）
2. 从 **ElevenLabsVoiceSelector** 选择目标语音（如"Jessica (female, american)"）
3. （可选）启用 **ElevenLabsInstantVoiceClone** 使用克隆语音
4. 将源音频和目标语音连接到 **ElevenLabsSpeechToSpeech**
5. 根据需要调整语音参数（相似度、稳定性、速度）
6. 点击 **运行** 或按 `Ctrl(cmd) + Enter`
7. 转换后的音频通过 **SaveAudioMP3** 保存

### 补充说明

- **ElevenLabsInstantVoiceClone** 及其关联的 **LoadAudio** 节点默认处于绕过状态，请用 `Ctrl-B` 启用
- 适用于口音转换、角色语音变换和配音工作
