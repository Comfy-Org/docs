**ElevenLabsTextToDialogue** 节点生成多说话人对话音频，每个参与者使用不同的语音。每位说话者都有自己的文字输入和语音选择。

<ReqHint/>
<UpdateReminder/>

## 支持模型

| 模型 | 描述 |
| :---- | :---------- |
| `eleven_v3` | 英语优化的高质量对话生成 |

### 文字转对话参数

| 参数 | 描述 |
| :-------- | :---------- |
| **similarity** | 语音相似度（0.0–1.0），默认 0.5 |
| **stability** | 语音稳定性（0.0–1.0），默认自动 |
| **model** | 语音模型选择 |
| **turn_count** | 对话轮次（说话人切换次数） |
| **text_1 / text_2** | 每位说话者的文字内容 |
| **output_format** | 音频输出格式（如 mp3_44100_192） |
| **seed** | 随机种子，用于可重现输出 |
| **seed_mode** | 种子模式：randomize / fixed |

该工作流提供两个选项：

1. **使用预设语音** — 从两个 **ElevenLabsVoiceSelector** 节点中选择不同的语音（如"Roger (male, american)"和"Laura (female, american)"）
2. **创建语音克隆** — 加载音频样本并启用 **ElevenLabsInstantVoiceClone**

使用模板作为参考时的操作步骤：

1. 使用 **ElevenLabsVoiceSelector** 节点为每位说话者选择语音
2. （可选）启用 **ElevenLabsInstantVoiceClone** 使用克隆语音
3. 在 **ElevenLabsTextToDialogue** 节点中输入每位说话者的对话文字
4. 设置 **turn_count** 控制说话人的交替次数
5. 点击 **运行** 或按 `Ctrl(cmd) + Enter`
6. 生成的对话音频通过 **SaveAudioMP3** 保存

### 补充说明

- 语音克隆部分默认处于绕过状态，请用 `Ctrl-B` 启用
- 对于多轮对话，节点会自动在两个语音之间交替
- 每位说话者的文字字段支持独立内容，实现自然的对话流程
