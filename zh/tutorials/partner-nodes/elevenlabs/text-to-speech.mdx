**ElevenLabsTextToSpeech** 节点使用 ElevenLabs 语音模型将文字转换为自然语音。它支持多语言语音和可配置的输出格式。

<ReqHint/>
<UpdateReminder/>

## 支持模型

| 模型 | 语言 |
| :---- | :-------- |
| `eleven_multilingual_v2` | 29 种语言的多语言支持 |
| `eleven_v3` | 英语优化，高质量 |

### 文字转语音参数

| 参数 | 描述 |
| :-------- | :---------- |
| **text** | 要转换为语音的文字内容 |
| **similarity** | 语音相似度（0.0–1.0），默认 0.5 |
| **stability** | 语音稳定性（0.0–1.0），默认自动 |
| **style_exaggeration** | 风格夸张度（auto / 0.0–1.0） |
| **speed** | 播放速度倍数 |
| **optimize_streaming_latency** | 延迟优化级别 |
| **output_format** | 音频输出格式（如 mp3_44100_192） |
| **seed** | 随机种子，用于可重现输出 |
| **seed_mode** | 种子模式：randomize / fixed |

该工作流提供两个选项：

1. **使用预设语音** — 从 **ElevenLabsVoiceSelector** 节点中选择一个语音（如"Roger (male, american)"）
2. **创建语音克隆** — 通过 **LoadAudio** 加载音频文件，连接到 **ElevenLabsInstantVoiceClone**，然后使用克隆的语音

使用模板作为参考时的操作步骤：

1. 在 **ElevenLabsVoiceSelector** 中选择预设语音，或提供音频样本并启用 **ElevenLabsInstantVoiceClone**
2. 在 **ElevenLabsTextToSpeech** 节点中输入文字
3. 选择模型（`eleven_multilingual_v2` 或 `eleven_v3`）
4. 根据需要调整语音参数（相似度、稳定性、速度）
5. 点击 **运行** 或按 `Ctrl(cmd) + Enter`
6. 生成的音频通过 **SaveAudioMP3** 节点保存

### 补充说明

- **ElevenLabsInstantVoiceClone** 节点默认处于绕过状态，使用语音克隆时请用 `Ctrl-B` 启用
- 输出格式默认使用 `mp3_44100_192` 高质量 MP3 音频
- 将 **seed** 设为 `0` 可忽略种子；大多数模型仅将其作为参考
- 定价详情请参见 [ElevenLabs 定价页](/zh/tutorials/partner-nodes/pricing#elevenlabs)
