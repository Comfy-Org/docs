[ElevenLabs](https://elevenlabs.io) 是领先的 AI 语音和音频平台。ComfyUI 通过 **ElevenLabs** 合作伙伴节点组连接，提供文字转语音、语音转语音、对话生成、音效生成、人声分离和语音转文字能力，全部集成在基于节点的流程中。

<ReqHint/>
<UpdateReminder/>

## 你可以做什么

- **文字转语音** — 使用预设语音或自定义语音克隆将文字转换为自然语音
- **语音转语音** — 转换现有音频的语音或风格，同时保留内容
- **文字转对话** — 为每个参与者使用不同的语音生成多说话人对话
- **文字转音效** — 根据文字描述生成自定义音效
- **人声分离** — 从音频文件中分离前景语音和背景噪音
- **语音转文字** — 将音频转录为文字，包含逐词时间戳和语言检测

## 关键节点

| 节点 | 描述 |
| :--- | :---------- |
| **ElevenLabsTextToSpeech** | 使用 ElevenLabs 语音模型将文字转换为语音 |
| **ElevenLabsSpeechToSpeech** | 转换输入音频的语音/风格 |
| **ElevenLabsTextToDialogue** | 生成多说话人之间的对话 |
| **ElevenLabsTextToSoundEffects** | 根据文字提示创建音效 |
| **ElevenLabsAudioIsolation** | 移除背景噪音，分离语音轨道 |
| **ElevenLabsSpeechToText** | 将音频转录为文字，包含逐词详细信息 |
| **ElevenLabsVoiceSelector** | 选择预设的 ElevenLabs 语音 |
| **ElevenLabsInstantVoiceClone** | 从上传的音频创建语音克隆 |

定价详情请参见 [合作伙伴节点定价](/zh/tutorials/partner-nodes/pricing#elevenlabs)。
