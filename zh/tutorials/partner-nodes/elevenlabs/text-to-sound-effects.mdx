**ElevenLabsTextToSoundEffects** 节点根据文字描述生成自定义音效。在你的 ComfyUI 工作流中直接为游戏、视频和交互体验创建音频素材。

<ReqHint/>
<UpdateReminder/>

## 支持模型

| 模型 | 描述 |
| :---- | :---------- |
| `eleven_sfx_v2` | 音效生成模型 |

### 文字转音效参数

| 参数 | 描述 |
| :-------- | :---------- |
| **text** | 要生成的音效描述 |
| **model** | 音效生成模型选择 |
| **duration_seconds** | 生成音效的持续时间 |
| **prompt_influence** | 对提示的跟随程度（0.0–1.0） |
| **output_format** | 音频输出格式（如 mp3_44100_192） |

该工作流简洁直接：

1. 在 **ElevenLabsTextToSoundEffects** 中输入音效描述（如"交通噪音，繁忙的高速公路上有喇叭声"）
2. 选择模型（`eleven_sfx_v2`）
3. 设置音效的 **duration_seconds**
4. 调整 **prompt_influence** 以平衡创意与提示遵循度
5. 点击 **运行** 或按 `Ctrl(cmd) + Enter`
6. 生成的音频通过 **SaveAudioMP3** 保存

### 示例提示词

| 提示词 | 最佳用途 |
| :----- | :------- |
| "交通噪音，繁忙的高速公路" | 城市环境氛围 |
| "雨打窗户，远处有雷声" | 氛围背景 |
| "脚步声在木地板上，缓慢而刻意" | 游戏/声音设计 |
| "门吱呀打开，悬疑氛围" | 电影/视频制作 |

### 补充说明

- 较长的持续时间会消耗更多积分。每分钟费率请参见 [ElevenLabs 定价页](/zh/tutorials/partner-nodes/pricing#elevenlabs)
- **prompt_influence** 设为 0.3 可在保真度和多样性之间取得平衡；较低的值允许更多的创造性诠释
- 音效按生成的音频分钟数计费
