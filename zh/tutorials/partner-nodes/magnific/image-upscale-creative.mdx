**Magnific 创意图像放大**节点利用 AI 生成细节来放大图像。不同于简单的插值放大，它会添加合理的纹理、清晰度和细节，生成看起来更自然的高分辨率结果。

### 操作步骤

1. 在 **Load Image** 中加载需要放大的图像
2. 在 **Magnific Image Upscale Creative** 中输入描述图像内容的 **prompt**（帮助引导细节生成）
3. 选择 **upscale_factor**（最高 16×）
4. 选择 **creativity** 级别控制新细节的生成量
5. 调整 **resemblance**（与原始图像相似度）、**hdr**、**sharpen** 和 **noise** 微调输出
6. 在高级选项中，**preset** 选择常见场景预设（建议 `automatic`）
7. 可选启用 **tune_color** 进行色彩校正
8. 点击 **Run** 或按 `Ctrl(cmd) + Enter` 运行

### 控件参考

| 控件 | 类型 | 默认值 | 说明 |
| :--- | :--- | :----- | :--- |
| prompt | 文本 | — | 描述图像内容以引导 AI 细节生成 |
| upscale_factor | 下拉 | `4x` | 放大倍数（例如 `2x`、`4x`、`8x`、`16x`） |
| creativity | 下拉 | `standard` | 创意细节添加量（`low` 低、`standard` 标准、`high` 高） |
| resemblance | 整数 | 2 | 放大结果与原始图像的相似度（0–10） |
| creativity_scale | 整数 | 3 | 创意细节注入规模（0–10） |
| detail_boost | 整数 | 1 | 细节增强程度（0–10） |
| hdr | 整数 | 0 | HDR 效果强度（0–10） |
| sharpen | 整数 | 0 | 锐化强度（0–10） |
| noise | 下拉 | `automatic` | 降噪模式 |
| tune_color | 布尔 | true | 启用色彩调校 |

### 补充说明

- **prompt** — 帮助模型理解图像内容，从而生成符合上下文的细节
- **creativity** — `standard` 平衡保真度和细节增强；`high` 添加更多推测性细节，适合风格化图像
- **resemblance** — 值越高结果越接近原始图像；值越低允许更多创意重构
- 模板中的 **ImageCompare** 节点可并排预览前后对比
