**Magnific 图像重打光**节点基于文本提示调整输入图像的光照。它在改变整体光照方向、氛围和气氛的同时保留面部特征。

### 操作步骤

1. 在 **Load Image** 中加载需要重打光的图像
2. （可选）将**参考图像**连接到 `reference_image` 输入，以匹配其光照方向/风格
3. 在 **Magnific Image Relight** 中配置描述期望光照的 **prompt**（提示词）
4. 调整 **strength**（0–100）控制重打光强度
5. 选择 **mode**（`standard` 标准或 `creative` 创意）决定重打光方式
6. 点击 **Run** 或按 `Ctrl(cmd) + Enter` 运行

### 控件参考

| 控件 | 类型 | 默认值 | 说明 |
| :--- | :--- | :----- | :--- |
| prompt | 文本 | — | 描述期望光照的文本（例如 "warm natural sunlight from upper left, soft afternoon golden hour lighting"） |
| strength | 整数 | 75 | 重打光效果强度（0–100） |
| mode | 下拉 | `standard` | `standard` 平衡重打光；`creative` 更戏剧化的光照变化 |
| preserve_faces | 布尔 | false | 是否保留原始面部外观 |
| reference_strength | 布尔 | true | 启用参考图像的光照影响 |
| preserve_reference_colors | 布尔 | true | 保留参考图像的颜色 |
| auto_crop | 下拉 | `disabled` | 输入图像宽高比的自动裁剪模式 |

### 输入

| 输入 | 类型 | 说明 |
| :--- | :--- | :--- |
| image | IMAGE | 需要重打光的源图像 |
| reference_image | IMAGE (可选) | 其光照风格/方向将影响结果 |

### 输出

| 输出 | 类型 | 说明 |
| :--- | :--- | :--- |
| IMAGE | IMAGE | 重打光后的输出图像 |

### 补充说明

- **prompt** — 尽量具体：描述光源位置、色温、阴影柔度及氛围
- **reference_image** — 连接后，节点使用 `reference_strength` 和 `preserve_reference_colors` 控制参考图的影响
- **preserve_faces** — 启用后保持面部特征不变，仅调整图像其他部分的光照
