### 操作步骤

1. 在 **Load Image** 中加载需要放大的图像
2. 在 **Magnific Image Upscale Precise V2** 中选择 **upscale_factor**（例如 `2x`）
3. 选择匹配图像内容的 **type**（`photo` 照片、`art` 艺术或 `general` 通用）
4. 调整 **resemblance**（结果与原始图像的相似度）和 **resolution**（细节精细度）
5. 设置 **sharpen** 控制边缘锐度
6. 可选启用 **add_grain** 添加自然的胶片质感
7. 点击 **Run** 或按 `Ctrl(cmd) + Enter` 运行

### 控件参考

| 控件 | 类型 | 默认值 | 说明 |
| :--- | :--- | :----- | :--- |
| upscale_factor | 下拉 | `2x` | 放大倍数（例如 `2x`、`4x`） |
| type | 下拉 | `photo` | 图像类型以优化处理：`photo` 照片、`art` 艺术、`general` 通用 |
| resemblance | 整数 | 7 | 与原始图像的保真度（0–10） |
| resolution | 整数 | 7 | 细节分辨率精细度（0–10） |
| sharpen | 整数 | 30 | 锐化强度（0–100） |
| add_grain | 布尔 | true | 为输出添加自然胶片颗粒 |

### 输入

| 输入 | 类型 | 说明 |
| :--- | :--- | :--- |
| image | IMAGE | 需要放大的源图像 |

### 输出

| 输出 | 类型 | 说明 |
| :--- | :--- | :--- |
| IMAGE | IMAGE | 精确放大后的输出图像 |

### 补充说明

- **type** — 照片用 `photo`，插画/画作用 `art`，混合内容用 `general`
- **resemblance** — 值越高放大结果越忠实于原始图像；值越低允许更多 AI 诠释
- **add_grain** — 添加微妙的颗粒纹理，让放大后的图像看起来更自然，尤其适合照片
- 模板中的 **ImageCompare** 节点可并排预览前后对比
