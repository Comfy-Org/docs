**Magnific 图像风格迁移**节点将参考图像的艺术风格、色调、纹理和构图感觉迁移到源图像上。您可以使用文本提示引导结果，并微调风格影响与内容保留之间的平衡。

### 操作步骤

1. 在第一个 **Load Image** 中加载需要重新风格的源图像
2. 在第二个 **Load Image** 中加载参考图像（其风格将被应用）
3. 在 **Magnific Image Style Transfer** 中配置描述期望输出的 **prompt**（提示词）
4. 调整 **content_weight**（保留源图像内容程度）和 **style_weight**（参考风格应用强度）
5. 选择 **style_fidelity**（`faithful` 忠实或 `creative` 创意）控制对参考风格的遵循程度
6. 选择 **text_fidelity**（`balanced` 平衡或 `strict` 严格）控制提示词遵循度
7. 可选启用 **preserve_faces** 保持面部特征不变
8. 点击 **Run** 或按 `Ctrl(cmd) + Enter` 运行

### 控件参考

| 控件 | 类型 | 默认值 | 说明 |
| :--- | :--- | :----- | :--- |
| prompt | 文本 | — | 引导风格迁移输出的文本描述 |
| content_weight | 整数 | 70 | 保留源图像内容的程度（0–100） |
| style_weight | 整数 | 41 | 参考图像风格应用强度（0–100） |
| style_fidelity | 下拉 | `faithful` | `faithful` 紧跟参考风格；`creative` 更多艺术化诠释 |
| text_fidelity | 下拉 | `balanced` | `balanced` 适度遵循提示；`strict` 更严格遵循 |
| preserve_faces | 布尔 | false | 是否保持面部特征不变 |
| use_original_color | 布尔 | true | 保留源图像的原始配色 |

### 输入

| 输入 | 类型 | 说明 |
| :--- | :--- | :--- |
| image | IMAGE | 需要重新风格的源图像 |
| reference_image | IMAGE | 其风格将被转移的参考图像 |

### 输出

| 输出 | 类型 | 说明 |
| :--- | :--- | :--- |
| IMAGE | IMAGE | 风格化后的输出图像 |

### 补充说明

- **prompt** — 描述艺术方向。例如："Fantasy portrait, retro anime hand-painted style, muted warm color palette, visible brushstroke texture"
- **content_weight vs style_weight** — content_weight 越高保留越多原始结构；style_weight 越高应用越多参考风格
- **style_fidelity** — 需要精确匹配参考风格时用 `faithful`；希望更多创意表达时用 `creative`
- **use_original_color** — 启用后，风格应用于形状和纹理，但保留原始配色
