Magnific AI 是专业的图像增强平台，以其先进的放大、重打光、风格迁移和皮肤增强能力闻名。通过 ComfyUI 的 Magnific 合作伙伴节点，您可以直接在工作流中使用这些能力，无需管理 API 密钥或离开 ComfyUI。

<ReqHint/>
<UpdateReminder/>

## 节点

| 节点 | 功能 |
| :--- | :--- |
| **重打光 (Image Relight)** | 通过文本提示调整光照。保留面部特征。可接受参考图像来匹配光照方向/风格。 |
| **风格迁移 (Image Style Transfer)** | 将参考图像的视觉风格迁移到源图像上，提供精细的风格影响和提示引导控制。 |
| **创意放大 (Upscale Creative)** | AI 驱动的放大，超越简单插值，增添细节。支持最高 16× 放大，带风格/纹理引导。 |
| **精确放大 (Upscale Precise)** | 以保真度为核心的 AI 放大。保留原始特征、纹理和细节，不引入意外伪影。 |
| **皮肤增强 (Skin Enhancer)** | 通过去除瑕疵、皱纹和皮肤不完美来改善面部皮肤纹理。提供创意、忠实和灵活三种模式。 |

## 定价

Magnific 节点按次计费，归属 **Freepik** 产品。详见[合作伙伴节点定价](/tutorials/partner-nodes/pricing#freepik)。
