HappyHorse 1.1 是阿里巴巴生产级视频生成模型的最新版本，现作为合作节点在 ComfyUI 中提供。它专为现实世界的创意生产而设计：短剧集、电商广告、品牌营销内容和游戏过场动画。

一个突出的特性是**原生同步音频生成**：HappyHorse 1.1 在单次渲染中即可生成对话、音效和背景音乐，音频与视觉时间线紧密绑定。无需额外的音频步骤。

版本 1.1 针对五项核心生产关键能力：动态表现力运动、角色一致性渲染、可靠提示词遵循、稳定文本渲染和真实的电影取景。

## 亮点

- **原生音视频同步**：对话、音效和背景音乐一次生成：无需额外步骤
- **三种创作路径**：文生视频（T2V）、图生视频（I2V）、参考生视频（R2V）
- **多图像 R2V**：每次生成最多使用 **9 张参考图像**，并保持身份一致性
- **多角色一致性**：多个角色参考保持清晰，无视觉交叉污染
- **角色 × 场景分离**：将角色和场景作为独立参考输入；背景变化时角色保持一致
- **长上下文提示词**：支持超过 2500 个字符的提示词；单个提示词可描述 6–8 个连续场景，模型自主分配时间和切换机位
- **电影化语言**：完全支持正反打、推轨镜头以及镜头间的连贯转场和节奏
- **灵活输出**：支持 720p 和 1080p，3–15 秒，画面比例 16:9 / 9:16 / 1:1 / 4:3 / 3:4 / 21:9 等
- **生产级视觉效果**：修复了 v1.0 中的油光皮肤和过度锐化问题，带来自然逼真的特写和系列内容

## 入门

1. 将 ComfyUI 更新到最新版本
2. 通过节点库找到 HappyHorse 节点（搜索 "HappyHorse"），或从模板库加载现成的模板
3. 选择模式：文生视频、图生视频或参考生视频
4. 连接提示词和任何参考图像，然后运行
5. 输出为 720p 或 1080p，并已包含音频
