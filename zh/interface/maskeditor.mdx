遮罩编辑器是 ComfyUI 内置的一个实用工具，让你可以直接在图像上创建和编辑遮罩，无需借助外部图像编辑软件。它支持多种绘图工具、图层管理、撤销/重做、画布变换以及 GPU 加速的笔刷渲染。

---

## 如何打开

有三种方式可以打开遮罩编辑器：

1. **Selection Toolbox** — 选中一个 Load Image 节点，然后点击节点上方选择工具栏中的 Mask 图标按钮。
2. **Image Overlay** — 悬停在图像预览上（gallery 模式），点击左上角出现的 **Edit or mask image** 按钮（mask 图标）。
3. **右键菜单** — 右键点击 Load Image 节点，从菜单中选择 `Open in Mask Editor`。

> 遮罩编辑器在 **Graph Mode** 和 **App Mode** 下均可使用。

## 界面概览

![遮罩编辑器主界面：工具栏、画布、设置面板](/images/interface/maskeditor/maskeditor_main_editor.jpg)

遮罩编辑器以模态对话框打开，包含四个主要区域：

| # | 区域 | 位置 | 说明 |
|---|------|------|------|
| 1 | **Top Bar** | 顶部 | 标题栏，包含撤销/重做、画布变换、反转/清除、保存/取消 |
| 2 | **Tool Panel** | 左侧 | 工具选择和缩放信息 |
| — | **Pointer Zone** | 中央 | 主画布区域，用于绘制和编辑 |
| 3 | **Side Panel** | 右侧 | 工具专属设置和图层控制 |

## 工具

左侧工具面板提供了五种工具。选择工具激活后，右侧面板会显示对应的设置项。

<img src="/images/interface/maskeditor/maskeditor_tool_panel.jpg" alt="遮罩编辑器工具面板，5 种工具带编号" style={{maxWidth: '300px'}} />

| # | 工具 | 图层 | 说明 |
|---|------|------|------|
| 1 | **Mask Pen** | Mask | 主要遮罩绘制工具。左键拖拽在图像上绘制遮罩。 |
| 2 | **Paint Pen** | RGB (Paint) | 直接在图像的 RGB 层上绘制。适用于局部重绘或原图修补。 |
| 3 | **Eraser** | Mask / RGB | 擦除已有遮罩或绘制层的部分内容。 |
| 4 | **Paint Bucket** | Mask | 基于颜色相似性的洪水填充工具。点击像素填充连通区域创建遮罩；点击已有遮罩的区域则擦除遮罩。Tolerance（容差）控制填充扩散范围。 |
| 5 | **Color Select** | Mask | 选择性地遮罩与目标颜色匹配的所有像素，支持高级匹配算法（Simple HSL / LAB）。 |

## TopBar 操作

![遮罩编辑器 TopBar 操作标注 1–10](/images/interface/maskeditor/maskeditor_topbar_actions.jpg)

顶部栏提供了画布级别的操作：

| # | 操作 | 说明 |
|---|------|------|
| 1 | **Undo** (Ctrl+Z) | 撤销上一步操作 |
| 2 | **Redo** (Ctrl+Shift+Z / Ctrl+Y) | 重做被撤销的操作 |
| 3 | **Rotate Left** | 将所有图层逆时针旋转 90° |
| 4 | **Rotate Right** | 将所有图层顺时针旋转 90° |
| 5 | **Mirror Horizontal** | 水平翻转所有图层 |
| 6 | **Mirror Vertical** | 垂直翻转所有图层 |
| 7 | **Invert** | 反转遮罩（黑白互换） |
| 8 | **Clear** | 清除整个遮罩 |
| 9 | **Save** | 保存遮罩并关闭编辑器 |
| 10 | **Cancel** | 关闭编辑器，不保存 |

> **注意**：旋转和镜像操作会同时影响所有三个图层（遮罩、绘制层和原图）。GPU 加速纹理会在变换后重新创建。

## 图层

右侧面板底部包含 **Layers**（图层）区域，可以独立控制每个图层的显示/隐藏：

- **Mask Layer**（遮罩层）：遮罩叠加层。可调整**不透明度**和**混合模式**：
  - `Black` — 遮罩以黑色叠加显示
  - `White` — 遮罩以白色叠加显示
  - `Negative` — 反转遮罩渲染
- **Paint Layer**（绘制层）：RGB 绘制层（使用 Paint Pen 时可见）
- **Base Image Layer**（原图层）：原始源图像

点击图层上的 **Activate Layer** 可切换绘制焦点到该图层。

## 画笔设置

当基于画笔的工具（Mask Pen、Paint Pen 或 Eraser）激活时，侧边面板显示画笔设置：

| 设置项 | 范围 | 说明 |
|--------|------|------|
| **Shape** | Arc / Rect | 圆形画笔或方形画笔 |
| **Color** | 任意十六进制颜色 | 绘制颜色（Paint Pen 用）或遮罩颜色 |
| **Thickness** | 1–250 | 画笔直径（像素） |
| **Opacity** | 0–1 | 画笔笔触透明度 |
| **Hardness** | 0–1 | 边缘柔和度。1 = 硬边缘，0 = 非常柔和 |
| **Step Size** | 1–100 | 画笔点之间的间距。值越低笔触越平滑 |

使用 **Reset to Default** 恢复默认画笔参数。

## 画布导航与快捷键

| 快捷键 | 操作 |
|--------|------|
| **Space + 拖拽** | 平移画布 |
| **Ctrl + 滚轮** | 放大/缩小 |
| **点击缩放百分比** | 将缩放重置为 100% |
| **Ctrl + Z** | 撤销 |
| **Ctrl + Shift + Z** 或 **Ctrl + Y** | 重做 |
| **Alt + 右键拖拽** | 调整画笔：左右改变**大小**，上下改变**硬度** |

---

## 设置

在 ComfyUI 的设置面板中，可以配置以下遮罩编辑器选项：

| 设置项 | 分类 | 默认值 | 说明 |
|--------|------|--------|------|
| **Brush adjustment speed multiplier** | Mask Editor → Brush Adjustment → Sensitivity | 1.0 (0.1–2.0) | 控制 Alt+右键拖拽调整画笔时的速度，值越大变化越快 |
| **Lock brush adjustment to dominant axis** | Mask Editor → Brush Adjustment → Use Dominant Axis | 开启 | 开启后，调整仅基于移动的主轴方向影响大小或硬度 |
