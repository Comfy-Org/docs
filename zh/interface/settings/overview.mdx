这个部分是关于 ComfyUI 前端设置菜单中详细的设置说明， 对于所有的用户设置将会自动保存到 `ComfyUI/user/default/comfy.settings.json` 文件夹

你可以使用 `ctrl + ,` 快捷键来打开设置面板， 然后点击对应的设置选项进行设置。

由于自定义节点也可以在菜单中注册对应的设置类目，在我们的官方文档说明中目前仅包含原生的设置内容，另外有部分选项设置 **仅针对 ComfyUI 桌面版** 有效，我们也在对应页面中做了注释说明。

## ComfyUI 设置菜单结构

设置面板以侧边栏形式组织为三个主要分组，具体结构取决于是否启用了团队工作区。

### 当团队工作区启用时：

1. **Workspace（工作区）**
   - 工作区设置（如有）
   - 积分

2. **General（通用）**
   - 用户 / 个人资料
   - **Comfy** — ComfyUI 核心设置（界面、队列、节点、工作流等）
   - Secrets（密钥）
   - **Lite Graph（画面）** — 画布渲染、节点、链接、分组等
   - **Appearance（外观）** — 主题、侧边栏位置、背景等
   - **3D** — 3D 节点初始化设置
   - **Mask Editor（蒙版编辑器）** — 蒙版编辑器偏好
   - 快捷键
   - 扩展
   - 关于
   - 服务器配置（仅桌面版）

3. **Other（其他）** — 第三方自定义节点注册的设置（仅当存在时显示）

### 当团队工作区禁用时（旧版）：

1. **Account（账户）**
   - 用户 / 个人资料
   - 订阅
   - Secrets（密钥）
   - 积分

2. **Application Settings（应用设置）** — 所有核心设置类别：
   - **Comfy**
   - **Lite Graph（画面）**
   - **Appearance（外观）**
   - **3D**
   - **Mask Editor（蒙版编辑器）**
   - **Other（其他）**（未分类设置）

3. **Special Settings（特殊设置）**
   - 快捷键
   - 扩展
   - 关于
   - 服务器配置（仅桌面版）

## 仅桌面版设置

以下设置仅在 ComfyUI Desktop 中可用：

- **服务器配置** — 访问和修改服务器配置设置
- **Comfy Desktop 通用设置** — 更新偏好、窗口样式、镜像源

## 仅云端面板

部分面板仅在登录 ComfyUI 账户后出现：

- **积分** — 购买积分及查看积分余额历史
- **Secrets（密钥）** — 管理 API 密钥和机密信息
- **Workspace（工作区）** — 团队工作区配置
- **订阅** — 管理订阅计划
