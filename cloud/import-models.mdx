## Overview

Comfy Cloud allows you to import models directly from Civitai. This feature is available for users with a **Creator** subscription or higher.

## Requirements

- **Subscription**: Creator tier or above
- **Supported source**: Civitai only

## How to import models

<Steps>
  <Step title="Open the Model Library">
    Click the **Models** button in the left sidebar to open the Model Library.
  </Step>
  
  <Step title="Click Import">
    In the Model Library popup, click the **Import** button.
  </Step>
  
  <Step title="Enter the Civitai download link">
    Paste the Civitai model download link to import the model.
  </Step>
</Steps>

## Limitations

- Currently only Civitai models are supported
- Requires Creator subscription or higher
