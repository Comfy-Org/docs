APP mode simplifies workflow interaction by allowing you to define custom input/output interfaces, making it easier to share and use workflows without editing nodes.

<Note>
APP mode is officially supported from ComfyUI front-end version 1.41.13. Earlier versions are in early test phase.
</Note>

## Enter APP mode

When the workflow is not in APP mode, access the APP builder to transform an existing workflow into an APP-enabled one.

## Build an APP mode workflow

### Select inputs

In the interface's top guided steps, click on the "Inputs" step. Choose necessary input parameters and options, like selecting an image, prompt input, or model selection. Clicking the relevant node will select it, and selected parameters will appear in the right sidebar.

### Select outputs

Click the "Outputs" step in the top guided steps. Pick outputs to display from the output nodes, such as the Preview node or Save image node.

### Preview

Click the "Preview" step to check the final APP appearance. Selected inputs show on the right, and output content previews in the canvas. Adjust earlier steps if issues arise.

### Set a default view

Click the "Set a default view" step. Choose APP mode or NODE graph mode as the default display. Selecting APP mode means the workflow opens in this mode by default. Click "Apply" to confirm.

## Save and share the workflow

### Save

Use the shortcut `Ctrl + S` to save the workflow for future use.

### Share

In the APP Mode interface, click the left menu and select "Share". Click "Create a link" to generate a sharing link, then "Copy" to copy it.

<Note>
Share links are only supported on Comfy Cloud.
</Note>
