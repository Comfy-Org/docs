The Mask Editor is a built-in tool in ComfyUI that lets you create and edit masks directly on your images — no external image editor needed. It supports multiple drawing tools, layer management, undo/redo, canvas transformations, and GPU-accelerated brush rendering.

---

## How to Open

There are three ways to open the Mask Editor:

1. **Right-click a Load Image node** → select `Open in Mask Editor` from the context menu.
2. **Hover over the image preview** (gallery mode) → click the **Edit or mask image** button (✏️ icon) that appears in the top-left corner.
3. **Select a Load Image node** → click the Mask icon button in the selection toolbar above the node.

> The Mask Editor is available in both **Graph Mode** and **App Mode**.

---

## Interface Overview

The Mask Editor opens as a modal dialog with four main areas:

| Area | Location | Description |
|------|----------|-------------|
| **TopBar** | Top | Header with undo/redo, canvas transforms, invert/clear, save/cancel |
| **Tool Panel** | Left | Tool selection and zoom info |
| **Pointer Zone** | Center | The main canvas area for drawing and editing |
| **Side Panel** | Right | Tool-specific settings and layer controls |

## Tools

The tool panel on the left side offers five tools. Select a tool to activate it, and the right panel will show its settings.

### Mask Pen <icon name="pen" />
The primary tool for drawing mask regions. Left-click and drag to paint mask on the image.

- **Active Layer**: Mask layer (automatically switched)
- **Settings**: Brush Shape, Color, Thickness, Opacity, Hardness, Step Size

### Paint Pen <icon name="paint-brush" />
Paint directly onto the image's RGB layer. Useful for inpainting or touching up the base image.

- **Active Layer**: RGB (paint) layer (automatically switched)
- **Settings**: Same brush settings as Mask Pen

### Eraser <icon name="eraser" />
Erase parts of an existing mask or paint layer.

- **Settings**: Same brush settings as Mask Pen

### Paint Bucket <icon name="fill" />
A flood-fill tool that fills connected regions of the mask based on color similarity. Click on any pixel to fill the surrounding area with a mask.

- **Settings**:
  - **Tolerance** (0–255): How similar neighboring pixels must be to be included in the fill. Higher values fill a wider area.
  - **Fill Opacity** (0–100): Opacity of the fill result.

### Color Select <icon name="eye-dropper" />
Selectively mask all pixels in the image that match a target color. This is useful for creating masks based on color ranges.

- **Settings**:
  - **Tolerance** (0–255): Color matching tolerance. Higher values include more shades.
  - **Selection Opacity** (0–100): How opaque the preview selection appears.
  - **Live Preview**: Toggle real-time preview of the selection as you adjust parameters.
  - **Apply to Whole Image**: When enabled, the selection applies to the entire image rather than just the clicked region.
  - **Method**: Choose the color comparison algorithm — `Simple` (RGB distance), `HSL`, or `LAB` (perceptually uniform).
  - **Stop at mask**: When enabled, the selection will not cross existing mask boundaries.
  - **Mask Tolerance** (0–255): How close to an existing mask edge the selection stops (only when "Stop at mask" is on).

## TopBar Actions

The top bar provides canvas-wide operations:

| Action | Description |
|--------|-------------|
| **Undo** (Ctrl+Z) | Revert the last action |
| **Redo** (Ctrl+Shift+Z / Ctrl+Y) | Reapply a reverted action |
| **Rotate Left** | Rotate all layers 90° counter-clockwise |
| **Rotate Right** | Rotate all layers 90° clockwise |
| **Mirror Horizontal** | Flip all layers horizontally |
| **Mirror Vertical** | Flip all layers vertically |
| **Invert** | Invert the mask (black ↔ white) |
| **Clear** | Clear the entire mask |
| **Save** | Save the mask and close the editor |
| **Cancel** | Close without saving |

> **Note**: Rotate and mirror operations affect all three layers (mask, paint, and base image) simultaneously. GPU-accelerated textures are recreated after transforms.

## Layers

The right panel includes a **Layers** section where you can toggle visibility of each layer independently:

- **Mask Layer**: The mask overlay. You can adjust its **opacity** and **blending mode**:
  - `Black` — Mask renders as black overlay
  - `White` — Mask renders as white overlay
  - `Negative` — Inverted mask rendering
- **Paint Layer**: The RGB paint layer (visible when using Paint Pen)
- **Base Image Layer**: The original source image

Click **Activate Layer** on a layer to switch drawing focus to it.

## Brush Settings

When a brush-based tool (Mask Pen, Paint Pen, or Eraser) is active, the side panel shows brush settings:

| Setting | Range | Description |
|---------|-------|-------------|
| **Shape** | Arc / Rect | Round brush or square brush |
| **Color** | Any hex color | The paint color (for Paint Pen) or mask color |
| **Thickness** | 1–250 | Brush diameter in pixels |
| **Opacity** | 0–1 | Brush stroke transparency |
| **Hardness** | 0–1 | Edge softness. 1 = hard edge, 0 = very soft |
| **Step Size** | 1–100 | Spacing between brush dabs. Lower values give smoother strokes |

Use **Reset to Default** to restore the default brush parameters.

## Canvas Navigation & Shortcuts

| Shortcut | Action |
|----------|--------|
| **Space + Drag** | Pan the canvas |
| **Ctrl + Scroll** | Zoom in/out |
| **Click zoom %** | Reset zoom to 100% |
| **Ctrl + Z** | Undo |
| **Ctrl + Shift + Z** or **Ctrl + Y** | Redo |
| **Alt + Right-click + Drag** | Adjust brush: left/right changes **size**, up/down changes **hardness** |

---

## Settings

In the ComfyUI Settings panel, you can configure the following Mask Editor options:

| Setting | Category | Default | Description |
|---------|----------|---------|-------------|
| **Brush adjustment speed multiplier** | Mask Editor → Brush Adjustment → Sensitivity | 1.0 (0.1–2.0) | How quickly the brush size and hardness change when using Alt+Right-click drag |
| **Lock brush adjustment to dominant axis** | Mask Editor → Brush Adjustment → Use Dominant Axis | On | When enabled, the adjustment affects only size OR hardness based on your dominant movement direction |
