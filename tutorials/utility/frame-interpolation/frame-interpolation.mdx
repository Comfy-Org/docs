Frame interpolation generates intermediate frames between existing frames, resulting in smoother motion and higher frame rate video output. ComfyUI includes native support for both **FILM** (Google) and **RIFE** (Real-Time Intermediate Flow Estimation) models.

<Img src="https://raw.githubusercontent.com/Comfy-Org/workflow_templates/main/templates/utility_video_frame_interpolation-1.webp" alt="Frame interpolation workflow comparison before and after" />

## Workflow overview

This workflow consists of three main steps:

1. **Load a video** — Upload any video file using the standard LoadVideo node
2. **Select interpolation model** — Choose between FILM and RIFE via the model subgraph configuration
3. **Output the interpolated video** — The result is saved with smooth, high-frame-rate motion

<Img src="https://raw.githubusercontent.com/Comfy-Org/workflow_templates/main/templates/utility_video_frame_interpolation-2.webp" alt="Frame interpolation workflow nodes" />

### Subgraph parameters

The workflow uses a Subgraph to expose the following controls:

- **model_name** — Select between FILM or RIFE frame interpolation models
- **multiplier** — Frame rate multiplier (e.g., 2x doubles the frame rate)
- **enable_fps_multiplier** — Toggle to apply multiplier based on original FPS

Download the model files and place them in the `ComfyUI/models/frame_interpolation/` folder:

## How to use

1. **Open the workflow** — Click the Run in Comfy Cloud button above or download the JSON and drag it into ComfyUI
2. **Load a video** — Use LoadVideo to select your input video
3. **Choose a model** — In the subgraph, select FILM or RIFE from the dropdown
4. **Set the multiplier** — 2x is recommended for most use cases; higher values produce smoother results but take longer
5. **Generate** — Run the workflow and download the output video
