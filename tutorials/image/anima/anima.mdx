**Anima** is an open text-to-image model by [CircleStone Labs](https://huggingface.co/circlestone-labs/Anima) in partnership with Comfy Org. With **2 billion** parameters, it is designed to generate high-quality **anime and non-photorealistic** images — ideal for characters, scenes, and concept art.

**Model highlights**:
- **Anime and artistic illustration focus** — tuned for non-photorealistic art styles
- **ComfyUI native support** — loaded with standard UNETLoader, CLIPLoader, and VAELoader nodes
- **Qwen-3 0.6B text encoder** — provides strong language understanding and prompt adherence
- **No input image needed** — pure text-to-image generation
- **Compact footprint** — 2B diffusion model, runs comfortably on consumer GPUs

**Related links**:
- [Hugging Face (Anima)](https://huggingface.co/circlestone-labs/Anima)

## Node guide

### Loading the model

The Anima workflow uses three standard ComfyUI model loader nodes — no custom nodes required.

- **UNETLoader** — loads the Anima diffusion model (`anima-base-v1.0.safetensors` or `anima-preview3-base.safetensors`)
- **CLIPLoader** — loads the Qwen-3 0.6B text encoder (`qwen_3_06b_base.safetensors`)
- **VAELoader** — loads the VAE (`qwen_image_vae.safetensors`)

### Workflow structure

The Anima workflow is packaged as a **subgraph** node inside ComfyUI's template system. The subgraph contains the following internal nodes:

- **UNETLoader** — loads the Anima diffusion model
- **CLIPLoader** + **CLIPTextEncode** (×2) — encodes both positive and negative prompts
- **EmptyLatentImage** — creates initial latent with configurable width and height
- **KSampler** — runs the denoising process
- **VAEDecode** — decodes the latent into an image

The subgraph exposes the following controls:

| Input | Type | Default | Description |
|-------|------|---------|-------------|
| prompt | STRING | — | Text prompt describing the anime image to generate |
| width | INT | 1024 | Image width |
| height | INT | 1024 | Image height |
| steps | INT | 30 | Number of denoising steps |
| cfg | FLOAT | 4.0 | Classifier-free guidance scale |
| seed | INT | random | Random seed for reproducibility |
| unet_name | COMBO | — | Anima diffusion model file |
| clip_name | COMBO | — | Text encoder model file |
| vae_name | COMBO | — | VAE model file |

## Limitations

- **No realism** — the model is intentionally designed for anime and illustration styles; realistic renderings are out of scope
- **Weak text rendering** — high-resolution text and complex typography may not render clearly
- **Plain styling** — the base model produces simpler compositions; advanced artistic styles may require refinement or fine-tuning
