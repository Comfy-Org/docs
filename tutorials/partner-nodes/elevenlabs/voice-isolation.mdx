The **ElevenLabsAudioIsolation** node separates foreground speech from background noise, producing a clean voice track from any audio input. It is useful for cleaning up recordings, podcasts, voiceovers, and field audio.

<ReqHint/>
<UpdateReminder/>

### Voice Isolation parameters

The **ElevenLabsAudioIsolation** node has no configurable parameters. Simply connect an audio source and it will automatically extract the voice track.

The workflow is a simple three-node chain:

1. Load the noisy audio file through **LoadAudio** (e.g. `noisy_background.mp3`)
2. Connect it to **ElevenLabsAudioIsolation** — the node automatically separates voice from background
3. The clean audio output is saved via **SaveAudioMP3**

If you use the template as a reference, a typical run looks like this:

1. Load your audio file in **LoadAudio**
2. Connect to **ElevenLabsAudioIsolation**
3. Click **Run** or press `Ctrl(cmd) + Enter`
4. The isolated voice track is saved to the specified output path

### Additional notes

- No manual tuning needed — the node handles noise profiling automatically
- Works on a wide range of audio inputs: interviews, podcasts, field recordings, phone calls
