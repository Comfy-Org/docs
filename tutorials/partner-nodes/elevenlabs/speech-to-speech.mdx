The **ElevenLabsSpeechToSpeech** node transforms the voice or speaking style of an input audio recording while preserving the original speech content. It can change accents, voice characteristics, and emotional delivery.

<ReqHint/>
<UpdateReminder/>

## Supported models

| Model | Description |
| :---- | :---------- |
| `eleven_multilingual_sts_v2` | Multi-language speech-to-speech conversion |

### Speech-to-Speech parameters

| Parameter | Description |
| :-------- | :---------- |
| **similarity** | Voice similarity (0.0–1.0), default 0.5 |
| **stability** | Voice stability (0.0–1.0), default auto |
| **style_exaggeration** | Style exaggeration (auto / 0.0–1.0) |
| **speed** | Playback speed multiplier |
| **optimize_streaming_latency** | Latency optimization level |
| **output_format** | Audio output format (e.g. mp3_44100_192) |
| **seed** | Random seed for reproducible output |
| **seed_mode** | Seed mode: randomize / fixed |
| **enable_logging** | Enable/disable logging |

The workflow has three sections:

1. **Source Audio** — load your input audio file via **LoadAudio**
2. **Use Preset Voice** — select a target voice from **ElevenLabsVoiceSelector**
3. **Create a Voice Clone** (optional) — load an audio sample and enable **ElevenLabsInstantVoiceClone** for custom voice cloning

If you use the template as a reference, a typical run looks like this:

1. Load the source audio through **LoadAudio** (e.g. `api_elevenlabs_speech_to_speech.mp3`)
2. Select a target voice from **ElevenLabsVoiceSelector** (e.g. "Jessica (female, american)")
3. (Optional) Enable **ElevenLabsInstantVoiceClone** for a cloned voice
4. Connect the source audio and target voice to **ElevenLabsSpeechToSpeech**
5. Adjust voice parameters (similarity, stability, speed) as needed
6. Click **Run** or press `Ctrl(cmd) + Enter`
7. The transformed audio is saved via **SaveAudioMP3**

### Additional notes

- The **ElevenLabsInstantVoiceClone** and associated **LoadAudio** nodes are bypassed by default. Use `Ctrl-B` to enable them
- Works well for accent conversion, character voice transformation, and voice acting
- For pricing, see [ElevenLabs on the pricing page](/tutorials/partner-nodes/pricing#elevenlabs)
