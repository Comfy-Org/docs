The **ElevenLabsTextToSoundEffects** node generates custom sound effects from text descriptions. Create audio assets for games, videos, and interactive experiences directly within your ComfyUI workflow.

<ReqHint/>
<UpdateReminder/>

## Supported models

| Model | Description |
| :---- | :---------- |
| `eleven_sfx_v2` | Sound effects generation model |

### Text-to-Sound Effects parameters

| Parameter | Description |
| :-------- | :---------- |
| **text** | Description of the sound effect to generate |
| **model** | Sound generation model selection |
| **duration_seconds** | Duration of the generated sound effect |
| **prompt_influence** | How closely to follow the prompt (0.0–1.0) |
| **output_format** | Audio output format (e.g. mp3_44100_192) |

The workflow is simple and focused:

1. Enter your sound effect description in **ElevenLabsTextToSoundEffects** (e.g. "Traffic noise with honking horns, busy highway")
2. Select the model (`eleven_sfx_v2`)
3. Set the **duration_seconds** for the sound effect
4. Adjust **prompt_influence** to balance creativity vs. prompt adherence
5. Click **Run** or press `Ctrl(cmd) + Enter`
6. The generated audio is saved via **SaveAudioMP3**

### Example prompts

| Prompt | Best use |
| :----- | :------- |
| "Traffic noise with honking horns, busy highway" | Urban environment ambience |
| "Rain on a window with distant thunder" | Atmospheric background |
| "Footsteps on wooden floor, slow and deliberate" | Game/sound design |
| "Door creaking open, suspenseful atmosphere" | Film/video production |

### Additional notes

- Longer durations consume more credits. See [ElevenLabs on the pricing page](/tutorials/partner-nodes/pricing#elevenlabs) for per-minute rates
- **prompt_influence** of 0.3 balances fidelity and variety; lower values allow more creative interpretation
- Sound effects are priced per minute of generated audio
