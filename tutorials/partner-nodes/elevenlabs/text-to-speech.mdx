The **ElevenLabsTextToSpeech** node converts text into natural-sounding speech using ElevenLabs voice models. It supports multilingual voices and configurable output formats.

<ReqHint/>
<UpdateReminder/>

## Supported models

| Model | Languages |
| :---- | :-------- |
| `eleven_multilingual_v2` | Multi-language support across 29 languages |
| `eleven_v3` | English-optimized, high quality |

### Text-to-Speech parameters

| Parameter | Description |
| :-------- | :---------- |
| **text** | The text content to convert to speech |
| **similarity** | Voice similarity (0.0–1.0), default 0.5 |
| **stability** | Voice stability (0.0–1.0), default auto |
| **style_exaggeration** | Style exaggeration (auto / 0.0–1.0) |
| **speed** | Playback speed multiplier |
| **optimize_streaming_latency** | Latency optimization level |
| **output_format** | Audio output format (e.g. mp3_44100_192) |
| **seed** | Random seed for reproducible output |
| **seed_mode** | Seed mode: randomize / fixed |

The workflow has two options:

1. **Use Preset Voice** — select a voice from the **ElevenLabsVoiceSelector** node (e.g. "Roger (male, american)")
2. **Create a Voice Clone** — load an audio file via **LoadAudio**, connect to **ElevenLabsInstantVoiceClone**, then use the cloned voice

If you use the template as a reference, a typical run looks like this:

1. Choose a preset voice from **ElevenLabsVoiceSelector**, or provide an audio sample and enable **ElevenLabsInstantVoiceClone**
2. Enter your text in the **ElevenLabsTextToSpeech** node
3. Select the model (`eleven_multilingual_v2` or `eleven_v3`)
4. Adjust voice parameters (similarity, stability, speed) as needed
5. Click **Run** or press `Ctrl(cmd) + Enter`
6. The generated audio is saved via **SaveAudioMP3** node

### Additional notes

- The **ElevenLabsInstantVoiceClone** node is bypassed by default. Use `Ctrl-B` to enable it when using voice cloning
- Output format defaults to `mp3_44100_192` for high-quality MP3 audio
- Set **seed** to `0` to omit; most models treat it as a hint only
- For pricing, see [ElevenLabs on the pricing page](/tutorials/partner-nodes/pricing#elevenlabs)
