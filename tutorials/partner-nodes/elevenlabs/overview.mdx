[ElevenLabs](https://elevenlabs.io) is a leading AI voice and audio platform. ComfyUI connects through the **ElevenLabs** partner node set, offering text-to-speech, speech-to-speech, dialogue generation, sound effects, voice isolation, and speech-to-text capabilities in a single node-based workflow.

<ReqHint/>
<UpdateReminder/>

## What you can do

- **Text-to-Speech** — convert text into natural-sounding speech using preset voices or custom voice clones
- **Speech-to-Speech** — transform the voice or style of an existing audio recording while preserving content
- **Text-to-Dialogue** — generate multi-speaker dialogue with distinct voices for each participant
- **Text-to-Sound Effects** — generate custom sound effects from text descriptions
- **Voice Isolation** — separate foreground voice from background noise in audio files
- **Speech-to-Text** — transcribe audio into text with word-level timestamps and language detection

## Key nodes

| Node | Description |
| :--- | :---------- |
| **ElevenLabsTextToSpeech** | Converts text to speech using ElevenLabs voice models |
| **ElevenLabsSpeechToSpeech** | Transforms voice/style of input audio |
| **ElevenLabsTextToDialogue** | Generates dialogue between multiple speakers |
| **ElevenLabsTextToSoundEffects** | Creates sound effects from text prompts |
| **ElevenLabsAudioIsolation** | Removes background noise, isolating the voice track |
| **ElevenLabsSpeechToText** | Transcribes audio to text with word-level details |
| **ElevenLabsVoiceSelector** | Selects a preset ElevenLabs voice |
| **ElevenLabsInstantVoiceClone** | Creates a voice clone from uploaded audio |

Pricing is listed under [Partner Node Pricing](/tutorials/partner-nodes/pricing#elevenlabs).
