The Seedance 2.0 Reference-to-Video node (`ByteDance2ReferenceNode`) generates video using reference images, video clips, and audio to guide the subject's appearance, motion, and rhythm. You can provide up to **9 reference images**, **3 reference videos**, and **3 reference audio clips** in a single workflow.

<UpdateReminder/>

## Workflow overview

The Reference-to-Video workflow uses multiple Load nodes feeding into the Seedance 2.0 Reference node:

1. **LoadImage (x4)** — Load reference images
2. **ByteDance2ReferenceNode** — Takes images, optional video/audio inputs, and a text prompt; outputs a VIDEO
3. **SaveVideo** — Saves the generated video to disk

## Node parameters

| Parameter | Description |
| :-------- | :---------- |
| `model` | Model name. Default: `Seedance 2.0` |
| `prompt` | Text description. Use structured prompts with shot timing, camera directions, and outfit descriptions |
| `resolution` | Output resolution. Options: `480p`, `720p`, `1080p` |
| `aspect_ratio` | Aspect ratio. Options: `16:9`, `9:16`, `1:1`, `4:5` |
| `duration` | Video duration in seconds |
| `generate_audio` | Whether to generate audio synchronized with the video |
| `seed` | Random seed for reproducibility |
| `cfg` | Classifier-free guidance scale |

## Inputs

| Input | Type | Max | Description |
| :---- | :--- | :-: | :---------- |
| `image_1` to `image_5` | IMAGE | 5 | Direct image inputs for reference appearance |
| `video_1` | VIDEO | 1 | Reference video input for motion guidance |
| `audio_1` | AUDIO | 1 | Reference audio input for rhythm and sync |
| `asset_1` | STRING | 1 | Asset ID for a verified real-human portrait |

## Running the workflow

1. **Load reference images** — Use LoadImage nodes to load images that define the subject's appearance. For a multi-outfit showcase, load one image per outfit
2. **Write a structured prompt** — Describe each segment with camera angles, transitions, and subject actions. Use timing breakdowns like `[00:00-00:02]` for precise control
3. **Set parameters** — Choose resolution, aspect ratio, duration, and audio generation
4. **Run the workflow** — Click the Queue button or use `Ctrl(Cmd) + Enter`
5. **Review the output** — The generated video appears in the SaveVideo node

## Tips

- **Mix multiple references** — Use images for appearance, video for motion patterns, and audio for rhythm
- **Structured prompts** — Use shot-by-shot formatting with time stamps for complex multi-scene videos
- **Outfit consistency** — For product showcases, the prompt should explicitly describe each reference outfit and its visibility in each time segment
- **Asset ID for real humans** — For verified real-person portraits, use the `asset_1` string input instead of direct image inputs
