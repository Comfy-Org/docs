The Seedance 2.0 Text-to-Video node (`ByteDance2TextToVideoNode`) generates video directly from a text description. Seedance 2.0 handles scene composition, motion, camera movement, and pacing based on your prompt.

<ReqHint/>
<UpdateReminder/>

## Workflow overview

The Text-to-Video workflow is the simplest Seedance 2.0 workflow. It has just two nodes:

1. **ByteDance2TextToVideoNode** — Accepts a text prompt and outputs a VIDEO
2. **SaveVideo** — Saves the generated video to disk

## Node parameters

| Parameter | Description |
| :-------- | :---------- |
| `model` | Model name. Default: `Seedance 2.0` |
| `prompt` | Text description of the video. Supports structured multi-shot prompts with camera directions, scene details, and timing |
| `resolution` | Output resolution. Options: `480p`, `720p`, `1080p` |
| `aspect_ratio` | Aspect ratio. Options: `16:9`, `9:16`, `1:1`, `4:5` |
| `duration` | Video duration in seconds |
| `generate_audio` | Whether to generate audio synchronized with the video |
| `seed` | Random seed for reproducibility. Options: `randomize`, `fixed`, or a specific value |
| `cfg` | Classifier-free guidance scale |

## Running the workflow

1. **Write your prompt** — Describe the scene, subject, action, camera movement, and mood in detail. For best results, use a structured format with timing breakdowns and camera directions
2. **Set parameters** — Choose resolution, aspect ratio, duration, and whether to generate audio
3. **Run the workflow** — Click the Queue button or use `Ctrl(Cmd) + Enter`
4. **Review the output** — The generated video appears in the SaveVideo node and is saved to `ComfyUI/output/`

## Tips

- **Detailed prompts** yield better results. Include subject appearance, environment, lighting, and camera movement
- **Use shot breakdowns** for multi-scene videos. Format each time segment with camera, action, and effect
- **Audio sync** is native — enable `generate_audio` for video with synchronized sound
- **Seed** randomness only affects whether to rerun; use `fixed` for reproducible results
