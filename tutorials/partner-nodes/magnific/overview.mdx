Magnific AI is a professional image enhancement platform known for its state-of-the-art upscaling, relighting, style transfer, and skin enhancement capabilities. Through ComfyUI's Magnific partner nodes, you can integrate these capabilities directly into your workflows without managing API keys or leaving ComfyUI.

<ReqHint/>
<UpdateReminder/>

## Nodes

| Node | What it does |
| :--- | :----------- |
| **Image Relight** | Adjust lighting with a text prompt. Preserves faces. Accepts an optional reference image to match lighting direction/style. |
| **Image Style Transfer** | Transfer the visual style of a reference image onto your source image, with granular controls for style influence and prompt guidance. |
| **Image Upscale Creative** | AI-powered upscaling that adds detail beyond simple interpolation. Supports up to 16× magnification with style/texture guidance. |
| **Image Upscale Precise** | AI upscaling focused on fidelity. Preserves original character, texture, and detail without adding unintended artifacts. |
| **Skin Enhancer** | Refine facial skin textures by removing blemishes, lines, and imperfections. Offers creative, faithful, and flexible modes. |

## Pricing

Magnific nodes are billed per run under the **Freepik** product. See [Partner Node Pricing](/tutorials/partner-nodes/pricing#freepik) for details.
