### Steps to run the workflow

1. In **Load Image** (first), load the input image you want to re-light
2. (Optional) In **Load Image** (second), load a reference image with the desired lighting look
3. In **Magnific Image Relight**, set the **prompt** describing the target lighting (e.g. "warm natural sunlight from upper left, soft afternoon golden hour lighting")
4. Adjust optional parameters: **prompt strength** (default 75), **relight mode** (standard or creative), **preserve background**, **generate shadows**, and **generate ambient light**
5. Click **Run** or press `Ctrl(cmd) + Enter`
6. View the re-lit result in **Save Image**

### Parameters

| Parameter | Default | Description |
|-----------|---------|-------------|
| **prompt** | — | Text description of the desired lighting environment |
| **prompt_strength** | 75 | How strongly the prompt influences the relighting (0-100) |
| **relight_mode** | standard | `standard` for balanced output, `creative` for more dramatic lighting changes |
| **preserve_background** | false | Keep the original background unchanged |
| **generate_shadows** | true | Enable shadow generation for more realistic lighting |
| **generate_ambient_light** | true | Enable ambient light generation |
| **reference_image** | disabled | How to use the reference image: `disabled`, `composition`, `lighting` |
