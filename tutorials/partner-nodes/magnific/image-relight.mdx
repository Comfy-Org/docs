The **Magnific Image Relight** node adjusts the lighting of an input image based on a text prompt. It preserves facial features while modifying the overall lighting direction, mood, and atmosphere.

### Steps to run

1. In **Load Image**, load the image you want to relight
2. (Optional) Connect a **reference image** to the `reference_image` input to match its lighting direction/style
3. In **Magnific Image Relight**, configure the **prompt** describing the desired lighting
4. Adjust **strength** (0–100) to control how strongly the relighting applies
5. Choose the **mode** (`standard` or `creative`) for the relighting approach
6. Click **Run** or press `Ctrl(cmd) + Enter`

### Widgets reference

| Widget | Type | Default | Description |
| :----- | :--- | :------ | :---------- |
| prompt | string | — | Text description of desired lighting (e.g., "warm natural sunlight from upper left, soft afternoon golden hour lighting") |
| strength | integer | 75 | How strongly the relighting effect is applied (0–100) |
| mode | dropdown | `standard` | `standard` — balanced relighting; `creative` — more dramatic lighting changes |
| preserve_faces | boolean | false | Whether to preserve original face appearance |
| reference_strength | boolean | true | Enable to use the reference image's lighting influence |
| preserve_reference_colors | boolean | true | Maintain colors from the reference image |
| auto_crop | dropdown | `disabled` | Automatic cropping mode for source image aspect ratio adjustment |

### Inputs

| Input | Type | Description |
| :---- | :--- | :---------- |
| image | IMAGE | The source image to relight |
| reference_image | IMAGE (optional) | An image whose lighting style/direction will influence the result |

### Outputs

| Output | Type | Description |
| :----- | :--- | :---------- |
| IMAGE | IMAGE | The relighted output image |

### Additional notes

- **prompt** — Be specific: describe light source position, color temperature, shadow softness, and mood
- **reference_image** — When connected, the node uses `reference_strength` and `preserve_reference_colors` to control the reference's influence
- **preserve_faces** — Enable to keep facial features unchanged while adjusting lighting on the rest of the image
