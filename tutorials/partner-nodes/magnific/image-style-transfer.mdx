### Steps to run the workflow

1. In **Load Image** (first), load the input image to transform
2. In **Load Image** (second), load the reference image whose style you want to transfer
3. In **Magnific Image Style Transfer**, edit the **prompt** describing the desired output style (e.g. "Fantasy portrait, glittering pink fantasy background, retro anime hand-painted style")
4. Adjust **strength** (default 70), **creativity** (default 41), **reference influence** (faithful or creative), **style fidelity** (balanced or higher), and enable **auto face detection** as needed
5. Click **Run** or press `Ctrl(cmd) + Enter`
6. View the stylized result in **Save Image**

### Parameters

| Parameter | Default | Description |
|-----------|---------|-------------|
| **prompt** | — | Text description of the desired style direction |
| **strength** | 70 | How strongly the style transfer affects the output (0-100) |
| **creativity** | 41 | Creativity level for style interpretation (0-100) |
| **reference_influence** | faithful | `faithful` stays close to the reference style, `creative` allows more interpretation |
| **style_fidelity** | balanced | `balanced` or higher fidelity to the original image |
| **auto_face_detection** | disabled | Toggle auto face detection on/off |
| **enhance_details** | true | Enhance fine details in the output |
