The **Magnific Image Style Transfer** node transfers the artistic style, color palette, texture, and composition feel of a reference image onto your source image. You can guide the result with a text prompt and fine-tune the balance between style influence and content preservation.

### Steps to run

1. In **Load Image** (first), load the source image you want to restyle
2. In **Load Image** (second), load the reference image whose style will be applied
3. In **Magnific Image Style Transfer**, configure the **prompt** describing the desired output
4. Adjust **content_weight** (how much of the source image content to preserve) and **style_weight** (how strongly the reference style is applied)
5. Choose **style_fidelity** (`faithful` or `creative`) to control how closely the node follows the reference style
6. Select **text_fidelity** (`balanced` or `strict`) for prompt adherence
7. Optionally enable **preserve_faces** to keep facial features intact
8. Click **Run** or press `Ctrl(cmd) + Enter`

### Widgets reference

| Widget | Type | Default | Description |
| :----- | :--- | :------ | :---------- |
| prompt | string | — | Text description guiding the style transfer output |
| content_weight | integer | 70 | How much of the source image content to preserve (0–100) |
| style_weight | integer | 41 | How strongly the reference image's style is applied (0–100) |
| style_fidelity | dropdown | `faithful` | `faithful` — closely follows reference style; `creative` — more artistic interpretation |
| text_fidelity | dropdown | `balanced` | `balanced` — moderate prompt following; `strict` — tighter prompt adherence |
| preserve_faces | boolean | false | Whether to preserve facial features unchanged |
| use_original_color | boolean | true | Maintain original colors from the source image |

### Additional notes

- **prompt** — Describe the artistic direction. For example: "Fantasy portrait, retro anime hand-painted style, muted warm color palette, visible brushstroke texture"
- **content_weight vs style_weight** — Higher content_weight keeps more of the original image's structure; higher style_weight applies more of the reference style
- **style_fidelity** — Use `faithful` when you want to match a specific artistic reference closely; use `creative` for more expressive results
- **use_original_color** — When enabled, the reference style is applied to shapes and textures but the original color scheme is retained
