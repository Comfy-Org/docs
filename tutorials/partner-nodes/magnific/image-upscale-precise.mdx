The **Magnific Image Upscale Precise** node upscales images at 2× with maximum fidelity to the original. It offers fine-grained controls for noise reduction, blur reduction, and detail enhancement.

### Steps to run the workflow

1. In **Load Image**, load the image you want to upscale
2. In **Magnific Image Upscale Precise**, select the **model type** (e.g. `photo` for photographic images)
3. Adjust **noise reduction** (1-10), **blur reduction** (1-10), and **detail enhancement** (0-100)
4. Enable **face enhancement** as needed
5. Click **Run** or press `Ctrl(cmd) + Enter`
6. Compare the result with the original using **Image Compare**, or save with **Save Image**

### Parameters

| Parameter | Default | Description |
|-----------|---------|-------------|
| **upscale_factor** | 2× | Scale factor (fixed at 2× for precise upscaling) |
| **model_type** | photo | Type of image for optimal processing: `photo`, `art`, `illustration`, etc. |
| **noise_reduction** | 7 | Noise reduction strength (1-10) |
| **blur_reduction** | 7 | Deblurring strength (1-10) |
| **detail** | 30 | Detail enhancement level (0-100) |
| **enhance_face** | true | Enable face enhancement |
