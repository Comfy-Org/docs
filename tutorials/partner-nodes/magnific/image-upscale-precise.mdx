The **Magnific Image Upscale Precise** (V2) node upscales images with a focus on fidelity. It preserves the original character, texture, and detail of the image while increasing resolution, making it ideal for photographs and realistic imagery where accuracy matters.

### Steps to run

1. In **Load Image**, load the image you want to upscale
2. In **Magnific Image Upscale Precise V2**, select the **upscale_factor** (e.g., `2x`)
3. Choose the **type** matching your image content (`photo`, `art`, or `general`)
4. Adjust **resemblance** (how closely the result matches the original) and **resolution** (detail refinement)
5. Set **sharpen** to control edge sharpness
6. Optionally enable **add_grain** for natural film-like texture
7. Click **Run** or press `Ctrl(cmd) + Enter`

### Widgets reference

| Widget | Type | Default | Description |
| :----- | :--- | :------ | :---------- |
| upscale_factor | dropdown | `2x` | Magnification factor (e.g., `2x`, `4x`) |
| type | dropdown | `photo` | Image type for optimized processing: `photo`, `art`, `general` |
| resemblance | integer | 7 | Fidelity to the original image (0–10) |
| resolution | integer | 7 | Detail resolution refinement (0–10) |
| sharpen | integer | 30 | Sharpening strength (0–100) |
| add_grain | boolean | true | Add natural film grain to the output |

### Additional notes

- **type** — Use `photo` for photographs, `art` for illustrations/paintings, `general` for mixed content
- **resemblance** — Higher values keep the upscaled result truer to the original. Lower values allow more AI interpretation
- **add_grain** — Adds subtle grain texture that can make upscaled images look more natural, especially useful for photographs
- The **ImageCompare** node in the template lets you preview the before/after side by side
