The **Magnific Image Upscale Creative** node upscales images while adding generative detail guided by a creative prompt. It supports up to 4× upscaling with configurable creativity, fidelity, and guidance parameters.

### Steps to run the workflow

1. In **Load Image**, load the image you want to upscale
2. In **Magnific Image Upscale Creative**, set a **prompt** describing the expected content of the output (e.g. "A photo from the 1950s in New York City of a woman sitting casually at the bus stand")
3. Choose the **upscale factor** (2×, 3×, or 4×) and **model** (standard)
4. Adjust **creativity** (1-10), **fidelity** (1-10), **guidance** (1-10), and **detail** level
5. Set **similarity mode** (automatic, creative, faithful) and enable **face enhancement** as needed
6. Click **Run** or press `Ctrl(cmd) + Enter`
7. Compare the result with the original using **Image Compare**, or save with **Save Image**

### Parameters

| Parameter | Default | Description |
|-----------|---------|-------------|
| **prompt** | — | Text prompt describing the image content and desired details |
| **upscale_factor** | 4× | Scale factor: 2×, 3×, or 4× |
| **model** | standard | Upscaling model variant |
| **creativity** | 2 | How much new detail the model can invent (1-10) |
| **fidelity** | 3 | How closely to preserve the original image (1-10) |
| **guidance** | 1 | How strongly the prompt guides detail generation (1-10) |
| **detail** | 0 | Detail enhancement level (0 = no enhancement) |
| **similarity** | automatic | `automatic`, `creative`, or `faithful` mode |
| **enhance_face** | true | Enable face enhancement |
