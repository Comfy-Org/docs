### Steps to run the workflow

1. In **Load Image**, load a portrait or close-up image
2. In **Magnific Skin Enhancer**, set the **smoothing** level (default 0) and **denoise** level (default 15)
3. Choose the **mode**: `creative` for more transformative enhancement or `natural` for subtle refinement
4. Click **Run** or press `Ctrl(cmd) + Enter`
5. Compare the before-and-after result using **Image Compare**

### Parameters

| Parameter | Default | Description |
|-----------|---------|-------------|
| **smoothing** | 0 | Skin smoothing strength (0-100). 0 = no smoothing |
| **denoise** | 15 | Noise reduction strength for skin areas (0-100) |
| **mode** | creative | `creative` for more visible enhancement, `natural` for subtle refinement |
