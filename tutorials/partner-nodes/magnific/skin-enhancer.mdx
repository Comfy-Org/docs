### Steps to run

1. In **Load Image**, load the portrait or close-up image to enhance
2. In **Magnific Skin Enhancer**, adjust **strength** (0–100) to control the intensity
3. Set **detail** (0–100) to control how much skin texture detail is preserved
4. Choose the **mode** (`creative`, `faithful`, or `flexible`)
5. Click **Run** or press `Ctrl(cmd) + Enter`
6. View the before/after comparison in the **ImageCompare** node

### Widgets reference

| Widget | Type | Default | Description |
| :----- | :--- | :------ | :---------- |
| strength | integer | 0 | How strongly the skin enhancement is applied (0–100). Start low for subtle retouching |
| detail | integer | 15 | How much original skin texture detail to retain (0–100). Higher values keep more natural texture |
| mode | dropdown | `creative` | `creative` — more aggressive smoothing; `faithful` — minimal, natural-looking correction; `flexible` — balanced between the two |

### Additional notes

- **strength** — Start with lower values (10–30) for subtle retouching. Increase for more aggressive correction
- **detail** — Higher values preserve more of the original skin texture, which is important for realistic results. Lower values give a smoother, more airbrushed look
- **mode**:
  - `creative` — best for portrait photography where some artistic smoothing is desired (most cost-effective)
  - `faithful` — best for editorial or commercial work where preserving natural skin structure is critical
  - `flexible` — a middle ground between creative and faithful
- Pricing per mode: `creative` 61.19 credits, `faithful` 78.07 credits, `flexible` 94.95 credits
