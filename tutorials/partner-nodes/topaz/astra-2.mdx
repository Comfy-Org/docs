Astra 2 is Topaz Labs' next-generation creative diffusion-based video upscaler, available as a partner node in ComfyUI. Building on Astra 1, it adds expanded control over detail generation and stylization, and transforms AI-generated footage by introducing new dynamic detail and texture with adjustable creativity, sharpness, and prompt-based guidance.

Astra 2 performs best on stylized and GenAI sources, as well as wide shots and large-scale scenes where additional detail can be creatively inferred. It enhances clarity, richness, and visual complexity, and is optimized for expressive, stylized workflows.

## Key capabilities

- **Creative detail synthesis** вЂ” Adds new texture and detail to GenAI sources rather than only sharpening existing pixels.
- **Expanded control** вЂ” Adjustable creativity and sharpness levels, plus optional prompt-based guidance for precise creative direction.
- **Strong on stylized and wide-shot content** вЂ” Optimized for stylized, GenAI, and detail-sparse scenes.

## Node parameters

| Parameter | Type | Required | Description |
| --- | --- | --- | --- |
| `creativity` | string (`low` \| `middle` \| `high`) | Yes | Strength of creative detail generation. |
| `prompt` | string | No | Prompt-based guidance for creative direction. |
| `input_frame_count` | integer | No | Maximum 9000 without a prompt; 450 with a prompt. |
| `input_frame_rate` | integer | No | Frame rate of the input video. |
| `input_height` | integer | No | Height of the input video. |
| `input_width` | integer | No | Width of the input video. |
| `output_height` | integer | No | Target output height. |
| `output_width` | integer | No | Target output width. |
| `realism` | decimal (`0.0`вЂ“`1.0`) | No | Realism control. |
| `sharp` | decimal (`0.0`вЂ“`1.0`) | No | Pre-enhance sharpness, default `0.5`. `0.0` applies a Gaussian blur, `0.5` is passthrough, `1.0` applies USM sharpening. |
