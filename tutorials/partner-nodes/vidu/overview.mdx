Vidu is a state-of-the-art AI video generation platform now available as ComfyUI Partner Nodes. With 11 workflow templates covering three generations (Q1, Q2, and Q3), Vidu offers text-to-video, image-to-video, reference-based video, start-end frame animation, and video extension capabilities.

## Vidu Generations

Vidu's capabilities are organized into three generation tiers, each offering different levels of quality, speed, and features:

### Vidu Q1 (Base Generation)

The foundational generation tier. Q1 provides reliable video generation with solid quality across multiple modes. Available nodes:

- **ViduTextToVideoNode** — Generate video from text prompts
- **ViduImageToVideoNode** — Animate a still image into video
- **ViduReferenceVideoNode** — Generate videos using multiple reference images for consistent character/scene elements
- **ViduStartEndToVideoNode** — Create smooth transitions between a start image and end image
- **ViduExtendVideoNode** — Extend a video by adding new content with a target end frame

The model parameter for Q1 nodes uses `viduq1`.

### Vidu Q2 (Fast Generation)

The second generation focused on speed and efficiency while maintaining high quality. Q2 introduces a new node naming convention (`Vidu2*`) and supports optimized models:

- **Vidu2TextToVideoNode** — Fast text-to-video generation
- **Vidu2ImageToVideoNode** — Fast image-to-video generation
- **Vidu2StartEndToVideoNode** — Fast first-last-frame to video generation
- **Vidu2ReferenceVideoNode** — Multi-subject reference video generation with named subject slots

Available models for Q2 include `viduq2`, `viduq2-pro`, `viduq2-pro-fast`, and more. Q2 nodes accept additional parameters like audio generation and detailed subject references.

### Vidu Q3 (Latest Generation)

The newest and most advanced generation. Q3 brings enhanced quality, cinematic control, and advanced capabilities:

- **Vidu3TextToVideoNode** — Highest quality text-to-video with multi-shot cinematic scene descriptions
- **Vidu3ImageToVideoNode** — Advanced image-to-video with scene transitions and audio support

Available models for Q3 include `viduq3-pro`. Q3 nodes support extended duration (up to 10 seconds), optional audio generation, and cinematic multi-shot prompts.

## Getting Started

To use Vidu Partner Nodes:

1. Ensure you have a ComfyUI account and are logged in
2. Make sure you're in a permitted network environment
3. Download a workflow template and drag it into ComfyUI
4. Adjust the prompt and parameters as needed
5. Click "Run" or use `Ctrl+Enter` to generate

All Vidu workflow templates are available in the [Workflow Templates](/interface/features/template) section.

## Model Parameters

Each Vidu generation supports different model parameters. Below is a quick reference:

| Generation | Model Values | Max Duration | Audio Support | Resolution |
|------------|-------------|-------------|---------------|------------|
| Q1 | `viduq1` | 5 seconds | No | 720p, 1080p |
| Q2 | `viduq2`, `viduq2-pro`, `viduq2-pro-fast`, etc. | 5 seconds | Yes (Q2 Reference) | 720p |
| Q3 | `viduq3-pro` | 6-10 seconds | Yes | 720p |
