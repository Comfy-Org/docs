Reve is now available inside ComfyUI via Partner Nodes. This model has a strong built-in aesthetic and can generate dynamic, high-quality images at up to **4K resolution**. The Create node offers fast generation times, allowing you to quickly iterate and tweak outputs using the Remix and Edit nodes.

## About Reve

**Key features:**
- **Strong built-in aesthetic**: Trained-in aesthetics produce high-quality results with fashion editorial, cinematic, and natural lighting styles
- **Up to 4K resolution**: Generate images with clarity, sharpness, and fine details at high resolutions
- **Fast generation**: Quick iteration times with the Create node
- **Image editing**: Add movement, change lighting conditions, and adjust color grading with the Edit node
- **Image remixing**: Use elements from reference images as layers to remix into a target image while maintaining consistency

## Get started

1. **Update** ComfyUI to the latest version or visit [Comfy Cloud](https://cloud.comfy.org/?utm_source=docs).
2. **Find** the Reve Image Create, Reve Image Remix, and Reve Image Edit nodes in the Node Library or Templates in the left sidebar.
3. **Drop a node** on your canvas and start creating.

## Available nodes

| Node | Type | Description |
|------|------|-------------|
| **Reve Image Create** | Text-to-Image | Generate high-quality images at up to 4K resolution with strong built-in aesthetics |
| **Reve Image Edit** | Image Edit | Edit lighting, color grading, and add movement to existing images |
| **Reve Image Remix** | Image Remix | Use reference images as layers to remix into a target image while maintaining consistency |
