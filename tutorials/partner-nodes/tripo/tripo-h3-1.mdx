We're excited to announce support for **Tripo H3.1** in ComfyUI via Partner Nodes. You can now generate high-detail 3D assets directly in your existing ComfyUI workflows, with better geometry density and surface quality for close-up production use.

Built for hero-level assets, Tripo H3.1 helps creators move from concept to usable models faster, whether the target is game production, marketing visuals, product rendering, or 3D printing pipelines.

## What's New with Tripo H3.1

Tripo H3.1 delivers the following quality improvements over previous Tripo model versions:

- **High-density geometry** — Sharper edges, cleaner silhouettes, and more precise forms for production-level 3D assets.
- **PBR-ready material behavior** — More reliable lighting response across different render environments.
- **Hero asset quality** — Stable detail retention in close-up shots, suitable for promotional key art and hero-level assets.
- **Cross-scenario reuse** — One model can work across game, marketing, and render pipelines without regeneration.

## Typical Use Cases

- **Game hero assets** — For close-up camera shots and promotional key art.
- **Marketing visuals** — Where geometric fidelity and material quality are critical.
- **Product visualization** — Cleaner shape definition in studio-style renders.
- **3D printing preparation** — Better structural clarity for improved handoff quality.

## How It Works

Tripo H3.1 is available as a model option in the existing Tripo Partner Nodes. You can use it with any Tripo generation workflow (Text-to-Model, Image-to-Model, Multi-view to Model) by selecting `v3.1` as the model version in the Tripo node parameters.

### Supported Nodes

Tripo H3.1 works with all existing Tripo generation nodes:

- **Tripo: Text to Model** — Generate 3D models from text prompts.
- **Tripo: Image to Model** — Generate 3D models from a single reference image.
- **Tripo: Multiview to Model** — Generate 3D models from multiple view images.
- **Tripo: Refine Draft model** — Refine preliminary model drafts (compatible with v3.1 and above).
- **Tripo: Rig** — Add skeleton rigging to generated models.
- **Tripo: Retarget Rigged Model** — Retarget animations to rigged models.
- **Tripo: Texture** — Add or modify textures on generated models.
- **Tripo: Conversion** — Convert between 3D formats.

## Get Started in ComfyUI

1. **Update ComfyUI** to the latest version.
2. Open the **Template Library** and search for "Tripo".
3. Select a Tripo workflow template (e.g., **Tripo3.0: Text to Model** or **Tripo3.0: Image to Model**).
4. In the Tripo node's `model` parameter, select `v3.1`.
5. Edit your prompt or reference image, then run the workflow.

### What's Different from Previous Versions

| Aspect | Tripo H3.1 | Previous Versions |
|--------|-----------|-------------------|
| Geometry Density | High-density, production-ready | Standard density |
| Surface Quality | Cleaner edges and silhouettes | Good, but may require polish for close-ups |
| Material Output | PBR-ready | Standard material maps |
| Refine Support | Supported | Supported (v1.4 only for Refine Draft) |

## Preview and Export

After generation, you can preview the model directly in ComfyUI:

1. The model output is connected to the `Preview 3D` node.
2. Click on the node to expand the preview menu.
3. Select **Export** to download the model in your preferred format.

For local users, the output file is saved at `ComfyUI/output/3d/`.
