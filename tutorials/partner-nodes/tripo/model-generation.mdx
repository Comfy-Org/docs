Tripo AI is a company focused on generative AI 3D modeling. It provides user-friendly platforms and API services that can quickly convert text prompts or 2D images (single or multiple) into high-quality 3D models.
ComfyUI has now natively integrated the corresponding Tripo API, allowing you to conveniently use the related nodes in ComfyUI for model generation.

Currently, ComfyUI's Partner nodes support the following Tripo model generation capabilities:
- Text-to-model
- Image-to-model
- Multi-view model generation
- Rig model
- Retarget rigged model

## What's New in Tripo H3.1

Tripo H3.1 represents a significant upgrade in model quality, designed for hero-level 3D asset production. With higher detail fidelity and improved geometry, it helps creators go from concept to production-ready models faster, whether the target is game development, marketing visuals, product rendering, or 3D printing.

Key improvements:

- **High-density geometry** — Sharper edges, cleaner silhouettes, and more precise forms that hold up in close-up shots.
- **PBR-ready material behavior** — More reliable lighting response across different render environments.
- **Hero asset quality** — Stable detail in close-up camera shots, suitable for promotional key art and hero assets.
- **Cross-scenario reuse** — One model can work across gameplay, marketing, and render pipelines.

### Typical Use Cases

- **Game hero assets** — For close camera shots and promotional key art.
- **Marketing visuals** — Where geometric fidelity and material quality are critical.
- **Product visualization** — Cleaner shape definition in studio-style renders.
- **3D printing preparation** — Better structural clarity for improved handoff quality.

To get started with Tripo H3.1:
1. Open the **Template Library** in ComfyUI and search for "Tripo".
2. Select the **Tripo3.0: Text to Model** or **Tripo3.0: Image to Model** workflow template.
3. In the Tripo node's `model` parameter, select `v3.1`.
4. Enter your prompt/reference image and run the workflow.

### 2. Complete the Workflow Execution Step by Step

![ComfyUI Tripo Text to Model Step Guide](/images/tutorial/api_nodes/tripo/tripo_text_to_model_step_guide.jpg)

You can refer to the numbers in the image to complete the basic text-to-model workflow execution:  
1. In the `Tripo: Text to Model` node, input your prompt in the `prompt` field
   - **model**: You can select different models. The latest **v3.1** provides high-density geometry and PBR-ready materials for production use. Select the model version that best fits your needs:
     - `v3.1` — Latest generation with high-density geometry, PBR-ready materials, and hero asset quality (recommended for new projects)
     - `v3.0-20250812` — Previous generation with strong baseline quality
     - `v2.5` and `v2.0` — Earlier versions for backward compatibility
     - `v1.4` — Legacy model, supports `Tripo: Refine Draft model` post-processing
   - **style**: You can set different styles
   - **texture_quality**: You can set different texture qualities
2. Click the `Run` button, or use the shortcut `Ctrl(cmd) + Enter` to execute model generation. After the workflow completes, the corresponding model will be automatically saved to the `ComfyUI/output/` directory
3. In the `Preview 3D` node, click to expand the menu
4. Select `Export` to directly export the corresponding model
