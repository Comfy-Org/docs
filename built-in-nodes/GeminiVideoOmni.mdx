This node generates a video with audio from a text prompt using Google's Gemini Omni Flash model. You can optionally provide reference images or videos to guide or edit the result.

## Inputs

| Parameter | Description | Data Type | Required | Range |
|-----------|-------------|-----------|----------|-------|
| `model` | The Gemini video model used to generate the video. | COMBO | Yes | `"Omni Flash"` |
| `seed` | Seed controls whether the node should re-run; results are non-deterministic regardless of seed. (default: 42) | INT | Yes | 0 to 2147483647 |

**Note:** The `model` parameter is a dynamic combo that expands to include additional inputs when "Omni Flash" is selected. These additional inputs include:
- `prompt` (STRING, required): The text prompt describing the desired video. Describe the desired length (3-10 seconds) and aspect ratio (16:9 or 9:16) directly in the prompt.
- `images` (IMAGE, optional): Reference images to guide the video generation. Maximum of 14 images.
- `videos` (VIDEO, optional): Reference videos to guide the video generation. Maximum of 3 videos, each with a maximum duration of 10 seconds.
- `temperature` (FLOAT, optional, default: 1.0): Controls randomness in generation.
- `top_p` (FLOAT, optional, default: 0.95): Controls nucleus sampling.

## Outputs

| Output Name | Description | Data Type |
|-------------|-------------|-----------|
| `video` | The generated video with audio. | VIDEO |
| `text` | The text response from the model, if any. | STRING |
