The EmptyCosmosLatentVideo node creates an empty latent video tensor with specified dimensions. It generates a zero-filled latent representation that can be used as a starting point for video generation workflows, with configurable width, height, length, and batch size parameters.

## Inputs

| Parameter | Description | Data Type | Required | Range |
| --- | --- | --- | --- | --- |
| `width` | The width of the latent video in pixels (default: 1280, must be divisible by 16) | INT | Yes | 16 to MAX_RESOLUTION |
| `height` | The height of the latent video in pixels (default: 704, must be divisible by 16) | INT | Yes | 16 to MAX_RESOLUTION |
| `length` | The number of frames in the latent video (default: 121, must be divisible by 8) | INT | Yes | 1 to MAX_RESOLUTION |
| `batch_size` | The number of latent videos to generate in a batch (default: 1) | INT | No | 1 to 4096 |

## Outputs

| Output Name | Description | Data Type |
| --- | --- | --- |
| `samples` | The generated empty latent video tensor with zero values | LATENT |
